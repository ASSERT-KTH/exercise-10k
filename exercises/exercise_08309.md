## Exercise
Write a program that takes in a user's input and determines the data type of the input. The input can be either an integer, float, or string. If the input is an integer or float, print out the data type and the value. If the input is a string, print out the data type, the length of the string, and the value.

## Reference Solution
```python
def determine_data_type(user_input):
    try:
        # Attempt to convert the input to an integer
        integer_input = int(user_input)
        print(f"Data type: Integer")
        print(f"Value: {integer_input}")
    except ValueError:
        try:
            # Attempt to convert the input to a float
            float_input = float(user_input)
            print(f"Data type: Float")
            print(f"Value: {float_input}")
        except ValueError:
            # If the input cannot be converted to an integer or float, it's a string
            print(f"Data type: String")
            print(f"Length: {len(user_input)}")
            print(f"Value: {user_input}")

def main():
    user_input = input("Please enter a value: ")
    determine_data_type(user_input)

if __name__ == "__main__":
    main()
```
