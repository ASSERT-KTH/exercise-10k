## Exercise
Create a function that takes a number as input and prints out whether it's even or odd.

## Reference Solution
```python
def check_even_or_odd(num: int) -> None:
    """
    Prints out whether the input number is even or odd.

    Args:
    num (int): The number to be checked.
    """
    if num % 2 == 0:
        print(f"{num} is even.")
    else:
        print(f"{num} is odd.")

# Example usage:
def main():
    check_even_or_odd(10)  # Output: 10 is even.
    check_even_or_odd(11)  # Output: 11 is odd.

if __name__ == "__main__":
    main()
```
