## Exercise
Write a program that finds the smallest number in a given list of numbers.

## Reference Solution
```python
def find_smallest_number(numbers):
    """
    This function finds the smallest number in a given list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        int: The smallest number in the list.
    """
    if len(numbers) == 0:
        raise ValueError("The list is empty")
    return min(numbers)

# Example usage
numbers = [12, 45, 7, 23, 56, 89, 34]
smallest_number = find_smallest_number(numbers)
print(f"The smallest number in the list is: {smallest_number}")
```
