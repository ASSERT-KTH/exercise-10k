## Exercise
Write a program that takes in a list of values as input and determines the data type of each value. The program should then print out the data type of each value in the list.

## Reference Solution
```python
def determineType(item):
    if isinstance(item, int):
        return "Integer"
    elif isinstance(item, float):
        return "Float"
    elif isinstance(item, str):
        return "String"
    elif isinstance(item, list):
        return "List"
    elif isinstance(item, tuple):
        return "Tuple"
    elif isinstance(item, dict):
        return "Dictionary"
    elif isinstance(item, set):
        return "Set"
    elif isinstance(item, bool):
        return "Boolean"
    else:
        return "Unknown"

def main():
    values = [1, 2.3, "hello", [1, 2, 3], (4, 5, 6), {"a": 1, "b": 2}, {7, 8, 9}, True, None]
    for item in values:
        print(f"{item}: {determineType(item)}")

if __name__ == "__main__":
    main()
```
