## Exercise
Create a function that takes a string and a repeat count as arguments and returns the repeated string.

## Reference Solution
```python
def repeat_string(input_str, repeat_count):
    """
    This function takes a string and a repeat count as arguments and returns the repeated string.

    Args:
        input_str (str): The string to be repeated.
        repeat_count (int): The number of times the string should be repeated.

    Returns:
        str: The repeated string.
    """

    # First, we need to check if the input string is indeed a string
    if not isinstance(input_str, str):
        raise TypeError("Input must be a string.")

    # Then, we need to check if the repeat count is an integer
    if not isinstance(repeat_count, int):
        raise TypeError("Repeat count must be an integer.")

    # Now, we can use Python's built-in string multiplication operator (*) to repeat the string
    # This operator repeats the string on its left side the number of times specified on its right side
    repeated_str = input_str * repeat_count

    # Finally, we return the repeated string
    return repeated_str
```
