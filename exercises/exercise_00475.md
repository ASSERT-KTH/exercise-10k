## Exercise
Write a program that takes a string as input and returns a wave pattern of the string

## Reference Solution
```python
def wave_pattern(s):
    """
    Generates a wave pattern of a given input string.

    Args:
        s (str): The input string.

    Returns:
        list: A list of strings representing the wave pattern.
    """
    result = []
    for i in range(len(s)):
        # Create a new string with the character at index i capitalized
        new_s = s[:i] + s[i].upper() + s[i+1:]
        result.append(new_s)
    return result

def main():
    # Example usage
    input_str = "hello"
    wave = wave_pattern(input_str)
    for i, pattern in enumerate(wave):
        print(f"Pattern {i+1}: {pattern}")

if __name__ == "__main__":
    main()
```
