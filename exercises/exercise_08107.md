## Exercise
Write a program that takes a string as input and returns the number of letters in the string.

## Reference Solution
```python
def count_letters(input_string):
    """
    Returns the number of letters in the input string.

    Args:
        input_string (str): The input string.

    Returns:
        int: The number of letters in the input string.
    """
    return sum(1 for char in input_string if char.isalpha())

def main():
    # Example usage
    input_string = input("Enter a string: ")
    letter_count = count_letters(input_string)
    print(f"The string '{input_string}' contains {letter_count} letters.")

if __name__ == "__main__":
    main()
```
