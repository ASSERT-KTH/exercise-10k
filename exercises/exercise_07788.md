## Exercise
Create a program that takes in a variable and its value as input, and then checks if the given value matches the declared data type. The data types to be validated are integer, float, string, and boolean. If the value matches the data type, the program should print 'Valid', otherwise it should print 'Invalid'. The program should handle potential errors during the validation process.

## Reference Solution
```python
def validation(variable, value):
    try:
        if type(variable) == int:
            if int(value):
                print("Valid")
            else:
                print("Invalid")
        elif type(variable) == float:
            if float(value):
                print("Valid")
            else:
                print("Invalid")
        elif type(variable) == str:
            if str(value):
                print("Valid")
            else:
                print("Invalid")
        elif type(variable) == bool:
            if bool(value):
                print("Valid")
            else:
                print("Invalid")
    except Exception as e:
        print(f"An error occurred: {e}")
        print("Invalid")

# Test the function
validation(10, 10)
validation(10.5, 10.5)
validation("Hello", "Hello")
validation(True, True)
validation("World", 10)
```
