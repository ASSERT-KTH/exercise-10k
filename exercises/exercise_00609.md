## Exercise
Write a program to calculate the absolute value of a given number.

## Reference Solution
```python
def calculate_absolute_value(num):
    """
    Calculate the absolute value of a given number.

    Args:
        num (int or float): The input number.

    Returns:
        int or float: The absolute value of the input number.
    """
    return abs(num)

def main():
    # Example usage
    num = float(input("Enter a number: "))
    absolute_value = calculate_absolute_value(num)
    print("The absolute value of {} is {}".format(num, absolute_value))

if __name__ == "__main__":
    main()
```
