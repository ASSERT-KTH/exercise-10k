## Exercise
Write a program to compare two numbers and display the larger one

## Reference Solution
```python
def compare_numbers(num1, num2):
    """
    Compare two numbers and return the larger one.
    
    Args:
        num1 (float): The first number.
        num2 (float): The second number.
    
    Returns:
        float: The larger number.
    """
    if num1 > num2:
        return num1
    else:
        return num2

def main():
    # Get user input
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))

    # Compare numbers and display the larger one
    larger_num = compare_numbers(num1, num2)
    print(f"The larger number is: {larger_num}")

if __name__ == "__main__":
    main()
```
