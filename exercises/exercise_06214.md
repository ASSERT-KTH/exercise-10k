## Exercise
Write a function that takes an array of integers and an integer as input, and returns a new array where each element is the corresponding element in the input array raised to the power of the given integer.

## Reference Solution
```python
def list_powers(lst, n):
    result = []
    for i in lst:
        if i == 0:
            result.append(1)
        else:
            result.append(i ** n)
    return result
```
