## Exercise
Create a program that calculates the result of a number raised to a power.

## Reference Solution
```python
def calculate_power(base, exponent):
    """
    Calculate the result of a number raised to a power.

    Args:
        base (float): The base number.
        exponent (float): The exponent to which the base is raised.

    Returns:
        float: The result of the power operation.
    """
    return base ** exponent

def main():
    # Example usage
    base = float(input("Enter the base number: "))
    exponent = float(input("Enter the exponent: "))

    result = calculate_power(base, exponent)
    print(f"{base} raised to the power of {exponent} is {result}")

if __name__ == "__main__":
    main()
```
