## Exercise
Write a function to retrieve the value associated with a given key from a dictionary, setting the key to a default value if it is not present

## Reference Solution
```python
def get_or_set(dictionary, key, default_value):
    """
    Retrieves the value associated with a given key from a dictionary.
    If the key is not present, it sets the key to a default value.

    Args:
        dictionary (dict): The dictionary to search in.
        key: The key to search for.
        default_value: The default value to set if the key is not present.

    Returns:
        The value associated with the key, or the default value if the key is not present.
    """
    if key not in dictionary:
        dictionary[key] = default_value
    return dictionary[key]

# Example usage:
my_dict = {"a": 1, "b": 2}
print(get_or_set(my_dict, "a", 0))  # Output: 1
print(get_or_set(my_dict, "c", 0))  # Output: 0
print(my_dict)  # Output: {'a': 1, 'b': 2, 'c': 0}
```

    value = dictionary.get(key)
    if value is None:
        dictionary[key] = default_value
        return default_value
    return value
