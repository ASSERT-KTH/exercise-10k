## Exercise
Create a function named 'most_frequent' that finds and returns the most frequent element in a given list.

## Reference Solution
```python
from collections import Counter

def most_frequent(lst):
    """
    Finds and returns the most frequent element in a given list.

    Args:
        lst (list): The input list.

    Returns:
        The most frequent element in the list.
    """
    if not lst:
        return None  # or raise an exception, depending on your requirements

    counter = Counter(lst)
    return counter.most_common(1)[0][0]

# Example usage
if __name__ == "__main__":
    lst = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]
    most_freq = most_frequent(lst)
    print(f"The most frequent element is: {most_freq}")
```
