## Exercise
Write a program that takes a list as input and processes it, then outputs the result.

## Reference Solution
```python
# Define a function to process the list
def process_list(input_list):
    # Check if the list is empty
    if not input_list:
        return "The list is empty."

    # Calculate the sum of the list elements
    total = sum(input_list)

    # Calculate the average of the list elements
    average = total / len(input_list)

    # Return the result as a string
    return f"Sum: {total}, Average: {average}"

# Main function
def main():
    # Get the list from the user
    input_list = input("Enter a list of numbers separated by spaces: ")

    # Convert the input string to a list of floats
    try:
        input_list = [float(num) for num in input_list.split()]
    except ValueError:
        print("Invalid input. Please enter numbers separated by spaces.")
        return

    # Process the list and print the result
    print(process_list(input_list))

# Run the main function
if __name__ == "__main__":
    main()
```
