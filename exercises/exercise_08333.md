## Exercise
Write a program that reads a string input from the user and checks if it matches a specific format. The format is 'DD/MM/YYYY', where 'DD' represents the day of the month, 'MM' represents the month, and 'YYYY' represents the year. If the input matches the format, the program should print 'Valid date' and the input date. If the input does not match the format, the program should print 'Invalid date' and prompt the user to enter the date again.

## Reference Solution
```python
import re

def main():
    while True:
        date = input("Enter a date (DD/MM/YYYY): ")
        regex_pattern = re.compile(r'^(\d{2})/(\d{2})/(\d{4})$')
        match = regex_pattern.match(date)

        if match:
            day, month, year = match.groups()
            day = int(day)
            month = int(month)
            year = int(year)

            if day > 0 and day <= 31 and month > 0 and month <= 12:
                print("Valid date:", date)
                break
            else:
                print("Invalid date. Please try again.")
        else:
            print("Invalid date. Please try again.")

if __name__ == "__main__":
    main()
```
