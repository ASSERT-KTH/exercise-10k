## Exercise
Create a function that uses map-reduce to calculate the sum of squares of a list of numbers. Then, call this function with a list of input numbers and print the result.

## Reference Solution
```python
def map_func(x):
    return x ** 2

def reduce_func(x, y):
    return x + y

def calculate_sum_squares(numbers):
    squares = list(map(map_func, numbers))
    result = reduce(reduce_func, squares)
    return result

numbers = [1, 2, 3, 4, 5]
result = calculate_sum_squares(numbers)
print(result)
```
