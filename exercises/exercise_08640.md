## Exercise
Write a function that converts a given string to uppercase.

## Reference Solution
```python
def convert_to_uppercase(input_string: str) -> str:
    """
    Converts a given string to uppercase.

    Args:
        input_string (str): The input string to be converted.

    Returns:
        str: The uppercase equivalent of the input string.
    """
    return input_string.upper()

# Example usage
def main():
    input_str = "Hello, World!"
    uppercase_str = convert_to_uppercase(input_str)
    print(f"Uppercase equivalent of '{input_str}': {uppercase_str}")

if __name__ == "__main__":
    main()
```
