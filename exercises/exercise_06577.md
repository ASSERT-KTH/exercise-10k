## Exercise
Create a dictionary and partition it into two separate dictionaries based on a given condition

## Reference Solution
```python
def partition_dictionary(input_dict, condition):
    """
    Partition a dictionary into two separate dictionaries based on a given condition.

    Args:
        input_dict (dict): The dictionary to be partitioned.
        condition (function): A function that takes a key-value pair and returns a boolean.

    Returns:
        tuple: Two dictionaries, the first containing key-value pairs that satisfy the condition,
               and the second containing key-value pairs that do not satisfy the condition.
    """
    dict_true = {key: value for key, value in input_dict.items() if condition(key, value)}
    dict_false = {key: value for key, value in input_dict.items() if not condition(key, value)}
    return dict_true, dict_false


# Example usage:
if __name__ == "__main__":
    # Create a dictionary
    input_dict = {"a": 1, "b": 2, "c": 3, "d": 4, "e": 5}

    # Define a condition (e.g., values greater than 3)
    def condition(key, value):
        return value > 3

    # Partition the dictionary
    dict_true, dict_false = partition_dictionary(input_dict, condition)

    # Print the results
    print("Dictionary with values greater than 3:")
    print(dict_true)
    print("Dictionary with values less than or equal to 3:")
    print(dict_false)
```
