## Exercise
Write a program that takes a list of strings and a separator as input and returns a single string with the strings joined by the separator.

## Reference Solution
```python
def join_strings(strings, separator):
    result = ""
    for i, string in enumerate(strings):
        result += string
        if i < len(strings) - 1:
            result += separator
    return result

# Example usage:
strings = ["Hello", "world", "this", "is", "a", "test"]
separator = ", "
result = join_strings(strings, separator)
print(result)
```
