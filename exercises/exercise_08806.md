## Exercise
Write a program that takes a value as input and returns its hyperbolic sine, cosine, and tangent.

## Reference Solution
```python
import math

def calculate_hyperbolic_functions(x):
    """
    Calculate the hyperbolic sine, cosine, and tangent of a given value.

    Args:
        x (float): The input value.

    Returns:
        tuple: A tuple containing the hyperbolic sine, cosine, and tangent of the input value.
    """
    # Calculate the hyperbolic sine
    sinh = (math.exp(x) - math.exp(-x)) / 2
    
    # Calculate the hyperbolic cosine
    cosh = (math.exp(x) + math.exp(-x)) / 2
    
    # Calculate the hyperbolic tangent
    if cosh == 0:
        raise ZeroDivisionError("Hyperbolic cosine cannot be zero.")
    else:
        tanh = sinh / cosh
    
    return sinh, cosh, tanh

def main():
    # Take the input value from the user
    x = float(input("Enter a value: "))

    # Calculate the hyperbolic functions
    sinh, cosh, tanh = calculate_hyperbolic_functions(x)

    # Print the results
    print(f"Hyperbolic sine: {sinh}")
    print(f"Hyperbolic cosine: {cosh}")
    print(f"Hyperbolic tangent: {tanh}")

if __name__ == "__main__":
    main()
```
