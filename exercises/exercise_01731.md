## Exercise
Write a program that takes a list of numbers and a target value as input, and returns a new list with the first occurrence of the target value removed.

## Reference Solution
```python
def solution(nums, val):
    nums.remove(val)
    return nums

# Example usage:
nums = [0,1,9,3,12]
val = 9
print(solution(nums, val))  # Output: [0, 1, 3, 12]
```
