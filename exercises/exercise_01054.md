## Exercise
Write a program that takes a list and a modification function as input, and returns the modified list.

## Reference Solution
```python
def modify_list(input_list, modification_function):
    """
    Applies a modification function to each element in the input list.

    Args:
        input_list (list): The input list to be modified.
        modification_function (function): A function that takes one argument and returns a modified value.

    Returns:
        list: The modified list.
    """
    return [modification_function(element) for element in input_list]

# Example usage:
def double_value(x):
    """Doubles the input value."""
    return x * 2

numbers = [1, 2, 3, 4, 5]
modified_numbers = modify_list(numbers, double_value)
print(modified_numbers)  # Output: [2, 4, 6, 8, 10]
```
