## Exercise
Write a program that asks for a sentence and prints out the last word.

## Reference Solution
```python
def get_last_word(sentence):
    """Returns the last word in a sentence."""
    words = sentence.split()
    if words:
        return words[-1]
    else:
        return None

def main():
    sentence = input("Please enter a sentence: ")
    last_word = get_last_word(sentence)
    if last_word:
        print("The last word is:", last_word)
    else:
        print("The sentence is empty.")

if __name__ == "__main__":
    main()
```
