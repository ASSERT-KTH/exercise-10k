## Exercise
Write a program that takes two numbers as input and returns the maximum of the two

## Reference Solution
```python
def get_max(a, b):
    """
    Returns the maximum of two numbers.

    Args:
        a (float): The first number.
        b (float): The second number.

    Returns:
        float: The maximum of a and b.
    """
    return max(a, b)

def main():
    # Get input from user
    a = float(input("Enter the first number: "))
    b = float(input("Enter the second number: "))

    # Calculate and print the maximum
    max_num = get_max(a, b)
    print(f"The maximum of {a} and {b} is: {max_num}")

if __name__ == "__main__":
    main()
```
