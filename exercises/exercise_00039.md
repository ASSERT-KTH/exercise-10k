## Exercise
Create a function that takes a string as input and returns True if it's a valid date in the format YYYY-MM-DD, False otherwise.

## Reference Solution
```python
from datetime import datetime

def is_valid_date(date_string):
    """
    Checks if a given string is a valid date in the format YYYY-MM-DD.

    Args:
        date_string (str): The input date string.

    Returns:
        bool: True if the date string is valid, False otherwise.
    """

    # First, we check if the input is a string. If it's not, we immediately return False.
    if not isinstance(date_string, str):
        return False

    # We define the expected date format.
    # This will be used to parse the input date string.
    date_format = "%Y-%m-%d"

    # We attempt to parse the date string using the defined format.
    try:
        # If the parsing is successful, it means the date string is valid.
        # We use the strptime function from the datetime module for this.
        datetime.strptime(date_string, date_format)
        
        # If no exception was raised, the date string is valid, so we return True.
        return True
    
    # If the parsing fails, a ValueError exception will be raised.
    except ValueError:
        # In this case, we return False, indicating that the date string is invalid.
        return False

# Example usage:
print(is_valid_date("2022-09-01"))  # Output: True
print(is_valid_date("2022-13-01"))  # Output: False
print(is_valid_date("2022-09-32"))  # Output: False
print(is_valid_date("2022-9-1"))    # Output: False
```
