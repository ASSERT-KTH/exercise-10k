## Exercise
Create a dictionary with 3 key-value pairs and map a function to the values.

## Reference Solution
```python
# Define a function to be mapped to the dictionary values
def square_value(value):
    """Returns the square of the input value."""
    return value ** 2

# Create a dictionary with 3 key-value pairs
data_dict = {
    'a': 1,
    'b': 2,
    'c': 3
}

# Map the function to the dictionary values
squared_values = {key: square_value(value) for key, value in data_dict.items()}

# Print the original dictionary and the resulting dictionary with squared values
print("Original Dictionary:")
print(data_dict)
print("\nDictionary with Squared Values:")
print(squared_values)
```
