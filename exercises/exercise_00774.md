## Exercise
Write a program that extracts an integer from a given string and prints out the integer.

## Reference Solution
```python
import re

def extract_integer(input_string):
    # Regular expression to match integers
    pattern = r'\d+'
    
    # Find all matches of the pattern in the string
    matches = re.findall(pattern, input_string)
    
    # Convert matches to integers and print them
    for match in matches:
        print(int(match))

# Test the function
input_string = "abc123def456ghi789"
extract_integer(input_string)
```
