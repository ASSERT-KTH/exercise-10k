## Exercise
Write a program that takes a user's input and determines the data type of the input. The input can be either an integer, a float, or a string. The program should then print out the data type of the input.

## Reference Solution
```python
# Enter your code here. Read input from STDIN. Print output to STDOUT

user_input = input()

try:
    user_input = int(user_input)
    print('This is an integer.')
except ValueError:
    try:
        user_input = float(user_input)
        print('This is a float.')
    except ValueError:
        print('This is a string.')
```
