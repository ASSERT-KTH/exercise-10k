## Exercise
Write a function that uses the zip function to pair corresponding elements from two lists. Use the function to pair the elements from the lists [1, 2, 3] and ['a', 'b', 'c'].

## Reference Solution
```python
def pair_elements(list1, list2):
    """
    Pair corresponding elements from two lists.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: A list of pairs, where each pair contains one element from list1 and one element from list2.
    """
    return list(zip(list1, list2))

# Example usage:
list1 = [1, 2, 3]
list2 = ['a', 'b', 'c']

paired_elements = pair_elements(list1, list2)
print(paired_elements)
```
