## Exercise
Create a program that takes two numbers and an operator as input and performs the corresponding arithmetic operation. The operator can be '+', '-', '*', or '/'. If the operator is '/' and the second number is 0, the program should display an error message.

## Reference Solution
```python
#include <iostream>
using namespace std;

int main() {
    int n1, n2;
    char op;

    cout << "Enter the first number: ";
    cin >> n1;
    cout << "Enter the operator (+, -, *, /): ";
    cin >> op;
    cout << "Enter the second number: ";
    cin >> n2;

    switch (op) {
        case '+':
            cout << "Result: " << n1 + n2 << endl;
            break;
        case '-':
            cout << "Result: " << n1 - n2 << endl;
            break;
        case '*':
            cout << "Result: " << n1 * n2 << endl;
            break;
        case '/':
            if (n2 == 0) {
                cout << "Error: Division by zero is not allowed!" << endl;
            } else {
                cout << "Result: " << (double)n1 / n2 << endl;
            }
            break;
        default:
            cout << "Error: Invalid operator!" << endl;
            break;
    }

    return 0;
}
```
