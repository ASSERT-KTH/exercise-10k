## Exercise
Understand the boolean data type by creating a variable and assigning true or false values

## Reference Solution
```python
# Create a boolean variable and assign it a value
is_admin = True
print(is_admin)  # Output: True

# Assign a false value to the variable
is_admin = False
print(is_admin)  # Output: False
```

```python
# Boolean AND operation
is_admin = True
is_logged_in = True
print(is_admin and is_logged_in)  # Output: True

# Boolean OR operation
is_admin = True
is_logged_in = False
print(is_admin or is_logged_in)  # Output: True

# Boolean NOT operation
is_admin = True
print(not is_admin)  # Output: False
```

```python
# Example use case: Conditional statement

if is_logged_in:
    print("Welcome, user!")
else:
    print("Please log in to continue.")
```
