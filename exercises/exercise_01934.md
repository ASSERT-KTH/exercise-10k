## Exercise
Write a program that declares a bytes variable and prints its elements.

## Reference Solution
```python
# Declare a bytes variable
byte_variable = b'Hello, World!'

# Print the bytes variable
print("Bytes Variable:")
print(byte_variable)

# Print the elements of the bytes variable
print("\nElements of Bytes Variable:")
for i, byte in enumerate(byte_variable):
    print(f"Index {i}: {byte}")
```
