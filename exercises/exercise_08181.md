## Exercise
Write a program that calculates the sum of all numbers from 1 to n using a for loop.

## Reference Solution
```python
def calculate_sum(n):
    total_sum = 0
    for i in range(1, n + 1):
        total_sum += i
    return total_sum

n = int(input("Enter a number: "))
result = calculate_sum(n)
print("The sum of numbers from 1 to", n, "is:", result)
```
