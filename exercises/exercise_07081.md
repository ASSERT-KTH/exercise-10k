## Exercise
Read a positive integer from the user and check if it is even or odd

## Reference Solution
```python
def check_even_or_odd():
    """
    Reads a positive integer from the user and checks if it is even or odd.
    """
    while True:
        try:
            num = int(input("Enter a positive integer: "))
            if num <= 0:
                print("Please enter a positive integer.")
            else:
                break
        except ValueError:
            print("Invalid input. Please enter a positive integer.")

    if num % 2 == 0:
        print(f"{num} is even.")
    else:
        print(f"{num} is odd.")

# Example usage
if __name__ == "__main__":
    check_even_or_odd()
```
