## Exercise
Create a program that takes a user's input and determines the data type of the input. The program should be able to identify integers, floats, and strings. It should also handle cases where the input cannot be classified into any of these categories.

## Reference Solution
```python
def determine_data_type(input_str):
    """
    Attempts to determine the data type of the input.
    
    Args:
        input_str (str): The user's input as a string.
    
    Returns:
        str: A string describing the data type of the input.
    """

    # First, we try to convert the input to an integer
    try:
        # If this succeeds, the input is an integer
        int(input_str)
        return "Integer"
    
    # If the above fails, we try to convert the input to a float
    except ValueError:
        try:
            # If this succeeds, the input is a float
            float(input_str)
            return "Float"
        
        # If the above fails, the input is neither an integer nor a float
        except ValueError:
            # In this case, we return "String" because the input can be treated as a string
            # This will catch any input that is not a number, including non-numeric characters and special characters
            return "String"

def main():
    # Get the user's input
    user_input = input("Please enter a value: ")
    
    # Determine the data type of the input
    data_type = determine_data_type(user_input)
    
    # Print the result
    print(f"The data type of your input is: {data_type}")

if __name__ == "__main__":
    main()
```
