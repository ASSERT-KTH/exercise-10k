## Exercise
Write a program that takes an array and a target element as input and returns the index of the target element if it exists in the array.

## Reference Solution
```python
def find_index(arr, x):
    try:
        idx = arr.index(x)
    except ValueError:
        idx = -1
    return idx

arr = [1, 2, 3, 4, 5, 6, 7, 8, 9]
x = 5
idx = find_index(arr, x)
print(idx)  # Output: 4
```
