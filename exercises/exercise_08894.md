## Exercise
Write a program that takes the length and width of a rectangle as input and returns the length of its diagonal.

## Reference Solution
```python
import math

def diagonal(length, width):
    """
    Calculate the length of the diagonal of a rectangle.

    Parameters:
    length (float): The length of the rectangle.
    width (float): The width of the rectangle.

    Returns:
    float: The length of the diagonal.
    """
    return math.sqrt(length ** 2 + width ** 2)

def main():
    length = float(input("Enter the length: "))
    width = float(input("Enter the width: "))
    print("The length of the diagonal is: ", diagonal(length, width))

if __name__ == "__main__":
    main()
```
