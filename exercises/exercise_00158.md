## Exercise
Create a function that takes in two dictionaries and returns a new dictionary containing the key-value pairs that exist in the first dictionary but not in the second

## Reference Solution
```python
def get_unique_pairs(dict1, dict2):
    """
    Returns a new dictionary containing key-value pairs that exist in dict1 but not in dict2.

    Args:
        dict1 (dict): The dictionary to get unique pairs from.
        dict2 (dict): The dictionary to compare against.

    Returns:
        dict: A new dictionary with unique key-value pairs.
    """
    return {key: value for key, value in dict1.items() if key not in dict2 or dict2[key] != value}

# Example usage
dict1 = {'a': 1, 'b': 2, 'c': 3}
dict2 = {'b': 2, 'c': 4, 'd': 5}

unique_pairs = get_unique_pairs(dict1, dict2)
print(unique_pairs)  # Output: {'a': 1, 'c': 3}
```
