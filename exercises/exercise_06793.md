## Exercise
Create a program that takes in a user's input and determines the data type of the input. The input can be an integer, float, or string. Use variables to store the input and the determined data type, and then print out the result.

## Reference Solution
```python
def determine_data_type():
    # Get user's input
    user_input = input("Please enter a value: ")

    # Try to convert the input to an integer
    try:
        int_input = int(user_input)
        data_type = "Integer"
    except ValueError:
        # If the input cannot be converted to an integer, try to convert it to a float
        try:
            float_input = float(user_input)
            data_type = "Float"
        except ValueError:
            # If the input cannot be converted to a float, it must be a string
            data_type = "String"

    # Print out the result
    print(f"The input '{user_input}' is of type {data_type}.")

# Call the function
determine_data_type()
```
