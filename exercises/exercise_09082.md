## Exercise
Write a program to calculate the sum of all elements in a given array. The array will contain only integers.

## Reference Solution
```python
def calculate_array_sum(array):
    """
    Calculates the sum of all elements in a given array.

    Args:
        array (list): A list of integers.

    Returns:
        int: The sum of all elements in the array.
    """
    return sum(array)

# Example usage:
def main():
    # Define an example array
    array = [1, 2, 3, 4, 5]

    # Calculate the sum of the array elements
    array_sum = calculate_array_sum(array)

    # Print the result
    print("Array:", array)
    print("Sum of array elements:", array_sum)

if __name__ == "__main__":
    main()
```
