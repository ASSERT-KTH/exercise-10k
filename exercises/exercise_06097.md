## Exercise
Create a program that uses recursive function calls and a for loop to sort a list using the quick sort algorithm.

## Reference Solution
```python
def quick_sort(arr):
    """
    Sorts a list using the quick sort algorithm.

    Args:
    arr (list): The list to be sorted.

    Returns:
    list: The sorted list.
    """

    # Base case: If the list is empty or contains one element, it's already sorted.
    if len(arr) <= 1:
        return arr

    # Select the pivot element (in this case, the middle element).
    pivot = arr[len(arr) // 2]

    # Initialize lists to store elements less than, equal to, and greater than the pivot.
    left = []
    middle = []
    right = []

    # Iterate over the list and distribute elements into the left, middle, and right lists.
    for x in arr:
        if x < pivot:
            left.append(x)
        elif x == pivot:
            middle.append(x)
        else:
            right.append(x)

    # Recursively sort the left and right lists, and combine the results with the middle list.
    return quick_sort(left) + middle + quick_sort(right)


# Example usage:
arr = [5, 2, 9, 1, 7, 3]
print("Original list:", arr)
print("Sorted list:", quick_sort(arr))
```
