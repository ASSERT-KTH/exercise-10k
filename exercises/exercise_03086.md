## Exercise
Use the map function to convert all strings in a list to uppercase.

## Reference Solution
```python
def convert_to_uppercase(strings):
    """
    Convert all strings in a list to uppercase using the map function.

    Args:
        strings (list): A list of strings.

    Returns:
        list: A list of uppercase strings.
    """
    # Use map function to apply str.upper to each string in the list
    uppercase_strings = list(map(str.upper, strings))
    
    return uppercase_strings

# Example usage
strings = ["hello", "world", "python"]
uppercase_strings = convert_to_uppercase(strings)

print(uppercase_strings)
```
