## Exercise
Write a program that takes an array and a target value as input, and returns the index of the target value in the array.

## Reference Solution
```python
def Search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

# Example usage
arr = [1, 2, 3, 4, 5]
target = 4
print(Search(arr, target))  # Output: 3

arr = [1, 2, 3, 4, 5]
target = 10
print(Search(arr, target))  # Output: -1
```
