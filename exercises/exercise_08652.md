## Exercise
Write a function that sorts a given list in ascending order.

## Reference Solution
```python
def sort_list(input_list):
    """
    Sorts a given list in ascending order.

    Args:
        input_list (list): The list to be sorted.

    Returns:
        list: The sorted list.
    """
    return sorted(input_list)

# Example usage:
if __name__ == "__main__":
    numbers = [64, 34, 25, 12, 22, 11, 90]
    print("Original list:", numbers)
    print("Sorted list:", sort_list(numbers))
```
