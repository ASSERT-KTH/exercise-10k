## Exercise
Create a function that uses debugging techniques such as print statements and a debugger to identify and fix a bug.

## Reference Solution
```python
def calculate_sum(numbers):
    """
    This function calculates the sum of a list of numbers.
    
    Args:
        numbers (list): A list of numbers.
    
    Returns:
        int: The sum of the numbers in the list.
    """
    
    # Initialize the sum variable to 0
    total = 0
    
    # Print the input list for debugging purposes
    print("Input list:", numbers)
    
    # Iterate over each number in the list
    for i, num in enumerate(numbers):
        # Print the current number being processed for debugging purposes
        print(f"Processing number {i+1}: {num}")
        
        # Check if the current number is a valid number (not None or NaN)
        import math
        if not isinstance(num, (int, float)) or math.isnan(num):
            # If the number is not valid, print an error message and skip it
            print(f"Error: Invalid number {num} at index {i}. Skipping...")
            continue
        
        # Add the current number to the total sum
        total += num
        
        # Print the updated total sum for debugging purposes
        print(f"Updated total: {total}")
    
    # Print the final total sum for debugging purposes
    print("Final total:", total)
    
    # Return the final total sum
    return total

# Test the function
numbers = [1, 2, 3, 4, 5]
result = calculate_sum(numbers)
print("Result:", result)
```
