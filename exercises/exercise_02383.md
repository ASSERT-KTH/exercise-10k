## Exercise
Write a program that generates the Fibonacci sequence up to the nth term using a for loop.

## Reference Solution
```python
#include <iostream>
using namespace std;

int main() {
    int a = 0;
    int b = 1;
    int c = 0;
    int n;
    cout << "Enter the value of n: ";
    cin >> n;

    for (int i = 1; i <= n; i++) {
        cout << a << " ";
        c = a + b;
        a = b;
        b = c;
    }

    return 0;
}
```
