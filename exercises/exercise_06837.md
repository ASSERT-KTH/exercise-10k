## Exercise
Create a function that takes an input string, reverses it, and checks if the reversed string is a palindrome. If the string is a palindrome, the function should return the reversed string; otherwise, it should return the original string.

## Reference Solution
```python
def check_palindrome(input_str):
    """
    This function checks if the reversed input string is a palindrome.
    If it is, the function returns the reversed string; otherwise, it returns the original string.

    Args:
        input_str (str): The input string to be checked.

    Returns:
        str: The reversed string if it's a palindrome; otherwise, the original string.
    """

    # First, we need to reverse the input string
    reversed_str = input_str[::-1]
    
    # Then, we need to check if the reversed string is equal to its reverse
    # If they are equal, then the reversed string is a palindrome
    if reversed_str == reversed_str[::-1]:
        
        # If the reversed string is a palindrome, we return it
        return reversed_str
    
    # If the reversed string is not a palindrome, we return the original string
    else:
        return input_str

# Test the function
print(check_palindrome("radar"))  # Should return "radar"
print(check_palindrome("hello"))  # Should return "hello"
```
