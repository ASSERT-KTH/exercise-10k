## Exercise
Write a program that joins a list of substrings into a single string.

## Reference Solution
```python
def join_substrings(substrings, separator=''):
    """
    Joins a list of substrings into a single string.

    Args:
        substrings (list): A list of substrings to join.
        separator (str): The separator to use between substrings. Defaults to an empty string.

    Returns:
        str: The joined string.
    """
    return separator.join(substrings)

def main():
    # Example usage
    substrings = ['Hello', 'world', 'this', 'is', 'a', 'test']
    separator = ' '
    joined_string = join_substrings(substrings, separator)
    print(joined_string)

if __name__ == '__main__':
    main()
```
