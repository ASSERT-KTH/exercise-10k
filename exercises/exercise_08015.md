## Exercise
Write a function to reverse the order of elements in a given list. The function should take a list as input and return the reversed list.

## Reference Solution
```python
def reverse_list(input_list):
    """
    This function takes a list as input and returns the reversed list.

    Args:
        input_list (list): The list to be reversed.

    Returns:
        list: The reversed list.
    """

    # First, we check if the input is indeed a list. If not, we raise a TypeError.
    if not isinstance(input_list, list):
        raise TypeError("Input must be a list.")

    # Now, we use Python's built-in list slicing feature to create a new list that is the reverse of the input list.
    # The syntax input_list[::-1] means start at the end of the list and end at position 0, move with the step -1.
    reversed_list = input_list[::-1]

    # Finally, we return the reversed list.
    return reversed_list

# Example usage:
numbers = [1, 2, 3, 4, 5]
print(reverse_list(numbers))  # Output: [5, 4, 3, 2, 1]
```
