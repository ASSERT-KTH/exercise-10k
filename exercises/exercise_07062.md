## Exercise
Write a program that takes a string as input and prints out the string with leading and trailing whitespaces removed.

## Reference Solution
```python
def remove_whitespaces(input_string):
    """
    Removes leading and trailing whitespaces from a string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The input string with leading and trailing whitespaces removed.
    """
    return input_string.strip()

def main():
    # Get the input string from the user
    input_string = input("Enter a string: ")

    # Remove leading and trailing whitespaces
    cleaned_string = remove_whitespaces(input_string)

    # Print the cleaned string
    print("Cleaned string:", cleaned_string)

if __name__ == "__main__":
    main()
```
