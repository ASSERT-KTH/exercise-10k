## Exercise
Write a program that takes an input from the user and checks its data type. The program should then print out the data type of the input.

## Reference Solution
```python
def check_data_type():
    user_input = input("Please enter something: ")
    
    try:
        # Try to convert the input to an integer
        int(user_input)
        print("The data type of the input is: integer")
    except ValueError:
        try:
            # Try to convert the input to a float
            float(user_input)
            print("The data type of the input is: float")
        except ValueError:
            # If it's neither an integer nor a float, it's a string
            print("The data type of the input is: string")

def main():
    check_data_type()

if __name__ == "__main__":
    main()
```
