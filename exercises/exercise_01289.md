## Exercise
Create a function that updates a dictionary with the key-value pairs from another dictionary.

## Reference Solution
```python
def update_dictionary(dict1, dict2):
    """
    Updates a dictionary with the key-value pairs from another dictionary.

    Args:
        dict1 (dict): The dictionary to be updated.
        dict2 (dict): The dictionary with the key-value pairs to update with.

    Returns:
        dict: The updated dictionary.
    """

    # First, we check if both inputs are dictionaries
    if not isinstance(dict1, dict) or not isinstance(dict2, dict):
        raise TypeError("Both inputs must be dictionaries")

    # We use the built-in update method of the dictionary class in Python
    # This method updates the dictionary with the key-value pairs from another dictionary or from an iterable of key-value pairs
    # It does not return anything, so we don't need to assign it to a variable
    dict1.update(dict2)

    # Finally, we return the updated dictionary
    return dict1

# Example usage:
dict1 = {"a": 1, "b": 2}
dict2 = {"b": 3, "c": 4}

print(update_dictionary(dict1, dict2))  # Output: {'a': 1, 'b': 3, 'c': 4}
```
