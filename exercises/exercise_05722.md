## Exercise
Create a function that takes a name as an argument and prints a personalized greeting.

## Reference Solution
```python
# Define a function called 'greet' that takes one argument, 'name'.
def greet(name):
    """
    Prints a personalized greeting based on the input name.

    Args:
        name (str): The name of the person to be greeted.
    """

    # Check if the input 'name' is a string. If not, raise a TypeError with a descriptive message.
    if not isinstance(name, str):
        raise TypeError("The 'name' argument must be a string.")

    # Use an f-string to format the greeting message with the input 'name'.
    # This allows for a more readable and efficient way of embedding expressions inside string literals.
    greeting = f"Hello, {name}!"

    # Print the greeting message.
    print(greeting)

# Example usage:
greet("John")  # Output: Hello, John!
```
