## Exercise
Create a function that finds the maximum value in a list of numbers.

## Reference Solution
```python
def find_max_value(numbers):
    """
    Finds the maximum value in a list of numbers.

    Args:
    numbers (list): A list of numbers.

    Returns:
    max_value: The maximum value in the list.

    Raises:
    ValueError: If the list is empty.
    """

    # Check if the list is empty
    if not numbers:
        raise ValueError("List is empty")

    # Initialize max_value with the first element of the list
    max_value = numbers[0]

    # Iterate over the list to find the maximum value
    for num in numbers:
        if num > max_value:
            max_value = num

    return max_value

# Example usage
numbers = [12, 45, 7, 23, 56, 89, 34]
print(find_max_value(numbers))  # Output: 89
```
