## Exercise
Create a program that takes in a list of values and determines the data type of each value. The program should then print out the data type of each value and the total count of each data type. The data types to consider are integers, floats, strings, and booleans.

## Reference Solution
```python
def determine_data_types(values):
    """
    This function determines the data type of each value in a list and prints out the data type of each value.
    It also prints out the total count of each data type.

    Args:
        values (list): A list of values

    Returns:
        None
    """

    # Initialize a dictionary to store the count of each data type
    data_type_counts = {
        "Integer": 0,
        "Float": 0,
        "String": 0,
        "Boolean": 0
    }

    # Iterate over each value in the list
    for value in values:
        # Check the type of the value
        if isinstance(value, int):
            print(f"{value} is an Integer")
            data_type_counts["Integer"] += 1
        elif isinstance(value, float):
            print(f"{value} is a Float")
            data_type_counts["Float"] += 1
        elif isinstance(value, str):
            print(f"{value} is a String")
            data_type_counts["String"] += 1
        elif isinstance(value, bool):
            print(f"{value} is a Boolean")
            data_type_counts["Boolean"] += 1
        else:
            print(f"{value} is of an unknown data type")

    # Print out the total count of each data type
    print("\nData Type Counts:")
    for data_type, count in data_type_counts.items():
        print(f"{data_type}: {count}")


# Example usage:
values = [1, 2.5, "hello", True, False, 3, 4.5, "world", 5]
determine_data_types(values)
```
