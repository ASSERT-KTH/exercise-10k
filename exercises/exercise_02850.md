## Exercise
Write a function to retrieve the value associated with a given key from a dictionary, providing a default value if the key is not present.

## Reference Solution
```python
def get_value_from_dict(dictionary, key, default_value=None):
    """
    Retrieves the value associated with a given key from a dictionary.
    
    If the key is not present in the dictionary, it returns a default value.
    
    Args:
        dictionary (dict): The dictionary to search in.
        key: The key to search for.
        default_value: The value to return if the key is not found. Defaults to None.
    
    Returns:
        The value associated with the key, or the default value if the key is not found.
    """

    # First, we check if the dictionary is actually a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("The first argument must be a dictionary.")

    # Then, we use the get method of the dictionary to retrieve the value
    # The get method returns the value for the given key if it exists in the dictionary
    # If the key does not exist, it returns the default value
    return dictionary.get(key, default_value)
```
