## Exercise
Create a program that uses functions to perform basic arithmetic operations. The program should have separate functions for addition, subtraction, multiplication, and division. Each function should take two numbers as input and return the result of the operation.

## Reference Solution
```python
def add(num1, num2):
    return num1 + num2

def sub(num1, num2):
    return num1 - num2

def mul(num1, num2):
    return num1 * num2

def div(num1, num2):
    if num2 != 0:
        return num1 / num2
    else:
        return None

# Example usage:
print(add(10, 5))  # Output: 15
print(sub(10, 5))  # Output: 5
print(mul(10, 5))  # Output: 50
print(div(10, 5))  # Output: 2.0
print(div(10, 0))  # Output: None
```
