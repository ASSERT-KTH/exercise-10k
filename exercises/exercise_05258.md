## Exercise
Write a program to check if a given number is an Armstrong number or not.

## Reference Solution
```python
def is_armstrong(num):
    """
    Checks if a given number is an Armstrong number.

    Args:
        num (int): The number to check.

    Returns:
        bool: True if the number is an Armstrong number, False otherwise.
    """
    # Convert the number to a string to easily get the number of digits
    num_str = str(num)
    num_digits = len(num_str)

    # Calculate the sum of the digits each raised to the power of the number of digits
    sum_of_digits = sum(int(digit) ** num_digits for digit in num_str)

    # Check if the sum is equal to the original number
    return num == sum_of_digits


def main():
    # Example usage:
    num = int(input("Enter a number: "))
    if is_armstrong(num):
        print(f"{num} is an Armstrong number.")
    else:
        print(f"{num} is not an Armstrong number.")


if __name__ == "__main__":
    main()
```
