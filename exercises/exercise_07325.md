## Exercise
Write a program that reads input from stdin and prints out the contents.

## Reference Solution
```python
import sys

def read_from_stdin():
    """
    Reads input from stdin and prints out the contents.
    """
    for line in sys.stdin:
        print(line, end='')

if __name__ == "__main__":
    read_from_stdin()
```
