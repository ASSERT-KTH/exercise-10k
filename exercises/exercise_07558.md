## Exercise
Write a function that takes a variable number of arguments and prints out their sum.

## Reference Solution
```python
# Define a function that takes a variable number of arguments
def print_sum(*args):
    """
    This function calculates the sum of all input arguments and prints it.

    Args:
        *args: A variable number of arguments

    Returns:
        None
    """

    # Initialize the sum variable to 0
    total_sum = 0
    
    # Iterate over each argument
    for arg in args:
        # Add the current argument to the sum
        total_sum += arg
    
    # Print the total sum
    print(total_sum)

# Test the function with some numbers
print_sum(1, 2, 3, 4, 5)
```
