## Exercise
Create a function that takes a string as input and returns the string converted to camelCase.

## Reference Solution
```python
def to_camel_case(input_string):
    """
    Converts a string to camelCase.

    Args:
        input_string (str): The input string to be converted.

    Returns:
        str: The input string converted to camelCase.
    """
    # Split the input string into words
    words = input_string.split()

    # If the input string is empty, return it as is
    if not words:
        return input_string

    # Convert the first word to lowercase
    camel_case_string = words[0].lower()

    # Convert the rest of the words to title case and append them to the result
    for word in words[1:]:
        camel_case_string += word.capitalize()

    return camel_case_string

# Example usage
if __name__ == "__main__":
    print(to_camel_case("hello world"))  # Output: "helloWorld"
    print(to_camel_case("this is a test"))  # Output: "thisIsATest"
    print(to_camel_case(""))  # Output: ""
```
