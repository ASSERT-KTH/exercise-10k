## Exercise
Create a function that takes a list and a reducing function as input and returns the result of applying the reducing function to the list.

## Reference Solution
```python
def reduce_list(input_list, reducing_function):
    """
    Applies a given reducing function to a list and returns the result.

    Args:
        input_list (list): The list to be reduced.
        reducing_function (function): A function that takes two arguments and returns one value.

    Returns:
        The result of applying the reducing function to the list.
    """

    # Check if the input list is empty
    if not input_list:
        raise ValueError("Input list cannot be empty")

    # Initialize the result with the first element of the list
    # This is because the reducing function will be applied from the second element onwards
    result = input_list[0]

    # Iterate over the list starting from the second element
    for i in range(1, len(input_list)):
        # Apply the reducing function to the current result and the current element
        # The result is updated at each step
        result = reducing_function(result, input_list[i])

    # Return the final result after applying the reducing function to all elements
    return result


# Example usage:
# Define a simple reducing function that adds two numbers
def add(a, b):
    return a + b

# Test the reduce_list function
numbers = [1, 2, 3, 4, 5]
print(reduce_list(numbers, add))  # Output: 15
```
