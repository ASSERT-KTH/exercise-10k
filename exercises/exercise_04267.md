## Exercise
Write a function that takes a dictionary as input and returns a list of all values

## Reference Solution
```python
def get_dict_values(input_dict):
    """
    Retrieves all values from a given dictionary.

    Args:
    input_dict (dict): The dictionary to retrieve values from.

    Returns:
    list: A list of all values in the dictionary.
    """
    return list(input_dict.values())

# Example usage
if __name__ == "__main__":
    example_dict = {
        "key1": "value1",
        "key2": "value2",
        "key3": "value3"
    }
    print(get_dict_values(example_dict))
```
