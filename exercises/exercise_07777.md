## Exercise
Write a program that takes in a user's input and determines the data type of the input. If the input can be converted to an integer, it should be classified as an integer. If it cannot be converted to an integer but can be converted to a float, it should be classified as a float. Otherwise, it should be classified as a string.

## Reference Solution
```python
def classify_input(input_str):
    """
    Classify the input string as an integer, float, or string.

    Args:
        input_str (str): The user's input.

    Returns:
        str: The data type of the input.
    """
    try:
        # Attempt to convert the input to an integer
        int(input_str)
        return "integer"
    except ValueError:
        try:
            # If integer conversion fails, attempt to convert to a float
            float(input_str)
            return "float"
        except ValueError:
            # If float conversion also fails, classify as a string
            return "string"


def main():
    # Get user input
    user_input = input("Please enter a value: ")

    # Classify the input
    data_type = classify_input(user_input)

    # Print the result
    print(f"The input '{user_input}' is a {data_type}.")


if __name__ == "__main__":
    main()
```
