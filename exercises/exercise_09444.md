## Exercise
Create a program that takes two numbers as input and prints the maximum of the two.

## Reference Solution
```python
def get_max(num1, num2):
    """
    Returns the maximum of two numbers.

    Args:
        num1 (float): The first number.
        num2 (float): The second number.

    Returns:
        float: The maximum of num1 and num2.
    """
    return max(num1, num2)

def main():
    # Get input from user
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))

    # Calculate and print the maximum
    max_num = get_max(num1, num2)
    print(f"The maximum of {num1} and {num2} is {max_num}")

if __name__ == "__main__":
    main()
```
