## Exercise
Write a program that takes a string as input and returns the number of special characters in the string.

## Reference Solution
```python
def count_special_chars(input_str):
    """
    Counts the number of special characters in a string.

    Args:
        input_str (str): The input string to count special characters from.

    Returns:
        int: The number of special characters in the string.
    """
    special_chars = "!@#$%^&*()-_=+[]{};:'<>,./?`~"
    count = 0

    for char in input_str:
        if char in special_chars:
            count += 1

    return count

# Example usage
input_str = "Hello, World! How's it going?"
print(f"Number of special characters: {count_special_chars(input_str)}")
```
