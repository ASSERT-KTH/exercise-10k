## Exercise
Write a program that takes a list of integers as input, sorts it in ascending order, and then removes the first and last elements. The program should then print the resulting list.

## Reference Solution
```python
def ordenar(secuencia):
    secuencia.sort()
    return secuencia[1:-1]

secuencia = [3, 5, 1, 7, 2, 9, 6, 8, 4]
secuencia = ordenar(secuencia)
print(secuencia)
```
