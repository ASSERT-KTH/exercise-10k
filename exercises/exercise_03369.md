## Exercise
Use a while loop to calculate the greatest common divisor (GCD) of two given numbers

## Reference Solution
```python
def calculate_gcd(num1, num2):
    """
    Calculate the GCD of two numbers using the Euclidean algorithm and a while loop.

    Args:
        num1 (int): The first number.
        num2 (int): The second number.

    Returns:
        int: The GCD of num1 and num2.
    """
    # Ensure num1 is the larger number
    if num2 > num1:
        num1, num2 = num2, num1

    while num2 != 0:
        # Store the remainder of num1 divided by num2
        remainder = num1 % num2
        # Update num1 to be the previous num2
        num1 = num2
        # Update num2 to be the remainder
        num2 = remainder

    return num1

# Example usage
num1 = 48
num2 = 18
gcd = calculate_gcd(num1, num2)
print(f"The GCD of {num1} and {num2} is: {gcd}")
```
