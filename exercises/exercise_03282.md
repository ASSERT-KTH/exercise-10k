## Exercise
Write a program that checks whether a given number is even or odd using a while loop.

## Reference Solution
```python
def check_even_odd(num):
    """
    Checks whether a given number is even or odd.
    
    Parameters:
    num (int): The number to check.
    
    Returns:
    str: 'Even' if the number is even, 'Odd' otherwise.
    """
    i = num
    while i != 0:
        i = i % 2
        if i == 0:
            return 'Even'
        else:
            return 'Odd'

num = int(input("Enter a number: "))
print(check_even_odd(num))
```
