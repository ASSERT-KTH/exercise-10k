## Exercise
Write a function that swaps the first and last elements of a given array. If the array has less than 2 elements, return the original array.

## Reference Solution
```python
def swap_first_last(arr):
    if len(arr) < 2:
        return arr
    else:
        temp = arr[0]
        arr[0] = arr[-1]
        arr[-1] = temp
        return arr

# Example usage:
arr1 = [1, 2, 3, 4, 5]
arr2 = [1]
arr3 = [1, 2]

print(swap_first_last(arr1))  # Output: [5, 2, 3, 4, 1]
print(swap_first_last(arr2))  # Output: [1]
print(swap_first_last(arr3))  # Output: [2, 1]
```
