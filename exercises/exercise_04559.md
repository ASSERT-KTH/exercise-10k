## Exercise
Create a program that converts a given string to sentence case.

## Reference Solution
```python
def convert_to_sentence_case(input_string):
    """
    Converts a given string to sentence case.

    Args:
        input_string (str): The input string to be converted.

    Returns:
        str: The input string in sentence case.
    """
    # Capitalize the first character and make the rest lowercase
    sentence_case_string = input_string.capitalize()
    
    return sentence_case_string


def main():
    # Example usage
    input_string = input("Enter a string: ")
    sentence_case_string = convert_to_sentence_case(input_string)
    print("Sentence case string: ", sentence_case_string)


if __name__ == "__main__":
    main()
```
