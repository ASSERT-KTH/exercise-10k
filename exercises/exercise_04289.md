## Exercise
Write a function to sort a dictionary's key-value pairs by key in ascending order.

## Reference Solution
```python
def sort_dict_by_key(dictionary):
    """
    This function sorts a dictionary's key-value pairs by key in ascending order.

    Args:
        dictionary (dict): The dictionary to be sorted.

    Returns:
        dict: A new dictionary with the key-value pairs sorted by key in ascending order.
    """

    # First, we check if the input is indeed a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("Input must be a dictionary")

    # We use the built-in sorted function to sort the dictionary items based on the keys
    # The sorted function returns a new list containing all items from the dictionary in ascending order of their keys
    sorted_items = sorted(dictionary.items())

    # We then use the dict function to convert the sorted list back into a dictionary
    # This is because dictionaries in Python 3.7 and later maintain the insertion order, so the resulting dictionary will have the key-value pairs sorted by key
    sorted_dict = dict(sorted_items)

    # Finally, we return the sorted dictionary
    return sorted_dict

# Example usage:
my_dict = {'c': 3, 'b': 2, 'a': 1}
sorted_my_dict = sort_dict_by_key(my_dict)
print(sorted_my_dict)  # Output: {'a': 1, 'b': 2, 'c': 3}
```
