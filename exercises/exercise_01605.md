## Exercise
Write a program that takes two integers as input and outputs their modulus (remainder).

## Reference Solution
```python
def calculate_modulus(dividend, divisor):
    """
    Calculate the modulus (remainder) of two integers.

    Args:
        dividend (int): The number being divided.
        divisor (int): The number by which we are dividing.

    Returns:
        int: The remainder of the division.
    """
    if divisor == 0:
        raise ZeroDivisionError("Cannot divide by zero")
    return dividend % divisor


def main():
    # Get user input
    dividend = int(input("Enter the dividend (first integer): "))
    divisor = int(input("Enter the divisor (second integer): "))

    try:
        # Calculate and print the modulus
        modulus = calculate_modulus(dividend, divisor)
        print(f"The modulus (remainder) of {dividend} and {divisor} is {modulus}")
    except ZeroDivisionError as e:
        print(f"Error: {e}")


if __name__ == "__main__":
    main()
```
