## Exercise
Write a function to create a dictionary using dictionary comprehension.

## Reference Solution
```python
def create_dictionary(keys, values):
    """
    Create a dictionary using dictionary comprehension.

    Args:
        keys (list): A list of keys for the dictionary.
        values (list): A list of values for the dictionary.

    Returns:
        dict: A dictionary where each key is mapped to a corresponding value.
    """
    return {key: value for key, value in zip(keys, values)}

# Example usage:
keys = ['apple', 'banana', 'cherry']
values = [1, 2, 3]

result = create_dictionary(keys, values)
print(result)  # Output: {'apple': 1, 'banana': 2, 'cherry': 3}
```

```python
def square_numbers(numbers):
    """
    Create a dictionary where each key is a number and its corresponding value is the square of that number.

    Args:
        numbers (list): A list of numbers.

    Returns:
        dict: A dictionary where each key is a number and its corresponding value is the square of that number.
    """
    return {num: num ** 2 for num in numbers}

# Example usage:
numbers = [1, 2, 3, 4, 5]
result = square_numbers(numbers)
print(result)  # Output: {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
```
