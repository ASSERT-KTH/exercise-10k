## Exercise
Write a program that takes a list, an index, and an element as input and inserts the element at the specified index in the list

## Reference Solution
```python
def insert_element(lst, index, element):
    """
    Inserts an element at a specified index in a list.

    Args:
        lst (list): The input list.
        index (int): The index where the element will be inserted.
        element: The element to be inserted.

    Returns:
        list: The list with the element inserted at the specified index.

    Raises:
        IndexError: If the index is out of range.
    """
    if index < 0 or index > len(lst):
        raise IndexError("Index out of range")
    return lst[:index] + [element] + lst[index:]

# Example usage:
my_list = [1, 2, 3, 4, 5]
index = 2
element = 10

print("Original List:", my_list)
print("Index:", index)
print("Element:", element)

result = insert_element(my_list, index, element)
print("Resulting List:", result)
```
