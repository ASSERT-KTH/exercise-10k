## Exercise
Write a program that prints numbers from 10 to 1 in reverse order using a for loop.

## Reference Solution
```python
def print_reverse_numbers():
    """Prints numbers from 10 to 1 in reverse order."""
    for i in range(10, 0, -1):
        print(i)

# Example usage:
if __name__ == "__main__":
    print_reverse_numbers()
```
