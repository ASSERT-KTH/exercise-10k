## Exercise
Write a program that takes a string input from the user and checks if it can be converted to an integer. If not, print an error message.

## Reference Solution
```python
try:
    number = input("Enter your number: ")
    number = int(number)
    print(number)
except ValueError:
    print("This is not a valid number")
```
