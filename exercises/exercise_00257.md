## Exercise
Write a program that takes two arrays as input, and returns True if the two arrays have no elements in common, and False otherwise.

## Reference Solution
```python
def check_no_common_elements(list_1, list_2):
    for element in list_1:
        if element in list_2:
            return False
    return True

# Test cases
print(check_no_common_elements([1, 2, 3], [4, 5, 6]))  # Output: True
print(check_no_common_elements([1, 2, 3], [2, 4, 6]))  # Output: False
```
