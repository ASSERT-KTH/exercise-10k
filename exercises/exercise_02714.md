## Exercise
Write a program to convert a given string to an integer.

## Reference Solution
```python
def string_to_int(s):
    try:
        return int(s)
    except ValueError:
        return "Invalid input. Please enter a string that represents an integer."

def main():
    s = input("Enter a string: ")
    print(string_to_int(s))

if __name__ == "__main__":
    main()
```
