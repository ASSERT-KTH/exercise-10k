## Exercise
Create a function to remove duplicate characters from a given string. The function should return the string without duplicates.

## Reference Solution
```python
def remove_duplicates(input_string):
    """
    This function removes duplicate characters from a given string and returns the resulting string.
    
    Parameters:
    input_string (str): The input string from which to remove duplicates.
    
    Returns:
    str: The input string without duplicates.
    """

    # Initialize an empty string to store the result
    result = ""

    # Initialize an empty set to keep track of characters we've seen so far
    seen = set()

    # Iterate over each character in the input string
    for char in input_string:
        
        # Check if the character is not in the seen set
        if char not in seen:
            
            # Add the character to the result string
            result += char
            
            # Add the character to the seen set
            seen.add(char)

    # Return the result string
    return result

# Test the function
print(remove_duplicates("hello"))  # Output: "helo"
```
