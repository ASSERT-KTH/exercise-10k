## Exercise
Write a program that takes an array as input and returns the standard deviation of the array.

## Reference Solution
```python
import numpy as np

def calculate_standard_deviation(array):
    """
    Calculate the standard deviation of an array.

    Parameters:
    array (list): The input array.

    Returns:
    float: The standard deviation of the array.
    """
    return np.std(array)

# Example usage:
array = [1, 2, 3, 4, 5]
print("Standard deviation:", calculate_standard_deviation(array))
```

```python
def calculate_mean(array):
    """
    Calculate the mean of an array.

    Parameters:
    array (list): The input array.

    Returns:
    float: The mean of the array.
    """
    return sum(array) / len(array)

def calculate_variance(array):
    """
    Calculate the variance of an array.

    Parameters:
    array (list): The input array.

    Returns:
    float: The variance of the array.
    """
    mean = calculate_mean(array)
    squared_diffs = [(x - mean) ** 2 for x in array]
    return sum(squared_diffs) / len(array)

def calculate_standard_deviation(array):
    """
    Calculate the standard deviation of an array.

    Parameters:
    array (list): The input array.

    Returns:
    float: The standard deviation of the array.
    """
    variance = calculate_variance(array)
    return variance ** 0.5

# Example usage:
array = [1, 2, 3, 4, 5]
print("Standard deviation:", calculate_standard_deviation(array))
```
