## Exercise
Write a program that takes an input and processes it using a function and a loop.

## Reference Solution
```python
def process_input(input_list):
    """
    Process the input list by squaring each number.

    Args:
        input_list (list): A list of numbers to be processed.

    Returns:
        list: A list of squared numbers.
    """
    squared_list = []
    for num in input_list:
        squared_list.append(num ** 2)
    return squared_list

def main():
    # Get user input
    input_str = input("Enter a list of numbers separated by spaces: ")

    # Convert input string to a list of integers
    input_list = [int(num) for num in input_str.split()]

    # Process the input list
    squared_list = process_input(input_list)

    # Print the result
    print("Squared List:", squared_list)

if __name__ == "__main__":
    main()
```
