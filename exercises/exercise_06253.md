## Exercise
Write a function that takes a dictionary and a key as input and returns the value for the given key if it exists in the dictionary. If not, it inserts the key with a default value and returns the default value

## Reference Solution
```python
def get_or_insert(dictionary, key, default_value=None):
    """
    Retrieves the value for a given key from a dictionary. If the key does not exist, 
    it inserts the key with a default value and returns the default value.

    Args:
        dictionary (dict): The dictionary to retrieve from or insert into.
        key: The key to retrieve or insert.
        default_value (any, optional): The default value to insert if the key does not exist. Defaults to None.

    Returns:
        any: The value for the given key, or the default value if the key did not exist.
    """
    if key not in dictionary:
        dictionary[key] = default_value
    return dictionary[key]

# Example usage
example_dict = {"a": 1, "b": 2}
print(get_or_insert(example_dict, "a"))  # Output: 1
print(get_or_insert(example_dict, "c", "default"))  # Output: default
print(example_dict)  # Output: {'a': 1, 'b': 2, 'c': 'default'}
```
