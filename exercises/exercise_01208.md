## Exercise
Write a function that calculates the sum of all elements in a given list.

## Reference Solution
```python
def sum_list_elements(input_list):
    """
    Calculates the sum of all elements in a given list.

    Args:
        input_list (list): A list of numbers.

    Returns:
        float: The sum of all elements in the list.

    Raises:
        TypeError: If the input is not a list.
        ValueError: If the list contains non-numeric values.
    """
    if not isinstance(input_list, list):
        raise TypeError("Input must be a list")

    try:
        return sum(input_list)
    except TypeError:
        raise ValueError("List must only contain numeric values")

# Example usage
def main():
    numbers = [1, 2, 3, 4, 5]
    result = sum_list_elements(numbers)
    print(f"The sum of the list elements is: {result}")

if __name__ == "__main__":
    main()
```
