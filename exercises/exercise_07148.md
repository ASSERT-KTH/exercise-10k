## Exercise
Create a function that checks if a given string is valid based on the following rules: it must be at least 5 characters long, must contain at least one uppercase letter, and must contain at least one digit. The function should return True if the string is valid and False otherwise.

## Reference Solution
```python
def check_string_validity(input_string):
    """
    Checks if a given string is valid based on the following rules:
    - It must be at least 5 characters long.
    - It must contain at least one uppercase letter.
    - It must contain at least one digit.

    Args:
        input_string (str): The input string to be checked.

    Returns:
        bool: True if the string is valid, False otherwise.
    """

    # First, we check if the length of the string is at least 5 characters
    # If it's not, we immediately return False because the string is invalid
    if len(input_string) < 5:
        return False

    # Next, we check if the string contains at least one uppercase letter
    # We use the any() function with a generator expression to check this
    # The isupper() method returns True if a character is uppercase, False otherwise
    # If no uppercase letter is found, we return False
    if not any(char.isupper() for char in input_string):
        return False

    # Finally, we check if the string contains at least one digit
    # We use the any() function with a generator expression to check this
    # The isdigit() method returns True if a character is a digit, False otherwise
    # If no digit is found, we return False
    if not any(char.isdigit() for char in input_string):
        return False

    # If the string passes all the checks, we return True
    # This means the string is valid
    return True
```
