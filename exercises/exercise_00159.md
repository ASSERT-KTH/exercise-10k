## Exercise
Create a function that takes in two dictionaries and returns a new dictionary containing the key-value pairs that exist in both dictionaries

## Reference Solution
```python
def common_key_value_pairs(dict1, dict2):
    """
    This function takes in two dictionaries and returns a new dictionary containing 
    the key-value pairs that exist in both dictionaries.

    Args:
        dict1 (dict): The first dictionary.
        dict2 (dict): The second dictionary.

    Returns:
        dict: A new dictionary containing common key-value pairs.
    """
    common_pairs = {key: value for key, value in dict1.items() if key in dict2 and dict2[key] == value}
    return common_pairs

# Example usage:
dict1 = {'a': 1, 'b': 2, 'c': 3}
dict2 = {'a': 1, 'b': 3, 'd': 4}

common_dict = common_key_value_pairs(dict1, dict2)
print(common_dict)  # Output: {'a': 1}
```
