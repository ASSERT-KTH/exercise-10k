## Exercise
Create a function that takes a variable number of arguments and returns their sum

## Reference Solution
```python
def variable_sum(*args):
    """
    Returns the sum of all input arguments.

    Args:
        *args: A variable number of arguments.

    Returns:
        The sum of all input arguments.

    Raises:
        TypeError: If any argument is not a number.
    """
    try:
        return sum(args)
    except TypeError:
        raise TypeError("All arguments must be numbers")

# Example usage:
print(variable_sum(1, 2, 3, 4, 5))  # Outputs: 15
print(variable_sum(10, 20, 30))  # Outputs: 60
```

def variable_sum(*args):
    """
    Returns the sum of all input arguments.

    Args:
        *args: A variable number of arguments.

    Returns:
        The sum of all input arguments.

    Raises:
        TypeError: If any argument is not a number.
    """
    for arg in args:
        if not isinstance(arg, (int, float)):
            raise TypeError("All arguments must be numbers")
    return sum(args)

# Example usage:
print(variable_sum(1, 2, 3, 4, 5))  # Outputs: 15
