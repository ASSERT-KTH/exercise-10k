## Exercise
Write a program that takes a list of integers as input, reverses the list, and then prints the element at the middle index of the reversed list. If the list has an even number of elements, print the average of the two middle elements.

## Reference Solution
```python
# Description: This program takes a list of integers as input, reverses the list, and prints the middle element (or average of the two middle elements if the list has an even number of elements).

# Author: [Your Name]
# Date: [Today's Date]

def main():
    # Get user input
    input_list = input("Enter a list of integers separated by spaces: ")

    # Parse input
    try:
        input_list = list(map(int, input_list.split()))
    except ValueError:
        print("Invalid input. Please enter a list of integers separated by spaces.")
        return

    # Reverse the list
    input_list = input_list[::-1]

    # Find the middle index
    middle_index = len(input_list) // 2

    # Print the middle element
    if len(input_list) % 2 != 0:
        print(input_list[middle_index])
    else:
        print((input_list[middle_index - 1] + input_list[middle_index]) / 2)

if __name__ == "__main__":
    main()
```
