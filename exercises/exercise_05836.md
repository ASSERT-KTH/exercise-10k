## Exercise
Write a function to encode a given string as HTML entities

## Reference Solution
```python
import html

def encode_html_entities(input_string):
    """
    Encodes a given string as HTML entities.

    Args:
        input_string (str): The input string to be encoded.

    Returns:
        str: The encoded string.
    """
    return html.escape(input_string)

# Example usage
if __name__ == "__main__":
    input_str = "<p>This is a test string with special characters: & < > ' \"</p>"
    encoded_str = encode_html_entities(input_str)
    print("Input String:")
    print(input_str)
    print("\nEncoded String:")
    print(encoded_str)
```
