## Exercise
Convert a string input to an integer and store it in a variable, then print the variable.

## Reference Solution
```python
def convert_and_print():
    # Get user input
    user_input = input("Please enter a string that can be converted to an integer: ")

    try:
        # Attempt to convert the string to an integer
        integer_value = int(user_input)
        
        # Print the integer value
        print("The integer value is:", integer_value)
    except ValueError:
        print("The input string cannot be converted to an integer.")

# Example usage
if __name__ == "__main__":
    convert_and_print()
```
