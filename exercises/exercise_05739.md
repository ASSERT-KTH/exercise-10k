## Exercise
Write a function that takes a function and some arguments, and returns a new function that is the partial application of the input function with the given arguments.

## Reference Solution

def partial_application(func, *args):
    """
    This function takes a function and some arguments, and returns a new function 
    that is the partial application of the input function with the given arguments.

    Args:
        func (function): The input function.
        *args: Variable number of arguments to be applied partially to the input function.

    Returns:
        function: A new function that is the partial application of the input function.
    """

    # We want to return a new function, so we define it inside our partial_application function
    def new_func(*new_args):
        
        # When the new function is called, it should call the original function with the 
        # original arguments and the new arguments
        # We use the * operator to unpack the arguments into a list, and then combine them
        return func(*args, *new_args)
    
    # Finally, we return the new function
    return new_func

# Example usage:
def add(a, b, c):
    return a + b + c

# Create a new function that adds 1 and 2 to the input
new_add = partial_application(add, 1, 2)

# Now we can call the new function with only one argument
print(new_add(3))  # Outputs: 6
