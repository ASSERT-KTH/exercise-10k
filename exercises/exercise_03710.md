## Exercise
Write a function that takes in two numbers and returns their greatest common divisor.

## Reference Solution
```python
import math

def calculate_gcd(num1: int, num2: int) -> int:
    """
    Calculate the greatest common divisor (GCD) of two numbers.

    Args:
        num1 (int): The first number.
        num2 (int): The second number.

    Returns:
        int: The GCD of num1 and num2.
    """
    # Use the math.gcd function for simplicity and efficiency
    return math.gcd(num1, num2)

# Example usage:
if __name__ == "__main__":
    num1 = 48
    num2 = 18
    gcd = calculate_gcd(num1, num2)
    print(f"The GCD of {num1} and {num2} is {gcd}.")
```
