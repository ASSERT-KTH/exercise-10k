## Exercise
Create a program that prints a pattern of numbers based on the input number. The pattern should be a square of numbers where each row contains the numbers from 1 to the input number. Use loops to generate the pattern.

## Reference Solution

    for j in range(1, 6):
        print(j, end=' ')
    print()
