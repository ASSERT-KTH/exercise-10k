## Exercise
Create a dictionary with 5 key-value pairs. Set a default value for a key using the .setdefault() method.

## Reference Solution
```python
# Create a dictionary
my_dict = {
    "name": "John Doe",
    "age": 30,
    "city": "New York",
    "country": "USA",
    "email": "john.doe@example.com"
}

# Set a default value for a key
default_value = "Not specified"
my_dict.setdefault("phone", default_value)

# Print the updated dictionary
print(my_dict)
```
