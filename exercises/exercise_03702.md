## Exercise
Write a function that takes in a list of numbers and returns the maximum value.

## Reference Solution
```python
def find_max(numbers):
    """
    Returns the maximum value in a list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        max_value (float): The maximum value in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not numbers:
        raise ValueError("Input list is empty")

    return max(numbers)

# Example usage
numbers = [4, 2, 9, 6, 5, 1]
max_value = find_max(numbers)
print("Maximum value:", max_value)
```
