## Exercise
Write a program that takes an employee's salary and years of service as input, and prints their salary after applying a 10% raise for each year of service.

## Reference Solution
```python
# Define a function to calculate the new salary
def calcular_aumento(salario, anos_servico):
    # Apply a 10% raise for each year of service
    salario_final = salario * (1 + (anos_servico * 0.10))
    return salario_final

# Get the salary and years of service from the user
salario = float(input("Digite seu salario: "))
anos_servico = int(input("Digite os anos de serviço: "))

# Calculate the new salary
salario_final = calcular_aumento(salario, anos_servico)

# Print the result
print("Salário final é %.2f" % salario_final)
```
