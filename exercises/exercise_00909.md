## Exercise
Create a program that maps a given data type to its corresponding value (e.g., 'int' to 1, 'float' to 2, 'str' to 3).

## Reference Solution
```python
def map_data_type(data_type):
    data_types = {
        'int': 1,
        'float': 2,
        'str': 3,
    }
    return data_types.get(data_type, 0)

# Example usage:
print(map_data_type('int'))  # Output: 1
print(map_data_type('float'))  # Output: 2
print(map_data_type('str'))  # Output: 3
print(map_data_type('bool'))  # Output: 0
```
