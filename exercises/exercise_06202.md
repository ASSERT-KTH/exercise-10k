## Exercise
Write a function that takes an array and an index as input, and returns the element at the specified index.

## Reference Solution
```python
def elementAt(arr, index):
    if index < 0 or index >= len(arr):
        return None
    return arr[index]

# Example usage:
arr = [1, 2, 3, 4, 5]
index = 3
result = elementAt(arr, index)
print(result)  # Output: 4
```
