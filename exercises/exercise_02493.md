## Exercise
Write a program to find the maximum contiguous subarray of a list of numbers

## Reference Solution
```python
def max_contiguous_subarray(nums):
    """
    Find the maximum contiguous subarray of a list of numbers.

    Args:
        nums (list): A list of numbers.

    Returns:
        tuple: A tuple containing the maximum sum and the subarray.
    """
    if not nums:
        return 0, []

    max_sum = float('-inf')
    current_sum = 0
    start = 0
    end = 0
    temp_start = 0

    for i in range(len(nums)):
        if current_sum < 0:
            current_sum = nums[i]
            temp_start = i
        else:
            current_sum += nums[i]

        if current_sum > max_sum:
            max_sum = current_sum
            start = temp_start
            end = i

    return max_sum, nums[start:end+1]

# Example usage:
nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
max_sum, subarray = max_contiguous_subarray(nums)
print(f"Maximum sum: {max_sum}")
print(f"Subarray: {subarray}")
```
