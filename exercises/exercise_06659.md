## Exercise
Generate a geometric progression sequence up to a given number

## Reference Solution
```python
def generate_geometric_progression(first_term, common_ratio, max_value):
    """
    Generates a geometric progression sequence up to a given number.

    Args:
        first_term (float): The first term of the sequence.
        common_ratio (float): The common ratio of the sequence.
        max_value (float): The maximum value of the sequence.

    Returns:
        list: A list of numbers representing the geometric progression sequence.
    """
    sequence = [first_term]
    while True:
        next_term = sequence[-1] * common_ratio
        if next_term > max_value:
            break
        sequence.append(next_term)
    return sequence

# Example usage:
first_term = 2
common_ratio = 2
max_value = 100

sequence = generate_geometric_progression(first_term, common_ratio, max_value)
print(sequence)
```
