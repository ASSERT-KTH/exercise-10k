## Exercise
Write a program that takes two numbers as input and returns the maximum of the two using conditional statements.

## Reference Solution
```python

# Get two numbers from the user
num1 = int(input("Enter first number: "))
num2 = int(input("Enter second number: "))

# Use if-else statement to find the maximum number

    max_num = num1
else:
    max_num = num2

# Print the result
print("Maximum number is:", max_num)
```
