## Exercise
Write a program that reads a string from the user and prints it out with a suffix added to each line.

## Reference Solution
```python
def print_with_suffix(input_string, suffix):
    """
    Prints the input string with a suffix added to each line.

    Args:
        input_string (str): The string to print.
        suffix (str): The suffix to add to each line.
    """
    lines = input_string.split('\n')
    for line in lines:
        print(line + suffix)

def main():
    input_string = input("Please enter a string: ")
    suffix = input("Please enter a suffix: ")
    print_with_suffix(input_string, suffix)

if __name__ == "__main__":
    main()
```
