## Exercise
Create a function that maps a list of numbers to their squares.

## Reference Solution
```python
def map_to_squares(numbers):
    """
    Maps a list of numbers to their squares.

    Args:
        numbers (list): A list of numbers.

    Returns:
        list: A new list where each number is mapped to its square.
    """
    return [num ** 2 for num in numbers]

# Example usage:
def main():
    numbers = [1, 2, 3, 4, 5]
    squares = map_to_squares(numbers)
    print(squares)  # Output: [1, 4, 9, 16, 25]

if __name__ == "__main__":
    main()
```
