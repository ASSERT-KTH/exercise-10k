## Exercise
Extract a substring from a given string starting at a specified index and ending at another specified index.

## Reference Solution
```python
def extract_substring(input_string, start_index, end_index):
    """
    Extracts a substring from the input string starting at the specified start index and ending at the specified end index.

    Args:
        input_string (str): The original string.
        start_index (int): The starting index of the substring.
        end_index (int): The ending index of the substring.

    Returns:
        str: The extracted substring.
    """

    # First, we need to check if the input string is actually a string
    if not isinstance(input_string, str):
        raise TypeError("Input must be a string.")

    # Then, we need to check if the start and end indices are integers
    if not isinstance(start_index, int) or not isinstance(end_index, int):
        raise TypeError("Start and end indices must be integers.")

    # We also need to check if the start index is less than or equal to the end index
    if start_index > end_index:
        raise ValueError("Start index cannot be greater than end index.")

    # And we need to check if the start and end indices are within the bounds of the string
    if start_index < 0 or end_index > len(input_string):
        raise ValueError("Start or end index is out of range.")

    # Now, we can use Python's string slicing feature to extract the substring
    # The slicing operation will return a new string that includes all characters from the start index up to but not including the end index
    substring = input_string[start_index:end_index]

    # Finally, we return the extracted substring
    return substring

# Test the function
print(extract_substring("Hello, World!", 7, 12))  # Outputs: "World"
```
