## Exercise
Write a program that takes an angle as input and returns its sine.

## Reference Solution
```python
import math

def calculate_sine(angle_degrees):
    """
    Calculate the sine of an angle.

    Args:
        angle_degrees (float): The angle in degrees.

    Returns:
        float: The sine of the angle.
    """
    # Convert the angle from degrees to radians
    angle_radians = math.radians(angle_degrees)
    
    # Calculate the sine of the angle
    sine = math.sin(angle_radians)
    
    return sine

def main():
    # Get the angle from the user
    angle_degrees = float(input("Enter an angle in degrees: "))

    # Calculate and display the sine of the angle
    sine = calculate_sine(angle_degrees)
    print(f"The sine of {angle_degrees} degrees is {sine}")

if __name__ == "__main__":
    main()
```
