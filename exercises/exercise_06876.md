## Exercise
Write a program that takes a string input from the user and checks if it is a palindrome. A palindrome is a word, phrase, number, or other sequence of characters that reads the same forward and backward (ignoring spaces, punctuation, and capitalization). If the input string is a palindrome, print 'Palindrome' otherwise print 'Not a Palindrome'. The program should also reverse the input string and print the reversed string.

## Reference Solution
```python
import re

def is_palindrome(str):
    # Remove non-alphanumeric characters and convert to lowercase
    s = re.sub(r'\W+', '', str).lower()

    # Check if the string is the same when reversed
    return s == s[::-1]

# Test the function
print(is_palindrome("A man, a plan, a canal: Panama"))  # True
print(is_palindrome("Not a palindrome"))  # False

# Get user input
str = input("Enter a string: ")

# Check if the input string is a palindrome
if is_palindrome(str):
    print("Palindrome")
else:
    print("Not a Palindrome")

# Reverse the input string
reversed_str = str[::-1]
print("Reversed string:", reversed_str)
```
