## Exercise
Demonstrate implicit type conversion in a programming language, where the type of a variable is automatically converted.

## Reference Solution
```python
# Implicit conversion of integer to float
a = 5
b = 2.0
result = a + b
print(result)  # Output: 7.0
print(type(result))  # Output: <class 'float'>

# Implicit conversion of integer to complex
c = 5
d = 2 + 3j
result = c + d
print(result)  # Output: (7+3j)
print(type(result))  # Output: <class 'complex'>

# Implicit conversion of string to integer (not applicable)
try:
    e = "5"
    f = 2
    result = e + f
    print(result)
except TypeError as e:
    print(e)  # Output: can only concatenate str (not "int") to str

# Implicit conversion of boolean to integer
g = True
h = 2
result = g + h
print(result)  # Output: 3
print(type(result))  # Output: <class 'int'>
```
