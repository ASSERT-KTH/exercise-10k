## Exercise
Given two strings, determine if the first string matches the second string where the second string can contain wildcard characters. The function should take in two strings as input and return true if the first string matches the second string and false otherwise.

## Reference Solution
```python
def isMatch(s, p):
    dp = [[False] * (len(p) + 1) for _ in range(len(s) + 1)]
    dp[-1][-1] = True
    for i in range(len(s), -1, -1):
        for j in range(len(p) - 1, -1, -1):
            match = i < len(s) and (p[j] == s[i] or p[j] == '.')
            if p[j] == '*':
                dp[i][j] = dp[i][j + 1] or (i < len(s) and dp[i + 1][j])
            else:
                dp[i][j] = match and dp[i + 1][j + 1]
    return dp[0][0]
```
