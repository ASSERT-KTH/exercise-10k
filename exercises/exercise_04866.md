## Exercise
Write a program that generates the first 10 numbers in the Fibonacci sequence using a for loop.

## Reference Solution
```python
def generate_fibonacci(n):
    """
    Generate the first n numbers in the Fibonacci sequence.

    Args:
        n (int): The number of Fibonacci numbers to generate.

    Returns:
        list: A list of the first n Fibonacci numbers.
    """
    fib_sequence = [0, 1]
    for i in range(2, n):
        fib_sequence.append(fib_sequence[i-1] + fib_sequence[i-2])
    return fib_sequence

# Generate the first 10 Fibonacci numbers
n = 10
fib_numbers = generate_fibonacci(n)

# Print the Fibonacci numbers
print("The first {} Fibonacci numbers are:".format(n))
print(fib_numbers)
```
