## Exercise
Write a function that takes a name as an argument and prints a personalized greeting message.

## Reference Solution
```python
# Define a function called 'print_greeting' that takes one argument, 'name'.
def print_greeting(name):
    """
    This function prints a personalized greeting message.

    Args:
        name (str): The name of the person to be greeted.

    Returns:
        None
    """

    # First, we check if the input 'name' is a string. If not, we raise a TypeError.
    if not isinstance(name, str):
        raise TypeError("The input 'name' must be a string.")

    # Now, we use the print function to output a greeting message with the provided name.
    # We use an f-string to insert the 'name' variable into the string.
    print(f"Hello, {name}! Welcome to our program.")

# Test the function with a sample name.
print_greeting("John")
```
