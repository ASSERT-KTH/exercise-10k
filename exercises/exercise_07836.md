## Exercise
Write a program that checks if a given element exists in a list or array. The program should take two inputs: the list/array and the target element. It should return 'True' if the element exists in the list/array, and 'False' otherwise.

## Reference Solution
```python
def check_element_exists(input_list, target_element):
    """
    This function checks if a given element exists in a list.

    Args:
        input_list (list): The list to be searched.
        target_element: The element to be searched for.

    Returns:
        bool: True if the element exists in the list, False otherwise.
    """

    # First, we need to check if the input is a list. If it's not, we raise a TypeError.
    if not isinstance(input_list, list):
        raise TypeError("Input must be a list.")

    # Now, we use the 'in' keyword in Python, which checks if an element is present in a list.
    # The 'in' keyword returns True if the element is found, and False otherwise.
    return target_element in input_list


# Test the function
print(check_element_exists([1, 2, 3, 4, 5], 3))  # Should return True
print(check_element_exists([1, 2, 3, 4, 5], 6))  # Should return False
```
