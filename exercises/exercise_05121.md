## Exercise
Write a program to calculate the sum of all elements in an array.

## Reference Solution
```python
def calculate_array_sum(array):
    """
    Calculate the sum of all elements in an array.

    Args:
        array (list): A list of numbers.

    Returns:
        int: The sum of all elements in the array.
    """
    return sum(array)

# Example usage
def main():
    array = [1, 2, 3, 4, 5]
    print("Array:", array)
    print("Sum:", calculate_array_sum(array))

if __name__ == "__main__":
    main()
```
