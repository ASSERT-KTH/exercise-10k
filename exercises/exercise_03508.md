## Exercise
Write a program that demonstrates the difference between integer division and floating-point division.

## Reference Solution
```python
# Integer Division
def integer_division(dividend, divisor):
    if divisor == 0:
        raise ZeroDivisionError("Cannot divide by zero")
    return dividend // divisor

# Floating-point Division
def float_division(dividend, divisor):
    if divisor == 0:
        raise ZeroDivisionError("Cannot divide by zero")
    return dividend / divisor

# Test the functions
dividend = 10
divisor = 3

print(f"Integer division of {dividend} by {divisor}: {integer_division(dividend, divisor)}")
print(f"Floating-point division of {dividend} by {divisor}: {float_division(dividend, divisor)}")

# Another example with a different dividend and divisor
dividend = 7
divisor = 2

print(f"\nInteger division of {dividend} by {divisor}: {integer_division(dividend, divisor)}")
print(f"Floating-point division of {dividend} by {divisor}: {float_division(dividend, divisor)}")
```
