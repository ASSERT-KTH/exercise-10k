## Exercise
Write a program that takes in a user's input and determines the data type of the input. The program should be able to identify if the input is an integer, float, or string. If the input is not a valid number, it should be treated as a string.

## Reference Solution
```python
class Solution:
    def __init__(self):
        pass

    def get_type(self, value):
        try:
            int(value)
            return "integer"
        except ValueError:
            try:
                float(value)
                return "float"
            except ValueError:
                return "string"

    def main(self):
        while True:
            value = input("Enter a value (or 'q' to quit): ")
            if value.lower() == 'q':
                break
            print(self.get_type(value))

if __name__ == "__main__":
    solution = Solution()
    solution.main()
```
