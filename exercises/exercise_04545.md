## Exercise
Create a program that splits a given string into substrings based on a delimiter.

## Reference Solution
```python
def split_string(input_str, delimiter):
    """
    Splits a string into substrings based on a delimiter.

    Args:
        input_str (str): The input string to be split.
        delimiter (str): The delimiter to split the string by.

    Returns:
        list: A list of substrings split by the delimiter.
    """
    return input_str.split(delimiter)

def main():
    # Example usage
    input_str = "hello,world,python,programming"
    delimiter = ","
    substrings = split_string(input_str, delimiter)
    
    print("Input String:", input_str)
    print("Delimiter:", delimiter)
    print("Substrings:", substrings)

if __name__ == "__main__":
    main()
```
