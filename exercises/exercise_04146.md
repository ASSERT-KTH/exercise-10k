## Exercise
Write a function to get a list of keys from a dictionary.

## Reference Solution
```python
def get_dict_keys(dictionary):
    """
    Returns a list of keys from a dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        list: A list of keys from the dictionary.
    """
    return list(dictionary.keys())

# Example usage
if __name__ == "__main__":
    my_dict = {"name": "John", "age": 30, "city": "New York"}
    keys = get_dict_keys(my_dict)
    print(keys)
```
