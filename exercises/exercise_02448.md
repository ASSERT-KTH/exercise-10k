## Exercise
Write a program that takes a list and an element as input and returns a new list containing the element inserted at a specified position.

## Reference Solution
```python
insertAtPosition = (list, element, position) => {
  const leftList = list.slice(0, position);
  const rightList = list.slice(position);
  const newList = [...leftList, element, ...rightList];
  return newList;
}

const list = [1, 2, 3, 4, 5];
const position = 2;
const element = 10;
const newList = insertAtPosition(list, element, position);
console.log(newList); // Output: [1, 2, 10, 3, 4, 5]
```
