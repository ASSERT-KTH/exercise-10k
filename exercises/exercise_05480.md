## Exercise
Write a program that takes an array as input and returns a new array containing only the unique elements of the original array.

## Reference Solution
```python
def unique_array(array):
    """
    This function takes an array as input, removes duplicate elements, and returns a new array.
    """
    unique = []
    for element in array:
        if element not in unique:
            unique.append(element)
    return unique

# Example usage:
array = [12, 24, 35, 24, 88, 120, 155, 88, 120, 155]
result = unique_array(array)
print(result)
```
