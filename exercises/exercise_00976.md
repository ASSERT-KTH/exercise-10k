## Exercise
Given two dictionaries, write a function to find the union between them, i.e., the key-value pairs that exist in either dictionary.

## Reference Solution
```python
def union_of_dictionaries(dict1, dict2):
    """
    This function finds the union between two dictionaries.
    
    Args:
    dict1 (dict): The first dictionary.
    dict2 (dict): The second dictionary.
    
    Returns:
    dict: A new dictionary containing the union of key-value pairs from both input dictionaries.
    """
    # Create a copy of the first dictionary to avoid modifying the original dictionary
    union_dict = dict1.copy()
    
    # Update the union dictionary with key-value pairs from the second dictionary
    union_dict.update(dict2)
    
    return union_dict

# Example usage:
dict1 = {'a': 1, 'b': 2, 'c': 3}
dict2 = {'c': 4, 'd': 5, 'e': 6}

union_dict = union_of_dictionaries(dict1, dict2)
print(union_dict)  # Output: {'a': 1, 'b': 2, 'c': 4, 'd': 5, 'e': 6}
```
