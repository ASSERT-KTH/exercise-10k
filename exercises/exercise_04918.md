## Exercise
Write a program that takes a password as input and prints whether it is strong (at least 8 characters, contains uppercase and lowercase letters, and at least one digit) or not.

## Reference Solution
```python
import re

def strong_password(password):
    if len(password) < 8:
        return False
    if not re.search("[a-z]", password):
        return False
    if not re.search("[A-Z]", password):
        return False
    if not re.search("[0-9]", password):
        return False
    return True

password = input("Enter a password: ")
if strong_password(password):
    print("The password is strong.")
else:
    print("The password is not strong.")
```
