## Exercise
Write a function that takes a list and a value as input, and returns the index of the value in the list if it exists

## Reference Solution
```python
def find_index(lst, value):
    """
    Find the index of a value in a list.

    Args:
        lst (list): The list to search in.
        value: The value to search for.

    Returns:
        int: The index of the value if found, -1 otherwise.
    """
    try:
        return lst.index(value)
    except ValueError:
        return -1

# Example usage:
numbers = [1, 2, 3, 4, 5]
print(find_index(numbers, 3))  # Output: 2
print(find_index(numbers, 6))  # Output: -1
```

def find_index(lst, value):
    """
    Find the index of a value in a list.

    Args:
        lst (list): The list to search in.
        value: The value to search for.

    Returns:
        int: The index of the value if found, -1 otherwise.
    """

        if x == value:
            return i
    return -1

# Example usage:
