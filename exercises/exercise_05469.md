## Exercise
Write a program that takes a list and an index as input and returns a new list with the element at the specified index removed.

## Reference Solution
```python
def remove_element(lst, i):
    new_lst = lst.copy()
    new_lst.pop(i)
    return new_lst

# Test the function
my_list = [1, 2, 3, 4, 5]
index = 2
print(remove_element(my_list, index))
```
