## Exercise
Create a list to store student grades, then calculate the average grade

## Reference Solution
```python
def calculate_average_grade(grades):
    """
    Calculate the average grade from a list of grades.

    Args:
        grades (list): A list of student grades.

    Returns:
        float: The average grade.
    """
    if not grades:
        return 0.0

    return sum(grades) / len(grades)


def main():
    # Create a list to store student grades
    student_grades = [85, 90, 78, 92, 88, 76, 95, 89]

    # Calculate the average grade
    average_grade = calculate_average_grade(student_grades)

    # Print the result
    print("Student Grades:", student_grades)
    print("Average Grade: {:.2f}".format(average_grade))


if __name__ == "__main__":
    main()
```
