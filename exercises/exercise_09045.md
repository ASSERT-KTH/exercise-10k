## Exercise
Create a function that takes two numbers as arguments and returns True if the first is greater than or equal to the second, False otherwise

## Reference Solution

def is_greater_or_equal(num1, num2):
    """
    Checks if the first number is greater than or equal to the second.

    Args:
        num1 (int or float): The first number.
        num2 (int or float): The second number.

    Returns:
        bool: True if num1 is greater than or equal to num2, False otherwise.
    """
    return num1 >= num2

# Example usage:
print(is_greater_or_equal(5, 3))  # True
print(is_greater_or_equal(3, 5))  # False
