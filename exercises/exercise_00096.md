## Exercise
Rotate an array of integers by a given number of positions

## Reference Solution
```python
def rotate_array(arr, num_positions):
    """
    Rotate an array of integers by a given number of positions.

    Args:
        arr (list): The input array of integers.
        num_positions (int): The number of positions to rotate.

    Returns:
        list: The rotated array.
    """
    num_positions = num_positions % len(arr)  # Handle cases where num_positions > len(arr)
    return arr[-num_positions:] + arr[:-num_positions]

# Example usage
arr = [1, 2, 3, 4, 5]
num_positions = 2
rotated_arr = rotate_array(arr, num_positions)
print(rotated_arr)  # Output: [4, 5, 1, 2, 3]
```
