## Exercise
Write a function to update a dictionary with key-value pairs from another dictionary using the update() method

## Reference Solution
```python
def update_dictionary(dict1, dict2):
    """
    Updates dict1 with key-value pairs from dict2.

    Args:
        dict1 (dict): The dictionary to be updated.
        dict2 (dict): The dictionary with the new key-value pairs.

    Returns:
        dict: The updated dictionary.
    """
    # Create a copy of dict1 to avoid modifying the original dictionary
    updated_dict = dict1.copy()
    
    # Update updated_dict with key-value pairs from dict2
    updated_dict.update(dict2)
    
    return updated_dict

# Example usage
if __name__ == "__main__":
    dict1 = {"a": 1, "b": 2, "c": 3}
    dict2 = {"c": 4, "d": 5, "e": 6}

    print("Original Dictionary 1:", dict1)
    print("Original Dictionary 2:", dict2)

    updated_dict = update_dictionary(dict1, dict2)
    print("Updated Dictionary:", updated_dict)
```
