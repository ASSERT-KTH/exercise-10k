## Exercise
Write a function that sorts a list in ascending order.

## Reference Solution
```python
def sort_list_ascending(input_list):
    """
    Sorts a list in ascending order.

    Args:
        input_list (list): The list to be sorted.

    Returns:
        list: The sorted list.
    """
    return sorted(input_list)

# Example usage:
numbers = [64, 34, 25, 12, 22, 11, 90]
print("Original list:", numbers)
sorted_numbers = sort_list_ascending(numbers)
print("Sorted list:", sorted_numbers)
```

```python
def sort_list_ascending_in_place(input_list):
    """
    Sorts a list in ascending order in-place.

    Args:
        input_list (list): The list to be sorted.
    """
    input_list.sort()

# Example usage:
numbers = [64, 34, 25, 12, 22, 11, 90]
```
