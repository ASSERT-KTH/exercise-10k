## Exercise
Create a function called greet that takes a name as input and returns a personalized greeting message. The greeting message should be in the format 'Hello, <name>!'. The function should handle both empty and non-empty strings as input.

## Reference Solution
```python
def greet(name):
    """
    This function generates a personalized greeting message based on the input name.
    
    Parameters:
    name (str): The name of the person to be greeted.
    
    Returns:
    str: A greeting message in the format 'Hello, <name>!'.
    """

    # First, we check if the input name is empty
    if not name:
        # If the name is empty, we return a default greeting message
        return 'Hello, World!'
    
    # If the name is not empty, we proceed with generating the personalized greeting message
    # We use string formatting to insert the name into the greeting message
    # The 'Hello, {}!' is a placeholder string where {} will be replaced by the name
    greeting_message = 'Hello, {}!'.format(name)
    
    # Finally, we return the generated greeting message
    return greeting_message

# Test the function with an empty string
print(greet(''))  # Output: Hello, World!

# Test the function with a non-empty string
print(greet('John'))  # Output: Hello, John!
```
