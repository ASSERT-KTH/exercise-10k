## Exercise
Create a program that uses functions to perform basic arithmetic operations. The program should have separate functions for addition, subtraction, multiplication, and division. It should also have a main function that takes user input for the operation and the numbers, then calls the corresponding function to perform the operation.

## Reference Solution
```python
def add(num1, num2):
    """
    This function adds two numbers.
    
    Parameters:
    num1 (float): The first number.
    num2 (float): The second number.
    
    Returns:
    float: The sum of num1 and num2.
    """
    return num1 + num2

def subtract(num1, num2):
    """
    This function subtracts num2 from num1.
    
    Parameters:
    num1 (float): The first number.
    num2 (float): The second number.
    
    Returns:
    float: The difference between num1 and num2.
    """
    return num1 - num2

def multiply(num1, num2):
    """
    This function multiplies two numbers.
    
    Parameters:
    num1 (float): The first number.
    num2 (float): The second number.
    
    Returns:
    float: The product of num1 and num2.
    """
    return num1 * num2

def divide(num1, num2):
    """
    This function divides num1 by num2.
    
    Parameters:
    num1 (float): The dividend.
    num2 (float): The divisor.
    
    Returns:
    float: The quotient of num1 and num2.
    
    Raises:
    ZeroDivisionError: If num2 is zero.
    """
    if num2 == 0:
        raise ZeroDivisionError("Cannot divide by zero.")
    return num1 / num2

def main():
    """
    This function takes user input for the operation and the numbers, 
    then calls the corresponding function to perform the operation.
    """
    print("Basic Arithmetic Operations")
    print("1. Addition")
    print("2. Subtraction")
    print("3. Multiplication")
    print("4. Division")
    
    choice = input("Enter your choice (1/2/3/4): ")
    
    if choice in ['1', '2', '3', '4']:
        num1 = float(input("Enter first number: "))
        num2 = float(input("Enter second number: "))
        
        if choice == '1':
            print(f"{num1} + {num2} = {add(num1, num2)}")
        elif choice == '2':
            print(f"{num1} - {num2} = {subtract(num1, num2)}")
        elif choice == '3':
            print(f"{num1} * {num2} = {multiply(num1, num2)}")
        elif choice == '4':
            try:
                print(f"{num1} / {num2} = {divide(num1, num2)}")
            except ZeroDivisionError as e:
                print(e)
    else:
        print("Invalid choice")

if __name__ == "__main__":
    main()
```
