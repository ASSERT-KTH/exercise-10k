## Exercise
Write a function to group a dictionary by its values

## Reference Solution
```python
from collections import defaultdict

def group_dict_by_values(d):
    """
    Group a dictionary by its values.

    Args:
        d (dict): The dictionary to group.

    Returns:
        dict: A dictionary where the keys are the unique values from the original dictionary
            and the values are lists of keys that correspond to each value.
    """
    grouped_dict = defaultdict(list)
    for key, value in d.items():
        grouped_dict[value].append(key)
    return dict(grouped_dict)

# Example usage:
d = {'a': 1, 'b': 2, 'c': 1, 'd': 3, 'e': 2, 'f': 1}
grouped = group_dict_by_values(d)
print(grouped)
```
