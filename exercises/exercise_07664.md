## Exercise
Write a function to find the next palindrome number after a given number.

## Reference Solution
```python
def next_palindrome(n):
    """
    Finds the next palindrome number after a given number.

    Args:
        n (int): The input number.

    Returns:
        int: The next palindrome number.
    """
    def is_palindrome(num):
        """Checks if a number is a palindrome."""
        return str(num) == str(num)[::-1]

    n += 1
    while not is_palindrome(n):
        n += 1
    return n

# Example usage:
if __name__ == "__main__":
    num = 123
    next_palindrome_num = next_palindrome(num)
    print(f"The next palindrome number after {num} is: {next_palindrome_num}")
```
