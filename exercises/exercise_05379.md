## Exercise
Write a program to zip two or more arrays into a single array of tuples.

## Reference Solution
```python
def zip_arrays(*args):
    """
    Zip two or more arrays into a single array of tuples.

    Args:
        *args: Variable number of arrays to be zipped.

    Returns:
        A list of tuples, where each tuple contains one element from each input array.
    """
    return list(zip(*args))

# Example usage
array1 = [1, 2, 3]
array2 = ['a', 'b', 'c']
array3 = [True, False, True]

zipped_array = zip_arrays(array1, array2, array3)
print(zipped_array)
```
