## Exercise
Create a program that takes a user input and checks its data type. The program should be able to identify whether the input is an integer, float, or string.

## Reference Solution
```python
def check_data_type(user_input):
    """
    Checks the data type of the user's input.

    Args:
        user_input (str): The user's input.

    Returns:
        str: The data type of the user's input.
    """

    # Try to convert the input to an integer
    try:
        int(user_input)
        return "integer"
    except ValueError:
        pass

    # Try to convert the input to a float
    try:
        float(user_input)
        return "float"
    except ValueError:
        pass

    # If the input is neither an integer nor a float, it's a string
    return "string"


def main():
    # Get the user's input
    user_input = input("Please enter a value: ")

    # Check the data type of the input
    data_type = check_data_type(user_input)

    # Print the data type
    print(f"The data type of '{user_input}' is {data_type}.")


if __name__ == "__main__":
    main()
```
