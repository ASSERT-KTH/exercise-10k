## Exercise
Create a variable of one data type and convert it to another data type, then print out the converted value.

## Reference Solution
```python
# Define a variable of one data type
original_value = "123"  # string data type

# Print the original value and its data type
print("Original Value:", original_value)
print("Original Data Type:", type(original_value))

# Convert the variable to another data type (integer)
converted_value = int(original_value)

# Print the converted value and its data type
print("Converted Value:", converted_value)
print("Converted Data Type:", type(converted_value))
```
