## Exercise
Write a program that calculates the volume of a cylinder given its radius and height.

## Reference Solution
```python
import math

def calculate_cylinder_volume(radius, height):
    """
    Calculates the volume of a cylinder given its radius and height.

    Args:
        radius (float): The radius of the cylinder.
        height (float): The height of the cylinder.

    Returns:
        float: The volume of the cylinder.
    """
    volume = math.pi * (radius ** 2) * height
    return volume

def main():
    # Get user input for radius and height
    radius = float(input("Enter the radius of the cylinder: "))
    height = float(input("Enter the height of the cylinder: "))

    # Calculate and display the volume
    volume = calculate_cylinder_volume(radius, height)
    print(f"The volume of the cylinder is: {volume:.2f} cubic units")

if __name__ == "__main__":
    main()
```
