## Exercise
Create a program that takes a number as input and returns the sum of its factors.

## Reference Solution
```python
def sum_of_factors(n):
    """
    Calculate the sum of factors for a given number.

    Args:
        n (int): The input number.

    Returns:
        int: The sum of factors of n.
    """
    factors = [i for i in range(1, n + 1) if n % i == 0]
    return sum(factors)

# Example usage:
num = int(input("Enter a number: "))
print("The sum of factors of", num, "is:", sum_of_factors(num))
```

            factors.add(i)
            factors.add(n // i)
