## Exercise
Write a program that takes a user input and determines its data type. The program should be able to identify integers, floats, and strings. It should then print out the data type of the input.

## Reference Solution
```python
def determine_data_type(input_value):
    """
    Determine the data type of the input value.
    """
    try:
        # Try to convert the input to an integer
        int(input_value)
        return "integer"
    except ValueError:
        try:
            # If it's not an integer, try to convert it to a float
            float(input_value)
            return "float"
        except ValueError:
            # If it's neither an integer nor a float, it's a string
            return "string"

def main():
    # Get user input
    user_input = input("Please enter a value: ")
    
    # Determine the data type of the input
    data_type = determine_data_type(user_input)
    
    # Print the data type
    print(f"The data type of the input '{user_input}' is {data_type}.")

if __name__ == "__main__":
    main()
```
