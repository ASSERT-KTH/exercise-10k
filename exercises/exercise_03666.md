## Exercise
Create a function that takes a name as input and prints a greeting, with a default name if none is provided.

## Reference Solution
```python
def print_greeting(name="World"):
    """
    Prints a greeting with the provided name.
    
    If no name is provided, it defaults to 'World'.
    
    Parameters:
    name (str): The name to be used in the greeting. Defaults to 'World'.
    """

    # First, we check if the name is not empty or None
    if name is not None and name != "":
        
        # If the name is provided, we print a greeting with the name
        print(f"Hello, {name}!")
    
    # If the name is not provided or is empty, we print a default greeting
    else:
        print("Hello, World!")

# Test the function
print_greeting("John")  # Outputs: Hello, John!
print_greeting()  # Outputs: Hello, World!
```
