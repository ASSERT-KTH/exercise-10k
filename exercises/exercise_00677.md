## Exercise
Given a dictionary {'name': 'Oliver', 'age': 30, 'city': 'Cairns'}, write a program to use the items() method to print all the key-value pairs in the dictionary.

## Reference Solution
```python
def print_dict_key_value_pairs(dictionary):
    """
    Prints all key-value pairs in the dictionary using items() method.
    
    Parameters:
    dictionary (dict): The dictionary to print key-value pairs from.
    """
    # Iterate over the dictionary items
    for key, value in dictionary.items():
        # Print each key-value pair
        print(f"{key}: {value}")

# Example dictionary
person_info = {'name': 'Oliver', 'age': 30, 'city': 'Cairns'}

# Print dictionary key-value pairs
print("Dictionary Key-Value Pairs:")
print_dict_key_value_pairs(person_info)
```
