## Exercise
Create a function that takes in a list of keys and a list of values, and returns a dictionary where each key is mapped to its corresponding value

## Reference Solution
```python
def create_dict(keys, values):
    """
    Creates a dictionary where each key is mapped to its corresponding value.

    Args:
        keys (list): A list of keys.
        values (list): A list of values.

    Returns:
        dict: A dictionary where each key is mapped to its corresponding value.

    Raises:
        ValueError: If the lengths of the keys and values lists do not match.
    """
    if len(keys) != len(values):
        raise ValueError("The lengths of the keys and values lists must match.")

    return dict(zip(keys, values))

# Example usage:
keys = ['name', 'age', 'city']
values = ['John Doe', 30, 'New York']
print(create_dict(keys, values))
```
