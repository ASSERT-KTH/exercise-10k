## Exercise
Write a function that takes two arrays as input, and returns their concatenation.

## Reference Solution

def concatenate_arrays(arr1, arr2):
    """
    Concatenates two input arrays.

    Args:
        arr1 (list): The first input array.
        arr2 (list): The second input array.

    Returns:
        list: The concatenation of arr1 and arr2.
    """
    return arr1 + arr2

# Example usage:
