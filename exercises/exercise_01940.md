## Exercise
Write a program that uses type hinting to specify the expected data type of a variable.

## Reference Solution
```python
def greeting(name: str) -> str:
    return 'Hello ' + name

print(greeting("John"))  # Outputs: Hello John
print(greeting(123))     # Outputs: Hello 123, but this will raise a warning in some IDEs or type checkers
```

```python
from typeguard import typechecked

@typechecked
def greeting(name: str) -> str:
    return 'Hello ' + name

print(greeting("John"))  # Outputs: Hello John
try:
    print(greeting(123))     # Raises a TypeError
except TypeError as e:
    print(e)
```
