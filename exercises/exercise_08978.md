## Exercise
Write a function that takes a string as input and returns True if the string contains only digits, False otherwise.

## Reference Solution
```python
def check_digits(s):
    """
    This function checks if a given string contains only digits.

    Args:
        s (str): The input string to be checked.

    Returns:
        bool: True if the string contains only digits, False otherwise.
    """

    # We'll use the built-in str.isdigit() method in Python, which returns True if all characters in the string are digits and there is at least one character, otherwise it returns False.
    return s.isdigit()

# Test the function
print(check_digits("12345"))  # Should return True
print(check_digits("123abc"))  # Should return False
```
