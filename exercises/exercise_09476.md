## Exercise
Write a program that takes an input string and converts it to different data types. The program should first check if the string can be converted to an integer, then a float, and finally a boolean. If the string cannot be converted to any of these types, it should be stored as a string.

## Reference Solution
```python
def convert_string_to_correct_type(input_string):
    try:
        converted_int = int(input_string)
        return converted_int
    except ValueError:
        try:
            converted_float = float(input_string)
            return converted_float
        except ValueError:
            if input_string.lower() == "true":
                return True
            elif input_string.lower() == "false":
                return False
            else:
                return input_string

# Examples
print(convert_string_to_correct_type("123"))  # Output: 123
print(convert_string_to_correct_type("123.45"))  # Output: 123.45
print(convert_string_to_correct_type("true"))  # Output: True
print(convert_string_to_correct_type("false"))  # Output: False
print(convert_string_to_correct_type("hello"))  # Output: "hello"
```
