## Exercise
Create a function that takes in a mathematical operation and two numbers, then performs the operation and returns the result.

## Reference Solution
```python
def perform_operation(operation, num1, num2):
    """
    Performs a mathematical operation on two numbers.

    Args:
        operation (str): The mathematical operation to perform. Supported operations are '+', '-', '*', '/'.
        num1 (float): The first number.
        num2 (float): The second number.

    Returns:
        float: The result of the operation.

    Raises:
        ValueError: If the operation is not supported.
        ZeroDivisionError: If the operation is division and the second number is zero.
    """
    if operation == '+':
        return num1 + num2
    elif operation == '-':
        return num1 - num2
    elif operation == '*':
        return num1 * num2
    elif operation == '/':
        if num2 == 0:
            raise ZeroDivisionError("Cannot divide by zero")
        return num1 / num2
    else:
        raise ValueError("Unsupported operation")

# Example usage
def main():
    print(perform_operation('+', 10, 5))  # Output: 15
    print(perform_operation('-', 10, 5))  # Output: 5
    print(perform_operation('*', 10, 5))  # Output: 50
    print(perform_operation('/', 10, 5))  # Output: 2.0

if __name__ == "__main__":
    main()
```
