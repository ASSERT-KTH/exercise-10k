## Exercise
Write a program that takes a string as input and returns the longest word in the string.

## Reference Solution
```python
def longest_word(input_string):
    """
    Returns the longest word in a given string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The longest word in the input string.
    """
    # Split the input string into words
    words = input_string.split()

    # Initialize the longest word and its length
    longest_word = max(words, key=len)

    return longest_word

def main():
    # Example usage
    input_string = "This is a sample string with multiple words"
    print("Input String:", input_string)
    print("Longest Word:", longest_word(input_string))

if __name__ == "__main__":
    main()
```
