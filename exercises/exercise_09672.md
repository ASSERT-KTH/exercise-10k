## Exercise
Write a program that asks for a sentence and reverses the order of characters in each word.

## Reference Solution
```python
def reverse_words_in_sentence(sentence):
    """
    Reverses the order of characters in each word of a sentence.

    Args:
        sentence (str): The input sentence.

    Returns:
        str: The sentence with each word's characters reversed.
    """
    words = sentence.split()  # Split the sentence into words
    reversed_words = [word[::-1] for word in words]  # Reverse each word
    reversed_sentence = ' '.join(reversed_words)  # Join the reversed words back into a sentence
    return reversed_sentence

def main():
    sentence = input("Please enter a sentence: ")
    reversed_sentence = reverse_words_in_sentence(sentence)
    print("Reversed sentence: ", reversed_sentence)

if __name__ == "__main__":
    main()
```
