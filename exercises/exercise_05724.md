## Exercise
Create a function that finds the maximum of three numbers.

## Reference Solution
```python
def max_of_three(a: float, b: float, c: float) -> float:
    """
    Returns the maximum of three numbers.

    Args:
        a (float): The first number.
        b (float): The second number.
        c (float): The third number.

    Returns:
        float: The maximum of the three numbers.
    """
    return max(a, b, c)

# Example usage:
if __name__ == "__main__":
    num1 = 10.5
    num2 = 20.2
    num3 = 15.7
    max_num = max_of_three(num1, num2, num3)
    print(f"The maximum of {num1}, {num2}, and {num3} is {max_num}")
```
