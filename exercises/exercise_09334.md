## Exercise
Write a program to demonstrate the concept of variable scope.

## Reference Solution
```python
# Global variable
global_variable = "I am a global variable"

def outer_function():
    # Local variable in outer function
    outer_variable = "I am a local variable in outer function"

    def inner_function():
        # Local variable in inner function
        inner_variable = "I am a local variable in inner function"
        print("Inner function:")
        print(f"  - global_variable: {global_variable}")
        print(f"  - outer_variable: {outer_variable}")
        print(f"  - inner_variable: {inner_variable}")

    print("Outer function:")
    print(f"  - global_variable: {global_variable}")
    print(f"  - outer_variable: {outer_variable}")
    # Trying to access inner_variable here will raise an error
    # print(f"  - inner_variable: {inner_variable}")
    inner_function()

# Trying to access outer_variable or inner_variable here will raise an error
# print(f"  - outer_variable: {outer_variable}")
# print(f"  - inner_variable: {inner_variable}")

outer_function()

print(f"  - global_variable: {global_variable}")

# print(f"  - outer_variable: {outer_variable}")
```
