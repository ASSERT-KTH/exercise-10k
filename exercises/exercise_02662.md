## Exercise
Write a program that takes a string and two integers as input, and returns the substring starting at the first integer and ending at the second integer.

## Reference Solution
```python
def substring(s, i, f):
    """
    Returns the substring of s starting at index i and ending at index f.
    """
    return s[i:f+1]

# Example usage:
s = "hello"
i = 1
f = 3
print(substring(s, i, f))  # Output: "ell"
```
