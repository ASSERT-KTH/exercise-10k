## Exercise
Write a function that takes a string and a delimiter as input and returns a list of substrings split by the delimiter.

## Reference Solution
```python
def split_string(input_str, delimiter):
    """
    This function splits a given input string into substrings based on a specified delimiter.

    Args:
        input_str (str): The input string to be split.
        delimiter (str): The delimiter used to split the input string.

    Returns:
        list: A list of substrings split by the delimiter.
    """

    # Check if the input string is not None and is of type string
    if not isinstance(input_str, str):
        raise TypeError("Input must be a string.")

    # Check if the delimiter is not None and is of type string
    if not isinstance(delimiter, str):
        raise TypeError("Delimiter must be a string.")

    # Use the built-in split method of Python strings to split the input string by the delimiter
    # This method returns a list of words in the string, using the delimiter as the separator
    substrings = input_str.split(delimiter)

    # Return the list of substrings
    return substrings

# Example usage:
print(split_string("hello,world,python", ","))  # Output: ['hello', 'world', 'python']
```
