## Exercise
Write a program to check whether a given string starts with a specified prefix.

## Reference Solution
```python
def check_prefix(string, prefix):
    """
    Checks if the given string starts with the specified prefix.

    Args:
        string (str): The input string to check.
        prefix (str): The prefix to look for.

    Returns:
        bool: True if the string starts with the prefix, False otherwise.
    """
    return string.startswith(prefix)

def main():
    # Example usage
    string = input("Enter a string: ")
    prefix = input("Enter a prefix: ")
    result = check_prefix(string, prefix)

    if result:
        print(f"'{string}' starts with '{prefix}'")
    else:
        print(f"'{string}' does not start with '{prefix}'")

if __name__ == "__main__":
    main()
```
