## Exercise
Create a function that takes a dictionary and returns a new dictionary where the key-value pairs are sorted by key.

## Reference Solution
```python
def sort_dict_by_key(input_dict):
    """
    This function takes a dictionary as input, sorts its key-value pairs by key, 
    and returns a new dictionary with the sorted key-value pairs.

    Args:
        input_dict (dict): The input dictionary to be sorted.

    Returns:
        dict: A new dictionary with key-value pairs sorted by key.
    """

    # First, we'll check if the input is indeed a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary")

    # We'll use the built-in sorted function to sort the dictionary items
    # The sorted function returns a new list containing all items from the input iterable (in this case, the dictionary items) in ascending order
    # We'll use a lambda function as the key to specify that we want to sort by the dictionary keys
    sorted_items = sorted(input_dict.items(), key=lambda x: x[0])

    # Now, we'll use the dict function to convert the sorted list of items back into a dictionary
    # The dict function takes an iterable of key-value pairs and returns a new dictionary
    # Since Python 3.7, dictionaries remember the order of items inserted, so this will give us a sorted dictionary
    sorted_dict = dict(sorted_items)

    # Finally, we'll return the sorted dictionary
    return sorted_dict

# Example usage:
input_dict = {'c': 3, 'b': 2, 'a': 1}
sorted_dict = sort_dict_by_key(input_dict)
print(sorted_dict)
```
