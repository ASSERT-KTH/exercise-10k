## Exercise
Write a program that takes a string as input and returns the compressed version of the string, where consecutive repeated characters are replaced by the character and the count of consecutive repetitions.

## Reference Solution
```python
def string_compression(input_str):
    compressed_str = ""
    count = 1

    for i in range(len(input_str) - 1):
        if input_str[i] == input_str[i + 1]:
            count += 1
        else:
            compressed_str += input_str[i] + str(count)
            count = 1

    compressed_str += input_str[-1] + str(count)

    return compressed_str

# Test the function
input_str = "aaaabcccaaa"
print(string_compression(input_str))  # Output: "a4b1c3a3"
```
