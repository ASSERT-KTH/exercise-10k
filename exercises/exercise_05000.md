## Exercise
Understand the concept of type casting and demonstrate it with an example.

## Reference Solution
```python
# Implicit type casting
a = 10  # integer
b = 5.5  # float
c = a + b  # implicit type casting of a to float
print(c)  # output: 15.5
print(type(c))  # output: <class 'float'>

# Explicit type casting
d = 10.5  # float
e = int(d)  # explicit type casting of d to integer
print(e)  # output: 10
print(type(e))  # output: <class 'int'>
```
