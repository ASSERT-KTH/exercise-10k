## Exercise
Write a program that takes an array and a predicate function as input and returns a new array containing only the elements that satisfy the predicate

## Reference Solution
```python
def filter_array(array, predicate):
    """
    Returns a new array containing only the elements that satisfy the predicate function.

    Args:
        array (list): The input array.
        predicate (function): A function that takes an element as input and returns a boolean.

    Returns:
        list: A new array containing only the elements that satisfy the predicate.
    """
    return [element for element in array if predicate(element)]
```

```python
# Define a predicate function that checks if a number is even
def is_even(x):
    return x % 2 == 0

# Create an input array
array = [1, 2, 3, 4, 5, 6]

# Filter the array using the predicate function
result = filter_array(array, is_even)

print(result)  # Output: [2, 4, 6]
```
