## Exercise
Find the median age of students in a dictionary where keys are student names and values are their corresponding ages

## Reference Solution
```python
import statistics

def find_median_age(students):
    """
    Calculate the median age of students.

    Args:
        students (dict): Dictionary where keys are student names and values are their corresponding ages.

    Returns:
        float: The median age of students.
    """
    # Extract ages from the dictionary
    ages = list(students.values())

    # Calculate the median age
    median_age = statistics.median(ages)

    return median_age

# Example usage
students = {
    "John": 20,
    "Alice": 22,
    "Bob": 21,
    "Charlie": 20,
    "David": 23
}

median_age = find_median_age(students)
print("Median age:", median_age)
```
