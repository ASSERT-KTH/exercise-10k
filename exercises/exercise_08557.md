## Exercise
Create a dictionary with 3 key-value pairs and get a value by its key.

## Reference Solution
```python
# Create a dictionary with 3 key-value pairs
my_dict = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

# Get a value by its key
def get_value_by_key(dictionary, key):
    """Retrieve a value from a dictionary by its key."""
    return dictionary.get(key)

# Example usage:
key_to_find = "age"
value = get_value_by_key(my_dict, key_to_find)

print(f"The value for key '{key_to_find}' is: {value}")
```
