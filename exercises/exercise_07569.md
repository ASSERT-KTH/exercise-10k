## Exercise
Create a function that raises an exception when given invalid input.

## Reference Solution
```python
def validate_input(input_value):
    """
    This function validates the input value and raises an exception if it's invalid.

    Args:
        input_value (int): The input value to be validated.

    Raises:
        ValueError: If the input value is not a positive integer.
    """

    # First, we check if the input value is an integer
    if not isinstance(input_value, int):
        # If it's not an integer, we raise a ValueError with a descriptive message
        raise ValueError("Input value must be an integer.")

    # Then, we check if the input value is positive
    if input_value <= 0:
        # If it's not positive, we raise a ValueError with a descriptive message
        raise ValueError("Input value must be a positive integer.")

    # If the input value passes both checks, we return True to indicate that it's valid
    return True

# Example usage:
try:
    validate_input(-5)
except ValueError as e:
    print(e)

try:
    validate_input("hello")
except ValueError as e:
    print(e)

try:
    validate_input(10)
    print("Input is valid")
except ValueError as e:
    print(e)
```
