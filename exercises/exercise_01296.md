## Exercise
Write a function that takes a dictionary and a function, and returns the result of applying the function to all values in the dictionary, in a cumulative way, so as to reduce the dictionary to a single output value.

def cumulative_reduce(dictionary, func):
    """
    This function applies a given function to all values in a dictionary in a cumulative way.
    
    Args:
        dictionary (dict): The input dictionary.
        func (function): The function to be applied to the dictionary values.
        
    Returns:
        result: The cumulative result of applying the function to all dictionary values.
    """
    
    # Check if the dictionary is empty
    if not dictionary:
        # If it is, return None
        return None
    
    # Initialize the result with the first value in the dictionary
    # We use the values() method to get a list of the dictionary's values
    # We assume that the function can handle the type of the first value
    result = next(iter(dictionary.values()))
    
    # Iterate over the remaining values in the dictionary
    # We use the values() method to get a list of the dictionary's values
    # We start from the second value (index 1) since we've already used the first one
    for value in list(dictionary.values())[1:]:
        # Apply the function to the current result and the current value
        # We update the result with the new value
        result = func(result, value)
    
    # Return the final result
    return result
