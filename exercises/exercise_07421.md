## Exercise
Create a program that takes a user's input and converts it into different data types. The program should be able to convert the input into an integer, float, and string. If the input cannot be converted into a specific data type, the program should display an error message.

## Reference Solution
```python
def convert_input():
    # Take user's input
    user_input = input("Please enter a value: ")

    # Try to convert the input into an integer
    try:
        int_value = int(user_input)
        print(f"Integer value: {int_value}")
    except ValueError:
        print("Error: Input cannot be converted into an integer.")

    # Try to convert the input into a float
    try:
        float_value = float(user_input)
        print(f"Float value: {float_value}")
    except ValueError:
        print("Error: Input cannot be converted into a float.")

    # Try to convert the input into a string
    try:
        str_value = str(user_input)
        print(f"String value: {str_value}")
    except Exception as e:
        print(f"Error: An unexpected error occurred - {str(e)}")

convert_input()
```
