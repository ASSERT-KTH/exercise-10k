## Exercise
Write a program that checks a person's age and prints out a message based on their age group using if/else statements

## Reference Solution
```python
def check_age(age):
    """
    Checks a person's age and prints out a message based on their age group.

    Args:
    age (int): The person's age.

    Returns:
    None
    """
    if age < 0:
        print("Age cannot be negative.")
    elif age <= 12:
        print("You are a kid.")
    elif age <= 19:
        print("You are a teenager.")
    elif age <= 64:
        print("You are an adult.")
    else:
        print("You are a senior citizen.")

def main():
    age = int(input("Enter your age: "))
    check_age(age)

if __name__ == "__main__":
    main()
```
