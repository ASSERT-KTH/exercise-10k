## Exercise
Write a program that asks the user for a string and then prints out its upper and lower case versions.

## Reference Solution
```python
def print_case_versions(input_string):
    """Prints upper and lower case versions of the input string."""
    print(f"Upper case: {input_string.upper()}")
    print(f"Lower case: {input_string.lower()}")

def main():
    user_string = input("Please enter a string: ")
    print_case_versions(user_string)

if __name__ == "__main__":
    main()
```
