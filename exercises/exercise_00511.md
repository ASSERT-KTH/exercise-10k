## Exercise
Write a program to insert a new element into a given list at a specific position.

## Reference Solution
```python
def insert_at_position(lst, new_element, position):
    """
    Inserts a new element into the list at the specified position.
    
    Parameters:
    lst (list): The input list.
    new_element: The element to be inserted.
    position (int): The position at which the element is to be inserted.
    
    Returns:
    list: The updated list with the new element inserted.
    """
    
    # Check if the position is within the list bounds
    if position < 0 or position > len(lst):
        raise IndexError("Position out of range")
    
    # Insert the new element at the specified position
    lst.insert(position, new_element)
    
    return lst

# Example usage
numbers = [1, 2, 3, 5, 6]
new_number = 4
position = 3

updated_numbers = insert_at_position(numbers, new_number, position)
print(updated_numbers)  # Output: [1, 2, 3, 4, 5, 6]
```
