## Exercise
Write a program that calculates the ceiling function of a given number.

## Reference Solution
```python
import math

def calculate_ceiling(number):
    """
    Calculate the ceiling function of a given number.

    Args:
        number (float): The input number.

    Returns:
        int: The ceiling of the input number.
    """
    return math.ceil(number)

# Example usage
def main():
    num = 4.7
    print(f"The ceiling of {num} is {calculate_ceiling(num)}")

if __name__ == "__main__":
    main()
```
