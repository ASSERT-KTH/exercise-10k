## Exercise
Write a function that returns a list of all key-value pairs in a dictionary.

## Reference Solution
```python
def get_key_value_pairs(dictionary):
    """
    This function takes a dictionary as input and returns a list of all key-value pairs in the dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        list: A list of tuples, where each tuple contains a key-value pair from the dictionary.
    """

    # First, we check if the input is indeed a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("Input must be a dictionary")

    # We use the items() method of the dictionary to get a view object that displays a list of all key-value pairs
    # This view object is a dict_items object, which is a view object that displays a list of all key-value pairs in a dictionary
    key_value_pairs = dictionary.items()

    # We convert the dict_items object to a list, which is what the function is supposed to return
    # We use the list() function to convert the dict_items object to a list
    key_value_pairs_list = list(key_value_pairs)

    # Finally, we return the list of key-value pairs
    return key_value_pairs_list

# Example usage:
my_dict = {"name": "John", "age": 30, "city": "New York"}
print(get_key_value_pairs(my_dict))
```
