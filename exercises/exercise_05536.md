## Exercise
Declare an array variable, assign values to it, and then print its values.

## Reference Solution
```python
# Import the array module
import array

# Declare an array variable
# 'i' is the typecode for signed integer
my_array = array.array('i')

# Assign values to the array
my_array.append(10)
my_array.append(20)
my_array.append(30)

# Print the values in the array
print("Values in the array:")
for value in my_array:
    print(value)

# Alternatively, you can use the print function directly on the array
print("\nArray values using print function:")
print(my_array)
```
