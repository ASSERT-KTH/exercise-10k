## Exercise
Write a function to delete a key-value pair from a dictionary.

## Reference Solution
```python
def delete_key_value_pair(dictionary, key):
    """
    Deletes a key-value pair from a dictionary.

    Args:
        dictionary (dict): The dictionary from which to delete the key-value pair.
        key: The key of the key-value pair to delete.

    Returns:
        dict: The updated dictionary with the key-value pair deleted.
    """
    if key in dictionary:
        del dictionary[key]
    return dictionary

# Example usage:
my_dict = {"name": "John", "age": 30, "city": "New York"}
print("Original dictionary:", my_dict)

updated_dict = delete_key_value_pair(my_dict, "age")
print("Updated dictionary:", updated_dict)
```
