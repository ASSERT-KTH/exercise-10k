## Exercise
Write a function that takes a variable number of arguments and returns their sum. Then, call this function with a list of input numbers and print the result.

## Reference Solution
```python
# Define a function that takes a variable number of arguments
def sum_args(*args):
    """
    This function takes a variable number of arguments and returns their sum.

    Args:
        *args: A variable number of arguments

    Returns:
        The sum of all input arguments
    """
    # Initialize the sum to 0
    total = 0
    # Iterate over each argument
    for arg in args:
        # Add the argument to the total
        total += arg
    # Return the total
    return total

# Create a list of input numbers
input_numbers = [1, 2, 3, 4, 5]

# Call the function with the input numbers and print the result
print(sum_args(*input_numbers))
```
