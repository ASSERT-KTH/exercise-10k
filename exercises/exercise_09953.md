## Exercise
Write a program that converts a string to an integer.

## Reference Solution
```python
def string_to_int(s):
    """
    Convert a string to an integer.

    Args:
        s (str): The string to be converted.

    Returns:
        int: The integer equivalent of the string.

    Raises:
        ValueError: If the string cannot be converted to an integer.
    """
    try:
        return int(s)
    except ValueError:
        print("Error: The string cannot be converted to an integer.")
        return None

# Example usage:
def main():
    string_input = input("Enter a string: ")
    result = string_to_int(string_input)
    if result is not None:
        print(f"The integer equivalent of '{string_input}' is: {result}")

if __name__ == "__main__":
    main()
```
