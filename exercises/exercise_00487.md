## Exercise
Write a program that takes two arrays as input, and returns a new array containing all elements from both arrays.

## Reference Solution
```python
function combinarListas(lista1, lista2) {
  const lista3 = [...lista1, ...lista2];
  return lista3;
}

const lista1 = [1, 2, 3, 4];
const lista2 = [5, 6, 7, 8];

const novaLista = combinarListas(lista1, lista2);
console.log(novaLista);
```
