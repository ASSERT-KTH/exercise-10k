## Exercise
Write a function that takes a string as input and returns the reversed string. The function should be case-sensitive and preserve the original casing of the input string.

## Reference Solution
```python
def reverse_string(input_str):
    """
    This function takes a string as input and returns the reversed string.
    The function is case-sensitive and preserves the original casing of the input string.

    Parameters:
    input_str (str): The input string to be reversed.

    Returns:
    str: The reversed string.
    """

    # We start by checking if the input is indeed a string
    if not isinstance(input_str, str):
        raise TypeError("Input must be a string.")

    # Now, we use Python's slicing feature to reverse the string
    # The syntax [::-1] means start at the end of the string and end at position 0, move with the step -1
    # This effectively reverses the string
    reversed_str = input_str[::-1]

    # Finally, we return the reversed string
    return reversed_str
```
