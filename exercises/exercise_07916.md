## Exercise
Create a program to calculate the perimeter of a circle given its radius.

## Reference Solution
```python
import math

def calculate_circle_perimeter(radius):
    """
    Calculate the perimeter of a circle given its radius.

    Args:
        radius (float): The radius of the circle.

    Returns:
        float: The perimeter of the circle.
    """
    if radius < 0:
        raise ValueError("Radius cannot be negative")
    perimeter = 2 * math.pi * radius
    return perimeter

def main():
    # Example usage
    radius = float(input("Enter the radius of the circle: "))
    try:
        perimeter = calculate_circle_perimeter(radius)
        print(f"The perimeter of the circle is: {perimeter:.2f}")
    except ValueError as e:
        print(e)

if __name__ == "__main__":
    main()
```
