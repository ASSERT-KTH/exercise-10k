## Exercise
Create a function that implements the linear search algorithm to find the index of a given element in a list of numbers. The function should take a list and a target element as arguments and return the index of the target element if it exists, or -1 otherwise.

## Reference Solution
```python
def linear_search(nums, target):
    """
    This function implements the linear search algorithm to find the index of a given element in a list of numbers.

    Args:
        nums (list): A list of numbers.
        target (int): The target element to be searched.

    Returns:
        int: The index of the target element if it exists, or -1 otherwise.
    """

    # Initialize the index to -1, assuming the target element does not exist in the list
    index = -1
    
    # Iterate over the list with the enumerate function to get both the index and value of each element
    for i, num in enumerate(nums):
        
        # Check if the current element is equal to the target element
        if num == target:
            
            # If the target element is found, update the index and break the loop
            index = i
            break
    
    # Return the index of the target element
    return index

# Example usage:
numbers = [1, 2, 3, 4, 5]
target_element = 3
print(linear_search(numbers, target_element))  # Output: 2
```
