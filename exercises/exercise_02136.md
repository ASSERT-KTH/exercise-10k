## Exercise
Write a function that takes a list and a chunk size as input and returns a new list containing chunks of the specified size.

## Reference Solution
```python
def chunk_list(input_list, chunk_size):
    """
    This function takes a list and a chunk size as input, 
    and returns a new list containing chunks of the specified size.

    Args:
        input_list (list): The list to be chunked.
        chunk_size (int): The size of each chunk.

    Returns:
        list: A list of chunks.
    """

    # Initialize an empty list to store the chunks
    chunks = []

    # Initialize a variable to keep track of the current index in the input list
    i = 0

    # Loop through the input list until we've processed all elements
    while i < len(input_list):
        
        # Calculate the end index of the current chunk
        # We use the minimum of i + chunk_size and len(input_list) to avoid going out of bounds
        end = min(i + chunk_size, len(input_list))
        
        # Append the current chunk to the list of chunks
        # We use list slicing to get the current chunk from the input list
        chunks.append(input_list[i:end])
        
        # Move on to the next chunk
        i += chunk_size
    
    # Return the list of chunks
    return chunks

# Example usage:
input_list = [1, 2, 3, 4, 5, 6, 7, 8, 9]
chunk_size = 3
print(chunk_list(input_list, chunk_size))  # Output: [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
```
