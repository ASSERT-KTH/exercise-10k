## Exercise
Create a dictionary with 5 key-value pairs. Calculate the sum of all the values in the dictionary.

## Reference Solution
```python
def calculate_dict_sum():
    # Create a dictionary with 5 key-value pairs
    my_dict = {
        "key1": 10,
        "key2": 20,
        "key3": 30,
        "key4": 40,
        "key5": 50
    }

    # Calculate the sum of all values in the dictionary
    total_sum = sum(my_dict.values())

    return total_sum

# Example usage:
if __name__ == "__main__":
    result = calculate_dict_sum()
    print("Sum of dictionary values:", result)
```
