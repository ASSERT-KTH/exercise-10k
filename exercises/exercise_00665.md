## Exercise
Given a dictionary {'name': 'Charlie', 'age': 30, 'city': 'Sydney'}, write a program to print all the values in the dictionary.

## Reference Solution
```python
def print_dict_values(dictionary):
    """
    Prints all values in a given dictionary.

    Args:
        dictionary (dict): The input dictionary.
    """
    for value in dictionary.values():
        print(value)


# Example usage
if __name__ == "__main__":
    person = {'name': 'Charlie', 'age': 30, 'city': 'Sydney'}
    print("Dictionary Values:")
    print_dict_values(person)
```
