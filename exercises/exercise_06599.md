## Exercise
Write a program to find the longest common prefix of two strings.

## Reference Solution
```python
def longest_common_prefix(str1, str2):
    """
    Returns the longest common prefix of two strings.

    Args:
        str1 (str): The first string.
        str2 (str): The second string.

    Returns:
        str: The longest common prefix.
    """
    prefix = ""
    for char1, char2 in zip(str1, str2):
        if char1 == char2:
            prefix += char1
        else:
            break
    return prefix

# Example usage:
def main():
    str1 = "hello world"
    str2 = "hello python"
    print("Longest common prefix:", longest_common_prefix(str1, str2))

if __name__ == "__main__":
    main()
```
