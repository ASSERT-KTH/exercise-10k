## Exercise
Generate a sequence of numbers based on a given formula or pattern.

## Reference Solution
```python
def fibonacci(n):
    sequence = [0, 1]
    while len(sequence) < n:
        sequence.append(sequence[-1] + sequence[-2])
    return sequence

print(fibonacci(10))
```
