## Exercise
Create a function that takes in a dictionary and returns a new dictionary containing the same key-value pairs, but with the keys converted to lowercase

## Reference Solution
```python
def convert_keys_to_lowercase(input_dict):
    """
    Creates a new dictionary with the same key-value pairs, 
    but with the keys converted to lowercase.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        dict: A new dictionary with lowercase keys.
    """
    return {key.lower(): value for key, value in input_dict.items()}

# Example usage:
input_dict = {"Name": "John", "Age": 30, "City": "New York"}
new_dict = convert_keys_to_lowercase(input_dict)
print(new_dict)  # Output: {'name': 'John', 'age': 30, 'city': 'New York'}
```
