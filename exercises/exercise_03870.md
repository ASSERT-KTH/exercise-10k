## Exercise
Create a dictionary with names of departments as keys and dictionaries of employee names and positions as values. Print the resulting dictionary.

## Reference Solution
```python
# Define the dictionary with department names as keys
# and dictionaries of employee names and positions as values
department_employees = {
    "Sales": {
        "John Doe": "Sales Manager",
        "Jane Smith": "Sales Representative",
        "Bob Johnson": "Sales Assistant"
    },
    "Marketing": {
        "Alice Brown": "Marketing Manager",
        "Mike Davis": "Marketing Specialist",
        "Emily Chen": "Marketing Coordinator"
    },
    "IT": {
        "David Lee": "IT Manager",
        "Sarah Taylor": "Software Developer",
        "Kevin White": "Network Administrator"
    }
}

# Print the resulting dictionary
print(department_employees)
```
