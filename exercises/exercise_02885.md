## Exercise
Create a program to calculate the result of exponentiation (e.g., 2^3) using a loop.

## Reference Solution
```python
def calculate_exponentiation(base, exponent):
    """
    Calculate the result of exponentiation using a loop.

    Args:
        base (int): The base number.
        exponent (int): The exponent.

    Returns:
        int: The result of the exponentiation.
    """
    result = 1
    for _ in range(abs(exponent)):
        result *= base

    # Handle negative exponents
    if exponent < 0:
        return 1 / result
    else:
        return result


def main():
    # Example usage
    base = 2
    exponent = 3
    print(f"{base}^{exponent} = {calculate_exponentiation(base, exponent)}")

    # Test with negative exponent
    base = 2
    exponent = -3
    print(f"{base}^{exponent} = {calculate_exponentiation(base, exponent)}")


if __name__ == "__main__":
    main()
```
