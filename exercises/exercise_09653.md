## Exercise
Write a program that asks for a number and prints it out.

## Reference Solution
```python
def print_number():
    # Ask for a number
    while True:
        try:
            num = float(input("Please enter a number: "))
            break
        except ValueError:
            print("Invalid input. Please enter a valid number.")

    # Print the number
    print("You entered:", num)

def main():
    print_number()

if __name__ == "__main__":
    main()
```
