## Exercise
Write a program that uses a for loop with a step of -2 to print the numbers from 10 to 0.

## Reference Solution
```python
def print_numbers():
    # Use range function to generate numbers from 10 to 0 with a step of -2
    for i in range(10, -1, -2):
        print(i)

# Call the function to print the numbers
print_numbers()
```
