## Exercise
Create a program that uses a nested for loop to print a multiplication table from 1 to n.

## Reference Solution
```python
def print_multiplication_table(n):
    """
    Prints a multiplication table from 1 to n.

    Args:
        n (int): The upper limit of the multiplication table.
    """
    # Print the header row
    print("   |", end="")
    for i in range(1, n + 1):
        print(f"{i:4d}", end="")
    print()
    print("---" + "---" * n)

    # Print the multiplication table
    for i in range(1, n + 1):
        print(f"{i:3d}|", end="")
        for j in range(1, n + 1):
            print(f"{i * j:4d}", end="")
        print()


def main():
    n = int(input("Enter the value of n: "))
    print_multiplication_table(n)


if __name__ == "__main__":
    main()
```
