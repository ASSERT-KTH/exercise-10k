## Exercise
Create a function to alternate the case of characters in a given string.

## Reference Solution
```python
def alternate_case(s):
    """
    Alternate the case of characters in a given string.

    Args:
        s (str): The input string.

    Returns:
        str: The string with alternating character case.
    """
    result = ""
    upper = True  # Start with upper case

    for char in s:
        if char.isalpha():  # Check if the character is a letter
            if upper:
                result += char.upper()
            else:
                result += char.lower()
            upper = not upper  # Alternate the case
        else:
            result += char  # Non-letter characters are not changed

    return result

# Example usage:
def main():
    input_str = "Hello, World!"
    print("Input:", input_str)
    print("Alternating case:", alternate_case(input_str))

if __name__ == "__main__":
    main()
```
