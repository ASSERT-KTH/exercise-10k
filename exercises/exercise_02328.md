## Exercise
Write a program to take a string input from the user and print it to the screen, but only if the input is not empty.

## Reference Solution
```python
def print_non_empty_string():
    user_input = input("Please enter a string: ")
    if user_input.strip() != "":
        print(user_input)

print_non_empty_string()
```

```python
def print_non_empty_string():
    while True:
        user_input = input("Please enter a string: ")
        if user_input.strip() != "":
            print(user_input)
            break
```
