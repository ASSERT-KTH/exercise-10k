## Exercise
Write a program that checks if a given number is positive, negative, or zero using if statements

## Reference Solution
```python
def check_number(num):
    if num > 0:
        print(f"{num} is a positive number.")
    elif num < 0:
        print(f"{num} is a negative number.")
    else:
        print(f"{num} is zero.")

# Example usage:
num = float(input("Enter a number: "))
check_number(num)
```
