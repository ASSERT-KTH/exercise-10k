## Exercise
Demonstrate the differences between integer, floating-point, and complex numbers in a programming language.

## Reference Solution
```python
# Integer
integer_num = 10
print(f"Integer: {integer_num}, Type: {type(integer_num)}")

# Floating-Point Number
float_num = 3.14
print(f"Floating-Point Number: {float_num}, Type: {type(float_num)}")

# Complex Number
complex_num = 3 + 4j
print(f"Complex Number: {complex_num}, Type: {type(complex_num)}")

# Operations
print("\nOperations:")
print("Integer + Integer:", 5 + 3)
print("Floating-Point + Floating-Point:", 2.5 + 1.8)
print("Complex + Complex:", (2 + 3j) + (4 + 5j))

# Type Conversions
print("\nType Conversions:")
print("Integer to Float:", float(10))
print("Float to Integer:", int(3.14))
print("Integer to Complex:", complex(10))
print("Float to Complex:", complex(3.14))
```
