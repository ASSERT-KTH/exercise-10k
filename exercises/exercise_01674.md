## Exercise
Develop a program that takes a phone number as input and checks if it is valid based on the following criteria: the phone number should have exactly 10 digits, and the first digit should not be zero using if/else statements

## Reference Solution
```python
def check_phone_number(phone_number):
    """
    Checks if a phone number is valid.
    
    A phone number is valid if it has exactly 10 digits and the first digit is not zero.
    
    Parameters:
    phone_number (str): The phone number to check.
    
    Returns:
    bool: True if the phone number is valid, False otherwise.
    """

    # Remove any non-digit characters from the phone number
    digits_only = ''.join(filter(str.isdigit, phone_number))
    
    # Check if the phone number has exactly 10 digits
    if len(digits_only) != 10:
        return False
    
    # Check if the first digit is not zero
    if digits_only[0] == '0':
        return False
    
    # If all checks pass, the phone number is valid
    return True


def main():
    phone_number = input("Enter a phone number: ")
    
    if check_phone_number(phone_number):
        print("The phone number is valid.")
    else:
        print("The phone number is not valid.")


if __name__ == "__main__":
    main()
```
