## Exercise
Create a program that takes a user's input and determines its data type. The input can be an integer, float, or string. The program should then print out the data type of the input.

## Reference Solution
```python
def determine_data_type(user_input):
    """
    Determine the data type of the user's input.
    
    Args:
        user_input (str): The input provided by the user.
    
    Returns:
        str: The data type of the user's input.
    """

    # Try to convert the input to an integer
    try:
        # If the input can be converted to an integer, it's an integer
        int(user_input)
        return "Integer"
    
    # If the input can't be converted to an integer, try to convert it to a float
    except ValueError:
        try:
            # If the input can be converted to a float, it's a float
            float(user_input)
            return "Float"
        
        # If the input can't be converted to a float, it's a string
        except ValueError:
            return "String"


def main():
    # Get the user's input
    user_input = input("Please enter a value: ")
    
    # Determine the data type of the input
    data_type = determine_data_type(user_input)
    
    # Print the data type
    print(f"The data type of '{user_input}' is {data_type}.")


if __name__ == "__main__":
    main()
```
