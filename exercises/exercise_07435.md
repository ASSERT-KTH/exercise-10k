## Exercise
Write a program that takes in a user's input and determines the data type of the input. The input can be either an integer, a float, or a string. The program should then print out the data type of the input.

## Reference Solution
```python
# Author: [Your Name]
# Date: [Current Date]
# Description: This program takes in a user's input and determines the data type of the input.
# Note: The input can be either an integer, a float, or a string.

user_input = input("Enter a value: ")

if user_input.isdigit():
    print("The data type of the input is: integer")
elif user_input.replace('.', '', 1).isdigit():
    print("The data type of the input is: float")
else:
    print("The data type of the input is: string")
```
