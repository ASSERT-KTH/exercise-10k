## Exercise
Write a program that converts a given string to uppercase and prints out the result.

## Reference Solution
```python
def convert_to_uppercase(input_string):
    """
    Converts a given string to uppercase and prints out the result.

    Args:
        input_string (str): The string to be converted.
    """
    uppercase_string = input_string.upper()
    print(uppercase_string)

def main():
    # Example usage
    input_str = "Hello, World!"
    print(f"Original string: {input_str}")
    print("Converting to uppercase...")
    convert_to_uppercase(input_str)

if __name__ == "__main__":
    main()
```
