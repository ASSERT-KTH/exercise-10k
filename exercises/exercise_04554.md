## Exercise
Write a program that counts the occurrences of a given character in a string.

## Reference Solution
```python
def count_char(string, char):
    """
    Counts the occurrences of a given character in a string.

    Args:
        string (str): The input string.
        char (str): The character to count.

    Returns:
        int: The number of occurrences of the character in the string.
    """
    return string.count(char)

def main():
    # Example usage
    string = "Hello, World!"
    char = "l"
    count = count_char(string, char)
    print(f"The character '{char}' occurs {count} times in the string '{string}'.")

if __name__ == "__main__":
    main()
```
