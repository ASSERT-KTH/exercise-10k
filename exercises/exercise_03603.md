## Exercise
Create a function that takes one number as an argument and another with a default value, then returns their sum

## Reference Solution
```python
def calculate_sum(num1, num2=5):
    """
    Calculate the sum of two numbers.

    Args:
        num1 (int): The first number.
        num2 (int, optional): The second number. Defaults to 5.

    Returns:
        int: The sum of num1 and num2.
    """
    return num1 + num2

# Example usage
print(calculate_sum(10))  # Output: 15 (10 + 5)
print(calculate_sum(10, 3))  # Output: 13 (10 + 3)
```
