## Exercise
Create a function that takes a string and a number as arguments and returns the string shifted by that many positions in the alphabet

## Reference Solution
```python
def caesar_cipher(text: str, shift: int) -> str:
    """
    Shifts each letter in the input string by a specified number of positions in the alphabet.

    Args:
        text (str): The input string to be shifted.
        shift (int): The number of positions to shift the string.

    Returns:
        str: The shifted string.
    """
    result = ""

    for char in text:
        if char.isalpha():
            ascii_offset = 65 if char.isupper() else 97
            result += chr((ord(char) - ascii_offset + shift) % 26 + ascii_offset)
        else:
            result += char

    return result

# Example usage:
if __name__ == "__main__":
    text = "Hello, World!"
    shift = 3
    print(f"Original text: {text}")
    print(f"Shifted text: {caesar_cipher(text, shift)}")
```
