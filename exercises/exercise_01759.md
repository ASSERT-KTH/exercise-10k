## Exercise
Given two lists of integers, write a program to find their symmetric difference.

## Reference Solution
```python
def symmetric_difference(list1, list2):
    """
    Returns the symmetric difference of two lists.
    
    :param list1: The first list of integers.
    :param list2: The second list of integers.
    :return: A list of integers that are in exactly one of the input lists.
    """
    # Convert lists to sets for efficient set operations
    set1 = set(list1)
    set2 = set(list2)
    
    # Calculate the symmetric difference using set operations
    symmetric_diff = set1.symmetric_difference(set2)
    
    # Convert the result back to a list and return it
    return list(symmetric_diff)

# Example usage:
list1 = [1, 2, 3, 4]
list2 = [3, 4, 5, 6]
print(symmetric_difference(list1, list2))  # Output: [1, 2, 5, 6]
```
