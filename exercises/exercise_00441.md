## Exercise
Write a program that validates user input to ensure it is a positive integer. If the input is invalid, prompt the user to enter a valid number.

## Reference Solution
```python
def get_positive_integer():
    """
    This function will keep asking the user for input until a positive integer is entered.
    """
    while True:
        try:
            # Try to convert the input to an integer
            num = int(input("Please enter a positive integer: "))
            # Check if the integer is positive
            if num > 0:
                return num
            else:
                print("The number is not positive. Please try again.")
        except ValueError:
            print("That's not a valid integer. Please try again.")

def main():
    # Call the function and store the result in a variable
    positive_integer = get_positive_integer()
    print(f"You have entered a valid positive integer: {positive_integer}")

if __name__ == "__main__":
    main()
```
