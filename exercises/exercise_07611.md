## Exercise
Write a program that uses a for loop to print the numbers from 1 to 10.

## Reference Solution
```python
# Define the range of numbers to print
start = 1
end = 10

# Use a for loop to print the numbers
for i in range(start, end + 1):
    print(i)
```
