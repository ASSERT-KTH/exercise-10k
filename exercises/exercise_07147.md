## Exercise
Create a function that takes a list of integers and a mapping function as input. The function should apply the mapping function to each element in the list and return the resulting list. The mapping function should take an integer and return its square.

## Reference Solution
```python
def square_mapping_function(number):
    """
    This function takes an integer as input and returns its square.
    
    Args:
        number (int): The input integer.
    
    Returns:
        int: The square of the input integer.
    """
    # The function simply returns the square of the input number
    return number ** 2


def apply_mapping_function(numbers, mapping_function):
    """
    This function applies a given mapping function to each element in a list of integers.
    
    Args:
        numbers (list): A list of integers.
        mapping_function (function): A function that takes an integer and returns a transformed integer.
    
    Returns:
        list: A list of integers where each element is the result of applying the mapping function to the corresponding element in the input list.
    """
    # Initialize an empty list to store the results
    result = []
    
    # Iterate over each number in the input list
    for number in numbers:
        # Apply the mapping function to the current number and append the result to the result list
        result.append(mapping_function(number))
    
    # Return the result list
    return result


# Example usage:
numbers = [1, 2, 3, 4, 5]
result = apply_mapping_function(numbers, square_mapping_function)
print(result)  # Output: [1, 4, 9, 16, 25]
```
