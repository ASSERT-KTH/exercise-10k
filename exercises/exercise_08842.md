## Exercise
Create a function that takes a list of numbers as input and returns the sum of all the numbers in the list

## Reference Solution
```python
def sum_list_numbers(numbers):
    """
    Calculate the sum of all numbers in a list.

    Args:
        numbers (list): A list of numbers.

    Returns:
        int or float: The sum of all numbers in the list.

    Raises:
        TypeError: If the input is not a list or if the list contains non-numeric values.
    """
    if not isinstance(numbers, list):
        raise TypeError("Input must be a list.")
    try:
        return sum(numbers)
    except TypeError:
        raise TypeError("List must only contain numeric values.")

# Example usage
numbers = [1, 2, 3, 4, 5]
result = sum_list_numbers(numbers)
print("Sum of list numbers:", result)
```
