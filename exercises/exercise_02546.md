## Exercise
Create a function that takes a variable number of arguments and returns their sum.

## Reference Solution
```python
def sum_args(*args):
    """
    This function takes a variable number of arguments and returns their sum.

    Args:
        *args: A variable number of arguments to be summed up.

    Returns:
        The sum of all the arguments passed to the function.
    """

    # Initialize the sum variable to 0. This variable will store the sum of all the arguments.
    total_sum = 0
    
    # Iterate over each argument in the args tuple.
    for arg in args:
        # Add the current argument to the total sum.
        total_sum += arg
    
    # Return the total sum after iterating over all the arguments.
    return total_sum

# Test the function with some examples.
print(sum_args(1, 2, 3, 4, 5))  # Output: 15
print(sum_args(10, 20, 30))  # Output: 60
print(sum_args(5, 5, 5, 5, 5))  # Output: 25
```
