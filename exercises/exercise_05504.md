## Exercise
Write a program to calculate the sum of all elements in an array of integers.

## Reference Solution
```python
def sum_of_elements(array):
    """
    This function calculates the sum of all elements in an array of integers.

    Parameters:
    array (list): A list of integers.

    Returns:
    int: The sum of all elements in the array.
    """
    return sum(array)


# Example usage:
array = [1, 2, 3, 4, 5]
print("Array:", array)
print("Sum of elements:", sum_of_elements(array))
```
