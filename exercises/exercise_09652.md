## Exercise
Write a program that asks for the user's name and prints out a greeting message.

## Reference Solution
```python
def get_user_name():
    """Asks the user for their name and returns it."""
    return input("Please enter your name: ")

def print_greeting(name):
    """Prints a greeting message for the given name."""
    print(f"Hello, {name}!")

def main():
    """Main function that runs the program."""
    user_name = get_user_name()
    print_greeting(user_name)

if __name__ == "__main__":
    main()
```
