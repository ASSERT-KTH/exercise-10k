## Exercise
Write a function to sort a list of integers in descending order.

## Reference Solution
```python
def sort_descending(nums):
    """
    Sorts a list of integers in descending order.

    Args:
        nums (list): A list of integers.

    Returns:
        list: The sorted list of integers in descending order.
    """
    return sorted(nums, reverse=True)

# Example usage:
if __name__ == "__main__":
    numbers = [5, 2, 9, 1, 7, 3]
    sorted_numbers = sort_descending(numbers)
    print(sorted_numbers)  # Output: [9, 7, 5, 3, 2, 1]
```

```python
def sort_descending_in_place(nums):
    """
    Sorts a list of integers in descending order in-place.

    Args:
        nums (list): A list of integers.
    """
    nums.sort(reverse=True)

# Example usage:
if __name__ == "__main__":
    numbers = [5, 2, 9, 1, 7, 3]
    sort_descending_in_place(numbers)
    print(numbers)  # Output: [9, 7, 5, 3, 2, 1]
```
