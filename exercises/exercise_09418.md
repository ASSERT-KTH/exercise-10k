## Exercise
Create a function that takes a number and returns 'even' if it's even, 'odd' if it's odd

## Reference Solution
```python
def even_or_odd(num: int) -> str:
    """
    Returns 'even' if the input number is even, 'odd' if it's odd.

    Args:
        num (int): The input number.

    Returns:
        str: 'even' or 'odd'.
    """
    if num % 2 == 0:
        return 'even'
    else:
        return 'odd'

# Example usage:
print(even_or_odd(10))  # Output: even
print(even_or_odd(11))  # Output: odd
```
