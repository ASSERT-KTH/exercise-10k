## Exercise
Write a program that takes a list of strings as input and prints out the strings joined into a single string based on a delimiter.

## Reference Solution
```python
#!/usr/bin/env 
# -*- coding: utf-8 -*-

__author__ = 'Cristian S. Rocha'

def join_strings(delimiter, strings):
    if delimiter == '':
        return delimiter.join(strings)
    else:
        return delimiter.join(strings)

if __name__ == '__main__':
    import sys
    delimiter = sys.argv[1]
    strings = sys.argv[2:]
    print(join_strings(delimiter, strings))
```
