## Exercise
Write a program that takes a string as input and returns True if the string is a palindrome and False otherwise.

## Reference Solution
```python
def is_palindrome(s: str) -> bool:
    """
    Checks if a string is a palindrome.

    Args:
        s (str): The input string.

    Returns:
        bool: True if the string is a palindrome, False otherwise.
    """
    s = ''.join(c for c in s if c.isalnum()).lower()  # Remove non-alphanumeric characters and convert to lowercase
    return s == s[::-1]  # Compare the string with its reverse

def main():
    # Example usage
    strings = ["radar", "hello", "A man, a plan, a canal: Panama"]
    for s in strings:
        print(f"'{s}' is a palindrome: {is_palindrome(s)}")

if __name__ == "__main__":
    main()
```
