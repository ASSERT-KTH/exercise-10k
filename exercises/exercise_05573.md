## Exercise
Create a program that takes a word as input and generates all possible anagrams of the word.

## Reference Solution
```python
import itertools

def generate_anagrams(word):
    """
    Generate all possible anagrams of a word.

    Args:
    word (str): The input word.

    Returns:
    list: A list of all possible anagrams.
    """
    # Remove any whitespace and convert to lowercase
    word = word.replace(" ", "").lower()
    
    # Generate all permutations of the word
    perms = itertools.permutations(word)
    
    # Convert each permutation to a string and store in a list
    anagrams = [''.join(p) for p in perms]
    
    # Remove duplicates by converting to a set and then back to a list
    anagrams = list(set(anagrams))
    
    return anagrams

def main():
    word = input("Enter a word: ")
    anagrams = generate_anagrams(word)
    print("Anagrams:")
    for anagram in anagrams:
        print(anagram)

if __name__ == "__main__":
    main()
```
