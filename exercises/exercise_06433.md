## Exercise
Create a program to print a chessboard pattern using nested for loops. The pattern should consist of alternating black and white squares.

## Reference Solution
```python
def print_chessboard(size):
    """
    Prints a chessboard pattern of given size.

    Args:
        size (int): Size of the chessboard.

    Returns:
        None
    """
    for i in range(size):
        for j in range(size):
            # Check if the square should be black or white
            if (i + j) % 2 == 0:
                print("B ", end="")  # Black square
            else:
                print("W ", end="")  # White square
        print()  # Move to the next line


def main():
    size = 8  # Default size of a chessboard
    print_chessboard(size)


if __name__ == "__main__":
    main()
```
