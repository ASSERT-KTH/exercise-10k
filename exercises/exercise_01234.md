## Exercise
Create a program that uses a for loop to print each element of a given tuple.

## Reference Solution
```python
def print_tuple_elements(tuple_to_print):
    """
    Prints each element of a given tuple.

    Args:
        tuple_to_print (tuple): The tuple to be printed.
    """
    for element in tuple_to_print:
        print(element)

# Example usage:
if __name__ == "__main__":
    # Define a tuple
    example_tuple = (1, 2, 3, 4, 5)
    
    # Print the tuple elements
    print("Elements of the tuple:")
    print_tuple_elements(example_tuple)
```
