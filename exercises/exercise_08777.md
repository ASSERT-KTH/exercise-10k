## Exercise
Write a function to find the common characters between two given strings.

## Reference Solution
```python
def common_characters(str1, str2):
    """
    Find the common characters between two given strings.

    Args:
    str1 (str): The first string.
    str2 (str): The second string.

    Returns:
    list: A list of common characters.
    """
    # Convert the strings to sets of characters
    set1 = set(str1)
    set2 = set(str2)

    # Find the intersection of the two sets
    common_chars = set1.intersection(set2)

    # Convert the set back to a list and return it
    return list(common_chars)

# Example usage
def main():
    str1 = "hello"
    str2 = "world"
    print(f"Common characters between '{str1}' and '{str2}': {common_characters(str1, str2)}")

if __name__ == "__main__":
    main()
```
