## Exercise
Write a program that checks if a student has passed or failed based on their score. If the score is 50 or above, the student has passed. Otherwise, they have failed.

## Reference Solution
```python
def check_pass_or_fail(score):
    """
    Checks if a student has passed or failed based on their score.

    Args:
        score (int): The score of the student.

    Returns:
        str: 'Passed' if the score is 50 or above, 'Failed' otherwise.
    """
    if score >= 50:
        return 'Passed'
    else:
        return 'Failed'

def main():
    score = float(input("Enter the student's score: "))
    result = check_pass_or_fail(score)
    print(f"The student has {result}.")

if __name__ == "__main__":
    main()
```
