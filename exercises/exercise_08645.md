## Exercise
Create a function that finds the index of a given element in a given list.

## Reference Solution
```python
def find_index(lst, elem):
    """
    Finds the index of a given element in a given list.

    Args:
        lst (list): The list to search in.
        elem: The element to search for.

    Returns:
        int: The index of the element in the list, or -1 if not found.
    """
    try:
        return lst.index(elem)
    except ValueError:
        return -1

# Example usage
def main():
    my_list = [1, 2, 3, 4, 5]
    print(find_index(my_list, 3))  # Output: 2
    print(find_index(my_list, 6))  # Output: -1

if __name__ == "__main__":
    main()
```
