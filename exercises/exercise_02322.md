## Exercise
Write a program to take a string input from the user and print it to the screen.

## Reference Solution
```python
def get_user_input():
    """Get string input from the user"""
    user_input = input("Please enter a string: ")
    return user_input

def print_to_screen(input_string):
    """Print the input string to the screen"""
    print("You entered:", input_string)

def main():
    user_input = get_user_input()
    print_to_screen(user_input)

if __name__ == "__main__":
    main()
```
