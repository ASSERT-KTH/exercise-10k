## Exercise
Write a program that takes a list and a transformation function as input, and returns a new list containing the transformed elements.

## Reference Solution
```python
def transform_list(input_list, transformation_function):
    """
    Applies a given transformation function to each element of a list and returns a new list containing the transformed elements.

    Args:
        input_list (list): The input list to be transformed.
        transformation_function (function): A function that takes one argument and returns a transformed value.

    Returns:
        list: A new list containing the transformed elements.
    """

    # Initialize an empty list to store the transformed elements.
    transformed_list = []

    # Iterate over each element in the input list.
    for element in input_list:
        # Apply the transformation function to the current element and store the result in a variable.
        transformed_element = transformation_function(element)
        
        # Append the transformed element to the transformed list.
        transformed_list.append(transformed_element)

    # Return the new list containing the transformed elements.
    return transformed_list

# Example usage:
# Define a transformation function that doubles a number.
def double_number(num):
    return num * 2

# Create a list of numbers.
numbers = [1, 2, 3, 4, 5]

# Transform the list of numbers by doubling each number.
transformed_numbers = transform_list(numbers, double_number)

# Print the transformed list.
print(transformed_numbers)  # Output: [2, 4, 6, 8, 10]
```
