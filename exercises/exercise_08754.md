## Exercise
Create a function to extract a substring from a given string.

## Reference Solution
```python
def extract_substring(input_string, start_index, end_index):
    """
    Extracts a substring from the input string.

    Args:
        input_string (str): The string from which to extract the substring.
        start_index (int): The starting index of the substring (inclusive).
        end_index (int): The ending index of the substring (exclusive).

    Returns:
        str: The extracted substring.

    Raises:
        ValueError: If the start index is greater than the end index.
        IndexError: If the start or end index is out of range.
    """
    if start_index > end_index:
        raise ValueError("Start index cannot be greater than end index")
    if start_index < 0 or end_index > len(input_string):
        raise IndexError("Start or end index is out of range")

    return input_string[start_index:end_index]


# Example usage:
if __name__ == "__main__":
    input_str = "Hello, World!"
    start_idx = 7
    end_idx = 12

    try:
        substring = extract_substring(input_str, start_idx, end_idx)
        print(f"Extracted substring: {substring}")
    except (ValueError, IndexError) as e:
        print(f"Error: {e}")
```
