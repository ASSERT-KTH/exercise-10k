## Exercise
Write a program to find the maximum sum of a subarray within a given array of numbers.

## Reference Solution
```python
def max_subarray_sum(arr):
    """
    Find the maximum sum of a subarray within a given array of numbers.

    Args:
        arr (list): A list of integers.

    Returns:
        int: The maximum sum of a subarray.
    """
    max_sum = float('-inf')  # Initialize max_sum to negative infinity
    current_sum = 0  # Initialize current_sum to 0

    for num in arr:
        current_sum = max(num, current_sum + num)  # Update current_sum
        max_sum = max(max_sum, current_sum)  # Update max_sum

    return max_sum

# Example usage
def main():
    arr = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
    max_sum = max_subarray_sum(arr)
    print("Maximum subarray sum:", max_sum)

if __name__ == "__main__":
    main()
```
