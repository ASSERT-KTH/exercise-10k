## Exercise
Write a function to use the clear() method to remove all key-value pairs from a dictionary.

## Reference Solution
```python
def clear_dictionary(dictionary):
    """
    Removes all key-value pairs from a dictionary using the clear() method.

    Args:
        dictionary (dict): The dictionary to be cleared.

    Returns:
        dict: The cleared dictionary.
    """
    # Check if the input is a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("Input must be a dictionary.")

    # Use the clear() method to remove all key-value pairs
    dictionary.clear()

    return dictionary


# Example usage:
my_dict = {"a": 1, "b": 2, "c": 3}
print("Original Dictionary:", my_dict)

cleared_dict = clear_dictionary(my_dict)
print("Cleared Dictionary:", cleared_dict)
```
