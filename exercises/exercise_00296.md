## Exercise
Create a program that calculates the absolute value of a given number.

## Reference Solution
```python
def calculate_absolute_value(num):
    """
    Calculate the absolute value of a given number.

    Args:
        num (int or float): The input number.

    Returns:
        int or float: The absolute value of the input number.
    """
    return abs(num)

def main():
    # Example usage
    numbers = [5, -3, 0, -10, 7.5, -2.5]

    for num in numbers:
        absolute_value = calculate_absolute_value(num)
        print(f"The absolute value of {num} is {absolute_value}")

if __name__ == "__main__":
    main()
```
