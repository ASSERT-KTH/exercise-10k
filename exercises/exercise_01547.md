## Exercise
Write a function to reverse the order of elements in a given list of integers.

## Reference Solution
```python
def reverse_list(input_list):
    """
    Reverses the order of elements in a given list of integers.

    Args:
        input_list (list): The list of integers to be reversed.

    Returns:
        list: The reversed list of integers.
    """
    return input_list[::-1]

# Example usage
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    print("Original List:", numbers)
    reversed_numbers = reverse_list(numbers)
    print("Reversed List:", reversed_numbers)
```
