## Exercise
Create a function that takes an array as input, and returns a new array containing the elements of the input array in sorted order.

## Reference Solution
```python
def sort_array(input_array):
    """
    Returns a new array containing the elements of the input array in sorted order.

    Args:
        input_array (list): The input array to be sorted.

    Returns:
        list: A new array containing the elements of the input array in sorted order.
    """
    # Create a copy of the input array to avoid modifying it directly
    array_copy = input_array.copy()

    # Use the built-in sorted function to sort the array in ascending order
    sorted_array = sorted(array_copy)

    # Return the sorted array
    return sorted_array

# Example usage:
input_array = [64, 34, 25, 12, 22, 11, 90]
print("Original array:", input_array)
print("Sorted array:", sort_array(input_array))
```
