## Exercise
Write a function that takes a variable number of arguments and returns their sum

## Reference Solution
```python
def sum_args(*args):
    """
    Returns the sum of all input arguments.

    Args:
        *args: A variable number of numeric arguments.

    Returns:
        The sum of all input arguments.

    Raises:
        TypeError: If a non-numeric argument is passed.
    """
    try:
        return sum(args)
    except TypeError:
        raise TypeError("All arguments must be numeric.")

# Example usage:
print(sum_args(1, 2, 3, 4, 5))  # Output: 15
print(sum_args(10, 20, 30))     # Output: 60
```
