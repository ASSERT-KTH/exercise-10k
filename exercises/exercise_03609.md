## Exercise
Create a function that uses memoization to optimize the calculation of the factorial of a number

## Reference Solution
```python
def memoized_factorial(n, memo={}):
    """
    Calculate the factorial of a number using memoization.

    Args:
    n (int): The number to calculate the factorial of.
    memo (dict): A dictionary to store the results of previous calculations.

    Returns:
    int: The factorial of the given number.
    """
    if n < 0:
        raise ValueError("n must be a non-negative integer")
    elif n == 0 or n == 1:
        return 1
    elif n in memo:
        return memo[n]
    else:
        result = n * memoized_factorial(n-1, memo)
        memo[n] = result
        return result

# Example usage:
print(memoized_factorial(5))  # Output: 120
print(memoized_factorial(10))  # Output: 3628800
```
