## Exercise
Write a program to demonstrate the error that occurs when trying to convert a variable from one data type to another that is not compatible.

## Reference Solution
```python
# Define a function to demonstrate the error
def demonstrate_incompatible_conversion():
    # Try to convert a string to an integer
    try:
        string_var = "Hello, World!"
        integer_var = int(string_var)
        print(f"Converted string to integer: {integer_var}")
    except ValueError as e:
        print(f"Error occurred: {e}")

    # Try to convert a list to a float
    try:
        list_var = [1, 2, 3]
        float_var = float(list_var)
        print(f"Converted list to float: {float_var}")
    except TypeError as e:
        print(f"Error occurred: {e}")

# Call the function to demonstrate the error
demonstrate_incompatible_conversion()
```
