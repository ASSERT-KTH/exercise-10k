## Exercise
Create a function that takes an email address as an argument and returns True if it is valid (contains '@') and False otherwise.

## Reference Solution
```python
def is_valid_email(email):
    """
    This function checks if an email address is valid by checking if it contains '@'.

    Args:
        email (str): The email address to be checked.

    Returns:
        bool: True if the email address is valid, False otherwise.
    """

    # First, we need to check if the input is a string. If it's not, we immediately return False.
    if not isinstance(email, str):
        return False
    
    # Now, we'll check if the email contains '@'. We can use the 'in' operator in Python for this.
    # The 'in' operator checks if a substring is present in a string and returns True if it is, False otherwise.
    return '@' in email

# Test the function
print(is_valid_email("test@example.com"))  # Should return True
print(is_valid_email("invalid_email"))  # Should return False
```
