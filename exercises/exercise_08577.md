## Exercise
Write a function to get all values from a dictionary

## Reference Solution
```python
def get_all_values(dictionary):
    """
    Retrieves all values from a dictionary.

    Args:
        dictionary (dict): The dictionary to retrieve values from.

    Returns:
        list: A list of all values in the dictionary.
    """
    return list(dictionary.values())


# Example usage:
example_dict = {
    "key1": "value1",
    "key2": "value2",
    "key3": "value3"
}

all_values = get_all_values(example_dict)
print(all_values)
```
