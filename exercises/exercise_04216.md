## Exercise
Remove duplicates from an array while preserving the original order.

## Reference Solution
```python
def remove_duplicates(arr):
    """
    Removes duplicates from an array while preserving the original order.
    
    Args:
        arr (list): The input array.
    
    Returns:
        list: The array with duplicates removed.
    """
    seen = set()
    result = []
    
    for element in arr:
        if element not in seen:
            seen.add(element)
            result.append(element)
    
    return result

# Example usage:
if __name__ == "__main__":
    arr = [1, 2, 3, 4, 2, 5, 6, 7, 8, 9, 5]
    print("Original Array:", arr)
    print("Array without Duplicates:", remove_duplicates(arr))
```
