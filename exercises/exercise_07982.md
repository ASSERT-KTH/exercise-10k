## Exercise
Write a program that finds the minimum value in a list of numbers.

## Reference Solution
```python
def find_min_value(numbers):
    """
    Finds the minimum value in a list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        int: The minimum value in the list.
    """
    if not numbers:
        raise ValueError("List is empty")

    min_value = numbers[0]
    for num in numbers:
        if num < min_value:
            min_value = num

    return min_value

# Example usage
numbers = [5, 2, 9, 1, 7, 3]
min_value = find_min_value(numbers)
print(f"Minimum value: {min_value}")  # Output: Minimum value: 1
```
