## Exercise
Write a function to convert a dictionary to a list of lists

## Reference Solution
```python
def dict_to_list_of_lists(input_dict):
    """
    Convert a dictionary to a list of lists, where each inner list contains a key-value pair.

    Args:
        input_dict (dict): The dictionary to convert.

    Returns:
        list: A list of lists, where each inner list contains a key-value pair from the dictionary.
    """
    return [[key, value] for key, value in input_dict.items()]


# Example usage:
if __name__ == "__main__":
    example_dict = {"name": "John", "age": 30, "city": "New York"}
    result = dict_to_list_of_lists(example_dict)
    print(result)
```
