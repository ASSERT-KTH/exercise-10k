## Exercise
Write a program that prints a pattern of asterisks (*) based on user input. The pattern should have a height of n rows, where n is the user's input. Each row should have one more asterisk than the previous row.

## Reference Solution
```python
import sys

def pattern(n):
    for i in range(1, n+1):
        print("*" * (i + 1))

if __name__ == "__main__":
    n = int(input("Enter a number: "))
    pattern(n)
```
