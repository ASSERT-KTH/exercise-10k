## Exercise
Write a program that iterates over a dictionary and prints its key-value pairs

## Reference Solution
```python
# Define a dictionary
my_dict = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

# Iterate over the dictionary and print key-value pairs
for key, value in my_dict.items():
    print(f"{key}: {value}")
```

```python
# Iterate over the dictionary's keys
for key in my_dict.keys():
    print(key)

# Iterate over the dictionary's values
for value in my_dict.values():
    print(value)
```
