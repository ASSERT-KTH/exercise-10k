## Exercise
Understand the concepts of NaN and Infinity by creating examples of each and printing their values.

## Reference Solution
```python
import math

# NaN (Not a Number) examples
print("NaN examples:")
print("0/0 =", 0/0)  # This will print 'nan' (Not a Number)
print("math.sqrt(-1) =", math.sqrt(-1))  # This will also print 'nan'

# Infinity examples
print("\nInfinity examples:")
print("1/0 =", 1/0)  # This will print 'inf' (Infinity)
print("math.inf =", math.inf)  # This will also print 'inf'
print("-math.inf =", -math.inf)  # This will print '-inf' (Negative Infinity)

# Checking for NaN and Infinity
print("\nChecking for NaN and Infinity:")
print("math.isnan(math.sqrt(-1)) =", math.isnan(math.sqrt(-1)))  # This will print 'True'
print("math.isinf(1/0) =", math.isinf(1/0))  # This will print 'True'
print("math.isinf(-1/0) =", math.isinf(-1/0))  # This will print 'True'
```
