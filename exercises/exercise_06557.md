## Exercise
Create a function that removes all duplicate characters from a given string.

## Reference Solution
```python
def remove_duplicates(input_string):
    """
    Removes all duplicate characters from a given string.

    Args:
        input_string (str): The input string.

    Returns:
        str: A new string with all duplicate characters removed.
    """
    seen = set()
    output_string = ""

    for char in input_string:
        if char not in seen:
            output_string += char
            seen.add(char)

    return output_string

# Example usage:
def main():
    input_str = "hello world"
    print("Original string:", input_str)
    print("String without duplicates:", remove_duplicates(input_str))

if __name__ == "__main__":
    main()
```
