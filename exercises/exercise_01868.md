## Exercise
Write a program to apply a transformation to each element of a list and print the resulting list.

## Reference Solution
```python
def transform_list(input_list, transformation):
    """
    Applies a transformation to each element of a list.

    Args:
        input_list (list): The list to be transformed.
        transformation (function): A function that takes one argument and returns a transformed value.

    Returns:
        list: The transformed list.
    """
    return [transformation(x) for x in input_list]

def square(x):
    """
    A simple transformation function that squares its input.

    Args:
        x (int or float): The number to be squared.

    Returns:
        int or float: The square of x.
    """
    return x ** 2

# Example usage:
numbers = [1, 2, 3, 4, 5]
transformed_numbers = transform_list(numbers, square)
print("Original list:", numbers)
print("Transformed list:", transformed_numbers)
```
