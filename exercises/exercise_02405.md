## Exercise
Write a function that takes a single argument and returns its type. Then, call this function with an input value and print the result.

## Reference Solution
```python
def get_type(input_value):
    """
    This function takes an input value and returns its type.

    Args:
        input_value (any): The input value to get the type of.

    Returns:
        type: The type of the input value.
    """

    # First, we need to use the built-in type() function in Python, which returns the type of an object.
    # We will store this type in a variable for clarity.
    input_type = type(input_value)

    # Now, we can simply return the type of the input value.
    return input_type


# Now, let's test our function with an example input value.
# We will use the number 10 as our input value.
input_value = 10

# We will call our function with the input value and store the result in a variable.
result = get_type(input_value)

# Finally, we will print the result to see the type of the input value.
print(result)
```
