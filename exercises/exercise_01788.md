## Exercise
Write a function to map a given function to each value in a dictionary and return a new dictionary containing the results.

## Reference Solution
```python
def map_dict(func, dictionary):
    """
    Maps a given function to each value in a dictionary and returns a new dictionary containing the results.

    Args:
        func (function): The function to be applied to each value in the dictionary.
        dictionary (dict): The dictionary whose values will be mapped to the function.

    Returns:
        dict: A new dictionary containing the results of applying the function to each value in the input dictionary.
    """

    # Initialize an empty dictionary to store the results
    result = {}

    # Iterate over each key-value pair in the input dictionary
    for key, value in dictionary.items():
        # Apply the given function to the current value and store the result in the result dictionary
        result[key] = func(value)

    # Return the result dictionary
    return result

# Example usage:
# Define a simple function that squares a number
def square(x):
    return x ** 2

# Create a sample dictionary
sample_dict = {'a': 1, 'b': 2, 'c': 3}

# Map the square function to the sample dictionary
result_dict = map_dict(square, sample_dict)

# Print the result dictionary
print(result_dict)  # Output: {'a': 1, 'b': 4, 'c': 9}
```
