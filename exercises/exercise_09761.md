## Exercise
Write a program to calculate the volume of a cone

## Reference Solution
```python
import math

def calculate_cone_volume(radius, height):
    """
    Calculate the volume of a cone.

    Args:
        radius (float): The radius of the cone.
        height (float): The height of the cone.

    Returns:
        float: The volume of the cone.
    """
    volume = (1.0/3) * math.pi * radius**2 * height
    return volume

def main():
    # Get user input for radius and height
    radius = float(input("Enter the radius of the cone: "))
    height = float(input("Enter the height of the cone: "))

    # Calculate and display the volume
    volume = calculate_cone_volume(radius, height)
    print(f"The volume of the cone is: {volume:.2f} cubic units")

if __name__ == "__main__":
    main()
```
