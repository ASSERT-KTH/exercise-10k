## Exercise
Write a program that converts a given hexadecimal number to decimal.

## Reference Solution
```python
def hex_to_decimal(hex_num):
    """
    Converts a hexadecimal number to decimal.

    Args:
        hex_num (str): The hexadecimal number to convert.

    Returns:
        int: The decimal equivalent of the hexadecimal number.

    Raises:
        ValueError: If the input is not a valid hexadecimal number.
    """
    try:
        decimal_num = int(hex_num, 16)
        return decimal_num
    except ValueError:
        print("Invalid hexadecimal number.")
        return None

def main():
    # Example usage:
    hex_num = input("Enter a hexadecimal number: ")
    decimal_num = hex_to_decimal(hex_num)
    if decimal_num is not None:
        print(f"The decimal equivalent of {hex_num} is {decimal_num}.")

if __name__ == "__main__":
    main()
```
