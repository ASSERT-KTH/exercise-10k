## Exercise
Write a function to check if two given numbers are twin primes.

## Reference Solution
```python
def is_prime(n):
    """Check if a number is prime."""
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True


def are_twin_primes(num1, num2):
    """
    Check if two given numbers are twin primes.
    
    Args:
        num1 (int): The first number.
        num2 (int): The second number.
    
    Returns:
        bool: True if the numbers are twin primes, False otherwise.
    """
    # Check if both numbers are prime
    if not is_prime(num1) or not is_prime(num2):
        return False
    
    # Check if the difference between the numbers is 2
    return abs(num1 - num2) == 2
```
