## Exercise
Write a program to sort a given 2D array of integers by a specified column.

## Reference Solution
```python
def sort_2d_array_by_column(array, column):
    """
    Sorts a given 2D array of integers by a specified column.

    Args:
        array (list): A 2D array of integers.
        column (int): The column to sort by.

    Returns:
        list: A sorted 2D array of integers.
    """
    # Check if the column index is valid
    if column < 0 or column >= len(array[0]):
        raise IndexError("Column index out of range")

    # Sort the array using the sorted function and a lambda function as the key
    return sorted(array, key=lambda x: x[column])

# Example usage
array = [[3, 6, 9], [1, 4, 7], [2, 5, 8]]
column = 1
sorted_array = sort_2d_array_by_column(array, column)

print("Original array:")
for row in array:
    print(row)

print("\nSorted array by column {}:".format(column))
for row in sorted_array:
    print(row)
```
