## Exercise
Create a program that echoes back every line of text the user types until they type 'quit'.

## Reference Solution
```python
def echo_program():
    """
    Echoes back every line of text the user types until they type 'quit'.
    """
    while True:
        user_input = input("Type something (or 'quit' to exit): ")
        if user_input.lower() == 'quit':
            print("Goodbye!")
            break
        print(f"You typed: {user_input}")

if __name__ == "__main__":
    echo_program()
```
