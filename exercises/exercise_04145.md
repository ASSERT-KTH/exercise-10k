## Exercise
Write a function to iterate over a dictionary and print its key-value pairs.

## Reference Solution
```python
def print_dict_key_value_pairs(input_dict):
    """
    Iterate over a dictionary and print its key-value pairs.

    Args:
        input_dict (dict): The dictionary to iterate over.

    Returns:
        None
    """
    for key, value in input_dict.items():
        print(f"Key: {key}, Value: {value}")

# Example usage
if __name__ == "__main__":
    example_dict = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }
    print_dict_key_value_pairs(example_dict)
```
