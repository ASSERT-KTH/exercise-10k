## Exercise
Write a program to find the maximum contiguous subarray sum of a given one-dimensional array of numbers. The subarray must have at least one element. If the array is empty, return 0.

## Reference Solution
```python
def max_subarray_sum(arr):
    """
    This function calculates the maximum contiguous subarray sum of a given one-dimensional array of numbers.
    
    Args:
        arr (list): A list of numbers.
    
    Returns:
        int: The maximum contiguous subarray sum.
    """
    
    # Check if the array is empty
    if not arr:
        return 0
    
    # Initialize the maximum current sum and the maximum global sum
    max_current = max_global = arr[0]
    
    # Iterate through the array starting from the second element
    for i in range(1, len(arr)):
        # Update the maximum current sum
        max_current = max(arr[i], max_current + arr[i])
        
        # Update the maximum global sum
        if max_current > max_global:
            max_global = max_current
    
    # Return the maximum global sum
    return max_global

# Example usage:
print(max_subarray_sum([-2, 1, -3, 4, -1, 2, 1, -5, 4]))  # Output: 6
print(max_subarray_sum([1, 2, 3, 4, 5]))  # Output: 15
print(max_subarray_sum([-1, -2, -3, -4, -5]))  # Output: -1
print(max_subarray_sum([]))  # Output: 0
```
