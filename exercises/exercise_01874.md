## Exercise
Write a program to zip two lists together into a list of pairs and print the resulting list.

## Reference Solution
```python
def zip_lists(list1, list2):
    """
    Zip two lists together into a list of pairs.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: A list of pairs, where each pair is a tuple containing one element from each input list.
    """
    return list(zip(list1, list2))

def main():
    # Example usage
    list1 = [1, 2, 3]
    list2 = ['a', 'b', 'c']

    zipped_list = zip_lists(list1, list2)
    print("Zipped List:", zipped_list)

if __name__ == "__main__":
    main()
```
