## Exercise
Create a dictionary with 5 key-value pairs. Remove and return a key-value pair using the .popitem() method.

## Reference Solution
```python
def main():
    # Create a dictionary
    my_dict = {
        "apple": 1,
        "banana": 2,
        "cherry": 3,
        "date": 4,
        "elderberry": 5
    }

    print("Original dictionary:")
    print(my_dict)

    # Remove and return a key-value pair using .popitem()
    removed_item = my_dict.popitem()

    print("\nRemoved key-value pair:")
    print(removed_item)

    print("\nUpdated dictionary:")
    print(my_dict)

if __name__ == "__main__":
    main()
```
