## Exercise
Write a program to access and print the elements of an array using indexing.

## Reference Solution
```python
# Define an array
array = [1, 2, 3, 4, 5]

# Print the entire array
print("Entire array:", array)

# Access and print individual elements using indexing
print("Element at index 0:", array[0])
print("Element at index 1:", array[1])
print("Element at index 2:", array[2])
print("Element at index 3:", array[3])
print("Element at index 4:", array[4])

# Access and print elements using negative indexing
print("Element at index -1:", array[-1])
print("Element at index -2:", array[-2])
print("Element at index -3:", array[-3])
print("Element at index -4:", array[-4])
print("Element at index -5:", array[-5])

# Print a range of elements using slicing
print("Elements from index 1 to 3:", array[1:4])
print("Elements from index 2 to the end:", array[2:])
print("Elements from the start to index 3:", array[:4])
```
