## Exercise
Create a прост calculator that takes a mathematical expression as input and evaluates it.

## Reference Solution
```python
# calculator.py

def calculate(expression):
    """
    Evaluates a mathematical expression.

    Args:
        expression (str): A string representing a mathematical expression.

    Returns:
        float: The result of the evaluated expression.
    """
    try:
        result = eval(expression)
        return result
    except Exception as e:
        print(f"An error occurred: {e}")
        return None

def main():
    print("Simple Calculator")
    print("------------------")

    while True:
        expression = input("Enter a mathematical expression (or 'quit' to exit): ")
        if expression.lower() == 'quit':
            break
        result = calculate(expression)
        if result is not None:
            print(f"Result: {result}")

if __name__ == "__main__":
    main()
```
