## Exercise
Write a program that takes a string as input and removes all duplicate characters from the string.

## Reference Solution
```python
def remove_duplicates(input_string):
    """
    Removes all duplicate characters from the input string, preserving the original order.

    Args:
        input_string (str): The input string.

    Returns:
        str: The input string with all duplicate characters removed.
    """
    seen = set()
    output_string = ""

    for char in input_string:
        if char not in seen:
            output_string += char
            seen.add(char)

    return output_string

def main():
    input_string = input("Enter a string: ")
    print("String without duplicates: ", remove_duplicates(input_string))

if __name__ == "__main__":
    main()
```
