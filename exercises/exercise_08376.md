## Exercise
Write a program that generates a geometric progression of a given length with a given common ratio.

## Reference Solution
```python
def generate_geometric_progression(first_term, common_ratio, length):
    """
    Generate a geometric progression of a given length with a given common ratio.

    Args:
        first_term (float): The first term of the progression.
        common_ratio (float): The common ratio of the progression.
        length (int): The length of the progression.

    Returns:
        list: A list of numbers representing the geometric progression.
    """
    progression = [first_term]
    for i in range(1, length):
        progression.append(progression[i-1] * common_ratio)
    return progression

# Example usage:
first_term = 2
common_ratio = 2
length = 10

progression = generate_geometric_progression(first_term, common_ratio, length)
print("Geometric Progression:")
for i, term in enumerate(progression):
    print(f"Term {i+1}: {term}")
```
