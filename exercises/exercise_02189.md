## Exercise
Write a function to implement a Caesar cipher, which shifts each character in a string by a specified number of positions. The function should take a string and a shift value as input, and return the encrypted string.

## Reference Solution
```python
def encrypt(text, shift):
    result = ""
    for char in text:
        if char.isalpha():
            ascii_offset = 65 if char.isupper() else 97
            result += chr((ord(char) - ascii_offset + shift) % 26 + ascii_offset)
        else:
            result += char
    return result

text = "Hello, World!"
shift = 3
print(encrypt(text, shift))
```
