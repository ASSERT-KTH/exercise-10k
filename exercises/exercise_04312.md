## Exercise
Write a function that takes another function as input and applies it to a list of numbers

## Reference Solution
```python
def apply_to_list(func, numbers):
    """
    Applies a given function to each number in a list.

    Args:
    - func (function): The function to be applied.
    - numbers (list): A list of numbers.

    Returns:
    - list: A list of results after applying the function to each number.
    """
    return [func(num) for num in numbers]

# Example usage:
def square(x):
    """Returns the square of a number."""
    return x ** 2

def cube(x):
    """Returns the cube of a number."""
    return x ** 3

numbers = [1, 2, 3, 4, 5]

squares = apply_to_list(square, numbers)
cubes = apply_to_list(cube, numbers)

print("Squares:", squares)
print("Cubes:", cubes)
```
