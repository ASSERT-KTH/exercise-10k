## Exercise
Create a function that returns the number of key-value pairs in a dictionary.

## Reference Solution
```python
def count_key_value_pairs(dictionary):
    """
    This function returns the number of key-value pairs in a dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        int: The number of key-value pairs in the dictionary.
    """

    # First, we check if the input is indeed a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("Input must be a dictionary")

    # Since dictionaries in Python are inherently collections of key-value pairs, 
    # we can simply return the length of the dictionary to get the number of key-value pairs
    # The built-in len() function in Python returns the number of items in an object
    return len(dictionary)
```
