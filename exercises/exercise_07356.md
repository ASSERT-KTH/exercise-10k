## Exercise
Understand the difference between reference and value types by creating examples of each and printing their values.

## Reference Solution
```python
# Reference Type Example: List
print("Reference Type Example: List")
list1 = [1, 2, 3]
list2 = list1  # Create a new reference to the same list

print("List 1:", list1)
print("List 2:", list2)

list2.append(4)  # Modify the list through list2
print("After modifying List 2:")
print("List 1:", list1)
print("List 2:", list2)

# Value Type Example: Integer
print("\nValue Type Example: Integer")
int1 = 5
int2 = int1  # Create a new copy of the integer value

print("Int 1:", int1)
print("Int 2:", int2)

int2 += 1  # Modify the integer value through int2
print("After modifying Int 2:")
print("Int 1:", int1)
print("Int 2:", int2)

# Value Type Example: String
print("\nValue Type Example: String")
str1 = "Hello"
str2 = str1  # Create a new copy of the string value

print("Str 1:", str1)
print("Str 2:", str2)

str2 += " World"  # Modify the string value through str2
print("After modifying Str 2:")
print("Str 1:", str1)
print("Str 2:", str2)
```
