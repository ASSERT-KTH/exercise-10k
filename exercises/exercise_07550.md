## Exercise
Write a program that takes a user's input and converts it into different data types. The user should be able to input a value and choose the data type they want to convert it into (e.g., integer, float, string). The program should then print out the converted value.

## Reference Solution
```python
def convert_type():
    user_input = input("Enter a value: ")
    data_type = input("Enter the data type you want to convert to (int, float, str): ")

    if data_type == "int":
        converted_value = int(user_input)
    elif data_type == "float":
        converted_value = float(user_input)
    elif data_type == "str":
        converted_value = str(user_input)
    else:
        print("Invalid data type. Please enter int, float, or str.")
        return

    print("The converted value is:", converted_value)

convert_type()
```
