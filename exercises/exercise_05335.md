## Exercise
Declare two boolean variables with the values True and False. Perform logical AND, OR, and NOT operations and print the results.

## Reference Solution
```python
# Declare boolean variables
bool_var1 = True
bool_var2 = False

# Perform logical AND operation
and_result = bool_var1 and bool_var2
print(f"Logical AND: {bool_var1} and {bool_var2} = {and_result}")

# Perform logical OR operation
or_result = bool_var1 or bool_var2
print(f"Logical OR: {bool_var1} or {bool_var2} = {or_result}")

# Perform logical NOT operation
not_result1 = not bool_var1
not_result2 = not bool_var2
print(f"Logical NOT: not {bool_var1} = {not_result1}")
print(f"Logical NOT: not {bool_var2} = {not_result2}")
```
