## Exercise
Write a function that finds the union between two dictionaries

## Reference Solution
```python
def dict_union(dict1, dict2):
    """
    Returns the union of two dictionaries.

    Args:
        dict1 (dict): The first dictionary.
        dict2 (dict): The second dictionary.

    Returns:
        dict: A new dictionary containing all key-value pairs from both input dictionaries.
    """
    return {**dict1, **dict2}

# Example usage:
dict1 = {'a': 1, 'b': 2}
dict2 = {'b': 3, 'c': 4}

result = dict_union(dict1, dict2)
print(result)  # Output: {'a': 1, 'b': 3, 'c': 4}
```
