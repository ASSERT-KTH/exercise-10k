## Exercise
Write a program that takes the total bill and the tip percentage as input and outputs the total amount to pay.

## Reference Solution
```python
def calculate_total_bill(bill, tip_percentage):
    """
    Calculate the total amount to pay based on the bill and tip percentage.

    Args:
        bill (float): The total bill.
        tip_percentage (float): The tip percentage.

    Returns:
        float: The total amount to pay.
    """
    tip_amount = bill * (tip_percentage / 100)
    total_amount = bill + tip_amount
    return total_amount


def main():
    # Get the total bill and tip percentage from the user
    bill = float(input("Enter the total bill: $"))
    tip_percentage = float(input("Enter the tip percentage (%): "))

    # Calculate and display the total amount to pay
    total_amount = calculate_total_bill(bill, tip_percentage)
    print(f"The total amount to pay is: ${total_amount:.2f}")


if __name__ == "__main__":
    main()
```
