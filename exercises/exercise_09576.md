## Exercise
Create a program that takes in a list of variables with their corresponding values and determines the data type of each variable. The program should then print out the variable name, its value, and its data type.

## Reference Solution
```python
def determine_data_type(variables):
    """
    This function determines the data type of each variable in the input list.
    
    Args:
        variables (dict): A dictionary where the keys are the variable names and the values are the variable values.
        
    Returns:
        None
    """

    # Iterate over each variable in the input dictionary
    for variable, value in variables.items():
        
        # Determine the data type of the variable
        data_type = type(value).__name__
        
        # Print the variable name, its value, and its data type
        print(f"Variable: {variable}, Value: {value}, Data Type: {data_type}")


# Example usage
if __name__ == "__main__":
    
    # Define a dictionary of variables with their corresponding values
    variables = {
        "name": "John Doe",
        "age": 30,
        "height": 1.75,
        "is_employed": True,
        "address": None
    }
    
    # Call the function with the example variables
    determine_data_type(variables)
```
