## Exercise
Write a program that finds the maximum value in an array of integers. The program should take an array of integers as input, iterate through the array to find the maximum value, and return the maximum value.

## Reference Solution
```python
#include <iostream>

using namespace std;

int main() {
    int n;
    cout << "Enter the number of elements: ";
    cin >> n;

    int arr[100];

    for (int i = 0; i < n; i++) {
        cout << "Enter element " << i + 1 << ": ";
        cin >> arr[i];
    }

    int max = arr[0];

    for (int i = 0; i < n; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }

    cout << "The maximum value is: " << max << endl;

    return 0;
}
```
