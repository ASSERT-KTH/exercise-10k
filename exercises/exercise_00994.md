## Exercise
Create a program that calculates the tip for a meal based on the total cost and tip percentage. The program uses if-else statements to check for valid input.

## Reference Solution
```python
// Variables
var TotalCost;
var TipPercentage;

// User Input
TotalCost = parseFloat(prompt("Enter the total cost of the meal:"));
TipPercentage = parseFloat(prompt("Enter the tip percentage (e.g., 10 for 10%):"));

// Validations
if (isNaN(TotalCost) || isNaN(TipPercentage)) {
  alert("Invalid input. Please enter a valid number.");
} else if (TotalCost <= 0 || TipPercentage <= 0) {
  alert("Invalid input. Please enter a positive number.");
} else {
  // Calculations
  var TipAmount = (TotalCost * TipPercentage) / 100;
  var TotalAmount = TotalCost + TipAmount;

  // Output
  alert("The tip amount is $" + TipAmount.toFixed(2));
  alert("The total amount is $" + TotalAmount.toFixed(2));
}
```
