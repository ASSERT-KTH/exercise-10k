## Exercise
Create a program that finds the maximum and minimum values in a list using if/else statements.

## Reference Solution
```python
def find_max_min(lst):
    """
    Find the maximum and minimum values in a list.

    Args:
        lst (list): The list of numbers.

    Returns:
        tuple: A tuple containing the maximum and minimum values.
    """
    if not lst:
        return None, None  # Return None if the list is empty

    max_val = min_val = lst[0]  # Initialize max_val and min_val with the first element

    for num in lst[1:]:
        if num > max_val:
            max_val = num
        elif num < min_val:
            min_val = num

    return max_val, min_val


# Example usage:
numbers = [5, 2, 9, 1, 7, 3, 6, 8, 4]
max_value, min_value = find_max_min(numbers)

if max_value is not None and min_value is not None:
    print(f"Maximum value: {max_value}")
    print(f"Minimum value: {min_value}")
else:
    print("The list is empty.")
```
