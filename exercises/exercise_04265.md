## Exercise
Write a function that takes a dictionary as input and prints all key-value pairs

## Reference Solution
```python
def print_dict_key_value_pairs(input_dict):
    """
    Prints all key-value pairs from the input dictionary.

    Args:
        input_dict (dict): The dictionary to print key-value pairs from.
    """
    for key, value in input_dict.items():
        print(f"{key}: {value}")

# Example usage:
if __name__ == "__main__":
    example_dict = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }
    print_dict_key_value_pairs(example_dict)
```
