## Exercise
Create a function that counts the number of occurrences of a specific character in a string.

## Reference Solution
```python
def count_char_occurrences(string, char):
    """
    Counts the number of occurrences of a specific character in a string.

    Args:
        string (str): The input string.
        char (str): The character to count occurrences of.

    Returns:
        int: The number of occurrences of the character in the string.
    """
    return string.count(char)

# Example usage
if __name__ == "__main__":
    string = "Hello, World!"
    char = "l"
    occurrences = count_char_occurrences(string, char)
    print(f"The character '{char}' occurs {occurrences} times in the string '{string}'.")
```
