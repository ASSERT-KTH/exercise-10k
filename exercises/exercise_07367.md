## Exercise
Create hexadecimal literals and print their values.

## Reference Solution
```python
# Create hexadecimal literals
hex_literal_1 = 0xA
hex_literal_2 = 0xFF
hex_literal_3 = 0x100

# Print the values
print("Hexadecimal Literals:")
print(f"0xA = {hex_literal_1} (decimal)")
print(f"0xFF = {hex_literal_2} (decimal)")
print(f"0x100 = {hex_literal_3} (decimal)")

# Print the hexadecimal values
print("\nHexadecimal Values:")
print(f"{hex_literal_1} (decimal) = {hex(hex_literal_1)} (hexadecimal)")

print(f"{hex_literal_3} (decimal) = {hex(hex_literal_3)} (hexadecimal)")
```
