## Exercise
Write a program that generates a multiplication table for a given number.

## Reference Solution
```python
def generate_multiplication_table(number, limit):
    """
    Generates a multiplication table for a given number up to a specified limit.

    Args:
        number (int): The number for which the multiplication table is generated.
        limit (int): The upper limit of the multiplication table.

    Returns:
        None
    """
    print(f"Multiplication Table for {number}:")
    for i in range(1, limit + 1):
        product = number * i
        print(f"{number} x {i} = {product}")


def main():
    # Get user input
    number = int(input("Enter a number: "))
    limit = int(input("Enter the upper limit: "))

    # Generate multiplication table
    generate_multiplication_table(number, limit)


if __name__ == "__main__":
    main()
```
