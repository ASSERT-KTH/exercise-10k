## Exercise
Write a program that sorts a list of numbers in ascending or descending order using if/else statements

## Reference Solution
```python
def sort_list(num_list, order):
    """
    Sorts a list of numbers in ascending or descending order.

    Args:
        num_list (list): A list of numbers.
        order (str): 'asc' for ascending, 'desc' for descending.

    Returns:
        list: The sorted list of numbers.
    """

    # Check if the order is 'asc' or 'desc'
    if order == 'asc':
        # Sort the list in ascending order
        return sorted(num_list)
    elif order == 'desc':
        # Sort the list in descending order
        return sorted(num_list, reverse=True)
    else:
        # If the order is neither 'asc' nor 'desc', raise an error
        raise ValueError("Invalid order. Use 'asc' for ascending or 'desc' for descending.")

# Example usage:
numbers = [64, 34, 25, 12, 22, 11, 90]
print("Original list:", numbers)

print("Sorted in ascending order:", sort_list(numbers, 'asc'))
print("Sorted in descending order:", sort_list(numbers, 'desc'))
```
