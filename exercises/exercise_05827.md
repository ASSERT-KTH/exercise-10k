## Exercise
Write a function to compress a given string by removing duplicate characters

## Reference Solution
```python
def compress_string(input_str):
    """
    Compress a given string by removing duplicate characters.

    Args:
    input_str (str): The input string to be compressed.

    Returns:
    str: The compressed string.
    """
    seen = set()
    compressed_str = ""

    for char in input_str:
        if char not in seen:
            compressed_str += char
            seen.add(char)

    return compressed_str

# Example usage
if __name__ == "__main__":
    input_str = "aaaabbbccddd"
    print("Original string:", input_str)
    print("Compressed string:", compress_string(input_str))
```
