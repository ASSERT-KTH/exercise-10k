## Exercise
Write a program that uses a for loop to print the first 10 numbers of a given number pattern.

## Reference Solution
```python
def fibonacci(n):
    """
    Generate the first n numbers of the Fibonacci sequence.
    
    Args:
        n (int): The number of Fibonacci numbers to generate.
    
    Returns:
        list: A list of the first n Fibonacci numbers.
    """
    fib_sequence = [0, 1]
    while len(fib_sequence) < n:
        fib_sequence.append(fib_sequence[-1] + fib_sequence[-2])
    return fib_sequence

def print_pattern(pattern):
    """
    Print the numbers in a given pattern.
    
    Args:
        pattern (list): A list of numbers to print.
    """
    for num in pattern:
        print(num)

def main():
    # Generate the first 10 Fibonacci numbers
    fib_numbers = fibonacci(10)
    
    # Print the Fibonacci numbers
    print("The first 10 Fibonacci numbers are:")
    print_pattern(fib_numbers)

if __name__ == "__main__":
    main()
```
