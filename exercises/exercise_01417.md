## Exercise
Write a program that checks if a given airport code is valid. If the code is in a dictionary of valid codes, print 'Valid', otherwise print 'Invalid'.

## Reference Solution
```python
# Dictionary of valid airport codes
valid_codes = {
    'JFK': 'John F. Kennedy International Airport',
    'LHR': 'London Heathrow Airport',
    'CDG': 'Paris Charles de Gaulle Airport',
    'AMS': 'Amsterdam Airport Schiphol',
    'FRA': 'Frankfurt Airport',
    'MAD': 'Adolfo Suárez Madrid–Barajas Airport',
    'BCN': 'Barcelona–El Prat Airport',
    'ATH': 'Athens International Airport',
    'IST': 'Istanbul Atatürk Airport',
    'DUB': 'Dublin Airport'
}

def check_airport_code(code):
    """
    Checks if a given airport code is valid.

    Args:
        code (str): The airport code to check.

    Returns:
        None
    """
    code = code.upper()  # Convert the code to uppercase
    if code in valid_codes:
        print('Valid')
    else:
        print('Invalid')

# Test the function
check_airport_code('JFK')  # Should print: Valid
check_airport_code('LAX')  # Should print: Invalid
```
