## Exercise
Understand the difference between null and undefined by declaring variables with these values and printing them.

## Reference Solution
```python
// Declaring variables with null and undefined values
let null_var = null;
let undefined_var = undefined;

// Printing the variables
console.log("Null Value:", null_var);
console.log("Undefined Value:", undefined_var);

// Checking the type of variables
console.log("Type of Null Value:", typeof null_var);
console.log("Type of Undefined Value:", typeof undefined_var);

// Checking for null and undefined
console.log("Is Null Var Null?", null_var === null);
console.log("Is Undefined Var Undefined?", undefined_var === undefined);
```
