## Exercise
Write a program that takes an array and an index as input, and returns the element at the specified index. If the index is out of bounds, the program should return a message indicating that the index is invalid.

## Reference Solution
```python
#include <stdio.h>

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    int index = 10;

    if (index >= 0 && index < 5) {
        printf("Element at index %d is %d\n", index, arr[index]);
    } else {
        printf("Index %d is invalid\n", index);
    }

    return 0;
}
```
