## Exercise
Write a function that takes a dictionary and a predicate as input and returns a new dictionary with only the key-value pairs that satisfy the predicate

## Reference Solution
```python
def filter_dict(dictionary, predicate):
    """
    This function filters a dictionary based on a given predicate.
    
    Args:
        dictionary (dict): The dictionary to be filtered.
        predicate (function): A function that takes a key-value pair and returns True if it satisfies the condition, False otherwise.
    
    Returns:
        dict: A new dictionary with only the key-value pairs that satisfy the predicate.
    """

    # Initialize an empty dictionary to store the filtered key-value pairs
    filtered_dict = {}

    # Iterate over each key-value pair in the input dictionary
    for key, value in dictionary.items():
        # Check if the current key-value pair satisfies the predicate
        if predicate(key, value):
            # If it does, add it to the filtered dictionary
            filtered_dict[key] = value

    # Return the filtered dictionary
    return filtered_dict


# Example usage:
# Define a predicate function that checks if the value is greater than 5
def greater_than_five(key, value):
    return value > 5

# Create a sample dictionary
sample_dict = {'a': 1, 'b': 6, 'c': 3, 'd': 7}

# Filter the dictionary using the predicate function
filtered_sample_dict = filter_dict(sample_dict, greater_than_five)

# Print the filtered dictionary
print(filtered_sample_dict)
```
