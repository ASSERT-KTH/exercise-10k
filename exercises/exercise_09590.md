## Exercise
Write a program that takes a list of numbers as input and returns the maximum value in the list. Use a basic algorithm to iterate through the list and keep track of the maximum value found so far.

## Reference Solution
```python
def max_element(my_list):
    if len(my_list) == 0:
        return None

    max_val = my_list[0]

    for i in range(1, len(my_list)):
        if my_list[i] > max_val:
            max_val = my_list[i]

    return max_val

# Testing the function
print(max_element([2, 5, 1, 7, 3, 9]))  # Expected output: 9
print(max_element([5, 5, 5, 5, 5]))  # Expected output: 5
print(max_element([]))  # Expected output: None
```
