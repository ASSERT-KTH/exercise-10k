## Exercise
Write a program that asks the user for a noun, verb, adjective, and adverb, then uses these words to fill in the blanks in a story.

## Reference Solution
```python
def get_word(prompt):
    """Gets a word from the user based on the given prompt."""
    while True:
        word = input(prompt)
        if word:
            return word
        else:
            print("Please enter a word.")

def main():
    print("Welcome to the story filler program!")
    print("Please provide the following words:")

    noun = get_word("Noun: ")
    verb = get_word("Verb: ")
    adjective = get_word("Adjective: ")
    adverb = get_word("Adverb: ")

    story = f"""
Once upon a time, in a land far, far away, there was a {adjective} {noun}.
This {noun} loved to {verb} {adverb} through the rolling hills and green forests.
One day, while {verb}ing {adverb}, the {noun} stumbled upon a hidden treasure.
From that day on, the {noun} lived happily ever after, {verb}ing {adverb} and enjoying its newfound wealth.
"""

    print("\nHere is your completed story:")
    print(story)

if __name__ == "__main__":
    main()
```
