## Exercise
Given an array of integers and a number k, rotate the array k times to the right. The function should take in an array of integers and a number k as input and return the rotated array.

## Reference Solution
```python
def rotate_array(nums, k):
    """
    Rotates the input array to the right by k steps.

    Args:
        nums (list): The input array of integers.
        k (int): The number of steps to rotate the array to the right.

    Returns:
        list: The rotated array.
    """
    # Calculate the effective number of steps to rotate, in case k is larger than the length of the array
    k = k % len(nums)
    
    # Use list slicing to rotate the array
    return nums[-k:] + nums[:-k]

# Example usage
if __name__ == "__main__":
    nums = [1, 2, 3, 4, 5, 6, 7]
    k = 3
    print("Original array:", nums)
    print("Rotated array:", rotate_array(nums, k))
```
