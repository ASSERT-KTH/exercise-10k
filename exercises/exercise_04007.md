## Exercise
Write a program that uses nested for loops to print a triangle pattern of asterisks (*).

## Reference Solution
```python
def print_triangle(height):
    """
    Prints a triangle pattern of asterisks (*).

    Args:
        height (int): The height of the triangle.

    Returns:
        None
    """
    for i in range(1, height + 1):
        # Print leading spaces
        for _ in range(height - i):
            print(" ", end=" ")
        # Print asterisks
        for _ in range(2 * i - 1):
            print("*", end=" ")
        print()

# Example usage
print_triangle(5)
```
