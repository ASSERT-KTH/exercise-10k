## Exercise
Write a program that takes an input from the user and converts it into different data types. The user should be able to input a value and choose the data type they want to convert it to. The program should then print out the converted value. If the conversion is not possible, the program should print an error message.

## Reference Solution
```python
# Importing Modules
import sys

# Defining the Function
def convertInput():
    # Getting User Input
    user_input = input("Enter a value: ")

    # Getting User Choice
    print("Choose a data type to convert to:")
    print("1. Integer")
    print("2. Float")
    print("3. String")
    print("4. Complex Number")
    choice = input("Enter your choice (1/2/3/4): ")

    # Converting the Input
    if choice == "1":
        try:
            converted_value = int(user_input)
            print(f"Converted value: {converted_value}")
        except ValueError:
            print("Error: Cannot convert the input to an integer.")
    elif choice == "2":
        try:
            converted_value = float(user_input)
            print(f"Converted value: {converted_value}")
        except ValueError:
            print("Error: Cannot convert the input to a float.")
    elif choice == "3":
        converted_value = str(user_input)
        print(f"Converted value: {converted_value}")
    elif choice == "4":
        try:
            converted_value = complex(user_input)
            print(f"Converted value: {converted_value}")
        except ValueError:
            print("Error: Cannot convert the input to a complex number.")
    else:
        print("Error: Invalid choice.")

# Calling the Function
convertInput()
```
