## Exercise
Write a program to print a set of unique numbers from 1 to 10 to the screen.

## Reference Solution
```python
def print_unique_numbers(start, end):
    """
    Prints a set of unique numbers from start to end.

    Args:
        start (int): The starting number.
        end (int): The ending number.
    """
    unique_numbers = set(range(start, end + 1))
    print("Unique numbers from {} to {}:".format(start, end))
    print(unique_numbers)


def main():
    start = 1
    end = 10
    print_unique_numbers(start, end)


if __name__ == "__main__":
    main()
```
