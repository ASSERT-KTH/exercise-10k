## Exercise
Create a program that takes in a student's score as input and outputs a grade based on the following conditions: score >= 90 (A), score >= 80 (B), score >= 70 (C), score >= 60 (D), score < 60 (F). The program should also handle cases where the input score is not a number or is not within the range of 0 to 100.

## Reference Solution
```python
const readline = require('readline');

const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout
});

const grade = (score) => {
  if (score >= 90) {
    return 'A';
  } else if (score >= 80) {
    return 'B';
  } else if (score >= 70) {
    return 'C';
  } else if (score >= 60) {
    return 'D';
  } else {
    return 'F';
  }
};

const getScoreFromUser = () => {
  rl.question('Enter a score between 0 and 100: ', (input) => {
    const score = parseInt(input);
    if (isNaN(score) || score < 0 || score > 100) {
      console.log('Invalid input. Please enter a number between 0 and 100.');
      getScoreFromUser();
    } else {
      const studentGrade = grade(score);
      console.log(`Your grade is: ${studentGrade}`);
      getScoreFromUser();
    }
  });
};

getScoreFromUser();
```
