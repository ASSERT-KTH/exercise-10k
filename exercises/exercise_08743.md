## Exercise
Hide complex data within a variable to prevent unauthorized access.

## Reference Solution
```python
class EncapsulatedData:
    def __init__(self, data):
        self.__data = data  # Private attribute

    def get_data(self):
        return self.__data  # Public method to access the data

    def set_data(self, new_data):
        self.__data = new_data  # Public method to modify the data

# Create an instance of the class
encapsulated_data = EncapsulatedData("Sensitive Information")

# Access the data using the public method
print(encapsulated_data.get_data())  # Output: Sensitive Information

# Modify the data using the public method
encapsulated_data.set_data("New Sensitive Information")
print(encapsulated_data.get_data())  # Output: New Sensitive Information

# Attempting to access the private attribute directly will result in an AttributeError
try:
    print(encapsulated_data.__data)
except AttributeError:
    print("Error: cannot access private attribute directly")
```
