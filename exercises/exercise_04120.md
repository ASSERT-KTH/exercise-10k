## Exercise
Write a program that recognizes and prints out the data type of a given variable.

## Reference Solution
```python
def print_data_type(var):
    """
    Prints the data type of a given variable.

    Args:
        var: The variable to check the data type of.
    """

    # Use the type() function to get the data type of the variable
    data_type = type(var)

    # Print the data type
    print(f"The data type of the variable is: {data_type.__name__}")


# Example usage:
if __name__ == "__main__":
    # Test with different data types
    print_data_type(10)  # Integer
    print_data_type(10.5)  # Float
    print_data_type("Hello, World!")  # String
    print_data_type(True)  # Boolean
    print_data_type([1, 2, 3])  # List
    print_data_type((1, 2, 3))  # Tuple
    print_data_type({"name": "John", "age": 30})  # Dictionary
    print_data_type({"apple", "banana", "cherry"})  # Set
```
