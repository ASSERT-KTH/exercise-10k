## Exercise
Create a function that takes a string and a separator as arguments and returns a list of substrings split by the separator

## Reference Solution
```python
def split_string(input_str, separator):
    """
    Splits a string into substrings based on a separator.

    Args:
        input_str (str): The input string to be split.
        separator (str): The separator used to split the string.

    Returns:
        list: A list of substrings split by the separator.
    """
    return input_str.split(separator)

# Example usage:
def main():
    input_str = "apple,banana,cherry"
    separator = ","
    result = split_string(input_str, separator)
    print(result)  # Output: ['apple', 'banana', 'cherry']

if __name__ == "__main__":
    main()
```
