## Exercise
Write a program to print a tuple of values to the screen.

## Reference Solution
```python
def print_tuple(values):
    """
    Prints a tuple of values to the screen.

    Args:
        values (tuple): A tuple of values to be printed.
    """
    print(values)

# Example usage:
if __name__ == "__main__":
    # Create a tuple of values
    my_tuple = (1, "hello", 3.14, True)

    # Print the tuple
    print_tuple(my_tuple)
```
