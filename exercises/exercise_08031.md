## Exercise
Write a program that uses an if statement to print 'Hello World' if the input is greater than 10.

## Reference Solution
```python
def print_hello_world():
    # Get user input
    user_input = int(input("Enter a number: "))

    # Check if input is greater than 10
    if user_input > 10:
        print("Hello World")

# Example usage
if __name__ == "__main__":
    print_hello_world()
```
