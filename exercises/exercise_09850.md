## Exercise
Write a function that takes a string input and returns the string reversed, with the additional complexity of also reversing the order of characters within each word, while maintaining the original word order. For example, the string 'hello world' should be transformed into 'olleh dlrow'. The input string will only contain spaces as delimiters and lowercase English letters.

## Reference Solution
```python
def reverse_string(input_str):
    """
    This function takes a string input and returns the string with each word's characters reversed, 
    while maintaining the original word order.

    Args:
        input_str (str): The input string to be processed.

    Returns:
        str: The processed string with each word's characters reversed.
    """

    # First, we split the input string into a list of words using space as the delimiter
    words = input_str.split(' ')

    # Then, we use a list comprehension to reverse each word in the list
    # The expression word[::-1] is using slicing to create a reversed copy of the word
    reversed_words = [word[::-1] for word in words]

    # Finally, we join the reversed words back into a string using space as the delimiter
    reversed_str = ' '.join(reversed_words)

    # We return the resulting string
    return reversed_str

# Test the function
print(reverse_string('hello world'))  # Output: 'olleh dlrow'
```
