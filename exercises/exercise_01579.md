## Exercise
Write a function that calls a given function with a given set of arguments.

## Reference Solution
```python
def call_function(func, *args, **kwargs):
    """
    Calls a given function with a given set of arguments.

    Args:
        func (function): The function to be called.
        *args: Variable number of non-keyword arguments.
        **kwargs: Variable number of keyword arguments.

    Returns:
        result: The result of the function call.
    """

    # First, we check if the input 'func' is callable. This is to prevent potential errors.
    if not callable(func):
        raise TypeError("The input 'func' is not callable.")

    # Now, we can call the function with the given arguments and keyword arguments.
    # The *args and **kwargs syntax allows us to pass a variable number of arguments to the function.
    # We store the result of the function call in the 'result' variable.
    result = func(*args, **kwargs)

    # Finally, we return the result of the function call.
    return result

# Example usage:
def add(a, b):
    return a + b

print(call_function(add, 3, 5))  # Outputs: 8
```
