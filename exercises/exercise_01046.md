## Exercise
Write a program that takes a list, an element, and an index as input, and inserts the element at the given index in the list.

## Reference Solution
```python
def insert_in_list(lst, element, index):
    if index <= len(lst):
        lst.insert(index, element)
        return lst
    else:
        return "Index out of range."

# Example usage
lst = [1, 2, 3, 4, 5]
element = 10
index = 3

result = insert_in_list(lst, element, index)
print(result)
```
