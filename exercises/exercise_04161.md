## Exercise
Create a program that takes a name as input and prints it 5 times.

## Reference Solution
```python
def print_name(name):
    """
    Prints the given name 5 times.
    
    Args:
        name (str): The name to be printed.
    """
    for _ in range(5):
        print(name)

def main():
    # Get the name from the user
    name = input("Please enter your name: ")
    
    # Print the name 5 times
    print_name(name)

if __name__ == "__main__":
    main()
```
