## Exercise
Write a function that takes a dictionary as input and returns a list of all the key-value pairs

## Reference Solution
```python
def get_key_value_pairs(dictionary):
    """
    Returns a list of tuples containing all key-value pairs in the dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        list: A list of tuples containing key-value pairs.
    """
    return list(dictionary.items())

# Example usage:
def main():
    example_dict = {"name": "John", "age": 30, "city": "New York"}
    key_value_pairs = get_key_value_pairs(example_dict)
    print(key_value_pairs)

if __name__ == "__main__":
    main()
```
