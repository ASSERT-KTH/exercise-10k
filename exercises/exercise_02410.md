## Exercise
Create a function with type hints that indicate the expected input and output types. Then, call this function with an input value and print the result.

## Reference Solution
```python
def greet(name: str) -> str:
    """
    Returns a personalized greeting message.

    Args:
    name (str): The name to include in the greeting.

    Returns:
    str: A greeting message with the provided name.
    """
    return f"Hello, {name}!"

# Example usage:
def main() -> None:
    name: str = "World"
    greeting: str = greet(name)
    print(greeting)

if __name__ == "__main__":
    main()
```
