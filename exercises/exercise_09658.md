## Exercise
Write a program that asks for a string and removes leading and trailing whitespace.

## Reference Solution
```python
def remove_whitespace(input_string):
    """
    Removes leading and trailing whitespace from a given string.

    Args:
        input_string (str): The string to process.

    Returns:
        str: The string with leading and trailing whitespace removed.
    """
    return input_string.strip()

def main():
    # Ask for a string
    input_string = input("Please enter a string: ")

    # Remove leading and trailing whitespace
    cleaned_string = remove_whitespace(input_string)

    # Print the result
    print("Cleaned string:", cleaned_string)

if __name__ == "__main__":
    main()
```
