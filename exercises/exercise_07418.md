## Exercise
Write a program that finds the maximum value in an array of integers. The program should take an array of integers as input, iterate through the array, and return the maximum value found.

## Reference Solution
```python
#include <stdio.h>

int maxValue(int* arr, int size) {
    int max = arr[0];
    for (int i = 1; i < size; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    return max;
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int size = sizeof(arr) / sizeof(arr[0]);
    printf("Max value: %d\n", maxValue(arr, size));
    return 0;
}
```
