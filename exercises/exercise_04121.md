## Exercise
Declare a variable to store a student's name and age, then print out the student's information

## Reference Solution
```python

student_name = "John Doe"
student_age = 20

# Print student information
print("Student Name:", student_name)
print("Student Age:", student_age)
```

```python
# Declare a dictionary to store student information
student_info = {
    "name": "John Doe",
    "age": 20
}

# Print student information
print("Student Name:", student_info["name"])
```
