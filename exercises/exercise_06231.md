## Exercise
Write a program that finds the minimum value in a given array of integers.

## Reference Solution
```python
def find_min_value(array):
    # Check if the array is empty
    if len(array) == 0:
        return None

    # Initialize the minimum value with the first element of the array
    min_value = array[0]

    # Iterate over the array to find the minimum value
    for num in array:
        if num < min_value:
            min_value = num

    return min_value


# Example usage
array = [10, 20, 3, 40, 5]
print("Minimum value in the array:", find_min_value(array))
```
