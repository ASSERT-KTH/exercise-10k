## Exercise
Create a function that takes a list of pairs as input and returns two separate lists: one with the first elements of the pairs and one with the second elements.

## Reference Solution
```python
def separate_pairs(pairs):
    """
    This function takes a list of pairs as input and returns two separate lists: 
    one with the first elements of the pairs and one with the second elements.

    Args:
        pairs (list): A list of pairs.

    Returns:
        tuple: Two separate lists, one with the first elements of the pairs and one with the second elements.
    """

    # Initialize two empty lists to store the first and second elements of the pairs
    first_elements = []
    
    # Initialize another list to store the second elements of the pairs
    second_elements = []

    # Iterate over each pair in the input list
    for pair in pairs:
        # Assuming each pair is a list or tuple with two elements, append the first element to the first_elements list
        first_elements.append(pair[0])
        
        # Append the second element to the second_elements list
        second_elements.append(pair[1])

    # Return the two lists as a tuple
    return first_elements, second_elements

# Example usage:
pairs = [(1, 2), (3, 4), (5, 6)]
first, second = separate_pairs(pairs)
print("First elements:", first)
print("Second elements:", second)
```
