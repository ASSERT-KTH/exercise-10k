## Exercise
Write a function that takes a dictionary as input and prints out all the key-value pairs

## Reference Solution
```python
def print_dictionary_key_value_pairs(input_dict):
    """
    This function takes a dictionary as input and prints out all the key-value pairs.

    Args:
        input_dict (dict): The dictionary to print key-value pairs from.
    """
    for key, value in input_dict.items():
        print(f"Key: {key}, Value: {value}")


# Example usage
if __name__ == "__main__":
    example_dict = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }
    print_dictionary_key_value_pairs(example_dict)
```
