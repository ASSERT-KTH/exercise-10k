## Exercise
Create a program that uses functions to perform basic arithmetic operations. The program should have separate functions for addition, subtraction, multiplication, and division. Each function should take two numbers as arguments and return the result of the operation. The main part of the program should prompt the user to choose an operation and input two numbers, then call the corresponding function and print the result.

## Reference Solution
```python
def add(num1, num2):
    result = num1 + num2
    return result

def sub(num1, num2):
    result = num1 - num2
    return result

def mul(num1, num2):
    result = num1 * num2
    return result

def div(num1, num2):
    result = num1 / num2
    return result

x = input("Enter the first number: ")
y = input("Enter the second number: ")

if x == "add":
    add(y, 3)
elif x == "sub":
    sub(y, 3)
elif x == "mul":
    mul(y, 3)
elif x == "div":
    div(y, 3)
else:
    print("Invalid operation")
```

```python
def add(num1, num2):
    result = num1 + num2
    return result

def sub(num1, num2):
    result = num1 - num2
    return result

def mul(num1, num2):
    result = num1 * num2
    return result

def div(num1, num2):
    result = num1 / num2
    return result

op = input("Enter the operation (add, sub, mul, div): ")
num1 = float(input("Enter the first number: "))
num2 = float(input("Enter the second number: "))

if op == "add":
    result = add(num1, num2)
    print(f"{num1} + {num2} = {result}")
elif op == "sub":
    result = sub(num1, num2)
    print(f"{num1} - {num2} = {result}")
elif op == "mul":
    result = mul(num1, num2)
    print(f"{num1} * {num2} = {result}")
elif op == "div":
    if num2 != 0:
        result = div(num1, num2)
        print(f"{num1} / {num2} = {result}")
    else:
        print("Error: Division by zero")
else:
    print("Invalid operation")
```
