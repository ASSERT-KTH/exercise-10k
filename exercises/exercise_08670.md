## Exercise
Write a program that asks the user for a string and counts the number of words in it.

## Reference Solution
```python
def count_words(input_string):
    """
    Counts the number of words in a given string.

    Args:
        input_string (str): The input string to count words from.

    Returns:
        int: The number of words in the input string.
    """
    words = input_string.split()
    return len(words)


def main():
    # Ask the user for a string
    user_string = input("Please enter a string: ")

    # Count the number of words in the string
    word_count = count_words(user_string)

    # Print the result
    print(f"The string contains {word_count} words.")


if __name__ == "__main__":
    main()
```
