## Exercise
Write a program that takes the cost and markup percentage as input and outputs the selling price.

## Reference Solution
```python
def calculate_selling_price(cost, markup_percentage):
    """
    Calculates the selling price based on the cost and markup percentage.

    Args:
        cost (float): The cost of the item.
        markup_percentage (float): The markup percentage.

    Returns:
        float: The selling price.
    """
    # Calculate the markup amount
    markup_amount = (cost * markup_percentage) / 100

    # Calculate the selling price
    selling_price = cost + markup_amount

    return selling_price

def main():
    # Get the cost and markup percentage from the user
    cost = float(input("Enter the cost: $"))
    markup_percentage = float(input("Enter the markup percentage (%): "))

    # Calculate the selling price
    selling_price = calculate_selling_price(cost, markup_percentage)

    # Print the selling price
    print(f"The selling price is: ${selling_price:.2f}")

if __name__ == "__main__":
    main()
```
