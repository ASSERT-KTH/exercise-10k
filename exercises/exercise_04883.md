## Exercise
Update the value of a specific key in a dictionary.

## Reference Solution
```python
def update_dict_key(dictionary, key, new_value):
    """
    Updates the value of a specific key in a dictionary.

    Args:
    - dictionary (dict): The dictionary to update.
    - key: The key to update.
    - new_value: The new value for the key.

    Returns:
    - The updated dictionary.
    """
    if key in dictionary:
        dictionary[key] = new_value
    else:
        print(f"Key '{key}' not found in the dictionary.")
    return dictionary

# Example usage:
my_dict = {"name": "John", "age": 30, "city": "New York"}
print("Original Dictionary:", my_dict)

updated_dict = update_dict_key(my_dict, "age", 31)
print("Updated Dictionary:", updated_dict)

# Try updating a non-existent key
updated_dict = update_dict_key(my_dict, "country", "USA")
print("Updated Dictionary:", updated_dict)
```
