## Exercise
Create a program that takes in two numbers and performs addition, subtraction, multiplication, and division based on the user's choice. The program should handle invalid inputs and division by zero errors.

## Reference Solution
```python
def calculator():
    """
    This function takes in two numbers and a choice of operation from the user.
    It then performs the chosen operation and displays the result.
    It also handles invalid inputs and division by zero errors.
    """
    print("Simple Calculator")
    print("1. Addition")
    print("2. Subtraction")
    print("3. Multiplication")
    print("4. Division")

    while True:
        try:
            choice = int(input("Enter your choice (1, 2, 3, 4): "))
            if choice not in [1, 2, 3, 4]:
                print("Invalid choice. Please enter a number between 1 and 4.")
                continue
            break
        except ValueError:
            print("Invalid input. Please enter a number.")

    while True:
        try:
            num1 = float(input("Enter first number: "))
            break
        except ValueError:
            print("Invalid input. Please enter a number.")

    while True:
        try:
            num2 = float(input("Enter second number: "))
            if choice == 4 and num2 == 0:
                print("Error! Division by zero is not allowed.")
                continue
            break
        except ValueError:
            print("Invalid input. Please enter a number.")

    if choice == 1:
        print(f"{num1} + {num2} = {num1 + num2}")
    elif choice == 2:
        print(f"{num1} - {num2} = {num1 - num2}")
    elif choice == 3:
        print(f"{num1} * {num2} = {num1 * num2}")
    else:
        print(f"{num1} / {num2} = {num1 / num2}")

if __name__ == "__main__":
    calculator()
```
