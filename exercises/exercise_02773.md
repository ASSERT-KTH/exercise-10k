## Exercise
Write a program that takes a word as input and generates all possible anagrams.

## Reference Solution
```python
import itertools

def generate_anagrams(word):
    """
    Generate all possible anagrams of a given word.

    Args:
        word (str): The input word.

    Returns:
        list: A list of all possible anagrams.
    """
    # Remove duplicates by converting to a set
    anagrams = set("".join(p) for p in itertools.permutations(word))
    return list(anagrams)

def main():
    # Get user input
    word = input("Enter a word: ")

    # Generate anagrams
    anagrams = generate_anagrams(word)

    # Print anagrams
    print("Anagrams:")
    for anagram in anagrams:
        print(anagram)

if __name__ == "__main__":
    main()
```
