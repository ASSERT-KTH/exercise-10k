## Exercise
Write a program that prints out a sequence of numbers based on a given pattern using if/else statements

## Reference Solution
```python
def print_sequence(n):
    for i in range(1, n+1):
        if i % 3 == 0 and i % 5 == 0:
            print("FizzBuzz")
        elif i % 3 == 0:
            print("Fizz")
        elif i % 5 == 0:
            print("Buzz")
        else:
            print(i)

# Test the function
print_sequence(20)
```
