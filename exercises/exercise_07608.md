## Exercise
Write a program that uses logical operators to compare two variables.

## Reference Solution
```python
# Define two variables
var1 = 10
var2 = 5

# Compare variables using logical operators
print("Logical AND (and):", var1 > 5 and var2 < 10)
print("Logical OR (or):", var1 > 5 or var2 < 0)
print("Logical NOT (not):", not var1 == 10)

# Compare variables using equality operators
print("Equal (==):", var1 == var2)
print("Not Equal (!=):", var1 != var2)
print("Greater Than (>):", var1 > var2)
print("Less Than (<):", var1 < var2)
print("Greater Than or Equal (>=):", var1 >= var2)
print("Less Than or Equal (<=):", var1 <= var2)
```
