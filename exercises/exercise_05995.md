## Exercise
Create a function that takes a list and an index as input, and returns the value at that index if it exists, handling the case where the index is out of range

## Reference Solution
```python
def get_value_at_index(lst, index):
    """
    Retrieves the value at a specified index in a list.

    Args:
        lst (list): The input list.
        index (int): The index to retrieve the value from.

    Returns:
        The value at the specified index if it exists, otherwise None.
    """
    try:
        return lst[index]
    except IndexError:
        return None

# Example usage:
numbers = [1, 2, 3, 4, 5]
print(get_value_at_index(numbers, 2))  # Output: 3
print(get_value_at_index(numbers, 10))  # Output: None
```

def get_value_at_index(lst, index):
    """
    Retrieves the value at a specified index in a list.

    Args:
        lst (list): The input list.
        index (int): The index to retrieve the value from.

    Returns:
        The value at the specified index if it exists, otherwise None.
    """
    if index < 0 or index >= len(lst):
        return None
    return lst[index]

# Example usage:
numbers = [1, 2, 3, 4, 5]
