## Exercise
Write a program that prints a table of numbers using nested for loops.

## Reference Solution
```python
# Define the number of rows and columns
rows = 5
columns = 5

# Print the table header
print("  |", end="")
for i in range(1, columns + 1):
    print(f" {i} |", end="")
print()

# Print the separator
print("-" * (columns * 4 + 3))

# Print the table
for i in range(1, rows + 1):
    print(f"{i} |", end="")
    for j in range(1, columns + 1):
        print(f" {i * j} |", end="")
    print()
```
