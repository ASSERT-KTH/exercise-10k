## Exercise
Create a program that takes a string as input and prints out its length.

## Reference Solution
```python
def print_string_length(input_string: str) -> None:
    """
    Prints the length of the input string.

    Args:
        input_string (str): The input string.
    """
    print(f"The length of the string is: {len(input_string)}")


def main() -> None:
    # Get the input string from the user
    input_string = input("Please enter a string: ")

    # Print the length of the string
    print_string_length(input_string)


if __name__ == "__main__":
    main()
```
