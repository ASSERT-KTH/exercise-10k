## Exercise
Write a program that iterates over the keys of a given dictionary

## Reference Solution
```python
def iterate_over_keys(input_dict):
    """
    Iterates over the keys of a given dictionary.

    Args:
        input_dict (dict): The dictionary to iterate over.

    Returns:
        None
    """
    for key in input_dict:
        print(f"Key: {key}")

# Example usage
if __name__ == "__main__":
    example_dict = {
        "name": "John",
        "age": 30,
        "city": "New York"
    }

    print("Dictionary Keys:")
    iterate_over_keys(example_dict)
```
