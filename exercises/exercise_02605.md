## Exercise
Write a program that iterates over the key-value pairs in a dictionary

## Reference Solution
```python
# Define a dictionary
my_dict = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

# Iterate over the key-value pairs in the dictionary
for key, value in my_dict.items():
    print(f"{key}: {value}")
```

```python
for key in my_dict.keys():
    print(f"{key}: {my_dict[key]}")

for value in my_dict.values():
    print(value)
```

```python
for pair in my_dict.items():
    key, value = pair
    print(f"{key}: {value}")
```
