## Exercise
Write a program to group a list of objects by a common attribute and print the resulting groups.

## Reference Solution
```python
from itertools import groupby

# Define a class for the objects
class Person:
    def __init__(self, name, age, city):
        self.name = name
        self.age = age
        self.city = city

    def __repr__(self):
        return f'{self.name}, {self.age}, {self.city}'

# Create a list of objects
people = [
    Person('John', 25, 'New York'),
    Person('Alice', 25, 'New York'),
    Person('Bob', 30, 'Chicago'),
    Person('Charlie', 25, 'Chicago'),
    Person('David', 30, 'Chicago'),
    Person('Eve', 25, 'New York')
]

# Sort the list of objects by the attribute to group by (age)
people.sort(key=lambda x: x.age)

# Group the list of objects by the attribute (age)
for age, group in groupby(people, key=lambda x: x.age):
    print(f'Age: {age}')
    for person in group:
        print(person)
    print()  # Empty line for better readability
```
