## Exercise
Write a program that takes an input from the user and determines its data type. The program should be able to identify if the input is an integer, float, or string. If the input is an integer or float, it should also check if it is positive, negative, or zero.

## Reference Solution
```python
def determine_data_type(input_value):
    """
    Determine the data type of the input value.

    Args:
        input_value (str): The input value from the user.

    Returns:
        str: A string describing the data type and value of the input.
    """
    try:
        # Try to convert the input to an integer
        int_value = int(input_value)
        if int_value > 0:
            return f"The input '{input_value}' is an integer and is positive."
        elif int_value < 0:
            return f"The input '{input_value}' is an integer and is negative."
        else:
            return f"The input '{input_value}' is an integer and is zero."
    except ValueError:
        try:
            # Try to convert the input to a float
            float_value = float(input_value)
            if float_value > 0:
                return f"The input '{input_value}' is a float and is positive."
            elif float_value < 0:
                return f"The input '{input_value}' is a float and is negative."
            else:
                return f"The input '{input_value}' is a float and is zero."
        except ValueError:
            # If the input cannot be converted to an integer or float, it is a string
            return f"The input '{input_value}' is a string."

def main():
    # Get the input from the user
    input_value = input("Please enter a value: ")

    # Determine the data type of the input
    print(determine_data_type(input_value))

if __name__ == "__main__":
    main()
```
