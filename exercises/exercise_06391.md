## Exercise
Create a program that takes in the initial balance, deposit amount, and withdrawal amount, then calculates the final balance. If the final balance is negative, print a warning message

## Reference Solution
```python
def calculate_balance(initial_balance, deposit_amount, withdrawal_amount):
    """
    Calculate the final balance after a deposit and withdrawal transaction.

    Args:
        initial_balance (float): The initial balance.
        deposit_amount (float): The deposit amount.
        withdrawal_amount (float): The withdrawal amount.

    Returns:
        float: The final balance.
    """
    final_balance = initial_balance + deposit_amount - withdrawal_amount
    return final_balance

def print_warning(final_balance):
    """
    Print a warning message if the final balance is negative.

    Args:
        final_balance (float): The final balance.
    """
    if final_balance < 0:
        print("Warning: Your account balance is negative.")

def main():
    # Get user input
    initial_balance = float(input("Enter the initial balance: "))
    deposit_amount = float(input("Enter the deposit amount: "))
    withdrawal_amount = float(input("Enter the withdrawal amount: "))

    # Calculate the final balance
    final_balance = calculate_balance(initial_balance, deposit_amount, withdrawal_amount)

    # Print the final balance
    print(f"Final balance: {final_balance:.2f}")

    # Print a warning message if necessary
    print_warning(final_balance)

if __name__ == "__main__":
    main()
```
