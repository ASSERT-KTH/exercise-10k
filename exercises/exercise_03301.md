## Exercise
Write a program that takes an array and an index as input, and returns the element at that index.

## Reference Solution
```python
def get_element_at_index(array, index):
    """
    Retrieves the element at a specified index from an array.

    Args:
        array (list): The input array.
        index (int): The index of the element to be retrieved.

    Returns:
        The element at the specified index if it exists, otherwise None.
    """

    # Check if the index is within the bounds of the array
    if index < 0 or index >= len(array):
        return None

    # Return the element at the specified index
    return array[index]


# Example usage
array = [1, 2, 3, 4, 5]
index = 2
print(get_element_at_index(array, index))  # Output: 3
```
