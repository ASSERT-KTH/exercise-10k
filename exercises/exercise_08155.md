## Exercise
Create an array of 5 integers and find the minimum element.

## Reference Solution
```python
def find_minimum_element(array):
    """
    Find the minimum element in an array.

    Args:
    array (list): A list of integers.

    Returns:
    int: The minimum element in the array.
    """
    return min(array)


# Create an array of 5 integers
array = [12, 45, 7, 23, 56]

# Find the minimum element
minimum_element = find_minimum_element(array)

print("Array:", array)
print("Minimum Element:", minimum_element)
```
