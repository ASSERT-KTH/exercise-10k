## Exercise
Write a program that takes a list of integers as input, and returns the maximum value in the list.

## Reference Solution
```python
def find_max_value(lst):
    """
    This function finds the maximum value in a given list of integers.

    Args:
        lst (list): A list of integers.

    Returns:
        int: The maximum value in the list.

    Raises:
        ValueError: If the list is empty.
    """
    # Check if the list is empty
    if not lst:
        raise ValueError("The list is empty")

    # Use the built-in max function to find the maximum value in the list
    max_value = max(lst)

    return max_value


# Example usage:
if __name__ == "__main__":
    numbers = [4, 2, 9, 6, 5, 1]
    print("Maximum value in the list:", find_max_value(numbers))
```
