## Exercise
Create a program that uses a while loop to continuously ask the user for a mathematical expression until the user types 'quit'. The program should then evaluate the expression using the eval function and print the result.

## Reference Solution
```python
while True:
    s = input('> ')
    if s == 'quit':
        break
    print(eval(s))
```
