## Exercise
Generate the Fibonacci sequence up to the nth number using a while loop.

## Reference Solution
```python
def generate_fibonacci(n):
    """
    Generate the Fibonacci sequence up to the nth number using a while loop.

    Args:
        n (int): The number of Fibonacci numbers to generate.

    Returns:
        list: A list of Fibonacci numbers up to the nth number.
    """
    # Initialize the first two Fibonacci numbers
    a, b = 0, 1
    # Initialize the list to store the Fibonacci sequence
    fibonacci_sequence = [a, b]

    # Generate the Fibonacci sequence using a while loop
    i = 2
    while i < n:
        # Calculate the next Fibonacci number
        next_number = a + b
        # Update the previous two numbers
        a, b = b, next_number
        # Append the next number to the Fibonacci sequence
        fibonacci_sequence.append(next_number)
        # Increment the counter
        i += 1

    return fibonacci_sequence

# Example usage
n = 10
fibonacci_sequence = generate_fibonacci(n)
print(f"The Fibonacci sequence up to the {n}th number is: {fibonacci_sequence}")
```
