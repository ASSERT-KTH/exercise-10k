## Exercise
Write a program that calculates the sum of two integers 10 and 20. Print the result.

## Reference Solution
```python
def calculate_sum(a, b):
    """
    Calculates the sum of two integers.
    
    Args:
        a (int): The first integer.
        b (int): The second integer.
    
    Returns:
        int: The sum of a and b.
    """
    return a + b

def main():
    # Define the integers to be summed
    num1 = 10
    num2 = 20
    
    # Calculate the sum
    result = calculate_sum(num1, num2)
    
    # Print the result
    print(f"The sum of {num1} and {num2} is: {result}")

if __name__ == "__main__":
    main()
```
