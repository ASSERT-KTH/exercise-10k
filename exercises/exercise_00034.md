## Exercise
Write a function that takes a string as input and returns True if it's a valid password, False otherwise. A valid password must contain at least 8 characters, one uppercase letter, one lowercase letter, and one digit.

## Reference Solution
```python
def is_valid_password(password: str) -> bool:
    """
    Checks if a given password is valid.
    
    A valid password must contain at least 8 characters, one uppercase letter, one lowercase letter, and one digit.

    Args:
        password (str): The password to be checked.

    Returns:
        bool: True if the password is valid, False otherwise.
    """

    # First, we check if the password has at least 8 characters
    if len(password) < 8:
        return False  # If not, we immediately return False

    # Initialize flags for uppercase, lowercase, and digit
    has_uppercase = False
    has_lowercase = False
    has_digit = False

    # Iterate over each character in the password
    for char in password:
        # Check if the character is an uppercase letter
        if char.isupper():
            has_uppercase = True  # If it is, set the flag to True
        # Check if the character is a lowercase letter
        elif char.islower():
            has_lowercase = True  # If it is, set the flag to True
        # Check if the character is a digit
        elif char.isdigit():
            has_digit = True  # If it is, set the flag to True

    # After checking all characters, we return True if all flags are True
    # This means the password has at least one uppercase letter, one lowercase letter, and one digit
    return has_uppercase and has_lowercase and has_digit
```
