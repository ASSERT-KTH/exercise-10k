## Exercise
Identify and print the primitive data types in the programming language, such as boolean, number, and string.

## Reference Solution
```python
# Define the primitive data types
data_types = {
    "Integer": int,
    "Float": float,
    "Complex": complex,
    "String": str,
    "Boolean": bool,
    "Bytes": bytes,
    "Bytearray": bytearray,
    "Memoryview": memoryview,
}

# Print the primitive data types
for data_type, value in data_types.items():
    print(f"{data_type}: {value}")

# Example usage
print("\nExample Usage:")
print("Integer: ", 10, type(10))
print("Float: ", 10.5, type(10.5))
print("Complex: ", 10 + 5j, type(10 + 5j))
print("String: ", "Hello", type("Hello"))
print("Boolean: ", True, type(True))
print("Bytes: ", b"Hello", type(b"Hello"))
print("Bytearray: ", bytearray(b"Hello"), type(bytearray(b"Hello")))
print("Memoryview: ", memoryview(b"Hello"), type(memoryview(b"Hello")))
```
