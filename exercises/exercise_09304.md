## Exercise
Create a function that takes a string as input and returns the longest word in the string.

## Reference Solution
```python
def longest_word(input_string):
    """
    Returns the longest word in the input string.

    Args:
        input_string (str): The input string to find the longest word in.

    Returns:
        str: The longest word in the input string.
    """
    # Split the input string into words
    words = input_string.split()

    # Initialize the longest word and its length
    longest_word = max(words, key=len)

    return longest_word

# Example usage:
input_string = "This is a test string with multiple words"
print(longest_word(input_string))
```
