## Exercise
Write a function to calculate the sum of deficient numbers less than a given limit

## Reference Solution
```python
def sum_of_deficient_numbers(limit):
    """
    Calculate the sum of deficient numbers less than a given limit.

    Args:
        limit (int): The upper limit for the deficient numbers.

    Returns:
        int: The sum of deficient numbers less than the limit.
    """
    def sum_of_proper_divisors(n):
        """
        Calculate the sum of proper divisors of a number.

        Args:
            n (int): The number for which to calculate the sum of proper divisors.

        Returns:
            int: The sum of proper divisors of the number.
        """
        return sum(i for i in range(1, n) if n % i == 0)

    return sum(num for num in range(1, limit) if num > sum_of_proper_divisors(num))
```
