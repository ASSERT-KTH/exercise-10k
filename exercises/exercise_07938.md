## Exercise
Create a program to calculate the force given the mass and acceleration.

## Reference Solution
```python
def calculate_force(mass, acceleration):
    """
    Calculate the force of an object given its mass and acceleration.

    Args:
        mass (float): The mass of the object in kg.
        acceleration (float): The acceleration of the object in m/s^2.

    Returns:
        float: The force of the object in N.
    """
    force = mass * acceleration
    return force

def main():
    print("Force Calculator Program")
    print("-------------------------")

    mass = float(input("Enter the mass of the object in kg: "))
    acceleration = float(input("Enter the acceleration of the object in m/s^2: "))

    force = calculate_force(mass, acceleration)

    print(f"The force of the object is {force} N")

if __name__ == "__main__":
    main()
```
