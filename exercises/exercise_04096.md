## Exercise
Create a function that can take either a string or a number as input and returns a greeting or the square of the number

## Reference Solution
```python
def greet_or_square(input_value):
    """
    Returns a greeting if input is a string, otherwise returns the square of the input number.

    Args:
        input_value (str or int/float): The input value to be processed.

    Returns:
        str or int/float: A greeting string or the square of the input number.
    """
    if isinstance(input_value, str):
        # Remove leading/trailing whitespaces and capitalize the first letter
        name = input_value.strip().capitalize()
        return f"Hello, {name}!"
    elif isinstance(input_value, (int, float)):
        return input_value ** 2
    else:
        raise ValueError("Input must be a string or a number")

# Example usage:
print(greet_or_square("john"))  # Output: Hello, John!
print(greet_or_square(5))       # Output: 25
print(greet_or_square(3.5))     # Output: 12.25
```
