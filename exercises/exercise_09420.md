## Exercise
Create a function that takes a dictionary and returns the sum of its values

## Reference Solution
```python
def sum_dict_values(dictionary):
    """
    Returns the sum of the values in the dictionary.

    Args:
        dictionary (dict): The dictionary to sum values from.

    Returns:
        int or float: The sum of the dictionary values.

    Raises:
        TypeError: If the dictionary contains non-numeric values.
    """
    try:
        return sum(dictionary.values())
    except TypeError:
        raise TypeError("Dictionary contains non-numeric values.")

# Example usage:
if __name__ == "__main__":
    example_dict = {"a": 1, "b": 2, "c": 3}
    print(sum_dict_values(example_dict))  # Output: 6
```
