## Exercise
Write a function that takes two numbers as arguments and returns the maximum value.

## Reference Solution
```python
def get_max_value(a: float, b: float) -> float:
    """
    Returns the maximum of two input numbers.

    Args:
        a (float): The first number.
        b (float): The second number.

    Returns:
        float: The maximum value between a and b.
    """
    return max(a, b)

# Example usage:
if __name__ == "__main__":
    num1 = 10.5
    num2 = 20.8
    max_value = get_max_value(num1, num2)
    print(f"The maximum value between {num1} and {num2} is {max_value}")
```
