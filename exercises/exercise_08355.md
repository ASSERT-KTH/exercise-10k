## Exercise
Write a program that takes an integer input from the user and prints out all the numbers from 1 to the input number. Use a for loop or a while loop to achieve this. If the input number is less than 1, print an error message.

## Reference Solution
```python
# Author: Your Name
# Date: Today's Date

# Get user input
numero = int(input("Ingrese el número: "))

# Check if input is valid
if numero < 1:
    print("Error: El número debe ser mayor que 0")
else:
    # Print numbers from 1 to input number
    for i in range(1, numero + 1):
        print(i)
```
