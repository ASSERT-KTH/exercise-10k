## Exercise
Write a function to filter a dictionary based on a condition

## Reference Solution
```python
def filter_dict(dictionary, condition):
    """
    Filter a dictionary based on a given condition.

    Args:
        dictionary (dict): The dictionary to be filtered.
        condition (function): A function that takes a key-value pair and returns a boolean value.

    Returns:
        dict: A new dictionary containing only the key-value pairs that satisfy the condition.
    """
    return {key: value for key, value in dictionary.items() if condition(key, value)}


# Example usage:
if __name__ == "__main__":
    # Define a dictionary
    my_dict = {"a": 1, "b": 2, "c": 3, "d": 4}

    # Define a condition function
    def is_even(key, value):
        return value % 2 == 0

    # Filter the dictionary
    filtered_dict = filter_dict(my_dict, is_even)

    # Print the filtered dictionary
    print(filtered_dict)
```
