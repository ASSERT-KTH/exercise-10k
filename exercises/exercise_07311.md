## Exercise
Write a program that takes a name as input from the user and prints out a greeting message.

## Reference Solution
```python
# Get the user's name
name = input("Please enter your name: ")

# Print out a greeting message
print("Hello, " + name + "! Welcome!")
```

```python
# Get the user's name
```
```python
while True:
    name = input("Please enter your name: ")
    if name.strip():
        break
    print("You didn't enter a name. Please try again.")

# Print out a greeting message
print(f"Hello, {name}! Welcome!")
```
