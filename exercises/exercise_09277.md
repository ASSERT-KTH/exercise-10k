## Exercise
Write a program that prints a pattern of asterisks (*) based on the input number. The pattern should be a square of asterisks with the size of the input number. For example, if the input is 5, the output should be a 5x5 square of asterisks.

## Reference Solution
```python
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);

    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            printf("*");
        }
        printf("\n");
    }

    return 0;
}
```
