## Exercise
Create a function to remove all vowels from a given string. The function should return the string without vowels.

## Reference Solution
```python
def remove_vowels(input_string):
    """
    This function removes all vowels from a given string.

    Args:
        input_string (str): The input string from which vowels are to be removed.

    Returns:
        str: The input string without vowels.
    """

    # First, we define the vowels in both lower and upper case
    vowels = 'aeiouAEIOU'

    # Then, we use a list comprehension to generate a new string without vowels
    # We iterate over each character in the input string
    # If the character is not a vowel, we include it in the new string
    no_vowels = [char for char in input_string if char not in vowels]

    # Finally, we join the characters in the list back into a string
    # We use the join() method, which concatenates all the characters in the list
    result = ''.join(no_vowels)

    # We return the resulting string
    return result

# Example usage:
print(remove_vowels('Hello World'))  # Output: 'Hll Wrld'
```
