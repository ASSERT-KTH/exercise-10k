## Exercise
Write a function that takes a list as input, and returns a sorted version of the list.

## Reference Solution
```python
def sort_list(input_list):
    """
    Returns a sorted version of the input list.

    Args:
        input_list (list): A list of elements to be sorted.

    Returns:
        list: A sorted list of elements.
    """
    return sorted(input_list)

# Example usage:
if __name__ == "__main__":
    numbers = [64, 34, 25, 12, 22, 11, 90]
    print("Original list:", numbers)
    print("Sorted list:", sort_list(numbers))
```
