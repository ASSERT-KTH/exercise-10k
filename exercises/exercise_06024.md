## Exercise
Write a function that takes two arguments, one with a default value. Use the function to demonstrate the use of default arguments.

## Reference Solution
```python
def greet(name, message="Hello"):
    """
    Prints a greeting message for a given name.

    Args:
        name (str): The name of the person to greet.
        message (str, optional): The greeting message. Defaults to "Hello".
    """
    print(f"{message}, {name}!")

# Example usage
if __name__ == "__main__":
    # Using the default message
    greet("John")
    
    # Providing a custom message
    greet("Jane", "Hi")
    
    # Using the default message again
    greet("Bob")
```
