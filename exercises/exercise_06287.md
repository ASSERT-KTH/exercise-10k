## Exercise
Use the get() method to retrieve the value associated with a given key

## Reference Solution
```python
# Create a dictionary
person = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

# Retrieve the value associated with the key "name"
name = person.get("name")
print(name)  # Output: John

# Retrieve the value associated with the key "country" (which does not exist)
country = person.get("country")
print(country)  # Output: None

# Retrieve the value associated with the key "country" with a default value
country = person.get("country", "USA")
print(country)  # Output: USA
```
