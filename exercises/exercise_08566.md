## Exercise
Create a dictionary with 3 key-value pairs and convert it to a set of keys.

## Reference Solution
```python
def dict_to_key_set():
    # Create a dictionary with 3 key-value pairs
    my_dict = {
        "name": "John",
        "age": 30,
        "city": "New York"
    }

    # Convert the dictionary to a set of keys
    key_set = set(my_dict.keys())

    return my_dict, key_set

# Example usage
if __name__ == "__main__":
    my_dict, key_set = dict_to_key_set()
    print("Dictionary:")
    print(my_dict)
    print("\nSet of Keys:")
    print(key_set)
```
