## Exercise
Create a function that takes an integer as an argument and returns True if it is even, False otherwise

## Reference Solution
```python
def is_even(n: int) -> bool:
    """
    Checks if a given integer is even.

    Args:
        n (int): The integer to check.

    Returns:
        bool: True if the number is even, False otherwise.
    """
    return n % 2 == 0

# Example usage:
if __name__ == "__main__":
    print(is_even(10))  # True
    print(is_even(11))  # False
```
