## Exercise
Write a program that takes an array and an index as input and returns a new array containing all elements except the one at the specified index.

## Reference Solution
```python
function removeAt(arr, idx) {
  return arr.slice(0, idx).concat(arr.slice(idx + 1));
}

console.log(removeAt([1, 2, 3, 4, 5], 1));
```
