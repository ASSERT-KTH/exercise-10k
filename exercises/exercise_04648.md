## Exercise
Create a program that simulates a coin flip. The program generates a random result (heads or tails) and displays the result.

## Reference Solution
```python
import random

def coin_flip():
    """
    Simulates a coin flip and returns the result.
    """
    return "heads" if random.randint(0, 1) == 0 else "tails"

def main():
    print("Coin Flip Simulator")
    print("--------------------")
    result = coin_flip()
    print(f"The result is: {result}")

if __name__ == "__main__":
    main()
```
