## Exercise
Implement a linear search algorithm to find a specific element in a list of numbers

## Reference Solution
```python
def linear_search(numbers, target):
    """
    Searches for a specific element in a list of numbers using linear search.

    Args:
    - numbers (list): A list of numbers.
    - target (int): The target element to search for.

    Returns:
    - index (int): The index of the target element if found, -1 otherwise.
    """
    for index, num in enumerate(numbers):
        if num == target:
            return index
    return -1

# Example usage:
numbers = [4, 2, 9, 6, 5, 1, 8, 3, 7]
target = 6
result = linear_search(numbers, target)

if result != -1:
    print(f"Target element {target} found at index {result}.")
else:
    print(f"Target element {target} not found in the list.")
```
