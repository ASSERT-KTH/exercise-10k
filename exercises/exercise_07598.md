## Exercise
Write a program that compares two variables and returns the result.

## Reference Solution
```python
def compare_variables(var1, var2):
    """
    Compare two variables and return the result.

    Args:
        var1: The first variable to compare.
        var2: The second variable to compare.

    Returns:
        A string describing the comparison result.
    """
    if var1 == var2:
        return f"{var1} is equal to {var2}"
    elif var1 > var2:
        return f"{var1} is greater than {var2}"
    else:
        return f"{var1} is less than {var2}"

def main():
    # Example usage
    var1 = 5
    var2 = 10
    result = compare_variables(var1, var2)
    print(result)

if __name__ == "__main__":
    main()
```
