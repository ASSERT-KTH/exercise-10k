## Exercise
Create a function to calculate the product of a list of numbers.

## Reference Solution
```python
def calculate_product(numbers):
    """
    Calculate the product of a list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        float: The product of all the numbers in the list.

    Raises:
        ValueError: If the list contains non-numeric values.
    """
    # Check if the list contains non-numeric values
    if not all(isinstance(num, (int, float)) for num in numbers):
        raise ValueError("The list must only contain numbers")

    # Initialize the product to 1
    product = 1

    # Calculate the product of all numbers in the list
    for num in numbers:
        product *= num

    return product

# Example usage
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    print(calculate_product(numbers))  # Output: 120
```
