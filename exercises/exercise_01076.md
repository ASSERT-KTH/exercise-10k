## Exercise
Create a program that uses a for loop to print a calendar for a given month and year.

## Reference Solution
```python
import calendar

def print_monthly_calendar(year, month):
    # Create a plain text calendar
    cal = calendar.monthcalendar(year, month)

    # Print the month and year
    print(calendar.month_name[month], year)
    print("Mo Tu We Th Fr Sa Su")

    # Print the days of the month
    for week in cal:
        for day in week:
            if day == 0:
                print("  ", end=" ")  # Print spaces for days not in the month
            else:
                print(f"{day:>2}", end=" ")  # Print days with leading spaces
        print()  # Print a newline after each week

# Test the function
print_monthly_calendar(2024, 9)
```
