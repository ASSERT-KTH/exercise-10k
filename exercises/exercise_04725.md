## Exercise
Create a function that takes a variable number of arguments and prints their sum.

## Reference Solution
```python
# Define a function that takes a variable number of arguments
def print_sum(*args):
    """
    This function takes a variable number of arguments and prints their sum.

    Args:
        *args: A variable number of arguments

    Returns:
        None
    """

    # Initialize a variable to store the sum
    total = 0
    
    # Check if all arguments are numbers
    if not all(isinstance(arg, (int, float)) for arg in args):
        # If not, print an error message
        print("Error: All arguments must be numbers")
    else:
        # Iterate over each argument
        for arg in args:
            # Add the argument to the total
            total += arg
        
        # Print the total
        print(total)

# Test the function with some examples
print_sum(1, 2, 3, 4, 5)  # Output: 15
print_sum(1.5, 2.5, 3.5)  # Output: 7.5
print_sum('a', 'b', 'c')  # Output: Error: All arguments must be numbers
```
