## Exercise
Create a function that takes a string as an argument and returns its uppercase version

## Reference Solution
```python
def to_uppercase(input_string: str) -> str:
    """
    Convert a string to its uppercase version.

    Args:
    input_string (str): The input string to be converted.

    Returns:
    str: The uppercase version of the input string.
    """
    return input_string.upper()

# Example usage
def main():
    input_str = "hello world"
    print(f"Original string: {input_str}")
    print(f"Uppercase string: {to_uppercase(input_str)}")

if __name__ == "__main__":
    main()
```
