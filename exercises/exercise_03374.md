## Exercise
Use a for loop to generate a list of prime numbers up to a given limit

## Reference Solution
```python
def generate_primes(limit):
    """
    Generate a list of prime numbers up to the given limit.

    Args:
        limit (int): The upper limit for generating prime numbers.

    Returns:
        list: A list of prime numbers up to the given limit.
    """
    primes = []
    for num in range(2, limit + 1):
        is_prime = True
        for i in range(2, int(num ** 0.5) + 1):
            if num % i == 0:
                is_prime = False
                break
        if is_prime:
            primes.append(num)
    return primes

# Example usage
limit = 50
primes = generate_primes(limit)
print(f"Prime numbers up to {limit}: {primes}")
```
