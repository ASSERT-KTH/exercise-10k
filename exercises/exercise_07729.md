## Exercise
Find all substrings with k distinct characters in a given string.

## Reference Solution
```python
from collections import defaultdict

def find_substrings_with_k_distinct_characters(s, k):
    """
    Find all substrings of `s` that contain exactly `k` distinct characters.

    Args:
        s (str): The input string.
        k (int): The number of distinct characters.

    Returns:
        list: A list of substrings with `k` distinct characters.
    """
    result = []
    for i in range(len(s)):
        char_freq = defaultdict(int)
        distinct_chars = 0
        for j in range(i, len(s)):
            if char_freq[s[j]] == 0:
                distinct_chars += 1
            char_freq[s[j]] += 1
            if distinct_chars == k:
                result.append(s[i:j+1])
    return result

# Example usage:
s = "abcabc"
k = 2
print(find_substrings_with_k_distinct_characters(s, k))
```
