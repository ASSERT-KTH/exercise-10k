## Exercise
Given a dictionary with names of books as keys and their authors as values, write a program to iterate through the dictionary and print each key-value pair.

## Reference Solution
```python
# Define a dictionary with book names as keys and authors as values
books = {
    "To Kill a Mockingbird": "Harper Lee",
    "1984": "George Orwell",
    "Pride and Prejudice": "Jane Austen",
    "The Great Gatsby": "F. Scott Fitzgerald",
    "The Catcher in the Rye": "J.D. Salinger"
}

# Iterate through the dictionary and print each key-value pair
for book, author in books.items():
    print(f"Book: {book}, Author: {author}")
```
