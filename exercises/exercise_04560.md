## Exercise
Write a program that translates a given string into Pig Latin.

## Reference Solution
```python
def translate_to_pig_latin(word):
    """
    Translates a given word into Pig Latin.
    
    Parameters:
    word (str): The word to be translated.
    
    Returns:
    str: The translated word in Pig Latin.
    """
    vowels = 'aeiou'
    if word[0].lower() in vowels:
        return word + 'way'
    else:
        vowel_index = 0
        for char in word:
            if char.lower() in vowels:
                break
            vowel_index += 1
        return word[vowel_index:] + word[:vowel_index] + 'ay'

def translate_sentence_to_pig_latin(sentence):
    """
    Translates a given sentence into Pig Latin.
    
    Parameters:
    sentence (str): The sentence to be translated.
    
    Returns:
    str: The translated sentence in Pig Latin.
    """
    words = sentence.split()
    translated_words = [translate_to_pig_latin(word) for word in words]
    return ' '.join(translated_words)

def main():
    sentence = input("Enter a sentence to translate: ")
    translated_sentence = translate_sentence_to_pig_latin(sentence)
    print("Translated sentence in Pig Latin: ", translated_sentence)

if __name__ == "__main__":
    main()
```
