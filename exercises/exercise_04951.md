## Exercise
Write a program that checks whether a person is an adult or a minor based on their age using conditional statements.

## Reference Solution
```python
def check_adult_or_minor(age):
    if age >= 18:
        return "Adult"
    else:
        return "Minor"

age = int(input("Enter your age: "))
result = check_adult_or_minor(age)
print(f"You are a {result}.")

# Example usage:
# Enter your age: 25
# You are a Adult.

# Enter your age: 17
# You are a Minor.
```
