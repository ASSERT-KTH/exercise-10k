## Exercise
Write a program that takes a string as input and returns whether the string is palindromic or not.

## Reference Solution
```python
def is_palindromic(s: str) -> bool:
    """
    Checks if a given string is palindromic.

    Args:
        s (str): The input string.

    Returns:
        bool: True if the string is palindromic, False otherwise.
    """
    s = ''.join(c for c in s if c.isalnum()).lower()  # Remove non-alphanumeric characters and convert to lowercase
    return s == s[::-1]  # Check if the string is equal to its reverse

# Example usage:
def main():
    input_str = input("Enter a string: ")
    if is_palindromic(input_str):
        print(f"'{input_str}' is a palindromic string.")
    else:
        print(f"'{input_str}' is not a palindromic string.")

if __name__ == "__main__":
    main()
```
