## Exercise
Write a program that takes a number as input and classifies it as 'positive', 'negative', or 'zero' using if/else statements

## Reference Solution
```python
def classify_number(num):
    """
    Classify a number as 'positive', 'negative', or 'zero'.

    Args:
        num (int): The number to classify.

    Returns:
        str: The classification of the number ('positive', 'negative', or 'zero').
    """
    if num > 0:
        return "positive"
    elif num < 0:
        return "negative"
    else:
        return "zero"

num = int(input("Enter a number: "))
print(classify_number(num))
```
