## Exercise
Create a function that takes an array and a mapping function as input and returns a new array containing the results of applying the mapping function to each element.

## Reference Solution
```python
def map_function(array, func):
    """
    This function takes an array and a mapping function as input, 
    applies the mapping function to each element in the array, 
    and returns a new array containing the results.

    Args:
        array (list): The input array.
        func (function): The mapping function to be applied.

    Returns:
        list: A new array containing the results of applying the mapping function.
    """

    # Initialize an empty list to store the results
    result = []

    # Iterate over each element in the input array
    for element in array:
        # Apply the mapping function to the current element
        mapped_element = func(element)
        
        # Append the result to the result list
        result.append(mapped_element)

    # Return the result list
    return result

# Example usage:
# Define a simple mapping function that squares a number
def square(x):
    return x ** 2

# Create an input array
array = [1, 2, 3, 4, 5]

# Apply the mapping function to the array
result = map_function(array, square)

# Print the result
print(result)  # Output: [1, 4, 9, 16, 25]
```
