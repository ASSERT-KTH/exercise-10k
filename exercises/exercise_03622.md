## Exercise
Write a program to find the maximum value in a given list of integers.

## Reference Solution
```python
def find_max_value(num_list):
    """
    Finds the maximum value in a given list of integers.

    Args:
        num_list (list): A list of integers.

    Returns:
        int: The maximum value in the list.
    """
    if not num_list:
        raise ValueError("List is empty")

    max_value = num_list[0]
    for num in num_list:
        if num > max_value:
            max_value = num

    return max_value

# Example usage
numbers = [4, 2, 9, 6, 5, 1]
max_val = find_max_value(numbers)
print(f"Maximum value: {max_val}")
```
