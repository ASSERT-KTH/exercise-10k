## Exercise
Write a program that takes an array of integers as input and returns the array with its elements reversed. For example, if the input array is [1, 2, 3, 4, 5], the output should be [5, 4, 3, 2, 1].

## Reference Solution
```python
def reverse_array(arr):
    """
    This function takes an array of integers as input and returns the array with its elements reversed.

    Parameters:
    arr (list): The input array of integers.

    Returns:
    list: The input array with its elements reversed.
    """
    return arr[::-1]

# Example usage:
input_array = [1, 2, 3, 4, 5]
print("Input array:", input_array)
print("Reversed array:", reverse_array(input_array))
```

```python
def reverse_array(arr):
    left = 0
    right = len(arr) - 1
    while left < right:
        arr[left], arr[right] = arr[right], arr[left]
        left += 1
        right -= 1
    return arr
```
