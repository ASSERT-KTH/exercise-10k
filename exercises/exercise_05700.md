## Exercise
Write a program that takes a string as input and checks if it's a palindrome using conditional statements.

## Reference Solution
```python
def is_palindrome(s):
    s = s.lower()  # Convert to lowercase
    reversed_s = s[::-1]  # Reverse the string

    if s == reversed_s:
        return True
    else:
        return False

# Test the function
s = input("Enter a string: ")
if is_palindrome(s):
    print(f"'{s}' is a palindrome.")
else:
    print(f"'{s}' is not a palindrome.")
```
