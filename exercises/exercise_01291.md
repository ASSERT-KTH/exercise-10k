## Exercise
Create a function that returns a list of all values in a dictionary.

## Reference Solution
```python
def get_dict_values(dictionary):
    """
    Returns a list of all values in a dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        list: A list of all values in the dictionary.
    """

    # First, we check if the input is indeed a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("Input must be a dictionary")

    # We can use the .values() method of dictionaries to get a view object that displays a list of all values in the dictionary
    dict_values = dictionary.values()

    # Since .values() returns a view object, we need to convert it to a list
    # We can do this using the list() function
    dict_values_list = list(dict_values)

    # Finally, we return the list of dictionary values
    return dict_values_list

# Example usage:
my_dict = {"a": 1, "b": 2, "c": 3}
print(get_dict_values(my_dict))  # Output: [1, 2, 3]
```
