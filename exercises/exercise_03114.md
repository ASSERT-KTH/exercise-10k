## Exercise
Create a dictionary with 5 key-value pairs. Check if a specific key exists in the dictionary.

## Reference Solution
```python
# Create a dictionary with 5 key-value pairs
my_dict = {
    "apple": 5,
    "banana": 10,
    "cherry": 15,
    "date": 20,
    "elderberry": 25
}

# Define the key to be searched
key_to_search = "banana"

# Check if the key exists in the dictionary
if key_to_search in my_dict:
    print(f"The key '{key_to_search}' exists in the dictionary with value: {my_dict[key_to_search]}")
else:
    print(f"The key '{key_to_search}' does not exist in the dictionary")

# Alternative approach using the get() method
value = my_dict.get(key_to_search)
if value is not None:
    print(f"The key '{key_to_search}' exists in the dictionary with value: {value}")
else:
    print(f"The key '{key_to_search}' does not exist in the dictionary")
```
