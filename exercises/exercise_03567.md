## Exercise
Write a function that reduces a dictionary to a single value

## Reference Solution
```python
def reduce_dict(dictionary, operation):
    """
    Reduces a dictionary to a single value by applying a specified operation.

    Args:
        dictionary (dict): The input dictionary.
        operation (str): The reduction operation to apply. Supported operations are 'sum', 'avg', 'max', and 'min'.

    Returns:
        The reduced value.

    Raises:
        ValueError: If the operation is not supported.
    """
    if operation == 'sum':
        return sum(dictionary.values())
    elif operation == 'avg':
        return sum(dictionary.values()) / len(dictionary)
    elif operation == 'max':
        return max(dictionary.values())
    elif operation == 'min':
        return min(dictionary.values())
    else:
        raise ValueError("Unsupported operation. Supported operations are 'sum', 'avg', 'max', and 'min'.")

# Example usage
if __name__ == "__main__":
    dictionary = {'a': 1, 'b': 2, 'c': 3, 'd': 4}
    print("Sum:", reduce_dict(dictionary, 'sum'))  # Output: 10
    print("Average:", reduce_dict(dictionary, 'avg'))  # Output: 2.5
    print("Max:", reduce_dict(dictionary, 'max'))  # Output: 4
    print("Min:", reduce_dict(dictionary, 'min'))  # Output: 1
```
