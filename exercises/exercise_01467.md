## Exercise
Write a program that takes two integer inputs from the user, performs integer division, and prints the result.

## Reference Solution
```python
#include <stdio.h>

int main() {
    int a, b;

    printf("Enter two integers: ");
    scanf("%d %d", &a, &b);

    if (b == 0) {
        printf("Error: Division by zero is not allowed.\n");
    } else {
        int result = a / b;
        printf("%d divided by %d is %d\n", a, b, result);
    }

    return 0;
}
```
