## Exercise
Create a function that finds the maximum value in a list of numbers. The function should take a list as an argument and return the maximum value.

## Reference Solution
```python
def find_max_value(num_list):
    """
    This function finds the maximum value in a list of numbers.

    Args:
        num_list (list): A list of numbers.

    Returns:
        max_val (int or float): The maximum value in the list.
    """

    # First, we need to check if the input is a list. If it's not, we raise a TypeError.
    if not isinstance(num_list, list):
        raise TypeError("Input must be a list.")

    # Then, we need to check if the list is empty. If it is, we raise a ValueError.
    if len(num_list) == 0:
        raise ValueError("List cannot be empty.")

    # Now, we need to check if all elements in the list are numbers. If not, we raise a ValueError.
    if not all(isinstance(num, (int, float)) for num in num_list):
        raise ValueError("All elements in the list must be numbers.")

    # Now, we can use the built-in max function to find the maximum value in the list.
    # The max function returns the largest item in an iterable or the largest of two or more arguments.
    max_val = max(num_list)

    # Finally, we return the maximum value.
    return max_val
```
