## Exercise
Write a program that checks if a given variable is a string.

## Reference Solution
```python
def is_string(var):
    """
    Checks if a given variable is a string.

    Args:
        var: The variable to check.

    Returns:
        bool: True if the variable is a string, False otherwise.
    """
    return isinstance(var, str)

# Example usage:
if __name__ == "__main__":
    var1 = "Hello, World!"
    var2 = 123
    var3 = 3.14
    var4 = True

    print(f"{var1} is a string: {is_string(var1)}")
    print(f"{var2} is a string: {is_string(var2)}")
    print(f"{var3} is a string: {is_string(var3)}")
    print(f"{var4} is a string: {is_string(var4)}")
```
