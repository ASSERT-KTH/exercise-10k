## Exercise
Create a program that takes a string as input, reverses it, and then outputs the result. The program should also handle cases where the input is not a string, and provide a meaningful error message.

## Reference Solution
```python
def reverse(s):
    if not isinstance(s, str):
        raise TypeError("Input must be a string.")
    return s[::-1]

# Test the function
```
