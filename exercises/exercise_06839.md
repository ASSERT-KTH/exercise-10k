## Exercise
Write a program that takes an array of integers as input and returns the maximum value in the array. The program should use a basic algorithm to iterate through the array and find the maximum value.

## Reference Solution
```python
#include <iostream>
using namespace std;

int main() {
    int a[] = {2, 4, 9, 6, 5, 1, 8, 3, 7};
    int n = sizeof(a) / sizeof(a[0]);
    int max = a[0];

    for (int i = 1; i < n; i++) {
        if (a[i] > max) {
            max = a[i];
        }
    }

    cout << "Maximum value in the array is: " << max << endl;

    return 0;
}
```
