## Exercise
Write a function to get a list of key-value pairs from a dictionary

## Reference Solution
```python
def get_key_value_pairs(dictionary):
    """
    Returns a list of key-value pairs from a dictionary.

    Args:
        dictionary (dict): The dictionary to get key-value pairs from.

    Returns:
        list: A list of tuples, where each tuple contains a key-value pair.
    """
    return list(dictionary.items())

# Example usage:
my_dict = {"name": "John", "age": 30, "city": "New York"}
key_value_pairs = get_key_value_pairs(my_dict)
print(key_value_pairs)
```
