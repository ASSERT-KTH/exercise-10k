## Exercise
Write a program that takes a string input from the user, reverses the order of its characters, and converts it to uppercase. The program should then print the modified string.

## Reference Solution
```python
# This program takes a string input from the user, reverses the order of its characters,
# and converts it to uppercase. The program then prints the modified string.

s = input("Enter a string: ")
s = s[::-1]
s = s.upper()
print(s)
```
