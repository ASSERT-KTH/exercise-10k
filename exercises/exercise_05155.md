## Exercise
Convert a list of key-value pairs to a dictionary

## Reference Solution
```python
# List of key-value pairs
key_value_pairs = [('a', 1), ('b', 2), ('c', 3)]

# Convert the list to a dictionary
dictionary = dict(key_value_pairs)

print(dictionary)  # Output: {'a': 1, 'b': 2, 'c': 3}
```

```python
# List of key-value pairs
key_value_pairs = [('a', 1), ('b', 2), ('c', 3)]

# Convert the list to a dictionary using a dictionary comprehension
dictionary = {key: value for key, value in key_value_pairs}

print(dictionary)  # Output: {'a': 1, 'b': 2, 'c': 3}
```

```python
# List of key-value pairs
key_value_pairs = [('a', 1), ('b', 2), ('c', 3)]

# Initialize an empty dictionary
dictionary = {}

# Convert the list to a dictionary using a loop
for key, value in key_value_pairs:
    dictionary[key] = value

print(dictionary)  # Output: {'a': 1, 'b': 2, 'c': 3}
```

```python
# List of key-value pairs with duplicate keys
key_value_pairs = [('a', 1), ('b', 2), ('a', 3)]

# Convert the list to a dictionary
dictionary = dict(key_value_pairs)

print(dictionary)  # Output: {'a': 3, 'b': 2}
```
