## Exercise
Perform common list operations such as indexing, slicing, and appending.

## Reference Solution
```python
# Create a list
my_list = [1, 2, 3, 4, 5]

# Indexing: Accessing a specific element
print(my_list[0])  # Output: 1
print(my_list[-1])  # Output: 5 (last element)

# Slicing: Accessing a subset of elements
print(my_list[1:3])  # Output: [2, 3]
print(my_list[:3])   # Output: [1, 2, 3]
print(my_list[2:])   # Output: [3, 4, 5]

# Appending: Adding an element to the end
my_list.append(6)
print(my_list)  # Output: [1, 2, 3, 4, 5, 6]

# Inserting: Adding an element at a specific position
my_list.insert(2, 10)
print(my_list)  # Output: [1, 2, 10, 3, 4, 5, 6]

# Removing: Removing the first occurrence of an element
my_list.remove(10)
print(my_list)  # Output: [1, 2, 3, 4, 5, 6]

# Pop: Removing an element at a specific position
popped_element = my_list.pop(2)
print(popped_element)  # Output: 3
print(my_list)  # Output: [1, 2, 4, 5, 6]

# Extend: Adding multiple elements to the end
my_list.extend([7, 8, 9])
print(my_list)  # Output: [1, 2, 4, 5, 6, 7, 8, 9]
```
