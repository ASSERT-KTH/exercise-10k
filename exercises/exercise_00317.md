## Exercise
Write a function that reduces a given array by applying a specified operation to all elements.

## Reference Solution
```python
from functools import reduce

def reduce_array(array, operation):
    """
    Reduces a given array by applying a specified operation to all elements.

    Args:
        array (list): The input array.
        operation (function): The operation to apply to the array elements.

    Returns:
        The result of the reduction operation.
    """

    # Check if the array is empty
    if not array:
        raise ValueError("Array is empty")

    # Apply the reduction operation
    result = reduce(operation, array)

    return result


# Example usage:
if __name__ == "__main__":
    # Define a simple addition operation
    def add(x, y):
        return x + y

    # Create an example array
    array = [1, 2, 3, 4, 5]

    # Reduce the array using the addition operation
    result = reduce_array(array, add)

    print("Result:", result)
```
