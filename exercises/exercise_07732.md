## Exercise
Write a program that takes in a variable and its data type as input from the user, and then validates if the variable's data type matches the given data type. If it does, print 'Valid', otherwise print 'Invalid'. The data types to consider are integer, float, and string.

## Reference Solution
```python
def validate_data_type(var, data_type):
    """
    Validate if the variable's data type matches the given data type.

    Parameters:
    var (any): The variable to check.
    data_type (str): The expected data type. Can be 'integer', 'float', or 'string'.

    Returns:
    str: 'Valid' if the data type matches, 'Invalid' otherwise.
    """
    # Define the mapping between the expected data types and the actual Python types
    type_mapping = {
        'integer': int,
        'float': float,
        'string': str
    }

    # Check if the given data type is valid
    if data_type not in type_mapping:
        raise ValueError(f"Invalid data type: {data_type}")

    # Check if the variable's type matches the expected type
    if isinstance(var, type_mapping[data_type]):
        return 'Valid'
    else:
        return 'Invalid'


def main():
    # Get the variable and its expected data type from the user
    var = input("Enter a variable: ")
    data_type = input("Enter the expected data type (integer, float, or string): ")

    # Try to convert the variable to a Python object
    try:
        if data_type == 'integer':
            var = int(var)
        elif data_type == 'float':
            var = float(var)
        elif data_type == 'string':
            var = str(var)
        else:
            raise ValueError(f"Invalid data type: {data_type}")
    except ValueError:
        print('Invalid')
        return

    # Validate the data type
    print(validate_data_type(var, data_type))


if __name__ == "__main__":
    main()
```
