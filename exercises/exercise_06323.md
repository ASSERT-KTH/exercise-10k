## Exercise
Write a program that takes an array as input, sorts it in ascending order, and returns the sorted array.

## Reference Solution
```python
import java.util.Arrays;

public class SortArray {
    public static void main(String[] args) {
        int[] array = {5, 1, 4, 2, 3};
        System.out.println(Arrays.toString(sortArray(array)));
    }

    public static int[] sortArray(int[] array) {
        Arrays.sort(array);
        return array;
    }
}
```
