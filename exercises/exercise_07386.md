## Exercise
Write a program that takes an array of integers as input and returns a new array where all elements are reversed. For example, if the input array is [1, 2, 3, 4, 5], the output array should be [5, 4, 3, 2, 1]. The program should handle arrays of any length and should not modify the original array.

## Reference Solution
```python
input_list = [1, 2, 3, 4, 5]

def reverse_list(input_list):
    reversed_list = input_list[::-1]
    return reversed_list

result_list = reverse_list(input_list)

print(result_list)
```
