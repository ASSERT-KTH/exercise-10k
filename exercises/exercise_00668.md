## Exercise
Given a dictionary {'name': 'Frank', 'age': 30, 'city': 'Melbourne'}, write a program to add a new key-value pair 'country': 'Australia'.

## Reference Solution
```python
# Define the dictionary
person = {'name': 'Frank', 'age': 30, 'city': 'Melbourne'}

# Print the original dictionary
print("Original Dictionary:")
print(person)

# Add a new key-value pair
person['country'] = 'Australia'

# Print the updated dictionary
print("\nUpdated Dictionary:")
print(person)
```
