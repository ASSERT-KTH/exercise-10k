## Exercise
Create a function that takes a number as an argument and returns its reciprocal, with error handling for division by zero

## Reference Solution
```python
def calculate_reciprocal(number):
    """
    Calculate the reciprocal of a given number.

    Args:
        number (float): The input number.

    Returns:
        float: The reciprocal of the input number.

    Raises:
        ZeroDivisionError: If the input number is zero.
        TypeError: If the input is not a number.
    """

    # Check if the input is a number
    if not isinstance(number, (int, float)):
        raise TypeError("Input must be a number.")

    # Check for division by zero
    if number == 0:
        raise ZeroDivisionError("Cannot calculate reciprocal of zero.")

    # Calculate and return the reciprocal
    return 1 / number


# Example usage:
if __name__ == "__main__":
    try:
        print(calculate_reciprocal(5))  # Output: 0.2
        print(calculate_reciprocal(0))  # Raises ZeroDivisionError
    except ZeroDivisionError as e:
        print(f"Error: {e}")
    except TypeError as e:
        print(f"Error: {e}")
```
