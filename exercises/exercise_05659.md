## Exercise
Write a function that reverses a given list of numbers.

## Reference Solution
```python
def reverse_list(numbers):
    """
    Reverses a given list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        list: A new list with the numbers in reverse order.
    """
    return numbers[::-1]

# Example usage:
numbers = [1, 2, 3, 4, 5]
reversed_numbers = reverse_list(numbers)
print(reversed_numbers)  # Output: [5, 4, 3, 2, 1]
```
