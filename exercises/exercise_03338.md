## Exercise
Write a program that takes two 2D arrays of integers as input and returns the sum of the two arrays, element-wise.

## Reference Solution

def matrix_add(A, B):
    """
    Returns the sum of two matrices, element-wise.

    Example:
    >>> matrix_add([[1, 2], [3, 4]], [[5, 6], [7, 8]])
    [[6, 8], [10, 12]]
    """
    return [[a + b for a, b in zip(row1, row2)] for row1, row2 in zip(A, B)]
