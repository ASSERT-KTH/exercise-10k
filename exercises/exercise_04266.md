## Exercise
Write a function that takes a dictionary as input and returns a list of all keys

## Reference Solution
```python
def get_dictionary_keys(input_dict):
    """
    Returns a list of all keys in the input dictionary.

    Args:
    input_dict (dict): The input dictionary.

    Returns:
    list: A list of all keys in the input dictionary.
    """
    return list(input_dict.keys())

# Example usage:
def main():
    example_dict = {"name": "John", "age": 30, "city": "New York"}
    keys = get_dictionary_keys(example_dict)
    print(keys)

if __name__ == "__main__":
    main()
```
