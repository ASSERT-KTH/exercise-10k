## Exercise
Write a program that takes a sorted list of integers and a target value as input and returns the index of the target value in the list using binary search.

## Reference Solution
```python
def binary_search(nums, target):
    left = 0
    right = len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] > target:
            right = mid - 1
        else:
            left = mid + 1
    return -1

nums = [1, 2, 3, 4, 5, 6, 7, 8, 9]
target = 6
res = binary_search(nums, target)
print(res)
```
