## Exercise
Create a program that checks if a given string contains only uppercase letters, only lowercase letters, or a mix of both.

## Reference Solution
```python
import re

def check_case(s):
    if re.match("^[A-Z]+$", s):
        return "The string contains only uppercase letters."
    elif re.match("^[a-z]+$", s):
        return "The string contains only lowercase letters."
    else:
        return "The string contains a mix of uppercase and lowercase letters."

def main():
    s = input("Enter a string: ")
    if not s.isalpha():
        print("The string should contain only letters.")
    else:
        print(check_case(s))

if __name__ == "__main__":
    main()
```
