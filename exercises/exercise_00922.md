## Exercise
Write a program that finds the maximum of two numbers using if/else statement

## Reference Solution
```python
def find_max(num1, num2):
    """
    This function finds the maximum of two numbers using if/else statement.

    Args:
        num1 (int): The first number.
        num2 (int): The second number.

    Returns:
        int: The maximum of the two numbers.
    """
    if num1 > num2:
        return num1
    else:
        return num2

# Example usage
def main():
    num1 = int(input("Enter the first number: "))
    num2 = int(input("Enter the second number: "))

    max_num = find_max(num1, num2)
    print("The maximum of", num1, "and", num2, "is:", max_num)

if __name__ == "__main__":
    main()
```
