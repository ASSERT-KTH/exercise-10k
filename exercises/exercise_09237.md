## Exercise
Write a program that declares and uses boolean variables.

## Reference Solution
```python
# Define a function to demonstrate boolean variables
def demonstrate_booleans():
    # Declare boolean variables
    is_admin = True
    is_guest = False

    # Print the values of the boolean variables
    print("Is Admin:", is_admin)
    print("Is Guest:", is_guest)

    # Use boolean variables in conditional statements
    if is_admin:
        print("Welcome, Admin!")
    else:
        print("Welcome, Guest!")

    # Use boolean variables in logical operations
    is_verified = True
    is_active = False
    print("Is Verified and Active:", is_verified and is_active)
    print("Is Verified or Active:", is_verified or is_active)
    print("Is Not Verified:", not is_verified)

# Call the function to demonstrate boolean variables
demonstrate_booleans()
```
