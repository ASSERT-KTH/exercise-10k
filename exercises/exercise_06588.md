## Exercise
Write a program to find the product of all numbers in an array.

## Reference Solution
```python
def product_of_array(arr):
    """
    Calculate the product of all numbers in an array.

    Args:
        arr (list): A list of numbers.

    Returns:
        int: The product of all numbers in the array.
    """
    product = 1
    for num in arr:
        product *= num
    return product

def main():
    # Example usage
    arr = [1, 2, 3, 4, 5]
    result = product_of_array(arr)
    print("Product of array elements:", result)

if __name__ == "__main__":
    main()
```
