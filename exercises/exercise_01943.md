## Exercise
Write a program that takes an array as input and returns its length.

## Reference Solution
```python
def get_array_length(array):
    """
    Returns the length of the input array.

    Args:
        array (list): The input array.

    Returns:
        int: The length of the array.
    """
    return len(array)

# Example usage
if __name__ == "__main__":
    array = [1, 2, 3, 4, 5]
    print("Array:", array)
    print("Length:", get_array_length(array))
```
