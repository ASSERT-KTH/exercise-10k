## Exercise
Write a program that swaps the values of two variables without using a temporary variable.

## Reference Solution
```python
def swap_variables(a, b):
    """
    Swaps the values of two variables without using a temporary variable.

    Args:
        a (any): The first variable.
        b (any): The second variable.

    Returns:
        tuple: A tuple containing the swapped values.
    """
    # Using tuple packing and unpacking to swap values
    a, b = b, a
    return a, b

def main():
    # Example usage
    var1 = 5
    var2 = 10

    print("Before swap:")
    print("var1:", var1)
    print("var2:", var2)

    var1, var2 = swap_variables(var1, var2)

    print("\nAfter swap:")
    print("var1:", var1)
    print("var2:", var2)

if __name__ == "__main__":
    main()
```
