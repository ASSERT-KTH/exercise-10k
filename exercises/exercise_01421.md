## Exercise
Create a function that takes an array and an index as input and returns the element at that index. If the index is out of range, return a message indicating that the index is out of range.

## Reference Solution

def get_element(array, index):
    """
    This function returns the element at a specified index in an array.
    
    If the index is out of range, it returns a message indicating that the index is out of range.

    Parameters:
    array (list): The input array.
    index (int): The index of the element to be retrieved.

    Returns:
    any: The element at the specified index if it exists, otherwise a message.
    """

    # First, we check if the index is within the valid range of the array
    # We use the len() function to get the number of elements in the array
    # The valid range of indices for an array is from 0 to len(array) - 1
    if index < 0 or index >= len(array):
        
        # If the index is out of range, we return a message indicating this
        return "Index out of range"
    
    # If the index is within the valid range, we return the element at that index
    # We use the index to access the corresponding element in the array
    else:
        return array[index]
