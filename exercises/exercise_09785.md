## Exercise
Write a program that reads input from the console and processes it.

## Reference Solution
```python
def process_input(input_str):
    """
    Processes the input string by converting it to uppercase, reversing it, and counting the number of vowels.

    Args:
        input_str (str): The input string to process.

    Returns:
        tuple: A tuple containing the uppercase string, reversed string, and vowel count.
    """
    uppercase_str = input_str.upper()
    reversed_str = input_str[::-1]
    vowel_count = sum(1 for char in input_str.lower() if char in 'aeiou')

    return uppercase_str, reversed_str, vowel_count


def main():
    # Read input from the console
    input_str = input("Please enter a string: ")

    # Process the input
    uppercase_str, reversed_str, vowel_count = process_input(input_str)

    # Print the results
    print("Uppercase String:", uppercase_str)
    print("Reversed String:", reversed_str)
    print("Vowel Count:", vowel_count)


if __name__ == "__main__":
    main()
```
