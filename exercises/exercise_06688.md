## Exercise
Create a program that takes two numbers as input and returns the result of raising the first number to the power of the second.

## Reference Solution
```python
def power(base, exponent):
    result = 1
    for i in range(abs(exponent)):
        result *= base
    if exponent < 0:
        return 1.0 / result
    return result

print(power(2, 3))  # Output: 8
print(power(2, -3))  # Output: 0.125
```
