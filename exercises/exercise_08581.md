## Exercise
Write a function to remove a key-value pair from a dictionary

## Reference Solution
```python
def remove_key_value_pair(dictionary, key):
    """
    Removes a key-value pair from a dictionary.

    Args:
        dictionary (dict): The dictionary to remove the key-value pair from.
        key: The key of the key-value pair to remove.

    Returns:
        dict: The dictionary with the key-value pair removed.
    """
    if key in dictionary:
        del dictionary[key]
    return dictionary

# Example usage:
my_dict = {"name": "John", "age": 30, "city": "New York"}
print("Original dictionary:", my_dict)

key_to_remove = "age"
updated_dict = remove_key_value_pair(my_dict, key_to_remove)
print("Dictionary after removing the key-value pair:", updated_dict)
```
