## Exercise
Create a program that finds the maximum number in a list using a while loop.

## Reference Solution
```python
def find_max_number(num_list):
    """
    This function finds the maximum number in a list using a while loop.

    Args:
        num_list (list): A list of numbers.

    Returns:
        max_num (int): The maximum number in the list.
    """
    # Check if the list is empty
    if len(num_list) == 0:
        return None

    # Initialize max_num with the first element of the list
    max_num = num_list[0]
    # Initialize index to 1 (since we've already checked the first element)
    index = 1

    # Loop through the list until we've checked all elements
    while index < len(num_list):
        # If current element is greater than max_num, update max_num
        if num_list[index] > max_num:
            max_num = num_list[index]
        # Move to the next element
        index += 1

    return max_num


# Example usage:
numbers = [4, 2, 9, 6, 5, 1, 8, 3, 7]
max_number = find_max_number(numbers)
print("Maximum number:", max_number)
```
