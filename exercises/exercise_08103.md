## Exercise
Write a program that takes a string and a substring as input and returns the index of the first occurrence of the substring in the string.

## Reference Solution
```python
def find_substring(string, substring):
    index = 0
    while index < len(string):
        if string[index:index + len(substring)] == substring:
            return index
        index += 1
    return -1

def main():
    string = "Hello, World!"
    substring = "World"
    print(find_substring(string, substring))

if __name__ == "__main__":
    main()
```
