## Exercise
Write a program that takes an array of integers as input, sorts it in ascending order, and then removes all duplicate elements. The program should then print the resulting array.

## Reference Solution
```python
def remove_duplicates(nums):
    nums.sort()
    i = 0
    while i < len(nums) - 1:
        if nums[i] == nums[i + 1]:
            nums.pop(i + 1)
        else:
            i += 1
    print(nums)

def main():
    nums = [1, 2, 3, 2, 4, 5, 5, 6, 7, 8, 8, 9]
    remove_duplicates(nums)

if __name__ == "__main__":
    main()
```
