## Exercise
Return a list of all keys in a dictionary

## Reference Solution
```python
def get_dict_keys(input_dict):
    """
    Returns a list of all keys in a dictionary.
    
    Args:
        input_dict (dict): The input dictionary.
    
    Returns:
        list: A list of dictionary keys.
    """
    return list(input_dict.keys())

# Example usage:
my_dict = {"name": "John", "age": 30, "city": "New York"}
keys = get_dict_keys(my_dict)
print(keys)  # Output: ['name', 'age', 'city']
```
