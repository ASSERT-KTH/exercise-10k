## Exercise
Write a program that calculates the average of a list of numbers using a for loop.

## Reference Solution
```python
def calculate_average(numbers):
    total = 0
    for num in numbers:
        total += num
    return total / len(numbers)

numbers = [1, 2, 3, 4, 5]
average = calculate_average(numbers)
print("The average is:", average)
```
