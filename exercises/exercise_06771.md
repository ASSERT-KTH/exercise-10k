## Exercise
Create a program that simulates rolling two six-sided dice. The program should ask for the user's name, then prompt them to roll the dice. After rolling, it should display the result, including the total of the two dice and a message based on the total (e.g., 'Good roll!' if the total is 7 or 11, 'Better luck next time!' if the total is 2, 3, or 12, and 'Okay roll!' for any other total). The program should use input/output operations to interact with the user.

## Reference Solution
```python
import random

def roll_dice():
    """Simulate rolling two six-sided dice."""
    dice1 = random.randint(1, 6)
    dice2 = random.randint(1, 6)
    return dice1, dice2

def main():
    """Main function to interact with the user."""
    print("Welcome to the Dice Roll Simulator!")
    name = input("Please enter your name: ")
    print(f"Hello, {name}!")

    while True:
        print("\nPress 'r' to roll the dice, or 'q' to quit.")
        choice = input().lower()
        if choice == 'q':
            print("Thanks for playing!")
            break
        elif choice == 'r':
            dice1, dice2 = roll_dice()
            total = dice1 + dice2
            print(f"\nYou rolled a {dice1} and a {dice2} for a total of {total}.")

            if total in [7, 11]:
                print("Good roll!")
            elif total in [2, 3, 12]:
                print("Better luck next time!")
            else:
                print("Okay roll!")
        else:
            print("Invalid choice. Please try again.")

if __name__ == "__main__":
    main()
```
