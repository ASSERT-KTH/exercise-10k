## Exercise
Write a function that takes a list and a predicate function as input and returns two new lists: one containing the elements for which the predicate returns True, and another containing the elements for which the predicate returns False.

## Reference Solution
```python
def split_list(input_list, predicate):
    """
    This function splits a given list into two separate lists based on a provided predicate function.
    
    Args:
        input_list (list): The input list to be split.
        predicate (function): A function that takes one argument and returns a boolean value.
    
    Returns:
        tuple: A tuple containing two lists. The first list contains elements for which the predicate returns True,
               and the second list contains elements for which the predicate returns False.
    """

    # Initialize two empty lists to store the elements for which the predicate returns True and False respectively.
    true_list = []
    
    # Initialize another empty list to store the elements for which the predicate returns False.
    false_list = []

    # Iterate over each element in the input list.
    for element in input_list:
        # For each element, apply the predicate function.
        if predicate(element):
            # If the predicate returns True, append the element to the true_list.
            true_list.append(element)
        else:
            # If the predicate returns False, append the element to the false_list.
            false_list.append(element)

    # Return a tuple containing the true_list and the false_list.
    return true_list, false_list

# Example usage:
# Define a predicate function that checks if a number is even.
def is_even(num):
    return num % 2 == 0

# Create a list of numbers.
numbers = [1, 2, 3, 4, 5, 6]

# Split the list using the is_even predicate.
even_numbers, odd_numbers = split_list(numbers, is_even)

print("Even numbers:", even_numbers)
print("Odd numbers:", odd_numbers)
```
