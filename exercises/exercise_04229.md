## Exercise
Write a program to convert a number to scientific notation.

## Reference Solution
```python
def convert_to_scientific_notation(num):
    """
    Converts a number to scientific notation.

    Args:
        num (float): The number to convert.

    Returns:
        str: The number in scientific notation.
    """
    return "{:e}".format(num)

def main():
    # Example usage
    num = float(input("Enter a number: "))
    scientific_notation = convert_to_scientific_notation(num)
    print(f"The scientific notation of {num} is: {scientific_notation}")

if __name__ == "__main__":
    main()
```
