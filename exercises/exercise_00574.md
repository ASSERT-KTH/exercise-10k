## Exercise
Write a function that removes duplicate characters from a given string.

## Reference Solution
```python
def remove_duplicate_chars(input_str):
    """
    Removes duplicate characters from a given string, preserving the original order.

    Args:
        input_str (str): The input string.

    Returns:
        str: The string with duplicate characters removed.
    """
    seen = set()
    result = []

    for char in input_str:
        if char not in seen:
            result.append(char)
            seen.add(char)

    return ''.join(result)

# Example usage
if __name__ == "__main__":
    input_str = "hello world"
    print("Original string:", input_str)
    print("String without duplicates:", remove_duplicate_chars(input_str))
```
