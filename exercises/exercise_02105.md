## Exercise
Write a function to iterate over the key-value pairs of a dictionary and print each pair

## Reference Solution
```python
def print_dict_key_value_pairs(dictionary):
    """
    Iterate over the key-value pairs of a dictionary and print each pair.

    Args:
        dictionary (dict): The dictionary to iterate over.

    Returns:
        None
    """
    for key, value in dictionary.items():
        print(f"Key: {key}, Value: {value}")


# Example usage:
if __name__ == "__main__":
    example_dict = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }

    print("Dictionary Key-Value Pairs:")
    print_dict_key_value_pairs(example_dict)
```
