## Exercise
Write a function that takes an array and a target element as input and returns the index of the target element in the array, or -1 if the target element is not found.

## Reference Solution
```python
public class Solution {
    public int searchInsert(int[] nums, int target) {
        int n = nums.length;
        int i = 0;

        while (i < n) {
            if (nums[i] == target) {
                return i;
            } else if (nums[i] > target) {
                return i;
            }
            i++;
        }

        return n;
    }
}
```
