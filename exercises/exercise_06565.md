## Exercise
Create a dictionary and iterate over its key-value pairs to print out the values

## Reference Solution
```python
# Create a dictionary
person = {
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}

# Iterate over key-value pairs and print values
for key, value in person.items():
    print(f"Key: {key}, Value: {value}")

# Alternatively, if you only want to print the values
for value in person.values():
    print(value)
```
