## Exercise
Create a function that takes an array of integers as input and returns the maximum value. The function should iterate over the array to find the maximum value.

## Reference Solution
```python
def find_max_value(array):
    """
    This function takes an array of integers as input and returns the maximum value.
    
    Parameters:
    array (list): A list of integers.
    
    Returns:
    int: The maximum value in the array.
    """

    # First, we check if the array is empty. If it is, we return None because there is no maximum value in an empty array.
    if not array:
        return None

    # We initialize the maximum value with the first element of the array. This is our initial guess for the maximum value.
    max_value = array[0]
    
    # Now, we iterate over the array starting from the second element (index 1).
    for i in range(1, len(array)):
        # For each element, we check if it is greater than our current maximum value.
        if array[i] > max_value:
            # If it is, we update our maximum value.
            max_value = array[i]
    
    # After iterating over the entire array, our maximum value is the largest value in the array.
    return max_value

# Example usage:
array = [5, 2, 9, 1, 7]
print(find_max_value(array))  # Output: 9
```
