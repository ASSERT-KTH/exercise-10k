## Exercise
Write a program that extracts a substring from a given string. The substring should start at a specified index and have a specified length.

## Reference Solution
```python
def extract_substring(input_string, start_index, length):
    """
    Extracts a substring from the input string.

    Args:
        input_string (str): The original string from which to extract the substring.
        start_index (int): The index at which to start extracting the substring.
        length (int): The length of the substring to extract.

    Returns:
        str: The extracted substring.
    """
    if start_index < 0 or start_index >= len(input_string):
        return "Invalid start index"

    if length <= 0:
        return "Invalid length"

    if start_index + length > len(input_string):
        return "Substring exceeds string length"

    return input_string[start_index:(start_index + length)]

# Example usage:
print(extract_substring("Hello, World!", 7, 5))  # Output: "World"
```
