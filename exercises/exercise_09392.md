## Exercise
Create a program that calculates the exponential function of a given number.

## Reference Solution
```python
import math

def calculate_exponential(x):
    """
    Calculate the exponential function of a given number.

    Args:
        x (float): The input number.

    Returns:
        float: The exponential function of x.
    """
    return math.exp(x)

def main():
    num = float(input("Enter a number: "))
    result = calculate_exponential(num)
    print(f"The exponential function of {num} is {result}")

if __name__ == "__main__":
    main()
```
