## Exercise
Create a program that takes in a list of variables and their corresponding values, and then determines the data type of each variable. The variables can be of type integer, float, or string. The program should print out the variable name and its corresponding data type.

## Reference Solution
```python
def determine_data_type(variables):
    for variable, value in variables.items():
        if isinstance(value, int):
            print(f"{variable}: integer")
        elif isinstance(value, float):
            print(f"{variable}: float")
        elif isinstance(value, str):
            print(f"{variable}: string")
        else:
            print(f"{variable}: unknown data type")

variables = {
    "var1": 10,
    "var2": 20.5,
    "var3": "hello",
    "var4": 30,
    "var5": 40.2,
    "var6": "world"
}

determine_data_type(variables)
```
