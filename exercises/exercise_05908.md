## Exercise
Write a function that takes a string, a substring, and a replacement string as input and returns the string with all occurrences of the substring replaced

## Reference Solution
```python
def replace_substring(input_string: str, substring: str, replacement: str) -> str:
    """
    Replaces all occurrences of a substring with a replacement string.

    Args:
        input_string (str): The input string.
        substring (str): The substring to replace.
        replacement (str): The replacement string.

    Returns:
        str: The input string with all occurrences of the substring replaced.
    """
    return input_string.replace(substring, replacement)

# Example usage:
if __name__ == "__main__":
    input_str = "Hello, world. Hello again."
    substring_to_replace = "Hello"
    replacement_str = "Hi"

    result = replace_substring(input_str, substring_to_replace, replacement_str)
    print(result)  # Output: "Hi, world. Hi again."
```
