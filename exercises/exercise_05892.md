## Exercise
Write a function that takes a 2D array as input and returns the transpose of the array.

## Reference Solution
```python
def transpose_2d_array(a):
    """
    This function takes a 2D array as input and returns its transpose.
    
    Parameters:
    a (list of lists): The input 2D array.
    
    Returns:
    list of lists: The transpose of the input array.
    """
    # Initialize an empty list to store the rows of the transposed array
    transposed_array = []
    
    # Calculate the number of rows and columns in the input array
    n_rows = len(a)
    n_cols = len(a[0])
    
    # Iterate over the columns of the input array
    for col in range(n_cols):
        # Initialize an empty list to store the elements of the current column
        new_row = []
        
        # Iterate over the rows of the input array
        for row in range(n_rows):
            # Append the element at the current column and row to the new row
            new_row.append(a[row][col])
        
        # Append the new row to the transposed array
        transposed_array.append(new_row)
    
    # Return the transposed array
    return transposed_array

# Example usage
a = [[1, 2, 3], [4, 5, 6]]
print(transpose_2d_array(a))  # Output: [[1, 4], [2, 5], [3, 6]]
```
