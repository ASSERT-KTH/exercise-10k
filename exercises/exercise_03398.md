## Exercise
Write a function to find the union of two lists.

## Reference Solution
```python
def union_of_lists(list1, list2):
    """
    Returns a new list containing all unique elements from both input lists.

    Args:
        list1 (list): The first input list.
        list2 (list): The second input list.

    Returns:
        list: A new list containing the union of both input lists.
    """
    return list(set(list1 + list2))

# Example usage:
if __name__ == "__main__":
    list1 = [1, 2, 3, 4]
    list2 = [3, 4, 5, 6]
    union_list = union_of_lists(list1, list2)
    print("Union List:", union_list)
```
