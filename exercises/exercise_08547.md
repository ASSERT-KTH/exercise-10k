## Exercise
Create a dictionary with 5 key-value pairs and filter out key-value pairs based on a condition.

## Reference Solution
```python
# Create a dictionary with 5 key-value pairs
data = {
    "John": 25,
    "Alice": 30,
    "Bob": 20,
    "Charlie": 35,
    "David": 15
}

# Function to filter key-value pairs
def filter_dict(data, min_age):
    """
    Filter key-value pairs in a dictionary based on a minimum age condition.

    Args:
    - data (dict): Dictionary containing names and ages.
    - min_age (int): Minimum age to filter by.

    Returns:
    - dict: Filtered dictionary.
    """
    return {name: age for name, age in data.items() if age >= min_age}

# Filter key-value pairs with age >= 25
filtered_data = filter_dict(data, 25)

# Print the original and filtered dictionaries
print("Original Dictionary:")
print(data)
print("\nFiltered Dictionary (Age >= 25):")
print(filtered_data)
```
