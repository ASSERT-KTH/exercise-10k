## Exercise
Write a function that partially applies another function, fixing some of its arguments to specific values.

## Reference Solution
```python
from functools import partial

def partial_application(func, *args, **kwargs):
    """
    Creates a new function with some arguments already applied.

    Args:
        func (function): The original function.
        *args: Positional arguments to be applied.
        **kwargs: Keyword arguments to be applied.

    Returns:
        function: A new function with the given arguments already applied.
    """

    # We use the partial function from the functools module, which does exactly what we need.
    # It returns a new function with the given arguments already applied.
    return partial(func, *args, **kwargs)


# Example usage:

def add(a, b, c):
    # A simple function that adds three numbers.
    return a + b + c

# Create a new function that adds 1 and 2 to its first argument.
add_1_and_2 = partial_application(add, 1, 2)

# Use the new function.
print(add_1_and_2(3))  # Outputs: 6
```
