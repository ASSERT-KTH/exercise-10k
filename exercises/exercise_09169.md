## Exercise
Create a string and classify each character as a letter, digit, or special character, then print out the classified characters.

## Reference Solution
```python
def classify_characters(input_string):
    """
    Classify each character in the input string as a letter, digit, or special character.

    Args:
        input_string (str): The string to be classified.

    Returns:
        None
    """
    for char in input_string:
        if char.isalpha():
            print(f"{char} is a letter.")
        elif char.isdigit():
            print(f"{char} is a digit.")
        else:
            print(f"{char} is a special character.")

def main():
    # Create a sample string
    sample_string = "Hello, World! 123"

    print("Character Classification:")
    print("---------------------------")
    classify_characters(sample_string)

if __name__ == "__main__":
    main()
```
