## Exercise
Write a function to use the popitem() method to remove and return an arbitrary key-value pair from a dictionary.

## Reference Solution
```python
def remove_arbitrary_pair(dictionary):
    """
    Removes and returns an arbitrary key-value pair from a dictionary.

    Args:
        dictionary (dict): The dictionary from which to remove a pair.

    Returns:
        tuple: The removed key-value pair.

    Raises:
        KeyError: If the dictionary is empty.
    """
    try:
        # Use the popitem() method to remove and return an arbitrary key-value pair
        pair = dictionary.popitem()
        return pair
    except KeyError:
        print("The dictionary is empty.")
        return None

# Example usage:
my_dict = {"apple": 1, "banana": 2, "cherry": 3}

print("Original dictionary:", my_dict)
removed_pair = remove_arbitrary_pair(my_dict)
print("Removed pair:", removed_pair)
print("Dictionary after removal:", my_dict)
```
