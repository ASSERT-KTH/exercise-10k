## Exercise
Create a function that uses currying to create a new function with one argument

## Reference Solution
```python
def curry_add(x):
    """
    A function that uses currying to create a new function with one argument.
    The new function adds x to its input.

    Args:
        x (int): The number to add.

    Returns:
        function: A new function that takes one argument and adds x to it.
    """
    def add(y):
        """
        A function that adds x to its input.

        Args:
            y (int): The number to add to x.

        Returns:
            int: The sum of x and y.
        """
        return x + y
    return add

# Example usage:
add_five = curry_add(5)
print(add_five(10))  # Outputs: 15

add_ten = curry_add(10)
print(add_ten(20))  # Outputs: 30
```
