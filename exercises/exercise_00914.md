## Exercise
Write a program that combines two lists of variables into a single list.

## Reference Solution
```python
def combine_lists(list1, list2):
    """
    Combines two lists into a single list.

    Args:
        list1 (list): The first list to combine.
        list2 (list): The second list to combine.

    Returns:
        list: A new list containing all elements from list1 and list2.
    """
    return list1 + list2


# Example usage:
list1 = [1, 2, 3, 4, 5]
list2 = ['a', 'b', 'c', 'd', 'e']

combined_list = combine_lists(list1, list2)

print("List 1:", list1)
print("List 2:", list2)
print("Combined List:", combined_list)
```
