## Exercise
Create a program that asks the user for a number and then checks if it is a palindrome (i.e., it reads the same forwards and backwards). Print out a message indicating whether the number is a palindrome or not

## Reference Solution
```python
def is_palindrome(num):
    """Checks if a number is a palindrome."""
    return str(num) == str(num)[::-1]

def main():
    # Ask the user for a number
    num = input("Enter a number: ")

    # Check if the input is a valid integer
    try:
        num = int(num)
    except ValueError:
        print("Invalid input. Please enter a valid integer.")
        return

    # Check if the number is a palindrome
    if is_palindrome(num):
        print(f"{num} is a palindrome.")
    else:
        print(f"{num} is not a palindrome.")

if __name__ == "__main__":
    main()
```
