## Exercise
Write a function that takes a list of strings and a delimiter as input and returns the joined string.

## Reference Solution
```python
def join_strings(strings, delimiter):
    """
    Join a list of strings with a specified delimiter.

    Args:
        strings (list): A list of strings to be joined.
        delimiter (str): The delimiter to join the strings with.

    Returns:
        str: The joined string.
    """
    return delimiter.join(strings)

# Example usage
if __name__ == "__main__":
    strings = ["Hello", "World", "This", "Is", "A", "Test"]
    delimiter = ", "
    print(join_strings(strings, delimiter))
```
