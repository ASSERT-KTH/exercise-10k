## Exercise
Write a program that asks the user for their favorite color and then prints a message saying they like that color.

## Reference Solution
```python
def get_favorite_color():
    """Asks the user for their favorite color"""
    favorite_color = input("What is your favorite color? ")
    return favorite_color

def print_color_preference(favorite_color):
    """Prints a message saying the user likes their favorite color"""
    print(f"You like {favorite_color}!")

def main():
    favorite_color = get_favorite_color()
    print_color_preference(favorite_color)

if __name__ == "__main__":
    main()
```
