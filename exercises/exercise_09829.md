## Exercise
Write a program that takes a string as input, checks if it contains any digits, and if not, reverses the string. The program should print 'Invalid input' if the string contains digits and the reversed string otherwise.

## Reference Solution
```python
def check(s):
    for i in range(len(s)):
        if s[i].isdigit():
            print("Invalid input")
            return
    print(s[::-1])

s = input()
check(s)
```
