## Exercise
Write a program that takes a user's name, age, and location as input and then prints out a greeting message with the provided information. The greeting message should include the user's name, age, and location in the following format: 'Hello, my name is {name}, I am {age} years old, and I am from {location}.',

## Reference Solution
```python
# Read user's name
name = input("What is your name? ")

# Read user's age
age = int(input("How old are you? "))

# Read user's location
location = input("Where are you from? ")

# Print greeting message
print("Hello, my name is", name, "I am", age, "years old, and I am from", location)
```
