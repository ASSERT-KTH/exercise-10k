## Exercise
Sort a dictionary by its values.

## Reference Solution
```python
def sort_dict_by_values(dictionary):
    """
    Sort a dictionary by its values.
    
    Args:
    dictionary (dict): The dictionary to sort.
    
    Returns:
    dict: A new dictionary sorted by its values.
    """
    return dict(sorted(dictionary.items(), key=lambda item: item[1]))

# Example usage:
my_dict = {'apple': 5, 'banana': 1, 'cherry': 3, 'date': 2, 'elderberry': 4}

sorted_dict = sort_dict_by_values(my_dict)
print(sorted_dict)
```
