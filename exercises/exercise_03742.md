## Exercise
Create a program to find the maximum number in a list of integers using a basic algorithm

## Reference Solution
```python
def find_max_number(num_list):
    """
    Finds the maximum number in a list of integers.

    Args:
        num_list (list): A list of integers.

    Returns:
        int: The maximum number in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not num_list:
        raise ValueError("Input list is empty")

    max_num = num_list[0]
    for num in num_list[1:]:
        if num > max_num:
            max_num = num

    return max_num


def main():
    # Example usage
    numbers = [12, 45, 7, 23, 56, 89, 34]
    max_number = find_max_number(numbers)
    print("Maximum number in the list:", max_number)


if __name__ == "__main__":
    main()
```
