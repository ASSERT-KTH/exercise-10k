## Exercise
Create a function that takes an array and a target value as input, then returns the indices of all occurrences of the target value in the array. If the target value is not found, return an empty list.

## Reference Solution
```python
def find_indices(array, target):
    """
    This function takes an array and a target value as input, 
    then returns the indices of all occurrences of the target value in the array.
    
    If the target value is not found, return an empty list.

    Parameters:
    array (list): The input list to search for the target value.
    target: The value to search for in the array.

    Returns:
    list: A list of indices where the target value is found in the array.
    """

    # Initialize an empty list to store the indices of the target value
    indices = []

    # Iterate over the array with the enumerate function to get both the index and value
    for i, value in enumerate(array):
        
        # Check if the current value is equal to the target value
        if value == target:
            
            # If it is, append the index to the indices list
            indices.append(i)

    # Return the list of indices
    return indices

# Example usage:
array = [1, 2, 3, 2, 4, 2, 5]
target = 2
print(find_indices(array, target))  # Output: [1, 3, 5]
```
