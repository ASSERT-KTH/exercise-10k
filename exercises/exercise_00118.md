## Exercise
Write a program to delete duplicate elements from an array.

## Reference Solution
```python
def delete_duplicates(array):
    """
    Deletes duplicate elements from an array.

    Args:
        array (list): The input array.

    Returns:
        list: The array with duplicate elements deleted.
    """
    return list(set(array))

# Example usage:
array = [1, 2, 2, 3, 4, 4, 5, 6, 6]
print("Original array:", array)
print("Array without duplicates:", delete_duplicates(array))
```

def delete_duplicates_ordered(array):
    """
    Deletes duplicate elements from an array while preserving order.

    Args:
        array (list): The input array.

    Returns:
        list: The array with duplicate elements deleted.
    """
    seen = set()
    return [x for x in array if not (x in seen or seen.add(x))]

# Example usage:
array = [1, 2, 2, 3, 4, 4, 5, 6, 6]
print("Original array:", array)
print("Array without duplicates:", delete_duplicates_ordered(array))
