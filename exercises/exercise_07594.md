## Exercise
Write a program that checks the scope of a given variable.

## Reference Solution
```python
# Global scope
x = 10

def outer_function():
    # Enclosing scope
    y = 20

    def inner_function():
        # Local scope
        z = 30
        print("Inside inner_function:")
        print("x (global):", x)
        print("y (enclosing):", y)
        print("z (local):", z)

    inner_function()
    print("Inside outer_function:")
    print("x (global):", x)
    print("y (enclosing):", y)
    # print("z (local):", z)  # This would raise an error

outer_function()
print("Global scope:")
print("x (global):", x)
# print("y (enclosing):", y)  # This would raise an error
# print("z (local):", z)  # This would raise an error
```
