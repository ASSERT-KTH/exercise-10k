## Exercise
Write a program to calculate the diameter of a sphere given its radius.

## Reference Solution
```python
def calculate_diameter(radius):
    """
    Calculate the diameter of a sphere given its radius.

    Args:
        radius (float): The radius of the sphere.

    Returns:
        float: The diameter of the sphere.
    """
    if radius < 0:
        raise ValueError("Radius cannot be negative")
    diameter = 2 * radius
    return diameter

def main():
    # Example usage
    radius = float(input("Enter the radius of the sphere: "))
    try:
        diameter = calculate_diameter(radius)
        print(f"The diameter of the sphere is: {diameter}")
    except ValueError as e:
        print(f"Error: {e}")

if __name__ == "__main__":
    main()
```
