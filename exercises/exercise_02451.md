## Exercise
Write a program that takes an array and a rotation amount as input and returns a new array containing the same elements rotated by the specified amount.

## Reference Solution
```python
def rotate_array(arr, num_positions):
    new_arr = arr[num_positions:] + arr[:num_positions]
    return new_arr

arr = [1, 2, 3, 4, 5]
num_positions = 2
print(rotate_array(arr, num_positions))  # Output: [3, 4, 5, 1, 2]
```
