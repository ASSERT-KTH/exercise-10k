## Exercise
Write a program to demonstrate the concept of null and undefined variables.

## Reference Solution
```python
// Declaring a variable without assigning a value to it
let null_variable = null;
let undefined_variable;

// Printing the null variable
console.log("Null Variable: ", null_variable);
console.log("Undefined Variable: ", undefined_variable);

// Checking if the variable is null or undefined
console.log("Is null_variable null? ", null_variable === null);
console.log("Is undefined_variable undefined? ", undefined_variable === undefined);
```
