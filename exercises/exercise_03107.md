## Exercise
Create a dictionary with 5 key-value pairs. Map each key to a new key using a given mapping.

## Reference Solution
```python
def map_keys(dictionary, key_mapping):
    """
    Maps each key in the dictionary to a new key using the given mapping.

    Args:
    dictionary (dict): The original dictionary.
    key_mapping (dict): A dictionary containing the key mappings.

    Returns:
    dict: A new dictionary with the keys mapped.
    """
    mapped_dictionary = {key_mapping[key]: value for key, value in dictionary.items() if key in key_mapping}
    return mapped_dictionary


# Example usage:
if __name__ == "__main__":
    # Create a dictionary with 5 key-value pairs
    original_dict = {
        "a": 1,
        "b": 2,
        "c": 3,
        "d": 4,
        "e": 5
    }

    # Define the key mapping
    key_mapping = {
        "a": "x",
        "b": "y",
        "c": "z",
        "d": "p",
        "e": "q"
    }

    # Map the keys
    mapped_dict = map_keys(original_dict, key_mapping)

    print("Original Dictionary:")
    print(original_dict)
    print("\nMapped Dictionary:")
    print(mapped_dict)
```
