## Exercise
Iterate over the key-value pairs in a dictionary and print each pair

## Reference Solution
```python
def print_dictionary_key_value_pairs(dictionary):
    """
    Iterate over the key-value pairs in a dictionary and print each pair.

    Args:
        dictionary (dict): The dictionary to iterate over.
    """
    for key, value in dictionary.items():
        print(f"Key: {key}, Value: {value}")


# Example usage
example_dictionary = {
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}

print_dictionary_key_value_pairs(example_dictionary)
```
