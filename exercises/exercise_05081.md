## Exercise
Write a program that takes a name as input and prints out a greeting message with the name.

## Reference Solution
```python
def print_greeting(name):
    """Prints out a greeting message with the given name."""
    print(f"Hello, {name}!")

def main():
    """Gets a name from the user and prints out a greeting message."""
    name = input("Please enter your name: ")
    print_greeting(name)

if __name__ == "__main__":
    main()
```
