## Exercise

## Reference Solution
```python
# Define integer variables
a = 10
b = 3

# Addition
addition_result = a + b
print(f"Addition: {a} + {b} = {addition_result}")

# Subtraction
subtraction_result = a - b
print(f"Subtraction: {a} - {b} = {subtraction_result}")

# Multiplication
multiplication_result = a * b
print(f"Multiplication: {a} * {b} = {multiplication_result}")

# Division (integer division)
division_result = a // b
print(f"Division (integer): {a} // {b} = {division_result}")

# Division (floating point division)
floating_point_division_result = a / b
print(f"Division (floating point): {a} / {b} = {floating_point_division_result}")

# Modulus (remainder)
modulus_result = a % b
print(f"Modulus: {a} % {b} = {modulus_result}")
```
