## Exercise
Retrieve a value from a dictionary using the get method

## Reference Solution
```python
def retrieve_value(dictionary, key, default=None):
    """
    Retrieves a value from a dictionary using the get() method.
    
    Args:
        dictionary (dict): The dictionary to retrieve the value from.
        key: The key to look up in the dictionary.
        default: The default value to return if the key is not found.
    
    Returns:
        The value associated with the key if it exists, otherwise the default value.
    """
    return dictionary.get(key, default)

# Example usage:
my_dict = {"name": "John", "age": 30}

# Retrieve a value that exists in the dictionary
print(retrieve_value(my_dict, "name"))  # Output: John

# Retrieve a value that does not exist in the dictionary
print(retrieve_value(my_dict, "city"))  # Output: None

# Retrieve a value that does not exist in the dictionary with a custom default value
print(retrieve_value(my_dict, "city", "New York"))  # Output: New York
```
