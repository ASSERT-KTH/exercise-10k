## Exercise
Create a function that takes a variable number of arguments and returns their sum

## Reference Solution
```python
def sum_arguments(*args):
    """
    This function takes a variable number of arguments and returns their sum.

    Args:
        *args: A variable number of arguments

    Returns:
        The sum of all arguments

    Raises:
        TypeError: If any argument is not a number
    """
    total = 0
    for arg in args:
        if not isinstance(arg, (int, float)):
            raise TypeError("All arguments must be numbers")
        total += arg
    return total

# Example usage:
def main():
    print(sum_arguments(1, 2, 3, 4, 5))  # Output: 15
    print(sum_arguments(10.5, 20.7))  # Output: 31.2
    try:
        print(sum_arguments(1, 'a', 3))
    except TypeError as e:
        print(e)

if __name__ == "__main__":
    main()
```
