## Exercise
Write a program that takes a string input from the user, checks if it is a palindrome, and if not, reverses the string and then checks again. The program should print out whether the original string is a palindrome or not, and if the reversed string is a palindrome or not.

## Reference Solution
```python
# Get user input
inp = input()

# Define the palindrome function
def palindrome(word):
    return word == word[::-1]

# Print whether the original string is a palindrome
print(f"Is '{inp}' a palindrome? {palindrome(inp)}")

# Reverse the string
inp = inp[::-1]

# Print whether the reversed string is a palindrome
print(f"Is '{inp}' a palindrome? {palindrome(inp)}")
```
