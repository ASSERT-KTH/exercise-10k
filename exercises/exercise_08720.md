## Exercise
Ask the user for a number and print out a pattern of that size.

## Reference Solution
```python
def print_pattern(size):
    """
    Prints a square pattern of asterisks (*) of the given size.
    
    Args:
    size (int): The size of the pattern.
    """
    for i in range(size):
        for j in range(size):
            print("*", end=" ")
        print()

def main():
    # Ask the user for a number
    while True:
        try:
            size = int(input("Enter a number: "))
            if size <= 0:
                print("Please enter a positive integer.")
            else:
                break
        except ValueError:
            print("Invalid input. Please enter a positive integer.")

    # Print the pattern
    print_pattern(size)

if __name__ == "__main__":
    main()
```
