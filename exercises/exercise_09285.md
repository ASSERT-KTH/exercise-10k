## Exercise
Create a program that uses functions to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The program should take in two numbers and an operator as input and then call the corresponding function to perform the operation.

## Reference Solution
```python
# Define the functions for each operation
def add(num1, num2):
    return num1 + num2

def sub(num1, num2):
    return num1 - num2

def mul(num1, num2):
    return num1 * num2

def div(num1, num2):
    return num1 / num2

# Main program
num1 = float(input("Enter the first number: "))
num2 = float(input("Enter the second number: "))
opr = input("Enter the operator (+, -, *, /): ")

if opr == "+":
    print("Result:", add(num1, num2))
elif opr == "-":
    print("Result:", sub(num1, num2))
elif opr == "*":
    print("Result:", mul(num1, num2))
elif opr == "/":
    if num2 != 0:
        print("Result:", div(num1, num2))
    else:
        print("Error: Division by zero is not allowed.")
else:
    print("Error: Invalid operator. Please enter +, -, *, or /.")
```
