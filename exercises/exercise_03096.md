## Exercise
Implement a function to find the first duplicate element in a given list.

## Reference Solution
```python
def find_first_duplicate(lst):
    """
    Find the first duplicate element in a given list.

    Args:
        lst (list): The input list.

    Returns:
        The first duplicate element, or None if no duplicates are found.
    """
    seen = set()
    for element in lst:
        if element in seen:
            return element
        seen.add(element)
    return None

# Example usage
if __name__ == "__main__":
    lst = [2, 1, 3, 2, 4, 5, 6]
    duplicate = find_first_duplicate(lst)
    if duplicate is not None:
        print(f"The first duplicate element is: {duplicate}")
    else:
        print("No duplicates found.")
```
