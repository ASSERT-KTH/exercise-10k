## Exercise
Create a program that asks for a user's name, age, and location, then prints out a formatted string containing this information.

## Reference Solution
```python
# Define a function to get user information
def get_user_info():
    # Ask for user's name, age, and location
    name = input("Please enter your name: ")
    age = input("Please enter your age: ")
    location = input("Please enter your location: ")

    # Return the user's information
    return name, age, location

# Define a function to print user information
def print_user_info(name, age, location):
    # Print out a formatted string containing the user's information
    print(f"Hello, my name is {name}. I am {age} years old and I am from {location}.")

# Main function
def main():
    # Get user's information
    name, age, location = get_user_info()

    # Print user's information
    print_user_info(name, age, location)

# Run the main function
if __name__ == "__main__":
    main()
```
