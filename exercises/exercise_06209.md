## Exercise
Write a function that takes an array as input, and returns the maximum element in the array.

## Reference Solution
```python
def find_max_element(array):
    """
    This function finds the maximum element in a given array.

    Args:
        array (list): A list of numbers.

    Returns:
        The maximum element in the array.

    Raises:
        ValueError: If the input array is empty.
    """
    if not array:
        raise ValueError("Input array is empty")

    return max(array)

# Example usage
if __name__ == "__main__":
    array = [1, 2, 3, 4, 5]
    max_element = find_max_element(array)
    print("Maximum element in the array:", max_element)
```
