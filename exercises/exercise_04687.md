## Exercise
Write a function that removes all vowels from a given string.

## Reference Solution
```python
def remove_vowels(input_string):
    """
    Removes all vowels from a given string.

    Args:
        input_string (str): The string from which to remove vowels.

    Returns:
        str: The input string with all vowels removed.
    """
    # Define a string of vowels
    vowels = 'aeiouAEIOU'

    # Use a list comprehension to create a new string without vowels
    return ''.join([char for char in input_string if char not in vowels])

# Example usage
def main():
    input_str = "Hello, World!"
    print(f"Original string: {input_str}")
    print(f"String without vowels: {remove_vowels(input_str)}")

if __name__ == "__main__":
    main()
```
