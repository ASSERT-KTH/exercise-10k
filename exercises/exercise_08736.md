## Exercise
Serialize and deserialize variables to and from a string representation.

## Reference Solution
```python
import json

def serialize_variables(variables):
    """
    Serialize variables to a string representation.
    
    Args:
        variables (dict): A dictionary of variables to be serialized.
        
    Returns:
        str: A string representation of the variables.
    """
    # Use json.dumps to convert the dictionary into a string
    serialized_variables = json.dumps(variables)
    return serialized_variables

def deserialize_variables(serialized_variables):
    """
    Deserialize variables from a string representation.
    
    Args:
        serialized_variables (str): A string representation of the variables.
        
    Returns:
        dict: A dictionary of variables deserialized from the string.
    """
    # Use json.loads to convert the string back into a dictionary
    variables = json.loads(serialized_variables)
    return variables

# Example usage:
variables = {'name': 'John', 'age': 30}
serialized_variables = serialize_variables(variables)
print(f"Serialized Variables: {serialized_variables}")

deserialized_variables = deserialize_variables(serialized_variables)
print(f"Deserialized Variables: {deserialized_variables}")
```
