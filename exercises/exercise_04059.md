## Exercise
Write a function that takes two dictionaries as input and loops through all the key-value pairs in both dictionaries, printing out each key and value.

## Reference Solution
```python
def print_key_value_pairs(dict1, dict2):
    """
    Prints out each key and value from two input dictionaries.

    Args:
        dict1 (dict): The first dictionary.
        dict2 (dict): The second dictionary.
    """

    # Print key-value pairs from the first dictionary
    print("Dictionary 1:")
    for key, value in dict1.items():
        print(f"Key: {key}, Value: {value}")

    # Print key-value pairs from the second dictionary
    print("\nDictionary 2:")
    for key, value in dict2.items():
        print(f"Key: {key}, Value: {value}")


# Example usage
if __name__ == "__main__":
    dict1 = {"name": "John", "age": 30}
    dict2 = {"city": "New York", "country": "USA"}

    print_key_value_pairs(dict1, dict2)
```
