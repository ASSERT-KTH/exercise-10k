## Exercise
Write a program to find the maximum number in an array of integers.

## Reference Solution
```python
def find_max(arr):
    """
    This function finds the maximum number in an array of integers.

    Args:
        arr (list): A list of integers.

    Returns:
        int: The maximum number in the array.
    """
    if not arr:
        raise ValueError("Array is empty")

    max_num = arr[0]
    for num in arr:
        if num > max_num:
            max_num = num

    return max_num

# Test the function
arr = [4, 2, 9, 6, 5, 1]
print("Maximum number in the array:", find_max(arr))
```
