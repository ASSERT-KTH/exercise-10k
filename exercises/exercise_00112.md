## Exercise
Write a program to rotate an array by a given number of positions.

## Reference Solution
```python
def rotate_array(arr, num_positions, direction):
    """
    Rotate an array by a given number of positions.

    Args:
        arr (list): The input array.
        num_positions (int): The number of positions to rotate.
        direction (str): The direction of rotation. Can be 'clockwise' or 'counterclockwise'.

    Returns:
        list: The rotated array.
    """
    num_positions = num_positions % len(arr)
    if direction == 'clockwise':
        return arr[-num_positions:] + arr[:-num_positions]
    elif direction == 'counterclockwise':
        return arr[num_positions:] + arr[:num_positions]
    else:
        raise ValueError("Invalid direction. Must be 'clockwise' or 'counterclockwise'.")

# Example usage
arr = [1, 2, 3, 4, 5]
num_positions = 2
direction = 'clockwise'

rotated_arr = rotate_array(arr, num_positions, direction)
print("Original array:", arr)
print("Rotated array:", rotated_arr)
```
