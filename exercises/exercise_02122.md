## Exercise
Write a function that takes a list and an index as input and returns the element at that index.

## Reference Solution
```python
def get_element_at_index(input_list, index):
    """
    This function retrieves an element from a list at a specified index.

    Args:
        input_list (list): The list from which to retrieve the element.
        index (int): The index of the element to be retrieved.

    Returns:
        The element at the specified index in the list, or None if the index is out of range.
    """

    # First, we check if the input is a list. If not, we raise a TypeError.
    if not isinstance(input_list, list):
        raise TypeError("Input must be a list.")

    # Then, we check if the index is an integer. If not, we raise a TypeError.
    if not isinstance(index, int):
        raise TypeError("Index must be an integer.")

    # Next, we check if the index is within the valid range for the list. 
    # If the index is out of range, we return None.
    if index < 0 or index >= len(input_list):
        return None

    # Finally, we return the element at the specified index.
    # We use the index to access the element in the list.
    return input_list[index]
```
