## Exercise
Create a function that takes in a dictionary and returns a new dictionary containing the same key-value pairs, but with the keys sorted in ascending order

## Reference Solution
```python
def sort_dict(input_dict):
    """
    Returns a new dictionary with the same key-value pairs as the input dictionary,
    but with the keys sorted in ascending order.

    Args:
        input_dict (dict): The input dictionary to be sorted.

    Returns:
        dict: A new dictionary with the sorted key-value pairs.
    """
    return dict(sorted(input_dict.items()))

# Example usage:
if __name__ == "__main__":
    input_dict = {"orange": 1, "apple": 2, "banana": 3}
    sorted_dict = sort_dict(input_dict)
    print(sorted_dict)
```
