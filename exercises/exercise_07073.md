## Exercise
Read a character from the user and print its ASCII value

## Reference Solution
```python
def print_ascii_value():
    """
    Reads a character from the user and prints its ASCII value.
    """
    char = input("Enter a character: ")
    
    # Check if user entered a single character
    if len(char) != 1:
        print("Error: Please enter a single character.")
        return
    
    ascii_value = ord(char)
    print(f"The ASCII value of '{char}' is {ascii_value}.")

def main():
    print_ascii_value()

if __name__ == "__main__":
    main()
```
