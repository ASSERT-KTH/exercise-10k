## Exercise
Create a function that takes a list of numbers and returns their sum, handling any exceptions that may occur

## Reference Solution
```python
def sum_numbers(numbers):
    """
    Calculate the sum of a list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        float: The sum of the numbers in the list.

    Raises:
        TypeError: If the input is not a list.
        ValueError: If the list contains non-numeric values.
    """

    # Check if input is a list
    if not isinstance(numbers, list):
        raise TypeError("Input must be a list.")

    # Initialize sum to 0
    total = 0

    # Iterate over each number in the list
    for num in numbers:
        # Check if the number is numeric
        if not isinstance(num, (int, float)):
            raise ValueError("List contains non-numeric values.")

        # Add the number to the total
        total += num

    # Return the total
    return total


# Example usage:
def main():
    try:
        numbers = [1, 2, 3, 4, 5]
        print(f"Sum: {sum_numbers(numbers)}")
    except (TypeError, ValueError) as e:
        print(f"Error: {e}")


if __name__ == "__main__":
    main()
```
