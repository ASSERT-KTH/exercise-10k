## Exercise
Create a program that uses a while loop to simulate a game of ping pong, alternating between 'ping' and 'pong' until a given number of rounds is reached.

## Reference Solution
```python
# 1. Introduction
# This program simulates a game of ping pong, alternating between 'ping' and 'pong'
# until a given number of rounds is reached.

# 2. Variables
rounds = 10  # Number of rounds to simulate
i = 0  # Counter variable

# 3. While Loop
while i < rounds:
    # 4. Conditional Statement
    if i % 2 == 0:
        print("ping")
    else:
        print("pong")
    # 5. Increment Counter
    i += 1
```
