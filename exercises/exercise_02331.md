## Exercise
Write a program to print a list of squares of numbers from 1 to 10 to the screen.

## Reference Solution
```python
def print_squares(n):
    """
    Prints a list of squares of numbers from 1 to n.
    """
    squares = [i ** 2 for i in range(1, n + 1)]
    print(squares)

def main():
    n = 10
    print(f"Squares of numbers from 1 to {n}:")
    print_squares(n)

if __name__ == "__main__":
    main()
```
