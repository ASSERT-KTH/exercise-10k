## Exercise
Write a program that takes an array as input, and returns a dictionary containing the frequency of each element in the array.

## Reference Solution
```python
// Define the function
function freqDict(arr) {
  var freq = {};
  for (var i = 0; i < arr.length; i++) {
    if (freq[arr[i]]) {
      freq[arr[i]]++;
    } else {
      freq[arr[i]] = 1;
    }
  }
  return freq;
}

// Example usage
var arr = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4];
var output = freqDict(arr);
console.log(output); // Output: {1: 1, 2: 2, 3: 3, 4: 4}
```
