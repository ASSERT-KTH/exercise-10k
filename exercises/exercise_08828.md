## Exercise
Write a program that prints a square pattern of asterisks (*) using nested loops. The size of the square should be determined by the user's input. Then, print a diamond pattern below the square using the same size.

## Reference Solution
```python
def print_square(size):
    """
    Prints a square pattern of asterisks.

    Args:
    size (int): The size of the square.

    Returns:
    None
    """
    for _ in range(size):
        print("* " * size)


def print_diamond(size):
    """
    Prints a diamond pattern of asterisks.

    Args:
    size (int): The size of the diamond.

    Returns:
    None
    """
    # Print the top half of the diamond
    for i in range(size):
        print(" " * (size - i - 1) + "* " * (i + 1))

    # Print the bottom half of the diamond
    for i in range(size - 2, -1, -1):
        print(" " * (size - i - 1) + "* " * (i + 1))


def main():
    # Get the size of the square and diamond from the user
    while True:
        try:
            size = int(input("Enter the size of the square and diamond: "))
            if size <= 0:
                print("Please enter a positive integer.")
                continue
            break
        except ValueError:
            print("Invalid input. Please enter an integer.")

    # Print the square
    print("Square pattern:")
    print_square(size)

    # Print the diamond
    print("\nDiamond pattern:")
    print_diamond(size)


if __name__ == "__main__":
    main()
```
