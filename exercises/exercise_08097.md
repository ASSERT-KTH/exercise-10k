## Exercise
Write a program that takes a string as input and returns the string in lower case.

## Reference Solution
```python
def convert_to_lowercase(input_string: str) -> str:
    """
    Converts a given string to lowercase.

    Args:
        input_string (str): The string to be converted.

    Returns:
        str: The input string in lowercase.
    """
    return input_string.lower()

def main():
    # Example usage
    input_string = input("Enter a string: ")
    lowercase_string = convert_to_lowercase(input_string)
    print("Lowercase string:", lowercase_string)

if __name__ == "__main__":
    main()
```
