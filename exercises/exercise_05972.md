## Exercise
Write a program that takes a list and an index as input, and removes the element at the specified index.

## Reference Solution
```python
class Solution:
    def removeElement(self, nums: list[int], val: int) -> int:
        n = len(nums)
        i = 0
        while i < n:
            if nums[i] == val:
                nums.pop(i)
                n -= 1
            else:
                i += 1
        return n
```
