## Exercise
Write a program that takes a string as input and returns the reversed string. The program should handle strings of any length and should not use any built-in string reversal functions.

## Reference Solution
```python
#include <iostream>
#include <string>

int main() {
    std::string str;
    std::cout << "Enter a string: ";
    std::cin >> str;
    int len = str.length();
    for (int i = 0; i < len; i++) {
        std::cout << str[len - i - 1];
    }
    return 0;
}
```
