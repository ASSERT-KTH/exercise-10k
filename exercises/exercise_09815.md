## Exercise
Create a program that validates and performs arithmetic operations based on user input. The program should take in two numbers and an operator (+, -, *, /) as input, validate the input, and then perform the operation. If the input is invalid, the program should output an error message.

## Reference Solution
```python
#include <iostream>
#include <stdexcept>

void calculate(double a, double b, char op) {
    double result;

    switch (op) {
        case '+':
            result = a + b;
            break;
        case '-':
            result = a - b;
            break;
        case '*':
            result = a * b;
            break;
        case '/':
            if (b != 0) {
                result = a / b;
            } else {
                throw std::runtime_error("Division by zero!");
            }
            break;
        default:
            throw std::runtime_error("Invalid operator!");
    }

    std::cout << "Result: " << result << std::endl;
}

int main() {
    try {
        double a, b;
        char op;

        std::cout << "Enter first number: ";
        std::cin >> a;

        std::cout << "Enter operator (+, -, *, /): ";
        std::cin >> op;

        std::cout << "Enter second number: ";
        std::cin >> b;

        calculate(a, b, op);
    } catch (const std::exception& e) {
        std::cerr << "Error: " << e.what() << std::endl;
        return 1;
    }

    return 0;
}
```
