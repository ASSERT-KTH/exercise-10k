## Exercise
Write a program that finds the minimum value in a list of numbers.

## Reference Solution
```python
def find_min_value(num_list):
    """
    Finds the minimum value in a list of numbers.

    Args:
        num_list (list): A list of numbers.

    Returns:
        int: The minimum value in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not num_list:
        raise ValueError("Input list is empty")

    return min(num_list)


def main():
    # Example usage
    numbers = [10, 20, 3, 45, 6]
    try:
        min_value = find_min_value(numbers)
        print(f"Minimum value: {min_value}")
    except ValueError as e:
        print(e)


if __name__ == "__main__":
    main()
```
