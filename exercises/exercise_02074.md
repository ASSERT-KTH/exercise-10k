## Exercise
Write a program that generates all combinations of a given list of numbers.

## Reference Solution
```python
import itertools

def generate_combinations(numbers, combination_length):
    """
    Generate all combinations of a given list of numbers.

    Args:
        numbers (list): The list of numbers.
        combination_length (int): The length of each combination.

    Returns:
        list: A list of all combinations.
    """
    return list(itertools.combinations(numbers, combination_length))

def main():
    numbers = [1, 2, 3, 4, 5]
    combination_length = 3

    combinations = generate_combinations(numbers, combination_length)
    print("Combinations:")
    for i, combination in enumerate(combinations):
        print(f"Combination {i+1}: {combination}")

if __name__ == "__main__":
    main()
```
