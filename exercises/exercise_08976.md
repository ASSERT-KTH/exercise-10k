## Exercise
Write a function that takes a string and a substring as input and returns True if the string ends with the substring, False otherwise.

## Reference Solution
```python
def check_ending(string, substring):
    """
    This function checks if a given string ends with a specified substring.

    Args:
        string (str): The original string to be checked.
        substring (str): The substring to be searched at the end of the string.

    Returns:
        bool: True if the string ends with the substring, False otherwise.
    """

    # First, we need to check if both inputs are strings. If not, we raise a TypeError.
    if not isinstance(string, str) or not isinstance(substring, str):
        raise TypeError("Both inputs must be strings.")

    # Now, we can use Python's built-in str.endswith() method to check if the string ends with the substring.
    # This method returns True if the string ends with the substring, and False otherwise.
    return string.endswith(substring)
```
