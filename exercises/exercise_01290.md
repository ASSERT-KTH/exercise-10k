## Exercise
Write a function that returns a list of all keys in a dictionary.

## Reference Solution
```python
def get_dict_keys(dictionary):
    """
    Returns a list of all keys in a dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        list: A list of all keys in the dictionary.
    """

    # First, we check if the input is actually a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("Input must be a dictionary")

    # We use the built-in dictionary method keys() to get a view object that displays a list of all keys available in the dictionary
    keys_view = dictionary.keys()

    # We convert the view object to a list using the list() function
    # This is because the problem statement asks for a list, and the keys() method returns a view object
    keys_list = list(keys_view)

    # Finally, we return the list of keys
    return keys_list

# Example usage:
my_dict = {"a": 1, "b": 2, "c": 3}
print(get_dict_keys(my_dict))  # Output: ['a', 'b', 'c']
```
