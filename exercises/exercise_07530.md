## Exercise
Write a program that asks the user for their name and age, then prints out a greeting message with their name and tells them how old they will be in 5 years.

## Reference Solution
```python
# Ask the user for their name
name = input("What is your name? ")

# Ask the user for their age
age = int(input("How old are you? "))

# Print out a greeting message with their name
print(f"Hello, {name}!")

# Calculate and print how old they will be in 5 years
print(f"In 5 years, you will be {age + 5} years old.")
```
