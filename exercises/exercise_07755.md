## Exercise
Create a function that checks if a given string is a palindrome. The function should take a string as input and return True if the string is a palindrome, False otherwise. The function should be case-insensitive and ignore non-alphanumeric characters.

## Reference Solution
```python
def is_palindrome(s: str) -> bool:
    """
    Checks if a given string is a palindrome. The function is case-insensitive and ignores non-alphanumeric characters.

    Args:
        s (str): The input string to be checked.

    Returns:
        bool: True if the string is a palindrome, False otherwise.
    """

    # First, we convert the string to lower case to make the function case-insensitive
    s = s.lower()

    # Then, we filter out non-alphanumeric characters from the string
    # We use a list comprehension to create a new string with only alphanumeric characters
    s = ''.join(char for char in s if char.isalnum())

    # Now, we compare the filtered string with its reverse
    # If they are equal, the string is a palindrome
    # We use slicing to reverse the string, which is a common Python idiom
    return s == s[::-1]
```
