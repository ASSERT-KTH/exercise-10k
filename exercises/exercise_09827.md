## Exercise
Write a program that takes a user's name as input and prints out a greeting message with their name. The program should also ask for the user's age and print out a message indicating whether they are eligible to vote or not.

## Reference Solution
```python
# This is a comment explaining the purpose of the program
# This program takes a user's name and age as input and prints out a greeting message
# and a message indicating whether they are eligible to vote or not

# Take the user's name as input
name = input("Enter your name: ")

# Print out a greeting message with the user's name
print("Hello, " + name + "!")

# Take the user's age as input
age = int(input("Enter your age: "))

# Print out a message indicating whether the user is eligible to vote or not
if age >= 18:
    print("You are eligible to vote!")
else:
    print("You are not eligible to vote yet.")
```
