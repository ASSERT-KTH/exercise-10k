## Exercise
Write a program that takes a string input from the user and converts it into different data types (integer, float, boolean). The program should handle potential errors during the conversion process and print the converted value along with its data type. If the conversion is not possible, it should print an error message.

## Reference Solution
```python
def convert_to_int(value):
    try:
        converted_value = int(value)
        print(f"Converted to int: {converted_value}")
    except ValueError:
        print("Error: Cannot convert to int")

def convert_to_float(value):
    try:
        converted_value = float(value)
        print(f"Converted to float: {converted_value}")
    except ValueError:
        print("Error: Cannot convert to float")

def convert_to_bool(value):
    try:
        if value.lower() == "true":
            converted_value = True
        elif value.lower() == "false":
            converted_value = False
        else:
            print("Error: Cannot convert to bool")
            return
        print(f"Converted to bool: {converted_value}")
    except Exception as e:
        print(f"Error: {str(e)}")

def main():
    value = input("Enter a string: ")
    convert_to_int(value)
    convert_to_float(value)
    convert_to_bool(value)

if __name__ == "__main__":
    main()
```
