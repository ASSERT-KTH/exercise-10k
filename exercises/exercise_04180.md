## Exercise
Write a program that simulates a chatbot conversation with the user.

## Reference Solution
```python
# Import the required libraries
import random

# Define a dictionary of chatbot responses
responses = {
    'hello': ['Hi, how are you?', 'Hello!', 'Hey, what\'s up?'],
    'how are you': ['I\'m doing great, thanks!', 'I\'m good, thanks for asking!', 'I\'m just a chatbot, I don\'t have feelings, but thanks for asking!'],
    'what is your purpose': ['My purpose is to assist and provide information to users.', 'I\'m here to help answer your questions and provide support.', 'I\'m a chatbot designed to simulate conversations and provide helpful responses.'],
    'default': ['I didn\'t understand that.', 'Can you please rephrase?', 'I\'m not sure what you mean.']
}

# Define a function to get the chatbot response
def get_response(user_input):
    user_input = user_input.lower()
    for keyword in responses:
        if keyword in user_input:
            return random.choice(responses[keyword])
    return random.choice(responses['default'])

# Define a function to simulate the chatbot conversation
def simulate_conversation():
    print('Welcome to the chatbot conversation simulator!')
    while True:
        user_input = input('User: ')
        if user_input.lower() == 'quit':
            print('Chatbot: Goodbye!')
            break
        print('Chatbot:', get_response(user_input))

# Call the simulate_conversation function
simulate_conversation()
```
