## Exercise
Create a program that generates all prime numbers up to a given number n using a while loop. The program should take an integer n as input and print all prime numbers from 2 to n.

## Reference Solution
```python
#include <iostream>
using namespace std;

int main() {
    int n, i = 2, j, p;

    cout << "Enter an integer: ";
    cin >> n;

    while (i < n) {
        p = 0;
        j = 2;
        while (j < i) {
            if (i % j == 0) {
                p = 1;
                break;
            }
            j++;
        }
        if (p == 0) {
            cout << i << " ";
        }
        i++;
    }

    return 0;
}
```
