## Exercise
Retrieve a list of key-value pairs in a dictionary using the items() method

## Reference Solution
```python
# Define a dictionary
my_dict = {
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}

# Retrieve key-value pairs using items()
key_value_pairs = list(my_dict.items())

# Print the key-value pairs
for pair in key_value_pairs:
    print(f"Key: {pair[0]}, Value: {pair[1]}")
```

```python
for key, value in my_dict.items():
    print(f"Key: {key}, Value: {value}")
```
