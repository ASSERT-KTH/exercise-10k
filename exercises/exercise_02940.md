## Exercise
Create a program that uses nested for loops to print a diamond shape of asterisks (*).

## Reference Solution
```python
def print_diamond(n):
    """Prints a diamond shape of asterisks."""
    
    # Print top half of diamond
    for i in range(n):
        print(' ' * (n - i - 1) + '*' * (2 * i + 1))
    
    # Print bottom half of diamond
    for i in range(n - 2, -1, -1):
        print(' ' * (n - i - 1) + '*' * (2 * i + 1))

def main():
    n = 5  # Change this to adjust the size of the diamond
    print_diamond(n)

if __name__ == "__main__":
    main()
```
