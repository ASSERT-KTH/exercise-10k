## Exercise
Store a date and time in a variable and print it in a specific format.

## Reference Solution
```python
from datetime import datetime

def store_and_print_datetime(year, month, day, hour, minute, second):
    """
    Stores a date and time in a variable and prints it in a specific format.
    
    Args:
        year (int): The year of the date.
        month (int): The month of the date.
        day (int): The day of the date.
        hour (int): The hour of the time.
        minute (int): The minute of the time.
        second (int): The second of the time.
    """

    # Store the date and time in a variable
    date_time = datetime(year, month, day, hour, minute, second)

    # Print the date and time in a specific format (YYYY-MM-DD HH:MM:SS)
    print(date_time.strftime("%Y-%m-%d %H:%M:%S"))

# Example usage
store_and_print_datetime(2024, 9, 16, 10, 30, 0)
```
