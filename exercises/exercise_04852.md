## Exercise
Write a program that takes a list as input, and returns its minimum element.

## Reference Solution
```python
def find_min_element(input_list):
    """
    This function finds the minimum element in a given list.

    Args:
        input_list (list): A list of numbers.

    Returns:
        The minimum element in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not input_list:
        raise ValueError("Input list is empty")

    return min(input_list)


# Example usage
def main():
    numbers = [12, 45, 7, 23, 56, 89, 34]
    try:
        min_element = find_min_element(numbers)
        print("Minimum element:", min_element)
    except ValueError as e:
        print("Error:", e)


if __name__ == "__main__":
    main()
```
