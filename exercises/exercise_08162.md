## Exercise
Create a list of 3 integers and remove the first occurrence of a specific element.

## Reference Solution
```python
def remove_first_occurrence(lst, target):
    """
    Removes the first occurrence of a target element in a list.

    Args:
        lst (list): The input list.
        target: The target element to remove.

    Returns:
        list: The modified list with the first occurrence of the target element removed.
    """
    try:
        # Find the index of the first occurrence of the target element
        idx = lst.index(target)
        
        # Remove the element at the found index
        lst.pop(idx)
        
        return lst
    
    except ValueError:
        # If the target element is not found, return the original list
        return lst


# Example usage:
numbers = [1, 2, 3, 2, 4]
target_element = 2

print("Original List:", numbers)
print("Target Element:", target_element)

modified_list = remove_first_occurrence(numbers, target_element)
print("Modified List:", modified_list)
```
