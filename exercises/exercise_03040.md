## Exercise
Write a program that takes a string as input and returns the string compressed using the Lzip algorithm.

## Reference Solution
```python
import lzma

def compress_string(input_string):
    # Convert the string to bytes
    input_bytes = input_string.encode('utf-8')

    # Compress the bytes using lzip
    compressed_bytes = lzma.compress(input_bytes)

    return compressed_bytes

def main():
    input_string = input("Enter a string to compress: ")
    compressed_bytes = compress_string(input_string)

    # Print the compressed bytes as a hexadecimal string
    print(compressed_bytes.hex())

if __name__ == "__main__":
    main()
```
