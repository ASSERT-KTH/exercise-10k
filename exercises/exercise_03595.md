## Exercise
Write a program that calculates the tip amount given the total bill and the tip percentage.

## Reference Solution
```python
def calculate_tip(total_bill, tip_percentage):
    """
    Calculates the tip amount given the total bill and the tip percentage.

    Args:
        total_bill (float): The total bill amount.
        tip_percentage (float): The tip percentage.

    Returns:
        float: The tip amount.
    """
    tip_amount = (total_bill / 100) * tip_percentage
    return tip_amount

def main():
    # Get the total bill from the user
    total_bill = float(input("Enter the total bill: $"))

    # Get the tip percentage from the user
    tip_percentage = float(input("Enter the tip percentage (%): "))

    # Calculate the tip amount
    tip_amount = calculate_tip(total_bill, tip_percentage)

    # Print the tip amount
    print(f"Tip amount: ${tip_amount:.2f}")

    # Print the total amount (bill + tip)
    print(f"Total amount: ${total_bill + tip_amount:.2f}")

if __name__ == "__main__":
    main()
```
