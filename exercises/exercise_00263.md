## Exercise
Write a program that calculates the simple interest given principal amount, rate of interest, and time period.

## Reference Solution
```python
def calculate_simple_interest(principal, rate, time):
    """
    Calculates the simple interest.

    Args:
        principal (float): The principal amount.
        rate (float): The rate of interest (in %).
        time (int): The time period (in years).

    Returns:
        float: The simple interest.
    """
    simple_interest = (principal * rate * time) / 100
    return simple_interest

def main():
    # Get user input
    principal = float(input("Enter the principal amount: "))
    rate = float(input("Enter the rate of interest (in %): "))
    time = int(input("Enter the time period (in years): "))

    # Calculate simple interest
    simple_interest = calculate_simple_interest(principal, rate, time)

    # Display result
    print(f"Simple Interest: {simple_interest:.2f}")
    print(f"Total Amount: {principal + simple_interest:.2f}")

if __name__ == "__main__":
    main()
```
