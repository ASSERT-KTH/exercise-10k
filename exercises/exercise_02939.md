## Exercise
Write a program that uses a while loop to count the number of digits in a given number.

## Reference Solution
```python
def count_digits(n):
    """
    Counts the number of digits in a given integer.

    Args:
        n (int): The number to count digits for.

    Returns:
        int: The number of digits in the given number.
    """
    count = 0
    while n != 0:
        n = n // 10  # integer division to remove the last digit
        count += 1
    return count

def main():
    num = int(input("Enter a number: "))
    if num == 0:
        print("The number 0 has 1 digit.")
    else:
        digit_count = count_digits(abs(num))  # handle negative numbers
        print(f"The number {num} has {digit_count} digits.")

if __name__ == "__main__":
    main()
```
