## Exercise
Create a function that takes an array and a reduction function as input, and returns the reduced value of the array elements.

## Reference Solution
```python
const reduce = (arr, fn) => {
  let acc = arr[0];
  for (let i = 1; i < arr.length; i++) {
    acc = fn(acc, arr[i]);
  }
  return acc;
};

const sum = (a, b) => a + b;
const multiply = (a, b) => a * b;

console.log(reduce([1, 2, 3, 4, 5], sum)); // Output: 15
console.log(reduce([1, 2, 3, 4, 5], multiply)); // Output: 120
```
