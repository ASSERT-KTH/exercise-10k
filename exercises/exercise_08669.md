## Exercise
Write a program that asks the user for a number and checks if it is a valid integer.

## Reference Solution
```python
def get_integer_from_user():
    """
    Asks the user for a number and checks if it is a valid integer.
    """
    while True:
        try:
            # Ask the user for a number
            user_input = input("Please enter a number: ")
            
            # Try to convert the input to an integer
            number = int(user_input)
            
            # If the conversion is successful, return the number
            return number
        except ValueError:
            # If the conversion fails, print an error message and ask again
            print("That's not a valid integer. Please try again.")

# Example usage:
number = get_integer_from_user()
print("You entered a valid integer:", number)
```
