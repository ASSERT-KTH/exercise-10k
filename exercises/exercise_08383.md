## Exercise
Write a program that calculates the Ackermann function of two given numbers.

## Reference Solution
```python
def ackermann(m, n):
    """
    Calculate the Ackermann function of two given numbers.

    Args:
        m (int): The first non-negative integer argument.
        n (int): The second non-negative integer argument.

    Returns:
        int: The result of the Ackermann function.
    """
    if m == 0:
        return n + 1
    elif m > 0 and n == 0:
        return ackermann(m - 1, 1)
    elif m > 0 and n > 0:
        return ackermann(m - 1, ackermann(m, n - 1))

# Example usage
if __name__ == "__main__":
    m = 3
    n = 4
    result = ackermann(m, n)
    print(f"Ackermann({m}, {n}) = {result}")
```
