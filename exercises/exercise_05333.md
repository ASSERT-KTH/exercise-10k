## Exercise
Declare a string variable with the value 'Hello World'. Extract the substring 'World' and print the result.

## Reference Solution
```python
def extract_substring():
    # Declare a string variable
    full_string = 'Hello World'

    # Extract the substring 'World'
    # Since 'World' is the second word in the string, we can split the string into words and access the second word
    substring = full_string.split()[1]

    # Print the extracted substring
    print(substring)

# Example usage
if __name__ == "__main__":
    extract_substring()
```
