## Exercise
Write a program that takes an array of numbers as input and returns the first duplicate number in the array.

## Reference Solution
```python
class Solution:
    def firstDuplicate(self, a):
        seen = set()
        for num in a:
            if num in seen:
                return num
            seen.add(num)
        return -1

# Example usage:
solution = Solution()
print(solution.firstDuplicate([2, 1, 3, 5, 3, 2]))  # Output: 3
print(solution.firstDuplicate([2, 4, 3, 5, 1]))  # Output: -1
```
