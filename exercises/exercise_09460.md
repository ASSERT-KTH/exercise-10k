## Exercise
Create a program that takes a string as input and parses it into an integer.

## Reference Solution
```python
def parse_string_to_int(input_string):
    """
    This function takes a string as input, attempts to parse it into an integer, 
    and returns the integer if successful. If the string cannot be parsed into 
    an integer, it returns an error message.

    Parameters:
    input_string (str): The string to be parsed into an integer.

    Returns:
    int or str: The parsed integer or an error message if parsing fails.
    """
    try:
        # Attempt to parse the string into an integer
        parsed_int = int(input_string)
        return parsed_int
    except ValueError:
        # If parsing fails, return an error message
        return "Error: Input string cannot be parsed into an integer."


def main():
    # Example usage
    input_str = input("Enter a string to parse into an integer: ")
    result = parse_string_to_int(input_str)
    print(result)


if __name__ == "__main__":
    main()
```
