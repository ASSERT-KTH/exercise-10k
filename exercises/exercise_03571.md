## Exercise
Write a function that searches a dictionary for a given key or value

## Reference Solution
```python
def search_dict(dictionary, target):
    """
    Searches a dictionary for a given key or value.

    Args:
        dictionary (dict): The dictionary to search.
        target (any): The key or value to search for.

    Returns:
        list: A list of tuples containing the key and value if the target is found.
    """
    found = []
    for key, value in dictionary.items():
        if key == target or value == target:
            found.append((key, value))
    return found

# Example usage
if __name__ == "__main__":
    example_dict = {
        "apple": 1,
        "banana": 2,
        "cherry": 3
    }
    print(search_dict(example_dict, "apple"))  # Search for key
    print(search_dict(example_dict, 2))  # Search for value
```
