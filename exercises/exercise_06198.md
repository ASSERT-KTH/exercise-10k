## Exercise
Write a function that takes a string as input and returns True if the string contains only digit characters, False otherwise.

## Reference Solution
```python
def check_digit_string(input_string):
    """
    This function checks if the input string contains only digit characters.

    Args:
        input_string (str): The input string to be checked.

    Returns:
        bool: True if the string contains only digit characters, False otherwise.
    """

    # We use the str.isdigit() method in Python, which returns True if all characters in the string are digits and there is at least one character, otherwise it returns False.
    return input_string.isdigit()

# Test the function
print(check_digit_string("12345"))  # Should return True
print(check_digit_string("123abc"))  # Should return False
```
