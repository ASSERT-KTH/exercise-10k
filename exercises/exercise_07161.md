## Exercise
Write a program that prints the numbers from 1 to 10 using a loop

## Reference Solution
```python
# Define a function to print numbers from 1 to n
def print_numbers(n):
    """
    Prints numbers from 1 to n using a loop.

    Args:
        n (int): The upper limit of the numbers to print.
    """
    for i in range(1, n + 1):
        print(i)

# Example usage:
if __name__ == "__main__":
    n = 10
    print(f"Printing numbers from 1 to {n}:")
    print_numbers(n)
```
