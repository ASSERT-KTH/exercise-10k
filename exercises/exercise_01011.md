## Exercise
Write a program that takes a string and an integer as input, and returns the rotated string, where the last n characters are moved to the beginning of the string.

## Reference Solution
```python
def rotate_string(s, n):
    return s[-n:] + s[:-n]

# Test the function
print(rotate_string("python", 2))  # Expected output: "onpyth"
```
