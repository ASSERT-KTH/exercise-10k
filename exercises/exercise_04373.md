## Exercise
Write a program that reduces a dictionary to a single value by applying a reduction function to its values.

## Reference Solution
```python
def reduce_dict(d, func):
    return func(*d.values())

def main():
    d = {'a': 1, 'b': 2, 'c': 3}
    res = reduce_dict(d, sum)
    print("Result:", res)

if __name__ == "__main__":
    main()
```
