## Exercise
Write a function that generates the Fibonacci sequence up to a given number.

## Reference Solution
```python
def generate_fibonacci(n):
    """
    Generates the Fibonacci sequence up to a given number.

    Args:
        n (int): The number of terms in the Fibonacci sequence.

    Returns:
        list: A list of integers representing the Fibonacci sequence.
    """
    if n <= 0:
        return []
    elif n == 1:
        return [0]
    elif n == 2:
        return [0, 1]

    fib_sequence = [0, 1]
    while len(fib_sequence) < n:
        fib_sequence.append(fib_sequence[-1] + fib_sequence[-2])

    return fib_sequence

# Example usage:
print(generate_fibonacci(10))
```
