## Exercise
Write a function to create a dictionary using dictionary comprehension

## Reference Solution
```python
def create_dictionary(keys, values):
    """
    Create a dictionary using dictionary comprehension.

    Args:
        keys (list): A list of keys for the dictionary.
        values (list): A list of values for the dictionary.

    Returns:
        dict: A dictionary with the given keys and values.
    """
    return {key: value for key, value in zip(keys, values)}

# Example usage:
keys = ['name', 'age', 'city']
values = ['John Doe', 30, 'New York']
result = create_dictionary(keys, values)
print(result)
```

```python
def square_numbers(n):
    """
    Create a dictionary with numbers from 1 to n as keys and their squares as values.

    Args:
        n (int): The upper limit for the numbers.

    Returns:
        dict: A dictionary with numbers and their squares.
    """
    return {i: i ** 2 for i in range(1, n + 1)}

# Example usage:
result = square_numbers(5)
print(result)
```
