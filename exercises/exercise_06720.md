## Exercise
Write a program that reads from standard input and writes to standard output

## Reference Solution
```python
import sys

def main():
    """
    Reads from standard input and writes to standard output.
    """
    for line in sys.stdin:
        # Remove newline character
        line = line.strip()
        # Process the line (in this case, just echo it)
        processed_line = line
        # Write to standard output
        sys.stdout.write(processed_line + "\n")

if __name__ == "__main__":
    main()
```
