## Exercise
Write a program that finds the index of a given element in a list.

## Reference Solution
```python
def find_index(lst, elem):
    try:
        return lst.index(elem)
    except ValueError:
        return -1  # Return -1 if the element is not found

# Test the function
my_list = [1, 2, 3, 4, 5]
element = 3
print(find_index(my_list, element))  # Output: 2

element = 6
print(find_index(my_list, element))  # Output: -1
```

```python
def find_indices(lst, elem):
    return [i for i, x in enumerate(lst) if x == elem]

# Test the function
my_list = [1, 2, 3, 3, 4, 5]
element = 3
print(find_indices(my_list, element))  # Output: [2, 3]
```
