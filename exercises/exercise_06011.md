## Exercise
Write a program that asks the user for a string, and prints out the number of characters in that string

## Reference Solution
```python
def count_characters():
    """
    Asks the user for a string and prints out the number of characters in that string.
    """
    user_string = input("Please enter a string: ")
    character_count = len(user_string)
    print(f"The string '{user_string}' has {character_count} characters.")

def main():
    count_characters()

if __name__ == "__main__":
    main()
```
