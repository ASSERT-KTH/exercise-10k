## Exercise
Write a program to find the first duplicate in an array of integers

## Reference Solution
```python
def find_first_duplicate(nums):
    """
    Finds the first duplicate in an array of integers.

    Args:
        nums (list): A list of integers.

    Returns:
        int: The first duplicate in the array, or None if no duplicates exist.
    """
    seen = set()
    for num in nums:
        if num in seen:
            return num
        seen.add(num)
    return None

# Example usage
if __name__ == "__main__":
    nums = [2, 1, 3, 5, 3, 2]
    print(find_first_duplicate(nums))  # Output: 3
```
