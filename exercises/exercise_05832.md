## Exercise
Write a function to generate a slug from a given string by replacing spaces with hyphens and removing special characters

## Reference Solution
```python
import re

def generate_slug(input_string):
    """
    Generate a slug from a given string by replacing spaces with hyphens and removing special characters.

    Args:
        input_string (str): The input string to generate a slug from.

    Returns:
        str: The generated slug.
    """
    # Remove special characters
    slug = re.sub(r'[^a-zA-Z0-9\s]', '', input_string)
    
    # Replace spaces with hyphens
    slug = slug.replace(' ', '-')

    # Convert to lowercase
    slug = slug.lower()

    return slug

# Example usage:
input_string = "Hello, World! This is a test string."
print(generate_slug(input_string))
```
