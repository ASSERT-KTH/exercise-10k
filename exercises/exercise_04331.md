## Exercise
Write a program to append a given element to an array.

## Reference Solution
```python
def append_to_array(array, element):
    """
    Appends a given element to an array.

    Args:
        array (list): The input array.
        element: The element to be appended.

    Returns:
        list: The updated array with the element appended.
    """
    array.append(element)
    return array

# Example usage:
array = [1, 2, 3, 4, 5]
element = 6
print("Original array:", array)
print("Element to be appended:", element)
print("Updated array:", append_to_array(array, element))
```

def append_to_array(array, element):
    """
    Appends a given element to an array without modifying the original array.

    Args:
        array (list): The input array.
        element: The element to be appended.

    Returns:
        list: A new array with the element appended.
    """
    new_array = array.copy()
    new_array.append(element)
    return new_array
