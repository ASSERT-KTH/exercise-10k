## Exercise
Write a function called 'calculate_average' that calculates the average of all numbers in a given list.

## Reference Solution
```python
def calculate_average(num_list):
    """
    Calculate the average of all numbers in a given list.

    Args:
        num_list (list): A list of numbers.

    Returns:
        float: The average of all numbers in the list.

    Raises:
        ValueError: If the list is empty.
        TypeError: If the list contains non-numeric values.
    """
    # Check if the list is empty
    if len(num_list) == 0:
        raise ValueError("List cannot be empty")

    # Check if all elements in the list are numeric
    if not all(isinstance(num, (int, float)) for num in num_list):
        raise TypeError("List must only contain numeric values")

    # Calculate the average
    average = sum(num_list) / len(num_list)

    return average


# Example usage
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    try:
        average = calculate_average(numbers)
        print(f"The average of {numbers} is: {average}")
    except (ValueError, TypeError) as e:
        print(f"Error: {e}")
```
