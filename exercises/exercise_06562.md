## Exercise
Write a function that takes a dictionary and a key as input and returns the corresponding value

## Reference Solution
```python
def get_dict_value(dictionary, key):
    """
    Retrieves the value associated with a given key from a dictionary.

    Args:
        dictionary (dict): The dictionary to search.
        key: The key to look for.

    Returns:
        The value associated with the key if found, otherwise None.
    """
    return dictionary.get(key)

# Example usage:
def main():
    example_dict = {"name": "John", "age": 30, "city": "New York"}
    key_to_find = "age"
    result = get_dict_value(example_dict, key_to_find)
    print(f"The value for key '{key_to_find}' is: {result}")

if __name__ == "__main__":
    main()
```
