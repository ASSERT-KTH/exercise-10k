## Exercise
Write a program to calculate the cube root of a number.

## Reference Solution
```python
import math

def calculate_cube_root(number):
    """
    Calculates the cube root of a given number.

    Args:
        number (float): The number to calculate the cube root of.

    Returns:
        float: The cube root of the given number.
    """
    if number < 0:
        return -math.pow(-number, 1/3)
    else:
        return math.pow(number, 1/3)

def main():
    # Example usage
    number = float(input("Enter a number: "))
    cube_root = calculate_cube_root(number)
    print(f"The cube root of {number} is {cube_root}")

if __name__ == "__main__":
    main()
```
