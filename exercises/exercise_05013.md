## Exercise
Write a program that checks the data type of the variable 'x' assigned the value 10. Print the data type to the console.

## Reference Solution
```python
def check_data_type():
    # Assign the value 10 to the variable x
    x = 10
    
    # Use the type() function to get the data type of x
    data_type = type(x)
    
    # Print the data type to the console
    print(f"The data type of x is: {data_type}")

# Call the function
check_data_type()
```
