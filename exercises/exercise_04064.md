## Exercise
Write a program that takes a string, a pattern, and a replacement as input and returns the string with all occurrences of the pattern replaced.

## Reference Solution
```python
import re

def replace(string, pattern, replacement):
    return re.sub(pattern, replacement, string)

string = input("Enter a string: ")
pattern = input("Enter a pattern: ")
replacement = input("Enter a replacement: ")

result = replace(string, pattern, replacement)
print(result)
```
