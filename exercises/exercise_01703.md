## Exercise
Write a program that uses a for loop to repeat a given string a specified number of times.

## Reference Solution
```python
def repeat_string(input_string, num_repeats):
    """
    Repeat a given string a specified number of times.

    Args:
        input_string (str): The string to be repeated.
        num_repeats (int): The number of times to repeat the string.

    Returns:
        str: The repeated string.
    """
    repeated_string = ""
    for _ in range(num_repeats):
        repeated_string += input_string
    return repeated_string

def main():
    input_string = input("Enter a string: ")
    num_repeats = int(input("Enter the number of times to repeat the string: "))
    repeated_string = repeat_string(input_string, num_repeats)
    print("Repeated string: ", repeated_string)

if __name__ == "__main__":
    main()
```
