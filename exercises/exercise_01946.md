## Exercise
Write a program that takes a list, an index, and a value as input and inserts the value at that index in the list.

## Reference Solution
```python
def insert_value_at_index(list1, index, value):
    list1.insert(index, value)
    return list1

list1 = [1, 2, 3, 4, 5]
index = 2
value = 10
print(insert_value_at_index(list1, index, value))
```
