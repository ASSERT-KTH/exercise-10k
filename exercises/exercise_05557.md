## Exercise
Write a program that finds the minimum of two numbers using a simple if statement.

## Reference Solution
```python
def find_minimum(num1, num2):
    """
    Returns the minimum of two numbers.

    Args:
        num1 (int): The first number.
        num2 (int): The second number.

    Returns:
        int: The minimum of num1 and num2.
    """
    if num1 < num2:
        return num1
    else:
        return num2

def main():
    # Example usage
    num1 = 10
    num2 = 20
    minimum = find_minimum(num1, num2)
    print(f"The minimum of {num1} and {num2} is {minimum}")

if __name__ == "__main__":
    main()
```
