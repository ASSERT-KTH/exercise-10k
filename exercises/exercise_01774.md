## Exercise
Write a function to find the maximum sum of a subarray in a given array of integers.

## Reference Solution
```python
def max_subarray_sum(array):
    """
    Calculate the maximum sum of a subarray in a given array of integers.

    Args:
        array (list): A list of integers.

    Returns:
        int: The maximum sum of a subarray.

    Raises:
        ValueError: If the input array is empty.
    """
    if not array:
        raise ValueError("Input array is empty")

    max_sum = float('-inf')  # Initialize max_sum to negative infinity
    current_sum = 0

    for num in array:
        current_sum = max(num, current_sum + num)
        max_sum = max(max_sum, current_sum)

    return max_sum

# Example usage
if __name__ == "__main__":
    array = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
    max_sum = max_subarray_sum(array)
    print(f"Maximum subarray sum: {max_sum}")
```
