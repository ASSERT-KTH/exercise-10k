## Exercise
Write a program that uses nested for loops to print a 5x5 spiral pattern of numbers.

## Reference Solution
```python
def print_spiral(n):
    # Create an empty 5x5 matrix
    matrix = [[0]*n for _ in range(n)]
    
    # Initialize the counter
    counter = 1
    
    # Initialize the start and end indices
    start_row, end_row = 0, n
    start_col, end_col = 0, n
    
    # Loop until the entire matrix is filled
    while start_row < end_row and start_col < end_col:
        
        # Fill the first row
        for i in range(start_col, end_col):
            matrix[start_row][i] = counter
            counter += 1
        start_row += 1
        
        # Fill the last column
        for i in range(start_row, end_row):
            matrix[i][end_col-1] = counter
            counter += 1
        end_col -= 1
        
        # Fill the last row
        if start_row < end_row:
            for i in range(end_col-1, start_col-1, -1):
                matrix[end_row-1][i] = counter
                counter += 1
            end_row -= 1
        
        # Fill the first column
        if start_col < end_col:
            for i in range(end_row-1, start_row-1, -1):
                matrix[i][start_col] = counter
                counter += 1
            start_col += 1
    
    # Print the spiral matrix
    for row in matrix:
        print(' '.join(str(num).ljust(2) for num in row))

# Test the function
print_spiral(5)
```
