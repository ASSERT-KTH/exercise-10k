## Exercise
Write a program that takes a user input and determines its data type. The program should be able to identify integers, floats, and strings. If the input cannot be classified into one of these types, it should print an error message.

## Reference Solution
```python
def determine_data_type(user_input):
    """
    This function determines the data type of a given user input.

    Args:
        user_input (str): The input provided by the user.

    Returns:
        str: The data type of the user input.
    """
    try:
        # Attempt to convert the input into an integer
        int(user_input)
        return "Integer"
    except ValueError:
        try:
            # If the input is not an integer, attempt to convert it into a float
            float(user_input)
            return "Float"
        except ValueError:
            # If the input is neither an integer nor a float, it is a string
            return "String"


def main():
    user_input = input("Please enter a value: ")
    data_type = determine_data_type(user_input)

    if data_type == "String":
        print(f"The input '{user_input}' is of type {data_type}.")
    else:
        print(f"The input {user_input} is of type {data_type}.")


if __name__ == "__main__":
    main()
```
