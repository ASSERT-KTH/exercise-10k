## Exercise
Write a program to subtract one number from another and display the result

## Reference Solution
```python
def subtract_numbers(num1, num2):
    """
    Subtracts num2 from num1 and returns the result.
    
    Args:
        num1 (float): The first number.
        num2 (float): The second number.
    
    Returns:
        float: The result of subtracting num2 from num1.
    """
    return num1 - num2

def main():
    # Get the numbers from the user
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))

    # Subtract the numbers and display the result
    result = subtract_numbers(num1, num2)
    print(f"{num1} - {num2} = {result}")

if __name__ == "__main__":
    main()
```
