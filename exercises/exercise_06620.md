## Exercise
Write a program that takes a string as input from the user and prints whether it is a palindrome or not.

## Reference Solution
```python
def is_palindrome(input_string):
    """
    Checks if the input string is a palindrome.

    Args:
        input_string (str): The input string to check.

    Returns:
        bool: True if the string is a palindrome, False otherwise.
    """
    input_string = input_string.lower()  # Convert to lowercase for comparison
    reversed_string = input_string[::-1]  # Reverse the string
    return input_string == reversed_string

def main():
    input_string = input("Enter a string: ")
    if is_palindrome(input_string):
        print(f'"{input_string}" is a palindrome.')
    else:
        print(f'"{input_string}" is not a palindrome.')

if __name__ == "__main__":
    main()
```
