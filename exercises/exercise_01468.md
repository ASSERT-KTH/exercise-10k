## Exercise
Write a program that takes two integer inputs from the user, calculates the modulus (remainder), and prints the result.

## Reference Solution
```python
# Get two integer inputs from the user
n1 = int(input("Enter first number: "))
n2 = int(input("Enter second number: "))

# Calculate the modulus (remainder)
s = n1 % n2

# Print the result
print("The modulus of", n1, "and", n2, "is:", s)
```
