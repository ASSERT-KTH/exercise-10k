## Exercise
Create a program that takes a word as input and scrambles its letters.

## Reference Solution
```python
import random

def scramble_word(word):
    """
    Scrambles the letters of the input word.

    Args:
        word (str): The word to scramble.

    Returns:
        str: The scrambled word.
    """
    word_list = list(word)  # Convert the word to a list
    random.shuffle(word_list)  # Shuffle the list
    return ''.join(word_list)  # Convert the list back to a string

def main():
    word = input("Enter a word: ")
    scrambled_word = scramble_word(word)
    print(f"Scrambled word: {scrambled_word}")

if __name__ == "__main__":
    main()
```
