## Exercise
Develop a program that translates English words to Pig Latin. The program should ask for the word, and then print out the translated word.

## Reference Solution
```python
def translate_to_pig_latin(word):
    """
    Translates an English word to Pig Latin.

    Args:
        word (str): The English word to translate.

    Returns:
        str: The translated Pig Latin word.
    """
    vowels = 'aeiou'
    if word[0].lower() in vowels:
        return word + 'way'
    else:
        vowel_index = 0
        for char in word:
            if char.lower() in vowels:
                break
            vowel_index += 1
        return word[vowel_index:] + word[:vowel_index] + 'ay'

def main():
    word = input("Enter a word to translate: ")
    print("The Pig Latin translation is:", translate_to_pig_latin(word))

if __name__ == "__main__":
    main()
```
