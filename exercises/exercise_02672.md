## Exercise
Write a program that takes a string and an integer as input, and returns the string encrypted using the Caesar cipher with the given shift.

## Reference Solution
```python
def caesar_cipher(s, shift):
    result = ""
    for char in s:
        if char.isalpha():
            ascii_offset = 97 if char.islower() else 65
            result += chr((ord(char) - ascii_offset + shift) % 26 + ascii_offset)
        else:
            result += char
    return result

# Test the function
print(caesar_cipher("Hello, World!", 3))  # Khoor, Zruog!
```
