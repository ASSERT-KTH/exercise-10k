## Exercise
Create a function that generates a random string of a specified length. The string should contain a mix of uppercase and lowercase letters, numbers, and special characters.

## Reference Solution
```python
import random
import string

def generate_random_string(length):
    """
    Generates a random string of a specified length containing a mix of 
    uppercase and lowercase letters, numbers, and special characters.

    Args:
        length (int): The length of the string to be generated.

    Returns:
        str: A random string of the specified length.
    """

    # First, we need to define the characters that can be included in the string
    # We'll use string.ascii_letters for uppercase and lowercase letters, 
    # string.digits for numbers, and string.punctuation for special characters
    all_characters = string.ascii_letters + string.digits + string.punctuation
    
    # Then, we'll use a list comprehension to generate a list of random characters
    # The length of the list will be equal to the specified length
    # We'll use random.choice to select a random character from all_characters
    random_characters = [random.choice(all_characters) for _ in range(length)]
    
    # Finally, we'll use ''.join() to concatenate the characters in the list into a single string
    random_string = ''.join(random_characters)
    
    return random_string

# Test the function
print(generate_random_string(10))
```
