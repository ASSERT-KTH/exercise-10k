## Exercise
Write a program that takes a list of integers and a predicate function as input, and returns a new list containing only the elements for which the predicate function returns True.

## Reference Solution
```python
def filter_list(value, predicate):
    if not isinstance(predicate, type(lambda: None)):
        raise TypeError("Predicate must be a function")

    if not isinstance(value, list):
        raise TypeError("Value must be a list")

    return list(filter(predicate, value))
```
