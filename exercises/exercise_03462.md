## Exercise
Write a program that asks for the user's name and then prints out a greeting with their name.

## Reference Solution
```python
# Get the user's name
def get_user_name():
    """Asks the user for their name."""
    user_name = input("Please enter your name: ")
    return user_name

# Print a greeting with the user's name
def print_greeting(name):
    """Prints a greeting with the provided name."""
    print(f"Hello, {name}!")

# Main function
def main():
    user_name = get_user_name()
    print_greeting(user_name)

# Run the main function
if __name__ == "__main__":
    main()
```
