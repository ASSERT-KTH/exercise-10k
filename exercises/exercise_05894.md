## Exercise
Write a function that takes an array and a number of positions as input and returns the array rotated by that number of positions.

## Reference Solution
```python
def rotate_array(arr, num_positions):
    """
    Rotate an array by a specified number of positions.

    Args:
        arr (list): The input array.
        num_positions (int): The number of positions to rotate the array.

    Returns:
        list: The rotated array.
    """
    num_positions = num_positions % len(arr)  # Handle cases where num_positions > len(arr)
    return arr[-num_positions:] + arr[:-num_positions]

# Example usage
def main():
    arr = [1, 2, 3, 4, 5]
    num_positions = 2
    print("Original array:", arr)
    print("Rotated array:", rotate_array(arr, num_positions))

if __name__ == "__main__":
    main()
```
