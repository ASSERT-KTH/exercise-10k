## Exercise
Write a program that takes a string as input and returns whether it contains only letters and spaces.

## Reference Solution
```python
def check_string(s):
    return all(c.isalpha() or c.isspace() for c in s)

# Test the function
print(check_string("Hello World"))  # True
print(check_string("Hello World!"))  # False
print(check_string("123"))  # False
print(check_string("a b c"))  # True
```

```python
import re

def check_string(s):
    return bool(re.match("^[a-zA-Z ]*$", s))

# Test the function
print(check_string("Hello World"))  # True
print(check_string("Hello World!"))  # False
print(check_string("123"))  # False
print(check_string("a b c"))  # True
```
