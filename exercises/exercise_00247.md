## Exercise
Write a program that takes an array and a target element as input, and returns the index of the target element if it exists in the array.

```python
def find_index(array, target):
    for i in range(len(array)):
        if array[i] == target:
            return i
    return -1

array = [1, 2, 3, 4, 5]
target = 3
print(find_index(array, target))  # Output: 2
```
