## Exercise
Write a program that prints a pattern of numbers using nested for loops.

## Reference Solution
```python
# Program to print a pattern of numbers using nested for loops

# Function to print the pattern
def print_pattern(n):
    """
    Prints a pattern of numbers from 1 to n.

    Args:
        n (int): The number up to which the pattern is printed.
    """
    for i in range(1, n + 1):
        for j in range(1, i + 1):
            print(j, end=" ")
        print()

# Example usage:
print("Pattern:")
print_pattern(5)
```
