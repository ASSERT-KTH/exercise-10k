## Exercise
Write a function that checks if a string contains a substring.

## Reference Solution
```python
def check_substring(main_string: str, sub_string: str) -> bool:
    """
    Checks if a string contains a substring.

    Args:
        main_string (str): The main string to search in.
        sub_string (str): The substring to search for.

    Returns:
        bool: True if the substring is found, False otherwise.
    """
    return sub_string in main_string

# Example usage
if __name__ == "__main__":
    main_str = "Hello, World!"
    sub_str = "World"
    result = check_substring(main_str, sub_str)
    print(f"'{main_str}' contains '{sub_str}': {result}")
```
