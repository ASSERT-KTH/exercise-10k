## Exercise
Create a program that takes two numbers as input and returns the absolute difference between them.

## Reference Solution
```python
def absolute_difference(num1, num2):
    return abs(num1 - num2)

# Example usage:
num1 = float(input("Enter the first number: "))
num2 = float(input("Enter the second number: "))

result = absolute_difference(num1, num2)
print("The absolute difference between the two numbers is: ", result)
```
