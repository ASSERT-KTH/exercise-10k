## Exercise
Write a function to reverse a given string. The function should take a string as input and return the reversed string.

## Reference Solution
```python
def reverse_string(input_str):
    """
    This function takes a string as input and returns the reversed string.

    Args:
        input_str (str): The input string to be reversed.

    Returns:
        str: The reversed string.
    """

    # First, we check if the input is indeed a string
    if not isinstance(input_str, str):
        raise TypeError("Input must be a string.")

    # We use Python's slicing feature to reverse the string
    # The syntax [start:stop:step] means start at the end of the string, 
    # stop at position 0, move with the step -1 which means one step backwards
    reversed_str = input_str[::-1]
    
    # Finally, we return the reversed string
    return reversed_str

# Example usage:
print(reverse_string("Hello World"))  # Output: "dlroW olleH"
```
