## Exercise
Create a function that takes a string as input and returns the number of vowels present in it.

## Reference Solution
```python
def count_vowels(input_string: str) -> int:
    """
    Counts the number of vowels in a given string.

    Args:
        input_string (str): The input string to count vowels from.

    Returns:
        int: The number of vowels in the input string.
    """
    vowels = 'aeiouAEIOU'
    return sum(1 for char in input_string if char in vowels)

# Example usage:
def main():
    input_str = "Hello World"
    print(f"Number of vowels in '{input_str}': {count_vowels(input_str)}")

if __name__ == "__main__":
    main()
```
