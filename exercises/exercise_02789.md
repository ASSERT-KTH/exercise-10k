## Exercise
Create a function that takes a list as input and returns a new list that is the sorted version of the original list.

## Reference Solution
```python
def sort_list(input_list):
    """
    This function creates a new sorted list from the input list.
    
    Parameters:
    input_list (list): The list that needs to be sorted.
    
    Returns:
    list: A new sorted list.
    """
    return sorted(input_list)

# Example usage:
input_list = [64, 34, 25, 12, 22, 11, 90]
print("Original list:", input_list)
print("Sorted list:", sort_list(input_list))
```
