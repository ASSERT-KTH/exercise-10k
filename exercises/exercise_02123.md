## Exercise
Create a function that takes an array and a threshold value as input and returns a new array containing only the elements greater than the threshold.

## Reference Solution
```python
def filter_threshold(array, threshold):
    """
    Returns a new list containing only the elements from the input array that are greater than the given threshold.

    Args:
        array (list): The input array to filter.
        threshold (int or float): The threshold value to filter against.

    Returns:
        list: A new list containing the filtered elements.
    """
    result = []
    for element in array:
        if element > threshold:
            result.append(element)
    return result

# Example usage
my_list = [1, 2, 3, 4, 5, 6]
filtered_list = filter_threshold(my_list, 3)
print(filtered_list)  # Output: [4, 5, 6]
```
