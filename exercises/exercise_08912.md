## Exercise
Write a program to find the longest common prefix of a given list of strings.

## Reference Solution
```python
def longest_common_prefix(strings):
    """
    Finds the longest common prefix of a given list of strings.

    Args:
        strings (list): A list of strings.

    Returns:
        str: The longest common prefix.
    """
    if not strings:
        return ""

    prefix = ""
    for chars in zip(*strings):
        if len(set(chars)) == 1:
            prefix += chars[0]
        else:
            break

    return prefix

# Example usage
strings = ["hello", "hello world", "hello python"]
print(longest_common_prefix(strings))  # Output: "hello"

strings = ["apple", "app", "apply"]
print(longest_common_prefix(strings))  # Output: "app"

strings = ["", "hello", "world"]
print(longest_common_prefix(strings))  # Output: ""

strings = []
print(longest_common_prefix(strings))  # Output: ""
```
