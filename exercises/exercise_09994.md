## Exercise
Use an immutable variable to store a value that should not be changed

## Reference Solution
```python
# Using tuple
MY_CONSTANT = (1, 2, 3)
try:
    MY_CONSTANT[0] = 10  # This will raise an error
except TypeError:
    print("Tuples are immutable")

# Using final keyword
from typing import Final
MY_CONSTANT: Final = 10
try:
    MY_CONSTANT = 20  # This will raise an error
except AttributeError:
    print("Final variables are immutable")
```

```python
public class ImmutableVariable {
    public static final int MY_CONSTANT = 10;

    public static void main(String[] args) {
        try {
            MY_CONSTANT = 20;  // This will raise an error
        } catch (Exception e) {
            System.out.println("Final variables are immutable");
        }
    }
}
```

```python
public class ImmutableVariable {
    public const int MY_CONSTANT = 10;

    public static void Main(string[] args) {
        try {
            // This will raise a compile-time error
            // MY_CONSTANT = 20;
        } catch (Exception e) {
            System.Console.WriteLine("Const variables are immutable");
        }
    }
}
```
