## Exercise
Write a function to get a list of all values in a dictionary

## Reference Solution
```python
def get_dict_values(dictionary):
    """
    Returns a list of all values in a dictionary.

    Args:
        dictionary (dict): The dictionary to retrieve values from.

    Returns:
        list: A list of dictionary values.
    """
    return list(dictionary.values())

# Example usage:
my_dict = {"name": "John", "age": 30, "city": "New York"}
print(get_dict_values(my_dict))  # Output: ['John', 30, 'New York']
```

def get_dict_values(dictionary):
    return [value for value in dictionary.values()]

# Example usage:
