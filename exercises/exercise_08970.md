## Exercise
Write a function that takes a string and a character to strip as input and returns the string with the character stripped from the start and end.

## Reference Solution
```python
function strip_char(s, ch):
    """
    Strips the specified character from the start and end of the string.
    """
    # Remove leading occurrences of the character
    s = s.lstrip(ch)
    
    # Remove trailing occurrences of the character
    s = s.rstrip(ch)
    
    return s

# Example usage:
print(strip_char("xyxHello Worldxxy", 'x'))  # Output: "xyHello World"
```
