## Exercise
Write a function that checks if a given number is even or odd.

## Reference Solution
```python
def check_even_or_odd(num: int) -> str:
    """
    Checks if a given number is even or odd.

    Args:
        num (int): The number to be checked.

    Returns:
        str: "Even" if the number is even, "Odd" otherwise.
    """
    if num % 2 == 0:
        return "Even"
    else:
        return "Odd"

# Example usage:
def main():
    num = int(input("Enter a number: "))
    result = check_even_or_odd(num)
    print(f"The number {num} is {result}.")

if __name__ == "__main__":
    main()
```
