## Exercise
Write a program that takes a string, a start index, and an end index as input and returns the substring between the start and end indices.

## Reference Solution
```python
def substring_between_indices(input_string, start_index, end_index):
    if start_index > end_index:
        return "Invalid input"
    if start_index < 0 or end_index < 0:
        return "Invalid input"
    if start_index >= len(input_string) or end_index >= len(input_string):
        return "Invalid input"
    return input_string[start_index:end_index]

print(substring_between_indices("hello world", 0, 5))
print(substring_between_indices("hello world", 6, 11))
print(substring_between_indices("hello world", -1, 11))
print(substring_between_indices("hello world", 11, 5))
print(substring_between_indices("hello world", 11, 20))
```
