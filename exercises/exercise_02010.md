## Exercise
Create a program that takes the title of a book and the status of the book as input and returns whether the book is available or not using conditional statements.

## Reference Solution
```python
def check_book_availability(title, status):
    """
    This function checks the availability of a book based on its status.

    Args:
        title (str): The title of the book.
        status (str): The status of the book. It can be 'available', 'not available', 'checked out', etc.

    Returns:
        str: A message indicating whether the book is available or not.
    """

    # Convert the status to lowercase to handle different cases
    status = status.lower()

    # Check if the book is available
    if status in ['available', 'on shelf', 'in stock']:
        return f"'{title}' is available."
    # Check if the book is not available
    elif status in ['not available', 'checked out', 'out of stock']:
        return f"'{title}' is not available."
    # If the status is neither available nor not available, return a message indicating that the status is unknown
    else:
        return f"The availability of '{title}' is unknown."


# Example usage
title = input("Enter the title of the book: ")
status = input("Enter the status of the book: ")

print(check_book_availability(title, status))
```
