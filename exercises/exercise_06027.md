## Exercise
Create two functions, one that doubles a number and another that adds 1 to a number. Use function composition to create a new function that doubles a number and then adds 1.

## Reference Solution
```python
# Define a function that doubles a number
def double(x):
    return x * 2

# Define a function that adds 1 to a number
def add_one(x):
    return x + 1

# Use function composition to create a new function that doubles a number and then adds 1
def double_and_add_one(x):
    return add_one(double(x))

# Test the new function
print(double_and_add_one(5))  # Expected output: 11
```

```python
double = lambda x: x * 2
add_one = lambda x: x + 1
double_and_add_one = lambda x: add_one(double(x))
print(double_and_add_one(5))  # Expected output: 11
```
