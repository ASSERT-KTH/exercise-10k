## Exercise
Create a string with non-ASCII characters and encode it using a specific encoding scheme, then print out the encoded string.

## Reference Solution
```python
# Define a string with non-ASCII characters
non_ascii_string = "Hëllö, Wørld!"

# Encode the string using UTF-8 encoding scheme
encoded_string = non_ascii_string.encode("utf-8")

# Print out the encoded string
print("Encoded String:", encoded_string)

# Print out the decoded string to verify the encoding
decoded_string = encoded_string.decode("utf-8")
print("Decoded String:", decoded_string)
```
