## Exercise
Create a function that takes an array of integers as input and returns the maximum value in the array. The function should handle arrays of varying lengths and should be able to find the maximum value even if it appears multiple times in the array.

## Reference Solution
```python
def find_max_value(array):
    """
    This function takes an array of integers as input and returns the maximum value in the array.
    
    Parameters:
    array (list): A list of integers.
    
    Returns:
    int: The maximum value in the array.
    """

    # First, we check if the input array is empty. If it is, we raise a ValueError because we can't find the maximum value of an empty array.
    if not array:
        raise ValueError("Array is empty")

    # We initialize the max_value variable with the first element of the array. This will be our initial guess for the maximum value.
    # We assume that the array has at least one element, which is a valid assumption because we checked for an empty array earlier.
    max_value = array[0]
    
    # We iterate over each element in the array, starting from the second element (index 1).
    for i in range(1, len(array)):
        # For each element, we check if it is greater than our current max_value.
        if array[i] > max_value:
            # If it is, we update max_value to be this new element.
            max_value = array[i]
    
    # After iterating over all elements in the array, max_value will hold the maximum value in the array.
    # We return this value.
    return max_value

# Example usage:
print(find_max_value([1, 2, 3, 4, 5]))  # Output: 5
print(find_max_value([5, 5, 5, 5, 5]))  # Output: 5
print(find_max_value([-1, -2, -3, -4, -5]))  # Output: -1
```
