## Exercise
Write a program that takes a string as input and returns the compressed string using Run-Length Encoding (RLE).

## Reference Solution
```python
def compress_string(input_str):
    """
    This function takes a string as input and returns the compressed string using Run-Length Encoding (RLE).

    Args:
        input_str (str): The input string to be compressed.

    Returns:
        str: The compressed string.
    """
    if not input_str:
        return ""

    compressed = []
    current_char = input_str[0]
    count = 1

    # Iterate over the input string from the second character to the end
    for char in input_str[1:]:
        if char == current_char:
            # If the current character is the same as the previous one, increment the count
            count += 1
        else:
            # If the current character is different from the previous one, append the previous character and count to the compressed list
            compressed.append(current_char + str(count))
            current_char = char
            count = 1

    # Append the last character and count to the compressed list
    compressed.append(current_char + str(count))

    # Join the compressed list into a string and return it
    return "".join(compressed)

input_str = "AAABBBCCCDDDD"
print("Input String: ", input_str)
print("Compressed String: ", compress_string(input_str))
```
