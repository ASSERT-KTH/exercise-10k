## Exercise
Write a program that declares a list variable and prints its elements.

## Reference Solution
```python
# Declare a list variable and populate it with sample elements
my_list = [1, 2, 3, 4, 5, "apple", "banana", "cherry"]

# Print each element in the list
def print_list_elements(input_list):
    """Prints each element in the input list."""
    for i, element in enumerate(input_list):
        print(f"Element {i+1}: {element}")

# Main function
def main():
    print("List Elements:")
    print_list_elements(my_list)

# Run the main function
if __name__ == "__main__":
    main()
```
