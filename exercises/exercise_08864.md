## Exercise
Write a function to calculate the digital root of a given number

## Reference Solution
```python
def digital_root(n: int) -> int:
    """
    Calculate the digital root of a given number.

    Args:
        n (int): The input number.

    Returns:
        int: The digital root of the input number.
    """
    while n > 9:
        n = sum(int(digit) for digit in str(n))
    return n

# Example usage:
if __name__ == "__main__":
    print(digital_root(16))  # Output: 7 (1 + 6 = 7)
    print(digital_root(942))  # Output: 6 (9 + 4 + 2 = 15, 1 + 5 = 6)
    print(digital_root(132189))  # Output: 6 (1 + 3 + 2 + 1 + 8 + 9 = 24, 2 + 4 = 6)
```
