## Exercise
Create a function that takes a list of numbers and returns their sum, using type hinting to specify the input and output types

## Reference Solution
```python
def sum_numbers(numbers: list[int]) -> int:
    """
    This function calculates the sum of a list of integers.
    
    Args:
        numbers (list[int]): A list of integers.
    
    Returns:
        int: The sum of the integers in the list.
    """
    
    # Initialize the sum variable to 0. This variable will store the sum of the numbers in the list.
    total_sum = 0
    
    # Iterate over each number in the list.
    for num in numbers:
        # Add the current number to the total sum.
        total_sum += num
    
    # Return the total sum after iterating over all numbers in the list.
    return total_sum

# Test the function
numbers = [1, 2, 3, 4, 5]
print(sum_numbers(numbers))  # Output: 15
```
