## Exercise
Write a function that takes a list and a predicate as input and returns a new list containing only the elements that satisfy the predicate.

## Reference Solution
```python
def filter_list(predicate, lst):
    """
    This function filters a given list based on a provided predicate.
    
    Args:
        predicate (function): A function that takes one argument and returns a boolean value.
        lst (list): The list to be filtered.
    
    Returns:
        list: A new list containing only the elements from the original list that satisfy the predicate.
    """

    # Initialize an empty list to store the filtered elements
    filtered_list = []

    # Iterate over each element in the input list
    for element in lst:
        # Apply the predicate to the current element
        if predicate(element):
            # If the predicate returns True, add the element to the filtered list
            filtered_list.append(element)

    # Return the filtered list
    return filtered_list

# Example usage:
# Define a predicate function that checks if a number is even
def is_even(num):
    return num % 2 == 0

# Create a list of numbers
numbers = [1, 2, 3, 4, 5, 6]

# Filter the list using the predicate
even_numbers = filter_list(is_even, numbers)

# Print the filtered list
print(even_numbers)  # Output: [2, 4, 6]
```
