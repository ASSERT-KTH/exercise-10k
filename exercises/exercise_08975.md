## Exercise
Write a function that takes a string and a substring as input and returns True if the string starts with the substring, False otherwise.

## Reference Solution
```python
def check_start(string, substring):
    """
    Checks if a given string starts with a specified substring.

    Args:
        string (str): The main string to be checked.
        substring (str): The substring to be searched at the start of the string.

    Returns:
        bool: True if the string starts with the substring, False otherwise.
    """

    # First, we need to check if both inputs are strings. If not, we raise a TypeError.
    if not isinstance(string, str) or not isinstance(substring, str):
        raise TypeError("Both inputs must be strings.")

    # Now, we use Python's built-in string method 'startswith' to check if the string starts with the substring.
    # This method returns True if the string starts with the substring, False otherwise.
    return string.startswith(substring)

# Example usage:
print(check_start("Hello World", "Hello"))  # Returns: True
print(check_start("Hello World", "World"))  # Returns: False
```
