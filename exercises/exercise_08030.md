## Exercise
Write a program that takes an array of integers as input and returns the maximum value in the array. The program should use a basic algorithm to iterate through the array and find the maximum value.

## Reference Solution
```python
#include <cassert>
#include <iostream>

int max(int* arr, int length) {
  assert(length > 0);
  int max_val = arr[0];
  for (int i = 1; i < length; i++) {
    if (arr[i] > max_val) {
      max_val = arr[i];
    }
  }
  return max_val;
}

int main() {
  int arr[] = {1, 2, 3, 4, 5};
  int length = sizeof(arr) / sizeof(arr[0]);
  std::cout << "Maximum value: " << max(arr, length) << std::endl;
  return 0;
}
```
