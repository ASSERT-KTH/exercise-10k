## Exercise
Create a function that takes an array of integers as input, calculates the mean, median, and standard deviation, and returns them in a list. The function should also remove any duplicate values from the array before performing the calculations.

## Reference Solution
```python
import numpy as np

def calculate_statistics(array):
    """
    This function calculates the mean, median, and standard deviation of an array of integers.
    
    It first removes any duplicate values from the array, then calculates the mean, median, and standard deviation.
    
    Args:
        array (list): A list of integers.
    
    Returns:
        list: A list containing the mean, median, and standard deviation of the input array.
    """

    # First, we convert the input list to a set to remove any duplicate values
    unique_array = set(array)
    
    # Then, we convert the set back to a list because numpy functions require lists or arrays as input
    unique_array = list(unique_array)
    
    # Check if the array is empty
    if len(unique_array) == 0:
        return [None, None, None]
    
    # Next, we calculate the mean of the array using numpy's mean function
    mean = np.mean(unique_array)
    
    # Then, we calculate the median of the array using numpy's median function
    median = np.median(unique_array)
    
    # Finally, we calculate the standard deviation of the array using numpy's std function
    std_dev = np.std(unique_array)
    
    # We return the calculated statistics in a list
    return [mean, median, std_dev]
```
