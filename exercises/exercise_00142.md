## Exercise
Create a function that takes in a dictionary and a key, and returns the value associated with that key if it exists, or a default value if it does not

## Reference Solution
```python
def get_dict_value(dictionary, key, default=None):
    """
    Retrieves the value associated with a key in a dictionary.

    Args:
        dictionary (dict): The dictionary to search.
        key: The key to search for.
        default: The value to return if the key is not found. Defaults to None.

    Returns:
        The value associated with the key, or the default value.
    """
    return dictionary.get(key, default)
```

```python
# Create a sample dictionary
sample_dict = {"name": "John", "age": 30}

# Retrieve a value that exists
print(get_dict_value(sample_dict, "name"))  # Output: John

# Retrieve a value that does not exist
print(get_dict_value(sample_dict, "city"))  # Output: None

# Retrieve a value that does not exist with a custom default
print(get_dict_value(sample_dict, "city", "New York"))  # Output: New York
```
