## Exercise
Create a function to check if a given string is a Sotropic number or not

## Reference Solution
```python
def is_sotropic(num):
    """
    Checks if a given number is a Sotropic number.

    Args:
        num (int): The number to check.

    Returns:
        bool: True if the number is Sotropic, False otherwise.
    """
    # Convert the number to a string to easily access its last digits
    num_str = str(num)
    # Calculate the square of the number
    square = num ** 2
    # Convert the square to a string
    square_str = str(square)
    # Check if the last digits of the square match the number
    return square_str.endswith(num_str)


# Example usage:
if __name__ == "__main__":
    numbers_to_check = [5, 6, 25, 76]
    for num in numbers_to_check:
        if is_sotropic(num):
            print(f"{num} is a Sotropic number.")
        else:
            print(f"{num} is not a Sotropic number.")
```
