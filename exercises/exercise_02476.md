## Exercise
Write a function that generates all possible anagrams of a given string.

## Reference Solution
```python
import itertools

def generate_anagrams(input_string):
    """
    Generates all possible anagrams of a given string.

    Args:
        input_string (str): The input string for which anagrams are to be generated.

    Returns:
        list: A list of all possible anagrams of the input string.
    """
    # Generate all permutations of the input string
    anagrams = [''.join(p) for p in itertools.permutations(input_string)]
    
    # Return the list of anagrams
    return anagrams

# Example usage
if __name__ == "__main__":
    input_str = "abc"
    anagrams = generate_anagrams(input_str)
    print("Anagrams of '{}'".format(input_str))
    for anagram in anagrams:
        print(anagram)
```
