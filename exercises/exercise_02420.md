## Exercise
Create a function that uses polymorphism to perform different actions. Then, call this function with an input value and print the result.

## Reference Solution
```python
def process_data(data):
    """
    Process data based on its type.

    Args:
        data (object): Input data of any type.

    Returns:
        str: Result of data processing.
    """
    if isinstance(data, int) or isinstance(data, float):
        # If data is a number, double its value
        return f"Doubled value: {data * 2}"
    elif isinstance(data, str):
        # If data is a string, convert it to uppercase
        return f"Uppercase string: {data.upper()}"
    elif isinstance(data, list) or isinstance(data, tuple):
        # If data is a list or tuple, return its length
        return f"Length: {len(data)}"
    else:
        # For any other data type, return a default message
        return "Unsupported data type"

# Example usage
if __name__ == "__main__":
    # Test with different input values
    inputs = [
        5,  # integer
        3.14,  # float
        "hello",  # string
        [1, 2, 3],  # list
        (4, 5, 6),  # tuple
        {"a": 1, "b": 2}  # dictionary (unsupported type)
    ]

    for data in inputs:
        result = process_data(data)
        print(f"Input: {data}, Result: {result}")
```
