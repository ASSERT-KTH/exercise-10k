## Exercise
Write a function that calculates the sum of all numbers in a given list.

## Reference Solution
```python
def calculate_sum(num_list):
    """
    This function calculates the sum of all numbers in a given list.
    
    Args:
        num_list (list): A list of numbers.
    
    Returns:
        int or float: The sum of all numbers in the list.
    """
    return sum(num_list)

# Example usage:
def main():
    numbers = [1, 2, 3, 4, 5]
    result = calculate_sum(numbers)
    print("The sum of the list is:", result)

if __name__ == "__main__":
    main()
```
