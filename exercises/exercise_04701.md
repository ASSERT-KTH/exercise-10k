## Exercise
Create a dictionary with 5 key-value pairs where the keys are names and the values are ages

## Reference Solution
```python
people = {
    "John": 25,
    "Emily": 30,
    "Michael": 35,
    "Sarah": 20,
    "William": 40
}
```

```python
# Accessing a value
print(people["John"])  # Output: 25

# Updating a value
people["John"] = 26
print(people["John"])  # Output: 26

# Adding a new key-value pair
people["James"] = 45
print(people)  # Output: {'John': 26, 'Emily': 30, 'Michael': 35, 'Sarah': 20, 'William': 40, 'James': 45}
```
