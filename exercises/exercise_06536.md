## Exercise
Write a function that returns a list of authors from a list of books.

## Reference Solution
```python
def extract_authors(books):
    """
    Extracts a list of unique authors from a list of books.

    Args:
        books (list): A list of dictionaries, where each dictionary represents a book with 'title' and 'author' keys.

    Returns:
        list: A list of unique authors.
    """
    authors = set()
    for book in books:
        authors.add(book['author'])
    return list(authors)

# Example usage:
books = [
    {'title': 'Book 1', 'author': 'Author A'},
    {'title': 'Book 2', 'author': 'Author B'},
    {'title': 'Book 3', 'author': 'Author A'},
    {'title': 'Book 4', 'author': 'Author C'}
]

authors = extract_authors(books)
print(authors)  # Output: ['Author A', 'Author B', 'Author C']
```
