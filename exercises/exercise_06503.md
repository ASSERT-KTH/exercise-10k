## Exercise
Swap the values of two variables without using a temporary variable.

## Reference Solution
```python
def swap_variables_arithmetic(a, b):
    """
    Swap the values of two variables using arithmetic operators.

    Args:
        a (int): The first variable.
        b (int): The second variable.

    Returns:
        tuple: The swapped values.
    """
    print("Before swap: a =", a, ", b =", b)
    a = a + b  # Add the values of a and b
    b = a - b  # Subtract the original value of b from the sum
    a = a - b  # Subtract the new value of b from the sum
    print("After swap: a =", a, ", b =", b)
    return a, b

# Example usage:
swap_variables_arithmetic(5, 10)
```

def swap_variables_tuple(a, b):
    """
    Swap the values of two variables using tuple packing and unpacking.

    Args:
        a (int): The first variable.
        b (int): The second variable.

    Returns:
        tuple: The swapped values.
    """
    print("Before swap: a =", a, ", b =", b)
    a, b = b, a  # Pack and unpack the values in reverse order
    print("After swap: a =", a, ", b =", b)
