## Exercise
Clean a dictionary by removing empty values and keys.

## Reference Solution
```python
def clean_dictionary(input_dict):
    """
    Clean a dictionary by removing empty values and keys.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        dict: The cleaned dictionary.
    """
    return {key: value for key, value in input_dict.items() if key and value}

# Example usage
input_dict = {
    "key1": "value1",
    "key2": "",
    "key3": None,
    "key4": "value4",
    "": "value5",
    "key6": []
}

cleaned_dict = clean_dictionary(input_dict)
print("Input Dictionary:", input_dict)
print("Cleaned Dictionary:", cleaned_dict)
```
