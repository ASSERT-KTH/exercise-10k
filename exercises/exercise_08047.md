## Exercise
Write a program that uses an if/else statement to respond to user input based on the intent of the message.

## Reference Solution
```python
def respond_to_message(message):
    # Intent dictionary to map intents to responses
    intent_dict = {
        "greeting": ["hello", "hi", "hey"],
        "goodbye": ["bye", "see you later", "goodbye"],
        "thanks": ["thanks", "thank you"]
    }

    # Convert message to lowercase for case-insensitive comparison
    message = message.lower()

    # Check if message is a greeting
    if any(intent in message for intent in intent_dict["greeting"]):
        return "Hello! How are you today?"

    # Check if message is a goodbye
    elif any(intent in message for intent in intent_dict["goodbye"]):
        return "See you later! Have a great day!"

    # Check if message is thanks
    elif any(intent in message for intent in intent_dict["thanks"]):
        return "You're welcome!"

    # If none of the above intents match, return a default response
    else:
        return "I'm not sure I understand what you mean. Could you please rephrase?"

def main():
    print("Welcome to the chatbot! Type 'quit' to exit.")
    while True:
        message = input("You: ")
        if message.lower() == "quit":
            break
        response = respond_to_message(message)
        print("Chatbot: ", response)

if __name__ == "__main__":
    main()
```
