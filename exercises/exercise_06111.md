## Exercise
Write a program that takes an array of integers and a rotation value as input, and returns a new array containing the elements rotated by the specified amount.

## Reference Solution
```python
function rotate(arr, numRotations) {
  return arr.slice(numRotations).concat(arr.slice(0, numRotations));
}

// Example usage:
const arr = [1, 2, 3, 4, 5];
console.log(rotate(arr, 2)); // Output: [3, 4, 5, 1, 2]
```
