## Exercise
Write a function that takes a string as input and returns the string with all duplicate characters removed.

## Reference Solution
```python
def remove_duplicates(input_string):
    """
    Removes all duplicate characters from a given string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The string with all duplicate characters removed.
    """
    return ''.join(dict.fromkeys(input_string))

# Example usage
if __name__ == "__main__":
    input_string = "Hello, World!"
    result = remove_duplicates(input_string)
    print("Input String:", input_string)
    print("Result:", result)
```
