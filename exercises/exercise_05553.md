## Exercise
Write a program that prints the first 20 numbers in the Pell sequence using a while loop.

## Reference Solution
```python
# Initialize the first two numbers in the Pell sequence
a, b = 0, 1

# Print the first two numbers
print(a)
print(b)

# Initialize a counter
i = 2

# Print the next 18 numbers in the sequence
while i < 20:
    # Calculate the next number in the sequence
    a, b = b, 2 * b + a
    # Print the next number
    print(b)
    # Increment the counter
    i += 1
```
