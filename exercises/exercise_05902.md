## Exercise
Write a function that takes a string as input and returns the string in all uppercase letters

## Reference Solution
```python
def convert_to_uppercase(input_string: str) -> str:
    """
    This function takes a string as input and returns the string in all uppercase letters.

    Args:
        input_string (str): The input string to be converted.

    Returns:
        str: The input string in all uppercase letters.
    """
    return input_string.upper()

# Example usage:
def main():
    input_str = "Hello, World!"
    print(convert_to_uppercase(input_str))  # Output: "HELLO, WORLD!"

if __name__ == "__main__":
    main()
```
