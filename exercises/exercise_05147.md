## Exercise
Get a list of all the values in a dictionary

## Reference Solution
```python
def get_dict_values(dictionary):
    """Return a list of dictionary values."""
    return list(dictionary.values())

# Example usage:
my_dict = {"a": 1, "b": 2, "c": 3}
values = get_dict_values(my_dict)
print(values)  # Output: [1, 2, 3]
```

def get_sorted_dict_values(dictionary):
    """Return a sorted list of dictionary values."""
    return sorted(dictionary.values())

# Example usage:
my_dict = {"a": 3, "b": 1, "c": 2}
