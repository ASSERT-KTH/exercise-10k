## Exercise
Write a program that takes an array and a reduction function as input, and returns the result of applying the reduction function to the array.

## Reference Solution
```python
def reduce(array, f):
    if array:
        result = array[0]
        for element in array[1:]:
            result = f(result, element)
        return result
    else:
        raise TypeError('reduce() of empty sequence with no initial value')
```
