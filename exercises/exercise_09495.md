## Exercise
Write a program that asks the user for their first and last names, then prints out a greeting message with their full name.

## Reference Solution
```python
def get_user_name():
    """Gets the user's first and last names."""
    first_name = input("Please enter your first name: ")
    last_name = input("Please enter your last name: ")
    return first_name, last_name

def print_greeting(first_name, last_name):
    """Prints out a greeting message with the user's full name."""
    full_name = first_name + " " + last_name
    print("Hello, " + full_name + "!")

def main():
    first_name, last_name = get_user_name()
    print_greeting(first_name, last_name)

if __name__ == "__main__":
    main()
```
