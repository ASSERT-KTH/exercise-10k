## Exercise
Extract the first character of a given string and print it out.

## Reference Solution
```python
def extract_first_char(input_string):
    """
    Extracts the first character of a given string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The first character of the input string.
    """
    if input_string:
        return input_string[0]
    else:
        return None

# Example usage:
input_str = "Hello, World!"
first_char = extract_first_char(input_str)

if first_char:
    print(f"The first character is: {first_char}")
else:
    print("The input string is empty.")
```
