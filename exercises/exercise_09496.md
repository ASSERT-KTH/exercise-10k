## Exercise
Write a program that prints a pattern of numbers using loops. The pattern should consist of numbers from 1 to n, where n is the input provided by the user. The pattern should be printed in the following format: 1, 2, 3, ..., n-2, n-1, n, n-1, n-2, ..., 3, 2, 1

## Reference Solution
```python
# Import the sys module
import sys

# Get the input from the user
n = int(input("Enter a number: "))

# Check if the input is valid
if n < 1:
    print("Invalid input. Please enter a positive integer.")
    sys.exit()

# Print the pattern
for i in range(1, n+1):
    print(i, end=', ')

for i in range(n-1, 0, -1):
    if i == 1:
        print(i)
    else:
        print(i, end=', ')
```
