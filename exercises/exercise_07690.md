## Exercise
Write a function that calculates the absolute value of a number.

## Reference Solution
```python
def calculate_absolute_value(number):
    """
    This function calculates the absolute value of a given number.

    Args:
        number (int or float): The input number.

    Returns:
        int or float: The absolute value of the input number.
    """

    # First, we check if the input is a number. If it's not, we raise a TypeError.
    if not isinstance(number, (int, float)):
        raise TypeError("Input must be a number.")

    # Now, we use the built-in abs() function in Python to calculate the absolute value.
    # The abs() function returns the absolute value of a number.
    absolute_value = abs(number)

    # Finally, we return the calculated absolute value.
    return absolute_value

# Example usage:
print(calculate_absolute_value(-5))  # Output: 5
print(calculate_absolute_value(10))  # Output: 10
```
