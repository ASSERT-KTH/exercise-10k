## Exercise
Write a program that takes two arrays of integers as input, and returns a new array containing only the elements that are in exactly one of the two arrays.

## Reference Solution
```python
def sym(*args):
    if len(args) == 1:
        return args[0]
    elif len(args) == 2:
        return [x for x in args[0] if x not in args[1]] + [x for x in args[1] if x not in args[0]]
    else:
        return sym(args[0], sym(*args[1:]))

# Test cases
print(sym([1, 2, 3], [5, 2, 1, 4]))  # Output: [3, 5, 4]
print(sym([1, 2, 3, 3], [5, 2, 1, 4]))  # Output: [3, 3, 5, 4]
print(sym([1, 2, 3], [5, 2, 1, 4, 4]))  # Output: [3, 5, 4, 4]
```
