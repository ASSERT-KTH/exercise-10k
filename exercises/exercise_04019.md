## Exercise
Write a program that uses nested for loops to print a 5x5 circle pattern of asterisks (*).

## Reference Solution
```python
import math

def print_circle_pattern(size):
    """
    Prints a circle pattern of asterisks (*) of the given size.

    Args:
        size (int): The size of the circle pattern.
    """

    # Calculate the center of the circle
    center = size // 2

    # Iterate over each row in the pattern
    for y in range(size):
        # Iterate over each column in the pattern
        for x in range(size):
            # Calculate the distance from the current point to the center of the circle
            distance = math.sqrt((x - center) ** 2 + (y - center) ** 2)

            # If the distance is less than or equal to the radius, print an asterisk
            if distance <= center:
                print("*", end=" ")
            else:
                # Otherwise, print a space
                print(" ", end=" ")
        # Move to the next line
        print()

# Example usage
print_circle_pattern(5)
```
