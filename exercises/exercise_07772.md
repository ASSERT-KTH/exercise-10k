## Exercise
Create a program that takes a string input from the user and outputs the string with each word capitalized and in reverse order. For example, if the input is 'hello world', the output should be 'World Hello'. The program should handle any number of words in the input string.

## Reference Solution
```python
import sys

def main():
    string = sys.stdin.readline().strip()
    words = string.split()
    capitalized_words = [word.capitalize() for word in words]
    reversed_words = capitalized_words[::-1]
    output = ' '.join(reversed_words)
    sys.stdout.write(output + '\n')

if __name__ == "__main__":
    main()
```
