## Exercise
Create a function that takes two arguments, adds them together, and returns the result. Then, call this function with two input numbers and print the sum.

## Reference Solution
```python
def add_numbers(a, b):
    """
    Adds two numbers together.

    Args:
        a (int): The first number.
        b (int): The second number.

    Returns:
        int: The sum of a and b.
    """
    return a + b

# Call the function with two input numbers and print the sum.
num1 = 5
num2 = 7
result = add_numbers(num1, num2)
print(f"The sum of {num1} and {num2} is: {result}")
```
