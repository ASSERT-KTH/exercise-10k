## Exercise
Write a program that takes a number as input and returns its floor.

## Reference Solution
```python
import math

def get_floor(num):
    """
    Returns the floor of a given number.

    Args:
        num (int or float): The input number.

    Returns:
        int: The floor of the input number.
    """
    return math.floor(num)

def main():
    # Example usage
    num = float(input("Enter a number: "))
    floor = get_floor(num)
    print("The floor of", num, "is", floor)

if __name__ == "__main__":
    main()
```
