## Exercise
Write a program that takes an array and a condition as input, and returns a new array with only the elements that satisfy the condition.

## Reference Solution
```python
def filter_elements(array, condition):
    new_array = []
    for a in array:
        if condition(a):
            new_array.append(a)
    return new_array
```
