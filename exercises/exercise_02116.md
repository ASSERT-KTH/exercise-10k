## Exercise
Write a function to retrieve the value associated with a given key from a dictionary, providing a default value if the key is not present

## Reference Solution
```python
def get_key_value(dictionary, key, default=None):
    """
    Retrieves the value associated with a given key from a dictionary.
    
    If the key is not present in the dictionary, it returns the default value.
    
    Args:
        dictionary (dict): The dictionary to search in.
        key: The key to search for.
        default: The default value to return if the key is not found. Defaults to None.
    
    Returns:
        The value associated with the key, or the default value if the key is not present.
    """

    # First, we check if the dictionary is actually a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("The first argument must be a dictionary")

    # Then, we try to get the value from the dictionary
    # The get method of a dictionary returns the value for the given key if it exists in the dictionary
    # If the key does not exist, it returns the default value
    return dictionary.get(key, default)

# Example usage:
my_dict = {"name": "John", "age": 30}
print(get_key_value(my_dict, "name"))  # Outputs: John
print(get_key_value(my_dict, "city", "New York"))  # Outputs: New York
```
