## Exercise
Write a function that takes a variable number of arguments and returns their sum, using function arity

## Reference Solution
```python
def sum_args(*args):
    """
    Returns the sum of all input arguments.
    
    Args:
        *args: A variable number of numeric arguments.
    
    Returns:
        The sum of all input arguments.
    """
    return sum(args)

# Example usage:
print(sum_args(1, 2, 3, 4, 5))  # Output: 15
```

```python
fn sum_args(args: &[i32]) -> i32 {
    """
    Returns the sum of all input arguments.
    
    Args:
        args: A slice of integers.
    
    Returns:
        The sum of all input arguments.
    """
    args.iter().sum::<i32>()
}

// Example usage:
fn main() {
    let args = [1, 2, 3, 4, 5];
    println!("{}", sum_args(&args));  // Output: 15
}
```

using System;
using System.Linq;

public class Program
{
    public static int SumArgs(params int[] args)
    {
        """
        Returns the sum of all input arguments.
        
        Args:
            args: A variable number of integers.
        
        Returns:
            The sum of all input arguments.
        """
        return args.Sum();
    }

    public static void Main()
    {
        Console.WriteLine(SumArgs(1, 2, 3, 4, 5));  // Output: 15
    }
}
