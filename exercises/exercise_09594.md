## Exercise
Write a program that reads a user-provided string and prints out its length and the string in uppercase

## Reference Solution
```python
def main():
    # Read user-provided string
    user_string = input("Please enter a string: ")

    # Calculate the length of the string
    string_length = len(user_string)

    # Convert the string to uppercase
    uppercase_string = user_string.upper()

    # Print out the length and uppercase string
    print("Length of the string: ", string_length)
    print("String in uppercase: ", uppercase_string)

if __name__ == "__main__":
    main()
```
