## Exercise
Write a program to convert time from 12-hour format to 24-hour format using conditional statements

## Reference Solution
```python
def convert_to_24_hour_format(time_str):
    """
    Converts time from 12-hour format to 24-hour format.
    
    Args:
        time_str (str): Time in 12-hour format (e.g., "08:30 PM")
    
    Returns:
        str: Time in 24-hour format (e.g., "20:30")
    """
    time_str = time_str.upper()  # Convert to uppercase for simplicity
    components = time_str.split()  # Split time and AM/PM
    
    time = components[0].split(":")  # Split hours and minutes
    hours = int(time[0])
    minutes = time[1]
    
    if components[1] == "PM" and hours != 12:
        # Add 12 to hours if PM and not 12
        hours += 12
    elif components[1] == "AM" and hours == 12:
        # Set hours to 0 if AM and 12
        hours = 0
    
    # Format the time as a string
    time_str = "{:02d}:{:02d}".format(hours, minutes)
    
    return time_str

def main():
    # Test the function
    time_str = input("Enter time in 12-hour format (e.g., 08:30 PM): ")
    print("Time in 24-hour format:", convert_to_24_hour_format(time_str))

if __name__ == "__main__":
    main()
```
