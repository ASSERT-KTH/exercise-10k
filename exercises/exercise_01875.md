## Exercise
Write a program to unzip a list of pairs into two separate lists and print the resulting lists.

## Reference Solution
```python
def unzip_pairs(pairs):
    """
    Unzip a list of pairs into two separate lists.

    Args:
        pairs (list): A list of pairs.

    Returns:
        tuple: Two separate lists.
    """
    list1, list2 = zip(*pairs)
    return list(list1), list(list2)


def main():
    # Define a list of pairs
    pairs = [(1, 'a'), (2, 'b'), (3, 'c')]

    # Unzip the list of pairs
    list1, list2 = unzip_pairs(pairs)

    # Print the resulting lists
    print("List 1:", list1)
    print("List 2:", list2)


if __name__ == "__main__":
    main()
```
