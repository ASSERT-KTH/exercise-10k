## Exercise
Create a program to retrieve the number of key-value pairs in a given dictionary.

## Reference Solution
```python
def get_key_value_pairs(dictionary):
    """
    Retrieves the number of key-value pairs in a given dictionary.

    Args:
        dictionary (dict): The dictionary to retrieve key-value pairs from.

    Returns:
        int: The number of key-value pairs in the dictionary.
    """
    return len(dictionary)

def main():
    # Create a sample dictionary
    sample_dict = {"apple": 1, "banana": 2, "cherry": 3}

    # Retrieve the number of key-value pairs
    num_pairs = get_key_value_pairs(sample_dict)

    # Print the result
    print(f"The dictionary has {num_pairs} key-value pairs.")

if __name__ == "__main__":
    main()
```
