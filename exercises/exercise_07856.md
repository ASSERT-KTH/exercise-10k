## Exercise
Create a program that determines whether a person is eligible to vote based on their age using if/else statements.

## Reference Solution
```python
def check_voting_eligibility(age):
    """
    Checks if a person is eligible to vote based on their age.

    Args:
        age (int): The age of the person.

    Returns:
        bool: True if the person is eligible to vote, False otherwise.
    """
    if age >= 18:
        return True
    else:
        return False

def main():
    age = int(input("Enter your age: "))
    if check_voting_eligibility(age):
        print("You are eligible to vote.")
    else:
        print("You are not eligible to vote.")

if __name__ == "__main__":
    main()
```
