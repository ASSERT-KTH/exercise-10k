## Exercise
Write a function to reduce a dictionary to a single value. For example, sum up the values of the dictionary.

## Reference Solution
```python
def reduce_dict(dictionary, reduction_function=sum):
    """
    Reduces a dictionary to a single value using a given reduction function.

    Args:
        dictionary (dict): The dictionary to reduce.
        reduction_function (function): The function to use for reduction. Defaults to sum.

    Returns:
        The reduced value.
    """
    return reduction_function(dictionary.values())

# Example usage:
if __name__ == "__main__":
    example_dict = {"a": 1, "b": 2, "c": 3}
    reduced_value = reduce_dict(example_dict)
    print(f"Sum of dictionary values: {reduced_value}")

    # Using a custom reduction function (e.g., max)
    custom_reduction_function = max
    custom_reduced_value = reduce_dict(example_dict, custom_reduction_function)
    print(f"Max of dictionary values: {custom_reduced_value}")
```
