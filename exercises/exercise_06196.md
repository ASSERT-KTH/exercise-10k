## Exercise
Write a function that takes a string and a substring as input and returns True if the string ends with the substring, False otherwise.

## Reference Solution
```python
def check_string_ends_with_substring(input_string, substring):
    """
    This function checks if the input string ends with the given substring.
    
    Args:
        input_string (str): The main string to be checked.
        substring (str): The substring to be searched at the end of the main string.
    
    Returns:
        bool: True if the string ends with the substring, False otherwise.
    """

    # First, we need to check if both the input string and the substring are not None
    if input_string is None or substring is None:
        raise TypeError("Both input string and substring must not be None")

    # Then, we need to check if both the input string and the substring are strings
    if not isinstance(input_string, str) or not isinstance(substring, str):
        raise TypeError("Both input string and substring must be strings")

    # Now, we can use Python's built-in str.endswith() method to check if the string ends with the substring
    # This method returns True if the string ends with the substring, False otherwise
    return input_string.endswith(substring)
```
