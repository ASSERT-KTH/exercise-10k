## Exercise
Create a program that checks whether a given number is positive, negative, or zero using conditional statements.

## Reference Solution
```python
def check_number(num):
    """
    Checks whether a given number is positive, negative, or zero.

    Args:
        num (int): The number to check.

    Returns:
        str: A string indicating whether the number is positive, negative, or zero.
    """
    if num > 0:
        return "Positive"
    elif num < 0:
        return "Negative"
    else:
        return "Zero"

def main():
    # Test the function
    numbers = [10, -5, 0, 7, -3]
    for num in numbers:
        print(f"The number {num} is {check_number(num)}")

if __name__ == "__main__":
    main()
```
