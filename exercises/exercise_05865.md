## Exercise
Determine whether a given number is even or odd.

## Reference Solution
```python
def is_even(num):
    """Return True if the number is even, False otherwise."""
    return num % 2 == 0

# Test the function
num = int(input("Enter a number: "))
if is_even(num):
    print(f"{num} is even.")
else:
    print(f"{num} is odd.")
```
