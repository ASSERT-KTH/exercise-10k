## Exercise
Write a function that takes a list and a target value as input, and returns the index of the first occurrence of the target value in the list, or -1 if the target value is not found.

## Reference Solution
```python
def searchInsert(nums, target):
    if target in nums:
        return nums.index(target)
    else:
        nums.append(target)
        nums.sort()
        return nums.index(target)

# Example usage:
nums = [1, 3, 5, 6]
target = 5
print(searchInsert(nums, target))  # Output: 2
```
