## Exercise
Write a program that checks if a given number is positive, negative, or zero. The program should ask the user for a number and then print out whether the number is positive, negative, or zero.

## Reference Solution
```python
# Ask the user for a number
num = float(input("Enter a number: "))

# Check if the number is positive, negative, or zero
if num > 0:
    print("The number is positive.")
elif num < 0:
    print("The number is negative.")
else:
    print("The number is zero.")
```
