## Exercise
Create a program that takes a person's name and age as input and generates a personalized greeting.

## Reference Solution
```python
def generate_greeting(name: str, age: int) -> str:
    """
    Generates a personalized greeting based on the person's age group.

    Args:
        name (str): The person's name.
        age (int): The person's age.

    Returns:
        str: A personalized greeting.
    """

    if age < 13:
        return f"Hello, young {name}!"
    elif 13 <= age < 20:
        return f"What's up, {name}?"
    elif 20 <= age < 65:
        return f"Hello, {name}! How's your day going?"
    else:
        return f"Hello, esteemed {name}!"

def main():
    name = input("Please enter your name: ")
    age = int(input("Please enter your age: "))

    greeting = generate_greeting(name, age)
    print(greeting)

if __name__ == "__main__":
    main()
```
