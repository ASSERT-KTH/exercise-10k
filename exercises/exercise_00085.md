## Exercise
Given a list of strings, find the longest common prefix among all the strings

## Reference Solution
```python
def longest_common_prefix(strs):
    """
    Finds the longest common prefix among all the strings in the given list.

    Args:
    strs (list): A list of strings.

    Returns:
    str: The longest common prefix.
    """
    if not strs:
        return ""

    # Find the shortest string in the list
    shortest_str = min(strs, key=len)

    # Iterate over the characters in the shortest string
    for i, char in enumerate(shortest_str):
        # Check if the character is the same in all strings
        for string in strs:
            if string[i] != char:
                # If the character is not the same, return the common prefix
                return shortest_str[:i]

    # If all characters are the same, return the shortest string
    return shortest_str

# Example usage
strs = ["flower", "flow", "flight"]
print(longest_common_prefix(strs))  # Output: "fl"
```
