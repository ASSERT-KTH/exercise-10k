## Exercise
Write a program that takes a string as input and prints its length, and then prints the string in uppercase and lowercase.

## Reference Solution
```python
# This program takes a string as input, prints its length, 
# and then prints the string in uppercase and lowercase.

string = input("Enter a string: ")
l = len(string)
print("The length of the string is: ", l)
print("The uppercase version of the string is: ", string.upper())
print("The lowercase version of the string is: ", string.lower())
```
