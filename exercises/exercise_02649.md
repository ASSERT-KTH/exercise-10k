## Exercise
Write a function to get a list of values from a dictionary

## Reference Solution
```python
def get_dict_values(dictionary):
    """
    Returns a list of values from a dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        list: A list of dictionary values.
    """
    return list(dictionary.values())

# Example usage
if __name__ == "__main__":
    example_dict = {"key1": "value1", "key2": "value2", "key3": "value3"}
    print(get_dict_values(example_dict))
```
