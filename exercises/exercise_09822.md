## Exercise
Create a program that generates a multiplication table for a given range of numbers. The program should take two integer inputs, 'start' and 'end', and print a multiplication table for numbers from 'start' to 'end'. The table should include the multiplication results for each number in the range with all other numbers in the range.

## Reference Solution
```python
# Header
# Author: [Your Name]
# Email: [Your Email]
# Date: [Current Date]
# Purpose: Generate a multiplication table for a given range of numbers.

# Importing Libraries
import numpy as np

# Defining the Function
def multiplication_table(start, end):
    # Creating the Table
    table = np.zeros((end - start + 1, end - start + 1), dtype=int)

    # Filling the Table
    for i in range(start, end + 1):
        for j in range(start, end + 1):
            table[i - start, j - start] = i * j

    # Printing the Table
    print(table)

# Main Function
def main():
    start = int(input("Enter the start of the range: "))
    end = int(input("Enter the end of the range: "))
    multiplication_table(start, end)

# Running the Main Function
if __name__ == "__main__":
    main()
```
