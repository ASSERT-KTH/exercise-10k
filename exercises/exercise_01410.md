## Exercise
Create a program that checks if a given age is valid for voting. If the age is 18 or above, print 'Valid', otherwise print 'Invalid'.

## Reference Solution
```python
def check_voting_age(age):
    """Checks if the given age is valid for voting."""
    if age >= 18:
        print('Valid')
    else:
        print('Invalid')

# Example usage
def main():
    age = int(input("Enter your age: "))
    check_voting_age(age)

if __name__ == "__main__":
    main()
```
