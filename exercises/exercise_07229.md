## Exercise
Create a program that calculates the area and perimeter of a rectangle using functions. The program should take the length and width of the rectangle as input, and then call two separate functions to calculate the area and perimeter, respectively.

## Reference Solution
```python
# Define the functions
def calcular_area(largo, ancho):
    return largo * ancho

def calcular_perimetro(largo, ancho):
    return 2 * (largo + ancho)

# Get user input
largo = int(input("Ingrese el largo: "))
ancho = int(input("Ingrese el ancho: "))

# Calculate area and perimeter
area = calcular_area(largo, ancho)
perimetro = calcular_perimetro(largo, ancho)

# Print results
print("El área del rectángulo es:", area)
print("El perímetro del rectángulo es:", perimetro)
```
