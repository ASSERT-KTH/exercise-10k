## Exercise
Write a function that removes all consonants from a given string.

## Reference Solution
```python
def remove_consonants(input_string: str) -> str:
    """
    Removes all consonants from a given string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The input string with all consonants removed.
    """
    vowels = 'aeiouAEIOU'
    return ''.join([char for char in input_string if char in vowels or not char.isalpha()])

# Example usage:
def main():
    input_str = "Hello, World!"
    print("Original string:", input_str)
    print("String without consonants:", remove_consonants(input_str))

if __name__ == "__main__":
    main()
```
