## Exercise
Create a function that applies a given function to each element of a list.

## Reference Solution
```python
def apply_to_each(lst, func):
    """
    Applies a given function to each element of a list.

    Args:
        lst (list): The input list.
        func (function): The function to be applied.

    Returns:
        list: The resulting list after applying the function.
    """
    return [func(element) for element in lst]

# Example usage:
def square(x):
    return x ** 2

numbers = [1, 2, 3, 4, 5]
result = apply_to_each(numbers, square)
print(result)  # Output: [1, 4, 9, 16, 25]
```

def apply_to_each(lst, func):
    return list(map(func, lst))

# Example usage:
numbers = [1, 2, 3, 4, 5]
result = apply_to_each(numbers, lambda x: x ** 2)
print(result)  # Output: [1, 4, 9, 16, 25]
