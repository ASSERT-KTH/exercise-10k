## Exercise
Write a program that searches for a target element in a sorted array using a basic algorithm such as linear search or binary search. The program should take as input a sorted array and a target element, and output the index of the target element if it is found in the array, or -1 if it is not found.

## Reference Solution

#include <iostream>

int binary_search(int arr[], int n, int target) {
    int left = 0;
    int right = n - 1;

    while (left <= right) {
        int mid = left + (right - left) / 2;

        if (arr[mid] == target) {
            return mid;
        } else if (arr[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }

    return -1;
}

int main() {
    int arr[] = {1, 2, 3, 4, 5, 6, 7, 8, 9};
    int n = sizeof(arr) / sizeof(arr[0]);
    int target = 5;

    int result = binary_search(arr, n, target);

    if (result != -1) {
        std::cout << "Target element found at index " << result << std::endl;
    } else {
        std::cout << "Target element not found in the array" << std::endl;
    }

    return 0;
}
