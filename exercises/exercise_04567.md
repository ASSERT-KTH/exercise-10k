## Exercise
Write a function that takes a dictionary and a function, then applies the function to all values in the dictionary and returns a single result

## Reference Solution
```python
def apply_function_to_dict_values(dictionary, func, reduction_func=None):
    """
    Applies a function to all values in a dictionary.

    Args:
        dictionary (dict): The dictionary to apply the function to.
        func (function): The function to apply to each value.
        reduction_func (function, optional): The function to use to reduce the results. Defaults to None.

    Returns:
        result: The result of applying the function to all values.
    """
    results = [func(value) for value in dictionary.values()]
    
    if reduction_func is not None:
        return reduction_func(results)
    elif len(results) == 1:
        return results[0]
    else:
        return results

# Example usage
def square(x):
    return x ** 2

def sum_results(results):
    return sum(results)

example_dict = {'a': 1, 'b': 2, 'c': 3}

result_with_reduction = apply_function_to_dict_values(example_dict, square, sum_results)
result_without_reduction = apply_function_to_dict_values(example_dict, square)

print("Result with reduction:", result_with_reduction)
print("Result without reduction:", result_without_reduction)
```
