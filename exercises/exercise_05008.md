## Exercise
Write a program that accesses the first character of the string 'Hello'. Print the character to the console.

## Reference Solution
```python
def print_first_character(input_string):
    """
    Prints the first character of the input string.

    Args:
        input_string (str): The input string.
    """
    # Check if the string is not empty
    if input_string:
        # Access the first character using indexing
        first_char = input_string[0]
        print(first_char)
    else:
        print("The input string is empty.")

# Example usage
input_str = 'Hello'
print_first_character(input_str)
```
