## Exercise
Write a program that takes a string as input from the user and prints the number of characters in the string.

## Reference Solution
```python
def count_characters():
    """
    Asks for user input and prints the number of characters in the input string.
    """
    user_input = input("Please enter a string: ")
    character_count = len(user_input)
    print(f"The string has {character_count} characters.")

def main():
    count_characters()

if __name__ == "__main__":
    main()
```
