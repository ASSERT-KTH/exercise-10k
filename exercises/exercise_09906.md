## Exercise
Write a program that checks whether a given number is palindromic (i.e. reads the same forwards and backwards).

## Reference Solution
```python
def is_palindromic(n):
    """
    Checks if a given number is palindromic.

    Args:
        n (int): The number to check.

    Returns:
        bool: True if the number is palindromic, False otherwise.
    """
    return str(n) == str(n)[::-1]

def main():
    # Example usage
    num = int(input("Enter a number: "))
    if is_palindromic(num):
        print(f"{num} is a palindromic number.")
    else:
        print(f"{num} is not a palindromic number.")

if __name__ == "__main__":
    main()
```
