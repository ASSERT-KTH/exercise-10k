## Exercise
Write a function to join multiple strings together with a delimiter.

## Reference Solution
```python
def join_strings(delimiter, *strings):
    """
    Join multiple strings together with a delimiter.

    Args:
        delimiter (str): The delimiter to use for joining the strings.
        *strings (str): The strings to join together.

    Returns:
        str: The joined string.
    """
    return delimiter.join(strings)

# Example usage:
if __name__ == "__main__":
    delimiter = ","
    strings = ["apple", "banana", "cherry"]
    joined_string = join_strings(delimiter, *strings)
    print(joined_string)  # Output: apple,banana,cherry
```
