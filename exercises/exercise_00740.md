## Exercise
Write a program that takes a number and a percentage as input and outputs the result of the percentage calculation.

## Reference Solution
```python
def calculate_percentage(number, percentage):
    """
    Calculate a percentage of a given number.

    Args:
        number (float): The number to calculate the percentage of.
        percentage (float): The percentage to calculate.

    Returns:
        float: The result of the percentage calculation.
    """
    # Calculate the percentage
    result = (percentage / 100) * number
    
    return result


def main():
    # Get the number and percentage from the user
    number = float(input("Enter a number: "))
    percentage = float(input("Enter a percentage: "))

    # Calculate and display the result
    result = calculate_percentage(number, percentage)
    print(f"{percentage}% of {number} is {result}")


if __name__ == "__main__":
    main()
```
