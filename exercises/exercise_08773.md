## Exercise
Write a function to count the total number of digits in a given string.

## Reference Solution
```python
def count_digits(string):
    """
    This function takes a string as input and returns the total number of digits in the string.

    Parameters:
    string (str): The input string to be processed.

    Returns:
    int: The total number of digits in the string.
    """

    # Initialize a counter variable to store the total number of digits
    total_digits = 0
    
    # Iterate over each character in the string
    for char in string:
        
        # Check if the character is a digit using the isdigit() method
        if char.isdigit():
            
            # If the character is a digit, increment the counter
            total_digits += 1
    
    # Return the total number of digits found in the string
    return total_digits

# Example usage:
print(count_digits("Hello123World456"))  # Output: 6
```
