## Exercise
Write a program that reads a user's name and age from the console, then prints out a greeting message with the user's name and a message indicating whether they are an adult or not based on their age.

ano = int(input('Em que ano você nasceu? '))

adulto = ano > 2004

print('Olá {}, você é adulto? {}'.format(nome, adulto))
