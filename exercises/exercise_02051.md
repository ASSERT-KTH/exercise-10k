## Exercise
Create a function to access a nested value in a dictionary.

## Reference Solution
```python
def access_nested_value(dictionary, keys):
    """
    Access a nested value in a dictionary.

    Args:
        dictionary (dict): The dictionary to access.
        keys (list): A list of keys to navigate through the dictionary.

    Returns:
        The value associated with the last key in the list, or None if the key is not found.
    """
    value = dictionary
    for key in keys:
        if isinstance(value, dict) and key in value:
            value = value[key]
        else:
            return None
    return value

# Example usage
if __name__ == "__main__":
    nested_dict = {
        "a": 1,
        "b": {
            "c": 2,
            "d": {
                "e": 3
            }
        }
    }

    print(access_nested_value(nested_dict, ["b", "d", "e"]))  # Output: 3
    print(access_nested_value(nested_dict, ["b", "c"]))  # Output: 2
    print(access_nested_value(nested_dict, ["a"]))  # Output: 1
    print(access_nested_value(nested_dict, ["f"]))  # Output: None
    print(access_nested_value(nested_dict, ["b", "f"]))  # Output: None
```
