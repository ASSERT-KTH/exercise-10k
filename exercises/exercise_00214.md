## Exercise
Write a function that prompts the user for their age and returns it.

## Reference Solution
```python
def get_user_age():
    """
    Prompts the user for their age and returns it as an integer.
    
    Returns:
        int: The user's age.
    """
    while True:
        try:
            age = int(input("Please enter your age: "))
            if age < 0:
                print("Age cannot be negative. Please try again.")
            else:
                return age
        except ValueError:
            print("Invalid input. Please enter a whole number.")

# Example usage
def main():
    user_age = get_user_age()
    print(f"Your age is: {user_age}")

if __name__ == "__main__":
    main()
```
