## Exercise
Create a function that takes keyword arguments and returns a formatted string.

## Reference Solution
```python
def format_string(**kwargs):
    """
    This function takes keyword arguments and returns a formatted string.
    
    Args:
        **kwargs: Keyword arguments to be formatted into a string.
    
    Returns:
        str: A formatted string containing all keyword arguments.
    """

    # Initialize an empty list to store the formatted key-value pairs
    formatted_pairs = []

    # Iterate over each key-value pair in the keyword arguments
    for key, value in kwargs.items():
        # Format the key-value pair as 'key: value' and add it to the list
        formatted_pairs.append(f"{key}: {value}")

    # Join all the formatted pairs with a comma and a space, and return the result
    return ", ".join(formatted_pairs)


# Test the function
print(format_string(name="John", age=30, city="New York"))
```
