## Exercise
Write a program that asks the user for their name and then prints out a greeting message with their name.

## Reference Solution
```python
def get_user_name():
    """Asks the user for their name and returns it."""
    user_name = input("Please enter your name: ")
    return user_name

def print_greeting(name):
    """Prints out a greeting message with the provided name."""
    print(f"Hello, {name}! It's nice to meet you.")

def main():
    """Gets the user's name and prints out a greeting message."""
    user_name = get_user_name()
    print_greeting(user_name)

if __name__ == "__main__":
    main()
```
