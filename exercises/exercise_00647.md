## Exercise
Write a program that reads a line of input from the user and prints out the number of characters it contains.

## Reference Solution
```python
#include <stdio.h>

int main() {
    int c, nl;
    nl = 0;
    while ((c = getchar()) != EOF && c != '\n') {
        nl++;
    }
    printf("%d\n", nl);
    return 0;
}
```
