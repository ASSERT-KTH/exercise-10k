## Exercise
Given a dictionary {'name': 'George', 'age': 25, 'city': 'Perth'}, write a program to remove the key-value pair 'age': 25.

## Reference Solution
```python
def remove_key_value_pair(dictionary, key):
    """
    Removes a key-value pair from a dictionary.

    Args:
        dictionary (dict): The dictionary to modify.
        key (str): The key of the pair to remove.

    Returns:
        dict: The modified dictionary.
    """
    if key in dictionary:
        # Using pop() method
        # dictionary.pop(key)
        
        # Using del statement
        del dictionary[key]
    return dictionary

# Example usage:
person = {'name': 'George', 'age': 25, 'city': 'Perth'}
print("Original dictionary:")
print(person)

person = remove_key_value_pair(person, 'age')
print("\nDictionary after removing 'age' key-value pair:")
print(person)
```
