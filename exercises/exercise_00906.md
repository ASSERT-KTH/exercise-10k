## Exercise
Write a program that sorts a list of variables in ascending or descending order based on their values.

## Reference Solution
```python
def sort_variables(variables, order='asc'):
    """
    Sorts a list of variables in ascending or descending order based on their values.

    Args:
    variables (dict): A dictionary where keys are variable names and values are variable values.
    order (str): The order of sorting. 'asc' for ascending and 'desc' for descending. Default is 'asc'.

    Returns:
    list: A list of tuples where each tuple contains a variable name and its value, sorted according to the specified order.
    """
    if order not in ['asc', 'desc']:
        raise ValueError("Invalid order. Order should be 'asc' or 'desc'.")

    # Sort variables by their values
    sorted_variables = sorted(variables.items(), key=lambda x: x[1])

    # If order is 'desc', reverse the sorted list
    if order == 'desc':
        sorted_variables.reverse()

    return sorted_variables

# Example usage
variables = {
    'var1': 10,
    'var2': 5,
    'var3': 15,
    'var4': 8
}

print(sort_variables(variables, 'asc'))
print(sort_variables(variables, 'desc'))
```
