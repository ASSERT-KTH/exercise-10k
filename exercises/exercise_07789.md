## Exercise
Create a program that takes a user input and determines its data type. The program should be able to identify if the input is an integer, float, or string. It should also handle cases where the input cannot be converted to any of these types.

## Reference Solution
```python
# Get user input
user_input = input("Enter a value: ")

# Try to convert input to integer
try:
    int(user_input)
    print("The data type of the input is: integer")
except ValueError:
    # Try to convert input to float
    try:
        float(user_input)
        print("The data type of the input is: float")
    except ValueError:
        # Input is not a number, must be a string
        print("The data type of the input is: string")
```
