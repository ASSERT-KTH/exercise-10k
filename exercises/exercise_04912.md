## Exercise
Write a program that takes a credit score as input and prints the corresponding credit rating based on the following criteria: Excellent (750-850), Good (700-749), Fair (650-699), Poor (below 650).

## Reference Solution
```python
# Author: [Your Name]
# Date: [Today's Date]
# Description: This program takes a credit score as input and prints the corresponding credit rating.

def credit_rating(score):
    if 750 <= score <= 850:
        return "Excellent"
    elif 700 <= score <= 749:
        return "Good"
    elif 650 <= score <= 699:
        return "Fair"
    else:
        return "Poor"

credit_score = int(input("Enter your credit score: "))
rating = credit_rating(credit_score)
print("Your credit rating is: ", rating)
```
