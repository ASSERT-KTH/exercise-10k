## Exercise
Write a program that enumerates the possible data types in a given programming language

## Reference Solution
```python
# Enumerate possible data types in Python

# Define a dictionary with data types as keys and examples as values
data_types = {
    "Integer": 1,
    "Float": 3.14,
    "Complex": 1 + 2j,
    "String": "Hello, World!",
    "Boolean": True,
    "List": [1, 2, 3],
    "Tuple": (1, 2, 3),
    "Dictionary": {"name": "John", "age": 30},
    "Set": {1, 2, 3},
    "NoneType": None
}

# Print each data type and its example
for data_type, example in data_types.items():
    print(f"{data_type}: {example} ({type(example).__name__})")
```
