## Exercise
Write a program to find the smallest element in a given array.

## Reference Solution
```python
def find_smallest_element(array):
    """
    Finds the smallest element in a given array.

    Args:
        array (list): The input array.

    Returns:
        int: The smallest element in the array.

    Raises:
        ValueError: If the input array is empty.
    """
    if not array:
        raise ValueError("Input array is empty")

    smallest_element = array[0]
    for element in array:
        if element < smallest_element:
            smallest_element = element

    return smallest_element


def main():
    # Example usage
    array = [5, 2, 8, 12, 3]
    print("Array:", array)
    smallest_element = find_smallest_element(array)
    print("Smallest element:", smallest_element)


if __name__ == "__main__":
    main()
```
