## Exercise
Write a program that takes an array of integers and an index as input, and returns the element at the specified index. If the index is out of bounds, return a message indicating that the index is invalid.

## Reference Solution
```python
def element_at_index(array, index):
    """
    This function returns the element at a given index in an array.

    Parameters:
    array (list): The input array.
    index (int): The index of the element to be returned.

    Returns:
    int: The element at the specified index in the array.
    """
    if index < len(array):
        return array[index]
    else:
        return "Invalid index"

# Example usage:
array = [1, 2, 3, 4, 5]
print(element_at_index(array, 2))  # Output: 3
print(element_at_index(array, 8))  # Output: Invalid index
```
