## Exercise
Create a program that converts time from 12-hour format to 24-hour format using if/else statements

## Reference Solution
```python
def convert_time(time_str):
    """
    Converts time from 12-hour format to 24-hour format.

    Args:
        time_str (str): Time in 12-hour format (e.g., "08:30 PM")

    Returns:
        str: Time in 24-hour format (e.g., "20:30")
    """
    components = time_str.split()
    time = components[0].split(":")
    hours = int(time[0])
    minutes = time[1]
    am_pm = components[1]

    if am_pm == "AM":
        if hours == 12:
            hours = 0
    else:
        if hours != 12:
            hours += 12

    return f"{hours:02d}:{minutes}"

def main():
    time_str = input("Enter time in 12-hour format (e.g., 08:30 PM): ")
    converted_time = convert_time(time_str)
    print(f"Time in 24-hour format: {converted_time}")

if __name__ == "__main__":
    main()
```
