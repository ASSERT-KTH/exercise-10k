## Exercise
Write a function that takes a function as an argument and returns a new function that applies the input function to each element of a list.

## Reference Solution
```python
def map_function(func):
    """
    This function takes another function as an argument and returns a new function.
    The new function applies the input function to each element of a list.

    Args:
        func (function): The input function to be applied to each element of the list.

    Returns:
        function: A new function that applies the input function to each element of a list.
    """

    # We define a new function inside our map_function. This new function will take a list as an argument.
    def new_function(lst):
        
        # We initialize an empty list to store the results of applying the input function to each element.
        result = []
        
        # We iterate over each element in the input list.
        for element in lst:
            
            # For each element, we apply the input function and append the result to our result list.
            result.append(func(element))
        
        # Finally, we return the result list.
        return result
    
    # We return the new function.
    return new_function

# Example usage:
def square(x):
    return x ** 2

map_to_square = map_function(square)
print(map_to_square([1, 2, 3, 4, 5]))  # Output: [1, 4, 9, 16, 25]
```
