## Exercise
Write a program that calculates the sum of all even numbers from 1 to a given number (inclusive) using a for loop. The program should ask the user for the input number and then print the sum.

## Reference Solution
```python
# Define the function calcula_soma
def calcula_soma(n):
    soma = 0
    for i in range(1, n+1):
        if i % 2 == 0:  # Check if number is even
            soma += i
    return soma

# Get user input
numero = int(input("Enter a number: "))

# Call the function and print the result
print(calcula_soma(numero))
```
