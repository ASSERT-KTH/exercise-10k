## Exercise
Write a function that calculates the sum of all numbers in a list.

## Reference Solution
```python
def calculate_list_sum(numbers):
    """
    Calculates the sum of all numbers in a list.

    Args:
        numbers (list): A list of numbers.

    Returns:
        int or float: The sum of all numbers in the list.
    """
    return sum(numbers)

# Example usage
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    result = calculate_list_sum(numbers)
    print(f"The sum of the list is: {result}")
```
