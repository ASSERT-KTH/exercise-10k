## Exercise
Write a function that takes a list of integers as input, and returns the minimum element in the list.

## Reference Solution
```python
def find_min_element(lst):
    """
    This function finds the minimum element in a given list of integers.

    Args:
        lst (list): A list of integers.

    Returns:
        int: The minimum element in the list.

    Raises:
        ValueError: If the input list is empty.
    """

    # Check if the list is empty
    if not lst:
        raise ValueError("Input list is empty")

    # Use the built-in min function to find the minimum element
    min_element = min(lst)

    return min_element


# Example usage:
if __name__ == "__main__":
    numbers = [10, 20, 3, 45, 6]
    min_num = find_min_element(numbers)
    print(f"Minimum element in the list: {min_num}")
```
