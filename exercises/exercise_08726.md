## Exercise
Understand the scope of variables and how they can be accessed within different blocks of code.

## Reference Solution
```python
# Global scope
x = 10

def my_function():
    # Local scope
    y = 20
    print("Inside function:", x)  # Accessing global variable
    print("Inside function:", y)  # Accessing local variable

    if True:
        # Block scope
        z = 30
        print("Inside block:", x)  # Accessing global variable
        print("Inside block:", y)  # Accessing local variable
        print("Inside block:", z)  # Accessing block variable

my_function()
print("Outside function:", x)  # Accessing global variable
# print("Outside function:", y)  # Error: y is not defined
# print("Outside function:", z)  # Error: z is not defined
```
