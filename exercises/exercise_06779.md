## Exercise
Write a program that uses a loop to print the squares of numbers from 1 to n. The user will input the value of n.

## Reference Solution
```python
def print_squares(n):
    """
    Prints the squares of numbers from 1 to n.
    
    Args:
        n (int): The upper limit of the range of numbers.
    """
    for i in range(1, n + 1):
        print(f"{i}^2 = {i ** 2}")


def main():
    # Get the value of n from the user
    while True:
        try:
            n = int(input("Enter a positive integer: "))
            if n <= 0:
                print("Please enter a positive integer.")
            else:
                break
        except ValueError:
            print("Invalid input. Please enter a positive integer.")

    # Print the squares of numbers from 1 to n
    print_squares(n)


if __name__ == "__main__":
    main()
```
