## Exercise
Write a program that uses a for loop to count the number of characters in a given string.

## Reference Solution
```python
def count_characters(input_string):
    """
    Counts the number of characters in a given string.

    Args:
        input_string (str): The input string to count characters from.

    Returns:
        int: The number of characters in the input string.
    """
    count = 0
    for _ in input_string:
        count += 1
    return count

def main():
    # Example usage
    input_str = "Hello, World!"
    char_count = count_characters(input_str)
    print(f"The string '{input_str}' has {char_count} characters.")

if __name__ == "__main__":
    main()
```
