## Exercise
Write a program that takes an integer as input and prints its square.

## Reference Solution

def print_square(n):
    """Prints the square of the input number."""
    square = n ** 2
    print(square)

# Get the input from the user
n = int(input("Enter an integer: "))

# Call the function with the input
print_square(n)
