## Exercise
Write a program that takes a list and a function as input, and returns the result of applying the function to all elements of the list.

## Reference Solution
```python
def applyToAll(lst, f):
    return [f(i) for i in lst]

# Test the function
print(applyToAll([1, 2, 3, 4, 5], lambda x: x**2))
```
