## Exercise
Write a program that takes input from the user and writes it to stdout.

## Reference Solution
```python
def get_user_input():
    """Get user input from stdin."""
    user_input = input("Please enter some text: ")
    return user_input

def write_to_stdout(user_input):
    """Write user input to stdout."""
    print(user_input)

def main():
    """Main program entry point."""
    user_input = get_user_input()
    write_to_stdout(user_input)

if __name__ == "__main__":
    main()
```
