## Exercise
Write a function to enumerate a list of integers, returning a list of pairs where each pair contains the index and value of each element.

## Reference Solution
```python
def enumerate_list(input_list):
    """
    This function takes a list of integers as input and returns a list of pairs.
    Each pair contains the index and value of each element in the input list.

    Args:
        input_list (list): A list of integers.

    Returns:
        list: A list of pairs where each pair contains the index and value of each element.
    """

    # Initialize an empty list to store the result
    result = []

    # Iterate over the input list using the enumerate function, which returns both the index and value of each element
    for index, value in enumerate(input_list):
        
        # Create a tuple to store the index and value of the current element
        pair = (index, value)
        
        # Append the pair to the result list
        result.append(pair)

    # Return the result list
    return result

# Example usage:
input_list = [1, 2, 3, 4, 5]
print(enumerate_list(input_list))  # Output: [(0, 1), (1, 2), (2, 3), (3, 4), (4, 5)]
```
