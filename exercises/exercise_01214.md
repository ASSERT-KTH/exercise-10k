## Exercise
Write a function that finds the minimum value in a given list.

## Reference Solution
```python
def find_min_value(lst):
    """
    This function finds the minimum value in a given list.

    Args:
        lst (list): The input list.

    Returns:
        The minimum value in the list, or None if the list is empty.
    """
    if not lst:
        return None
    return min(lst)

# Example usage:
numbers = [10, 5, 8, 3, 1, 4]
print(find_min_value(numbers))  # Output: 1
```

def find_min_value_manual(lst):
    """
    This function finds the minimum value in a given list manually.

    Args:
        lst (list): The input list.

    Returns:
        The minimum value in the list, or None if the list is empty.
    """
    if not lst:
        return None
    min_val = lst[0]
    for num in lst:
        if num < min_val:
            min_val = num
    return min_val

# Example usage:
numbers = [10, 5, 8, 3, 1, 4]
print(find_min_value_manual(numbers))  # Output: 1
