## Exercise
Create an enum data type 'Color' with values 'RED', 'GREEN', and 'BLUE'.

## Reference Solution
```python
from enum import Enum

class Color(Enum):
    """
    Enum data type representing colors.
    """
    RED = 1
    GREEN = 2
    BLUE = 3

# Example usage:
def main():
    print(Color.RED)  # Color.RED
    print(Color.RED.name)  # RED
    print(Color.RED.value)  # 1

    # Iterate over enum values
    for color in Color:
        print(f"{color.name}: {color.value}")

if __name__ == "__main__":
    main()
```
