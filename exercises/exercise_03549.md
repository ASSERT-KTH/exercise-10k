## Exercise
Write a program that takes an array and a partition size as input, and returns a new array containing the partitioned sub-arrays.

## Reference Solution
```python
function chunkArrayInGroups(arr, size) {
  var newArr = [];
  var i = 0;
  while (i < arr.length) {
    newArr.push(arr.slice(i, i + size));
    i += size;
  }
  return newArr;
}

chunkArrayInGroups([0, 1, 2, 3, 4, 5], 2);
```
