## Exercise
Create a program that asks the user for their name, age, and favorite hobby. The program should then output a customized message that includes the user's name, age, and hobby, and also generates a random lottery number between 1 and 100. The output message should be in the format: 'Hello, {name}! You are {age} years old and enjoy {hobby}. Your lucky lottery number is: {lotto_number}'.

## Reference Solution
```python
import random

def main():
    name = input("Enter your name: ")
    age = int(input("Enter your age: "))
    hobby = input("Enter your favorite hobby: ")

    lottery_number = random.randint(1, 100)

    print(f"Hello, {name}! You are {age} years old and enjoy {hobby}. Your lucky lottery number is: {lottery_number}")

if __name__ == "__main__":
    main()
```
