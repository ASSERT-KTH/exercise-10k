## Exercise
Write a program to perform basic arithmetic operations (addition, subtraction, multiplication, division) using variables of different data types.

## Reference Solution
```python
# Define variables of different data types
int_var1 = 10
int_var2 = 5

float_var1 = 10.5
float_var2 = 2.5

complex_var1 = 3 + 4j
complex_var2 = 2 + 3j

# Perform arithmetic operations
# Addition
print("Addition:")
print("Integer:", int_var1 + int_var2)
print("Float:", float_var1 + float_var2)
print("Complex:", complex_var1 + complex_var2)

# Subtraction
print("\nSubtraction:")
print("Integer:", int_var1 - int_var2)
print("Float:", float_var1 - float_var2)
print("Complex:", complex_var1 - complex_var2)

# Multiplication
print("\nMultiplication:")
print("Integer:", int_var1 * int_var2)
print("Float:", float_var1 * float_var2)
print("Complex:", complex_var1 * complex_var2)

# Division
print("\nDivision:")
print("Integer:", int_var1 / int_var2)
print("Float:", float_var1 / float_var2)
print("Complex:", complex_var1 / complex_var2)
```
