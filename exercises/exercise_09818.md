## Exercise
Write a function that finds the maximum element in an array of integers. The function should take an array as input and return the maximum element. If the array is empty, the function should return null.

## Reference Solution
```python
def max_element(array):
    if not array:
        return None
    max = array[0]
    for i in range(1, len(array)):
        if array[i] > max:
            max = array[i]
    return max
```
