## Exercise
Write a program that prints a pattern of stars using a for loop.

## Reference Solution
```python
def print_star_pattern(n):
    """
    Prints a pattern of stars with n rows.

    Args:
        n (int): The number of rows in the pattern.
    """
    for i in range(1, n + 1):
        # Print i number of stars in each row
        for _ in range(i):
            print("*", end=" ")
        # Move to the next line after each row
        print()

# Example usage
if __name__ == "__main__":
    num_rows = 5
    print_star_pattern(num_rows)
```
