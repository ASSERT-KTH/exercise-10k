## Exercise
Write a program that takes a word as input and checks if it is a palindrome.

## Reference Solution
```python
def is_palindrome(word: str) -> bool:
    """
    Checks if a word is a palindrome.

    Args:
        word (str): The word to check.

    Returns:
        bool: True if the word is a palindrome, False otherwise.
    """
    word = word.lower()  # Convert to lowercase
    word = ''.join(e for e in word if e.isalnum())  # Remove non-alphanumeric characters
    return word == word[::-1]  # Check if the word is the same when reversed

def main():
    word = input("Enter a word: ")
    if is_palindrome(word):
        print(f"'{word}' is a palindrome.")
    else:
        print(f"'{word}' is not a palindrome.")

if __name__ == "__main__":
    main()
```
