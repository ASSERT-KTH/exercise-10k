## Exercise
Write a function called 'contains_substring' that checks if a given string contains a specified substring.

## Reference Solution
```python
def contains_substring(string, substring):
    """
    Checks if a given string contains a specified substring.

    Args:
        string (str): The main string to be checked.
        substring (str): The substring to be searched for in the main string.

    Returns:
        bool: True if the string contains the substring, False otherwise.
    """

    # First, we need to check if both inputs are strings. If not, we raise a TypeError.
    if not isinstance(string, str) or not isinstance(substring, str):
        raise TypeError("Both inputs must be strings.")

    # Now, we use the 'in' operator in Python, which checks if a substring is present in a string.
    # This operator returns True if the substring is found, and False otherwise.
    return substring in string


# Example usage:
print(contains_substring("Hello, World!", "World"))  # Returns: True
print(contains_substring("Hello, World!", "Universe"))  # Returns: False
```
