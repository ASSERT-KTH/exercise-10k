## Exercise
Write a program that takes a weight in pounds as input and prints out the equivalent weight in kilograms.

## Reference Solution
```python
def pounds_to_kilograms(pounds):
    """
    Converts weight in pounds to kilograms.

    Args:
        pounds (float): Weight in pounds.

    Returns:
        float: Weight in kilograms.
    """
    kilograms = pounds * 0.453592
    return kilograms

def main():
    # Get user input
    pounds = float(input("Enter weight in pounds: "))

    # Convert pounds to kilograms
    kilograms = pounds_to_kilograms(pounds)

    # Print result
    print(f"{pounds} pounds is equivalent to {kilograms:.2f} kilograms.")

if __name__ == "__main__":
    main()
```
