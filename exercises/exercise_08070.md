## Exercise
Write a function that takes another function as an argument and returns the result of applying the function to a value

## Reference Solution
```python
def apply_function(func, value):
    """
    Applies a given function to a specified value and returns the result.

    Args:
        func (function): The function to be applied.
        value: The value to which the function is applied.

    Returns:
        The result of applying the function to the value.
    """
    return func(value)

# Example usage
def square(x):
    """Returns the square of x."""
    return x ** 2

def main():
    value = 5
    result = apply_function(square, value)
    print(f"The square of {value} is {result}")

if __name__ == "__main__":
    main()
```
