## Exercise
Write a program that takes an array and a value as input and returns the index of the value in the array.

## Reference Solution
```python
def find_index(array, value):
    """
    This function finds the index of a value in an array.

    Args:
        array (list): The input array.
        value: The value to be searched in the array.

    Returns:
        int: The index of the value in the array.

    Raises:
        ValueError: If the value is not found in the array.
    """
    try:
        # Use the index() method to find the index of the value
        index = array.index(value)
        return index
    except ValueError:
        # If the value is not found, raise a ValueError with a custom message
        raise ValueError(f"The value {value} is not found in the array.")

# Example usage:
array = [1, 2, 3, 4, 5]
value = 3
print(find_index(array, value))  # Output: 2
```
