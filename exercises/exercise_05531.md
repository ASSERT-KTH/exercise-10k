## Exercise
Write a program to demonstrate the concept of NaN and Infinity values in a programming language.

## Reference Solution
```python
import math

# Demonstrate NaN (Not a Number)
print("NaN (Not a Number) examples:")
print("float('nan'):", float('nan'))
print("0.0 / 0.0:", 0.0 / 0.0)
print("math.sqrt(-1):", math.sqrt(-1))

# Demonstrate Infinity
print("\nInfinity examples:")
print("float('inf'):", float('inf'))
print("float('-inf'):", float('-inf'))
print("1.0 / 0.0:", 1.0 / 0.0)
print("-1.0 / 0.0:", -1.0 / 0.0)

# Check if a number is NaN
print("\nChecking if a number is NaN:")
print("math.isnan(float('nan')):", math.isnan(float('nan')))
print("math.isnan(0.0 / 0.0):", math.isnan(0.0 / 0.0))
print("math.isnan(1.0):", math.isnan(1.0))

# Check if a number is Infinity
print("\nChecking if a number is Infinity:")
print("math.isinf(float('inf')):", math.isinf(float('inf')))
print("math.isinf(float('-inf')):", math.isinf(float('-inf')))
print("math.isinf(1.0):", math.isinf(1.0))
```
