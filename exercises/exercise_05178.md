## Exercise
Write a function to convert a set of keys to a dictionary with default values.

## Reference Solution
```python
def keys_to_dict(keys, default_value=None):
    """
    This function converts a set of keys to a dictionary with default values.
    
    Args:
        keys (set): A set of keys to be converted into a dictionary.
        default_value (any, optional): The default value for each key in the dictionary. Defaults to None.
    
    Returns:
        dict: A dictionary where each key from the input set is mapped to the default value.
    """
    
    # Initialize an empty dictionary to store the keys and their corresponding default values
    result_dict = {}
    
    # Iterate over each key in the input set
    for key in keys:
        # For each key, assign the default value in the dictionary
        result_dict[key] = default_value
    
    # Return the resulting dictionary
    return result_dict

# Example usage:
keys = {'a', 'b', 'c'}
default_value = 0
print(keys_to_dict(keys, default_value))  # Output: {'a': 0, 'b': 0, 'c': 0}
```
