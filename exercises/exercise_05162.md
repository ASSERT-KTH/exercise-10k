## Exercise
Write a function to create a dictionary from a list of tuples, where each tuple contains a key-value pair.

## Reference Solution
```python
def create_dict(tuple_list):
    """
    This function creates a dictionary from a list of tuples, 
    where each tuple contains a key-value pair.

    Args:
        tuple_list (list): A list of tuples, where each tuple contains a key-value pair.

    Returns:
        dict: A dictionary created from the list of tuples.
    """

    # Initialize an empty dictionary to store the key-value pairs
    result_dict = {}

    # Iterate over each tuple in the list
    for tup in tuple_list:
        
        # Check if the tuple has exactly two elements (key and value)
        if len(tup) == 2:
            
            # Extract the key and value from the tuple
            key, value = tup
            
            # Add the key-value pair to the dictionary
            result_dict[key] = value
        
        # If the tuple does not have exactly two elements, raise a ValueError
        else:
            raise ValueError("Each tuple must have exactly two elements")

    # Return the created dictionary
    return result_dict

# Example usage:
tuple_list = [(1, 'a'), (2, 'b'), (3, 'c')]
print(create_dict(tuple_list))  # Output: {1: 'a', 2: 'b', 3: 'c'}
```
