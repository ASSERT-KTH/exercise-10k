## Exercise
Calculate a person's age based on their birth year and the current year.

## Reference Solution
```python
def calculate_age(birth_year, current_year):
    """
    Calculates a person's age based on their birth year and the current year.

    Args:
        birth_year (int): The person's birth year.
        current_year (int): The current year.

    Returns:
        int: The person's age.
    """
    age = current_year - birth_year
    return age

# Example usage
birth_year = 1990
current_year = 2024
age = calculate_age(birth_year, current_year)
print(f"Age: {age}")
```
