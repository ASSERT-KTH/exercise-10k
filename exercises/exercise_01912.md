## Exercise
Calculate the tip amount given the total bill and tip percentage.

## Reference Solution
```python
def calculate_tip(total_bill, tip_percentage):
    """
    Calculate the tip amount given the total bill and tip percentage.

    Args:
        total_bill (float): The total bill amount.
        tip_percentage (float): The tip percentage.

    Returns:
        float: The calculated tip amount.
    """
    tip_amount = (total_bill * tip_percentage) / 100
    return tip_amount

# Example usage:
total_bill = 100.0
tip_percentage = 15.0
tip_amount = calculate_tip(total_bill, tip_percentage)
print(f"Tip Amount: ${tip_amount:.2f}")
```
