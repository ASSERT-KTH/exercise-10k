## Exercise
Write a program that uses functions to calculate the area and perimeter of a rectangle. The program should take the length and width of the rectangle as input and then call two separate functions to calculate the area and perimeter. The results should then be printed out to the user.

## Reference Solution
```python
#include <stdio.h>

int main() {
    int l, w;

    printf("Enter length: ");
    scanf("%d", &l);

    printf("Enter width: ");
    scanf("%d", &w);

    int a = area(l, w);
    int p = per(l, w);

    printf("Area: %d\n", a);
    printf("Perimeter: %d\n", p);

    return 0;
}

int area(int l, int w) {
    return l * w;
}

int per(int l, int w) {
    return 2 * (l + w);
}
```
