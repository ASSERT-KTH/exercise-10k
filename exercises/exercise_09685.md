## Exercise
Create a function that takes a string as input and checks if it contains only alphanumeric characters and spaces. If the string is valid, return 'Valid', otherwise return 'Invalid'. The function should be case-sensitive.

## Reference Solution
```python
def check_string(s):
    """
    This function checks if a given string contains only alphanumeric characters and spaces.
    
    Parameters:
    s (str): The input string to be checked.
    
    Returns:
    str: 'Valid' if the string is valid, 'Invalid' otherwise.
    """

    # We want to check each character in the string, so we'll use a loop to iterate over the string
    for char in s:
        
        # For each character, we'll use the isalnum() method to check if it's alphanumeric and the isspace() method to check if it's a space
        # If the character is neither alphanumeric nor a space, we immediately return 'Invalid'
        if not (char.isalnum() or char.isspace()):
            return 'Invalid'
    
    # If we've checked all characters and haven't returned 'Invalid', then the string is valid
    return 'Valid'
```
