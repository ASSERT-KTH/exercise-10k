## Exercise
Write a program that asks the user for their name and age, and then prints out a personalized letter to them. The letter should include the user's name and age.

## Reference Solution
```python
# Ask the user for their name and age
name = input("Please enter your name: ")
age = int(input("Please enter your age: "))

# Print out a personalized letter to the user
print("Dear " + name + ",")
print("I hope this letter finds you well. You are currently " + str(age) + " years old.")
print("I wish you all the best for the future.")
print("Sincerely,")
print("Your Friend")
```
