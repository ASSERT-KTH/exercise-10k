## Exercise
Create a function that takes two numbers as arguments and returns True if the first is greater than the second, False otherwise

## Reference Solution
```python
def is_greater_than(a: float, b: float) -> bool:
    """
    Checks if the first number is greater than the second.

    Args:
    a (float): The first number.
    b (float): The second number.

    Returns:
    bool: True if a is greater than b, False otherwise.
    """
    return a > b

# Example usage
if __name__ == "__main__":
    print(is_greater_than(5, 3))  # True
    print(is_greater_than(3, 5))  # False
    print(is_greater_than(5, 5))  # False
```
