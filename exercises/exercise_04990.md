## Exercise
Write a program to check the data type of a given variable.

## Reference Solution
```python
def check_data_type(var):
    """
    This function checks the data type of a given variable.

    Args:
        var (any): The variable to check the data type of.

    Returns:
        str: The data type of the variable.
    """
    return type(var).__name__

# Example usage
if __name__ == "__main__":
    # Test with different data types
    variables = [
        10,  # integer
        10.5,  # float
        "Hello",  # string
        True,  # boolean
        [1, 2, 3],  # list
        (1, 2, 3),  # tuple
        {"a": 1, "b": 2},  # dictionary
        {1, 2, 3},  # set
        None  # NoneType
    ]

    for var in variables:
        print(f"Variable: {var}, Data Type: {check_data_type(var)}")
```
