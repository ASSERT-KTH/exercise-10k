## Exercise
Write a program that finds the maximum number in a given list of numbers.

## Reference Solution
```python
def find_max_number(numbers):
    """
    This function finds the maximum number in a given list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        max_number (int or float): The maximum number in the list.
    """
    if not numbers:
        raise ValueError("The list is empty")

    max_number = numbers[0]
    for num in numbers:
        if num > max_number:
            max_number = num

    return max_number


def main():
    numbers = [12, 45, 7, 23, 56, 89, 34]
    print("The list of numbers is: ", numbers)
    print("The maximum number in the list is: ", find_max_number(numbers))


if __name__ == "__main__":
    main()
```
