## Exercise
Write a program that takes a student's score as input and prints out the corresponding grade based on the following criteria: A (90-100), B (80-89), C (70-79), D (60-69), F (below 60). Use if/else statements to determine the grade.

## Reference Solution
```python
#include <iostream>

using namespace std;

int main() {
    int x;
    cin >> x;

    if (x >= 90 && x <= 100) {
        cout << "A";
    } else if (x >= 80 && x < 90) {
        cout << "B";
    } else if (x >= 70 && x < 80) {
        cout << "C";
    } else if (x >= 60 && x < 70) {
        cout << "D";
    } else {
        cout << "F";
    }

    return 0;
}
```
