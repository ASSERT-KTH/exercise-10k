## Exercise
Write a program that decodes bytes into a string using a specific encoding.

## Reference Solution
```python
import sys

def decode_bytes(bytes, encoding):
    try:
        return bytes.decode(encoding)
    except UnicodeDecodeError:
        return bytes

if __name__ == "__main__":
    bytes_obj = sys.stdin.buffer.read()
    decoded_str = decode_bytes(bytes_obj, 'utf-8')
    print(decoded_str)
```
