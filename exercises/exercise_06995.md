## Exercise
Write a program that takes the length and width of a rectangle as input and returns its perimeter.

## Reference Solution
```python
def calculate_perimeter(length, width):
    """
    Calculate the perimeter of a rectangle.

    Args:
        length (float): The length of the rectangle.
        width (float): The width of the rectangle.

    Returns:
        float: The perimeter of the rectangle.
    """
    if length <= 0 or width <= 0:
        raise ValueError("Length and width must be greater than 0")

    perimeter = 2 * (length + width)
    return perimeter


def main():
    # Get the length and width from the user
    length = float(input("Enter the length of the rectangle: "))
    width = float(input("Enter the width of the rectangle: "))

    try:
        # Calculate the perimeter
        perimeter = calculate_perimeter(length, width)
        print(f"The perimeter of the rectangle is {perimeter:.2f} units.")
    except ValueError as e:
        print(e)


if __name__ == "__main__":
    main()
```
