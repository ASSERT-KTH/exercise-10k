## Exercise
Create a program that checks if a given input is of a specific data type. The program should take two parameters: the input value and the expected data type. It should then return 'Valid' if the input matches the expected data type, and 'Invalid' otherwise.

## Reference Solution
```python
def check_data_type(input_value, expected_type):
    """
    This function checks if the given input value matches the expected data type.

    Parameters:
    input_value (any): The value to be checked.
    expected_type (type): The expected data type.

    Returns:
    str: 'Valid' if the input value matches the expected data type, 'Invalid' otherwise.
    """

    # First, we need to check if the input value is of the expected type
    # We can use the isinstance() function in Python for this purpose
    # The isinstance() function returns True if the object (first argument) is an instance or subclass of the class (second argument)
    
    # Check if the input value is of the expected type
    is_valid = isinstance(input_value, expected_type)
    
    # If the input value is of the expected type, return 'Valid'
    # Otherwise, return 'Invalid'
    
    # Use a conditional statement to return 'Valid' or 'Invalid' based on the is_valid variable
    if is_valid:
        # If the input value is of the expected type, return 'Valid'
        return 'Valid'
    else:
        # If the input value is not of the expected type, return 'Invalid'
        return 'Invalid'

# Example usage:
print(check_data_type(10, int))  # Output: Valid
print(check_data_type('hello', int))  # Output: Invalid
```
