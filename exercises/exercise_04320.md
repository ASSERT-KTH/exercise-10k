## Exercise
Write a function that uses lambda functions to filter a list of numbers

## Reference Solution
```python
def filter_numbers(numbers, condition):
    """
    Filter a list of numbers based on a given condition.

    Args:
        numbers (list): A list of numbers to filter.
        condition (str): A string indicating the condition to filter by.
            Options: 'even', 'odd', 'positive', 'negative', 'zero'

    Returns:
        list: A filtered list of numbers.
    """
    conditions = {
        'even': lambda x: x % 2 == 0,
        'odd': lambda x: x % 2 != 0,
        'positive': lambda x: x > 0,
        'negative': lambda x: x < 0,
        'zero': lambda x: x == 0
    }

    if condition in conditions:
        return list(filter(conditions[condition], numbers))
    else:
        raise ValueError("Invalid condition")

# Example usage
if __name__ == "__main__":
    numbers = [-5, -2, 0, 2, 5, 10]
    print("Original list:", numbers)

    print("Even numbers:", filter_numbers(numbers, 'even'))
    print("Odd numbers:", filter_numbers(numbers, 'odd'))
    print("Positive numbers:", filter_numbers(numbers, 'positive'))
    print("Negative numbers:", filter_numbers(numbers, 'negative'))
    print("Zero:", filter_numbers(numbers, 'zero'))
```
