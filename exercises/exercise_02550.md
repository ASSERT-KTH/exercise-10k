## Exercise
Use the reduce function to apply a given function to all elements of a list.

## Reference Solution
```python
import functools

def add(x, y):
    """Add two numbers."""
    return x + y

# Example list
numbers = [1, 2, 3, 4, 5]

# Use reduce to calculate the sum
total = functools.reduce(add, numbers)

print(total)  # Output: 15
```

```python
import functools

numbers = [1, 2, 3, 4, 5]
total = functools.reduce(lambda x, y: x + y, numbers)
print(total)  # Output: 15
```

```python
import functools

# Multiplication
numbers = [1, 2, 3, 4, 5]
product = functools.reduce(lambda x, y: x * y, numbers)
print(product)  # Output: 120

# Concatenation
strings = ['h', 'e', 'l', 'l', 'o']
message = functools.reduce(lambda x, y: x + y, strings)
print(message)  # Output: hello
```
