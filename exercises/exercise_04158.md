## Exercise
Write a function to filter a dictionary based on a condition.

## Reference Solution
```python
def filter_dict(input_dict, condition_func):
    """
    Filters a dictionary based on a condition.

    Args:
        input_dict (dict): The dictionary to be filtered.
        condition_func (function): A function that takes in a key-value pair and returns True if the condition is met, False otherwise.

    Returns:
        dict: A new dictionary that includes only the key-value pairs where the condition is met.
    """
    return {key: value for key, value in input_dict.items() if condition_func(key, value)}

# Example usage:
def condition_func(key, value):
    # Example condition: filter out values that are not greater than 5
    return value > 5

input_dict = {'a': 1, 'b': 2, 'c': 6, 'd': 7, 'e': 3}
filtered_dict = filter_dict(input_dict, condition_func)

print(filtered_dict)  # Output: {'c': 6, 'd': 7}
```
