## Exercise
Write a program that takes a Markdown string as input and outputs the parsed HTML.

## Reference Solution
```python
import markdown

def markdown_to_html(markdown_string):
    """
    Convert a Markdown string to HTML.

    Args:
        markdown_string (str): The Markdown string to convert.

    Returns:
        str: The parsed HTML string.
    """
    return markdown.markdown(markdown_string)

def main():
    # Example usage
    markdown_string = """
# Heading
## Subheading
This is a paragraph of text with **bold** and *italic* text.
"""
    html_string = markdown_to_html(markdown_string)
    print(html_string)

if __name__ == "__main__":
    main()
```
