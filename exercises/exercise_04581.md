## Exercise
Create a simple function that takes a name as input and prints out a greeting message

## Reference Solution
```python
def print_greeting(name: str) -> None:
    """
    Prints a greeting message for the given name.

    Args:
        name (str): The name to be used in the greeting.
    """
    print(f"Hello, {name}!")

# Example usage:
def main():
    name = input("Please enter your name: ")
    print_greeting(name)

if __name__ == "__main__":
    main()
```
