## Exercise
Write a program that takes a voltage as input and prints 'Safe' if the voltage is between 0 and 12, and 'Danger' if the voltage is outside this range.

## Reference Solution
```python
# Importing the required module
import sys

# Defining the function
def voltage_check(voltage):
    if 0 <= voltage <= 12:
        print("Safe")
    else:
        print("Danger")

# Checking the command-line argument
if len(sys.argv) > 1:
    # Converting the command-line argument and calling the function
    voltage = int(sys.argv[1])
    voltage_check(voltage)
else:
    # Providing a default input
    voltage_check(10)
```
