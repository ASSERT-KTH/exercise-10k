## Exercise
Write a program to declare and use boolean variables to represent true or false values.

## Reference Solution
```python
# Declare boolean variables
is_true = True
is_false = False

# Print the values of boolean variables
print("Is True:", is_true)
print("Is False:", is_false)

# Use boolean variables in conditional statements
if is_true:
    print("The statement is true.")
else:
    print("The statement is false.")

if is_false:
    print("The statement is true.")
else:
    print("The statement is false.")

# Use boolean variables in logical operations
print("And Operation:", is_true and is_false)
print("Or Operation:", is_true or is_false)
print("Not Operation:", not is_true)

# Use boolean variables in functions
def check_boolean(value):
    if value:
        return "The value is true."
    else:
        return "The value is false."

print(check_boolean(is_true))
print(check_boolean(is_false))
```
