## Exercise
Write a program that takes a list and an element as input and appends the element to the end of the list

def append_to_list(input_list, element):
    """
    Appends an element to the end of a list.

    Args:
        input_list (list): The list to which the element is to be appended.
        element: The element to be appended.

    Returns:
        list: The list with the element appended.
    """
    input_list.append(element)
    return input_list

# Example usage:
