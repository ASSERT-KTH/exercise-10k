## Exercise
Write a program to find the maximum element in a given array.

## Reference Solution
```python
def find_max_element(array):
    """
    Finds the maximum element in a given array.

    Args:
    array (list): The input array.

    Returns:
    int: The maximum element in the array.
    """
    if not array:
        raise ValueError("Array is empty")

    max_element = array[0]
    for element in array:
        if element > max_element:
            max_element = element

    return max_element


def main():
    # Example usage
    array = [10, 20, 30, 40, 50]
    max_element = find_max_element(array)
    print("Maximum element:", max_element)


if __name__ == "__main__":
    main()
```
