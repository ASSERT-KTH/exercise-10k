## Exercise
Write a function to remove non-digit characters from a given string.

## Reference Solution
```python
def remove_non_digit_chars(input_str):
    """
    Removes non-digit characters from a given string.

    Args:
        input_str (str): The input string.

    Returns:
        str: The input string with all non-digit characters removed.
    """
    return ''.join(filter(str.isdigit, input_str))

# Example usage:
if __name__ == "__main__":
    input_str = "abc123def456"
    result_str = remove_non_digit_chars(input_str)
    print(f"Input String: {input_str}")
    print(f"Result String: {result_str}")
```
