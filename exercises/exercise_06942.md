## Exercise
Create a function that uses a decorator to modify its behavior

## Reference Solution
```python
def counter_decorator(func):
    """
    Decorator that adds a counter to a function.
    """
    count = 0
    def wrapper(*args, **kwargs):
        nonlocal count
        count += 1
        print(f"Function {func.__name__} called {count} times.")
        return func(*args, **kwargs)
    return wrapper

@counter_decorator
def add(a, b):
    """
    Function that adds two numbers.
    """
    return a + b

# Example usage:
if __name__ == "__main__":
    print(add(1, 2))  # Function add called 1 times. Output: 3
    print(add(3, 4))  # Function add called 2 times. Output: 7
    print(add(5, 6))  # Function add called 3 times. Output: 11
```
