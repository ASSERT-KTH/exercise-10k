## Exercise
Write a program that takes an array of integers as input, and returns the sum of all the elements in the array.

## Reference Solution
```python
def sum_array_elements(array):
    """
    This function calculates the sum of all elements in an array.

    Args:
        array (list): A list of integers.

    Returns:
        int: The sum of all elements in the array.
    """

    # Initialize the sum variable to 0
    total_sum = 0

    # Iterate over each element in the array
    for num in array:
        # Add the current element to the total sum
        total_sum += num

    # Return the total sum
    return total_sum


# Example usage
array = [1, 2, 3, 4, 5]
print(sum_array_elements(array))  # Output: 15
```
