## Exercise
Create a function that takes two values as arguments and returns True if they are equal, False otherwise

## Reference Solution
```python
def are_equal(a, b):
    """
    Returns True if two values are equal, False otherwise.

    Args:
        a: The first value to compare.
        b: The second value to compare.

    Returns:
        bool: Whether the two values are equal.
    """
    return a == b

# Example usage:
print(are_equal(5, 5))  # Output: True
print(are_equal(5, 3))  # Output: False
print(are_equal("hello", "hello"))  # Output: True
print(are_equal("hello", "world"))  # Output: False
```
