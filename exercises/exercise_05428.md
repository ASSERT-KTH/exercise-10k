## Exercise
Write a program that converts currency from USD to EUR or vice versa. If the user chooses to convert from USD to EUR, use the exchange rate 1 USD = 0.88 EUR. If the user chooses to convert from EUR to USD, use the exchange rate 1 EUR = 1.14 USD.

## Reference Solution
```python
def convert_usd_to_eur(amount):
    """
    Converts USD to EUR using the exchange rate 1 USD = 0.88 EUR.

    Args:
    amount (float): The amount in USD to be converted.

    Returns:
    float: The converted amount in EUR.
    """
    return amount * 0.88

def convert_eur_to_usd(amount):
    """
    Converts EUR to USD using the exchange rate 1 EUR = 1.14 USD.

    Args:
    amount (float): The amount in EUR to be converted.

    Returns:
    float: The converted amount in USD.
    """
    return amount * 1.14

def main():
    print("Currency Converter Program")
    print("-------------------------------")

    while True:
        print("1. Convert USD to EUR")
        print("2. Convert EUR to USD")
        print("3. Quit")

        choice = input("Enter your choice (1/2/3): ")

        if choice == "1":
            amount = float(input("Enter the amount in USD: "))
            converted_amount = convert_usd_to_eur(amount)
            print(f"{amount} USD is equal to {converted_amount:.2f} EUR")
        elif choice == "2":
            amount = float(input("Enter the amount in EUR: "))
            converted_amount = convert_eur_to_usd(amount)
            print(f"{amount} EUR is equal to {converted_amount:.2f} USD")
        elif choice == "3":
            print("Goodbye!")
            break
        else:
            print("Invalid choice. Please try again.")

if __name__ == "__main__":
    main()
```
