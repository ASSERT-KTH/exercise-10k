## Exercise
Write a program that takes an array and a chunk size as input, and returns a new array containing chunks of the specified size.

## Reference Solution
```python
function chunkArrayInGroups(arr, size) {
  var newArr = [];
  for (var i = 0; i < arr.length; i += size) {
    newArr.push(arr.slice(i, i + size));
  }
  return newArr;
}

console.log(chunkArrayInGroups([0, 1, 2, 3, 4, 5], 2));
```
