## Exercise
Write a program that calculates the percentage of a number based on the user's input. The program uses if-else statements to check for valid input.

## Reference Solution
```python
def calculate_percentage():
    # Get the number and percentage from the user
    num = input("Enter a number: ")
    percent = input("Enter a percentage: ")

    # Check if the inputs are valid numbers
    if num.replace('.', '', 1).replace('-', '', 1).isdigit() and percent.replace('.', '', 1).replace('-', '', 1).isdigit():
        num = float(num)
        percent = float(percent)

        # Check if the percentage is within valid range (0-100)
        if 0 <= percent <= 100:
            # Calculate the percentage of the number
            result = (percent / 100) * num
            print(f"{percent}% of {num} is {result}")
        else:
            print("Invalid percentage. Please enter a percentage between 0 and 100.")
    else:
        print("Invalid input. Please enter a valid number and percentage.")

# Call the function
calculate_percentage()
```
