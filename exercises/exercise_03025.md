## Exercise
Write a program that takes a string and a shift value as input, and returns the string shifted by the specified value.

## Reference Solution
```python
#!/usr/bin/perl

use strict;
use warnings;

my $str = $ARGV[0];
my $shift = $ARGV[1];

print shift_string($str, $shift);

sub shift_string {
    my ($str, $shift) = @_;
    $str =~ s/([a-z])/$1 eq 'z' ? 'a' : chr(ord($1) + $shift)/eg;
    $str =~ s/([A-Z])/$1 eq 'Z' ? 'A' : chr(ord($1) + $shift)/eg;
    return $str;
}
```
