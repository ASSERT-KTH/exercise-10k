## Exercise
Develop a program that simulates a dice roller, where the user can roll a dice and the program generates a random result (1-6)

## Reference Solution
```python
import random

def roll_dice():
    """Simulates a dice roll and returns a random result (1-6)"""
    return random.randint(1, 6)

def main():
    print("Welcome to the Dice Roller Program!")
    while True:
        user_input = input("Press 'r' to roll the dice, 'q' to quit: ")
        if user_input.lower() == 'r':
            result = roll_dice()
            print(f"You rolled a {result}!")
        elif user_input.lower() == 'q':
            print("Goodbye!")
            break
        else:
            print("Invalid input. Please try again.")

if __name__ == "__main__":
    main()
```
