## Exercise
Write a program that takes a list as input and returns a reversed version of the list.

## Reference Solution
```python
def reverse_list(input_list):
    """
    This function takes a list as input and returns a reversed version of the list.

    Args:
        input_list (list): The list to be reversed.

    Returns:
        list: The reversed list.
    """
    return input_list[::-1]

# Example usage:
input_list = [1, 2, 3, 4, 5]
print("Original List:", input_list)
print("Reversed List:", reverse_list(input_list))
```
