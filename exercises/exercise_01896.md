## Exercise
Use escape sequences to represent special characters in strings

## Reference Solution
```python
# New line
print("Hello\nWorld")

# Tab
print("Hello\tWorld")

# Double quote
print("Hello \"World\"")

# Backslash
print("Hello \\ World")
```

```python
# Raw string
print(r"Hello \n World")
```

```python
# Unicode escape sequence
print("\u0041")  # Prints 'A'
print("\U00000041")  # Prints 'A'
```
