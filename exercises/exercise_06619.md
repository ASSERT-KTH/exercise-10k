## Exercise
Create a program that takes a string as input from the user and prints the number of vowels in the string.

## Reference Solution
```python
def count_vowels(input_string):
    """
    Count the number of vowels in a given string.

    Args:
        input_string (str): The input string to count vowels in.

    Returns:
        int: The number of vowels in the input string.
    """
    vowels = 'aeiouAEIOU'
    return sum(1 for char in input_string if char in vowels)


def main():
    # Get user input
    input_string = input("Please enter a string: ")

    # Count vowels and print result
    vowel_count = count_vowels(input_string)
    print(f"The string '{input_string}' contains {vowel_count} vowel(s).")


if __name__ == "__main__":
    main()
```
