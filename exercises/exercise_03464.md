## Exercise
Write a program that asks for a text and then prints out the text in uppercase.

## Reference Solution
```python
def print_uppercase(text):
    """Prints the input text in uppercase."""
    print(text.upper())

def main():
    # Ask for user input
    text = input("Please enter a text: ")
    
    # Print the text in uppercase
    print("Text in uppercase:")
    print_uppercase(text)

if __name__ == "__main__":
    main()
```
