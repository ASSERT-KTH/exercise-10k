## Exercise
Write a program to check if a string contains a given substring.

## Reference Solution
```python
def check_substring(main_string, sub_string):
    """
    Checks if the main string contains the given substring.

    Args:
        main_string (str): The main string to search in.
        sub_string (str): The substring to search for.

    Returns:
        bool: True if the substring is found, False otherwise.
    """
    return sub_string in main_string

def main():
    main_string = input("Enter the main string: ")
    sub_string = input("Enter the substring to search for: ")

    if check_substring(main_string, sub_string):
        print(f"The substring '{sub_string}' is found in the main string.")
    else:
        print(f"The substring '{sub_string}' is not found in the main string.")

if __name__ == "__main__":
    main()
```
