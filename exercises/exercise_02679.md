## Exercise
Write a program that takes a string as input and returns the string decompressed using run-length encoding.

## Reference Solution
```python
def decompressRLElist(nums):
    ans = []
    for i in range(0, len(nums), 2):
        ans += [nums[i+1]] * nums[i]
    return ans

# Example input:
nums = [1, 2, 3, 4]
print(decompressRLElist(nums))  # Output: [2, 4, 4, 4]
```
