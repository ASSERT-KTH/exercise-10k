## Exercise
Write a function that takes a dictionary as input and removes all the key-value pairs

## Reference Solution
```python
def clear_dictionary(input_dict):
    """
    Removes all key-value pairs from a dictionary.

    Args:
        input_dict (dict): The dictionary to be cleared.

    Returns:
        dict: The cleared dictionary.
    """
    input_dict.clear()
    return input_dict

# Example usage
example_dict = {"key1": "value1", "key2": "value2"}
print("Before clearing:", example_dict)
cleared_dict = clear_dictionary(example_dict)
print("After clearing:", cleared_dict)
```
