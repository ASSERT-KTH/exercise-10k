## Exercise
Write a program that prints a diamond pattern of asterisks (*) using nested for loops.

## Reference Solution
```python
def print_diamond(n):
    # Print the top half of the diamond
    for i in range(n):
        print(' ' * (n - i - 1) + '*' * (2 * i + 1))

    # Print the bottom half of the diamond
    for i in range(n - 2, -1, -1):
        print(' ' * (n - i - 1) + '*' * (2 * i + 1))

# Test the function
print_diamond(5)
```
