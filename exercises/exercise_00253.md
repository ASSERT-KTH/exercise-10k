## Exercise
Write a program that takes two arrays as input, and returns a new array that contains only the elements that are in the first array but not in the second array.

## Reference Solution

def diff_array(arr1, arr2) {
  return arr1.filter(function(val) {
    return arr2.indexOf(val) === -1;
  });
}
