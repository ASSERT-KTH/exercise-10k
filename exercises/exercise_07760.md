## Exercise
Create a program that takes a user's name as input and prints out a personalized greeting message. The program should also ask for the user's age and print out a message indicating whether they are eligible to vote or not.

## Reference Solution
```python
# Author: [Your Name]
# Date: [Today's Date]
# Description: This program takes a user's name and age as input and prints out a personalized greeting message and a voting eligibility message.
# Version: 1.0

# Get the user's name and age
name = input("What is your name? ")
age = int(input("What is your age? "))

# Print a personalized greeting message
print(f"Hello, {name}!")

# Print a message indicating whether the user is eligible to vote or not
if age >= 18:
    print(f"You are {age} years old and eligible to vote.")
else:
    print(f"You are {age} years old and not eligible to vote.")

# End of the program
```
