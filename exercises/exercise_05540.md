## Exercise
Declare a set variable, assign values to it, and then print its values.

## Reference Solution
```python
# Declare a set variable
my_set = set()

# Assign values to the set
my_set.add(1)
my_set.add(2)
my_set.add(3)
my_set.add(2)  # Duplicate values are automatically ignored in sets

# Alternatively, you can assign values directly during declaration
# my_set = {1, 2, 3}

# Print the set values
print("Set Values:", my_set)
```
