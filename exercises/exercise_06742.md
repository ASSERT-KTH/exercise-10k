## Exercise
Convert a string to an integer and store it in a variable.

## Reference Solution
```python
def convert_string_to_int(string):
    """
    Converts a string to an integer.

    Args:
    string (str): The string to be converted.

    Returns:
    int: The integer equivalent of the input string.

    Raises:
    ValueError: If the string cannot be converted to an integer.
    """
    try:
        # Attempt to convert the string to an integer
        integer = int(string)
        return integer
    except ValueError:
        # Handle the case where the string cannot be converted
        print("The string cannot be converted to an integer.")
        return None

# Example usage
string_to_convert = "123"
result = convert_string_to_int(string_to_convert)
print(f"The integer equivalent of '{string_to_convert}' is: {result}")
```
