## Exercise
Write a program that takes a student's grade as input and determines whether the student passes or fails based on a conditional statement. The passing grade is 50 or above.

## Reference Solution
```python
# Conditional Statements

# A conditional statement is used to execute a block of code if a certain condition is true.

# Syntax
# if condition:
#     # code to execute if condition is true

# Example
grade = 65

if grade >= 50:
    print("Passing")
else:
    print("Failing")
```
