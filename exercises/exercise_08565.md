## Exercise
Create a dictionary with 3 key-value pairs and convert it to a list of lists.

## Reference Solution
```python
def dict_to_list_of_lists(input_dict):
    """
    This function takes a dictionary as input and converts it to a list of lists.
    
    Args:
    input_dict (dict): The input dictionary.
    
    Returns:
    list: A list of lists, where each sublist contains a key-value pair from the input dictionary.
    """
    return [[key, value] for key, value in input_dict.items()]


# Example usage
if __name__ == "__main__":
    # Create a dictionary with 3 key-value pairs
    example_dict = {
        "Name": "John Doe",
        "Age": 30,
        "City": "New York"
    }
    
    # Convert the dictionary to a list of lists
    list_of_lists = dict_to_list_of_lists(example_dict)
    
    print(list_of_lists)
```
