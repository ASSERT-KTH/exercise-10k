## Exercise
Write a program that takes a string as input and returns a list of all substrings.

## Reference Solution
```python
def get_all_substrings(input_string):
    """
    Returns a list of all substrings of the input string.

    Args:
        input_string (str): The input string.

    Returns:
        list: A list of all substrings.
    """
    substrings = []
    for i in range(len(input_string)):
        for j in range(i + 1, len(input_string) + 1):
            substrings.append(input_string[i:j])
    return substrings

def main():
    input_string = input("Enter a string: ")
    all_substrings = get_all_substrings(input_string)
    print("All substrings:")
    for substring in all_substrings:
        print(substring)

if __name__ == "__main__":
    main()
```
