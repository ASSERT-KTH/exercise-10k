## Exercise
Write a program that finds the maximum of two numbers using a simple if statement.

## Reference Solution
```python
def find_max(a, b):
    if a > b:
        return a
    else:
        return b

# Test the function
num1 = 10
num2 = 20
print(f"The maximum of {num1} and {num2} is: {find_max(num1, num2)}")
```
