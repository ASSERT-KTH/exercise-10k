## Exercise
Write a function that takes a dictionary and a map function as input and returns a new dictionary containing the results of applying the map function to each value in the dictionary.

## Reference Solution
```python
def map_dict_values(input_dict, map_func):
    """
    This function takes a dictionary and a map function as input, 
    applies the map function to each value in the dictionary, 
    and returns a new dictionary containing the results.

    Args:
        input_dict (dict): The input dictionary.
        map_func (function): The function to be applied to each value in the dictionary.

    Returns:
        dict: A new dictionary containing the results of applying the map function to each value in the input dictionary.
    """

    # Initialize an empty dictionary to store the results
    result_dict = {}

    # Iterate over each key-value pair in the input dictionary
    for key, value in input_dict.items():
        # Apply the map function to the current value
        mapped_value = map_func(value)
        
        # Store the result in the result dictionary with the same key
        result_dict[key] = mapped_value
    
    # Return the result dictionary
    return result_dict

# Example usage:
# Define a simple map function that squares its input
def square(x):
    return x ** 2

# Create a sample dictionary
input_dict = {'a': 1, 'b': 2, 'c': 3}

# Apply the map function to the dictionary
result = map_dict_values(input_dict, square)

# Print the result
print(result)  # Output: {'a': 1, 'b': 4, 'c': 9}
```
