## Exercise
Create a program that takes in two numbers and an operation (+, -, *, /) as input from the user and performs the specified operation. The program should then output the result.

## Reference Solution
```python
def calculator():
    # Get user input
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))
    operation = input("Enter the operation (+, -, *, /): ")

    # Perform the specified operation
    if operation == "+":
        result = num1 + num2
    elif operation == "-":
        result = num1 - num2
    elif operation == "*":
        result = num1 * num2
    elif operation == "/":
        # Check for division by zero
        if num2 != 0:
            result = num1 / num2
        else:
            print("Error: Division by zero is not allowed.")
            return
    else:
        print("Error: Invalid operation. Please enter +, -, *, or /. ")
        return

    # Output the result
    print(f"{num1} {operation} {num2} = {result}")


if __name__ == "__main__":
    calculator()
```
