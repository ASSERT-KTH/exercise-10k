## Exercise
Write a function that takes a dictionary and a key, then returns the value associated with that key. If the key does not exist, return a default value

## Reference Solution
```python
def get_value(dictionary, key, default=None):
    """
    Retrieves the value associated with the given key from the dictionary.
    
    Args:
        dictionary (dict): The dictionary to search in.
        key: The key to look for.
        default: The default value to return if the key is not found. Defaults to None.
    
    Returns:
        The value associated with the key, or the default value if the key is not found.
    """
    return dictionary.get(key, default)

# Example usage:
if __name__ == "__main__":
    example_dict = {"name": "John", "age": 30}
    
    # Retrieving an existing key
    print(get_value(example_dict, "name"))  # Output: John
    
    # Retrieving a non-existing key with default value
    print(get_value(example_dict, "city", "New York"))  # Output: New York
    
    # Retrieving a non-existing key without specifying a default value
    print(get_value(example_dict, "country"))  # Output: None
```
