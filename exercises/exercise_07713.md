## Exercise
Remove all consonants from a given string.

## Reference Solution
```python
import re

def remove_consonants(input_string):
    """
    Remove all consonants from a given string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The input string with all consonants removed.
    """
    # Use regular expression to replace consonants with an empty string
    return re.sub(r'[^aeiouAEIOU\s]', '', input_string)

# Example usage:
input_string = "Hello, World!"
print("Original String:", input_string)
print("String without consonants:", remove_consonants(input_string))
```
