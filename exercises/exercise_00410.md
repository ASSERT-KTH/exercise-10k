## Exercise
Given a rotated sorted array, find the element at a given index.

## Reference Solution
```python
def find_element(rotated_array, index):
    """
    Finds the element at a given index in a rotated sorted array.

    Args:
        rotated_array (list): The rotated sorted array.
        index (int): The index of the element to find.

    Returns:
        The element at the given index if it exists, otherwise None.
    """
    if not rotated_array:
        return None  # or raise an error
    if index < 0 or index >= len(rotated_array):
        return None  # or raise an error
    return rotated_array[index]

# Example usage
rotated_array = [4, 5, 6, 1, 2, 3]
index = 3
print(find_element(rotated_array, index))  # Output: 1
```
