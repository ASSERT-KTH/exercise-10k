## Exercise
Write a function that takes two dictionaries and returns a new dictionary with the key-value pairs that exist in both dictionaries

## Reference Solution
```python
def common_key_value_pairs(dict1, dict2):
    """
    Returns a new dictionary containing key-value pairs common to both input dictionaries.

    Args:
        dict1 (dict): The first dictionary.
        dict2 (dict): The second dictionary.

    Returns:
        dict: A new dictionary with the common key-value pairs.
    """
    common_pairs = {key: dict1[key] for key in dict1 if key in dict2 and dict1[key] == dict2[key]}
    return common_pairs

# Example usage
dict1 = {'a': 1, 'b': 2, 'c': 3}
dict2 = {'a': 1, 'b': 20, 'c': 3}

common_dict = common_key_value_pairs(dict1, dict2)
print(common_dict)  # Output: {'a': 1, 'c': 3}
```
