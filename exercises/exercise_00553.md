## Exercise
Write a program that takes a string as input and returns the string with all duplicate characters removed.

## Reference Solution
```python
def remove_duplicates(input_string):
    """
    Remove duplicate characters from a string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The string with all duplicate characters removed.
    """
    seen = set()
    output_string = ""

    for char in input_string:
        if char not in seen:
            seen.add(char)
            output_string += char

    return output_string

def main():
    # Example usage:
    input_str = "hello world"
    print("Input:", input_str)
    print("Output:", remove_duplicates(input_str))

if __name__ == "__main__":
    main()
```
