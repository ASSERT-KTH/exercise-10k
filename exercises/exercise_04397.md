## Exercise
Use regular expression patterns with variables

## Reference Solution
```python
import re

# Define a variable pattern
pattern = r"\d{4}-\d{2}-\d{2}"  # matches dates in YYYY-MM-DD format

# Use the pattern with re.search
date_string = "My birthday is 1990-02-12"
if re.search(pattern, date_string):
    print("Date found")
else:
    print("Date not found")

# Define a variable pattern with a variable
year = 2022
pattern = rf"\d{{4}}-{year}"  # matches dates in YYYY-2022 format

# Use the pattern with re.search
date_string = "My anniversary is 2010-2022"
if re.search(pattern, date_string):
    print("Date found")
else:
    print("Date not found")
```
