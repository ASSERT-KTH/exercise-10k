## Exercise
Write a program that uses a for loop to print each element of a given array.

## Reference Solution
```python
# Define the array
array = [1, 2, 3, 4, 5]

# Use a for loop to print each element
for element in array:
    print(element)
```

```python
array = [1, 2, 3, 4, 5]

for index, element in enumerate(array):
    print(f"Index: {index}, Value: {element}")
```
