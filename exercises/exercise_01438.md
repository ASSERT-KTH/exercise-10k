## Exercise
Create a function that takes a string as input and returns a list of elements parsed from the string.

## Reference Solution
```python
def parse_string(input_str, delimiter=','):
    """
    This function takes a string as input and returns a list of elements parsed from the string.
    
    Args:
        input_str (str): The input string to be parsed.
        delimiter (str): The delimiter used to split the string. Defaults to ','.
    
    Returns:
        list: A list of elements parsed from the string.
    """

    # First, we check if the input is a string. If it's not, we raise a TypeError.
    if not isinstance(input_str, str):
        raise TypeError("Input must be a string.")

    # We then check if the delimiter is a string. If it's not, we raise a TypeError.
    if not isinstance(delimiter, str):
        raise TypeError("Delimiter must be a string.")

    # If the input string is empty, we return an empty list.
    if not input_str:
        return []

    # If the delimiter is an empty string, we return a list of individual characters in the input string.
    if delimiter == '':
        return list(input_str)

    # Now, we use the split() method of the string class to split the string into a list of elements.
    # The split() method splits a string into a list where each word is a list item.
    # We pass the delimiter to the split() method to specify the separator.
    parsed_list = input_str.split(delimiter)

    # Finally, we return the parsed list.
    return parsed_list

# Example usage:
print(parse_string("apple,banana,orange"))  # Output: ['apple', 'banana', 'orange']
print(parse_string("hello world", ' '))  # Output: ['hello', 'world']
```
