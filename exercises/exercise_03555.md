## Exercise
Write a program that takes two arrays as input, and returns a new array containing the elements that are in either array but not in both.

## Reference Solution
```python
def findUniqueNumbers(array1, array2):
    new_array = []
    for x in array1:
        if x not in array2:
            new_array.append(x)
    for x in array2:
        if x not in array1:
            new_array.append(x)
    return new_array

array1 = [3, 2, 1]
array2 = [2, 4, 5]
print(findUniqueNumbers(array1, array2))
```
