## Exercise
Write a function that raises an exception

## Reference Solution
```python
def raise_exception(message="An error occurred"):
    """
    Raises a ValueError exception with a custom message.

    Args:
        message (str): The error message. Defaults to "An error occurred".
    """
    raise ValueError(message)

# Example usage:
try:
    raise_exception("Something went wrong")
except ValueError as e:
    print(f"Caught exception: {e}")
```
