## Exercise
Create a function that takes in a dictionary and returns a new dictionary containing the same values, but with the keys and values swapped

## Reference Solution
```python
def swap_dict_keys_values(input_dict):
    """
    Returns a new dictionary with the keys and values of the input dictionary swapped.

    Args:
        input_dict (dict): The dictionary to swap keys and values for.

    Returns:
        dict: A new dictionary with the keys and values swapped.

    Raises:
        ValueError: If the input dictionary has non-unique values.
    """
    # Check if the input dictionary has unique values
    if len(input_dict.values()) != len(set(input_dict.values())):
        raise ValueError("Input dictionary has non-unique values")

    # Use dictionary comprehension to swap keys and values
    swapped_dict = {value: key for key, value in input_dict.items()}

    return swapped_dict

# Example usage:
input_dict = {"a": 1, "b": 2, "c": 3}
swapped_dict = swap_dict_keys_values(input_dict)
print(swapped_dict)  # Output: {1: 'a', 2: 'b', 3: 'c'}
```
