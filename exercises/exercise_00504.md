## Exercise
Given an array of integers, write a program to find the index of a specific target element.

## Reference Solution
```python
def find_index(arr, target):
    """
    Find the index of a specific target element in an array.

    Args:
        arr (list): The input array of integers.
        target (int): The target element to find.

    Returns:
        int: The index of the target element if found, -1 otherwise.
    """
    try:
        # Use the index method to find the index of the target element
        index = arr.index(target)
        return index
    except ValueError:
        # If the target element is not found, return -1
        return -1


# Example usage:
arr = [1, 2, 3, 4, 5]
target = 3
index = find_index(arr, target)

if index != -1:
    print(f"Target element {target} found at index {index}.")
else:
    print(f"Target element {target} not found in the array.")
```
