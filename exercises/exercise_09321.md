## Exercise
Write a program to declare and initialize variables of different data types, such as integer, float, and string.

## Reference Solution
```python
# Declare and initialize integer variable
integer_variable = 10
print("Integer Variable:", integer_variable)
print("Data Type of Integer Variable:", type(integer_variable))

# Declare and initialize float variable
float_variable = 3.14
print("\nFloat Variable:", float_variable)
print("Data Type of Float Variable:", type(float_variable))

# Declare and initialize string variable
string_variable = "Hello, World!"
print("\nString Variable:", string_variable)
print("Data Type of String Variable:", type(string_variable))

# Declare and initialize boolean variable
boolean_variable = True
print("\nBoolean Variable:", boolean_variable)
print("Data Type of Boolean Variable:", type(boolean_variable))

# Declare and initialize list variable
list_variable = [1, 2, 3, 4, 5]
print("\nList Variable:", list_variable)
print("Data Type of List Variable:", type(list_variable))

# Declare and initialize tuple variable
tuple_variable = (1, 2, 3, 4, 5)
print("\nTuple Variable:", tuple_variable)
print("Data Type of Tuple Variable:", type(tuple_variable))

# Declare and initialize dictionary variable
dictionary_variable = {"name": "John", "age": 30}
print("\nDictionary Variable:", dictionary_variable)
print("Data Type of Dictionary Variable:", type(dictionary_variable))

# Declare and initialize set variable
set_variable = {1, 2, 3, 4, 5}
print("\nSet Variable:", set_variable)
print("Data Type of Set Variable:", type(set_variable))
```
