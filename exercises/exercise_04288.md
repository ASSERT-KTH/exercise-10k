## Exercise
Write a function to sort a dictionary's values in ascending order.

## Reference Solution
```python
def sort_dict_values(dictionary):
    """
    Sorts a dictionary's values in ascending order.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        list: A list of the dictionary's values in ascending order.
    """
    return sorted(dictionary.values())

# Example usage:
if __name__ == "__main__":
    my_dict = {'a': 5, 'b': 1, 'c': 3, 'd': 2, 'e': 4}
    sorted_values = sort_dict_values(my_dict)
    print("Sorted values:", sorted_values)
```
