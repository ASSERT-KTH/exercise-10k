## Exercise
Write a function to find the union of two dictionaries.

## Reference Solution
```python
def dict_union(dict1, dict2):
    """
    Returns a new dictionary containing all key-value pairs from both input dictionaries.
    
    :param dict1: The first dictionary.
    :param dict2: The second dictionary.
    :return: A new dictionary containing all key-value pairs from both input dictionaries.
    """
    # Create a copy of the first dictionary to avoid modifying it directly
    result = dict1.copy()
    
    # Update the result dictionary with key-value pairs from the second dictionary
    result.update(dict2)
    
    return result

# Example usage:
dict1 = {'a': 1, 'b': 2}
dict2 = {'b': 3, 'c': 4}

union_dict = dict_union(dict1, dict2)
print(union_dict)  # Output: {'a': 1, 'b': 3, 'c': 4}
```

def dict_union(dict1, dict2):
    """
    Returns a new dictionary containing all key-value pairs from both input dictionaries.
    
    :param dict1: The first dictionary.
    :param dict2: The second dictionary.
    :return: A new dictionary containing all key-value pairs from both input dictionaries.

# Example usage:
