## Exercise
Create a function that takes a string and a number as arguments and returns the string repeated that many times

## Reference Solution
```python
def repeat_string(input_str: str, num_repeats: int) -> str:
    """
    Repeat a string a specified number of times.

    Args:
        input_str (str): The string to be repeated.
        num_repeats (int): The number of times to repeat the string.

    Returns:
        str: The repeated string.
    """
    return input_str * num_repeats

# Example usage:
def main():
    input_str = "Hello, World! "
    num_repeats = 3
    print(repeat_string(input_str, num_repeats))

if __name__ == "__main__":
    main()
```
