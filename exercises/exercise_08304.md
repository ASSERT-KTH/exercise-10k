## Exercise
Write a program that prints the multiplication tables for numbers from 1 to 5. The program should use a loop to iterate over the numbers and print the multiplication table for each number.

## Reference Solution
```python
using System;

class Program
{
    static void Main()
    {
        for (int i = 1; i <= 5; i++)
        {
            Console.WriteLine($"Multiplication table for {i}:");
            for (int j = 1; j <= 10; j++)
            {
                Console.WriteLine($"{i} x {j} = {i * j}");
            }
            Console.WriteLine();
        }
    }
}
```
