## Exercise
Write a program that generates the first n neon numbers, where n is a user-inputted value. A neon number is a number whose sum of digits is equal to the number itself. Use a while loop to generate the numbers.

## Reference Solution
```python
def sum_of_digits(num):
    """
    Calculates the sum of digits of a number.
    
    Args:
        num (int): The number to calculate the sum of digits for.
    
    Returns:
        int: The sum of digits of the number.
    """
    return sum(int(digit) for digit in str(num))


def generate_neon_numbers(n):
    """
    Generates the first n neon numbers.
    
    Args:
        n (int): The number of neon numbers to generate.
    
    Returns:
        list: A list of the first n neon numbers.
    """
    neon_numbers = []
    num = 1
    while len(neon_numbers) < n:
        if num == sum_of_digits(num):
            neon_numbers.append(num)
        num += 1
    return neon_numbers


def main():
    # Get user input
    n = int(input("Enter the number of neon numbers to generate: "))
    
    # Generate neon numbers
    neon_numbers = generate_neon_numbers(n)
    
    # Print the neon numbers
    print("The first {} neon numbers are:".format(n))
    print(neon_numbers)


if __name__ == "__main__":
    main()
```
