## Exercise
Write a function that removes a given key from a dictionary if it exists, and returns the updated dictionary.

## Reference Solution
```python
def remove_key(dictionary, key):
    """
    Removes a given key from a dictionary if it exists, and returns the updated dictionary.

    Args:
        dictionary (dict): The dictionary to be updated.
        key (any): The key to be removed.

    Returns:
        dict: The updated dictionary.
    """

    # First, we check if the key exists in the dictionary
    if key in dictionary:
        
        # If the key exists, we use the del statement to remove the key-value pair from the dictionary
        del dictionary[key]
    
    # Whether the key existed or not, we return the updated dictionary
    # This is because the del statement only removes the key if it exists, and does nothing if it doesn't
    return dictionary

# Example usage:
dict1 = {'a': 1, 'b': 2, 'c': 3}
print(remove_key(dict1, 'b'))  # Outputs: {'a': 1, 'c': 3}
```
