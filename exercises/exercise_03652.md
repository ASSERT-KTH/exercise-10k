## Exercise
Create a function that takes a number as an argument and returns True if it is prime and False otherwise

## Reference Solution
```python
import math

def is_prime(n):
    """
    Checks if a number is prime.

    Args:
        n (int): The number to check.

    Returns:
        bool: True if the number is prime, False otherwise.
    """
    if n <= 1:
        return False
    if n == 2:
        return True
    if n % 2 == 0:
        return False
    sqrt_n = math.sqrt(n)
    for i in range(3, int(sqrt_n) + 1, 2):
        if n % i == 0:
            return False
    return True

# Example usage
if __name__ == "__main__":
    numbers = [2, 3, 4, 5, 6, 7, 8, 9, 10]
    for num in numbers:
        print(f"Is {num} prime? {is_prime(num)}")
```
