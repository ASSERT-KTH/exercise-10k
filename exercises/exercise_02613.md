## Exercise
Remove and return the value of a key in a dictionary

## Reference Solution
```python
def remove_and_return_key_value(dictionary, key):
    """
    Removes and returns the value of a key in a dictionary.

    Args:
        dictionary (dict): The dictionary to remove the key from.
        key: The key to remove and return the value of.

    Returns:
        The value of the removed key, or None if the key does not exist.
    """
    return dictionary.pop(key, None)

# Example usage
my_dict = {'a': 1, 'b': 2, 'c': 3}
key_to_remove = 'b'
value = remove_and_return_key_value(my_dict, key_to_remove)
print(f"Removed key '{key_to_remove}' with value: {value}")
print(f"Updated dictionary: {my_dict}")
```
