## Exercise
Write a program that takes a list, an index, and a new element as input and replaces the element at the specified index with the new element.

## Reference Solution
```python
def replaceInList(list, index, new_element):
    list[index] = new_element
    return list

def main():
    l = [1, 5, 7, 8, 9]
    print("Original List:", l)
    index = 3
    new_element = 4
    print("Index:", index)
    print("New Element:", new_element)
    print("List after replacement:", replaceInList(l, index, new_element))

main()
```
