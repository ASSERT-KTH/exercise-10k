## Exercise
Understand how data types are represented in memory by creating examples of each and printing their representations.

## Reference Solution
```python
import sys

# Boolean
bool_example = True
print(f"Boolean: {bool_example} - Memory Size: {sys.getsizeof(bool_example)} bytes")

# Integer
int_example = 10
print(f"Integer: {int_example} - Memory Size: {sys.getsizeof(int_example)} bytes")

# Float
float_example = 10.5
print(f"Float: {float_example} - Memory Size: {sys.getsizeof(float_example)} bytes")

# Complex Number
complex_example = 10 + 5j
print(f"Complex Number: {complex_example} - Memory Size: {sys.getsizeof(complex_example)} bytes")

# String
str_example = "Hello, World!"
print(f"String: {str_example} - Memory Size: {sys.getsizeof(str_example.encode('utf-8'))} bytes")

# List
list_example = [1, 2, 3, 4, 5]
print(f"List: {list_example} - Memory Size: {sys.getsizeof(list_example)} bytes")

# Tuple
tuple_example = (1, 2, 3, 4, 5)
print(f"Tuple: {tuple_example} - Memory Size: {sys.getsizeof(tuple_example)} bytes")

# Dictionary
dict_example = {"name": "John", "age": 30}
print(f"Dictionary: {dict_example} - Memory Size: {sys.getsizeof(dict_example)} bytes")

# Set
set_example = {1, 2, 3, 4, 5}
print(f"Set: {set_example} - Memory Size: {sys.getsizeof(set_example)} bytes")

# None
none_example = None
print(f"None: {none_example} - Memory Size: {sys.getsizeof(none_example)} bytes")
```
