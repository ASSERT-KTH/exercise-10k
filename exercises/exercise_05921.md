## Exercise
Write a program that calculates the sum of all integers from 1 to n, where n is a given positive integer.

## Reference Solution
```python
#include <iostream>

int sum_of_integers(int n) {
    int sum = 0;
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    return sum;
}

int main() {
    int n;
    std::cout << "Enter a positive integer: ";
    std::cin >> n;
    if (n <= 0) {
        std::cout << "Error: Input must be a positive integer." << std::endl;
        return 1;
    }
    int result = sum_of_integers(n);
    std::cout << "The sum of integers from 1 to " << n << " is: " << result << std::endl;
    return 0;
}
```
