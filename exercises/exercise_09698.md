## Exercise
Create a program that takes a user's input and determines its data type. The input can be an integer, float, or string. The program should then print out the determined data type.

## Reference Solution
```python
def determine_data_type(user_input):
    """
    This function determines the data type of the user's input.
    
    Args:
        user_input (str): The user's input.
    
    Returns:
        str: The determined data type.
    """

    # Try to convert the input to an integer
    try:
        # If the input can be converted to an integer, it's an integer
        int(user_input)
        return "Integer"
    except ValueError:
        # If the input can't be converted to an integer, try to convert it to a float
        try:
            float(user_input)
            return "Float"
        except ValueError:
            # If the input can't be converted to a float, it's a string
            return "String"

def main():
    # Get the user's input
    user_input = input("Please enter a value: ")
    
    # Determine the data type of the user's input
    data_type = determine_data_type(user_input)
    
    # Print out the determined data type
    print(f"The data type of '{user_input}' is: {data_type}")

if __name__ == "__main__":
    main()
```
