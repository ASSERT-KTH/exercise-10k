## Exercise
Write a program that checks if a given variable is an integer.

## Reference Solution
```python
def is_integer(var):
    """Check if a variable is an integer."""
    return isinstance(var, int)

# Example usage
def main():
    variables = [1, 2.5, "hello", True, None]

    for var in variables:
        print(f"Is {var} an integer? {is_integer(var)}")

if __name__ == "__main__":
    main()
```
