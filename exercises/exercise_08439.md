## Exercise
Create a program that takes in a month and outputs the corresponding season using conditional statements.

## Reference Solution
```python
def get_season(month):
    """
    Returns the season corresponding to the input month.
    
    Parameters:
    month (str): The month of the year (e.g., 'January', 'February', etc.)
    
    Returns:
    str: The season corresponding to the input month
    """
    month = month.lower()
    
    # Check for spring months
    if month in ['march', 'april', 'may']:
        return 'Spring'
    
    # Check for summer months
    elif month in ['june', 'july', 'august']:
        return 'Summer'
    
    # Check for autumn months
    elif month in ['september', 'october', 'november']:
        return 'Autumn'
    
    # Check for winter months
    elif month in ['december', 'january', 'february']:
        return 'Winter'
    
    # If none of the above conditions are met, return an error message
    else:
        return 'Invalid month'

def main():
    month = input("Enter a month: ")
    season = get_season(month)
    print(f"The season for {month} is {season}")

if __name__ == "__main__":
    main()
```
