## Exercise
Given a string, validate whether it is a valid IP address or not. The function should take in a string as input and return true if it is a valid IP address and false otherwise.

## Reference Solution
```python
import ipaddress

def is_valid_ip(ip: str) -> bool:
    """
    Validate whether a given string is a valid IP address or not.

    Args:
        ip (str): The IP address to be validated.

    Returns:
        bool: True if the IP address is valid, False otherwise.
    """
    try:
        ipaddress.ip_address(ip)
        return True
    except ValueError:
        return False

# Example usage:
if __name__ == "__main__":
    print(is_valid_ip("192.168.0.1"))  # True
    print(is_valid_ip("256.1.1.1"))    # False
    print(is_valid_ip("2001:0db8:85a3:0000:0000:8a2e:0370:7334"))  # True
    print(is_valid_ip("2001:0db8:85a3:0000:0000:8a2e:0370:733g"))  # False
```
