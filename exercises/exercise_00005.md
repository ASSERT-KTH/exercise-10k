## Exercise
Write a program to identify the data type of a given variable.

## Reference Solution
```python
def get_data_type(variable):
    """
    Returns the data type of the given variable.
    
    Args:
        variable: The variable to check.
    
    Returns:
        The data type of the variable.
    """
    return type(variable)

def main():
    # Example usage
    variables = [
        10,  # integer
        3.14,  # float
        "Hello",  # string
        True,  # boolean
        [1, 2, 3],  # list
        (1, 2, 3),  # tuple
        {"a": 1, "b": 2},  # dictionary
        {1, 2, 3},  # set
        None  # NoneType
    ]

    for variable in variables:
        data_type = get_data_type(variable)
        print(f"Variable: {variable}, Data Type: {data_type.__name__}")

if __name__ == "__main__":
    main()
```
