## Exercise
Write a function that takes a string, a character to replace, and a replacement character as input and returns the string with the character replaced.

## Reference Solution
```python
def replace_char(string, char, new_char):
    string = string.replace(char, new_char)
    return string

# Example usage:
print(replace_char("Hello, World!", "H", "J"))
```
