## Exercise
Create a function that takes an array and a value as input and returns a new array where all elements are replaced with the given value.

## Reference Solution
```python
def replace_elements_with_value(array, value):
    new_array = []
    for element in array:
        new_array.append(value)
    return new_array

# Example usage:
array = [1, 2, 3, 4, 5]
value = 10
result = replace_elements_with_value(array, value)
print(result)  # Output: [10, 10, 10, 10, 10]
```
