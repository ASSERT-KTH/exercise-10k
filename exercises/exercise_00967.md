## Exercise
Given a dictionary and a function, write a function to apply the function to each value in the dictionary and return a new dictionary with the results.

## Reference Solution
```python
def apply_function_to_dict_values(dictionary, func):
    """
    Applies a given function to each value in a dictionary and returns a new dictionary with the results.

    Args:
        dictionary (dict): The input dictionary.
        func (function): The function to be applied to each value in the dictionary.

    Returns:
        dict: A new dictionary where each value is the result of applying the function to the corresponding value in the input dictionary.
    """

    # Initialize an empty dictionary to store the results
    result_dict = {}

    # Iterate over each key-value pair in the input dictionary
    for key, value in dictionary.items():
        # Apply the function to the current value and store the result in the result dictionary
        result_dict[key] = func(value)

    # Return the result dictionary
    return result_dict

# Example usage:
# Define a simple function that squares a number
def square(x):
    return x ** 2

# Create a dictionary with some numbers
numbers_dict = {'a': 1, 'b': 2, 'c': 3}

# Apply the square function to each value in the dictionary
result = apply_function_to_dict_values(numbers_dict, square)
print(result)  # Output: {'a': 1, 'b': 4, 'c': 9}
```
