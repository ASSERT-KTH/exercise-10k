## Exercise
Create a function to validate if a given string contains only alphanumeric characters.

## Reference Solution
```python
def is_alphanumeric(s: str) -> bool:
    """
    Validate if a given string contains only alphanumeric characters.

    Args:
    s (str): The input string to be validated.

    Returns:
    bool: True if the string is alphanumeric, False otherwise.
    """
    return s.isalnum()

# Example usage:
def main():
    # Test cases
    test_strings = ["HelloWorld", "Hello World", "Hello123", "!@#$"]

    for s in test_strings:
        print(f"Is '{s}' alphanumeric? {is_alphanumeric(s)}")

if __name__ == "__main__":
    main()
```
