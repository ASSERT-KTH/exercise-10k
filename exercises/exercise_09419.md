## Exercise
Create a function that takes a list and returns the sum of its elements

## Reference Solution
```python
def sum_elements(lst):
    """
    Calculate the sum of all elements in a list.

    Args:
        lst (list): A list of numbers.

    Returns:
        int or float: The sum of all elements in the list.

    Raises:
        TypeError: If the list contains non-numeric values.
    """
    try:
        return sum(lst)
    except TypeError:
        raise TypeError("The list must only contain numeric values.")

# Example usage
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    result = sum_elements(numbers)
    print(f"The sum of the elements is: {result}")
```
