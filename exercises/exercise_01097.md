## Exercise
Write a program that takes the length, width, and height of a rectangular prism as input and outputs the surface area.

## Reference Solution
```python
def calculate_surface_area(length, width, height):
    # Calculate the surface area of the rectangular prism
    surface_area = 2 * (length * width + width * height + height * length)

    return surface_area

def main():
    # Get the dimensions of the rectangular prism from the user
    length = float(input("Enter the length of the rectangular prism: "))
    width = float(input("Enter the width of the rectangular prism: "))
    height = float(input("Enter the height of the rectangular prism: "))

    # Calculate and display the surface area
    surface_area = calculate_surface_area(length, width, height)
    print(f"The surface area of the rectangular prism is: {surface_area}")

if __name__ == "__main__":
    main()
```
