## Exercise
Create a function that takes a list of integers as input and returns the maximum value. The function should iterate through the list to find the maximum value.

## Reference Solution
```python
def find_max_value(lst):
    """
    This function finds the maximum value in a given list of integers.

    Args:
        lst (list): A list of integers.

    Returns:
        int: The maximum value in the list.
    """

    # First, we check if the list is empty. If it is, we raise a ValueError because we can't find the maximum value of an empty list.
    if not lst:
        raise ValueError("List is empty")

    # We initialize the max_value variable with the first element of the list. This will be our initial guess for the maximum value.
    max_value = lst[0]

    # We iterate over each element in the list starting from the second element (index 1).
    for i in range(1, len(lst)):
        # For each element, we check if it's greater than our current max_value.
        if lst[i] > max_value:
            # If it is, we update max_value to be this new number.
            max_value = lst[i]

    # After iterating over the entire list, max_value will hold the maximum value in the list, which we then return.
    return max_value
```
