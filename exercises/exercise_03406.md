## Exercise
Write a program to iterate over the key-value pairs of a given dictionary and print each pair.

## Reference Solution
```python
def print_dictionary_pairs(dictionary):
    """
    Prints each key-value pair in the given dictionary.

    Args:
        dictionary (dict): The dictionary to iterate over.
    """
    for key, value in dictionary.items():
        print(f"Key: {key}, Value: {value}")


# Example usage
if __name__ == "__main__":
    example_dict = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }

    print("Dictionary Key-Value Pairs:")
    print_dictionary_pairs(example_dict)
```
