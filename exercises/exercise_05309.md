## Exercise
Create a function that takes another function as input and applies it to a list of numbers

## Reference Solution
```python
def apply_function(func, numbers):
    """
    Applies a given function to each number in a list.

    Args:
        func (function): The function to apply.
        numbers (list): A list of numbers.

    Returns:
        list: A list of results after applying the function to each number.
    """
    return [func(num) for num in numbers]

# Example usage:
def square(x):
    """Returns the square of a number."""
    return x ** 2

def double(x):
    """Returns the double of a number."""
    return x * 2

numbers = [1, 2, 3, 4, 5]

print("Squares:")
print(apply_function(square, numbers))

print("\nDoubles:")
print(apply_function(double, numbers))
```
