## Exercise
Write a program that takes two lists as input, and returns the cartesian product of the two lists.

## Reference Solution
```python
import itertools

def cartesian_product(list1, list2):
    """
    Returns the cartesian product of two lists.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: The cartesian product of the two lists.
    """
    return list(itertools.product(list1, list2))

# Example usage:
if __name__ == "__main__":
    list1 = [1, 2, 3]
    list2 = ['a', 'b', 'c']

    result = cartesian_product(list1, list2)
    print(result)
```
