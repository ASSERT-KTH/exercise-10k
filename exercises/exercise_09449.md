## Exercise
Write a program that simulates a coin flip and prints the result (heads or tails).

## Reference Solution
```python
import random

def coin_flip():
    """
    Simulates a coin flip and returns the result.
    """
    # Generate a random number (0 or 1)
    flip = random.randint(0, 1)
    
    # Convert the number to heads or tails
    if flip == 0:
        return "Heads"
    else:
        return "Tails"

def main():
    # Simulate a coin flip and print the result
    result = coin_flip()
    print(f"The coin landed on: {result}")

if __name__ == "__main__":
    main()
```
