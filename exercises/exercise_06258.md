## Exercise
Write a function that takes two dictionaries as input and merges them into one dictionary

## Reference Solution
```python
def merge_dictionaries(dict1, dict2):
    """
    Merge two dictionaries into one.

    Args:
        dict1 (dict): The first dictionary.
        dict2 (dict): The second dictionary.

    Returns:
        dict: A new dictionary containing all items from both input dictionaries.
    """
    merged_dict = dict1.copy()  # Create a copy to avoid modifying the original dictionary
    merged_dict.update(dict2)
    return merged_dict

# Example usage:
dict1 = {'a': 1, 'b': 2}
dict2 = {'b': 3, 'c': 4}

merged_dict = merge_dictionaries(dict1, dict2)
print(merged_dict)  # Output: {'a': 1, 'b': 3, 'c': 4}
```
