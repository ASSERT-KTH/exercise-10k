## Exercise
Write a program to ask the user for their name and print it to the console.

## Reference Solution
```python
def get_user_name():
    """Asks the user for their name and returns it."""
    user_name = input("Please enter your name: ")
    return user_name

def print_user_name(user_name):
    """Prints the user's name to the console."""
    print(f"Hello, {user_name}!")

def main():
    user_name = get_user_name()
    print_user_name(user_name)

if __name__ == "__main__":
    main()
```
