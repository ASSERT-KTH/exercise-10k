## Exercise
Write a program that takes a string as input and returns the number of unique characters in the string.

## Reference Solution
```python
def count_unique_chars(input_string):
    """
    Returns the number of unique characters in the input string.

    Args:
        input_string (str): The input string.

    Returns:
        int: The number of unique characters.
    """
    # Convert the string to a set to remove duplicates
    unique_chars = set(input_string)
    
    # Return the number of unique characters
    return len(unique_chars)

def main():
    # Example usage
    input_string = "Hello, World!"
    unique_char_count = count_unique_chars(input_string)
    print(f"The string '{input_string}' has {unique_char_count} unique characters.")

if __name__ == "__main__":
    main()
```
