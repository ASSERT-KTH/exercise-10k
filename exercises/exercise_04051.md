## Exercise
Write a function that takes a dictionary as input and loops through all the key-value pairs in the dictionary, printing out each key and value.

## Reference Solution
```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-

d = {'a': 1, 'b': 2, 'c': 3, 'd': 4, 'e': 5}
for k, v in d.items():
    print(f'{k} = {v}')
```
