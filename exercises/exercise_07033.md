## Exercise
Create a program that takes in a variable and its value, then checks and prints out the data type of the variable. The program should handle the basic data types such as integers, floats, strings, and booleans.

## Reference Solution
```python
def check_data_type(var_name, var_value):
    """
    This function checks and prints the data type of a given variable.

    Args:
        var_name (str): The name of the variable.
        var_value: The value of the variable.

    Returns:
        None
    """
    print(f"The data type of {var_name} is {type(var_value).__name__}.")

def main():
    # Example usage:
    var1_name = "age"
    var1_value = 25
    check_data_type(var1_name, var1_value)

    var2_name = "height"
    var2_value = 170.5
    check_data_type(var2_name, var2_value)

    var3_name = "name"
    var3_value = "John Doe"
    check_data_type(var3_name, var3_value)

    var4_name = "is_admin"
    var4_value = True
    check_data_type(var4_name, var4_value)

if __name__ == "__main__":
    main()
```
