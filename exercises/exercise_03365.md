## Exercise
Use a for loop to count the number of vowels in a given string

## Reference Solution
```python
def count_vowels(input_string):
    """
    Counts the number of vowels in a given string.

    Args:
        input_string (str): The input string to count vowels from.

    Returns:
        int: The number of vowels in the input string.
    """
    vowels = 'aeiouAEIOU'
    count = 0

    for char in input_string:
        if char in vowels:
            count += 1

    return count

# Example usage:
input_str = "Hello World"
print(f"Number of vowels in '{input_str}': {count_vowels(input_str)}")
```
