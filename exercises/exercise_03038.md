## Exercise
Write a program that takes a string as input and returns the pluralized form of the string.

## Reference Solution
```python
import inflect

def pluralize_string(input_string):
    """
    Returns the pluralized form of the input string.

    Args:
        input_string (str): The string to be pluralized.

    Returns:
        str: The pluralized form of the input string.
    """
    p = inflect.engine()
    return p.plural(input_string)

def main():
    # Example usage
    input_string = "cat"
    print(f"The plural form of '{input_string}' is: {pluralize_string(input_string)}")

if __name__ == "__main__":
    main()
```
