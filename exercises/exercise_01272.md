## Exercise
Write a program to flatten a nested list into a single list.

## Reference Solution
```python
def flatten(nested_list):
    """
    Recursively flattens a nested list into a single list.
    
    Args:
        nested_list (list): The nested list to be flattened.
    
    Returns:
        list: The flattened list.
    """
    flat_list = []
    for element in nested_list:
        if isinstance(element, list):
            # If the element is a list, recursively flatten it and extend the flat list
            flat_list.extend(flatten(element))
        else:
            # If the element is not a list, append it to the flat list
            flat_list.append(element)
    return flat_list

# Example usage:
nested_list = [1, 2, [3, 4, [5, 6]], 7, [8, [9]]]
print("Nested List:", nested_list)
print("Flattened List:", flatten(nested_list))
```
