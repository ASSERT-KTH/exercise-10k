## Exercise
Create a program that uses a for loop to generate all anagrams of a given string.

## Reference Solution
```python
import itertools

def generate_anagrams(input_string):
    """
    Generate all possible anagrams of the input string.

    Args:
    input_string (str): The string for which anagrams are to be generated.

    Returns:
    list: A list of all possible anagrams.
    """
    # Generate all permutations of the input string
    anagrams = [''.join(p) for p in itertools.permutations(input_string)]
    
    return anagrams

def main():
    # Test the function
    input_string = "abc"
    anagrams = generate_anagrams(input_string)
    
    print("Anagrams of '{}' are:".format(input_string))
    for i, anagram in enumerate(anagrams):
        print("{}. {}".format(i+1, anagram))

if __name__ == "__main__":
    main()
```
