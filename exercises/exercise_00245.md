## Exercise
Write a program that takes an array and a function as input, and returns a new array with the function applied to each element.

## Reference Solution
```python
def map_function(array, function):
    new_array = []
    for i in array:
        new_array.append(function(i))
    return new_array

def multiply_by_2(x):
    return x * 2

array = [1, 2, 3, 4, 5]
print(map_function(array, multiply_by_2))
```
