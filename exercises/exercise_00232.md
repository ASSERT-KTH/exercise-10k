## Exercise
Write a program to find the longest common prefix among two or more strings.

## Reference Solution
```python
def longest_common_prefix(strings):
    """
    Finds the longest common prefix among two or more strings.

    Args:
        strings (list): A list of strings.

    Returns:
        str: The longest common prefix.
    """
    if not strings:
        return ""

    shortest_str = min(strings, key=len)

    for i, char in enumerate(shortest_str):
        for string in strings:
            if string[i] != char:
                return shortest_str[:i]

    return shortest_str

def main():
    # Example usage
    strings = ["hello", "hello world", "hello python"]
    prefix = longest_common_prefix(strings)
    print(f"Longest common prefix: {prefix}")

if __name__ == "__main__":
    main()
```
