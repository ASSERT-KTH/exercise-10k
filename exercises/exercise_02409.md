Write a function with a docstring that explains its purpose and usage. Then, call this function with an input value and print the result.

def calculate_square(number):
    """
    This function calculates the square of a given number.

    Args:
        number (int or float): The input number to be squared.

    Returns:
        int or float: The square of the input number.
    """
    return number ** 2

# Call the function with an input value and print the result
result = calculate_square(5)
print("The square of 5 is: ", result)
