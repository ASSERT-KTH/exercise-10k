## Exercise
Create a function to filter a dictionary based on a given condition and return a new dictionary containing only the key-value pairs that satisfy the condition.

## Reference Solution
```python
def filter_dictionary(input_dict, condition):
    """
    Filters a dictionary based on a given condition and returns a new dictionary containing only the key-value pairs that satisfy the condition.

    Args:
        input_dict (dict): The dictionary to be filtered.
        condition (function): A function that takes two arguments (key, value) and returns True if the key-value pair satisfies the condition, False otherwise.

    Returns:
        dict: A new dictionary containing only the key-value pairs that satisfy the condition.
    """

    # Initialize an empty dictionary to store the filtered key-value pairs
    filtered_dict = {}

    # Iterate over each key-value pair in the input dictionary
    for key, value in input_dict.items():
        
        # Check if the key-value pair satisfies the condition
        if condition(key, value):
            
            # If the condition is satisfied, add the key-value pair to the filtered dictionary
            filtered_dict[key] = value
    
    # Return the filtered dictionary
    return filtered_dict

# Example usage:
# Define a condition function that checks if the value is greater than 10
def condition(key, value):
    return value > 10

# Create a sample dictionary
input_dict = {'a': 5, 'b': 15, 'c': 20, 'd': 8}

# Filter the dictionary using the condition function
filtered_dict = filter_dictionary(input_dict, condition)

# Print the filtered dictionary
print(filtered_dict)
```
