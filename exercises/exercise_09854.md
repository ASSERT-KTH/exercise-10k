## Exercise
Given an array of integers, write a function that finds the maximum contiguous subarray sum. The function should take an array of integers as input and return the maximum sum.

## Reference Solution
```python
def maxSubArray(nums):
    if not nums:
        return 0
    dp = [0] * len(nums)
    dp[0] = nums[0]
    for i in range(1, len(nums)):
        dp[i] = max(dp[i-1] + nums[i], nums[i])
    return max(dp)
```
