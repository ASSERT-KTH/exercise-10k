## Exercise
Write a program that takes a string as input and returns the string with all consonants removed.

## Reference Solution
```python
def remove_consonants(input_string):
    """
    Removes all consonants from the input string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The input string with all consonants removed.
    """
    vowels = 'aeiouAEIOU'
    return ''.join([char for char in input_string if char in vowels or not char.isalpha()])

# Example usage
def main():
    input_string = "Hello, World!"
    print("Input String:", input_string)
    print("Output String:", remove_consonants(input_string))

if __name__ == "__main__":
    main()
```
