## Exercise
Write a program that simulates a dice roll and prints out the result.

## Reference Solution
```python
import random

def roll_dice(sides=6):
    """
    Simulates a dice roll with the specified number of sides.
    
    Args:
        sides (int): The number of sides on the dice. Defaults to 6.
    
    Returns:
        int: The result of the dice roll.
    """
    return random.randint(1, sides)

def main():
    print("Welcome to the Dice Roll Simulator!")
    num_sides = int(input("Enter the number of sides on the dice (default is 6): ") or 6)
    result = roll_dice(num_sides)
    print(f"You rolled a {result} on a {num_sides}-sided dice.")

if __name__ == "__main__":
    main()
```
