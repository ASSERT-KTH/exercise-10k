## Exercise
Create a function named 'product_of_list' that calculates the product of all numbers in a given list.

## Reference Solution
```python
def product_of_list(num_list):
    """
    Calculate the product of all numbers in a given list.

    Args:
        num_list (list): A list of numbers.

    Returns:
        float: The product of all numbers in the list.

    Raises:
        ValueError: If the list is empty.
        TypeError: If the list contains non-numeric values.
    """
    if not num_list:
        raise ValueError("List is empty")

    product = 1
    for num in num_list:
        if not isinstance(num, (int, float)):
            raise TypeError("List contains non-numeric values")
        product *= num

    return product

# Example usage
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    result = product_of_list(numbers)
    print(f"Product of the list: {result}")
```
