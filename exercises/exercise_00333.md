## Exercise
Write a program to demonstrate the binary and hexadecimal representations of variables of different data types.

## Reference Solution
```python
# Define variables of different data types
int_var = 10
float_var = 10.5
bool_var = True
char_var = 'A'
str_var = "Hello, World!"

# Print binary and hexadecimal representations
print("Integer Variable:")
print(f"  Decimal: {int_var}")
print(f"  Binary: {bin(int_var)}")
print(f"  Hexadecimal: {hex(int_var)}")

print("\nFloat Variable:")
print(f"  Decimal: {float_var}")
print(f"  Binary (approximate): {bin(int(float_var))}")
print(f"  Hexadecimal (approximate): {hex(int(float_var))}")

print("\nBoolean Variable:")
print(f"  Decimal: {int(bool_var)}")
print(f"  Binary: {bin(int(bool_var))}")
print(f"  Hexadecimal: {hex(int(bool_var))}")

print("\nCharacter Variable:")
print(f"  Character: {char_var}")
print(f"  ASCII Value: {ord(char_var)}")
print(f"  Binary: {bin(ord(char_var))}")
print(f"  Hexadecimal: {hex(ord(char_var))}")

print("\nString Variable:")
print(f"  String: {str_var}")
print("  Binary and hexadecimal representations of each character:")
for char in str_var:
    print(f"    {char}:")
    print(f"      ASCII Value: {ord(char)}")
    print(f"      Binary: {bin(ord(char))}")
    print(f"      Hexadecimal: {hex(ord(char))}")
```
