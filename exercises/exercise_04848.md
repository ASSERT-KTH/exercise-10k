## Exercise
Write a program that takes a list and an element as input, and returns the index of the element if it exists in the list.

## Reference Solution
```python
def get_element_index(lst, element):
    if element in lst:
        return lst.index(element)
    else:
        return -1

# Example usage:
lst = [1, 2, 3, 4, 5]
element = 3
print(get_element_index(lst, element))  # Output: 2

lst = [1, 2, 3, 4, 5]
element = 6
print(get_element_index(lst, element))  # Output: -1
```
