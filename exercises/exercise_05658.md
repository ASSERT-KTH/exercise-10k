## Exercise
Create a function that repeats a given string a specified number of times.

## Reference Solution
```python
def repeat_string(input_string: str, num_repeats: int) -> str:
    """
    Repeats a given string a specified number of times.

    Args:
        input_string (str): The string to be repeated.
        num_repeats (int): The number of times to repeat the string.

    Returns:
        str: The repeated string.
    """
    return input_string * num_repeats

# Example usage
def main():
    input_str = "Hello, World! "
    num_repeats = 3
    repeated_str = repeat_string(input_str, num_repeats)
    print(repeated_str)

if __name__ == "__main__":
    main()
```
