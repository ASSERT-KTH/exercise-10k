## Exercise
Write a function that takes a dictionary as input and returns a list of all key-value pairs

## Reference Solution
```python
def get_key_value_pairs(input_dict):
    """
    Returns a list of tuples, where each tuple is a key-value pair from the input dictionary.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        list: A list of tuples, where each tuple is a key-value pair.
    """
    return list(input_dict.items())

# Example usage:
if __name__ == "__main__":
    example_dict = {"a": 1, "b": 2, "c": 3}
    key_value_pairs = get_key_value_pairs(example_dict)
    print(key_value_pairs)
```
