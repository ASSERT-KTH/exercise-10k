## Exercise
Write a program that takes a string input from the user, checks if it's a palindrome, reverses the string, and then prints out both the original string and its reversed version. The program should also handle cases where the input string contains spaces or special characters.

## Reference Solution
```python
import re

def main():
    string = input("Enter a string: ")

    def reverseString(s):
        s = re.sub(r'\W+', '', s).lower()  # remove spaces and special characters, convert to lowercase
        reversed_s = s[::-1]  # reverse the string
        return reversed_s

    def isPalindrome(s):
        s = re.sub(r'\W+', '', s).lower()  # remove spaces and special characters, convert to lowercase
        return s == s[::-1]  # check if the string is the same when reversed

    print(f"Original string: {string}")
    print(f"Reversed string: {reverseString(string)}")
    print(f"Is palindrome: {isPalindrome(string)}")

if __name__ == "__main__":
    main()
```
