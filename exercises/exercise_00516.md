## Exercise
Given an array of integers, write a program to swap two specific elements.

## Reference Solution
```python
def swap_elements(arr, idx1, idx2):
    """
    Swap two elements in an array.

    Args:
        arr (list): The input array.
        idx1 (int): The index of the first element to swap.
        idx2 (int): The index of the second element to swap.

    Returns:
        list: The modified array with the elements swapped.
    """
    # Check if the indices are valid
    if idx1 < 0 or idx1 >= len(arr) or idx2 < 0 or idx2 >= len(arr):
        raise IndexError("Invalid index")

    # Swap the elements
    arr[idx1], arr[idx2] = arr[idx2], arr[idx1]

    return arr

# Example usage
arr = [1, 2, 3, 4, 5]
idx1 = 1
idx2 = 3

print("Original array:", arr)
print("Swapping elements at indices", idx1, "and", idx2)
arr = swap_elements(arr, idx1, idx2)
print("Modified array:", arr)
```
