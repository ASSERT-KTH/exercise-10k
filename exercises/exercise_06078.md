## Exercise
Create a program that prints a triangle of asterisks using a for loop.

## Reference Solution
```python
def print_triangle(size):
    """
    Prints a triangle of asterisks to the console.

    Args:
        size (int): The size of the triangle.
    """
    for i in range(size):
        # Print spaces to align the triangle
        print(' ' * (size - i - 1), end='')
        # Print asterisks for each row
        print('* ' * (i + 1))

def main():
    # Get the size of the triangle from the user
    size = int(input("Enter the size of the triangle: "))
    # Print the triangle
    print_triangle(size)

if __name__ == "__main__":
    main()
```
