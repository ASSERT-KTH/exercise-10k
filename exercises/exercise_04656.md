## Exercise
Create a program that displays the weather forecast. The user can choose the location, and the program displays the current weather conditions.

## Reference Solution
```python
import requests
import json

def get_weather(api_key, location):
    """
    Fetch the current weather conditions for a given location.

    Args:
        api_key (str): The OpenWeatherMap API key.
        location (str): The city or zip code of the location.

    Returns:
        dict: The current weather conditions.
    """
    base_url = "http://api.openweathermap.org/data/2.5/weather"
    params = {
        "q": location,
        "appid": api_key,
        "units": "metric"
    }
    response = requests.get(base_url, params=params)
    return response.json()

def display_weather(weather):
    """
    Display the current weather conditions.

    Args:
        weather (dict): The current weather conditions.
    """
    print(f"Location: {weather['name']}")
```
