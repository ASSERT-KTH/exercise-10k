## Exercise
Develop a program that prints a diamond shape using a while loop.

## Reference Solution
```python
def print_diamond(n):
    """
    Prints a diamond shape of size n.
    
    Args:
        n (int): The size of the diamond.
    """
    # Print the top half of the diamond
    i = 1
    while i <= n:
        # Print spaces
        print(' ' * (n - i), end='')
        # Print asterisks
        print('* ' * i)
        i += 1

    # Print the bottom half of the diamond
    i = n - 1
    while i > 0:
        # Print spaces
        print(' ' * (n - i), end='')
        # Print asterisks
        print('* ' * i)
        i -= 1


def main():
    """
    Asks the user for the size of the diamond and prints it.
    """
    n = int(input("Enter the size of the diamond: "))
    print_diamond(n)


if __name__ == "__main__":
    main()
```
