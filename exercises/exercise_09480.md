## Exercise
Write a program that prints the following pattern: 1, 3, 5, 7, 9, ... , n. The pattern should stop when the last number printed is greater than or equal to the input number n. Use a loop to generate the pattern.

## Reference Solution
```python
n = int(input("Enter a number: "))

i = 1
while i <= n:
    print(i)
    i += 2
```
