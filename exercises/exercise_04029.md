## Exercise
Create a function to find the first duplicate in an array of integers

## Reference Solution
```python
def find_first_duplicate(arr):
    """
    Finds the first duplicate in an array of integers.

    Args:
    arr (list): A list of integers.

    Returns:
    int or None: The first duplicate integer in the array, or None if no duplicates are found.
    """
    seen = set()
    for num in arr:
        if num in seen:
            return num
        seen.add(num)
    return None

# Example usage:
if __name__ == "__main__":
    arr = [2, 1, 3, 5, 3, 2]
    duplicate = find_first_duplicate(arr)
    if duplicate is not None:
        print(f"The first duplicate in the array is {duplicate}")
    else:
        print("No duplicates found in the array")
```
