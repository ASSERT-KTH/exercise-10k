## Exercise
Write a function that takes a string as input and returns the string with all consonants removed.

def remove_consonants(input_string):
    """
    This function takes a string as input and returns the string with all consonants removed.

    Parameters:
    input_string (str): The input string to be processed.

    Returns:
    str: The input string with all consonants removed.
    """

    # First, we define the vowels in both lower and upper case
    vowels = 'aeiouAEIOU'

    # Then, we initialize an empty string to store the result
    result = ''

    # We iterate over each character in the input string
    for char in input_string:
        # For each character, we check if it is a vowel
        if char in vowels:
            # If it is a vowel, we add it to the result string
            result += char

    # Finally, we return the result string
    return result
