## Exercise
Write a program that prints all numbers from 1 to n using a for loop, where n is given as input by the user.

## Reference Solution
```python
# Get the input from the user
n = int(input("Enter a number: "))

# Use a for loop to print numbers from 1 to n
for i in range(1, n+1):
    print(i)
```
