## Exercise
Use built-in functions to perform common data type operations

## Reference Solution
```python
# Convert a string to an integer
num = int("123")
print(num)  # Output: 123

# Convert a float to an integer
num = int(123.45)
print(num)  # Output: 123

# Convert an integer to a string
str_num = str(123)
print(str_num)  # Output: "123"

# Convert a string to a boolean
bool_value = bool("True")
print(bool_value)  # Output: True
```

```python
# Check the data type of a value
num = 123
print(type(num))  # Output: <class 'int'>

# Check if a value is an instance of a particular data type
num = 123
print(isinstance(num, int))  # Output: True
```

```python
# Get the length of a string
str_value = "Hello World"
print(len(str_value))  # Output: 11

# Convert a string to lowercase
str_value = "Hello World"
print(str_value.lower())  # Output: "hello world"

# Convert a string to uppercase
str_value = "Hello World"
print(str_value.upper())  # Output: "HELLO WORLD"

# Split a string into a list of substrings
str_value = "Hello World"
print(str_value.split())  # Output: ["Hello", "World"]

# Join a list of strings into a single string
str_list = ["Hello", "World"]
print(" ".join(str_list))  # Output: "Hello World"
```
