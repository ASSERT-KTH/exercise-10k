## Exercise
Write a program that generates a sequence of numbers based on the user's input. If the input is 'even', print the first 10 even numbers, if 'odd', print the first 10 odd numbers, and if 'both', print the first 10 numbers.

## Reference Solution
```python
def print_numbers(choice):
    if choice == 'even':
        for i in range(10):
            print(i * 2)
    elif choice == 'odd':
        for i in range(10):
            print(i * 2 + 1)
    elif choice == 'both':
        for i in range(10):
            print(i + 1)

def main():
    choice = input("Enter your choice (even, odd, or both): ")
    print_numbers(choice)

if __name__ == "__main__":
    main()
```
