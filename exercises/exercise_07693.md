## Exercise
Write a function that checks if a string contains a digit.

## Reference Solution
```python
def contains_digit(s: str) -> bool:
    """
    Checks if a string contains a digit.

    Args:
        s (str): The input string.

    Returns:
        bool: True if the string contains a digit, False otherwise.
    """
    return any(char.isdigit() for char in s)

# Example usage:
print(contains_digit("Hello World"))  # False
print(contains_digit("Hello123"))  # True
```

import re

def contains_digit(s: str) -> bool:

    Args:
        s (str): The input string.
