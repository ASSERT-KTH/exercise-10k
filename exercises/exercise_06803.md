## Exercise
Given a string input, write a function that checks if the string is a palindrome and then reverses the string. A palindrome is a string that reads the same backward as forward. If the string is a palindrome, the function should return the reversed string; otherwise, return 'Not a palindrome'

## Reference Solution
```python
def check_and_reverse(input_string: str) -> str:
    """
    Checks if the input string is a palindrome and returns the reversed string if it is.
    Otherwise, returns 'Not a palindrome'.

    Args:
        input_string (str): The string to check.

    Returns:
        str: The reversed string if the input is a palindrome, 'Not a palindrome' otherwise.
    """
    # Remove any leading/trailing whitespace and convert to lowercase for comparison
    cleaned_string = input_string.strip().lower()
    
    # Check if the cleaned string is the same when reversed
    if cleaned_string == cleaned_string[::-1]:
        # If it's a palindrome, return the reversed original string
        return input_string[::-1]
    else:
        # If it's not a palindrome, return 'Not a palindrome'
        return 'Not a palindrome'

# Example usage
if __name__ == "__main__":
    print(check_and_reverse("madam"))  # Expected output: "madam"
    print(check_and_reverse("hello"))  # Expected output: "Not a palindrome"
    print(check_and_reverse("A man a plan a canal Panama"))  # Expected output: "amanaP lanac a nalp a nam A"
```
