## Exercise
Write two functions, one that doubles a number and another that adds 1 to a number. Then, use function composition to create a new function that doubles a number and adds 1 to the result.

## Reference Solution
```python
def double_number(x: int) -> int:
    """
    Doubles the input number.

    Args:
    x (int): The number to be doubled.

    Returns:
    int: The doubled number.
    """
    return x * 2


def add_one(x: int) -> int:
    """
    Adds 1 to the input number.

    Args:
    x (int): The number to be incremented.

    Returns:
    int: The incremented number.
    """
    return x + 1


def double_and_add_one(x: int) -> int:
    """
    Doubles the input number and adds 1 to the result.

    Args:
    x (int): The number to be processed.

    Returns:
    int: The result of doubling the number and adding 1.
    """
    return add_one(double_number(x))


# Example usage:
if __name__ == "__main__":
    num = 5
    result = double_and_add_one(num)
    print(f"Double {num} and add 1: {result}")
```
