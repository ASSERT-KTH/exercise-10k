## Exercise
Write a program that checks if a given element is present in an array. The program should take two parameters: the array and the element to be searched. It should return 'True' if the element is found, 'False' otherwise.

## Reference Solution
```python
def linear_search(arr, target):
    for i in arr:
        if i == target:
            return True
    return False

# Test the function
print(linear_search([1, 2, 3, 4, 5], 2))  # Should return True
print(linear_search([1, 2, 3, 4, 5], 6))  # Should return False
```
