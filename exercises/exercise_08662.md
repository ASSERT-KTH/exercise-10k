## Exercise
Write a program that asks the user for their name and prints it to the screen.

## Reference Solution
```python
def get_user_name():
    """Gets the user's name from input."""
    user_name = input("Please enter your name: ")
    return user_name

def print_user_name(user_name):
    """Prints the user's name to the screen."""
    print(f"Hello, {user_name}!")

def main():
    """Main function to run the program."""
    user_name = get_user_name()
    print_user_name(user_name)

if __name__ == "__main__":
    main()
```
