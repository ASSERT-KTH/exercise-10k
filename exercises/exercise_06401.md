## Exercise
Write a program that takes an integer as input and checks if it is greater than 10. If it is, print 'Greater than 10'. Otherwise, print 'Less than or equal to 10'.

## Reference Solution
```python
x = int(input("Enter an integer: "))
if x > 10:
    print("Greater than 10")
else:
    print("Less than or equal to 10")
```
