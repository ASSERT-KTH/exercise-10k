## Exercise
Write a function to convert a dictionary to a list of tuples, where each tuple contains a key-value pair.

## Reference Solution
```python
def dict_to_tuple_list(dictionary):
    """
    This function takes a dictionary as input and returns a list of tuples.
    Each tuple in the list contains a key-value pair from the dictionary.

    Args:
        dictionary (dict): The dictionary to be converted.

    Returns:
        list: A list of tuples, where each tuple contains a key-value pair.
    """

    # Initialize an empty list to store the tuples
    tuple_list = []

    # Iterate over each key-value pair in the dictionary
    for key, value in dictionary.items():
        # Create a tuple containing the key-value pair
        pair_tuple = (key, value)
        
        # Append the tuple to the list
        tuple_list.append(pair_tuple)

    # Return the list of tuples
    return tuple_list
```
