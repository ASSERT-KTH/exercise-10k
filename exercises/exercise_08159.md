## Exercise
Create an array of 5 integers and reverse the order of elements.

## Reference Solution
```python
def reverse_array():
    # Create an array of 5 integers
    arr = [1, 2, 3, 4, 5]
    
    # Print the original array
    print("Original Array:", arr)
    
    # Reverse the array using slicing
    reversed_arr = arr[::-1]
    
    # Print the reversed array
    print("Reversed Array:", reversed_arr)

# Call the function
reverse_array()
```
