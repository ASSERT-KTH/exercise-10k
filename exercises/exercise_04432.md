## Exercise
Create a program that acts as an alarm clock using a while loop to wait for the alarm time.

## Reference Solution
```python
import datetime
import time
import winsound

def set_alarm(alarm_time, sound_duration=1000):
    """
    Set an alarm for a specific time.

    Args:
        alarm_time (str): The time to set the alarm for in 'HH:MM' format.
        sound_duration (int): The duration of the alarm sound in milliseconds. Defaults to 1000.
    """
    print(f"Alarm set for {alarm_time}")

    while True:
        current_time = datetime.datetime.now().strftime("%H:%M")
        if current_time == alarm_time:
            print("Wake Up!")
            winsound.Beep(2500, sound_duration)  # 2500 Hz frequency, 1000 ms duration
            break
        time.sleep(1)  # Wait for 1 second before checking the time again

def main():
    alarm_time = input("Enter the alarm time in 'HH:MM' format: ")
    set_alarm(alarm_time)

if __name__ == "__main__":
    main()
```
