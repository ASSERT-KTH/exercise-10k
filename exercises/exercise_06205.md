## Exercise
Write a function that takes a list and an element as input, and returns the index of the element in the list if it exists, or -1 if it does not.

## Reference Solution
```python
def index_of(list, element):
    for i in range(len(list)):
        if list[i] == element:
            return i
    return -1
```
