## Exercise
Write a program that asks for a string and replaces a substring with another string.

## Reference Solution
```python
def replace_substring():
    # Get the original string from the user
    original_string = input("Enter a string: ")
    
    # Get the substring to replace from the user
    substring_to_replace = input("Enter a substring to replace: ")
    
    # Get the replacement string from the user
    replacement_string = input("Enter a replacement string: ")
    
    # Replace the substring with the replacement string
    modified_string = original_string.replace(substring_to_replace, replacement_string)
    
    # Print the modified string
    print("Modified string:", modified_string)

# Example usage
if __name__ == "__main__":
    replace_substring()
```
