## Exercise
Create a 3x3x3 multi-dimensional array and access its elements.

## Reference Solution
```python
import numpy as np

def main():
    # Create a 3x3x3 multi-dimensional array
    array_3d = np.arange(27).reshape(3, 3, 3)
    
    print("3D Array:")
    print(array_3d)
    
    # Accessing elements
    print("\nAccessing Elements:")
    print("Element at position (0, 0, 0):", array_3d[0, 0, 0])
    print("Element at position (1, 2, 2):", array_3d[1, 2, 2])
    print("2D Slice at position (0, :, :)", array_3d[0, :, :])
    print("1D Slice at position (0, 0, :)", array_3d[0, 0, :])

if __name__ == "__main__":
    main()
```
