## Exercise
Create a function to curry another function and use it to calculate the sum.

## Reference Solution
```python
def curry(func):
    """
    Curry a function.

    Args:
        func (function): The function to curry.

    Returns:
        function: The curried function.
    """
    def curried(*args, **kwargs):
        if len(args) + len(kwargs) >= func.__code__.co_argcount:
            return func(*args, **kwargs)
        return lambda *x, **y: curried(*(args + x), **dict(kwargs, **y))
    return curried

# Example usage:
def add(a, b, c):
    """
    Calculate the sum of three numbers.

    Args:
        a (int): The first number.
        b (int): The second number.
        c (int): The third number.

    Returns:
        int: The sum of the three numbers.
    """
    return a + b + c

# Curry the add function
curried_add = curry(add)

# Calculate the sum using the curried function
result = curried_add(1)(2)(3)
print(result)  # Output: 6
```
