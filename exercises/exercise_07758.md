## Exercise
Create a program that reads a user's name, age, and favorite color from the input, then prints out a personalized greeting message, including the user's name, age, and favorite color, using proper formatting and grammar. Ensure the program handles invalid inputs, such as non-integer age values.

## Reference Solution
```python
# Importing Modules
import sys

# Defining Variables
nome = None
idade = None

# Reading Input
nome = input("Digite seu nome: ")
idade = input("Digite sua idade: ")

# Handling Invalid Inputs
try:
    idade = int(idade)
except ValueError:
    print("Erro: Idade inválida.")
    sys.exit(1)

# Reading Favorite Color
cor_favorita = input("Digite sua cor favorita: ")

# Printing Greeting Message
print(f"Olá, {nome}! Você tem {idade} anos e sua cor favorita é {cor_favorita}.")
```
