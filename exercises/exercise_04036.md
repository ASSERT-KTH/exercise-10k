## Exercise
Implement a function to find the minimum number of operations needed to transform one string into another

## Reference Solution
```python
def levenshtein_distance(s1, s2):
    """
    Calculate the minimum number of operations needed to transform s1 into s2.

    Args:
        s1 (str): The source string.
        s2 (str): The target string.

    Returns:
        int: The minimum number of operations needed.
    """
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    # Initialize the base cases
    for i in range(m + 1):
        dp[i][0] = i
    for j in range(n + 1):
        dp[0][j] = j

    # Fill in the rest of the table
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            cost = 0 if s1[i - 1] == s2[j - 1] else 1
            dp[i][j] = min(
                dp[i - 1][j] + 1,      # Deletion
                dp[i][j - 1] + 1,      # Insertion
                dp[i - 1][j - 1] + cost  # Substitution or no operation
            )

    return dp[m][n]
```

```python
print(levenshtein_distance("kitten", "sitting"))  # Output: 3
print(levenshtein_distance("hello", "world"))  # Output: 4
print(levenshtein_distance("", "abc"))  # Output: 3
print(levenshtein_distance("abc", ""))  # Output: 3
print(levenshtein_distance("", ""))  # Output: 0
```
