## Exercise
Write a function that calculates the length of a string.

## Reference Solution
```python
def calculate_string_length(input_string: str) -> int:
    """
    Calculate the length of a given string.

    Args:
        input_string (str): The input string.

    Returns:
        int: The length of the input string.
    """
    return len(input_string)

# Example usage:
def main():
    input_str = "Hello, World!"
    length = calculate_string_length(input_str)
    print(f"The length of '{input_str}' is: {length}")

if __name__ == "__main__":
    main()
```
