## Exercise
Write a program that prints the binary representation of a given number using a while loop.

## Reference Solution
```python
# https://www.hackerrank.com/challenges/30-binary-numbers/problem

def main():
    n = int(input())
    l = []
    while n > 0:
        if n % 2 == 0:
            l.append(0)
        else:
            l.append(1)
        n = n // 2
    l.reverse()
    print(''.join(map(str, l)))

if __name__ == '__main__':
    main()
```
