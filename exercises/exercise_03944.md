## Exercise
Write a program to accept a name and a message as input and print out a personalized message.

## Reference Solution
```python
def print_personalized_message():
    """
    Prints a personalized message based on user input.
    """
    name = input("Please enter your name: ")
    message = input("Please enter a message: ")

    # Print the personalized message
    print(f"Hello, {name}! {message}")

def main():
    print_personalized_message()

if __name__ == "__main__":
    main()
```
