## Exercise
Create a function that takes a variable number of arguments and returns their sum

## Reference Solution
```python
def sum_variables(*args):
    """
    Returns the sum of a variable number of arguments.

    Args:
        *args: A variable number of numeric arguments.

    Returns:
        The sum of all input arguments.

    Raises:
        TypeError: If any non-numeric argument is passed.
    """
    try:
        return sum(args)
    except TypeError:
        raise TypeError("All arguments must be numeric.")

# Example usage:
if __name__ == "__main__":
    print(sum_variables(1, 2, 3, 4, 5))  # Output: 15
    print(sum_variables(10, 20))  # Output: 30
    print(sum_variables(-5, 5))  # Output: 0
    try:
        print(sum_variables(1, 'a', 3))
    except TypeError as e:
        print(e)  # Output: All arguments must be numeric.
```
