## Exercise
Write a program that uses a for loop to print each character of a given string.

## Reference Solution
```python
def print_characters(input_string):
    """
    Prints each character of the input string.
    
    Args:
        input_string (str): The string to print characters from.
    """
    for char in input_string:
        print(char)

def main():
    # Example usage
    input_string = "Hello, World!"
    print_characters(input_string)

if __name__ == "__main__":
    main()
```
