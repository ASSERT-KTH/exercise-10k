## Exercise
Write a program that takes a string input from the user and outputs the reversed string in uppercase. The program should handle strings of varying lengths and validate the input to ensure it only contains alphabetic characters.

## Reference Solution
```python
# Initial setup
frase = input('Digite uma frase: ')
frase = ''.join(filter(str.isalpha, frase))  # Remove non-alphabetic characters

# Calculations
tamanho = len(frase)
inverso = ''

# Loop to reverse the string
for i in range(tamanho - 1, -1, -1):
    inverso += frase[i]

# Output
print(f'Frase invertida: {inverso.upper()}')
```
