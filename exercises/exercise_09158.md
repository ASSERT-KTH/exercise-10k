## Exercise
Create two variables and swap their values without using a temporary variable, then print out the swapped values.

## Reference Solution
```python
# Create two variables
a = 5
b = 10

# Print original values
print("Original values:")
print(f"a = {a}")
print(f"b = {b}")

# Swap values without using a temporary variable
a, b = b, a

# Print swapped values
print("\nSwapped values:")
print(f"a = {a}")
print(f"b = {b}")
```
