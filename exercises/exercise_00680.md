## Exercise
Given a dictionary {'name': 'Ruby', 'age': 30, 'city': 'Rockhampton'}, write a program to iterate over the key-value pairs in the dictionary and print each key and value.

## Reference Solution
```python
def print_key_value_pairs(dictionary):
    """
    Iterate over the key-value pairs in a dictionary and print each key and value.

    Args:
        dictionary (dict): The dictionary to iterate over.
    """
    for key, value in dictionary.items():
        print(f"Key: {key}, Value: {value}")


# Example usage
if __name__ == "__main__":
    person = {'name': 'Ruby', 'age': 30, 'city': 'Rockhampton'}
    print_key_value_pairs(person)
```
