## Exercise
Create a program that displays a calendar for a given month and year. If the user chooses to view the calendar for the current month, use the current date to determine the month and year.

## Reference Solution
```python
import calendar
import datetime

def display_calendar(year, month):
    """Displays a calendar for the given month and year."""
    print(calendar.month(year, month))

def main():
    print("Calendar Program")
    print("-----------------")
    print("1. View calendar for current month")
    print("2. View calendar for specific month and year")

    choice = input("Enter your choice (1/2): ")

    if choice == "1":
        # Get current date
        current_date = datetime.date.today()
        year = current_date.year
        month = current_date.month
        display_calendar(year, month)
    elif choice == "2":
        # Get month and year from user
        year = int(input("Enter year: "))
        month = int(input("Enter month (1-12): "))
        display_calendar(year, month)
    else:
        print("Invalid choice")

if __name__ == "__main__":
    main()
```
