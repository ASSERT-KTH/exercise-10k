## Exercise
Create a function named 'find_index' that finds and returns the index of a specific element in a given list.

## Reference Solution
```python
def find_index(lst, element):
    """
    This function finds and returns the index of a specific element in a given list.
    
    Parameters:
    lst (list): The list to search for the element.
    element: The element to find in the list.
    
    Returns:
    int: The index of the element if found, -1 otherwise.
    """
    
    # Check if the input is a list, raise TypeError if not
    if not isinstance(lst, list):
        raise TypeError("Input must be a list.")
    
    # Try to find the index of the element in the list
    try:
        # Use the index method of the list to find the index of the element
        # This method raises a ValueError if the element is not found
        index = lst.index(element)
        
        # If the element is found, return its index
        return index
    
    # If the element is not found, catch the ValueError and return -1
    except ValueError:
        # Return -1 to indicate that the element was not found
        return -1

# Example usage:
my_list = [1, 2, 3, 4, 5]
print(find_index(my_list, 3))  # Output: 2
print(find_index(my_list, 6))  # Output: -1
```
