## Exercise
Create a program that prints out the current date and time.

## Reference Solution
```python
# Import the datetime module
from datetime import datetime

def print_current_date_time():
    """
    Prints out the current date and time.
    """
    # Get the current date and time
    current_date_time = datetime.now()
    
    # Print the current date and time
    print("Current Date and Time: ", current_date_time.strftime("%Y-%m-%d %H:%M:%S"))

def main():
    print_current_date_time()

if __name__ == "__main__":
    main()
```
