## Exercise
Use a for loop to find the largest number in a given list of numbers

## Reference Solution
```python
def find_largest_number(num_list):
    """
    Finds the largest number in a given list of numbers.

    Args:
        num_list (list): A list of numbers.

    Returns:
        int: The largest number in the list.
    """
    largest_num = num_list[0]
    for num in num_list:
        if num > largest_num:
            largest_num = num
    return largest_num

# Example usage:
numbers = [4, 2, 9, 6, 5, 1, 8, 3, 7]
print("Largest number:", find_largest_number(numbers))
```
