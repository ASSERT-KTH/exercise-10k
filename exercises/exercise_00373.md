## Exercise
Create a function that takes a string input, reverses the order of its characters, and converts all characters to uppercase. The function should return the resulting string.

## Reference Solution
```python
def reverse_and_uppercase(input_string):
    """
    This function takes a string input, reverses the order of its characters, 
    and converts all characters to uppercase. The function returns the resulting string.

    Parameters:
    input_string (str): The input string to be reversed and uppercased.

    Returns:
    str: The reversed and uppercased string.
    """

    # First, we need to check if the input is indeed a string
    if not isinstance(input_string, str):
        raise TypeError("Input must be a string")

    # Now, we'll use slicing to reverse the string. The syntax [::-1] means start at the end of the string and end at position 0, move with the step -1
    reversed_string = input_string[::-1]
    
    # Next, we'll use the upper() method to convert all characters in the reversed string to uppercase
    uppercased_string = reversed_string.upper()
    
    # Finally, we'll return the resulting string
    return uppercased_string

# Test the function
print(reverse_and_uppercase("hello world"))
```
