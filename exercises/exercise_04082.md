## Exercise
Create a function that takes a name as input and returns a personalized greeting

## Reference Solution
```python
def personalized_greeting(name: str) -> str:
    """
    Returns a personalized greeting based on the input name.

    Args:
        name (str): The name to be used in the greeting.

    Returns:
        str: A personalized greeting.
    """
    return f"Hello, {name}!"

# Example usage
def main():
    name = input("Please enter your name: ")
    greeting = personalized_greeting(name)
    print(greeting)

if __name__ == "__main__":
    main()
```
