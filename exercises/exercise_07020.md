## Exercise
Write a program to find the maximum contiguous subarray sum of a given one-dimensional array of numbers. The subarray must have at least one element. If the array is empty, return 0.

## Reference Solution
```python
def max_subarray_sum(arr):
    """
    This function calculates the maximum contiguous subarray sum.
    
    Args:
    arr (list): A list of integers.
    
    Returns:
    int: The maximum contiguous subarray sum. If the array is empty, it returns 0.
    """
    
    # Check if the array is empty
    if not arr:
        return 0
    
    # Initialize the maximum sum and current sum
    max_sum = float('-inf')
    current_sum = 0
    
    # Iterate over the array
    for num in arr:
        # Update the current sum
        current_sum = max(num, current_sum + num)
        
        # Update the maximum sum
        max_sum = max(max_sum, current_sum)
    
    return max_sum

# Example usage:
arr = [-2, -3, 4, -1, -2, 1, 5, -3]
print(max_subarray_sum(arr))  # Output: 7

arr = []
print(max_subarray_sum(arr))  # Output: 0
```
