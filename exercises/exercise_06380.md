## Exercise
Write a program that takes two lists as input and returns a new list containing the cartesian product of the two lists

## Reference Solution
```python
import itertools

def cartesian_product(list1, list2):
    """
    Calculate the cartesian product of two lists.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: A new list containing the cartesian product of the two lists.
    """
    return list(itertools.product(list1, list2))

# Example usage
list1 = [1, 2, 3]
list2 = ['a', 'b', 'c']

print(cartesian_product(list1, list2))
```

```python
def cartesian_product(list1, list2):
    """
    Calculate the cartesian product of two lists.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: A new list containing the cartesian product of the two lists.
    """
    result = []
    for elem1 in list1:
        for elem2 in list2:
            result.append((elem1, elem2))
    return result

# Example usage
list1 = [1, 2, 3]
list2 = ['a', 'b', 'c']

print(cartesian_product(list1, list2))
```
