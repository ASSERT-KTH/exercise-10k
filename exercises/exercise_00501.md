## Exercise
Write a program to calculate the sum of all elements in a given list of integers.

## Reference Solution
```python
def calculate_sum(numbers):
    """
    Calculate the sum of all elements in a list of integers.

    Args:
        numbers (list): A list of integers.

    Returns:
        int: The sum of all elements in the list.
    """
    return sum(numbers)

# Example usage:
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    total_sum = calculate_sum(numbers)
    print(f"The sum of the list is: {total_sum}")
```
