## Exercise
Write a function that takes a dictionary as input and filters out key-value pairs where the value is less than 5

## Reference Solution
```python
def filter_dict(input_dict):
    """
    Filters out key-value pairs from a dictionary where the value is less than 5.

    Args:
        input_dict (dict): The input dictionary to be filtered.

    Returns:
        dict: A new dictionary containing only the key-value pairs where the value is 5 or greater.
    """
    return {key: value for key, value in input_dict.items() if value >= 5}

# Example usage:
if __name__ == "__main__":
    example_dict = {
        "a": 1,
        "b": 5,
        "c": 10,
        "d": 3,
        "e": 7
    }
    filtered_dict = filter_dict(example_dict)
    print("Original Dictionary:", example_dict)
    print("Filtered Dictionary:", filtered_dict)
```
