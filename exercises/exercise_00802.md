## Exercise
Convert a user-provided integer into a floating-point number and print the result.

## Reference Solution
```python
def convert_to_float():
    # Get user input
    user_input = input("Enter an integer: ")

    # Validate input
    try:
        # Convert input to integer
        integer = int(user_input)
        
        # Convert integer to floating-point number
        float_number = float(integer)
        
        # Print the result
        print(f"The integer {integer} as a floating-point number is: {float_number}")
    
    except ValueError:
        print("Invalid input. Please enter an integer.")

# Example usage
if __name__ == "__main__":
    convert_to_float()
```
