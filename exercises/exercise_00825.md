## Exercise
Given a string, count the frequency of each word and return a dictionary with the words as keys and their frequencies as values.

## Reference Solution
```python
from collections import Counter
import re

def word_frequency_counter(input_string):
    """
    Count the frequency of each word in a given string.

    Args:
        input_string (str): The input string.

    Returns:
        dict: A dictionary with words as keys and their frequencies as values.
    """
    # Convert the string to lowercase and remove punctuation
    cleaned_string = re.sub(r'[^\w\s]', '', input_string.lower())
    
    # Split the string into words
    words = cleaned_string.split()
    
    # Count the frequency of each word
    frequency_dict = Counter(words)
    
    return dict(frequency_dict)

# Example usage
input_string = "This is a test string. This string is just a test."
print(word_frequency_counter(input_string))
```
