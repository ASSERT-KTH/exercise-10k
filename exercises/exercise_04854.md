## Exercise
Write a program that takes a list, an element, and an index as input, and returns the list with the element inserted at the specified index.

## Reference Solution
```python
def insert_element(lst, element, index):
    lst.insert(index, element)
    return lst

lst = [1, 2, 3, 4, 5]
element = 6
index = 3

result = insert_element(lst, element, index)
print(result)
```
