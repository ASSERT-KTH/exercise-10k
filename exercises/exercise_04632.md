## Exercise
Write a program that takes a list and a grouping function as input, and returns a dictionary with the grouped elements.

## Reference Solution
```python
from itertools import groupby

def group_list(input_list, f):
    result = {}
    for key, group in groupby(sorted(input_list, key=f), f):
        result[key] = list(group)
    return result

# Example usage:
my_list = [1, 2, 3, 4, 5, 6, 7, 8, 9]
print(group_list(my_list, lambda x: x % 2))
```
