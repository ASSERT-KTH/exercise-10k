## Exercise
Create a function that takes a string as an argument and returns its length. The function should have type hints for the input and output types.

## Reference Solution
```python
# Define a function named 'string_length' that takes one argument 'input_string' of type 'str' and returns an integer.
def string_length(input_string: str) -> int:
    """
    This function calculates the length of a given string.

    Args:
        input_string (str): The input string to calculate the length of.

    Returns:
        int: The length of the input string.
    """

    # Use the built-in len() function in Python to get the length of the input string.
    # The len() function returns the number of items in an object.
    length = len(input_string)

    # Return the calculated length.
    return length

# Test the function with a sample string.
test_string = "Hello, World!"
print(string_length(test_string))  # Output: 13
```
