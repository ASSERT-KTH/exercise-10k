## Exercise
Write a function that takes a list and a function as input, and returns the result of applying the function to all elements in the list.

## Reference Solution
```python
def apply_function(lst, func):
    """
    Applies a given function to all elements in a list and returns the results.

    Args:
        lst (list): The input list.
        func (function): The function to be applied to each element in the list.

    Returns:
        list: A new list containing the results of applying the function to each element in the input list.
    """

    # Initialize an empty list to store the results of applying the function to each element
    result = []

    # Iterate over each element in the input list
    for element in lst:
        # Apply the function to the current element and append the result to the result list
        result.append(func(element))

    # Return the result list
    return result

# Example usage:
numbers = [1, 2, 3, 4, 5]
def square(x):
    return x ** 2

print(apply_function(numbers, square))  # Output: [1, 4, 9, 16, 25]
```
