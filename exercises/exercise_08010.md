## Exercise
Write a program that checks the data type of a given input and prints out whether it is an integer, float, or string. The program should also handle cases where the input is not one of these three data types.

## Reference Solution
```python
def check_input_type(input):
    if isinstance(input, int):
        print("Input is an integer.")
    elif isinstance(input, float):
        print("Input is a float.")
    elif isinstance(input, str):
        print("Input is a string.")
    else:
        print("Input data type not recognized.")

# Test the function
check_input_type(10)
check_input_type(10.5)
check_input_type("Hello")
check_input_type([1, 2, 3])
```
