## Exercise
Write a program that takes in a username and checks if it is valid using conditional statements.

## Reference Solution
```python
def validate_username(username):
    """
    Checks if a username is valid.

    Args:
        username (str): The username to be validated.

    Returns:
        bool: True if the username is valid, False otherwise.
    """
    # Check if the username is between 3 and 15 characters long
    if not 3 <= len(username) <= 15:
        return False

    # Check if the username contains only letters, numbers, and underscores
    if not all(char.isalnum() or char == '_' for char in username):
        return False

    # Check if the username starts with a letter
    if not username[0].isalpha():
        return False

    return True

def main():
    username = input("Enter a username: ")
    if validate_username(username):
        print("The username is valid.")
    else:
        print("The username is not valid.")

if __name__ == "__main__":
    main()
```
