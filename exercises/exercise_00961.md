## Exercise
Create a dictionary with the given keys and values, and print out the resulting dictionary.

## Reference Solution
```python
def create_dictionary(keys, values):
    """
    Creates a dictionary from the given keys and values.
    
    Args:
        keys (list): A list of keys for the dictionary.
        values (list): A list of values for the dictionary.
    
    Returns:
        dict: A dictionary created from the given keys and values.
    """
    if len(keys) != len(values):
        raise ValueError("The number of keys and values must be equal.")
    
    dictionary = dict(zip(keys, values))
    return dictionary

# Example usage
keys = ["name", "age", "city"]
values = ["John Doe", 30, "New York"]

resulting_dict = create_dictionary(keys, values)
print(resulting_dict)
```
