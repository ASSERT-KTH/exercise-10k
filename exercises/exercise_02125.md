## Exercise
Create a function that takes an array as input and returns a new array containing the same elements in sorted order.

## Reference Solution
```python
def sort_array(arr):
    """
    Sorts an array in ascending order.

    Args:
        arr (list): The input array to be sorted.

    Returns:
        list: A new array containing the same elements in sorted order.
    """
    return sorted(arr)

# Example usage:
if __name__ == "__main__":
    # Create a sample array
    sample_array = [5, 2, 9, 1, 7, 3]

    # Sort the array
    sorted_array = sort_array(sample_array)

    # Print the original and sorted arrays
    print("Original Array:", sample_array)
    print("Sorted Array:", sorted_array)
```
