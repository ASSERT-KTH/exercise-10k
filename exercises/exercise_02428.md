## Exercise
Create a program to find all prime numbers up to n, using a for loop and conditional statements.

## Reference Solution
```python
def find_primes(n):
    primes = []
    for i in range(2, n + 1):
        is_prime = True
        for j in range(2, int(i ** 0.5) + 1):
            if i % j == 0:
                is_prime = False
                break
        if is_prime:
            primes.append(i)
    return primes

n = int(input("Enter a number: "))
print(find_primes(n))
```
