## Exercise
Perform explicit type casting on a variable to change its data type, then print the result.

## Reference Solution
```python
# Initial variable
num_str = "123"

# Print initial type
print(f"Initial type: {type(num_str)}")

# Explicit type casting to integer
num_int = int(num_str)

# Print type after casting
print(f"Type after casting: {type(num_int)}")

# Print the result
print(f"Result after casting: {num_int}")
```

```python
# String to float
num_str = "123.45"
num_float = float(num_str)
print(f"String to float: {num_float}, Type: {type(num_float)}")

# Integer to string
num_int = 123
num_str = str(num_int)
print(f"Integer to string: {num_str}, Type: {type(num_str)}")

# List to tuple
num_list = [1, 2, 3]
num_tuple = tuple(num_list)
print(f"List to tuple: {num_tuple}, Type: {type(num_tuple)}")

# Set to list
num_set = {1, 2, 3}
num_list = list(num_set)
print(f"Set to list: {num_list}, Type: {type(num_list)}")
```
