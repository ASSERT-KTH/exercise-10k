## Exercise
Given an array of strings, write a function to remove the first occurrence of a specified element.

## Reference Solution
```python
def remove_first_occurrence(array, target):
    """
    Removes the first occurrence of a specified element from an array of strings.

    Args:
    array (list): The array of strings.
    target (str): The target string to be removed.

    Returns:
    list: The array with the first occurrence of the target string removed.
    """
    try:
        # Get the index of the first occurrence of the target string
        index = array.index(target)
        
        # Remove the element at the index
        array.pop(index)
        
        # Return the updated array
        return array
    
    except ValueError:
        # If the target string is not found in the array, return the original array
        return array

# Example usage:
array = ["apple", "banana", "orange", "banana", "grape"]
target = "banana"
print(remove_first_occurrence(array, target))  # Output: ["apple", "orange", "banana", "grape"]
```
