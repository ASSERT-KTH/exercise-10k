## Exercise
Write a program that takes in a variable and its value as input, and then determines the data type of the variable. The variable can be an integer, float, string, or boolean. The program should output the data type of the variable.

## Reference Solution
```python
def typeof(var):
    """
    Determine the data type of a variable.

    Args:
        var: The variable to check.

    Returns:
        The data type of the variable as a string.
    """
    if isinstance(var, bool):
        return "boolean"
    elif isinstance(var, int):
        return "integer"
    elif isinstance(var, str):
        return "string"
    elif isinstance(var, float):
        return "float"
    else:
        return "unknown"

x = 5
print(typeof(x))  # Output: "integer"
```
