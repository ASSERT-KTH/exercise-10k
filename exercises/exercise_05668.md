## Exercise
Write a program that takes a number as input and outputs its square root.

## Reference Solution
```python
import math

def calculate_square_root(num):
    if num < 0:
        return "Error: Square root of negative number is not a real number."
    else:
        return math.sqrt(num)

# Take user input
num = float(input("Enter a number: "))

# Calculate and print square root
print("The square root of", num, "is", calculate_square_root(num))
```
