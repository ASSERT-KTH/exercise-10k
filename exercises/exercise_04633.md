## Exercise
Write a program that takes an array and a subset size as input, and returns all possible subsets of the specified size.

## Reference Solution
```python
import itertools

def get_subsets(array, subset_size):
    """
    Generate all possible subsets of a specified size from the given array.

    Args:
        array (list): The input array.
        subset_size (int): The size of the subsets to generate.

    Returns:
        list: A list of all possible subsets of the specified size.

    Raises:
        ValueError: If the subset size is larger than the array size.
    """
    if subset_size > len(array):
        raise ValueError("Subset size cannot be larger than array size")

    # Use itertools.combinations to generate all possible subsets of the specified size
    subsets = list(itertools.combinations(array, subset_size))

    return subsets


# Example usage:
if __name__ == "__main__":
    array = [1, 2, 3, 4, 5]
    subset_size = 3

    subsets = get_subsets(array, subset_size)
    print("Subsets of size", subset_size, ":", subsets)
```
