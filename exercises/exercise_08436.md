## Exercise
Write a program that takes in an hour and outputs the corresponding time of day using conditional statements.

## Reference Solution
```python
def get_time_of_day(hour):
    """
    Returns the time of day based on the given hour.

    Args:
        hour (int): The hour of the day (0-23)

    Returns:
        str: The time of day (e.g. "Morning", "Afternoon", etc.)
    """
    if hour < 0 or hour > 23:
        return "Invalid hour"

    if hour < 6:
        return "Late Night"
    elif hour < 12:
        return "Morning"
    elif hour < 17:
        return "Afternoon"
    elif hour < 20:
        return "Evening"
    else:
        return "Night"

def main():
    hour = int(input("Enter an hour (0-23): "))
    print(get_time_of_day(hour))

if __name__ == "__main__":
    main()
```
