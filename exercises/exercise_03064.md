## Exercise
Write a program that takes an integer as input and prints out whether it is even or odd.

## Reference Solution
```python
def check_even_or_odd(num: int) -> None:
    """
    Checks if a number is even or odd and prints the result.

    Args:
        num (int): The number to check.
    """
    if num % 2 == 0:
        print(f"{num} is even.")
    else:
        print(f"{num} is odd.")

def main() -> None:
    # Get user input
    num = int(input("Enter an integer: "))

    # Check if the number is even or odd
    check_even_or_odd(num)

if __name__ == "__main__":
    main()
```
