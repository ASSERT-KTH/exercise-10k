## Exercise
Explain the concept of a set data type and provide examples of declaring and using sets.

## Reference Solution
```python
# Declare an empty set
empty_set = set()

# Declare a set with initial elements
fruits = {"apple", "banana", "cherry"}

# Declare a set from a list
numbers = set([1, 2, 2, 3, 4, 4, 5])
print(numbers)  # Output: {1, 2, 3, 4, 5}
```

```python
# Create a set
colors = {"red", "green", "blue"}

# Add a single element
colors.add("yellow")
print(colors)  # Output: {'red', 'green', 'blue', 'yellow'}

# Add multiple elements
colors.update(["purple", "orange"])
print(colors)  # Output: {'red', 'green', 'blue', 'yellow', 'purple', 'orange'}
```

```python
# Create a set
numbers = {1, 2, 3, 4, 5}

# Remove a specified element
numbers.remove(3)
print(numbers)  # Output: {1, 2, 4, 5}

# Remove an arbitrary element
removed_number = numbers.pop()
print(numbers)  # Output: {1, 2, 4} or {1, 2, 5}
print(removed_number)  # Output: 5 or 4
```

```python
# Create two sets
set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}

# Union
union_set = set1 | set2
print(union_set)  # Output: {1, 2, 3, 4, 5, 6}

# Intersection
intersection_set = set1 & set2
print(intersection_set)  # Output: {3, 4}

# Difference
difference_set = set1 - set2
print(difference_set)  # Output: {1, 2}

# Symmetric Difference
symmetric_difference_set = set1 ^ set2
print(symmetric_difference_set)  # Output: {1, 2, 5, 6}
```
