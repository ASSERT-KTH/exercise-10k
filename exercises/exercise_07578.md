## Exercise
Write a function that uses internationalization to provide support for multiple languages or regions.

## Reference Solution
```python
import gettext

# Set up internationalization
def setup_i18n(language):
    """
    Set up internationalization for the given language.

    Args:
        language (str): The language code (e.g., 'en', 'fr', 'es')
    """
    lang = gettext.translation('myapp', languages=[language], fallback=True)
    lang.install()

# Define a function that uses internationalization
def greet(name, language):
    """
    Greet the user in their preferred language.

    Args:
        name (str): The user's name
        language (str): The language code (e.g., 'en', 'fr', 'es')
    """
    setup_i18n(language)
    print(gettext.gettext(f"Hello, {name}!"))

# Example usage
if __name__ == "__main__":
    greet("John", "en")  # Output: Hello, John!
    greet("Jean", "fr")  # Output: Bonjour, Jean!
    greet("Juan", "es")  # Output: Hola, Juan!
```
