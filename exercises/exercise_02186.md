## Exercise
Create a function to check if two given strings are anagrams of each other. The function should return True if the strings are anagrams, and False otherwise.

## Reference Solution
```python
def are_anagrams(str1, str2):
    """
    This function checks if two given strings are anagrams of each other.
    
    Args:
        str1 (str): The first string to be compared.
        str2 (str): The second string to be compared.
    
    Returns:
        bool: True if the strings are anagrams, False otherwise.
    """

    # First, we convert both strings to lower case to make the comparison case-insensitive
    str1 = str1.lower()
    
    # We do the same for the second string
    str2 = str2.lower()
    
    # We remove any spaces from both strings, as anagrams typically do not consider spaces
    str1 = str1.replace(" ", "")
    
    # We do the same for the second string
    str2 = str2.replace(" ", "")
    
    # We check if the sorted versions of both strings are equal
    # If they are, then the original strings are anagrams of each other
    return sorted(str1) == sorted(str2)

# Test the function
print(are_anagrams("Listen", "Silent"))  # Should return True
print(are_anagrams("Hello", "World"))  # Should return False
```
