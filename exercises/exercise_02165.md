## Exercise
Ask the user for the side length of a cube and print its volume.

## Reference Solution
```python
def calculate_cube_volume(side_length):
    """
    Calculate the volume of a cube given its side length.

    Args:
        side_length (float): The length of a side of the cube.

    Returns:
        float: The volume of the cube.
    """
    return side_length ** 3

def main():
    # Ask the user for the side length of the cube
    while True:
        try:
            side_length = float(input("Enter the side length of the cube: "))
            if side_length <= 0:
                print("Side length must be a positive number.")
            else:
                break
        except ValueError:
            print("Invalid input. Please enter a number.")

    # Calculate and print the volume of the cube
    volume = calculate_cube_volume(side_length)
    print(f"The volume of the cube is: {volume:.2f} cubic units")

if __name__ == "__main__":
    main()
```
