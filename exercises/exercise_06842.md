## Exercise
Write a program that takes an array of integers as input and returns a new array where the elements are reversed. For example, if the input array is [1, 2, 3, 4, 5], the output array should be [5, 4, 3, 2, 1]. The program should handle arrays of varying lengths and should not modify the original array.

## Reference Solution
```python
function reverse(arr) {
  const arr2 = [];
  for (let i = 0; i < arr.length; i++) {
    arr2.push(arr[arr.length - i - 1]);
  }
  return arr2;
}

const arr = [1, 2, 3, 4, 5];
const reversedArr = reverse(arr);
console.log(reversedArr); // [5, 4, 3, 2, 1]
```
