## Exercise
Write a program that takes an array of integers and two indices as input, and swaps the elements at the given indices. For example, if the input array is [1, 2, 3, 4, 5] and the indices are 1 and 3, the output should be [1, 4, 3, 2, 5].

## Reference Solution
```python
#include <iostream>

int main() {
    int n[5] = {1, 2, 3, 4, 5};
    int i = 1, j = 3;
    int temp;

    std::cout << "Array before swapping: ";
    for (int k = 0; k < 5; k++) {
        std::cout << n[k] << " ";
    }
    std::cout << std::endl;

    temp = n[i];
    n[i] = n[j];
    n[j] = temp;

    std::cout << "Array after swapping: ";
    for (int k = 0; k < 5; k++) {
        std::cout << n[k] << " ";
    }
    std::cout << std::endl;

    return 0;
}
```
