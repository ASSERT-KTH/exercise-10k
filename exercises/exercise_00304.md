## Exercise
Write a function that filters out elements from a list that do not meet a specified condition.

## Reference Solution
```python
def filter_list(condition, lst):
    """
    Filters out elements from a list that do not meet a specified condition.

    Args:
        condition (function): A function that takes one argument and returns a boolean value.
        lst (list): The list to be filtered.

    Returns:
        list: A new list containing only the elements that meet the condition.
    """
    return [element for element in lst if condition(element)]
```

```python
def is_odd(num):
    return num % 2 != 0

numbers = [1, 2, 3, 4, 5, 6]
filtered_numbers = filter_list(is_odd, numbers)
print(filtered_numbers)  # Output: [1, 3, 5]
```

```python
def starts_with_vowel(s):
    return s.lower()[0] in 'aeiou'

words = ['apple', 'banana', 'orange', 'kiwi']
filtered_words = filter_list(starts_with_vowel, words)
print(filtered_words)  # Output: ['apple', 'orange']
```
