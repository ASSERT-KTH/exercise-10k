## Exercise
Write a program that checks if a number is positive, negative, or zero. If the number is positive, print 'Positive'. If the number is negative, print 'Negative'. If the number is zero, print 'Zero'.

## Reference Solution
```python
def check_number(num):
    """
    Checks if a number is positive, negative, or zero.

    Args:
    num (int or float): The number to check.

    Returns:
    None
    """
    if num > 0:
        print('Positive')
    elif num < 0:
        print('Negative')
    else:
        print('Zero')

# Example usage:
if __name__ == "__main__":
    num = float(input("Enter a number: "))
    check_number(num)
```
