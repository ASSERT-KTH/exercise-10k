## Exercise
Write a program that takes a list of integers and a lambda function as input, and returns a new list containing the results of applying the lambda function to each element in the original list.

## Reference Solution
```python
# -*- coding: utf-8 -*-

from functools import reduce
from typing import List, Callable

def my_map(f: Callable[[int], int], lst: List[int]) -> List[int]:
    """
    Applies a given lambda function to each element in a list of integers.
    
    Parameters:
    f (Callable[[int], int]): A lambda function that takes an integer and returns an integer.
    lst (List[int]): A list of integers.
    
    Returns:
    List[int]: A new list containing the results of applying the lambda function to each element in the original list.
    """
    return list(map(f, lst))

# Example usage:
numbers = [1, 2, 3, 4, 5]
double_numbers = my_map(lambda x: x * 2, numbers)
print(double_numbers)  # Output: [2, 4, 6, 8, 10]
```
