## Exercise
Write a program that asks the user for a string, and prints out the string with all characters converted to uppercase or lowercase

## Reference Solution
```python
def convert_case():
    """
    Asks the user for a string and prints out the string with all characters converted to either uppercase or lowercase.
    """
    user_string = input("Please enter a string: ")
    print("Choose an option:")
    print("1. Convert to uppercase")
    print("2. Convert to lowercase")

    option = input("Enter your choice (1/2): ")

    if option == "1":
        print(user_string.upper())
    elif option == "2":
        print(user_string.lower())
    else:
        print("Invalid option. Please choose 1 or 2.")

def main():
    convert_case()

if __name__ == "__main__":
    main()
```
