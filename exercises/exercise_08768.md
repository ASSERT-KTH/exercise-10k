## Exercise
Create a function to remove non-space characters from a given string.

## Reference Solution
```python
def remove_non_space_characters(input_string):
    """
    Removes non-space characters from a given string.

    Args:
        input_string (str): The input string to process.

    Returns:
        str: A string containing only the space characters from the input string.
    """
    return ''.join(char for char in input_string if char.isspace())

# Example usage
if __name__ == "__main__":
    input_str = "Hello, World! This is a test string."
    result = remove_non_space_characters(input_str)
    print(f"Input string: {input_str}")
    print(f"Result string: '{result}'")
```
