## Exercise
Apply a function to each value in a dictionary.

## Reference Solution
```python
# Define a function to apply to each value in the dictionary
def square(x):
    """
    This function takes a number as input and returns its square.
    
    Parameters:
    x (int or float): The number to be squared.
    
    Returns:
    int or float: The square of the input number.
    """
    return x ** 2

# Define a dictionary
my_dict = {'a': 1, 'b': 2, 'c': 3, 'd': 4}

# Apply the function to each value in the dictionary
for key, value in my_dict.items():
    # Apply the function to the value and update the dictionary
    my_dict[key] = square(value)

# Print the updated dictionary
print(my_dict)
```
