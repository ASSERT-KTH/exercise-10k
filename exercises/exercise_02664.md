## Exercise
Write a program that takes a string and a character as input, and returns the number of occurrences of the character in the string.

## Reference Solution
```python
#include <stdio.h>
#include <string.h>

int count(char string[], char ch) {
    int count = 0;
    for (int i = 0; i < strlen(string); i++) {
        if (string[i] == ch) {
            count++;
        }
    }
    return count;
}

int main() {
    char string[] = "hello";
    char ch = 'l';
    printf("%d\n", count(string, ch));
    return 0;
}
```
