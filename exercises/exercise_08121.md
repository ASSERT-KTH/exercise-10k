## Exercise
Write a program that checks if a given number is positive, negative, or zero using if/else statements

## Reference Solution
```python
def check_number(num):
    """
    Checks if a given number is positive, negative, or zero.

    Args:
        num (int): The number to check.

    Returns:
        str: A string indicating whether the number is positive, negative, or zero.
    """
    if num > 0:
        return "Positive"
    elif num < 0:
        return "Negative"
    else:
        return "Zero"

def main():
    # Example usage
    num = int(input("Enter a number: "))
    result = check_number(num)
    print(f"The number {num} is {result}.")

if __name__ == "__main__":
    main()
```
