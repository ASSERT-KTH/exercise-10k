## Exercise
Create a program that simulates a coin flip. The program should print out either 'Heads' or 'Tails' randomly.

## Reference Solution
```python
import random

def coin_flip():
    """Simulate a coin flip and return the result."""
    return 'Heads' if random.randint(0, 1) == 0 else 'Tails'

def main():
    """Simulate a coin flip and print the result."""
    result = coin_flip()
    print(result)

if __name__ == "__main__":
    main()
```
