## Exercise
Work with the date and time data type to store and manipulate dates and times

## Reference Solution
```python
import datetime
```

```python
date_object = datetime.date(2022, 12, 25)
print(date_object)  # Output: 2022-12-25
```

```python
time_object = datetime.time(12, 30, 0)
print(time_object)  # Output: 12:30:00
```

```python
datetime_object = datetime.datetime(2022, 12, 25, 12, 30, 0)
print(datetime_object)  # Output: 2022-12-25 12:30:00
```

```python
date_object = datetime.date(2022, 12, 25)
new_date_object = date_object + datetime.timedelta(days=5)
print(new_date_object)  # Output: 2022-12-30
```

```python
datetime_object = datetime.datetime(2022, 12, 25, 12, 30, 0)
new_datetime_object = datetime_object - datetime.timedelta(hours=2)
print(new_datetime_object)  # Output: 2022-12-25 10:30:00
```

```python
date_object = datetime.date(2022, 12, 25)
formatted_date = date_object.strftime("%Y-%m-%d")
print(formatted_date)  # Output: 2022-12-25
```

```python
datetime_object = datetime.datetime(2022, 12, 25, 12, 30, 0)
formatted_datetime = datetime_object.strftime("%Y-%m-%d %H:%M:%S")
print(formatted_datetime)  # Output: 2022-12-25 12:30:00
```

```python
date_string = "2022-12-25"
date_object = datetime.datetime.strptime(date_string, "%Y-%m-%d").date()
print(date_object)  # Output: 2022-12-25
```

```python
datetime_string = "2022-12-25 12:30:00"
datetime_object = datetime.datetime.strptime(datetime_string, "%Y-%m-%d %H:%M:%S")
print(datetime_object)  # Output: 2022-12-25 12:30:00
```
