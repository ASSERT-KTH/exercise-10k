## Exercise
Write a function that takes two numbers as input and returns the first number raised to the power of the second.

## Reference Solution
```python
def potencia(base, exponente):
    resultado = base ** exponente
    return resultado

base = int(input("Ingrese el primer numero: "))
exponente = int(input("Ingrese el segundo numero: "))

print("El resultado es: ", potencia(base, exponente))
```
