## Exercise
Create a program that checks if a given year is a leap year. The program should take the year as input and print 'True' if it is a leap year, 'False' otherwise.

## Reference Solution
```python
def is_leap(year):
    if year % 4 == 0:
        if year % 100 == 0:
            if year % 400 == 0:
                return True
            else:
                return False
        else:
            return True
    else:
        return False

year = int(input())
print(is_leap(year))
```
