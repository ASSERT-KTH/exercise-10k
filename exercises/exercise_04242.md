## Exercise
Create a program that takes a student's score as input and prints the corresponding grade (A, B, C, D, F) based on the following criteria: 90-100 (A), 80-89 (B), 70-79 (C), 60-69 (D), below 60 (F).

## Reference Solution
```python
# Author: [Your Name]
# Date: [Today's Date]
# Description: This program takes a student's score as input and prints the corresponding grade.

def grade(score):
    if score >= 90:
        return "A"
    elif score >= 80:
        return "B"
    elif score >= 70:
        return "C"
    elif score >= 60:
        return "D"
    else:
        return "F"

score = float(input("Enter a score: "))
print("Grade:", grade(score))
```
