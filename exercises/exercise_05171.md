## Exercise
Write a function to check if a key exists in a dictionary.

## Reference Solution
```python
def check_key(dictionary, key):
    """
    Checks if a key exists in a dictionary.

    Args:
        dictionary (dict): The dictionary to be checked.
        key (any): The key to be searched for.

    Returns:
        bool: True if the key exists, False otherwise.
    """

    # First, we need to check if the dictionary is actually a dictionary
    # This is to prevent an AttributeError when trying to use the 'in' keyword
    if not isinstance(dictionary, dict):
        raise TypeError("The first argument must be a dictionary")

    # Now, we can use the 'in' keyword to check if the key exists in the dictionary
    # The 'in' keyword returns True if the key is found, and False otherwise
    return key in dictionary

# Example usage:
my_dict = {"a": 1, "b": 2, "c": 3}
print(check_key(my_dict, "a"))  # Output: True
print(check_key(my_dict, "d"))  # Output: False
```
