## Exercise
Create a program that takes in a string and outputs the reversed string based on the following conditions: if the string is empty, output an error message, if the string has an odd length, reverse the string, if the string has an even length, output the original string.

## Reference Solution
```python
def reverse_string(s):
    """
    This function takes in a string and outputs the reversed string based on its length.

    If the string is empty, it outputs an error message.
    If the string has an odd length, it reverses the string.
    If the string has an even length, it outputs the original string.

    Args:
        s (str): The input string.

    Returns:
        str: The output string.
    """

    # Check if the string is empty
    if not s:
        # If the string is empty, output an error message
        return "Error: Input string is empty."

    # Check if the string has an odd or even length
    if len(s) % 2 != 0:
        # If the string has an odd length, reverse the string
        return s[::-1]
    else:
        # If the string has an even length, output the original string
        return s


# Example usage
def main():
    print(reverse_string(""))  # Output: Error: Input string is empty.
    print(reverse_string("a"))  # Output: a
    print(reverse_string("ab"))  # Output: ab
    print(reverse_string("abc"))  # Output: cba


if __name__ == "__main__":
    main()
```
