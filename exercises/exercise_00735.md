## Exercise
Write a program that takes a decimal number as input and outputs its equivalent fraction.

## Reference Solution
```python
from fractions import Fraction

def decimal_to_fraction(decimal):
    """
    Convert a decimal number to its equivalent fraction.

    Args:
        decimal (float): The decimal number to convert.

    Returns:
        str: The equivalent fraction as a string.
    """
    fraction = Fraction(decimal).limit_denominator()
    return f"{fraction.numerator}/{fraction.denominator}"

def main():
    # Get the decimal number from the user
    decimal = float(input("Enter a decimal number: "))

    # Convert the decimal to a fraction
    fraction = decimal_to_fraction(decimal)

    # Print the result
    print(f"The equivalent fraction is: {fraction}")

if __name__ == "__main__":
    main()
```
