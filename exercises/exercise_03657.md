## Exercise
Create a function that takes a string as an argument and returns the string in title case

## Reference Solution
```python
def to_title_case(input_string: str) -> str:
    """
    Converts a string to title case.

    Args:
        input_string (str): The input string to be converted.

    Returns:
        str: The input string in title case.
    """
    return input_string.title()

# Example usage:
def main():
    input_str = "hello world"
    print(to_title_case(input_str))  # Outputs: "Hello World"

if __name__ == "__main__":
    main()
```
