## Exercise
Write a function that takes a dictionary and a condition, then returns a new dictionary with only the key-value pairs that meet the condition

## Reference Solution
```python
def filter_dict(dictionary, condition):
    """
    Filter a dictionary based on a given condition.

    Args:
    - dictionary (dict): The input dictionary to be filtered.
    - condition (function): A function that takes a key-value pair and returns a boolean.

    Returns:
    - dict: A new dictionary with only the key-value pairs that meet the condition.
    """
    return {key: value for key, value in dictionary.items() if condition(key, value)}

# Example usage:
if __name__ == "__main__":
    # Define a sample dictionary
    sample_dict = {
        "a": 1,
        "b": 2,
        "c": 3,
        "d": 4,
    }

    # Define a condition function
    def is_even(key, value):
        return value % 2 == 0

    # Filter the dictionary
    filtered_dict = filter_dict(sample_dict, is_even)

    # Print the result
    print("Original Dictionary:", sample_dict)
    print("Filtered Dictionary:", filtered_dict)
```
