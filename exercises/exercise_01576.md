## Exercise
Create a function that uses the reduce function to apply a given function to a list of numbers and return the result.

## Reference Solution
```python
from functools import reduce

def reduce_function(func, numbers):
    """
    This function uses the reduce function to apply a given function to a list of numbers and return the result.

    Args:
        func (function): A function that takes two arguments.
        numbers (list): A list of numbers.

    Returns:
        The result of applying the function to the list of numbers.
    """

    # Check if the input list is empty
    if not numbers:
        return None  # Return None for empty list

    # We use the reduce function to apply the given function to the list of numbers
    # The reduce function applies a rolling computation to sequential pairs of values in a list
    # For example, reduce(lambda x, y: x+y, [1, 2, 3, 4]) is equivalent to ((1+2)+3)+4
    result = reduce(func, numbers)

    # Finally, we return the result
    return result


# Example usage:
# Define a function that adds two numbers
def add(x, y):
    return x + y

# Create a list of numbers
numbers = [1, 2, 3, 4]

# Use the reduce_function to apply the add function to the list of numbers
result = reduce_function(add, numbers)

# Print the result
print(result)  # Output: 10
```
