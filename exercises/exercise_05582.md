## Exercise
Write a function that takes a name as an argument and returns a personalized greeting.

## Reference Solution
```python
def personalized_greeting(name: str) -> str:
    """
    Returns a personalized greeting based on the provided name.

    Args:
        name (str): The name to be used in the greeting.

    Returns:
        str: A personalized greeting.
    """
    return f"Hello, {name}!"

# Example usage:
if __name__ == "__main__":
    name = "John"
    print(personalized_greeting(name))  # Output: Hello, John!
```
