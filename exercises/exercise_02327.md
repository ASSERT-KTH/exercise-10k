## Exercise
Write a program to print a quote to the screen, with the quote and the author on separate lines.

## Reference Solution
```python
def print_quote(quote, author):
    """
    Prints a quote to the screen, with the quote and the author on separate lines.

    Args:
        quote (str): The quote to be printed.
        author (str): The author of the quote.
    """
    print(f"\"{quote}\"")
    print(f"- {author}")

def main():
    # Example usage
    quote = "The only way to do great work is to love what you do."
    author = "Steve Jobs"
    print_quote(quote, author)

if __name__ == "__main__":
    main()
```
