## Exercise
Write a program that takes a user input and determines its data type. The program should be able to identify whether the input is an integer, float, or string. If the input cannot be classified into these categories, it should print an error message.

## Reference Solution
```python
def get_user_input():
    """Gets user input from the console."""
    user_input = input("Please enter a value: ")
    return user_input

def determine_data_type(user_input):
    """Determines the data type of the user input."""
    try:
        # Try to convert the input to an integer
        int(user_input)
        return "Integer"
    except ValueError:
        try:
            # If it's not an integer, try to convert it to a float
            float(user_input)
            return "Float"
        except ValueError:
            # If it's neither an integer nor a float, it's a string
            return "String"

def main():
    user_input = get_user_input()
    data_type = determine_data_type(user_input)
    print(f"The data type of '{user_input}' is: {data_type}")

if __name__ == "__main__":
    main()
```
