## Exercise
Write a program that takes an input and validates it using a conditional statement.

## Reference Solution
```python
def validate_input(input_value):
    """
    Validates if the input is a positive integer.

    Args:
        input_value (str): The input to be validated.

    Returns:
        bool: True if the input is a positive integer, False otherwise.
    """
    try:
        # Attempt to convert the input to an integer
        num = int(input_value)
        
        # Check if the number is positive
        if num > 0:
            return True
        else:
            return False
    except ValueError:
        # If the input cannot be converted to an integer, it's not valid
        return False

def main():
    # Get the input from the user
    input_value = input("Enter a positive integer: ")

    # Validate the input
    if validate_input(input_value):
        print("The input is a positive integer.")
    else:
        print("The input is not a positive integer.")

if __name__ == "__main__":
    main()
```
