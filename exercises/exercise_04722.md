## Exercise
Write a function that takes a name as an argument and prints a personalized greeting message.

## Reference Solution
```python
# Define a function called 'personalized_greeting' that takes one argument, 'name'.
def personalized_greeting(name):
    """
    This function prints a personalized greeting message.

    Args:
        name (str): The name of the person to be greeted.

    Returns:
        None
    """

    # Check if the input 'name' is a string. If not, raise a TypeError with a descriptive message.
    if not isinstance(name, str):
        raise TypeError("The input 'name' must be a string.")

    # Use an f-string to format the greeting message with the provided 'name'.
    # This allows us to embed the 'name' variable inside the string.
    greeting_message = f"Hello, {name}!"

    # Print the formatted greeting message.
    print(greeting_message)

# Example usage:
personalized_greeting("John")
```
