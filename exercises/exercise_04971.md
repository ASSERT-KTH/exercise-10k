## Exercise
Write a function that joins a list of substrings into a single string based on a delimiter

## Reference Solution
```python
def join_substrings(substrings, delimiter):
    """
    Joins a list of substrings into a single string based on a delimiter.

    Args:
        substrings (list): A list of substrings to join.
        delimiter (str): The delimiter to use for joining.

    Returns:
        str: The joined string.
    """
    return delimiter.join(substrings)

# Example usage:
substrings = ["Hello", "world", "this", "is", "a", "test"]
delimiter = ", "
joined_string = join_substrings(substrings, delimiter)
print(joined_string)
```
