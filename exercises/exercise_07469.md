## Exercise
Write a program that converts a given dictionary into a list of its keys

## Reference Solution
```python
def dict_keys_to_list(input_dict):
    """
    Converts a dictionary into a list of its keys.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        list: A list of dictionary keys.
    """
    return list(input_dict.keys())

# Example usage
if __name__ == "__main__":
    example_dict = {"name": "John", "age": 30, "city": "New York"}
    keys_list = dict_keys_to_list(example_dict)
    print(keys_list)
```
