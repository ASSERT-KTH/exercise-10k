## Exercise
Write a program to sort an array in ascending order and print the sorted array.

## Reference Solution
```python
def sort_array(array):
    """
    Sorts an array in ascending order and prints the sorted array.

    Args:
        array (list): The input array to be sorted.
    """
    # Use the built-in sorted function to sort the array in ascending order
    sorted_array = sorted(array)

    # Print the sorted array
    print("Sorted Array:", sorted_array)


# Example usage
if __name__ == "__main__":
    # Define an example array
    example_array = [64, 34, 25, 12, 22, 11, 90]

    # Print the original array
    print("Original Array:", example_array)

    # Call the sort_array function to sort and print the array
    sort_array(example_array)
```
