## Exercise
Write a function that takes a dictionary as input and loops through all the values in the dictionary, printing out each value.

## Reference Solution
```python
def print_dict_values(input_dict):
    """
    Prints all values in the input dictionary.
    
    Args:
        input_dict (dict): The dictionary to print values from.
    """
    for value in input_dict.values():
        print(value)

# Example usage:
example_dict = {
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}

print("Dictionary Values:")
print_dict_values(example_dict)
```
