## Exercise
Create a function to initialize a dictionary with default values for a given set of keys.

## Reference Solution
```python
def initialize_dict(keys, default_value):
    """
    Initialize a dictionary with default values for a given set of keys.

    Args:
        keys (list): A list of keys to be used in the dictionary.
        default_value (any): The default value to be assigned to each key.

    Returns:
        dict: A dictionary with the given keys and default values.
    """

    # We start by creating an empty dictionary that will store our key-value pairs.
    result_dict = {}

    # Then, we iterate over each key in the given list of keys.
    for key in keys:
        # For each key, we assign the default value in the dictionary.
        result_dict[key] = default_value

    # Finally, we return the resulting dictionary with the default values.
    return result_dict


# Example usage:
keys = ['a', 'b', 'c']
default_value = 0
print(initialize_dict(keys, default_value))  # Output: {'a': 0, 'b': 0, 'c': 0}
```
