## Exercise
Write a function to check if a key is present in a dictionary.

## Reference Solution
```python
def check_key(dictionary, key):
    """
    This function checks if a key is present in a dictionary.

    Args:
        dictionary (dict): The dictionary to be checked.
        key (any): The key to be searched for.

    Returns:
        bool: True if the key is present, False otherwise.
    """

    # First, we need to check if the input is indeed a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("The input is not a dictionary.")

    # Now, we can use the 'in' keyword in Python to check if the key is present in the dictionary
    # The 'in' keyword returns True if the key is found, and False otherwise
    return key in dictionary

# Example usage:
my_dict = {'a': 1, 'b': 2, 'c': 3}
print(check_key(my_dict, 'a'))  # Output: True
print(check_key(my_dict, 'd'))  # Output: False
```
