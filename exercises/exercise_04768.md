## Exercise
Write a program that converts a temperature from Celsius to Fahrenheit or vice versa. If the user inputs 'C', the program converts the temperature from Celsius to Fahrenheit. If the user inputs 'F', the program converts the temperature from Fahrenheit to Celsius.

## Reference Solution
```python
def convert_celsius_to_fahrenheit(celsius):
    """Converts Celsius to Fahrenheit."""
    return (celsius * 9/5) + 32

def convert_fahrenheit_to_celsius(fahrenheit):
    """Converts Fahrenheit to Celsius."""
    return (fahrenheit - 32) * 5/9

def main():
    print("Temperature Conversion Program")
    print("--------------------------------")

    while True:
        choice = input("Enter 'C' to convert Celsius to Fahrenheit, 'F' to convert Fahrenheit to Celsius, or 'Q' to quit: ")
        
        if choice.upper() == 'C':
            celsius = float(input("Enter temperature in Celsius: "))
            fahrenheit = convert_celsius_to_fahrenheit(celsius)
            print(f"{celsius}째C is equal to {fahrenheit}째F")
        elif choice.upper() == 'F':
            fahrenheit = float(input("Enter temperature in Fahrenheit: "))
            celsius = convert_fahrenheit_to_celsius(fahrenheit)
            print(f"{fahrenheit}째F is equal to {celsius}째C")
        elif choice.upper() == 'Q':
            print("Goodbye!")
            break
        else:
            print("Invalid choice. Please try again.")

if __name__ == "__main__":
    main()
```
