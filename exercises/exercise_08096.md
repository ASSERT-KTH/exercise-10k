## Exercise
Write a program that takes a string as input and returns the string in upper case.

## Reference Solution
```python
def convert_to_upper_case(input_string: str) -> str:
    """
    Converts a given string to upper case.

    Args:
        input_string (str): The input string to be converted.

    Returns:
        str: The input string in upper case.
    """
    return input_string.upper()

def main():
    # Example usage
    input_str = input("Enter a string: ")
    upper_case_str = convert_to_upper_case(input_str)
    print("Upper case string:", upper_case_str)

if __name__ == "__main__":
    main()
```
