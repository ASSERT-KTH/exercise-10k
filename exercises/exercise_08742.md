## Exercise
Hide data within a variable to prevent unauthorized access.

## Reference Solution
```python
class MyClass:
    def __init__(self, data):
        self.__private_data = data  # Private variable

    def get_data(self):
        return self.__private_data  # Getter method

    def set_data(self, data):
        self.__private_data = data  # Setter method

# Create an instance of MyClass
obj = MyClass("Hello, World!")

# Try to access the private variable directly
try:
    print(obj.__private_data)
except AttributeError:
    print("Error: cannot access private variable directly")

# Use the getter method to access the private variable
print(obj.get_data())  # Output: Hello, World!

# Use the setter method to modify the private variable
obj.set_data("New data")
print(obj.get_data())  # Output: New data
```
