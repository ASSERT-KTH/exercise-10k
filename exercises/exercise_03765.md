## Exercise
Remove a specific key-value pair from a dictionary.

## Reference Solution
```python
def remove_key_value_pair(dictionary, key):
    """
    Removes a specific key-value pair from a dictionary.

    Args:
        dictionary (dict): The dictionary from which to remove the key-value pair.
        key (any): The key of the key-value pair to remove.

    Returns:
        dict: The updated dictionary with the key-value pair removed.
    """
    if key in dictionary:
        del dictionary[key]
    return dictionary

# Example usage:
my_dict = {"a": 1, "b": 2, "c": 3}
print(remove_key_value_pair(my_dict, "b"))  # Output: {'a': 1, 'c': 3}
```
