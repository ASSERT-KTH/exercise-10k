## Exercise
Write a function that checks if a given number is even or odd and returns the corresponding result.

## Reference Solution
```python
def check_even_or_odd(num: int) -> str:
    """
    Checks if a given number is even or odd and returns the corresponding result.

    Args:
        num (int): The number to check.

    Returns:
        str: "Even" if the number is even, "Odd" otherwise.
    """
    if num % 2 == 0:
        return "Even"
    else:
        return "Odd"

# Example usage:
def main():
    numbers = [10, 23, 44, 57, 92]
    for num in numbers:
        result = check_even_or_odd(num)
        print(f"{num} is {result}")

if __name__ == "__main__":
    main()
```
