## Exercise
Create a function that takes an array of integers as input and returns the maximum value in the array. If the array is empty, the function should return a message indicating that the array is empty.

## Reference Solution
```python
def find_max_value(array):
    """
    This function takes an array of integers as input and returns the maximum value in the array.
    If the array is empty, the function returns a message indicating that the array is empty.

    Parameters:
    array (list): A list of integers.

    Returns:
    int or str: The maximum value in the array if it's not empty, otherwise a message indicating that the array is empty.
    """

    # First, we check if the array is empty
    if not array:
        # If the array is empty, we return a message indicating that the array is empty
        return "The array is empty."

    # If the array is not empty, we use the built-in max function to find the maximum value
    # The max function returns the largest item in an iterable or the largest of two or more arguments
    max_value = max(array)

    # Finally, we return the maximum value
    return max_value

# Test the function
print(find_max_value([1, 2, 3, 4, 5]))  # Output: 5
print(find_max_value([]))  # Output: The array is empty.
```
