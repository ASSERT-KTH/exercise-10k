## Exercise
Create a function that takes a list of numbers as input and returns the maximum value. The function should be able to handle lists of varying lengths and should return an error message if the list is empty.

## Reference Solution
```python
def find_max_value(num_list):
    """
    This function finds the maximum value in a given list of numbers.

    Args:
        num_list (list): A list of numbers.

    Returns:
        max_value (int or float): The maximum value in the list.
        error_message (str): An error message if the list is empty.
    """

    # First, we check if the list is empty
    if not num_list:
        # If the list is empty, we return an error message
        return "Error: The list is empty."

    # If the list is not empty, we use the built-in max() function to find the maximum value
    # The max() function returns the largest item in an iterable or the largest of two or more arguments
    max_value = max(num_list)

    # Finally, we return the maximum value
    return max_value

# Test the function
print(find_max_value([1, 2, 3, 4, 5]))  # Output: 5
print(find_max_value([-1, -2, -3, -4, -5]))  # Output: -1
print(find_max_value([]))  # Output: Error: The list is empty.
```
