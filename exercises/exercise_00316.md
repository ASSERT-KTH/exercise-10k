## Exercise
Write a function that applies a specified transformation to each element of a given list.

## Reference Solution
```python
def transform_list(input_list, transformation):
    """
    Applies a specified transformation to each element of a given list.

    Args:
        input_list (list): The list to be transformed.
        transformation (function): A function that takes one argument and returns a transformed value.

    Returns:
        list: A new list with the transformation applied to each element.
    """

    # Initialize an empty list to store the transformed elements
    transformed_list = []

    # Iterate over each element in the input list
    for element in input_list:
        # Apply the transformation to the current element
        transformed_element = transformation(element)
        
        # Append the transformed element to the transformed list
        transformed_list.append(transformed_element)

    # Return the transformed list
    return transformed_list

# Example usage:
# Define a transformation function that squares a number
def square(x):
    return x ** 2

# Create a list of numbers
numbers = [1, 2, 3, 4, 5]

# Apply the square transformation to the list of numbers
squared_numbers = transform_list(numbers, square)

# Print the transformed list
print(squared_numbers)  # Output: [1, 4, 9, 16, 25]
```
