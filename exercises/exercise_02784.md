## Exercise
Write a program that searches for a specific element in a given array and returns its index if found.

## Reference Solution
```python
def find_element(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

# Example usage
arr = [1, 2, 3, 4, 5]
target = 3
result = find_element(arr, target)
if result != -1:
    print(f"Element {target} found at index {result}.")
else:
    print(f"Element {target} not found in the array.")
```
