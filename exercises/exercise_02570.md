## Exercise
Write a program that finds the first duplicate in an array using a while loop.

## Reference Solution
```python
def find_first_duplicate(arr):
    """
    Find the first duplicate in an array.

    Args:
    arr (list): The input array.

    Returns:
    int: The first duplicate in the array. If no duplicates are found, returns None.
    """
    seen = set()
    i = 0
    while i < len(arr):
        if arr[i] in seen:
            return arr[i]
        seen.add(arr[i])
        i += 1
    return None

# Example usage
arr = [2, 1, 3, 5, 3, 2]
print(find_first_duplicate(arr))  # Output: 3
```
