## Exercise
Create a program that takes a user's name and age as input, validates if the age is within the range of 1 to 120, and then outputs a greeting message with the user's name and age. If the age is not within the valid range, output an error message.

## Reference Solution
```python
// JavaScript code

// Define variables
var nome = "";
var idade = 0;

// Get input from user
nome = prompt("Enter your name: ");
idade = parseInt(prompt("Enter your age: "));

// Validate input
if (idade >= 1 && idade <= 120) {
  // Output greeting message
  alert("Hello, " + nome + "! You are " + idade + " years old.");
} else {
  // Output error message
  alert("Invalid age. Please enter an age between 1 and 120.");
}
```
