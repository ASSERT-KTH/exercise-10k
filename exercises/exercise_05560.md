## Exercise
Write a program that calculates the sum of digits in a given integer using a for loop.

## Reference Solution
```python
def sum_of_digits(n):
    n = abs(n)  # handle negative numbers
    sum = 0
    for digit in str(n):
        sum += int(digit)
    return sum

num = int(input("Enter an integer: "))
print("The sum of the digits is:", sum_of_digits(num))
```
