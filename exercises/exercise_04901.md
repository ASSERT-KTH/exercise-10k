## Exercise
Write a program that checks if a given number is greater than 10. If it is, print 'The number is greater than 10'. Otherwise, print 'The number is less than or equal to 10'.

## Reference Solution
```python
def check_number(num):
    """
    Checks if a given number is greater than 10.

    Args:
        num (int): The number to check.

    Returns:
        None
    """
    if num > 10:
        print('The number is greater than 10')
    else:
        print('The number is less than or equal to 10')

# Example usage:
num = int(input("Enter a number: "))
check_number(num)
```
