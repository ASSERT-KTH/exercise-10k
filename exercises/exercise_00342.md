## Exercise
Create a function that filters a list based on a given condition.

## Reference Solution
```python
def filter_list(condition, lst):
    """
    Filters a list based on a given condition.

    Args:
    condition (function): A function that takes one argument and returns a boolean value.
    lst (list): The list to be filtered.

    Returns:
    list: A new list containing only the elements for which the condition is True.
    """
    return [element for element in lst if condition(element)]

# Example usage:
# Define a condition function
def is_even(num):
    return num % 2 == 0

# Define a list
numbers = [1, 2, 3, 4, 5, 6]

# Filter the list using the condition function
even_numbers = filter_list(is_even, numbers)

print(even_numbers)  # Output: [2, 4, 6]
```
