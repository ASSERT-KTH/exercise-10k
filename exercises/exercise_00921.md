## Exercise
Write a program that checks if a given number is positive or negative using if/else statement

## Reference Solution
```python
def check_number_sign(num):
    """
    Checks if a given number is positive or negative.

    Args:
        num (int): The number to check.

    Returns:
        str: A message indicating whether the number is positive or negative.
    """
    if num > 0:
        return f"{num} is a positive number."
    elif num < 0:
        return f"{num} is a negative number."
    else:
        return f"{num} is neither positive nor negative (it's zero)."

def main():
    # Example usage:
    num = int(input("Enter a number: "))
    print(check_number_sign(num))

if __name__ == "__main__":
    main()
```
