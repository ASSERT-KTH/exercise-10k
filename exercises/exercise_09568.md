## Exercise
Write a program that takes an array of integers as input and returns the maximum value in the array. The program should use a basic algorithm to iterate through the array and keep track of the maximum value found so far.

## Reference Solution
```python
public class Main {
    public static int max(int[] a) {
        int max = a[0];
        for (int i = 1; i < a.length; i++) {
            if (a[i] > max) {
                max = a[i];
            }
        }
        return max;
    }

    public static void main(String[] args) {
        int[] a = {4, 5, 2, 25, 7, 89, 32, 56, 75};
        System.out.println(max(a));
    }
}
```
