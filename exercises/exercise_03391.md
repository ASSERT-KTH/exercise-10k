## Exercise
Write a function to find the average of all elements in a list.

## Reference Solution
```python
def calculate_average(lst):
    """
    Calculate the average of all elements in a list.

    Args:
        lst (list): A list of numbers.

    Returns:
        float: The average of the list elements.

    Raises:
        ValueError: If the list is empty.
        TypeError: If the list contains non-numeric values.
    """
    if not lst:
        raise ValueError("Cannot calculate average of an empty list")

    try:
        return sum(lst) / len(lst)
    except TypeError:
        raise TypeError("List contains non-numeric values")

# Example usage:
numbers = [1, 2, 3, 4, 5]
print(calculate_average(numbers))  # Output: 3.0
```
