## Exercise
Create a function that takes a dictionary and a function, and returns a new dictionary where each value is the result of applying the function to the corresponding value in the original dictionary.

## Reference Solution
```python
def transform_dict(input_dict, func):
    """
    This function takes a dictionary and a function as input, applies the function to each value in the dictionary, 
    and returns a new dictionary with the results.

    Args:
        input_dict (dict): The input dictionary.
        func (function): The function to be applied to each value in the dictionary.

    Returns:
        dict: A new dictionary where each value is the result of applying the function to the corresponding value in the original dictionary.
    """

    # Initialize an empty dictionary to store the results
    result = {}

    # Iterate over each key-value pair in the input dictionary
    for key, value in input_dict.items():
        
        # Apply the function to the current value and store the result in the new dictionary
        result[key] = func(value)
    
    # Return the new dictionary with the transformed values
    return result

# Example usage:
# Define a simple function to square a number
def square(x):
    return x ** 2

# Create a dictionary with some numbers
numbers = {'a': 1, 'b': 2, 'c': 3}

# Use the transform_dict function to square each number in the dictionary
squared_numbers = transform_dict(numbers, square)

# Print the result
print(squared_numbers)  # Output: {'a': 1, 'b': 4, 'c': 9}
```
