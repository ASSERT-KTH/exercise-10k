## Exercise
Write a function called 'find_first_duplicate' that finds and returns the first duplicate in a given list.

## Reference Solution
```python
def find_first_duplicate(lst):
    """
    Finds and returns the first duplicate in a given list.

    Args:
    lst (list): The input list.

    Returns:
    The first duplicate in the list, or None if no duplicates are found.
    """
    seen = set()
    for item in lst:
        if item in seen:
            return item
        seen.add(item)
    return None

# Example usage
if __name__ == "__main__":
    print(find_first_duplicate([2, 1, 3, 5, 3, 2]))  # Output: 3
    print(find_first_duplicate([1, 2, 3, 4, 5]))  # Output: None
```
