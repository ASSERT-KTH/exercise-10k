## Exercise
Write a program to convert a decimal number to its hexadecimal representation.

## Reference Solution
```python
def decimal_to_hexadecimal(decimal_number):
    """
    Convert a decimal number to its hexadecimal representation.

    Args:
        decimal_number (int): The decimal number to convert.

    Returns:
        str: The hexadecimal representation of the decimal number.
    """
    return hex(decimal_number)[2:].upper()

def main():
    decimal_number = int(input("Enter a decimal number: "))
    hexadecimal_number = decimal_to_hexadecimal(decimal_number)
    print(f"The hexadecimal representation of {decimal_number} is {hexadecimal_number}")

if __name__ == "__main__":
    main()
```
