## Exercise
Write a program to check if a given string is symmetric.

## Reference Solution
```python
def is_symmetric(s: str) -> bool:
    """
    Check if a given string is symmetric.

    Args:
        s (str): The input string.

    Returns:
        bool: True if the string is symmetric, False otherwise.
    """
    return s == s[::-1]

def main():
    # Example usage:
    strings = ["madam", "hello", "radar", "python"]

    for s in strings:
        if is_symmetric(s):
            print(f"'{s}' is a symmetric string.")
        else:
            print(f"'{s}' is not a symmetric string.")

if __name__ == "__main__":
    main()
```
