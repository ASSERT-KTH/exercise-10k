## Exercise
Write a program that checks the data type of a given variable.

## Reference Solution
```python
def check_data_type(var):
    """
    Checks the data type of a given variable.

    Args:
        var: The variable to check.

    Returns:
        str: The data type of the variable.
    """
    return type(var).__name__

# Example usage:
if __name__ == "__main__":
    # Test variables
    test_variables = [
        10,  # Integer
        3.14,  # Float
        "Hello",  # String
        True,  # Boolean
        [1, 2, 3],  # List
        {"a": 1, "b": 2},  # Dictionary
        (1, 2, 3),  # Tuple
        None,  # NoneType
    ]

    # Check data type for each test variable
    for var in test_variables:
        print(f"Variable: {var}, Data Type: {check_data_type(var)}")
```
