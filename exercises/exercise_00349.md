## Exercise
Write a program to insert an element at a specific position in an array.

## Reference Solution
```python
def insert_element(array, position, element):
    """
    Inserts an element at a specific position in an array.

    Args:
        array (list): The input array.
        position (int): The position where the element will be inserted.
        element: The element to be inserted.

    Returns:
        list: The updated array with the element inserted.
    """
    # Check if the position is within the bounds of the array
    if position < 0 or position > len(array):
        raise IndexError("Position out of range")

    # Insert the element at the specified position
    array.insert(position, element)

    return array


# Example usage:
array = [1, 2, 3, 5]
position = 3
element = 4

print("Original array:", array)
print("Position:", position)
print("Element to be inserted:", element)

updated_array = insert_element(array, position, element)

print("Updated array:", updated_array)
```
