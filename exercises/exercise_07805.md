## Exercise
Write a program that converts a time from one format to another (e.g. from 12-hour to 24-hour format).

## Reference Solution
```python
from datetime import datetime

def convert_to_24_hour_format(time_str):
    """
    Converts a time from 12-hour format to 24-hour format.

    Args:
        time_str (str): Time in 12-hour format (e.g. "08:30 AM")

    Returns:
        str: Time in 24-hour format (e.g. "08:30")
    """
    # Parse the input time string
    time_obj = datetime.strptime(time_str, "%I:%M %p")
    
    # Convert the time to 24-hour format
    time_24_hour = time_obj.strftime("%H:%M")
    
    return time_24_hour

def main():
    # Example usage
    time_str = input("Enter a time in 12-hour format (e.g. 08:30 AM): ")
    time_24_hour = convert_to_24_hour_format(time_str)
    print(f"Time in 24-hour format: {time_24_hour}")

if __name__ == "__main__":
    main()
```
