## Exercise
Write a program to calculate the standard deviation of all elements in a given array.

## Reference Solution
```python
import math

def calculate_mean(array):
    """
    Calculate the mean of an array.

    Args:
        array (list): A list of numbers.

    Returns:
        float: The mean of the array.
    """
    return sum(array) / len(array)


def calculate_variance(array):
    """
    Calculate the variance of an array.

    Args:
        array (list): A list of numbers.

    Returns:
        float: The variance of the array.
    """
    mean = calculate_mean(array)
    return sum((x - mean) ** 2 for x in array) / len(array)


def calculate_standard_deviation(array):
    """
    Calculate the standard deviation of an array.

    Args:
        array (list): A list of numbers.

    Returns:
        float: The standard deviation of the array.
    """
    variance = calculate_variance(array)
    return math.sqrt(variance)


def main():
    # Example array
    array = [1, 2, 3, 4, 5]

    # Calculate and print the standard deviation
    standard_deviation = calculate_standard_deviation(array)
    print("Standard Deviation:", standard_deviation)


if __name__ == "__main__":
    main()
```
