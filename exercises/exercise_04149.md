## Exercise
Write a function to check if a key is in a dictionary.

## Reference Solution

def check_key(dictionary, key):
    """
    Checks if a key is present in a dictionary.

    Args:
        dictionary (dict): The dictionary to check.
        key (any): The key to look for.

    Returns:
        bool: True if the key is in the dictionary, False otherwise.
    """
    return key in dictionary

# Example usage:
my_dict = {"name": "John", "age": 30}
print(check_key(my_dict, "name"))  # Output: True
print(check_key(my_dict, "city"))  # Output: False
