## Exercise
Write a program that takes a number as input and calculates its absolute value.

## Reference Solution
```python
def calculate_absolute_value():
    # Take a number as input from the user
    num = float(input("Enter a number: "))

    # Calculate the absolute value of the number
    absolute_value = abs(num)

    # Print the absolute value
    print("The absolute value of", num, "is", absolute_value)

# Call the function
calculate_absolute_value()
```
