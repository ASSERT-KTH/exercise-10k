## Exercise
Write a program to initialize variables with different data types (integer, float, string, boolean).

## Reference Solution
```python
# Initialize variables with different data types

# Integer
integer_variable = 10
print("Integer Variable:", integer_variable)
print("Data Type:", type(integer_variable))

# Float
float_variable = 10.5
print("\nFloat Variable:", float_variable)
print("Data Type:", type(float_variable))

# String
string_variable = "Hello, World!"
print("\nString Variable:", string_variable)
print("Data Type:", type(string_variable))

# Boolean
boolean_variable = True
print("\nBoolean Variable:", boolean_variable)
print("Data Type:", type(boolean_variable))
```
