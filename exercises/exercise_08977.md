Write a function that takes a string as input and returns True if the string contains only alphabets, False otherwise.

## Reference Solution

```python
def check_alphabets(input_string):
    """
    This function checks if a given string contains only alphabets.

    Args:
        input_string (str): The input string to be checked.

    Returns:
        bool: True if the string contains only alphabets, False otherwise.
    """

    # We'll use the built-in isalpha() method in Python, which returns True if all characters in the string are alphabets.
    # However, this method considers spaces and punctuation as non-alphabet characters. So, we'll iterate over each character in the string.

    # Initialize a flag variable to True, assuming the string contains only alphabets.
    is_alphabet = True

    # Iterate over each character in the input string.
    for char in input_string:
        
        # Check if the character is not an alphabet using the isalpha() method.
        if not char.isalpha():
            
            # If the character is not an alphabet, set the flag variable to False and break the loop.
            is_alphabet = False
            break

    # Return the flag variable, which indicates whether the string contains only alphabets or not.
    return is_alphabet
```
