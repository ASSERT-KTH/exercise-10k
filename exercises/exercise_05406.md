## Exercise
Write a function called 'find_max' that finds and returns the maximum number in a given list of integers.

## Reference Solution
```python
def find_max(numbers):
    """
    Finds and returns the maximum number in a given list of integers.

    Args:
        numbers (list): A list of integers.

    Returns:
        int: The maximum number in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not numbers:
        raise ValueError("Input list is empty")
    return max(numbers)

# Example usage:
def main():
    numbers = [12, 45, 7, 23, 56, 89, 34]
    try:
        max_number = find_max(numbers)
        print(f"The maximum number in the list is: {max_number}")
    except ValueError as e:
        print(e)

if __name__ == "__main__":
    main()
```
