## Exercise
Write a function that takes two lists as input, and returns a new list that contains all the elements from both lists, without duplicates.

## Reference Solution
```python
def combine_lists(list1, list2):
    """
    Combines two lists and removes duplicates.

    Args:
        list1 (list): The first list to combine.
        list2 (list): The second list to combine.

    Returns:
        list: A new list containing all unique elements from both input lists.
    """
    return list(set(list1 + list2))

# Example usage:
list1 = [1, 2, 3, 4]
list2 = [3, 4, 5, 6]

result = combine_lists(list1, list2)
print(result)  # Output: [1, 2, 3, 4, 5, 6]
```

```python
def combine_lists_ordered(list1, list2):
    """
    Combines two lists and removes duplicates while preserving order.

    Args:
        list1 (list): The first list to combine.
        list2 (list): The second list to combine.

    Returns:
        list: A new list containing all unique elements from both input lists.
    """
    seen = set()
    result = []
    for item in list1 + list2:
        if item not in seen:
            result.append(item)
            seen.add(item)
    return result

# Example usage:
list1 = [1, 2, 3, 4]
list2 = [3, 4, 5, 6]

result = combine_lists_ordered(list1, list2)
print(result)  # Output: [1, 2, 3, 4, 5, 6]
```
