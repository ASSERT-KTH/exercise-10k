## Exercise
Write a function to sort the values of a dictionary and return them as a list.

## Reference Solution
```python
def sort_dict_values(dictionary):
    """
    This function takes a dictionary as input, sorts its values, and returns them as a list.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        list: A list of sorted values from the dictionary.
    """

    # First, we need to get the values from the dictionary. We can do this using the .values() method.
    dict_values = dictionary.values()

    # Next, we need to sort these values. We can use the sorted() function for this.
    # The sorted() function returns a new sorted list from the elements of any sequence.
    sorted_values = sorted(dict_values)

    # Finally, we return the sorted values.
    return sorted_values

# Example usage:
my_dict = {'a': 5, 'b': 2, 'c': 8, 'd': 3}
print(sort_dict_values(my_dict))  # Output: [2, 3, 5, 8]
```
