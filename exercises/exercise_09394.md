## Exercise
Create a program that calculates the absolute value of a given number.

## Reference Solution
```python
def calculate_absolute_value(number):
    """
    This function calculates the absolute value of a given number.

    Args:
        number (int or float): The input number.

    Returns:
        int or float: The absolute value of the input number.
    """
    return abs(number)

def main():
    # Get the input number from the user
    number = float(input("Enter a number: "))

    # Calculate the absolute value
    absolute_value = calculate_absolute_value(number)

    # Print the result
    print(f"The absolute value of {number} is {absolute_value}")

if __name__ == "__main__":
    main()
```
