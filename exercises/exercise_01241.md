## Exercise
Write a simple function that takes a name as an argument and prints out a greeting message.

## Reference Solution
```python
def print_greeting(name: str) -> None:
    """
    Prints out a greeting message with the provided name.

    Args:
        name (str): The person's name.
    """
    print(f"Hello, {name}!")

# Example usage
def main() -> None:
    name = "John"
    print_greeting(name)

if __name__ == "__main__":
    main()
```
