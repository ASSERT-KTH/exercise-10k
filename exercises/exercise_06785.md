## Exercise
Write a program that takes an integer n as input and prints all numbers from n to 1 in reverse order, separated by a space, using a for loop or a while loop.

## Reference Solution
```python
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    for (int i = n; i >= 1; i--) {
        cout << i << " ";
    }

    return 0;
}
```
