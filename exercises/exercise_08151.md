## Exercise
Create an array of 5 integers and print each element.

## Reference Solution
```python
def main():
    # Create an array of 5 integers
    array = [1, 2, 3, 4, 5]
    
    # Print each element in the array
    for i, element in enumerate(array):
        print(f"Element at index {i}: {element}")

if __name__ == "__main__":
    main()
```
