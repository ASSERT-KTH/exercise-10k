## Exercise
Write a program that calculates the sum of all integers from 1 to a given number.

## Reference Solution
```python
def sum_of_integers(n):
    """
    Calculate the sum of all integers from 1 to a given number.

    Args:
        n (int): The upper limit of the range.

    Returns:
        int: The sum of all integers from 1 to n.
    """
    return sum(range(1, n + 1))

# Example usage:
num = int(input("Enter a positive integer: "))
result = sum_of_integers(num)
print(f"The sum of integers from 1 to {num} is: {result}")
```
