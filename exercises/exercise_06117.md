## Exercise
Write a program that takes an array of integers and a comparator function as input, and returns two new arrays: one containing the integers for which the comparator function returns True, and one containing the integers for which the comparator function returns False.

## Reference Solution
```python
def partition(arr, comparator):
    true_arr = [num for num in arr if comparator(num)]
    false_arr = [num for num in arr if not comparator(num)]
    return true_arr, false_arr

# Example usage:
def comparator(n):
    return n % 2 == 0

arr = [1, 2, 3, 4, 5, 6]
true_arr, false_arr = partition(arr, comparator)
print(true_arr)  # Output: [2, 4, 6]
print(false_arr)  # Output: [1, 3, 5]
```
