## Exercise
Write a function that takes a string, start index, and end index as input and returns the sliced string.

## Reference Solution
```python
def slice_string(input_str, start_idx, end_idx):
    """
    This function takes a string, start index, and end index as input and returns the sliced string.

    Args:
        input_str (str): The input string.
        start_idx (int): The start index of the slice.
        end_idx (int): The end index of the slice.

    Returns:
        str: The sliced string.
    """

    # First, we need to check if the input string is indeed a string
    if not isinstance(input_str, str):
        raise TypeError("Input must be a string.")

    # Then, we need to check if the start and end indices are integers
    if not isinstance(start_idx, int) or not isinstance(end_idx, int):
        raise TypeError("Start and end indices must be integers.")

    # We also need to check if the start index is less than or equal to the end index
    if start_idx > end_idx:
        raise ValueError("Start index cannot be greater than end index.")

    # We should also check if the start and end indices are within the bounds of the string
    if start_idx < 0 or end_idx > len(input_str):
        raise ValueError("Start or end index is out of range.")

    # Now, we can use Python's built-in string slicing feature to get the sliced string
    # The slicing operation will return a new string that includes all characters from the start index up to but not including the end index
    sliced_str = input_str[start_idx:end_idx]

    # Finally, we return the sliced string
    return sliced_str
```
