## Exercise
Write a program that takes in a user's input and determines the data type of the input. The program should check if the input is an integer, a float, or a string. If the input is an integer or a float, the program should print out the data type and the value. If the input is a string, the program should print out the data type and the length of the string.

## Reference Solution
```python
# -*- coding: utf-8 -*-

# Importing Modules
import sys

# try-except Block
try:
    # Getting User Input
    a = input("Enter something: ")

    # Checking Data Type
    if a.isdigit():
        print(f"Data type: integer, Value: {int(a)}")
    elif a.replace('.', '', 1).isdigit():
        print(f"Data type: float, Value: {float(a)}")
    else:
        print(f"Data type: string, Length: {len(a)}")

# except Block
except EOFError:
    print("Error: Invalid input")
    sys.exit(1)
```
