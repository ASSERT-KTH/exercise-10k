## Exercise
Create a function that uses the map function to apply a given function to a list of numbers

## Reference Solution
```python
def apply_function_to_list(func, numbers):
    """
    Applies a given function to each number in a list using map().

    Args:
        func (function): The function to be applied to each number.
        numbers (list): A list of numbers.

    Returns:
        list: A list of results after applying the function to each number.
    """
    return list(map(func, numbers))

# Example usage:
def square(x):
    """Returns the square of a number."""
    return x ** 2

numbers = [1, 2, 3, 4, 5]
result = apply_function_to_list(square, numbers)
print(result)  # Output: [1, 4, 9, 16, 25]

# Another example with a lambda function:
numbers = [1, 2, 3, 4, 5]
result = apply_function_to_list(lambda x: x * 3, numbers)
print(result)  # Output: [3, 6, 9, 12, 15]
```
