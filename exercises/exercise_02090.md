## Exercise
Write a program to calculate the result of exponentiation (a^b) given two numbers a and b.

## Reference Solution
```python
def calculate_exponentiation(a, b):
    """
    Calculate the result of exponentiation (a^b) given two numbers a and b.

    Args:
        a (float): The base number.
        b (float): The exponent.

    Returns:
        float: The result of a^b.
    """
    return a ** b


def main():
    # Example usage:
    a = float(input("Enter the base number (a): "))
    b = float(input("Enter the exponent (b): "))

    result = calculate_exponentiation(a, b)

    print(f"The result of {a}^{b} is {result}")


if __name__ == "__main__":
    main()
```
