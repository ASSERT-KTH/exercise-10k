## Exercise
Write a function that takes a dictionary and a key as input and removes the key-value pair for the given key from the dictionary if it exists. If the key does not exist, return 'Key not found'.

## Reference Solution
```python
def remove_key(dictionary, key):
    """
    This function removes a key-value pair from a dictionary if the key exists.
    
    Args:
        dictionary (dict): The dictionary to remove the key-value pair from.
        key (any): The key to be removed.
    
    Returns:
        dict: The updated dictionary.
    """

    # First, we check if the key exists in the dictionary
    if key in dictionary:
        
        # If the key exists, we use the pop method to remove the key-value pair
        dictionary.pop(key)
        
        # After removing the key, we return the updated dictionary
        return dictionary
    
    # If the key does not exist, we return 'Key not found'
    else:
        return 'Key not found'

# Example usage:
my_dict = {'a': 1, 'b': 2, 'c': 3}
print(remove_key(my_dict, 'b'))  # Output: {'a': 1, 'c': 3}
print(remove_key(my_dict, 'd'))  # Output: 'Key not found'
```
