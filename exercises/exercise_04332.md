## Exercise
Write a program to insert a given element at a specific index in an array.

## Reference Solution
```python
def insert_element(array, element, index):
    """
    Inserts an element at a specific index in an array.

    Args:
        array (list): The input array.
        element: The element to be inserted.
        index (int): The index at which the element is to be inserted.

    Returns:
        list: The array with the element inserted at the specified index.

    Raises:
        IndexError: If the index is out of range.
    """

    # Check if the index is within the valid range
    if index < 0 or index > len(array):
        raise IndexError("Index out of range")

    # Use list slicing to insert the element at the specified index
    return array[:index] + [element] + array[index:]


# Example usage:
if __name__ == "__main__":
    array = [1, 2, 3, 5]
    element = 4
    index = 3

    try:
        result = insert_element(array, element, index)
        print("Array after insertion:", result)
    except IndexError as e:
        print("Error:", e)
```
