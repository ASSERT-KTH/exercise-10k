## Exercise
Write a program that takes two lists of integers as input, and returns a new list containing pairs of corresponding elements from the two input lists.

## Reference Solution
```python
def pair_elements(list1, list2):
    result = []
    for i in range(len(list1)):
        result.append((list1[i], list2[i]))
    return result

def main():
    list1 = [1, 2, 3, 4, 5]
    list2 = [6, 7, 8, 9, 10]
    result = pair_elements(list1, list2)
    for i in range(len(result)):
        print(f"Pair {i+1}: {result[i]}")

if __name__ == "__main__":
    main()
```
