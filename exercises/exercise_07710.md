## Exercise
Compress a given string by replacing repeated characters with the character and the count of repetitions.

## Reference Solution
```python
def compress_string(input_str):
    """
    Compresses a given string by replacing repeated characters with the character and the count of repetitions.

    Args:
        input_str (str): The input string to be compressed.

    Returns:
        str: The compressed string.
    """
    if not input_str:
        return ""

    compressed_str = []
    count = 1

    for i in range(1, len(input_str)):
        if input_str[i] == input_str[i - 1]:
            count += 1
        else:
            compressed_str.append(input_str[i - 1] + (str(count) if count > 1 else ""))
            count = 1

    compressed_str.append(input_str[-1] + (str(count) if count > 1 else ""))

    return "".join(compressed_str)


# Example usage
input_str = "AAABBBCCC"
compressed = compress_string(input_str)
print(f"Input: {input_str}")
print(f"Output: {compressed}")
```
