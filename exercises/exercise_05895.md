## Exercise
Write a function that takes two lists as input and returns a list of pairs, where each pair contains one element from each list.

## Reference Solution
```python
def pair_lists(list1, list2):
    """
    This function takes two lists as input and returns a list of pairs.
    Each pair contains one element from each list.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: A list of pairs, where each pair contains one element from each list.
    """

    # Initialize an empty list to store the pairs
    pairs = []

    # Calculate the minimum length between the two lists
    # This is to avoid index out of range error when one list is longer than the other
    min_length = min(len(list1), len(list2))

    # Iterate over the range of the minimum length
    for i in range(min_length):
        # For each iteration, create a pair with one element from each list
        pair = (list1[i], list2[i])
        
        # Append the pair to the pairs list
        pairs.append(pair)

    # Return the list of pairs
    return pairs

# Test the function
list1 = [1, 2, 3]
list2 = ['a', 'b', 'c']
print(pair_lists(list1, list2))  # Output: [(1, 'a'), (2, 'b'), (3, 'c')]
```
