## Exercise
Create a dictionary with 5 key-value pairs. Clear the dictionary using the .clear() method.

## Reference Solution
```python
def main():
    # Create a dictionary with 5 key-value pairs
    my_dict = {
        "key1": "value1",
        "key2": "value2",
        "key3": "value3",
        "key4": "value4",
        "key5": "value5"
    }
    
    # Print the dictionary before clearing
    print("Dictionary before clearing:")
    print(my_dict)
    
    # Clear the dictionary using the .clear() method
    my_dict.clear()
    
    # Print the dictionary after clearing
    print("\nDictionary after clearing:")
    print(my_dict)

if __name__ == "__main__":
    main()
```
