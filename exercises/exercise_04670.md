## Exercise
Write a program that takes a string as input and replaces all consonants with a specified character.

## Reference Solution
```python
def replace_consonants(input_str, replacement_char):
    """
    Replaces all consonants in the input string with the specified replacement character.

    Args:
        input_str (str): The input string.
        replacement_char (str): The character to replace consonants with.

    Returns:
        str: The modified string with consonants replaced.
    """
    # Define the set of vowels
    vowels = set('aeiouAEIOU')

    # Use a list comprehension to iterate over each character in the input string
    # If the character is a letter and not a vowel, replace it with the replacement character
    # Otherwise, keep the original character
    modified_str = ''.join(
        [char if not char.isalpha() or char in vowels else replacement_char for char in input_str]
    )

    return modified_str

# Example usage:
input_str = "Hello, World!"
replacement_char = '*'
print(replace_consonants(input_str, replacement_char))
```
