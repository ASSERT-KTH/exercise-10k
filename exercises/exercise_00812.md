## Exercise
Use an enumerated data type to store a set of named values and print the value of an enumeration member.

## Reference Solution
```python
from enum import Enum

# Define an enumeration class
class Color(Enum):
    """Enumeration of colors"""
    RED = 1
    GREEN = 2
    BLUE = 3

def main():
    # Access and print the value of an enumeration member
    print(Color.RED)  # Output: Color.RED
    print(Color.RED.name)  # Output: RED
    print(Color.RED.value)  # Output: 1

    # Iterate over the enumeration members and print their values
    for color in Color:
        print(f"{color.name}: {color.value}")

if __name__ == "__main__":
    main()
```
