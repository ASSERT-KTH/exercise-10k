## Exercise
Write a program that takes a string as input and returns a dictionary with the frequency of each word in the string.

## Reference Solution
```python
import re
from collections import Counter

def word_frequency(string):
    """
    Returns a dictionary with the frequency of each word in the string.

    Args:
        string (str): The input string.

    Returns:
        dict: A dictionary with words as keys and their frequencies as values.
    """
    # Convert the string to lowercase
    string = string.lower()
    
    # Replace non-alphanumeric characters with spaces
    string = re.sub(r'[^a-z0-9\s]', ' ', string)
    
    # Split the string into words
    words = string.split()
    
    # Count the frequency of each word
    frequency = Counter(words)
    
    return dict(frequency)

# Example usage
if __name__ == "__main__":
    input_string = "This is a test string. The string is just a test."
    frequency_dict = word_frequency(input_string)
    print(frequency_dict)
```
