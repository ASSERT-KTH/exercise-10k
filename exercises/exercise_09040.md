## Exercise
Create a function that takes a boolean value as an argument and returns its negation

## Reference Solution
```python
def negate_boolean(value: bool) -> bool:
    """
    Returns the negation of the input boolean value.

    Args:
        value (bool): The input boolean value.

    Returns:
        bool: The negation of the input boolean value.
    """
    return not value

# Example usage:
if __name__ == "__main__":
    print(negate_boolean(True))  # Output: False
    print(negate_boolean(False))  # Output: True
```
