## Exercise
Write a function that takes another function as an argument and returns its result

## Reference Solution
```python
def execute_function(func, *args, **kwargs):
    """
    Execute a given function with provided arguments and keyword arguments.

    Args:
        func (function): The function to be executed.
        *args: Variable number of positional arguments.
        **kwargs: Variable number of keyword arguments.

    Returns:
        result: The result of the executed function.
    """
    return func(*args, **kwargs)

# Example usage:
def add(a, b):
    """Return the sum of two numbers."""
    return a + b

def greet(name):
    """Return a greeting message."""
    return f"Hello, {name}!"

# Execute the add function with arguments
result = execute_function(add, 3, 5)
print(result)  # Output: 8

# Execute the greet function with keyword arguments
result = execute_function(greet, name="John")
print(result)  # Output: Hello, John!
```
