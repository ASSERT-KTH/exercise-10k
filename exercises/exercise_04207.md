## Exercise
Write a function to rotate the elements of an array by a specified number of positions.

## Reference Solution
```python
def rotate_array(arr, num_positions):
    """
    Rotate the elements of an array by a specified number of positions.

    Args:
        arr (list): The input array.
        num_positions (int): The number of positions to rotate the array.

    Returns:
        list: The rotated array.
    """
    num_positions = num_positions % len(arr)  # Handle cases where num_positions > len(arr)
    return arr[-num_positions:] + arr[:-num_positions]

# Example usage:
if __name__ == "__main__":
    arr = [1, 2, 3, 4, 5]
    num_positions = 2
    rotated_arr = rotate_array(arr, num_positions)
    print(f"Original array: {arr}")
    print(f"Rotated array by {num_positions} positions: {rotated_arr}")
```
