## Exercise
Create a program that calculates the volume of a cube given its side length.

## Reference Solution
```python
def calculate_cube_volume(side_length):
    """
    Calculates the volume of a cube given its side length.

    Args:
        side_length (float): The length of the cube's side.

    Returns:
        float: The volume of the cube.
    """
    volume = side_length ** 3
    return volume

def main():
    # Get the side length from the user
    side_length = float(input("Enter the side length of the cube: "))

    # Calculate the volume
    volume = calculate_cube_volume(side_length)

    # Print the result
    print(f"The volume of the cube is: {volume:.2f} cubic units")

if __name__ == "__main__":
    main()
```
