## Exercise
Given a dictionary {'name': 'Alice', 'age': 20, 'city': 'Tokyo'}, write a program to print all the keys in the dictionary.

## Reference Solution
```python
def print_dict_keys(dictionary):
    """
    Prints all the keys in the dictionary.
    
    Args:
        dictionary (dict): The input dictionary.
    """
    # Get the keys from the dictionary
    keys = dictionary.keys()
    
    # Print the keys
    for key in keys:
        print(key)

def main():
    # Example dictionary
    person = {'name': 'Alice', 'age': 20, 'city': 'Tokyo'}
    
    # Print the keys
    print("Dictionary Keys:")
    print_dict_keys(person)

if __name__ == "__main__":
    main()
```
