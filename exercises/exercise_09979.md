## Exercise
Write a program that demonstrates the concept of variable shadowing.

## Reference Solution
```python
# Global variable
global_var = 10

def demonstrate_shadowing():
    """
    Demonstrates variable shadowing by declaring a local variable with the same name as a global variable.
    """
    # Local variable with the same name as the global variable
    global_var = 20
    
    # Accessing the local variable
    print("Local variable:", global_var)

    # Trying to access the global variable
    print("Trying to access global variable directly:", global_var)

# Accessing the global variable outside the function
print("Global variable before function call:", global_var)

demonstrate_shadowing()

# Accessing the global variable after the function call
print("Global variable after function call:", global_var)
```
