## Exercise
Write a program that takes an integer as input and prints out the sum of its digits. The program should use a while loop to extract each digit from the number and add it to a running total. If the input number is negative, the program should print out an error message and exit.

## Reference Solution
```python
# Initial Comments
# Author: [Your Name]
# Date: [Today's Date]
# Description: This program takes an integer as input and prints out the sum of its digits.

# Importing Modules
import sys

# Defining Variables
number = 12345  # example input
total = 0
digit = 0

# Program Logic
while number != 0:
    digit = number % 10
    total += digit
    number = number // 10

print(total)
```
