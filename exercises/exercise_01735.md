## Exercise
Write a program that takes a number as input and returns the sum of its digits.

## Reference Solution
```python
def sum_of_digits(n):
    return sum(int(digit) for digit in str(n))

# Test the function
number = int(input("Enter a number: "))
print("The sum of its digits is:", sum_of_digits(number))
```
