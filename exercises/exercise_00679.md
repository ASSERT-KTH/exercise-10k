## Exercise
Given a dictionary {'name': 'Quinn', 'age': 20, 'city': 'Mackay'}, write a program to use the values() method to print all the values in the dictionary.

## Reference Solution
```python
def print_dict_values():
    # Define the dictionary
    person = {'name': 'Quinn', 'age': 20, 'city': 'Mackay'}

    # Print all the values in the dictionary
    print("Dictionary Values:")
    for value in person.values():
        print(value)

# Example usage
if __name__ == "__main__":
    print_dict_values()
```
