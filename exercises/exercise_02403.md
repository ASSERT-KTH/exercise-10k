## Exercise
Write a function that takes one argument with a default value and returns its double. Then, call this function with and without an input number and print the results.

## Reference Solution
```python
def double_number(num=5):
    """
    Returns the double of the input number.

    Args:
        num (int, optional): The number to be doubled. Defaults to 5.

    Returns:
        int: The double of the input number.
    """
    return num * 2

# Call the function with an input number
print("Double of 10:", double_number(10))

# Call the function without an input number (using default value)
print("Double of default value:", double_number())
```
