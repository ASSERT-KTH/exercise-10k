## Exercise
Write a function that takes a list of integers as input, and returns a new list containing only the unique elements from the input list.

## Reference Solution
```python
def get_unique_elements(input_list):
    """
    This function takes a list of integers as input, and returns a new list containing only the unique elements from the input list.

    Args:
        input_list (list): A list of integers.

    Returns:
        list: A new list containing only the unique elements from the input list.
    """

    # First, we check if the input is a list. If it's not, we raise a TypeError.
    if not isinstance(input_list, list):
        raise TypeError("Input must be a list.")

    # Then, we check if all elements in the list are integers. If any element is not an integer, we raise a ValueError.
    if not all(isinstance(num, int) for num in input_list):
        raise ValueError("All elements in the list must be integers.")

    # We use a set to store unique elements. Sets in Python are unordered collections of unique elements.
    # We convert the input list to a set to remove duplicates.
    unique_set = set(input_list)

    # We convert the set back to a list, because the problem requires the output to be a list.
    # We use the list() function to convert the set to a list.
    unique_list = list(unique_set)

    # Finally, we return the list of unique elements.
    return unique_list

# Example usage:
print(get_unique_elements([1, 2, 2, 3, 4, 4, 5, 6, 6]))  # Output: [1, 2, 3, 4, 5, 6]
```
