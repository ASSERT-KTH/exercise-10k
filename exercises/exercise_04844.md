## Exercise
Write a program that takes two lists as input, and returns their concatenation.

## Reference Solution
```python
def concatenate_lists(list1, list2):
    """
    Concatenates two lists.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: The concatenation of list1 and list2.
    """
    return list1 + list2

# Example usage
def main():
    list1 = [1, 2, 3]
    list2 = [4, 5, 6]
    print("List 1:", list1)
    print("List 2:", list2)
    print("Concatenation:", concatenate_lists(list1, list2))

if __name__ == "__main__":
    main()
```
