## Exercise
Write a program that takes the day of the week as input and prints out the corresponding day number (Monday = 1, Tuesday = 2, ..., Sunday = 7).

## Reference Solution
```python
def findDay(date):
    days = ["monday", "tuesday", "wednesday", "thursday", "friday", "saturday", "sunday"]
    return days.index(date.lower()) + 1

date = input("Enter day of the week: ")
print(findDay(date))
```
