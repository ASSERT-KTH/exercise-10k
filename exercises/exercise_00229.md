## Exercise
Create a function to validate if a given string contains only digits.

## Reference Solution
```python
def validate_digits(input_string: str) -> bool:
    """
    Validate if a given string contains only digits.

    Args:
        input_string (str): The input string to be validated.

    Returns:
        bool: True if the string contains only digits, False otherwise.
    """
    return input_string.isdigit()

# Example usage:
if __name__ == "__main__":
    print(validate_digits("12345"))  # True
    print(validate_digits("123abc"))  # False
    print(validate_digits(""))  # False
    print(validate_digits("123.45"))  # False
```
