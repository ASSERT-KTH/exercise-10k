## Exercise
Write a function that takes a string as input and returns the minimized string by removing consecutive duplicate characters

## Reference Solution
```python
def minimize_string(input_str):
    """
    This function minimizes a string by removing consecutive duplicate characters.

    Args:
        input_str (str): The input string.

    Returns:
        str: The minimized string.
    """
    if not input_str:
        return ""

    minimized_str = [input_str[0]]
    for char in input_str[1:]:
        if char != minimized_str[-1]:
            minimized_str.append(char)

    return "".join(minimized_str)


# Example usage
if __name__ == "__main__":
    input_str = "aaabbbccc"
    minimized = minimize_string(input_str)
    print(f"Input: {input_str}")
    print(f"Minimized: {minimized}")
```
