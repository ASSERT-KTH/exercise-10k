## Exercise
Create a function that checks if a given string is valid according to the following rules: it must contain at least one uppercase letter, one lowercase letter, one digit, and one special character. The function should return 'Valid' if the string meets all the rules, and 'Invalid' otherwise.

## Reference Solution
```python
def check_string(s):
    """
    Checks if a given string is valid according to the following rules:
    it must contain at least one uppercase letter, one lowercase letter, one digit, and one special character.

    Args:
        s (str): The input string to be checked.

    Returns:
        str: 'Valid' if the string meets all the rules, 'Invalid' otherwise.
    """

    # First, we'll assume the string is invalid until we've checked all the rules
    result = 'Invalid'

    # Check if the string contains at least one uppercase letter
    has_uppercase = any(char.isupper() for char in s)
    
    # Check if the string contains at least one lowercase letter
    has_lowercase = any(char.islower() for char in s)
    
    # Check if the string contains at least one digit
    has_digit = any(char.isdigit() for char in s)
    
    # Check if the string contains at least one special character
    # We'll use the fact that special characters are neither letters nor digits
    has_special_char = any(not char.isalnum() for char in s)
    
    # If all rules are met, update the result to 'Valid'
    if has_uppercase and has_lowercase and has_digit and has_special_char:
        result = 'Valid'
    
    # Return the result
    return result
```
