## Exercise
Create a list of 5 integers and print a subset of elements using slicing.

## Reference Solution
```python
# Create a list of integers
numbers = [1, 2, 3, 4, 5]

# Print the original list
print("Original List:", numbers)

# Print a subset of elements using slicing
# Syntax: list_name[start_index:end_index]
print("Subset of elements from index 1 to 3:", numbers[1:4])

# Print the first two elements
print("First two elements:", numbers[:2])

# Print the last two elements
print("Last two elements:", numbers[-2:])

# Print all elements except the first one
print("All elements except the first one:", numbers[1:])
```
