## Exercise
Write a function that takes a dictionary and a key as input and returns the corresponding value, or a default value if the key is not found

## Reference Solution
```python
def get_dict_value(dictionary, key, default=None):
    """
    Retrieves the value associated with a given key from a dictionary.
    
    Args:
        dictionary (dict): The dictionary to search.
        key: The key to look for.
        default: The value to return if the key is not found. Defaults to None.
    
    Returns:
        The value associated with the key, or the default value if not found.
    """
    return dictionary.get(key, default)

# Example usage:
example_dict = {"name": "John", "age": 30}
print(get_dict_value(example_dict, "name"))  # Output: John
print(get_dict_value(example_dict, "city", "Not specified"))  # Output: Not specified
```
