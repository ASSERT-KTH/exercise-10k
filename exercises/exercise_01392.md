## Exercise
Create a program that uses a for loop to encrypt a message using a Caesar cipher with a shift of 3.

## Reference Solution
```python
def caesar_cipher(message, shift):
    """
    Encrypts a message using a Caesar cipher with the given shift.

    Args:
        message (str): The message to be encrypted.
        shift (int): The shift value for the Caesar cipher.

    Returns:
        str: The encrypted message.
    """
    encrypted_message = ""
    for char in message:
        if char.isalpha():
            ascii_offset = 65 if char.isupper() else 97
            encrypted_char = chr((ord(char) - ascii_offset + shift) % 26 + ascii_offset)
            encrypted_message += encrypted_char
        else:
            encrypted_message += char
    return encrypted_message

def main():
    message = input("Enter a message to encrypt: ")
    shift = 3
    encrypted_message = caesar_cipher(message, shift)
    print(f"Encrypted message: {encrypted_message}")

if __name__ == "__main__":
    main()
```
