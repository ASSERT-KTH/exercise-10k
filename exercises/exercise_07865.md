## Exercise
Write a program that checks if a character is a vowel using conditional statements.

## Reference Solution
```python
def is_vowel(char):
    # Convert the character to lower case
    char = char.lower()
    
    # Check if the character is a vowel
    if char in 'aeiou':
        return True
    else:
        return False

# Test the function
char = input("Enter a character: ")
if is_vowel(char):
    print(char, "is a vowel.")
else:
    print(char, "is not a vowel.")
```
