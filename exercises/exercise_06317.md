## Exercise
Split a dictionary into two dictionaries based on a certain condition

## Reference Solution
```python
def split_dict(input_dict, condition):
    """
    Splits a dictionary into two dictionaries based on a certain condition.

    Args:
        input_dict (dict): The input dictionary to be split.
        condition (function): A function that takes a key-value pair as input and returns a boolean value.

    Returns:
        tuple: A tuple containing two dictionaries. The first dictionary contains key-value pairs for which the condition is True, and the second dictionary contains key-value pairs for which the condition is False.
    """
    dict_true = {key: value for key, value in input_dict.items() if condition(key, value)}
    dict_false = {key: value for key, value in input_dict.items() if not condition(key, value)}
    return dict_true, dict_false

# Example usage
input_dict = {'a': 1, 'b': 2, 'c': 3, 'd': 4}

# Define a condition function that checks if the value is even
def is_even(key, value):
    return value % 2 == 0

dict_even, dict_odd = split_dict(input_dict, is_even)

print("Dictionary with even values:")
print(dict_even)
print("Dictionary with odd values:")
print(dict_odd)
```
