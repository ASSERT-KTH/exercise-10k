## Exercise
Write a program that takes a string input and checks if it is a palindrome. If the string is a palindrome, the program should reverse the string and return the reversed string. If the string is not a palindrome, the program should return a message indicating that the string is not a palindrome.

## Reference Solution
```python
def palindrome(str):
    if str == str[::-1]:
        return str[::-1]
    else:
        return str + " is not a palindrome"

# Example usage:
print(palindrome("radar"))  # Output: "radar"
print(palindrome("python"))  # Output: "python is not a palindrome"
```
