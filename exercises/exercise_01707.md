## Exercise
Write a program that uses a for loop to print a given pattern (e.g. a square of asterisks).

## Reference Solution
```python
def print_square(size):
    """
    Prints a square of asterisks of a given size.
    
    Args:
        size (int): The size of the square.
    """
    for i in range(size):
        for j in range(size):
            print("*", end=" ")
        print()

# Example usage
print_square(5)
```
