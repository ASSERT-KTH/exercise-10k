## Exercise
Create a program that simulates a coin flip and prints out the result using an if statement

## Reference Solution
```python
import random

def coin_flip():
    # Generate a random number (0 or 1) to simulate the coin flip
    flip = random.randint(0, 1)
    
    # Use an if statement to determine the result of the flip
    if flip == 0:
        result = "Heads"
    else:
        result = "Tails"
    
    return result

def main():
    # Call the coin_flip function and print the result
    result = coin_flip()
    print(f"The coin landed on: {result}")

if __name__ == "__main__":
    main()
```
