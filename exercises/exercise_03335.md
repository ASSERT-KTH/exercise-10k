## Exercise
Write a program that takes a list of integers and a function as input and returns a new list containing the result of applying the function to each element in the original list.

## Reference Solution
```python
from typing import List, Callable

def map_func(lst: List[int], func: Callable[[int], int]) -> List[int]:
    """
    Applies a function to each element in a list and returns a new list with the results.
    
    Args:
        lst (List[int]): The input list of integers.
        func (Callable[[int], int]): The function to be applied to each element.
    
    Returns:
        List[int]: A new list containing the results of applying the function to each element.
    """
    new_lst = []  # Initialize an empty list to store the results
    for elem in lst:  # Iterate over each element in the input list
        new_lst.append(func(elem))  # Apply the function to each element and append the result to the new list
    return new_lst  # Return the new list

# Example usage:
def square(x: int) -> int:
    """
    Returns the square of a number.
    
    Args:
        x (int): The input number.
    
    Returns:
        int: The square of the input number.
    """
    return x * x

numbers = [1, 2, 3, 4, 5]  # Example list of integers
result = map_func(numbers, square)  # Apply the square function to each element in the list
print(result)  # Print the resulting list: [1, 4, 9, 16, 25]
```
