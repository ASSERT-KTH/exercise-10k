## Exercise
Write a function to get the number of key-value pairs in a dictionary

## Reference Solution
```python
def get_key_value_pairs_count(dictionary):
    """
    Returns the number of key-value pairs in a dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        int: The number of key-value pairs in the dictionary.
    """
    return len(dictionary)

# Example usage
my_dict = {"name": "John", "age": 30, "city": "New York"}
print(get_key_value_pairs_count(my_dict))  # Output: 3
```
