## Exercise
Declare a variable using the enum data type to represent a set of named values.

## Reference Solution
```python
from enum import Enum

# Define an Enum class
class Color(Enum):
    RED = 1
    GREEN = 2
    BLUE = 3

# Declare a variable using the enum data type
color = Color.GREEN

# Example usage:
print(color)  # Output: Color.GREEN
print(color.name)  # Output: GREEN
print(color.value)  # Output: 2
```
