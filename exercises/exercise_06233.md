## Exercise
Write a program that finds the first duplicate in a given list, i.e., the first element that appears more than once.

## Reference Solution
```python
def find_first_duplicate(lst):
    """
    Finds the first duplicate in a given list.

    Args:
        lst (list): The input list.

    Returns:
        The first duplicate element if found, None otherwise.
    """
    seen = set()
    for num in lst:
        if num in seen:
            return num
        seen.add(num)
    return None

# Example usage
if __name__ == "__main__":
    numbers = [2, 1, 3, 5, 3, 2]
    duplicate = find_first_duplicate(numbers)
    if duplicate:
        print(f"The first duplicate is: {duplicate}")
    else:
        print("No duplicates found.")
```
