## Exercise
Create a program that prints out a sequence of numbers from 1 to n, where n is provided by the user. The program should use a loop to print out the numbers.

## Reference Solution
```python
using System;

class PrintNumberSequence
{
    static void Main()
    {
        int n = int.Parse(Console.ReadLine());

        for (int i = 1; i <= n; i++)
        {
            Console.WriteLine(i);
        }
    }
}
```
