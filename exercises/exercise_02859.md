## Exercise
Write a function to set a default value for a given key in a dictionary if the key is not present.

## Reference Solution

def set_default_value(dictionary, key, default_value):
    """
    Sets a default value for a given key in a dictionary if the key is not present.

    Args:
        dictionary (dict): The dictionary to be updated.
        key (any): The key to be checked.
        default_value (any): The default value to be set.

    Returns:
        dict: The updated dictionary.
    """

    # First, we check if the key is already present in the dictionary
    if key not in dictionary:
        
        # If the key is not present, we add it to the dictionary with the default value
        dictionary[key] = default_value
    
    # Finally, we return the updated dictionary
    return dictionary

# Example usage:
my_dict = {'a': 1, 'b': 2}
updated_dict = set_default_value(my_dict, 'c', 3)
print(updated_dict)  # Output: {'a': 1, 'b': 2, 'c': 3}
