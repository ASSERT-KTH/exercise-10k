## Exercise
Write a program that converts a given string into an integer

## Reference Solution
```python
def string_to_int(s):
    """
    Converts a string into an integer.

    Args:
        s (str): The input string.

    Returns:
        int: The integer equivalent of the input string.

    Raises:
        ValueError: If the input string is not a valid integer.
    """
    try:
        return int(s)
    except ValueError:
        print("Error: The input string is not a valid integer.")
        return None

def main():
    # Example usage
    input_string = input("Enter a string: ")
    result = string_to_int(input_string)
    if result is not None:
        print(f"The integer equivalent of '{input_string}' is: {result}")

if __name__ == "__main__":
    main()
```
