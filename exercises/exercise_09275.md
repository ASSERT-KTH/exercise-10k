## Exercise
Write a program that takes a string as input and returns the reversed string. The program should handle strings of any length and should not use any built-in string reversal functions.

## Reference Solution
```python
class Solution:
    def reverseString(self, s):
        l = 0
        r = len(s) - 1
        while l < r:
            s[l], s[r] = s[r], s[l]
            l += 1
            r -= 1
        return s

# Test the solution
s = Solution()
print(s.reverseString("Hello, World!"))  # Output: "!dlroW ,olleH"
print(s.reverseString("abcdefg"))  # Output: "gfedcba"
```
