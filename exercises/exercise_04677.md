## Exercise
Write a program that takes two strings as input and returns the characters that are in the first string but not in the second string.

## Reference Solution
```python
def find_unique_chars(s1, s2):
    for c in s2:
        s1 = s1.replace(c, "")
    return s1

s1 = "hello"
s2 = "world"
print(find_unique_chars(s1, s2))
```
