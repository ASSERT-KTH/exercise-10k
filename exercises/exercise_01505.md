## Exercise
Write a program that splits a given string into substrings based on a specified separator.

## Reference Solution
```python
def split_string(input_string, separator):
    """
    Splits a given string into substrings based on a specified separator.

    Args:
        input_string (str): The input string to be split.
        separator (str): The separator used to split the string.

    Returns:
        list: A list of substrings split from the input string.
    """
    return input_string.split(separator)


def main():
    # Example usage
    input_string = "apple,banana,orange"
    separator = ","
    result = split_string(input_string, separator)
    print("Input String:", input_string)
    print("Separator:", separator)
    print("Result:", result)


if __name__ == "__main__":
    main()
```
