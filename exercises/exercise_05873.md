## Exercise
Calculate the sum of the digits of a given number.

## Reference Solution
```python
def sum_of_digits(n: int) -> int:
    """
    Calculate the sum of the digits of a given integer.

    Args:
    n (int): The input integer.

    Returns:
    int: The sum of the digits of the input integer.
    """
    return sum(int(digit) for digit in str(abs(n)))

# Example usage
def main():
    num = 12345
    print(f"The sum of the digits of {num} is: {sum_of_digits(num)}")

if __name__ == "__main__":
    main()
```
