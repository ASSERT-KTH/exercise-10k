## Exercise
Write a program that takes a person's account balance and transaction amount as input and returns whether the transaction is valid or not using conditional statements.

## Reference Solution
```python
#include <iostream>

using namespace std;

int main() {
    float accountBalance = 1000.0, transactionAmount = 500.0;

    if (accountBalance >= transactionAmount) {
        cout << "Transaction is valid";
    } else {
        cout << "Transaction is invalid";
    }

    return 0;
}
```
