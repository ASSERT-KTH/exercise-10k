## Exercise
Create a program that converts currency from one unit to another (e.g., USD to EUR, EUR to GBP). The program uses if-else statements to determine the conversion factor.

## Reference Solution
```python
# Define conversion factors
conversion_factors = {
    'USD': {'EUR': 0.88, 'GBP': 0.76, 'JPY': 110.65},
    'EUR': {'USD': 1.14, 'GBP': 0.86, 'JPY': 125.68},
    'GBP': {'USD': 1.32, 'EUR': 1.16, 'JPY': 145.55},
    'JPY': {'USD': 0.0091, 'EUR': 0.0079, 'GBP': 0.0069}
}

def convert_currency(amount, from_currency, to_currency):
    if from_currency == to_currency:
        return amount
    elif from_currency in conversion_factors and to_currency in conversion_factors[from_currency]:
        return amount * conversion_factors[from_currency][to_currency]
    else:
        return "Conversion not supported"

# Test the function
print(convert_currency(100, 'USD', 'EUR'))  # Should print 88.0
print(convert_currency(100, 'EUR', 'USD'))  # Should print 114.0
print(convert_currency(100, 'GBP', 'JPY'))  # Should print 14555.0
```
