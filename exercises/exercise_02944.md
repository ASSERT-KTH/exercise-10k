## Exercise
Write a program to generate the first n Fibonacci numbers.

## Reference Solution
```python
def fibonacci_recursive(n):
    """
    Generate the first n Fibonacci numbers using recursion.
    
    Args:
        n (int): The number of Fibonacci numbers to generate.
    
    Returns:
        list: A list of the first n Fibonacci numbers.
    """
    def fib(a, b, n):
        if n == 0:
            return []
        elif n == 1:
            return [a]
        elif n == 2:
            return [a, b]
        else:
            return [a] + fib(b, a + b, n - 1)
    
    return fib(0, 1, n)

def fibonacci_iterative(n):
    """
    Generate the first n Fibonacci numbers using iteration.
    
    Args:
        n (int): The number of Fibonacci numbers to generate.
    
    Returns:
        list: A list of the first n Fibonacci numbers.
    """
    fib_sequence = [0, 1]
    while len(fib_sequence) < n:
        fib_sequence.append(fib_sequence[-1] + fib_sequence[-2])
    return fib_sequence[:n]

def main():
    n = int(input("Enter the number of Fibonacci numbers to generate: "))
    print("Recursive approach:")
    print(fibonacci_recursive(n))
    print("Iterative approach:")
    print(fibonacci_iterative(n))

if __name__ == "__main__":
    main()
```
