## Exercise
Ask the user for a string and print out the number of characters it contains.

## Reference Solution
```python
def count_characters():
    """
    Asks the user for a string and prints out the number of characters it contains.
    """
    user_string = input("Please enter a string: ")
    character_count = len(user_string)
    print(f"The string contains {character_count} characters.")

def main():
    count_characters()

if __name__ == "__main__":
    main()
```
