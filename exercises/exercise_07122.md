## Exercise
Create a program that takes in a user's input and determines the data type of the input. The input can be an integer, float, or string. The program should then output the data type and the input value.

## Reference Solution
```python
def determine_data_type(user_input):
    """
    This function determines the data type of the user's input and returns the data type and the input value.

    Args:
        user_input (str): The user's input.

    Returns:
        tuple: A tuple containing the data type and the input value.
    """

    # Try to convert the input to an integer
    try:
        # If the input can be converted to an integer, it's an integer
        int_input = int(user_input)
        return "integer", int_input

    # If the input can't be converted to an integer, try to convert it to a float
    except ValueError:
        try:
            # If the input can be converted to a float, it's a float
            float_input = float(user_input)
            return "float", float_input

        # If the input can't be converted to a float, it's a string
        except ValueError:
            return "string", user_input


def main():
    # Get the user's input
    user_input = input("Please enter a value: ")

    # Determine the data type of the input
    data_type, input_value = determine_data_type(user_input)

    # Print the data type and the input value
    print(f"The data type of the input is: {data_type}")
    print(f"The input value is: {input_value}")


if __name__ == "__main__":
    main()
```
