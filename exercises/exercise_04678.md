## Exercise
Write a program that takes two strings as input and returns the characters that are in exactly one of the strings.

## Reference Solution
```python
def unique_chars(str1, str2):
    set1 = set(str1)
    set2 = set(str2)

    unique_chars = (set1 - set2) | (set2 - set1)
    return unique_chars

# Test the function
str1 = "hello"
str2 = "world"

unique_chars_set = unique_chars(str1, str2)
print("Characters in exactly one of the strings:")
for char in unique_chars_set:
    print(char)
```
