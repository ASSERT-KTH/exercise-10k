## Exercise
Write a program that takes an array and an element as input, removes all occurrences of the element from the array, and returns the resulting array.

## Reference Solution
```python
def removeElement(array, element):
    # array = [1, 2, 3, 2, 3, 4, 2, 5, 6, 7, 8, 9]
    # element = 2
    # array = [1, 3, 3, 4, 5, 6, 7, 8, 9]
    while element in array:
        array.remove(element)
    return array
```
