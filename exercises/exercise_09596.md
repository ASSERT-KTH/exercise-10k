## Exercise
Write a program that reads user input in a loop until the user enters 'quit'

## Reference Solution
```python
def main():
    """Reads user input in a loop until the user enters 'quit'."""
    while True:
        user_input = input("Please enter a message ('quit' to exit): ")
        if user_input.lower() == 'quit':
            print("Exiting the program. Goodbye!")
            break
        else:
            print(f"You entered: {user_input}")

if __name__ == "__main__":
    main()
```
