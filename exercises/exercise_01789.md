## Exercise
Create a function to reduce a dictionary to a single value by applying a given function to all key-value pairs.

## Reference Solution
```python
def reduce_dict(dictionary, func):
    """
    Reduces a dictionary to a single value by applying a given function to all key-value pairs.

    Args:
        dictionary (dict): The dictionary to be reduced.
        func (function): A function that takes two arguments and returns one value.

    Returns:
        The result of applying the function to all key-value pairs in the dictionary.
    """

    # Initialize the result with the first key-value pair in the dictionary
    # We use the items() method to get a list of tuples containing each key-value pair
    result = dictionary[list(dictionary.keys())[0]]

    # Iterate over each key-value pair in the dictionary
    for key, value in dictionary.items():
        # For each pair, apply the given function to the current result and the value
        # The result of this operation becomes the new result
        # This process continues until all key-value pairs have been processed
        result = func(result, value)

    # After all pairs have been processed, return the final result
    return result

# Example usage:
# Define a dictionary
my_dict = {'a': 1, 'b': 2, 'c': 3}

# Define a function to add two numbers
def add(x, y):
    return x + y

# Reduce the dictionary using the add function
result = reduce_dict(my_dict, add)
print(result)  # Output: 6
```
