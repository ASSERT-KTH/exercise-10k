## Exercise
Write a program that checks a person's age and prints 'Child' if under 13, 'Teen' if between 13 and 19, and 'Adult' otherwise

## Reference Solution
```python
def check_age(age):
    """
    Checks a person's age and returns their age category.

    Args:
        age (int): The person's age.

    Returns:
        str: The person's age category ('Child', 'Teen', or 'Adult').
    """
    if age < 13:
        return 'Child'
    elif 13 <= age <= 19:
        return 'Teen'
    else:
        return 'Adult'

def main():
    # Example usage:
    age = int(input("Enter your age: "))
    category = check_age(age)
    print(f"Your age category is: {category}")

if __name__ == "__main__":
    main()
```
