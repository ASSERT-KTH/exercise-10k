## Exercise
Write a program that prints the numbers from 1 to n, replacing multiples of 3 with 'Fizz' and multiples of 5 with 'Buzz'. Use a for loop to iterate over the numbers.

## Reference Solution
```python
def fizzbuzz(n):
    for i in range(1, n+1):
        if i % 3 == 0 and i % 5 == 0:
            print('FizzBuzz')
        elif i % 3 == 0:
            print('Fizz')
        elif i % 5 == 0:
            print('Buzz')
        else:
            print(i)

fizzbuzz(20)
```
