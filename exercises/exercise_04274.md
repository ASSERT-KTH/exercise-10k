## Exercise
Write a function that takes a dictionary as input and removes all key-value pairs using the clear() method

## Reference Solution
```python
def clear_dict(input_dict):
    """
    Removes all key-value pairs from a dictionary using the clear() method.

    Args:
        input_dict (dict): The dictionary to be cleared.

    Returns:
        None
    """
    # Check if input is a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary.")

    # Clear the dictionary
    input_dict.clear()

# Example usage
if __name__ == "__main__":
    example_dict = {"key1": "value1", "key2": "value2"}
    print("Before clearing:", example_dict)
    clear_dict(example_dict)
    print("After clearing:", example_dict)
```
