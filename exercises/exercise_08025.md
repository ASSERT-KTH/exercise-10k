## Exercise
Write a program to find the maximum contiguous subarray sum of a given one-dimensional array of numbers. The subarray must have at least one element. If the array is empty, return 0.

## Reference Solution
```python
def max_subarray_sum(nums):
    """
    This function calculates the maximum contiguous subarray sum.

    Args:
        nums (list): A list of integers.

    Returns:
        int: The maximum contiguous subarray sum. If the array is empty, return 0.
    """
    if not nums:  # Check if the array is empty
        return 0

    max_current = max_global = nums[0]  # Initialize the maximum current and global sum
    for num in nums[1:]:  # Iterate over the array starting from the second element
        max_current = max(num, max_current + num)  # Update the maximum current sum
        max_global = max(max_global, max_current)  # Update the maximum global sum

    return max_global

# Example usage:
print(max_subarray_sum([-2, 1, -3, 4, -1, 2, 1, -5, 4]))  # Output: 6
print(max_subarray_sum([1, 2, 3, 4, 5]))  # Output: 15
print(max_subarray_sum([-1, -2, -3, -4, -5]))  # Output: -1
print(max_subarray_sum([]))  # Output: 0
```
