## Exercise
Write a function that takes a list and a predicate as input, filters the list to include only elements that satisfy the predicate, and returns the filtered list.

## Reference Solution
```python
def filter_list(input_list, predicate):
    """
    This function filters the input list to include only elements that satisfy the predicate.

    Args:
        input_list (list): The list to be filtered.
        predicate (function): A function that takes one argument and returns a boolean value.

    Returns:
        list: The filtered list.
    """

    # Initialize an empty list to store the filtered elements
    filtered_list = []

    # Iterate over each element in the input list
    for element in input_list:
        # Check if the current element satisfies the predicate
        if predicate(element):
            # If it does, append it to the filtered list
            filtered_list.append(element)

    # Return the filtered list
    return filtered_list

# Example usage:
# Define a predicate function that checks if a number is even
def is_even(num):
    return num % 2 == 0

# Create a list of numbers
numbers = [1, 2, 3, 4, 5, 6]

# Filter the list using the is_even predicate
filtered_numbers = filter_list(numbers, is_even)

# Print the filtered list
print(filtered_numbers)  # Output: [2, 4, 6]
```
