## Exercise
Write a program that uses nested for loops to print a 5x5 square of asterisks.

## Reference Solution
```python
# Define the size of the square
size = 5

# Use nested for loops to print the square
for i in range(size):
    for j in range(size):
        print("*", end="")
    print()
```
