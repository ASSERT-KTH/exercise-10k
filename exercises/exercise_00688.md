## Exercise
Write a program to find the absolute value of a given number.

## Reference Solution
```python
def calculate_absolute_value(num):
    """
    Calculate the absolute value of a given number.

    Args:
        num (float): The input number.

    Returns:
        float: The absolute value of the input number.
    """
    return abs(num)


def main():
    # Example usage
    num = float(input("Enter a number: "))
    absolute_value = calculate_absolute_value(num)
    print(f"The absolute value of {num} is {absolute_value}")


if __name__ == "__main__":
    main()
```
