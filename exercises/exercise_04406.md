## Exercise
Create a program that finds the maximum number in a list using a for loop.

## Reference Solution
```python
def find_max_number(num_list):
    """
    Finds the maximum number in a list.

    Args:
        num_list (list): A list of numbers.

    Returns:
        int: The maximum number in the list.
    """
    if not num_list:
        raise ValueError("List is empty")

    max_num = num_list[0]
    for num in num_list:
        if num > max_num:
            max_num = num

    return max_num

# Example usage
numbers = [4, 2, 9, 6, 5, 1, 8, 3, 7]
max_number = find_max_number(numbers)
print("Maximum number:", max_number)
```
