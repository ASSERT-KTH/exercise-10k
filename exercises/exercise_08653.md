## Exercise
Create a function that sorts a given list in descending order.

## Reference Solution
```python
def sort_descending(input_list):
    """
    Sorts a given list in descending order.

    Args:
        input_list (list): The list to be sorted.

    Returns:
        list: The sorted list in descending order.
    """
    return sorted(input_list, reverse=True)


# Example usage
if __name__ == "__main__":
    numbers = [4, 2, 9, 6, 5, 1]
    sorted_numbers = sort_descending(numbers)
    print("Sorted list:", sorted_numbers)
```
