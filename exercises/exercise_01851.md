## Exercise
Write a program that takes two arrays of integers as input, and returns a new array containing all the elements from both arrays, without duplicates.

## Reference Solution
```python
from typing import List

def merge(nums1: List[int], nums2: List[int]) -> List[int]:
    nums = list(set(nums1 + nums2))
    nums.sort()
    return nums
```
