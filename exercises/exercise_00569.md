## Exercise
Write a function that pads a given string with a specified character to a specified length.

## Reference Solution
```python
def pad_string(input_str, char, length):
    """
    Pads a given string with a specified character to a specified length.

    Args:
        input_str (str): The input string to be padded.
        char (str): The character used for padding.
        length (int): The desired length of the output string.

    Returns:
        str: The padded string.
    """
    if len(input_str) >= length:
        return input_str
    else:
        padding_length = length - len(input_str)
        padding = char * padding_length
        return input_str + padding

# Example usage:
def main():
    input_str = "Hello"
    char = " "
    length = 10
    print(pad_string(input_str, char, length))

if __name__ == "__main__":
    main()
```
