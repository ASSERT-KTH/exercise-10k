## Exercise
Write a program that uses a while loop to generate and print a sequence of numbers, starting from a given number and incrementing by a given step.

## Reference Solution
```python
#include <stdio.h>

int main() {
    int n = 11;
    int i = 1;
    int step = 2;

    while (i < n) {
        printf("%d\n", i);
        i += step;
    }

    return 0;
}
```
