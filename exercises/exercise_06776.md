## Exercise
Create a program that uses functions to perform basic arithmetic operations. The program should have separate functions for addition, subtraction, multiplication, and division. Each function should take two numbers as arguments and return the result of the operation. The main part of the program should prompt the user to choose an operation and input two numbers, then call the corresponding function and print the result.

## Reference Solution
```python
def add(num1, num2):
    return num1 + num2

def subtract(num1, num2):
    return num1 - num2

def multiply(num1, num2):
    return num1 * num2

def divide(num1, num2):
    if num2 == 0:
        return "Error: Division by zero is not allowed."
    else:
        return num1 / num2

def main():
    print("Arithmetic Operations")
    print("1. Addition")
    print("2. Subtraction")
    print("3. Multiplication")
    print("4. Division")
    choice = input("Enter your choice (1-4): ")
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))

    if choice == "1":
        result = add(num1, num2)
    elif choice == "2":
        result = subtract(num1, num2)
    elif choice == "3":
        result = multiply(num1, num2)
    elif choice == "4":
        result = divide(num1, num2)
    else:
        result = "Error: Invalid choice. Please try again."

    print("Result: ", result)

main()
```
