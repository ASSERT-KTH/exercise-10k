## Exercise
Create a program that simulates a coin flip. If the result is 'heads', print 'Heads', otherwise print 'Tails'.

## Reference Solution
```python
import random

def coin_flip():
    """
    Simulates a coin flip and returns the result.
    """
    result = random.choice(['heads', 'tails'])
    return result

def main():
    result = coin_flip()
    if result == 'heads':
        print('Heads')
    else:
        print('Tails')

if __name__ == "__main__":
    main()
```
