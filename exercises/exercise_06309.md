## Exercise
Sort the keys of a dictionary in ascending order

## Reference Solution
```python
def sort_dict_keys(dictionary):
    """
    Returns a list of dictionary keys sorted in ascending order.

    Args:
        dictionary (dict): The dictionary to sort keys from.

    Returns:
        list: A list of sorted dictionary keys.
    """
    return sorted(dictionary.keys())

# Example usage
my_dict = {"c": 3, "b": 2, "a": 1, "d": 4}
sorted_keys = sort_dict_keys(my_dict)
print(sorted_keys)  # Output: ['a', 'b', 'c', 'd']
```

```python
def sort_dict_items(dictionary):
    """
    Returns a dictionary with items sorted by keys in ascending order.

    Args:
        dictionary (dict): The dictionary to sort items from.

    Returns:
        dict: A dictionary with sorted items.
    """
    return dict(sorted(dictionary.items()))

# Example usage
my_dict = {"c": 3, "b": 2, "a": 1, "d": 4}
sorted_dict = sort_dict_items(my_dict)
print(sorted_dict)  # Output: {'a': 1, 'b': 2, 'c': 3, 'd': 4}
```
