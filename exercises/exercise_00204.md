## Exercise
Write a function that takes three numbers as arguments and returns the maximum value.

## Reference Solution
```python
def find_max(a: float, b: float, c: float) -> float:
    """
    Returns the maximum value among three numbers.

    Args:
        a (float): The first number.
        b (float): The second number.
        c (float): The third number.

    Returns:
        float: The maximum value.
    """
    return max(a, b, c)

# Example usage:
def main():
    num1 = 10.5
    num2 = 20.8
    num3 = 15.2
    max_value = find_max(num1, num2, num3)
    print(f"The maximum value is: {max_value}")

if __name__ == "__main__":
    main()
```
