## Exercise
Create a function that takes a list of numbers and returns the maximum and minimum values.

## Reference Solution
```python
def find_max_min(numbers):
    """
    This function takes a list of numbers and returns the maximum and minimum values.

    Args:
        numbers (list): A list of numbers.

    Returns:
        tuple: A tuple containing the maximum and minimum values.

    Raises:
        ValueError: If the input list is empty.
    """
    if not numbers:
        raise ValueError("Input list cannot be empty")

    return max(numbers), min(numbers)

# Example usage
def main():
    numbers = [10, 20, 30, 40, 50]
    max_value, min_value = find_max_min(numbers)
    print(f"Maximum value: {max_value}")
    print(f"Minimum value: {min_value}")

if __name__ == "__main__":
    main()
```
