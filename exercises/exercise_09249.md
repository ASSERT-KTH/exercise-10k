## Exercise
Write a program that iterates over a dictionary and prints the key-value pairs.

## Reference Solution
```python
def print_dict_key_value_pairs(dictionary):
    """
    Prints the key-value pairs of a dictionary.

    Args:
        dictionary (dict): The dictionary to iterate over.
    """
    for key, value in dictionary.items():
        print(f"{key}: {value}")


def main():
    # Example dictionary
    example_dict = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }

    # Print the key-value pairs
    print_dict_key_value_pairs(example_dict)


if __name__ == "__main__":
    main()
```
