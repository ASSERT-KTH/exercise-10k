## Exercise
Write a function to check the type of a variable and return the correct type

## Reference Solution
```python
def check_type(var):
    """
    This function checks the type of a variable and returns the correct type.

    Args:
        var: The variable to be checked.

    Returns:
        str: The type of the variable.
    """
    return type(var).__name__

# Example usage:
if __name__ == "__main__":
    # Test with different types
    print(check_type(10))      # Output: int
    print(check_type(10.5))    # Output: float
    print(check_type("Hello")) # Output: str
    print(check_type(True))    # Output: bool
    print(check_type([1, 2]))  # Output: list
    print(check_type((1, 2)))  # Output: tuple
    print(check_type({"a": 1}))# Output: dict
    print(check_type({1, 2}))  # Output: set
    print(check_type(None))    # Output: NoneType
```
