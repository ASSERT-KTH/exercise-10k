## Exercise
Given a dictionary, write a function to access a specific key and return its corresponding value.

## Reference Solution
```python
def get_dict_value(dictionary, key):
    """
    Access a specific key in a dictionary and return its corresponding value.

    Args:
        dictionary (dict): The input dictionary.
        key: The key to access.

    Returns:
        The value corresponding to the given key, or None if the key does not exist.

    Raises:
        TypeError: If the input is not a dictionary.
    """
    if not isinstance(dictionary, dict):
        raise TypeError("Input must be a dictionary")

    return dictionary.get(key)

# Example usage:
example_dict = {"name": "John", "age": 30, "city": "New York"}
key_to_access = "age"

value = get_dict_value(example_dict, key_to_access)
print(f"The value for '{key_to_access}' is: {value}")
```
