## Exercise
Write a program to find the most frequent element in a given list.

## Reference Solution
```python
from collections import Counter

def most_frequent_element(lst):
    """
    Returns the most frequent element in the given list.
    
    Args:
        lst (list): The input list.
    
    Returns:
        The most frequent element in the list.
    """
    if not lst:
        return None
    
    counter = Counter(lst)
    return counter.most_common(1)[0][0]

# Example usage:
numbers = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]
print(most_frequent_element(numbers))  # Output: 4

words = ['apple', 'banana', 'apple', 'orange', 'apple']
print(most_frequent_element(words))  # Output: 'apple'
```
