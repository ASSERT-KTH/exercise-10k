## Exercise
Write a program that takes a number as input and returns its absolute value

## Reference Solution
```python
def absolute_value(num):
    """
    Returns the absolute value of a given number.

    Args:
        num (int): The input number.

    Returns:
        int: The absolute value of the input number.
    """
    return abs(num)

# Example usage
def main():
    num = int(input("Enter a number: "))
    print("The absolute value is:", absolute_value(num))

if __name__ == "__main__":
    main()
```
