## Exercise
Write a program to find the minimum number from a given list of numbers.

## Reference Solution
```python
def find_min_number(num_list):
    """
    Find the minimum number from a given list of numbers.

    Args:
        num_list (list): A list of numbers.

    Returns:
        int: The minimum number in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not num_list:
        raise ValueError("Input list is empty")

    return min(num_list)


# Example usage:
def main():
    numbers = [12, 45, 7, 23, 56, 89, 34]
    try:
        min_number = find_min_number(numbers)
        print("Minimum number:", min_number)
    except ValueError as e:
        print("Error:", e)


if __name__ == "__main__":
    main()
```
