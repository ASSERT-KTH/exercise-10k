## Exercise
Create a function that updates the value associated with a given key in a dictionary, if the key exists, otherwise adds a new key-value pair.

## Reference Solution

def update_dict(dictionary, key, value):
    """
    Updates the value associated with a given key in a dictionary if the key exists.
    Otherwise, adds a new key-value pair to the dictionary.

    Args:
        dictionary (dict): The dictionary to be updated.
        key (any): The key to be searched and updated in the dictionary.
        value (any): The new value associated with the given key.

    Returns:
        dict: The updated dictionary.
    """

    # First, we check if the key already exists in the dictionary
    # If it does, we update its value; otherwise, we add a new key-value pair
    if key in dictionary:
        # If the key exists, we update its value
        dictionary[key] = value
    else:
        # If the key does not exist, we add a new key-value pair
        dictionary[key] = value

    # Finally, we return the updated dictionary
    return dictionary
