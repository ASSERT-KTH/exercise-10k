## Exercise
Write a program that takes an array of integers and a target number as input, and returns the indices of all occurrences of the target number in the array. If the target number is not found, return an empty list.

## Reference Solution
```python
def twoSum(nums, target):
    nums = [(v, i) for i, v in enumerate(nums)]
    nums.sort()
    l, r = 0, len(nums) - 1
    ans = []
    while l < r:
        if nums[l][0] + nums[r][0] == target:
            ans.append([nums[l][1], nums[r][1]])
            l += 1
            r -= 1
        elif nums[l][0] + nums[r][0] < target:
            l += 1
        else:
            r -= 1
    return ans

print(twoSum([2, 7, 11, 15], 9))  # Output: [[0, 1]]
```
