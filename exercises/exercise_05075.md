## Exercise
Declare a variable and increment or decrement it using pre- or post-increment/decrement operators.

## Reference Solution
```python
#include <stdio.h>

int main() {
    int x = 5;

    // Pre-increment
    printf("Pre-increment: x = %d, ++x = %d\n", x, ++x);
    x = 5; // Reset x

    // Post-increment
    printf("Post-increment: x = %d, x++ = %d\n", x, x++);
    x = 5; // Reset x

    // Pre-decrement
    printf("Pre-decrement: x = %d, --x = %d\n", x, --x);
    x = 5; // Reset x

    // Post-decrement
    printf("Post-decrement: x = %d, x-- = %d\n", x, x--);

    return 0;
}
```
