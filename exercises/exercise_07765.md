## Exercise
Create a function called calculator that takes in two numbers and an operation as input. The operation can be either 'add', 'subtract', 'multiply', or 'divide'. The function should then perform the operation on the two numbers and return the result.

## Reference Solution
```python
def calculator(num1, num2, operation):
    """
    This function performs basic arithmetic operations on two numbers.

    Args:
        num1 (float): The first number.
        num2 (float): The second number.
        operation (str): The operation to be performed. It can be 'add', 'subtract', 'multiply', or 'divide'.

    Returns:
        float: The result of the operation.

    Raises:
        ValueError: If the operation is not one of the four basic arithmetic operations.
        ZeroDivisionError: If the operation is 'divide' and the second number is zero.
    """

    # First, we check if the operation is valid
    if operation not in ['add', 'subtract', 'multiply', 'divide']:
        raise ValueError("Invalid operation. Operation must be 'add', 'subtract', 'multiply', or 'divide'.")

    # Then, we perform the operation based on the input
    # For addition, we simply add the two numbers
    if operation == 'add':
        return num1 + num2
    
    # For subtraction, we subtract the second number from the first
    elif operation == 'subtract':
        return num1 - num2
    
    # For multiplication, we multiply the two numbers
    elif operation == 'multiply':
        return num1 * num2
    
    # For division, we divide the first number by the second
    # We also need to check for division by zero
    elif operation == 'divide':
        # Check for division by zero
        if num2 == 0:
            raise ZeroDivisionError("Cannot divide by zero.")
        return num1 / num2

# Example usage:
print(calculator(10, 2, 'add'))  # Output: 12
print(calculator(10, 2, 'subtract'))  # Output: 8
print(calculator(10, 2, 'multiply'))  # Output: 20
print(calculator(10, 2, 'divide'))  # Output: 5.0
```
