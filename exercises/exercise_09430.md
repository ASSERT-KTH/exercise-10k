## Exercise
Create a function that takes a list of numbers and returns their sum, including a docstring to document the function

## Reference Solution
```python
def sum_numbers(numbers):
    """
    Calculate the sum of a list of numbers.

    Args:
        numbers (list): A list of numbers to be summed.

    Returns:
        float: The sum of the numbers in the list.

    Raises:
        TypeError: If the input is not a list.
        ValueError: If the list contains non-numeric values.
    """
    # Check if input is a list
    if not isinstance(numbers, list):
        raise TypeError("Input must be a list")

    # Initialize sum to 0
    total_sum = 0

    # Iterate over each number in the list
    for num in numbers:
        # Check if the number is numeric
        if not isinstance(num, (int, float)):
            raise ValueError("List must only contain numeric values")

        # Add the number to the sum
        total_sum += num

    # Return the sum
    return total_sum


# Example usage:
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    result = sum_numbers(numbers)
    print(f"The sum of {numbers} is: {result}")
```
