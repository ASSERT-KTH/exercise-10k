## Exercise
Write a function to check if a given value is of a specific data type, then test it with the value 123 and '123'.

## Reference Solution
```python
def check_data_type(value, data_type):
    """
    Checks if a given value is of a specific data type.

    Args:
        value (any): The value to check.
        data_type (type): The expected data type.

    Returns:
        bool: True if the value is of the expected data type, False otherwise.
    """
    return isinstance(value, data_type)

# Test the function
print(check_data_type(123, int))  # Expected output: True
print(check_data_type('123', str))  # Expected output: True
print(check_data_type(123, str))  # Expected output: False
print(check_data_type('123', int))  # Expected output: False
```
