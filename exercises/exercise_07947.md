## Exercise
Write a program to calculate the size of different data types (e.g., int, float, char) in bytes.

## Reference Solution
```python
import sys

def calculate_data_type_sizes():
    # List of data types to calculate sizes for
    data_types = [
        int,
        float,
        bool,
        complex,
        str,
        bytes,
        bytearray,
        memoryview,
        tuple,
        list,
        dict,
        set,
        frozenset,
    ]

    # Print header
    print(f"{'Data Type':^20} | {'Size (bytes)':^15}")
    print("-" * 40)

    # Calculate and print sizes for each data type
    for data_type in data_types:
        size = sys.getsizeof(data_type())
        print(f"{data_type.__name__:^20} | {size:^15}")

if __name__ == "__main__":
    calculate_data_type_sizes()
```
