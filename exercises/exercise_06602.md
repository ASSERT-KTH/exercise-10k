## Exercise
Write a program that asks the user for their name and then prints out a greeting message with the name.

## Reference Solution
```python
def main():
    # Ask the user for their name
    name = input("What is your name? ")

    # Print out a greeting message with the name
    print(f"Hello, {name}!")

if __name__ == "__main__":
    main()
```
