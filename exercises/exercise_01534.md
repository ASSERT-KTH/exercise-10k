## Exercise
Write a program that finds the longest common prefix of two strings.

## Reference Solution
```python
def longest_common_prefix(str1, str2):
    """
    Returns the longest common prefix of two strings.

    Args:
        str1 (str): The first input string.
        str2 (str): The second input string.

    Returns:
        str: The longest common prefix of str1 and str2.
    """
    prefix = ""
    for char1, char2 in zip(str1, str2):
        if char1 == char2:
            prefix += char1
        else:
            break
    return prefix
```
