## Exercise
Write a program that takes a list of shopping items as input, where each item is on a new line, and outputs a formatted list with each item numbered and followed by a period. The input will be read from the console, and the output should also be written to the console.

## Reference Solution
```python
import sys

def shop_list():
    items = []

    for line in sys.stdin:
        line = line.strip()
        items.append(line)

    for i in items:
        print(f'{items.index(i) + 1}. {i}')

if __name__ == "__main__":
    shop_list()
```
