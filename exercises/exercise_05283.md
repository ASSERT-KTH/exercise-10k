## Exercise
Create a program that checks whether a user-inputted number is prime or not. Use a while loop to check divisibility from 2 to the square root of the number.

## Reference Solution
```python
import math

def is_prime(num):
    """Check if a number is prime."""
    if num < 2:
        return False

    # Check divisibility from 2 to the square root of the number
    i = 2
    while i <= math.sqrt(num):
        if num % i == 0:
            return False
        i += 1

    return True

def main():
    # Get user input
    num = int(input("Enter a number: "))

    # Check if the number is prime
    if is_prime(num):
        print(f"{num} is a prime number.")
    else:
        print(f"{num} is not a prime number.")

if __name__ == "__main__":
    main()
```
