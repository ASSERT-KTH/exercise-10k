## Exercise
Convert a given string to an integer and print it out

## Reference Solution
```python
def string_to_int(input_str):
    """
    Convert a given string to an integer and print it out.

    Args:
        input_str (str): The input string to be converted.

    Returns:
        None
    """
    try:
        # Attempt to convert the string to an integer
        integer = int(input_str)
        print(f"The integer equivalent of '{input_str}' is: {integer}")
    except ValueError:
        print(f"Error: Unable to convert '{input_str}' to an integer.")

# Example usage
if __name__ == "__main__":
    input_str = "123"
    string_to_int(input_str)
```
