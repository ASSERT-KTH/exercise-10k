## Exercise
Write a program that takes a list and a grouping function as input, and returns a dictionary where the keys are the groups and the values are lists of elements that belong to each group.

## Reference Solution
```python
from collections import defaultdict

def group_by(list_, f):
    groups = defaultdict(list)
    for item in list_:
        groups[f(item)].append(item)
    return dict(groups)

# Example usage:
numbers = [1, 2, 3, 4, 5, 6]
def type_of(x):
    return x % 2 == 0

result = group_by(numbers, type_of)
print(result)  # Output: {False: [1, 3, 5], True: [2, 4, 6]}
```
