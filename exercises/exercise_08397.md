## Exercise
Create a function that takes a string as input and checks if it contains only letters and spaces. If the string is valid, return the string with all letters converted to uppercase. If the string is not valid, return an error message.

## Reference Solution
```python
def check_string(input_str):
    """
    This function checks if the input string contains only letters and spaces.
    If the string is valid, it returns the string with all letters converted to uppercase.
    If the string is not valid, it returns an error message.

    Parameters:
    input_str (str): The input string to be checked.

    Returns:
    str: The input string in uppercase if valid, otherwise an error message.
    """

    # First, we want to check if all characters in the string are either letters or spaces
    # We can use the isalpha() method to check if a character is a letter, and the isspace() method to check if a character is a space
    # We'll use a for loop to iterate over each character in the string
    for char in input_str:
        
        # If the character is not a letter and not a space, we immediately return an error message
        if not char.isalpha() and not char.isspace():
            return "Error: String contains invalid characters."

    # If we've made it through the entire string without returning an error message, then the string is valid
    # Now, we want to convert all the letters in the string to uppercase
    # We can use the upper() method to do this
    # This method returns a copy of the string with all the letters converted to uppercase
    return input_str.upper()

# Test the function
print(check_string("Hello World"))  # Should print "HELLO WORLD"
print(check_string("Hello123"))  # Should print "Error: String contains invalid characters."
```
