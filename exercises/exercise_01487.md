## Exercise
Write a program that demonstrates the concept of variable scope in programming.

## Reference Solution
```python
# Global variable
global_variable = "I am a global variable"

def outer_function():
    # Enclosing variable
    enclosing_variable = "I am an enclosing variable"

    def inner_function():
        # Local variable
        local_variable = "I am a local variable"
        print("Inner function:")
        print(f"  Local variable: {local_variable}")
        print(f"  Enclosing variable: {enclosing_variable}")
        print(f"  Global variable: {global_variable}")

    inner_function()
    print("Outer function:")
    print(f"  Enclosing variable: {enclosing_variable}")
    print(f"  Global variable: {global_variable}")

outer_function()
print("Global scope:")
print(f"  Global variable: {global_variable}")

try:
    print(f"  Enclosing variable: {enclosing_variable}")
except NameError:
    print("  Enclosing variable is not accessible in the global scope")

try:
    print(f"  Local variable: {local_variable}")
except NameError:
    print("  Local variable is not accessible in the global scope")
```
