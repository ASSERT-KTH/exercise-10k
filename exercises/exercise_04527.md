## Exercise
Write a program that takes an array and an index as input and returns the array with the element at the specified index removed.

## Reference Solution
```python
def removeElement(arr, index):
    return arr[:index] + arr[index + 1:]

arr = [1, 2, 3, 4, 5]
print(removeElement(arr, 2))
```
