## Exercise
Write a function that takes a list and two indices as input, and returns a slice of the list between those indices.

## Reference Solution
```python
def slice_list(input_list, start_index, end_index):
    """
    This function takes a list and two indices as input, and returns a slice of the list between those indices.

    Args:
        input_list (list): The input list to be sliced.
        start_index (int): The starting index of the slice.
        end_index (int): The ending index of the slice.

    Returns:
        list: A slice of the input list between the start and end indices.
    """

    # First, we need to check if the input is a list. If not, we raise a TypeError.
    if not isinstance(input_list, list):
        raise TypeError("Input must be a list.")

    # Then, we need to check if the start and end indices are integers. If not, we raise a TypeError.
    if not isinstance(start_index, int) or not isinstance(end_index, int):
        raise TypeError("Start and end indices must be integers.")

    # Next, we need to check if the start index is less than or equal to the end index. If not, we raise a ValueError.
    if start_index > end_index:
        raise ValueError("Start index cannot be greater than end index.")

    # Finally, we can use Python's list slicing feature to get the desired slice.
    # We use the slicing syntax input_list[start_index:end_index] to get the slice.
    # This will return a new list containing all elements from the start index up to but not including the end index.
    return input_list[start_index:end_index]
```
