## Exercise
Write a program to count the number of positive numbers in an array.

## Reference Solution
```python
def count_positive_numbers(array):
    """
    Counts the number of positive numbers in an array.

    Args:
        array (list): A list of integers.

    Returns:
        int: The count of positive numbers in the array.
    """
    count = 0
    for num in array:
        if num > 0:
            count += 1
    return count

# Example usage
def main():
    array = [1, -2, 3, 0, 4, -5, 6]
    print("Array:", array)
    print("Count of positive numbers:", count_positive_numbers(array))

if __name__ == "__main__":
    main()
```
