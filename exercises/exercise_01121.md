## Exercise
Create a program that takes a user's name as input and prints out a greeting message

## Reference Solution
```python
def greet_user(name):
    """Prints out a greeting message for the given user."""
    print(f"Hello, {name}!")

def main():
    name = input("What is your name? ")
    greet_user(name)

if __name__ == "__main__":
    main()
```
