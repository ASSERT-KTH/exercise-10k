## Exercise
Convert a dictionary to a list of key-value pairs

## Reference Solution
```python
def dict_to_list(dictionary):
    """
    Convert a dictionary to a list of key-value pairs.
    
    Args:
        dictionary (dict): The dictionary to convert.
    
    Returns:
        list: A list of tuples, where each tuple is a key-value pair from the dictionary.
    """
    return list(dictionary.items())

# Example usage
my_dict = {'a': 1, 'b': 2, 'c': 3}
key_value_pairs = dict_to_list(my_dict)
print(key_value_pairs)  # Output: [('a', 1), ('b', 2), ('c', 3)]
```
