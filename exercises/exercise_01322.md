## Exercise
Write a function that takes a string as input and returns True if the string is a palindrome, False otherwise.

## Reference Solution
```python
def is_palindrome(s: str) -> bool:
    """
    Checks if a string is a palindrome.

    Args:
        s (str): The input string.

    Returns:
        bool: True if the string is a palindrome, False otherwise.
    """
    s = ''.join(c for c in s if c.isalnum()).lower()  # Remove non-alphanumeric characters and convert to lowercase
    return s == s[::-1]  # Check if the string is equal to its reverse

# Example usage:
if __name__ == "__main__":
    print(is_palindrome("radar"))  # True
    print(is_palindrome("hello"))  # False
    print(is_palindrome("A man, a plan, a canal: Panama"))  # True
    print(is_palindrome("Not a palindrome"))  # False
```
