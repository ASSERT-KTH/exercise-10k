## Exercise
Write a program that takes a string as input and returns the string reversed. For example, if the input is 'hello', the output should be 'olleh'.

## Reference Solution
```python
def reverse_string(input_str):
    """
    Reverses the input string.

    Args:
        input_str (str): The input string to be reversed.

    Returns:
        str: The reversed string.
    """
    return input_str[::-1]

def main():
    # Example usage
    input_str = 'hello'
    reversed_str = reverse_string(input_str)
    print(f"Input: {input_str}")
    print(f"Reversed: {reversed_str}")

if __name__ == "__main__":
    main()
```
