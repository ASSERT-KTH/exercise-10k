## Exercise
Write a function that takes an array of integers as input and returns a new array where all the elements are in reverse order. The function should not modify the original array.

## Reference Solution
```python
def reverse_list(array):
    new_array = []
    for i in range(len(array) - 1, -1, -1):
        new_array.append(array[i])
    return new_array

# Test cases
print(reverse_list([1, 2, 3, 4, 5]))  # Output: [5, 4, 3, 2, 1]
print(reverse_list([10, 20, 30, 40, 50]))  # Output: [50, 40, 30, 20, 10]
```
