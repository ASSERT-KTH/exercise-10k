## Exercise
Write a program that takes a string as input and prints out the number of characters in the string

## Reference Solution
```python
def count_characters(input_string):
    """
    Returns the number of characters in the input string.
    
    Args:
        input_string (str): The input string to count characters from.
    
    Returns:
        int: The number of characters in the input string.
    """
    return len(input_string)

def main():
    # Get user input
    input_string = input("Enter a string: ")
    
    # Count characters and print result
    character_count = count_characters(input_string)
    print(f"The string '{input_string}' has {character_count} characters.")

if __name__ == "__main__":
    main()
```
