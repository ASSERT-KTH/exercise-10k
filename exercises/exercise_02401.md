## Exercise
Write a function that takes a single argument and returns its square. Then, call this function with the input number and print the result.

## Reference Solution
```python
def square(num):
    """
    Returns the square of a given number.

    Args:
        num (int or float): The input number.

    Returns:
        int or float: The square of the input number.
    """
    return num ** 2

# Example usage
def main():
    input_number = 5  # Replace with your desired input number
    result = square(input_number)
    print(f"The square of {input_number} is: {result}")

if __name__ == "__main__":
    main()
```
