## Exercise
Write a program to find the maximum value in a list of integers.

## Reference Solution
```python
def find_max_value(lst):
    """
    This function finds the maximum value in a list of integers.

    Args:
        lst (list): A list of integers.

    Returns:
        int: The maximum value in the list.

    Raises:
        ValueError: If the list is empty.
    """
    if not lst:
        raise ValueError("List is empty")

    return max(lst)


def main():
    # Example usage:
    numbers = [12, 45, 7, 23, 56, 89, 34]
    try:
        max_value = find_max_value(numbers)
        print(f"Maximum value in the list: {max_value}")
    except ValueError as e:
        print(e)


if __name__ == "__main__":
    main()
```
