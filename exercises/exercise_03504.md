## Exercise
Write a program that declares and assigns values to variables of different data types (integer, float, string, boolean).

## Reference Solution
```python
# Integer variable
integer_var = 10
print("Integer variable:", integer_var)
print("Data type of integer_var:", type(integer_var))

# Float variable
float_var = 10.5
print("\nFloat variable:", float_var)
print("Data type of float_var:", type(float_var))

# String variable
string_var = "Hello, World!"
print("\nString variable:", string_var)
print("Data type of string_var:", type(string_var))

# Boolean variable
boolean_var = True
print("\nBoolean variable:", boolean_var)
print("Data type of boolean_var:", type(boolean_var))
```
