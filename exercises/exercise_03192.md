## Exercise
Write a program that takes three numbers as input and uses if/else statements to determine the smallest number.

## Reference Solution
```python
def smallest_number(a, b, c):
    """
    This function determines the smallest number among three input numbers.

    Args:
    a (int): The first number.
    b (int): The second number.
    c (int): The third number.

    Returns:
    int: The smallest number.
    """
    if a <= b and a <= c:
        return a
    elif b <= a and b <= c:
        return b
    else:
        return c

def main():
    # Example usage:
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))
    num3 = float(input("Enter the third number: "))

    smallest = smallest_number(num1, num2, num3)
    print(f"The smallest number is: {smallest}")

if __name__ == "__main__":
    main()
```
