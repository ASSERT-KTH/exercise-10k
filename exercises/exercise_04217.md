## Exercise
Calculate the median of an array.

## Reference Solution
```python
def calculate_median(array):
    """
    Calculate the median of an array.

    Args:
        array (list): The input array.

    Returns:
        float: The median of the array.
    """
    # Sort the array
    array.sort()

    # Determine the length of the array
    length = len(array)

    # Calculate the median
    if length % 2 == 0:  # Even length
        mid1 = array[length // 2 - 1]
        mid2 = array[length // 2]
        median = (mid1 + mid2) / 2
    else:  # Odd length
        median = array[length // 2]

    return median

# Example usage
array = [1, 3, 5, 7, 9]
median = calculate_median(array)
print("Median:", median)  # Output: 5
```
