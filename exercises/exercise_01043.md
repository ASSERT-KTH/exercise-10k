## Exercise
Write a program that takes two arrays as input, and returns their concatenation.

## Reference Solution
```python
def concatenate_arrays(array1, array2):
    """
    Concatenates two input arrays.

    Args:
        array1 (list): The first array.
        array2 (list): The second array.

    Returns:
        list: The concatenated array.
    """
    return array1 + array2


# Example usage:
if __name__ == "__main__":
    array1 = [1, 2, 3]
    array2 = [4, 5, 6]

    concatenated = concatenate_arrays(array1, array2)
```
