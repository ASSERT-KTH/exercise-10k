## Exercise
Write a function that takes an array of integers as input and returns the reversed array. The function should not use any built-in reverse functions.

## Reference Solution
```python
def reverse_array(arr):
    start = 0
    end = len(arr) - 1
    while start < end:
        arr[start], arr[end] = arr[end], arr[start]
        start += 1
        end -= 1
    return arr

arr = [1, 2, 3, 4, 5]
print(reverse_array(arr))
```
