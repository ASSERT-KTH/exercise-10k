## Exercise
Write a program that uses string formatting to insert values into a string.

## Reference Solution
```python
# Define a function to demonstrate string formatting
def demonstrate_string_formatting():
    # Define variables to be inserted into the string
    name = "John Doe"
    age = 30
    city = "New York"

    # Using f-string formatting (Python 3.6+)
    formatted_string_f = f"My name is {name}, I am {age} years old, and I live in {city}."
    print("Using f-string formatting:")
    print(formatted_string_f)

    # Using str.format() method
    formatted_string_format = "My name is {}, I am {} years old, and I live in {}.".format(name, age, city)
    print("\nUsing str.format() method:")
    print(formatted_string_format)

    # Using % operator
    formatted_string_percent = "My name is %s, I am %d years old, and I live in %s." % (name, age, city)
    print("\nUsing % operator:")
    print(formatted_string_percent)

# Call the function to demonstrate string formatting
demonstrate_string_formatting()
```
