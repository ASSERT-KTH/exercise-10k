## Exercise
Write a program that takes a list and a value as input and returns the index of the first occurrence of the value in the list. If the value is not found, return -1.

## Reference Solution
```python
def find_index(l, val):
    try:
        return l.index(val)
    except ValueError:
        return -1

# Test cases
print(find_index([1, 2, 3], 2))  # Expected output: 1
print(find_index([1, 2, 3], 4))  # Expected output: -1
```
