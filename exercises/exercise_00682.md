## Exercise
Write a program to print the multiplication table for a given number.

## Reference Solution
```python
def print_multiplication_table(num, limit):
    """
    Prints the multiplication table for a given number up to a specified limit.

    Args:
        num (int): The number for which the multiplication table is to be printed.
        limit (int): The upper limit of the multiplication table.
    """
    print(f"Multiplication Table for {num}:")
    for i in range(1, limit + 1):
        print(f"{num} x {i} = {num * i}")


def main():
    num = int(input("Enter a number: "))
    limit = int(input("Enter the limit: "))
    print_multiplication_table(num, limit)


if __name__ == "__main__":
    main()
```
