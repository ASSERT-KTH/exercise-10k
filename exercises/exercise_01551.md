## Exercise
Write a function to insert a new element at a specified position in a given list of integers.

## Reference Solution
```python
def insert_element(lst, element, position):
    """
    Inserts a new element at a specified position in a given list of integers.

    Args:
        lst (list): The list of integers.
        element (int): The new element to be inserted.
        position (int): The position where the new element is to be inserted.

    Returns:
        list: The updated list with the new element inserted.
    """
    # Check if the position is within the bounds of the list
    if position < 0 or position > len(lst):
        raise IndexError("Position out of range")

    # Insert the element at the specified position
    lst.insert(position, element)

    return lst

# Example usage:
numbers = [1, 2, 3, 5]
new_number = 4
position = 3

updated_numbers = insert_element(numbers, new_number, position)
print(updated_numbers)  # Output: [1, 2, 3, 4, 5]
```
