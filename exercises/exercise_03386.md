## Exercise
Write a function to find the minimum value in an array of integers.

## Reference Solution
```python
def find_minimum_value(array):
    """
    Finds the minimum value in an array of integers.

    Args:
        array (list): A list of integers.

    Returns:
        int: The minimum value in the array.

    Raises:
        ValueError: If the array is empty.
    """
    if not array:
        raise ValueError("Array is empty")

    return min(array)

# Example usage
def main():
    array = [10, 20, 3, 45, 6]
    print("Minimum value:", find_minimum_value(array))

if __name__ == "__main__":
    main()
```
