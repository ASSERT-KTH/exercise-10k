## Exercise
Develop a program to compare the prices of two items and determine which one is cheaper using conditional statements

## Reference Solution
```python
def compare_prices(item1_price, item2_price):
    """
    Compare the prices of two items and return the cheaper one.

    Args:
        item1_price (float): The price of the first item.
        item2_price (float): The price of the second item.

    Returns:
        str: A message indicating which item is cheaper.
    """
    if item1_price < item2_price:
        return f"Item 1 is cheaper by ${item2_price - item1_price:.2f}"
    elif item2_price < item1_price:
        return f"Item 2 is cheaper by ${item1_price - item2_price:.2f}"
    else:
        return "Both items have the same price"

def main():
    # Get the prices of the two items from the user
    item1_price = float(input("Enter the price of Item 1: $"))
    item2_price = float(input("Enter the price of Item 2: $"))

    # Compare the prices and print the result
    print(compare_prices(item1_price, item2_price))

if __name__ == "__main__":
    main()
```
