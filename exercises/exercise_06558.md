## Exercise
Write a function that splits a given string into substrings based on a specified delimiter.

## Reference Solution
```python
def split_string(input_str, delimiter):
    """
    This function splits a given string into substrings based on a specified delimiter.

    Args:
        input_str (str): The input string to be split.
        delimiter (str): The delimiter used to split the string.

    Returns:
        list: A list of substrings split from the input string.
    """

    # First, we check if the input string is indeed a string
    if not isinstance(input_str, str):
        raise TypeError("Input must be a string.")

    # Then, we check if the delimiter is also a string
    if not isinstance(delimiter, str):
        raise TypeError("Delimiter must be a string.")

    # We use the built-in split method of Python strings, which splits a string into a list where each word is a list item
    # This method takes an optional argument, which is the delimiter to split the string by
    # If no delimiter is provided, it defaults to any whitespace character
    substrings = input_str.split(delimiter)

    # Finally, we return the list of substrings
    return substrings

# Example usage:
print(split_string("hello,world,this,is,a,test", ","))  # Output: ['hello', 'world', 'this', 'is', 'a', 'test']
```
