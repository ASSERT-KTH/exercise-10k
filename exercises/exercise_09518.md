## Exercise
Write a program that takes a string as input and returns the reversed string. The program should handle strings of any length and should not use any built-in reverse functions.

## Reference Solution
```python
def reverse(input_str):
    output_str = ""
    for i in range(len(input_str)):
        output_str += input_str[i]
    return output_str

# Test the function
```

```python
def reverse(input_str):
    output_str = ""
    i = len(input_str) - 1
    while i >= 0:
```
