## Exercise
Write a function that takes a variable number of arguments and returns their sum.

## Reference Solution
```python
# Define a function that takes a variable number of arguments
def sum_variables(*args):
    """
    This function takes a variable number of arguments and returns their sum.

    Args:
        *args: A variable number of arguments

    Returns:
        The sum of all the arguments
    """

    # Initialize the sum to 0
    total_sum = 0
    
    # Iterate over each argument
    for arg in args:
        # Add the current argument to the total sum
        total_sum += arg
    
    # Return the total sum
    return total_sum

# Test the function
print(sum_variables(1, 2, 3, 4, 5))  # Output: 15
```
