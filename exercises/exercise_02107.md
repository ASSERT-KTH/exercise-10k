## Exercise
Write a function to create a copy of a dictionary

## Reference Solution
```python
def create_dict_copy(input_dict):
    """
    Create a copy of a dictionary using the copy() method.

    Args:
        input_dict (dict): The dictionary to be copied.

    Returns:
        dict: A copy of the input dictionary.
    """
    return input_dict.copy()

# Example usage:
input_dict = {'a': 1, 'b': 2, 'c': 3}
dict_copy = create_dict_copy(input_dict)
print(dict_copy)  # Output: {'a': 1, 'b': 2, 'c': 3}
```
