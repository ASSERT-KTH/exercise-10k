## Exercise
Use the map function to apply a given function to each element of a list.

## Reference Solution
```python
def square(num):
    """Return the square of a number."""
    return num ** 2

# Define a list of numbers
numbers = [1, 2, 3, 4, 5]

# Use map() to apply the square function to each number
squares = list(map(square, numbers))

print(squares)  # Output: [1, 4, 9, 16, 25]
```

```python
numbers = [1, 2, 3, 4, 5]
squares = list(map(lambda x: x ** 2, numbers))
print(squares)  # Output: [1, 4, 9, 16, 25]
```

```python
def multiply(a, b):
    return a * b

numbers1 = [1, 2, 3]
numbers2 = [4, 5, 6]

products = list(map(multiply, numbers1, numbers2))
```
