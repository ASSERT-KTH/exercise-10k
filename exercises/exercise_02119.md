## Exercise
Write a function to create a new dictionary using a dictionary comprehension

## Reference Solution
```python
def create_new_dict(input_dict, conversion_func):
    """
    Creates a new dictionary using dictionary comprehension.

    Args:
    input_dict (dict): The input dictionary.
    conversion_func (function): A function to apply to each value in the dictionary.

    Returns:
    dict: A new dictionary with the converted values.
    """
    return {key: conversion_func(value) for key, value in input_dict.items()}

# Example usage:
numbers = {'a': 1, 'b': 2, 'c': 3}

# Define a conversion function to square the numbers
def square(x):
    return x ** 2

new_dict = create_new_dict(numbers, square)
print(new_dict)  # Output: {'a': 1, 'b': 4, 'c': 9}
```
