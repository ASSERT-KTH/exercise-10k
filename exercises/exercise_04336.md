## Exercise
Write a program to find the union of two arrays.

## Reference Solution
```python
def union(arr1, arr2):
    """
    Returns the union of two input arrays.

    Args:
        arr1 (list): The first input array.
        arr2 (list): The second input array.

    Returns:
        list: The union of arr1 and arr2.
    """
    return list(set(arr1 + arr2))

def main():
    # Example usage
    arr1 = [1, 2, 3, 4, 5]
    arr2 = [4, 5, 6, 7, 8]

    print("Array 1:", arr1)
    print("Array 2:", arr2)
    print("Union:", union(arr1, arr2))

if __name__ == "__main__":
    main()
```
