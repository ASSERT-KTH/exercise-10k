## Exercise
Write a function that counts the number of letters in a given string.

## Reference Solution
```python
def count_letters(input_string: str) -> int:
    """
    Counts the number of letters in a given string.

    Args:
        input_string (str): The input string to count letters from.

    Returns:
        int: The number of letters in the input string.
    """
    return sum(1 for char in input_string if char.isalpha())

# Example usage:
def main():
    input_str = "Hello, World!"
    letter_count = count_letters(input_str)
    print(f"The string '{input_str}' contains {letter_count} letters.")

if __name__ == "__main__":
    main()
```
