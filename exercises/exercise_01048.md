## Exercise
Write a program that takes a list, an element, and an index as input, and replaces the element at the given index in the list with the new element.

## Reference Solution
```python
def replace_all(my_list, search_element, replace_element):
    for i in range(len(my_list)):
        if my_list[i] == search_element:
            my_list[i] = replace_element
    return my_list

def replace_element(my_list, idx, replace_element):
    if idx < len(my_list):
        my_list[idx] = replace_element
    return my_list

# Example usage
my_list = [1, 2, 3, 4, 5]
idx = 3
replace_element = 6
result = replace_element(my_list, idx, replace_element)
print(result)
```
