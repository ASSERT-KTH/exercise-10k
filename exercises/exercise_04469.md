## Exercise
Write a program to find the minimum number in a list using a for loop

## Reference Solution
```python
def find_min_number(num_list):
    """
    Find the minimum number in a list using a for loop.

    Args:
        num_list (list): A list of numbers.

    Returns:
        int: The minimum number in the list.
    """
    if not num_list:
        return None  # Return None if the list is empty

    min_num = num_list[0]  # Initialize the minimum number with the first element

    for num in num_list:
        if num < min_num:
            min_num = num  # Update the minimum number if a smaller number is found

    return min_num


# Example usage:
numbers = [10, 5, 8, 12, 3]
min_number = find_min_number(numbers)
print("Minimum number in the list:", min_number)  # Output: 3
```
