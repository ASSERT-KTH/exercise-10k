## Exercise
Create a program that takes a word as input and prints out its length.

## Reference Solution
```python
def get_word_length(word):
    """
    Returns the length of the input word.

    Args:
        word (str): The input word.

    Returns:
        int: The length of the word.
    """
    return len(word)

def main():
    # Get user input
    word = input("Enter a word: ")

    # Calculate and print word length
    word_length = get_word_length(word)
    print(f"The length of '{word}' is {word_length}.")

if __name__ == "__main__":
    main()
```
