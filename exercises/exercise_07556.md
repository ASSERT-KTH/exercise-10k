## Exercise
Write a function that takes another function as input and applies it to a given number.

## Reference Solution
```python
# Define a function that takes another function and a number as input.
def apply_function(func, num):
    """
    Applies a given function to a given number.

    Args:
        func (function): The function to be applied.
        num (int or float): The number to which the function is applied.

    Returns:
        The result of applying the function to the number.
    """

    # Check if the input is a function and a number
    if not callable(func) or not isinstance(num, (int, float)):
        raise TypeError("Input must be a function and a number")

    # Apply the function to the number and return the result
    # We assume that the function can handle the given number as input
    return func(num)


# Example usage:
# Define a simple function that squares a number
def square(x):
    return x ** 2

# Apply the square function to the number 5
result = apply_function(square, 5)
print(result)  # Outputs: 25
```
