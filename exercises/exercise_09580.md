## Exercise
Write a program to find the maximum value in a list of integers. The program should take a list of integers as input, and then return the maximum value in the list.

## Reference Solution

```python
def find_max_value(lst):
    if not lst:
        return None
    return max(lst)

# Example usage:
numbers = [12, 45, 7, 23, 56, 89, 34]
print(find_max_value(numbers))  # Output: 89
```

```python
def find_max_value(lst):
    if not lst:
        return None
    max_value = lst[0]
    for num in lst[1:]:
        if num > max_value:
            max_value = num
    return max_value

# Example usage:
numbers = [12, 45, 7, 23, 56, 89, 34]
```
