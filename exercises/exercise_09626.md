## Exercise
Create a function that takes a name and age as keyword arguments and prints out a personalized greeting.

## Reference Solution
```python
def print_greeting(**kwargs):
    """
    This function prints out a personalized greeting based on the provided name and age.

    Args:
        **kwargs: A dictionary of keyword arguments. It should contain 'name' and 'age' as keys.

    Returns:
        None
    """

    # First, we check if the 'name' key exists in the kwargs dictionary
    if 'name' not in kwargs:
        raise ValueError("The 'name' keyword argument is required.")

    # Then, we check if the 'age' key exists in the kwargs dictionary
    if 'age' not in kwargs:
        raise ValueError("The 'age' keyword argument is required.")

    # Now, we extract the name and age from the kwargs dictionary
    name = kwargs['name']
    age = kwargs['age']

    # We then print out the personalized greeting
    print(f"Hello, my name is {name} and I am {age} years old.")

# To test the function
print_greeting(name='John', age=30)
```
