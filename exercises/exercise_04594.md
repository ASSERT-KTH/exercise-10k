## Exercise
Write a function that takes a shape and its dimensions as input and returns the area

## Reference Solution
```python
import math

def calculate_area(shape, **dimensions):
    """
    Calculates the area of a given shape based on its dimensions.

    Args:
        shape (str): The type of shape (rectangle, square, circle, triangle)
        **dimensions: The dimensions of the shape (e.g. length, width, radius, base, height)

    Returns:
        float: The area of the shape
    """
    if shape == 'rectangle':
        # Rectangle area = length * width
        return dimensions['length'] * dimensions['width']
    elif shape == 'square':
        # Square area = side * side
        return dimensions['side'] ** 2
    elif shape == 'circle':
        # Circle area = Ï€ * radius^2
        return math.pi * (dimensions['radius'] ** 2)
    elif shape == 'triangle':
        # Triangle area = (base * height) / 2
        return (dimensions['base'] * dimensions['height']) / 2
    else:
        raise ValueError("Invalid shape. Supported shapes are: rectangle, square, circle, triangle")

# Example usage
print(calculate_area('rectangle', length=4, width=5))  # Output: 20
print(calculate_area('square', side=4))  # Output: 16
print(calculate_area('circle', radius=3))  # Output: 28.274333882308138
print(calculate_area('triangle', base=3, height=4))  # Output: 6.0
```
