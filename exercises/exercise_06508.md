## Exercise
Write a program to perform basic string operations (concatenation, substring, length).

## Reference Solution
```python
def main():
    str1 = input("Enter first string: ")
    str2 = input("Enter second string: ")

    # Concatenation
    print("Concatenation of '{}' and '{}' is: {}".format(str1, str2, str1 + str2))

    # Substring
    start = int(input("Enter start index for substring: "))
    end = int(input("Enter end index for substring: "))
    print("Substring of '{}' from index {} to {} is: {}".format(str1, start, end, str1[start:end]))

    # Length
    print("Length of '{}' is: {}".format(str1, len(str1)))
    print("Length of '{}' is: {}".format(str2, len(str2)))

if __name__ == "__main__":
    main()
```
