## Exercise
Write a program that takes a list as input and returns the average value of the list.

## Reference Solution
```python
def calculate_average(lst):
    """
    Calculate the average value of a list.

    Args:
        lst (list): A list of numbers.

    Returns:
        float: The average value of the list.

    Raises:
        ValueError: If the list is empty.
        TypeError: If the list contains non-numeric values.
    """
    if not lst:
        raise ValueError("Cannot calculate average of an empty list")

    try:
        return sum(lst) / len(lst)
    except TypeError:
        raise TypeError("List contains non-numeric values")


# Example usage:
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    try:
        average = calculate_average(numbers)
        print(f"The average value of the list is: {average}")
    except (ValueError, TypeError) as e:
        print(f"Error: {e}")
```
