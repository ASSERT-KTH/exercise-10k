## Exercise
Perform basic tuple operations (access, unpack) and print out the resulting values.

## Reference Solution
```python
# Define a tuple
my_tuple = (1, 2, 3, 4, 5)

# Accessing tuple elements
print("Accessing tuple elements:")
print("First element:", my_tuple[0])
print("Last element:", my_tuple[-1])

# Unpacking a tuple
print("\nUnpacking a tuple:")
a, b, c, d, e = my_tuple
print("Unpacked values: a =", a, ", b =", b, ", c =", c, ", d =", d, ", e =", e)

# Unpacking a tuple with variable length
print("\nUnpacking a tuple with variable length:")
my_tuple_var = (1, 2, 3, 4, 5)
first, *middle, last = my_tuple_var
print("First element:", first)
print("Middle elements:", middle)
print("Last element:", last)

# Accessing tuple elements using slicing
print("\nAccessing tuple elements using slicing:")
print("First three elements:", my_tuple[:3])
print("Last three elements:", my_tuple[-3:])

# Tuple concatenation
print("\nTuple concatenation:")
tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)
print("Tuple 1:", tuple1)
print("Tuple 2:", tuple2)
print("Concatenated tuple:", tuple1 + tuple2)

# Tuple repetition
print("\nTuple repetition:")
print("Repeated tuple:", my_tuple * 2)
```
