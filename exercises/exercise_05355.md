## Exercise
Create a function that takes an array and a modification function as input, and applies the modification function to each element of the array.

## Reference Solution
```python
def apply_modification(arr, func):
    """
    Applies a modification function to each element of an array.

    Args:
        arr (list): The input array.
        func (function): The modification function.

    Returns:
        list: The modified array.
    """

    # Initialize an empty list to store the modified elements
    modified_arr = []

    # Iterate over each element in the input array
    for element in arr:
        # Apply the modification function to the current element
        modified_element = func(element)
        
        # Append the modified element to the modified array
        modified_arr.append(modified_element)

    # Return the modified array
    return modified_arr

# Example usage:
# Define a modification function that squares its input
def square(x):
    return x ** 2

# Create an example array
arr = [1, 2, 3, 4, 5]

# Apply the modification function to the array
modified_arr = apply_modification(arr, square)

# Print the modified array
print(modified_arr)  # Output: [1, 4, 9, 16, 25]
```
