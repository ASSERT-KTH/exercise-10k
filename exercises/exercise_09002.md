## Exercise
Write a function that takes a string as input and returns the count of words.

## Reference Solution
```python
def count_words(input_string):
    """
    Returns the count of words in the input string.

    Args:
        input_string (str): The input string.

    Returns:
        int: The count of words in the input string.
    """
    # Remove leading and trailing whitespace
    input_string = input_string.strip()
    
    # Split the string into words
    words = input_string.split()
    
    # Return the count of words
    return len(words)

# Example usage
def main():
    input_str = "This is an example string"
    word_count = count_words(input_str)
    print(f"The word count is: {word_count}")

if __name__ == "__main__":
    main()
```
