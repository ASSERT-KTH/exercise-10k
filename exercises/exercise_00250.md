## Exercise
Write a function that takes a list as input, and returns a reversed version of the list.

## Reference Solution
```python
def reverse_list(input_list):
    """
    Returns a reversed version of the input list.

    Args:
        input_list (list): The list to be reversed.

    Returns:
        list: The reversed list.
    """
    return input_list[::-1]

# Example usage
def main():
    my_list = [1, 2, 3, 4, 5]
    print("Original List:", my_list)
    print("Reversed List:", reverse_list(my_list))

if __name__ == "__main__":
    main()
```
