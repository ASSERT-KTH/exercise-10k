## Exercise
Declare a variable to store a person's name and age, then print out the person's information

## Reference Solution
```python
# Declare a dictionary to store person's information
person = {
    "name": "John Doe",
    "age": 30
}

# Print out the person's information
print("Name:", person["name"])
print("Age:", person["age"])
```

```python
# Define a class to represent a person
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def print_info(self):
        print("Name:", self.name)
        print("Age:", self.age)

# Create a person object and print its information
person = Person("John Doe", 30)
person.print_info()
```
