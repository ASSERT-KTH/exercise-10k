## Exercise
Create a program that takes a letter as input and returns whether it is a vowel or consonant using conditional statements.

## Reference Solution
```python
# Define a function to check if a letter is a vowel or consonant
def check_letter(letter):
    # Convert the letter to lowercase
    letter = letter.lower()
    
    # Check if the letter is a vowel
    if letter in 'aeiou':
        return "Vowel"
    # Check if the letter is a consonant (assuming only alphabets are input)
    elif letter.isalpha():
        return "Consonant"
    else:
        return "Not a letter"

# Get user input
letter = input("Enter a letter: ")

# Check the letter and print the result
print(check_letter(letter))
```
