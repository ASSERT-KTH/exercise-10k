## Exercise
Create a function that takes a name and age as keyword arguments and returns a string with the name and age

## Reference Solution
```python
def generate_string(**kwargs):
    """
    Generate a string with name and age.

    Args:
        **kwargs: Keyword arguments. Expected keys are 'name' and 'age'.

    Returns:
        str: A string with the name and age.

    Raises:
        KeyError: If either 'name' or 'age' is not provided.
    """
    required_keys = ['name', 'age']

    # Check if all required keys are present
    if not all(key in kwargs for key in required_keys):
        raise KeyError("Both 'name' and 'age' are required.")

    # Extract name and age from kwargs
    name = kwargs['name']
    age = kwargs['age']

    # Validate age
    if not isinstance(age, int) or age < 0:
        raise ValueError("Age must be a non-negative integer.")

    # Generate the string
    return f"{name} is {age} years old."


# Example usage
if __name__ == "__main__":
    try:
        print(generate_string(name="John", age=30))
    except (KeyError, ValueError) as e:
        print(f"Error: {e}")
```
