## Exercise
Write a function that returns the current time in hours, minutes, and seconds.

## Reference Solution
```python
from datetime import datetime

def get_current_time():
    """
    Returns the current time in hours, minutes, and seconds.
    
    Returns:
        dict: A dictionary containing the current time in hours, minutes, and seconds.
    """
    now = datetime.now()
    current_time = {
        "hours": now.hour,
        "minutes": now.minute,
        "seconds": now.second
    }
    return current_time

def main():
    current_time = get_current_time()
    print(f"Current Time: {current_time['hours']} hours, {current_time['minutes']} minutes, {current_time['seconds']} seconds")

if __name__ == "__main__":
    main()
```
