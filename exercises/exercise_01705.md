## Exercise
Write a program that uses a for loop to print the elements of a given array.

## Reference Solution
```python
def print_array_elements(array):
    """
    This function prints the elements of a given array using a for loop.

    Args:
        array (list): The input array to be printed.
    """
    for element in array:
        print(element)

# Example usage
array = [1, 2, 3, 4, 5]
print("Array elements:")
print_array_elements(array)
```
