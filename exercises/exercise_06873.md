## Exercise
Write a program that takes a user's input and converts it into different data types (integer, float, string). The program should handle potential errors during conversion.

## Reference Solution
```python
def convert_data_types(user_input):
    """
    Converts user's input into different data types (integer, float, string).
    
    Args:
        user_input (str): Input provided by the user.
    
    Returns:
        tuple: A tuple containing the converted integer, float, and string data types.
    """
    try:
        # Attempt to convert user's input to integer
        integer_value = int(user_input)
    except ValueError:
        print("Error: Unable to convert input to integer.")
        integer_value = None
    
    try:
        # Attempt to convert user's input to float
        float_value = float(user_input)
    except ValueError:
        print("Error: Unable to convert input to float.")
        float_value = None
    
    # No need to try-except for string conversion, as it's always possible
    string_value = str(user_input)
    
    return integer_value, float_value, string_value

def main():
    # Get user's input
    user_input = input("Please enter a value: ")
    
    # Convert user's input to different data types
    integer_value, float_value, string_value = convert_data_types(user_input)
    
    # Print the results
    print(f"Integer Value: {integer_value}")
    print(f"Float Value: {float_value}")
    print(f"String Value: {string_value}")

if __name__ == "__main__":
    main()
```
