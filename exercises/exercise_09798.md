## Exercise
Write a program that takes an input and processes it using a lambda function.

## Reference Solution
```python
# Define a lambda function to square a number
square = lambda x: x ** 2

def main():
    # Get user input
    num = float(input("Enter a number: "))

    # Use the lambda function to square the input
    result = square(num)

    # Print the result
    print(f"The square of {num} is {result}")

if __name__ == "__main__":
    main()
```
