## Exercise
Write a program that calculates the maximum and minimum of two given numbers.

## Reference Solution
```python
def calculate_max_min(num1, num2):
    """
    Calculate the maximum and minimum of two numbers.

    Args:
        num1 (int): The first number.
        num2 (int): The second number.

    Returns:
        tuple: A tuple containing the maximum and minimum of the two numbers.
    """
    max_num = max(num1, num2)
    min_num = min(num1, num2)
    
    return max_num, min_num


def main():
    # Get the two numbers from the user
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))

    # Calculate the maximum and minimum
    max_num, min_num = calculate_max_min(num1, num2)

    # Print the results
    print(f"Maximum: {max_num}")
    print(f"Minimum: {min_num}")


if __name__ == "__main__":
    main()
```
