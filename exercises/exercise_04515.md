## Exercise
Create a program that generates all possible anagrams of a user-inputted word.

## Reference Solution
```python
import itertools

def generate_anagrams(word):
    """
    Generates all possible anagrams of a given word.

    Args:
        word (str): The input word.

    Returns:
        list: A list of all possible anagrams.
    """
    # Remove any whitespace and convert to lowercase
    word = word.replace(" ", "").lower()

    # Generate all permutations of the word
    permutations = itertools.permutations(word)

    # Convert the permutations to a list of strings
    anagrams = ["".join(p) for p in permutations]

    # Remove duplicates by converting to a set
    unique_anagrams = set(anagrams)

    return list(unique_anagrams)

def main():
    word = input("Enter a word: ")
    anagrams = generate_anagrams(word)

    print("Anagrams:")
    for anagram in anagrams:
        print(anagram)

if __name__ == "__main__":
    main()
```
