## Exercise
Declare variables for storing different types of data such as integers, floats, and strings. Then, identify and print the data type of each variable.

## Reference Solution
```python
# Declare variables for different data types
integer_variable = 10
float_variable = 3.14
string_variable = "Hello, World!"

# Identify and print the data type of each variable
print("Variable Name\tData Type\tValue")
print(f"integer_variable\t{type(integer_variable)}\t\t{integer_variable}")
print(f"float_variable\t{type(float_variable)}\t\t{float_variable}")
print(f"string_variable\t{type(string_variable)}\t\t{string_variable}")
```
