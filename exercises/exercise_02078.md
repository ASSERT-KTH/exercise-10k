## Exercise
Write a program that generates a random permutation of a given list of numbers.

## Reference Solution
```python
import random

def generate_random_permutation(numbers):
    """
    Generates a random permutation of the given list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        list: A random permutation of the input list.
    """
    # Create a copy of the input list to avoid modifying it in place
    numbers_copy = numbers.copy()
    
    # Use random.shuffle to generate a random permutation
    random.shuffle(numbers_copy)
    
    return numbers_copy

# Example usage
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    print("Original list:", numbers)
    print("Random permutation:", generate_random_permutation(numbers))
```
