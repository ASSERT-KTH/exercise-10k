## Exercise
Given an array of integers, write a program to sort the elements in ascending order.

## Reference Solution
```python
def sort_array(arr):
    """
    Sorts the elements of an array in ascending order.

    Args:
        arr (list): The input array of integers.

    Returns:
        list: The sorted array of integers.
    """
    return sorted(arr)

# Example usage
arr = [64, 34, 25, 12, 22, 11, 90]
print("Original array:", arr)
print("Sorted array:", sort_array(arr))
```

```python
def bubble_sort(arr):
    """
    Sorts the elements of an array in ascending order using Bubble Sort.

    Args:
        arr (list): The input array of integers.

    Returns:
        list: The sorted array of integers.
    """
    n = len(arr)
    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr

# Example usage
arr = [64, 34, 25, 12, 22, 11, 90]
print("Original array:", arr)
print("Sorted array:", bubble_sort(arr))
```
