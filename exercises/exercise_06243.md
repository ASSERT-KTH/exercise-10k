## Exercise
Write a function that takes a dictionary and a list of key-value pairs as input and updates the dictionary with the new values

## Reference Solution
```python
def update_dictionary(input_dict, key_value_pairs):
    """
    This function updates a dictionary with new key-value pairs.

    Args:
        input_dict (dict): The dictionary to be updated.
        key_value_pairs (list): A list of tuples containing the new key-value pairs.

    Returns:
        dict: The updated dictionary.
    """

    # First, we check if the input dictionary is indeed a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input is not a dictionary")

    # Then, we check if the list of key-value pairs is indeed a list
    if not isinstance(key_value_pairs, list):
        raise TypeError("Key-value pairs are not in a list")

    # We iterate over each key-value pair in the list
    for pair in key_value_pairs:
        # We check if the pair is indeed a tuple with two elements
        if not isinstance(pair, tuple) or len(pair) != 2:
            raise TypeError("Each key-value pair must be a tuple with two elements")

        # We extract the key and value from the tuple
        key, value = pair

        # We update the dictionary with the new key-value pair
        input_dict[key] = value

    # Finally, we return the updated dictionary
    return input_dict

# Example usage:
input_dict = {"a": 1, "b": 2}
key_value_pairs = [("a", 3), ("c", 4)]
print(update_dictionary(input_dict, key_value_pairs))  # Output: {'a': 3, 'b': 2, 'c': 4}
```
