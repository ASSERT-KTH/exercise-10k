## Exercise
Create a function that takes an array as input and returns the reversed array.

## Reference Solution
```python
def reverse_array(arr):
    """
    Reverses the input array.

    Args:
    arr (list): The input array to be reversed.

    Returns:
    list: The reversed array.
    """
    return arr[::-1]

# Example usage:
if __name__ == "__main__":
    input_array = [1, 2, 3, 4, 5]
    reversed_array = reverse_array(input_array)
    print("Input Array:", input_array)
    print("Reversed Array:", reversed_array)
```
