## Exercise
Create a program that takes in the student's name and attendance percentage, then prints out the attendance report. If the attendance percentage is above 90, print 'Excellent attendance', otherwise print 'Good attendance' or 'Fair attendance'

## Reference Solution
```python
def attendance_report(name: str, attendance_percentage: float) -> None:
    """
    Prints out an attendance report based on the attendance percentage.

    Args:
        name (str): The student's name.
        attendance_percentage (float): The student's attendance percentage.
    """
    if attendance_percentage > 90:
        attendance_status = 'Excellent attendance'
    elif attendance_percentage >= 75:
        attendance_status = 'Good attendance'
    else:
        attendance_status = 'Fair attendance'

    print(f"Name: {name}")
    print(f"Attendance Percentage: {attendance_percentage}%")
    print(f"Attendance Status: {attendance_status}")


def main() -> None:
    """
    Main function to get user input and print attendance report.
    """
    name = input("Enter student's name: ")
    attendance_percentage = float(input("Enter attendance percentage: "))

    attendance_report(name, attendance_percentage)


if __name__ == "__main__":
    main()
```
