## Exercise
Write a program that takes a string as input and returns the number of digits in the string.

## Reference Solution
```python
def count_digits(input_str):
    """
    Counts the number of digits in a string.

    Args:
        input_str (str): The input string.

    Returns:
        int: The number of digits in the string.
    """
    count = 0
    for char in input_str:
        if char.isdigit():
            count += 1
    return count

# Example usage:
input_str = "Hello123World456"
print(f"Number of digits: {count_digits(input_str)}")
```
