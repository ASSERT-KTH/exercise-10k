## Exercise
Create a program to determine the type of a triangle based on its sides. If all sides are equal, it is an equilateral triangle. If two sides are equal, it is an isosceles triangle. If all sides are different, it is a scalene triangle.

## Reference Solution
```python
def determine_triangle_type(a: float, b: float, c: float) -> str:
    """
    Determine the type of a triangle based on its sides.

    Args:
    a (float): The length of side a.
    b (float): The length of side b.
    c (float): The length of side c.

    Returns:
    str: The type of the triangle (equilateral, isosceles, or scalene).
    """
    # Check if all sides are equal (equilateral)
    if a == b == c:
        return "Equilateral"

    # Check if two sides are equal (isosceles)
    elif a == b or a == c or b == c:
        return "Isosceles"

    # If all sides are different, it's a scalene triangle
    else:
        return "Scalene"


def main():
    # Example usage
    print("Triangle Type Determiner")
    print("-------------------------")

    a = float(input("Enter the length of side a: "))
    b = float(input("Enter the length of side b: "))
    c = float(input("Enter the length of side c: "))

    # Check if the sides can form a valid triangle
    if a + b > c and a + c > b and b + c > a:
        triangle_type = determine_triangle_type(a, b, c)
        print(f"The triangle is {triangle_type}.")
    else:
        print("The sides cannot form a valid triangle.")


if __name__ == "__main__":
    main()
```
