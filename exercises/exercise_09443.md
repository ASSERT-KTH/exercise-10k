## Exercise
Write a program that checks if a given number is even or odd and prints a corresponding message.

## Reference Solution
```python
def check_even_odd(num):
    if num % 2 == 0:
        print(f"{num} is even.")
    else:
        print(f"{num} is odd.")

# Test the function
check_even_odd(10)  # Output: 10 is even.
check_even_odd(11)  # Output: 11 is odd.
```
