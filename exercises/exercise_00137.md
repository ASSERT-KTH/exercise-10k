## Exercise
Write a program that converts a given decimal number to a simplified fraction.

## Reference Solution
```python
from fractions import Fraction

def decimal_to_fraction(decimal):
    """
    Converts a given decimal number to a simplified fraction.

    Args:
        decimal (float): The decimal number to convert.

    Returns:
        str: The simplified fraction as a string.
    """
    fraction = Fraction(decimal).limit_denominator()
    return f"{fraction.numerator}/{fraction.denominator}"

# Example usage:
decimal = 0.5
print(f"The decimal {decimal} is equivalent to the fraction: {decimal_to_fraction(decimal)}")
```
