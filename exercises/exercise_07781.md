## Exercise
Create a program that defines a function to perform basic arithmetic operations. The function should take two numbers and an operator as input and return the result of the operation. The operators to be supported are addition, subtraction, multiplication, and division. Handle invalid operators and division by zero errors.

## Reference Solution
```python
#include <iostream>

int main(int argc, const char * argv[]) {
    // Function to perform arithmetic operations
    double arithmetic(double x, double y, char op) {
        switch (op) {
            case '+':
                return x + y;
            case '-':
                return x - y;
            case '*':
                return x * y;
            case '/':
                if (y == 0) {
                    throw std::runtime_error("Division by zero!");
                }
                return x / y;
            default:
                throw std::invalid_argument("Invalid operator!");
        }
    }

    // Example usage
    double x = 10.0;
    double y = 2.0;
    char op = '+';

    try {
        double result = arithmetic(x, y, op);
        std::cout << "Result: " << result << std::endl;
    } catch (const std::exception& e) {
        std::cerr << "Error: " << e.what() << std::endl;
        return 1;
    }

    return 0;
}
```
