## Exercise
Write a program that takes two strings as input, reverses the order of characters in the first string, and then concatenates the reversed string with the second string.

## Reference Solution
```python
def reverse_and_concat(str1, str2):
    a = str1[::-1]
    b = a + str2
    return b

x = "hello"
y = "world"

print(reverse_and_concat(x, y))
```
