## Exercise
Write a function to update the value of a specific key in a dictionary

## Reference Solution
```python
def update_dict_key(dictionary, key, new_value):
    """
    Updates the value of a specific key in a dictionary.

    Args:
        dictionary (dict): The dictionary to be updated.
        key (any): The key to be updated.
        new_value (any): The new value for the key.

    Returns:
        dict: The updated dictionary.
    """
    dictionary[key] = new_value
    return dictionary

# Example usage:
my_dict = {"name": "John", "age": 30}
print("Original Dictionary:", my_dict)

updated_dict = update_dict_key(my_dict, "age", 31)
print("Updated Dictionary:", updated_dict)

# Add a new key
updated_dict = update_dict_key(updated_dict, "city", "New York")
print("Updated Dictionary with new key:", updated_dict)
```
