## Exercise
Write a program that determines the day of the week based on the date. If the date is between Monday and Friday, it is a weekday. If the date is Saturday or Sunday, it is a weekend.

## Reference Solution
```python
from datetime import datetime

def is_weekend(date_str):
    """
    Determine whether a given date is a weekend or not.

    Args:
        date_str (str): The date as a string in YYYY-MM-DD format.

    Returns:
        bool: True if the date is a weekend, False otherwise.
    """
    date = datetime.strptime(date_str, "%Y-%m-%d")
    return date.weekday() >= 5


def get_day_of_week(date_str):
    """
    Determine the day of the week for a given date.

    Args:
        date_str (str): The date as a string in YYYY-MM-DD format.

    Returns:
        str: The day of the week.
    """
    date = datetime.strptime(date_str, "%Y-%m-%d")
    return date.strftime("%A")


def main():
    date_str = input("Enter a date in YYYY-MM-DD format: ")
    if is_weekend(date_str):
        print(f"{get_day_of_week(date_str)} is a weekend.")
    else:
        print(f"{get_day_of_week(date_str)} is a weekday.")


if __name__ == "__main__":
    main()
```
