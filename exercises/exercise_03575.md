## Exercise
Write a function that inherits properties from a parent dictionary to a child dictionary

## Reference Solution
```python
def inherit_properties(parent, child):
    """
    Creates a new dictionary that inherits properties from a parent dictionary.
    
    Args:
    parent (dict): The parent dictionary.
    child (dict): The child dictionary.
    
    Returns:
    dict: A new dictionary containing all key-value pairs from the parent and child dictionaries.
    """
    # Create a new dictionary to store the inherited properties
    inherited_dict = parent.copy()
    
    # Update the new dictionary with the child dictionary's key-value pairs
    inherited_dict.update(child)
    
    return inherited_dict

# Example usage:
parent_dict = {"name": "John", "age": 30, "city": "New York"}
child_dict = {"age": 31, "country": "USA"}

inherited_dict = inherit_properties(parent_dict, child_dict)

print(inherited_dict)
```
