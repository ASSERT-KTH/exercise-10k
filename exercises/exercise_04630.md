## Exercise
Write a program that takes a nested list as input, and returns the flattened list.

## Reference Solution
```python
def flatten(nested_list):
    """
    This function takes a nested list as input and returns a flattened list.

    Args:
        nested_list (list): A list that can contain nested lists.

    Returns:
        list: A flattened list.
    """
    flat_list = []
    for element in nested_list:
        if isinstance(element, list):
            # If the element is a list, recursively call the flatten function
            flat_list.extend(flatten(element))
        else:
            # If the element is not a list, append it to the flat list
            flat_list.append(element)
    return flat_list

# Example usage
if __name__ == "__main__":
    nested_list = [1, 2, [3, 4, [5, 6]], 7, [8, 9]]
    print("Nested List:", nested_list)
    print("Flattened List:", flatten(nested_list))
```
