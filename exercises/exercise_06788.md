## Exercise
Create a function that takes a string as input and returns the reversed string. However, the function should also validate if the input string is a palindrome. If it is a palindrome, the function should return the string 'Palindrome' instead of reversing the string.

## Reference Solution
```python
def reverse_or_palindrome(input_string):
    """
    This function checks if the input string is a palindrome. If it is, the function returns 'Palindrome'.
    Otherwise, it returns the reversed string.

    Parameters:
    input_string (str): The input string to be checked.

    Returns:
    str: 'Palindrome' if the input string is a palindrome, otherwise the reversed string.
    """

    # First, we need to remove any leading or trailing whitespaces from the input string
    input_string = input_string.strip()

    # Then, we convert the input string to lowercase to make the comparison case-insensitive
    input_string = input_string.lower()

    # Now, we check if the input string is equal to its reverse
    # If they are equal, the string is a palindrome
    if input_string == input_string[::-1]:
        # If the string is a palindrome, we return 'Palindrome'
        return 'Palindrome'
    else:
        # If the string is not a palindrome, we return the reversed string
        return input_string[::-1]
```
