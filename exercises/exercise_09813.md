## Exercise
Declare and initialize variables of different data types, including integer, float, string, and boolean. Write a program that takes the user's name and age as input, declares these as string and integer respectively, and then prints out a greeting message with the user's name and age.

## Reference Solution
```python
# Declare and initialize variables of different data types
integer_var = 10
float_var = 3.14
string_var = "Hello, World!"
boolean_var = True

print("Declared Variables:")
print(f"Integer: {integer_var}")
print(f"Float: {float_var}")
print(f"String: {string_var}")
print(f"Boolean: {boolean_var}")

# Take user's name and age as input
user_name = input("Please enter your name: ")
user_age = int(input("Please enter your age: "))

# Print out a greeting message with the user's name and age
print(f"\nHello, {user_name}! You are {user_age} years old.")

# Declare and initialize variables of different data types using the user's input
user_integer_var = user_age
user_float_var = float(user_age)
user_string_var = str(user_age)
user_boolean_var = user_age > 18

print("\nDeclared Variables using User's Input:")
print(f"Integer: {user_integer_var}")
print(f"Float: {user_float_var}")
print(f"String: {user_string_var}")
print(f"Boolean (is user an adult): {user_boolean_var}")
```
