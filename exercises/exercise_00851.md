## Exercise
Create a function that takes a list of functions as an argument and returns a new function that chains each function together.

## Reference Solution
```python
def chain_functions(functions):
    """
    This function takes a list of functions as an argument and returns a new function 
    that chains each function together. The output of each function is used as the 
    input for the next function in the list.

    Args:
        functions (list): A list of functions where each function takes one argument.

    Returns:
        function: A new function that chains each function together.
    """

    # Define a new function that will be returned by chain_functions
    def chained_function(*args, **kwargs):
        
        # Initialize the result with the input to the chained function
        result = args[0] if args else kwargs.get('input', None)
        
        # Iterate over each function in the list of functions
        for func in functions:
            # Apply the current function to the result and update the result
            result = func(result)
        
        # Return the final result after applying all functions
        return result
    
    # Return the chained function
    return chained_function

# Example usage:
def double(x):
    return x * 2

def add_one(x):
    return x + 1

def square(x):
    return x ** 2

chained = chain_functions([double, add_one, square])
print(chained(5))  # Output: 121
```
