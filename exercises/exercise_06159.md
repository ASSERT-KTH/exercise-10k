## Exercise
Write a function to check if a string ends with a given substring.

## Reference Solution
```python
def ends_with(main_string: str, sub_string: str) -> bool:
    """
    Checks if the main string ends with the sub string.

    Args:
    main_string (str): The string to check.
    sub_string (str): The substring to look for.

    Returns:
    bool: True if the main string ends with the sub string, False otherwise.
    """
    return main_string.endswith(sub_string)

# Example usage:
if __name__ == "__main__":
    main_str = "Hello, World!"
    sub_str = "World!"
    print(ends_with(main_str, sub_str))  # Output: True

    main_str = "Hello, World!"
    sub_str = "Universe"
    print(ends_with(main_str, sub_str))  # Output: False
```
