## Exercise
Create a program that converts decimal numbers to hexadecimal using if/else statements

## Reference Solution
```python
def decimal_to_hexadecimal(decimal):
    """
    Converts a decimal number to hexadecimal.

    Args:
        decimal (int): The decimal number to convert.

    Returns:
        str: The hexadecimal representation of the decimal number.
    """
    hex_map = "0123456789ABCDEF"
    hexadecimal = ""

    while decimal > 0:
        remainder = decimal % 16
        hexadecimal = hex_map[remainder] + hexadecimal
        decimal = decimal // 16

    return hexadecimal

def main():
    decimal = int(input("Enter a decimal number: "))
    hexadecimal = decimal_to_hexadecimal(decimal)
    print(f"The hexadecimal representation of {decimal} is {hexadecimal}")

if __name__ == "__main__":
    main()
```
