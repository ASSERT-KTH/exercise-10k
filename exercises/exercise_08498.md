## Exercise
Write a program that takes an input string, attempts to convert it into an integer, a float, and a boolean, and then prints out the type of the input string along with its converted value if possible. If the conversion fails, print an error message for that specific data type.

## Reference Solution
```python
def main():
    # Input string
    a = input("Enter a string: ")

    # Integer conversion
    try:
        c = int(a)
        print(f"Integer: {c}")
    except ValueError:
        print("Error: cannot be converted to integer")

    # Float conversion
    try:
        d = float(a)
        print(f"Float: {d}")
    except ValueError:
        print("Error: cannot be converted to float")

    # Boolean conversion
    if a == "True" or a == "False":
        e = a == "True"
        print(f"Boolean: {e}")
    else:
        print("Error: cannot be converted to boolean")

if __name__ == "__main__":
    main()
```
