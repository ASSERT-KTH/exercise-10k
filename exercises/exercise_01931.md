## Exercise
Write a program that declares a dictionary variable and prints its key-value pairs.

## Reference Solution
```python
# Declare a dictionary variable
person = {
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}

# Print the key-value pairs
for key, value in person.items():
    print(f"{key}: {value}")
```
