## Exercise
Write a function that takes a list of keys and a value as input and returns a dictionary with the keys and the given value

## Reference Solution
```python
def create_dict(keys, value):
    """
    Create a dictionary with the given keys and value.

    Args:
        keys (list): A list of keys for the dictionary.
        value: The value to be assigned to each key in the dictionary.

    Returns:
        dict: A dictionary with the given keys and value.
    """
    return {key: value for key in keys}

# Example usage:
def main():
    keys = ['a', 'b', 'c']
    value = 10
    result = create_dict(keys, value)
    print(result)

if __name__ == "__main__":
    main()
```
