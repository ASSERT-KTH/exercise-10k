## Exercise
Write a program to remove duplicate elements from a given array.

## Reference Solution
```python
def remove_duplicates(array):
    """
    Removes duplicate elements from a given array.

    Args:
        array (list): The input array.

    Returns:
        list: A new array with duplicate elements removed.
    """
    return list(set(array))

# Example usage:
array = [1, 2, 2, 3, 4, 4, 5, 6, 6]
print("Original array:", array)
print("Array without duplicates:", remove_duplicates(array))
```

def remove_duplicates_ordered(array):
    """
    Removes duplicate elements from a given array, preserving the original order.

    Args:
        array (list): The input array.

    Returns:
        list: A new array with duplicate elements removed.
    """
    seen = set()
    return [x for x in array if not (x in seen or seen.add(x))]

# Example usage:
array = [1, 2, 2, 3, 4, 4, 5, 6, 6]
print("Original array:", array)
print("Array without duplicates:", remove_duplicates_ordered(array))
