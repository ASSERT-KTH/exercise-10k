## Exercise
Write a function that takes a variable number of arguments and returns their sum.

## Reference Solution
```python
def sum_args(*args):
    """
    This function takes a variable number of arguments and returns their sum.

    Args:
        *args: A variable number of arguments

    Returns:
        The sum of all the arguments
    """

    # Initialize the sum to 0. This variable will store the sum of all the arguments.
    total_sum = 0

    # Iterate over each argument in the args list.
    for arg in args:
        # Check if the argument is a number. If it's not, raise a TypeError.
        if not isinstance(arg, (int, float)):
            raise TypeError("All arguments must be numbers")
        
        # Add the current argument to the total sum.
        total_sum += arg
    
    # Return the total sum after iterating over all arguments.
    return total_sum

# Test the function with some examples.
print(sum_args(1, 2, 3, 4, 5))  # Output: 15
print(sum_args(10, 20, 30))  # Output: 60
```
