## Exercise
Read an integer from the user and print out its double.

## Reference Solution
```python
def print_double():
    # Read an integer from the user
    while True:
        try:
            num = int(input("Please enter an integer: "))
            break
        except ValueError:
            print("That's not a valid integer. Please try again.")

    # Calculate and print the double
    double_num = num * 2
    print("The double of", num, "is", double_num)

# Example usage:
if __name__ == "__main__":
    print_double()
```
