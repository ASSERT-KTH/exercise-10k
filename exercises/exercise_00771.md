## Exercise
Write a program that initializes a variable with a default value and then changes it based on user input.

## Reference Solution
```python
# Initialize a variable with a default value
default_value = "Hello, World!"

# Print the default value
print("Default Value:", default_value)

# Ask the user for input
user_input = input("Enter a new value: ")

# Change the variable based on user input
default_value = user_input

# Print the new value
print("New Value:", default_value)
```
