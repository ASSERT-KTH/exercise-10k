## Exercise
Write a function that takes in a name and returns a personalized greeting message.

## Reference Solution
```python
def generate_greeting(name: str) -> str:
    """
    Returns a personalized greeting message.

    Args:
        name (str): The name to be used in the greeting.

    Returns:
        str: A greeting message with the given name.
    """
    return f"Hello, {name}!"

# Example usage:
def main():
    name = input("Please enter your name: ")
    greeting = generate_greeting(name)
    print(greeting)

if __name__ == "__main__":
    main()
```
