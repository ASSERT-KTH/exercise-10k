## Exercise
Write a program that uses nested for loops to print a diamond pattern of asterisks (*).

## Reference Solution
```python
def print_diamond(n):
    # Print the top half of the diamond
    for i in range(n):
        # Print leading spaces
        for j in range(n - i - 1):
            print(" ", end="")
        # Print asterisks
        for k in range(2 * i + 1):
            print("*", end="")
        print()

    # Print the bottom half of the diamond
    for i in range(n-2, -1, -1):
        # Print leading spaces
        for j in range(n - i - 1):
            print(" ", end="")
        # Print asterisks
        for k in range(2 * i + 1):
            print("*", end="")
        print()

# Test the function
print_diamond(5)
```
