## Exercise
Declare a variable and create a deep copy of it using a built-in function or library.

## Reference Solution
```python
import copy

# Declare a variable (a list in this case)
original_list = [[1, 2, 3], [4, 5, 6]]

# Create a deep copy of the variable
copied_list = copy.deepcopy(original_list)

# Modify the original list
original_list.append([7, 8, 9])
original_list[0][0] = 'X'

# Print the original and copied lists
print("Original List:", original_list)
print("Copied List:", copied_list)
```
