## Exercise
Write a function that takes a string as input and returns a list of sentences.

## Reference Solution
```python
import re

def split_into_sentences(text):
    """
    Splits a given text into individual sentences.

    Args:
        text (str): The input text.

    Returns:
        list: A list of sentences.
    """
    # Use regular expression to split the text into sentences
    sentences = re.split(r'[.!?]\s*', text)
    
    # Remove any empty strings from the list
    sentences = [sentence for sentence in sentences if sentence]
    
    return sentences

# Example usage
if __name__ == "__main__":
    text = "Hello world! This is a test. How are you?"
    print(split_into_sentences(text))
```
