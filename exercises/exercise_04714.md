## Exercise
Remove and return an arbitrary key-value pair from a dictionary using the popitem method

## Reference Solution
```python
def remove_and_return_arbitrary_pair(dictionary):
    """
    Removes and returns an arbitrary key-value pair from a dictionary.
    
    Args:
        dictionary (dict): The dictionary from which to remove the pair.
    
    Returns:
        tuple: A tuple containing the key and value of the removed pair.
    """
    if not dictionary:
        return None
    
    # Remove and return an arbitrary key-value pair
    return dictionary.popitem()

# Example usage
example_dict = {"apple": 1, "banana": 2, "cherry": 3}
print("Original dictionary:", example_dict)
removed_pair = remove_and_return_arbitrary_pair(example_dict)
print("Removed pair:", removed_pair)
print("Dictionary after removal:", example_dict)
```
