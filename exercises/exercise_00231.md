## Exercise
Create a function to remove duplicate characters from a given string.

## Reference Solution
```python
def remove_duplicate_chars(input_str):
    """
    Removes duplicate characters from a given string.

    Args:
        input_str (str): The input string.

    Returns:
        str: The input string with duplicate characters removed.
    """
    seen = set()
    result = []

    for char in input_str:
        if char not in seen:
            seen.add(char)
            result.append(char)

    return ''.join(result)

# Example usage:
if __name__ == "__main__":
    input_str = "Hello, World!"
    print("Original String:", input_str)
    print("String without duplicates:", remove_duplicate_chars(input_str))
```
