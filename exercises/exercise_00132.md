## Exercise
Create a program that calculates the area, perimeter, and hypotenuse of a right triangle given its legs.

## Reference Solution
```python
import math

def calculate_right_triangle_properties(a, b):
    """
    Calculate the area, perimeter, and hypotenuse of a right triangle given its legs.

    Args:
        a (float): The length of the first leg.
        b (float): The length of the second leg.

    Returns:
        dict: A dictionary containing the area, perimeter, and hypotenuse of the triangle.
    """
    # Calculate the hypotenuse using the Pythagorean theorem
    c = math.sqrt(a**2 + b**2)

    # Calculate the area of the triangle
    area = 0.5 * a * b

    # Calculate the perimeter of the triangle
    perimeter = a + b + c

    return {
        "area": area,
        "perimeter": perimeter,
        "hypotenuse": c
    }

def main():
    # Get the lengths of the legs from the user
    a = float(input("Enter the length of the first leg: "))
    b = float(input("Enter the length of the second leg: "))

    # Calculate the properties of the triangle
    properties = calculate_right_triangle_properties(a, b)

    # Print the results
    print("Right Triangle Properties:")
    print(f"Area: {properties['area']:.2f}")
    print(f"Perimeter: {properties['perimeter']:.2f}")
    print(f"Hypotenuse: {properties['hypotenuse']:.2f}")

if __name__ == "__main__":
    main()
```
