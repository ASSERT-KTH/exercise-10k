## Exercise
Write a program that takes two lists as input and returns their concatenation.

## Reference Solution
```python
def concatenate(list1, list2):
    out = []
    for i in range(0, len(list1)):
        out.append(list1[i])
    for i in range(0, len(list2)):
        out.append(list2[i])
    return out

print(concatenate([1, 2, 3], [4, 5, 6]))  # Output: [1, 2, 3, 4, 5, 6]
```
