## Exercise
Write a program that takes an array and an index as input, and returns the element at that index. If the index is out of range, return an error message.

## Reference Solution
```python
#include <stdio.h>

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    int index = 2;

    if (index >= 0 && index <= 4) {
        printf("Element at index %d: %d\n", index, arr[index]);
    } else {
        printf("Error: Index out of range.\n");
    }

    return 0;
}
```
