## Exercise
Create a function that takes three numbers as arguments and returns the minimum of the three

## Reference Solution
```python
def min_of_three(a: float, b: float, c: float) -> float:
    """
    Returns the minimum of three numbers.

    Args:
        a (float): The first number.
        b (float): The second number.
        c (float): The third number.

    Returns:
        float: The minimum of the three numbers.
    """
    return min(a, b, c)

# Example usage
if __name__ == "__main__":
    print(min_of_three(10, 20, 30))  # Output: 10
    print(min_of_three(-5, 0, 5))     # Output: -5
    print(min_of_three(100, 200, 50)) # Output: 50
```
