## Exercise
Merge two dictionaries into one

## Reference Solution
```python
dict1 = {'a': 1, 'b': 2}
dict2 = {'c': 3, 'd': 4}

dict1.update(dict2)

print(dict1)  # Output: {'a': 1, 'b': 2, 'c': 3, 'd': 4}
```

```python
dict1 = {'a': 1, 'b': 2}
dict2 = {'c': 3, 'd': 4}

merged_dict = {**dict1, **dict2}

print(merged_dict)  # Output: {'a': 1, 'b': 2, 'c': 3, 'd': 4}
```

```python
dict1 = {'a': 1, 'b': 2}
dict2 = {'c': 3, 'd': 4}

merged_dict = dict1 | dict2

print(merged_dict)  # Output: {'a': 1, 'b': 2, 'c': 3, 'd': 4}
```
