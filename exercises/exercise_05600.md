## Exercise
Write a function that takes three numbers as arguments and returns the maximum value.

## Reference Solution
```python
def find_max(a: float, b: float, c: float) -> float:
    """
    Returns the maximum value of three numbers.

    Args:
        a (float): The first number.
        b (float): The second number.
        c (float): The third number.

    Returns:
        float: The maximum value.
    """
    return max(a, b, c)

# Example usage:
if __name__ == "__main__":
    print(find_max(10, 20, 30))  # Output: 30
    print(find_max(-10, -20, -30))  # Output: -10
    print(find_max(5, 5, 5))  # Output: 5
```
