## Exercise
Create a function that takes a function with multiple arguments and returns a new function that takes one argument at a time.

## Reference Solution
```python
def curry(func):
    """
    Creates a curried version of a function.

    Args:
        func: The function to be curried.

    Returns:
        A new function that takes one argument at a time.
    """
    def inner(*args, **kwargs):
        if len(args) + len(kwargs) >= func.__code__.co_argcount:
            return func(*args, **kwargs)
        return lambda *x, **y: inner(*(args + x), **dict(kwargs, **y))
    return inner

# Example usage:
def add(a, b, c):
    """Adds three numbers."""
    return a + b + c

# Create a curried version of the add function
curried_add = curry(add)

# Use the curried function
result = curried_add(1)(2)(3)
print(result)  # Output: 6
```
