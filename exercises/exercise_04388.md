## Exercise
Use enum data type to represent a fixed set of values

## Reference Solution
```python
from enum import Enum

class DayOfWeek(Enum):
    MONDAY = 1
    TUESDAY = 2
    WEDNESDAY = 3
    THURSDAY = 4
    FRIDAY = 5
    SATURDAY = 6
    SUNDAY = 7
```

```python
def get_day_name(day: DayOfWeek) -> str:
    return day.name

print(get_day_name(DayOfWeek.MONDAY))  # Output: MONDAY
```
