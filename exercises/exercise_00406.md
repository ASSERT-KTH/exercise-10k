## Exercise
Find the second largest element in an array.

## Reference Solution
```python
def find_second_largest(arr):
    """
    Find the second largest element in an array.

    Args:
        arr (list): A list of numbers.

    Returns:
        The second largest element in the array, or None if it does not exist.
    """
    if len(arr) < 2:
        return None

    max_val = float('-inf')
    second_max_val = float('-inf')

    for num in arr:
        if num > max_val:
            second_max_val = max_val
            max_val = num
        elif num < max_val and num > second_max_val:
            second_max_val = num

    if second_max_val == float('-inf'):
        return None
    else:
        return second_max_val

# Example usage
arr = [10, 20, 30, 40, 50]
print(find_second_largest(arr))  # Output: 40
```
