## Exercise
Create a program that demonstrates the modulus operation.

## Reference Solution
```python
def demonstrate_modulus():
    """
    Demonstrates the modulus operation.
    """
    # Define two integers
    dividend = 17
    divisor = 5

    # Calculate the quotient and remainder using the modulus operation
    quotient = dividend // divisor
    remainder = dividend % divisor

    # Print the results
    print(f"Dividend: {dividend}")
    print(f"Divisor: {divisor}")
    print(f"Quotient: {quotient}")
    print(f"Remainder: {remainder}")

    # Example use case: check if a number is even or odd
    number = 23
    if number % 2 == 0:
        print(f"{number} is even.")
    else:
        print(f"{number} is odd.")

if __name__ == "__main__":
    demonstrate_modulus()
```
