## Exercise
Write a program that asks for two strings and checks if the first string is a suffix of the second string.

## Reference Solution
```python
def check_suffix():
    # Get the two strings from the user
    str1 = input("Enter the first string: ")
    str2 = input("Enter the second string: ")

    # Check if str1 is a suffix of str2
    if str2.endswith(str1):
        print(f"'{str1}' is a suffix of '{str2}'")
    else:
        print(f"'{str1}' is not a suffix of '{str2}'")

if __name__ == "__main__":
    check_suffix()
```
