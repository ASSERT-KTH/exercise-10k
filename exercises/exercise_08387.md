## Exercise
Write a program that calculates the nth Pell number using dynamic programming.

## Reference Solution
```python
def pell_number(n):
    # Create a list to store the Pell numbers
    pell = [0] * (n + 1)

    # Initialize the first two Pell numbers
    if n >= 0:
        pell[0] = 0
    if n >= 1:
        pell[1] = 1

    # Calculate the remaining Pell numbers
    for i in range(2, n + 1):
        pell[i] = 2 * pell[i - 1] + pell[i - 2]

    # Return the nth Pell number
    return pell[n]

# Test the function
n = int(input("Enter the value of n: "))
print(f"The {n}th Pell number is: {pell_number(n)}")
```
