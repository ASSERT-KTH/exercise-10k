## Exercise
Find the oldest student in a dictionary where keys are student names and values are their corresponding ages

## Reference Solution
```python
def find_oldest_student(students):
    """
    Find the oldest student in a dictionary.

    Args:
        students (dict): A dictionary where keys are student names and values are their corresponding ages.

    Returns:
        tuple: A tuple containing the name and age of the oldest student.
    """
    if not students:
        return None

    oldest_student = max(students, key=students.get)
    return oldest_student, students[oldest_student]

# Example usage:
students = {
    "John": 20,
    "Alice": 22,
    "Bob": 19,
    "Charlie": 25
}

oldest = find_oldest_student(students)
if oldest:
    print(f"The oldest student is {oldest[0]} with an age of {oldest[1]}")
else:
    print("No students found.")
```
