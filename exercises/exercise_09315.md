## Exercise
Write a function that takes a string as input and returns the string with words reversed and separated by spaces.

## Reference Solution
```python
def reverse_words(input_string):
    """
    Reverses each word in a given string and joins them back together with spaces.

    Args:
        input_string (str): The input string to be processed.

    Returns:
        str: The string with words reversed and separated by spaces.
    """
    # Split the input string into words
    words = input_string.split()

    # Reverse each word
    reversed_words = [word[::-1] for word in words]

    # Join the reversed words back together with spaces
    reversed_string = ' '.join(reversed_words)

    return reversed_string

# Example usage
def main():
    input_string = "Hello World"
    print(reverse_words(input_string))  # Output: "olleH dlroW"

if __name__ == "__main__":
    main()
```
