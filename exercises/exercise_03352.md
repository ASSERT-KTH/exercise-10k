## Exercise
Write a program that uses a for loop to check if a given phone number is valid (10 digits, starts with a digit between 2 and 9).

## Reference Solution
```python
def is_valid_phone_number(phone_number):
    # Check if phone number has 10 digits
    if len(phone_number) != 10:
        return False

    # Check if phone number starts with a digit between 2 and 9
    if not 2 <= int(phone_number[0]) <= 9:
        return False

    # Check if all characters are digits
    for digit in phone_number:
        if not digit.isdigit():
            return False

    return True

phone_number = input("Enter a phone number: ")
if is_valid_phone_number(phone_number):
    print("Phone number is valid")
else:
    print("Phone number is not valid")
```
