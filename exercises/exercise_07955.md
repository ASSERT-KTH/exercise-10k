## Exercise
Write a program to convert a string to its UTF-8 encoded representation.

## Reference Solution
```python
def string_to_utf8(input_string):
    """
    This function converts a string to its UTF-8 encoded representation.

    Args:
        input_string (str): The string to be converted.

    Returns:
        bytes: The UTF-8 encoded representation of the input string.
    """
    # Use the encode() method to convert the string to UTF-8
    utf8_bytes = input_string.encode('utf-8')

    return utf8_bytes


def main():
    # Example usage:
    input_string = "Hello, World!"
    utf8_bytes = string_to_utf8(input_string)

    print(f"Input String: {input_string}")
    print(f"UTF-8 Encoded Representation: {utf8_bytes}")


if __name__ == "__main__":
    main()
```
