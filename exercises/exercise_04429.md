## Exercise
Write a program that generates a sequence of numbers where each number is twice the previous number using a for loop.

## Reference Solution
```python
#include <cstdio>

int main() {
    int num[8];
    int i;
    num[0] = 1;
    for(i = 1; i < 8; i++) {
        num[i] = num[i-1] * 2;
    }
    for(i = 0; i < 8; i++) {
        printf("%d ", num[i]);
    }
    printf("\n");
    return 0;
}
```
