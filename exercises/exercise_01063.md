## Exercise
Write a program that uses a for loop to print a pattern of asterisks (*) where each row has one more asterisk than the previous row.

## Reference Solution
```python
def print_asterisk_pattern(n):
    """
    Prints a pattern of asterisks where each row has one more asterisk than the previous row.

    Args:
        n (int): The number of rows in the pattern.
    """
    for i in range(1, n + 1):
        # Print i asterisks in each row
        print('* ' * i)

# Test the function
print_asterisk_pattern(5)
```
