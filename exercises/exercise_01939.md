## Exercise
Write a program that demonstrates dynamic typing (e.g., changing the data type of a variable).

## Reference Solution
```python
# Dynamic Typing Example in Python

def main():
    # Initialize a variable with an integer value
    my_variable = 10
    print("Initial type and value:", type(my_variable), my_variable)

    # Change the type to float
    my_variable = 10.5
    print("Type changed to float:", type(my_variable), my_variable)

    # Change the type to string
    my_variable = "Hello, World!"
    print("Type changed to string:", type(my_variable), my_variable)

    # Change the type to list
    my_variable = [1, 2, 3, 4, 5]
    print("Type changed to list:", type(my_variable), my_variable)

    # Change the type to dictionary
    my_variable = {"name": "John", "age": 30}
    print("Type changed to dictionary:", type(my_variable), my_variable)

if __name__ == "__main__":
    main()
```
