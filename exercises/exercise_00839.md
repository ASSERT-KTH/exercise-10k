## Exercise
Given a string and a regex pattern, check if the string matches the pattern and return True or False.

## Reference Solution
```python
import re

def match_pattern(string, pattern):
    """
    Checks if a string matches a regex pattern.

    Args:
    string (str): The input string to check.
    pattern (str): The regex pattern to match.

    Returns:
    bool: True if the string matches the pattern, False otherwise.
    """
    try:
        # Use re.fullmatch to ensure the entire string matches the pattern
        return bool(re.fullmatch(pattern, string))
    except re.error:
        # Handle invalid regex patterns
        print("Invalid regex pattern")
        return False

# Example usage
if __name__ == "__main__":
    string = "hello123"
    pattern = "hello\d+"
    result = match_pattern(string, pattern)
    print(f"String '{string}' matches pattern '{pattern}': {result}")
```
