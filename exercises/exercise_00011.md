## Exercise
Write a program to define and call a function with arguments and return values.

## Reference Solution
```python
# Define a function with arguments and return values
def greet(name: str, age: int) -> str:
    """
    Returns a greeting message with the given name and age.
    
    Args:
        name (str): The person's name.
        age (int): The person's age.
    
    Returns:
        str: A greeting message.
    """
    return f"Hello, {name}! You are {age} years old."

# Call the function with arguments
def main():
    name = "John"
    age = 30
    greeting = greet(name, age)
    print(greeting)

if __name__ == "__main__":
    main()
```
