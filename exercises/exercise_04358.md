## Exercise
Write a program that checks the orientation of a rectangle (landscape or portrait) based on its dimensions

## Reference Solution
```python
def check_orientation(width, height):
    """
    Checks the orientation of a rectangle based on its dimensions.

    Args:
        width (int): The width of the rectangle.
        height (int): The height of the rectangle.

    Returns:
        str: The orientation of the rectangle ("landscape" or "portrait").
    """
    if width > height:
        return "landscape"
    elif width < height:
        return "portrait"
    else:
        return "square"

def main():
    # Example usage:
    width = int(input("Enter the width of the rectangle: "))
    height = int(input("Enter the height of the rectangle: "))

    orientation = check_orientation(width, height)
    print(f"The orientation of the rectangle is: {orientation}")

if __name__ == "__main__":
    main()
```
