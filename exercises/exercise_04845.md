## Exercise
Write a program that takes an array as input, and returns its sorted version.

## Reference Solution
```python
def sort_array(array):
    """
    Returns the sorted version of the input array.

    Args:
        array (list): The input array to be sorted.

    Returns:
        list: The sorted version of the input array.
    """
    return sorted(array)

# Example usage:
def main():
    array = [64, 34, 25, 12, 22, 11, 90]
    print("Original array:", array)
    print("Sorted array:", sort_array(array))

if __name__ == "__main__":
    main()
```
