## Exercise
Create a program that calculates the fractional part of a given number.

## Reference Solution
```python
def calculate_fractional_part(number):
    """
    Calculate the fractional part of a given number.

    Args:
        number (float): The input number.

    Returns:
        float: The fractional part of the input number.
    """
    return number - int(number)

def main():
    # Example usage
    number = float(input("Enter a number: "))
    fractional_part = calculate_fractional_part(number)
    print(f"The fractional part of {number} is {fractional_part}")

if __name__ == "__main__":
    main()
```
