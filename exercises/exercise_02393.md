## Exercise
Write a program that calculates the greatest common divisor (GCD) of two given numbers using a while loop.

## Reference Solution
```python
#include <iostream>

using namespace std;

int main() {
    int a, b, r;

    cout << "Enter first number: ";
    cin >> a;

    cout << "Enter second number: ";
    cin >> b;

    while (b != 0) {
        r = a % b;
        a = b;
        b = r;
    }

    cout << "GCD: " << a << endl;

    return 0;
}
```
