Given a dictionary {'name': 'Jane', 'age': 30, 'city': 'London'}, write a program to access and print the value of the 'name' key.

    person = {'name': 'Jane', 'age': 30, 'city': 'London'}

    # Access and print the value of the 'name' key
    name = person.get('name')
    print("The person's name is:", name)
