## Exercise
Write a program that declares a boolean variable and checks its value using an if statement.

## Reference Solution
```python
def main():
    # Declare a boolean variable
    is_true = True

    # Check the value of the boolean variable
    if is_true:
        print("The boolean variable is True.")
    else:
        print("The boolean variable is False.")

if __name__ == "__main__":
    main()
```
