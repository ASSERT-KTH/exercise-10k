## Exercise
Create a program to calculate the result of a exponentiation operation using a while loop.

## Reference Solution
```python
def exponentiation(base, exponent):
    """
    Calculate the result of an exponentiation operation using a while loop.

    Args:
        base (int): The base number.
        exponent (int): The exponent.

    Returns:
        int: The result of the exponentiation operation.
    """
    result = 1
    i = 0
    while i < abs(exponent):
        result *= base
        i += 1
    if exponent < 0:
        return 1 / result
    else:
        return result

def main():
    # Example usage:
    base = int(input("Enter the base: "))
    exponent = int(input("Enter the exponent: "))
    result = exponentiation(base, exponent)
    print(f"The result of {base}^{exponent} is {result}")

if __name__ == "__main__":
    main()
```
