## Exercise
Write a function to create a dictionary using dictionary comprehension

## Reference Solution
```python
def create_dict(keys, values):
    """
    Creates a dictionary using dictionary comprehension.

    Args:
        keys (list): A list of keys for the dictionary.
        values (list): A list of values for the dictionary.

    Returns:
        dict: A dictionary with the given keys and values.
    """
    return {key: value for key, value in zip(keys, values)}

# Example usage:
keys = ['a', 'b', 'c']
values = [1, 2, 3]
result = create_dict(keys, values)
print(result)  # Output: {'a': 1, 'b': 2, 'c': 3}
```

def create_dict_with_condition(keys, values):
    """
    Creates a dictionary using dictionary comprehension with a conditional statement.

    Args:
        keys (list): A list of keys for the dictionary.
        values (list): A list of values for the dictionary.

    Returns:
        dict: A dictionary with the given keys and values, excluding values that are less than 2.
    """
    return {key: value for key, value in zip(keys, values) if value >= 2}

# Example usage:
keys = ['a', 'b', 'c']
values = [1, 2, 3]
result = create_dict_with_condition(keys, values)
print(result)  # Output: {'b': 2, 'c': 3}
