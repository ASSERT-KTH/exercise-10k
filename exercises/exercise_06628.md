## Exercise
Create a function that returns another function with a closed-over variable

## Reference Solution
```python
def create_counter(initial_value=0):
    """
    Returns a counter function with a closed-over variable.

    Args:
        initial_value (int): The initial value of the counter. Defaults to 0.

    Returns:
        function: A counter function that increments and returns the current count.
    """
    count = initial_value

    def counter():
        nonlocal count
        count += 1
        return count

    return counter

# Example usage
if __name__ == "__main__":
    # Create two separate counters
    counter1 = create_counter()
    counter2 = create_counter(10)

    # Use the counters
    print(counter1())  # Output: 1
    print(counter1())  # Output: 2
    print(counter2())  # Output: 11
    print(counter2())  # Output: 12
```
