## Exercise
Create a function that takes a list as input and returns True if it contains duplicate elements, False otherwise.

## Reference Solution
```python
def has_duplicates(input_list):
    """
    Checks if a list contains duplicate elements.

    Args:
        input_list (list): The input list to be checked.

    Returns:
        bool: True if the list contains duplicate elements, False otherwise.
    """

    # First, we check if the input is indeed a list
    if not isinstance(input_list, list):
        raise TypeError("Input must be a list")

    # We convert the list to a set, which automatically removes duplicates
    # If the lengths of the original list and the set are different, then there were duplicates
    # This is because sets in Python cannot contain duplicate values
    set_from_list = set(input_list)
    
    # Now we compare the lengths of the original list and the set
    # If they are not equal, then there were duplicates in the list
    return len(input_list) != len(set_from_list)

# Example usage:
print(has_duplicates([1, 2, 3, 4, 5]))  # False
print(has_duplicates([1, 2, 3, 4, 4]))  # True
```
