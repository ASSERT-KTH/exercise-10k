## Exercise
Create a function that takes a list, a start index, an end index, and a list of values as input and assigns the values to the slice of the list from the start index to the end index.

## Reference Solution
```python
def assign_values_to_slice(input_list, start_index, end_index, values):
    """
    Assigns the given values to the slice of the input list from the start index to the end index.

    Args:
        input_list (list): The list to be modified.
        start_index (int): The start index of the slice.
        end_index (int): The end index of the slice.
        values (list): The list of values to be assigned to the slice.

    Returns:
        list: The modified list.
    """

    # Check if the start index is less than the end index. If not, raise a ValueError.
    if start_index > end_index:
        raise ValueError("Start index cannot be greater than end index")

    # Check if the length of the values list is equal to the length of the slice. If not, raise a ValueError.
    if end_index - start_index != len(values):
        raise ValueError("Length of values list must be equal to the length of the slice")

    # Use list slicing to assign the values to the slice of the input list.
    # The slice of the input list from the start index to the end index is replaced with the values list.
    input_list[start_index:end_index] = values

    # Return the modified list.
    return input_list

# Example usage:
input_list = [1, 2, 3, 4, 5]
start_index = 1
end_index = 3
values = [10, 20]
print(assign_values_to_slice(input_list, start_index, end_index, values))  # Output: [1, 10, 20, 4, 5]
```
