## Exercise
Write a function to chain multiple dictionaries together to form a new dictionary.

## Reference Solution
```python
def chain_dicts(*dicts):
    """
    This function chains multiple dictionaries together to form a new dictionary.
    
    Args:
        *dicts (dict): A variable number of dictionaries to be chained together.
    
    Returns:
        dict: A new dictionary that is the result of chaining all input dictionaries together.
    """
    
    # Initialize an empty dictionary to store the chained result
    chained_dict = {}
    
    # Iterate over each input dictionary
    for d in dicts:
        # Update the chained dictionary with the current dictionary
        # This will overwrite any existing keys with the same name
        chained_dict.update(d)
    
    # Return the chained dictionary
    return chained_dict

# Example usage:
dict1 = {'a': 1, 'b': 2}
dict2 = {'b': 3, 'c': 4}
dict3 = {'c': 5, 'd': 6}

result = chain_dicts(dict1, dict2, dict3)
print(result)  # Output: {'a': 1, 'b': 3, 'c': 5, 'd': 6}
```
