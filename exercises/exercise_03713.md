## Exercise
Create a function that takes in a list and a number, and returns all combinations of the list of the specified length.

## Reference Solution
```python
import itertools

def get_combinations(input_list, length):
    """
    Returns all combinations of the input list of the specified length.

    Args:
        input_list (list): The list to generate combinations from.
        length (int): The length of the combinations to generate.

    Returns:
        list: A list of tuples, where each tuple is a combination of the input list.
    """
    return list(itertools.combinations(input_list, length))

# Example usage
def main():
    input_list = [1, 2, 3, 4]
    length = 2
    combinations = get_combinations(input_list, length)
    print(f"Combinations of length {length}: {combinations}")

if __name__ == "__main__":
    main()
```
