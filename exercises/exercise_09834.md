## Exercise
Write a program that takes in a user's input and determines its data type. The program should then print out the data type of the input. For example, if the input is 'hello', the program should print 'string'. If the input is '123', the program should print 'integer'.

## Reference Solution
```python
def check_data_type(user_input):
    """
    Checks the data type of the user's input.

    Args:
        user_input (str): The user's input.

    Returns:
        str: The data type of the input.
    """
    try:
        # Try to convert the input to an integer
        int(user_input)
        return 'integer'
    except ValueError:
        try:
            # If not an integer, try to convert it to a float
            float(user_input)
            return 'float'
        except ValueError:
            # If not a float, it's a string
            return 'string'

def main():
    user_input = input("Please enter a value: ")
    data_type = check_data_type(user_input)
    print(f"The data type of '{user_input}' is: {data_type}")

if __name__ == "__main__":
    main()
```
