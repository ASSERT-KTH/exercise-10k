## Exercise
Create a program that uses a for loop to calculate the first n Fibonacci numbers.

## Reference Solution
```python
def calculate_fibonacci(n):
    """
    Calculate the first n Fibonacci numbers.

    Args:
        n (int): The number of Fibonacci numbers to calculate.

    Returns:
        list: A list of the first n Fibonacci numbers.
    """
    if n <= 0:
        return []
    elif n == 1:
        return [0]
    elif n == 2:
        return [0, 1]

    fib_sequence = [0, 1]
    for i in range(2, n):
        fib_sequence.append(fib_sequence[i-1] + fib_sequence[i-2])

    return fib_sequence

def main():
    n = int(input("Enter the number of Fibonacci numbers to calculate: "))
    fib_numbers = calculate_fibonacci(n)
    print("The first {} Fibonacci numbers are:".format(n))
    print(fib_numbers)

if __name__ == "__main__":
    main()
```
