## Exercise
Write a program to format a given time in a specific format (e.g. HH:MM:SS AM/PM).

## Reference Solution
```python
from datetime import datetime

def format_time(time_str):
    """
    Formats a given time in the format 'HH:MM:SS AM/PM'.

    Args:
        time_str (str): Time in the format 'HH:MM:SS' or 'HH:MM'.

    Returns:
        str: Time in the format 'HH:MM:SS AM/PM'.
    """
    try:
        # Check if time_str has seconds
        if len(time_str.split(':')) == 3:
            dt = datetime.strptime(time_str, '%H:%M:%S')
        else:
            dt = datetime.strptime(time_str, '%H:%M')

        # Format time in 'HH:MM:SS AM/PM' format
        formatted_time = dt.strftime('%I:%M:%S %p')

        return formatted_time
    except ValueError:
        print("Invalid time format. Please use 'HH:MM:SS' or 'HH:MM'.")

# Example usage:
time_str = "14:30:00"
print(format_time(time_str))  # Output: 02:30:00 PM

time_str = "14:30"
print(format_time(time_str))  # Output: 02:30:00 PM
```
