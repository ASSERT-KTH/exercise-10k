## Exercise
Write a program that converts a string to an integer and vice versa, and prints out the converted values.

## Reference Solution
```python
#include <iostream>
#include <string>

using namespace std;

int main() {
    string s = "12345";
    int num = 67890;

    // Convert string to integer
    int convertToInt = stoi(s);

    // Convert integer to string
    string convertToString = to_string(num);

    // Print converted values
    cout << "Converted string to integer: " << convertToInt << endl;
    cout << "Converted integer to string: " << convertToString << endl;

    return 0;
}
```
