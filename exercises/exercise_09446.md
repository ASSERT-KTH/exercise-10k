## Exercise
Create a program that checks if a given number is positive, negative, or zero and prints a corresponding message.

## Reference Solution
```python
#include <stdio.h>

int main() {
    int n;

    printf("Enter a number: ");
    scanf("%d", &n);

    if (n > 0) {
        printf("You entered a positive number.\n");
    } else if (n < 0) {
        printf("You entered a negative number.\n");
    } else {
        printf("You entered zero.\n");
    }

    return 0;
}
```
