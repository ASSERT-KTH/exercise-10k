## Exercise
Write a program to check if a given string contains any duplicate pairs of characters

## Reference Solution
```python
def has_duplicate_pairs(s):
    """Check if a string has any duplicate pairs of characters."""
    seen = set()
    for i in range(len(s) - 1):
        pair = s[i:i+2]
        if pair in seen:
            return True
        seen.add(pair)
    return False

# Example usage
if __name__ == "__main__":
    strings_to_check = ["hello", "world", "abab", "abcdefg"]
    for s in strings_to_check:
        print(f"'{s}': {has_duplicate_pairs(s)}")
```
