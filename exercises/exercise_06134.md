## Exercise
Ask the user for a number and print out the sequence of numbers from 1 to the input number.

## Reference Solution
```python
def print_sequence():
    """
    Asks the user for a number and prints out the sequence of numbers from 1 to the input number.
    """
    # Ask the user for a number
    while True:
        try:
            num = int(input("Please enter a positive integer: "))
            if num <= 0:
                print("Please enter a positive integer.")
            else:
                break
        except ValueError:
            print("That's not a valid number!")

    # Print the sequence of numbers
    for i in range(1, num + 1):
        print(i)

if __name__ == "__main__":
    print_sequence()
```
