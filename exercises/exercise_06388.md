## Exercise
Develop a program that takes in a temperature in Celsius and converts it to Fahrenheit. If the temperature is above 100, print a warning message

## Reference Solution
```python
def celsius_to_fahrenheit(celsius):
    """
    Convert temperature from Celsius to Fahrenheit.

    Args:
        celsius (float): Temperature in Celsius.

    Returns:
        float: Temperature in Fahrenheit.
    """
    fahrenheit = (celsius * 9/5) + 32
    return fahrenheit

def main():
    # Get temperature in Celsius from user
    celsius = float(input("Enter temperature in Celsius: "))

    # Convert temperature to Fahrenheit
    fahrenheit = celsius_to_fahrenheit(celsius)

    # Print temperature in Fahrenheit
    print(f"{celsius}°C is equal to {fahrenheit}°F")

    # Check if temperature is above 100°C and print warning message
    if celsius > 100:
        print("Warning: Temperature is above 100°C!")

if __name__ == "__main__":
    main()
```
