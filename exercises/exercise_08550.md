## Exercise
Create a dictionary with 5 key-value pairs and sort the key-value pairs based on the keys.

## Reference Solution
```python
# Create a dictionary with 5 key-value pairs
my_dict = {
    "orange": 5,
    "apple": 10,
    "banana": 7,
    "grape": 3,
    "mango": 8
}

# Sort the key-value pairs based on the keys
sorted_dict = dict(sorted(my_dict.items()))

# Print the sorted dictionary
print("Sorted Dictionary:")
for key, value in sorted_dict.items():
    print(f"{key}: {value}")
```
