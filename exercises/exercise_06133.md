## Exercise
Create a program that responds to basic user inputs like 'hello' and 'goodbye'.

## Reference Solution
```python
# Define a dictionary to map user inputs to responses
responses = {
    'hello': 'Hello! How can I assist you today?',
    'hi': 'Hello! How can I assist you today?',
    'goodbye': 'Goodbye! It was nice chatting with you.',
    'bye': 'Goodbye! It was nice chatting with you.',
    'help': 'I can respond to basic greetings like "hello" and "goodbye".',
    'default': 'I did not understand your input. Please try again.'
}

def get_response(user_input):
    """Return a response based on the user's input."""
    user_input = user_input.lower()  # Convert input to lowercase
    return responses.get(user_input, responses['default'])

def main():
    print("Welcome to the basic user input response program!")
    while True:
        user_input = input("You: ")
        if user_input.lower() == 'quit':
            break
        print("Program: ", get_response(user_input))

if __name__ == "__main__":
    main()
```
