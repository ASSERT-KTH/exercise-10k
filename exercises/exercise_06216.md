## Exercise
Write a function that takes an array of integers as input, and returns a new array where each element is the absolute value of the corresponding element in the input array.

## Reference Solution
```python
def absolute_array(arr):
    """
    This function takes an array of integers as input, 
    and returns a new array where each element is the absolute value of the corresponding element in the input array.

    Parameters:
    arr (list): A list of integers.

    Returns:
    list: A new list with absolute values of the input list.
    """

    # First, we check if the input is a list. If not, we raise a TypeError.
    if not isinstance(arr, list):
        raise TypeError("Input must be a list")

    # Then, we initialize an empty list to store the absolute values.
    # This list will be returned at the end of the function.
    result = []

    # We iterate over each element in the input list.
    for num in arr:
        # For each element, we calculate its absolute value using the built-in abs() function.
        absolute_num = abs(num)
        
        # We append the absolute value to our result list.
        result.append(absolute_num)

    # Finally, we return the result list.
    return result

# Example usage:
print(absolute_array([-1, 2, -3, 4, -5]))  # Output: [1, 2, 3, 4, 5]
```
