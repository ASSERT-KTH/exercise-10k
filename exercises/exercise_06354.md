## Exercise
Explain the concept of a dictionary data type and provide examples of declaring and using dictionaries.

## Reference Solution
```python
# Using the dict constructor
my_dict = dict(name="John", age=30)
print(my_dict)  # Output: {'name': 'John', 'age': 30}

# Using dictionary literal syntax
my_dict = {"name": "John", "age": 30}
print(my_dict)  # Output: {'name': 'John', 'age': 30}
```

```python
my_dict = {"name": "John", "age": 30}
print(my_dict["name"])  # Output: John
print(my_dict.get("age"))  # Output: 30
```

```python
my_dict = {"name": "John", "age": 30}
my_dict["age"] = 31
print(my_dict)  # Output: {'name': 'John', 'age': 31}
```

```python
my_dict = {"name": "John", "age": 30}
my_dict["city"] = "New York"
print(my_dict)  # Output: {'name': 'John', 'age': 30, 'city': 'New York'}
```

```python
my_dict = {"name": "John", "age": 30, "city": "New York"}
del my_dict["age"]
print(my_dict)  # Output: {'name': 'John', 'city': 'New York'}
```

```python
my_dict = {"name": "John", "age": 30, "city": "New York"}
for key, value in my_dict.items():
    print(f"{key}: {value}")
# Output:
# name: John
# age: 30
# city: New York
```

```python
person = {
    "name": "John Doe",
    "age": 30,
    "address": {
        "street": "123 Main St",
        "city": "New York",
        "state": "NY",
        "zip": "10001"
    },
    "interests": ["reading", "hiking", "coding"]
}

print(person["name"])  # Output: John Doe
print(person["address"]["city"])  # Output: New York
print(person["interests"])  # Output: ['reading', 'hiking', 'coding']
```
