## Exercise
Write a function to validate if a given value is of type integer, then test it with the value 123 and '123'.

## Reference Solution
```python
def is_integer(value):
    """
    Validate if a given value is of type integer.

    Args:
        value: The value to be checked.

    Returns:
        bool: True if the value is an integer, False otherwise.
    """
    return isinstance(value, int)

# Test the function
print(is_integer(123))  # Expected output: True
print(is_integer('123'))  # Expected output: False
```
