## Exercise
Create a program that reads a line of input from the user and prints out the number of words it contains.

## Reference Solution
```python
def count_words(input_string):
    """
    Counts the number of words in a given string.

    Args:
        input_string (str): The input string.

    Returns:
        int: The number of words in the input string.
    """
    words = input_string.split()
    return len(words)

def main():
    # Read a line of input from the user
    user_input = input("Please enter a line of text: ")

    # Count the number of words
    word_count = count_words(user_input)

    # Print the result
    print(f"The input contains {word_count} words.")

if __name__ == "__main__":
    main()
```
