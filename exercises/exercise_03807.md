## Exercise
Write a program to calculate the first n triangle numbers using a for loop.

## Reference Solution
```python
def calculate_triangle_numbers(n):
    """
    Calculate the first n triangle numbers.

    Args:
        n (int): The number of triangle numbers to calculate.

    Returns:
        list: A list of the first n triangle numbers.
    """
    triangle_numbers = []
    for i in range(1, n + 1):
        # Calculate the ith triangle number using the formula
        triangle_number = i * (i + 1) // 2
        triangle_numbers.append(triangle_number)
    return triangle_numbers

def main():
    # Example usage
    n = int(input("Enter the number of triangle numbers to calculate: "))
    triangle_numbers = calculate_triangle_numbers(n)
    print("The first {} triangle numbers are:".format(n))
    for i, num in enumerate(triangle_numbers, start=1):
        print("T{} = {}".format(i, num))

if __name__ == "__main__":
    main()
```
