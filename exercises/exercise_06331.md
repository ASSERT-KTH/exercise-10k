## Exercise
Write a program that takes an array and a function as input, applies the function to each element of the array, and returns a new array with the results.

## Reference Solution
```python
// Define the function applyToArray
function applyToArray(array, func) {
  const result = [];
  for (let i = 0; i < array.length; i++) {
    result.push(func(array[i]));
  }
  return result;
}

// Define a function that doubles a number
function double(num) {
  return num * 2;
}

// Call the applyToArray function
const numbers = [1, 2, 3, 4, 5];
const doubledNumbers = applyToArray(numbers, double);

// Log the result
console.log(doubledNumbers); // Output: [2, 4, 6, 8, 10]
```
