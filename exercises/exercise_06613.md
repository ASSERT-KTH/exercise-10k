## Exercise
Create a program that takes two strings as input from the user and prints the longer string.

## Reference Solution
```python
def print_longer_string():
    # Take two strings as input from the user
    str1 = input("Enter the first string: ")
    str2 = input("Enter the second string: ")

    # Compare the lengths of the two strings
    if len(str1) > len(str2):
        # If str1 is longer, print it
        print("The longer string is:", str1)
    elif len(str2) > len(str1):
        # If str2 is longer, print it
        print("The longer string is:", str2)
    else:
        # If both strings are of equal length, print a message
        print("Both strings are of equal length.")

# Call the function
print_longer_string()
```
