## Exercise
Write a program that takes a list of integers as input, and returns a single value representing the reduction of the list according to a specified operation (e.g. sum, product, etc.).

## Reference Solution
```python
from functools import reduce

def main():
    # Read input from user
    lista = [int(x) for x in input().split()]

    # Define operations
    sum_list = lambda x, y: x + y
    prod_list = lambda x, y: x * y
    max_list = lambda x, y: max(x, y)
    min_list = lambda x, y: min(x, y)

    # Apply operations using reduce function
    sum_result = reduce(sum_list, lista)
    prod_result = reduce(prod_list, lista)
    max_result = reduce(max_list, lista)
    min_result = reduce(min_list, lista)

    # Print results
    print("Sum:", sum_result)
    print("Product:", prod_result)
    print("Max:", max_result)
    print("Min:", min_result)

if __name__ == "__main__":
    main()
```
