## Exercise
Write a program that calculates the average grade of a student from a list of grades using a for loop.

## Reference Solution
```python
def calculate_average_grade(grades):
    """
    Calculate the average grade from a list of grades.

    Args:
        grades (list): A list of grades.

    Returns:
        float: The average grade.
    """
    total = 0
    for grade in grades:
        total += grade
    average = total / len(grades)
    return average

def main():
    # Example usage
    grades = [85, 90, 78, 92, 88]
    average_grade = calculate_average_grade(grades)
    print(f"Average Grade: {average_grade:.2f}")

if __name__ == "__main__":
    main()
```
