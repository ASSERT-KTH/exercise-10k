## Exercise
Write a program that reverses a given integer.

## Reference Solution
```python
def reverse_integer(n):
    """
    Reverses a given integer.

    Args:
        n (int): The integer to be reversed.

    Returns:
        int: The reversed integer.
    """
    sign = -1 if n < 0 else 1
    n *= sign

    # Convert integer to string, reverse, and convert back to integer
    reversed_n = int(str(n)[::-1])

    return sign * reversed_n

# Example usage
if __name__ == "__main__":
    num = int(input("Enter an integer: "))
    reversed_num = reverse_integer(num)
    print("Reversed integer:", reversed_num)
```
