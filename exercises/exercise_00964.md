## Exercise
Write a function to remove a key from a dictionary. If the key does not exist, return 'Key not found'.

## Reference Solution
```python
def remove_key(dictionary, key):
    """
    Removes a key from a dictionary.

    Args:
        dictionary (dict): The dictionary to modify.
        key: The key to remove.

    Returns:
        dict or str: The modified dictionary if the key exists, 'Key not found' otherwise.
    """
    if key in dictionary:
        del dictionary[key]
        return dictionary
    else:
        return 'Key not found'

# Example usage:
my_dict = {'a': 1, 'b': 2, 'c': 3}
print(remove_key(my_dict, 'b'))  # Output: {'a': 1, 'c': 3}
print(remove_key(my_dict, 'd'))  # Output: 'Key not found'
```
