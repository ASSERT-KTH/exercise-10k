## Exercise
Write a program that takes a number as input and outputs its floor and ceiling values.

## Reference Solution
```python
import math

def calculate_floor_and_ceiling(num):
    """
    Calculate the floor and ceiling values of a given number.

    Args:
    num (float): The input number.

    Returns:
    tuple: A tuple containing the floor and ceiling values of the input number.
    """
    floor_value = math.floor(num)
    ceiling_value = math.ceil(num)
    return floor_value, ceiling_value

def main():
    # Get the input number from the user
    num = float(input("Enter a number: "))

    # Calculate the floor and ceiling values
    floor_value, ceiling_value = calculate_floor_and_ceiling(num)

    # Print the results
    print(f"Floor value: {floor_value}")
    print(f"Ceiling value: {ceiling_value}")

if __name__ == "__main__":
    main()
```
