## Exercise
Write a program that takes two strings as input and returns the string with the two strings concatenated and the first character of the second string capitalized.

## Reference Solution
```python
def concatenar_con_mayuscula(s1, s2):
    s2_mayuscula = s2.capitalize()
    resultado = s1 + s2_mayuscula
    return resultado

# Example usage:
print(concatenar_con_mayuscula("Hola", "mundo"))  # Output: "Holamundo"
```
