## Exercise
Create a function named 'celsius_to_fahrenheit' that takes a temperature in Celsius as input and returns the equivalent temperature in Fahrenheit.

## Reference Solution
```python
def celsius_to_fahrenheit(celsius):
    """
    This function converts a temperature in Celsius to Fahrenheit.

    Args:
        celsius (float): The temperature in Celsius.

    Returns:
        float: The equivalent temperature in Fahrenheit.
    """

    # First, we need to check if the input is a number. If it's not, we raise a TypeError.
    if not isinstance(celsius, (int, float)):
        raise TypeError("Input must be a number.")

    # Now, we apply the formula to convert Celsius to Fahrenheit: (°C × 9/5) + 32 = °F.
    # We multiply the Celsius temperature by 9/5.
    fahrenheit = celsius * 9 / 5
    
    # Then, we add 32 to the result to get the Fahrenheit temperature.
    fahrenheit += 32
    
    # Finally, we return the calculated Fahrenheit temperature.
    return fahrenheit

# Example usage:
print(celsius_to_fahrenheit(30))  # Output: 86.0
```
