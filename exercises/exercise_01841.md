## Exercise
Write a program that takes an array of integers and an index as input, and returns the value at that index in the array.

## Reference Solution
```python
def get_value(array, index):
    if index < len(array):
        return array[index]
    else:
        return 0

# Example usage:
array = []
for i in range(100):
    array.append(i)

print(get_value(array, 0))  # Output: 0
print(get_value(array, 50))  # Output: 50
print(get_value(array, 100))  # Output: 0
```
