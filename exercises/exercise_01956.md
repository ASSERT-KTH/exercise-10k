## Exercise
Write a program that takes a list as input and returns the mode value of the list.

## Reference Solution
```python
from collections import Counter

def calculate_mode(input_list):
    """
    Calculate the mode value of a list.

    Args:
        input_list (list): The input list.

    Returns:
        The mode value of the list.
    """
    # Count the frequency of each value in the list
    frequency_count = Counter(input_list)

    # Find the value with the maximum frequency
    mode_value = max(frequency_count, key=frequency_count.get)

    return mode_value

# Example usage
if __name__ == "__main__":
    input_list = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]
    mode_value = calculate_mode(input_list)
    print("Mode value:", mode_value)
```
