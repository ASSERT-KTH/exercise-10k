## Exercise
Create a function that checks if a given element exists in a list of integers. The function should take two parameters: a list of integers and the target element to be searched. It should return True if the target element exists in the list, False otherwise.

## Reference Solution
```python
def check_element(lst, target):
    """
    Checks if a given element exists in a list of integers.

    Parameters:
    lst (list): A list of integers.
    target (int): The target element to be searched.

    Returns:
    bool: True if the target element exists in the list, False otherwise.
    """

    # First, we need to check if the input is a list and the target is an integer
    if not isinstance(lst, list) or not all(isinstance(i, int) for i in lst):
        raise TypeError("The first argument must be a list of integers.")
    if not isinstance(target, int):
        raise TypeError("The second argument must be an integer.")

    # We can use the 'in' operator in Python, which checks if a value exists in a sequence or not
    # This operator returns True if the value is found, and False otherwise
    return target in lst

# Example usage:
numbers = [1, 2, 3, 4, 5]
print(check_element(numbers, 3))  # Output: True
print(check_element(numbers, 6))  # Output: False
```
