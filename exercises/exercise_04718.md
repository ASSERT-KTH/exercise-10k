## Exercise
Create a dictionary from a list of keys and a value

## Reference Solution
```python
keys = ['a', 'b', 'c']
value = 'default'

# Create a dictionary with the given keys and value
dictionary = dict.fromkeys(keys, value)

print(dictionary)
```
