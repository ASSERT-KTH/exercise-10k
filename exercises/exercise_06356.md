## Exercise
Explain the concept of type checking and provide examples of using type checking functions to determine the data type of a variable.

## Reference Solution
```python
# Define variables with different data types
var_int = 10
var_float = 3.14
var_str = "Hello, World!"
var_list = [1, 2, 3]
var_dict = {"name": "John", "age": 30}

# Use type() to get the data type of each variable
print("Type of var_int:", type(var_int))
print("Type of var_float:", type(var_float))
print("Type of var_str:", type(var_str))
print("Type of var_list:", type(var_list))
print("Type of var_dict:", type(var_dict))

# Use isinstance() to check if a variable is an instance of a specific type
print("Is var_int an integer?", isinstance(var_int, int))
print("Is var_float a float?", isinstance(var_float, float))
print("Is var_str a string?", isinstance(var_str, str))
print("Is var_list a list?", isinstance(var_list, list))
print("Is var_dict a dictionary?", isinstance(var_dict, dict))
```
