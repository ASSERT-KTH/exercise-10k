## Exercise
Create a function that uses memoization to cache its results

## Reference Solution
```python
def memoize(func):
    """
    A decorator to memoize the results of a function.
    """
    cache = dict()

    def memoized_func(*args):
        if args in cache:
            return cache[args]
        result = func(*args)
        cache[args] = result
        return result

    return memoized_func


@memoize
def fibonacci(n):
    """
    Calculate the nth Fibonacci number.

    Args:
    n (int): The position of the Fibonacci number to calculate.

    Returns:
    int: The nth Fibonacci number.
    """
    if n < 2:
        return n
    return fibonacci(n-1) + fibonacci(n-2)


# Example usage:
if __name__ == "__main__":
    for i in range(10):
        print(f"Fibonacci number {i} is: {fibonacci(i)}")
```
