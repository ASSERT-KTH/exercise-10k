## Exercise
Write a program that checks the data type of a given variable and prints it out.

## Reference Solution
```python
def check_data_type(var):
    """
    Checks the data type of a given variable and prints it out.
    
    Args:
        var: The variable to check the data type of.
    """
    data_type = type(var)
    print(f"The data type of the variable is: {data_type.__name__}")

# Example usage:
if __name__ == "__main__":
    # Test with different data types
    check_data_type(10)  # int
    check_data_type(3.14)  # float
    check_data_type("Hello")  # str
    check_data_type(True)  # bool
    check_data_type([1, 2, 3])  # list
    check_data_type((1, 2, 3))  # tuple
    check_data_type({"a": 1, "b": 2})  # dict
```
