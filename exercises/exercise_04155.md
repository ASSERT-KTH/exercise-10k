## Exercise
Write a function to use the update() method to update a dictionary with key-value pairs from another dictionary.

## Reference Solution
```python
def update_dictionary(dict1, dict2):
    """
    Updates a dictionary with key-value pairs from another dictionary.

    Args:
        dict1 (dict): The dictionary to be updated.
        dict2 (dict): The dictionary with key-value pairs to update from.

    Returns:
        dict: The updated dictionary.
    """
    # Create a copy of the original dictionary to avoid modifying it directly
    updated_dict = dict1.copy()
    
    # Use the update() method to update the dictionary
    updated_dict.update(dict2)
    
    return updated_dict

# Example usage:
dict1 = {"a": 1, "b": 2}
dict2 = {"b": 3, "c": 4}

updated_dict = update_dictionary(dict1, dict2)
print(updated_dict)  # Output: {'a': 1, 'b': 3, 'c': 4}
```
