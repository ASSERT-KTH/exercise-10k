## Exercise
Given a dictionary {'name': 'Bob', 'age': 25, 'city': 'Paris'}, write a program to update the value of the 'age' key to 26.

## Reference Solution
```python
def update_age(dictionary):
    """Updates the 'age' value in the dictionary to 26."""
    dictionary['age'] = 26
    return dictionary

def main():
    # Create a dictionary
    person = {'name': 'Bob', 'age': 25, 'city': 'Paris'}
    
    # Print the original dictionary
    print("Original Dictionary:")
    print(person)
    
    # Update the 'age' value
    updated_person = update_age(person)
    
    # Print the updated dictionary
    print("\nUpdated Dictionary:")
    print(updated_person)

if __name__ == "__main__":
    main()
```
