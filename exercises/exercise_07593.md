## Exercise
Write a program that checks if a given variable is non-numeric.

## Reference Solution
```python
def is_non_numeric(var):
    """
    Checks if a given variable is non-numeric.

    Args:
        var: The variable to check.

    Returns:
        bool: True if the variable is non-numeric, False otherwise.
    """
    try:
        float(var)
        return False
    except ValueError:
        return True

# Example usage:
def main():
    variables = ["hello", 123, 3.14, "123", None, True]

    for var in variables:
        print(f"Is '{var}' non-numeric? {is_non_numeric(var)}")

if __name__ == "__main__":
    main()
```
