## Exercise
Declare an enum data type, assign a value to a variable of that type, and then print its value.

## Reference Solution
```python
# Define an enum data type
from enum import Enum

class Color(Enum):
    RED = 1
    GREEN = 2
    BLUE = 3

# Assign a value to a variable of the enum type
color = Color.GREEN

# Print the value of the enum variable
print(color)  # Output: Color.GREEN
print(color.name)  # Output: GREEN
print(color.value)  # Output: 2
```
