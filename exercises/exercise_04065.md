## Exercise
Write a program that takes a string and a separator as input and returns a list of substrings split by the separator.

## Reference Solution
```python
def split_string_by_separator(str, separator):
    list = []
    while separator in str:
        index = str.find(separator)
        list.append(str[:index])
        str = str[index + 1:]
    list.append(str)
    return list

string = "This is a test string"
separator = " "
print(split_string_by_separator(string, separator))
```
