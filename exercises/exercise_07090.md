## Exercise
Read a sentence from the user and find the shortest word

## Reference Solution
```python
def find_shortest_word(sentence):
    """
    Find the shortest word in a sentence.

    Args:
        sentence (str): The input sentence.

    Returns:
        str: The shortest word in the sentence.
    """
    # Remove punctuation and convert to lowercase
    sentence = ''.join(e for e in sentence if e.isalnum() or e.isspace()).lower()
    
    # Split the sentence into words
    words = sentence.split()
    
    # Find the shortest word
    shortest_word = min(words, key=len)
    
    return shortest_word

# Read a sentence from the user
sentence = input("Enter a sentence: ")

# Find and print the shortest word
shortest_word = find_shortest_word(sentence)
print("The shortest word is:", shortest_word)
```
