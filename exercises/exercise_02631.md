## Exercise
Write a program that takes a player's choice as input and generates a random computer choice for a game of Rock, Paper, Scissors.

## Reference Solution
```python
import random

player = input("Enter a choice (rock, paper, scissors): ")
possible_choices = ["rock", "paper", "scissors"]
computer = random.choice(possible_choices)

print(f"\nYou chose {player}, computer chose {computer}.\n")
```
