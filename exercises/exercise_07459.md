## Exercise
Write a program that takes a user's input and converts it into different data types. The program should be able to convert the input into an integer, a float, and a string. If the input cannot be converted into a certain data type, the program should print an error message.

## Reference Solution
```python
# Importing Modules and Defining Functions
import sys

def convert_to_int(s):
    try:
        return int(s)
    except ValueError:
        print("Cannot convert input to an integer.")
        return None

def convert_to_float(s):
    try:
        return float(s)
    except ValueError:
        print("Cannot convert input to a float.")
        return None

def convert_to_str(s):
    return str(s)

# Main Program
user_input = input("Enter a value: ")

# Converting Input to Different Data Types
int_val = convert_to_int(user_input)
float_val = convert_to_float(user_input)
str_val = convert_to_str(user_input)

# Printing Results
if int_val is not None:
    print(f"Input as an integer: {int_val}")
if float_val is not None:
    print(f"Input as a float: {float_val}")
print(f"Input as a string: {str_val}")

# Exit
sys.exit()
```
