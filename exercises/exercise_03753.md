## Exercise
Write a program to check whether a given list contains duplicate elements or not using a basic algorithm

## Reference Solution
```python
def contains_duplicates(lst):
    """
    Checks if a list contains duplicate elements.

    Args:
        lst (list): The input list to check.

    Returns:
        bool: True if the list contains duplicates, False otherwise.
    """
    for i in range(len(lst)):
        for j in range(i + 1, len(lst)):
            if lst[i] == lst[j]:
                return True
    return False

# Example usage
if __name__ == "__main__":
    # Test case 1: List with duplicates
    lst1 = [1, 2, 3, 2, 4]
    print(f"List {lst1} contains duplicates: {contains_duplicates(lst1)}")

    # Test case 2: List without duplicates
    lst2 = [1, 2, 3, 4, 5]
    print(f"List {lst2} contains duplicates: {contains_duplicates(lst2)}")
```
