## Exercise
Write a program that iterates over the values of a given dictionary

## Reference Solution
```python
def iterate_dict_values(dictionary):
    """
    Iterate over the values of a given dictionary.

    Args:
        dictionary (dict): The dictionary to iterate over.

    Returns:
        None
    """
    for value in dictionary.values():
        print(value)

# Example usage
if __name__ == "__main__":
    example_dict = {
        "key1": "value1",
        "key2": "value2",
        "key3": "value3"
    }

    print("Dictionary values:")
    iterate_dict_values(example_dict)
```
