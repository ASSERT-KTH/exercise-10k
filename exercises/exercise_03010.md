## Exercise
Write a program to validate whether a given string contains only alphanumeric characters.

## Reference Solution
```python
def is_alphanumeric(s: str) -> bool:
    """
    Validate whether a given string contains only alphanumeric characters.

    Args:
        s (str): The input string to be validated.

    Returns:
        bool: True if the string contains only alphanumeric characters, False otherwise.
    """
    return s.isalnum()


# Example usage
def main():
    test_strings = [
        "HelloWorld",
        "Hello World",
        "Hello123",
        "!@#$%",
        "Hello123!",
    ]

    for s in test_strings:
        print(f"String: {s}, Alphanumeric: {is_alphanumeric(s)}")


if __name__ == "__main__":
    main()
```
