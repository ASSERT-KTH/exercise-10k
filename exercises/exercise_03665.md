## Exercise
Write a function that takes two numbers as input and returns the maximum of the two.

## Reference Solution
```python
def max_of_two_numbers(num1, num2):
    """
    This function compares two numbers and returns the maximum value.

    Parameters:
    num1 (int or float): The first number for comparison.
    num2 (int or float): The second number for comparison.

    Returns:
    int or float: The maximum of the two input numbers.
    """

    # First, we need to check if both inputs are numbers
    # We can use the isinstance function to check the type of the input
    if not isinstance(num1, (int, float)) or not isinstance(num2, (int, float)):
        raise TypeError("Both inputs must be numbers")

    # Now, we can use the built-in max function in Python to find the maximum of the two numbers
    # The max function takes in multiple arguments and returns the largest one
    # We pass num1 and num2 as arguments to the max function
    max_value = max(num1, num2)

    # Finally, we return the maximum value
    return max_value

# Example usage:
print(max_of_two_numbers(5, 10))  # Output: 10
print(max_of_two_numbers(-5, 0))  # Output: 0
```
