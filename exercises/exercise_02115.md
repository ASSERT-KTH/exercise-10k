## Exercise
Write a function to remove and return an arbitrary key-value pair from a dictionary

## Reference Solution
```python
def remove_arbitrary_pair(dictionary):
    """
    Removes and returns an arbitrary key-value pair from a dictionary.

    Args:
        dictionary (dict): The dictionary from which to remove a key-value pair.

    Returns:
        tuple or None: A tuple containing the removed key-value pair, or None if the dictionary is empty.
    """
    if not dictionary:
        return None

    # Get an arbitrary key from the dictionary
    key = next(iter(dictionary))

    # Remove and return the key-value pair
    value = dictionary.pop(key)
    return key, value

# Example usage:
my_dict = {"a": 1, "b": 2, "c": 3}
print(remove_arbitrary_pair(my_dict))  # Output: ('a', 1) (or any other key-value pair)
print(my_dict)  # Output: {'b': 2, 'c': 3}
```
