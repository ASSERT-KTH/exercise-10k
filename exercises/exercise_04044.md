## Exercise
Write a function that takes a dictionary as input and returns a list of tuples containing all the key-value pairs in the dictionary.

## Reference Solution
```python
def dict_to_tuple_list(input_dict):
    """
    This function takes a dictionary as input and returns a list of tuples containing all the key-value pairs in the dictionary.

    Args:
        input_dict (dict): The dictionary to be converted into a list of tuples.

    Returns:
        list: A list of tuples containing all the key-value pairs in the dictionary.
    """

    # First, we check if the input is indeed a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary")

    # We use the items() method of the dictionary to get a view object that displays a list of a dictionaryâ€™s key-value tuple pairs
    dict_items = input_dict.items()

    # We convert the view object returned by items() into a list
    # This is because the problem statement asks for a list of tuples
    tuple_list = list(dict_items)

    # Finally, we return the list of tuples
    return tuple_list

# Example usage:
input_dict = {"a": 1, "b": 2, "c": 3}
print(dict_to_tuple_list(input_dict))  # Output: [('a', 1), ('b', 2), ('c', 3)]
```
