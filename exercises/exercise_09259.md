## Exercise
Write a program that finds the union of two sets and prints the result.

## Reference Solution
```python
def set_union(set1, set2):
    """
    Returns the union of two sets.

    Args:
        set1 (set): The first set.
        set2 (set): The second set.

    Returns:
        set: The union of set1 and set2.
    """
    return set1.union(set2)

def main():
    # Example usage
    set1 = {1, 2, 3, 4}
    set2 = {3, 4, 5, 6}

    print("Set 1:", set1)
    print("Set 2:", set2)

    union = set_union(set1, set2)
    print("Union:", union)

if __name__ == "__main__":
    main()
```
