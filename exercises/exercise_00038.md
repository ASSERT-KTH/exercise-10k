## Exercise
Write a function that takes a list as input and returns a list of all possible permutations of the elements.

## Reference Solution
```python
import itertools

def generate_permutations(input_list):
    """
    Generates all possible permutations of the input list.

    Args:
        input_list (list): The input list.

    Returns:
        list: A list of tuples, where each tuple is a permutation of the input list.
    """
    return list(itertools.permutations(input_list))

# Example usage:
if __name__ == "__main__":
    input_list = [1, 2, 3]
    permutations = generate_permutations(input_list)
    print(permutations)
```
