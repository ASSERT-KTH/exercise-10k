## Exercise
Create a function that takes a size parameter and prints a square of that size using nested loops

## Reference Solution
```python
def print_square(size, symbol='*'):
    """
    Prints a square of a given size using a specified symbol.

    Args:
        size (int): The size of the square.
        symbol (str, optional): The symbol to use for printing the square. Defaults to '*'.
    """
    for i in range(size):
        for j in range(size):
            print(symbol, end=' ')
        print()

# Example usage:
print_square(5)
print_square(3, '#')
```
