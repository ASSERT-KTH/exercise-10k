## Exercise
Write a function that takes a function as input and applies it to a list of numbers.

## Reference Solution
```python
def apply_function(func, num_list):
    """
    Applies a given function to a list of numbers.

    Args:
        func (function): The function to be applied.
        num_list (list): A list of numbers.

    Returns:
        list: A list of results after applying the function.
    """

    # Initialize an empty list to store the results
    result = []

    # Iterate over each number in the input list
    for num in num_list:
        # Apply the function to the current number and append the result to the result list
        result.append(func(num))

    # Return the list of results
    return result

# Example usage:
# Define a simple function that squares a number
def square(x):
    return x ** 2

# Create a list of numbers
numbers = [1, 2, 3, 4, 5]

# Apply the square function to the list of numbers
squared_numbers = apply_function(square, numbers)

# Print the result
print(squared_numbers)
```
