## Exercise
Write a function that takes another function as an argument and applies it to a given value.

## Reference Solution
```python
def apply_function(func, value):
    """
    Applies a given function to a specified value.

    Args:
        func (function): The function to be applied.
        value: The value to which the function will be applied.

    Returns:
        result: The result of applying the function to the value.
    """

    # First, we check if the provided 'func' is actually a function
    if not callable(func):
        raise TypeError("The first argument must be a function.")

    # Now, we apply the function to the given value and store the result
    # We use a try-except block to catch any potential errors that might occur during function application
    try:
        # Apply the function to the value and store the result
        result = func(value)
    except Exception as e:
        # If an error occurs, print the error message and return None
        print(f"An error occurred: {str(e)}")
        return None

    # Finally, we return the result of applying the function to the value
    return result


# Example usage:
def square(x):
    return x ** 2

print(apply_function(square, 5))  # Outputs: 25
```
