## Exercise
Create a program that calculates the length of a given string.

## Reference Solution
```python
def calculate_string_length(input_string):
    """
    Calculate the length of a given string.

    Args:
        input_string (str): The input string.

    Returns:
        int: The length of the input string.
    """
    return len(input_string)

def main():
    # Get the input string from the user
    input_string = input("Enter a string: ")

    # Calculate the length of the input string
    string_length = calculate_string_length(input_string)

    # Print the result
    print(f"The length of the string '{input_string}' is: {string_length}")

if __name__ == "__main__":
    main()
```
