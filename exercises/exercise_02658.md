## Exercise
Write a function to create a new dictionary using dictionary comprehension

## Reference Solution
```python
def create_dictionary(keys, values):
    """
    Create a dictionary using dictionary comprehension.

    Args:
        keys (list): A list of keys for the dictionary.
        values (list): A list of values for the dictionary.

    Returns:
        dict: A dictionary where each key is mapped to its corresponding value.

    Raises:
        ValueError: If the lengths of the keys and values lists do not match.
    """
    if len(keys) != len(values):
        raise ValueError("The lengths of the keys and values lists must match.")

    # Use dictionary comprehension to create a new dictionary
    new_dict = {key: value for key, value in zip(keys, values)}

    return new_dict


# Example usage:
if __name__ == "__main__":
    keys = ['apple', 'banana', 'cherry']
    values = [1, 2, 3]

    new_dict = create_dictionary(keys, values)
    print(new_dict)
```
