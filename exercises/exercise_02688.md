## Exercise
Write a program that takes three numbers as input and determines the maximum of the three.

## Reference Solution
```python
def find_max(a, b, c):
    max_val = max(a, b, c)
    return max_val

a = float(input("Enter the first number: "))
b = float(input("Enter the second number: "))
c = float(input("Enter the third number: "))

max_val = find_max(a, b, c)
print(f"The maximum of {a}, {b}, and {c} is {max_val}.")
```
