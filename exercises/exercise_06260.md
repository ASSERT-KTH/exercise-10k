## Exercise
Write a function that takes a dictionary as input and maps each value to its square

## Reference Solution
```python
def square_dict_values(input_dict):
    """
    Maps each value in the input dictionary to its square.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        dict: A new dictionary with squared values.
    """
    return {key: value ** 2 for key, value in input_dict.items()}

# Example usage:
if __name__ == "__main__":
    input_dict = {"a": 1, "b": 2, "c": 3, "d": 4}
    squared_dict = square_dict_values(input_dict)
    print("Input Dictionary:", input_dict)
    print("Squared Dictionary:", squared_dict)
```
