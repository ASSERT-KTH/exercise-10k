## Exercise
Create a function that checks if a given string is valid based on the following rules: it must contain at least one uppercase letter, one lowercase letter, and one number. The function should return True if the string is valid and False otherwise.

## Reference Solution
```python
def check_string(s):
    """
    Checks if a given string is valid based on the following rules:
    it must contain at least one uppercase letter, one lowercase letter, and one number.

    Args:
        s (str): The input string to be checked.

    Returns:
        bool: True if the string is valid, False otherwise.
    """

    # First, we'll assume the string is not valid
    is_valid = False

    # Check if the string has at least one uppercase letter
    has_uppercase = any(char.isupper() for char in s)
    
    # Check if the string has at least one lowercase letter
    has_lowercase = any(char.islower() for char in s)
    
    # Check if the string has at least one digit
    has_digit = any(char.isdigit() for char in s)
    
    # If the string has at least one uppercase letter, one lowercase letter, and one digit, it's valid
    if has_uppercase and has_lowercase and has_digit:
        is_valid = True
    
    # Return the result
    return is_valid
```
