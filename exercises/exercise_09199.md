## Exercise
Swap the values of two variables

## Reference Solution
```python
def swap_variables_temp(a, b):
    """
    Swap the values of two variables using a temporary variable.

    Args:
        a (any): The first variable.
        b (any): The second variable.

    Returns:
        tuple: The swapped variables.
    """
    temp = a
    a = b
    b = temp
    return a, b

# Example usage
x = 5
y = 10
print("Before swap: x =", x, ", y =", y)
x, y = swap_variables_temp(x, y)
print("After swap: x =", x, ", y =", y)
```
