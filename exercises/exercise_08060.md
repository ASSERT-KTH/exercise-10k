## Exercise
Write a program that uses an if/else statement to respond to user input based on the context of the conversation.

## Reference Solution
```python
def get_user_input():
    """Gets user input"""
    user_input = input("Please enter a message: ")
    return user_input.lower()

def respond_to_input(user_input):
    """Responds to user input based on context"""
    if "help" in user_input:
        return "I'd be happy to help you. What do you need assistance with?"
    elif "goodbye" in user_input or "bye" in user_input:
        return "It was nice chatting with you. Goodbye!"
    else:
        return "I'm not sure I understand. Could you please rephrase?"

def main():
    print("Welcome to the conversation response program!")
    while True:
        user_input = get_user_input()
        response = respond_to_input(user_input)
        print(response)
        if "goodbye" in user_input or "bye" in user_input:
            break

if __name__ == "__main__":
    main()
```
