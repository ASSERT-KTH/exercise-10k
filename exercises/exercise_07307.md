## Exercise
Create a function that takes a string, a substring to replace, and a replacement substring as arguments and returns the modified string

## Reference Solution
```python
def replace_substring(original_string: str, substring_to_replace: str, replacement_substring: str) -> str:
    """
    Replaces a substring with a replacement substring in a given string.

    Args:
        original_string (str): The original string.
        substring_to_replace (str): The substring to replace.
        replacement_substring (str): The replacement substring.

    Returns:
        str: The modified string.
    """
    return original_string.replace(substring_to_replace, replacement_substring)

# Example usage:
def main():
    original_string = "Hello, world!"
    substring_to_replace = "world"
    replacement_substring = "Python"

    modified_string = replace_substring(original_string, substring_to_replace, replacement_substring)
    print(modified_string)  # Output: "Hello, Python!"

if __name__ == "__main__":
    main()
```
