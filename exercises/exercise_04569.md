## Exercise
Write a function that takes two dictionaries and returns a new dictionary with the key-value pairs from both dictionaries

## Reference Solution
```python
def merge_dictionaries(dict1, dict2):
    """
    Merge two dictionaries into a new dictionary.

    Args:
        dict1 (dict): The first dictionary.
        dict2 (dict): The second dictionary.

    Returns:
        dict: A new dictionary with the key-value pairs from both input dictionaries.
    """
    return {**dict1, **dict2}

# Example usage
dict1 = {'a': 1, 'b': 2}
dict2 = {'c': 3, 'd': 4}

merged_dict = merge_dictionaries(dict1, dict2)
print(merged_dict)  # Output: {'a': 1, 'b': 2, 'c': 3, 'd': 4}
```
