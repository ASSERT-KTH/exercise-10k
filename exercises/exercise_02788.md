## Exercise
Write a program that removes a specified element from a given array.

## Reference Solution
```python
def remove_element(array, target):
    """
    Removes the first occurrence of a specified element from an array.

    Args:
        array (list): The input array.
        target: The element to be removed.

    Returns:
        list: The array with the target element removed.
    """
    try:
        # Find the index of the target element
        index = array.index(target)
        
        # Remove the target element
        array.pop(index)
        
        return array
    
    except ValueError:
        # If the target element is not found, return the original array
        return array


# Example usage:
array = [1, 2, 3, 4, 5, 2, 6]
target = 2
print("Original array:", array)
print("Array after removing", target, ":", remove_element(array, target))
```
