## Exercise
Write a program that takes a string and a list of strings as input, and returns a list of all anagrams of the string in the list.

## Reference Solution
```python
def find_anagrams(input_str, str_list):
    """
    This function finds all anagrams of a given string in a list of strings.

    Args:
        input_str (str): The string for which anagrams are to be found.
        str_list (list): A list of strings in which anagrams are to be searched.

    Returns:
        list: A list of all anagrams of the input string in the given list.
    """

    # Sort the characters in the input string
    sorted_input = sorted(input_str)

    # Initialize an empty list to store the anagrams
    anagrams = []

    # Iterate over each string in the list
    for string in str_list:
        # Sort the characters in the current string
        sorted_string = sorted(string)

        # Check if the sorted strings are equal
        if sorted_string == sorted_input:
            # If they are equal, it means the current string is an anagram of the input string
            anagrams.append(string)

    # Return the list of anagrams
    return anagrams


# Example usage:
input_str = "listen"
str_list = ["silent", "enlist", "inlets", "abc", "def"]
print(find_anagrams(input_str, str_list))  # Output: ['silent', 'enlist', 'inlets']
```
