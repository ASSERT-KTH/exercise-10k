## Exercise
Write a function to convert a string to lowercase.

## Reference Solution
```python
def convert_to_lowercase(input_string: str) -> str:
    """
    Converts a given string to lowercase.

    Args:
    input_string (str): The string to be converted.

    Returns:
    str: The input string in lowercase.
    """
    return input_string.lower()

# Example usage:
if __name__ == "__main__":
    original_string = "Hello, World!"
    print(f"Original string: {original_string}")
    lowercase_string = convert_to_lowercase(original_string)
    print(f"Lowercase string: {lowercase_string}")
```
