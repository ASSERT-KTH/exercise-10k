## Exercise
Create a program that finds the maximum value in an array using a for loop.

## Reference Solution
```python
def find_max(array):
    # Initialize max_value to the first element in the array
    max_value = array[0]

    # Iterate over the array
    for num in array:
        # Check if the current number is greater than max_value
        if num > max_value:
            # If it is, update max_value
            max_value = num

    # Return the maximum value
    return max_value

# Test the function
array = [10, 20, 30, 40, 50]
print("The maximum value in the array is:", find_max(array))
```
