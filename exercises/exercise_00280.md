## Exercise
Write a program that calculates the tip amount and total cost of a bill given its amount and tip percentage.

## Reference Solution
```python
def calculate_tip(bill_amount, tip_percentage):
    """
    Calculate the tip amount and total cost of a bill.

    Args:
        bill_amount (float): The amount of the bill.
        tip_percentage (float): The tip percentage.

    Returns:
        tuple: A tuple containing the tip amount and total cost.
    """
    tip_amount = bill_amount * (tip_percentage / 100)
    total_cost = bill_amount + tip_amount
    return tip_amount, total_cost

def main():
    # Get the bill amount and tip percentage from the user
    bill_amount = float(input("Enter the bill amount: $"))
    tip_percentage = float(input("Enter the tip percentage (%): "))

    # Calculate the tip amount and total cost
    tip_amount, total_cost = calculate_tip(bill_amount, tip_percentage)

    # Print the results
    print(f"Bill Amount: ${bill_amount:.2f}")
    print(f"Tip Percentage: {tip_percentage}%")
    print(f"Tip Amount: ${tip_amount:.2f}")
    print(f"Total Cost: ${total_cost:.2f}")

if __name__ == "__main__":
    main()
```
