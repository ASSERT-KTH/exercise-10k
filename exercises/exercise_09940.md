## Exercise
Write a function that takes an array of integers as input and returns the maximum value in the array. If the array is empty, return a message indicating that the array is empty.

## Reference Solution
```python
def find_max(arr):
    if not arr:
        return "The array is empty."
    return max(arr)

# Test the function
print(find_max([1, 2, 3, 4, 5]))  # Output: 5
print(find_max([]))  # Output: The array is empty.
```
