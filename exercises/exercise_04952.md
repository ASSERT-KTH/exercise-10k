## Exercise
Create a program that determines the time of day (morning, afternoon, evening) based on a given hour (0-23) using conditional statements.

## Reference Solution
```python
hour = int(input("Enter the hour of the day: "))

if hour < 0 or hour > 23:
    print("Invalid hour. Please enter a value between 0 and 23.")
elif hour < 12:
    print("Good morning!")
elif hour < 17:
    print("Good afternoon!")
else:
    print("Good evening!")
```
