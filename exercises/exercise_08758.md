## Exercise
Create a function to join a list of strings into a single string.

## Reference Solution
```python
def join_strings(string_list, separator=''):
    """
    Join a list of strings into a single string.

    Args:
        string_list (list): A list of strings to be joined.
        separator (str): The separator to use between strings. Defaults to an empty string.

    Returns:
        str: The joined string.
    """
    return separator.join(string_list)

# Example usage:
strings_to_join = ['Hello', 'world', 'this', 'is', 'a', 'test']
joined_string = join_strings(strings_to_join, ' ')
print(joined_string)  # Output: Hello world this is a test
```
