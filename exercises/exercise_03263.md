## Exercise
Write a program that uses a for loop to print out the numbers from 10 to 1 in descending order.

## Reference Solution
```python
def print_descending_numbers(start, end):
    """
    Prints numbers from start to end in descending order.

    Args:
        start (int): The starting number.
        end (int): The ending number.
    """
    for i in range(start, end - 1, -1):
        print(i)

def main():
    start = 10
    end = 1
    print_descending_numbers(start, end)

if __name__ == "__main__":
    main()
```
