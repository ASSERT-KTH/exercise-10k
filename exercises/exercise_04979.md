## Exercise
Write a function that removes all consonants from a given string

## Reference Solution
```python
def remove_consonants(input_string):
    """
    Removes all consonants from a given string.

    Args:
        input_string (str): The string from which consonants will be removed.

    Returns:
        str: A new string with all consonants removed.
    """
    # Define a string of vowels (both lowercase and uppercase)
    vowels = 'aeiouAEIOU'

    # Use a list comprehension to filter out consonants
    # and then join the remaining characters into a string
    return ''.join([char for char in input_string if char in vowels or not char.isalpha()])

# Example usage
if __name__ == "__main__":
    input_str = "Hello, World!"
    print("Original String:", input_str)
    print("String without consonants:", remove_consonants(input_str))
```
