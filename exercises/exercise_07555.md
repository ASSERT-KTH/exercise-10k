## Exercise
Create a function that takes a list of numbers as input and prints out their sum.

## Reference Solution
```python
def print_sum(numbers):
    """
    Calculate the sum of a list of numbers and print the result.

    Args:
        numbers (list): A list of numbers.

    Returns:
        None
    """
    total = sum(numbers)
    print("The sum of the numbers is:", total)

# Example usage
def main():
    numbers = [1, 2, 3, 4, 5]
    print_sum(numbers)

if __name__ == "__main__":
    main()
```
