## Exercise
Create a function that calculates the average of all elements in a given list.

## Reference Solution
```python
def calculate_average(lst):
    """
    Calculate the average of all elements in a given list.

    Args:
        lst (list): A list of numbers.

    Returns:
        float: The average of all elements in the list.

    Raises:
        ValueError: If the list is empty.
        TypeError: If the list contains non-numeric values.
    """
    if not lst:
        raise ValueError("List cannot be empty")

    try:
        return sum(lst) / len(lst)
    except TypeError:
        raise TypeError("List must contain only numeric values")


# Example usage
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    average = calculate_average(numbers)
    print(f"The average of the list is: {average}")
```
