## Exercise
Write a program that takes two command line arguments, a name and an age, and prints out a greeting message.

## Reference Solution
```python
import sys

def main():
    # Check if the user provided exactly two command line arguments
    if len(sys.argv) != 3:
        print("Usage: python greeting.py <name> <age>")
        sys.exit(1)

    name = sys.argv[1]
    age = sys.argv[2]

    # Print out a greeting message
    print(f"Hello, {name}! You are {age} years old.")

if __name__ == "__main__":
    main()
```
