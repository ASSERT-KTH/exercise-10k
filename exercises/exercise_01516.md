## Exercise
Write a program that finds the first duplicate character in a given string.

## Reference Solution
```python
def find_first_duplicate(s):
    """
    Finds the first duplicate character in a given string.

    Args:
        s (str): The input string.

    Returns:
        str: The first duplicate character, or None if no duplicates are found.
    """
    seen = set()
    for char in s:
        if char in seen:
            return char
        seen.add(char)
    return None

def main():
    # Example usage:
    s = "abcad"
    duplicate = find_first_duplicate(s)
    if duplicate:
        print(f"The first duplicate character in '{s}' is: {duplicate}")
    else:
        print(f"No duplicate characters found in '{s}'")

if __name__ == "__main__":
    main()
```
