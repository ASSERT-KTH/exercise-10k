## Exercise
Understand the concepts of NaN (Not a Number) and Infinity in programming.

## Reference Solution
```python
import math

# NaN
print(float('nan'))  # Output: nan
print(float('inf') / float('inf'))  # Output: nan
print(math.isnan(float('nan')))  # Output: True

# Infinity
print(float('inf'))  # Output: inf
print(1 / 0)  # Raises ZeroDivisionError
print(float('inf') + 1)  # Output: inf
print(math.isinf(float('inf')))  # Output: True
```
