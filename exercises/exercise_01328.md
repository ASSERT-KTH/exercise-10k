## Exercise
Write a function that takes a string as input and returns the string with all vowels removed.

## Reference Solution
```python
def remove_vowels(input_string: str) -> str:
    """
    This function takes a string as input and returns the string with all vowels removed.

    Args:
        input_string (str): The input string to remove vowels from.

    Returns:
        str: The input string with all vowels removed.
    """
    # Define a string of vowels
    vowels = 'aeiouAEIOU'
    
    # Use a list comprehension to filter out vowels from the input string
    # and then join the characters back into a string
    return ''.join([char for char in input_string if char not in vowels])

# Example usage
if __name__ == "__main__":
    input_str = "Hello World"
    print(f"Original String: {input_str}")
    print(f"String without vowels: {remove_vowels(input_str)}")
```
