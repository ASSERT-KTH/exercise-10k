## Exercise
Write a function to check if a string starts with a given substring.

## Reference Solution
```python
def check_start(string, substring):
    """
    Checks if a given string starts with a specified substring.

    Args:
        string (str): The main string to be checked.
        substring (str): The substring to be searched at the start of the main string.

    Returns:
        bool: True if the string starts with the substring, False otherwise.
    """

    # First, we need to check if both the string and the substring are not None
    if string is None or substring is None:
        raise ValueError("Both string and substring must not be None")

    # We also need to check if both the string and the substring are strings
    if not isinstance(string, str) or not isinstance(substring, str):
        raise TypeError("Both string and substring must be strings")

    # Now, we can use Python's built-in str.startswith() method to check if the string starts with the substring
    # This method returns True if the string starts with the substring, and False otherwise
    return string.startswith(substring)

# Example usage:
print(check_start("Hello, World!", "Hello"))  # Returns: True
print(check_start("Hello, World!", "World"))  # Returns: False
```
