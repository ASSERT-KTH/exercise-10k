## Exercise
Create a function that implements a hash table to store key-value pairs. The function should take a key and a value as arguments and return the value associated with the key.

## Reference Solution
```python
def hash_table(key, value=None, table={}):
    """
    A function that implements a hash table to store key-value pairs.
    
    Args:
        key (any): The key to be used in the hash table.
        value (any, optional): The value associated with the key. Defaults to None.
        table (dict, optional): The hash table. Defaults to {}.
    
    Returns:
        any: The value associated with the key if value is None, otherwise None.
    """

    # If the value is not None, we are inserting or updating a key-value pair
    if value is not None:
        # Insert or update the key-value pair in the hash table
        table[key] = value
    
    # If the value is None, we are retrieving a value by its key
    else:
        # Return the value associated with the key if it exists in the hash table
        return table.get(key)

# Example usage:
print(hash_table('name', 'John'))  # Insert a key-value pair
print(hash_table('name'))  # Retrieve a value by its key
```
