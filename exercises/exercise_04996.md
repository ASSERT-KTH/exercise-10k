## Exercise
Understand the difference between null and undefined variables.

## Reference Solution
```python
// Null example
let name = null; // name is defined but has no value
console.log(name); // Output: null

// Undefined example
let age; // age is not defined or initialized
console.log(age); // Output: undefined

// Example of accessing a non-existent property
let person = { name: 'John' };
console.log(person.age); // Output: undefined
```
