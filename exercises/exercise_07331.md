## Exercise
Write a program that reads input from a URL and prints out the contents.

## Reference Solution
```python
import requests

def read_url_contents(url):
    """
    Reads the contents of a URL and prints it out.

    Args:
        url (str): The URL to read from.
    """
    try:
        # Send a GET request to the URL
        response = requests.get(url)

        # Check if the request was successful
        if response.status_code == 200:
            # Print the contents of the URL
            print(response.text)
        else:
            print(f"Failed to retrieve contents. Status code: {response.status_code}")
    except requests.exceptions.RequestException as e:
        print(f"An error occurred: {e}")

# Example usage
if __name__ == "__main__":
    url = input("Enter a URL: ")
    read_url_contents(url)
```
