## Exercise
Write a program that takes a character as input and determines whether it is a vowel or not.

## Reference Solution
```python
def is_vowel(char):
    """
    Checks if a character is a vowel.

    Args:
        char (str): The input character.

    Returns:
        bool: True if the character is a vowel, False otherwise.
    """
    vowels = 'aeiouAEIOU'
    return char in vowels

def main():
    char = input("Enter a character: ")
    if len(char) != 1:
        print("Please enter a single character.")
    elif is_vowel(char):
        print(f"'{char}' is a vowel.")
    else:
        print(f"'{char}' is not a vowel.")

if __name__ == "__main__":
    main()
```
