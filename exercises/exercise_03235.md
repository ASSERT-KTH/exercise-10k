## Exercise
Write a program to generate all permutations of a given string.

## Reference Solution
```python
import itertools

def generate_permutations(s):
    """
    Generate all permutations of a given string.
    
    Args:
        s (str): The input string.
    
    Returns:
        list: A list of all permutations.
    """
    # Using itertools.permutations to generate all permutations
    perms = [''.join(p) for p in itertools.permutations(s)]
    
    return perms

def main():
    input_str = input("Enter a string: ")
    perms = generate_permutations(input_str)
    
    print("Permutations:")
    for i, perm in enumerate(perms):
        print(f"{i+1}. {perm}")

if __name__ == "__main__":
    main()
```
