## Exercise
Write a program that takes an array and an index as input, and returns the element at that index.

## Reference Solution
```python
#include <stdio.h>

int getElement(int array[], int index) {
    return array[index];
}

int main() {
    int array[] = {1, 2, 3, 4, 5};
    int index = 3;
    int result = getElement(array, index);
    printf("The element at index %d is: %d\n", index, result);
    return 0;
}
```
