## Exercise
Write a program that formats a given string by replacing specified placeholders with actual values.

## Reference Solution
```python
def format_string(template, values):
    """
    Formats a given string by replacing specified placeholders with actual values.

    Args:
        template (str): The input string with placeholders.
        values (dict): A dictionary containing the placeholder keys and their corresponding values.

    Returns:
        str: The formatted string with placeholders replaced.
    """
    for key, value in values.items():
        template = template.replace(f"{{{key}}}", str(value))
    return template

def main():
    # Example usage
    template = "Hello, my name is {name} and I am {age} years old."
    values = {
        "name": "John Doe",
        "age": 30
    }
    formatted_string = format_string(template, values)
    print(formatted_string)

if __name__ == "__main__":
    main()
```
