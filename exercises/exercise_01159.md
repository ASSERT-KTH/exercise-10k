## Exercise
Write a program that generates anagrams of a given word.

## Reference Solution
```python
import itertools

def generate_anagrams(word):
    """
    Generate all possible anagrams of a given word.

    Args:
        word (str): The input word.

    Returns:
        list: A list of all possible anagrams.
    """
    # Remove any whitespace and convert to lowercase
    word = word.replace(" ", "").lower()

    # Generate all permutations of the word
    permutations = itertools.permutations(word)

    # Convert each permutation to a string and store in a set to remove duplicates
    anagrams = set("".join(p) for p in permutations)

    return list(anagrams)

def main():
    word = input("Enter a word: ")
    anagrams = generate_anagrams(word)

    print("Anagrams:")
    for anagram in anagrams:
        print(anagram)

if __name__ == "__main__":
    main()
```
