## Exercise
Declare variables with primitive data types and print their values

## Reference Solution
```python
# Declare variables with primitive data types
boolean_value = True
integer_value = 10
float_value = 3.14
string_value = "Hello, World!"
character_value = 'A'

# Print the values of the variables
print("Boolean Value:", boolean_value)
print("Integer Value:", integer_value)
print("Float Value:", float_value)
print("String Value:", string_value)
print("Character Value:", character_value)

# Print the data types of the variables
print("\nData Types:")
print("Boolean Value:", type(boolean_value))
print("Integer Value:", type(integer_value))
print("Float Value:", type(float_value))
print("String Value:", type(string_value))
print("Character Value:", type(character_value))
```
