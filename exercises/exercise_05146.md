## Exercise
Get a list of all the keys in a dictionary

## Reference Solution

```python
# Example dictionary
my_dict = {'a': 1, 'b': 2, 'c': 3}

# Method 1: Using .keys()
keys = list(my_dict.keys())
print(keys)  # Output: ['a', 'b', 'c']

# Method 2: Using list comprehension
keys = [key for key in my_dict]
print(keys)  # Output: ['a', 'b', 'c']
```
