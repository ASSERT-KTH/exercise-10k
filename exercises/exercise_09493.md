## Exercise
Write a program to find the maximum sum of a subarray within a given one-dimensional array of numbers. The subarray must be a contiguous subset of the original array.

## Reference Solution
```python
def max_subarray_sum(arr):
    """
    Find the maximum sum of a subarray within a given one-dimensional array of numbers.

    Args:
        arr (list): A list of numbers.

    Returns:
        int: The maximum sum of a subarray.

    Raises:
        ValueError: If the input array is empty.
    """
    if not arr:
        raise ValueError("Input array is empty")

    max_sum = float('-inf')  # Initialize max_sum to negative infinity
    current_sum = 0  # Initialize current_sum to 0

    for num in arr:
        current_sum = max(num, current_sum + num)  # Decide whether to continue or start a new subarray
        max_sum = max(max_sum, current_sum)  # Update max_sum if current_sum is greater

    return max_sum

# Example usage:
if __name__ == "__main__":
    arr = [-2, -3, 4, -1, -2, 1, 5, -3]
    max_sum = max_subarray_sum(arr)
    print(f"Maximum subarray sum: {max_sum}")
```
