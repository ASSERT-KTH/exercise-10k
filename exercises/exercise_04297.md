## Exercise
Write a function to map a dictionary's values to a new dictionary using a given function.

## Reference Solution
```python
def map_dict_values(dictionary, func):
    """
    Maps a dictionary's values to a new dictionary using a given function.

    Args:
        dictionary (dict): The input dictionary.
        func (function): A function that takes one argument and returns a value.

    Returns:
        dict: A new dictionary with the same keys as the input dictionary, but with the values transformed by the given function.
    """

    # Initialize an empty dictionary to store the results
    result = {}

    # Iterate over each key-value pair in the input dictionary
    for key, value in dictionary.items():
        
        # Apply the given function to the current value and store the result in the new dictionary
        result[key] = func(value)

    # Return the new dictionary with the transformed values
    return result

# Example usage:
# Define a simple function that doubles its input
def double(x):
    return x * 2

# Create a sample dictionary
sample_dict = {'a': 1, 'b': 2, 'c': 3}

# Map the dictionary's values using the double function
new_dict = map_dict_values(sample_dict, double)

# Print the resulting dictionary
print(new_dict)  # Output: {'a': 2, 'b': 4, 'c': 6}
```
