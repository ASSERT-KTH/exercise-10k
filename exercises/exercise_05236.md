## Exercise
Write a program that takes a weather forecast as input, and prints 'Sunny' if the forecast is 'Clear', 'Rainy' if the forecast is 'Rain', and 'Snowy' if the forecast is 'Snow'.

## Reference Solution
```python
use std::collections::HashMap;

fn get_weather(forecast: &str) -> String {
    let mut weather: HashMap<&str, &str> = HashMap::new();

    weather.insert("Clear", "Sunny");
    weather.insert("Rain", "Rainy");
    weather.insert("Snow", "Snowy");

    let weather = weather.get(forecast);

    match weather {
        Some(x) => String::from(x),
        None => String::from("Unknown"),
    }
}

fn main() {
    let forecast = "Clear";
    println!("{}", get_weather(forecast));
}
```
