## Exercise
Write a program that asks for a sentence and prints out the shortest word.

## Reference Solution
```python
def shortest_word(sentence):
    """
    Find the shortest word in a sentence.

    Args:
        sentence (str): Input sentence.

    Returns:
        str: The shortest word in the sentence.
    """
    words = sentence.split()
    shortest = min(words, key=len)
    return shortest

def main():
    # Ask for a sentence
    sentence = input("Please enter a sentence: ")

    # Find and print the shortest word
    shortest = shortest_word(sentence)
    print("The shortest word is:", shortest)

if __name__ == "__main__":
    main()
```
