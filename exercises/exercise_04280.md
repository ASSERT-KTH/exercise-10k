## Exercise
Write a function that takes a list of tuples as input and returns a dictionary created using dictionary comprehension

## Reference Solution
```python
def tuples_to_dict(tuple_list):
    """
    Creates a dictionary from a list of tuples using dictionary comprehension.

    Args:
        tuple_list (list): A list of tuples, where each tuple contains a key-value pair.

    Returns:
        dict: A dictionary created from the input list of tuples.
    """
    return {key: value for key, value in tuple_list}

# Example usage:
if __name__ == "__main__":
    tuple_list = [(1, "a"), (2, "b"), (3, "c")]
    result_dict = tuples_to_dict(tuple_list)
    print(result_dict)
```
