## Exercise
Write a program that takes a list of integers, reverses the order of the elements, and then sorts the list in ascending order. The program should also remove any duplicate elements from the list.

## Reference Solution
```python
def reverse_sort_list(nums):
    # Reverse the input list
    nums = nums[::-1]
    
    # Remove duplicates and sort the list
    nums = sorted(set(nums))
    
    return nums

# Example usage
nums = [1, 2, 3, 4, 5, 5, 6, 7, 8, 9]
print(reverse_sort_list(nums))  # Output: [1, 2, 3, 4, 5, 6, 7, 8, 9]
```
