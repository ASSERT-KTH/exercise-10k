## Exercise
Write a function that uses the map function to apply a given function to each element of a list.

## Reference Solution
```python
def square(num):
    return num ** 2

def apply_function(func, lst):
    """
    Applies a given function to each element of a list using the map function.

    Args:
        func (function): The function to be applied.
        lst (list): The list of elements to apply the function to.

    Returns:
        list: A new list with the results of applying the function to each element.
    """
    
    # We use the map function to apply the given function to each element of the list.
    # The map function returns a map object, which is an iterator, so we convert it to a list.
    mapped_list = list(map(func, lst))
    
    # We return the new list with the results of applying the function to each element.
    return mapped_list

# Example usage:
numbers = [1, 2, 3, 4, 5]
squares = apply_function(square, numbers)
print(squares)  # Output: [1, 4, 9, 16, 25]
```
