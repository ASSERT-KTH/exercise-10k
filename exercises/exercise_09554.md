## Exercise
Create a function that takes in a string representing the name of another function and a variable number of arguments. The function should then call the function with the given name and pass in the provided arguments. If the function does not exist, it should return an error message.

## Reference Solution
```python
def dynamic_function_caller(func_name, *args, **kwargs):
    """
    Dynamically calls a function with the given name and passes in the provided arguments.
    
    Args:
        func_name (str): The name of the function to be called.
        *args: Variable number of non-keyword arguments to be passed to the function.
        **kwargs: Variable number of keyword arguments to be passed to the function.
    
    Returns:
        The result of the function call if the function exists, otherwise an error message.
    """

    # First, we need to check if the function with the given name exists in the global namespace
    if func_name in globals():
        
        # If the function exists, we retrieve it from the global namespace
        func = globals()[func_name]
        
        # We then check if the retrieved object is callable (i.e., it's a function)
        if callable(func):
            
            # If the object is callable, we call it with the provided arguments and return the result
            try:
                return func(*args, **kwargs)
            except Exception as e:
                # If an exception occurs during the function call, we return an error message
                return f"An error occurred: {str(e)}"
        
        # If the object is not callable, we return an error message
        else:
            return f"'{func_name}' is not a function"
    
    # If the function does not exist in the global namespace, we return an error message
    else:
        return f"Function '{func_name}' not found"

# Example usage:
def add(a, b):
    return a + b

print(dynamic_function_caller('add', 3, 5))  # Output: 8
print(dynamic_function_caller('non_existent_function', 3, 5))  # Output: Function 'non_existent_function' not found
```
