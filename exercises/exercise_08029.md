## Exercise
Write a program to find the maximum contiguous subarray sum of a given one-dimensional array of numbers. The subarray must have at least one element.

## Reference Solution
```python
def max_subarray_sum(arr):
    """
    Find the maximum contiguous subarray sum of a given array.

    Args:
        arr (list): A one-dimensional array of numbers.

    Returns:
        int: The maximum contiguous subarray sum.
    """
    if not arr:
        raise ValueError("Array cannot be empty")

    max_sum = float('-inf')
    current_sum = 0

    for num in arr:
        current_sum = max(num, current_sum + num)
        max_sum = max(max_sum, current_sum)

    return max_sum


# Example usage
if __name__ == "__main__":
    arr = [-2, -3, 4, -1, -2, 1, 5, -3]
    max_sum = max_subarray_sum(arr)
    print("Maximum contiguous subarray sum:", max_sum)
```
