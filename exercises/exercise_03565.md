## Exercise
Write a function that filters a dictionary based on a given condition and returns the filtered dictionary

## Reference Solution
```python
def filter_dict(input_dict, condition):
    """
    Filters a dictionary based on a given condition.

    Args:
        input_dict (dict): The dictionary to be filtered.
        condition (function): A function that takes a key-value pair and returns a boolean.

    Returns:
        dict: The filtered dictionary.
    """
    return {key: value for key, value in input_dict.items() if condition(key, value)}

# Example usage
if __name__ == "__main__":
    # Define a sample dictionary
    sample_dict = {"a": 1, "b": 2, "c": 3, "d": 4}

    # Define a condition function
    def condition(key, value):
        return value > 2

    # Filter the dictionary
    filtered_dict = filter_dict(sample_dict, condition)

    # Print the filtered dictionary
    print(filtered_dict)
```
