## Exercise
Write a function that takes an array of integers as input and returns the maximum value in the array. The function should iterate through the array to find the maximum value.

## Reference Solution
```python
def find_max(arr):
    max_num = arr[0]
    for num in arr:
        if num > max_num:
            max_num = num
    return max_num

# Example usage:
arr = [12, 45, 7, 23, 56, 89, 34]
print(find_max(arr))  # Output: 89
```
