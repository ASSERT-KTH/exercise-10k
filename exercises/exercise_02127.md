## Exercise
Create a function that takes an array and a target value as input and returns the index of the target value if it exists in the array.

## Reference Solution
```python
def find_index(arr, target):
    for i, item in enumerate(arr):
        if item == target:
            return i
    return -1

# Test the function
arr = [1, 3, 5, 7, 9, 2, 4, 6, 8, 10]
target = 7
print(find_index(arr, target))  # Output: 3
```
