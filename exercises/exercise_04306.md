## Exercise
Write a function that takes a temperature in Celsius as input and returns the equivalent temperature in Fahrenheit

## Reference Solution
```python
def celsius_to_fahrenheit(celsius: float) -> float:
    """
    Converts a temperature in Celsius to Fahrenheit.

    Args:
    celsius (float): The temperature in Celsius.

    Returns:
    float: The equivalent temperature in Fahrenheit.
    """
    if celsius < -273.15:
        raise ValueError("Temperature cannot be lower than absolute zero")

    fahrenheit = (celsius * 9/5) + 32
    return fahrenheit

# Example usage
```
