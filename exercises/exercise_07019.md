## Exercise
Write a program that takes a string input from the user, reverses the string, and checks if the original string is a palindrome. If the string is a palindrome, print 'Palindrome' otherwise print 'Not a Palindrome'. The program should handle strings with spaces and special characters.

## Reference Solution
```python
import re

def is_palindrome(s):
    # Remove spaces and special characters
    s = re.sub(r'[^A-Za-z0-9]', '', s)
    # Reverse the string
    rev = s[::-1]
    # Compare the original string with the reversed string
    if s == rev:
        return True
    else:
        return False

# Get user input
s = input("Enter a string: ")
# Check if the input string is a palindrome
if is_palindrome(s):
    print("Palindrome")
else:
    print("Not a Palindrome")
```
