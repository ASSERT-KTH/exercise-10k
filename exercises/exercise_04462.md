## Exercise
Write a program to print numbers from 1 to N using a while loop

## Reference Solution
```python
def print_numbers(n):
    """
    Prints numbers from 1 to n using a while loop.

    Args:
        n (int): The upper limit of the range of numbers to print.
    """
    i = 1
    while i <= n:
        print(i)
        i += 1

# Example usage:
n = int(input("Enter a positive integer: "))
print_numbers(n)
```
