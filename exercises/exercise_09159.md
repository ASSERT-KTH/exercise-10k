## Exercise
Create a program that reads input from the user and prints out a greeting message, then print out the greeting.

## Reference Solution
```python
def print_greeting():
    """Prints a greeting message to the user."""
    user_name = input("Please enter your name: ")
    greeting_message = f"Hello, {user_name}! Welcome to our program."
    print(greeting_message)

def main():
    """Main function to execute the program."""
    print_greeting()

if __name__ == "__main__":
    main()
```
