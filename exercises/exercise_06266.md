## Exercise
Return a list of all keys in a dictionary

## Reference Solution
```python
def get_dict_keys(dictionary):
    """Return a list of dictionary keys."""
    return list(dictionary.keys())

# Example usage:
my_dict = {'a': 1, 'b': 2, 'c': 3}
keys = get_dict_keys(my_dict)
print(keys)  # Output: ['a', 'b', 'c']
```
