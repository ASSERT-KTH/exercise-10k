## Exercise
Write a program that counts the number of consonants in a given string.

## Reference Solution
```python
def count_consonants(input_string):
    """
    Counts the number of consonants in a given string.

    Args:
        input_string (str): The input string to count consonants in.

    Returns:
        int: The number of consonants in the input string.
    """
    vowels = 'aeiou'
    consonant_count = 0

    for char in input_string.lower():
        if char.isalpha() and char not in vowels:
            consonant_count += 1

    return consonant_count

def main():
    input_string = input("Enter a string: ")
    consonant_count = count_consonants(input_string)
    print(f"The number of consonants in the string is: {consonant_count}")

if __name__ == "__main__":
    main()
```
