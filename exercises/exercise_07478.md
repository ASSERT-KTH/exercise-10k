## Exercise
Write a program that updates the value of a given key in a dictionary

## Reference Solution
```python
def update_dict_value(dictionary, key, new_value):
    """
    Updates the value of a given key in a dictionary.

    Args:
        dictionary (dict): The dictionary to update.
        key (any): The key to update.
        new_value (any): The new value for the key.

    Returns:
        dict: The updated dictionary.
    """
    if key in dictionary:
        dictionary[key] = new_value
        print(f"The value of '{key}' has been updated to '{new_value}'.")
    else:
        print(f"The key '{key}' does not exist in the dictionary.")
    return dictionary

# Example usage:
my_dict = {"name": "John", "age": 30, "city": "New York"}
print("Original dictionary:")
print(my_dict)

updated_dict = update_dict_value(my_dict, "age", 31)
print("\nUpdated dictionary:")
print(updated_dict)
```
