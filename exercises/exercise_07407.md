## Exercise
Create a program that takes two numbers as input and performs all basic arithmetic operations (addition, subtraction, multiplication, division) on them. The program should then output the results of each operation in a clear and readable format.

## Reference Solution
```python
# Basic Arithmetic Operations Program

# Get two numbers from the user
num1 = float(input("Enter first number: "))
num2 = float(input("Enter second number: "))

# Perform arithmetic operations
add = num1 + num2
sub = num1 - num2
mul = num1 * num2
div = num1 / num2

# Print the results
print("\nResults:")
print("--------")
print(f"{num1} + {num2} = {add}")
print(f"{num1} - {num2} = {sub}")
print(f"{num1} * {num2} = {mul}")
print(f"{num1} / {num2} = {div}")

# Example usage
# Enter first number: 10
# Enter second number: 2
#
# Results:
# --------
# 10.0 + 2.0 = 12.0
# 10.0 - 2.0 = 8.0
# 10.0 * 2.0 = 20.0
# 10.0 / 2.0 = 5.0
```
