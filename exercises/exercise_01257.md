## Exercise
Write a higher-order function that applies a given function to each element of a list.

## Reference Solution
```python
def apply_function_to_list(func, lst):
    """
    Applies a given function to each element of a list.

    Args:
        func (function): The function to be applied to each element.
        lst (list): The list of elements.

    Returns:
        list: A new list with the function applied to each element.
    """

    # Initialize an empty list to store the results
    result = []

    # Iterate over each element in the input list
    for element in lst:
        # Apply the given function to the current element and append the result to the result list
        result.append(func(element))

    # Return the result list
    return result

# Example usage:
# Define a simple function that squares a number
def square(x):
    return x ** 2

# Create a list of numbers
numbers = [1, 2, 3, 4, 5]

# Apply the square function to each number in the list
squared_numbers = apply_function_to_list(square, numbers)

# Print the result
print(squared_numbers)  # Output: [1, 4, 9, 16, 25]
```
