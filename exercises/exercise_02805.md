## Exercise
Write a program to ask the user for a string and determine if it is a valid integer.

## Reference Solution
```python
def is_valid_integer(input_string):
    """Check if a string can be converted to a valid integer."""
    try:
        int(input_string)
        return True
    except ValueError:
        return False

def main():
    """Ask the user for a string and check if it's a valid integer."""
    input_string = input("Please enter a string: ")
    if is_valid_integer(input_string):
        print(f"'{input_string}' is a valid integer.")
    else:
        print(f"'{input_string}' is not a valid integer.")

if __name__ == "__main__":
    main()
```
