## Exercise
Write a function that uses inheritance to create a subclass. Then, call this function with an input value and print the result.

## Reference Solution
```python
class Shape {
    constructor(area) {
        this.area = area;
    }

    calculateArea() {
        return this.area;
    }
}

class Square extends Shape {
    constructor(sideLength) {
        super(sideLength ** 2);
    }
}

function calculateSquareArea(sideLength) {
    let square = new Square(sideLength);
    return square.calculateArea();
}

// Test the function
console.log(calculateSquareArea(5));  // Output: 25
```
