## Exercise
Write a program that takes a day of the week as input and prints whether it is a weekday or a weekend.

## Reference Solution
```python
def is_weekday(day):
    if day in ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday"]:
        return True
    elif day in ["Saturday", "Sunday"]:
        return False

day = "Wednesday"
print(is_weekday(day))
```
