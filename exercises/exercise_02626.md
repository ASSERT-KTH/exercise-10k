## Exercise
Create a program that takes a word as input and generates a scrambled version of it.

## Reference Solution
```python
import random

def scramble_word(word):
    """
    Generates a scrambled version of the input word.

    Args:
        word (str): The input word to be scrambled.

    Returns:
        str: The scrambled version of the input word.
    """
    # Convert the word to a list of characters
    char_list = list(word)

    # Shuffle the list of characters
    random.shuffle(char_list)

    # Join the shuffled characters back into a string
    scrambled_word = ''.join(char_list)

    return scrambled_word

def main():
    # Get the input word from the user
    word = input("Enter a word: ")

    # Scramble the word
    scrambled = scramble_word(word)

    # Print the scrambled word
    print("Scrambled word:", scrambled)

if __name__ == "__main__":
    main()
```
