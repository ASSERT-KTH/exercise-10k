## Exercise
Write a function that takes a dictionary as input and returns a list of all the key-value pairs

## Reference Solution
```python
def get_key_value_pairs(input_dict):
    """
    Returns a list of tuples, where each tuple contains a key-value pair from the input dictionary.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        list[tuple]: A list of tuples, where each tuple contains a key-value pair.
    """
    return list(input_dict.items())

# Example usage:
input_dict = {
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}

key_value_pairs = get_key_value_pairs(input_dict)
print(key_value_pairs)
```

```python
def get_key_value_pairs(input_dict):
    """
    Returns a list of tuples, where each tuple contains a key-value pair from the input dictionary.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        list[tuple]: A list of tuples, where each tuple contains a key-value pair.
    """
    return [(key, input_dict[key]) for key in input_dict]

# Example usage:
input_dict = {
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}

key_value_pairs = get_key_value_pairs(input_dict)
```
