## Exercise
Write a function that uses type hints to specify the expected types of its parameters and return value.

## Reference Solution
```python
def greet(name: str) -> str:
    """
    This function takes a name as input and returns a greeting message.

    Args:
        name (str): The name of the person to be greeted.

    Returns:
        str: A greeting message.
    """

    # First, we want to check if the input is indeed a string
    if not isinstance(name, str):
        raise TypeError("The name must be a string.")

    # Now, we can create the greeting message
    # We use an f-string to insert the name into the message
    greeting_message = f"Hello, {name}!"

    # Finally, we return the greeting message
    return greeting_message

# Example usage:
print(greet("John"))  # Output: Hello, John!
```
