## Exercise
Create a function that takes two values as arguments and returns True if they are not equal, False otherwise

## Reference Solution
```python
def are_not_equal(a, b):
    """
    Checks if two input values are not equal.

    Args:
    a (any): The first value to compare.
    b (any): The second value to compare.

    Returns:
    bool: True if the values are not equal, False otherwise.
    """
    return a != b

# Example usage:
print(are_not_equal(5, 10))  # Expected output: True
print(are_not_equal(5, 5))   # Expected output: False
print(are_not_equal("a", "b"))  # Expected output: True
print(are_not_equal("a", "a"))  # Expected output: False
```
