## Exercise
Write a function to iterate over the keys and values of a dictionary and print them

## Reference Solution
```python
def print_dict_keys_values(input_dict):
    """
    Iterate over the keys and values of a dictionary and print them.

    Args:
        input_dict (dict): The dictionary to iterate over.

    Returns:
        None
    """

    # Check if the input is a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary.")

    # Iterate over the dictionary items (key-value pairs)
    for key, value in input_dict.items():
        # Print the key-value pair
        print(f"Key: {key}, Value: {value}")


# Example usage:
my_dict = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

print_dict_keys_values(my_dict)
```
