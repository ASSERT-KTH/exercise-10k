```python
def find_highest_number(array):
    """
    This function identifies the highest number in a given array.

    Args:
        array (list): A list of numbers.

    Returns:
        int: The highest number in the array.

    Raises:
        ValueError: If the array is empty.
    """
    if not array:
        raise ValueError("Array cannot be empty")

    highest_number = array[0]
    for num in array:
        if num > highest_number:
            highest_number = num

    return highest_number


# Example usage:
numbers = [12, 45, 7, 23, 56, 89, 34]
print(find_highest_number(numbers))  # Output: 89
```
