## Exercise
Create two functions, one that doubles a number and another that adds 1 to a number. Use these functions to create a new function that doubles a number and adds 1.

## Reference Solution
```python
def double_number(num: float) -> float:
    """
    Doubles a given number.

    Args:
        num (float): The number to be doubled.

    Returns:
        float: The doubled number.
    """
    return num * 2


def add_one(num: float) -> float:
    """
    Adds 1 to a given number.

    Args:
        num (float): The number to add 1 to.

    Returns:
        float: The number with 1 added.
    """
    return num + 1


def double_and_add_one(num: float) -> float:
    """
    Doubles a number and then adds 1.

    Args:
        num (float): The number to be doubled and then add 1 to.

    Returns:
        float: The result of doubling the number and adding 1.
    """
    doubled = double_number(num)
    result = add_one(doubled)
    return result


# Example usage
if __name__ == "__main__":
    number = 5.0
    result = double_and_add_one(number)
    print(f"Doubling {number} and adding 1 results in: {result}")
```
