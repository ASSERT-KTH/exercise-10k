## Exercise
Given an array of integers, select a random element from the array. The function should take in an array of integers as input and return a random element from the array.

## Reference Solution
```python
import random

def select_random_element(array):
    """
    Selects a random element from the input array.

    Args:
        array (list): A list of integers.

    Returns:
        int: A random element from the array.
    """
    if not array:
        raise ValueError("Array is empty")

    return random.choice(array)

# Example usage:
array = [1, 2, 3, 4, 5]
random_element = select_random_element(array)
print(random_element)
```
