## Exercise
Develop a program that manages events. The program should take the event name, date, and ticket type as input, and print the event details.

## Reference Solution
```python
class Event:
    def __init__(self, name, date, ticket_type):
        """
        Initialize an Event object.

        Args:
            name (str): The name of the event.
            date (str): The date of the event.
            ticket_type (str): The type of ticket for the event.
        """
        self.name = name
        self.date = date
        self.ticket_type = ticket_type

    def print_details(self):
        """
        Print the event details.
        """
        print("Event Details:")
        print(f"Name: {self.name}")
        print(f"Date: {self.date}")
        print(f"Ticket Type: {self.ticket_type}")


def main():
    """
    The main function of the program.
    """
    print("Event Management Program")
    print("-------------------------")

    # Get event details from user
    name = input("Enter event name: ")
    date = input("Enter event date (YYYY-MM-DD): ")
    ticket_type = input("Enter ticket type: ")

    # Create an Event object
    event = Event(name, date, ticket_type)

    # Print event details
    event.print_details()


if __name__ == "__main__":
    main()
```
