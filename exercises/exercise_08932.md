## Exercise
Write a program to convert a user's input from a string to an integer, and then print out the double of the input number.

## Reference Solution
```python
def convert_and_double():
    """
    Converts user's string input to an integer and prints out the double of the input number.
    """
    # Get user's input
    user_input = input("Please enter a number: ")

    try:
        # Convert input to an integer
        num = int(user_input)
        
        # Calculate and print the double of the input number
        double_num = num * 2
        print(f"The double of {num} is: {double_num}")
    
    except ValueError:
        # Handle invalid input
        print("Invalid input. Please enter a valid number.")

if __name__ == "__main__":
    convert_and_double()
```
