## Exercise
Write a program that takes a list of integers as input and returns the maximum value in the list. If the list is empty, return a message indicating that the list is empty.

## Reference Solution
```python
def max_value_list(l):
    if len(l) == 0:
        return "The list is empty"
    else:
        return max(l)

# Test the function
l = [10, 20, 30, 40, 50]
print(max_value_list(l))
```
