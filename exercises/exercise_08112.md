## Exercise
Write a program that takes a string as input and returns the shortest word in the string.

## Reference Solution
```python
def shortest_word(input_string):
    """
    Returns the shortest word in the input string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The shortest word in the input string.
    """
    words = input_string.split()
    if not words:
        return None
    return min(words, key=len)

def main():
    input_string = input("Enter a string: ")
    shortest = shortest_word(input_string)
    if shortest:
        print(f"The shortest word is: {shortest}")
    else:
        print("The input string is empty.")

if __name__ == "__main__":
    main()
```
