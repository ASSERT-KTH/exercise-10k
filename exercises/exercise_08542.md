## Exercise
Create a dictionary with 5 key-value pairs and print the keys.

## Reference Solution
```python
def main():
    # Create a dictionary with 5 key-value pairs
    my_dict = {
        "apple": 1,
        "banana": 2,
        "cherry": 3,
        "date": 4,
        "elderberry": 5
    }

    # Print the keys
    print("Keys:")
    for key in my_dict.keys():
        print(key)

    # Alternatively, you can use list() function to print keys
    print("\nKeys using list() function:")
    print(list(my_dict.keys()))

if __name__ == "__main__":
    main()
```
