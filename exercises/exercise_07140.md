## Exercise
Write a program that takes a list of integers as input and returns the maximum value in the list. Assume the list has at least one element.

## Reference Solution
```python
def maxListValue(inputList):
    maxVal = inputList[0]
    for i in range(1, len(inputList)):
        if inputList[i] > maxVal:
            maxVal = inputList[i]
    return maxVal

# Example usage
inputList = [1, 2, 3, 4, 5]
print(maxListValue(inputList))  # Output: 5
```
