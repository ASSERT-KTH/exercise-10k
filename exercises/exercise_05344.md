## Exercise
Write a function that takes an array and an aggregation function (such as sum, product, or average) as input, and returns the aggregated value of the array elements.

## Reference Solution
```python
from functools import reduce
from operator import mul

def aggregate(arr, agg_func):
    if agg_func == "sum":
        return sum(arr)
    elif agg_func == "product":
        return reduce(mul, arr, 1)
    elif agg_func == "average":
        return sum(arr) / len(arr)
    else:
        raise ValueError("Invalid aggregation function")
```
