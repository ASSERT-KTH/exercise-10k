## Exercise
Write a program that takes a string as input and returns all possible anagrams of the string.

## Reference Solution
```python
import itertools

def generate_anagrams(input_string):
    """
    Generate all possible anagrams of a given string.

    Args:
        input_string (str): The input string.

    Returns:
        list: A list of all possible anagrams.
    """
    # Remove any whitespace from the input string
    input_string = input_string.replace(" ", "")

    # Generate all permutations of the input string
    perms = itertools.permutations(input_string)

    # Convert the permutations to a list of strings and return
    return [''.join(p) for p in set(perms)]


def main():
    input_string = input("Enter a string: ")
    anagrams = generate_anagrams(input_string)

    print("Anagrams:")
    for i, anagram in enumerate(anagrams):
        print(f"{i+1}. {anagram}")


if __name__ == "__main__":
    main()
```
