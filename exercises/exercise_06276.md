## Exercise
Group a list of items by a common key using a dictionary

## Reference Solution
```python
def group_by_key(items, key):
    """
    Group a list of items by a common key using a dictionary.

    Args:
        items (list): The list of items to group.
        key (str): The key to group by.

    Returns:
        dict: A dictionary with the grouped items.
    """
    grouped = {}
    for item in items:
        if item[key] not in grouped:
            grouped[item[key]] = []
        grouped[item[key]].append(item)
    return grouped

# Example usage
items = [
    {"name": "John", "age": 25},
    {"name": "Jane", "age": 25},
    {"name": "Bob", "age": 30},
    {"name": "Alice", "age": 30},
]

grouped_items = group_by_key(items, "age")
print(grouped_items)
```
