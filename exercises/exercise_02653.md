## Exercise
Write a function to remove and return the last inserted key-value pair from a dictionary using the popitem() method

## Reference Solution
```python
def remove_last_inserted_pair(dictionary):
    """
    Removes and returns the last inserted key-value pair from a dictionary.
    
    Args:
    dictionary (dict): The input dictionary.
    
    Returns:
    tuple: A tuple containing the last inserted key-value pair. If the dictionary is empty, it returns None.
    """
    if dictionary:
        # Use popitem() to remove and return the last inserted key-value pair
        return dictionary.popitem()
    else:
        # If the dictionary is empty, return None
        return None

# Example usage:
my_dict = {'a': 1, 'b': 2, 'c': 3}
print("Original dictionary:", my_dict)
last_inserted_pair = remove_last_inserted_pair(my_dict)
print("Last inserted pair:", last_inserted_pair)
print("Updated dictionary:", my_dict)
```
