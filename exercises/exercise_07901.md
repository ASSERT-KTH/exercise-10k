## Exercise
Write a program that takes a list of integers as input and returns the maximum value in the list. The program should handle lists of varying lengths and should be able to handle empty lists by returning a message indicating that the list is empty.

## Reference Solution
```python
class MaxIntFinder:
    def __init__(self, int_list):
        self.int_list = int_list

    def find_max(self):
        if len(self.int_list) == 0:
            return "List is empty"
        else:
            sorted_int_list = sorted(self.int_list)
            return sorted_int_list[-1]

maxint = MaxIntFinder([1, 2, 3, 4, 5])
print(maxint.find_max())

maxint = MaxIntFinder([])
print(maxint.find_max())
```
