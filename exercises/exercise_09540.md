## Exercise
Write a program that takes in a user's input and determines the data type of the input. The program should check if the input is an integer, float, or string, and print out the corresponding data type.

## Reference Solution
```python
# This program determines the data type of the user's input

# Ask the user for input
user_input = input("Please enter a number or a word: ")

# Check if the input is an integer
try:
    int(user_input)
    print("You've entered an integer.")
except ValueError:
    # Check if the input is a float
    try:
        float(user_input)
        print("You've entered a float.")
    except ValueError:
        # If the input is neither an integer nor a float, it must be a string
        print("You've entered a string.")
```
