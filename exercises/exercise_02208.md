## Exercise
Write a program that takes a user's name as input and checks if it contains only letters and spaces. Print 'Valid name' if valid, otherwise print 'Invalid name'

## Reference Solution
```python
def validate_name(name):
    """
    Checks if a name contains only letters and spaces.

    Args:
        name (str): The name to validate.

    Returns:
        bool: True if the name is valid, False otherwise.
    """
    return name.replace(' ', '').isalpha()


def main():
    # Get user input
    name = input("Please enter your name: ")

    # Validate the name
    if validate_name(name):
        print("Valid name")
    else:
        print("Invalid name")


if __name__ == "__main__":
    main()
```
