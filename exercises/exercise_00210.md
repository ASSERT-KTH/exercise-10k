## Exercise
Write a function that filters a list based on a given condition.

## Reference Solution
```python
def filter_list(input_list, condition):
    """
    Filters a list based on a given condition.

    Args:
    input_list (list): The list to be filtered.
    condition (function): A function that takes one argument and returns a boolean value.

    Returns:
    list: The filtered list.
    """
    return [item for item in input_list if condition(item)]

# Example usage:
def is_even(num):
    """
    Checks if a number is even.

    Args:
    num (int): The number to check.

    Returns:
    bool: True if the number is even, False otherwise.
    """
    return num % 2 == 0

numbers = [1, 2, 3, 4, 5, 6]
filtered_numbers = filter_list(numbers, is_even)
print(filtered_numbers)  # Output: [2, 4, 6]
```
