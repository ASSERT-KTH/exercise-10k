## Exercise
Write a program to demonstrate the initialization of variables with default values.

## Reference Solution
```python
# Initialize variables with default values
name = "John Doe"
age = 30
city = "New York"

# Print the default values
print("Default Values:")
print(f"Name: {name}")
print(f"Age: {age}")
print(f"City: {city}")

# Update the variables
name = "Jane Doe"
age = 25
city = "Los Angeles"

# Print the updated values
print("\nUpdated Values:")
print(f"Name: {name}")
print(f"Age: {age}")
print(f"City: {city}")

# Function with default values
def greet(name = "World", age = 0):
    print(f"Hello, {name}! You are {age} years old.")

# Call the function with default values
print("\nCalling the function with default values:")
greet()

# Call the function with custom values
print("\nCalling the function with custom values:")
greet("John Doe", 30)
```
