## Exercise
Write a program that takes a number as input, and prints 'Prime' if the number is prime, 'Not Prime' otherwise.

## Reference Solution
```python
def is_prime(n):
    """Check if a number is prime."""
    if n < 2:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True

def main():
    """Get user input and print whether the number is prime."""
    num = int(input("Enter a number: "))
    if is_prime(num):
        print("Prime")
    else:
        print("Not Prime")

if __name__ == "__main__":
    main()
```
