## Exercise
Write a function to update the value of a key in a dictionary. If the key does not exist, add it to the dictionary.

## Reference Solution
```python
def update_dict_key(dictionary, key, value):
    """
    Updates the value of a key in a dictionary. If the key does not exist, adds it to the dictionary.

    Args:
        dictionary (dict): The dictionary to be updated.
        key (any): The key to be updated or added.
        value (any): The value associated with the key.

    Returns:
        dict: The updated dictionary.
    """
    # Check if the key exists in the dictionary
    if key in dictionary:
        # Update the value associated with the key
        dictionary[key] = value
    else:
        # Add the key-value pair to the dictionary
        dictionary[key] = value
    return dictionary

# Example usage
my_dict = {"name": "John", "age": 30}
print("Original Dictionary:", my_dict)

updated_dict = update_dict_key(my_dict, "age", 31)
print("Updated Dictionary (existing key):", updated_dict)

updated_dict = update_dict_key(updated_dict, "city", "New York")
print("Updated Dictionary (new key):", updated_dict)
```
