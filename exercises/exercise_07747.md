## Exercise
Write a program that prints the numbers from 1 to 10 using a for loop, then prints the numbers from 10 to 1 using a while loop.

## Reference Solution
```python
# Print numbers from 1 to 10 using a for loop
for i in range(1, 11):
    print(i)

# Print numbers from 10 to 1 using a while loop
i = 10
while i >= 1:
    print(i)
    i -= 1
```
