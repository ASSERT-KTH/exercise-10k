## Exercise
Write a function called 'find_min' that finds and returns the minimum number in a given list of integers.

## Reference Solution
```python
def find_min(numbers):
    """
    Finds and returns the minimum number in a given list of integers.

    Args:
        numbers (list): A list of integers.

    Returns:
        int: The smallest integer in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not numbers:
        raise ValueError("Input list is empty")
    return min(numbers)

# Example usage:
if __name__ == "__main__":
    numbers = [12, 45, 7, 23, 56, 89, 34]
    print("Minimum number in the list:", find_min(numbers))
```
