## Exercise
Write a program that finds the minimum element in an array.

## Reference Solution
```python
def find_minimum(array):
    """
    Finds the minimum element in an array.

    Args:
        array (list): A list of integers.

    Returns:
        int: The minimum element in the array.
    """
    if not array:
        raise ValueError("Array is empty")

    minimum = array[0]
    for element in array:
        if element < minimum:
            minimum = element

    return minimum

# Example usage:
def main():
    array = [5, 2, 8, 12, 3]
    minimum = find_minimum(array)
    print(f"Minimum element: {minimum}")

if __name__ == "__main__":
    main()
```
