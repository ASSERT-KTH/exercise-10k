## Exercise
Create a function named 'sum_of_squares' that calculates the sum of squares of all numbers in a given list.

```python
def sum_of_squares(numbers):
    """
    Calculates the sum of squares of all numbers in a given list.

    Args:
        numbers (list): A list of numbers.

    Returns:
        int: The sum of squares of all numbers in the list.

    Raises:
        TypeError: If the input is not a list or if the list contains non-numeric values.
    """
    # Check if input is a list
    if not isinstance(numbers, list):
        raise TypeError("Input must be a list")

    # Initialize sum of squares to 0
    sum_squares = 0

    # Iterate over each number in the list
    for num in numbers:
        # Check if number is numeric
        if not isinstance(num, (int, float)):
            raise TypeError("List must only contain numeric values")
        
        # Add square of number to sum
        sum_squares += num ** 2

    return sum_squares

# Example usage
numbers = [1, 2, 3, 4, 5]
result = sum_of_squares(numbers)
print(result)  # Output: 55
```
