## Exercise
Create a program that calculates the result of a number raised to a power.

## Reference Solution
```python
def power(base, exponent):
    """
    Calculate the result of a number raised to a power.

    Args:
        base (float): The base number.
        exponent (float): The exponent.

    Returns:
        float: The result of the base raised to the exponent.
    """
    return base ** exponent

def main():
    # Get the base and exponent from the user
    base = float(input("Enter the base: "))
    exponent = float(input("Enter the exponent: "))

    # Calculate the result
    result = power(base, exponent)

    # Print the result
    print(f"{base} raised to the power of {exponent} is {result}")

if __name__ == "__main__":
    main()
```
