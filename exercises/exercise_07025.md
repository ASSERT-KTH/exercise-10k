## Exercise
Create a program that takes a string input from the user and checks if it's a palindrome. If the string is not a palindrome, the program should reverse the string and output the reversed string. If the string is a palindrome, the program should output a message stating that the string is a palindrome.

## Reference Solution
```python
#include <iostream>
#include <string>

using namespace std;

int main() {
    string palabra;
    cout << "Ingrese una palabra: ";
    cin >> palabra;

    string palabra2 = palabra;
    int i = palabra.length() - 1;
    int j = 0;

    while (i >= 0) {
        palabra2[j] = palabra[i];
        i--;
        j++;
    }

    if (palabra == palabra2) {
        cout << "La palabra ingresada es un palindromo" << endl;
    } else {
        cout << "La palabra ingresada no es un palindromo" << endl;
        cout << "La palabra al reves es " << palabra2 << endl;
    }

    return 0;
}
```
