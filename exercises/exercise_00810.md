## Exercise
Cast a variable from one data type to another and print the result.

## Reference Solution
```python
# Integer to Float
integer_value = 10
float_value = float(integer_value)
print(f"Integer {integer_value} cast to float: {float_value}")

# Float to Integer
float_value = 10.5
integer_value = int(float_value)
print(f"Float {float_value} cast to integer: {integer_value}")

# String to Integer
string_value = "10"
integer_value = int(string_value)
print(f"String '{string_value}' cast to integer: {integer_value}")

# Integer to String
integer_value = 10
string_value = str(integer_value)
print(f"Integer {integer_value} cast to string: '{string_value}'")
```
