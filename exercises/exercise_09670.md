## Exercise
Write a program that asks for two strings and alternates their characters.

## Reference Solution
```python
def alternate_characters(str1, str2):
    """
    Alternate characters from two strings.

    Args:
        str1 (str): The first string.
        str2 (str): The second string.

    Returns:
        str: The string with alternating characters.
    """
    result = ""
    max_len = max(len(str1), len(str2))

    for i in range(max_len):
        if i < len(str1):
            result += str1[i]
        if i < len(str2):
            result += str2[i]

    return result

def main():
    # Ask for two strings
    str1 = input("Enter the first string: ")
    str2 = input("Enter the second string: ")

    # Alternate characters
    alternated_str = alternate_characters(str1, str2)

    # Print the result
    print("Alternated string:", alternated_str)

if __name__ == "__main__":
    main()
```
