## Exercise
Write a program that simulates a coin flip and prints out the result. Use if/else statements to determine the result.

## Reference Solution
```python
import random

def coin_flip():
    """
    Simulates a coin flip and returns the result.
    """
    # Generate a random number between 0 and 1
    flip = random.random()

    # Use if/else statement to determine the result
    if flip < 0.5:
        return "Heads"
    else:
        return "Tails"

def main():
    # Call the coin_flip function and print the result
    result = coin_flip()
    print(f"The coin landed on: {result}")

if __name__ == "__main__":
    main()
```
