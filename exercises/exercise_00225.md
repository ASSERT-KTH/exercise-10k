## Exercise
Create a function to concatenate two or more strings.

## Reference Solution
```python
def concatenate_strings(*args):
    """
    Concatenates two or more strings.

    Args:
        *args: A variable number of strings to be concatenated.

    Returns:
        str: The concatenated string.
    """
    return ''.join(args)

# Example usage
def main():
    print(concatenate_strings("Hello, ", "world!"))  # Output: "Hello, world!"
    print(concatenate_strings("This", " is", " a", " test."))  # Output: "This is a test."

if __name__ == "__main__":
    main()
```
