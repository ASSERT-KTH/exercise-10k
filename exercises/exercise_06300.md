## Exercise
Create a dictionary from two lists using the zip() function

## Reference Solution
```python
# Define two lists
keys = ['apple', 'banana', 'cherry']
values = [1, 2, 3]

# Use zip() and dict() to create a dictionary
fruit_dict = dict(zip(keys, values))

# Print the resulting dictionary
print(fruit_dict)
```

```python
{'apple': 1, 'banana': 2, 'cherry': 3}
```

```python
from itertools import zip_longest

keys = ['apple', 'banana', 'cherry']
values = [1, 2]

fruit_dict = dict(zip_longest(keys, values, fillvalue=None))

print(fruit_dict)
```

```python
{'apple': 1, 'banana': 2, 'cherry': None}
```
