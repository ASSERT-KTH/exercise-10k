## Exercise
Write a program that simulates flipping a coin and prints out the result.

## Reference Solution
```python
import random

def flip_coin():
    """Simulates flipping a coin and returns the result."""
    return "Heads" if random.randint(0, 1) == 0 else "Tails"

def main():
    """Runs the coin flip simulator and prints out the result."""
    result = flip_coin()
    print(f"You flipped a coin and got: {result}")

if __name__ == "__main__":
    main()
```
