## Exercise
Write a program that takes an age as input and prints out whether the person is an adult or not.

## Reference Solution
```python
def check_adult(age):
    """
    Checks if a person is an adult based on their age.

    Args:
        age (int): The age of the person.

    Returns:
        bool: True if the person is an adult, False otherwise.
    """
    return age >= 18

def main():
    # Get user input
    age = int(input("Enter your age: "))

    # Check if the person is an adult
    if check_adult(age):
        print("You are an adult.")
    else:
        print("You are not an adult.")

if __name__ == "__main__":
    main()
```
