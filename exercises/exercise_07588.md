## Exercise
Write a program that checks if a given variable is a list.

## Reference Solution
```python
def is_list(var):
    """
    Checks if a variable is a list.

    Args:
        var: The variable to check.

    Returns:
        bool: True if the variable is a list, False otherwise.
    """
    return isinstance(var, list)

# Example usage:
def main():
    # Test cases
    test_cases = [
        [1, 2, 3],  # List
        "hello",  # String
        123,  # Integer
        45.67,  # Float
        True,  # Boolean
        {"a": 1},  # Dictionary
        (1, 2, 3),  # Tuple
    ]

    for var in test_cases:
        print(f"Is {var} a list? {is_list(var)}")

if __name__ == "__main__":
    main()
```
