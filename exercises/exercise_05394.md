## Exercise
Write a function that finds the index of a given element in a list.

## Reference Solution
```python
def find_index(lst, element):
    """
    Find the index of a given element in a list.

    Args:
        lst (list): The list to search in.
        element: The element to search for.

    Returns:
        int: The index of the element if found, -1 otherwise.
    """
    try:
        return lst.index(element)
    except ValueError:
        return -1

# Example usage:
numbers = [1, 2, 3, 4, 5]
print(find_index(numbers, 3))  # Output: 2
print(find_index(numbers, 6))  # Output: -1
```

```python
def find_index_manual(lst, element):
    """
    Find the index of a given element in a list (manual implementation).

    Args:
        lst (list): The list to search in.
        element: The element to search for.

    Returns:
        int: The index of the element if found, -1 otherwise.
    """
    for i, x in enumerate(lst):
        if x == element:
            return i
    return -1

# Example usage:
numbers = [1, 2, 3, 4, 5]
print(find_index_manual(numbers, 3))  # Output: 2
print(find_index_manual(numbers, 6))  # Output: -1
```
