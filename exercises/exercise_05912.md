## Exercise
Write a function that takes two strings as input and returns the concatenation of the two strings

## Reference Solution
```python
def concatenate_strings(str1: str, str2: str) -> str:
    """
    Concatenates two input strings.

    Args:
        str1 (str): The first input string.
        str2 (str): The second input string.

    Returns:
        str: The concatenation of str1 and str2.
    """
    return str1 + str2

# Example usage:
if __name__ == "__main__":
    str1 = "Hello, "
    str2 = "World!"
    print(concatenate_strings(str1, str2))  # Output: Hello, World!
```
