## Exercise
Create a function that takes a string as input and returns its uppercase equivalent.

## Reference Solution
```python
def convert_to_uppercase(input_string):
    """
    This function takes a string as input and returns its uppercase equivalent.

    Args:
        input_string (str): The input string to be converted to uppercase.

    Returns:
        str: The uppercase equivalent of the input string.
    """

    # First, we check if the input is indeed a string. If not, we raise a TypeError.
    if not isinstance(input_string, str):
        raise TypeError("Input must be a string.")

    # Now, we use the built-in upper() method in Python, which converts all lowercase characters in a string into uppercase characters.
    # We store the result in a variable called 'uppercase_string'.
    uppercase_string = input_string.upper()

    # Finally, we return the 'uppercase_string'.
    return uppercase_string

# Example usage:
print(convert_to_uppercase("hello world"))  # Outputs: "HELLO WORLD"
```
