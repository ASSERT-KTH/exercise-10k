## Exercise
Write a program to find the minimum value in a given list of integers.

## Reference Solution
```python
def find_minimum_value(lst):
    """
    Find the minimum value in a given list of integers.

    Args:
        lst (list): A list of integers.

    Returns:
        int: The minimum value in the list.
    """
    if not lst:
        raise ValueError("List is empty")

    return min(lst)


def main():
    # Example usage:
    numbers = [5, 2, 9, 1, 7, 3]
    min_value = find_minimum_value(numbers)
    print(f"Minimum value: {min_value}")


if __name__ == "__main__":
    main()
```
