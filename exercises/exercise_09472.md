## Exercise
Write a program that takes an array of integers as input and returns the maximum value in the array. The program should use a basic algorithm to iterate over the array and keep track of the maximum value found so far.

## Reference Solution
```python
class Solution:
    def maxElement(self, arr):
        max = arr[0]
        for i in range(1, len(arr)):
            if arr[i] > max:
                max = arr[i]
        return max

# Testing the function
arr = [10, 324, 45, 90, 9808]
print(Solution().maxElement(arr))  # Output: 9808
```
