## Exercise
Write a program that calculates the sum of the squares of all numbers from 1 to n, where n is the user's input.

## Reference Solution
```python
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int n = scanner.nextInt();
        int i = 1;
        int sum = 0;

        while (i <= n) {
            int square = i * i;
            sum += square;
            i++;
        }

        System.out.println(sum);
    }
}
```
