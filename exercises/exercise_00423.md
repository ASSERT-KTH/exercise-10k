## Exercise
Write a program that asks the user for a number and prints it back to the console.

## Reference Solution
```python
def get_user_number():
    """Gets a number from the user."""
    while True:
        try:
            user_number = float(input("Please enter a number: "))
            return user_number
        except ValueError:
            print("That's not a valid number. Please try again.")

def main():
    """Main function to run the program."""
    user_number = get_user_number()
    print(f"You entered: {user_number}")

if __name__ == "__main__":
    main()
```
