## Exercise
Create a program that takes an integer as input and prints 'Even' if the number is even, otherwise prints 'Odd'

## Reference Solution
```python
def check_even_or_odd(num):
    """
    Checks if a number is even or odd.

    Args:
        num (int): The number to check.

    Returns:
        str: 'Even' if the number is even, 'Odd' otherwise.
    """
    if num % 2 == 0:
        return 'Even'
    else:
        return 'Odd'

def main():
    # Get user input
    num = int(input("Enter an integer: "))

    # Check if the number is even or odd
    result = check_even_or_odd(num)

    # Print the result
    print(result)

if __name__ == "__main__":
    main()
```
