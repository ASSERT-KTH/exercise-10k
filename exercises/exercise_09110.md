## Exercise
Write a program that takes a list of integers as input, reverses the order of the elements, and then rotates the list by a specified number of positions. The rotation should be to the right, meaning that the last elements of the list should move to the beginning. The number of positions to rotate should also be provided as input.

## Reference Solution
```python
#include <iostream>
#include <vector>

int main() {
    std::vector<int> v;
    int n, m;

    std::cin >> n;
    for (int i = 0; i < n; i++) {
        int a;
        std::cin >> a;
        v.push_back(a);
    }

    std::cin >> m;
    for (int i = 0; i < m; i++) {
        int temp = v.back();
        v.pop_back();
        v.insert(v.begin(), temp);
    }

    for (int i = 0; i < n; i++) {
        std::cout << v[i];
        if (i != n - 1) {
            std::cout << " ";
        }
    }

    return 0;
}
```
