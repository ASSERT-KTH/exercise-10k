## Exercise
Create a program that takes in a variable and its value, then determines and prints out the data type of the variable. The program should be able to handle at least five different data types: integer, float, string, boolean, and list. Use a dictionary to map the variable names to their respective values and iterate over the dictionary to print out the variable names and their corresponding data types.

## Reference Solution
```python
# Variables and Dictionary
x = 10
y = 10.5
z = "hello"
a = True
b = [1, 2, 3]

dic = {"x": x, "y": y, "z": z, "a": a, "b": b}

# Loop and Print
for k, v in dic.items():
    print(f"variable: {k}, value: {v}, datatype: {type(v).__name__}")
```
