## Exercise
Write a function that takes in another function and a list of numbers, and applies the given function to each number in the list.

## Reference Solution
```python
def apply_function(func, num_list):
    """
    Applies a given function to each number in a list.

    Args:
        func (function): The function to be applied.
        num_list (list): A list of numbers.

    Returns:
        list: A new list with the function applied to each number.
    """

    # Initialize an empty list to store the results
    result = []

    # Iterate over each number in the input list
    for num in num_list:
        # Apply the given function to the current number and append the result to the result list
        result.append(func(num))

    # Return the result list
    return result

# Example usage:
# Define a simple function that squares a number
def square(x):
    return x ** 2

# Create a list of numbers
numbers = [1, 2, 3, 4, 5]

# Apply the square function to each number in the list
squared_numbers = apply_function(square, numbers)

# Print the result
print(squared_numbers)  # Output: [1, 4, 9, 16, 25]
```
