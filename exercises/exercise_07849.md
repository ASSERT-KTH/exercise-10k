## Exercise
Write a program that takes an input string and determines the data type of the value represented by the string. The data types to consider are integers, floats, and strings. If the string represents an integer, output 'int'. If it represents a float, output 'float'. If it represents neither an integer nor a float, output 'string'.

## Reference Solution
```python
# Author: [Your Name]
# Date: [Today's Date]
# Description: This program determines the data type of a value represented by an input string.

def check_type(s):
    try:
        int(s)
        return 'int'
    except ValueError:
        try:
            float(s)
            return 'float'
        except ValueError:
            return 'string'

def main():
    s = input("Enter a string: ")
    print(check_type(s))

if __name__ == "__main__":
    main()
```
