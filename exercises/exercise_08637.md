## Exercise
Create a function that finds the maximum number in a given list.

## Reference Solution
```python
def find_max_number(num_list):
    """
    This function finds the maximum number in a given list.

    Args:
        num_list (list): A list of numbers.

    Returns:
        max_num (float): The maximum number in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not num_list:
        raise ValueError("Input list is empty")

    return max(num_list)

# Example usage
if __name__ == "__main__":
    numbers = [4, 2, 9, 6, 5, 1, 8, 3, 7]
    max_num = find_max_number(numbers)
    print("Maximum number in the list:", max_num)
```
