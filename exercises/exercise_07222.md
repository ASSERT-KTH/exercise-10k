## Exercise
Write a program that takes a user input and determines its data type. The program should be able to identify if the input is an integer, float, or string.

## Reference Solution
```python
def determine_data_type(user_input):
    """
    This function determines the data type of the user's input.
    
    Args:
        user_input (str): The input provided by the user.
    
    Returns:
        str: The data type of the user's input.
    """
    
    # Try to convert the input to an integer
    try:
        int(user_input)
        return "Integer"
    
    # If the input cannot be converted to an integer, try to convert it to a float
    except ValueError:
        try:
            float(user_input)
            return "Float"
        
        # If the input cannot be converted to a float, it must be a string
        except ValueError:
            return "String"


def main():
    # Get user input
    user_input = input("Please enter a value: ")
    
    # Determine the data type of the input
    data_type = determine_data_type(user_input)
    
    # Print the data type
    print(f"The data type of '{user_input}' is {data_type}.")


if __name__ == "__main__":
    main()
```
