## Exercise
Write a program that uses a for loop to print each element of a given list.

## Reference Solution
```python
# Define a function to print list elements
def print_list_elements(input_list):
    """
    Prints each element of the given list.
    
    Args:
        input_list (list): The list to print elements from.
    """
    # Use a for loop to iterate over the list
    for element in input_list:
        # Print each element
        print(element)

# Example usage
if __name__ == "__main__":
    # Define a sample list
    sample_list = [1, 2, 3, 4, 5, "apple", "banana", "cherry"]
    
    # Call the function to print list elements
    print("Printing list elements:")
    print_list_elements(sample_list)
```
