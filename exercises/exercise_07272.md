## Exercise
Write a program that takes in a user's input and determines the data type of the input. The input can be either an integer, float, or string. The program should then print out the data type of the input.

## Reference Solution
```python
# Get user input
user_input = input()

# Determine data type
try:
    int(user_input)
    print("This is an integer.")
except ValueError:
    try:
        float(user_input)
        print("This is a float.")
    except ValueError:
        print("This is a string.")
```
