## Exercise
Write a program to convert a value to a boolean.

## Reference Solution
```python
def convert_to_boolean(value):
    """
    Converts a value to a boolean.

    Args:
        value: The value to be converted.

    Returns:
        bool: The boolean representation of the input value.
    """
    return bool(value)


# Example usage:
if __name__ == "__main__":
    # Test cases
    test_cases = [0, 1, 0.0, 1.0, "", "hello", [], [1, 2, 3], {}, {"a": 1}, None]

    for test_case in test_cases:
        print(f"Boolean representation of {test_case}: {convert_to_boolean(test_case)}")
```
