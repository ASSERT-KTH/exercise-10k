## Exercise
Write a program that takes a favorite color as input and prints out a message with the color.

## Reference Solution
```python
def print_color_message(color):
    """Prints a message with the given color."""
    print(f"Your favorite color is {color}! That's a great choice.")

def main():
    """Gets the favorite color from the user and prints the message."""
    color = input("What is your favorite color? ")
    print_color_message(color)

if __name__ == "__main__":
    main()
```
