## Exercise
Write a function to find the union of two arrays.

## Reference Solution
```python
def union_of_arrays(arr1, arr2):
    """
    Returns the union of two arrays.

    Parameters:
    arr1 (list): The first array.
    arr2 (list): The second array.

    Returns:
    list: The union of the two arrays.
    """
    return list(set(arr1 + arr2))
```
