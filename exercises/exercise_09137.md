## Exercise
Write a program that takes a student's score as input and checks if the student has passed or failed. If the score is 50 or above, the student has passed; otherwise, the student has failed.

## Reference Solution
```python
score = int(input("Enter score: "))

if score >= 50:
    print("You passed")
else:
    print("You failed")
```
