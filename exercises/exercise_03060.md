## Exercise
Create a function to memoize a given function, storing its results to avoid redundant calculations.

## Reference Solution
```python
def memoize(func):
    """
    A decorator function to memoize a given function, storing its results to avoid redundant calculations.

    Args:
        func (function): The function to be memoized.

    Returns:
        function: A memoized version of the input function.
    """

    # Create a dictionary to store the results of the function for different inputs
    cache = {}

    # Define a wrapper function that will be returned by the decorator
    def wrapper(*args, **kwargs):
        
        # Create a key based on the arguments passed to the function
        key = str(args) + str(kwargs)
        
        # Check if the result for the given key is already in the cache
        if key in cache:
            # If it is, return the cached result
            return cache[key]
        else:
            # If not, calculate the result using the original function
            result = func(*args, **kwargs)
            
            # Store the result in the cache for future use
            cache[key] = result
            
            # Return the result
            return result

    # Return the wrapper function
    return wrapper


# Example usage:

# Define a function to be memoized
@memoize
def add(a, b):
    print("Calculating...")
    return a + b

# Test the memoized function
print(add(2, 3))  # Calculating... and then prints 5
print(add(2, 3))  # Prints 5 without calculating
```
