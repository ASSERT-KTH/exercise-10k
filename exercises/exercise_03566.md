## Exercise
Write a function that maps a dictionary to a new dictionary with transformed values

## Reference Solution
```python
def transform_dict_values(input_dict, transform_func):
    """
    Maps a dictionary to a new dictionary with transformed values.

    Args:
        input_dict (dict): The input dictionary.
        transform_func (function): A function that takes a value and returns a transformed value.

    Returns:
        dict: A new dictionary with the transformed values.
    """
    return {key: transform_func(value) for key, value in input_dict.items()}

# Example usage
if __name__ == "__main__":
    # Define a transformation function
    def double_value(x):
        return x * 2

    # Create an input dictionary
    input_dict = {"a": 1, "b": 2, "c": 3}

    # Transform the dictionary values
    transformed_dict = transform_dict_values(input_dict, double_value)

    # Print the transformed dictionary
    print(transformed_dict)  # Output: {'a': 2, 'b': 4, 'c': 6}
```
