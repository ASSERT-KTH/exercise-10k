## Exercise
Write a function that takes a list as input and returns a new list containing the same elements in a random order.

## Reference Solution
```python
import random

def randomize_list(input_list):
    """
    Returns a new list containing the same elements as the input list, 
    but in a random order.

    Args:
        input_list (list): The list to be randomized.

    Returns:
        list: A new list with the same elements in a random order.
    """
    # Create a copy of the input list to avoid modifying the original list
    list_copy = input_list.copy()
    
    # Use the random.shuffle function to randomize the order of elements in the copied list
    random.shuffle(list_copy)
    
    # Return the randomized list
    return list_copy

# Example usage:
if __name__ == "__main__":
    # Define a sample list
    sample_list = [1, 2, 3, 4, 5]
    print("Original List:", sample_list)

    # Randomize the list
    randomized_list = randomize_list(sample_list)
    print("Randomized List:", randomized_list)
```
