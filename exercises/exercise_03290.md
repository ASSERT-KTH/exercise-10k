## Exercise
Write a program that finds all prime numbers up to a given number using a for loop.

## Reference Solution
```python
def find_primes(n):
    primes = []
    for possiblePrime in range(2, n + 1):
        isPrime = True
        for num in range(2, int(possiblePrime ** 0.5) + 1):
            if possiblePrime % num == 0:
                isPrime = False
                break
        if isPrime:
            primes.append(possiblePrime)
    return primes

n = int(input("Enter a number: "))
print("Prime numbers up to", n, "are:")
print(find_primes(n))
```
