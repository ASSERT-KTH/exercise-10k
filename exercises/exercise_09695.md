## Exercise
Create a program that takes in a user's name, age, and location as input, then prints out a formatted string containing this information.

## Reference Solution
```python
def print_user_info():
    # Get user input
    name = input("Please enter your name: ")
    age = int(input("Please enter your age: "))
    location = input("Please enter your location: ")

    # Print out the user information in a formatted string
    print(f"Hello, my name is {name}, I am {age} years old, and I am from {location}.")

# Call the function to start the program
print_user_info()
```
