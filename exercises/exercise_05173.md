## Exercise
Write a function to count the number of keys in a dictionary.

## Reference Solution
```python
def count_dict_keys(dictionary):
    """
    Returns the number of keys in a dictionary.

    Args:
        dictionary (dict): The dictionary to count keys from.

    Returns:
        int: The number of keys in the dictionary.
    """
    return len(dictionary)

# Example usage
my_dict = {"apple": 1, "banana": 2, "cherry": 3}
key_count = count_dict_keys(my_dict)
print(f"Number of keys: {key_count}")
```
