## Exercise
Write a program that takes an integer as input and prints whether it is even or odd.

## Reference Solution
```python
def check_even_odd(num):
    if num % 2 == 0:
        print(f"{num} is even.")
    else:
        print(f"{num} is odd.")

num = int(input("Enter an integer: "))
check_even_odd(num)
```

```python
while True:
    try:
        num = int(input("Enter an integer: "))
        check_even_odd(num)
        break
    except ValueError:
        print("Invalid input. Please enter a valid integer.")
```
