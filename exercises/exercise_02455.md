## Exercise
Write a program that takes two arrays as input and returns a new array containing the elements from both arrays interleaved.

## Reference Solution
```python
def interleave(arr1, arr2):
    result = []
    while arr1 and arr2:
        result.append(arr1.pop(0))
        result.append(arr2.pop(0))
    result += arr1
    result += arr2
    return result

# Test the function
arr1 = [1, 2, 3]
arr2 = ['a', 'b', 'c']
print(interleave(arr1, arr2))  # Output: [1, 'a', 2, 'b', 3, 'c']
```
