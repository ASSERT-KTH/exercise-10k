## Exercise
Develop a program to validate a PIN code. The PIN code should be 4 digits long and should not contain any repeating digits.

## Reference Solution
```python
def validate_pin(pin_code):
    """
    Validates a PIN code.

    Args:
        pin_code (str): The PIN code to validate.

    Returns:
        bool: True if the PIN code is valid, False otherwise.
    """
    # Check if the PIN code is 4 digits long
    if len(pin_code) != 4:
        return False

    # Check if the PIN code contains only digits
    if not pin_code.isdigit():
        return False

    # Check if the PIN code contains any repeating digits
    if len(set(pin_code)) != 4:
        return False

    return True


def main():
    pin_code = input("Enter a 4-digit PIN code: ")
    if validate_pin(pin_code):
        print("Valid PIN code")
    else:
        print("Invalid PIN code")


if __name__ == "__main__":
    main()
```
