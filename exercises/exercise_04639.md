## Exercise
Write a program that takes an array as input, and returns the reversed array.

## Reference Solution
```python
def reverse_array(array):
    """
    Reverses the input array.

    Args:
        array (list): The input array to be reversed.

    Returns:
        list: The reversed array.
    """
    return array[::-1]

# Example usage:
def main():
    array = [1, 2, 3, 4, 5]
    print("Original array:", array)
    reversed_array = reverse_array(array)
    print("Reversed array:", reversed_array)

if __name__ == "__main__":
    main()
```
