## Exercise
Create a program that reads a text from the user and prints out the text with all vowels removed.

## Reference Solution
```python
def remove_vowels(text):
    """
    Removes all vowels from the given text.
    
    Args:
        text (str): The input text.
    
    Returns:
        str: The text without vowels.
    """
    vowels = 'aeiouAEIOU'
    return ''.join([char for char in text if char not in vowels])

def main():
    # Read text from the user
    text = input("Please enter a text: ")
    
    # Remove vowels from the text
    text_without_vowels = remove_vowels(text)
    
    # Print the text without vowels
    print("Text without vowels:")
    print(text_without_vowels)

if __name__ == "__main__":
    main()
```
