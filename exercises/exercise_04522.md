## Exercise
Write a program that takes a list and two indices as input and returns a slice of the list from the start index to the end index.

## Reference Solution
```python
def list_slice(input_list, start, end):
    return input_list[start:end]

# Example usage:
my_list = [1, 2, 3, 4, 5, 6]
print(list_slice(my_list, 1, 4))  # Output: [2, 3, 4]
```
