## Exercise
Create a program that takes in two numbers and an operator (+, -, *, /) as input from the user, then performs the specified operation and prints out the result.

## Reference Solution
```python
def calculator():
    # Take in two numbers and an operator from the user
    num1 = float(input("Enter the first number: "))
    operator = input("Enter the operator (+, -, *, /): ")
    num2 = float(input("Enter the second number: "))

    # Perform the specified operation
    if operator == "+":
        result = num1 + num2
    elif operator == "-":
        result = num1 - num2
    elif operator == "*":
        result = num1 * num2
    elif operator == "/":
        # Check for division by zero
        if num2 != 0:
            result = num1 / num2
        else:
            print("Error: Division by zero is not allowed.")
            return
    else:
        print("Error: Invalid operator. Please enter one of the following: +, -, *, /")
        return

    # Print out the result
    print(f"{num1} {operator} {num2} = {result}")

# Call the function
calculator()
```
