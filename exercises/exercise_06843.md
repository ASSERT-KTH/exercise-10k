## Exercise
Write a program that asks the user to input a value and then checks the data type of the input value. If the input is an integer, print 'Integer'. If the input is a float, print 'Float'. If the input is a string, print 'String'.

## Reference Solution
```python
# Prompt the user to enter a value
value = input("Enter a value: ")

# Try to convert the value to an integer
try:
    int(value)
    print("Integer")
except ValueError:
    # If conversion to integer fails, try to convert to float
    try:
        float(value)
        print("Float")
    except ValueError:
        # If conversion to float fails, it's a string
        print("String")
```
