## Exercise
Write a program that takes two strings as input and returns True if they are anagrams of each other and False otherwise.

## Reference Solution
```python
def are_anagrams(s1, s2):
    if len(s1) != len(s2):
        return False

    s1 = sorted(s1)
    s2 = sorted(s2)

    for i in range(len(s1)):
        if s1[i] != s2[i]:
            return False

    return True

# Example usage
s1 = "listen"
s2 = "silent"
print(are_anagrams(s1, s2))  # Output: True
```
