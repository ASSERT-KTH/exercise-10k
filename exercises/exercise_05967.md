## Exercise
Write a program that takes an array and a mapping function as input, and returns a new array that contains the results of applying the mapping function to each element of the input array.

## Reference Solution
```python
const map = (array, mapping) => {
  const result = [];
  for (let i = 0; i < array.length; i++) {
    result.push(mapping(array[i], i));
  }
  return result;
};
```
