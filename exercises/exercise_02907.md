## Exercise
Write a program to split a given string into substrings based on a delimiter.

## Reference Solution
```python
def split_string(input_str, delimiter):
    """
    Splits a given string into substrings based on a delimiter.

    Args:
        input_str (str): The input string to be split.
        delimiter (str): The delimiter used to split the string.

    Returns:
        list: A list of substrings.
    """
    return input_str.split(delimiter)

def main():
    # Example usage
    input_str = "apple,banana,cherry,date"
    delimiter = ","
    substrings = split_string(input_str, delimiter)
    
    print("Input String:", input_str)
    print("Delimiter:", delimiter)
    print("Substrings:", substrings)

if __name__ == "__main__":
    main()
```
