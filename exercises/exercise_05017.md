## Exercise
Write a program that formats the string 'Hello, {}!' with the name 'John'. Print the formatted string to the console.

## Reference Solution
```python
def print_formatted_string(name):
    # Use string formatting to insert the name into the string
    formatted_string = 'Hello, {}!'.format(name)
    print(formatted_string)

def main():
    name = 'John'
    print_formatted_string(name)

if __name__ == "__main__":
    main()
```
