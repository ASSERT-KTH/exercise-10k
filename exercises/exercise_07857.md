## Exercise
Write a program that finds the maximum of three numbers using conditional statements.

## Reference Solution
```python
def find_max(a, b, c):
    if a >= b and a >= c:
        return a
    elif b >= a and b >= c:
        return b
    else:
        return c

# Test the function
num1 = int(input("Enter the first number: "))
num2 = int(input("Enter the second number: "))
num3 = int(input("Enter the third number: "))

max_num = find_max(num1, num2, num3)
print("The maximum number is:", max_num)
```
