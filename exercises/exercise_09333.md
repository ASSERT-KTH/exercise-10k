## Exercise
Write a program to perform bitwise operations using variables, such as AND, OR, and XOR.

## Reference Solution
```python
# Define two variables
a = 5   # Binary: 101
b = 3   # Binary: 011

# Print the binary representation of the variables
print(f"Binary representation of {a}: {bin(a)[2:]}")
print(f"Binary representation of {b}: {bin(b)[2:]}")

# Perform bitwise operations
and_result = a & b
or_result = a | b
xor_result = a ^ b

# Print the results
print(f"a AND b: {and_result} ({bin(and_result)[2:]})")
print(f"a OR b: {or_result} ({bin(or_result)[2:]})")
print(f"a XOR b: {xor_result} ({bin(xor_result)[2:]})")
```
