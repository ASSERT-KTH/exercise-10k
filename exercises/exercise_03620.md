## Exercise
Write a function that uses function overloading to calculate the sum of two numbers or the concatenation of two strings

## Reference Solution
```python
from functools import singledispatch

@singledispatch
def calculate(a, b):
    """
    Calculates the sum of two numbers or the concatenation of two strings.

    Args:
        a (int or str): The first number or string.
        b (int or str): The second number or string.

    Returns:
        int or str: The sum of two numbers or the concatenation of two strings.
    """
    raise TypeError("Unsupported type")

@calculate.register
def _(a: int, b: int):
    """
    Calculates the sum of two integers.

    Args:
        a (int): The first integer.
        b (int): The second integer.

    Returns:
        int: The sum of two integers.
    """
    return a + b

@calculate.register
def _(a: str, b: str):
    """
    Concatenates two strings.

    Args:
        a (str): The first string.
        b (str): The second string.

    Returns:
        str: The concatenation of two strings.
    """
    return a + b

# Example usage:
print(calculate(1, 2))  # Outputs: 3
print(calculate("Hello, ", "World!"))  # Outputs: Hello, World!
```
