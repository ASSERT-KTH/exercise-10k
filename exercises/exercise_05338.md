## Exercise
Declare a string variable with the value 'madam'. Check if the string is a palindrome and print 'Yes' if it is, 'No' otherwise.

## Reference Solution
```python
def check_palindrome(input_string):
    """Check if a given string is a palindrome."""
    # Remove any leading or trailing whitespace and convert to lowercase
    input_string = input_string.strip().lower()
    
    # Compare the string with its reverse
    if input_string == input_string[::-1]:
        return 'Yes'
    else:
        return 'No'

def main():
    # Declare a string variable with the value 'madam'
    string_variable = 'madam'
    
    # Check if the string is a palindrome and print the result
    print(check_palindrome(string_variable))

if __name__ == "__main__":
    main()
```
