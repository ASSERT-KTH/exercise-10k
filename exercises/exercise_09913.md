## Exercise
Write a program that responds to basic user queries (e.g. hello, goodbye, thanks).

## Reference Solution
```python
class QueryResponder:
    def __init__(self):
        self.responses = {
            'hello': 'Hi, how can I assist you today?',
            'hi': 'Hello! What can I help you with?',
            'goodbye': 'Goodbye! It was nice chatting with you.',
            'bye': 'See you later!',
            'thanks': 'You\'re welcome!',
            'thank you': 'You\'re welcome!',
        }

    def respond(self, query):
        """Respond to a user query."""
        query = query.lower()
        if query in self.responses:
            return self.responses[query]
        else:
            return 'Sorry, I didn\'t understand that.'

def main():
    responder = QueryResponder()
    print('Welcome to the query responder! Type "quit" to exit.')
    while True:
        query = input('You: ')
        if query.lower() == 'quit':
            break
        print('Responder:', responder.respond(query))

if __name__ == '__main__':
    main()
```
