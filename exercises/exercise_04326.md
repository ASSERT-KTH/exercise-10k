## Exercise
Write a program to find the minimum element in an array.

## Reference Solution
```python
def find_min_element(array):
    """
    This function finds the minimum element in an array.

    Args:
        array (list): A list of integers.

    Returns:
        int: The minimum element in the array.
    """
    if len(array) == 0:
        raise ValueError("Array is empty")

    min_element = array[0]
    for element in array:
        if element < min_element:
            min_element = element

    return min_element


# Example usage:
if __name__ == "__main__":
    array = [10, 20, 3, 40, 5]
    print("Array:", array)
    print("Minimum element:", find_min_element(array))
```
