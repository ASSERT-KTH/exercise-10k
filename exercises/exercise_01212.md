## Exercise
Write a function that finds the index of a given element in a list.

## Reference Solution
```python
def find_index(lst, element):
    """
    Finds the index of a given element in a list.

    Args:
        lst (list): The list to search.
        element: The element to find.

    Returns:
        int: The index of the element if found, -1 otherwise.
    """
    try:
        return lst.index(element)
    except ValueError:
        return -1

# Example usage:
my_list = [1, 2, 3, 4, 5]
print(find_index(my_list, 3))  # Output: 2
print(find_index(my_list, 6))  # Output: -1
```
