## Exercise
Write a program that takes a user input and checks its data type. If the input is a string, check if it can be converted to an integer or a float. If it can, print the converted value. If not, print the original string.

## Reference Solution
```python
x = input("enter a string: ")

if not isinstance(x, str):
    print("Error: input is not a string")
else:
    try:
        y = int(x)
        print(f"converted to int: {y}")
    except ValueError:
        try:
            z = float(x)
            print(f"converted to float: {z}")
        except ValueError:
            print(f"original string: {x}")
```
