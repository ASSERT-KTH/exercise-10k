## Exercise
Create a program that takes two numbers as input and returns the percent increase from the first number to the second number.

## Reference Solution
```python
def calculate_percent_increase(initial_value, final_value):
    """
    Calculates the percent increase from the initial value to the final value.

    Args:
        initial_value (float): The initial value.
        final_value (float): The final value.

    Returns:
        float: The percent increase.
    """
    if initial_value == 0:
        raise ValueError("Initial value cannot be zero.")

    percent_increase = ((final_value - initial_value) / initial_value) * 100
    return percent_increase

def main():
    # Get user input
    initial_value = float(input("Enter the initial value: "))
    final_value = float(input("Enter the final value: "))

    # Calculate percent increase
    try:
        percent_increase = calculate_percent_increase(initial_value, final_value)
        print(f"The percent increase is: {percent_increase:.2f}%")
    except ValueError as e:
        print(e)

if __name__ == "__main__":
    main()
```
