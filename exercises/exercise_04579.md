## Exercise
Write a function that takes a dictionary and a condition, then returns two dictionaries with the key-value pairs that meet and do not meet the condition

## Reference Solution
```python
def split_dict(input_dict, condition):
    """
    Splits a dictionary into two dictionaries based on a given condition.

    Args:
        input_dict (dict): The input dictionary.
        condition (function): A function that takes a key-value pair and returns a boolean.

    Returns:
        tuple: Two dictionaries. The first dictionary contains key-value pairs that meet the condition,
               and the second dictionary contains key-value pairs that do not meet the condition.
    """

    meet_condition = {key: value for key, value in input_dict.items() if condition(key, value)}
    do_not_meet_condition = {key: value for key, value in input_dict.items() if not condition(key, value)}

    return meet_condition, do_not_meet_condition


# Example usage
if __name__ == "__main__":
    # Define a sample dictionary
    sample_dict = {
        "a": 1,
        "b": 2,
        "c": 3,
        "d": 4,
        "e": 5
    }

    # Define a condition function
    def is_even(key, value):
        return value % 2 == 0

    # Split the dictionary
    meet, do_not_meet = split_dict(sample_dict, is_even)

    # Print the results
    print("Key-value pairs that meet the condition:")
    print(meet)
    print("\nKey-value pairs that do not meet the condition:")
    print(do_not_meet)
```
