## Exercise
Calculate the average grade of a student given a list of scores.

## Reference Solution
```python
def calculate_average_grade(scores):
    """
    Calculate the average grade given a list of scores.

    Args:
        scores (list): A list of scores.

    Returns:
        float: The average grade.
    """
    if not scores:
        return 0

    return sum(scores) / len(scores)

# Example usage
scores = [85, 90, 78, 92, 88]
average_grade = calculate_average_grade(scores)
print("Average Grade:", average_grade)
```
