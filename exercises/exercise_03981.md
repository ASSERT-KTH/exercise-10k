## Exercise
Write a program that takes a string input from the user and prints the reversed string.

## Reference Solution
```python
def reverse_string(input_str):
    """
    This function takes a string as input and returns the reversed string.
    
    Parameters:
    input_str (str): The input string to be reversed.
    
    Returns:
    str: The reversed input string.
    """
    return input_str[::-1]

def main():
    # Get user input
    user_input = input("Please enter a string: ")
    
    # Reverse the input string
    reversed_str = reverse_string(user_input)
    
    # Print the reversed string
    print("The reversed string is: ", reversed_str)

if __name__ == "__main__":
    main()
```
