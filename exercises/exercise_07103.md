## Exercise
Write a function that takes a list of integers as input and returns the reversed list. The function should not use any built-in list reversal methods.

def reverse_list(input_list):
    """
    This function takes a list of integers as input and returns the reversed list.
    
    The function does not use any built-in list reversal methods. Instead, it uses a simple iterative approach to reverse the list.

    Args:
        input_list (list): A list of integers.

    Returns:
        list: The reversed list of integers.
    """

    # Initialize an empty list to store the reversed elements
    reversed_list = []

    # Iterate over the input list in reverse order
    # We use the range function to generate indices from the last index to the first index
    for i in range(len(input_list) - 1, -1, -1):
        
        # Append each element from the input list to the reversed list
        # This effectively reverses the order of the elements
        reversed_list.append(input_list[i])

    # Return the reversed list
    return reversed_list

# Example usage:
numbers = [1, 2, 3, 4, 5]
print(reverse_list(numbers))  # Output: [5, 4, 3, 2, 1]
