## Exercise
Create a dictionary with 3 key-value pairs and copy the dictionary.

## Reference Solution
```python
# Create a dictionary with 3 key-value pairs
original_dict = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

# Print the original dictionary
print("Original Dictionary:")
print(original_dict)

# Copy the dictionary
copied_dict = original_dict.copy()

# Print the copied dictionary
print("\nCopied Dictionary:")
print(copied_dict)

# Modify the original dictionary to demonstrate that the copy is independent
original_dict["age"] = 31

# Print the original and copied dictionaries after modification
print("\nAfter Modifying the Original Dictionary:")
print("Original Dictionary:")
print(original_dict)
print("Copied Dictionary:")
print(copied_dict)
```
