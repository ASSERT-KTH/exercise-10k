## Exercise
Write a program that checks the data type of a given variable and prints its type.

## Reference Solution
```python
def check_data_type(var):
    """
    Checks the data type of a given variable and prints its type.
    
    Parameters:
    var (any): The variable to check the data type of.
    """
    print(f"The data type of the variable is: {type(var)}")


# Test the function with different data types
if __name__ == "__main__":
    check_data_type(10)  # integer
    check_data_type(10.5)  # float
    check_data_type("Hello, World!")  # string
    check_data_type(True)  # boolean
    check_data_type([1, 2, 3])  # list
    check_data_type((1, 2, 3))  # tuple
    check_data_type({"name": "John", "age": 30})  # dictionary
    check_data_type({1, 2, 3})  # set
```
