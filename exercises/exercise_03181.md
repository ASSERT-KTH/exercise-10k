## Exercise
Create a program that takes a math operation and two numbers as input. Use if/else statements to perform the correct operation and print the result.

## Reference Solution
```python
def math_operation():
    # Get user input
    num1 = float(input("Enter the first number: "))
    operation = input("Enter the math operation (+, -, *, /): ")
    num2 = float(input("Enter the second number: "))

    # Perform math operation
    if operation == "+":
        result = num1 + num2
        print(f"{num1} + {num2} = {result}")
    elif operation == "-":
        result = num1 - num2
        print(f"{num1} - {num2} = {result}")
    elif operation == "*":
        result = num1 * num2
        print(f"{num1} * {num2} = {result}")
    elif operation == "/":
        if num2 != 0:
            result = num1 / num2
            print(f"{num1} / {num2} = {result}")
        else:
            print("Error: Division by zero is not allowed.")
    else:
        print("Error: Invalid math operation.")

# Call the function
math_operation()
```
