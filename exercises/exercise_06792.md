## Exercise
Create a program that takes in two numbers and an arithmetic operation (+, -, *, /) as input, and then performs the specified operation on the two numbers and outputs the result.

## Reference Solution
```python
def arithmetic_operation(num1, num2, operation):
    """
    This function performs an arithmetic operation on two numbers.

    Args:
        num1 (float): The first number.
        num2 (float): The second number.
        operation (str): The arithmetic operation to be performed. It can be '+', '-', '*', or '/'.

    Returns:
        float: The result of the arithmetic operation.
    """

    # Check if the operation is valid
    if operation not in ['+', '-', '*', '/']:
        raise ValueError("Invalid operation. Operation must be '+', '-', '*', or '/'.")

    # Perform the operation
    if operation == '+':
        result = num1 + num2
    elif operation == '-':
        result = num1 - num2
    elif operation == '*':
        result = num1 * num2
    elif operation == '/':
        # Check for division by zero
        if num2 == 0:
            raise ZeroDivisionError("Cannot divide by zero.")
        result = num1 / num2

    return result


def main():
    # Get the numbers and operation from the user
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))
    operation = input("Enter the operation (+, -, *, /): ")

    # Perform the operation and print the result
    try:
        result = arithmetic_operation(num1, num2, operation)
        print(f"{num1} {operation} {num2} = {result}")
    except (ValueError, ZeroDivisionError) as e:
        print(e)


if __name__ == "__main__":
    main()
```
