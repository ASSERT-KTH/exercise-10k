## Exercise
Create a grouped dictionary and ungroup it into a single-level dictionary

## Reference Solution
```python
# Grouped dictionary creation
grouped_dict = {
    'Group1': {'a': 1, 'b': 2},
    'Group2': {'c': 3, 'd': 4},
    'Group3': {'e': 5, 'f': 6}
}

# Print the grouped dictionary
print("Grouped Dictionary:")
print(grouped_dict)

# Ungroup the dictionary
ungrouped_dict = {f"{key}_{subkey}": value for key, subdict in grouped_dict.items() for subkey, value in subdict.items()}

# Print the ungrouped dictionary
print("\nUngrouped Dictionary:")
print(ungrouped_dict)
```
