## Exercise
Write a function to update a value in a dictionary by its key

## Reference Solution
```python
def update_dict_value(dictionary, key, new_value):
    """
    Updates a value in a dictionary by its key.

    Args:
        dictionary (dict): The dictionary to be updated.
        key (any): The key of the value to be updated.
        new_value (any): The new value to be assigned to the key.

    Returns:
        dict: The updated dictionary.
    """
    dictionary[key] = new_value
    return dictionary

# Example usage:
if __name__ == "__main__":
    example_dict = {"name": "John", "age": 30}
    print("Original Dictionary:", example_dict)

    updated_dict = update_dict_value(example_dict, "age", 31)
    print("Updated Dictionary:", updated_dict)

    # Adding a new key-value pair
    updated_dict = update_dict_value(updated_dict, "city", "New York")
    print("Updated Dictionary with new key-value pair:", updated_dict)
```
