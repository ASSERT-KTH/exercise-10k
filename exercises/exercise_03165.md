## Exercise
Develop a function that calculates the sum of all elements in an array.

## Reference Solution
```python
def calculate_array_sum(array):
    """
    Calculate the sum of all elements in an array.

    Args:
        array (list): A list of numbers.

    Returns:
        int: The sum of all elements in the array.

    Raises:
        TypeError: If the input is not a list.
        ValueError: If the list contains non-numeric values.
    """
    if not isinstance(array, list):
        raise TypeError("Input must be a list.")

    try:
        return sum(array)
    except TypeError:
        raise ValueError("List must only contain numeric values.")

# Example usage:
def main():
    array = [1, 2, 3, 4, 5]
    print(calculate_array_sum(array))  # Output: 15

if __name__ == "__main__":
    main()
```
