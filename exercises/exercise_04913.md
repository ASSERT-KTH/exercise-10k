## Exercise
Write a program that takes a person's weight and height as input and calculates their BMI. Based on the BMI, print the corresponding weight status (Underweight, Normal, Overweight, Obese).

## Reference Solution
```python
def calculate_bmi(weight, height):
    """
    Calculate the BMI of a person.

    Args:
        weight (float): Weight in kg.
        height (float): Height in meters.

    Returns:
        float: BMI value.
    """
    bmi = weight / (height ** 2)
    return bmi

def get_weight_status(bmi):
    """
    Determine the weight status based on the BMI.

    Args:
        bmi (float): BMI value.

    Returns:
        str: Weight status.
    """
    if bmi < 18.5:
        return "Underweight"
    elif 18.5 <= bmi < 25:
        return "Normal"
    elif 25 <= bmi < 30:
        return "Overweight"
    else:
        return "Obese"

def main():
    # Get user input
    weight = float(input("Enter your weight in kg: "))
    height = float(input("Enter your height in meters: "))

    # Calculate BMI
    bmi = calculate_bmi(weight, height)

    # Determine weight status
    weight_status = get_weight_status(bmi)

    # Print results
    print(f"Your BMI is: {bmi:.2f}")
    print(f"Your weight status is: {weight_status}")

if __name__ == "__main__":
    main()
```
