## Exercise
Develop a program that takes in the student's name, score, and attendance percentage, then prints out the grade report. If the score is above 80 and attendance percentage is above 90, print 'Excellent', otherwise print 'Good' or 'Fair'

## Reference Solution
```python
def calculate_grade(score, attendance):
    """
    Calculate the grade based on score and attendance.

    Args:
    score (float): The student's score.
    attendance (float): The student's attendance percentage.

    Returns:
    str: The grade ('Excellent', 'Good', or 'Fair').
    """
    if score > 80 and attendance > 90:
        return 'Excellent'
    elif score > 70 or attendance > 80:
        return 'Good'
    else:
        return 'Fair'


def print_grade_report(name, score, attendance):
    """
    Print the grade report for a student.

    Args:
    name (str): The student's name.
    score (float): The student's score.
    attendance (float): The student's attendance percentage.
    """
    grade = calculate_grade(score, attendance)
    print(f"Name: {name}")
    print(f"Score: {score}")
    print(f"Attendance: {attendance}%")
    print(f"Grade: {grade}")


def main():
    name = input("Enter the student's name: ")
    score = float(input("Enter the student's score: "))
    attendance = float(input("Enter the student's attendance percentage: "))

    print_grade_report(name, score, attendance)


if __name__ == "__main__":
    main()
```
