## Exercise
Write a program that uses a for loop to print the numbers from 1 to 10.

## Reference Solution
```python
def print_numbers():
    """Prints numbers from 1 to 10 using a for loop."""
    for i in range(1, 11):
        print(i)

# Example usage:
if __name__ == "__main__":
    print_numbers()
```
