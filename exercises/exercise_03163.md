## Exercise
Create a function that determines whether a given number is even or odd.

## Reference Solution
```python
def is_even_or_odd(num: int) -> str:
    """
    Returns 'even' if the number is even, 'odd' otherwise.

    Args:
        num (int): The number to check.

    Returns:
        str: 'even' or 'odd'.
    """
    if num % 2 == 0:
        return 'even'
    else:
        return 'odd'

# Example usage
def main():
    numbers = [10, 11, 20, 21, 30]
    for num in numbers:
        print(f"{num} is {is_even_or_odd(num)}")

if __name__ == "__main__":
    main()
```
