## Exercise
Find the common keys between two dictionaries

## Reference Solution
```python
def find_common_keys(dict1, dict2):
    """
    Find the common keys between two dictionaries.

    Args:
        dict1 (dict): The first dictionary.
        dict2 (dict): The second dictionary.

    Returns:
        set: A set of common keys between the two dictionaries.
    """
    return set(dict1.keys()).intersection(set(dict2.keys()))

# Example usage:
dict1 = {'a': 1, 'b': 2, 'c': 3}
dict2 = {'b': 20, 'c': 30, 'd': 40}

common_keys = find_common_keys(dict1, dict2)
print(common_keys)  # Output: {'b', 'c'}
```
