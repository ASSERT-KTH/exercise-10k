## Exercise
Create a function that takes two arrays as input, and returns a new array containing the elements of the input arrays, paired together in tuples.

## Reference Solution
```python
def pair_elements(array1, array2):
    """
    This function takes two arrays as input and returns a new array containing the elements of the input arrays, 
    paired together in tuples. If the arrays are of different lengths, the extra elements from the longer array 
    will be ignored.

    Args:
        array1 (list): The first input array.
        array2 (list): The second input array.

    Returns:
        list: A new array containing the paired elements from the input arrays.
    """

    # We will use the built-in zip function in Python, which makes an iterator that aggregates elements from each of the iterables.
    # Stop when the shortest input iterable is exhausted.
    paired_array = list(zip(array1, array2))
    
    # Return the resulting array.
    return paired_array

# Test the function
array1 = [1, 2, 3]
array2 = ['a', 'b', 'c']
print(pair_elements(array1, array2))  # Output: [(1, 'a'), (2, 'b'), (3, 'c')]
```
