## Exercise
Create a function that takes a number as an argument and returns its absolute value.

## Reference Solution

```python
# Define a function named 'absolute_value' that takes one argument 'num'.
def absolute_value(num):
    """
    This function calculates the absolute value of a given number.

    Args:
        num (int or float): The number for which the absolute value is to be calculated.

    Returns:
        int or float: The absolute value of the input number.
    """

    # Check if the input number is less than 0, because absolute value of a negative number is its positive counterpart.
    if num < 0:
        # If the number is negative, multiply it by -1 to make it positive.
        return -num
    
    # If the number is not negative, its absolute value is the number itself.
    return num

# Test the function with some examples.
print(absolute_value(-5))  # Output: 5
print(absolute_value(10))  # Output: 10
print(absolute_value(-3.5))  # Output: 3.5
```
