## Exercise
Create an expression with bitwise operators and evaluate it, then print out the result.

## Reference Solution
```python
# Define the variables
a = 5   # Binary: 101
b = 3   # Binary: 011

# Perform bitwise operations
result_and = a & b  # Binary AND: 101 & 011 = 001
result_or = a | b   # Binary OR: 101 | 011 = 111
result_xor = a ^ b  # Binary XOR: 101 ^ 011 = 110
result_shift_left = a << 1  # Binary left shift: 101 << 1 = 1010
result_shift_right = a >> 1  # Binary right shift: 101 >> 1 = 10

# Print the results
print("Bitwise AND:", result_and)
print("Bitwise OR:", result_or)
print("Bitwise XOR:", result_xor)
print("Left Shift:", result_shift_left)
print("Right Shift:", result_shift_right)
```
