## Exercise
Write a program that takes a string as input and returns True if the string contains only alphanumeric characters and False otherwise.

## Reference Solution
```python
def check_if_alphanumeric(data):
    """
    Checks if a given string contains only alphanumeric characters.
    """
    for char in data:
        if not char.isalnum():
            return False
    return True

# Test the function
print(check_if_alphanumeric("HelloWorld"))  # Expected output: True
print(check_if_alphanumeric("Hello World"))  # Expected output: False
```
