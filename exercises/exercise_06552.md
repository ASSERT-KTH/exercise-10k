## Exercise
Write a function that converts a given string to camel case format.

## Reference Solution
```python
def convert_to_camel_case(input_string):
    """
    Converts a given string to camel case format.

    Args:
        input_string (str): The input string to be converted.

    Returns:
        str: The input string in camel case format.
    """
    words = input_string.split()
    camel_case_string = words[0].lower() + ''.join(word.capitalize() for word in words[1:])
    return camel_case_string

# Example usage:
print(convert_to_camel_case("hello world"))  # Output: "helloWorld"
print(convert_to_camel_case("this is a test"))  # Output: "thisIsATest"
```
