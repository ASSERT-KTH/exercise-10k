## Exercise
Create a program that checks whether a given number is even or odd.

## Reference Solution
```python
def check_parity(num):
    """
    Checks whether a given number is even or odd.

    Args:
        num (int): The number to check.

    Returns:
        str: "Even" if the number is even, "Odd" otherwise.
    """
    if num % 2 == 0:
        return "Even"
    else:
        return "Odd"

def main():
    # Example usage
    num = int(input("Enter a number: "))
    result = check_parity(num)
    print(f"The number {num} is {result}.")

if __name__ == "__main__":
    main()
```
