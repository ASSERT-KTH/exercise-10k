## Exercise
Create a function that splits a given string into substrings separated by a given delimiter.

## Reference Solution
```python
def split_string(input_string, delimiter):
    """
    Splits a given string into substrings separated by a given delimiter.

    Args:
        input_string (str): The string to be split.
        delimiter (str): The delimiter to split the string by.

    Returns:
        list: A list of substrings separated by the delimiter.
    """
    return input_string.split(delimiter)

# Example usage:
if __name__ == "__main__":
    input_str = "apple,banana,cherry"
    delimiter = ","
    print(split_string(input_str, delimiter))  # Output: ['apple', 'banana', 'cherry']
```
