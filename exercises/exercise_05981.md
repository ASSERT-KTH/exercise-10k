## Exercise
Create a function that takes a single argument and returns its double value

## Reference Solution
```python
def double_value(x):
    """
    Returns the double value of the input.

    Args:
        x (int or float): The input value.

    Returns:
        int or float: The double value of the input.
    """
    return x * 2
```

```python
print(double_value(5))   # Output: 10
print(double_value(3.5)) # Output: 7.0
```
