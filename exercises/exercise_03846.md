## Exercise
Write a program that uses a for loop to print all prime numbers up to a given number.

## Reference Solution
```python
def print_primes(n):
    """Prints all prime numbers up to n."""
    for num in range(2, n + 1):
        is_prime = True
        for i in range(2, int(num ** 0.5) + 1):
            if num % i == 0:
                is_prime = False
                break
        if is_prime:
            print(num)

# Example usage:
print_primes(100)
```
