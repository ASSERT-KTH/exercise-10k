## Exercise
Write a program that takes in a list of variables with their respective data types and validates if the provided value matches the given data type. The data types to consider are integers, floats, and strings. If the value matches the data type, print 'Valid'; otherwise, print 'Invalid'.

## Reference Solution
```python
def validate_data_type(variables):
    """
    Validate if the provided value matches the given data type.

    Args:
    variables (dict): A dictionary where keys are variable names and values are tuples of (value, data_type).

    Returns:
    None
    """
    for variable, (value, data_type) in variables.items():
        try:
            if data_type == 'int':
                int(value)
            elif data_type == 'float':
                float(value)
            elif data_type == 'str':
                str(value)
            else:
                print(f"Invalid data type for variable {variable}. Supported data types are 'int', 'float', and 'str'.")
                continue

            print(f"Variable {variable}: Valid")
        except ValueError:
            print(f"Variable {variable}: Invalid")


# Example usage
variables = {
    'var1': ('10', 'int'),
    'var2': ('10.5', 'float'),
    'var3': ('hello', 'str'),
    'var4': ('10.5', 'int'),
    'var5': ('hello', 'float'),
    'var6': ('123', 'str')  # This will be valid because all values can be converted to string
}

validate_data_type(variables)
```
