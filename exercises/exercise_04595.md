## Exercise
Create a function that takes a number as input and returns all prime numbers up to that number

## Reference Solution
```python
def get_prime_numbers(n):
    """
    Returns a list of all prime numbers up to n.

    Args:
    n (int): The upper limit for generating prime numbers.

    Returns:
    list: A list of prime numbers.
    """
    sieve = [True] * (n + 1)
    sieve[0:2] = [False, False]  # 0 and 1 are not prime numbers

    # Iterate from 2 to sqrt(n)
    for current_prime in range(2, int(n**0.5) + 1):
        if sieve[current_prime]:
            # Mark multiples of current_prime as not prime
            for multiple in range(current_prime**2, n + 1, current_prime):
                sieve[multiple] = False

    # Return a list of all prime numbers in the range
    return [num for num, is_prime in enumerate(sieve) if is_prime]


# Example usage:
if __name__ == "__main__":
    n = 50
    prime_numbers = get_prime_numbers(n)
    print(f"Prime numbers up to {n}: {prime_numbers}")
```
