## Exercise
Create a dictionary with 5 key-value pairs and print out the values for given keys

## Reference Solution
```python
def main():
    # Create a dictionary
    student_grades = {
        "John": 85,
        "Alice": 92,
        "Bob": 78,
        "Charlie": 88,
        "David": 95
    }

    # Define the keys for which we want to access the values
    keys_to_access = ["John", "Alice", "Charlie"]

    # Access and print the values for the given keys
    for key in keys_to_access:
        if key in student_grades:
            print(f"The grade for {key} is: {student_grades[key]}")
        else:
            print(f"No grade found for {key}")

if __name__ == "__main__":
    main()
```
