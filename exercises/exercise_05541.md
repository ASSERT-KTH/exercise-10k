## Exercise
Write a program that prints numbers from 1 to 10 using a for loop.

## Reference Solution
```python
# Print numbers from 1 to 10 using a for loop

def print_numbers():
    # Loop through numbers from 1 to 10
    for i in range(1, 11):
        print(i)

# Call the function
print_numbers()
```
