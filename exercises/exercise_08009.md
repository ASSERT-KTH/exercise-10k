## Exercise
Create a function that validates whether a given string is valid according to specific rules. A string is valid if it starts with a vowel, ends with a consonant, and contains at least one digit. The string can contain any combination of uppercase and lowercase letters, as well as numbers. The function should return True if the string is valid and False otherwise.

## Reference Solution
```python
def validate_string(input_str):
    """
    This function validates whether a given string is valid according to specific rules.
    A string is valid if it starts with a vowel, ends with a consonant, and contains at least one digit.
    
    Parameters:
    input_str (str): The input string to be validated.
    
    Returns:
    bool: True if the string is valid, False otherwise.
    """

    # First, we check if the input string is empty
    if not input_str:
        return False  # If it's empty, the string is not valid

    # Then, we check if the string starts with a vowel
    # We use the lower() method to handle both uppercase and lowercase letters
    if input_str[0].lower() not in 'aeiou':
        return False  # If it doesn't start with a vowel, the string is not valid

    # Next, we check if the string ends with a consonant
    # We use the lower() method to handle both uppercase and lowercase letters
    # We also use the isalpha() method to check if the last character is a letter
    if not input_str[-1].isalpha() or input_str[-1].lower() in 'aeiou':
        return False  # If it doesn't end with a consonant, the string is not valid

    # Finally, we check if the string contains at least one digit
    # We use the any() function with a generator expression to check this
    if not any(char.isdigit() for char in input_str):
        return False  # If it doesn't contain a digit, the string is not valid

    # If all checks pass, the string is valid
    return True
```
