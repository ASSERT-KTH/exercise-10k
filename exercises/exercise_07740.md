## Exercise
Write a program that uses a while loop to print the numbers from 1 to 10, but only the odd numbers. After each odd number, print a message saying 'Odd number found!'

## Reference Solution
```python
def print_odd_numbers():
    i = 1
    while i <= 10:
        if i % 2 != 0:
            print(i)
            print('Odd number found!')
        i += 1

print_odd_numbers()
```
