## Exercise
Create a program that uses a for loop to print each key-value pair in an object.

## Reference Solution
```python
def print_key_value_pairs(obj):
    """
    Prints each key-value pair in the given object.
    
    Args:
        obj (dict): The object to print key-value pairs from.
    """
    for key, value in obj.items():
        print(f"Key: {key}, Value: {value}")


# Example usage
if __name__ == "__main__":
    example_object = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }

    print("Key-Value Pairs:")
    print_key_value_pairs(example_object)
```
