## Exercise
Write a program that prints the factors of a given number using a for loop.

## Reference Solution
```python
def print_factors(num):
    """
    Prints the factors of a given number.

    Args:
        num (int): The number to find factors for.
    """
    print(f"Factors of {num}:")
    for i in range(1, num + 1):
        if num % i == 0:
            print(i)

def main():
    num = int(input("Enter a number: "))
    print_factors(num)

if __name__ == "__main__":
    main()
```
