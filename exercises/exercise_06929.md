## Exercise
Create a function that takes another function as an argument and applies it to a list of numbers

## Reference Solution
```python
def apply_function(func, numbers):
    """
    Applies a given function to a list of numbers.

    Args:
        func (function): The function to be applied.
        numbers (list): A list of numbers.

    Returns:
        list: A list of results after applying the function.
    """
    return [func(num) for num in numbers]

# Example usage:
def square(x):
    """Returns the square of a number."""
    return x ** 2

numbers = [1, 2, 3, 4, 5]
result = apply_function(square, numbers)
print(result)  # Output: [1, 4, 9, 16, 25]
```
