## Exercise
Declare constant variables, assign values to them, and then print their values.

## Reference Solution
```python
from typing import Final

# Declare and assign constant variables
PI: Final[float] = 3.14159
GRAVITY: Final[float] = 9.81

def main():
    # Print the values of the constant variables
    print("Value of PI:", PI)
    print("Value of GRAVITY:", GRAVITY)

    # Attempting to reassign a constant will result in a warning
    try:
        PI = 3.14
    except AttributeError:
        print("Cannot reassign a constant variable.")

if __name__ == "__main__":
    main()
```
