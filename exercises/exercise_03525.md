## Exercise
Write a program that prints a pattern of stars (*) using a for loop.

## Reference Solution
```python
# Define the number of rows
rows = 5

# Print the pattern
for i in range(rows):
    # Print spaces
    for j in range(rows - i - 1):
        print(' ', end=' ')
    
    # Print stars
    for k in range(2 * i + 1):
        print('*', end=' ')
    
    # Move to the next line
    print()
```
