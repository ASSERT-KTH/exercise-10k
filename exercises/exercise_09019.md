## Exercise
Create a function that uses generators to produce a sequence of values on-the-fly.

## Reference Solution
```python
def fibonacci(n):
    """
    Generates the Fibonacci sequence up to the nth number.

    Args:
        n (int): The number of Fibonacci numbers to generate.

    Yields:
        int: The next number in the Fibonacci sequence.
    """
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

# Example usage:
if __name__ == "__main__":
    n = 10  # Generate the first 10 Fibonacci numbers
    print("Fibonacci sequence up to {} numbers:".format(n))
    for i, num in enumerate(fibonacci(n)):
        print("F({}) = {}".format(i, num))
```
