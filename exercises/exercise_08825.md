## Exercise
Write a program that takes a string as input and outputs the reversed string. The program should handle strings of varying lengths and containing different characters, including spaces and punctuation.

## Reference Solution
```python
def reverse_strings(string):
    reversed_string = ""
    for char in string:
        reversed_string = char + reversed_string
    return reversed_string

print(reverse_strings("Hello world!"))
```
