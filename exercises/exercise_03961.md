## Exercise
Write a program that checks whether a given number is prime or not. The program should take an integer as input and output 'Prime' if the number is prime, and 'Not Prime' otherwise.

## Reference Solution
```python
n = int(input())
if n <= 1:
    print("Not Prime")
else:
    a = [True] * (n + 1)
    a[0] = False
    a[1] = False
    p = 2
    while p * p <= n:
        if a[p]:
            for i in range(p * p, n + 1, p):
                a[i] = False
        p += 1
    if a[n]:
        print("Prime")
    else:
        print("Not Prime")
```
