## Exercise
Write a program that parses a string into an array of numbers

## Reference Solution
```python
import re

def parse_string_to_numbers(input_string):
    """
    Parse a string into an array of numbers.

    Args:
        input_string (str): The input string to parse.

    Returns:
        list: A list of numbers parsed from the input string.
    """
    # Use regular expression to find all numbers in the string
    numbers = re.findall(r"[-+]?\d*\.\d+|[-+]?\d+", input_string)
    
    # Convert the numbers from string to float or int
    numbers = [int(num) if num.isdigit() or (num.startswith('-') and num[1:].isdigit()) else float(num) for num in numbers]
    
    return numbers

# Example usage
if __name__ == "__main__":
    input_string = "The numbers are 10, 20.5, -30, and 40.25"
    numbers = parse_string_to_numbers(input_string)
    print(numbers)
```
