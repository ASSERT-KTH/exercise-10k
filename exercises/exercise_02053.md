## Exercise
Create a function to transform a dictionary by applying a function to each value.

## Reference Solution
```python
def transform_dict(input_dict, func):
    """
    This function transforms a dictionary by applying a given function to each value.

    Args:
        input_dict (dict): The dictionary to be transformed.
        func (function): The function to be applied to each value in the dictionary.

    Returns:
        dict: A new dictionary with the function applied to each value.
    """

    # Initialize an empty dictionary to store the transformed values
    transformed_dict = {}

    # Iterate over each key-value pair in the input dictionary
    for key, value in input_dict.items():
        
        # Apply the given function to the current value
        transformed_value = func(value)
        
        # Store the transformed value in the new dictionary with the same key
        transformed_dict[key] = transformed_value
    
    # Return the new dictionary with the transformed values
    return transformed_dict

# Example usage:
# Define a simple function to double a number
def double(x):
    return x * 2

# Create a dictionary
my_dict = {'a': 1, 'b': 2, 'c': 3}

# Transform the dictionary by doubling each value
transformed_my_dict = transform_dict(my_dict, double)

print(transformed_my_dict)  # Output: {'a': 2, 'b': 4, 'c': 6}
```
