## Exercise
Write a program that takes a string as input and returns a list of all email addresses found in the string.

## Reference Solution
```python
import re

def find_emails(text):
    """
    Finds all email addresses in the given text.

    Args:
    text (str): The text to search for email addresses.

    Returns:
    list: A list of email addresses found in the text.
    """

    # Regular expression for matching most common email address formats
    email_regex = r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b'

    # Find all matches of the regular expression in the text
    emails = re.findall(email_regex, text)

    return emails


# Test the function
text = "You can contact me at john.doe@example.com or jane.doe@example.org for more information."
print(find_emails(text))
```
