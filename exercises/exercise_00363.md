## Exercise
Write a program that takes a string as input, reverses its order, and converts all characters to uppercase. The program should then print the modified string.

## Reference Solution
```python
# This is a comment explaining the purpose of the code
# The code takes a string as input, reverses its order, and converts all characters to uppercase

def main():
    # Take a string as input from the user
    s = input()

    # Reverse the order of the string
    s = s[::-1]

    # Convert the string to uppercase
    s = s.upper()

    # Print the modified string
    print(s)

# Call the main function
main()
```
