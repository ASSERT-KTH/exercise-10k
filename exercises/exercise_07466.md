## Exercise
Write a program that converts a given list into a string

## Reference Solution
```python
def list_to_string(input_list):
    """
    Converts a given list into a string.

    Args:
        input_list (list): The list to be converted.

    Returns:
        str: A string representation of the input list.
    """
    return ', '.join(map(str, input_list))

# Example usage
def main():
    input_list = [1, 2, 3, 'hello', 'world']
    output_string = list_to_string(input_list)
    print("Input List:", input_list)
    print("Output String:", output_string)

if __name__ == "__main__":
    main()
```
