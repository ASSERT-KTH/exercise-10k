## Exercise
Write a function to map a function to each value in a dictionary

## Reference Solution
```python
def map_function_to_dict_values(dictionary, func):
    """
    Maps a given function to each value in a dictionary.

    Args:
        dictionary (dict): The dictionary to map the function to.
        func (function): The function to apply to each value.

    Returns:
        dict: A new dictionary with the function applied to each value.
    """
    return {key: func(value) for key, value in dictionary.items()}

# Example usage:
numbers = {'a': 1, 'b': 2, 'c': 3}

def square(x):
    return x ** 2

squared_numbers = map_function_to_dict_values(numbers, square)
print(squared_numbers)  # Output: {'a': 1, 'b': 4, 'c': 9}
```
