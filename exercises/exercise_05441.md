## Exercise
Write a program that takes a name as input and prints out a greeting message

## Reference Solution
```python
def print_greeting(name: str) -> None:
    """
    Prints out a greeting message with the given name.

    Args:
        name (str): The name to include in the greeting.
    """
    print(f"Hello, {name}!")

def main() -> None:
    """
    Asks for a name and prints out a greeting message.
    """
    name = input("Please enter your name: ")
    print_greeting(name)

if __name__ == "__main__":
    main()
```
