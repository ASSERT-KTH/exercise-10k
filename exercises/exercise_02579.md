## Exercise
Write a program that generates the first n Fibonacci numbers using a for loop.

## Reference Solution
```python
def generate_fibonacci(n):
    """
    Generate the first n Fibonacci numbers.

    Args:
        n (int): The number of Fibonacci numbers to generate.

    Returns:
        list: A list of the first n Fibonacci numbers.
    """

    # Handle edge cases
    if n <= 0:
        return []
    elif n == 1:
        return [0]
    elif n == 2:
        return [0, 1]

    # Initialize the list with the first two Fibonacci numbers
    fibonacci_numbers = [0, 1]

    # Generate the remaining Fibonacci numbers
    for i in range(2, n):
        # Calculate the next Fibonacci number as the sum of the previous two
        next_number = fibonacci_numbers[i - 1] + fibonacci_numbers[i - 2]
        fibonacci_numbers.append(next_number)

    return fibonacci_numbers


# Example usage
if __name__ == "__main__":
    n = 10
    print(f"The first {n} Fibonacci numbers are: {generate_fibonacci(n)}")
```
