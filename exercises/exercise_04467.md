## Exercise
Write a program to print the elements of a list in reverse order using a while loop

## Reference Solution
```python
def print_list_in_reverse(input_list):
    """
    Prints the elements of a list in reverse order using a while loop.

    Args:
        input_list (list): The list to be printed in reverse order.
    """
    # Get the length of the list
    list_length = len(input_list)
    
    # Initialize the index to the last element of the list
    index = list_length - 1
    
    # Use a while loop to print the elements in reverse order
    while index >= 0:
        print(input_list[index])
        # Decrement the index to move to the previous element
        index -= 1


# Example usage
if __name__ == "__main__":
    # Create a sample list
    sample_list = [1, 2, 3, 4, 5]
    
    # Print the list in reverse order
    print("Elements in reverse order:")
    print_list_in_reverse(sample_list)
```
