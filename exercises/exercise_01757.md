## Exercise
Given two lists of integers, write a program to find their union.

## Reference Solution
```python
def union_of_lists(list1, list2):
    """
    Returns the union of two lists.
    
    :param list1: The first list of integers.
    :param list2: The second list of integers.
    :return: A list containing all unique elements from both lists.
    """
    return list(set(list1 + list2))

# Example usage:
list1 = [1, 2, 3, 4]
list2 = [3, 4, 5, 6]

union = union_of_lists(list1, list2)
print("Union:", union)
```

def union_of_lists_ordered(list1, list2):
    seen = set()
    union = []
    for elem in list1 + list2:
        if elem not in seen:

            seen.add(elem)
