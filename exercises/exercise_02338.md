## Exercise
Write a program to take a user's name as input and print a greeting message with their name, using string manipulation to remove any whitespace.

## Reference Solution
```python
// https://www.youtube.com/watch?v=VKjWX7fUeM8

#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    cin >> name;
    name.erase(remove(name.begin(), name.end(), ' '), name.end());
    cout << "Hello, " << name << "!";
    return 0;
}
```
