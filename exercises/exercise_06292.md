## Exercise
Remove a key-value pair from a dictionary using the pop() method

## Reference Solution
```python
dict.pop(key, default=None)
```

```python
# Create a dictionary
person = {
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}

# Print the original dictionary
print("Original Dictionary:")
print(person)

# Remove the 'age' key-value pair
removed_value = person.pop("age")

# Print the updated dictionary
print("\nUpdated Dictionary:")
print(person)

# Print the removed value
print("\nRemoved Value:")
print(removed_value)
```

```python
# Try to remove a non-existent key with a default value
removed_value = person.pop("country", "Key not found")
print("Removed Value (with default):")
print(removed_value)
```
