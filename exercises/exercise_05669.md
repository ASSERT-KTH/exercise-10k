## Exercise
Write a program that takes a number as input and outputs its absolute value.

## Reference Solution
```python
def absolute_value(num):
    """
    Returns the absolute value of the input number.

    Args:
        num (int): The input number.

    Returns:
        int: The absolute value of the input number.
    """
    return abs(num)

def main():
    # Get user input
    num = int(input("Enter a number: "))

    # Calculate and print the absolute value
    print("The absolute value of", num, "is", absolute_value(num))

if __name__ == "__main__":
    main()
```
