## Exercise
Write a program to create a dice roll simulator that rolls a dice and displays the result.

## Reference Solution
```python
import random

def roll_dice():
    """Simulates the roll of a dice and returns the result."""
    return random.randint(1, 6)

def main():
    print("Welcome to the Dice Roll Simulator!")
    roll_again = 'y'
    while roll_again.lower() == 'y':
        result = roll_dice()
        print(f"You rolled a {result}.")
        roll_again = input("Roll again? (y/n): ")
    print("Thanks for playing!")

if __name__ == "__main__":
    main()
```
