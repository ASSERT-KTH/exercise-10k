## Exercise
Create an array of 5 integers and find the index of a specific element.

## Reference Solution
```python
def find_index(array, target):
    """
    Finds the index of a target element in the given array.

    Args:
        array (list): The input array of integers.
        target (int): The target element to find.

    Returns:
        int: The index of the target element if found, -1 otherwise.
    """
    try:
        return array.index(target)
    except ValueError:
        return -1


# Create an array of 5 integers
array = [10, 20, 30, 40, 50]

# Find the index of a specific element (e.g., 30)
target = 30
index = find_index(array, target)

if index != -1:
    print(f"Element {target} found at index {index}.")
else:
    print(f"Element {target} not found in the array.")
```
