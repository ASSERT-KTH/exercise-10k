## Exercise
Write a program that takes a string as input and returns a new string with duplicate words removed.

## Reference Solution
```python
def remove_duplicate_words(input_string):
    """
    Removes duplicate words from a string, maintaining the original order.

    Args:
    input_string (str): The input string.

    Returns:
    str: A new string with duplicate words removed.
    """
    words = input_string.split()
    seen = set()
    unique_words = []

    for word in words:
        # Convert to lower case for case-insensitive comparison
        lower_word = word.lower()
        if lower_word not in seen:
            unique_words.append(word)
            seen.add(lower_word)

    return ' '.join(unique_words)


# Example usage
if __name__ == "__main__":
    input_string = "Hello world this is a test hello world"
    result = remove_duplicate_words(input_string)
    print("Original String:", input_string)
    print("String without duplicates:", result)
```
