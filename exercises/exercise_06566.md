## Exercise
Write a function that concatenates two dictionaries into one

## Reference Solution
```python
def concat_dicts(dict1, dict2):
    """
    Concatenates two dictionaries into one.

    Args:
        dict1 (dict): The first dictionary.
        dict2 (dict): The second dictionary.

    Returns:
        dict: A new dictionary that is the concatenation of dict1 and dict2.
    """
    return {**dict1, **dict2}

# Example usage:
dict1 = {'a': 1, 'b': 2}
dict2 = {'b': 3, 'c': 4}

result = concat_dicts(dict1, dict2)
print(result)  # Output: {'a': 1, 'b': 3, 'c': 4}
```
