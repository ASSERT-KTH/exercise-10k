## Exercise
Write a program that takes an array and a rotation amount as input, and returns the array rotated by the specified amount.

## Reference Solution
```python
def rotate(array, rotation_amount):
    rotation_amount = rotation_amount % len(array)
    return array[-rotation_amount:] + array[:-rotation_amount]

arr = [1, 2, 3, 4, 5, 6, 7]
r = 3
print(rotate(arr, r))  # Output: [5, 6, 7, 1, 2, 3, 4]
```
