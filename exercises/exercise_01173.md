## Exercise
Write a function to remove duplicate characters from a given string

## Reference Solution
```python
def remove_duplicates(input_string):
    """
    Removes duplicate characters from a given string.

    Args:
        input_string (str): The input string.

    Returns:
        str: A new string with all duplicate characters removed.
    """
    seen = set()
    result = []

    for char in input_string:
        if char not in seen:
            seen.add(char)
            result.append(char)

    return ''.join(result)

# Example usage:
if __name__ == "__main__":
    input_str = "Programming"
    print("Original String:", input_str)
    print("String without duplicates:", remove_duplicates(input_str))
```
