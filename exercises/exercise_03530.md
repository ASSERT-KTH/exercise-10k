## Exercise
Create a program that prints a triangle of stars (*) using a while loop.

## Reference Solution
```python
def print_triangle(height):
    """
    Prints a triangle of stars (*) with the given height.

    Args:
        height (int): The height of the triangle.
    """
    row = 0
    while row < height:
        # Print spaces before the stars
        space_count = height - row - 1
        print(' ' * space_count, end='')
        
        # Print the stars
        star_count = 2 * row + 1
        print('*' * star_count)
        
        row += 1

# Example usage
height = 5
print_triangle(height)
```
