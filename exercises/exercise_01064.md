## Exercise
Create a program that uses a for loop to find the maximum number in a list of numbers.

## Reference Solution
```python
def find_max_number(num_list):
    """
    This function finds the maximum number in a list of numbers.

    Args:
        num_list (list): A list of numbers.

    Returns:
        max_num (int or float): The maximum number in the list.
    """
    # Check if the list is empty
    if not num_list:
        return None

    # Initialize the maximum number with the first number in the list
    max_num = num_list[0]

    # Iterate over the list to find the maximum number
    for num in num_list:
        if num > max_num:
            max_num = num

    return max_num


# Example usage
numbers = [4, 2, 9, 6, 5, 1, 8, 3, 7]
max_number = find_max_number(numbers)
print("Maximum number in the list:", max_number)
```
