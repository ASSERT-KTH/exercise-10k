## Exercise
Write a program that takes a user's name and age as input and prints out a greeting message with their name and age. The program should also ask for the user's favorite color and print it out.

## Reference Solution
```python
# Get user input
name = input("What is your name? ")
age = input("How old are you? ")
color = input("What is your favorite color? ")

# Print greeting message
print("Hello " + name + " you are " + age + " years old")

# Print favorite color
print("Your favorite color is " + color)
```
