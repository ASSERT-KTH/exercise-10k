## Exercise
Write a program to take a name as input from the user and print out a greeting message

## Reference Solution
```python
def print_greeting(name):
    """Prints a greeting message with the given name."""
    print(f"Hello, {name}!")

def main():
    """Asks the user for their name and prints a greeting message."""
    name = input("Please enter your name: ")
    print_greeting(name)

if __name__ == "__main__":
    main()
```
