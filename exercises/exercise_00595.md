## Exercise
Write a program that takes two strings as input and returns true if the first string is a subsequence of the second string, and false otherwise.

## Reference Solution
```python
def is_subsequence(str1, str2):
    if not str1:
        return True
    if not str2:
        return False

    i = 0
    for j in range(len(str2)):
        if str1[i] == str2[j]:
            i += 1
        if i == len(str1):
            return True

    return False

# Test cases
print(is_subsequence("abc", "ahbgdc"))  # True
print(is_subsequence("axc", "ahbgdc"))  # False
print(is_subsequence("", "ahbgdc"))  # True
print(is_subsequence("abc", ""))  # False
```
