## Exercise
Create a program that takes in the current temperature and prints out an alert message. If the temperature is above 30, print 'Heat warning!', if it's below 0, print 'Cold warning!', otherwise print 'Normal temperature'

## Reference Solution
```python
def temperature_alert(temperature):
    """
    Prints out an alert message based on the temperature level.

    Args:
        temperature (float): The current temperature.
    """
    if temperature > 30:
        print("Heat warning!")
    elif temperature < 0:
        print("Cold warning!")
    else:
        print("Normal temperature")

def main():
    # Example usage
    temperature = float(input("Enter the current temperature: "))
    temperature_alert(temperature)

if __name__ == "__main__":
    main()
```
