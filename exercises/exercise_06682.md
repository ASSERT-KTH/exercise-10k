## Exercise
Generate a multiplication table for a given number up to 10.

## Reference Solution
```python
def generate_multiplication_table(number):
    """
    Generate a multiplication table for a given number up to 10.

    Args:
        number (int): The number for which to generate the multiplication table.

    Returns:
        None
    """
    print(f"Multiplication Table for {number}:")
    for i in range(1, 11):
        print(f"{number} x {i} = {number * i}")

# Example usage:
number = int(input("Enter a number: "))
generate_multiplication_table(number)
```
