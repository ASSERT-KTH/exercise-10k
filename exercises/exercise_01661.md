## Exercise
Write a program that takes an integer as input and prints 'even' if the number is even and 'odd' if the number is odd using if/else statements

## Reference Solution
```python
def check_even_odd(num):
    if num % 2 == 0:
        print("even")
    else:
        print("odd")

# Test the function
num = int(input("Enter an integer: "))
check_even_odd(num)
```
