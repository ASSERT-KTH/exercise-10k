## Exercise
Write a function that reverses the order of elements in a given array.

## Reference Solution
```python
def reverse_array(input_array):
    """
    Reverses the order of elements in a given array.

    Args:
        input_array (list): The input array to be reversed.

    Returns:
        list: A new array with the elements in reverse order.
    """
    return input_array[::-1]

# Example usage
if __name__ == "__main__":
    input_array = [1, 2, 3, 4, 5]
    reversed_array = reverse_array(input_array)
    print("Input Array:", input_array)
    print("Reversed Array:", reversed_array)
```
