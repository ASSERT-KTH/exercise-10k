## Exercise
Write a program that filters a dictionary based on a condition and returns a new dictionary with the filtered key-value pairs.

## Reference Solution
```python
def filter_dict(d, condition):
    if not callable(condition):
        raise TypeError("Condition must be a callable")
    return {k: v for k, v in d.items() if condition(k, v)}

def is_even(k, v):
    return v % 2 == 0

d = {"a": 1, "b": 2, "c": 3, "d": 4}
print(filter_dict(d, is_even))  # Output: {"b": 2, "d": 4}
```
