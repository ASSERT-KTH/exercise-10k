## Exercise
Write a program that takes a string as input and returns the string encoded in a specified encoding (e.g. ASCII, UTF-8).

## Reference Solution
```python
def string_encoding(string, encoding):
    if encoding == "ASCII":
        encoded_string = string.encode("ascii")
    elif encoding == "UTF-8":
        encoded_string = string.encode("utf-8")
    else:
        print("Invalid encoding. Supported encodings are ASCII and UTF-8.")
        return None
    
    return encoded_string

input_string = input("Enter a string: ")
input_encoding = input("Enter the encoding (ASCII or UTF-8): ")

encoded_string = string_encoding(input_string, input_encoding)

if encoded_string is not None:
    print(f"Encoded string: {encoded_string}")
    print(f"Encoded string length: {len(encoded_string)}")
    print(f"Encoded string type: {type(encoded_string)}")
```
