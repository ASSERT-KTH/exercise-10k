## Exercise
Create a function that takes a string as an argument and returns a dictionary representing the JSON data

## Reference Solution
```python
import json

def json_string_to_dict(json_str):
    """
    Convert a JSON string to a dictionary.

    Args:
        json_str (str): The JSON string to convert.

    Returns:
        dict: The dictionary representation of the JSON data.

    Raises:
        json.JSONDecodeError: If the input string is not valid JSON.
    """
    try:
        return json.loads(json_str)
    except json.JSONDecodeError as e:
        print(f"Invalid JSON: {e}")
        return None

# Example usage:
if __name__ == "__main__":
    json_str = '{"name": "John", "age": 30, "city": "New York"}'
    result = json_string_to_dict(json_str)
    print(result)
```
