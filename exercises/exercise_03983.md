## Exercise
Write a program that generates all possible anagrams of a given string.

## Reference Solution
```python
import itertools

def generate_anagrams(input_string):
    """
    Generates all possible anagrams of a given string.

    Args:
        input_string (str): The input string for which anagrams are to be generated.

    Returns:
        list: A list of all possible anagrams.
    """
    # Remove any whitespace from the input string
    input_string = input_string.replace(" ", "")

    # Generate all permutations of the input string
    permutations = itertools.permutations(input_string)

    # Convert the permutations to a list of strings and return
    return [''.join(p) for p in permutations]

def main():
    input_string = input("Enter a string: ")
    anagrams = generate_anagrams(input_string)

    print("Anagrams:")
    for anagram in anagrams:
        print(anagram)

if __name__ == "__main__":
    main()
```
