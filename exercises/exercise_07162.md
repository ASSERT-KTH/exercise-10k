## Exercise
Write a program that accesses and prints the first and last elements of an array

## Reference Solution
```python
# Define the array
my_array = [1, 2, 3, 4, 5]

# Check if the array is not empty
if len(my_array) > 0:
    # Print the first element
    print("First element: ", my_array[0])
    
    # Print the last element
    print("Last element: ", my_array[-1])
else:
    print("The array is empty.")
```
