## Exercise
Find the maximum element in an array and return its value and index.

## Reference Solution
```python
def find_max_element(arr):
    """
    Finds the maximum element in an array and returns its value and index.

    Args:
        arr (list): The input array of integers.

    Returns:
        tuple: A tuple containing the maximum element's value and index.
    """
    if not arr:
        raise ValueError("Array is empty")

    max_value = arr[0]
    max_index = 0

    for i, value in enumerate(arr):
        if value > max_value:
            max_value = value
            max_index = i

    return max_value, max_index

# Example usage:
arr = [3, 5, 1, 9, 2, 8]
max_value, max_index = find_max_element(arr)

print(f"Maximum element: {max_value}")
print(f"Index: {max_index}")
```
