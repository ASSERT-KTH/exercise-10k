## Exercise
Create a program that takes an array of integers and an index as input, and returns the element at the specified index. If the index is out of range, return a message indicating that the index is invalid.

## Reference Solution
```python
def find_element(arr, index):
    if index < 0 or index >= len(arr):
        return "Invalid index"
    return arr[index]

# Test cases
arr = [1, 2, 3, 4, 5]
print(find_element(arr, 0))  # Output: 1
print(find_element(arr, 4))  # Output: 5
print(find_element(arr, 5))  # Output: Invalid index
```
