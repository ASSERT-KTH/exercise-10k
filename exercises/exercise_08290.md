## Exercise
Create a program that calculates the length of the hypotenuse of a right triangle.

## Reference Solution
```python
import math

def calculate_hypotenuse(a, b):
    """
    Calculate the length of the hypotenuse of a right triangle.

    Args:
        a (float): The length of side a.
        b (float): The length of side b.

    Returns:
        float: The length of the hypotenuse.
    """
    return math.sqrt(a**2 + b**2)

def main():
    # Get the lengths of the sides from the user
    a = float(input("Enter the length of side a: "))
    b = float(input("Enter the length of side b: "))

    # Calculate and display the length of the hypotenuse
    hypotenuse = calculate_hypotenuse(a, b)
    print(f"The length of the hypotenuse is: {hypotenuse:.2f}")

if __name__ == "__main__":
    main()
```
