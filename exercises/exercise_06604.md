## Exercise
Write a program that prints a given quote to the console, including the quotation marks.

## Reference Solution
```python
def print_quote(quote: str) -> None:
    """
    Prints a given quote to the console, including the quotation marks.

    Args:
        quote (str): The quote to be printed.
    """
    print(f'"{quote}"')

# Example usage:
def main() -> None:
    quote = "Be the change you wish to see in the world."
    print_quote(quote)

if __name__ == "__main__":
    main()
```
