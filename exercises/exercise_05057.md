## Exercise
Create a program that converts a decimal number to a fraction.

## Reference Solution
```python
from fractions import Fraction

def decimal_to_fraction(decimal):
    """
    Converts a decimal number to a fraction.

    Args:
        decimal (float): The decimal number to convert.

    Returns:
        str: The fraction representation of the decimal number.
    """
    fraction = Fraction(decimal).limit_denominator()
    return f"{fraction.numerator}/{fraction.denominator}"

def main():
    # Example usage:
    decimal_num = float(input("Enter a decimal number: "))
    fraction = decimal_to_fraction(decimal_num)
    print(f"The fraction representation of {decimal_num} is: {fraction}")

if __name__ == "__main__":
    main()
```
