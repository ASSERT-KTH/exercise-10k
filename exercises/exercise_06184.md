## Exercise
Write a function that takes a string and an index as input and returns the character at the specified index.

## Reference Solution
```python
def get_character_at_index(input_string, index):
    """
    This function returns the character at a specified index in a given string.

    Args:
        input_string (str): The input string from which the character will be retrieved.
        index (int): The index of the character to be retrieved.

    Returns:
        str: The character at the specified index in the input string.

    Raises:
        TypeError: If the input is not a string or the index is not an integer.
        IndexError: If the index is out of range.
    """

    # First, we check if the input is a string and the index is an integer
    if not isinstance(input_string, str) or not isinstance(index, int):
        raise TypeError("Input must be a string and index must be an integer.")

    # Then, we check if the index is within the valid range
    # We use the len() function to get the length of the string
    if index < 0 or index >= len(input_string):
        raise IndexError("Index out of range.")

    # Finally, we return the character at the specified index
    # We use the indexing operator [] to access the character at the specified index
    return input_string[index]
```
