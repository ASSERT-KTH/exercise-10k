## Exercise
Write a function to check if a specific key exists in a dictionary

## Reference Solution
```python
def check_key_existence(dictionary, key):
    """
    Checks if a specific key exists in a dictionary.

    Args:
        dictionary (dict): The dictionary to be checked.
        key (any): The key to be searched for.

    Returns:
        bool: True if the key exists, False otherwise.
    """
    return key in dictionary

# Example usage:
if __name__ == "__main__":
    example_dict = {"name": "John", "age": 30, "city": "New York"}
    key_to_check = "age"

    result = check_key_existence(example_dict, key_to_check)
    print(f"Key '{key_to_check}' exists: {result}")
```
