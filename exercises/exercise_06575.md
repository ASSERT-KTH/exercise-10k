## Exercise
Create two dictionaries and merge them into a single dictionary

## Reference Solution
```python
# Create the first dictionary
dict1 = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

# Create the second dictionary
dict2 = {
    "country": "USA",
    "job": "Software Engineer",
    "hobby": "Reading"
}

# Merge the dictionaries
merged_dict = {**dict1, **dict2}

# Print the merged dictionary
print(merged_dict)
```
