## Exercise
Write a function that takes a string as input and returns the string with the first letter capitalized.

## Reference Solution
```python
def capitalize_first_letter(input_string: str) -> str:
    """
    Capitalize the first letter of a given string.

    Args:
        input_string (str): The string to capitalize.

    Returns:
        str: The input string with the first letter capitalized.
    """
    if not input_string:  # Check if the string is empty
        return input_string
    return input_string[0].upper() + input_string[1:]

# Example usage:
if __name__ == "__main__":
    input_str = "hello world"
    print(capitalize_first_letter(input_str))  # Output: "Hello world"
```
