## Exercise
Create a function to remove non-letter characters from a given string.

## Reference Solution
```python
def remove_non_letters(input_string: str) -> str:
    """
    Remove non-letter characters from a given string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The string with non-letter characters removed.
    """
    return ''.join(char for char in input_string if char.isalpha())

# Example usage:
def main():
    input_str = "Hello, World! 123"
    print("Original string:", input_str)
    print("String with non-letters removed:", remove_non_letters(input_str))

if __name__ == "__main__":
    main()
```
