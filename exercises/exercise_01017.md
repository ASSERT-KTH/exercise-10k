## Exercise
Write a program that takes a string and two integers as input, and returns the substring between the given start and end indices.

## Reference Solution
```python
#include <iostream>
#include <string>

int main() {
    std::string str;
    int a, b;

    std::cout << "Enter a string: ";
    std::cin >> str;

    std::cout << "Enter start index: ";
    std::cin >> a;

    std::cout << "Enter end index: ";
    std::cin >> b;

    for (int i = a; i < b; i++) {
        std::cout << str[i];
    }

    return 0;
}
```
