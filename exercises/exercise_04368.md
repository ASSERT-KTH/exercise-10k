## Exercise
Write a program that sorts a dictionary's keys in ascending order and prints them.

## Reference Solution
```python
# Define a dictionary
my_dict = {'c': 3, 'b': 2, 'a': 1, 'e': 5, 'd': 4}

# Sort the dictionary's keys in ascending order
sorted_keys = sorted(my_dict.keys())

# Print the sorted keys
print("Sorted keys:")
for key in sorted_keys:
    print(key)
```
