## Exercise
Write a program that uses nested loops to print a square pattern of asterisks.

## Reference Solution
```python
def print_square(size):
    """
    Prints a square pattern of asterisks with the specified size.

    Args:
        size (int): The size of the square.
    """
    # Loop through each row in the square
    for i in range(size):
        # Loop through each column in the row
        for j in range(size):
            # Print an asterisk
            print("* ", end="")
        # Move to the next line
        print()

# Example usage:
print_square(5)
```
