## Exercise
Write a program that prints all even numbers from 1 to a given input number using a for loop. If the input number is less than 1, the program should print an error message.

## Reference Solution
```python
for i in range(1, 10):
    if i % 2 == 0:
        print(i)

print("Error: input value less than 1")
```

```python
n = int(input("Enter a number: "))
if n < 1:
    print("Error: input value less than 1")
else:
    for i in range(1, n+1):
        if i % 2 == 0:
            print(i)
```
