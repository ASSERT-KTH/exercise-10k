## Exercise
Write a program that takes in a user's input and determines the data type of the input. The input can be either an integer, float, or string. Print out the determined data type.

## Reference Solution
```python
def determine_data_type(user_input):
    """
    Determine the data type of the user's input.

    Args:
        user_input (str): The user's input.

    Returns:
        str: The determined data type.
    """
    try:
        # Try to convert the input to an integer
        int(user_input)
        return "Integer"
    except ValueError:
        try:
            # If it's not an integer, try to convert it to a float
            float(user_input)
            return "Float"
        except ValueError:
            # If it's neither an integer nor a float, it's a string
            return "String"

def main():
    user_input = input("Please enter a value: ")
    data_type = determine_data_type(user_input)
    print(f"The data type of '{user_input}' is: {data_type}")

if __name__ == "__main__":
    main()
```
