## Exercise
Write a function that uses a try-except block to handle an exception

## Reference Solution
```python
def divide_numbers(dividend, divisor):
    """
    Divide two numbers and handle potential ZeroDivisionError.

    Args:
    - dividend (float): The number being divided.
    - divisor (float): The number by which we are dividing.

    Returns:
    - float: The result of the division, or an error message if division by zero occurs.
    """
    try:
        # Attempt to divide the numbers
        result = dividend / divisor
        return result
    except ZeroDivisionError:
        # Handle the exception if the divisor is zero
        return "Error: Cannot divide by zero."
    except TypeError:
        # Handle the exception if inputs are not numbers
        return "Error: Both inputs must be numbers."

# Example usage
if __name__ == "__main__":
    print(divide_numbers(10, 2))  # Output: 5.0
    print(divide_numbers(10, 0))  # Output: Error: Cannot divide by zero.
    print(divide_numbers("ten", 2))  # Output: Error: Both inputs must be numbers.
```
