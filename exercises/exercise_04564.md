## Exercise
Write a function that takes a dictionary and prints out each key-value pair

## Reference Solution
```python
def print_dict(input_dict):
    """
    Prints out each key-value pair in the input dictionary.

    Args:
        input_dict (dict): The dictionary to print.
    """
    for key, value in input_dict.items():
        print(f"{key}: {value}")

# Example usage:
def main():
    example_dict = {
        "name": "John",
        "age": 30,
        "city": "New York"
    }
    print_dict(example_dict)

if __name__ == "__main__":
    main()
```
