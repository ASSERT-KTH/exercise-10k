## Exercise
Find the difference between two dictionaries

## Reference Solution
```python
def dict_diff(dict1, dict2):
    """Return a dictionary with keys that are in dict1 but not in dict2, or have different values."""
    return {key: dict1[key] for key in dict1 if key not in dict2 or dict1[key] != dict2[key]}

# Example usage:
dict1 = {'a': 1, 'b': 2, 'c': 3}
dict2 = {'a': 1, 'b': 3, 'd': 4}
print(dict_diff(dict1, dict2))  # Output: {'b': 2, 'c': 3}
```
