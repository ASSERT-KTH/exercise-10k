## Exercise
Write a program that takes an array and an index as input and returns the element at that index.

## Reference Solution
```python
# Define the function to get the element at a given index
def get_element_at_index(array, index):
    """
    Returns the element at the specified index in the given array.

    Args:
        array (list): The input array.
        index (int): The index of the element to retrieve.

    Returns:
        any: The element at the specified index.

    Raises:
        IndexError: If the index is out of range.
    """
    # Check if the index is within the bounds of the array
    if index < 0 or index >= len(array):
        raise IndexError("Index out of range")

    # Return the element at the specified index
    return array[index]

# Example usage
array = [1, 2, 3, 4, 5]
index = 2

try:
    element = get_element_at_index(array, index)
    print(f"Element at index {index}: {element}")
except IndexError as e:
    print(f"Error: {e}")

# Output: Element at index 2: 3
```
