## Exercise
Write a program that prints all prime numbers from 1 to 50 using a for loop and a helper function.

## Reference Solution
```python
def is_prime(n):
    if n <= 1:
        return False
    elif n <= 3:
        return True
    elif n % 2 == 0 or n % 3 == 0:
        return False
    i = 5
    while i * i <= n:
        if n % i == 0 or n % (i + 2) == 0:
            return False
        i += 6
    return True

for num in range(1, 51):
    if is_prime(num):
        print(num)
```
