## Exercise
Write a program to print a list of numbers to the screen, one number per line.

## Reference Solution
```python
def print_numbers(numbers):
    """
    Prints a list of numbers to the screen, one number per line.

    Args:
        numbers (list): A list of numbers to print.
    """
    for number in numbers:
        print(number)

# Example usage
numbers = [1, 2, 3, 4, 5]
print_numbers(numbers)
```
