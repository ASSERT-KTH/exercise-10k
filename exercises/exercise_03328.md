## Exercise
Write a program that takes an array of integers as input and returns the sum of all the elements in the array.

## Reference Solution
```python
def sum_of_elements(array):
    """
    This function calculates the sum of all elements in a given array.

    Args:
        array (list): A list of integers.

    Returns:
        int: The sum of all elements in the array.

    Raises:
        TypeError: If the input is not a list.
        ValueError: If the list contains non-integer values.
    """
    # Check if the input is a list
    if not isinstance(array, list):
        raise TypeError("Input must be a list.")

    # Check if the list is empty
    if len(array) == 0:
        return 0

    # Check if all elements in the list are integers
    if not all(isinstance(element, int) for element in array):
        raise ValueError("All elements in the list must be integers.")

    # Calculate the sum of all elements in the array
    total_sum = sum(array)

    return total_sum

# Example usage:
array = [1, 2, 3, 4, 5]
result = sum_of_elements(array)
print("The sum of all elements in the array is:", result)
```
