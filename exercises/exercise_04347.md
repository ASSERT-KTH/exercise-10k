## Exercise
Write a program that checks if a given year is a leap year and prints 'Leap Year' if true, otherwise prints 'Not a Leap Year'

## Reference Solution
```python
def check_leap_year(year):
    """Check if a given year is a leap year."""
    if year % 4 == 0:
        if year % 100 == 0:
            if year % 400 == 0:
                return True
            else:
                return False
        else:
            return True
    else:
        return False

def main():
    year = int(input("Enter a year: "))
    if check_leap_year(year):
        print(f"{year} is a Leap Year")
    else:
        print(f"{year} is Not a Leap Year")

if __name__ == "__main__":
    main()
```
