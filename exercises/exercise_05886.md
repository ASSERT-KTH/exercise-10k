## Exercise
Write a function that takes an array as input and returns a new array containing the same elements in sorted order.

## Reference Solution
```python
def sort_array(input_array):
    """
    Returns a new array containing the same elements as the input array, 
    but in sorted order.

    Args:
    input_array (list): The input array to be sorted.

    Returns:
    list: A new array containing the sorted elements.
    """
    return sorted(input_array)

# Example usage:
def main():
    # Test the function with an example array
    example_array = [64, 34, 25, 12, 22, 11, 90]
    print("Original array:", example_array)
    print("Sorted array:", sort_array(example_array))

if __name__ == "__main__":
    main()
```
