## Exercise
Write a program that checks if a given string is a substring of another string

## Reference Solution
```python
def is_substring(main_string, sub_string):
    """
    Checks if a given string is a substring of another string.

    Args:
        main_string (str): The main string to check.
        sub_string (str): The substring to look for.

    Returns:
        bool: True if the substring is found, False otherwise.
    """
    return sub_string in main_string


def main():
    # Example usage
    main_string = "Hello, World!"
    sub_string = "World"
    result = is_substring(main_string, sub_string)

    print(f"'{sub_string}' is a substring of '{main_string}': {result}")


if __name__ == "__main__":
    main()
```
