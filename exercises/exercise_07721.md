## Exercise
Remove consecutive duplicate characters from a given string.

## Reference Solution
```python
def remove_consecutive_duplicates(input_string):
    """
    Removes consecutive duplicate characters from a given string.
    
    Args:
        input_string (str): The input string to process.
    
    Returns:
        str: The string with consecutive duplicates removed.
    """
    result = []
    for char in input_string:
        if not result or char != result[-1]:
            result.append(char)
    return ''.join(result)

# Example usage:
input_string = "aaabbbcccdddeee"
print("Input String:", input_string)
print("String after removing consecutive duplicates:", remove_consecutive_duplicates(input_string))
```
