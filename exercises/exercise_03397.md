## Exercise
Write a function to find the intersection of two lists.

## Reference Solution
```python
def intersect(list1, list2):
    """
    Returns the intersection of two lists.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: The intersection of list1 and list2.
    """
    return list(set(list1) & set(list2))

# Example usage
if __name__ == "__main__":
    list1 = [1, 2, 3, 4, 5]
    list2 = [4, 5, 6, 7, 8]
    print(intersect(list1, list2))  # Output: [4, 5]
```

```python
def intersect_ordered(list1, list2):
    """
    Returns the intersection of two lists, preserving the original order.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: The intersection of list1 and list2, in the order they appear in list1.
    """
    set2 = set(list2)
    return [element for element in list1 if element in set2]

# Example usage
if __name__ == "__main__":
    list1 = [1, 2, 3, 4, 5]
    list2 = [4, 5, 6, 7, 8]
    print(intersect_ordered(list1, list2))  # Output: [4, 5]
```
