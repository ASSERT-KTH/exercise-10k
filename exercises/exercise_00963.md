## Exercise
Write a function to update the value associated with a given key in a dictionary. If the key does not exist, add it to the dictionary with the provided value.

## Reference Solution
```python
def update_dictionary(dictionary, key, value):
    """
    Updates the value associated with a given key in a dictionary. 
    If the key does not exist, adds it to the dictionary with the provided value.

    Args:
        dictionary (dict): The dictionary to be updated.
        key (any): The key to be updated or added.
        value (any): The new value associated with the key.

    Returns:
        dict: The updated dictionary.
    """

    # First, we check if the key already exists in the dictionary
    if key in dictionary:
        # If the key exists, we update its value
        dictionary[key] = value
    else:
        # If the key does not exist, we add it to the dictionary with the provided value
        dictionary[key] = value

    # Finally, we return the updated dictionary
    return dictionary

# Example usage:
my_dict = {"name": "John", "age": 30}
print(update_dictionary(my_dict, "age", 31))  # Updates the age
print(update_dictionary(my_dict, "city", "New York"))  # Adds a new key-value pair
```
