## Exercise
Write a function that combines two lists into a list of pairs.

## Reference Solution
```python
def combine_lists(list1, list2):
    """
    Combines two lists into a list of pairs.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: A list of pairs.
    """
    return list(zip(list1, list2))

# Example usage:
list1 = [1, 2, 3]
list2 = ['a', 'b', 'c']

result = combine_lists(list1, list2)
print(result)  # Output: [(1, 'a'), (2, 'b'), (3, 'c')]
```

import itertools

def combine_lists(list1, list2):
    """
    Combines two lists into a list of pairs, filling in missing values with None.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: A list of pairs.
    """
    return list(itertools.zip_longest(list1, list2))

# Example usage:
list1 = [1, 2, 3]
list2 = ['a', 'b']
