## Exercise
Create a program that takes a user's full name and age as input, validates if the age is within the range of 1 to 120, and then formats the output to display the user's name in title case and the age with a descriptive message based on the age group (child: 1-12, teenager: 13-19, adult: 20-64, senior: 65 and above). If the age is not within the valid range, display an error message.

## Reference Solution
```python
# Define a function to format the output
format_output = lambda name, age: f"{name.title()}: {get_age_group(age)}"

# Define a function to determine the age group
def get_age_group(age):
    if age <= 12:
        return "Child"
    elif age <= 19:
        return "Teenager"
    elif age <= 64:
        return "Adult"
    else:
        return "Senior"

# Define a function to get the user's full name
def get_full_name():
    return input("Enter your full name: ")

# Define a function to get the user's age
def get_age():
    while True:
        try:
            age = int(input("Enter your age: "))
            if 1 <= age <= 120:
                return age
            else:
                print("Error: Age must be between 1 and 120.")
        except ValueError:
            print("Error: Invalid input. Please enter a whole number.")

# Main program
def main():
    full_name = get_full_name()
    age = get_age()
    print(format_output(full_name, age))

# Run the main program
main()
```
