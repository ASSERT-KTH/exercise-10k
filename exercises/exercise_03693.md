## Exercise
Write a program to calculate the tip amount for a given bill amount and tip percentage.

## Reference Solution
```python
def calculate_tip(bill_amount, tip_percentage):
    """
    Calculate the tip amount for a given bill amount and tip percentage.

    Args:
        bill_amount (float): The amount of the bill.
        tip_percentage (float): The tip percentage as a decimal.

    Returns:
        float: The tip amount.
    """
    tip_amount = bill_amount * tip_percentage
    return tip_amount

def main():
    # Get the bill amount from the user
    bill_amount = float(input("Enter the bill amount: $"))

    # Get the tip percentage from the user
    tip_percentage = float(input("Enter the tip percentage (e.g. 15 for 15%): "))
    tip_percentage = tip_percentage / 100  # Convert to decimal

    # Calculate the tip amount
    tip_amount = calculate_tip(bill_amount, tip_percentage)

    # Print the result
    print(f"The tip amount is: ${tip_amount:.2f}")
    print(f"The total amount (bill + tip) is: ${bill_amount + tip_amount:.2f}")

if __name__ == "__main__":
    main()
```
