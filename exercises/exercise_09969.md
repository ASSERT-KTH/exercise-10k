## Exercise
Write a program that splits a string into substrings based on a delimiter.

## Reference Solution
```python
def split_string(input_string, delimiter):
    """
    Splits a string into substrings based on a delimiter.

    Args:
        input_string (str): The string to be split.
        delimiter (str): The delimiter to split the string by.

    Returns:
        list: A list of substrings.
    """
    return input_string.split(delimiter)

def main():
    # Example usage
    input_string = "hello,world,python,programming"
    delimiter = ","
    substrings = split_string(input_string, delimiter)
    
    print("Input String:", input_string)
    print("Delimiter:", delimiter)
    print("Substrings:", substrings)

if __name__ == "__main__":
    main()
```
