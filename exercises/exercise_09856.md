## Exercise
Create a program that takes user input and checks if it is a valid integer. If the input is valid, print the integer. If not, print an error message and ask for input again. Continue this process until a valid integer is entered.

## Reference Solution
```python
import sys

def main():
    while True:
        user_input = input("Enter a valid integer: ")
        if user_input.isnumeric():
            print(int(user_input))
            break
        elif user_input == "-":
            print("Error: Input cannot be negative.")
        else:
            print("Error: Input is not a valid integer.")
            continue
        break

if __name__ == "__main__":
    main()
```
