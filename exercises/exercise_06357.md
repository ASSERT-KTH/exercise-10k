## Exercise
Swap the values of two variables without using a temporary variable. Explain the process and provide examples.

## Reference Solution
```python
def swap_arithmetic(a, b):
    """
    Swap the values of two variables using arithmetic operations.

    Args:
        a (int): The first variable.
        b (int): The second variable.

    Returns:
        tuple: The swapped values.
    """
    print("Before swap: a =", a, ", b =", b)
    a = a + b  # Store the sum of a and b in a
    b = a - b  # Store the original value of a in b
    a = a - b  # Store the original value of b in a
    print("After swap: a =", a, ", b =", b)
    return a, b

# Example usage:
a = 5
b = 10
swap_arithmetic(a, b)
```

def swap_tuple(a, b):
    """
    Swap the values of two variables using tuple packing and unpacking.

    Args:
        a (int): The first variable.
        b (int): The second variable.

    Returns:
        tuple: The swapped values.
    """
    print("Before swap: a =", a, ", b =", b)
    a, b = b, a  # Swap the values using tuple packing and unpacking
    print("After swap: a =", a, ", b =", b)
    return a, b

# Example usage:
