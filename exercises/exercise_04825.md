## Exercise
Write a program to manipulate a string by converting it to uppercase, lowercase, and title case, and print out the results.

## Reference Solution
```python
def manipulate_string(input_string):
    """
    Manipulate a string by converting it to uppercase, lowercase, and title case.

    Args:
        input_string (str): The string to be manipulated.

    Returns:
        dict: A dictionary containing the manipulated strings.
    """
    # Convert the string to uppercase
    uppercase_string = input_string.upper()

    # Convert the string to lowercase
    lowercase_string = input_string.lower()

    # Convert the string to title case
    title_case_string = input_string.title()

    # Return a dictionary with the manipulated strings
    return {
        "original": input_string,
        "uppercase": uppercase_string,
        "lowercase": lowercase_string,
        "title_case": title_case_string
    }

def main():
    # Test the function
    input_string = "hElLo, wOrLd!"
    result = manipulate_string(input_string)

    # Print the results
    for key, value in result.items():
        print(f"{key.capitalize().replace('_', ' ')}: {value}")

if __name__ == "__main__":
    main()
```
