## Exercise
Create a program that checks a student's grade and prints out a message based on their score using if/else statements

## Reference Solution
```python
def check_grade(score):
    """
    Checks a student's grade and prints out a message based on their score.
    
    Args:
        score (float): The student's score.
    """
    if score >= 90:
        print("Grade: A. Excellent job! Your score is", score)
    elif score >= 80:
        print("Grade: B. Good job! Your score is", score)
    elif score >= 70:
        print("Grade: C. Fair job! Your score is", score)
    elif score >= 60:
        print("Grade: D. Needs improvement! Your score is", score)
    else:
        print("Grade: F. Failed! Your score is", score)

def main():
    # Get the student's score
    score = float(input("Enter the student's score: "))
    
    # Check the grade
    check_grade(score)

if __name__ == "__main__":
    main()
```
