## Exercise
Write a program to print the ASCII art representation of a character.

## Reference Solution
```python
import pyfiglet

def print_ascii_art(char):
    """
    Prints the ASCII art representation of a character.

    Args:
        char (str): The character to print.

    Returns:
        None
    """
    ascii_art = pyfiglet.figlet_format(char)
    print(ascii_art)

def main():
    char = input("Enter a character: ")
    print_ascii_art(char)

if __name__ == "__main__":
    main()
```
