## Exercise
Create a function that takes in an array and a target value, then returns the index of the target value in the array if it exists. If the target value does not exist, return a message indicating that the value was not found.

## Reference Solution
```python
def find_index(array, target):
    """
    This function finds the index of a target value in an array.

    Args:
        array (list): The list to search in.
        target: The value to search for.

    Returns:
        int or str: The index of the target value if found, otherwise a message indicating that the value was not found.
    """
    try:
        # Attempt to find the index of the target value
        index = array.index(target)
        return index
    except ValueError:
        # If the target value is not found, return a message
        return "Target value not found in array."

# Example usage
if __name__ == "__main__":
    array = [1, 2, 3, 4, 5]
    target = 3
    result = find_index(array, target)
    print(result)  # Output: 2

    target = 6
    result = find_index(array, target)
    print(result)  # Output: Target value not found in array.
```
