## Exercise
Write a function that takes a dictionary as input and clears all the key-value pairs

## Reference Solution
```python
def clear_dictionary(input_dict):
    """
    Clears all key-value pairs from the input dictionary.

    Args:
        input_dict (dict): The dictionary to be cleared.

    Returns:
        None
    """
    input_dict.clear()

# Example usage:
my_dict = {"key1": "value1", "key2": "value2"}
print("Before clearing:", my_dict)

clear_dictionary(my_dict)
print("After clearing:", my_dict)
```
