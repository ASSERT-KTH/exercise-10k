## Exercise
Write a function to calculate the sum of digit factorials of a given number

## Reference Solution
```python
import math

def digit_factorial_sum(n):
    """
    Calculate the sum of digit factorials of a given number.

    Args:
    n (int): The input number.

    Returns:
    int: The sum of digit factorials.
    """
    return sum(math.factorial(int(digit)) for digit in str(n))

# Example usage:
def main():
    num = 145
    result = digit_factorial_sum(num)
    print(f"The sum of digit factorials of {num} is: {result}")

if __name__ == "__main__":
    main()
```
