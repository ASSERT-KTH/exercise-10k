## Exercise
Create a function that finds the union of two given lists.

## Reference Solution

def union_of_lists(list1, list2):
    """
    Returns the union of two given lists.

    Args:
        list1 (list): The first input list.
        list2 (list): The second input list.

    Returns:
        list: A new list containing all unique elements from both input lists.
    """
    return list(set(list1 + list2))
