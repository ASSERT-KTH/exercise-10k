## Exercise
Write a function that normalizes a dictionary by converting its keys to a standard format

## Reference Solution
```python
def normalize_dict_keys(input_dict):
    """
    Normalize a dictionary by converting its keys to lowercase with underscores instead of spaces.

    Args:
        input_dict (dict): The dictionary to be normalized.

    Returns:
        dict: The normalized dictionary.
    """
    normalized_dict = {key.replace(' ', '_').lower(): value for key, value in input_dict.items()}
    return normalized_dict

# Example usage
if __name__ == "__main__":
    input_dict = {
        "First Key": "Value 1",
        "Second Key": "Value 2",
        "Third Key": "Value 3"
    }

    normalized_dict = normalize_dict_keys(input_dict)
    print("Normalized Dictionary:")
    for key, value in normalized_dict.items():
        print(f"{key}: {value}")
```
