## Exercise
Write a program that asks for a sentence and prints out the first word.

## Reference Solution
```python
def get_first_word(sentence):
    """
    Extracts the first word from a given sentence.

    Args:
        sentence (str): The input sentence.

    Returns:
        str: The first word of the sentence.
    """
    words = sentence.split()
    if words:
        return words[0]
    else:
        return None

def main():
    # Ask the user for a sentence
    sentence = input("Please enter a sentence: ")

    # Extract and print the first word
    first_word = get_first_word(sentence)
    if first_word:
        print("The first word is:", first_word)
    else:
        print("The sentence is empty.")

if __name__ == "__main__":
    main()
```
