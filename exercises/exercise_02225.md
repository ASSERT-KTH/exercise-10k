## Exercise
Write a program that takes an array and a target value as input, and returns the index of the target value in the array. If the target value is not found, return -1.

## Reference Solution
```python
def find_index(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

# Example usage:
arr = [1, 2, 3, 4, 5]
target = 3
print(find_index(arr, target))  # Output: 2
```
