## Exercise
Write a function to find the maximum contiguous subarray sum of a given one-dimensional array of numbers. The function should take an array of integers as input and return the maximum sum.

## Reference Solution
```python
def maxSubArray(nums):
    if not nums:
        return 0

    max_sum = float('-inf')
    current_sum = 0

    for num in nums:
        current_sum = max(num, current_sum + num)
        max_sum = max(max_sum, current_sum)

    return max_sum
```
