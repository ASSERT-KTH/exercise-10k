## Exercise
Write a program that takes a string as input from the user and validates if it's a numeric value. If the input is numeric, format it as a currency and print it out. If not, print an error message indicating that the input is not a valid number.

## Reference Solution
```python
import sys

def checkIfNumeric():
    text = input("Enter a value: ")
    if text.isnumeric():
        print(f"R${float(text):.2f}")
    else:
        print("Invalid input. Please enter a numeric value.")

checkIfNumeric()
```
