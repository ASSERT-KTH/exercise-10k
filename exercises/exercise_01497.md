## Exercise
Write a program that checks the data type of a variable and prints it out.

## Reference Solution
```python
def check_data_type(var):
    """
    This function checks the data type of a variable and prints it out.
    
    Parameters:
    var (any): The variable to check the data type of.
    """
    data_type = type(var)
    print(f"The data type of the variable is: {data_type.__name__}")

# Example usage:
check_data_type(10)  # Output: The data type of the variable is: int
check_data_type(10.5)  # Output: The data type of the variable is: float
check_data_type("Hello")  # Output: The data type of the variable is: str
check_data_type(True)  # Output: The data type of the variable is: bool
check_data_type([1, 2, 3])  # Output: The data type of the variable is: list
check_data_type({"a": 1, "b": 2})  # Output: The data type of the variable is: dict
```
