## Exercise
Create a program that counts down from a given number using a for loop, printing each number on a new line.

## Reference Solution
```python
def countdown(n):
    """
    Counts down from n to 1, printing each number on a new line.
    
    Args:
        n (int): The starting number for the countdown.
    """
    for i in range(n, 0, -1):
        print(i)

# Example usage
n = 10
print(f"Counting down from {n}:")
countdown(n)
```
