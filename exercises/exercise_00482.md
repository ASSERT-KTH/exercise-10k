## Exercise
Write a program that takes a list and a filter function as input, and returns a new list containing only the elements that pass the filter function.

## Reference Solution
```python
def filter_list(list, filter_func):
    new_list = []
    for element in list:
        if filter_func(element):
            new_list.append(element)
    return new_list

def is_even(n):
    return n % 2 == 0

def main():
    list = [1, 2, 3, 4, 5, 6, 7, 8, 9]
    result_list = filter_list(list, is_even)
    print(result_list)

if __name__ == "__main__":
    main()
```
