## Exercise
Write a function that takes an array and two indices as input, and returns a slice of the array from the first index to the second index.

## Reference Solution
```python
def slice_array(arr, start, end):
    """
    Returns a slice of the input array from the start index to the end index.

    Args:
        arr: The input array.
        start: The starting index of the slice.
        end: The ending index of the slice.

    Returns:
        A new array that is a slice of the input array.
    """
    return arr[start:end]
```
