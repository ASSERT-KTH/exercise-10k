## Exercise
Write a program to reverse a given string. The program should take a string as input and output the reversed string.

## Reference Solution
```python
def reverse_string(s):
    """Return the reversed string."""
    return s[::-1]

def main():
    """Get user input and print the reversed string."""
    user_input = input("Enter a string: ")
    reversed_s = reverse_string(user_input)
    print("Reversed string:", reversed_s)

if __name__ == "__main__":
    main()
```
