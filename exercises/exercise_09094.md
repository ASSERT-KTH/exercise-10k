## Exercise
Write a program that prints all the numbers from 1 to 100, but for multiples of 3, print 'Loop' instead of the number. For multiples of 5, print 'Py'. For numbers which are multiples of both 3 and 5, print 'LoopPy'. Use a for loop to iterate through the numbers and conditional statements to check for multiples.

## Reference Solution
```python
for num in range(1, 101):
    if num % 3 == 0 and num % 5 == 0:
        print("LoopPy")
    elif num % 3 == 0:
        print("Loop")
    elif num % 5 == 0:
        print("Py")
    else:
        print(num)
```
