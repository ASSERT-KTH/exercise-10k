## Exercise
Write a program that takes a string as input and returns all possible anagrams of the string.

## Reference Solution
```python
from itertools import permutations

def generate_anagrams(input_string):
    """
    Generate all possible anagrams of a given string.

    Args:
        input_string (str): The input string for which anagrams are to be generated.

    Returns:
        list: A list of all possible anagrams of the input string.
    """
    # Remove duplicate anagrams by converting to set
    anagrams = set(''.join(p) for p in permutations(input_string))
    
    return list(anagrams)

def main():
    # Example usage
    input_string = "abc"
    anagrams = generate_anagrams(input_string)
    
    print("Anagrams of '{}':".format(input_string))
    for anagram in anagrams:
        print(anagram)

if __name__ == "__main__":
    main()
```
