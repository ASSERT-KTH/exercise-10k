## Exercise
Write a program that takes a string as input and returns the normalized version of the string, where all whitespace characters are replaced by a single space.

## Reference Solution
```python
import re

def normalize_string(s):
    s = re.sub(r'\s+', ' ', s)
    s = s.strip()
    return s

# Test the function
s = "Hello,   World!  "
print(normalize_string(s))
```
