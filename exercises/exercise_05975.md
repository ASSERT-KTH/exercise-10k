## Exercise
Write a program that takes an array and a rotation amount as input, and returns a new array that is the result of rotating the input array by the specified amount.

## Reference Solution
```python
def rotate(lists, n):
    if len(lists) == 0:
        return lists
    n = n % len(lists)
    return lists[-n:] + lists[:-n]
```
