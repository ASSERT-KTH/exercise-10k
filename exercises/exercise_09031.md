## Exercise
Create a function that takes three numbers as arguments and returns the maximum of the three

## Reference Solution
```python
def max_of_three(a: float, b: float, c: float) -> float:
    """
    Returns the maximum of three numbers.

    Args:
        a (float): The first number.
        b (float): The second number.
        c (float): The third number.

    Returns:
        float: The maximum of the three numbers.
    """
    return max(a, b, c)

# Example usage:
if __name__ == "__main__":
    print(max_of_three(10, 20, 30))  # Output: 30
    print(max_of_three(-10, -20, -30))  # Output: -10
    print(max_of_three(5, 5, 5))  # Output: 5
```
