## Exercise
Write a closure to create a function that adds a given number to its input.

## Reference Solution
```python
// Function to create a closure that adds a given number to its input
function createAdder(num) {
    // The closure has access to 'num' even after createAdder has returned
    return function add(input) {
        return input + num;
    }
}

// Example usage:
// Create a function that adds 5 to its input
const addFive = createAdder(5);

// Use the addFive function
console.log(addFive(10));  // Outputs: 15

// Create a function that adds 3 to its input
const addThree = createAdder(3);

// Use the addThree function
console.log(addThree(10));  // Outputs: 13
```
