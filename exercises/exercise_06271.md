## Exercise
Create a nested dictionary with 2 levels of nesting

## Reference Solution
```python
# Create a nested dictionary
nested_dict = {
    "level1_key1": {
        "level2_key1": "value1",
        "level2_key2": "value2"
    },
    "level1_key2": {
        "level2_key3": "value3",
        "level2_key4": "value4"
    }
}

# Print the nested dictionary
print("Nested Dictionary:")
print(nested_dict)

# Accessing values in the nested dictionary
print("\nAccessing Values:")
print("Value of level2_key1:", nested_dict["level1_key1"]["level2_key1"])
print("Value of level2_key4:", nested_dict["level1_key2"]["level2_key4"])
```
