## Exercise
Write a program that takes the length and width of a rectangle as input, and returns the area and perimeter.

## Reference Solution
```python
def calculate_rectangle_properties(length, width):
    # Calculate the area
    area = length * width

    # Calculate the perimeter
    perimeter = 2 * (length + width)

    return area, perimeter

length = 5
width = 3

area, perimeter = calculate_rectangle_properties(length, width)

print(f"Area: {area}")
print(f"Perimeter: {perimeter}")
```
