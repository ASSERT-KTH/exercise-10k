## Exercise
Apply a function to each value in a dictionary using dictionary comprehension

## Reference Solution
```python
def square(x):
    """Return the square of a number"""
    return x ** 2

# Define a dictionary
numbers = {'a': 1, 'b': 2, 'c': 3, 'd': 4}

# Apply the square function to each value in the dictionary
squared_numbers = {key: square(value) for key, value in numbers.items()}

print(squared_numbers)
```
