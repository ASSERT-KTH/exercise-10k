## Exercise
Declare and use boolean variables to represent true or false conditions.

## Reference Solution
```python
# Declare boolean variables
is_admin = True
is_logged_in = False

# Use boolean variables in conditional statements
if is_admin:
    print("You are an admin.")
else:
    print("You are not an admin.")

# Use boolean variables in logical operations
is_valid_user = is_logged_in and is_admin
print(f"Is valid user: {is_valid_user}")

# Toggle a boolean variable
is_logged_in = not is_logged_in
print(f"Is logged in: {is_logged_in}")
```
