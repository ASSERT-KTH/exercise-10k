## Exercise
Write a function that returns another function

## Reference Solution
```python
def outer_function(greeting):
    """
    Returns a function that prints a personalized greeting.

    Args:
        greeting (str): The greeting to be used.

    Returns:
        function: A function that takes a name and prints a personalized greeting.
    """
    def inner_function(name):
        print(f"{greeting}, {name}!")

    return inner_function

# Example usage:
hello = outer_function("Hello")
hi = outer_function("Hi")

hello("Alice")  # Output: Hello, Alice!
hi("Bob")       # Output: Hi, Bob!
```
