## Exercise
Write a function that takes an array of integers as input and returns the reversed array. The function should not use any built-in reverse functions.

## Reference Solution

```python
def main():
    array = [1, 2, 3, 4, 5, 6]
    reversed_array = reverse_array(array)
    print(array)
    print(reversed_array)

def reverse_array(array):
    left_index = 0
    right_index = len(array) - 1
    while left_index < right_index:
        temp = array[left_index]
        array[left_index] = array[right_index]
        array[right_index] = temp
        left_index += 1
        right_index -= 1
    return array

main()
```
