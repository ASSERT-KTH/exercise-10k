## Exercise
Cast a variable of one data type to another data type.

## Reference Solution
```python
# Cast a string to an integer
my_string = "123"
my_int = int(my_string)
print(my_int)  # Output: 123

# Cast an integer to a float
my_int = 123
my_float = float(my_int)
print(my_float)  # Output: 123.0
```

```python
// Cast a string to an integer
String myString = "123";
int myInt = Integer.parseInt(myString);
System.out.println(myInt);  // Output: 123

// Cast an integer to a float
int myInt2 = 123;
float myFloat = (float) myInt2;
System.out.println(myFloat);  // Output: 123.0
```

```python
// Cast a string to an integer
std::string myString = "123";
int myInt = std::stoi(myString);
std::cout << myInt << std::endl;  // Output: 123

// Cast an integer to a float
int myInt2 = 123;
float myFloat = static_cast<float>(myInt2);
std::cout << myFloat << std::endl;  // Output: 123.0
```

```python
# Implicit casting of an integer to a float
my_int = 123
my_float = 123.45 + my_int
print(my_float)  # Output: 256.45
```

```python
// Implicit casting of an integer to a float
int myInt = 123;
float myFloat = 123.45f + myInt;
System.out.println(myFloat);  // Output: 246.45
```
