## Exercise
Write a program that takes two strings as input and returns true if the first string is a rotation of the second string, and false otherwise.

## Reference Solution
```python
def isRotation(s1, s2):
    if len(s1) != len(s2):
        return False

    temp = s1 + s1

    for i in range(len(temp)):
        if temp[i:i+len(s1)] == s2:
            return True

    return False

# Test the function
s1 = "abcd"
s2 = "dabc"

if isRotation(s1, s2):
    print("True")
else:
    print("False")
```
