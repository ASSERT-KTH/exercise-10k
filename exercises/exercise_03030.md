## Exercise
Write a program that takes a string and a rotation value as input, and returns the string rotated by the specified value.

## Reference Solution
```python
def rotate_string(s, n):
    if n > len(s):
        n = n % len(s)
    return s[-n:] + s[:-n]

print(rotate_string("Hello, World!", 5))
print(rotate_string("I love Python!", 13))
```
