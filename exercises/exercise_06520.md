## Exercise
Write a program to represent different data types (int, float, str, etc.) in a specific format.

## Reference Solution
```python
# Define a dictionary with different data types
data_types = {
    "Integer": 10,
    "Float": 10.5,
    "String": "Hello, World!",
    "Boolean": True,
    "List": [1, 2, 3, 4, 5],
    "Tuple": (1, 2, 3, 4, 5),
    "Dictionary": {"key1": "value1", "key2": "value2"},
    "Set": {1, 2, 3, 4, 5},
}

# Print the data types and their values
for data_type, value in data_types.items():
    print(f"{data_type}: {value} ({type(value).__name__})")
```
