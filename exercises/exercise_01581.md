## Exercise
Write a program to take a user's name as input and print out a greeting message.

## Reference Solution
```python
def print_greeting(name):
    """
    Prints out a greeting message to the user.

    Args:
        name (str): The user's name.
    """
    print(f"Hello, {name}!")

def main():
    # Get the user's name
    name = input("Please enter your name: ")

    # Print out the greeting message
    print_greeting(name)

if __name__ == "__main__":
    main()
```
