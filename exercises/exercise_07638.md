## Exercise
Create a program that uses a while loop to simulate the Collatz Conjecture for a given number.

## Reference Solution
```python
def collatz_conjecture(n):
    """
    Simulates the Collatz Conjecture for a given number.

    Args:
        n (int): The starting number.

    Returns:
        list: A list of numbers generated by the Collatz Conjecture process.
    """
    sequence = [n]
    while n != 1:
        if n % 2 == 0:
            n = n // 2
        else:
            n = 3 * n + 1
        sequence.append(n)
    return sequence

def main():
    num = int(input("Enter a number: "))
    sequence = collatz_conjecture(num)
    print("Collatz Conjecture sequence:")
    print(sequence)

if __name__ == "__main__":
    main()
```
