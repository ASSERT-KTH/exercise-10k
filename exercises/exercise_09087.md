## Exercise
Create a program that prints a square pattern of asterisks (*) using loops. The size of the square should be determined by the user's input. For example, if the user inputs 5, the program should print a 5x5 square of asterisks.

## Reference Solution
```python
#include <iostream>
using namespace std;

int main() {
  int n;
  cout << "Enter a number: ";
  cin >> n;

  for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
      cout << "* ";
    }
    cout << endl;
  }

  return 0;
}
```
