## Exercise
Write a function that takes a string as input and returns the string with all HTML tags removed.

## Reference Solution
```python
import re

def remove_html_tags(input_string):
    """
    This function takes a string as input, removes all HTML tags, and returns the resulting string.

    Args:
        input_string (str): The input string that may contain HTML tags.

    Returns:
        str: The input string with all HTML tags removed.
    """

    # First, we check if the input is a string. If it's not, we raise a TypeError.
    if not isinstance(input_string, str):
        raise TypeError("Input must be a string.")

    # We use the sub function from the re module, which replaces substrings in a string based on a regular expression pattern.
    # The pattern '<.*?>' matches any HTML tag. The '?' after '*' makes the match non-greedy, so it stops at the first '>'.
    # We replace each match with an empty string, effectively removing the HTML tags.
    cleaned_string = re.sub('<.*?>', '', input_string)

    # Finally, we return the cleaned string.
    return cleaned_string
```
