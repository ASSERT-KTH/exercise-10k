## Exercise
Write a function that joins a list of substrings into a single string separated by a given delimiter.

## Reference Solution
```python
def join_substrings(substrings, delimiter):
    """
    Join a list of substrings into a single string separated by a given delimiter.

    Args:
        substrings (list): A list of substrings to be joined.
        delimiter (str): The delimiter to separate the substrings.

    Returns:
        str: The joined string.
    """
    return delimiter.join(substrings)

# Example usage:
def main():
    substrings = ["Hello", "World", "This", "Is", "A", "Test"]
    delimiter = ", "
    print(join_substrings(substrings, delimiter))

if __name__ == "__main__":
    main()
```
