## Exercise
Write a program to print the elements of a list in a specific order (e.g. odd numbers first, then even numbers) using a while loop

## Reference Solution
```python
def print_odd_even_numbers(num_list):
    # Initialize two lists to store odd and even numbers
    odd_numbers = [num for num in num_list if num % 2 != 0]
    even_numbers = [num for num in num_list if num % 2 == 0]

    # Initialize index for while loop
    i = 0
    j = 0

    # Print odd numbers first, then even numbers
    while i < len(odd_numbers) or j < len(even_numbers):
        if i < len(odd_numbers):
            print(odd_numbers[i])
            i += 1
        if j < len(even_numbers):
            print(even_numbers[j])
            j += 1

# Test the function
```
