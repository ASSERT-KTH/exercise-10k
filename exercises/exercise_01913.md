## Exercise
Write a program that calculates the percentage increase given the original and new values.

## Reference Solution
```python
def calculate_percentage_increase(original_value, new_value):
    """
    Calculate the percentage increase given the original and new values.

    Args:
        original_value (float): The original value.
        new_value (float): The new value.

    Returns:
        float: The percentage increase.
    """
    if original_value == 0:
        raise ValueError("Original value cannot be zero")

    percentage_increase = ((new_value - original_value) / original_value) * 100
    return percentage_increase


def main():
    original_value = float(input("Enter the original value: "))
    new_value = float(input("Enter the new value: "))

    try:
        percentage_increase = calculate_percentage_increase(original_value, new_value)
        print(f"The percentage increase is: {percentage_increase:.2f}%")
    except ValueError as e:
        print(e)


if __name__ == "__main__":
    main()
```
