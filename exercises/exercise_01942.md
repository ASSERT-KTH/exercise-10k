## Exercise
Write a program that takes a list and two indices as input and returns a slice of the list from the first index to the second index.

## Reference Solution
```python
def list_slice(lst, start, stop):
    return lst[start:stop]
```

```python
my_list = [1, 2, 3, 4, 5]
print(list_slice(my_list, 1, 3))  # Output: [2, 3]
```
