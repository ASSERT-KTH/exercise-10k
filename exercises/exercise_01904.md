## Exercise
Calculate and display the circumference and area of a circle given the radius.

## Reference Solution
```python
import math

def calculate_circle_properties(radius):
    """
    Calculates the circumference and area of a circle given the radius.

    Args:
    radius (float): The radius of the circle.

    Returns:
    tuple: A tuple containing the circumference and area of the circle.
    """
    if radius < 0:
        raise ValueError("Radius cannot be negative")

    circumference = 2 * math.pi * radius
    area = math.pi * (radius ** 2)

    return circumference, area

def main():
    radius = float(input("Enter the radius of the circle: "))
    circumference, area = calculate_circle_properties(radius)

    print(f"Circumference: {circumference:.2f}")
    print(f"Area: {area:.2f}")

if __name__ == "__main__":
    main()
```
