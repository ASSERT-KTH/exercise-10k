## Exercise
Create a program to transform a given dictionary into a different data structure, such as a list of tuples.

## Reference Solution
```python
def dict_to_tuple_list(input_dict):
    """
    Transforms a dictionary into a list of tuples.

    Args:
        input_dict (dict): The dictionary to be transformed.

    Returns:
        list: A list of tuples, where each tuple contains a key-value pair from the input dictionary.
    """
    return list(input_dict.items())

# Example usage:
if __name__ == "__main__":
    example_dict = {"name": "John", "age": 30, "city": "New York"}
    print("Input Dictionary:")
    print(example_dict)

    tuple_list = dict_to_tuple_list(example_dict)
    print("\nTransformed List of Tuples:")
    print(tuple_list)
```
