## Exercise
Create a program that checks if a given string is a palindrome. The program should ignore non-alphanumeric characters and be case-insensitive. It should return 'True' if the string is a palindrome and 'False' otherwise.

## Reference Solution
```python
def check_palindrome(input_string):
    s = ''.join(filter(str.isalnum, input_string)).lower()
    return s == s[::-1]

# Test cases
print(check_palindrome("A man, a plan, a canal: Panama"))  # True
print(check_palindrome("Not a palindrome"))  # False
print(check_palindrome("Was it a car or a cat I saw?"))  # True
print(check_palindrome("No 'x' in Nixon"))  # True
print(check_palindrome("Able was I ere I saw Elba"))  # True
```
