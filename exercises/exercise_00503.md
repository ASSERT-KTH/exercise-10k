## Exercise
Write a program to extract a subset of elements from a given list using slicing.

## Reference Solution
```python
def extract_subset(input_list, start, end):
    """
    Extracts a subset of elements from the input list using slicing.
    
    Args:
        input_list (list): The input list from which elements are to be extracted.
        start (int): The starting index of the subset.
        end (int): The ending index of the subset.
        
    Returns:
        list: A subset of elements from the input list.
    """
    return input_list[start:end]

def main():
    # Example usage
    input_list = [1, 2, 3, 4, 5, 6, 7, 8, 9]
    start = 2
    end = 6
    
    subset = extract_subset(input_list, start, end)
    print("Input List:", input_list)
    print("Start Index:", start)
    print("End Index:", end)
    print("Extracted Subset:", subset)

if __name__ == "__main__":
    main()
```
