## Exercise
Create a function that uses a docstring and type hints to document its behavior and specify the types of its arguments and return value

## Reference Solution
```python
def greet(name: str) -> str:
    """
    Returns a greeting message for the given name.

    Args:
        name (str): The name to include in the greeting.

    Returns:
        str: A greeting message including the given name.
    """
    return f"Hello, {name}!"

# Test the function
print(greet("John"))  # Output: Hello, John!
```
