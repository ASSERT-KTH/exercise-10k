## Exercise
Write a program that takes an array of integers as input and returns the maximum value in the array. Assume the array has at least one element.

## Reference Solution
```python
function maxElement(arr) {
  let max = arr[0];
  for (let i = 1; i < arr.length; i++) {
    if (arr[i] > max) {
      max = arr[i];
    }
  }
  return max;
}

let arr = [];
for (let i = 0; i < 10; i++) {
  arr.push(Math.floor(Math.random() * 100));
}

console.log(arr);
console.log(maxElement(arr));
```
