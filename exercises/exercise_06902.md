## Exercise
Write a program that takes an integer as input and prints out whether it is positive, negative, or zero. The program should also check if the number is a multiple of 3 and print out a corresponding message.

## Reference Solution
```python
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int x = scanner.nextInt();

        if (x > 0) {
            System.out.println("Positive");
        } else if (x < 0) {
            System.out.println("Negative");
        } else {
            System.out.println("Zero");
        }

        if (x % 3 == 0) {
            System.out.println("Multiple of 3");
        }
    }
}
```
