## Exercise
Create a program to calculate the sum of powers of a number (e.g., 2^1 + 2^2 + 2^3 + ... + 2^n).

## Reference Solution
```python
def sum_of_powers(base, n):
    """
    Calculate the sum of powers of a number.

    Args:
        base (int): The base number.
        n (int): The maximum power.

    Returns:
        int: The sum of powers of the base number up to n.
    """
    return sum(base ** i for i in range(1, n + 1))


def main():
    # Example usage:
    base = int(input("Enter the base number: "))
    n = int(input("Enter the maximum power: "))

    result = sum_of_powers(base, n)
    print(f"The sum of powers of {base} up to {n} is: {result}")


if __name__ == "__main__":
    main()
```
