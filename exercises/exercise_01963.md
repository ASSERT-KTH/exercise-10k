## Exercise
Create a function that calculates the result of a mathematical expression. The function should take a string as an argument, where the string is a mathematical expression, and return the result.

## Reference Solution
```python
def calculate_expression(expression):
    """
    This function calculates the result of a mathematical expression.
    
    Args:
        expression (str): A string representing a mathematical expression.
    
    Returns:
        float: The result of the mathematical expression.
    """
    
    # Check if the input is a string. If not, raise a TypeError.
    if not isinstance(expression, str):
        raise TypeError("The input must be a string.")
    
    # Try to evaluate the expression. If it's not a valid mathematical expression, a SyntaxError will be raised.
    try:
        # Use the eval function to evaluate the expression. The eval function parses the expression passed to this method and executes Python expression(s) passed as a string.
        result = eval(expression)
        
        # Return the result of the expression.
        return result
    
    # Catch any exceptions that occur during evaluation, such as SyntaxError or NameError.
    except Exception as e:
        # Raise a ValueError with the exception message.
        raise ValueError(f"Failed to evaluate the expression: {str(e)}")

# Test the function
print(calculate_expression("1 + 2 * 3"))  # Output: 7
print(calculate_expression("10 / 2"))  # Output: 5.0
print(calculate_expression("5 - 3 + 2"))  # Output: 4
```
