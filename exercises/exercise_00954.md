## Exercise
Write a program that calculates the maximum of a list of numbers using a while loop.

## Reference Solution
```python
def calculate_max(numbers):
    max_num = numbers[0]
    i = 1
    while i < len(numbers):
        if numbers[i] > max_num:
            max_num = numbers[i]
        i += 1
    return max_num

# Test the function
numbers = [12, 45, 7, 23, 56, 89, 34]
print("The maximum number in the list is:", calculate_max(numbers))
```
