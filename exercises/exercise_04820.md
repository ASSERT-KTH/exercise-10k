## Exercise
Given a string, convert it into a zigzag pattern and then read off the characters from the top row to the bottom row. The function should take in a string and a number of rows as input and return the converted string.

## Reference Solution
```python
def convert(s, numRows):
    if numRows == 1 or numRows >= len(s):
        return s

    rows = [''] * numRows
    index, step = 0, 1

    for char in s:
        rows[index] += char
        if index == 0:
            step = 1
        elif index == numRows - 1:
            step = -1
        index += step

    return ''.join(rows)
```
