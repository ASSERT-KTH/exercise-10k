## Exercise
Swap the values of two variables without using a temporary variable.

## Reference Solution
```python
def swap_arithmetic(a, b):
    """Swap values using arithmetic operations."""
    a = a + b  # a now contains the sum of a and b
    b = a - b  # b now contains the original value of a
    a = a - b  # a now contains the original value of b
    return a, b

# Example usage:
a, b = 5, 10
print("Before swap:", a, b)
a, b = swap_arithmetic(a, b)
print("After swap:", a, b)
```

def swap_tuple(a, b):
    """Swap values using tuple packing and unpacking."""
    a, b = b, a  # direct swap
    return a, b
