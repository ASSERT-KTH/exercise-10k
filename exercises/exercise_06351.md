## Exercise
Explain the concept of a list data type and provide examples of declaring and using lists.

## Reference Solution
```python
# Declare a list
my_list = [1, 2, 3, "hello", 4.5]

# Access an element by index
print(my_list[0])  # Output: 1

# Append an element to the end of the list
my_list.append(6)
print(my_list)  # Output: [1, 2, 3, 'hello', 4.5, 6]

# Iterate over the list using a for loop
for element in my_list:
    print(element)
```

```python
// Declare a list
let myList = [1, 2, 3, "hello", 4.5];

// Access an element by index
console.log(myList[0]);  // Output: 1

// Append an element to the end of the list
myList.push(6);
console.log(myList);  // Output: [1, 2, 3, 'hello', 4.5, 6]

// Iterate over the list using a for loop
for (let element of myList) {
    console.log(element);
}
```
