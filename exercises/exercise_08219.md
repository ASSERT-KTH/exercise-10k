## Exercise
Write a program that checks if a given character is a vowel using if/else statements

## Reference Solution
```python
def is_vowel(char):
    """
    Checks if a given character is a vowel.

    Args:
        char (str): The character to check.

    Returns:
        bool: True if the character is a vowel, False otherwise.
    """
    char = char.lower()  # Convert to lowercase for simplicity
    if char in 'aeiou':
        return True
    else:
        return False

def main():
    char = input("Enter a character: ")
    if is_vowel(char):
        print(f"'{char}' is a vowel.")
    else:
        print(f"'{char}' is not a vowel.")

if __name__ == "__main__":
    main()
```
