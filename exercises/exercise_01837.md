## Exercise
Write a program to calculate the product of all elements in a given array.

## Reference Solution
```python
def calculate_array_product(array):
    """
    Calculates the product of all elements in a given array.

    Args:
        array (list): A list of numbers.

    Returns:
        float: The product of all elements in the array.
    """
    product = 1
    for num in array:
        product *= num
    return product

def main():
    # Example usage
    array = [1, 2, 3, 4, 5]
    print("Array:", array)
    print("Product:", calculate_array_product(array))

if __name__ == "__main__":
    main()
```
