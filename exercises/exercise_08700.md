## Exercise
Ask the user for a string and print out the number of words it contains.

## Reference Solution
```python
def count_words():
    """
    Asks the user for a string and prints out the number of words it contains.
    """
    user_string = input("Please enter a string: ")
    word_count = len(user_string.split())
    print(f"The string contains {word_count} words.")

def main():
    count_words()

if __name__ == "__main__":
    main()
```
