## Exercise
Create a function to calculate the reverse of a given number.

## Reference Solution
```python
def reverse_number(n):
    """
    Reverses a given integer.

    Args:
        n (int): The number to be reversed.

    Returns:
        int: The reversed number.
    """
    sign = -1 if n < 0 else 1
    n *= sign

    # Reverse the number
    reversed_n = int(str(n)[::-1])

    return sign * reversed_n

# Example usage:
if __name__ == "__main__":
    num = 12345
    print(f"Reverse of {num} is: {reverse_number(num)}")
```
