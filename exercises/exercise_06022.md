## Exercise
Write a function that takes a name as an argument and returns a personalized greeting message. Use the function to greet a user.

## Reference Solution
```python
def generate_greeting(name: str) -> str:
    """
    Generates a personalized greeting message.

    Args:
        name (str): The name of the person to greet.

    Returns:
        str: A greeting message with the provided name.
    """
    return f"Hello, {name}!"

def main():
    # Get the user's name
    user_name = input("Please enter your name: ")

    # Generate and print the greeting message
    greeting_message = generate_greeting(user_name)
    print(greeting_message)

if __name__ == "__main__":
    main()
```
