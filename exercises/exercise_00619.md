## Exercise
Write a program to calculate the range of a set of numbers.

## Reference Solution
```python
def calculate_range(numbers):
    """
    Calculate the range of a set of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        float: The range of the numbers.
    """
    if not numbers:
        raise ValueError("Input list is empty")

    return max(numbers) - min(numbers)


def main():
    # Example usage
    numbers = [1, 2, 3, 4, 5]
    range_value = calculate_range(numbers)
    print(f"The range of the numbers {numbers} is {range_value}")


if __name__ == "__main__":
    main()
```
