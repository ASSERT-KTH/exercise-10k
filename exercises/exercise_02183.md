## Exercise
Write a function to concatenate two or more strings. The function should take multiple strings as input and return a single string that is the result of concatenating all the input strings.

## Reference Solution
```python
def concatenate_strings(*args):
    """
    This function concatenates two or more strings.
    
    Args:
        *args: A variable number of strings to be concatenated.
    
    Returns:
        A single string that is the result of concatenating all the input strings.
    """

    # Initialize an empty string to store the concatenated result
    result = ""

    # Iterate over each string in the input arguments
    for arg in args:
        # Check if the current argument is a string
        if isinstance(arg, str):
            # Concatenate the current string to the result
            result += arg
        else:
            # If the argument is not a string, raise a TypeError
            raise TypeError("All arguments must be strings")

    # Return the concatenated string
    return result

# Example usage:
print(concatenate_strings("Hello", " ", "World"))  # Outputs: "Hello World"
```
