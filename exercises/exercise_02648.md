## Exercise
Write a function to get a list of keys from a dictionary

## Reference Solution
```python
def get_dictionary_keys(dictionary):
    """
    Returns a list of keys from a given dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        list: A list of dictionary keys.
    """
    return list(dictionary.keys())

# Example usage:
if __name__ == "__main__":
    example_dict = {"name": "John", "age": 30, "city": "New York"}
    keys = get_dictionary_keys(example_dict)
    print(keys)  # Output: ['name', 'age', 'city']
```
