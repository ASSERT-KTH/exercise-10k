## Exercise
Write a function to remove duplicate values from a dictionary

## Reference Solution
```python
def remove_duplicate_values(dictionary):
    """
    Remove duplicate values from a dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        dict: A new dictionary with unique values.
    """
    seen = set()
    return {key: value for key, value in dictionary.items() if not (value in seen or seen.add(value))}

# Example usage:
dictionary = {
    "apple": 1,
    "banana": 2,
    "cherry": 1,
    "date": 3,
    "elderberry": 2
}

unique_dict = remove_duplicate_values(dictionary)
print(unique_dict)
```
