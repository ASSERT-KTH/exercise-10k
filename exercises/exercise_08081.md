## Exercise
Write a function that uses a generator expression to create a generator

## Reference Solution
```python
def generate_squares(start, end):
    """
    Generate the squares of numbers in a given range.

    Args:
        start (int): The start of the range (inclusive).
        end (int): The end of the range (inclusive).

    Yields:
        int: The square of a number in the range.
    """
    return (x ** 2 for x in range(start, end + 1))

# Example usage:
if __name__ == "__main__":
    start_range = 1
    end_range = 5

    print(f"Generating squares from {start_range} to {end_range}:")
    for square in generate_squares(start_range, end_range):
        print(square)
```
