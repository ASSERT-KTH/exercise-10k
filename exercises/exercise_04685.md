## Exercise
Write a function that takes a string as input and returns a dictionary with the frequency of each word.

## Reference Solution
```python
from collections import Counter
import re

def word_frequency(input_string):
    """
    Calculate the frequency of each word in a string.

    Args:
        input_string (str): The input string.

    Returns:
        dict: A dictionary with words as keys and their frequencies as values.
    """
    # Remove punctuation and convert to lower case
    cleaned_string = re.sub(r'[^\w\s]', '', input_string).lower()
    
    # Split the string into words
    words = cleaned_string.split()
    
    # Count the frequency of each word
    frequency = Counter(words)
    
    return dict(frequency)

# Example usage
if __name__ == "__main__":
    input_string = "This is a test string. The string is just a test."
    print(word_frequency(input_string))
```
