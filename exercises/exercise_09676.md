## Exercise
Write a program that asks for two strings and checks if the first string is a prefix of the second string.

## Reference Solution
```python
def is_prefix(str1, str2):
    """
    Checks if str1 is a prefix of str2.
    
    Args:
        str1 (str): The potential prefix string.
        str2 (str): The string to check against.
    
    Returns:
        bool: True if str1 is a prefix of str2, False otherwise.
    """
    return str2.startswith(str1)

def main():
    # Get input strings from user
    str1 = input("Enter the first string: ")
    str2 = input("Enter the second string: ")
    
    # Check if str1 is a prefix of str2
    if is_prefix(str1, str2):
        print(f"'{str1}' is a prefix of '{str2}'")
    else:
        print(f"'{str1}' is not a prefix of '{str2}'")

if __name__ == "__main__":
    main()
```
