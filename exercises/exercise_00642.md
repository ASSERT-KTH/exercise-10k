## Exercise
Create a program that continuously reads and prints input from the user until the user types 'exit'.

## Reference Solution
```python
def main():
    while True:
        user_input = input("Please enter a message (type 'exit' to quit): ")
        if user_input.lower() == 'exit':
            print("Exiting program.")
            break
        else:
            print(f"You entered: {user_input}")

if __name__ == "__main__":
    main()
```
