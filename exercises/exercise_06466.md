## Exercise
Write a function that takes a dictionary as input and returns a list of all the keys

## Reference Solution
```python
def get_dict_keys(input_dict):
    """
    Returns a list of all keys in the input dictionary.

    Args:
        input_dict (dict): The dictionary to extract keys from.

    Returns:
        list: A list of dictionary keys.
    """
    return list(input_dict.keys())

# Example usage:
my_dict = {"name": "John", "age": 30, "city": "New York"}
print(get_dict_keys(my_dict))  # Output: ['name', 'age', 'city']
```
