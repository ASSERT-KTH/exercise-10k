## Exercise
Create a dictionary with 5 key-value pairs. Check if the dictionary contains a specific key.

## Reference Solution
```python
# Create a dictionary with 5 key-value pairs
my_dict = {
    "name": "John",
    "age": 30,
    "city": "New York",
    "country": "USA",
    "job": "Developer"
}

# Define a function to check if a key exists in the dictionary
def check_key(dictionary, key):
    """
    Check if a key exists in the dictionary.

    Args:
        dictionary (dict): The dictionary to search in.
        key (str): The key to search for.

    Returns:
        bool: True if the key exists, False otherwise.
    """
    return key in dictionary

# Check if the dictionary contains a specific key
key_to_check = "age"
result = check_key(my_dict, key_to_check)

# Print the result
print(f"Does the dictionary contain the key '{key_to_check}'? {result}")

# Alternative way to check without a function
key_to_check = "city"
result = key_to_check in my_dict
print(f"Does the dictionary contain the key '{key_to_check}'? {result}")
```
