## Exercise
Write a function to convert a dictionary to a list of tuples

## Reference Solution
```python
def dict_to_tuple_list(dictionary):
    """
    Converts a dictionary to a list of tuples.
    
    Args:
        dictionary (dict): The dictionary to be converted.
    
    Returns:
        list: A list of tuples, where each tuple contains a key-value pair from the dictionary.
    """
    return list(dictionary.items())

# Example usage:
if __name__ == "__main__":
    example_dict = {"apple": 1, "banana": 2, "cherry": 3}
    tuple_list = dict_to_tuple_list(example_dict)
    print(tuple_list)
```
