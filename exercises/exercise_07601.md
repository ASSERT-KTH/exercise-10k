## Exercise
Write a program that checks if a given variable is null.

## Reference Solution
```python
def check_null(var):
    """
    Checks if a given variable is null.

    Args:
        var: The variable to check.

    Returns:
        bool: True if the variable is null, False otherwise.
    """
    return var is None


# Example usage:
if __name__ == "__main__":
    null_var = None
    not_null_var = "Hello, World!"

    print(f"Is null_var null? {check_null(null_var)}")
    print(f"Is not_null_var null? {check_null(not_null_var)}")
```
