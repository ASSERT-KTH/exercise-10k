## Exercise
Write a function that takes a name as an argument and prints a greeting message. The function should have a default argument for the name.

## Reference Solution

# Define a function called 'greet' with a default argument 'name' set to 'World'.
def greet(name='World'):
    """
    Prints a greeting message with the provided name.

    Args:
        name (str): The name to be used in the greeting. Defaults to 'World'.

    Returns:
        None
    """

    # Use an f-string to format the greeting message with the provided name.
    # This allows us to insert the value of 'name' directly into the string.
    greeting_message = f"Hello, {name}!"

    # Print the greeting message.
    print(greeting_message)


# Test the function by calling it without any arguments.
# This should print 'Hello, World!' because 'World' is the default name.
greet()

# Test the function by calling it with a custom name.
# This should print 'Hello, Alice!' because we passed 'Alice' as the name.
greet('Alice')
