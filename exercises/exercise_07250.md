## Exercise
Create a program that takes in two numbers and an arithmetic operation (+, -, *, /) as input and performs the given operation on the two numbers. The program should then output the result of the operation.

## Reference Solution
```python
def main():
    print("Arithmetic Operations Calculator")
    print("--------------------------------")

    x = int(input("Enter the first number: "))
    y = int(input("Enter the second number: "))
    z = input("Enter the arithmetic operation (+, -, *, /): ")

    if z == "+":
        print("Result:", x + y)
    elif z == "-":
        print("Result:", x - y)
    elif z == "*":
        print("Result:", x * y)
    elif z == "/":
        if y == 0:
            print("Error: Division by zero is not allowed.")
        else:
            print("Result:", x / y)
    else:
        print("Error: Invalid operation. Please enter +, -, *, or /.")

if __name__ == "__main__":
    main()
```
