## Exercise
Write a program that takes a list, an index, and an element as input and inserts the element at the specified index in the list.

## Reference Solution
```python
def insert_element(lst, index, element):
    if index >= 0 and index <= len(lst):
        lst.insert(index, element)
        return lst
    else:
        return "Invalid index"
```
