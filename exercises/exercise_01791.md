## Exercise
Create a function to nest a dictionary within another dictionary and return the resulting nested dictionary.

## Reference Solution
```python
def nest_dictionaries(outer_dict, inner_dict, key='nested'):
    """
    Nest a dictionary within another dictionary.

    Args:
    - outer_dict (dict): The outer dictionary.
    - inner_dict (dict): The inner dictionary to be nested.
    - key (str): The key under which the inner dictionary will be nested. Defaults to 'nested'.

    Returns:
    - dict: The resulting nested dictionary.
    """
    # Create a copy of the outer dictionary to avoid modifying the original
    nested_dict = outer_dict.copy()
    
    # Update the copy with the inner dictionary
    nested_dict[key] = inner_dict
    
    return nested_dict

# Example usage:
outer_dict = {'a': 1, 'b': 2}
inner_dict = {'x': 10, 'y': 20}

nested_result = nest_dictionaries(outer_dict, inner_dict)
print(nested_result)

# Custom key example:
custom_nested_result = nest_dictionaries(outer_dict, inner_dict, key='custom')
print(custom_nested_result)
```
