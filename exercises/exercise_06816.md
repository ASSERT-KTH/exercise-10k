## Exercise
Write a program that prints a pyramid pattern of asterisks (*) based on the input number of rows. The pattern should be centered and symmetrical. For example, if the input is 5, the output should be a pyramid with 5 rows.

## Reference Solution
```python
n = int(input("Enter the number of rows: "))

for i in range(n):
    print(" " * (n - i - 1) + "*" * (2 * i + 1))
```
