## Exercise
Write a program that takes a number as input and returns the product of its digits.

## Reference Solution
```python
def product_of_digits(n):
    """
    This function calculates the product of the digits of a given number.

    Args:
        n (int): The input number.

    Returns:
        int: The product of the digits of the input number.
    """
    product = 1
    for digit in str(n):
        product *= int(digit)
    return product

# Test the function
num = int(input("Enter a number: "))
print("The product of its digits is:", product_of_digits(num))
```
