## Exercise
Write a program to calculate the floor and ceiling of a given number.

## Reference Solution
```python
import math

def calculate_floor_and_ceiling(num):
    """
    Calculate the floor and ceiling of a given number.

    Args:
        num (float): The input number.

    Returns:
        tuple: A tuple containing the floor and ceiling of the input number.
    """
    floor_val = math.floor(num)
    ceil_val = math.ceil(num)
    return floor_val, ceil_val

def main():
    # Example usage
    num = float(input("Enter a number: "))
    floor_val, ceil_val = calculate_floor_and_ceiling(num)
    print(f"Floor: {floor_val}")
    print(f"Ceiling: {ceil_val}")

if __name__ == "__main__":
    main()
```
