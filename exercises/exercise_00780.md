## Exercise
Write a program that splits a given string into substrings based on a given separator and prints out the result.

## Reference Solution
```python
def split_string(input_str, separator):
    """
    Splits a given string into substrings based on a given separator.

    Args:
        input_str (str): The input string to be split.
        separator (str): The separator used to split the string.

    Returns:
        list: A list of substrings split from the input string.
    """
    try:
        # Check if the input string and separator are of type string
        if not isinstance(input_str, str) or not isinstance(separator, str):
            raise TypeError("Both input string and separator must be of type string.")

        # Split the input string into substrings based on the separator
        substrings = input_str.split(separator)

        # Print the result
        print("Substrings:")
        for i, substring in enumerate(substrings):
            print(f"Substring {i+1}: {substring}")

        return substrings

    except TypeError as e:
        print(f"Error: {e}")

# Example usage
input_str = "apple,banana,cherry"
separator = ","
split_string(input_str, separator)
```
