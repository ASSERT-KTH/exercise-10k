## Exercise
Write a program that asks for a string and encodes each character using its ASCII value.

## Reference Solution
```python
def ascii_encoder(input_string):
    """
    Encodes each character in the input string using its ASCII value.

    Args:
        input_string (str): The input string to be encoded.

    Returns:
        list: A list of ASCII values corresponding to each character in the input string.
    """
    ascii_values = [ord(char) for char in input_string]
    return ascii_values

def main():
    # Ask for user input
    input_string = input("Please enter a string: ")

    # Encode the input string using ASCII values
    encoded_string = ascii_encoder(input_string)

    # Print the encoded string
    print("Encoded string (ASCII values):")
    print(encoded_string)

    # Print the encoded string with character and ASCII value mapping
    print("\nCharacter-ASCII Value Mapping:")
    for char, ascii_value in zip(input_string, encoded_string):
        print(f"{char} -> {ascii_value}")

if __name__ == "__main__":
    main()
```
