## Exercise
Write a program to take a string input from the user and print it to the screen, converting it to uppercase or lowercase.

## Reference Solution
```python
def convert_case(input_string, case):
    """
    Converts the input string to the specified case.

    Args:
    input_string (str): The string to be converted.
    case (str): The case to convert to. It can be 'upper' or 'lower'.

    Returns:
    str: The converted string.
    """
    if case == 'upper':
        return input_string.upper()
    elif case == 'lower':
        return input_string.lower()
    else:
        return "Invalid case. Please specify 'upper' or 'lower'."

def main():
    user_string = input("Please enter a string: ")
    print("Do you want to convert to:")
    print("1. Uppercase")
    print("2. Lowercase")
    choice = input("Enter your choice (1/2): ")
    
    if choice == '1':
        print(convert_case(user_string, 'upper'))
    elif choice == '2':
        print(convert_case(user_string, 'lower'))
    else:
        print("Invalid choice. Please try again.")

if __name__ == "__main__":
    main()
```
