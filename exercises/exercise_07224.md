## Exercise
Write a program that finds the maximum value in an array of integers. The program should take an array as input and return the maximum value. If the array is empty, the program should return a message indicating that the array is empty.

## Reference Solution
```python
def findMax(array):
    if not array:
        return "Array is empty"
    return max(array)

array = [1, 2, 3, 4, 5]
print(findMax(array))  # Output: 5
```
