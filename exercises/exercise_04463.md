## Exercise
Write a program to find the maximum number in a list using a for loop

## Reference Solution
```python
def find_max_number(num_list):
    """
    This function finds the maximum number in a given list.

    Args:
        num_list (list): A list of numbers.

    Returns:
        int: The maximum number in the list.
    """
    # Check if the list is empty
    if not num_list:
        return None

    # Initialize the maximum number with the first element of the list
    max_number = num_list[0]

    # Iterate over the list to find the maximum number
    for num in num_list:
        if num > max_number:
            max_number = num

    return max_number


# Example usage
if __name__ == "__main__":
    numbers = [12, 45, 7, 23, 56, 89, 34]
    max_num = find_max_number(numbers)
    print("Maximum number in the list:", max_num)
```
