## Exercise
Create a JSON string with 3 key-value pairs and deserialize it to a dictionary.

## Reference Solution
```python
import json

def create_and_deserialize_json():
    # Create a JSON string with 3 key-value pairs
    json_string = '{"name": "John", "age": 30, "city": "New York"}'

    # Deserialize the JSON string to a dictionary
    dictionary = json.loads(json_string)

    return dictionary

def main():
    dictionary = create_and_deserialize_json()
    print(dictionary)

if __name__ == "__main__":
    main()
```
