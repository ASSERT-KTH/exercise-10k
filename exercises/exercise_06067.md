## Exercise
Write a program that calculates the sum of all elements in an array using a for loop.

## Reference Solution
```python
def calculate_array_sum(array):
    """
    Calculate the sum of all elements in the given array.
    
    Args:
    array (list): The input array.
    
    Returns:
    int: The sum of all elements in the array.
    """
    total_sum = 0  # Initialize the sum variable to 0
    
    # Iterate through each element in the array
    for element in array:
        total_sum += element  # Add the current element to the total sum
    
    return total_sum  # Return the calculated sum


# Example usage:
if __name__ == "__main__":
    array = [1, 2, 3, 4, 5]  # Sample array
    print("Array:", array)
    print("Sum of array elements:", calculate_array_sum(array))
```
