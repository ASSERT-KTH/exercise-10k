## Exercise
Write a function to get the number of key-value pairs in a dictionary

## Reference Solution
```python
def get_dict_length(dictionary):
    """
    Returns the number of key-value pairs in a dictionary.

    Args:
        dictionary (dict): The dictionary to get the length of.

    Returns:
        int: The number of key-value pairs in the dictionary.
    """
    return len(dictionary)

# Example usage
my_dict = {"key1": "value1", "key2": "value2", "key3": "value3"}
print(get_dict_length(my_dict))  # Output: 3
```
