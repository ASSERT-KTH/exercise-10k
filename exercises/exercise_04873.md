## Exercise
Write a program that generates a random password of length 10 using a while loop.

## Reference Solution
```python
import random
import string

def generate_password(length):
    """
    Generates a random password of given length.

    Args:
    length (int): The length of the password.

    Returns:
    str: The generated password.
    """
    password = ''
    counter = 0
    while counter < length:
        # Choose a random character from the string of all possible characters
        char = random.choice(string.ascii_letters + string.digits + string.punctuation)
        password += char
        counter += 1

    return password

# Generate a password of length 10
password = generate_password(10)
print("Generated Password : ", password)
```
