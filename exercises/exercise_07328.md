## Exercise
Write a program that takes input from the user and writes it to stderr.

## Reference Solution
```python
import sys

def main():
    user_input = input("Please enter a message: ")
    sys.stderr.write(user_input + "\n")
    sys.stderr.flush()

if __name__ == "__main__":
    main()
```

```python
python program.py > output.txt
```
