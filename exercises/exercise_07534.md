## Exercise
Write a program that takes an integer grade as input and prints out whether the grade is passing (60 or above) or failing (below 60) using conditional statements (if/else).

## Reference Solution
```python
#include <iostream>

int main() {
    int grade;
    std::cout << "Enter your grade: ";
    std::cin >> grade;

    if (grade >= 60) {
        std::cout << "Passing" << std::endl;
    } else {
        std::cout << "Failing" << std::endl;
    }

    return 0;
}
```
