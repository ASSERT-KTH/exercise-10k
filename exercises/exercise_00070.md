## Exercise
Write a function to split a given string into a list of substrings based on a delimiter

## Reference Solution
```python
def split_string(input_string: str, delimiter: str) -> list[str]:
    """
    Splits a given string into a list of substrings based on a delimiter.

    Args:
        input_string (str): The input string to be split.
        delimiter (str): The delimiter used to split the string.

    Returns:
        list[str]: A list of substrings split based on the delimiter.
    """
    return input_string.split(delimiter)

# Example usage:
def main():
    input_str = "apple,banana,cherry"
    delimiter = ","
    result = split_string(input_str, delimiter)
    print(result)  # Output: ['apple', 'banana', 'cherry']

if __name__ == "__main__":
    main()
```
