## Exercise
Create a program that uses nested for loops to print a star pattern of a given size.

## Reference Solution
```python
def print_star_pattern(size):
    """
    Prints a star pattern of a given size.

    Args:
        size (int): The size of the star pattern.

    Returns:
        None
    """
    for i in range(size):
        for j in range(size - i - 1):
            print(" ", end="")
        for k in range(2 * i + 1):
            print("*", end="")
        print()

def main():
    size = int(input("Enter the size of the star pattern: "))
    print_star_pattern(size)

if __name__ == "__main__":
    main()
```
