## Exercise
Write a function to filter a dictionary based on a given condition.

## Reference Solution
```python
def filter_dict(input_dict, condition):
    """
    Filter a dictionary based on a given condition.

    Args:
    input_dict (dict): The dictionary to be filtered.
    condition (function): A function that takes a key-value pair and returns True if the pair should be included in the filtered dictionary.

    Returns:
    dict: A new dictionary with only the key-value pairs that meet the specified condition.
    """
    return {key: value for key, value in input_dict.items() if condition(key, value)}

# Example usage:
numbers = {'a': 1, 'b': 2, 'c': 3, 'd': 4}

# Define a condition function that checks if the value is even
def is_even(key, value):
    return value % 2 == 0

# Filter the dictionary using the condition function
filtered_numbers = filter_dict(numbers, is_even)
print(filtered_numbers)  # Output: {'b': 2, 'd': 4}
```
