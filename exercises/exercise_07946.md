## Exercise
Write a program to demonstrate the concept of variable scope in programming.

## Reference Solution
```python
# Global scope
x = 10  # global variable

def outer_function():
    # Enclosing scope
    y = 20  # nonlocal variable

    def inner_function():
        # Local scope
        z = 30  # local variable
        print("Inner function:")
        print("x (global):", x)
        print("y (nonlocal):", y)
        print("z (local):", z)

    inner_function()
    print("Outer function:")
    print("x (global):", x)
    print("y (nonlocal):", y)
    # print("z (local):", z)  # This would raise a NameError

outer_function()
print("Global scope:")
print("x (global):", x)
# print("y (nonlocal):", y)  # This would raise a NameError
# print("z (local):", z)  # This would raise a NameError
```
