## Exercise
Create a function that splits a given string into a list of substrings separated by a given separator.

## Reference Solution
```python
def split_string_by_separator(input_string, separator):
    """
    This function splits a given string into a list of substrings separated by a given separator.

    Args:
        input_string (str): The input string to be split.
        separator (str): The separator used to split the input string.

    Returns:
        list: A list of substrings separated by the given separator.
    """

    # First, we check if the input is a string and the separator is a string
    if not isinstance(input_string, str) or not isinstance(separator, str):
        raise TypeError("Both input_string and separator must be strings")

    # If the separator is an empty string, we return a list of individual characters in the input string
    if separator == "":
        # We use a list comprehension to create a list of individual characters
        return [char for char in input_string]
    
    # We use the built-in split method of Python strings, which splits a string into a list where each word is a list item
    # We pass the separator as an argument to the split method
    # The split method returns a list of substrings separated by the given separator
    substrings = input_string.split(separator)

    # Finally, we return the list of substrings
    return substrings

# Example usage:
print(split_string_by_separator("hello,world,python", ","))  # Output: ['hello', 'world', 'python']
```
