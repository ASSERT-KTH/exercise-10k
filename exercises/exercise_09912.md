## Exercise
Write a program that checks whether a given phone number is valid.

## Reference Solution
```python
import re

def validate_phone_number(phone_number):
    """
    Validate a phone number based on the format and digits.

    Args:
        phone_number (str): The phone number to be validated.

    Returns:
        bool: True if the phone number is valid, False otherwise.
    """
    pattern = re.compile(r'^(\d{3}-\d{3}-\d{4}|\(\d{3}\) \d{3}-\d{4})$')
    return bool(pattern.match(phone_number))

def main():
    phone_number = input("Enter a phone number: ")
    if validate_phone_number(phone_number):
        print("The phone number is valid.")
    else:
        print("The phone number is not valid.")

if __name__ == "__main__":
    main()
```
