## Exercise
Write a program to check if an array is symmetric.

## Reference Solution
```python
def is_symmetric(array):
    """
    Checks if an array is symmetric.

    Args:
        array (list): The input array.

    Returns:
        bool: True if the array is symmetric, False otherwise.
    """
    return array == array[::-1]


def main():
    # Example usage
    arrays = [
        [1, 2, 3, 2, 1],
        [1, 2, 3, 4, 5],
        [1, 1, 1, 1, 1]
    ]

    for array in arrays:
        print(f"Array: {array}")
        print(f"Is symmetric: {is_symmetric(array)}")
        print()


if __name__ == "__main__":
    main()
```
