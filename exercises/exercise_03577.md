## Exercise
Write a function that finds the intersection between two dictionaries

## Reference Solution
```python
def dict_intersection(dict1, dict2):
    """
    Finds the intersection between two dictionaries.

    Args:
        dict1 (dict): The first dictionary.
        dict2 (dict): The second dictionary.

    Returns:
        dict: A dictionary containing the key-value pairs common to both input dictionaries.
    """
    intersection = {key: dict1[key] for key in dict1 if key in dict2 and dict1[key] == dict2[key]}
    return intersection

# Example usage:
if __name__ == "__main__":
    dict1 = {'a': 1, 'b': 2, 'c': 3}
    dict2 = {'b': 2, 'c': 3, 'd': 4}
    print(dict_intersection(dict1, dict2))  # Output: {'b': 2, 'c': 3}
```
