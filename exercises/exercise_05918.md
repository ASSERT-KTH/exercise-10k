## Exercise
Write a function that takes a string and a shift value as input and returns the string with each character shifted by the shift value

## Reference Solution
```python
def shift_string(s, shift):
    """
    Shifts each character in the string by the given shift value.

    Args:
        s (str): The input string.
        shift (int): The shift value.

    Returns:
        str: The string with each character shifted by the shift value.
    """
    result = ""
    for char in s:
        if char.isalpha():
            ascii_offset = 97 if char.islower() else 65
            result += chr((ord(char) - ascii_offset + shift) % 26 + ascii_offset)
        else:
            result += char
    return result

# Example usage
if __name__ == "__main__":
    s = "Hello, World!"
    shift = 3
    print(f"Original string: {s}")
    print(f"Shifted string: {shift_string(s, shift)}")
```
