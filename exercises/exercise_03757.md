## Exercise
Write a program to generate a frequency dictionary for a given list of elements using a basic algorithm

## Reference Solution
```python
def generate_frequency_dict(elements):
    """
    Generate a frequency dictionary for a given list of elements.

    Args:
        elements (list): The list of elements to generate the frequency dictionary for.

    Returns:
        dict: The frequency dictionary where each key is an element and each value is its frequency.
    """
    frequency_dict = {}
    for element in elements:
        if element in frequency_dict:
            frequency_dict[element] += 1
        else:
            frequency_dict[element] = 1
    return frequency_dict

# Example usage
if __name__ == "__main__":
    elements = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]
    frequency_dict = generate_frequency_dict(elements)
    print("Frequency Dictionary:")
    for element, frequency in frequency_dict.items():
        print(f"{element}: {frequency}")
```
