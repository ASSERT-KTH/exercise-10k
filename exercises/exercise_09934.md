## Exercise
Write a function that takes an array of integers as input and returns the reversed array. The function should not use any built-in reverse functions.

## Reference Solution
```python
def reverse(arr):
    left = 0
    right = len(arr) - 1

    while left < right:
        temp = arr[left]
        arr[left] = arr[right]
        arr[right] = temp
        left += 1
        right -= 1

    return arr

arr = [4, 5, 1, 2]
print(reverse(arr))
```
