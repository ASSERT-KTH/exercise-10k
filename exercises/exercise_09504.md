## Exercise
Write a program that takes a string as input and returns the reversed string. For example, if the input is 'hello', the output should be 'olleh'. The program should handle strings of any length and should not use any built-in string reversal functions.

## Reference Solution
```python
def reverse_string(input_string):
    reversed_string = ""
    for char in input_string:
        reversed_string = char + reversed_string
    return reversed_string

print(reverse_string(input("Enter a string: ")))
```
