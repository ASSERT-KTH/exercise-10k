## Exercise
Reduce a dictionary to a single value using the reduce() function

## Reference Solution
```python
from functools import reduce

# Example dictionary
my_dict = {'a': 1, 'b': 2, 'c': 3, 'd': 4}

# Convert dictionary values to a list
dict_values = list(my_dict.values())

# Define a function to add two numbers
def add(x, y):
    return x + y

# Use reduce() to sum all values in the list
result = reduce(add, dict_values)

print("Sum of dictionary values:", result)
```
