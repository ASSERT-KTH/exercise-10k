## Exercise
Determine the ASCII value of a given character.

## Reference Solution
```python
def get_ascii_value(char):
    """
    Returns the ASCII value of a given character.

    Args:
        char (str): The character to get the ASCII value for.

    Returns:
        int: The ASCII value of the character.
    """
    return ord(char)

# Example usage:
char = 'A'
ascii_value = get_ascii_value(char)
print(f"The ASCII value of '{char}' is: {ascii_value}")
```
