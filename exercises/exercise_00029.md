## Exercise
Create a function that takes a number and a limit as input and returns a list of multiples up to the limit.

## Reference Solution
```python
def generate_multiples(num, limit):
    """
    Generate a list of multiples of a given number up to a specified limit.

    Args:
        num (int): The number for which to generate multiples.
        limit (int): The upper limit for the multiples.

    Returns:
        list: A list of multiples of the given number up to the limit.
    """
    return [i for i in range(num, limit + 1) if i % num == 0]

# Example usage:
def main():
    num = 3  # Number for which to generate multiples
    limit = 20  # Upper limit for the multiples
    multiples = generate_multiples(num, limit)
    print(f"Multiples of {num} up to {limit}: {multiples}")

if __name__ == "__main__":
    main()
```
