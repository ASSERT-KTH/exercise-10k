## Exercise
Write a function that takes a dictionary as input and returns the number of key-value pairs

## Reference Solution
```python
def count_key_value_pairs(input_dict):
    """
    Returns the number of key-value pairs in the input dictionary.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        int: The number of key-value pairs.
    """
    return len(input_dict)

# Example usage:
input_dict = {"name": "John", "age": 30, "city": "New York"}
print(count_key_value_pairs(input_dict))  # Output: 3
```
