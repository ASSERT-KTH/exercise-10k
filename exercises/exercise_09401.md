## Exercise
Write a program that calculates the number of digits in a given integer.

## Reference Solution
```python
def count_digits(n):
    """
    Calculate the number of digits in a given integer.

    Args:
        n (int): The input integer.

    Returns:
        int: The number of digits in the input integer.
    """
    return len(str(abs(n)))

def main():
    # Example usage
    num = int(input("Enter an integer: "))
    digit_count = count_digits(num)
    print(f"The number {num} has {digit_count} digits.")

if __name__ == "__main__":
    main()
```

import math

def count_digits(n):
