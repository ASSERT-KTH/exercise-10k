## Exercise
Create a function that iterates over a dictionary and prints each key-value pair.

## Reference Solution
```python
def print_dict_key_value_pairs(dictionary):
    """
    This function iterates over a dictionary and prints each key-value pair.

    Args:
        dictionary (dict): The dictionary to be iterated over.

    Returns:
        None
    """

    # First, we check if the input is indeed a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("Input must be a dictionary")

    # We then iterate over each key-value pair in the dictionary using the items() method
    for key, value in dictionary.items():
        
        # For each key-value pair, we print the key and the value
        print(f"Key: {key}, Value: {value}")

# Example usage:
example_dict = {"name": "John", "age": 30, "city": "New York"}
print_dict_key_value_pairs(example_dict)
```
