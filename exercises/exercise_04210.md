## Exercise
Find the first occurrence of a subarray within a larger array.

## Reference Solution
```python
def find_subarray(main_array, sub_array):
    """
    Find the first occurrence of a subarray within a larger array.

    Args:
        main_array (list): The larger array in which we want to find the subarray.
        sub_array (list): The subarray that we want to find in the larger array.

    Returns:
        int: The index of the first occurrence of the subarray. Returns -1 if not found.
    """
    sub_array_len = len(sub_array)
    main_array_len = len(main_array)

    # Check if the subarray is longer than the main array
    if sub_array_len > main_array_len:
        return -1

    # Iterate over the main array with a sliding window of size equal to the subarray length
    for i in range(main_array_len - sub_array_len + 1):
        # Check if the current window matches the subarray
        if main_array[i:i + sub_array_len] == sub_array:
            return i

    # If no match is found, return -1
    return -1


# Example usage

    main_array = [1, 2, 3, 4, 5, 6, 7, 8, 9]
```
