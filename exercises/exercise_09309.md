## Exercise
Write a function that takes a string and a separator as input and returns a list of substrings separated by the separator.

## Reference Solution
```python
def split_string(input_str, separator):
    """
    This function takes a string and a separator as input, 
    and returns a list of substrings separated by the separator.

    Args:
        input_str (str): The input string to be split.
        separator (str): The separator used to split the string.

    Returns:
        list: A list of substrings separated by the separator.
    """

    # First, we check if the input string is None or not a string
    if not isinstance(input_str, str):
        raise TypeError("Input must be a string.")

    # Then, we check if the separator is None or not a string
    if not isinstance(separator, str):
        raise TypeError("Separator must be a string.")

    # Now, we use the built-in split method of Python strings to split the string
    # This method splits the string into a list where each word is a list item
    # We pass the separator as an argument to this method
    substrings = input_str.split(separator)

    # Finally, we return the list of substrings
    return substrings

# Example usage:
print(split_string("hello world", " "))  # Output: ['hello', 'world']
```
