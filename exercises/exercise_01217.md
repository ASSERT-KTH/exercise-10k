## Exercise
Create a function that finds the last element of a given list.

## Reference Solution
```python
def find_last_element(input_list):
    """
    Finds the last element of a given list.

    Args:
        input_list (list): The input list.

    Returns:
        any: The last element of the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not input_list:
        raise ValueError("Input list is empty")
    return input_list[-1]

# Example usage
if __name__ == "__main__":
    my_list = [1, 2, 3, 4, 5]
    try:
        last_element = find_last_element(my_list)
        print(f"The last element is: {last_element}")
    except ValueError as e:
        print(f"Error: {e}")
```
