## Exercise
Create a program that prints the numbers from 1 to n, where n is provided by the user. Use a for loop to iterate through the numbers and print each one on a new line.

## Reference Solution
```python
# Get the user input for n
n = int(input("Enter a number: "))

# Use a for loop to print the numbers from 1 to n
for i in range(1, n + 1):
    print(i)
```
