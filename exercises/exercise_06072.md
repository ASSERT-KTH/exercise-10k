## Exercise
Create a program that reverses a given string using a for loop.

## Reference Solution
```python
def reverse_string(input_str):
    """
    Reverses a given string using a for loop.

    Args:
        input_str (str): The string to be reversed.

    Returns:
        str: The reversed string.
    """
    reversed_str = ""
    for char in input_str:
        reversed_str = char + reversed_str
    return reversed_str

def main():
    # Example usage
    input_str = "Hello, World!"
    print(f"Original String: {input_str}")
    print(f"Reversed String: {reverse_string(input_str)}")

if __name__ == "__main__":
    main()
```
