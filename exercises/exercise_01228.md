## Exercise
Create a program that uses a for loop to print each key-value pair of a given dictionary.

## Reference Solution
```python
# Define a dictionary
my_dict = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

# Print each key-value pair of the dictionary
for key, value in my_dict.items():
    print(f"{key}: {value}")
```
