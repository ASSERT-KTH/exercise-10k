## Exercise
Write a program that prints a pattern of asterisks (*) using a for loop. The pattern should have n rows and n columns.

## Reference Solution
```python
def print_asterisk_pattern(n):
    """
    Prints a pattern of asterisks (*) with n rows and n columns.

    Args:
        n (int): The number of rows and columns in the pattern.

    Returns:
        None
    """
    for i in range(n):
        for j in range(n):
            print("*", end=" ")
        print()

# Get the input from the user
n = int(input("Enter the number of rows and columns: "))

# Call the function to print the pattern
print_asterisk_pattern(n)
```
