## Exercise
Declare a global variable 'globalVar' and access it within a function.

```python
def access_global_variable():

    Access the global variable within a function.

# Call the function to access the global variable
```

```python

globalVar = "I'm a global variable"

def modify_global_variable():
    """
    Modify the global variable within a function.
    """

    globalVar = "I'm a modified global variable"  # Modify the global variable
    print("Modified global variable within a function:")
    print(globalVar)

# Call the function to modify the global variable
modify_global_variable()
```
