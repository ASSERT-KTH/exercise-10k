## Exercise
Write a program that declares and initializes variables of different data types (integer, float, string, boolean) and prints out their values.

## Reference Solution

/* 
 * This program demonstrates how to declare and initialize variables of different data types in C.
 * It shows how to declare and initialize variables of integer, float, and character types.
 * It also prints out the values of the variables using printf() function.
 */

#include <stdio.h>

int main() {
    // Declare and initialize variables
    int a = 5;
    float b = 3.5;
    char c = 'A';
    char d[] = "Hello, World!";
    _Bool e = 1;

    // Print out variable values
    printf("a = %d\n", a);
    printf("b = %f\n", b);
    printf("c = %c\n", c);
    printf("d = %s\n", d);
    printf("e = %d\n", e);

    return 0;
}
