## Exercise
Create a function that takes a string as an argument and returns a list of lists, where each sublist is a row in the CSV data

## Reference Solution
```python
import csv
import io

def csv_string_to_list(csv_string):
    """
    Parse a CSV string and return a list of lists, where each sublist is a row in the CSV data.

    Args:
        csv_string (str): The CSV string to be parsed.

    Returns:
        list: A list of lists, where each sublist is a row in the CSV data.
    """
    csv_io = io.StringIO(csv_string)
    csv_reader = csv.reader(csv_io)
    csv_data = [row for row in csv_reader]
    return csv_data

# Example usage:
csv_string = "Name,Age,Country\nJohn,25,USA\nAlice,30,UK"
print(csv_string_to_list(csv_string))
```

```python
[['Name', 'Age', 'Country'], ['John', '25', 'USA'], ['Alice', '30', 'UK']]
```
