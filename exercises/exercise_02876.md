## Exercise
Create a function to remove all duplicate characters from a given string

## Reference Solution
```python
def remove_duplicates(input_string):
    """
    Removes all duplicate characters from a given string.

    Args:
        input_string (str): The input string.

    Returns:
        str: A new string with all duplicate characters removed.
    """
    seen = set()
    result = []

    for char in input_string:
        if char not in seen:
            seen.add(char)
            result.append(char)

    return ''.join(result)

# Example usage:
input_string = "Hello, World!"
print("Original string:", input_string)
print("String without duplicates:", remove_duplicates(input_string))
```
