## Exercise
Use a for loop to generate a palindrome of a given length

## Reference Solution
```python
def generate_palindrome(length):
    """
    Generate a palindrome of a given length.

    Args:
        length (int): The length of the palindrome.

    Returns:
        str: The generated palindrome.
    """
    palindrome = ""
    first_half = ""

    # Generate the first half of the palindrome
    for i in range(length // 2):
        first_half += "a"

    # Generate the palindrome by concatenating the first half, 
    # a middle character if the length is odd, and the reverse of the first half
    if length % 2 == 0:
        palindrome = first_half + first_half[::-1]
    else:
        palindrome = first_half + "b" + first_half[::-1]

    return palindrome


# Example usage:
length = 10
print(generate_palindrome(length))
```
