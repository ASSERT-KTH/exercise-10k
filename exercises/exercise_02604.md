## Exercise
Remove a key-value pair from a dictionary

## Reference Solution
```python
# Create a dictionary
my_dict = {'a': 1, 'b': 2, 'c': 3}

# Remove a key-value pair
del my_dict['b']

print(my_dict)  # Output: {'a': 1, 'c': 3}
```

```python
# Create a dictionary
my_dict = {'a': 1, 'b': 2, 'c': 3}

# Remove a key-value pair and return the value
value = my_dict.pop('b')

print(my_dict)  # Output: {'a': 1, 'c': 3}
print(value)    # Output: 2
```

```python
# Create a dictionary
my_dict = {'a': 1, 'b': 2, 'c': 3}

# Remove an arbitrary key-value pair and return it as a tuple
key, value = my_dict.popitem()

print(my_dict)
print(key, value)
```

```python
my_dict = {'a': 1, 'b': 2}
```
