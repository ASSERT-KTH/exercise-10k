## Exercise
Write a function that takes a string as input and returns the uppercase version of the string.

## Reference Solution
```python
def uppercase_string(input_str: str) -> str:
    """
    Returns the uppercase version of the input string.

    Args:
        input_str (str): The input string to be converted to uppercase.

    Returns:
        str: The uppercase version of the input string.
    """
    return input_str.upper()

# Example usage
def main():
    input_str = "Hello, World!"
    uppercase_str = uppercase_string(input_str)
    print(f"Input String: {input_str}")
    print(f"Uppercase String: {uppercase_str}")

if __name__ == "__main__":
    main()
```
