## Exercise
Create a function that takes a name and an optional age parameter, and returns a string with the name and age

## Reference Solution
```python
def create_greeting(name: str, age: int = None) -> str:
    """
    Creates a greeting string with the provided name and optional age.

    Args:
        name (str): The name to include in the greeting.
        age (int, optional): The age to include in the greeting. Defaults to None.

    Returns:
        str: A string containing the name and age (if provided).
    """

    if age is not None:
        return f"Hello, my name is {name} and I am {age} years old."
    else:
        return f"Hello, my name is {name}."

# Example usage
if __name__ == "__main__":
    print(create_greeting("John", 30))  # Output: Hello, my name is John and I am 30 years old.
    print(create_greeting("Jane"))  # Output: Hello, my name is Jane.
```
