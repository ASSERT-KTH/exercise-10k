Create a function that takes keyword arguments and returns their values

## Reference Solution

def get_keyword_args(**kwargs):
    """
    Returns the values of keyword arguments as a dictionary.

    Args:
        **kwargs: Variable number of keyword arguments.

    Returns:
        dict: Dictionary containing the values of keyword arguments.
    """
    return kwargs

# Example usage
if __name__ == "__main__":
    result = get_keyword_args(name="John", age=30, city="New York")
    print(result)
