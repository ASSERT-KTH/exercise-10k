## Exercise
Replace all occurrences of a substring in a given string with another substring

## Reference Solution
```python
def replace_substring(input_string, old_substring, new_substring):
    """
    Replaces all occurrences of a substring in a given string with another substring.

    Args:
        input_string (str): The original string.
        old_substring (str): The substring to be replaced.
        new_substring (str): The substring to replace with.

    Returns:
        str: The modified string with all occurrences of the old substring replaced.
    """
    return input_string.replace(old_substring, new_substring)


# Example usage:
if __name__ == "__main__":
    input_str = "Hello, world! Hello again!"
    old_substr = "Hello"
    new_substr = "Hi"

    modified_str = replace_substring(input_str, old_substr, new_substr)
    print(f"Original String: {input_str}")
    print(f"Modified String: {modified_str}")
```
