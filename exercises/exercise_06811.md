## Exercise
Create a program that uses functions to perform basic calculator operations such as addition, subtraction, multiplication, and division. The program should take in two numbers and an operator as input, then use a function to perform the corresponding operation and return the result.

## Reference Solution
```python
def add(x, y):
    return x + y

def sub(x, y):
    return x - y

def mul(x, y):
    return x * y

def div(x, y):
    if y == 0:
        return "Error: Division by zero"
    return x / y

def calculator(x, y, op):
    if op == '+':
        return add(x, y)
    elif op == '-':
        return sub(x, y)
    elif op == '*':
        return mul(x, y)
    elif op == '/':
        return div(x, y)
    else:
        return "Error: Invalid operator"

# Example usage:
print(calculator(10, 2, '+'))  # Output: 12
print(calculator(10, 2, '-'))  # Output: 8
print(calculator(10, 2, '*'))  # Output: 20
print(calculator(10, 2, '/'))  # Output: 5.0
print(calculator(10, 0, '/'))  # Output: Error: Division by zero
print(calculator(10, 2, '%'))  # Output: Error: Invalid operator
```
