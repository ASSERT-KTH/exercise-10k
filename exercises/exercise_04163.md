## Exercise
Create a program that takes a name and age as input and prints a personalized message.

## Reference Solution
```python
def print_personalized_message(name: str, age: int) -> None:
    """
    Prints a personalized message based on the provided name and age.

    Args:
        name (str): The person's name.
        age (int): The person's age.
    """
    message = f"Hello, {name}! You are {age} years old."
    print(message)

def main() -> None:
    """
    Gets the name and age from the user and prints a personalized message.
    """
    name = input("Please enter your name: ")
    while True:
        try:
            age = int(input("Please enter your age: "))
            break
        except ValueError:
            print("Invalid age. Please enter a whole number.")
    
    print_personalized_message(name, age)

if __name__ == "__main__":
    main()
```
