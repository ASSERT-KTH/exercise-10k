## Exercise
Write a function that takes a dictionary as input and removes and returns an arbitrary key-value pair using the popitem() method

## Reference Solution
```python
def pop_arbitrary_pair(input_dict):
    """
    Removes and returns an arbitrary key-value pair from the input dictionary.

    Args:
        input_dict (dict): The dictionary from which to remove a key-value pair.

    Returns:
        tuple: The removed key-value pair. If the dictionary is empty, returns None.
    """
    if not input_dict:
        return None
    return input_dict.popitem()

# Example usage
if __name__ == "__main__":
    example_dict = {"apple": 1, "banana": 2, "cherry": 3}
    print("Original dictionary:", example_dict)
    
    removed_pair = pop_arbitrary_pair(example_dict)
    print("Removed pair:", removed_pair)
    print("Dictionary after removal:", example_dict)
```
