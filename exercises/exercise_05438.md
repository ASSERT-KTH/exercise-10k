## Exercise
Write a program that calculates statistical measures such as mean, median, and mode. If the user chooses to calculate the mean, use the formula mean = sum(x) / n. If the user chooses to calculate the median, use the formula median = x[n/2] if n is odd, or median = (x[n/2 - 1] + x[n/2]) / 2 if n is even.

## Reference Solution
```python
import statistics

def calculate_mean(numbers):
    """Calculates the mean of a list of numbers."""
    return sum(numbers) / len(numbers)

def calculate_median(numbers):
    """Calculates the median of a list of numbers."""
    n = len(numbers)
    numbers.sort()
    
    if n % 2 == 1:
        return numbers[n // 2]
    else:
        return (numbers[n // 2 - 1] + numbers[n // 2]) / 2

def calculate_mode(numbers):
    """Calculates the mode of a list of numbers."""
    return statistics.mode(numbers)

def main():
    print("Statistical Measures Calculator")
    print("-------------------------------")
    
    numbers = input("Enter a list of numbers separated by spaces: ")
    numbers = [float(num) for num in numbers.split()]
    
    while True:
        print("\nChoose an option:")
        print("1. Calculate Mean")
        print("2. Calculate Median")
        print("3. Calculate Mode")
        print("4. Quit")
        
        option = input("Enter your choice (1-4): ")
        
        if option == "1":
            print(f"Mean: {calculate_mean(numbers)}")
        elif option == "2":
            print(f"Median: {calculate_median(numbers)}")
        elif option == "3":
            try:
                print(f"Mode: {calculate_mode(numbers)}")
            except statistics.StatisticsError:
                print("No unique mode found. The list is multimodal.")
        elif option == "4":
            break
        else:
            print("Invalid option. Please choose again.")

if __name__ == "__main__":
    main()
```
