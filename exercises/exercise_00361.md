## Exercise
Create a program that takes a string input from the user, reverses it, and then prints out the reversed string. The program should also handle cases where the input string is empty or contains only spaces.

## Reference Solution
```python
def reverse_string(s):
    if not s:
        return "The string is empty."
    elif s.strip() == "":
        return "The string is empty or contains only spaces."
    else:
        return s[::-1]

s = input("Enter a string: ")
print(reverse_string(s))
```
