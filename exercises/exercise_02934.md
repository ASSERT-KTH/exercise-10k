## Exercise
Create a program that uses a for loop to count the number of vowels in a given string.

## Reference Solution
```python
def count_vowels(input_string):
    """
    Counts the number of vowels in a given string.

    Args:
        input_string (str): The input string to count vowels from.

    Returns:
        int: The number of vowels in the input string.
    """
    vowels = 'aeiouAEIOU'
    count = 0

    for char in input_string:
        if char in vowels:
            count += 1

    return count

def main():
    # Example usage
    input_string = "Hello, World!"
    print(f"Input String: {input_string}")
    print(f"Number of Vowels: {count_vowels(input_string)}")

if __name__ == "__main__":
    main()
```
