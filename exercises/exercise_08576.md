## Exercise
Write a function to get all keys from a dictionary

## Reference Solution
```python
def get_all_keys(dictionary):
    """
    Returns a list of all keys in the dictionary.

    Args:
        dictionary (dict): The dictionary to retrieve keys from.

    Returns:
        list: A list of all keys in the dictionary.
    """
    return list(dictionary.keys())

# Example usage
my_dict = {"name": "John", "age": 30, "city": "New York"}
keys = get_all_keys(my_dict)
print(keys)  # Output: ['name', 'age', 'city']
```

def get_all_keys(dictionary):

    Returns:
        list: A list of all keys in the dictionary.
