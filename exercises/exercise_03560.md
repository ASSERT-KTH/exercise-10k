## Exercise
Write a program that takes a list as input, and returns a list of tuples containing each element and its rank (i.e., its position in the sorted list).

## Reference Solution
```python
def rank_elements(input_list):
    sorted_list = sorted(input_list)
    ranked_list = []
    
    for element in input_list:
        rank = sorted_list.index(element) + 1
        ranked_list.append((element, rank))
    
    return ranked_list


input_list = [5, 2, 8, 12, 3]
print(rank_elements(input_list))
```

from scipy.stats import rankdata

def rank_elements(input_list):
    ranks = rankdata(input_list, method='ordinal')
    return list(zip(input_list, ranks))
