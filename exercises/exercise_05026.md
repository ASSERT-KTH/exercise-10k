## Exercise
Write a program that checks if a given string is a palindrome using a while loop.

## Reference Solution
```python
def is_palindrome(s):
    s = ''.join(c for c in s if c.isalnum()).lower()  # remove non-alphanumeric characters and convert to lowercase
    left = 0
    right = len(s) - 1

    while left < right:
        if s[left] != s[right]:
            return False
        left += 1
        right -= 1

    return True

# Test the function
print(is_palindrome("A man, a plan, a canal: Panama"))  # True
print(is_palindrome("Hello"))  # False
```
