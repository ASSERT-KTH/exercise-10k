## Exercise
Write a program that converts a given string to upper case.

## Reference Solution
```python
def convert_to_upper_case(input_string: str) -> str:
    """
    Converts a given string to upper case.

    Args:
        input_string (str): The input string to be converted.

    Returns:
        str: The upper case equivalent of the input string.
    """
    return input_string.upper()

def main():
    # Example usage:
    input_string = "Hello, World!"
    upper_case_string = convert_to_upper_case(input_string)
    print(f"Input String: {input_string}")
    print(f"Upper Case String: {upper_case_string}")

if __name__ == "__main__":
    main()
```
