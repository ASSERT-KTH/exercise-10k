## Exercise
Write a program that sorts a dictionary's values in ascending order and prints them.

## Reference Solution
```python
def sort_dict_values(dictionary):
    """
    Sorts a dictionary's values in ascending order and prints them.

    Args:
        dictionary (dict): The dictionary to sort values from.

    Returns:
        list: A list of sorted values.
    """
    # Check if the input is a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("Input must be a dictionary")

    # Sort the dictionary's values in ascending order
    sorted_values = sorted(dictionary.values())

    # Print the sorted values
    print("Sorted values:")
    for value in sorted_values:
        print(value)

    return sorted_values

# Example usage
if __name__ == "__main__":
    example_dict = {"a": 5, "b": 1, "c": 9, "d": 3, "e": 7}
    print("Original dictionary:", example_dict)
    sort_dict_values(example_dict)
```
