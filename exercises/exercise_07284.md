## Exercise
Create a function that takes a name as an argument and returns a greeting, with a default name of 'World'

## Reference Solution
```python
def greet(name: str = 'World') -> str:
    """
    Returns a greeting message with a given name.

    Args:
        name (str, optional): The name to include in the greeting. Defaults to 'World'.

    Returns:
        str: A greeting message.
    """
    return f"Hello, {name}!"

# Example usage
def main():
    print(greet())  # Output: Hello, World!
    print(greet('Alice'))  # Output: Hello, Alice!

if __name__ == "__main__":
    main()
```
