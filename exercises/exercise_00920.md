## Exercise
Write a program that serializes a given list of variables into a single string.

## Reference Solution
```python
import json

def serialize_variables(variables):
    """
    Serialize a list of variables into a single string.

    Args:
        variables (list): A list of variables to be serialized.

    Returns:
        str: A JSON string representing the serialized variables.
    """
    return json.dumps(variables)

# Example usage:
variables = ["apple", 123, True, 3.14, None]
serialized_string = serialize_variables(variables)
print(serialized_string)
```
