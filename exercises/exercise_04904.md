## Exercise
Write a program that takes two numbers and an operator (+, -, *, /) as input. Based on the operator, perform the corresponding arithmetic operation and print the result.

## Reference Solution
```python
def arithmetic_operation(num1, num2, operator):
    """
    Performs arithmetic operations based on the given operator.

    Args:
        num1 (float): The first number.
        num2 (float): The second number.
        operator (str): The operator to use. Can be '+', '-', '*', or '/'.

    Returns:
        float: The result of the arithmetic operation.
    """

    # Check if the operator is valid
    if operator not in ['+', '-', '*', '/']:
        raise ValueError("Invalid operator. Only '+', '-', '*', and '/' are supported.")

    # Perform the corresponding arithmetic operation
    if operator == '+':
        result = num1 + num2
    elif operator == '-':
        result = num1 - num2
    elif operator == '*':
        result = num1 * num2
    elif operator == '/':
        # Check for division by zero
        if num2 == 0:
            raise ZeroDivisionError("Cannot divide by zero.")
        result = num1 / num2

    return result


def main():
    # Get user input
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))
    operator = input("Enter the operator (+, -, *, /): ")

    try:
        # Perform the arithmetic operation and print the result
        result = arithmetic_operation(num1, num2, operator)
        print(f"{num1} {operator} {num2} = {result}")
    except (ValueError, ZeroDivisionError) as e:
        print(f"Error: {e}")


if __name__ == "__main__":
    main()
```
