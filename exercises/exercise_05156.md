## Exercise
Find the student(s) with a specific age in a dictionary where keys are student names and values are their corresponding ages

## Reference Solution
```python
def find_students_by_age(students, target_age):
    """
    Find students with a specific age in a dictionary.

    Args:
        students (dict): A dictionary where keys are student names and values are their corresponding ages.
        target_age (int): The age to search for.

    Returns:
        list: A list of student names with the target age.
    """
    return [name for name, age in students.items() if age == target_age]


# Example usage
if __name__ == "__main__":
    students = {
        "John": 20,
        "Alice": 22,
        "Bob": 20,
        "Charlie": 21,
        "David": 20
    }

    target_age = 20
    result = find_students_by_age(students, target_age)

    print(f"Students with age {target_age}: {result}")
```
