## Exercise
Write a function to retrieve a list of all values in a dictionary.

## Reference Solution
```python
def get_dict_values(input_dict):
    """
    This function retrieves a list of all values in a given dictionary.

    Args:
        input_dict (dict): The dictionary from which to retrieve values.

    Returns:
        list: A list of all values in the dictionary.
    """

    # First, we check if the input is indeed a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary")

    # We can use the .values() method of dictionaries to get a view object that displays a list of all values in the dictionary
    dict_values = input_dict.values()

    # We then convert this view object into a list using the list() function
    # This is because the problem statement asks for a list, and the .values() method returns a view object
    dict_values_list = list(dict_values)

    # Finally, we return this list of dictionary values
    return dict_values_list

# Example usage:
example_dict = {"a": 1, "b": 2, "c": 3}
print(get_dict_values(example_dict))  # Output: [1, 2, 3]
```
