## Exercise
Write a program that takes a list and a transformation function as input and returns a new list containing the results of applying the transformation to each element

## Reference Solution
```python
def transform_list(input_list, transformation_function):
    """
    Applies a given transformation function to each element of a list.

    Args:
        input_list (list): The list to be transformed.
        transformation_function (function): A function that takes one argument and returns a transformed value.

    Returns:
        list: A new list containing the results of applying the transformation to each element of the input list.
    """
    # Initialize an empty list to store the transformed elements
    transformed_list = []

    # Iterate over each element in the input list
    for element in input_list:
        # Apply the transformation function to the current element
        transformed_element = transformation_function(element)
        
        # Append the transformed element to the transformed list
        transformed_list.append(transformed_element)

    # Return the transformed list
    return transformed_list

# Example usage:
numbers = [1, 2, 3, 4, 5]

# Define a transformation function that doubles each number
def double_number(x):
    return x * 2

# Apply the transformation function to the list of numbers
double_numbers = transform_list(numbers, double_number)

# Print the transformed list
print(double_numbers)  # Output: [2, 4, 6, 8, 10]
```
