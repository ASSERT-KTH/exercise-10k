## Exercise
Write a program that groups the keys in a dictionary by their values

## Reference Solution
```python
def group_keys_by_values(dictionary):
    """
    Group the keys in a dictionary by their values.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        dict: A dictionary where the keys are the original values and the values are lists of keys.
    """
    result = {}
    for key, value in dictionary.items():
        if value not in result:
            result[value] = [key]
        else:
            result[value].append(key)
    return result

# Example usage
if __name__ == "__main__":
    dictionary = {
        "a": 1,
        "b": 2,
        "c": 1,
        "d": 3,
        "e": 2,
        "f": 1
    }
    print("Original Dictionary:")
    print(dictionary)

    grouped_dictionary = group_keys_by_values(dictionary)
    print("\nGrouped Dictionary:")
    print(grouped_dictionary)
```
