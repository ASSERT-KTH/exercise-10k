## Exercise
Write a function to reverse the order of elements in an array. The function should take an array as input and return the reversed array.

## Reference Solution
```python
def reverse_elements(arr):
    arr = arr[::-1]
    return arr

arr = [1, 2, 3, 4, 5]
print(reverse_elements(arr))
```
