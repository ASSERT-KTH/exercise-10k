## Exercise
Create a program to calculate the perimeter of a trapezoid given its bases and sides.

## Reference Solution
```python
def calculate_trapezoid_perimeter(base1, base2, side1, side2):
    """
    Calculate the perimeter of a trapezoid.

    Args:
        base1 (float): The length of the first base.
        base2 (float): The length of the second base.
        side1 (float): The length of the first side.
        side2 (float): The length of the second side.

    Returns:
        float: The perimeter of the trapezoid.
    """
    perimeter = base1 + base2 + side1 + side2
    return perimeter

def main():
    # Example usage
    base1 = float(input("Enter the length of the first base: "))
    base2 = float(input("Enter the length of the second base: "))
    side1 = float(input("Enter the length of the first side: "))
    side2 = float(input("Enter the length of the second side: "))

    perimeter = calculate_trapezoid_perimeter(base1, base2, side1, side2)
    print(f"The perimeter of the trapezoid is: {perimeter}")

if __name__ == "__main__":
    main()
```
