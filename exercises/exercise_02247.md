## Exercise
Write a program that takes a number as input and returns whether it is even or odd.

## Reference Solution
```python
def is_even(n):
    """
    Returns whether a number is even or odd.

    Args:
        n (int): The number to check.

    Returns:
        str: "Even" if the number is even, "Odd" otherwise.
    """
    if n % 2 == 0:
        return "Even"
    else:
        return "Odd"

def main():
    num = int(input("Enter a number: "))
    result = is_even(num)
    print(f"The number {num} is {result}.")

if __name__ == "__main__":
    main()
```
