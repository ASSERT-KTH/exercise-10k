## Exercise
Create a program that takes a string as input and counts the number of words in it

## Reference Solution
```python
def count_words(input_string):
    """
    Counts the number of words in a given string.

    Args:
        input_string (str): The input string to count words from.

    Returns:
        int: The number of words in the input string.
    """
    # Remove leading and trailing whitespace
    input_string = input_string.strip()
    
    # Split the string into words
    words = input_string.split()
    
    # Return the number of words
    return len(words)

def main():
    # Get user input
    input_string = input("Enter a string: ")
    
    # Count the words
    word_count = count_words(input_string)
    
    # Print the result
    print(f"The string contains {word_count} words.")

if __name__ == "__main__":
    main()
```
