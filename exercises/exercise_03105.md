## Exercise
Create two dictionaries with 5 key-value pairs each. Merge the two dictionaries into a single dictionary.

## Reference Solution
```python
# Create the first dictionary
dict1 = {
    "key1": "value1",
    "key2": "value2",
    "key3": "value3",
    "key4": "value4",
    "key5": "value5"
}

# Create the second dictionary
dict2 = {
    "key6": "value6",
    "key7": "value7",
    "key8": "value8",
    "key9": "value9",
    "key10": "value10"
}

# Merge the two dictionaries into a single dictionary
merged_dict = {**dict1, **dict2}

# Print the merged dictionary
print("Merged Dictionary:")
print(merged_dict)
```
