## Exercise
Write a program that returns the number of key-value pairs in a dictionary

def count_key_value_pairs(dictionary):
    """
    Returns the number of key-value pairs in a dictionary.

    Args:
        dictionary (dict): The dictionary to count key-value pairs from.

    Returns:
        int: The number of key-value pairs in the dictionary.
    """
    return len(dictionary)


# Example usage:
