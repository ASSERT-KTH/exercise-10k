## Exercise
Create a program that takes in a mathematical expression as input from the user, evaluates it, and prints the result. The expression should be a simple arithmetic operation involving two numbers and one of the four basic arithmetic operators (+, -, *, /).

## Reference Solution
```python
def evaluate_expression():
    """
    Evaluates a simple arithmetic expression involving two numbers and one of the four basic arithmetic operators.
    
    Parameters:
    None
    
    Returns:
    None
    """
    # Get user input for the mathematical expression
    expression = input("Enter a mathematical expression (e.g., 2+2, 5*3, 10/2): ")
    
    try:
        # Evaluate the expression using eval function
        result = eval(expression)
        
        # Print the result
        print(f"The result of the expression '{expression}' is: {result}")
    
    except ZeroDivisionError:
        # Handle division by zero error
        print("Error: Division by zero is not allowed.")
    
    except SyntaxError:
        # Handle invalid syntax error
        print("Error: Invalid syntax. Please enter a valid mathematical expression.")
    
    except Exception as e:
        # Handle any other exceptions
        print(f"An error occurred: {str(e)}")

# Example usage:
if __name__ == "__main__":
    evaluate_expression()
```
