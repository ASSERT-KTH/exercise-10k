## Exercise
Write a program that takes an array and an index as input, and returns the value at the specified index. If the index is out of bounds, return an error message.

## Reference Solution
```python
class Solution {
public:
    int findnum(vector<int>& nums, int n) {
        if (n > nums.size() - 1) {
            return -1;
        } else {
            return nums[n];
        }
    }
};
```
