## Exercise
Create a function to find the nearest number to a given target in a list of numbers.

## Reference Solution
```python
def find_nearest_number(numbers, target):
    """
    Find the nearest number to a given target in a list of numbers.

    Args:
        numbers (list): A list of numbers.
        target (int or float): The target number.

    Returns:
        int or float: The nearest number to the target.

    Raises:
        ValueError: If the input list is empty.
    """
    if not numbers:
        raise ValueError("Input list is empty")

    return min(numbers, key=lambda x: abs(x - target))

# Example usage
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    target = 3.7
    nearest_number = find_nearest_number(numbers, target)
    print(f"The nearest number to {target} is {nearest_number}")
```
