## Exercise
Write a program to apply a transformation to each element of a list.

## Reference Solution
```python
def transform_list(input_list, transformation_function):
    """
    Applies a transformation function to each element of a list.

    Args:
        input_list (list): The list to be transformed.
        transformation_function (function): A function that takes one argument and returns a transformed value.

    Returns:
        list: A new list with the transformation applied to each element.
    """
    return [transformation_function(element) for element in input_list]


# Example usage:
def double_value(x):
    """
    Doubles the input value.
    """
    return x * 2


numbers = [1, 2, 3, 4, 5]
transformed_numbers = transform_list(numbers, double_value)

print("Original list:", numbers)
print("Transformed list:", transformed_numbers)
```
