## Exercise
Create a function that takes two strings as input and returns True if the strings are anagrams, False otherwise.

## Reference Solution
```python
def are_anagrams(str1, str2):
    """
    This function checks if two input strings are anagrams.
    
    Args:
        str1 (str): The first input string.
        str2 (str): The second input string.
    
    Returns:
        bool: True if the strings are anagrams, False otherwise.
    """

    # First, we convert both strings to lower case to handle case-insensitive comparison
    str1 = str1.lower()
    
    # We do the same for the second string
    str2 = str2.lower()
    
    # Then, we remove any spaces from the strings
    str1 = str1.replace(" ", "")
    
    # We do the same for the second string
    str2 = str2.replace(" ", "")
    
    # Next, we check if the sorted versions of both strings are equal
    # If they are, then the original strings are anagrams
    # We use the sorted function to sort the characters in the strings
    # We convert the strings to lists before sorting because strings are immutable in Python
    # We convert the sorted lists back to strings using the join method
    return sorted(str1) == sorted(str2)

# Test the function
print(are_anagrams("Listen", "Silent"))  # Should return True
print(are_anagrams("Hello", "World"))  # Should return False
```
