## Exercise
Create a function that takes a string as an argument and returns its lowercase version

## Reference Solution
```python
def to_lowercase(input_string: str) -> str:
    """
    Returns the lowercase version of the input string.

    Args:
        input_string (str): The string to be converted to lowercase.

    Returns:
        str: The lowercase version of the input string.
    """
    return input_string.lower()

# Example usage:
if __name__ == "__main__":
    print(to_lowercase("HELLO WORLD"))  # Output: "hello world"
    print(to_lowercase("Mixed Case String"))  # Output: "mixed case string"
```
