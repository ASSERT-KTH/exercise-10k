## Exercise
Write a function to find the product of all elements in an array.

## Reference Solution
```python
def array_product(arr):
    """
    Calculate the product of all elements in an array.

    Args:
        arr (list): A list of numbers.

    Returns:
        float: The product of all elements in the array.

    Raises:
        ValueError: If the input array is empty.
    """
    if not arr:
        raise ValueError("Input array is empty")

    product = 1
    for num in arr:
        product *= num

    return product

# Example usage:
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    result = array_product(numbers)
    print(f"The product of the array elements is: {result}")
```
