## Exercise
Create a program that uses a dictionary to store student grades. The program should take in a student's name and their grades for three subjects: math, science, and English. The program should then calculate the average grade and store it in the dictionary. Finally, the program should print out the student's name, their grades for each subject, and their average grade.

## Reference Solution
```python
def get_student_grades():
    """Get student grades from user."""
    student_name = input("Enter the student's name: ")

    # Get grades for each subject
    math_grade = float(input("Enter the student's math grade: "))
    science_grade = float(input("Enter the student's science grade: "))
    english_grade = float(input("Enter the student's English grade: "))

    return student_name, math_grade, science_grade, english_grade


def calculate_average_grade(math_grade, science_grade, english_grade):
    """Calculate the average grade for a student."""
    average_grade = (math_grade + science_grade + english_grade) / 3
    return average_grade


def store_student_grades(student_name, math_grade, science_grade, english_grade, average_grade):
    """Store student grades in a dictionary."""
    student_grades = {
        "Name": student_name,
        "Math Grade": math_grade,
        "Science Grade": science_grade,
        "English Grade": english_grade,
        "Average Grade": average_grade
    }
    return student_grades


def print_student_grades(student_grades):
    """Print out the student's grades."""
    print("Student's Name:", student_grades["Name"])
    print("Math Grade:", student_grades["Math Grade"])
    print("Science Grade:", student_grades["Science Grade"])
    print("English Grade:", student_grades["English Grade"])
    print("Average Grade:", student_grades["Average Grade"])


def main():
    student_name, math_grade, science_grade, english_grade = get_student_grades()
    average_grade = calculate_average_grade(math_grade, science_grade, english_grade)
    student_grades = store_student_grades(student_name, math_grade, science_grade, english_grade, average_grade)
    print_student_grades(student_grades)


if __name__ == "__main__":
    main()
```
