## Exercise
Create a function that takes a number as input and returns its square root.

## Reference Solution
```python
import math

def calculate_square_root(number):
    """
    This function calculates the square root of a given number.

    Args:
        number (float): The input number for which the square root is to be calculated.

    Returns:
        float: The square root of the input number.
    """

    # First, we check if the input number is negative, because square root of negative numbers is not defined in real mathematics
    if number < 0:
        raise ValueError("Input number cannot be negative")

    # Now, we use the math.sqrt function from Python's math module to calculate the square root
    # This function returns the square root of the given number
    square_root = math.sqrt(number)

    # Finally, we return the calculated square root
    return square_root

# Test the function
print(calculate_square_root(9))  # Output: 3.0
```
