## Exercise
Write a program to simulate a coin flip using a for loop. The program should simulate a specified number of coin flips and print the results.

## Reference Solution
```python
import random

def coin_flip_simulation(num_flips):
    """
    Simulates a coin flip a specified number of times and prints the results.

    Args:
        num_flips (int): The number of coin flips to simulate.

    Returns:
        None
    """

    # Initialize variables to track the number of heads and tails
    num_heads = 0
    num_tails = 0

    # Simulate the coin flips
    print("Simulating {} coin flips...".format(num_flips))
    for i in range(num_flips):
        # Generate a random number, 0 or 1, to simulate the coin flip
        flip = random.randint(0, 1)
        
        # Print the result of the flip
        if flip == 0:
            print("Flip {}: Heads".format(i + 1))
            num_heads += 1
        else:
            print("Flip {}: Tails".format(i + 1))
            num_tails += 1

    # Print the final results
    print("\nFinal Results:")
    print("Heads: {}".format(num_heads))
    print("Tails: {}".format(num_tails))

    # Calculate and print the percentage of heads and tails
    heads_percentage = (num_heads / num_flips) * 100
    tails_percentage = (num_tails / num_flips) * 100
    print("Heads Percentage: {:.2f}%".format(heads_percentage))
    print("Tails Percentage: {:.2f}%".format(tails_percentage))

# Example usage:
num_flips = 10
coin_flip_simulation(num_flips)
```
