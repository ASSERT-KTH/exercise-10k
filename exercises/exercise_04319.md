## Exercise
Create a function that can handle multiple types of input and return the correct output

## Reference Solution
```python
def process_input(input_value):
    """
    This function processes different types of input and returns the correct output.

    Args:
        input_value (any): The input value to be processed.

    Returns:
        any: The processed output.
    """

    # Check if input is an integer
    if isinstance(input_value, int):
        # If input is an integer, return its square
        return input_value ** 2

    # Check if input is a string
    elif isinstance(input_value, str):
        # If input is a string, return its length
        return len(input_value)

    # Check if input is a list or tuple
    elif isinstance(input_value, (list, tuple)):
        # If input is a list or tuple, return its sum
        return sum(input_value)

    # Check if input is a dictionary
    elif isinstance(input_value, dict):
        # If input is a dictionary, return its keys
        return list(input_value.keys())

    # If input is none of the above, return an error message
    else:
        return "Error: Input type not supported"

# Example usage:
print(process_input(5))  # Output: 25
print(process_input("Hello"))  # Output: 5
print(process_input([1, 2, 3, 4, 5]))  # Output: 15
print(process_input({"a": 1, "b": 2, "c": 3}))  # Output: ['a', 'b', 'c']
print(process_input(3.14))  # Output: Error: Input type not supported
```
