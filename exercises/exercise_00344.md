## Exercise
Create a function that sorts a list in ascending or descending order.

## Reference Solution
```python
def sort_list(input_list, order='asc'):
    """
    Sorts a list in ascending or descending order.

    Args:
        input_list (list): The list to be sorted.
        order (str, optional): The order of sorting. Defaults to 'asc'.
            - 'asc' for ascending order
            - 'desc' for descending order

    Returns:
        list: The sorted list.
    """
    if order == 'asc':
        return sorted(input_list)
    elif order == 'desc':
        return sorted(input_list, reverse=True)
    else:
        raise ValueError("Invalid order. Please use 'asc' or 'desc'.")

# Example usage:
if __name__ == "__main__":
    numbers = [64, 34, 25, 12, 22, 11, 90]
    print("Original list:", numbers)
    print("Sorted list (ascending):", sort_list(numbers, 'asc'))
    print("Sorted list (descending):", sort_list(numbers, 'desc'))
```
