## Exercise
Write a program that takes a string as input and returns a diamond pattern of the string

## Reference Solution
```python
def diamond_pattern(input_string):
    """
    Returns a diamond pattern of the input string.

    Args:
        input_string (str): The input string to create a diamond pattern from.

    Returns:
        str: The diamond pattern of the input string.
    """
    lines = []
    n = len(input_string)

    # Create the top half of the diamond
    for i in range(n):
        line = ' ' * (n - i - 1) + input_string[:i + 1]
        lines.append(line)

    # Create the bottom half of the diamond
    for i in range(n - 2, -1, -1):
        line = ' ' * (n - i - 1) + input_string[:i + 1]
        lines.append(line)

    # Join the lines into a single string
    diamond = '\n'.join(lines)

    return diamond

# Example usage
input_string = "Hello"
print(diamond_pattern(input_string))
```
