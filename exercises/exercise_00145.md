## Exercise
Create a function that takes in a dictionary and prints out each key-value pair

## Reference Solution
```python
def print_dictionary(dictionary):
    """
    Prints out each key-value pair in the input dictionary.
    
    Args:
        dictionary (dict): The dictionary to print key-value pairs from.
    """
    
    # Check if the input is a dictionary
    if not isinstance(dictionary, dict):
        raise ValueError("Input must be a dictionary")
    
    # Iterate over the dictionary items and print each key-value pair
    for key, value in dictionary.items():
        print(f"{key}: {value}")


# Example usage:
if __name__ == "__main__":
    example_dict = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }
    print_dictionary(example_dict)
```
