## Exercise
Develop a program that finds a substring within a given string using if/else statements.

## Reference Solution
```python
def find_substring(string, substring):
    """
    Checks if a substring exists within a given string.

    Args:
        string (str): The main string to search in.
        substring (str): The substring to search for.

    Returns:
        bool: True if the substring exists, False otherwise.
    """
    if substring in string:
        print(f"The substring '{substring}' exists in the string '{string}'.")
        return True
    else:
        print(f"The substring '{substring}' does not exist in the string '{string}'.")
        return False

def main():
    string = input("Enter the main string: ")
    substring = input("Enter the substring to search for: ")
    find_substring(string, substring)

if __name__ == "__main__":
    main()
```
