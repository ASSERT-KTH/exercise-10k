## Exercise
Create a program that can compress and decompress strings using Run-Length Encoding (RLE). The program should take a string as input, compress it using RLE, and then decompress it back to the original string. The compressed string should be in the format of 'charactercount' (e.g., 'a3' for 'aaa'). The program should handle strings with repeated characters and single characters.

## Reference Solution
```python
def comp(s):
    res = []
    i = 0
    while i < len(s):
        count = 1
        while i + 1 < len(s) and s[i] == s[i+1]:
            i += 1
            count += 1
        res.append(s[i])
        if count > 1:
            res.append(str(count))
        i += 1
    return ''.join(res)

def decomp(s):
    res = []
    i = 0
    while i < len(s):
        count = ''
        while i < len(s) and s[i].isdigit():
            count += s[i]
            i += 1
        if count:
            res.append(s[i-1] * int(count))
        else:
            res.append(s[i])
        i += 1
    return ''.join(res)

def main():
    s = input("Enter a string: ")
    s1 = comp(s)
    print("Compressed string:", s1)
    s2 = decomp(s1)
    print("Decompressed string:", s2)

if __name__ == "__main__":
    main()
```
