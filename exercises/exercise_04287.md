## Exercise
Write a function to sort a dictionary's keys in ascending order.

## Reference Solution
```python
def sort_dict_keys(dictionary):
    """
    This function sorts a dictionary's keys in ascending order.
    
    Args:
        dictionary (dict): The dictionary to be sorted.
    
    Returns:
        dict: A new dictionary with the keys sorted in ascending order.
    """
    
    # First, we check if the input is indeed a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("Input must be a dictionary")
    
    # We use the built-in sorted function to sort the dictionary's keys
    # The sorted function returns a new list containing all items from the dictionary's keys in ascending order
    sorted_keys = sorted(dictionary.keys())
    
    # We use a dictionary comprehension to create a new dictionary with the sorted keys
    # The dictionary comprehension is a compact way to create a new dictionary from an existing iterable
    sorted_dict = {key: dictionary[key] for key in sorted_keys}
    
    # Finally, we return the new dictionary with the sorted keys
    return sorted_dict

# Example usage:
my_dict = {'c': 3, 'b': 2, 'a': 1}
sorted_my_dict = sort_dict_keys(my_dict)
print(sorted_my_dict)  # Output: {'a': 1, 'b': 2, 'c': 3}
```
