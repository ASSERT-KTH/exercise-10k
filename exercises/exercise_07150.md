## Exercise
Create a program that takes a string input from the user and validates if it contains only alphanumeric characters and spaces. If the input is valid, print out the input string. Otherwise, print an error message.

## Reference Solution
```python
import re

# Define a function to validate the input string
def validate_string(s):
    pattern = re.compile(r'^[\w\s]*$')
    if pattern.match(s):
        print(s)
    else:
        print("Error: Input string contains non-alphanumeric characters or non-spaces.")

if __name__ == '__main__':
    # Get the input string from the user
    s = input("Enter a string: ")

    # Call the validate_string function
    validate_string(s)
```
