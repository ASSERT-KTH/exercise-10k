## Exercise
Write a program that takes a string input from the user, reverses the order of its characters, and then prints out the reversed string. The program should handle both lowercase and uppercase letters, as well as any special characters or numbers that might be included in the input.

```python

# Get user input
s = input("Enter a string: ")

# Reverse the string
s = s[::-1]

# Print the reversed string
print("Reversed string: ", s)
```
