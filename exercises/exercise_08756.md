## Exercise
Create a function to convert a given string to uppercase or lowercase.

## Reference Solution
```python
def convert_case(input_string, mode):
    """
    Converts the input string to uppercase or lowercase.

    Args:
        input_string (str): The string to be converted.
        mode (str): The conversion mode. Can be 'upper' or 'lower'.

    Returns:
        str: The converted string.

    Raises:
        ValueError: If the mode is not 'upper' or 'lower'.
    """
    if mode == 'upper':
        return input_string.upper()
    elif mode == 'lower':
        return input_string.lower()
    else:
        raise ValueError("Invalid mode. Mode can be 'upper' or 'lower'.")

# Example usage:
def main():
    input_string = "Hello, World!"
    print(convert_case(input_string, 'upper'))  # Output: HELLO, WORLD!
    print(convert_case(input_string, 'lower'))  # Output: hello, world!

if __name__ == "__main__":
    main()
```
