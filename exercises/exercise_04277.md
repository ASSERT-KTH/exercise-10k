## Exercise
Write a function that takes a dictionary as input and returns the number of keys and values using the len() function

## Reference Solution
```python
def count_dict_keys_values(input_dict):
    """
    Returns the number of keys and values in a dictionary.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        tuple: A tuple containing the number of keys and values.
    """
    # Check if input is a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary")

    # Get the number of key-value pairs
    num_pairs = len(input_dict)

    # Return the number of keys and values
    return num_pairs, num_pairs

# Example usage
def main():
    example_dict = {"key1": "value1", "key2": "value2", "key3": "value3"}
    num_keys, num_values = count_dict_keys_values(example_dict)
    print(f"Number of keys: {num_keys}")
    print(f"Number of values: {num_values}")

if __name__ == "__main__":
    main()
```
