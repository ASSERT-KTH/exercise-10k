## Exercise
Use an enum type to store a set of named values

## Reference Solution
```python
from enum import Enum

class Color(Enum):
    """Enum class representing different colors"""
    RED = 1
    GREEN = 2
    BLUE = 3

# Accessing enum values
print(Color.RED)  # Color.RED
print(Color.RED.name)  # RED
print(Color.RED.value)  # 1

# Iterating over enum values
for color in Color:
    print(f"{color.name}: {color.value}")

# Using enum values in a function
def get_color_name(color: Color) -> str:
    """Returns the name of the given color"""
    return color.name

print(get_color_name(Color.GREEN))  # GREEN
```
