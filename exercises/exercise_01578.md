## Exercise
Create a function that determines the type of a given function.

## Reference Solution
```python
def get_function_type(func):
    """
    This function determines the type of a given function.
    
    Args:
        func (function): The function to be checked.
    
    Returns:
        str: A string describing the type of the function.
    """

    # First, we check if the input is a function
    if not callable(func):
        return "Input is not a function"

    # We use the built-in type() function in Python to get the type of the function
    func_type = type(func)

    # We then use the __name__ attribute of the type to get its name
    func_type_name = func_type.__name__

    # Finally, we return the name of the function type
    return func_type_name

# Example usage:
def example_function():
    pass

print(get_function_type(example_function))  # Output: function
```
