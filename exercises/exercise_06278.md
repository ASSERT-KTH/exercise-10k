## Exercise
Map a function to each value in a dictionary

## Reference Solution
```python
def square(x):
    """Returns the square of x"""
    return x ** 2

# Define a dictionary
data = {'a': 1, 'b': 2, 'c': 3, 'd': 4}

# Map the square function to each value in the dictionary
result = {key: square(value) for key, value in data.items()}

print(result)  # Output: {'a': 1, 'b': 4, 'c': 9, 'd': 16}
```

result = dict(map(lambda item: (item[0], square(item[1])), data.items()))
