## Exercise
Create a function that simulates a coin flip and returns the result.

## Reference Solution
```python
import random

def coin_flip():
    """
    Simulates a coin flip and returns the result.

    Returns:
        str: The result of the coin flip, either "Heads" or "Tails".
    """
    flip = random.choice(["Heads", "Tails"])
    return flip

# Example usage:
def main():
    print("Coin Flip Simulator")
    print("--------------------")
    result = coin_flip()
    print(f"The coin landed on: {result}")

if __name__ == "__main__":
    main()
```
