## Exercise
Given a set of variables with different data types, write a program to identify and print the data type of each variable. The variables will be given as input in the form of a dictionary where keys are variable names and values are the corresponding values.

## Reference Solution
```python
def print_data_types(variables):
    """
    Prints the data type of each variable in the given dictionary.

    Args:
        variables (dict): A dictionary where keys are variable names and values are the corresponding values.
    """
    # Iterate over each key-value pair in the dictionary
    for variable, value in variables.items():
        # Use the type() function to get the data type of the value
        data_type = type(value).__name__
        
        # Print the variable name and its data type
        print(f"Variable: {variable}, Data Type: {data_type}")


# Example usage:
variables = {
    "var1": 10,
    "var2": 3.14,
    "var3": "Hello",
    "var4": True,
    "var5": [1, 2, 3],
    "var6": {"a": 1, "b": 2},
    "var7": (1, 2, 3),
    "var8": {"apple", "banana", "cherry"}
}

print_data_types(variables)
```
