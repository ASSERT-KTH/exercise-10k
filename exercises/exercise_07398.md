## Exercise
Given an array of integers, find the maximum contiguous subarray sum. The subarray must have at least one element. If the array is empty, return 0.

## Reference Solution
```python
def max_subarray_sum(arr):
    """
    Returns the maximum contiguous subarray sum.

    Args:
    arr (list): A list of integers.

    Returns:
    int: The maximum contiguous subarray sum.
    """
    if not arr:
        return 0

    max_sum = float('-inf')
    current_sum = 0

    for num in arr:
        current_sum = max(num, current_sum + num)
        max_sum = max(max_sum, current_sum)

    return max_sum

# Example usage
arr = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
print(max_subarray_sum(arr))  # Output: 6
```
