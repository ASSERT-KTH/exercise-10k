## Exercise
Use a while loop to calculate the sum of digits of a given number

## Reference Solution
```python
def sum_of_digits(n):
    """
    Calculate the sum of digits of a given number.
    
    Args:
        n (int): The number for which to calculate the sum of digits.
    
    Returns:
        int: The sum of digits of the given number.
    """
    sum = 0
    while n != 0:
        # Extract the last digit
        digit = n % 10
        # Add the digit to the sum
        sum += digit
        # Remove the last digit from the number
        n //= 10
    return sum

# Example usage
number = 12345
result = sum_of_digits(number)
print(f"The sum of digits of {number} is {result}")
```
