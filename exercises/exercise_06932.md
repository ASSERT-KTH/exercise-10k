## Exercise
Create a function that uses nested loops to print a matrix of numbers

## Reference Solution
```python
def print_matrix(rows, cols):
    """
    Prints a matrix of numbers.

    Args:
        rows (int): The number of rows in the matrix.
        cols (int): The number of columns in the matrix.
    """
    num = 1
    for i in range(rows):
        for j in range(cols):
            print(f"{num:2d}", end=" ")
            num += 1
        print()

# Example usage
if __name__ == "__main__":
    rows = 5
    cols = 5
    print(f"Matrix ({rows}x{cols}):")
    print_matrix(rows, cols)
```
