## Exercise
Write a program that takes a student's score as input and outputs their corresponding grade based on the following conditions: score >= 90 (A), 80 <= score < 90 (B), 70 <= score < 80 (C), 60 <= score < 70 (D), score < 60 (F). The program should also handle cases where the input score is not within the valid range of 0 to 100.

## Reference Solution
```python
def get_grade(score):
    if score >= 90:
        return 'A'
    elif score >= 80:
        return 'B'
    elif score >= 70:
        return 'C'
    elif score >= 60:
        return 'D'
    else:
        return 'F'

def get_letter_grade(score):
    if score >= 90:
        return 'A'
    elif score >= 80:
        return 'B'
    elif score >= 70:
        return 'C'
    elif score >= 60:
        return 'D'
    else:
        return 'F'

# Example usage:
score = 85
print(get_grade(score))
print(get_letter_grade(score))
```
