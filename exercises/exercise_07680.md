## Exercise
Write a function that counts the occurrences of a character in a string.

## Reference Solution
```python
def count_character(string, char):
    """
    Counts the occurrences of a character in a string.

    Args:
        string (str): The input string.
        char (str): The character to count.

    Returns:
        int: The number of occurrences of the character in the string.
    """
    return string.count(char)

# Example usage
def main():
    string = "Hello, World!"
    char = "l"
    count = count_character(string, char)
    print(f"The character '{char}' occurs {count} times in the string '{string}'.")

if __name__ == "__main__":
    main()
```
