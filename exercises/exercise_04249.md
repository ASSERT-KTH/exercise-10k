## Exercise
Develop a program that converts a temperature from Celsius to Fahrenheit or vice versa. The program should take the temperature and conversion type as input, and print the converted temperature.

## Reference Solution

def convert_temperature(temp, conversion_type):
    """
    Converts a temperature from Celsius to Fahrenheit or vice versa.

    Args:
        temp (float): The temperature to be converted.
        conversion_type (str): The type of conversion. 'c_to_f' for Celsius to Fahrenheit, 'f_to_c' for Fahrenheit to Celsius.

    Returns:
        float: The converted temperature.
    """
    if conversion_type == 'c_to_f':
        # Convert Celsius to Fahrenheit
        converted_temp = (temp * 9/5) + 32
    elif conversion_type == 'f_to_c':
        # Convert Fahrenheit to Celsius
        converted_temp = (temp - 32) * 5/9
    else:
        raise ValueError("Invalid conversion type. Please use 'c_to_f' or 'f_to_c'.")

    return converted_temp
