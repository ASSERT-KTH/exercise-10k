## Exercise
Create a function that takes a list of numbers and returns the square of each number using the map function

## Reference Solution
```python
def square_numbers(numbers):
    """
    Returns a list of squares of the input numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        list: A list of squares of the input numbers.
    """
    return list(map(lambda x: x ** 2, numbers))

# Example usage
numbers = [1, 2, 3, 4, 5]
squares = square_numbers(numbers)
print(squares)  # Output: [1, 4, 9, 16, 25]
```

```python
def square(x):
    """
    Returns the square of a number.

    Args:
        x (int): A number.

    Returns:
        int: The square of the input number.
    """
    return x ** 2

def square_numbers(numbers):
    """
    Returns a list of squares of the input numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        list: A list of squares of the input numbers.
    """
    return list(map(square, numbers))

# Example usage
```
