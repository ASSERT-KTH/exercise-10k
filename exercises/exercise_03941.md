## Exercise
Write a program to accept a name and age as input and print out a greeting message with the name and age.

## Reference Solution
```python
def print_greeting(name, age):
    """
    Prints a greeting message with the given name and age.
    
    Parameters:
    name (str): The name of the person.
    age (int): The age of the person.
    """
    print(f"Hello, {name}! You are {age} years old.")

def main():
    """
    Asks for the user's name and age, then prints a greeting message.
    """
    name = input("Please enter your name: ")
    while True:
        try:
            age = int(input("Please enter your age: "))
            break
        except ValueError:
            print("That's not a valid age. Please try again.")
    
    print_greeting(name, age)

if __name__ == "__main__":
    main()
```
