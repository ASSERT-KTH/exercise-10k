## Exercise
Create a program that checks the data type of a given input and prints out whether it is an integer, float, or string. Use variables to store the input and its corresponding data type.

## Reference Solution
```python
def check_data_type(input_value):
    """
    Checks the data type of a given input and prints out whether it is an integer, float, or string.

    Args:
    input_value: The value to check the data type for.
    """
    # Store the input value in a variable
    user_input = input_value

    # Use the type() function to get the data type of the input
    input_type = type(user_input)

    # Check the data type and print the result
    if input_type == int:
        print(f"The data type of {user_input} is integer.")
    elif input_type == float:
        print(f"The data type of {user_input} is float.")
    elif input_type == str:
        print(f"The data type of {user_input} is string.")
    else:
        print(f"The data type of {user_input} is {input_type.__name__}.")

def main():
    # Test the function with different data types
    check_data_type(10)  # Integer
    check_data_type(10.5)  # Float
    check_data_type("Hello, World!")  # String

if __name__ == "__main__":
    main()
```
