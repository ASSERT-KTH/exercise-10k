## Exercise
Write a program that takes an array and a modification function as input, and returns the modified array.

## Reference Solution
```python
def modify_array(array, modification_function):
    """
    This function takes an array and a modification function as input, 
    and returns the modified array.

    Args:
        array (list): The input array to be modified.
        modification_function (function): A function that takes an element as input and returns the modified element.

    Returns:
        list: The modified array.
    """
    # Initialize an empty list to store the modified elements
    modified_array = []

    # Iterate over each element in the input array
    for element in array:
        # Apply the modification function to the current element and append the result to the modified array
        modified_array.append(modification_function(element))

    # Return the modified array
    return modified_array


# Example usage:
# Define a modification function that doubles the input number
def double_number(num):
    return num * 2

# Test the modify_array function
array = [1, 2, 3, 4, 5]
modified_array = modify_array(array, double_number)

# Print the modified array
print("Original Array:", array)
print("Modified Array:", modified_array)
```
