## Exercise
Write a program that takes a string as input, reverses the string, and checks if the original string is a palindrome. A palindrome is a word, phrase, number, or other sequences of characters that reads the same forward and backward (ignoring spaces, punctuation, and capitalization).

## Reference Solution
```python
import re

def reverse_string(s):
    """Reverses the input string."""
    return s[::-1]

def is_palindrome(s):
    """Checks if the input string is a palindrome."""
    # Remove spaces and punctuation, and convert to lowercase
    cleaned_s = re.sub(r'\W+', '', s).lower()
    # Compare the cleaned string with its reverse
    return cleaned_s == reverse_string(cleaned_s)

def main():
    # Get the input string from the user
    s = input("Enter a string: ")
    
    # Reverse the string
    reversed_s = reverse_string(s)
    print(f"Reversed string: {reversed_s}")
    
    # Check if the string is a palindrome
    if is_palindrome(s):
        print(f"'{s}' is a palindrome.")
    else:
        print(f"'{s}' is not a palindrome.")

if __name__ == "__main__":
    main()
```
