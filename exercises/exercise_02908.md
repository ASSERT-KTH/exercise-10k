## Exercise
Create a program to join a list of strings into a single string.

## Reference Solution
```python
def join_strings(string_list, separator=" "):
    """
    Joins a list of strings into a single string.

    Args:
        string_list (list): A list of strings to be joined.
        separator (str, optional): The separator to use between strings. Defaults to " ".

    Returns:
        str: The joined string.
    """
    return separator.join(string_list)


def main():
    # Example usage
    strings_to_join = ["Hello", "World", "This", "is", "a", "test"]
    joined_string = join_strings(strings_to_join, separator="-")
    print(joined_string)


if __name__ == "__main__":
    main()
```
