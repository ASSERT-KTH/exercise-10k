## Exercise
Understand the concept of variable scope and declare variables with local and global scope.

## Reference Solution
```python
# Global variable
global_variable = "I am global"

def my_function():
    # Local variable
    local_variable = "I am local"
    print("Inside function:")
    print("Global variable:", global_variable)
    print("Local variable:", local_variable)

my_function()

print("Outside function:")
print("Global variable:", global_variable)
# Trying to access local variable outside function will raise an error
try:
    print("Local variable:", local_variable)
except NameError:
    print("Local variable is not accessible outside function")
```

```python
global_variable = "I am global"

def my_function():
    global global_variable
    global_variable = "I am modified"
    print("Inside function:")
    print("Global variable:", global_variable)

my_function()

print("Outside function:")
print("Global variable:", global_variable)
```
