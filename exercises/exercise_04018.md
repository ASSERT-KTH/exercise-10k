## Exercise
Write a program that uses a for loop to sum all the odd numbers from 1 to 10.

## Reference Solution
```python
def sum_odd_numbers():
    total = 0
    for i in range(1, 11):
        if i % 2 != 0:
            total += i
    return total

print(sum_odd_numbers())
```
