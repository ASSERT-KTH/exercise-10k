## Exercise
Write a function to check if a string consists only of letters and digits.

## Reference Solution
```python
def check_string(s):
    """
    Checks if a string consists only of letters and digits.

    Args:
        s (str): The input string to be checked.

    Returns:
        bool: True if the string consists only of letters and digits, False otherwise.
    """
    return s.isalnum()
```

```python
print(check_string("HelloWorld123"))  # True
print(check_string("Hello World123"))  # False (contains space)
print(check_string("HelloWorld!23"))  # False (contains special character)
print(check_string("abcdefg"))  # True
print(check_string("1234567"))  # True
```
