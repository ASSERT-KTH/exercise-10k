## Exercise
Create a dictionary with 3 key-value pairs and loop through the key-value pairs.

## Reference Solution
```python
def main():
    # Create a dictionary with 3 key-value pairs
    my_dict = {
        "name": "John",
        "age": 30,
        "city": "New York"
    }

    # Loop through the key-value pairs
    for key, value in my_dict.items():
        print(f"{key}: {value}")

if __name__ == "__main__":
    main()
```
