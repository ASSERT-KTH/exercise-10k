## Exercise
Write a program to print a pattern of numbers from 1 to n, where n is the input provided by the user. The pattern should be printed using a for loop. For example, if the user inputs 5, the program should print: 1, 2, 3, 4, 5.

## Reference Solution
```python
# Author: [Your Name]
# Program: Print numbers from 1 to n using a for loop.

n = int(input("Enter a number: "))

for i in range(1, n+1):
    print(i, end=" ")
```
