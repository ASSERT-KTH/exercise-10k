## Exercise
Given a dictionary {'name': 'Kate', 'age': 20, 'city': 'Darwin'}, write a program to use the get() method to access the value of the 'name' key.

## Reference Solution
```python
def access_dictionary_value():
    # Define the dictionary
    person = {'name': 'Kate', 'age': 20, 'city': 'Darwin'}
    
    # Use the get() method to access the value of the 'name' key
    name = person.get('name')
    
    # Print the accessed value
    print("The person's name is:", name)

# Example usage
if __name__ == "__main__":
    access_dictionary_value()
```
