## Exercise
Write a program that takes an array and a partition size as input and returns a new array containing the same elements partitioned into subarrays of the specified size.

## Reference Solution
```python
function partition(arr, size) {
  let result = [];
  for (let i = 0; i < arr.length; i += size) {
    let chunk = arr.slice(i, i + size);
    result.push(chunk);
  }
  return result;
}
```
