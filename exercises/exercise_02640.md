## Exercise
Create a program that takes a phone number as input and generates a validation message.

## Reference Solution
```python
import re

def validate_phone_number(phone_number):
    """
    Validate a phone number and return a validation message.
    
    Args:
        phone_number (str): The phone number to validate.
    
    Returns:
        str: A validation message indicating whether the phone number is valid or not.
    """
    pattern = re.compile(r'^\d{3}-\d{3}-\d{4}$')
    if pattern.match(phone_number):
        return f"The phone number {phone_number} is valid."
    else:
        return f"The phone number {phone_number} is invalid. Please use the format XXX-XXX-XXXX."

def main():
    phone_number = input("Enter a phone number: ")
    print(validate_phone_number(phone_number))

if __name__ == "__main__":
    main()
```
