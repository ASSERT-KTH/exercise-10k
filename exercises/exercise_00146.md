## Exercise
Create a function that takes in a dictionary and a filter function, and returns a new dictionary containing only the key-value pairs that pass the filter

## Reference Solution
```python
def filter_dict(input_dict, filter_func):
    """
    Filters a dictionary based on a given function.

    Args:
    input_dict (dict): The dictionary to be filtered.
    filter_func (function): A function that takes in a key-value pair and returns a boolean.

    Returns:
    dict: A new dictionary containing only the key-value pairs that pass the filter.
    """
    return {key: value for key, value in input_dict.items() if filter_func(key, value)}

# Example usage:
def example_filter(key, value):
    # Filter out values that are not integers
    return isinstance(value, int)

example_dict = {
    'a': 1,
    'b': 'hello',
    'c': 3.5,
    'd': 4
}

filtered_dict = filter_dict(example_dict, example_filter)
print(filtered_dict)  # Output: {'a': 1, 'd': 4}
```
