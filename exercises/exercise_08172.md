## Exercise
Create a list of 5 integers and find the first element that meets a condition.

## Reference Solution
```python
def find_first_element(numbers, condition):
    """
    Find the first element in the list that meets the condition.

    Args:
        numbers (list): A list of integers.
        condition (function): A function that takes an integer and returns a boolean.

    Returns:
        The first element that meets the condition, or None if no element meets the condition.
    """
    for num in numbers:
        if condition(num):
            return num
    return None

# Define the condition function
def is_even(num):
    """
    Check if a number is even.

    Args:
        num (int): The number to check.

    Returns:
        True if the number is even, False otherwise.
    """
    return num % 2 == 0

# Create a list of 5 integers
numbers = [1, 3, 5, 7, 10]

# Find the first even number in the list
first_even = find_first_element(numbers, is_even)

print("List of numbers:", numbers)
print("First even number:", first_even)
```
