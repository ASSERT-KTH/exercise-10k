## Exercise
Write a program that takes a list as input, and returns a new list containing the same elements in sorted order.

## Reference Solution
```python
def sort_list(input_list):
    """
    Returns a new list containing the same elements as the input list, in sorted order.

    Args:
        input_list (list): The list to be sorted.

    Returns:
        list: A new list containing the same elements as the input list, in sorted order.
    """
    return sorted(input_list)

# Example usage
if __name__ == "__main__":
    input_list = [5, 2, 8, 1, 9]
    sorted_list = sort_list(input_list)
    print("Input List:", input_list)
    print("Sorted List:", sorted_list)
```
