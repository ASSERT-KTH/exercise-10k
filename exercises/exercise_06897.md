## Exercise
Write a program that checks if a given element is present in an array. The program should take two inputs: the array and the element to be checked. It should return 'True' if the element is present in the array, 'False' otherwise.

## Reference Solution
```python
def check_element(array, element):
    if element in array:
        return True
    else:
        return False

array = [1, 2, 3, 4, 5]
element = 3

result = check_element(array, element)
print(result)
```
