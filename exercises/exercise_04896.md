## Exercise
Return a view object that displays a list of all key-value pairs in a dictionary.

## Reference Solution
```python
class DictionaryView:
    def __init__(self, dictionary):
        """
        Initialize the DictionaryView object.

        Args:
        dictionary (dict): The dictionary to display.
        """
        self.dictionary = dictionary

    def display(self):
        """
        Display a list of all key-value pairs in the dictionary.

        Returns:
        list: A list of tuples containing the key-value pairs.
        """
        return list(self.dictionary.items())

    def print_view(self):
        """
        Print the dictionary view to the console.
        """
        for key, value in self.display():
            print(f"Key: {key}, Value: {value}")


# Example usage
if __name__ == "__main__":
    example_dict = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }

    view = DictionaryView(example_dict)
    print("Dictionary View:")
    view.print_view()
```
