## Exercise
Write a program to calculate the range of a list of numbers

## Reference Solution
```python
def calculate_range(numbers):
    """
    Calculate the range of a list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        float: The range of the list of numbers.
    """
    if not numbers:
        raise ValueError("List of numbers cannot be empty")

    return max(numbers) - min(numbers)


def main():
    # Example usage:
    numbers = [10, 20, 30, 40, 50]
    try:
        range_value = calculate_range(numbers)
        print(f"The range of the list is: {range_value}")
    except ValueError as e:
        print(e)


if __name__ == "__main__":
    main()
```
