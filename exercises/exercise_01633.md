## Exercise
Create a program that takes a list as input and returns True if it contains any duplicate elements, False otherwise.

## Reference Solution
```python
def has_duplicates(input_list):
    """
    Checks if a list contains any duplicate elements.

    Args:
        input_list (list): The list to check for duplicates.

    Returns:
        bool: True if the list contains any duplicate elements, False otherwise.
    """
    return len(input_list) != len(set(input_list))

# Example usage:
if __name__ == "__main__":
    print(has_duplicates([1, 2, 3, 4, 5]))  # False
    print(has_duplicates([1, 2, 3, 4, 4]))  # True
    print(has_duplicates(['a', 'b', 'c', 'd']))  # False
    print(has_duplicates(['a', 'b', 'c', 'a']))  # True
```
