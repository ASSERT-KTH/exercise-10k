## Exercise
Write a program that uses a for loop to check if a password meets certain criteria (e.g., length, uppercase, lowercase, digits).

## Reference Solution
```python
def validate_password(password):
    """
    Validate a password based on certain criteria.

    Args:
        password (str): The password to validate.

    Returns:
        bool: True if the password is valid, False otherwise.
    """
    # Define the criteria
    criteria = [
        lambda p: len(p) >= 8,  # Minimum length of 8 characters
        lambda p: any(c.isupper() for c in p),  # At least one uppercase letter
        lambda p: any(c.islower() for c in p),  # At least one lowercase letter
        lambda p: any(c.isdigit() for c in p),  # At least one digit
    ]

    # Use a for loop to check each criterion
    for criterion in criteria:
        if not criterion(password):
            return False

    return True


def main():
    # Get the password from the user
    password = input("Enter a password: ")

    # Validate the password
    if validate_password(password):
        print("Password is valid.")
    else:
        print("Password is not valid. Please ensure it meets the following criteria:")
        print("* Minimum length of 8 characters")
        print("* At least one uppercase letter")
        print("* At least one lowercase letter")
        print("* At least one digit")


if __name__ == "__main__":
    main()
```
