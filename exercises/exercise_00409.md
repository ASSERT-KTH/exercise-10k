## Exercise
Write a program to find the first duplicate in an array.

## Reference Solution
```python
def find_first_duplicate(arr):
    """
    Finds the first duplicate in an array.

    Args:
        arr (list): The input array.

    Returns:
        int: The first duplicate element, or None if no duplicates are found.
    """
    seen = set()
    for num in arr:
        if num in seen:
            return num
        seen.add(num)
    return None

# Example usage
arr = [2, 1, 3, 5, 3, 2]
duplicate = find_first_duplicate(arr)
if duplicate is not None:
    print(f"The first duplicate is: {duplicate}")
else:
    print("No duplicates found")
```
