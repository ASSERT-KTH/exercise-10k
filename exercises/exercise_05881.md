## Exercise
Write a function that takes a list of integers as input and returns the reversed list.

## Reference Solution
```python
def reverse_list(input_list):
    """
    This function takes a list of integers as input, reverses it, and returns the reversed list.

    Args:
        input_list (list): A list of integers.

    Returns:
        list: The reversed list of integers.
    """

    # First, we check if the input is a list. If not, we raise a TypeError.
    if not isinstance(input_list, list):
        raise TypeError("Input must be a list.")

    # Then, we check if all elements in the list are integers. If not, we raise a ValueError.
    if not all(isinstance(num, int) for num in input_list):
        raise ValueError("All elements in the list must be integers.")

    # Now, we use Python's built-in list slicing feature to create a new list that is the reverse of the input list.
    # The syntax input_list[start:stop:step] means start at the start index, stop at the stop index, and step by the step value.
    # By using ::-1, we're telling Python to start at the end of the list, stop at the beginning, and step backwards by 1.
    reversed_list = input_list[::-1]

    # Finally, we return the reversed list.
    return reversed_list
```
