## Exercise
Write a program that asks the user for a string and a number, and prints the string repeated that many times

## Reference Solution
```python
def repeat_string():
    """
    Asks the user for a string and a number, and prints the string repeated that many times.
    """
    # Get user input
    user_string = input("Please enter a string: ")
    while True:
        try:
            repeat_count = int(input("Please enter the number of times to repeat the string: "))
            if repeat_count < 0:
                print("Please enter a non-negative number.")
            else:
                break
        except ValueError:
            print("Invalid input. Please enter a whole number.")

    # Repeat the string and print the result
    repeated_string = user_string * repeat_count
    print(repeated_string)

# Example usage
if __name__ == "__main__":
    repeat_string()
```
