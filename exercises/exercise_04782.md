## Exercise
Write a program that asks the user for their name and then prints out a greeting message.

## Reference Solution
```python
def get_user_name():
    """Get the user's name from input."""
    user_name = input("Please enter your name: ")
    return user_name

def print_greeting(name):
    """Print a greeting message for the given name."""
    print(f"Hello, {name}! Welcome!")

def main():
    """Main function to run the program."""
    user_name = get_user_name()
    print_greeting(user_name)

if __name__ == "__main__":
    main()
```
