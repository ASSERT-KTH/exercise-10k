## Exercise
Create a program that takes a number as input and returns the sum of its digits.

## Reference Solution
```python
def sum_of_digits(n):
    """
    Calculate the sum of digits of a given number.

    Args:
        n (int): The input number.

    Returns:
        int: The sum of digits of the input number.
    """
    return sum(int(digit) for digit in str(abs(n)))

# Test the function
num = int(input("Enter a number: "))
print("The sum of its digits is:", sum_of_digits(num))
```
