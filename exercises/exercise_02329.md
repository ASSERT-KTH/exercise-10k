## Exercise
Write a program to print a range of numbers to the screen, from 1 to 10.

## Reference Solution
```python
def print_range(start, end):
    """
    Prints a range of numbers from start to end (inclusive).
    
    Args:
        start (int): The starting number of the range.
        end (int): The ending number of the range.
    """
    for i in range(start, end + 1):
        print(i)

def main():
    start = 1
    end = 10
    print_range(start, end)

if __name__ == "__main__":
    main()
```
