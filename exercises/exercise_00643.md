## Exercise
Write a program that reads a single line of input from the user and checks if it contains only digits. If it does, print 'Valid'. Otherwise, print 'Invalid'.

## Reference Solution
```python
import sys

def check_line(line):
    if line.isdigit():
        return 'Valid'
    else:
        return 'Invalid'

line = sys.stdin.readline().strip()

print(check_line(line))
```
