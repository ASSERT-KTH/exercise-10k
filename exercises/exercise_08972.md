## Exercise
Write a function that takes a string and a character as input and returns the count of the character in the string.

## Reference Solution
```python
def count_char(string, char):
    """
    Returns the count of a character in a string.

    Args:
        string (str): The input string.
        char (str): The character to count.

    Returns:
        int: The count of the character in the string.
    """
    return string.count(char)

# Example usage:
def main():
    string = "Hello, World!"
    char = "l"
    count = count_char(string, char)
    print(f"The character '{char}' appears {count} times in the string '{string}'.")

if __name__ == "__main__":
    main()
```
