## Exercise
Create a function that finds the first occurrence of a substring within a larger string.

## Reference Solution
```python
def find_substring(main_string, sub_string):
    """
    Finds the first occurrence of a substring within a larger string.

    Args:
        main_string (str): The larger string to search in.
        sub_string (str): The substring to search for.

    Returns:
        int: The index of the first occurrence of the substring. Returns -1 if not found.
    """
    try:
        index = main_string.index(sub_string)
        return index
    except ValueError:
        return -1

# Example usage:
main_str = "Hello, world! Hello again!"
sub_str = "Hello"
print(find_substring(main_str, sub_str))  # Output: 0
print(find_substring(main_str, "Goodbye"))  # Output: -1
```
