## Exercise
Write a function that takes a single argument with a default value and returns its double

## Reference Solution
```python
def double_value(x=5):
    """
    This function takes an integer as input and returns its double.
    
    Args:
        x (int): The input number. Defaults to 5.
    
    Returns:
        int: The double of the input number.
    """
    return x * 2

# Test the function
print(double_value())  # Output: 10
print(double_value(10))  # Output: 20
```
