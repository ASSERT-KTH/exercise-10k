## Exercise
Log complex variables to track changes and improve debugging.

## Reference Solution
```python
import logging
import json

# Configure logging
logging.basicConfig(level=logging.DEBUG)

# Example of logging a complex variable
complex_variable = {
    "key1": "value1",
    "key2": {
        "nested_key1": "nested_value1",
        "nested_key2": "nested_value2"
    }
}

logging.debug("Complex variable: %s", json.dumps(complex_variable, indent=4))
```

```python
const winston = require('winston');

// Configure logging
const logger = winston.createLogger({
  level: 'debug',
  format: winston.format.json(),
  transports: [
    new winston.transports.Console()
  ]
});

// Example of logging a complex variable
const complexVariable = {
  key1: "value1",
  key2: {
    nestedKey1: "nestedValue1",
    nestedKey2: "nestedValue2"
  }
};

logger.debug("Complex variable: %o", complexVariable);
```

```python
import logging
import json

# Before modification
logging.debug("Before modification: %s", json.dumps(complex_variable, indent=4))

# Modify the complex variable
```
