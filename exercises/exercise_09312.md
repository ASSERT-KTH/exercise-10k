## Exercise
Create a function that takes a string as input and returns the string with all duplicate characters removed.

## Reference Solution
```python
def remove_duplicates(input_string):
    """
    Removes all duplicate characters from a given string.

    Args:
    input_string (str): The input string from which to remove duplicates.

    Returns:
    str: The input string with all duplicate characters removed.
    """
    seen = set()
    result = []
    
    for char in input_string:
        if char not in seen:
            result.append(char)
            seen.add(char)
    
    return ''.join(result)

# Example usage:
if __name__ == "__main__":
    input_str = "Hello, World!"
    print("Original string:", input_str)
    print("String with duplicates removed:", remove_duplicates(input_str))
```
