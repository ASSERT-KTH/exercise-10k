## Exercise
Write a function to check if a value exists in a dictionary.

## Reference Solution
```python
def check_value_in_dict(dictionary, value):
    """
    Checks if a value exists in a dictionary.

    Args:
        dictionary (dict): The dictionary to search in.
        value: The value to search for.

    Returns:
        bool: True if the value exists, False otherwise.
    """
    return value in dictionary.values()

# Example usage
if __name__ == "__main__":
    my_dict = {"a": 1, "b": 2, "c": 3}
    print(check_value_in_dict(my_dict, 2))  # Output: True
    print(check_value_in_dict(my_dict, 4))  # Output: False
```
