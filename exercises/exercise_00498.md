## Exercise
Write a program that takes a list and a predicate function as input, and returns two separate lists containing elements that satisfy and don't satisfy the predicate function.

## Reference Solution
```python
def separate_list(lst, predicate):
    if not lst:
        return [], []

    satisfied = []
    not_satisfied = []

    for i in lst:
        if predicate(i):
            satisfied.append(i)
        else:
            not_satisfied.append(i)

    return satisfied, not_satisfied

# Example usage:
def is_odd(n):
    return n % 2 != 0

numbers = [1, 2, 3, 4, 5, 6]
odd_numbers, even_numbers = separate_list(numbers, is_odd)
print("Odd numbers:", odd_numbers)
print("Even numbers:", even_numbers)
```
