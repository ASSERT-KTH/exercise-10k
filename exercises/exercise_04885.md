## Exercise
Iterate over the key-value pairs of a dictionary and print them out.

## Reference Solution
```python
# Define a dictionary
my_dict = {
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}

# Iterate over key-value pairs and print them
for key, value in my_dict.items():
    print(f"Key: {key}, Value: {value}")
```
