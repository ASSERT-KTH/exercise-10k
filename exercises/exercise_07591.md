## Exercise
Write a program that checks if a given variable is a set.

## Reference Solution
```python
def is_set(var):
    """
    Checks if a given variable is a set.

    Args:
        var: The variable to check.

    Returns:
        bool: True if the variable is a set, False otherwise.
    """
    return isinstance(var, set)

# Example usage:
if __name__ == "__main__":
    my_set = {1, 2, 3}
    my_list = [1, 2, 3]
    my_dict = {"a": 1, "b": 2}

    print(is_set(my_set))  # Expected output: True
    print(is_set(my_list))  # Expected output: False
    print(is_set(my_dict))  # Expected output: False
```
