## Exercise
Create a program that asks the user for a date in the format MM/DD/YYYY and then checks if it is a valid date. Print out a message indicating whether the date is valid or not

## Reference Solution
```python
from datetime import datetime

def validate_date(date_str):
    """Validate a date string in the format MM/DD/YYYY."""
    try:
        datetime.strptime(date_str, '%m/%d/%Y')
        return True
    except ValueError:
        return False

def main():
    date_str = input("Please enter a date in the format MM/DD/YYYY: ")
    if validate_date(date_str):
        print(f"The date {date_str} is valid.")
    else:
        print(f"The date {date_str} is not valid.")

if __name__ == "__main__":
    main()
```
