## Exercise
Write a program that removes duplicate characters from a given string.

## Reference Solution
```python
def remove_duplicates(input_string):
    """
    Remove duplicate characters from a given string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The string with duplicate characters removed.
    """
    seen = set()
    result = []

    for char in input_string:
        if char not in seen:
            seen.add(char)
            result.append(char)

    return ''.join(result)

# Example usage:
def main():
    input_string = "Hello, World!"
    print("Original String:", input_string)
    print("String without duplicates:", remove_duplicates(input_string))

if __name__ == "__main__":
    main()
```
