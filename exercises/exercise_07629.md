## Exercise
Write a program that uses a for loop to translate a given string into Pig Latin.

## Reference Solution
```python
def pig_latin_translator(input_string):
    """
    Translates a given string into Pig Latin.

    Args:
        input_string (str): The input string to be translated.

    Returns:
        str: The translated string in Pig Latin.
    """
    vowels = 'aeiou'
    words = input_string.split()
    translated_words = []

    for word in words:
        if word[0].lower() in vowels:
            translated_word = word + 'way'
        else:
            translated_word = word[1:] + word[0] + 'ay'
        translated_words.append(translated_word)

    return ' '.join(translated_words)


# Example usage
def main():
    input_string = "hello world python"
    print("Input String:", input_string)
    print("Pig Latin Translation:", pig_latin_translator(input_string))


if __name__ == "__main__":
    main()
```
