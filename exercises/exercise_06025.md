## Exercise
Create a function that takes a variable number of arguments and returns their sum. Use the function to calculate the sum of 1, 2, 3, and 4.

## Reference Solution
```python
def sum_variables(*args):
    """
    Calculate the sum of variable arguments.

    Args:
    *args: A variable number of numeric arguments.

    Returns:
    The sum of all arguments.

    Raises:
    TypeError: If any argument is not a number.
    """
    try:
        return sum(args)
    except TypeError:
        raise TypeError("All arguments must be numbers.")

# Example usage:
if __name__ == "__main__":
    result = sum_variables(1, 2, 3, 4)
    print(f"The sum is: {result}")
```
