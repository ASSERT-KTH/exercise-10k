## Exercise
Write a program that takes a list of numbers as input and calculates the sum of all elements in the list.

## Reference Solution
```python
def calculate_sum(numbers):
    """
    This function calculates the sum of all elements in a given list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        int: The sum of all elements in the list.
    """
    return sum(numbers)

# Example usage:
numbers = [1, 2, 3, 4, 5]
result = calculate_sum(numbers)
print("The sum of the list is:", result)
```
