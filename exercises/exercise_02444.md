## Exercise
Write a program that takes a list and a mapping function as input and returns a new list containing the results of applying the mapping function to each element.

## Reference Solution
```python
def map_list(input_list, mapping_function):
    new_list = []
    for x in input_list:
        new_list.append(mapping_function(x))
    return new_list

list = [1, 2, 3]
result = map_list(list, lambda x: x*x)
print(result)
```
