## Exercise
Write a program that takes two arrays as input, and returns a boolean indicating whether the first array is a subset of the second array.

## Reference Solution
```python
def is_subset(arr1, arr2):
    return set(arr1).issubset(set(arr2))

# Test the function
print(is_subset([1, 2, 3], [1, 2, 3, 4, 5]))  # True
print(is_subset([1, 2, 3, 6], [1, 2, 3, 4, 5]))  # False
```
