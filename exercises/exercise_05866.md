## Exercise
Find the index of a specific element in a given list.

## Reference Solution
```python
def find_index(lst, element):
    """
    Find the index of a specific element in a given list.

    Args:
        lst (list): The list to search in.
        element: The element to search for.

    Returns:
        int: The index of the element if found, -1 otherwise.
    """
    try:
        return lst.index(element)
    except ValueError:
        return -1

# Example usage
my_list = [1, 2, 3, 4, 5]
element = 3
index = find_index(my_list, element)

if index != -1:
    print(f"The element {element} is at index {index}.")
else:
    print(f"The element {element} is not in the list.")
```
