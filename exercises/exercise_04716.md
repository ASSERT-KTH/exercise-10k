## Exercise
Update a dictionary with key-value pairs using keyword arguments

## Reference Solution
```python
def update_dict(original_dict, **kwargs):
    """
    Update a dictionary with key-value pairs using keyword arguments.

    Args:
        original_dict (dict): The dictionary to be updated.
        **kwargs: Key-value pairs to update the dictionary.

    Returns:
        dict: The updated dictionary.
    """
    original_dict.update(kwargs)
    return original_dict

# Example usage:
original_dict = {"a": 1, "b": 2}
updated_dict = update_dict(original_dict, c=3, d=4)
print(updated_dict)  # Output: {'a': 1, 'b': 2, 'c': 3, 'd': 4}
```
