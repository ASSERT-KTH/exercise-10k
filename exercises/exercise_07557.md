## Exercise
Create a function that takes a name as input and prints out a greeting message, with a default greeting if no name is provided.

## Reference Solution
```python
def print_greeting(name=None):
    """
    Prints out a greeting message with the provided name.
    
    If no name is provided, it will use a default greeting.

    Args:
        name (str, optional): The name to be used in the greeting. Defaults to None.
    """

    # First, we check if the name is None, which means no name was provided
    if name is None:
        # If no name is provided, we use a default greeting
        greeting_message = "Hello, World!"
    else:
        # If a name is provided, we use it in the greeting message
        greeting_message = f"Hello, {name}!"

    # Finally, we print out the greeting message
    print(greeting_message)

# Test the function with a name
print_greeting("John")

# Test the function without a name
print_greeting()
```
