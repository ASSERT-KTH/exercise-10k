## Exercise
Create a function that takes a person's name as an argument and returns a greeting message.

## Reference Solution
```python
# Define a function called 'greet' that takes one argument 'name'.
def greet(name):
    """
    This function generates a greeting message based on the input name.

    Args:
        name (str): The person's name.

    Returns:
        str: A greeting message with the person's name.
    """

    # Check if the input is a string. If not, raise a TypeError.
    if not isinstance(name, str):
        raise TypeError("Input must be a string.")

    # Use an f-string to format the greeting message with the person's name.
    # This is a more modern and efficient way of formatting strings in Python.
    greeting_message = f"Hello, {name}!"

    # Return the greeting message.
    return greeting_message

# Test the function with a sample name.
print(greet("John"))  # Output: Hello, John!
```
