## Exercise
Write a program that finds the maximum contiguous subarray of an array of integers. The program should take an array of integers as input and return the maximum sum of a contiguous subarray. If the array is empty, the program should return 0.

## Reference Solution
```python
def maxSubArray(nums):
    if not nums:
        return 0

    max_ = nums[0]
    cur = nums[0]

    for num in nums[1:]:
        cur = max(num, cur + num)
        max_ = max(max_, cur)

    return max_

# Example usage:
print(maxSubArray([-2, 1, -3, 4, -1, 2, 1, -5, 4]))  # Output: 6
```
