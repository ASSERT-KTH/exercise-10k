## Exercise
Create a program that uses a for loop to convert an integer to a roman numeral.

## Reference Solution
```python
def int_to_roman(num):
    """
    Converts an integer to a Roman numeral.

    :param num: The integer to convert.
    :return: The Roman numeral representation of the integer.
    """
    roman_numerals = [
        (1000, 'M'),
        (900, 'CM'),
        (500, 'D'),
        (400, 'CD'),
        (100, 'C'),
        (90, 'XC'),
        (50, 'L'),
        (40, 'XL'),
        (10, 'X'),
        (9, 'IX'),
        (5, 'V'),
        (4, 'IV'),
        (1, 'I')
    ]

    result = ''
    for value, numeral in roman_numerals:
        while num >= value:
            num -= value
            result += numeral

    return result


def main():
    num = int(input("Enter an integer: "))
    if 0 < num < 4000:
        print(int_to_roman(num))
    else:
        print("Please enter a number between 1 and 3999.")


if __name__ == "__main__":
    main()
```
