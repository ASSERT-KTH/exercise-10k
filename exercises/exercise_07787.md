## Exercise
Write a program that takes a user input and determines its data type. If the input is a string, check if it can be converted to an integer or a float. If the input can be converted, print out the converted value and its data type. If the input cannot be converted, print out an error message.

## Reference Solution
```python
def check_data_type(user_input):
    # Check if input is a string
    if isinstance(user_input, str):
        # Try to convert string to integer
        try:
            int_value = int(user_input)
            print(f"Converted integer value: {int_value} (Type: int)")
            return
        except ValueError:
            pass

        # Try to convert string to float
        try:
            float_value = float(user_input)
            print(f"Converted float value: {float_value} (Type: float)")
            return
        except ValueError:
            print(f"Error: '{user_input}' cannot be converted to a number.")
            return

    # If input is not a string, print its type
    print(f"Input type: {type(user_input).__name__}")


def main():
    user_input = input("Enter a value: ")
    check_data_type(user_input)


if __name__ == "__main__":
    main()
```
