## Exercise
Create a program that uses a for loop to count down from a given number and print each number on a new line.

## Reference Solution
```python
def count_down(n):
    """
    Counts down from n and prints each number on a new line.

    Args:
        n (int): The number to start counting down from.
    """
    for i in range(n, 0, -1):
        print(i)

# Example usage:
count_down(10)
```
