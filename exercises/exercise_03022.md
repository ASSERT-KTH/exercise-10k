## Exercise
Write a program that takes a string and a substring as input, and returns the number of occurrences of the substring in the string.

## Reference Solution
```python
def count_substring(string, sub_string):
    """
    This function counts the number of occurrences of a substring in a given string.

    Args:
        string (str): The main string.
        sub_string (str): The substring to be searched.

    Returns:
        int: The number of occurrences of the substring in the string.
    """

    # Initialize a counter variable to store the count of occurrences
    count = 0
    
    # Check if the substring is not empty
    if len(sub_string) != 0:
        
        # Iterate over the string with a sliding window of size equal to the length of the substring
        for i in range(len(string) - len(sub_string) + 1):
            
            # Check if the current window matches the substring
            if string[i:i+len(sub_string)] == sub_string:
                
                # If it matches, increment the counter
                count += 1
                
    # Return the count of occurrences
    return count


# Test the function
string = "ABCDCDC"
sub_string = "CDC"
print(count_substring(string, sub_string))  # Output: 2
```
