## Exercise
Write a program to create a copy of a given dictionary.

## Reference Solution
```python
def copy_dictionary(input_dict):
    """
    Creates a copy of the input dictionary.

    Args:
        input_dict (dict): The dictionary to be copied.

    Returns:
        dict: A copy of the input dictionary.
    """
    return input_dict.copy()


# Example usage:
if __name__ == "__main__":
    original_dict = {"a": 1, "b": 2, "c": 3}
    print("Original Dictionary:", original_dict)

    copied_dict = copy_dictionary(original_dict)
    print("Copied Dictionary:", copied_dict)

    # Modify the copied dictionary to demonstrate it's a separate copy
    copied_dict["d"] = 4
    print("Modified Copied Dictionary:", copied_dict)
    print("Original Dictionary (unchanged):", original_dict)
```
