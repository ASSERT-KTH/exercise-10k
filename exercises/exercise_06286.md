## Exercise
Check if a given key is present in a dictionary

## Reference Solution
```python
def check_key(dictionary, key):
    """
    Check if a given key is present in a dictionary.

    Args:
        dictionary (dict): The dictionary to check.
        key (any): The key to look for.

    Returns:
        bool: True if the key is present, False otherwise.
    """
    return key in dictionary

# Example usage:
my_dict = {"name": "John", "age": 30}
key_to_check = "name"
result = check_key(my_dict, key_to_check)

print(f"Is '{key_to_check}' present in the dictionary? {result}")

key_to_check = "city"
result = check_key(my_dict, key_to_check)
```
