## Exercise
Write a program that takes a list and a slice specification as input, and returns the slice of the list.

## Reference Solution
```python
def slice_list(input_list, slice_spec):
    if len(slice_spec) == 2:
        start, end = slice_spec
        return input_list[start:end]
    elif len(slice_spec) == 3:
        start, end, step = slice_spec
        return input_list[start:end:step]
    else:
        raise ValueError("Invalid slice specification")

# Example usage:
def main():
    input_list = [1, 2, 3, 4, 5, 6, 7, 8, 9]
    slice_spec = [1, 7, 2]
    result = slice_list(input_list, slice_spec)
    print(result)

main()
```
