## Exercise
Write a program that asks the user for a number and prints it to the screen

## Reference Solution
```python
def get_user_number():
    """Asks the user for a number and returns it."""
    while True:
        try:
            user_number = float(input("Please enter a number: "))
            return user_number
        except ValueError:
            print("Invalid input. Please enter a valid number.")

def print_number(number):
    """Prints the given number to the screen."""
    print(f"You entered: {number}")

def main():
    user_number = get_user_number()
    print_number(user_number)

if __name__ == "__main__":
    main()
```
