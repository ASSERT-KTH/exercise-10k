## Exercise
Write a program that converts a given string to uppercase using a for loop.

## Reference Solution
```python
def convert_to_uppercase(input_string):
    """
    Converts a given string to uppercase using a for loop.

    Args:
        input_string (str): The string to be converted.

    Returns:
        str: The input string in uppercase.
    """
    uppercase_string = ""
    for char in input_string:
        # Check if the character is an alphabet letter
        if char.isalpha():
            # Convert the character to uppercase
            uppercase_string += char.upper()
        else:
            # If not a letter, add it as is
            uppercase_string += char
    return uppercase_string


def main():
    # Example usage
    input_str = "Hello, World!"
    print("Input String:", input_str)
    print("Uppercase String:", convert_to_uppercase(input_str))


if __name__ == "__main__":
    main()
```
