## Exercise
Write a function that checks if a given list is a subset of another list.

## Reference Solution
```python
def is_subset(main_list, subset_list):
    """
    Checks if subset_list is a subset of main_list.

    Args:
        main_list (list): The main list to check against.
        subset_list (list): The list to check as a subset.

    Returns:
        bool: True if subset_list is a subset of main_list, False otherwise.
    """
    return set(subset_list).issubset(set(main_list))

# Example usage:
if __name__ == "__main__":
    main_list = [1, 2, 3, 4, 5]
    subset_list1 = [1, 2, 3]
    subset_list2 = [1, 2, 6]

    print(f"Is {subset_list1} a subset of {main_list}? {is_subset(main_list, subset_list1)}")
    print(f"Is {subset_list2} a subset of {main_list}? {is_subset(main_list, subset_list2)}")
```
