## Exercise
Create a program that takes two integers as input and outputs the maximum and minimum values.

## Reference Solution
```python
def find_max_min(a, b):
    """
    This function takes two integers as input and returns the maximum and minimum values.
    
    Parameters:
    a (int): The first integer.
    b (int): The second integer.
    
    Returns:
    tuple: A tuple containing the maximum and minimum values.
    """
    max_val = max(a, b)
    min_val = min(a, b)
    
    return max_val, min_val

def main():
    # Get the input from the user
    a = int(input("Enter the first integer: "))
    b = int(input("Enter the second integer: "))
    
    # Find the maximum and minimum values
    max_val, min_val = find_max_min(a, b)
    
    # Print the results
    print(f"Maximum value: {max_val}")
    print(f"Minimum value: {min_val}")

if __name__ == "__main__":
    main()
```
