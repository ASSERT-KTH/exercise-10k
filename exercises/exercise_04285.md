## Exercise
Write a function to iterate over a dictionary's key-value pairs and print each pair.

## Reference Solution
```python
def print_dict_key_value_pairs(dictionary):
    """
    Iterate over a dictionary's key-value pairs and print each pair.

    Args:
        dictionary (dict): The dictionary to iterate over.
    """
    for key, value in dictionary.items():
        print(f"Key: {key}, Value: {value}")

# Example usage:
if __name__ == "__main__":
    example_dict = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }
    print_dict_key_value_pairs(example_dict)
```
