## Exercise
Create a function to calculate the sum of a series of numbers.

## Reference Solution
```python
def series_sum(numbers):
    """
    Calculate the sum of a series of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        float: The sum of the series.
    """
    return sum(numbers)

# Example usage:
numbers = [1, 2, 3, 4, 5]
result = series_sum(numbers)
print("The sum of the series is:", result)
```
