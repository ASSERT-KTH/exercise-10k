## Exercise
Create a program that advises the user to take an umbrella or sunscreen based on the weather condition using if/else statements.

## Reference Solution
```python
def weather_advisor(weather_condition):
    """
    Advises the user to take an umbrella or sunscreen based on the weather condition.

    Args:
        weather_condition (str): The current weather condition.

    Returns:
        str: The advice based on the weather condition.
    """
    if weather_condition.lower() == "rainy":
        return "Take an umbrella with you."
    elif weather_condition.lower() in ["sunny", "cloudy"]:
        return "Apply sunscreen to protect your skin."
    else:
        return "Check the weather forecast for more information."

def main():
    print("Weather Advisor Program")
    print("------------------------")
    
    weather_condition = input("Enter the current weather condition (e.g., rainy, sunny, cloudy): ")
    advice = weather_advisor(weather_condition)
    
    print(advice)

if __name__ == "__main__":
    main()
```
