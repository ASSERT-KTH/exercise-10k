## Exercise
Write a program to transpose a list of lists and print the resulting list.

## Reference Solution
```python
def transpose_list(input_list):
    """
    Transpose a list of lists.

    Args:
        input_list (list): A list of lists to be transposed.

    Returns:
        list: The transposed list.
    """
    return list(map(list, zip(*input_list)))

def main():
    # Example usage
    input_list = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
    print("Input List:")
    for row in input_list:
        print(row)

    transposed_list = transpose_list(input_list)
    print("\nTransposed List:")
    for row in transposed_list:
        print(row)

if __name__ == "__main__":
    main()
```
