## Exercise
Write a function that takes a list as input and returns the sum of its elements.

## Reference Solution
```python
def sum_list_elements(input_list):
    """
    Calculates the sum of all elements in a list.

    Args:
    input_list (list): A list of numbers.

    Returns:
    int or float: The sum of all elements in the list.

    Raises:
    TypeError: If the input is not a list or if the list contains non-numeric values.
    """
    if not isinstance(input_list, list):
        raise TypeError("Input must be a list.")
    try:
        return sum(input_list)
    except TypeError:
        raise TypeError("List must only contain numeric values.")

# Example usage
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    result = sum_list_elements(numbers)
    print("Sum of list elements:", result)
```
