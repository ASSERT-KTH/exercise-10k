## Exercise
Write a program that finds the maximum contiguous subarray of a given list of integers. The program should take a list of integers as input and output the maximum subarray and its sum.

## Reference Solution
```python
def maxsubarray(arr):
    max_ = arr[0]
    max_sum = arr[0]
    start = 0
    end = 0
    temp = arr[0]
    temp_start = 0

    for i in range(1, len(arr)):
        if arr[i] < temp + arr[i]:
            temp = temp + arr[i]
        else:
            temp = arr[i]
            temp_start = i

        if temp > max_:
            max_ = temp
            max_sum = max_
            start = temp_start
            end = i

    return arr[start:end+1], max_sum

# Example usage
arr = [2, 3, 4, 5, 7]
result = maxsubarray(arr)
print("Maximum contiguous subarray:", result[0])
print("Sum of maximum contiguous subarray:", result[1])
```
