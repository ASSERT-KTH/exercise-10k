## Exercise
Create a function that encodes a given string using the Base64 encoding scheme.

## Reference Solution
```python
import base64

def encode_base64(input_string):
    """
    Encodes a given string using the Base64 encoding scheme.

    Args:
        input_string (str): The string to be encoded.

    Returns:
        str: The Base64 encoded string.
    """

    # First, we need to convert the input string into bytes because the base64 library in Python works with bytes-like objects.
    input_bytes = input_string.encode('utf-8')

    # Now, we use the base64.b64encode function to encode the bytes into Base64.
    encoded_bytes = base64.b64encode(input_bytes)

    # Finally, we convert the encoded bytes back into a string using the decode method.
    encoded_string = encoded_bytes.decode('utf-8')

    return encoded_string

# Example usage:
print(encode_base64("Hello, World!"))
```
