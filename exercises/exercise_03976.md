## Exercise
Implement the heapsort algorithm to sort a list of integers in ascending order. The program should take a list of integers as input and output the sorted list.

## Reference Solution
```python
def heapsort(lst):
    def heapify(lst, n, i):
        largest = i
        left = 2 * i + 1
        right = 2 * i + 2

        if left < n and lst[left] > lst[largest]:
            largest = left

        if right < n and lst[right] > lst[largest]:
            largest = right

        if largest != i:
            lst[i], lst[largest] = lst[largest], lst[i]
            heapify(lst, n, largest)

    n = len(lst)
    for i in range(n // 2 - 1, -1, -1):
        heapify(lst, n, i)

    for i in range(n - 1, 0, -1):
        lst[0], lst[i] = lst[i], lst[0]
        heapify(lst, i, 0)

    return lst

lst = [12, 11, 13, 5, 6, 7]
sorted_lst = heapsort(lst)
print(sorted_lst)
```
