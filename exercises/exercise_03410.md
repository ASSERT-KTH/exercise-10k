## Exercise
Write a program to retrieve all keys from a given dictionary.

## Reference Solution
```python
def get_dictionary_keys(input_dict):
    """
    Retrieves all keys from a given dictionary.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        list: A list of all keys in the dictionary.
    """
    return list(input_dict.keys())

# Example usage
def main():
    example_dict = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }
    keys = get_dictionary_keys(example_dict)
    print("Dictionary Keys:", keys)

if __name__ == "__main__":
    main()
```
