## Exercise
Write a program that prints prime numbers from 1 to 20 using a for loop.

## Reference Solution
```python
def is_prime(n):
    """Check if a number is prime."""
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

def print_primes(n):
    """Print prime numbers from 1 to n."""
    for i in range(1, n+1):
        if is_prime(i):
            print(i)

print_primes(20)
```
