## Exercise
Write a program that takes a string input from the user and checks if it's a palindrome. If the string is a palindrome, the program should reverse the string and print it. Otherwise, it should print an error message indicating that the input string is not a palindrome.

## Reference Solution
```python
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    cin >> s;
    string temp = s;
    reverse(temp.begin(), temp.end());
    if (temp == s) {
        cout << temp << endl;
    } else {
        cout << "NO" << endl;
    }
    return 0;
}
```
