## Exercise
Write a program that converts a string to uppercase and lowercase.

## Reference Solution
```python
def convert_string_case(input_string):
    """
    Converts a string to uppercase and lowercase.

    Args:
        input_string (str): The input string to convert.

    Returns:
        tuple: A tuple containing the uppercase and lowercase versions of the input string.
    """
    uppercase_string = input_string.upper()
    lowercase_string = input_string.lower()
    return uppercase_string, lowercase_string

def main():
    # Get user input
    input_string = input("Enter a string: ")

    # Convert string case
    uppercase_string, lowercase_string = convert_string_case(input_string)

    # Print results
    print("Uppercase:", uppercase_string)
    print("Lowercase:", lowercase_string)

if __name__ == "__main__":
    main()
```
