## Exercise
Create a function that finds the index of a given element in a list. Use the function to find the index of the element 'b' in the list ['a', 'b', 'c'].

## Reference Solution
```python
def find_index(lst, elem):
    """
    This function finds the index of a given element in a list.
    
    Parameters:
    lst (list): The list to search in.
    elem: The element to search for.
    
    Returns:
    int: The index of the element if found, -1 otherwise.
    """
    
    # Try to find the index of the element in the list
    try:
        # Use the index method of the list to find the index of the element
        index = lst.index(elem)
        
        # Return the index
        return index
    
    # If the element is not found, the index method will raise a ValueError
    except ValueError:
        # Return -1 to indicate that the element was not found
        return -1

# Use the function to find the index of 'b' in ['a', 'b', 'c']
print(find_index(['a', 'b', 'c'], 'b'))  # Output: 1
```
