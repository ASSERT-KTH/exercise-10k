## Exercise
Write a program that prints the numbers from 1 to n, where n is a user-defined input. Use a for loop to iterate through the numbers and print each one on a new line.

## Reference Solution
```python
# main
n = int(input("Enter a number: "))
for i in range(1, n+1):
    print(i)
```
