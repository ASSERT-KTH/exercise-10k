## Exercise
Create a function to check if a given number is a Harshad number or not

## Reference Solution
```python
def is_harshad_number(n: int) -> bool:
    """
    Checks if a given number is a Harshad number.

    Args:
    n (int): The number to check.

    Returns:
    bool: True if the number is a Harshad number, False otherwise.
    """
    # Calculate the sum of the digits of the number
    digit_sum = sum(int(digit) for digit in str(n))
    
    # Check if the number is divisible by the sum of its digits
    return n % digit_sum == 0

# Example usage
if __name__ == "__main__":
    numbers_to_check = [18, 19, 36]
    for num in numbers_to_check:
        print(f"{num} is a Harshad number: {is_harshad_number(num)}")
```
