## Exercise
Write a program that uses a for loop to print the numbers from 10 to 1.

## Reference Solution
```python
# Decrementing Numbers Program

def print_decrementing_numbers():
    """
    Prints the numbers from 10 to 1 using a for loop.
    """
    for i in range(10, 0, -1):
        print(i)

# Example usage:
if __name__ == "__main__":
    print("Numbers from 10 to 1:")
    print_decrementing_numbers()
```
