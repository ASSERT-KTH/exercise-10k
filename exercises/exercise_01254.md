## Exercise
Write a function that applies a given function to a list of arguments.

## Reference Solution
```python
def apply_function(func, args):
    """
    Applies a given function to a list of arguments.

    Args:
        func (function): The function to be applied.
        args (list): A list of arguments to be passed to the function.

    Returns:
        result: The result of applying the function to the arguments.
    """

    # First, we check if the input function is callable
    if not callable(func):
        raise TypeError("The input 'func' must be a callable function.")

    # Then, we check if the input arguments is a list
    if not isinstance(args, list):
        raise TypeError("The input 'args' must be a list of arguments.")

    # Now, we try to apply the function to the arguments
    try:
        # We use the built-in * operator to unpack the list of arguments
        # This will pass each argument in the list as a separate argument to the function
        result = func(*args)
        
        # Finally, we return the result of applying the function to the arguments
        return result
    
    # If any error occurs during the application of the function, we catch the exception and raise it
    except Exception as e:
        raise Exception(f"An error occurred while applying the function: {str(e)}")

# Example usage:
def add(a, b):
    return a + b

print(apply_function(add, [1, 2]))  # Outputs: 3
```
