## Exercise
Write a program to declare and use variables with proper naming conventions.

## Reference Solution
```python
# Declare variables
first_name = "John"  # string variable
last_name = "Doe"  # string variable
age = 30  # integer variable
height = 1.75  # float variable
is_employed = True  # boolean variable

# Use variables
def print_user_info():
    """Prints user information"""
    print(f"Name: {first_name} {last_name}")
    print(f"Age: {age}")
    print(f"Height: {height} meters")
    print(f"Employed: {is_employed}")

# Main function
def main():
    print_user_info()

# Run the main function
if __name__ == "__main__":
    main()
```
