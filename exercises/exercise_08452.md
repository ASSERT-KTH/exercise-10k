## Exercise
Generate a multiplication table for a given number

## Reference Solution
```python
def generate_multiplication_table(number, max_multiplier=10):
    """
    Generates a multiplication table for a given number.

    Args:
        number (int): The number for which to generate the multiplication table.
        max_multiplier (int): The maximum multiplier to include in the table. Defaults to 10.

    Returns:
        None
    """
    print(f"Multiplication Table for {number}:")
    for multiplier in range(1, max_multiplier + 1):
        product = number * multiplier
        print(f"{number} x {multiplier} = {product}")

# Example usage:
number = int(input("Enter a number: "))
generate_multiplication_table(number)
```
