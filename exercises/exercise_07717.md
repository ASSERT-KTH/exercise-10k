## Exercise
Segment a given string into substrings separated by a specified delimiter.

## Reference Solution
```python
def segment_string(input_str, delimiter):
    """
    Segment a given string into substrings separated by a specified delimiter.

    Args:
        input_str (str): The input string to be segmented.
        delimiter (str): The delimiter used to segment the string.

    Returns:
        list: A list of substrings separated by the delimiter.
    """
    return input_str.split(delimiter)

# Example usage
input_str = "apple,banana,orange"
delimiter = ","
result = segment_string(input_str, delimiter)
print(result)  # Output: ["apple", "banana", "orange"]
```
