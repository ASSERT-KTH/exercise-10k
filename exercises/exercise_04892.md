## Exercise
Use the get() method to access a key and return a default value if the key is not found.

## Reference Solution
```python
# Define a dictionary
my_dict = {"name": "John", "age": 30}

# Access a key that exists
print(my_dict.get("name"))  # Output: John

# Access a key that does not exist with a default value
print(my_dict.get("city", "New York"))  # Output: New York

# Access a key that does not exist without a default value
print(my_dict.get("country"))  # Output: None
```
