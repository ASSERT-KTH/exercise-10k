## Exercise
Write a program that takes an input and processes it using a loop.

## Reference Solution
```python
def process_input(input_str):
    """
    Processes the input string using a loop.
    """
    result = ""
    for char in input_str:
        # Convert each character to uppercase
        result += char.upper()
    return result

def main():
    # Get input from the user
    input_str = input("Enter a string: ")
    
    # Process the input using the loop
    result = process_input(input_str)
    
    # Print out the result
    print("Result:", result)

if __name__ == "__main__":
    main()
```
