## Exercise
Write a program that takes a list of strings and a delimiter as input and returns the joined string.

## Reference Solution
```python
def main(strings, delimiter):
    joined_string = delimiter.join(strings)
    return joined_string

# Test the function
print(main(["Hello", "World"], " "))
print(main(["Hello", "World"], ""))
```
