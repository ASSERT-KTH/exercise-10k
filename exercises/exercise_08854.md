## Exercise
Write a function to calculate the minimum of a list of numbers

## Reference Solution
```python
def calculate_minimum(numbers):
    """
    Calculate the minimum value in a list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        float: The minimum value in the list.
    """
    if not numbers:
        raise ValueError("List cannot be empty")

    return min(numbers)

# Example usage:
def main():
    numbers = [12, 45, 7, 23, 56, 89, 34]
    minimum = calculate_minimum(numbers)
    print(f"Minimum value: {minimum}")

if __name__ == "__main__":
    main()
```
