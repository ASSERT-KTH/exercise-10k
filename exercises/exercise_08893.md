## Exercise
Write a program that takes the radius and height of a cone as input and returns its surface area.

## Reference Solution
```python
import math

def calculate_cone_surface_area(radius, height):
    """
    Calculate the surface area of a cone.

    Args:
        radius (float): The radius of the cone.
        height (float): The height of the cone.

    Returns:
        float: The surface area of the cone.
    """
    if radius < 0 or height < 0:
        raise ValueError("Radius and height must be non-negative")

    slant_height = math.sqrt(radius**2 + height**2)
    surface_area = math.pi * radius**2 + math.pi * radius * slant_height
    return surface_area

def main():
    # Get user input
    radius = float(input("Enter the radius of the cone: "))
    height = float(input("Enter the height of the cone: "))

    # Calculate surface area
    try:
        surface_area = calculate_cone_surface_area(radius, height)
        print(f"The surface area of the cone is: {surface_area:.2f}")
    except ValueError as e:
        print(e)

if __name__ == "__main__":
    main()
```
