## Exercise
Write a program to declare and use constant variables.

## Reference Solution
```python
# Import the enum module for creating enumerations
from enum import Enum

# Define a class for constants using Enum
class Constants(Enum):
    """Class for storing constant values."""
    PI = 3.14159
    EULER_CONSTANT = 2.71828
    GRAVITY = 9.81

def main():
    # Access and print constant values
    print("Value of PI:", Constants.PI.value)
    print("Value of Euler's Constant:", Constants.EULER_CONSTANT.value)
    print("Value of Gravity:", Constants.GRAVITY.value)

    # Attempting to change a constant value will result in an error
    try:
        Constants.PI = 3.14
    except AttributeError:
        print("Error: Cannot modify a constant value.")

if __name__ == "__main__":
    main()
```
