## Exercise
Declare an enum with values representing the days of the week, then print the value of Monday.

## Reference Solution
```python
from enum import Enum

class DayOfWeek(Enum):
    """Enum representing the days of the week."""
    MONDAY = 1
    TUESDAY = 2
    WEDNESDAY = 3
    THURSDAY = 4
    FRIDAY = 5
    SATURDAY = 6
    SUNDAY = 7

def main():
    # Print the value of Monday
    print(DayOfWeek.MONDAY.value)

if __name__ == "__main__":
    main()
```
