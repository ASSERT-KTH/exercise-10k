## Exercise
Write a function that takes a list and a transformation function as input and returns a new list containing the transformed elements.

## Reference Solution
```python
def transform_list(input_list, transformation_function):
    """
    This function takes a list and a transformation function as input, 
    applies the transformation function to each element in the list, 
    and returns a new list containing the transformed elements.

    Args:
        input_list (list): The list of elements to be transformed.
        transformation_function (function): A function that takes one argument and returns a transformed value.

    Returns:
        list: A new list containing the transformed elements.
    """

    # Initialize an empty list to store the transformed elements
    transformed_list = []

    # Iterate over each element in the input list
    for element in input_list:
        # Apply the transformation function to the current element
        transformed_element = transformation_function(element)
        
        # Append the transformed element to the transformed list
        transformed_list.append(transformed_element)

    # Return the transformed list
    return transformed_list

# Example usage:
# Define a transformation function that squares a number
def square(x):
    return x ** 2

# Create a list of numbers
numbers = [1, 2, 3, 4, 5]

# Transform the list of numbers using the square function
squared_numbers = transform_list(numbers, square)

# Print the transformed list
print(squared_numbers)  # Output: [1, 4, 9, 16, 25]
```
