## Exercise
Update the value associated with a given key in a dictionary

## Reference Solution
```python
def update_dictionary_value(dictionary, key, new_value):
    """
    Updates the value associated with a given key in a dictionary.

    Args:
        dictionary (dict): The dictionary to update.
        key: The key to update.
        new_value: The new value to associate with the key.

    Returns:
        dict: The updated dictionary.
    """
    if key in dictionary:
        dictionary[key] = new_value
    else:
        print(f"Key '{key}' not found in dictionary.")
    return dictionary

# Example usage:
my_dict = {"name": "John", "age": 30}
updated_dict = update_dictionary_value(my_dict, "age", 31)
print(updated_dict)  # Output: {'name': 'John', 'age': 31}
```
