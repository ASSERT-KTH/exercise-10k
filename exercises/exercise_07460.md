## Exercise
Create a program that defines a set of functions to perform basic arithmetic operations like addition, subtraction, multiplication, and division. The program should take two numbers and an operation as input and return the result of the operation.

## Reference Solution
```python
# Define the main function
def main():
    print("Arithmetic Operations Program")
    print("-------------------------------")

    # Get user input
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))
    operation = input("Enter the operation (+, -, *, /): ")

    # Perform the operation
    if operation == "+":
        print(f"{num1} + {num2} = {add(num1, num2)}")
    elif operation == "-":
        print(f"{num1} - {num2} = {sub(num1, num2)}")
    elif operation == "*":
        print(f"{num1} * {num2} = {mul(num1, num2)}")
    elif operation == "/":
        print(f"{num1} / {num2} = {div(num1, num2)}")
    else:
        print("Invalid operation. Please enter +, -, *, or /.")

# Define the arithmetic operations functions
def add(x, y):
    return x + y

def sub(x, y):
    return x - y

def mul(x, y):
    return x * y

def div(x, y):
    if y == 0:
        print("Error: Division by zero is not allowed.")
        return None
    return x / y

# Call the main function
main()
```
