## Exercise
Given an array of integers, write a function to find the maximum sum of a contiguous subarray within the array. The function should take an array of integers as input and return the maximum sum.

## Reference Solution
```python
def maxSubArray(nums):
    if not nums:
        return 0

    max_sum = float('-inf')
    current_sum = 0

    for num in nums:
        current_sum = max(num, current_sum + num)
        max_sum = max(max_sum, current_sum)

    return max_sum

# Test case
nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
result = maxSubArray(nums)
print(result)  # Output: 6
```
