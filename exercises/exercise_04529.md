## Exercise
Write a program that takes an array as input and returns the minimum and maximum elements in the array.

## Reference Solution
```python
def find_min_max(array):
    """
    This function takes an array as input and returns the minimum and maximum elements in the array.

    Args:
        array (list): A list of numbers.

    Returns:
        tuple: A tuple containing the minimum and maximum elements in the array.

    Raises:
        ValueError: If the input array is empty.
    """

    # Check if the array is empty
    if not array:
        raise ValueError("Array cannot be empty")

    # Initialize minimum and maximum values with the first element of the array
    min_value = array[0]
    max_value = array[0]

    # Iterate over the array to find the minimum and maximum values
    for num in array:
        if num < min_value:
            min_value = num
        elif num > max_value:
            max_value = num

    # Return the minimum and maximum values as a tuple
    return min_value, max_value


# Example usage:
if __name__ == "__main__":
    array = [10, 20, 30, 40, 50]
    min_val, max_val = find_min_max(array)
    print("Minimum value:", min_val)
    print("Maximum value:", max_val)
```
