## Exercise
Write a program that uses a for loop to print each element in an array.

## Reference Solution
```python
def print_array_elements(array):
    """
    Prints each element in the given array.

    Args:
        array (list): The input array.
    """
    for element in array:
        print(element)

# Example usage:
if __name__ == "__main__":
    # Define an example array
    example_array = [1, 2, 3, 4, 5]

    # Print the array elements
    print("Array elements:")
    print_array_elements(example_array)
```
