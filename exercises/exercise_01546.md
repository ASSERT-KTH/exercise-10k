## Exercise
Given an array of integers, write a function to find the index of a specified element.

## Reference Solution
```python
def find_index(arr, target):
    """
    Find the index of a specified element in an array.

    Args:
        arr (list): The input array of integers.
        target (int): The element to find.

    Returns:
        int: The index of the target element if found, -1 otherwise.
    """
    for i, num in enumerate(arr):
        if num == target:
            return i
    return -1

# Example usage
arr = [1, 2, 3, 4, 5]
target = 3
index = find_index(arr, target)
if index != -1:
    print(f"Element {target} found at index {index}.")
else:
    print(f"Element {target} not found in the array.")
```
