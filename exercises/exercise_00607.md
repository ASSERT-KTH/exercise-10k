## Exercise
Write a program to raise a number to a given power and print the result.

## Reference Solution
```python
def calculate_power(base, exponent):
    """
    Calculate the result of raising the base to the exponent.

    Args:
        base (float): The base number.
        exponent (float): The power to which the base is raised.

    Returns:
        float: The result of the power operation.
    """
    result = base ** exponent
    return result

def main():
    # Get user input for base and exponent
    base = float(input("Enter the base number: "))
    exponent = float(input("Enter the exponent: "))

    # Calculate and print the result
    result = calculate_power(base, exponent)
    print(f"{base} raised to the power of {exponent} is: {result}")

if __name__ == "__main__":
    main()
```
