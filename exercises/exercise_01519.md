## Exercise
Write a program that generates all possible anagrams of a given string.

## Reference Solution
```python
import itertools

def generate_anagrams(s):
    # Generate all permutations of the input string
    perms = itertools.permutations(s)

    # Convert each permutation to a string and store in a set to remove duplicates
    anagrams = set("".join(p) for p in perms)

    return anagrams

def main():
    input_string = input("Enter a string: ")
    anagrams = generate_anagrams(input_string)

    print("Anagrams:")
    for anagram in anagrams:
        print(anagram)

if __name__ == "__main__":
    main()
```
