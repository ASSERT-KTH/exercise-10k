## Exercise
Write a function that takes two dictionaries as input and updates the first dictionary with the key-value pairs from the second dictionary

## Reference Solution
```python
def update_dict(dict1, dict2):
    """
    Updates the first dictionary with the key-value pairs from the second dictionary.

    Args:
        dict1 (dict): The dictionary to be updated.
        dict2 (dict): The dictionary with the new key-value pairs.

    Returns:
        dict: The updated dictionary.
    """
    dict1.update(dict2)
    return dict1

# Example usage
dict1 = {'a': 1, 'b': 2}
dict2 = {'b': 3, 'c': 4}

print("Before update:")
print("dict1:", dict1)
print("dict2:", dict2)

updated_dict = update_dict(dict1, dict2)

print("\nAfter update:")
print("updated_dict:", updated_dict)
```
