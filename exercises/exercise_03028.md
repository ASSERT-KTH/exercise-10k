## Exercise
Write a program that takes a string as input and returns the string compressed by removing repeated characters.

## Reference Solution
```python
def compress_string(input_str):
    """
    Compress a string by removing repeated characters.

    Args:
        input_str (str): The input string to be compressed.

    Returns:
        str: The compressed string.
    """
    if not input_str:
        return ""

    compressed_str = ""
    last_char = input_str[0]

    for char in input_str:
        if char != last_char:
            compressed_str += char
            last_char = char

    return compressed_str

# Example usage:
def main():
    input_str = "AAABBBCCCDDDD"
    print("Input String:", input_str)
    print("Compressed String:", compress_string(input_str))

if __name__ == "__main__":
    main()
```
