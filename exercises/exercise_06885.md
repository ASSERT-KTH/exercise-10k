## Exercise
Write a program that takes in a user's input and checks the data type of the input. If the input is an integer, it should print 'Integer'. If the input is a float, it should print 'Float'. If the input is a string, it should print 'String'.

## Reference Solution
```python
def check_data_type(user_input):
    try:
        # Try to convert the input to an integer
        int(user_input)
        print('Integer')
    except ValueError:
        try:
            # If it's not an integer, try to convert it to a float
            float(user_input)
            print('Float')
        except ValueError:
            # If it's neither an integer nor a float, it's a string
            print('String')

def main():
    user_input = input("Please enter your input: ")
    check_data_type(user_input)

if __name__ == "__main__":
    main()
```
