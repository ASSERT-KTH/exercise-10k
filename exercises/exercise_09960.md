## Exercise
Write a program that compares the data types of two variables and prints out the result.

## Reference Solution
```python
def compare_data_types(var1, var2):
    type_var1 = type(var1)
    type_var2 = type(var2)

    if type_var1 == type_var2:
        print(f"The data types of {var1} and {var2} are the same: {type_var1.__name__}")
    else:
        print(f"The data types of {var1} and {var2} are different: {type_var1.__name__} and {type_var2.__name__}")

# Test the function
var1 = 10
var2 = 20
compare_data_types(var1, var2)

var1 = "Hello"
var2 = 10
compare_data_types(var1, var2)

var1 = 10.5
var2 = 20.5
compare_data_types(var1, var2)
```
