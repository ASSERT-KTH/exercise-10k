## Exercise
Write a program that takes a string as input and returns the reversed string using conditional statements.

## Reference Solution
```python
def reverse_string(input_str):
    reversed_str = ""
    for char in input_str:
        if not reversed_str:
            reversed_str = char
        else:
            reversed_str = char + reversed_str
    return reversed_str

# Test the function
input_str = "Hello World"
print("Original String: ", input_str)
print("Reversed String: ", reverse_string(input_str))
```
