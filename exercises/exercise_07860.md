## Exercise
Create a program that finds the maximum of two numbers using if/else statements.

## Reference Solution
```python
def find_max(num1, num2):
    """
    Returns the maximum of two numbers.

    Args:
        num1 (int): The first number.
        num2 (int): The second number.

    Returns:
        int: The maximum of num1 and num2.
    """
    if num1 > num2:
        return num1
    else:
        return num2

def main():
    # Example usage
    num1 = 10
    num2 = 20
    max_num = find_max(num1, num2)
    print(f"The maximum of {num1} and {num2} is {max_num}")

if __name__ == "__main__":
    main()
```
