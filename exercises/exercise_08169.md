## Exercise
Create an array of 5 integers and filter out elements that do not meet a condition.

## Reference Solution
```python
# Import the necessary module
import numpy as np

# Create an array of 5 integers
numbers = np.array([1, 2, 3, 4, 5])

# Define the condition
condition = numbers > 3

# Filter out elements that do not meet the condition
filtered_numbers = numbers[condition]

# Print the original array and the filtered array
print("Original Array:", numbers)
print("Filtered Array:", filtered_numbers)
```
