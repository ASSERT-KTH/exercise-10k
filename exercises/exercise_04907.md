## Exercise
Write a program that takes a person's age as input and prints whether they are eligible to vote (18 or older) or not.

## Reference Solution
```python
def check_voting_eligibility():
    age = int(input("Enter your age: "))

    if age >= 18:
        print("You are eligible to vote.")
    else:
        print("You are not eligible to vote.")

# Call the function
check_voting_eligibility()
```
