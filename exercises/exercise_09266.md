## Exercise
Write a program that takes an array of integers as input and returns the maximum value in the array. The program should iterate over the array, compare each element with the current maximum, and update the maximum if a larger value is found.

## Reference Solution
```python
def findMax(arr):
    # Initialize max_val with the first element of the array
    max_val = arr[0]

    # Iterate over the array starting from the second element (index 1)
    for i in range(1, len(arr)):
        # Compare the current element with the current max_val
        if arr[i] > max_val:
            # If the current element is greater, update max_val
            max_val = arr[i]

    # Return the maximum value found
    return max_val

# Example usage:
arr = [1, 2, 3, 4, 5]
max_value = findMax(arr)
print("Maximum value in the array:", max_value)
```
