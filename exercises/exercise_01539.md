## Exercise
Write a program that sorts a list of strings in alphabetical order.

## Reference Solution
```python
def sort_strings(strings):
    """
    Sorts a list of strings in alphabetical order.

    Args:
        strings (list): A list of strings to be sorted.

    Returns:
        list: The sorted list of strings.
    """
    return sorted(strings)

def main():
    # Example usage
    strings = ["orange", "apple", "banana", "grape"]
    sorted_strings = sort_strings(strings)
    print("Sorted strings:")
    for string in sorted_strings:
        print(string)

if __name__ == "__main__":
    main()
```
