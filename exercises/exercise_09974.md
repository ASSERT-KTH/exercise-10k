## Exercise
Write a program that encodes a string into bytes using a specific encoding.

## Reference Solution
```python
def encode_string(input_string, encoding):
    try:
        encoded_bytes = input_string.encode(encoding)
        return encoded_bytes
    except LookupError:
        print(f"Error: Unknown encoding '{encoding}'")
        return None
    except UnicodeError as e:
        print(f"Error: {e}")
        return None

# Example usage:
input_string = "Hello, World!"
encoding = "utf-8"
encoded_bytes = encode_string(input_string, encoding)

if encoded_bytes is not None:
    print(f"Encoded bytes: {encoded_bytes}")
```
