## Exercise
Write a program that prints out a greeting based on the current time of day using if statements

## Reference Solution
```python
import datetime

def get_greeting():
    """
    Returns a greeting based on the current time of day.
    """
    current_time = datetime.datetime.now()
    current_hour = current_time.hour

    if 0 <= current_hour < 12:
        return "Good morning!"
    elif 12 <= current_hour < 17:
        return "Good afternoon!"
    else:
        return "Good evening!"

def main():
    print(get_greeting())

if __name__ == "__main__":
    main()
```
