## Exercise
Write a program that takes two lists as input and returns a new list containing all elements from both lists

## Reference Solution
```python
def combine_lists(list1, list2):
    """
    Combines two lists into a new list.

    Args:
        list1 (list): The first list to combine.
        list2 (list): The second list to combine.

    Returns:
        list: A new list containing all elements from both input lists.
    """
    return list1 + list2

# Example usage:
list1 = [1, 2, 3]
list2 = [4, 5, 6]

combined_list = combine_lists(list1, list2)
print(combined_list)  # Output: [1, 2, 3, 4, 5, 6]
```
