## Exercise
Write a program that takes two numbers as input and returns the minimum of the two.

## Reference Solution
```python
def find_min(a, b):
    """
    Returns the minimum of two numbers.

    Args:
        a (int): The first number.
        b (int): The second number.

    Returns:
        int: The minimum of a and b.
    """
    if a < b:
        return a
    else:
        return b

# Example usage:
num1 = int(input("Enter the first number: "))
num2 = int(input("Enter the second number: "))

min_num = find_min(num1, num2)
print("The minimum of the two numbers is:", min_num)
```
