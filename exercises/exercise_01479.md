## Exercise
Write a program that declares and initializes an enumeration variable, then prints its value.

## Reference Solution
```python
# Define an enumeration class
from enum import Enum

class Color(Enum):
    RED = 1
    GREEN = 2
    BLUE = 3

# Declare and initialize an enumeration variable
color = Color.GREEN

# Print the value of the enumeration variable
print(color.value)
```
