## Exercise
Create two dictionaries with 3 key-value pairs each and update one dictionary with the other.

## Reference Solution
```python
# Create the first dictionary
dict1 = {
    "key1": "value1",
    "key2": "value2",
    "key3": "value3"
}

# Create the second dictionary
dict2 = {
    "key2": "new_value2",
    "key3": "new_value3",
    "key4": "value4"
}

# Print the original dictionaries
print("Original dict1:")
print(dict1)
print("\nOriginal dict2:")
print(dict2)

# Update dict1 with dict2
dict1.update(dict2)

# Print the updated dictionary
print("\nUpdated dict1:")
print(dict1)
```
