## Exercise
Write a program that reads a character from the user and prints it back out.

## Reference Solution
```python
def read_and_print_character():
    """
    Reads a character from the user and prints it back out.
    """
    user_input = input("Please enter a character: ")
    
    # Check if the input is a single character
    if len(user_input) != 1:
        print("Error: Please enter only one character.")
    else:
        print("You entered:", user_input)

def main():
    read_and_print_character()

if __name__ == "__main__":
    main()
```
