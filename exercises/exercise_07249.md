## Exercise
Write a program that takes a string input from the user, checks if it's a palindrome, and then reverses the string. If the string is a palindrome, the program should print 'Palindrome' before printing the reversed string. Otherwise, it should only print the reversed string.

## Reference Solution
```python
input_str = input("Enter a string: ")
def check_p(str):
    if str == str[::-1]:
        print("Palindrome")
    return str[::-1]

print(check_p(input_str))
```
