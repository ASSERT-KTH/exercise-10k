## Exercise
Write a program that checks whether a given month number (1-12) falls within a specific semester (e.g., spring: January-May) using conditional statements.

## Reference Solution
```python
#include <iostream>
using namespace std;

int main() {
    int month;
    cin >> month;

    if (month >= 1 && month <= 5) {
        cout << "Spring" << endl;
    } else if (month >= 6 && month <= 8) {
        cout << "Summer" << endl;
    } else if (month >= 9 && month <= 12) {
        cout << "Autumn" << endl;
    } else {
        cout << "Invalid month" << endl;
    }

    return 0;
}
```
