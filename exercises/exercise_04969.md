## Exercise
Write a function that checks if a string contains only digits

## Reference Solution
```python
def contains_only_digits(input_string: str) -> bool:
    """
    Checks if a string contains only digits.

    Args:
        input_string (str): The input string to check.

    Returns:
        bool: True if the string contains only digits, False otherwise.
    """
    return input_string.isdigit()

# Example usage:
if __name__ == "__main__":
    print(contains_only_digits("12345"))  # True
    print(contains_only_digits("123abc"))  # False
    print(contains_only_digits(""))  # False
```
