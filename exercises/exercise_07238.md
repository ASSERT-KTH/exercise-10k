## Exercise
Write a program that prints the numbers from 1 to 10 using a for loop, then prints the numbers from 10 to 1 using a while loop.

## Reference Solution
```python
#include <iostream>
using namespace std;

int main() {
    // Using a for loop to print numbers from 1 to 10
    for (int i = 1; i <= 10; i++) {
        cout << i << endl;
    }

    // Using a while loop to print numbers from 10 to 1
    int j = 10;
    while (j >= 1) {
        cout << j << endl;
        j--;
    }

    return 0;
}
```
