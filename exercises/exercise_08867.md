## Exercise
Create a function to check if a given number is an abundant number or not

## Reference Solution
```python
def is_abundant_number(n):
    """
    Checks if a given number is an abundant number.

    Args:
        n (int): The number to check.

    Returns:
        bool: True if the number is abundant, False otherwise.
    """
    if n < 1:
        raise ValueError("Input must be a positive integer.")

    # Calculate the sum of proper divisors
    divisor_sum = sum(i for i in range(1, n) if n % i == 0)

    # Check if the sum of proper divisors is greater than the number
    return divisor_sum > n


# Example usage
if __name__ == "__main__":
    numbers_to_check = [12, 15, 20]
    for num in numbers_to_check:
        result = is_abundant_number(num)
        print(f"{num} is abundant: {result}")
```
