## Exercise
Create a program that takes in a user's input and determines the data type of the input. The input can be an integer, float, or string. The program should then print out the data type of the input.

## Reference Solution
```python
def get_data_type():
    """Gets the user's input and determines its data type."""
    user_input = input("Please enter a value: ")
    
    try:
        # Attempt to convert the input to an integer
        int(user_input)
        data_type = "Integer"
    except ValueError:
        try:
            # Attempt to convert the input to a float
            float(user_input)
            data_type = "Float"
        except ValueError:
            # If the input cannot be converted to an integer or float, it's a string
            data_type = "String"
    
    print(f"The data type of the input is: {data_type}")

def main():
    get_data_type()

if __name__ == "__main__":
    main()
```
