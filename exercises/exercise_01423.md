## Exercise
Create a function that takes an array and a predicate function as input and returns a new array containing only the elements for which the predicate function returns True.

## Reference Solution
```python
def filter_array(array, predicate):
    """
    This function filters an array based on a given predicate function.
    
    Args:
        array (list): The input array to be filtered.
        predicate (function): A function that takes one argument and returns a boolean value.
    
    Returns:
        list: A new array containing only the elements for which the predicate function returns True.
    """

    # Initialize an empty list to store the filtered elements
    filtered_array = []

    # Iterate over each element in the input array
    for element in array:
        # Apply the predicate function to the current element
        if predicate(element):
            # If the predicate function returns True, append the element to the filtered array
            filtered_array.append(element)

    # Return the filtered array
    return filtered_array

# Example usage:
# Define a predicate function that checks if a number is even
def is_even(num):
    return num % 2 == 0

# Create an array of numbers
numbers = [1, 2, 3, 4, 5, 6]

# Filter the array using the predicate function
even_numbers = filter_array(numbers, is_even)

# Print the filtered array
print(even_numbers)  # Output: [2, 4, 6]
```
