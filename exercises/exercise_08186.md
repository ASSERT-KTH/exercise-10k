## Exercise
Write a program that prints a table of numbers using a nested for loop.

## Reference Solution
```python
def print_table(rows, cols):
    """
    Prints a table of numbers with the specified number of rows and columns.
    
    Args:
        rows (int): The number of rows in the table.
        cols (int): The number of columns in the table.
    """
    for i in range(1, rows + 1):
        for j in range(1, cols + 1):
            # Print the product of the row and column numbers
            print(f"{i * j:4d}", end='')
        print()

# Example usage
if __name__ == "__main__":
    rows = 5
    cols = 5
    print(f"Table of numbers ({rows}x{cols}):")
    print_table(rows, cols)
```
