## Exercise
Create a function to find the first duplicate in a list of numbers.

## Reference Solution
```python
def find_first_duplicate(nums):
    """
    Find the first duplicate in a list of numbers.

    Args:
        nums (list): A list of numbers.

    Returns:
        int or None: The first duplicate number, or None if no duplicates are found.
    """
    seen = set()
    for num in nums:
        if num in seen:
            return num
        seen.add(num)
    return None

# Example usage:
if __name__ == "__main__":
    numbers = [2, 1, 3, 5, 3, 2]
    duplicate = find_first_duplicate(numbers)
    if duplicate is not None:
        print(f"The first duplicate is: {duplicate}")
    else:
        print("No duplicates found.")
```
