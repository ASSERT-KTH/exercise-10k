## Exercise
Write a program that uses a for loop to implement a Caesar cipher, which shifts each letter in a given string by a specified number of positions.

## Reference Solution
```python
s = "Hello, world!"
shift = 3

for i in range(len(s)):
    if s[i].isalpha():
        ascii_offset = 65 if s[i].isupper() else 97
        print(chr((ord(s[i]) - ascii_offset + shift) % 26 + ascii_offset), end='')
    else:
        print(s[i], end='')
```
