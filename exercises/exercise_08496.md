## Exercise
Write a program that inverts a given dictionary. The inversion process means swapping the keys with the values. If the original dictionary has non-unique values, the inverted dictionary should have lists of keys as values. For example, if the input dictionary is {a: 1, b: 2, c: 1}, the output should be {1: ['a', 'c'], 2: ['b']}. Handle the case where the input dictionary is empty.

## Reference Solution
```python
def invert_dictionary(input_dict):
    """
    Inverts a dictionary by swapping keys with values.
    If the original dictionary has non-unique values, the inverted dictionary will have lists of keys as values.

    Args:
        input_dict (dict): The dictionary to be inverted.

    Returns:
        dict: The inverted dictionary.
    """
    inverted_dict = {}
    for key, value in input_dict.items():
        if value in inverted_dict:
            # If the value already exists in the inverted dictionary, append the key to its list
            inverted_dict[value].append(key)
        else:
            # If the value does not exist in the inverted dictionary, add it with the key as a list
            inverted_dict[value] = [key]
    return inverted_dict

def main():
    # Example usage
    input_dict = {'a': 1, 'b': 2, 'c': 1}
    print("Input Dictionary:", input_dict)
    inverted_dict = invert_dictionary(input_dict)
    print("Inverted Dictionary:", inverted_dict)

    # Handle the case where the input dictionary is empty
    empty_dict = {}
    print("\nInput Dictionary:", empty_dict)
    inverted_empty_dict = invert_dictionary(empty_dict)
    print("Inverted Dictionary:", inverted_empty_dict)

if __name__ == "__main__":
    main()
```
