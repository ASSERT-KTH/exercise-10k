## Exercise
Write a program that checks if a given variable is a complex number.

## Reference Solution
```python
def is_complex_number(var):
    """
    Checks if a given variable is a complex number.
    
    Args:
        var: The variable to check.
    
    Returns:
        bool: True if the variable is a complex number, False otherwise.
    """
    return isinstance(var, complex)

# Example usage:
print(is_complex_number(5 + 3j))  # Output: True
print(is_complex_number(5))      # Output: False
print(is_complex_number("5+3j")) # Output: False (because it's a string, not a complex number)
```

def is_complex_string(s):
    """
    Checks if a given string represents a complex number.
    
    Args:
        s (str): The string to check.
    
    Returns:
        bool: True if the string represents a complex number, False otherwise.
    """
    try:
        complex(s)
        return True
    except ValueError:
        return False

# Example usage:
print(is_complex_string("5+3j"))  # Output: True
print(is_complex_string("5"))     # Output: True (because "5" can be interpreted as a complex number with zero imaginary part)
print(is_complex_string("abc"))   # Output: False
