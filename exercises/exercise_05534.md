## Exercise
Declare a date and time variable, assign a value to it, and then print its value.

## Reference Solution
```python
from datetime import datetime

def main():
    # Declare a date and time variable
    date_time_variable = None
    
    # Assign a value to the date and time variable
    date_time_variable = datetime(2024, 9, 16, 10, 30, 0)
    
    # Print the value of the date and time variable
    print("Date and Time:", date_time_variable)

if __name__ == "__main__":
    main()
```
