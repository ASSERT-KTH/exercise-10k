## Exercise
Write a function that takes a list as input and returns the minimum element.

## Reference Solution
```python
def find_minimum_element(input_list):
    """
    Returns the minimum element in the input list.

    Args:
        input_list (list): A list of elements.

    Returns:
        The minimum element in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not input_list:
        raise ValueError("Input list is empty")

    return min(input_list)

# Example usage
def main():
    numbers = [12, 45, 7, 23, 56, 89, 34]
    print("Minimum element:", find_minimum_element(numbers))

if __name__ == "__main__":
    main()
```
