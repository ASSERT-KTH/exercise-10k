## Exercise
Write a function to convert a string to uppercase.

## Reference Solution
```python
def convert_to_uppercase(input_string: str) -> str:
    """
    Converts a string to uppercase.

    Args:
        input_string (str): The string to be converted.

    Returns:
        str: The input string in uppercase.
    """
    return input_string.upper()

# Example usage:
if __name__ == "__main__":
    input_str = "hello world"
    print(convert_to_uppercase(input_str))  # Output: "HELLO WORLD"
```
