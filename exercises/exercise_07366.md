## Exercise
Create octal literals and print their values.

## Reference Solution
```python
# Create octal literals
octal_literal_1 = 0o10
octal_literal_2 = 0o20
octal_literal_3 = 0o30

# Print the values of the octal literals
print("Octal Literal 1:", octal_literal_1)
print("Octal Literal 2:", octal_literal_2)
print("Octal Literal 3:", octal_literal_3)

# Convert octal literals to decimal and hexadecimal
print("\nOctal Literal 1 in Decimal:", int(str(octal_literal_1), 8))
print("Octal Literal 1 in Hexadecimal:", hex(int(str(octal_literal_1), 8)))

print("Octal Literal 2 in Decimal:", int(str(octal_literal_2), 8))
print("Octal Literal 2 in Hexadecimal:", hex(int(str(octal_literal_2), 8)))

print("Octal Literal 3 in Decimal:", int(str(octal_literal_3), 8))
print("Octal Literal 3 in Hexadecimal:", hex(int(str(octal_literal_3), 8)))
```
