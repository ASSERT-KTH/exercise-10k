## Exercise
Create a function that calculates the maximum of two lists.

## Reference Solution
```python
def max_of_two_lists(list1, list2):
    """
    Calculate the maximum of two lists.

    Args:
        list1 (list): The first list of numbers.
        list2 (list): The second list of numbers.

    Returns:
        list: A list containing the maximum values from the input lists.
    """
    # Check if both lists are not empty
    if not list1 or not list2:
        raise ValueError("Both lists must be non-empty")

    # Calculate the maximum of each list
    max_list1 = max(list1)
    max_list2 = max(list2)

    # Return the maximum values
    return [max_list1, max_list2]

def main():
    # Example usage
    list1 = [10, 20, 30]
    list2 = [40, 50, 60]
    result = max_of_two_lists(list1, list2)
    print("Maximum of list1:", result[0])
    print("Maximum of list2:", result[1])

if __name__ == "__main__":
    main()
```
