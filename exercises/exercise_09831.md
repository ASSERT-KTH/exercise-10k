## Exercise
Write a program that takes an array of integers as input, reverses the order of its elements, and then sorts the reversed array in ascending order. The program should output the modified array.

## Reference Solution
```python
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;
    int arr[n];
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }
    reverse(arr, arr + n);
    sort(arr, arr + n);
    for (int i = 0; i < n; i++) {
        cout << arr[i] << endl;
    }
    return 0;
}
```
