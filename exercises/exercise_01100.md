## Exercise
Write a program that generates a random number between two given numbers.

## Reference Solution
```python
import random

def generate_random_number(min_num, max_num):
    """
    Generates a random number between min_num and max_num (inclusive).
    
    Args:
    min_num (int): The minimum number in the range.
    max_num (int): The maximum number in the range.
    
    Returns:
    int: A random number between min_num and max_num.
    """
    return random.randint(min_num, max_num)

# Example usage:
min_num = int(input("Enter the minimum number: "))
max_num = int(input("Enter the maximum number: "))

random_num = generate_random_number(min_num, max_num)
print("Random number between {} and {}: {}".format(min_num, max_num, random_num))
```
