## Exercise
Write a function that takes a string, old substring, and new substring as input and returns the string with the old substring replaced by the new substring.

## Reference Solution
```python
def replace_substring(input_string: str, old_substring: str, new_substring: str) -> str:
    """
    Replaces all occurrences of old_substring with new_substring in input_string.

    Args:
        input_string (str): The string in which to replace substrings.
        old_substring (str): The substring to be replaced.
        new_substring (str): The substring to replace with.

    Returns:
        str: The input string with all occurrences of old_substring replaced by new_substring.
    """
    return input_string.replace(old_substring, new_substring)

# Example usage
if __name__ == "__main__":
    original_string = "Hello, world! Hello again!"
    old_sub = "Hello"
    new_sub = "Hi"
    print(f"Original string: {original_string}")
    print(f"Replacing '{old_sub}' with '{new_sub}':")
    print(replace_substring(original_string, old_sub, new_sub))
```
