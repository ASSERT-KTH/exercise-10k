## Exercise
Write a function to check if a given string contains all unique characters

## Reference Solution
```python
def has_unique_chars(input_string):
    """
    Checks if a given string contains all unique characters.

    Args:
        input_string (str): The input string to be checked.

    Returns:
        bool: True if the string contains all unique characters, False otherwise.
    """
    return len(input_string) == len(set(input_string))

# Example usage
if __name__ == "__main__":
    print(has_unique_chars("abcdefg"))  # True
    print(has_unique_chars("aabbcc"))   # False
```
