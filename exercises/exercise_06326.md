## Exercise
Write a function that takes a list, an element, and an index as input, inserts the element at the specified index, and returns the resulting list.

## Reference Solution
```python
def insert_element(lst, element, index):
    """
    Inserts an element at a specified index in a list and returns the resulting list.

    Args:
        lst (list): The input list.
        element: The element to be inserted.
        index (int): The index at which the element is to be inserted.

    Returns:
        list: The resulting list after insertion.
    """

    # Check if the index is within the bounds of the list
    if index < 0 or index > len(lst):
        raise IndexError("Index out of range")

    # Create a copy of the original list to avoid modifying it directly
    new_lst = lst.copy()

    # Use list slicing to split the list into two parts at the specified index
    # The part before the index will be the first slice, and the part after the index will be the second slice
    first_part = new_lst[:index]
    second_part = new_lst[index:]

    # Combine the first slice, the element to be inserted, and the second slice to form the new list
    new_lst = first_part + [element] + second_part

    # Return the new list
    return new_lst
```
