## Exercise
Write a program to slice a list and extract a subset of elements.

## Reference Solution
```python
def slice_list(input_list, start, end):
    """
    Slice a list and extract a subset of elements.

    Args:
        input_list (list): The input list to be sliced.
        start (int): The starting index of the slice.
        end (int): The ending index of the slice.

    Returns:
        list: The sliced list.
    """
    return input_list[start:end]

def main():
    # Example usage
    input_list = [1, 2, 3, 4, 5, 6, 7, 8, 9]
    start = 2
    end = 6

    sliced_list = slice_list(input_list, start, end)
    print("Original List:", input_list)
    print("Sliced List:", sliced_list)

if __name__ == "__main__":
    main()
```
