## Exercise
Write a program that finds the index of a target element in a list using a for loop.

## Reference Solution
```python
def find_index(lst, target):
    """
    Find the index of a target element in a list.

    Args:
        lst (list): The list to search.
        target: The target element to find.

    Returns:
        int: The index of the target element if found, -1 otherwise.
    """
    for i, element in enumerate(lst):
        if element == target:
            return i
    return -1

# Example usage:
numbers = [4, 2, 9, 6, 5, 1, 8, 3, 7]
target_num = 6

index = find_index(numbers, target_num)

if index != -1:
    print(f"Target {target_num} found at index {index}.")
else:
    print(f"Target {target_num} not found in the list.")
```
