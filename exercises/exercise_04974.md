## Exercise
Write a function that removes leading and trailing whitespace from a string

## Reference Solution
```python
def remove_whitespace(input_string: str) -> str:
    """
    Removes leading and trailing whitespace from a string.

    Args:
        input_string (str): The input string to be processed.

    Returns:
        str: The input string with leading and trailing whitespace removed.
    """
    return input_string.strip()

# Example usage
if __name__ == "__main__":
    original_string = "   Hello, World!   "
    print(f"Original String: '{original_string}'")
    cleaned_string = remove_whitespace(original_string)
    print(f"Cleaned String: '{cleaned_string}'")
```
