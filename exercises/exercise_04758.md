## Exercise
Write a program that takes two numbers as input and returns the minimum of the two

## Reference Solution
```python
def find_minimum(num1, num2):
    """
    Returns the minimum of two numbers.

    Args:
        num1 (int or float): The first number.
        num2 (int or float): The second number.

    Returns:
        int or float: The minimum of the two numbers.
    """
    return min(num1, num2)

def main():
    # Get user input
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))

    # Find and print the minimum
    minimum = find_minimum(num1, num2)
    print(f"The minimum of {num1} and {num2} is {minimum}")

if __name__ == "__main__":
    main()
```
