## Exercise
Write a function that takes a string as input and returns True if the string contains only uppercase characters, False otherwise.

## Reference Solution
```python
def check_uppercase(input_string):
    """
    This function checks if a given string contains only uppercase characters.

    Args:
        input_string (str): The input string to be checked.

    Returns:
        bool: True if the string contains only uppercase characters, False otherwise.
    """

    # First, we need to check if the input is a string. If it's not, we raise a TypeError.
    if not isinstance(input_string, str):
        raise TypeError("Input must be a string.")

    # We use the isupper() method in Python, which returns True if all cased characters in the string are uppercase and there is at least one cased character, otherwise False.
    # This method considers non-alphabet characters as caseless, so they don't affect the result.
    return input_string.isupper()
```
