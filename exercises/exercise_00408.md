## Exercise
Write a program to find the common elements in two arrays.

## Reference Solution
```python
def common_elements(arr1, arr2):
    """
    Returns a list of common elements in two arrays.

    Args:
        arr1 (list): The first array.
        arr2 (list): The second array.

    Returns:
        list: A list of common elements.
    """
    return list(set(arr1) & set(arr2))

# Example usage
if __name__ == "__main__":
    arr1 = [1, 2, 3, 4, 5]
    arr2 = [4, 5, 6, 7, 8]
    common = common_elements(arr1, arr2)
    print("Common elements:", common)
```
