## Exercise
Write a program that takes a string from the user, splits it into words, and prints out the words one per line.

## Reference Solution
```python
def print_words_per_line(input_string):
    """
    This function takes a string, splits it into words, and prints each word on a new line.

    Args:
        input_string (str): The string to be split into words.
    """
    # Split the string into words
    words = input_string.split()

    # Print each word on a new line
    for word in words:
        print(word)


def main():
    # Get the string from the user
    user_string = input("Please enter a string: ")

    # Call the function to print words per line
    print_words_per_line(user_string)


if __name__ == "__main__":
    main()
```
