## Exercise
Write a program that checks if a given number is even or odd using if/else conditional statements.

## Reference Solution
```python
def check_even_odd(num):
    if num % 2 == 0:
        print(num, "is even.")
    else:
        print(num, "is odd.")

# Test the function
check_even_odd(10)
check_even_odd(11)
```
