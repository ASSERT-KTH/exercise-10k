## Exercise
Write a program that takes an array and a target value as input, and returns the index of the target value in the array. If the target value is not found, return -1.

## Reference Solution
```python
def findIndex(arr, target):
    """
    This function finds the index of a target value in an array.
    
    Parameters:
    arr (list): The array to search in.
    target: The value to search for.
    
    Returns:
    int: The index of the target value if found, -1 otherwise.
    """
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

# Example usage:
arr = [1, 2, 3, 4, 5]
target = 3
index = findIndex(arr, target)
print(f"The index of {target} in the array is: {index}")
```
