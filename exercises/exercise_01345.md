## Exercise
Write a program to manipulate a given string by converting it to uppercase, lowercase, and reversing it

## Reference Solution
```python
def string_manipulation(input_string):
    """
    This function takes a string as input and returns its uppercase, lowercase, and reversed versions.
    
    Parameters:
    input_string (str): The string to be manipulated.
    
    Returns:
    tuple: A tuple containing the uppercase, lowercase, and reversed versions of the input string.
    """
    uppercase_string = input_string.upper()
    lowercase_string = input_string.lower()
    reversed_string = input_string[::-1]
    
    return uppercase_string, lowercase_string, reversed_string

def main():
    input_string = input("Enter a string: ")
    uppercase, lowercase, reversed_string = string_manipulation(input_string)
    
    print(f"Uppercase: {uppercase}")
    print(f"Lowercase: {lowercase}")
    print(f"Reversed: {reversed_string}")

if __name__ == "__main__":
    main()
```
