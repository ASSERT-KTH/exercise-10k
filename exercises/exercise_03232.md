## Exercise
Write a program to find the first missing positive integer in an array.

## Reference Solution
```python
def first_missing_positive(nums):
    """
    Finds the first missing positive integer in an array.

    Args:
        nums (list): A list of integers.

    Returns:
        int: The first missing positive integer.
    """
    if not nums:
        return 1

    # Remove non-positive numbers and duplicates
    nums = list(set([num for num in nums if num > 0]))

    # Sort the array
    nums.sort()

    # Check if 1 is missing
    if nums[0] > 1:
        return 1

    # Iterate over the array to find the first missing positive integer
    for i in range(len(nums) - 1):
        if nums[i + 1] - nums[i] > 1:
            return nums[i] + 1

    # If no missing integer is found, return the next integer after the last one
    return nums[-1] + 1


# Example usage
if __name__ == "__main__":
    print(first_missing_positive([3, 4, -1, 1]))  # Output: 2
    print(first_missing_positive([1, 2, 0]))  # Output: 3
    print(first_missing_positive([-1, -2, 0]))  # Output: 1
```
