## Exercise
Write a program that takes a list and a reducing function as input, and returns the result of applying the reducing function to all elements of the list.

## Reference Solution
```python
def reduce_list(list_1, func):
    if len(list_1) < 2:
        return list_1
    result = list_1[0]
    for i in range(1, len(list_1)):
        result = func(result, list_1[i])
    return result
```
