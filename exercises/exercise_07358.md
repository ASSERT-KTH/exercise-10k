## Exercise
Understand how type inference works by declaring variables without explicit type annotations and printing their types.

## Reference Solution
```python
# Declare variables without explicit type annotations
var_int = 10
var_float = 10.5
var_str = "Hello, World!"
var_bool = True
var_list = [1, 2, 3]
var_dict = {"name": "John", "age": 30}
var_tuple = (1, 2, 3)

# Print the types of the variables
print(f"Type of var_int: {type(var_int)}")
print(f"Type of var_float: {type(var_float)}")
print(f"Type of var_str: {type(var_str)}")
print(f"Type of var_bool: {type(var_bool)}")
print(f"Type of var_list: {type(var_list)}")
print(f"Type of var_dict: {type(var_dict)}")
print(f"Type of var_tuple: {type(var_tuple)}")
```
