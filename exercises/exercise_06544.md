## Exercise
Create a function that compresses a given string by removing duplicate characters.

## Reference Solution
```python
def compress_string(input_str):
    """
    Compresses a given string by removing duplicate characters.

    Args:
        input_str (str): The input string to be compressed.

    Returns:
        str: The compressed string with no duplicate characters.
    """
    if not input_str:
        return ""

    seen = set()
    compressed = []

    for char in input_str:
        if char not in seen:
            seen.add(char)
            compressed.append(char)

    return "".join(compressed)

# Example usage:
def main():
    input_str = "Hello, World!"
    compressed_str = compress_string(input_str)
    print(f"Input String: {input_str}")
    print(f"Compressed String: {compressed_str}")

if __name__ == "__main__":
    main()
```
