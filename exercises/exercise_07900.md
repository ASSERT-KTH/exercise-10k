## Exercise
Create a function that takes an array and an index as input, and returns the element at the specified index. If the index is out of range, return a message indicating that the index is invalid.

## Reference Solution
```python
def get_element_at_index(array, index):
    """
    Returns the element at the specified index in the given array.
    
    If the index is out of range, returns a message indicating that the index is invalid.

    Parameters:
    array (list): The input array.
    index (int): The index of the element to be retrieved.

    Returns:
    any: The element at the specified index if it exists, otherwise a message indicating that the index is invalid.
    """

    # First, we need to check if the index is within the valid range for the given array
    # We use the len() function to get the number of elements in the array
    # The valid range of indices for an array is from 0 to len(array) - 1
    # So, we check if the index is greater than or equal to 0 and less than the length of the array
    if index < 0 or index >= len(array):
        
        # If the index is out of range, we return a message indicating that the index is invalid
        return "Invalid index. Index is out of range."
    
    # If the index is within the valid range, we return the element at the specified index
    # We use the indexing operator [] to access the element at the specified index
    # This will return the element at the specified index if it exists
    return array[index]
```
