## Exercise
Manipulate a string by concatenating, uppercasing, and lowercasing it, then print out the results.

## Reference Solution
```python
def manipulate_string():
    # Original string
    original_string = "Hello, World!"

    # Concatenating strings
    concatenated_string = original_string + " This is a concatenated string."
    print("Concatenated String:")
    print(concatenated_string)

    # Uppercasing a string
    uppercased_string = original_string.upper()
    print("\nUpper Cased String:")
    print(uppercased_string)

    # Lowercasing a string
    lowercased_string = original_string.lower()
    print("\nLower Cased String:")
    print(lowercased_string)

# Example usage
if __name__ == "__main__":
    manipulate_string()
```
