## Exercise
Create a function that takes another function as an argument and applies it to a list of numbers.

## Reference Solution
```python
def apply_to_list(func, num_list):
    """
    Applies a given function to each number in a list.

    Args:
    - func (function): The function to apply to the list.
    - num_list (list): A list of numbers.

    Returns:
    - list: A list of results after applying the function to each number.
    """
    return [func(num) for num in num_list]

# Example usage:
def square(x):
    """Returns the square of a number."""
    return x ** 2

numbers = [1, 2, 3, 4, 5]
result = apply_to_list(square, numbers)
print(result)  # Output: [1, 4, 9, 16, 25]
```
