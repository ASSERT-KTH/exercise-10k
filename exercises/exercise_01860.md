## Exercise
Write a program that takes a list of integers and a key function as input, and returns a dictionary containing the elements from the input list, grouped by the key function.

## Reference Solution
```python
from itertools import groupby
from operator import itemgetter

def group_by_key(lst, key):
    return dict((k, list(g)) for k, g in groupby(sorted(lst, key=itemgetter(0)), key=key))

# Example usage:
numbers = [1, 2, 3, 4, 5, 6]
result = group_by_key(numbers, lambda x: x % 2)
print(result)  # Output: {0: [2, 4, 6], 1: [1, 3, 5]}
```
