## Exercise
Create a function to find the intersection of two dictionaries and return a new dictionary containing only the key-value pairs that exist in both dictionaries.

## Reference Solution
```python
def intersect_dictionaries(dict1, dict2):
    """
    This function finds the intersection of two dictionaries and returns a new dictionary containing only the key-value pairs that exist in both dictionaries.

    Parameters:
    dict1 (dict): The first dictionary.
    dict2 (dict): The second dictionary.

    Returns:
    dict: A new dictionary containing the intersection of dict1 and dict2.
    """

    # Initialize an empty dictionary to store the intersection
    intersection_dict = {}

    # Iterate over the key-value pairs in the first dictionary
    for key, value in dict1.items():
        # Check if the key exists in the second dictionary and if the values are equal
        if key in dict2 and dict2[key] == value:
            # If the key exists and the values are equal, add the key-value pair to the intersection dictionary
            intersection_dict[key] = value

    # Return the intersection dictionary
    return intersection_dict


# Example usage:
dict1 = {'a': 1, 'b': 2, 'c': 3}
dict2 = {'a': 1, 'b': 3, 'd': 4}

print(intersect_dictionaries(dict1, dict2))  # Output: {'a': 1}
```
