## Exercise
Create a program that takes two numbers and an arithmetic operation as input and performs the operation on the numbers. The operations can be addition, subtraction, multiplication, or division.

## Reference Solution
```python
def calculate(num1, num2, operation):
    """
    Perform arithmetic operation on two numbers.

    Args:
        num1 (float): The first number.
        num2 (float): The second number.
        operation (str): The arithmetic operation. Can be '+', '-', '*', '/'.

    Returns:
        float: The result of the operation.
    """
    if operation == '+':
        return num1 + num2
    elif operation == '-':
        return num1 - num2
    elif operation == '*':
        return num1 * num2
    elif operation == '/':
        if num2 != 0:
            return num1 / num2
        else:
            raise ZeroDivisionError("Cannot divide by zero")
    else:
        raise ValueError("Invalid operation. Must be '+', '-', '*', or '/'")


def main():
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))
    operation = input("Enter the operation (+, -, *, /): ")

    try:
        result = calculate(num1, num2, operation)
        print(f"{num1} {operation} {num2} = {result}")
    except (ZeroDivisionError, ValueError) as e:
        print(e)


if __name__ == "__main__":
    main()
```
