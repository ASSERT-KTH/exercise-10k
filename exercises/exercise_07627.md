## Exercise
Write a program that uses a for loop to find the maximum value in a given list of numbers.

## Reference Solution
```python
def find_max_value(num_list):
    """
    This function finds the maximum value in a given list of numbers.

    Args:
        num_list (list): A list of numbers.

    Returns:
        max_value (int or float): The maximum value in the list.
    """
    if not num_list:
        return None

    max_value = num_list[0]
    for num in num_list:
        if num > max_value:
            max_value = num

    return max_value


# Example usage:
numbers = [4, 2, 9, 6, 5, 1, 8, 3, 7]
max_val = find_max_value(numbers)

if max_val is not None:
    print(f"The maximum value in the list is: {max_val}")
else:
    print("The list is empty.")
```
