## Exercise
Write a program to round a given number up to the nearest integer

## Reference Solution
```python
import math

def round_up(num):
    """
    Rounds a given number up to the nearest integer.

    Args:
        num (float): The number to be rounded up.

    Returns:
        int: The rounded up integer.
    """
    return math.ceil(num)

# Example usage:
if __name__ == "__main__":
    num = 4.7
    rounded_num = round_up(num)
    print(f"The number {num} rounded up is: {rounded_num}")
```
