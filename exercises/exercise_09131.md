## Exercise
Create a program that takes a user's input and checks its data type. The program should be able to identify whether the input is an integer, float, or string. If the input is an integer or float, it should also check if it is positive, negative, or zero.

## Reference Solution
```python
def get_input():
    """Get user input."""
    return input("Please enter a value: ")


def check_data_type(input_value):
    """Check the data type of the input."""
    try:
        # Try to convert the input to an integer
        int_value = int(input_value)
        print("The input is an integer.")
        check_sign(int_value)
    except ValueError:
        try:
            # If it's not an integer, try to convert it to a float
            float_value = float(input_value)
            print("The input is a float.")
            check_sign(float_value)
        except ValueError:
            # If it's neither an integer nor a float, it's a string
            print("The input is a string.")


def check_sign(value):
    """Check if the number is positive, negative, or zero."""
    if value > 0:
        print("The number is positive.")
    elif value < 0:
        print("The number is negative.")
    else:
        print("The number is zero.")


def main():
    """Main function."""
    input_value = get_input()
    check_data_type(input_value)


if __name__ == "__main__":
    main()
```
