## Exercise
Create a program that takes a list as input and returns True if it is sorted in ascending order, False otherwise.

## Reference Solution
```python
def is_sorted(lst):
    for i in range(1, len(lst)):
        if lst[i - 1] > lst[i]:
            return False
    return True

print(is_sorted([1, 2, 3, 4, 5]))  # True
print(is_sorted([1, 3, 2, 4, 5]))  # False
print(is_sorted([5, 4, 3, 2, 1]))  # False
```
