## Exercise
Create a function that takes a set and returns the size of the set

## Reference Solution
```python
def set_size(input_set):
    """
    Calculate the size of a set.

    Args:
        input_set (set): The input set.

    Returns:
        int: The size of the set.
    """
    return len(input_set)

# Example usage:
if __name__ == "__main__":
    example_set = {1, 2, 3, 4, 5}
    print(f"The size of the set is: {set_size(example_set)}")
```
