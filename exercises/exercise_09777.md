## Exercise
Write a program that takes a string as input and prints it with a specific format (e.g. title case, uppercase, etc.).

## Reference Solution
```python
def print_string_formats(input_string):
    # Title case
    print(f"Title Case: {input_string.title()}")

    # Uppercase
    print(f"Uppercase: {input_string.upper()}")

    # Lowercase
    print(f"Lowercase: {input_string.lower()}")

    # Capitalize
    print(f"Capitalize: {input_string.capitalize()}")

    # Swapcase
    print(f"Swapcase: {input_string.swapcase()}")


def main():
    input_string = input("Enter a string: ")
    print_string_formats(input_string)


if __name__ == "__main__":
    main()
```
