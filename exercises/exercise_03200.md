## Exercise
Write a program that takes a string as input and uses if/else statements to check if it's a palindrome (e.g., reads the same forwards and backwards).

## Reference Solution
```python
def is_palindrome(s):
    """
    Checks if a given string is a palindrome.

    Args:
    s (str): The input string.

    Returns:
    bool: True if the string is a palindrome, False otherwise.
    """
    # Remove any non-alphanumeric characters and convert to lowercase
    cleaned_s = ''.join(c for c in s if c.isalnum()).lower()
    
    # Compare the cleaned string with its reverse
    if cleaned_s == cleaned_s[::-1]:
        return True
    else:
        return False

# Test the function
def main():
    s = input("Enter a string: ")
    if is_palindrome(s):
        print(f'"{s}" is a palindrome.')
    else:
        print(f'"{s}" is not a palindrome.')

if __name__ == "__main__":
    main()
```
