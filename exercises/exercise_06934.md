## Exercise
Create a function that takes a value as an argument and returns its type, with type checking to ensure the input is a number

## Reference Solution
```python
def get_type(value):
    """
    Returns the type of the input value, with type checking to ensure the input is a number.

    Args:
        value: The input value to check.

    Returns:
        The type of the input value if it's a number, otherwise raises a TypeError.

    Raises:
        TypeError: If the input value is not a number.
    """
    if isinstance(value, (int, float)):
        return type(value)
    else:
        raise TypeError("Input value must be a number")
```

```python
# Test with an integer
print(get_type(10))  # Output: <class 'int'>

# Test with a float
print(get_type(3.14))  # Output: <class 'float'>

# Test with a non-numeric value
try:
    print(get_type("hello"))
except TypeError as e:
    print(e)  # Output: Input value must be a number
```
