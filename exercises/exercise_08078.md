## Exercise
Write a function with a docstring that describes the function

## Reference Solution

def greet(name: str) -> None:
    """
    Prints a personalized greeting message to the console.

    Args:
        name (str): The name of the person to be greeted.

    Returns:
        None
    """
    print(f"Hello, {name}!")

# Example usage:
greet("John")
