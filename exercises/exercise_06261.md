## Exercise
Create a dictionary with 5 key-value pairs, where each key is a name and each value is an age

```python
people = {
    "John": 25,
    "Emily": 30,
    "Michael": 35,

    "William": 40
}
```

```python
print(people["John"])  # Outputs: 25
```

```python
for name, age in people.items():
```
