## Exercise
Write a program to print a pattern of asterisks (*) using nested for loops, where the number of asterisks in each row increases by one.

## Reference Solution
```python
def print_pattern(n):
    for i in range(n):
        for j in range(i + 1):
            print("*", end="")
        print()

print_pattern(5)
```
