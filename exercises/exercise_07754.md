## Exercise
Create a function that takes a string as input and checks if it is a palindrome. If it is, the function should return the reversed string. If not, it should return a message stating that the input string is not a palindrome. The function should handle strings with spaces and special characters by ignoring them and considering only alphanumeric characters.

## Reference Solution
```python
def check_palindrome(s):
    """
    Checks if the input string is a palindrome. If it is, returns the reversed string.
    If not, returns a message stating that the input string is not a palindrome.
    
    The function ignores spaces and special characters by considering only alphanumeric characters.

    Parameters:
    s (str): The input string to be checked.

    Returns:
    str: The reversed string if the input string is a palindrome, otherwise a message stating that the input string is not a palindrome.
    """

    # First, we remove all non-alphanumeric characters from the string and convert it to lower case
    cleaned_s = ''.join(char.lower() for char in s if char.isalnum())

    # Then, we compare the cleaned string with its reverse
    # If they are equal, the string is a palindrome
    if cleaned_s == cleaned_s[::-1]:
        # If the string is a palindrome, we return the reversed string
        return cleaned_s[::-1]
    else:
        # If the string is not a palindrome, we return a message stating that the input string is not a palindrome
        return "The input string is not a palindrome."

# Test the function
print(check_palindrome("A man, a plan, a canal: Panama"))  # Should return "amanaplanacanalpanama"
print(check_palindrome("Hello"))  # Should return "The input string is not a palindrome."
```
