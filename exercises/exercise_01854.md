## Exercise
Write a program that takes a list of integers and a sublist as input, and returns True if the sublist is contiguous within the original list, False otherwise.

## Reference Solution
```python
def is_sublist_contiguous(main_list, sub_list):
    length = len(main_list)
    sub_length = len(sub_list)

    for i in range(length - sub_length + 1):
        if main_list[i:i + sub_length] == sub_list:
            return True

    return False

# Test the function
main_list = [1, 2, 3, 4, 5, 6, 7, 8, 9]
sub_list = [3, 4, 5]
print(is_sublist_contiguous(main_list, sub_list))  # Expected output: True

main_list = [1, 2, 3, 4, 5, 6, 7, 8, 9]
sub_list = [3, 5, 7]
print(is_sublist_contiguous(main_list, sub_list))  # Expected output: False
```
