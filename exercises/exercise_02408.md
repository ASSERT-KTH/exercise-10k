## Exercise
Create a function that takes keyword arguments and returns a formatted string. Then, call this function with keyword arguments and print the result.

## Reference Solution
```python
def formatted_string(**kwargs):
    """
    This function takes keyword arguments and returns a formatted string.
    
    Args:
        **kwargs: Variable number of keyword arguments.
    
    Returns:
        str: A formatted string containing the keyword arguments.
    """
    
    # Initialize an empty list to store the formatted key-value pairs
    formatted_pairs = []
    
    # Iterate over each key-value pair in kwargs
    for key, value in kwargs.items():
        # Format the key-value pair as a string and add it to the list
        formatted_pairs.append(f"{key}: {value}")
    
    # Join all the formatted pairs into a single string separated by commas
    result = ", ".join(formatted_pairs)
    
    # Return the formatted string
    return result


# Call the function with keyword arguments and print the result
print(formatted_string(name="John", age=30, city="New York"))
```
