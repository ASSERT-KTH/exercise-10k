## Exercise
Write a program that takes a list as input, and returns a new list containing only the unique elements of the original list.

## Reference Solution
```python
def get_unique_elements(input_list):
    """
    Returns a new list containing only the unique elements of the original list.

    Args:
        input_list (list): The input list.

    Returns:
        list: A new list containing only the unique elements.
    """
    seen = set()
    unique_list = [x for x in input_list if not (x in seen or seen.add(x))]
    return unique_list

# Example usage
input_list = [1, 2, 2, 3, 4, 4, 5, 6, 6]
unique_list = get_unique_elements(input_list)
print(unique_list)  # Output: [1, 2, 3, 4, 5, 6]
```
