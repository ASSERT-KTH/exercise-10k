## Exercise
Write a function that uses the map function to apply a given function to a list of numbers and return the results.

## Reference Solution
```python
def apply_function(func, numbers):
    """
    Applies a given function to a list of numbers and returns the results.

    Args:
        func (function): The function to be applied to the list of numbers.
        numbers (list): A list of numbers.

    Returns:
        list: A list of results after applying the function to each number.
    """

    # First, we check if the input is a list and if all elements in the list are numbers
    if not isinstance(numbers, list) or not all(isinstance(num, (int, float)) for num in numbers):
        raise TypeError("Input must be a list of numbers")

    # We use the map function to apply the given function to each number in the list
    # The map function returns a map object, which is an iterator, so we need to convert it to a list
    # We use the list function to convert the map object to a list
    results = list(map(func, numbers))

    # Finally, we return the list of results
    return results

# Example usage:
def square(x):
    return x ** 2

numbers = [1, 2, 3, 4, 5]
print(apply_function(square, numbers))  # Output: [1, 4, 9, 16, 25]
```
