## Exercise
Write a higher-order function that filters a list based on a given predicate function.

## Reference Solution
```python
def filter_list(predicate, lst):
    """
    This function filters a list based on a given predicate function.
    
    Args:
        predicate (function): A function that takes one argument and returns a boolean value.
        lst (list): The list to be filtered.
    
    Returns:
        list: A new list containing only the elements for which the predicate function returns True.
    """
    
    # Initialize an empty list to store the filtered elements
    filtered_list = []
    
    # Iterate over each element in the input list
    for element in lst:
        # Apply the predicate function to the current element
        if predicate(element):
            # If the predicate function returns True, add the element to the filtered list
            filtered_list.append(element)
    
    # Return the filtered list
    return filtered_list

# Example usage:
# Define a predicate function that checks if a number is even
def is_even(num):
    return num % 2 == 0

# Create a list of numbers
numbers = [1, 2, 3, 4, 5, 6]

# Filter the list using the predicate function
even_numbers = filter_list(is_even, numbers)

# Print the filtered list
print(even_numbers)  # Output: [2, 4, 6]
```
