## Exercise
Create a calculator program that takes in a mathematical expression as a string and evaluates it to produce the result.

## Reference Solution
```python
def calculator(expression):
    """
    Evaluates a mathematical expression and returns the result.

    Args:
    expression (str): A string representing a mathematical expression.

    Returns:
    float: The result of the evaluated expression.
    """
    try:
        result = eval(expression)
        return result
    except Exception as e:
        return str(e)

def main():
    # Example usage:
    expressions = [
        "1 + 2 * 3",
        "10 / 2",
        "5 - 3 + 2",
        "7 * 8 - 2",
        "10 ** 2"
    ]

    for expression in expressions:
        print(f"Expression: {expression}")
        print(f"Result: {calculator(expression)}")
        print()

if __name__ == "__main__":
    main()
```
