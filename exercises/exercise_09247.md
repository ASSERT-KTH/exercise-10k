## Exercise
Write a program that uses string formatting to print a formatted string.

## Reference Solution
```python
# Define a function to print a formatted string
def print_formatted_string(name, age, city):
    # Use string formatting to print the formatted string
    formatted_string = f"My name is {name}, I am {age} years old, and I live in {city}."
    print(formatted_string)

# Main function
def main():
    # Get input from the user
    name = input("Enter your name: ")
    age = int(input("Enter your age: "))
    city = input("Enter your city: ")

    # Call the function to print the formatted string
    print_formatted_string(name, age, city)

# Run the main function
if __name__ == "__main__":
    main()
```
