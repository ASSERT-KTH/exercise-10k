## Exercise
Write a program that asks for a string and extracts a substring.

## Reference Solution
```python
def extract_substring():
    # Get the original string from the user
    original_string = input("Please enter a string: ")

    # Get the start and end indices from the user
    start_index = int(input("Please enter the start index of the substring: "))
    end_index = int(input("Please enter the end index of the substring: "))

    # Validate the indices
    if start_index < 0 or end_index > len(original_string) or start_index > end_index:
        print("Invalid indices. Please try again.")
        return extract_substring()

    # Extract the substring
    substring = original_string[start_index:end_index]

    # Print the substring
    print("The extracted substring is:", substring)

# Call the function
extract_substring()
```
