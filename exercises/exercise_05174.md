## Exercise
Write a function to count the number of unique values in a dictionary.

## Reference Solution
```python
def count_unique_values(dictionary):
    """
    Counts the number of unique values in a dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        int: The number of unique values in the dictionary.
    """
    return len(set(dictionary.values()))

# Example usage:
if __name__ == "__main__":
    dictionary = {
        "key1": "value1",
        "key2": "value1",
        "key3": "value2",
        "key4": "value3",
    }
    unique_count = count_unique_values(dictionary)
    print(f"Number of unique values: {unique_count}")
```
