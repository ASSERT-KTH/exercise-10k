## Exercise
Write a program that prints the numbers from 1 to 10 using a for loop, then prints the numbers from 10 to 1 using a while loop.

## Reference Solution
```python
// #include <bits/stdc++.h>
#include <iostream>

using namespace std;

void solve() {
  for (int i = 1; i <= 10; i++) {
    cout << i << endl;
  }
  int i = 10;
  while (i >= 1) {
    cout << i << endl;
    i--;
  }
}

int main() {
  solve();
  return 0;
}
```
