## Exercise
Write a program to generate a multiplication table up to 10x10 using nested for loops.

## Reference Solution
```python
# Print the header row
print("   |", end="")  # Empty space for the first column
for i in range(1, 11):  # Loop from 1 to 10
    print(f"{i:4}", end="")  # Print each number with a width of 4
print()  # Move to the next line
print("-" * 45)  # Print a horizontal line

# Print each row
for i in range(1, 11):  # Outer loop for the rows
    print(f"{i:3}|", end="")  # Print the row number with a width of 3
    for j in range(1, 11):  # Inner loop for the columns
        print(f"{i*j:4}", end="")  # Print the product of i and j with a width of 4
    print()  # Move to the next line
```
