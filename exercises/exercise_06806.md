## Exercise
Write a program that takes a string input from the user, reverses the string, and checks if the original string is a palindrome. A palindrome is a string that reads the same backward as forward. The program should print the reversed string and a message indicating whether the original string is a palindrome or not.

## Reference Solution
```python
# Author: [Your Name]
# Date: [Current Date]

# Get user input
str = input("Enter a string: ")

# Reverse the string
rev_str = str[::-1]

# Check if the string is a palindrome
if str == rev_str:
    print("The string is a palindrome.")
else:
    print("The string is not a palindrome.")

# Print the reversed string
print("The reversed string is: ", rev_str)

# Print the original string
print("The original string is: ", str)
```
