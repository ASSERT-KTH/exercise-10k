## Exercise
Write a program that asks the user for a temperature in Celsius or Fahrenheit and converts it to the other unit. The user should be able to input the temperature and the unit (C or F).

## Reference Solution
```python
def convert_temperature(temp, unit):
    """
    Converts temperature from Celsius to Fahrenheit or vice versa.

    Args:
        temp (float): The temperature to convert.
        unit (str): The unit of the temperature (C or F).

    Returns:
        tuple: A tuple containing the converted temperature and the new unit.
    """
    if unit.upper() == 'C':
        converted_temp = temp * 9 / 5 + 32
        new_unit = 'F'
    elif unit.upper() == 'F':
        converted_temp = (temp - 32) * 5 / 9
        new_unit = 'C'
    else:
        raise ValueError("Invalid unit. Please enter C or F.")

    return converted_temp, new_unit


def main():
    temp = float(input("Enter the temperature: "))
    unit = input("Enter the unit (C or F): ")

    try:
        converted_temp, new_unit = convert_temperature(temp, unit)
        print(f"{temp} {unit} is equal to {converted_temp:.2f} {new_unit}")
    except ValueError as e:
        print(e)


if __name__ == "__main__":
    main()
```
