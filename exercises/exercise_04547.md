## Exercise
Create a program that concatenates two or more given strings.

## Reference Solution
```python
def concatenate_strings(*args):
    """
    Concatenates two or more given strings.

    Args:
        *args (str): Variable number of string arguments.

    Returns:
        str: The concatenated string.
    """
    return ''.join(args)


def main():
    # Example usage:
    strings_to_concatenate = ["Hello, ", "world! ", "This ", "is ", "a ", "test."]
    print("Strings to concatenate: ", strings_to_concatenate)
    concatenated_string = concatenate_strings(*strings_to_concatenate)
    print("Concatenated string: ", concatenated_string)


if __name__ == "__main__":
    main()
```
