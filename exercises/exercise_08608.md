## Exercise
Write a program that takes a string and an integer as input, and returns the string with each character shifted by the integer.

## Reference Solution
```python
import sys

def shift_string(s, shift):
    return s[shift:] + s[:shift]

def main():
    s = input("Enter a string: ")
    shift = int(input("Enter a shift value: "))
    result = shift_string(s, shift)
    print(result)

if __name__ == "__main__":
    main()
```
