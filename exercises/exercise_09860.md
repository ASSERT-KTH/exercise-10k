## Exercise
Create a function that takes an input string and encodes it using a combination of string manipulation techniques: reverse the string, replace all vowels with their corresponding ASCII values, and then concatenate the result with its mirror image (the string reversed again). The function should handle both lowercase and uppercase letters and should not modify non-alphabetic characters.

## Reference Solution
```python
def encode_string(input_str):
    """
    This function takes an input string, reverses it, replaces vowels with their ASCII values, 
    and then concatenates the result with its mirror image.

    Parameters:
    input_str (str): The input string to be encoded.

    Returns:
    str: The encoded string.
    """

    # First, we define the vowels in both lowercase and uppercase
    vowels = 'aeiouAEIOU'

    # We reverse the input string
    reversed_str = input_str[::-1]

    # We initialize an empty string to store the result after replacing vowels with ASCII values
    ascii_replaced_str = ''

    # We iterate over each character in the reversed string
    for char in reversed_str:
        # If the character is a vowel, we replace it with its ASCII value
        if char in vowels:
            ascii_replaced_str += str(ord(char))
        # If the character is not a vowel, we keep it as it is
        else:
            ascii_replaced_str += char

    # We create the mirror image of the result by reversing it again
    mirror_image = ascii_replaced_str[::-1]

    # Finally, we concatenate the result with its mirror image
    encoded_str = ascii_replaced_str + mirror_image

    return encoded_str
```
