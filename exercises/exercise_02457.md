## Exercise
Write a program that takes an array as input and returns a new array containing all possible permutations of the elements.

## Reference Solution
```python
import itertools

def get_permutations(arr):
    """
    Returns a new array containing all possible permutations of the elements in the input array.

    Args:
        arr (list): The input array.

    Returns:
        list: A new array containing all possible permutations of the elements in the input array.
    """
    # Use itertools.permutations to generate all permutations of the input array
    perms = list(itertools.permutations(arr))
    
    # Convert the permutations to a list of lists
    perms = [list(perm) for perm in perms]
    
    return perms

# Example usage:
arr = [1, 2, 3]
perms = get_permutations(arr)
for perm in perms:
    print(perm)
```
