## Exercise
Create a function to remove duplicates from a sorted array

## Reference Solution
```python
def remove_duplicates(sorted_array):
    """
    Removes duplicates from a sorted array.

    Args:
        sorted_array (list): A sorted list of elements.

    Returns:
        list: A new list with duplicates removed.
    """
    if not sorted_array:
        return []

    result = [sorted_array[0]]
    for i in range(1, len(sorted_array)):
        if sorted_array[i] != sorted_array[i - 1]:
            result.append(sorted_array[i])

    return result

# Example usage:
def main():
    sorted_array = [1, 1, 2, 2, 2, 3, 4, 4, 5, 6, 6]
    print("Original Array:", sorted_array)
    print("Array without duplicates:", remove_duplicates(sorted_array))

if __name__ == "__main__":
    main()
```
