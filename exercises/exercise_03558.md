## Exercise
Write a program that takes two lists as input, and returns a boolean indicating whether the two lists are disjoint (i.e., have no elements in common).

## Reference Solution
```python
def disjoint(a, b):
    """Return True if the two lists are disjoint, False otherwise."""
    for x in a:
        if x in b:
            return False
    return True

# Test the function
a = [1, 2, 3]
b = [4, 5, 6]
print(disjoint(a, b))  # Expected output: True

a = [1, 2, 3]
b = [3, 4, 5]
print(disjoint(a, b))  # Expected output: False
```
