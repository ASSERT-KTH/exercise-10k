## Exercise
Write a function to sort a list of integers in ascending order.

## Reference Solution
```python
def sort_list_ascending(numbers):
    """
    Sorts a list of integers in ascending order.

    Args:
        numbers (list): A list of integers.

    Returns:
        list: The sorted list of integers.
    """
    return sorted(numbers)

# Example usage:
numbers = [64, 34, 25, 12, 22, 11, 90]
sorted_numbers = sort_list_ascending(numbers)
print(sorted_numbers)

def sort_list_ascending_manual(numbers):
    """
    Sorts a list of integers in ascending order using the quicksort algorithm.

    Args:
        numbers (list): A list of integers.

    Returns:
        list: The sorted list of integers.
    """
    if len(numbers) <= 1:
        return numbers
    pivot = numbers[len(numbers) // 2]
    left = [x for x in numbers if x < pivot]
    middle = [x for x in numbers if x == pivot]
    right = [x for x in numbers if x > pivot]
    return sort_list_ascending_manual(left) + middle + sort_list_ascending_manual(right)

# Example usage:
numbers = [64, 34, 25, 12, 22, 11, 90]
sorted_numbers = sort_list_ascending_manual(numbers)
print(sorted_numbers)
```
