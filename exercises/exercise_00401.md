## Exercise
Write a program to calculate the sum of all elements in an array.

## Reference Solution
```python
def calculate_array_sum(array):
    """
    This function calculates the sum of all elements in an array.

    Args:
        array (list): A list of numbers.

    Returns:
        int: The sum of all elements in the array.
    """
    return sum(array)


def main():
    # Test the function with an example array
    array = [1, 2, 3, 4, 5]
    print("Array:", array)
    print("Sum of array elements:", calculate_array_sum(array))


if __name__ == "__main__":
    main()
```
