## Exercise
Create a program that calculates the Value-Added Tax (VAT) amount based on the price and VAT rate provided by the user.

## Reference Solution
```python
def calculate_vat(price, vat_rate):
    """
    Calculate the VAT amount based on the price and VAT rate.

    Args:
        price (float): The price of the item.
        vat_rate (float): The VAT rate as a decimal (e.g., 20% = 0.20).

    Returns:
        float: The VAT amount.
    """
    vat_amount = price * vat_rate
    return vat_amount

def main():
    # Get user input
    price = float(input("Enter the price: £"))
    vat_rate = float(input("Enter the VAT rate (in %): ")) / 100

    # Calculate VAT amount
    vat_amount = calculate_vat(price, vat_rate)

    # Calculate total price
    total_price = price + vat_amount

    # Display results
    print(f"Price: £{price:.2f}")
    print(f"VAT rate: {vat_rate * 100}%")
    print(f"VAT amount: £{vat_amount:.2f}")
    print(f"Total price: £{total_price:.2f}")

if __name__ == "__main__":
    main()
```
