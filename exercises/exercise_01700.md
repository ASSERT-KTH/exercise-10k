## Exercise
Write a function to rotate a list by a given number of positions.

## Reference Solution
```python
def rotate_list(input_list, positions):
    """
    Rotate a list by a given number of positions.

    Args:
        input_list (list): The list to be rotated.
        positions (int): The number of positions to rotate the list.

    Returns:
        list: The rotated list.
    """
    if len(input_list) == 0:
        return input_list

    positions = positions % len(input_list)
    return input_list[-positions:] + input_list[:-positions]

# Example usage:
def main():
    input_list = [1, 2, 3, 4, 5]
    positions = 2
    print("Original List:", input_list)
    print("Rotated List:", rotate_list(input_list, positions))

if __name__ == "__main__":
    main()
```
