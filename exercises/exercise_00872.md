## Exercise
Write a program that takes in a number and outputs whether it is even or odd based on the following conditions: if the number is divisible by 2, output 'Even', if the number is not divisible by 2, output 'Odd'.

## Reference Solution
```python
# Define the function
def is_even(x):
    if x % 2 == 0:
        return 'Even'
    else:
        return 'Odd'

# Call the function
print(is_even(9))
```
