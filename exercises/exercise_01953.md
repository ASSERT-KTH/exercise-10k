## Exercise
Write a program that takes an array as input and returns the sum of all elements in the array.

## Reference Solution
```python
// define the function
function sum(arr) {
  // initialize sum variable
  var sum = 0;

  // loop through array
  for (var i = 0; i < arr.length; i++) {
    // add current element to sum
    sum += arr[i];
  }

  // return sum
  return sum;
}

// call the function with a sample array
var arr = [1, 2, 3, 4, 5];
console.log(sum(arr)); // output: 15
```
