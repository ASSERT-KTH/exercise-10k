## Exercise
Create a program that uses functions to perform basic arithmetic operations. The program should take in two numbers and an operation as input, and then call the corresponding function to perform the operation. The operations that need to be supported are addition, subtraction, multiplication, and division.

## Reference Solution
```python
# Define functions for each arithmetic operation
def add(num1, num2):
    """
    Adds two numbers.

    Args:
        num1 (float): The first number.
        num2 (float): The second number.

    Returns:
        float: The sum of num1 and num2.
    """
    return num1 + num2


def subtract(num1, num2):
    """
    Subtracts num2 from num1.

    Args:
        num1 (float): The first number.
        num2 (float): The second number.

    Returns:
        float: The difference of num1 and num2.
    """
    return num1 - num2


def multiply(num1, num2):
    """
    Multiplies two numbers.

    Args:
        num1 (float): The first number.
        num2 (float): The second number.

    Returns:
        float: The product of num1 and num2.
    """
    return num1 * num2


def divide(num1, num2):
    """
    Divides num1 by num2.

    Args:
        num1 (float): The dividend.
        num2 (float): The divisor.

    Returns:
        float: The quotient of num1 and num2.

    Raises:
        ZeroDivisionError: If num2 is zero.
    """
    if num2 == 0:
        raise ZeroDivisionError("Cannot divide by zero")
    return num1 / num2


def main():
    # Get user input
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))
    operation = input("Enter the operation (+, -, *, /): ")

    # Call the corresponding function based on the operation
    operations = {
        "+": add,
        "-": subtract,
        "*": multiply,
        "/": divide
    }

    if operation in operations:
        try:
            result = operations[operation](num1, num2)
            print(f"{num1} {operation} {num2} = {result}")
        except ZeroDivisionError as e:
            print(str(e))
    else:
        print("Invalid operation")


if __name__ == "__main__":
    main()
```
