## Exercise
Write a function to get the values of a dictionary

## Reference Solution
```python
def get_dict_values(input_dict):
    """
    Returns a list of values from the input dictionary.

    Args:
        input_dict (dict): The dictionary to extract values from.

    Returns:
        list: A list of dictionary values.
    """
    return list(input_dict.values())

# Example usage:
if __name__ == "__main__":
    example_dict = {"key1": "value1", "key2": "value2", "key3": "value3"}
    print(get_dict_values(example_dict))
```
