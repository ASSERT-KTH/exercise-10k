## Exercise
Write a program that takes a string input from the user, checks if it can be converted to a floating-point number, and prints a success message if valid.

## Reference Solution
```python
def main():
    input_str = input()
    try:
        float(input_str)
        print("Success")
    except ValueError:
        pass

main()
```
