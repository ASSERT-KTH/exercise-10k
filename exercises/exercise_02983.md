## Exercise
Write a program that finds the first duplicate in a given list of numbers.

## Reference Solution
```python
def find_first_duplicate(numbers):
    """
    Finds the first duplicate in a given list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        int or None: The first duplicate number, or None if no duplicates are found.
    """
    seen = set()
    for num in numbers:
        if num in seen:
            return num
        seen.add(num)
    return None

# Example usage
numbers = [2, 1, 3, 5, 3, 2]
first_duplicate = find_first_duplicate(numbers)
if first_duplicate is not None:
    print(f"The first duplicate is: {first_duplicate}")
else:
    print("No duplicates found.")
```
