## Exercise
Write a program that compares two strings using variables and prints out the result of the comparison.

## Reference Solution
```python
# Define two string variables
str1 = "Hello"
str2 = "World"

# Compare the two strings
if str1 == str2:
    print(f"The strings '{str1}' and '{str2}' are equal.")
else:
    print(f"The strings '{str1}' and '{str2}' are not equal.")

# Compare the lengths of the two strings
if len(str1) > len(str2):
    print(f"The string '{str1}' is longer than '{str2}'.")
elif len(str1) < len(str2):
    print(f"The string '{str1}' is shorter than '{str2}'.")
else:
    print(f"The strings '{str1}' and '{str2}' are of equal length.")

# Compare the strings lexicographically
if str1 < str2:
    print(f"The string '{str1}' comes before '{str2}' alphabetically.")
elif str1 > str2:
    print(f"The string '{str1}' comes after '{str2}' alphabetically.")
else:
    print(f"The strings '{str1}' and '{str2}' are equal alphabetically.")
```
