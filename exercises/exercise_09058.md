## Exercise
Create a program that prints out the day of the week based on a given number using if/else statements

## Reference Solution
```python
def print_day_of_week(day_number):
    """
    Prints the day of the week based on the given number.

    Args:
        day_number (int): The number of the day (1-7), where 1 = Monday and 7 = Sunday.
    """
    if day_number == 1:
        print("Monday")
    elif day_number == 2:
        print("Tuesday")
    elif day_number == 3:
        print("Wednesday")
    elif day_number == 4:
        print("Thursday")
    elif day_number == 5:
        print("Friday")
    elif day_number == 6:
        print("Saturday")
    elif day_number == 7:
        print("Sunday")
    else:
        print("Invalid day number. Please enter a number between 1 and 7.")

def main():
    day_number = int(input("Enter the day number (1-7): "))
    print_day_of_week(day_number)

if __name__ == "__main__":
    main()
```
