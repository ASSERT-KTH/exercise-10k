## Exercise
Write a program that takes a string input from the user, reverses the string, and then concatenates it with the original string. The program should output the resulting string.

## Reference Solution
```python
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    cin >> s;
    string ans = s;
    reverse(s.begin(), s.end());
    ans += s;
    cout << ans << endl;
    return 0;
}
```
