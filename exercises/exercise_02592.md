## Exercise
Write a program to find the most frequent element in a given list of numbers.

## Reference Solution
```python
from collections import Counter

def most_frequent_element(numbers):
    """
    Find the most frequent element in a list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        int: The most frequent element in the list.
    """
    # Count the frequency of each element in the list
    frequency = Counter(numbers)

    # Find the element with the highest frequency
    most_frequent = max(frequency, key=frequency.get)

    return most_frequent

def main():
    # Example usage
    numbers = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]
    most_frequent = most_frequent_element(numbers)
    print("Most frequent element:", most_frequent)

if __name__ == "__main__":
    main()
```
