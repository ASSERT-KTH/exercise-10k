## Exercise
Write a function to print all the key-value pairs in a dictionary.

## Reference Solution
```python
def print_dict_key_value_pairs(input_dict):
    """
    Prints all key-value pairs in a dictionary.

    Args:
        input_dict (dict): The dictionary to print key-value pairs from.
    """
    for key, value in input_dict.items():
        print(f"{key}: {value}")


# Example usage:
if __name__ == "__main__":
    example_dict = {"name": "John", "age": 30, "city": "New York"}
    print("Dictionary Key-Value Pairs:")
    print_dict_key_value_pairs(example_dict)
```
