## Exercise
Write a function to create a new dictionary with specified keys and values using the fromkeys() method

## Reference Solution
```python
def create_dictionary(keys, value=None):
    """
    Create a new dictionary with specified keys and values.

    Args:
        keys (list): A list of keys for the dictionary.
        value (any, optional): The value for all keys. Defaults to None.

    Returns:
        dict: A new dictionary with the specified keys and values.
    """
    return dict.fromkeys(keys, value)

# Example usage
keys = ['apple', 'banana', 'cherry']
value = 'fruit'

new_dict = create_dictionary(keys, value)

print(new_dict)
```

```python
def create_dictionary(keys, value=None):
    """
    Create a new dictionary with specified keys and values.

    Args:
        keys (list): A list of keys for the dictionary.
        value (any, optional): The value for all keys. Defaults to None.

    Returns:
        dict: A new dictionary with the specified keys and values.
    """
    if isinstance(value, list):
        return {key: value.copy() for key in keys}
    else:
        return dict.fromkeys(keys, value)

# Example usage
keys = ['apple', 'banana', 'cherry']
value = [1, 2, 3]

new_dict = create_dictionary(keys, value)

print(new_dict)
```

```python
{'apple': [1, 2, 3], 'banana': [1, 2, 3], 'cherry': [1, 2, 3]}
```
