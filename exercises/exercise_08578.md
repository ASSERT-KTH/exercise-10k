## Exercise
Write a function to get all key-value pairs from a dictionary

## Reference Solution
```python
def get_key_value_pairs(dictionary):
    """
    Returns a list of tuples containing all key-value pairs from the dictionary.

    Args:
        dictionary (dict): The dictionary to retrieve key-value pairs from.

    Returns:
        list[tuple]: A list of tuples, where each tuple contains a key-value pair.
    """
    return list(dictionary.items())

# Example usage:
if __name__ == "__main__":
    example_dict = {"name": "John", "age": 30, "city": "New York"}
    key_value_pairs = get_key_value_pairs(example_dict)
    print(key_value_pairs)
```
