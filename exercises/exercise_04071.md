## Exercise
Write a program that takes a string as input and returns the string with all leading and trailing whitespace removed.

## Reference Solution
```python
def trim(s):
    if len(s) == 0:
        return s
    start = 0
    end = len(s) - 1
    while start <= end and s[start] == ' ':
        start += 1
    while start <= end and s[end] == ' ':
        end -= 1
    return s[start:end+1]

# Test the function
s = '   hello world   '
print('[' + s + ']')  # ['   hello world   ']
print('[' + trim(s) + ']')  # [hello world]
```
