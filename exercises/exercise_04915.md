## Exercise
Write a program that simulates a coin flip and prints the result (Heads or Tails).

## Reference Solution
```python
import random

def coin_flip():
    """
    Simulates a coin flip and returns the result.
    """
    flip = random.randint(0, 1)
    if flip == 0:
        return "Heads"
    else:
        return "Tails"

def main():
    """
    Prints the result of a coin flip.
    """
    result = coin_flip()
    print(f"The coin landed on: {result}")

if __name__ == "__main__":
    main()
```
