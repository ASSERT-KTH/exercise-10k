## Exercise
Write a function that takes a list of pairs as input and returns two lists: one containing the first elements of each pair and one containing the second elements of each pair.

## Reference Solution
```python
def separate_pairs(pairs):
    """
    This function takes a list of pairs as input and returns two lists: 
    one containing the first elements of each pair and one containing the second elements of each pair.

    Args:
        pairs (list): A list of pairs.

    Returns:
        tuple: Two lists, the first containing the first elements of each pair and the second containing the second elements of each pair.
    """

    # Initialize two empty lists to store the first and second elements of each pair
    first_elements = []
    
    # Initialize the second list
    second_elements = []

    # Iterate over each pair in the input list
    for pair in pairs:
        # Append the first element of the pair to the first list
        first_elements.append(pair[0])
        
        # Append the second element of the pair to the second list
        second_elements.append(pair[1])

    # Return the two lists
    return first_elements, second_elements

# Example usage:
pairs = [(1, 2), (3, 4), (5, 6)]
first, second = separate_pairs(pairs)
print("First elements:", first)
print("Second elements:", second)
```
