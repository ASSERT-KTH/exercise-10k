## Exercise
Write a program that asks for a sentence and reverses the order of its words.

## Reference Solution
```python
def reverse_words(sentence):
    """
    Reverses the order of words in a sentence.

    Args:
        sentence (str): The input sentence.

    Returns:
        str: The sentence with words in reversed order.
    """
    words = sentence.split()  # Split the sentence into words
    reversed_sentence = ' '.join(reversed(words))  # Reverse the words and join them back into a sentence
    return reversed_sentence

def main():
    sentence = input("Please enter a sentence: ")  # Ask for user input
    reversed_sentence = reverse_words(sentence)  # Reverse the words in the sentence
    print("Reversed sentence: ", reversed_sentence)  # Print the result

if __name__ == "__main__":
    main()
```
