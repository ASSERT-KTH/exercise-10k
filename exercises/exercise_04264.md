## Exercise
Write a function that takes a dictionary and a key as input and removes the key-value pair from the dictionary

## Reference Solution
```python
def remove_key_value_pair(dictionary, key):
    """
    Removes the key-value pair from the dictionary if the key exists.

    Args:
        dictionary (dict): The dictionary to modify.
        key: The key to remove from the dictionary.

    Returns:
        dict: The modified dictionary.
    """
    if key in dictionary:
        del dictionary[key]
    return dictionary

# Example usage
if __name__ == "__main__":
    my_dict = {"name": "John", "age": 30, "city": "New York"}
    print("Original dictionary:", my_dict)

    key_to_remove = "age"
    updated_dict = remove_key_value_pair(my_dict, key_to_remove)
    print("Dictionary after removing '{}':".format(key_to_remove), updated_dict)
```
