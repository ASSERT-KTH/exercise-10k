## Exercise
Write a program that demonstrates variable shadowing by declaring a variable with the same name as a global variable.

## Reference Solution
```python
# Global variable
x = 10

def local_scope():
    # Local variable with the same name as the global variable
    x = 20
    print("Local variable x:", x)

def modify_global_variable():
    global x  # Use the global keyword to modify the global variable
    x = 30
    print("Global variable x modified to:", x)

def main():
    print("Initial global variable x:", x)
    local_scope()
    print("Global variable x after local scope:", x)
    modify_global_variable()
    print("Global variable x after modification:", x)

if __name__ == "__main__":
    main()
```
