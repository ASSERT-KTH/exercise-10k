## Exercise
Write a function that takes a list of key-value pairs as input and returns a dictionary

## Reference Solution
```python
def create_dict(pairs):
    """
    Convert a list of key-value pairs to a dictionary.

    Args:
        pairs (list): A list of tuples, where each tuple contains a key-value pair.

    Returns:
        dict: A dictionary created from the input key-value pairs.
    """
    return dict(pairs)


# Example usage:
if __name__ == "__main__":
    key_value_pairs = [("apple", 5), ("banana", 7), ("orange", 3)]
    result_dict = create_dict(key_value_pairs)
    print(result_dict)
```
