## Exercise
Create a function that uses type checking to verify the types of its parameters and return value.

## Reference Solution
```python
# Define a function with type checking for parameters and return value
def add_numbers(a: int, b: int) -> int:
    """
    This function adds two integers together.

    Args:
        a (int): The first integer to add.
        b (int): The second integer to add.

    Returns:
        int: The sum of the two integers.
    """

    # Check if both parameters are integers
    if not isinstance(a, int) or not isinstance(b, int):
        raise TypeError("Both parameters must be integers.")

    # Add the two numbers together
    result = a + b

    # Check if the result is an integer
    if not isinstance(result, int):
        raise TypeError("The result must be an integer.")

    # Return the result
    return result

# Test the function
print(add_numbers(5, 7))  # Output: 12
```
