## Exercise
Write a function to filter a dictionary based on a condition. For example, filter out key-value pairs where the value is less than 5.

## Reference Solution
```python
def filter_dict(input_dict, condition_func):
    """
    Filters a dictionary based on a given condition.

    Args:
    - input_dict (dict): The dictionary to be filtered.
    - condition_func (function): A function that takes a value and returns a boolean.

    Returns:
    - dict: A new dictionary with key-value pairs where the condition is met.
    """
    return {key: value for key, value in input_dict.items() if condition_func(value)}

# Example usage:
if __name__ == "__main__":
    # Define a sample dictionary
    sample_dict = {"a": 1, "b": 2, "c": 6, "d": 3, "e": 7}

    # Define a condition function
    def is_greater_than_five(value):
        return value > 5

    # Filter the dictionary
    filtered_dict = filter_dict(sample_dict, is_greater_than_five)

    # Print the result
    print("Original Dictionary:", sample_dict)
    print("Filtered Dictionary:", filtered_dict)
```
