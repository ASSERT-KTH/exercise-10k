## Exercise
Write a program that creates a weather forecast using if/else statement

## Reference Solution
```python
def get_weather_forecast(temperature, precipitation):
    """
    Returns a weather forecast based on temperature and precipitation.
    
    Parameters:
    temperature (float): Current temperature in degrees Celsius.
    precipitation (float): Current precipitation in mm.
    
    Returns:
    str: Weather forecast.
    """
    if temperature > 25:
        if precipitation > 0:
            return "Hot and rainy"
        else:
            return "Hot and sunny"
    elif temperature >= 15 and temperature <= 25:
        if precipitation > 0:
            return "Mild and rainy"
        else:
            return "Mild and sunny"
    else:
        if precipitation > 0:
            return "Cold and rainy"
        else:
            return "Cold and sunny"

def main():
    temperature = float(input("Enter current temperature (°C): "))
    precipitation = float(input("Enter current precipitation (mm): "))
    
    forecast = get_weather_forecast(temperature, precipitation)
    print("Weather Forecast:", forecast)

if __name__ == "__main__":
    main()
```
