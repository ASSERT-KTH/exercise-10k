## Exercise
Write a program that checks if a student has passed or failed based on their grade. If the grade is 50 or above, the student passes. Otherwise, they fail.

## Reference Solution
```python
def check_grade(grade):
    """
    Checks if a student has passed or failed based on their grade.

    Args:
        grade (float): The student's grade.

    Returns:
        str: "Pass" if the grade is 50 or above, "Fail" otherwise.
    """
    if grade >= 50:
        return "Pass"
    else:
        return "Fail"

def main():
    # Example usage
    grade = float(input("Enter the student's grade: "))
    result = check_grade(grade)
    print(f"The student has {result}ed.")

if __name__ == "__main__":
    main()
```
