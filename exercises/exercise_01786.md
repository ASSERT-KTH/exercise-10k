## Exercise
Write a function to iterate over the key-value pairs of a dictionary and perform a given operation on each pair.

## Reference Solution
```python
def iterate_dict(dictionary, operation):
    """
    Iterate over the key-value pairs of a dictionary and perform a given operation on each pair.

    Args:
        dictionary (dict): The dictionary to iterate over.
        operation (function): A function that takes two arguments (key and value) and performs an operation on them.

    Returns:
        None
    """

    # Check if the input is a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("Input must be a dictionary")

    # Check if the operation is a callable function
    if not callable(operation):
        raise TypeError("Operation must be a callable function")

    # Iterate over each key-value pair in the dictionary
    for key, value in dictionary.items():
        # Perform the given operation on the current key-value pair
        operation(key, value)


# Example usage:

# Define a dictionary
my_dict = {"a": 1, "b": 2, "c": 3}

# Define an operation function
def print_key_value(key, value):
    # Print the key-value pair
    print(f"{key}: {value}")

# Iterate over the dictionary and perform the operation
iterate_dict(my_dict, print_key_value)
```
