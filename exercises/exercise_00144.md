## Exercise
Create a function that takes in a dictionary and a key, and removes the key-value pair from the dictionary if it exists

## Reference Solution
```python
def remove_key(dictionary, key):
    """
    Removes a key-value pair from a dictionary if it exists.

    Args:
        dictionary (dict): The dictionary to modify.
        key: The key to remove.

    Returns:
        dict: The modified dictionary.
    """
    if key in dictionary:
        del dictionary[key]
    return dictionary

# Example usage
if __name__ == "__main__":
    my_dict = {"a": 1, "b": 2, "c": 3}
    print("Original dictionary:", my_dict)
    key_to_remove = "b"
    modified_dict = remove_key(my_dict, key_to_remove)
    print("Dictionary after removing key '{}':".format(key_to_remove), modified_dict)
```
