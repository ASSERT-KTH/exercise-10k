## Exercise
Write a program that takes an array and a reduction function as input, and returns the reduced value.

## Reference Solution
```python
def reduce(arr, fn):
    val = arr[0]
    for i in range(1, len(arr)):
        val = fn(arr[i], val)
    return val
```
