## Exercise
Create a function that takes a string input and checks if it's a palindrome. If the string is not a palindrome, the function should reverse the string and return the reversed string along with a boolean indicating whether the original string was a palindrome.

## Reference Solution
```python
def check_palindrome(input_str):
    """
    Checks if the input string is a palindrome. If not, it reverses the string and returns the reversed string along with a boolean indicating whether the original string was a palindrome.

    Args:
        input_str (str): The input string to be checked.

    Returns:
        tuple: A tuple containing the reversed string and a boolean indicating whether the original string was a palindrome.
    """

    # First, we remove any leading or trailing whitespace from the input string
    input_str = input_str.strip()

    # Then, we convert the input string to lowercase to make the comparison case-insensitive
    input_str = input_str.lower()

    # Now, we compare the input string with its reverse
    # We use Python's slicing feature to reverse the string
    reversed_str = input_str[::-1]

    # We check if the input string is equal to its reverse
    is_palindrome = input_str == reversed_str

    # If the input string is not a palindrome, we return the reversed string and False
    # Otherwise, we return the reversed string and True
    return reversed_str, is_palindrome


# Example usage:
input_str = "radar"
reversed_str, is_palindrome = check_palindrome(input_str)
print(f"Reversed String: {reversed_str}")
print(f"Is Palindrome: {is_palindrome}")
```
