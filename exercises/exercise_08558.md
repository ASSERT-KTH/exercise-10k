## Exercise
Create a dictionary with 3 key-value pairs and get the key-value pairs.

## Reference Solution
```python
# Create a dictionary with 3 key-value pairs
my_dict = {
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}

# Get the key-value pairs
key_value_pairs = my_dict.items()

# Print the key-value pairs
for key, value in key_value_pairs:
    print(f"Key: {key}, Value: {value}")
```
