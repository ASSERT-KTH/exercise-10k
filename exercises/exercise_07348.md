## Exercise
Convert the float 123.45 to an integer, rounding down to the nearest whole number, then print the result.

## Reference Solution
```python
import math

def convert_float_to_int(float_num):
    """
    Converts a float to an integer, rounding down to the nearest whole number.
    
    Args:
        float_num (float): The float number to be converted.
    
    Returns:
        int: The integer equivalent of the float number.
    """
    return math.floor(float_num)

# Example usage
float_num = 123.45
result = convert_float_to_int(float_num)
print(result)
```
