## Exercise
Write a program to concatenate two strings using the correct operator.

## Reference Solution
```python
def concatenate_strings(str1, str2):
    """
    Concatenates two strings using the '+' operator.

    Args:
        str1 (str): The first string.
        str2 (str): The second string.

    Returns:
        str: The concatenated string.
    """
    return str1 + str2

def main():
    # Example usage
    str1 = "Hello, "
    str2 = "World!"
    result = concatenate_strings(str1, str2)
    print(result)

if __name__ == "__main__":
    main()
```
