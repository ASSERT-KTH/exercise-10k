## Exercise
Create a function that takes a name and an age as input, with the age defaulting to 25 if not provided.

## Reference Solution
```python
# Define a function with name and age as parameters, defaulting age to 25 if not provided
def create_person(name, age=25):
    """
    Create a person object with a name and an age.

    Args:
        name (str): The name of the person.
        age (int, optional): The age of the person. Defaults to 25.

    Returns:
        dict: A dictionary representing the person.
    """
    
    # Check if the name is a string and the age is an integer
    if not isinstance(name, str) or not isinstance(age, int):
        raise TypeError("Name must be a string and age must be an integer.")
    
    # Create a dictionary to store the person's information
    person = {
        # Store the name in the dictionary
        "name": name,
        # Store the age in the dictionary
        "age": age
    }
    
    # Return the person dictionary
    return person

# Test the function
print(create_person("John"))
print(create_person("Jane", 30))
```
