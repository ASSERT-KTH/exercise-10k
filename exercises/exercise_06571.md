## Exercise
Create a dictionary and map its values to a new dictionary

## Reference Solution
```python
# Create the original dictionary
original_dict = {
    "key1": "value1",
    "key2": "value2",
    "key3": "value3"
}

# Print the original dictionary
print("Original Dictionary:")
print(original_dict)

# Map the values to a new dictionary
new_dict = {f"new_key_{i}": value for i, value in enumerate(original_dict.values())}

# Print the new dictionary
print("\nNew Dictionary:")
print(new_dict)
```
