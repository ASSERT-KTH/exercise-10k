## Exercise
Create a copy of a dictionary.

## Reference Solution
```python
# Original dictionary
original_dict = {'a': 1, 'b': 2, 'c': 3}

# Method 1: Using the copy() method
copied_dict1 = original_dict.copy()

# Method 2: Using the dict() constructor
copied_dict2 = dict(original_dict)

print("Original Dictionary:", original_dict)
print("Copied Dictionary 1:", copied_dict1)
print("Copied Dictionary 2:", copied_dict2)
```

```python
import copy

original_dict = {'a': 1, 'b': [2, 3], 'c': {'d': 4}}

deep_copied_dict = copy.deepcopy(original_dict)

print("Original Dictionary:", original_dict)
print("Deep Copied Dictionary:", deep_copied_dict)
```
