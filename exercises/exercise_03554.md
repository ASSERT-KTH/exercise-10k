## Exercise
Write a program that takes two lists as input, and returns a new list containing the elements that are in the first list but not in the second.

## Reference Solution
```python
def find_uniq(list1, list2):
    return [x for x in list1 if x not in list2]

# Test
list1 = [1, 2, 3, 4, 5]
list2 = [4, 5, 6, 7, 8]
print(find_uniq(list1, list2))
```
