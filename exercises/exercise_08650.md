## Exercise
Write a function that finds the intersection of two given lists.

## Reference Solution
```python
def list_intersection(list1, list2):
    """
    Returns a new list containing the intersection of two input lists.

    :param list1: The first input list.
    :param list2: The second input list.
    :return: A new list containing the common elements.
    """
    return [value for value in list1 if value in list2]

# Example usage
if __name__ == "__main__":

    list2 = [4, 5, 6, 7, 8]
    print("List 1:", list1)
    print("List 2:", list2)
    print("Intersection:", list_intersection(list1, list2))
```
