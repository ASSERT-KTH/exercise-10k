## Exercise
Write a program to find the union of two or more arrays.

## Reference Solution
```python
def array_union(*arrays):
    """
    Calculate the union of two or more arrays.
    
    Args:
        *arrays: Variable number of lists.
    
    Returns:
        A list containing the union of all input lists.
    """
    return list(set().union(*arrays))

# Example usage
if __name__ == "__main__":
    # Define example arrays
    array1 = [1, 2, 3, 4]
    array2 = [3, 4, 5, 6]
    array3 = [5, 6, 7, 8]

    # Calculate the union of the arrays
    union = array_union(array1, array2, array3)
    print("Union:", union)
```
