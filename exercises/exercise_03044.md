## Exercise
Write a function that takes a list of numbers as input and returns the maximum value.

## Reference Solution
```python
def find_max_value(numbers):
    """
    Returns the maximum value in a list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        float: The maximum value in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not numbers:
        raise ValueError("Input list is empty")
    return max(numbers)

# Example usage:
def main():
    numbers = [1, 2, 3, 4, 5]
    try:
        max_value = find_max_value(numbers)
        print(f"Maximum value: {max_value}")
    except ValueError as e:
        print(f"Error: {e}")

if __name__ == "__main__":
    main()
```
