## Exercise
Write a program that finds the maximum subarray sum in a list using a for loop.

## Reference Solution
```python
def max_subarray_sum(lst):
    """
    Find the maximum subarray sum in a list.

    Args:
        lst (list): A list of integers.

    Returns:
        int: The maximum subarray sum.
    """
    max_sum = float('-inf')  # Initialize max_sum to negative infinity
    current_sum = 0  # Initialize current_sum to 0

    for num in lst:
        current_sum = max(num, current_sum + num)  # Update current_sum
        max_sum = max(max_sum, current_sum)  # Update max_sum

    return max_sum

# Example usage:
numbers = [-2, -3, 4, -1, -2, 1, 5, -3]
max_sum = max_subarray_sum(numbers)
print("Maximum subarray sum:", max_sum)
```
