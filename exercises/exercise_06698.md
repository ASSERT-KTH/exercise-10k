## Exercise
Create a program that takes a number as input and returns its absolute value.

## Reference Solution
```python
def get_absolute_value(num):
    """
    Returns the absolute value of a given number.
    
    Parameters:
    num (int or float): The input number.
    
    Returns:
    int or float: The absolute value of the input number.
    """
    return abs(num)

def main():
    # Get user input
    num = float(input("Enter a number: "))
    
    # Calculate and display the absolute value
    absolute_value = get_absolute_value(num)
    print(f"The absolute value of {num} is {absolute_value}")

if __name__ == "__main__":
    main()
```
