## Exercise
Write a function to find the symmetric difference of two lists.

## Reference Solution
```python
def symmetric_difference(list1, list2):
    """
    Returns the symmetric difference of two lists.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: The symmetric difference of list1 and list2.
    """
    return list(set(list1) ^ set(list2))

# Example usage:
if __name__ == "__main__":
    list1 = [1, 2, 3, 4]
    list2 = [3, 4, 5, 6]
    print(symmetric_difference(list1, list2))  # Output: [1, 2, 5, 6]
```

def symmetric_difference_ordered(list1, list2):
    """
    Returns the symmetric difference of two lists, preserving the original order.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: The symmetric difference of list1 and list2.
    """
    set1 = set(list1)
    set2 = set(list2)
    result = [x for x in list1 if x not in set2] + [x for x in list2 if x not in set1]
    return result

# Example usage:
if __name__ == "__main__":
