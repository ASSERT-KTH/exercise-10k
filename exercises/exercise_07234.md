## Exercise
Write a program that checks if a given string is a valid password and then formats it according to specific rules. A valid password must be at least 8 characters long, contain at least one lowercase letter, one uppercase letter, and one digit. If the password is valid, format it by converting the first letter to uppercase and the rest to lowercase, then append ' is valid' to the end.

## Reference Solution
```python
import re

def validPassword(password):
    # Check if password is at least 8 characters long
    if len(password) < 8:
        return False
    
    # Check if password contains at least one lowercase letter
    if not re.search("[a-z]", password):
        return False
    
    # Check if password contains at least one uppercase letter
    if not re.search("[A-Z]", password):
        return False
    
    # Check if password contains at least one digit
    if not re.search("[0-9]", password):
        return False
    
    # If all conditions are met, return True
    return True

def formatPassword(password):
    # Convert first letter to uppercase and rest to lowercase
    formatted_password = password[0].upper() + password[1:].lower()
    
    # Append ' is valid' to the end
    formatted_password += ' is valid'
    
    return formatted_password

password = "Password123"
if validPassword(password):
    print(formatPassword(password))
else:
    print("Invalid password")
```
