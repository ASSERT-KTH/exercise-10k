## Exercise
Given an array of integers, write a program to find the maximum contiguous subarray. The maximum contiguous subarray is the subarray with the largest sum of its elements. Your program should return the sum of the maximum contiguous subarray.

## Reference Solution
```python
class Solution:
    def maxSubArray(self, nums):
        max_sum = float('-inf')
        current_sum = 0

        for num in nums:
            current_sum = max(num, current_sum + num)
            max_sum = max(max_sum, current_sum)

        return max_sum
```
