## Exercise
Write a program that generates the Fibonacci sequence up to the 10th number using a while loop.

## Reference Solution
```python
<?php
$a = 0;
$b = 1;

while ($a < 100) {
    echo $a . " ";
    $b = $a + $b;
    $a = $b - $a;
}
?>
```
