## Exercise
Write a program to calculate the absolute value of a given number using if/else statements

## Reference Solution
```python
def calculate_absolute_value(num):
    """
    This function calculates the absolute value of a given number.
    
    Args:
        num (int): The number for which the absolute value is to be calculated.
    
    Returns:
        int: The absolute value of the given number.
    """
    if num < 0:
        # If the number is negative, multiply it by -1 to make it positive
        return num * -1
    else:
        # If the number is non-negative, return the number as it is
        return num


# Test the function
num = int(input("Enter a number: "))
print("The absolute value of", num, "is:", calculate_absolute_value(num))
```
