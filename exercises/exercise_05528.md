## Exercise
Write a program to demonstrate the scope of variables in different blocks of code.

## Reference Solution
```python
# Global variable
global_variable = "I am global"

def outer_function():
    # Enclosing variable
    enclosing_variable = "I am enclosing"

    def inner_function():
        # Local variable
        local_variable = "I am local"

        print("Inside inner_function:")
        print(f"Global variable: {global_variable}")
        print(f"Enclosing variable: {enclosing_variable}")
        print(f"Local variable: {local_variable}")

    inner_function()

    print("\nInside outer_function:")
    print(f"Global variable: {global_variable}")
    print(f"Enclosing variable: {enclosing_variable}")
    # print(f"Local variable: {local_variable}")  # Uncomment to see NameError

outer_function()

print("\nInside main block:")
print(f"Global variable: {global_variable}")
# print(f"Enclosing variable: {enclosing_variable}")  # Uncomment to see NameError
# print(f"Local variable: {local_variable}")  # Uncomment to see NameError
```
