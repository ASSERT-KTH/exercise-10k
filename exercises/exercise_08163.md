## Exercise
Create an array of 5 integers and calculate the average of all elements.

## Reference Solution
```python
def calculate_average(array):
    """
    Calculate the average of all elements in the array.

    Args:
        array (list): A list of integers.

    Returns:
        float: The average of all elements in the array.
    """
    return sum(array) / len(array)


def main():
    # Create an array of 5 integers
    array = [10, 20, 30, 40, 50]

    # Calculate the average of all elements
    average = calculate_average(array)

    print("Array:", array)
    print("Average:", average)


if __name__ == "__main__":
    main()
```
