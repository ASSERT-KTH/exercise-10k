## Exercise
Write a program that takes a number as input and returns its rounded value.

## Reference Solution
```python
def round_number(num):
    """
    Rounds a given number to the nearest integer.

    Args:
        num (float): The number to be rounded.

    Returns:
        int: The rounded number.
    """
    return round(num)

def main():
    # Get user input
    num = float(input("Enter a number: "))

    # Round the number
    rounded_num = round_number(num)

    # Print the result
    print("The rounded number is:", rounded_num)

if __name__ == "__main__":
    main()
```
