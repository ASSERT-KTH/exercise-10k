## Exercise
Write a program that prints the multiplication table for a given number.

## Reference Solution
```python
def print_multiplication_table(number, limit=10):
    """
    Prints the multiplication table for a given number up to a specified limit.

    Args:
        number (int): The number for which to print the multiplication table.
        limit (int, optional): The maximum multiplier to include in the table. Defaults to 10.
    """
    print(f"Multiplication Table for {number}:")
    for i in range(1, limit + 1):
        print(f"{number} x {i} = {number * i}")

def main():
    number = int(input("Enter a number: "))
    limit = int(input("Enter the limit (default is 10): ") or 10)
    print_multiplication_table(number, limit)

if __name__ == "__main__":
    main()
```
