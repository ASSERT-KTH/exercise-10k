## Exercise
Given a list of integers, find the first duplicate element

## Reference Solution
```python
def find_first_duplicate(nums):
    """
    Find the first duplicate element in a list of integers.

    Args:
        nums (list): A list of integers.

    Returns:
        int: The first duplicate element, or None if no duplicates are found.
    """
    seen = {}
    for num in nums:
        if num in seen:
            return num
        seen[num] = True
    return None

# Example usage:
numbers = [2, 1, 3, 5, 3, 2]
print(find_first_duplicate(numbers))  # Output: 3
```
