## Exercise
Write a program that takes in a variable and its value as input, and then determines the data type of the variable. The program should handle at least 5 different data types, including integers, floats, strings, booleans, and lists. The program should output the determined data type and provide feedback if the input is invalid.

## Reference Solution
```python
# Author: [Your Name]
# Date: [Today's Date]
# Description: A program that determines the data type of a variable.

# Importing Modules
import sys

# Main Function
def main():
    # Variable Declaration
    variable = input("Enter the variable: ")
    value = input("Enter the value: ")

    # Checking for Integer
    if value.isdigit():
        print(f"The data type of {variable} is integer.")
    # Checking for Float
    elif value.replace('.', '', 1).isdigit():
        print(f"The data type of {variable} is float.")
    # Checking for String
    elif value.startswith('"') and value.endswith('"'):
        print(f"The data type of {variable} is string.")
    # Checking for Boolean
    elif value.lower() == 'true' or value.lower() == 'false':
        print(f"The data type of {variable} is boolean.")
    # Checking for List
    elif value.startswith('[') and value.endswith(']'):
        print(f"The data type of {variable} is list.")
    # Error Handling
    else:
        print(f"Invalid input. The value '{value}' does not match any of the supported data types.")

# Execution
if __name__ == "__main__":
    main()
```
