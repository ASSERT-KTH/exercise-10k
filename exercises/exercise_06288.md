## Exercise
Update a dictionary with new key-value pairs using the update() method

## Reference Solution
```python
def update_dictionary():
    # Create an initial dictionary
    initial_dict = {
        "name": "John",
        "age": 30,
        "city": "New York"
    }

    # New key-value pairs to update
    new_pairs = {
        "country": "USA",
        "job": "Software Engineer"
    }

    # Update the dictionary with new key-value pairs
    initial_dict.update(new_pairs)

    return initial_dict

# Main function
def main():
    updated_dict = update_dictionary()
    print("Updated Dictionary:", updated_dict)

if __name__ == "__main__":
    main()
```
