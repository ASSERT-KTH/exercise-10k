## Exercise
Write a function to retrieve a list of all keys in a dictionary.

## Reference Solution
```python
def get_dictionary_keys(dictionary):
    """
    Retrieves a list of all keys in a dictionary.

    Args:
        dictionary (dict): The dictionary to retrieve keys from.

    Returns:
        list: A list of dictionary keys.
    """
    return list(dictionary.keys())

# Example usage:
def main():
    example_dict = {"name": "John", "age": 30, "city": "New York"}
    keys = get_dictionary_keys(example_dict)
    print(keys)

if __name__ == "__main__":
    main()
```
