## Exercise
Write a program to find the floor and ceiling of a given number.

## Reference Solution
```python
import math

def calculate_floor_and_ceiling(num):
    """
    Calculate the floor and ceiling of a given number.

    Args:
        num (float): The input number.

    Returns:
        tuple: A tuple containing the floor and ceiling of the input number.
    """
    floor_val = math.floor(num)
    ceiling_val = math.ceil(num)

    return floor_val, ceiling_val

def main():
    num = float(input("Enter a number: "))
    floor_val, ceiling_val = calculate_floor_and_ceiling(num)

    print(f"Floor of {num}: {floor_val}")
    print(f"Ceiling of {num}: {ceiling_val}")

if __name__ == "__main__":
    main()
```
