## Exercise
Create a function to calculate the length of a given string.

## Reference Solution
```python
def calculate_string_length(input_string):
    """
    This function calculates the length of a given string.

    Args:
        input_string (str): The input string to calculate the length for.

    Returns:
        int: The length of the input string.
    """

    # First, we check if the input is indeed a string. If not, we raise a TypeError.
    if not isinstance(input_string, str):
        raise TypeError("Input must be a string.")

    # Now, we use the built-in len() function in Python, which returns the number of items in an object.
    # For strings, this means the number of characters.
    # We store this value in the 'length' variable.
    length = len(input_string)

    # Finally, we return the calculated length.
    return length

# Test the function
print(calculate_string_length("Hello, World!"))
```
