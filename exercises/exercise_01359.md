## Exercise
Write a program to demonstrate the use of command line arguments

## Reference Solution
```python
import sys

def main():
    # Check if the correct number of arguments are provided
    if len(sys.argv) != 3:
        print("Usage: python program.py <name> <age>")
        sys.exit(1)

    # Extract the name and age from the command line arguments
    name = sys.argv[1]
    age = sys.argv[2]

    # Print out a greeting message
    print(f"Hello, {name}! You are {age} years old.")

if __name__ == "__main__":
    main()
```
