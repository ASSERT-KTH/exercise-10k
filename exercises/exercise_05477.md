## Exercise
Write a program that takes a list and a function as input and applies the function to the first two elements of the list, then to the result and the next element, and so on.

## Reference Solution
```python
def foldr(op, l):
    if len(l) == 0:
        raise ValueError("Input list cannot be empty")
    elif len(l) == 1:
        return l[0]
    else:
        return op(foldr(op, l[1:]), l[0])

def add(x, y):
    return x + y

# Test the code
lst = [1, 2, 3, 4, 5]
result = foldr(add, lst)
print(result)  # Output: 15
```
