## Exercise
Write a program that takes a sentence as input from the user and prints it in sentence case (first letter capitalized, rest lowercase).

## Reference Solution
```python
def convert_to_sentence_case(sentence):
    """
    Converts a sentence to sentence case (first letter capitalized, rest lowercase).
    
    Args:
        sentence (str): The input sentence.
    
    Returns:
        str: The sentence in sentence case.
    """
    return sentence.capitalize()

def main():
    # Get user input
    sentence = input("Please enter a sentence: ")
    
    # Convert to sentence case
    sentence_case = convert_to_sentence_case(sentence)
    
    # Print the result
    print("Sentence in sentence case: ", sentence_case)

if __name__ == "__main__":
    main()
```
