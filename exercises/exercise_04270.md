## Exercise
Write a function that takes a dictionary and a key as input and returns the corresponding value using the get() method

## Reference Solution
```python
def get_dict_value(dictionary, key):
    """
    Retrieves the value associated with a given key from a dictionary.

    Args:
        dictionary (dict): The dictionary to search in.
        key (any): The key to search for.

    Returns:
        any: The value associated with the key, or None if not found.
    """
    return dictionary.get(key)

# Example usage:
example_dict = {"name": "John", "age": 30}
print(get_dict_value(example_dict, "name"))  # Output: John
print(get_dict_value(example_dict, "age"))   # Output: 30
print(get_dict_value(example_dict, "city"))  # Output: None
```
