## Exercise
Create a dictionary with 3 key-value pairs and clear the dictionary.

## Reference Solution
```python
# Create a dictionary with 3 key-value pairs
my_dict = {
    "key1": "value1",
    "key2": "value2",
    "key3": "value3"
}

# Print the created dictionary
print("Created Dictionary:", my_dict)

# Clear the dictionary
my_dict.clear()

# Print the dictionary after clearance
print("Dictionary after clearance:", my_dict)
```
