## Exercise
Write a function that takes in a list and returns True if it contains any duplicates, False otherwise.

## Reference Solution
```python
def has_duplicates(lst):
    """
    Checks if a list contains any duplicate elements.

    Args:
        lst (list): The input list to check for duplicates.

    Returns:
        bool: True if the list contains duplicates, False otherwise.
    """
    return len(lst) != len(set(lst))

# Example usage
if __name__ == "__main__":
    print(has_duplicates([1, 2, 3, 4, 5]))  # False
    print(has_duplicates([1, 2, 2, 4, 5]))  # True
    print(has_duplicates(["a", "b", "c", "d"]))  # False
    print(has_duplicates(["a", "b", "b", "d"]))  # True
```
