## Exercise
Write a program to validate user input to ensure it is a positive integer.

## Reference Solution
```python
def validate_positive_integer(input_value):
    """
    Validate if the input is a positive integer.

    Args:
        input_value (str): The user input to be validated.

    Returns:
        bool: True if the input is a positive integer, False otherwise.
    """
    try:
        integer_value = int(input_value)
        return integer_value > 0
    except ValueError:
        return False

def main():
    while True:
        user_input = input("Please enter a positive integer (or 'quit' to exit): ")
        if user_input.lower() == 'quit':
            break
        if validate_positive_integer(user_input):
            print(f"{user_input} is a valid positive integer.")
        else:
            print(f"{user_input} is not a valid positive integer. Please try again.")

if __name__ == "__main__":
    main()
```
