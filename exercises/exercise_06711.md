## Exercise
Write a program that takes a user's name as input and prints out a greeting message

## Reference Solution
```python
def print_greeting(name: str) -> None:
    """
    Prints a greeting message to the user.

    Args:
        name (str): The user's name.
    """
    print(f"Hello, {name}!")

def main() -> None:
    """
    Gets the user's name and prints a greeting message.
    """
    name = input("Please enter your name: ")
    print_greeting(name)

if __name__ == "__main__":
    main()
```
