## Exercise
Write a program to reduce an array to a single value using a reduction function.

## Reference Solution
```python
from functools import reduce

def reduction_function(x, y):
    """
    Reduction function that takes two elements and returns their sum.
    
    Args:
    x (int): The first element.
    y (int): The second element.
    
    Returns:
    int: The sum of x and y.
    """
    return x + y

def reduce_array(array):
    """
    Reduces an array to a single value using a reduction function.
    
    Args:
    array (list): The input array.
    
    Returns:
    int: The reduced value.
    """
    if not array:
        return None
    return reduce(reduction_function, array)

# Example usage
array = [1, 2, 3, 4, 5]
reduced_value = reduce_array(array)
print(f"Reduced value: {reduced_value}")
```
