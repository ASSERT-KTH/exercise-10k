## Exercise
Write a program that uses a for loop to count the number of vowels in a given string.

## Reference Solution
```python
def count_vowels(input_string):
    """
    Counts the number of vowels in a given string.

    Args:
        input_string (str): The input string to count vowels in.

    Returns:
        int: The number of vowels in the input string.
    """
    vowels = 'aeiouAEIOU'
    count = 0
    for char in input_string:
        if char in vowels:
            count += 1
    return count

def main():
    input_string = input("Enter a string: ")
    vowel_count = count_vowels(input_string)
    print(f"The string '{input_string}' contains {vowel_count} vowels.")

if __name__ == "__main__":
    main()
```
