## Exercise
Return a list of all key-value pairs in a dictionary

## Reference Solution
```python
def get_key_value_pairs(dictionary):
    """
    Returns a list of all key-value pairs in a dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        list: A list of tuples, where each tuple is a key-value pair.
    """
    return list(dictionary.items())

# Example usage:
my_dict = {"name": "John", "age": 30, "city": "New York"}
pairs = get_key_value_pairs(my_dict)
print(pairs)
```
