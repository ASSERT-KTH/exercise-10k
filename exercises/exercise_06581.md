## Exercise
Write a program to find the maximum number from a given list of numbers.

## Reference Solution
```python
def find_max_number(numbers):
    """
    Finds the maximum number from a given list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        int: The maximum number in the list.
    """
    if not numbers:
        raise ValueError("Input list is empty")

    return max(numbers)

# Example usage
if __name__ == "__main__":
    numbers = [12, 45, 7, 23, 56, 89, 34]
    max_number = find_max_number(numbers)
    print(f"Maximum number: {max_number}")
```
