## Exercise
Write a program that converts a float to an integer.

## Reference Solution
```python
def convert_float_to_int(float_num):
    """
    Converts a float to an integer by truncating the decimal part.

    Args:
        float_num (float): The floating-point number to convert.

    Returns:
        int: The integer equivalent of the input float.
    """
    return int(float_num)

def main():
    # Example usage
    float_num = 3.14159
    int_num = convert_float_to_int(float_num)
    print(f"Float: {float_num}, Integer: {int_num}")

if __name__ == "__main__":
    main()
```
