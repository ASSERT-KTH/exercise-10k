## Exercise
Write a program to swap the values of two variables without using a temporary variable.

## Reference Solution
```python
def swap_variables(a, b):
    """
    Swap the values of two variables without using a temporary variable.

    Args:
        a (any): The first variable.
        b (any): The second variable.

    Returns:
        tuple: A tuple containing the swapped values.
    """
    a, b = b, a
    return a, b

# Example usage:
if __name__ == "__main__":
    var1 = 5
    var2 = 10

    print("Before swap:")
    print(f"var1 = {var1}")
    print(f"var2 = {var2}")

    var1, var2 = swap_variables(var1, var2)

    print("\nAfter swap:")
    print(f"var1 = {var1}")
    print(f"var2 = {var2}")
```
