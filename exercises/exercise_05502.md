## Exercise
Write a program to find the maximum value in an array of integers.

## Reference Solution
```python
def find_max_value(array):
    """
    This function finds the maximum value in an array of integers.

    Args:
        array (list): A list of integers.

    Returns:
        int: The maximum value in the array.

    Raises:
        ValueError: If the input array is empty.
    """
    if len(array) == 0:
        raise ValueError("Input array is empty")

    max_value = array[0]
    for value in array:
        if value > max_value:
            max_value = value

    return max_value


# Example usage:
def main():
    numbers = [12, 45, 7, 23, 56, 89, 34]
    print("Array of numbers:", numbers)
    print("Maximum value:", find_max_value(numbers))


if __name__ == "__main__":
    main()
```
