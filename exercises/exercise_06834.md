## Exercise
Write a program that takes user input for their name, age, and height, and then stores this information in variables of appropriate data types. Finally, print out the user's information in a formatted string.

## Reference Solution
```python
# Take user input for name, age, and height
name = input("Please enter your name: ")
age = int(input("Please enter your age: "))
height = float(input("Please enter your height in meters: "))

# Print out the user's information in a formatted string
print(f"Your name is {name}, you are {age} years old, and your height is {height} meters.")
```
