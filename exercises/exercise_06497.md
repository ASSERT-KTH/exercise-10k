## Exercise
Write a program that asks the user for a word and then prints out a scrambled version of the word.

## Reference Solution
```python
import random

def scramble_word(word):
    """Scrambles the input word"""
    word_list = list(word)
    random.shuffle(word_list)
    return ''.join(word_list)

def main():
    word = input("Please enter a word: ")
    scrambled_word = scramble_word(word)
    print("Scrambled word: ", scrambled_word)

if __name__ == "__main__":
    main()
```
