## Exercise
Write a program that takes an input and validates it using a try-except block.

## Reference Solution
```python
def get_valid_input():
    """
    Get a valid integer input from the user.
    """
    while True:
        try:
            # Attempt to convert the input to an integer
            user_input = int(input("Please enter an integer: "))
            return user_input
        except ValueError:
            # If the input is not an integer, display an error message
            print("Invalid input. Please enter a valid integer.")

def main():
    # Call the function to get a valid input
    valid_input = get_valid_input()
    print(f"Valid input: {valid_input}")

if __name__ == "__main__":
    main()
```
