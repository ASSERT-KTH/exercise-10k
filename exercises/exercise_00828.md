## Exercise
Given a string, find all duplicate characters and return them in a list.

## Reference Solution
```python
def find_duplicate_chars(input_str):
    """
    Finds all duplicate characters in a given string and returns them in a list.

    Args:
        input_str (str): The input string to find duplicate characters in.

    Returns:
        list: A list of duplicate characters in the input string.
    """
    char_count = {}
    duplicate_chars = []

    # Count the occurrences of each character
    for char in input_str:
        if char in char_count:
            char_count[char] += 1
        else:
            char_count[char] = 1

    # Find the characters that appear more than once
    for char, count in char_count.items():
        if count > 1 and char not in duplicate_chars:
            duplicate_chars.append(char)

    return duplicate_chars

# Example usage:
input_str = "programming"
print(find_duplicate_chars(input_str))  # Output: ['g', 'm']
```
