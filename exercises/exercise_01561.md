## Exercise
Write a function that takes a single number as input and returns its double.

## Reference Solution
```python
def double_number(num: float) -> float:
    """
    Returns the double of the input number.

    Args:
        num (float): The number to be doubled.

    Returns:
        float: The double of the input number.
    """
    return num * 2

# Example usage:
def main():
    num = 5.0
    doubled = double_number(num)
    print(f"The double of {num} is {doubled}")

if __name__ == "__main__":
    main()
```
