## Exercise
Write a function to group a dictionary by its values and return a new dictionary containing the grouped key-value pairs.

## Reference Solution
```python
def group_dict_by_values(input_dict):
    """
    Group a dictionary by its values and return a new dictionary containing the grouped key-value pairs.

    Args:
        input_dict (dict): The input dictionary to be grouped.

    Returns:
        dict: A new dictionary with the grouped key-value pairs.
    """
    # Initialize an empty dictionary to store the grouped key-value pairs
    grouped_dict = {}

    # Iterate over each key-value pair in the input dictionary
    for key, value in input_dict.items():
        # If the value is not already a key in the grouped dictionary, add it with an empty list as its value
        if value not in grouped_dict:
            grouped_dict[value] = []
        # Append the key to the list of keys for the current value
        grouped_dict[value].append(key)

    return grouped_dict

    # Group the input dictionary by its values
    grouped_dict = group_dict_by_values(input_dict)
```
