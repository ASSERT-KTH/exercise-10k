## Exercise
Create a function that takes two numbers as arguments, with default values, and returns their sum

## Reference Solution
```python
def calculate_sum(a: float = 0, b: float = 0) -> float:
    """
    This function calculates the sum of two numbers.

    Args:
        a (float): The first number. Defaults to 0.
        b (float): The second number. Defaults to 0.

    Returns:
        float: The sum of the two numbers.
    """
    return a + b

# Example usage
if __name__ == "__main__":
    print(calculate_sum(5, 7))  # Output: 12
    print(calculate_sum(10))    # Output: 10
    print(calculate_sum())      # Output: 0
```
