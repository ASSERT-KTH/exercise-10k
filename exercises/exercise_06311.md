## Exercise
Filter a dictionary to include only key-value pairs where the value meets a certain condition

## Reference Solution
```python
def filter_dict(input_dict, condition_func):
    """
    Filter a dictionary to include only key-value pairs where the value meets a certain condition.

    Args:
        input_dict (dict): The dictionary to be filtered.
        condition_func (function): A function that takes a value and returns True if the value meets the condition.

    Returns:
        dict: A new dictionary with only the key-value pairs where the value meets the condition.
    """
    return {key: value for key, value in input_dict.items() if condition_func(value)}


# Example usage:
numbers = {'a': 1, 'b': 2, 'c': 3, 'd': 4, 'e': 5}

# Filter numbers greater than 3
filtered_numbers = filter_dict(numbers, lambda x: x > 3)
print(filtered_numbers)  # Output: {'d': 4, 'e': 5}

# Filter even numbers
filtered_numbers = filter_dict(numbers, lambda x: x % 2 == 0)
print(filtered_numbers)  # Output: {'b': 2, 'd': 4}
```
