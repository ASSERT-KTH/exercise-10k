## Exercise
Write a function that takes a variable number of arguments and returns their sum

## Reference Solution
```python
def variable_sum(*args):
    """
    Returns the sum of a variable number of arguments.

    Args:
        *args: A variable number of arguments to be summed.

    Returns:
        The sum of the input arguments.
    """
    return sum(args)

# Example usage
if __name__ == "__main__":
    print(variable_sum(1, 2, 3, 4, 5))  # Output: 15
    print(variable_sum(10, 20))  # Output: 30
    print(variable_sum(1.5, 2.5, 3.5))  # Output: 7.5
```
