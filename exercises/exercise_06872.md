## Exercise
Write a program that calculates the sum of all even numbers from 1 to a given number using a for loop or a while loop.

## Reference Solution
```python
i = 1
even_sum = 0
num = int(input("Enter a number: "))

while i <= num:
    if i % 2 == 0:
        even_sum += i
    i += 1

print(even_sum)
```
