## Exercise
Create a program that checks if a given string is a palindrome. If the string is equal to its reverse, print 'Palindrome', otherwise print 'Not a palindrome'.

## Reference Solution
```python
def is_palindrome(s: str) -> None:
    """
    Checks if a given string is a palindrome.

    Args:
    s (str): The input string.

    Returns:
    None
    """
    # Remove non-alphanumeric characters and convert to lowercase
    cleaned_s = ''.join(c for c in s if c.isalnum()).lower()
    
    # Compare the cleaned string with its reverse
    if cleaned_s == cleaned_s[::-1]:
        print('Palindrome')
    else:
        print('Not a palindrome')

def main() -> None:
    # Example usage
    s = input("Enter a string: ")
    is_palindrome(s)

if __name__ == "__main__":
    main()
```
