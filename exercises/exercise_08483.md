## Exercise
Write a program that asks the user for their age, height, and name. Store this information in variables of appropriate data types. Then, print out the user's information along with the data type of each variable.

## Reference Solution
```python
# User input and variable declaration
nome = input("Digite seu nome: ")
idade = int(input("Digite sua idade: "))
altura = float(input("Digite sua altura (em metros): "))

# Printing user information
print(f"nome: {nome} ({type(nome).__name__})")
print(f"idade: {idade} ({type(idade).__name__})")
print(f"altura: {altura} ({type(altura).__name__})")
```
