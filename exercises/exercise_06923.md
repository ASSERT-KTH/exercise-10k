## Exercise
Create a function that takes two numbers as arguments and returns their sum

## Reference Solution

def add_numbers(a, b):
    """
    Returns the sum of two numbers.

    Args:
        a (int or float): The first number.
        b (int or float): The second number.

    Returns:
        int or float: The sum of a and b.
    """
    return a + b

# Example usage:
