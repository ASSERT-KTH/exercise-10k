## Exercise
Write a program to calculate the average of all elements in a given array.

## Reference Solution
```python
def calculate_average(array):
    """
    Calculate the average of all elements in the given array.

    Args:
        array (list): A list of numbers.

    Returns:
        float: The average of the array elements.
    """
    if not array:
        raise ValueError("Array cannot be empty")

    return sum(array) / len(array)


def main():
    # Example usage
    array = [1, 2, 3, 4, 5]
    try:
        average = calculate_average(array)
        print(f"The average of the array elements is: {average}")
    except ValueError as e:
        print(e)


if __name__ == "__main__":
    main()
```
