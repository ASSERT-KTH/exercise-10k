## Exercise
Create a program that uses functions to perform basic arithmetic operations. The program should take two numbers and an operation as input and return the result. The operations to be supported are addition, subtraction, multiplication, and division.

## Reference Solution
```python
# Define functions for each operation
def addition(x, y):
    return x + y

def subtraction(x, y):
    return x - y

def multiplication(x, y):
    return x * y

def division(x, y):
    return x / y

# Main function
def main():
    print("Basic Arithmetic Operations")
    print("--------------------------------")

    # Get user input
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))
    operation = input("Enter the operation (+, -, *, /): ")

    # Perform the operation
    if operation == "+":
        result = addition(num1, num2)
    elif operation == "-":
        result = subtraction(num1, num2)
    elif operation == "*":
        result = multiplication(num1, num2)
    elif operation == "/":
        if num2 != 0:
            result = division(num1, num2)
        else:
            result = "Error: Division by zero"
    else:
        result = "Error: Invalid operation"

    # Print the result
    print(f"{num1} {operation} {num2} = {result}")

    # Loop back to the main function
    main()

# Call the main function
main()
```
