## Exercise
Write a function to find the maximum element in an array of integers. The function should take an array as input and return the maximum element. If the array is empty, the function should return a message indicating that the array is empty.

## Reference Solution
```python
def find_max_element(array):
    """
    This function finds the maximum element in an array of integers.
    
    Args:
        array (list): A list of integers.
    
    Returns:
        int: The maximum element in the array. If the array is empty, it returns a message indicating that the array is empty.
    """
    
    # First, we check if the array is empty
    if not array:
        
        # If the array is empty, we return a message indicating that the array is empty
        return "The array is empty"
    
    # If the array is not empty, we use the built-in max function to find the maximum element
    # The max function returns the largest item in an iterable or the largest of two or more arguments
    max_element = max(array)
    
    # Finally, we return the maximum element
    return max_element

# Example usage:
print(find_max_element([1, 2, 3, 4, 5]))  # Output: 5
print(find_max_element([]))  # Output: The array is empty
```
