## Exercise
Write a program to remove duplicate words from a given string.

## Reference Solution
```python
def remove_duplicate_words(input_string):
    """
    Removes duplicate words from a given string.

    Args:
        input_string (str): The input string to remove duplicates from.

    Returns:
        str: The input string with duplicate words removed.
    """
    # Split the input string into a list of words
    words = input_string.split()

    # Use a dictionary to keep track of unique words
    unique_words = {}
    for word in words:
        # Convert the word to lowercase to handle case-insensitive duplicates
        word_lower = word.lower()
        unique_words[word_lower] = word

    # Join the unique words back into a string
    output_string = ' '.join(unique_words.values())

    return output_string

# Example usage
input_string = "Hello world this is a test this test is only a test"
print("Input String:", input_string)
print("Output String:", remove_duplicate_words(input_string))
```
