## Exercise
Write a program to print a number pattern using a for loop.

## Reference Solution
```python
def print_number_pattern(n):
    """
    Prints a number pattern with n rows.

    Args:
        n (int): The number of rows in the pattern.
    """
    num = 1
    for i in range(1, n + 1):
        for j in range(1, i + 1):
            print(num, end=" ")
            num += 1
        print()

def main():
    n = 5  # Change this to print a different number of rows
    print_number_pattern(n)

if __name__ == "__main__":
    main()
```
