## Exercise
Write a function that takes a dictionary as input and removes and returns the last inserted key-value pair

## Reference Solution
```python
def remove_last_inserted_pair(dictionary):
    """
    Removes and returns the last inserted key-value pair from a dictionary.
    
    Args:
        dictionary (dict): The input dictionary.
    
    Returns:
        tuple: A tuple containing the last inserted key-value pair. 
               If the dictionary is empty, returns None.
    """
    if not dictionary:
        return None
    
    # Get the last inserted key (which is the last key in the dictionary)
    last_key = list(dictionary.keys())[-1]
    
    # Get the corresponding value
    last_value = dictionary[last_key]
    
    # Remove the last inserted key-value pair from the dictionary
    del dictionary[last_key]
    
    # Return the removed key-value pair
    return last_key, last_value

# Example usage:
my_dict = {'a': 1, 'b': 2, 'c': 3}
print("Original Dictionary:", my_dict)

removed_pair = remove_last_inserted_pair(my_dict)
print("Removed Key-Value Pair:", removed_pair)
print("Updated Dictionary:", my_dict)
```
