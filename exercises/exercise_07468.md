## Exercise
Write a program that converts a given set into a list

## Reference Solution
```python
def convert_set_to_list(input_set):
    """
    Converts a given set into a list.

    Args:
        input_set (set): The set to be converted.

    Returns:
        list: A list containing all elements from the input set.
    """
    return list(input_set)


# Example usage:
if __name__ == "__main__":
    # Create a sample set
    sample_set = {"apple", "banana", "cherry"}

    # Convert the set to a list
    result_list = convert_set_to_list(sample_set)

    # Print the result
    print("Original Set:", sample_set)
    print("Converted List:", result_list)
```
