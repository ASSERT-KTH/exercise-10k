## Exercise
Write a program that takes a string and two integers as input, and returns the substring starting at the first integer and ending at the second integer.

## Reference Solution
```python
public class Main {
    public static String substring(String s, int start, int end) {
        StringBuilder sb = new StringBuilder();
        for (int i = start; i < end; i++) {
            sb.append(s.charAt(i));
        }
        return sb.toString();
    }

    public static void main(String[] args) {
        System.out.println(substring("Hello", 1, 4));
    }
}
```
