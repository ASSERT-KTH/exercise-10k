## Exercise
Write a program that takes a list and a target element as input and returns the index of the target element if it exists in the list.

## Reference Solution
```python
def find_index(list, target):
    for i, element in enumerate(list):
        if element == target:
            return i
    return -1

# Example usage
my_list = [1, 2, 3, 4, 5]
print(find_index(my_list, 3))  # Output: 2
```
