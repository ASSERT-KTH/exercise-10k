## Exercise
Write a program that takes a list as input and returns a new list containing the same elements in reverse order.

## Reference Solution
```python
def reverse_list(input_list):
    """
    This function takes a list as an input and returns a new list containing the same elements in reverse order.

    Args:
        input_list (list): The list to be reversed.

    Returns:
        list: A new list containing the same elements in reverse order.
    """
    # Create a new list to store the reversed elements
    new_list = []

    # Iterate over the input list in reverse order
    for element in reversed(input_list):
        # Append each element to the new list
        new_list.append(element)

    # Return the new list
    return new_list

# Example usage
input_list = [1, 2, 3, 4, 5]
output_list = reverse_list(input_list)
print(output_list)  # Output: [5, 4, 3, 2, 1]
```
