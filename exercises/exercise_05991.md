## Exercise
Create a function that takes a number as input and prints a square of asterisks of that size

## Reference Solution
```python
def print_asterisk_square(size):
    """
    Prints a square of asterisks of the given size.

    Args:
        size (int): The size of the square.
    """
    for _ in range(size):
        print('* ' * size)

# Example usage:
print_asterisk_square(5)
```
