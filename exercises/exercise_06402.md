## Exercise
Write a program that takes an integer as input and classifies it as positive, negative, or zero. Use if/else statements to determine the classification.

## Reference Solution
```python
# Read an integer from the user
n = int(input())

# Classify the integer
if n > 0:
    result = "Positive"
elif n < 0:
    result = "Negative"
else:
    result = "Zero"

# Print the result
print(result)
```
