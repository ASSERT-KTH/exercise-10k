## Exercise
Create a program that prints out a weather forecast based on the temperature. If the temperature is above 25, print 'Hot', between 15 and 25 print 'Mild', between 5 and 15 print 'Cool', and below 5 print 'Cold'.

## Reference Solution
```python
def get_weather_forecast(temperature):
    """
    Returns the weather forecast based on the temperature.

    Args:
        temperature (float): The temperature in degrees Celsius.

    Returns:
        str: The weather forecast.
    """
    if temperature > 25:
        return 'Hot'
    elif 15 <= temperature <= 25:
        return 'Mild'
    elif 5 <= temperature < 15:
        return 'Cool'
    else:
        return 'Cold'

def main():
    temperature = float(input("Enter the temperature in degrees Celsius: "))
    forecast = get_weather_forecast(temperature)
    print(f"The weather forecast is: {forecast}")

if __name__ == "__main__":
    main()
```
