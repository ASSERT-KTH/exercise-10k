## Exercise
Write a function that takes a list of numbers as input and returns the minimum value

## Reference Solution
```python
def find_minimum_value(numbers):
    """
    Returns the minimum value from a list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        The minimum value in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not numbers:
        raise ValueError("Input list is empty")
    return min(numbers)

# Example usage
numbers = [12, 45, 7, 23, 56, 89, 34]
print(find_minimum_value(numbers))  # Output: 7
```

def find_minimum_value_manual(numbers):
    """
    Returns the minimum value from a list of numbers (manual implementation).

    Args:
        numbers (list): A list of numbers.

    Returns:
        The minimum value in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not numbers:
        raise ValueError("Input list is empty")
    min_value = numbers[0]
    for num in numbers[1:]:
        if num < min_value:
            min_value = num
    return min_value

# Example usage
