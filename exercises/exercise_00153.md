## Exercise
Create a function that takes in a dictionary and returns a new dictionary containing the same key-value pairs, but with the keys normalized to a standard format

## Reference Solution
```python
def normalize_dict_keys(input_dict):
    """
    This function takes in a dictionary and returns a new dictionary containing the same key-value pairs,
    but with the keys normalized to a standard format (lowercase, no leading/trailing whitespaces, and no special characters).

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        dict: A new dictionary with normalized keys.
    """

    # Initialize an empty dictionary to store the normalized key-value pairs
    normalized_dict = {}

    # Iterate over each key-value pair in the input dictionary
    for key, value in input_dict.items():
        # Remove leading/trailing whitespaces from the key
        key = key.strip()
        
        # Convert the key to lowercase
        key = key.lower()
        
        # Replace special characters in the key with underscores
        key = ''.join(e if e.isalnum() else '_' for e in key)
        
        # Add the normalized key-value pair to the normalized dictionary
        normalized_dict[key] = value
    
    # Return the normalized dictionary
    return normalized_dict

# Example usage:
input_dict = {"Key1": "Value1", " Key2 ": "Value2", "Key3!": "Value3"}
normalized_dict = normalize_dict_keys(input_dict)
print(normalized_dict)  # Output: {'key1': 'Value1', 'key2': 'Value2', 'key3_': 'Value3'}
```
