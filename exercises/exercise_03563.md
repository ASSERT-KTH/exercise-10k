## Exercise
Write a function that returns all values from a given dictionary

## Reference Solution
```python
def get_dictionary_values(dictionary):
    """
    Returns a list of all values in the given dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        list: A list of dictionary values.
    """
    return list(dictionary.values())

# Example usage:
if __name__ == "__main__":
    example_dict = {"key1": "value1", "key2": 2, "key3": True}
    values = get_dictionary_values(example_dict)
    print(values)
```
