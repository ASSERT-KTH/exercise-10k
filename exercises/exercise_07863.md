## Exercise
Write a program that checks if a string contains a specific substring using conditional statements.

## Reference Solution
```python
def check_substring(main_string, sub_string):
    """
    Checks if a main string contains a specific substring.

    Args:
        main_string (str): The string to be checked.
        sub_string (str): The substring to be searched for.

    Returns:
        bool: True if the substring is found, False otherwise.
    """
    if sub_string in main_string:
        return True
    else:
        return False


# Example usage:
main_str = "Hello, World!"
sub_str = "World"
print(f"Is '{sub_str}' in '{main_str}'? {check_substring(main_str, sub_str)}")

main_str = "Hello, Universe!"
sub_str = "World"
print(f"Is '{sub_str}' in '{main_str}'? {check_substring(main_str, sub_str)}")
```
