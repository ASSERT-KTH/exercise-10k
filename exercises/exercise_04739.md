## Exercise
Create a function that uses a generator to produce a sequence of values on the fly.

## Reference Solution
```python
def fibonacci(n):
    """
    Generate the Fibonacci sequence up to the nth number.

    Args:
        n (int): The number of Fibonacci numbers to generate.

    Yields:
        int: The next Fibonacci number in the sequence.
    """
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

# Example usage:
if __name__ == "__main__":
    n = 10  # Generate the first 10 Fibonacci numbers
    print("Fibonacci sequence up to {} numbers:".format(n))
    for num in fibonacci(n):
        print(num)
```
