## Exercise
Write a function that takes a dictionary and a filter condition as input and returns a new dictionary containing only the key-value pairs that meet the filter condition.

## Reference Solution
```python
def filter_dict(input_dict, filter_condition):
    """
    This function filters a dictionary based on a given condition.

    Args:
        input_dict (dict): The dictionary to be filtered.
        filter_condition (function): A function that takes a key-value pair and returns True if the pair meets the condition, False otherwise.

    Returns:
        dict: A new dictionary containing only the key-value pairs that meet the filter condition.
    """

    # Initialize an empty dictionary to store the filtered key-value pairs
    filtered_dict = {}

    # Iterate over each key-value pair in the input dictionary
    for key, value in input_dict.items():
        
        # Check if the current key-value pair meets the filter condition
        if filter_condition(key, value):
            
            # If the condition is met, add the key-value pair to the filtered dictionary
            filtered_dict[key] = value
    
    # Return the filtered dictionary
    return filtered_dict

# Example usage:
# Define a filter condition function
def condition(key, value):
    return value > 10

# Create a sample dictionary
sample_dict = {'a': 5, 'b': 15, 'c': 20, 'd': 8}

# Use the filter_dict function
filtered_sample = filter_dict(sample_dict, condition)

# Print the result
print(filtered_sample)
```
