## Exercise
Write a function to remove a given key from a dictionary

## Reference Solution
```python
def remove_key(dictionary, key):
    """
    Removes a given key from a dictionary.

    Args:
        dictionary (dict): The input dictionary.
        key: The key to be removed.

    Returns:
        dict: A new dictionary with the specified key removed.
    """
    new_dict = dictionary.copy()  # Create a copy of the original dictionary
    if key in new_dict:
        del new_dict[key]  # Remove the key from the new dictionary
    return new_dict

# Example usage:
original_dict = {"a": 1, "b": 2, "c": 3}
key_to_remove = "b"

new_dict = remove_key(original_dict, key_to_remove)
print("Original Dictionary:", original_dict)
print("New Dictionary:", new_dict)
```
