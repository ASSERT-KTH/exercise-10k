## Exercise
Declare a character variable with the value '*'. Print a 5x5 square using the character.

## Reference Solution
```python
def print_square(size, char):
    """
    Prints a square of a given size using a specified character.

    Args:
        size (int): The size of the square.
        char (str): The character to use when printing the square.
    """
    for _ in range(size):
        print(char * size)


def main():
    # Declare a character variable with the value '*'
    char = '*'
    size = 5

    # Print a 5x5 square using the character
    print_square(size, char)


if __name__ == "__main__":
    main()
```
