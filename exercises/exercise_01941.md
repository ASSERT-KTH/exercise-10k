## Exercise
Write a program that takes an array and an index as input and returns the element at that index. If the index is out of range, return an error message.

## Reference Solution
```python
// array = [1, 2, 3, 4, 5, 6]
// index = 2

function getElement(array, index) {
  if (index >= 0 && index < array.length) {
    return array[index];
  } else {
    return "error";
  }
}
```
