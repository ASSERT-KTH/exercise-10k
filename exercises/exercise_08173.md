## Exercise
Create an array of 5 integers and check if any element meets a condition.

## Reference Solution
```python
def check_condition(array, condition):
    """
    Check if any element in the array meets the given condition.

    Args:
        array (list): The list of integers to check.
        condition (function): A function that takes an integer and returns a boolean.

    Returns:
        bool: True if any element meets the condition, False otherwise.
    """
    return any(condition(x) for x in array)


# Create an array of 5 integers
array = [5, 15, 3, 8, 12]

# Define the condition (in this case, x > 10)
condition = lambda x: x > 10

# Check if any element meets the condition
result = check_condition(array, condition)

print(f"Array: {array}")
print(f"Condition: x > 10")
print(f"Result: {result}")
```
