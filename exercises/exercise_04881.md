## Exercise
Create a dictionary with 5 key-value pairs, then print out the keys and values.

## Reference Solution
```python
# Create a dictionary with 5 key-value pairs
my_dict = {
    "name": "John",
    "age": 30,
    "city": "New York",
    "country": "USA",
    " occupation": "Software Engineer"
}

# Print out the keys
print("Keys:")
for key in my_dict.keys():
    print(key)

# Print out the values
print("\nValues:")
for value in my_dict.values():
    print(value)

# Print out the key-value pairs
print("\nKey-Value Pairs:")
for key, value in my_dict.items():
    print(f"{key}: {value}")
```
