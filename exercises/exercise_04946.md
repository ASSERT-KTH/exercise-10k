## Exercise
Create a program that finds the maximum of two numbers using conditional statements.

## Reference Solution
```python
def find_max(a, b):
    if a > b:
        return a
    else:
        return b

# Example usage
num1 = 10
num2 = 20
max_num = find_max(num1, num2)
print(f"The maximum of {num1} and {num2} is {max_num}.")
```
