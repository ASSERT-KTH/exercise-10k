## Exercise
Write a program that takes two arrays as input, and returns a new array containing the elements that are present in either array but not in both.

## Reference Solution
```python
def symmetric_difference(a, b):
    return list(set(a).symmetric_difference(set(b)))

def symmetric_difference_two(a, b):
    return list(set(a).symmetric_difference(set(b)))

def symmetric_difference_three(a, b):
    return list(set(a) ^ set(b))

def symmetric_difference_four(a, b):
    return list(set(a) ^ set(b))

def symmetric_difference_five(a, b):
    return list(set(a) ^ set(b))

# Test the functions
a = [1, 2, 3, 4, 5]
b = [4, 5, 6, 7, 8]
```
