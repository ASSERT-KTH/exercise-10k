## Exercise
Create a program that uses nested for loops to print a calendar for a given month and year.

## Reference Solution
```python
import calendar

def print_calendar(year, month):
    # Create a calendar object
    cal = calendar.monthcalendar(year, month)

    # Print the month and year
    print(calendar.month_name[month], year)
    print("Mo Tu We Th Fr Sa Su")

    # Iterate over each week in the month
    for week in cal:
        # Iterate over each day in the week
        for day in week:
            # If the day is 0, it means it's not in the current month
            if day == 0:
                print("  ", end="")
            else:
                # Print the day, padding with a space if necessary
                print(f"{day:2d}", end=" ")
        # Start a new line after each week
        print()

# Test the function
year = 2024
month = 9
print_calendar(year, month)
```
