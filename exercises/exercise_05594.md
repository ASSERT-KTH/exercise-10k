## Exercise
Write a function that takes a number as an argument and returns True if it's prime, False otherwise.

## Reference Solution
```python
def is_prime(n):
    """
    Checks if a number is prime.

    Args:
        n (int): The number to check.

    Returns:
        bool: True if the number is prime, False otherwise.
    """
    if n <= 1:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True

# Example usage:
if __name__ == "__main__":
    print(is_prime(11))  # True
    print(is_prime(15))  # False
```
