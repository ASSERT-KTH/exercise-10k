## Exercise
Write a function to convert a set of keys to a dictionary with default values

## Reference Solution
```python
def convert_to_dict(keys, default_value=None):
    """
    This function converts a set of keys into a dictionary with default values.

    Args:
        keys (list): A list of keys to be converted into a dictionary.
        default_value (any, optional): The default value for each key. Defaults to None.

    Returns:
        dict: A dictionary with the given keys and default values.
    """

    # Initialize an empty dictionary to store the result
    result = {}

    # Iterate over each key in the given list of keys
    for key in keys:
        # For each key, assign the default value in the result dictionary
        result[key] = default_value

    # Return the resulting dictionary
    return result

# Example usage:
keys = ['a', 'b', 'c']
default_value = 10
print(convert_to_dict(keys, default_value))  # Output: {'a': 10, 'b': 10, 'c': 10}
```
