## Exercise
Write a program that compares the memory usage of different data types (integer, float, string, list, tuple, dictionary, set).

## Reference Solution
```python
import sys

def compare_memory_usage():
    # Define the data types
    data_types = {
        "Integer": 1,
        "Float": 1.0,
        "String": "Hello, World!",
        "List": [1, 2, 3, 4, 5],
        "Tuple": (1, 2, 3, 4, 5),
        "Dictionary": {"key1": 1, "key2": 2},
        "Set": {1, 2, 3, 4, 5},
    }

    # Compare the memory usage
    print("Memory Usage Comparison:")
    print("---------------------------")
    for data_type, value in data_types.items():
        memory_usage = sys.getsizeof(value)
        print(f"{data_type}: {memory_usage} bytes")

if __name__ == "__main__":
    compare_memory_usage()
```
