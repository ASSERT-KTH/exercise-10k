## Exercise
Write a program to round a given number down to the nearest integer

## Reference Solution
```python
import math

def round_down(num):
    """
    Rounds a given number down to the nearest integer.

    Args:
        num (float): The number to be rounded down.

    Returns:
        int: The rounded down integer.
    """
    return math.floor(num)

# Example usage
if __name__ == "__main__":
    num = 3.7
    rounded_down = round_down(num)
    print(f"The number {num} rounded down to the nearest integer is: {rounded_down}")
```
