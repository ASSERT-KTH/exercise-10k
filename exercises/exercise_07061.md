## Exercise
Write a program that takes a string as input and prints out the string in upper case and lower case.

## Reference Solution
```python
def print_string_in_different_cases(input_string):
    """
    Prints the input string in upper case and lower case.

    Args:
        input_string (str): The input string to be converted.
    """
    print("Upper Case: ", input_string.upper())
    print("Lower Case: ", input_string.lower())

def main():
    # Get the input string from the user
    input_string = input("Enter a string: ")
    
    # Print the string in upper case and lower case
    print_string_in_different_cases(input_string)

if __name__ == "__main__":
    main()
```
