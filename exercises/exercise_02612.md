## Exercise
Write a program that retrieves the value of a key in a dictionary with a default value if the key is not present

## Reference Solution
```python
def get_dict_value(dictionary, key, default_value=None):
    """
    Retrieves the value of a key in a dictionary with a default value if the key is not present.

    Args:
        dictionary (dict): The dictionary to retrieve the value from.
        key (any): The key to retrieve the value for.
        default_value (any, optional): The default value to return if the key is not present. Defaults to None.

    Returns:
        any: The value of the key if present, otherwise the default value.
    """
    return dictionary.get(key, default_value)

# Example usage:
my_dict = {"name": "John", "age": 30}
print(get_dict_value(my_dict, "name"))  # Output: John
print(get_dict_value(my_dict, "city", "New York"))  # Output: New York
print(get_dict_value(my_dict, "city"))  # Output: None
```
