## Exercise
Create a function that takes in a dictionary and a mapping function, and returns a new dictionary containing the results of applying the mapping function to each value

## Reference Solution
```python
def map_dict_values(input_dict, mapping_func):
    """
    This function takes in a dictionary and a mapping function, 
    and returns a new dictionary containing the results of applying 
    the mapping function to each value.

    Args:
        input_dict (dict): The input dictionary.
        mapping_func (function): A function that takes one argument and returns one value.

    Returns:
        dict: A new dictionary with the same keys as the input dictionary, 
              but with values that are the result of applying the mapping function.
    """

    # Initialize an empty dictionary to store the results
    result_dict = {}

    # Iterate over each key-value pair in the input dictionary
    for key, value in input_dict.items():
        
        # Apply the mapping function to the current value
        new_value = mapping_func(value)
        
        # Store the new value in the result dictionary with the same key
        result_dict[key] = new_value
    
    # Return the result dictionary
    return result_dict
```
