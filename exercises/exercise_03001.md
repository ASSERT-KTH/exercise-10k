## Exercise
Write a program to calculate the length of a given string.

## Reference Solution
```python
def calculate_string_length(input_string):
    """
    Calculate the length of a given string.

    Args:
        input_string (str): The input string.

    Returns:
        int: The length of the input string.
    """
    return len(input_string)

def main():
    # Example usage
    input_string = input("Enter a string: ")
    length = calculate_string_length(input_string)
    print(f"The length of the string '{input_string}' is {length}.")

if __name__ == "__main__":
    main()
```
