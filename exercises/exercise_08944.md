## Exercise
Write a program to invert a boolean value.

## Reference Solution
```python
def invert_boolean(value: bool) -> bool:
    """
    Inverts a boolean value.

    Args:
        value (bool): The boolean value to invert.

    Returns:
        bool: The inverted boolean value.
    """
    return not value

# Example usage
if __name__ == "__main__":
    print(invert_boolean(True))  # Output: False
    print(invert_boolean(False))  # Output: True
```
