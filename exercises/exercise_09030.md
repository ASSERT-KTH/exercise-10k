## Exercise
Create a function that takes a number as an argument and returns True if it is positive, False otherwise

## Reference Solution
```python
def is_positive(num: float) -> bool:
    """
    Checks if a given number is positive.

    Args:
    num (float): The number to check.

    Returns:
    bool: True if the number is positive, False otherwise.
    """
    return num > 0

# Example usage:
if __name__ == "__main__":
    print(is_positive(10))  # Output: True
    print(is_positive(-5))  # Output: False
    print(is_positive(0))   # Output: False
```
