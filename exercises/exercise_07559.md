## Exercise
Create a function that takes a name and age as input and prints out a greeting message, with default values for both parameters.

## Reference Solution
```python
# Define a function with default values for name and age
def greeting(name = "World", age = 0):
    """
    Prints out a greeting message with the given name and age.

    Args:
        name (str): The name to be used in the greeting. Defaults to "World".
        age (int): The age to be used in the greeting. Defaults to 0.
    """

    # Check if the age is valid (greater than 0)
    if age < 0:
        print("Invalid age. Please enter a non-negative integer.")
    else:
        # Print out the greeting message
        print(f"Hello, {name}! You are {age} years old.")

# Test the function with default values
greeting()

# Test the function with custom values
greeting("John", 30)

# Test the function with invalid age
greeting("Jane", -1)
```
