## Exercise
Write a program that takes a list of numbers as input and returns the maximum number in the list. The program should handle empty lists and lists with duplicate maximum numbers.

## Reference Solution
```python
def find_max_number(lst):
    if len(lst) == 0:
        return None
    max_number = lst[0]
    for i in range(len(lst)):
        if lst[i] > max_number:
            max_number = lst[i]
    return max_number

# Test cases
print(find_max_number([5, 3, 7, 1, 9, 2]))  # 9
print(find_max_number([1, 2, 3, 4, 5]))  # 5
print(find_max_number([5, 5, 5, 5, 5]))  # 5
print(find_max_number([]))  # None
```
