## Exercise
Write a function to find the median of a list of numbers.

## Reference Solution
```python
def calculate_median(numbers):
    """
    Calculate the median of a list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        float: The median of the list of numbers.
    """
    # Check if the list is empty
    if not numbers:
        raise ValueError("The list is empty")

    # Sort the list
    numbers.sort()

    # Calculate the length of the list
    n = len(numbers)

    # If the list has an odd number of elements, return the middle value
    if n % 2 != 0:
        return numbers[n // 2]
    # If the list has an even number of elements, return the average of the two middle values
    else:
        return (numbers[n // 2 - 1] + numbers[n // 2]) / 2

# Example usage:
numbers = [1, 3, 5, 7, 9]
print(calculate_median(numbers))  # Output: 5

numbers = [1, 2, 3, 4, 5, 6]
print(calculate_median(numbers))  # Output: 3.5
```
