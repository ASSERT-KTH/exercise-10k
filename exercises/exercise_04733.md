## Exercise
Create a lambda function that takes a number as an argument and returns its double.

## Reference Solution
```python
double_number = lambda x: x * 2
```

```python
# Define the lambda function
double_number = lambda x: x * 2

# Test the function
number = 5
result = double_number(number)

print(f"The double of {number} is {result}")
```

```python
# Define a list of numbers
numbers = [1, 2, 3, 4, 5]

# Use the lambda function with map() to double the numbers
doubled_numbers = list(map(lambda x: x * 2, numbers))

print(doubled_numbers)
```
