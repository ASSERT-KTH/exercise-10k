## Exercise
Group a dictionary by its values

## Reference Solution
```python
def group_dict_by_values(input_dict):
    """
    Group a dictionary by its values.

    Args:
    input_dict (dict): The dictionary to be grouped.

    Returns:
    dict: A dictionary with values from the input dictionary as keys and a list of corresponding keys as values.
    """
    result = {}
    for key, value in input_dict.items():
        if value not in result:
            result[value] = [key]
        else:
            result[value].append(key)
    return result

# Example usage:
input_dict = {
    'a': 1,
    'b': 2,
    'c': 1,
    'd': 3,
    'e': 2,
    'f': 1
}

grouped_dict = group_dict_by_values(input_dict)
print(grouped_dict)
```
