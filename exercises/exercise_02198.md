## Exercise
Create a function to encode a given string using run-length encoding. The function should return the encoded string.

## Reference Solution
```python
def run_length_encoding(input_string):
    """
    This function encodes a given string using run-length encoding.
    
    Args:
        input_string (str): The input string to be encoded.
    
    Returns:
        str: The encoded string.
    """
    
    # Initialize an empty string to store the encoded result
    encoded_string = ""
    
    # If the input string is empty, return an empty string
    if not input_string:
        return encoded_string
    
    # Initialize counters for the current character and its count
    current_char = input_string[0]
    current_count = 1
    
    # Iterate over the input string starting from the second character
    for char in input_string[1:]:
        
        # If the current character matches the previous one, increment the count
        if char == current_char:
            current_count += 1
        else:
            # Otherwise, append the count and character to the encoded string
            encoded_string += str(current_count) + current_char
            
            # Update the current character and reset the count
            current_char = char
            current_count = 1
    
    # Append the last character and its count to the encoded string
    encoded_string += str(current_count) + current_char
    
    return encoded_string

# Example usage:
print(run_length_encoding("AAABBBCCC"))  # Output: "3A3B3C"
```
