## Exercise
Given a flattened dictionary, write a function to denormalize it by reconstructing the nested dictionaries.

## Reference Solution
```python
def denormalize(flattened_dict, separator='.'):
    """
    Denormalize a flattened dictionary.

    Args:
        flattened_dict (dict): The flattened dictionary to denormalize.
        separator (str, optional): The separator used in the flattened dictionary keys. Defaults to '.'.

    Returns:
        dict: The denormalized dictionary.
    """
    denormalized_dict = {}
    for key, value in flattened_dict.items():
        levels = key.split(separator)
        current_dict = denormalized_dict
        for level in levels[:-1]:
            if level not in current_dict:
                current_dict[level] = {}
            current_dict = current_dict[level]
        current_dict[levels[-1]] = value
    return denormalized_dict


# Example usage:
flattened_dict = {
    'a.b.c': 1,
    'a.b.d': 2,
    'e.f.g': 3,
    'e.f.h': 4
}

denormalized = denormalize(flattened_dict)
print(denormalized)
```
