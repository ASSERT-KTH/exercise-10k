## Exercise
Write a program to apply a function to each element of a list.

## Reference Solution
```python
def apply_function_to_list(input_list, func):
    """
    Applies a given function to each element of a list.

    Args:
        input_list (list): The list of elements to apply the function to.
        func (function): The function to apply to each element.

    Returns:
        list: A new list with the function applied to each element.
    """
    return [func(element) for element in input_list]

# Example usage:
def square(x):
    """Returns the square of a number."""
    return x ** 2

numbers = [1, 2, 3, 4, 5]
squared_numbers = apply_function_to_list(numbers, square)

print("Original List:", numbers)
print("Squared Numbers:", squared_numbers)
```
