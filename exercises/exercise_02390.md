## Exercise
Write a program that performs a binary search on a sorted list to find a target value using a while loop.

## Reference Solution
```python
# Binary Search using while loop

def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = left + (right - left) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

# Testing the function
arr = [2, 3, 4, 10, 40]
target = 10
print(binary_search(arr, target))  # Output: 3
```
