## Exercise
Write a function to validate if a given string is a valid IP address

## Reference Solution
```python
import ipaddress

def validate_ip_address(ip):
    """
    Validate if a given string is a valid IP address.
    
    Args:
        ip (str): The IP address to be validated.
    
    Returns:
        bool: True if the IP address is valid, False otherwise.
    """
    try:
        ipaddress.ip_address(ip)
        return True
    except ValueError:
        return False

# Example usage:
if __name__ == "__main__":
    print(validate_ip_address("192.168.1.1"))  # IPv4
    print(validate_ip_address("2001:0db8:85a3:0000:0000:8a2e:0370:7334"))  # IPv6
    print(validate_ip_address("256.1.1.1"))  # Invalid IPv4
    print(validate_ip_address("2001:0db8:85a3:0000:0000:8a2e:0370:733g"))  # Invalid IPv6
```
