## Exercise
Write a program to find the common elements in two arrays.

## Reference Solution
```python
def find_common_elements(array1, array2):
    """
    Find the common elements in two arrays.

    Args:
        array1 (list): The first array.
        array2 (list): The second array.

    Returns:
        list: A list of common elements.
    """
    return list(set(array1) & set(array2))

# Example usage:
array1 = [1, 2, 3, 4, 5]
array2 = [4, 5, 6, 7, 8]

common_elements = find_common_elements(array1, array2)
print("Common elements:", common_elements)
```
