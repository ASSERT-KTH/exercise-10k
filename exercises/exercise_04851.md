## Exercise
Write a program that takes an array as input, and returns its maximum element.

## Reference Solution
```python
def find_max_element(arr):
    """
    This function finds the maximum element in a given array.

    Args:
        arr (list): The input array.

    Returns:
        The maximum element in the array.

    Raises:
        ValueError: If the input array is empty.
    """
    if not arr:
        raise ValueError("Input array is empty")

    return max(arr)


def main():
    # Example usage
    arr = [12, 45, 7, 23, 56, 89, 34]
    max_element = find_max_element(arr)
    print(f"Maximum element in the array: {max_element}")


if __name__ == "__main__":
    main()
```
