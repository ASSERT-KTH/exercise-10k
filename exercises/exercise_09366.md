## Exercise
Write a program that demonstrates array indexing.

## Reference Solution
```python
# Define an array
array = [1, 2, 3, 4, 5]

# Print the entire array
print("Entire array:", array)

# Accessing elements using indexing
print("Element at index 0:", array[0])
print("Element at index 1:", array[1])
print("Element at index 4:", array[4])

# Accessing elements from the end of the array
print("Last element:", array[-1])
print("Second last element:", array[-2])

# Slicing the array
print("Elements from index 1 to 3:", array[1:4])
print("Elements from index 2 to the end:", array[2:])
print("Elements from the start to index 3:", array[:4])

# Modifying elements using indexing
array[0] = 10
print("Modified array:", array)

# Trying to access an index out of range
try:
    print("Element at index 10:", array[10])
except IndexError:
    print("Error: Index out of range")
```
