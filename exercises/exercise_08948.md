## Exercise
Evaluate a boolean expression and print out the result.

## Reference Solution
```python
def evaluate_boolean_expression(expression):
    """
    Evaluate a boolean expression and return the result.
    
    Args:
    expression (str): A boolean expression as a string.
    
    Returns:
    bool: The result of the evaluated expression.
    """
    try:
        # Evaluate the expression using eval()
        result = eval(expression)
        
        # Check if the result is a boolean value
        if not isinstance(result, bool):
            raise ValueError("The expression does not evaluate to a boolean value.")
        
        return result
    
    except Exception as e:
        print(f"An error occurred: {e}")
        return None

# Example usage:
expression = "True and False"
result = evaluate_boolean_expression(expression)

if result is not None:
    print(f"The result of the expression '{expression}' is: {result}")
```
