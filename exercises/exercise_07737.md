## Exercise
Create a program that takes in a user's input and determines the data type of the input. The program should be able to differentiate between integers, floats, and strings. If the input is an integer or a float, the program should print out the data type and the value. If the input is a string, the program should check if the string can be converted into an integer or a float, and print out the converted value and its data type if possible.

## Reference Solution
```python
def determine_data_type(input_str):
    """
    This function determines the data type of the input.
    It can differentiate between integers, floats, and strings.
    If the input is an integer or a float, it prints out the data type and the value.
    If the input is a string, it checks if the string can be converted into an integer or a float,
    and prints out the converted value and its data type if possible.

    Args:
        input_str (str): The user's input.

    Returns:
        None
    """

    # First, we try to convert the input to an integer
    try:
        # Attempt to convert the input to an integer
        int_val = int(input_str)
        # If successful, print the data type and the value
        print(f"Data type: Integer, Value: {int_val}")
        return
    except ValueError:
        # If the input cannot be converted to an integer, we try to convert it to a float
        pass

    # Now, we try to convert the input to a float
    try:
        # Attempt to convert the input to a float
        float_val = float(input_str)
        # If successful, print the data type and the value
        print(f"Data type: Float, Value: {float_val}")
        return
    except ValueError:
        # If the input cannot be converted to a float, we consider it as a string
        pass

    # At this point, we know the input is a string
    print(f"Input '{input_str}' is a string.")

    # We try to convert the string to an integer
    try:
        # Attempt to convert the string to an integer
        int_val = int(input_str)
        # If successful, print the converted value and its data type
        print(f"The string can be converted to an integer: {int_val}")
    except ValueError:
        # If the string cannot be converted to an integer, we try to convert it to a float
        pass

    # We try to convert the string to a float
    try:
        # Attempt to convert the string to a float
        float_val = float(input_str)
        # If successful, print the converted value and its data type
        print(f"The string can be converted to a float: {float_val}")
    except ValueError:
        # If the string cannot be converted to a float, we do nothing
        pass


def main():
    # Take the user's input
    user_input = input("Please enter a value: ")
    
    # Call the function to determine the data type
    determine_data_type(user_input)


if __name__ == "__main__":
    main()
```
