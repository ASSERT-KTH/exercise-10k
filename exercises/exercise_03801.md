## Exercise
Write a program to print a pattern of asterisks (*) using a for loop. The pattern should have 5 rows and each row should have an increasing number of asterisks (1, 2, 3, 4, 5).

## Reference Solution
```python
# Define the number of rows in the pattern
num_rows = 5

# Loop through each row
for i in range(1, num_rows + 1):
    # Print asterisks for the current row
    print('*' * i)
```
