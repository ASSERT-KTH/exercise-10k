## Exercise
Write a program that finds the common prefix of two lists, i.e., the longest sequence of elements that are equal at the beginning of both lists.

## Reference Solution
```python
def common_prefix(list1, list2):
    prefix = []
    for a, b in zip(list1, list2):
        if a == b:
            prefix.append(a)
        else:
            break
    return prefix

# Test the function
list1 = [1, 2, 3, 4, 5]
list2 = [1, 2, 3, 6, 7]
print(common_prefix(list1, list2))
```
