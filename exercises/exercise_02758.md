## Exercise
Write a program that calculates the selling price of an item given its cost and markup percentage.

## Reference Solution
```python
def calculate_selling_price(cost, markup_percentage):
    # Calculate the markup amount
    markup_amount = (cost / 100) * markup_percentage
    
    # Calculate the selling price
    selling_price = cost + markup_amount
    
    return selling_price

def main():
    # Get the cost of the item from the user
    cost = float(input("Enter the cost of the item: $"))
    
    # Get the markup percentage from the user
    markup_percentage = float(input("Enter the markup percentage (%): "))
    
    # Calculate the selling price
    selling_price = calculate_selling_price(cost, markup_percentage)
    
    # Print the selling price
    print(f"The selling price of the item is: ${selling_price:.2f}")

if __name__ == "__main__":
    main()
```
