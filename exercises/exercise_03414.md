## Exercise
Write a program to filter a given dictionary based on a specific condition.

## Reference Solution
```python
def filter_dict(input_dict, condition):
    """
    Filters a dictionary based on a given condition.

    Args:
        input_dict (dict): The dictionary to be filtered.
        condition (function): A function that takes in a key-value pair and returns True or False.

    Returns:
        dict: The filtered dictionary.
    """
    return {key: value for key, value in input_dict.items() if condition(key, value)}


# Example usage:
if __name__ == "__main__":
    # Define a sample dictionary
    sample_dict = {
        "a": 1,
        "b": 2,
        "c": 3,
        "d": 4,
        "e": 5
    }

    # Define a condition function
    def is_even(key, value):
        return value % 2 == 0

    # Filter the dictionary
    filtered_dict = filter_dict(sample_dict, is_even)

    # Print the filtered dictionary
    print("Original Dictionary:")
    print(sample_dict)
    print("Filtered Dictionary:")
    print(filtered_dict)
```
