## Exercise
Write a function that uses profiling to measure its execution time or other performance metrics.

## Reference Solution
```python
import cProfile
import pstats

def my_function():
    """
    This function simulates a time-consuming operation.
    
    It uses a loop to iterate over a large range of numbers, 
    which makes it a good candidate for profiling.
    """
    # Simulate some time-consuming operation
    result = 0
    for i in range(10000000):
        result += i
    return result

def profile_function(func):
    """
    This function uses profiling to measure the execution time of another function.
    
    Args:
        func (function): The function to be profiled.
    
    Returns:
        None
    """
    # Create a profiler object
    profiler = cProfile.Profile()
    
    # Start profiling
    profiler.enable()
    
    # Call the function to be profiled
    func()
    
    # Stop profiling
    profiler.disable()
    
    # Print the profiling statistics
    stats = pstats.Stats(profiler)
    stats.sort_stats('cumulative')
    stats.print_stats()

# Profile my_function
profile_function(my_function)
```
