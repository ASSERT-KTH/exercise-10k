## Exercise
Write a program that takes an array and a rotation amount as input, and returns a new array containing the elements of the original array, but rotated by the specified amount.

## Reference Solution
```python
def rotate(arr, d):
    n = len(arr)
    d = d % n
    return arr[d:] + arr[:d]

arr = [1, 2, 3, 4, 5]
d = 2

print("Original array:", arr)
print("Rotated array:", rotate(arr, d))
```
