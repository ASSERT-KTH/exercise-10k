## Exercise
Write a program to compress a given string by removing duplicate characters

## Reference Solution
```python
def compress_string(input_str):
    """
    Compress a given string by removing duplicate characters.

    Args:
        input_str (str): The input string to be compressed.

    Returns:
        str: The compressed string with no duplicate characters.
    """
    if not input_str:
        return ""

    seen = set()
    compressed_str = ""

    for char in input_str:
        if char not in seen:
            compressed_str += char
            seen.add(char)

    return compressed_str

# Example usage
def main():
    input_str = "aaaabbbccddd"
    print("Input String:", input_str)
    compressed_str = compress_string(input_str)
    print("Compressed String:", compressed_str)

if __name__ == "__main__":
    main()
```
