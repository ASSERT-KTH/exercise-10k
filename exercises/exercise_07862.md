## Exercise
Create a program that converts time from 12-hour format to 24-hour format using conditional statements.

## Reference Solution
```python
def convert_to_24_hour(time_str):
    components = time_str.split()
    time_components = components[0].split(":")
    hours = int(time_components[0])
    minutes = int(time_components[1])
    seconds = int(time_components[2])
    period = components[1]

    if period == "AM":
        if hours == 12:
            hours = 0
    elif period == "PM":
        if hours != 12:
            hours += 12

    return f"{hours:02d}:{minutes:02d}:{seconds:02d}"

time_str = input("Enter time in 12-hour format (HH:MM:SS AM/PM): ")
print(f"Time in 24-hour format: {convert_to_24_hour(time_str)}")
```
