## Exercise
Write a program to concatenate two or more arrays.

## Reference Solution
```python
import numpy as np

def concatenate_arrays(*arrays):
    """
    Concatenates two or more arrays into a single array.

    Args:
        *arrays: Variable number of arrays to concatenate.

    Returns:
        A single array containing all elements from the input arrays.
    """
    return np.concatenate(arrays)

# Example usage
if __name__ == "__main__":
    array1 = np.array([1, 2, 3])
    array2 = np.array([4, 5, 6])
    array3 = np.array([7, 8, 9])

    concatenated_array = concatenate_arrays(array1, array2, array3)
    print("Concatenated array:", concatenated_array)
```
