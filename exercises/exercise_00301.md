## Exercise
Write a function that calculates the sum of all elements in a given array.

## Reference Solution
```python
def array_sum(array):
    """
    Calculates the sum of all elements in a given array.

    Args:
        array (list): A list of numbers.

    Returns:
        int: The sum of all elements in the array.

    Raises:
        TypeError: If the input is not a list.
        ValueError: If the list contains non-numeric values.
    """
    if not isinstance(array, list):
        raise TypeError("Input must be a list")

    try:
        return sum(array)
    except TypeError:
        raise ValueError("List must contain only numeric values")


# Example usage:
if __name__ == "__main__":
    numbers = [1, 2, 3, 4, 5]
    result = array_sum(numbers)
    print(f"Sum of the array: {result}")
```
