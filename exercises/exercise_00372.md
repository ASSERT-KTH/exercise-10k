## Exercise
Write a program to insert an element at a specified position in an array. The program should take as input the array, the element to be inserted, and the position where the element should be inserted. If the position is greater than the length of the array, the element should be appended to the end of the array.

## Reference Solution
```python
def insert_element(array, element, index):
    if index > len(array):
        array.append(element)
    else:
        array.insert(index, element)
    return array

array = [1, 2, 3, 4, 5]
element = 6
index = 3

print("Original array:", array)
print("Element to be inserted:", element)
print("Position of element:", index)

array = insert_element(array, element, index)

print("Array after insertion:", array)
```
