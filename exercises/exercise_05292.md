## Exercise
Write a program that generates the first n Fibonacci numbers, where n is a user-inputted value. Use a while loop to generate the numbers.

## Reference Solution
```python
def fibonacci(n):
    """
    Generate the first n Fibonacci numbers.
    
    Args:
        n (int): The number of Fibonacci numbers to generate.
    
    Returns:
        list: A list of the first n Fibonacci numbers.
    """
    # Initialize the list to store Fibonacci numbers
    fib_sequence = [0, 1]
    
    # Check if n is less than or equal to 0
    if n <= 0:
        return []
    
    # Check if n is 1 or 2
    if n == 1:
        return [0]
    elif n == 2:
        return fib_sequence
    
    # Generate Fibonacci numbers using a while loop
    i = 2
    while i < n:
        # Calculate the next Fibonacci number
        next_fib = fib_sequence[i-1] + fib_sequence[i-2]
        # Append the next Fibonacci number to the list
        fib_sequence.append(next_fib)
        # Increment the counter
        i += 1
    
    # Return the list of Fibonacci numbers
    return fib_sequence

def main():
    # Get user input for n
    n = int(input("Enter the number of Fibonacci numbers to generate: "))
    
    # Generate and print the Fibonacci numbers
    fib_numbers = fibonacci(n)
    print("The first {} Fibonacci numbers are:".format(n))
    print(fib_numbers)

if __name__ == "__main__":
    main()
```
