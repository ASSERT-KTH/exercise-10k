## Exercise
Write a function to count the total number of spaces in a given string.

## Reference Solution
```python
def count_spaces(input_string: str) -> int:
    """
    Counts the total number of spaces in a given string.

    Args:
        input_string (str): The input string to count spaces from.

    Returns:
        int: The total number of spaces in the input string.
    """
    return input_string.count(' ')

# Example usage
def main():
    input_str = "Hello World, this is a test string."
    space_count = count_spaces(input_str)
    print(f"Total spaces in the string: {space_count}")

if __name__ == "__main__":
    main()
```
