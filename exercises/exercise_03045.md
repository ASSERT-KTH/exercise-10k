## Exercise
Create two functions: one to double a number and another to add 1 to a number. Then, use function composition to create a new function that doubles a number and then adds 1.

## Reference Solution
```python
# Define a function to double a number
def double(x: int) -> int:
    """Returns the double of a given number."""
    return x * 2

# Define a function to add 1 to a number
def add_one(x: int) -> int:
    """Returns the given number plus one."""
    return x + 1

# Use function composition to create a new function
def double_and_add_one(x: int) -> int:
    """Returns the double of a given number plus one."""
    return add_one(double(x))

# Example usage
if __name__ == "__main__":
    num = 5
    result = double_and_add_one(num)
    print(f"Double {num} and add 1: {result}")
```
