## Exercise
Write a program that takes a string as input, reverses its order, and converts it to uppercase. The program should handle strings with both letters and numbers.

## Reference Solution
```python
def reverse_string(input_string):
    reversed_string = input_string[::-1]  # reverse the string
    return reversed_string.upper()  # convert to uppercase

input_string = "Hello123"
print(reverse_string(input_string))
```
