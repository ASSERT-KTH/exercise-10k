## Exercise
Write a function to find the maximum contiguous subarray of an array of integers. The function should take an array of integers as input and return the maximum sum of a contiguous subarray.

## Reference Solution
```python
def maximum_subarray(arr):
    """
    This function finds the maximum sum of a contiguous subarray within a given array of integers.

    Parameters:
    arr (list): A list of integers.

    Returns:
    int: The maximum sum of a contiguous subarray.
    """
    max_sum = float('-inf')
    current_sum = 0

    for num in arr:
        current_sum = max(num, current_sum + num)
        max_sum = max(max_sum, current_sum)

    return max_sum

# Example usage:
arr = [-2, -3, 4, -1, -2, 1, 5, -3]
result = maximum_subarray(arr)
print(result)  # Output: 7
```
