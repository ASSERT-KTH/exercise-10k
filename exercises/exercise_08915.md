## Exercise
Write a program to find the maximum subarray sum of a given list of numbers.

## Reference Solution
```python
def max_subarray_sum(nums):
    """
    Find the maximum subarray sum of a given list of numbers.

    Args:
    - nums (list): A list of numbers.

    Returns:
    - max_sum (int): The maximum subarray sum.
    """
    if not nums:
        return 0

    max_sum = float('-inf')
    current_sum = 0

    for num in nums:
        current_sum = max(num, current_sum + num)
        max_sum = max(max_sum, current_sum)

    return max_sum

# Example usage
if __name__ == "__main__":
    numbers = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
    max_sum = max_subarray_sum(numbers)
    print("Maximum subarray sum:", max_sum)
```
