## Exercise
Write a program that calculates the sum of all even numbers from 1 to a given number n using a for loop.

## Reference Solution
```python
#include <iostream>

using namespace std;

int main() {
    int n, somma = 0;

    cout << "Inserisci un numero: ";
    cin >> n;

    for (int i = 1; i <= n; i++) {
        if (i % 2 == 0) {
            somma += i;
        }
    }

    cout << "La somma dei numeri pari e' " << somma << endl;

    return 0;
}
```
