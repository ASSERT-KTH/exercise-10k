## Exercise
Write a program that takes a list and a target element as input, and returns the index of the target element in the list if found, or -1 otherwise.

## Reference Solution
```python
def find_index(my_list, target):
    for e in range(len(my_list)):
        if my_list[e] == target:
            return e
    return -1

my_list = [1, 2, 3, 4, 5]
index = find_index(my_list, 3)
print(index)
```
