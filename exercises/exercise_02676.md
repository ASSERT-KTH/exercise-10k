## Exercise
Write a program that takes a string as input and returns the string in title case.

## Reference Solution
```python
def convert_to_title_case(input_string):
    """
    Converts the input string to title case.

    Args:
        input_string (str): The string to be converted.

    Returns:
        str: The input string in title case.
    """
    return input_string.title()

def main():
    input_string = input("Enter a string: ")
    title_case_string = convert_to_title_case(input_string)
    print("Title Case String: ", title_case_string)

if __name__ == "__main__":
    main()
```
