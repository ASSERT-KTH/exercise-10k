## Exercise
Write a program that takes two numbers as input and returns the result of the floor division.

## Reference Solution
```python
def floor_division(num1, num2):
    """
    This function performs floor division on two numbers.

    Args:
        num1 (int): The dividend.
        num2 (int): The divisor.

    Returns:
        int: The result of the floor division.
    """
    if num2 == 0:
        raise ZeroDivisionError("Cannot divide by zero")
    return num1 // num2

# Example usage:
num1 = int(input("Enter the first number: "))
num2 = int(input("Enter the second number: "))

result = floor_division(num1, num2)
print(f"The result of {num1} divided by {num2} is: {result}")
```
