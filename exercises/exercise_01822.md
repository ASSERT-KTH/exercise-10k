## Exercise
Write a program to access and print the elements at specific indices in a given array.

## Reference Solution
```python
def print_elements_at_indices(array, indices):
    """
    Prints the elements at specific indices in a given array.

    Args:
        array (list): The input array.
        indices (list): A list of indices.

    Returns:
        None
    """
    # Check if the array is empty
    if not array:
        print("The array is empty.")
        return

    # Check if the indices list is empty
    if not indices:
        print("The indices list is empty.")
        return

    # Iterate over the indices
    for index in indices:
        # Check if the index is within the bounds of the array
        if index < 0 or index >= len(array):
            print(f"Index {index} is out of range.")
        else:
            print(f"Element at index {index}: {array[index]}")


# Example usage:
array = [1, 2, 3, 4, 5]
indices = [0, 2, 4]
print_elements_at_indices(array, indices)
```
