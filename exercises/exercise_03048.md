## Exercise
Simulate function overloading by creating a function that can take either two numbers or four numbers as input and return their sum.

## Reference Solution
```python
def sum_numbers(*args):
    """
    Simulates function overloading by taking either two numbers or four numbers as input and returning their sum.

    Args:
        *args: A variable number of arguments

    Returns:
        The sum of the input numbers
    """

    # Check if the number of arguments is not 2 or 4
    if len(args) not in [2, 4]:
        raise ValueError("Input should be either two numbers or four numbers")

    # Initialize the sum variable to zero
    total = 0

    # Iterate over each argument
    for arg in args:
        # Check if the argument is a number
        if not isinstance(arg, (int, float)):
            raise TypeError("All inputs must be numbers")
        
        # Add the argument to the total
        total += arg

    # Return the total sum
    return total

# Test the function with two numbers
print(sum_numbers(1, 2))  # Output: 3

# Test the function with four numbers
print(sum_numbers(1, 2, 3, 4))  # Output: 10
```
