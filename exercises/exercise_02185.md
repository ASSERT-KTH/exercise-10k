## Exercise
Write a function to convert a given string to uppercase or lowercase. The function should take a string and a case conversion option as input, and return the converted string.

## Reference Solution
```python
def convert_case(input_str, case_option):
    """
    This function converts a given string to uppercase or lowercase.

    Args:
        input_str (str): The input string to be converted.
        case_option (str): The case conversion option. It can be 'upper' or 'lower'.

    Returns:
        str: The converted string.
    """

    # First, we check if the input string is actually a string
    if not isinstance(input_str, str):
        raise TypeError("Input must be a string.")

    # Then, we check if the case option is valid
    if case_option not in ['upper', 'lower']:
        raise ValueError("Invalid case option. It must be 'upper' or 'lower'.")

    # Now, we use the built-in string methods to convert the case
    # If the case option is 'upper', we use the upper() method
    if case_option == 'upper':
        # The upper() method returns a string where all characters are in uppercase
        return input_str.upper()
    
    # If the case option is 'lower', we use the lower() method
    elif case_option == 'lower':
        # The lower() method returns a string where all characters are in lowercase
        return input_str.lower()

# Example usage:
print(convert_case('Hello World', 'upper'))  # Outputs: HELLO WORLD
print(convert_case('Hello World', 'lower'))  # Outputs: hello world
```
