## Exercise
Write a function to merge two sorted lists into one sorted list.

## Reference Solution
```python
def merge_sorted_lists(list1, list2):
    """
    Merge two sorted lists into one sorted list.

    Args:
        list1 (list): The first sorted list.
        list2 (list): The second sorted list.

    Returns:
        list: A new sorted list containing all elements from both input lists.
    """
    result = []
    i, j = 0, 0

    # Compare elements from both lists and append the smaller one to the result list
    while i < len(list1) and j < len(list2):
        if list1[i] < list2[j]:
            result.append(list1[i])
            i += 1
        else:
            result.append(list2[j])
            j += 1

    # Append any remaining elements from both lists
    result.extend(list1[i:])
    result.extend(list2[j:])

    return result

# Example usage
list1 = [1, 3, 5, 7]
list2 = [2, 4, 6, 8]
print(merge_sorted_lists(list1, list2))  # Output: [1, 2, 3, 4, 5, 6, 7, 8]
```
