## Exercise
Write a program that checks the data type of a given variable and prints out whether it is an integer, float, or string.

## Reference Solution
```python
def check_data_type(var):
    if isinstance(var, int):
        print(f"The variable is an integer.")
    elif isinstance(var, float):
        print(f"The variable is a float.")
    elif isinstance(var, str):
        print(f"The variable is a string.")
    else:
        print(f"The variable is of type {type(var).__name__}.")

# Test the function
check_data_type(10)  # integer
check_data_type(10.5)  # float
check_data_type("Hello")  # string
check_data_type([1, 2, 3])  # list
check_data_type({"a": 1, "b": 2})  # dictionary
```
