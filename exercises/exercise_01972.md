## Exercise
Create a function that checks if a given string is a palindrome. The function should take a string as an argument and return a boolean indicating whether the string is a palindrome.

## Reference Solution
```python
def is_palindrome(s: str) -> bool:
    """
    Checks if a given string is a palindrome.

    Args:
        s (str): The input string to check.

    Returns:
        bool: True if the string is a palindrome, False otherwise.
    """

    # First, we remove any non-alphanumeric characters from the string and convert it to lowercase
    # This is because we want to ignore case sensitivity and non-alphanumeric characters when checking for palindromes
    s = ''.join(c for c in s if c.isalnum()).lower()

    # Then, we compare the string with its reverse
    # If they are equal, then the string is a palindrome
    # We use Python's slicing feature to reverse the string
    return s == s[::-1]
```
