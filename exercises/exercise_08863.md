## Exercise
Create a function to check if a given string is a repdigit or not

## Reference Solution
```python
def is_repdigit(s):
    """
    Checks if a given string is a repdigit or not.

    Args:
        s (str): The input string.

    Returns:
        bool: True if the string is a repdigit, False otherwise.
    """
    # Check if the string is not empty and all characters are digits
    if not s or not s.isdigit():
        return False

    # Check if all digits are the same
    return len(set(s)) == 1

# Example usage
print(is_repdigit("111"))  # True
print(is_repdigit("123"))  # False
print(is_repdigit("2222"))  # True
print(is_repdigit("abc"))  # False
print(is_repdigit(""))  # False
```
