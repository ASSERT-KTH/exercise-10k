## Exercise
Demonstrate the concept of dynamic typing in programming

## Reference Solution
```python
# No need to declare the type of the variable
x = 5
print(type(x))  # Output: <class 'int'>

# Change the type of the variable
x = "Hello, World!"
print(type(x))  # Output: <class 'str'>
```

```python
// Need to declare the type of the variable
int x = 5;
System.out.println(x.getClass().getName());  // Output: int

// Cannot change the type of the variable
// x = "Hello, World!";  // Error: incompatible types
```
