## Exercise
Create two functions, one that triples a number and another that adds 1 to the result, then compose them to create a new function

## Reference Solution
```python
def triple(x):
    """
    Triples the input number.

    Args:
        x (int or float): The number to be tripled.

    Returns:
        int or float: The triple of the input number.
    """
    return 3 * x


def increment(x):
    """
    Adds 1 to the input number.

    Args:
        x (int or float): The number to be incremented.

    Returns:
        int or float: The input number plus one.
    """
    return x + 1


def triple_and_increment(x):
    """
    Composed function that first triples the input number and then increments the result.

    Args:
        x (int or float): The number to be tripled and then incremented.

    Returns:
        int or float: The result of first tripling the input number and then adding 1.
    """
    return increment(triple(x))


# Example usage
if __name__ == "__main__":
    number = 5
    result = triple_and_increment(number)
    print(f"Tripling {number} and then incrementing the result gives: {result}")
```
