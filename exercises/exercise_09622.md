## Exercise
Create a function that takes a name as an argument and prints out a personalized greeting.

## Reference Solution
```python
# Define a function called 'greeting' that takes one argument 'name'.
def greeting(name):
    """
    Prints out a personalized greeting.

    Args:
        name (str): The name to be used in the greeting.

    Returns:
        None
    """

    # Inside the function, we want to print out a greeting message that includes the name.
    # We use an f-string to insert the name into the string.
    print(f"Hello, {name}!")

# Now we can call the function with a name to see the greeting.
greeting("John")
```
