## Exercise
Write a program that reverses the order of a given list of variables.

## Reference Solution
```python
def reverse_list(input_list):
    """
    Reverses the order of a given list.

    Args:
    input_list (list): The list to be reversed.

    Returns:
    list: The reversed list.
    """
    return input_list[::-1]


# Example usage:
my_list = [1, 2, 3, 4, 5]
print("Original list:", my_list)
print("Reversed list:", reverse_list(my_list))
```

```python
def reverse_list_in_place(input_list):
    """
    Reverses the order of a given list in-place.

    Args:
    input_list (list): The list to be reversed.
    """
    input_list.reverse()


# Example usage:
my_list = [1, 2, 3, 4, 5]
print("Original list:", my_list)
reverse_list_in_place(my_list)
print("Reversed list:", my_list)
```
