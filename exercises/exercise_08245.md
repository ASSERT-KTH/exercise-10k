## Exercise
Develop a program that greets the user based on the current time of day using if/else statements.

## Reference Solution
```python
from datetime import datetime

def get_greeting():
    """
    Returns a greeting based on the current time of day.
    """
    now = datetime.now()
    current_hour = now.hour

    if current_hour < 12:
        return "Good morning!"
    elif current_hour < 18:
        return "Good afternoon!"
    else:
        return "Good evening!"

def main():
    print(get_greeting())

if __name__ == "__main__":
    main()
```
