## Exercise
Write a function to access a character at a specific index in a string.

## Reference Solution
```python
def access_char_at_index(string, index):
    """
    This function accesses a character at a specific index in a string.

    Args:
        string (str): The input string.
        index (int): The index of the character to be accessed.

    Returns:
        str: The character at the specified index in the string.

    Raises:
        TypeError: If the input string is not of type 'str' or the index is not of type 'int'.
        IndexError: If the index is out of range.
    """

    # First, we check if the input string is indeed a string and the index is an integer
    if not isinstance(string, str) or not isinstance(index, int):
        raise TypeError("Input string must be of type 'str' and index must be of type 'int'.")

    # Then, we check if the index is within the valid range for the given string
    if index < 0 or index >= len(string):
        raise IndexError("Index out of range.")

    # Now, we can safely access the character at the specified index
    # We use the built-in indexing feature of Python strings to do this
    return string[index]

# Example usage:
print(access_char_at_index("Hello, World!", 4))  # Outputs: 'o'
```
