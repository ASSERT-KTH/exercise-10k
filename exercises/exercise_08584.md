## Exercise
Write a function to filter dictionary items based on a condition

## Reference Solution
```python
def filter_dict_items(dictionary, condition):
    """
    Filters dictionary items based on a given condition.

    Args:
        dictionary (dict): The dictionary to filter.
        condition (function): A function that takes in a key-value pair and returns a boolean.

    Returns:
        dict: A new dictionary containing only the items that meet the condition.
    """
    return {key: value for key, value in dictionary.items() if condition(key, value)}

# Example usage:
# Define a dictionary
example_dict = {
    "apple": 1,
    "banana": 2,
    "cherry": 3,
    "date": 4,
    "elderberry": 5
}

# Define a condition function
def condition(key, value):
    return value > 2

# Filter the dictionary
filtered_dict = filter_dict_items(example_dict, condition)

# Print the filtered dictionary
print(filtered_dict)
```
