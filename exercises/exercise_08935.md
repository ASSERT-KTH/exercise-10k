## Exercise
Evaluate a boolean expression and print out the result.

## Reference Solution
```python
def evaluate_boolean_expression(expression):
    """
    Evaluate a boolean expression and print out the result.

    Args:
        expression (str): A boolean expression as a string.

    Returns:
        bool: The result of the boolean expression.
    """
    try:
        # Use eval() function to evaluate the boolean expression
        result = eval(expression)
        return result
    except Exception as e:
        print(f"Error: {str(e)}")
        return None

# Example usage
if __name__ == "__main__":
    expression = "True and False"
    result = evaluate_boolean_expression(expression)
    if result is not None:
        print(f"The result of '{expression}' is: {result}")
```
