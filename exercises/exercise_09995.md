## Reference Solution

```python
# Dynamic typing in Python
def greet(name):
    """Print a greeting message."""
    print(f"Hello, {name}!")

# Using the function with a string
greet("John")  # Output: Hello, John!

# Using the function with an integer
greet(123)  # Output: Hello, 123!
```
