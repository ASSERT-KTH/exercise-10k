## Exercise
Write a program that checks if a list of integers is sorted in ascending order.

## Reference Solution
```python
def is_sorted_ascending(lst):
    """
    Checks if a list of integers is sorted in ascending order.

    Args:
        lst (list): A list of integers.

    Returns:
        bool: True if the list is sorted in ascending order, False otherwise.
    """
    return all(lst[i] <= lst[i + 1] for i in range(len(lst) - 1))

def main():
    # Example usage:
    lst1 = [1, 2, 3, 4, 5]
    lst2 = [5, 4, 3, 2, 1]

    print(f"Is {lst1} sorted in ascending order? {is_sorted_ascending(lst1)}")
    print(f"Is {lst2} sorted in ascending order? {is_sorted_ascending(lst2)}")

if __name__ == "__main__":
    main()
```
