## Exercise
Write a function that takes a list of integers as input and returns the maximum value in the list. If the list is empty, the function should return a message indicating that the list is empty.

## Reference Solution
```python
def find_max_value(lst):
    """
    This function finds the maximum value in a given list of integers.
    
    If the list is empty, it returns a message indicating that the list is empty.
    
    Parameters:
    lst (list): A list of integers.
    
    Returns:
    int or str: The maximum value in the list if it's not empty, otherwise a message.
    """

    # First, we check if the list is empty
    if not lst:
        # If the list is empty, we return a message indicating that the list is empty
        return "The list is empty."

    # If the list is not empty, we use the built-in max function to find the maximum value
    # The max function returns the largest item in an iterable or the largest of two or more arguments
    max_value = max(lst)
    
    # Finally, we return the maximum value
    return max_value

# Example usage:
print(find_max_value([1, 2, 3, 4, 5]))  # Output: 5
print(find_max_value([]))  # Output: The list is empty.
```
