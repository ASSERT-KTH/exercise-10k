## Exercise
Write a program that prints the ASCII values of characters in a given string using a for loop.

## Reference Solution
```python
#include <stdio.h>

int main() {
    int i;
    char str[] = "C Programming";

    for (i = 0; str[i] != '\0'; i++) {
        printf("ASCII value of character '%c' is %d\n", str[i], str[i]);
    }

    return 0;
}
```
