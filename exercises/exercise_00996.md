## Exercise
Create a program that calculates the Body Mass Index (BMI) based on the user's weight and height. The program uses if-else statements to determine the BMI category.

## Reference Solution
```python
# Get user's weight and height
weight = float(input("Enter your weight in kg: "))
height = float(input("Enter your height in meters: "))

# Calculate BMI
bmi = weight / (height ** 2)

# Determine BMI category
if bmi < 18.5:
    category = "Underweight"
elif bmi < 25:
    category = "Normal"
elif bmi < 30:
    category = "Overweight"
else:
    category = "Obese"

# Print the result
print(f"Your BMI is: {bmi:.2f}")
print(f"Your BMI category is: {category}")
```
