## Exercise
Create a function that checks if a given number is within a certain range.

## Reference Solution
```python
def is_in_range(num, lower, upper, inclusive=True):
    """
    Checks if a given number is within a certain range.

    Args:
        num (int or float): The number to check.
        lower (int or float): The lower bound of the range.
        upper (int or float): The upper bound of the range.
        inclusive (bool, optional): Whether the range is inclusive. Defaults to True.

    Returns:
        bool: True if the number is within the range, False otherwise.
    """

    if inclusive:
        return lower <= num <= upper
    else:
        return lower < num < upper

# Example usage:
if __name__ == "__main__":
    print(is_in_range(5, 1, 10))  # Returns: True

    print(is_in_range(1, 1, 10, inclusive=False))  # Returns: False
    print(is_in_range(10, 1, 10, inclusive=False))  # Returns: False
```
