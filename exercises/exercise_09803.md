## Exercise
Create a function that takes an array of integers as input and returns the maximum value in the array. If the array is empty, the function should return null.

## Reference Solution
```python
def find_max_value(array):
    """
    This function finds the maximum value in an array of integers.
    
    Args:
        array (list): A list of integers.
    
    Returns:
        int: The maximum value in the array. If the array is empty, it returns None.
    """
    
    # First, we check if the array is empty
    if not array:
        
        # If the array is empty, we return None as per the problem statement
        return None
    
    # If the array is not empty, we use the built-in max function to find the maximum value
    # The max function returns the largest item in an iterable or the largest of two or more arguments
    max_value = max(array)
    
    # Finally, we return the maximum value
    return max_value
```
