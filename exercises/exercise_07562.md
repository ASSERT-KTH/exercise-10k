## Exercise
Write a function that calculates the Fibonacci sequence using memoization to optimize performance.

## Reference Solution
```python
from functools import lru_cache

@lru_cache(maxsize=64)
def fibonacci(n):
    if n < 2:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)
```
