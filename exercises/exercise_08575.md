## Exercise
Write a function to get the length of a dictionary

## Reference Solution
```python
def get_dict_length(input_dict):
    """
    Returns the length of a dictionary.

    Args:
        input_dict (dict): The dictionary to get the length of.

    Returns:
        int: The number of key-value pairs in the dictionary.
    """
    return len(input_dict)

# Example usage:
my_dict = {"apple": 1, "banana": 2, "cherry": 3}
print(get_dict_length(my_dict))  # Output: 3
```
