## Exercise
Write a program to create a Person class with attributes like name, age, and occupation, and methods to display person's details.

## Reference Solution
```python
class Person:
    """
    A class representing a person with attributes like name, age, and occupation.
    """

    def __init__(self, name, age, occupation):
        """
        Initializes a Person object with the given name, age, and occupation.

        Args:
            name (str): The person's name.
            age (int): The person's age.
            occupation (str): The person's occupation.
        """
        self.name = name
        self.age = age
        self.occupation = occupation

    def display_details(self):
        """
        Displays the person's details.
        """
        print(f"Name: {self.name}")
        print(f"Age: {self.age} years old")
        print(f"Occupation: {self.occupation}")


def main():
    # Create a Person object
    person = Person("John Doe", 30, "Software Developer")

    # Display the person's details
    print("Person Details:")
    person.display_details()


if __name__ == "__main__":
    main()
```
