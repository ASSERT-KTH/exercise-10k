## Exercise
Write a program that takes two arrays as input, and returns True if the first array is a subset of the second array, and False otherwise.

## Reference Solution
```python
class Solution:
    def isSubset(self, A, B):
        if len(A) == 0:
            return True

        for i in A:
            if i not in B:
                return False

        return True
```

```python
solution = Solution()
print(solution.isSubset([1, 2, 3], [1, 2, 3, 4, 5]))  # True
print(solution.isSubset([1, 2, 3], [1, 2, 4, 5]))  # False
```
