## Exercise
Write a program that takes a number as input and prints the sequence of numbers from 1 to the input number.

## Reference Solution
```python
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cout << i << endl;
    }
    return 0;
}
```
