## Exercise
Create a function that takes a number as input and returns True if it is prime and False otherwise.

## Reference Solution
```python
def is_prime(n):
    """
    Checks if a number is prime.

    Args:
        n (int): The number to check.

    Returns:
        bool: True if the number is prime, False otherwise.
    """
    if n <= 1:  # 0 and 1 are not prime numbers
        return False
    for i in range(2, int(n ** 0.5) + 1):  # Check up to the square root of n
        if n % i == 0:  # If n is divisible by any number, it's not prime
            return False
    return True  # If n is not divisible by any number, it's prime

# Example usage
if __name__ == "__main__":
    print(is_prime(25))  # False
    print(is_prime(23))  # True
    print(is_prime(37))  # True
    print(is_prime(48))  # False
```
