## Exercise
Create a function that uses a generator to produce a sequence of numbers

## Reference Solution
```python
def fibonacci(n):
    """
    Generate a sequence of Fibonacci numbers up to the nth number.
    
    Args:
        n (int): The number of Fibonacci numbers to generate.
    
    Yields:
        int: The next Fibonacci number in the sequence.
    """
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

# Example usage:
if __name__ == "__main__":
    n = 10  # Generate the first 10 Fibonacci numbers
    print(f"First {n} Fibonacci numbers:")
    for i, num in enumerate(fibonacci(n)):
        print(f"F({i}) = {num}")
```
