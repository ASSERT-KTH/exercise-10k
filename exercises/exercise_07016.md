## Exercise
Write a program that declares and initializes variables of different data types, including integers, floats, and strings. The program should then print out the values and data types of these variables.

```python
# Declare and initialize variables
my_int = 3
my_float = 4.5
my_string = "Hello"
my_string2 = "World"

# Print variables and their data types
print("Value:", my_int, "Type:", type(my_int))
print("Value:", my_float, "Type:", type(my_float))
print("Value:", my_string, "Type:", type(my_string))
print("Value:", my_string2, "Type:", type(my_string2))
```
