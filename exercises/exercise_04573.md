## Exercise
Write a function that takes two dictionaries and returns a new dictionary with the key-value pairs that exist in the first dictionary but not in the second

## Reference Solution
```python
def unique_key_value_pairs(dict1, dict2):
    """
    Returns a new dictionary with key-value pairs that exist in dict1 but not in dict2.

    Args:
        dict1 (dict): The first dictionary.
        dict2 (dict): The second dictionary.

    Returns:
        dict: A new dictionary with unique key-value pairs from dict1.
    """
    return {key: dict1[key] for key in dict1 if key not in dict2 or dict1[key] != dict2[key]}

# Example usage:
dict1 = {'a': 1, 'b': 2, 'c': 3, 'd': 4}
dict2 = {'b': 2, 'c': 5, 'e': 6}

unique_pairs = unique_key_value_pairs(dict1, dict2)
print(unique_pairs)  # Output: {'a': 1, 'c': 3, 'd': 4}
```
