## Exercise
Write a function that converts a string to upper or lower case based on a given flag

## Reference Solution
```python
def convert_case(input_str, to_upper):
    """
    Converts a string to upper or lower case based on a given flag.

    Args:
        input_str (str): The input string to be converted.
        to_upper (bool): A flag indicating whether to convert to upper case (True) or lower case (False).

    Returns:
        str: The converted string.
    """
    if to_upper:
        return input_str.upper()
    else:
        return input_str.lower()

# Example usage:
def main():
    input_str = "Hello, World!"
    print("Original string:", input_str)

    # Convert to upper case
    upper_case_str = convert_case(input_str, True)
    print("Upper case string:", upper_case_str)

    # Convert to lower case
    lower_case_str = convert_case(input_str, False)
    print("Lower case string:", lower_case_str)

if __name__ == "__main__":
    main()
```
