## Exercise
Write a program that takes an integer as input and prints out the numbers from 1 to the input number, with each number on a new line. Use a for loop to achieve this.

## Reference Solution
```python
<?php
$n = 100; // replace with the desired input number
for ($i = 1; $i <= $n; $i++) {
    echo $i . "\n";
}
?>
```
