## Exercise
Write a program that takes an array as input and returns the sum of all its elements

## Reference Solution
```python
def array_sum(arr):
    """
    Calculate the sum of all elements in an array.

    Args:
        arr (list): The input array.

    Returns:
        int: The sum of all elements in the array.
    """
    return sum(arr)

# Example usage
def main():
    # Define an example array
    arr = [1, 2, 3, 4, 5]

    # Calculate the sum of the array
    total = array_sum(arr)

    # Print the result
    print("Array:", arr)
    print("Sum:", total)

if __name__ == "__main__":
    main()
```
