## Exercise
Create a program that uses functions to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The program should take two numbers and an operation as input and return the result of the operation.

## Reference Solution
```python
# Arithmetic Operations Program

# Define the functions
def add(num1, num2):
    """Return the sum of two numbers."""
    return num1 + num2

def sub(num1, num2):
    """Return the difference of two numbers."""
    return num1 - num2

def mul(num1, num2):
    """Return the product of two numbers."""
    return num1 * num2

def div(num1, num2):
    """Return the quotient of two numbers. Raises ZeroDivisionError if num2 is zero."""
    if num2 == 0:
        raise ZeroDivisionError("Cannot divide by zero!")
    return num1 / num2

# Main Program
num1 = 10
num2 = 2

# Call the functions
add_result = add(num1, num2)
sub_result = sub(num1, num2)
mul_result = mul(num1, num2)
div_result = div(num1, num2)

# Print the results
print(f"Addition: {num1} + {num2} = {add_result}")
print(f"Subtraction: {num1} - {num2} = {sub_result}")
print(f"Multiplication: {num1} * {num2} = {mul_result}")
print(f"Division: {num1} / {num2} = {div_result}")
```
