## Exercise
Create a program that calculates the sum of digits in a given integer.

## Reference Solution
```python
def calculate_digit_sum(n):
    """
    Calculate the sum of digits in a given integer.

    Args:
        n (int): The input integer.

    Returns:
        int: The sum of digits in the input integer.
    """
    return sum(int(digit) for digit in str(abs(n)))

def main():
    # Example usage
    num = int(input("Enter an integer: "))
    digit_sum = calculate_digit_sum(num)
    print(f"The sum of digits in {num} is: {digit_sum}")

if __name__ == "__main__":
    main()
```
