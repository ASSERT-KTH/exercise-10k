## Exercise
Write a program that checks if a given number is positive, negative, or zero using conditional statements.

## Reference Solution
```python
using System;

class Program
{
    static void Main()
    {
        int num = 0;

        if (num > 0)
        {
            Console.WriteLine("Positive number");
        }
        else if (num < 0)
        {
            Console.WriteLine("Negative number");
        }
        else
        {
            Console.WriteLine("Zero");
        }
    }
}
```
