## Exercise
Create a function that uses an anonymous function (lambda) to calculate the square of a number. Use the function to calculate the square of 4.

## Reference Solution
```python
# Define a function that uses lambda to calculate the square of a number
calculate_square = lambda x: x ** 2

# Use the function to calculate the square of 4
result = calculate_square(4)

# Print the result
print("The square of 4 is:", result)
```
