## Exercise
Write a program that counts the number of words in a given string using a for loop.

## Reference Solution
```python
def count_words(input_string):
    word_count = 0
    in_word = False

    for char in input_string:
        if char.isspace():
            in_word = False
        else:
            if not in_word:
                word_count += 1
                in_word = True

    return word_count


# Test the function
input_str = "This is a test string with multiple words"
print("Number of words:", count_words(input_str))
```
