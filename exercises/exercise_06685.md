## Exercise
Create a program that takes the side length of a cube as input and returns its volume.

## Reference Solution
```python
def calculate_cube_volume(side_length):
    """
    Calculates the volume of a cube.

    Args:
        side_length (float): The length of a side of the cube.

    Returns:
        float: The volume of the cube.
    """
    if side_length <= 0:
        raise ValueError("Side length must be a positive number")
    return side_length ** 3

def main():
    # Get the side length from the user
    while True:
        try:
            side_length = float(input("Enter the side length of the cube: "))
            if side_length <= 0:
                print("Side length must be a positive number")
            else:
                break
        except ValueError:
            print("Invalid input. Please enter a number.")

    # Calculate and print the volume
    volume = calculate_cube_volume(side_length)
    print(f"The volume of the cube is {volume:.2f} cubic units.")

if __name__ == "__main__":
    main()
```
