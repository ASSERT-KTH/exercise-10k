## Exercise
Write a program that uses a for loop to print a given matrix.

## Reference Solution
```python
def print_matrix(matrix):
    """
    Prints a given matrix.

    Args:
        matrix (list of lists): The matrix to print.
    """
    for row in matrix:
        for element in row:
            print(element, end=' ')
        print()

# Example usage
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

print("Given Matrix:")
print_matrix(matrix)
```
