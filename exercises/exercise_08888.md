## Exercise
Write a program that takes the lengths of the legs of a right triangle as input and returns the length of the hypotenuse.

## Reference Solution
```python
#include <cstdio>
#include <cmath>

int main() {
    double a, b, c;
    while (scanf("%lf%lf", &a, &b) == 2) {
        c = sqrt(a * a + b * b);
        printf("%.2f\n", c);
    }
    return 0;
}
```
