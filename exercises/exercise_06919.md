## Exercise
Write a program that prints all numbers from 1 to the input number using a for loop or a while loop.

## Reference Solution
```python
def print_numbers(n):
    for i in range(1, n + 1):
        print(i)

num = int(input("Enter a number: "))
print_numbers(num)
```

```python
def print_numbers(n):
    i = 1
    while i <= n:
        print(i)
        i += 1

num = int(input("Enter a number: "))
print_numbers(num)
```
