## Exercise
Create a function that takes a variable number of arguments and returns their sum

## Reference Solution
```python
def variable_sum(*args):
    """
    Returns the sum of all input arguments.

    Args:
        *args: A variable number of numeric arguments.

    Returns:
        The sum of all input arguments.

    Raises:
        TypeError: If non-numeric arguments are provided.
    """
    try:
        return sum(args)
    except TypeError:
        raise TypeError("All arguments must be numeric.")

# Example usage
if __name__ == "__main__":
    print(variable_sum(1, 2, 3, 4, 5))  # Output: 15
    print(variable_sum(10, 20))  # Output: 30
    try:
        print(variable_sum(1, 'a', 3))  # Raises TypeError
    except TypeError as e:
        print(e)  # Output: All arguments must be numeric.
```
