## Exercise
Create a function that takes a list as input and returns the average of its elements.

## Reference Solution
```python
def calculate_average(input_list):
    """
    Calculate the average of a list of numbers.

    Args:
        input_list (list): A list of numbers.

    Returns:
        float: The average of the input list.

    Raises:
        ValueError: If the input list is empty.
        TypeError: If the input list contains non-numeric values.
    """
    if not input_list:
        raise ValueError("Input list is empty")

    try:
        return sum(input_list) / len(input_list)
    except TypeError:
        raise TypeError("Input list contains non-numeric values")

# Example usage:
numbers = [1, 2, 3, 4, 5]
average = calculate_average(numbers)
print(f"The average is: {average}")
```
