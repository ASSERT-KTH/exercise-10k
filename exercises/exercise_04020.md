## Exercise
Write a program that uses nested for loops to print a 5x5 mosaic pattern of asterisks (*) and dots (.).

## Reference Solution
```python
# Header
# Author: [Your Name]
# Date: [Today's Date]

# Program Description
# This program prints a 5x5 mosaic pattern of asterisks (*) and dots (.).

# Code
for i in range(5):
    for j in range(5):
        if (i + j) % 2 == 0:
            print('*', end=' ')
        else:
            print('.', end=' ')
    print()
```
