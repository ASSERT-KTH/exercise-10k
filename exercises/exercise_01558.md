## Exercise
Given a list of pairs, write a function to separate the pairs into two separate lists.

## Reference Solution
```python
def separate_pairs(pairs):
    """
    Separate pairs into two separate lists.

    Args:
        pairs (list): A list of pairs.

    Returns:
        tuple: Two lists, the first containing the first elements of the pairs and the second containing the second elements.
    """
    list1, list2 = zip(*pairs)
    return list(list1), list(list2)

# Example usage:
pairs = [(1, 2), (3, 4), (5, 6)]
list1, list2 = separate_pairs(pairs)
print("List 1:", list1)
print("List 2:", list2)
```
