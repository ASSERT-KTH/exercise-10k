## Exercise
Write a program to find the first duplicate element in an array.

## Reference Solution
```python
def find_first_duplicate(arr):
    """
    Find the first duplicate element in an array.

    Args:
        arr (list): The input array.

    Returns:
        int: The first duplicate element if found, otherwise None.
    """
    seen = set()
    for num in arr:
        if num in seen:
            return num
        seen.add(num)
    return None

# Example usage
arr = [2, 1, 3, 5, 3, 2]
print(find_first_duplicate(arr))  # Output: 3
```
