## Exercise
Write a program that asks for a sentence and splits it into individual words.

## Reference Solution
```python
def split_sentence_into_words(sentence):
    """
    Splits a sentence into individual words.

    Args:
        sentence (str): The input sentence.

    Returns:
        list: A list of words in the sentence.
    """
    words = sentence.split()
    return words

def main():
    # Ask for a sentence
    sentence = input("Please enter a sentence: ")

    # Split the sentence into words
    words = split_sentence_into_words(sentence)

    # Print the words
    print("Words in the sentence:")
    for i, word in enumerate(words):
        print(f"{i+1}. {word}")

if __name__ == "__main__":
    main()
```
