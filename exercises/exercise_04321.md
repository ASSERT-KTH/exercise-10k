## Exercise
Write a program to find the first element of an array.

## Reference Solution
```python
def find_first_element(array):
    """
    Returns the first element of the given array.

    Args:
        array (list): The input array.

    Returns:
        The first element of the array, or None if the array is empty.
    """
    if len(array) == 0:
        return None
    return array[0]

# Example usage
array = [1, 2, 3, 4, 5]
first_element = find_first_element(array)

print("First element:", first_element)

# Edge case: empty array
empty_array = []
first_element = find_first_element(empty_array)

print("First element of empty array:", first_element)
```
