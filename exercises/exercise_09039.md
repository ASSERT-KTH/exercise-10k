## Exercise
Create a function that takes two boolean values as arguments and returns True if either is True, False otherwise

## Reference Solution
```python
def boolean_or(a: bool, b: bool) -> bool:
    """
    Returns True if either of the input boolean values is True, False otherwise.

    Args:
        a (bool): The first boolean value.
        b (bool): The second boolean value.

    Returns:
        bool: The result of the Boolean OR operation.
    """
    return a or b

# Example usage:

    print(boolean_or(False, False)) # Expected output: False
```
