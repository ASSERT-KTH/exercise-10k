## Exercise
Create a program that simulates a chatbot. The program should ask for user input and respond based on a set of predefined rules (e.g., 'hello' -> 'Hi, how are you?').

## Reference Solution
```python
# Import the `random` module for generating random responses
import random

# Define the chatbot rules
chatbot_rules = {
    'hello': 'Hi, how are you?',
    'hi': 'Hello! How can I assist you today?',
    'how are you': 'I\'m doing great, thanks for asking!',
    'what is your purpose': 'I\'m here to assist and chat with you.',
    'default': 'Sorry, I didn\'t understand that. Can you please rephrase?'
}

# Define a function to get the chatbot response
def get_chatbot_response(user_input):
    """Returns the chatbot response based on the user input."""
    user_input = user_input.lower()  # Convert to lowercase for case-insensitive matching
    for key in chatbot_rules:
        if key in user_input:
            return chatbot_rules[key]
    return chatbot_rules['default']  # Return the default response if no match is found

# Define the main function
def main():
    print("Welcome to the chatbot simulator!")
    while True:
        user_input = input("You: ")
        if user_input.lower() == 'quit':  # Exit the program if the user types 'quit'
            print("Chatbot: Goodbye!")
            break
        chatbot_response = get_chatbot_response(user_input)
        print(f"Chatbot: {chatbot_response}")

# Run the main function
if __name__ == "__main__":
    main()
```
