## Exercise
Write a program that calculates the selling price of a product given its cost and markup percentage.

## Reference Solution
```python
def calculate_selling_price(cost, markup_percentage):
    """
    Calculate the selling price of a product given its cost and markup percentage.

    Args:
        cost (float): The cost of the product.
        markup_percentage (float): The markup percentage.

    Returns:
        float: The selling price of the product.
    """
    markup_amount = (cost / 100) * markup_percentage
    selling_price = cost + markup_amount
    return selling_price


def main():
    cost = float(input("Enter the cost of the product: "))
    markup_percentage = float(input("Enter the markup percentage: "))

    selling_price = calculate_selling_price(cost, markup_percentage)

    print(f"The selling price of the product is: ${selling_price:.2f}")


if __name__ == "__main__":
    main()
```
