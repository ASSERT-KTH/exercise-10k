## Exercise
Write a function that checks if a number is positive.

## Reference Solution
```python
def is_positive(num: float) -> bool:
    """
    Checks if a number is positive.

    Args:
        num (float): The number to check.

    Returns:
        bool: True if the number is positive, False otherwise.
    """
    return num > 0
```

```python
def main():
    print(is_positive(5))   # Output: True
    print(is_positive(-3))  # Output: False
    print(is_positive(0))   # Output: False

if __name__ == "__main__":
    main()
```
