## Exercise
Write a program that takes an array and an index as input, and returns a new array with the element at that index removed.

## Reference Solution
```python
function removeElement(array, index) {
  return array.slice(0, index).concat(array.slice(index + 1));
}

// Example usage:
const arr = [3, 4, 5, 6, 7, 8, 9];
console.log(removeElement(arr, 3)); // Output: [3, 4, 5, 7, 8, 9]
```
