## Exercise
Write a program that takes a string as input and returns the shortest word in the string.

## Reference Solution
```python
def shortest_word(input_string):
    """
    Returns the shortest word in the input string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The shortest word in the input string.
    """
    # Split the input string into words
    words = input_string.split()

    # Initialize the shortest word and its length
    shortest_word = min(words, key=len)

    return shortest_word

def main():
    # Example usage
    input_string = "This is a sample string"
    print("Input String:", input_string)
    print("Shortest Word:", shortest_word(input_string))

if __name__ == "__main__":
    main()
```
