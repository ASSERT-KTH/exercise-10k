## Exercise
Write a function to check if a given string contains a substring.

## Reference Solution
```python
def check_substring(main_string: str, sub_string: str) -> bool:
    """
    Checks if a given string contains a specified substring.

    Args:
        main_string (str): The main string to search in.
        sub_string (str): The substring to search for.

    Returns:
        bool: True if the substring is found, False otherwise.
    """
    return sub_string.lower() in main_string.lower()

# Example usage
if __name__ == "__main__":
    main_str = "Hello, World!"
    sub_str = "world"
    result = check_substring(main_str, sub_str)
    print(f"Substring '{sub_str}' found: {result}")
```
