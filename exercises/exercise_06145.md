## Exercise
Write a function to extract a substring from a given string.

## Reference Solution
```python
def extract_substring(input_string: str, start_index: int, end_index: int) -> str:
    """
    Extracts a substring from the input string based on the start and end indices.

    Args:
        input_string (str): The original string.
        start_index (int): The starting index of the substring (inclusive).
        end_index (int): The ending index of the substring (exclusive).

    Returns:
        str: The extracted substring.
    """
    if not isinstance(input_string, str):
        raise TypeError("Input must be a string.")
    if not isinstance(start_index, int) or not isinstance(end_index, int):
        raise TypeError("Start and end indices must be integers.")
    if start_index < 0 or end_index > len(input_string) or start_index > end_index:
        raise ValueError("Invalid start or end index.")

    return input_string[start_index:end_index]

# Example usage:
def main():
    input_str = "Hello, World!"
    start_idx = 7
    end_idx = 12
    print(extract_substring(input_str, start_idx, end_idx))  # Output: "World"

if __name__ == "__main__":
    main()
```
