## Exercise
Create a function that takes a person's name as input and prints out a greeting message.

## Reference Solution
```python
def print_greeting(name: str) -> None:
    """
    Prints out a greeting message for the given person.

    Args:
        name (str): The person's name.
    """
    print(f"Hello, {name}!")

# Example usage:
def main() -> None:
    name = input("Please enter your name: ")
    print_greeting(name)

if __name__ == "__main__":
    main()
```
