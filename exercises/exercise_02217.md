## Exercise
Write a program that takes a string as input and checks if it is a palindrome. Print 'Palindrome' if the string is a palindrome, otherwise print 'Not a palindrome'

## Reference Solution
```python
def is_palindrome(s: str) -> bool:
    """
    Checks if a string is a palindrome.

    Args:
    s (str): The input string.

    Returns:
    bool: True if the string is a palindrome, False otherwise.
    """
    s = ''.join(c for c in s if c.isalnum()).lower()  # remove non-alphanumeric characters and convert to lowercase
    return s == s[::-1]  # check if the string is equal to its reverse

def main():
    input_str = input("Enter a string: ")
    if is_palindrome(input_str):
        print("Palindrome")
    else:
        print("Not a palindrome")

if __name__ == "__main__":
    main()
```
