## Exercise
Write a function that takes a string as input and returns the lowercase version of the string.

## Reference Solution
```python
def to_lowercase(input_string: str) -> str:
    """
    Returns the lowercase version of the input string.

    Args:
        input_string (str): The string to be converted to lowercase.

    Returns:
        str: The lowercase version of the input string.
    """
    return input_string.lower()

# Example usage:
def main():
    input_str = "Hello, World!"
    print(f"Original string: {input_str}")
    print(f"Lowercase string: {to_lowercase(input_str)}")

if __name__ == "__main__":
    main()
```
