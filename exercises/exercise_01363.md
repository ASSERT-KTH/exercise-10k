## Exercise
Write a program to display the multiplication table for a given number.

## Reference Solution
```python
def print_multiplication_table(number, limit=10):
    """
    Prints the multiplication table for the given number up to the specified limit.

    Args:
        number (int): The number for which the multiplication table is to be printed.
        limit (int, optional): The upper limit of the multiplication table. Defaults to 10.
    """
    print(f"Multiplication Table for {number}:")
    for i in range(1, limit + 1):
        print(f"{number} x {i} = {number * i}")

def main():
    number = int(input("Enter a number: "))
    limit = int(input("Enter the upper limit of the multiplication table (default: 10): ") or 10)
    print_multiplication_table(number, limit)

if __name__ == "__main__":
    main()
```
