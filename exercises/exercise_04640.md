## Exercise
Write a program that takes a list as input, and returns all possible permutations of the list.

## Reference Solution
```python
import itertools

def get_permutations(input_list):
    """
    Returns all possible permutations of the input list.

    Args:
        input_list (list): The input list.

    Returns:
        list: A list of tuples, where each tuple is a permutation of the input list.
    """
    return list(itertools.permutations(input_list))

def main():
    # Example usage
    input_list = [1, 2, 3]
    permutations = get_permutations(input_list)
    print("Input List:", input_list)
    print("Permutations:")
    for permutation in permutations:
        print(permutation)

if __name__ == "__main__":
    main()
```
