## Exercise
Write a program that uses a for loop to print a diamond pattern based on user input.

## Reference Solution
```python
def print_diamond(size):
    """
    Prints a diamond pattern of a given size.
    
    Args:
    size (int): The size of the diamond pattern.
    """
    # Print the top half of the diamond
    for i in range(size):
        # Print spaces before the asterisks
        print(" " * (size - i - 1), end="")
        # Print asterisks
        print("* " * (i + 1))
    
    # Print the bottom half of the diamond
    for i in range(size - 2, -1, -1):
        # Print spaces before the asterisks
        print(" " * (size - i - 1), end="")
        # Print asterisks
        print("* " * (i + 1))

def main():
    # Get the size of the diamond from the user
    size = int(input("Enter the size of the diamond: "))
    
    # Print the diamond pattern
    print_diamond(size)

if __name__ == "__main__":
    main()
```
