## Exercise
Create a function that returns a closure that adds a given number to its input

## Reference Solution
```python
def create_adder(num):
    """
    Creates a closure that adds a given number to its input.

    Args:
    num (int): The number to be added.

    Returns:
    function: A closure that takes one argument and returns the sum of num and the input.
    """
    def adder(x):
        # The closure has access to the variables in the surrounding scope
        return x + num
    return adder

# Example usage:
if __name__ == "__main__":
    # Create a closure that adds 5 to its input
    add_five = create_adder(5)
    
    # Use the closure
    print(add_five(10))  # Outputs: 15
```
