## Exercise
Write a program that determines whether a given number is even or odd using conditional statements.

## Reference Solution
```python
def check_even_odd(num):
    if num % 2 == 0:
        print(f"{num} is an even number.")
    else:
        print(f"{num} is an odd number.")

# Test the function
num = int(input("Enter a number: "))
check_even_odd(num)
```
