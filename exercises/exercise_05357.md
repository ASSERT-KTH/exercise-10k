## Exercise
Create a function that takes an array and a mapping function as input, and applies the mapping function to each element of the array, returning a new array containing the mapped elements.

## Reference Solution
```python
def map_array(array, mapping_function):
    """
    Applies a given mapping function to each element of an input array and returns a new array containing the mapped elements.

    Args:
        array (list): The input array to be mapped.
        mapping_function (function): A function that takes one argument and returns a value.

    Returns:
        list: A new array containing the mapped elements.
    """

    # Initialize an empty list to store the mapped elements
    mapped_array = []

    # Iterate over each element in the input array
    for element in array:
        # Apply the mapping function to the current element and store the result in a variable
        mapped_element = mapping_function(element)
        
        # Append the mapped element to the mapped array
        mapped_array.append(mapped_element)

    # Return the mapped array
    return mapped_array

# Example usage:
# Define a mapping function that squares a number
def square(x):
    return x ** 2

# Create an array of numbers
numbers = [1, 2, 3, 4, 5]

# Map the square function to the array of numbers
squared_numbers = map_array(numbers, square)

# Print the result
print(squared_numbers)  # Output: [1, 4, 9, 16, 25]
```
