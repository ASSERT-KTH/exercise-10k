## Exercise
Write a program that takes a nested list as input, and returns a new list containing all elements flattened.

## Reference Solution
```python
def flatten(nested_list):
    """
    This function takes a nested list as input and returns a new list containing all elements flattened.

    Args:
        nested_list (list): A list that may contain nested lists.

    Returns:
        list: A new list with all elements flattened.
    """
    flat_list = []
    for element in nested_list:
        if isinstance(element, list):
            flat_list.extend(flatten(element))
        else:
            flat_list.append(element)
    return flat_list

# Example usage
if __name__ == "__main__":
    nested_list = [1, 2, [3, 4, [5, 6]], 7, [8, 9]]
    print("Original Nested List:", nested_list)
    print("Flattened List:", flatten(nested_list))
```
