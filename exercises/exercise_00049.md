## Exercise
Validate an IP address

## Reference Solution
```python
import ipaddress

def validate_ip(address):
    """
    Validate an IP address.

    Args:
        address (str): The IP address to validate.

    Returns:
        bool: True if the IP address is valid, False otherwise.
    """
    try:
        ipaddress.ip_address(address)
        return True
    except ValueError:
        return False

# Example usage:
print(validate_ip("192.168.1.1"))  # True
print(validate_ip("256.1.1.1"))  # False
print(validate_ip("2001:0db8:85a3:0000:0000:8a2e:0370:7334"))  # True
print(validate_ip("2001:0db8:85a3:0000:0000:8a2e:0370:733g"))  # False
```
