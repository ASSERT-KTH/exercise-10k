## Exercise
Declare two integer variables with the values 10 and 20. Swap the values of the variables and print the new values.

## Reference Solution
```python
# Declare two integer variables
var1 = 10
var2 = 20

# Print the original values
print("Original Values:")
print(f"var1: {var1}")
print(f"var2: {var2}")

# Swap the values of the variables
var1, var2 = var2, var1

# Print the new values
print("\nNew Values:")
print(f"var1: {var1}")
print(f"var2: {var2}")
```
