## Exercise
Given a dictionary and a function, write a function to apply the function to all values in the dictionary and return a single result.

## Reference Solution
```python
def apply_function_to_dict_values(dictionary, func):
    """
    Applies a given function to all values in a dictionary and returns a single result.
    
    Parameters:
    dictionary (dict): The dictionary to apply the function to.
    func (function): The function to apply to the dictionary values.
    
    Returns:
    result: The result of applying the function to the dictionary values.
    """

    # Initialize a variable to store the result. This will be used to store the result of the function application.
    result = None
    
    # Check if the dictionary is not empty
    if dictionary:
        
        # Get the first value from the dictionary
        first_value = next(iter(dictionary.values()))
        
        # Apply the function to the first value and store the result
        result = func(first_value)
        
        # Iterate over the rest of the dictionary values
        for value in dictionary.values():
            
            # If this is not the first value, apply the function to the current result and the current value
            if value != first_value:
                # Apply the function to the current result and the current value
                result = func(result, value)
    
    # Return the final result
    return result

# Example usage:
def add(x, y):
    """Adds two numbers."""
    return x + y

numbers = {'a': 1, 'b': 2, 'c': 3}
print(apply_function_to_dict_values(numbers, add))  # Output: 6
```
