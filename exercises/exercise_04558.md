## Exercise
Write a program that converts a given string to title case.

## Reference Solution
```python
def convert_to_title_case(input_string: str) -> str:
    """
    Converts a given string to title case.

    Args:
        input_string (str): The string to be converted.

    Returns:
        str: The input string in title case.
    """
    return input_string.title()


def main():
    # Example usage
    input_string = "hello world"
    title_case_string = convert_to_title_case(input_string)
    print(f"Input String: {input_string}")
    print(f"Title Case String: {title_case_string}")


if __name__ == "__main__":
    main()
```
