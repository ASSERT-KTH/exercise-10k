## Exercise
Write a program that takes a string as input and returns True if it has balanced parentheses and False otherwise.

## Reference Solution
```python
def has_balanced_parentheses(s):
    """
    Checks if a string has balanced parentheses.

    Args:
    s (str): The input string.

    Returns:
    bool: True if the string has balanced parentheses, False otherwise.
    """
    stack = []
    parentheses_map = {')': '(', '}': '{', ']': '['}

    for char in s:
        # If the character is an opening parenthesis, push it onto the stack.
        if char in parentheses_map.values():
            stack.append(char)
        # If the character is a closing parenthesis, check if the stack is empty or if the top of the stack does not match.
        elif char in parentheses_map:
            if not stack or stack.pop() != parentheses_map[char]:
                return False

    # If the stack is empty after iterating over the string, the parentheses are balanced.
    return not stack


# Example usage:
if __name__ == "__main__":
    print(has_balanced_parentheses("((()))"))  # True
    print(has_balanced_parentheses("(()"))  # False
    print(has_balanced_parentheses("({[]})"))  # True
    print(has_balanced_parentheses("({[})"))  # False
```
