## Exercise
Sort the values of a dictionary in ascending order

## Reference Solution
```python
def sort_dict_values(dictionary):
    """
    Sorts the values of a dictionary in ascending order.

    Args:
        dictionary (dict): The dictionary to sort.

    Returns:
        list: A list of sorted values.
    """
    return sorted(dictionary.values())

# Example usage
my_dict = {'a': 5, 'b': 1, 'c': 3, 'd': 2, 'e': 4}
sorted_values = sort_dict_values(my_dict)
print(sorted_values)
```
