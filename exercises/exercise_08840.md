## Exercise
Write a program that takes a string input from the user, checks if it can be converted to an integer, and then prints out the integer value if successful. If the input cannot be converted, print an error message. Additionally, the program should also handle cases where the input is empty.

## Reference Solution
```python
// Author: [Your Name]
// Date: [Today's Date]

#include <iostream>
#include <string>

using namespace std;

int main() {
    string input;

    cout << "Enter a string: ";
    cin >> input;

    if (input.empty()) {
        cout << "Error: Input is empty." << endl;
        return 1;
    }

    try {
        int integer = stoi(input);
        cout << "Integer value: " << integer << endl;
    } catch (invalid_argument) {
        cout << "Error: Input cannot be converted to an integer." << endl;
    } catch (out_of_range) {
        cout << "Error: Input is out of range for an integer." << endl;
    }

    return 0;
}
```
