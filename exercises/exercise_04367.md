## Exercise
Write a program that iterates over a dictionary and prints its key-value pairs.

## Reference Solution
```python
def print_dict_key_value_pairs(dictionary):
    """
    Iterate over a dictionary and print its key-value pairs.

    Args:
        dictionary (dict): The dictionary to iterate over.
    """
    for key, value in dictionary.items():
        print(f"Key: {key}, Value: {value}")


def main():
    # Example dictionary
    example_dict = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }

    print("Dictionary Key-Value Pairs:")
    print_dict_key_value_pairs(example_dict)


if __name__ == "__main__":
    main()
```
