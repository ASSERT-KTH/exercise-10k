## Exercise
Implement the quicksort algorithm to sort a list of integers in ascending order. The program should take a list of integers as input and output the sorted list.

## Reference Solution
```python
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

def main():
    n = int(input())
    arr = list(map(int, input().split()))
    print(quicksort(arr))

if __name__ == "__main__":
    main()
```
