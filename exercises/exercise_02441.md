## Exercise
Write a program that takes an array and an index as input and returns the element at that index.

## Reference Solution
```python
def findElement(arr, index):
    if index < len(arr):
        return arr[index]
    else:
        return -1

# Test the function
arr = [1, 2, 3, 4, 5]
index = 2
print(findElement(arr, index))
```
