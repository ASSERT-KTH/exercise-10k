## Exercise
Store a collection of key-value pairs in an object variable and print the values.

## Reference Solution
```python
# Create a dictionary (object) to store key-value pairs
data = {
    "name": "John Doe",
    "age": 30,
    "city": "New York",
    "country": "USA"
}

# Print the values
for key, value in data.items():
    print(f"{key}: {value}")
```

```python
class Person:
    def __init__(self, name, age, city, country):
        self.name = name
        self.age = age
        self.city = city
        self.country = country

    def print_values(self):
        for key, value in vars(self).items():
            print(f"{key}: {value}")

# Create an instance of the Person class
person = Person("John Doe", 30, "New York", "USA")

# Print the values
person.print_values()
```
