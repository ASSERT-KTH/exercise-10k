## Exercise
Write a function to return a dictionary sorted by its values

## Reference Solution
```python
def sort_dict_by_values(input_dict):
    """
    Returns a dictionary sorted by its values.

    Args:
        input_dict (dict): The dictionary to be sorted.

    Returns:
        dict: A dictionary sorted by its values.
    """
    return dict(sorted(input_dict.items(), key=lambda item: item[1]))

# Example usage:
input_dict = {'a': 3, 'b': 2, 'c': 1}
sorted_dict = sort_dict_by_values(input_dict)
print(sorted_dict)
```
