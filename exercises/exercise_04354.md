## Exercise
Write a program that checks a bank account balance and prints 'Low Balance' if the balance is below a certain threshold (e.g., $100), otherwise prints 'Sufficient Balance'

## Reference Solution
```python
def check_balance(balance, threshold=100):
    """
    Checks the bank account balance and prints 'Low Balance' if the balance is below a certain threshold,
    otherwise prints 'Sufficient Balance'.

    Args:
        balance (float): The current bank account balance.
        threshold (float, optional): The minimum balance required to avoid 'Low Balance' message. Defaults to 100.

    Returns:
        None
    """

    if balance < threshold:
        print("Low Balance")
    else:
        print("Sufficient Balance")


# Example usage:
def main():
    balance = float(input("Enter your bank account balance: $"))
    check_balance(balance)


if __name__ == "__main__":
    main()
```
