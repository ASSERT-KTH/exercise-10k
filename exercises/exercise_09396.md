## Exercise
Create a program that calculates the floor function of a given number.

## Reference Solution
```python
import math

def calculate_floor(number):
    """
    Calculates the floor function of a given number.

    Args:
        number (float): The input number.

    Returns:
        int: The floor of the input number.
    """
    return math.floor(number)

def main():
    # Get the input number from the user
    number = float(input("Enter a number: "))

    # Calculate the floor of the input number
    floor = calculate_floor(number)

    # Print the result
    print(f"The floor of {number} is {floor}")

if __name__ == "__main__":
    main()
```
