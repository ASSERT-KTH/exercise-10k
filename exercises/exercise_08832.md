## Exercise
Create a program that calculates the area and perimeter of a rectangle. The program should use two separate functions, one for calculating the area and one for calculating the perimeter. The length and width of the rectangle should be provided as input to the program.

## Reference Solution
```python
# Define functions
def area(length, width):
    return length * width

def perimeter(length, width):
    return 2 * (length + width)

# Main program
print("Rectangle Area and Perimeter Calculator")
print("----------------------------------------")

# Get input from user
length = int(input("Enter the length of the rectangle: "))
width = int(input("Enter the width of the rectangle: "))

# Calculate and display area and perimeter
print("Area:", area(length, width))
print("Perimeter:", perimeter(length, width))
```
