## Exercise
Write a program that finds the maximum number in a list of numbers using a for loop.

## Reference Solution
```python
def find_max(numbers):
    """
    This function finds the maximum number in a list of numbers.
    
    Args:
        numbers (list): A list of numbers.
    
    Returns:
        max_num (int or float): The maximum number in the list.
    """
    # Check if the list is empty
    if len(numbers) == 0:
        return None
    
    # Initialize max_num with the first element of the list
    max_num = numbers[0]
    
    # Iterate over the list of numbers
    for num in numbers:
        # Check if the current number is greater than max_num
        if num > max_num:
            # Update max_num if the current number is greater
            max_num = num
    
    # Return the maximum number
    return max_num

# Example usage:
numbers = [4, 2, 9, 6, 5, 1, 8, 3, 7]
max_num = find_max(numbers)
print("Maximum number in the list:", max_num)
```
