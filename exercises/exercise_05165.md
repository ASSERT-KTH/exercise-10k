## Exercise
Write a function to iterate over the keys and values of a dictionary and print them.

## Reference Solution
```python
def print_dict_keys_values(dictionary):
    """
    Iterate over the keys and values of a dictionary and print them.

    Args:
        dictionary (dict): The dictionary to iterate over.
    """
    for key, value in dictionary.items():
        print(f"Key: {key}, Value: {value}")

# Example usage:
def main():
    example_dict = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }
    print_dict_keys_values(example_dict)

if __name__ == "__main__":
    main()
```
