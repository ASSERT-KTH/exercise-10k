## Exercise
Create a program that calculates the percentage of a given number.

## Reference Solution
```python
def calculate_percentage(number, percentage):
    """
    Calculate the percentage of a given number.

    Args:
        number (float): The number to calculate the percentage of.
        percentage (float): The percentage to calculate.

    Returns:
        float: The calculated percentage.
    """
    return (percentage / 100) * number


def main():
    # Get user input
    number = float(input("Enter the number: "))
    percentage = float(input("Enter the percentage: "))

    # Calculate the percentage
    result = calculate_percentage(number, percentage)

    # Print the result
    print(f"{percentage}% of {number} is {result}")


if __name__ == "__main__":
    main()
```
