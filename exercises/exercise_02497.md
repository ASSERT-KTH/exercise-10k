## Exercise
Write a program to find the minimum element in a rotated sorted list of numbers

## Reference Solution
```python
def find_min(nums):
    """
    Find the minimum element in a rotated sorted list of numbers.

    Args:
        nums (list): A list of integers.

    Returns:
        int: The minimum element in the list.
    """
    left, right = 0, len(nums) - 1
    while left < right:
        mid = left + (right - left) // 2
        if nums[mid] > nums[right]:
            left = mid + 1
        else:
            right = mid
    return nums[left]

# Example usage
if __name__ == "__main__":
    nums = [4, 5, 6, 7, 0, 1, 2]
    min_element = find_min(nums)
    print("Minimum element:", min_element)
```
