## Exercise
Write a program that takes a string as input and returns a dictionary with the frequency of each word in the string.

## Reference Solution
```python
import re
from collections import Counter

def word_frequency(input_string):
    """
    Returns a dictionary with the frequency of each word in the input string.

    Args:
        input_string (str): The input string.

    Returns:
        dict: A dictionary with word frequencies.
    """
    # Convert to lowercase and remove punctuation
    cleaned_string = re.sub(r'[^\w\s]', '', input_string.lower())
    
    # Split into words
    words = cleaned_string.split()
    
    # Count word frequencies
    frequency = Counter(words)
    
    return dict(frequency)

# Example usage
if __name__ == "__main__":
    input_string = "This is a test string. The string is just a test."
    frequency = word_frequency(input_string)
    print(frequency)
```
