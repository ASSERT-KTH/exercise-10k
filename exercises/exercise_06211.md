## Exercise
Write a function that takes an array of integers as input, and returns a new array where each element is the cumulative sum of the elements up to that point in the input array.

## Reference Solution
```python
def cumulative_sum(arr):
    temp_list = []
    sum = 0
    for i in range(len(arr)):
        sum += arr[i]
        temp_list.append(sum)
    return temp_list

# Test the function
arr = [1, 2, 3, 4]
result = cumulative_sum(arr)
print(result)  # Output: [1, 3, 6, 10]
```
