## Exercise
Create a function to check if a given string is a pangram.

## Reference Solution
```python
def is_pangram(input_string):
    """
    Checks if a given string is a pangram.

    Args:
    input_string (str): The string to check.

    Returns:
    bool: True if the string is a pangram, False otherwise.
    """
    alphabet = "abcdefghijklmnopqrstuvwxyz"
    input_string = input_string.lower()
    for char in alphabet:
        if char not in input_string:
            return False
    return True

# Example usage:
print(is_pangram("The quick brown fox jumps over the lazy dog"))  # True
print(is_pangram("Hello World"))  # False
```
