## Exercise
Write a program to demonstrate type casting, including casting between numbers and strings.

## Reference Solution
```python
# Type Casting in Python

# Introduction
print("Type Casting in Python")
print("------------------------")

# Casting from string to integer
string_number = "123"
print(f"Original string: {string_number}")
integer_number = int(string_number)
print(f"Cast to integer: {integer_number} (type: {type(integer_number)})")

# Casting from string to float
string_float = "123.45"
print(f"\nOriginal string: {string_float}")
float_number = float(string_float)
print(f"Cast to float: {float_number} (type: {type(float_number)})")

# Casting from integer to string
integer_number = 123
print(f"\nOriginal integer: {integer_number}")
string_number = str(integer_number)
print(f"Cast to string: {string_number} (type: {type(string_number)})")

# Casting from float to string
float_number = 123.45
print(f"\nOriginal float: {float_number}")
string_float = str(float_number)
print(f"Cast to string: {string_float} (type: {type(string_float)})")

# Casting from integer to float
integer_number = 123
print(f"\nOriginal integer: {integer_number}")
float_number = float(integer_number)
print(f"Cast to float: {float_number} (type: {type(float_number)})")

# Casting from float to integer (note: this will truncate the decimal part)
float_number = 123.45
print(f"\nOriginal float: {float_number}")
integer_number = int(float_number)
print(f"Cast to integer: {integer_number} (type: {type(integer_number)})")
```
