## Exercise
Write a program that asks the user for a string and checks if it contains any vowels.

## Reference Solution
```python
def check_for_vowels(input_string):
    """
    Checks if a string contains any vowels.

    Args:
        input_string (str): The string to check.

    Returns:
        bool: True if the string contains at least one vowel, False otherwise.
    """
    vowels = 'aeiouAEIOU'
    for char in input_string:
        if char in vowels:
            return True
    return False

def main():
    # Ask the user for a string
    user_string = input("Please enter a string: ")

    # Check if the string contains any vowels
    if check_for_vowels(user_string):
        print("The string contains at least one vowel.")
    else:
        print("The string does not contain any vowels.")

if __name__ == "__main__":
    main()
```
