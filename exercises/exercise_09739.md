## Exercise
Write a program that takes an angle as input and returns its cosine.

## Reference Solution
```python
import math

def calculate_cosine(angle_in_degrees):
    """
    Calculate the cosine of an angle.

    Args:
        angle_in_degrees (float): The angle in degrees.

    Returns:
        float: The cosine of the angle.
    """
    # Convert the angle from degrees to radians
    angle_in_radians = math.radians(angle_in_degrees)

    # Calculate the cosine of the angle
    cosine = math.cos(angle_in_radians)

    return cosine

# Example usage
if __name__ == "__main__":
    angle_in_degrees = float(input("Enter an angle in degrees: "))
    cosine = calculate_cosine(angle_in_degrees)
    print(f"The cosine of {angle_in_degrees} degrees is {cosine}")
```
