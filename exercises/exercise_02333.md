## Exercise
Write a program to print a dictionary of key-value pairs to the screen.

## Reference Solution
```python
# Define a dictionary
my_dict = {
    "name": "John",
    "age": 30,
    "city": "New York",
    "job": "Engineer"
}

# Print the dictionary
print("Dictionary:")
print(my_dict)

# Print key-value pairs
print("\nKey-Value Pairs:")
for key, value in my_dict.items():
    print(f"{key}: {value}")
```
