## Exercise
Create a function that takes a number as an argument and returns 'Fizz' if the number is divisible by 3, 'Buzz' if it is divisible by 5, and 'Fizz Buzz' if it is divisible by both

## Reference Solution
```python
def fizz_buzz(n: int) -> str:
    """
    Returns 'Fizz' if n is divisible by 3, 'Buzz' if n is divisible by 5, 
    and 'Fizz Buzz' if n is divisible by both.

    Args:
        n (int): The input number.

    Returns:
        str: The Fizz Buzz result.
    """
    if n % 3 == 0 and n % 5 == 0:
        return 'Fizz Buzz'
    elif n % 3 == 0:
        return 'Fizz'
    elif n % 5 == 0:
        return 'Buzz'
    else:
        return str(n)

# Example usage
if __name__ == "__main__":
    print(fizz_buzz(15))  # Output: Fizz Buzz
    print(fizz_buzz(3))   # Output: Fizz
    print(fizz_buzz(5))   # Output: Buzz
    print(fizz_buzz(4))   # Output: 4
```
