## Exercise
Write a function that takes multiple arguments and returns a curried version of itself

## Reference Solution
```python
def curry(func):
    """
    Curry a function with multiple arguments.

    Args:
    func: The function to be curried.

    Returns:
    A curried version of the function.
    """
    def inner(*args, **kwargs):
        if len(args) + len(kwargs) >= func.__code__.co_argcount:
            return func(*args, **kwargs)
        return lambda *x, **y: inner(*(args + x), **dict(kwargs, **y))
    return inner

# Example usage:

def add(a, b, c):
    """
    Add three numbers.

    Args:
    a (int): The first number.
    b (int): The second number.
    c (int): The third number.

    Returns:
    int: The sum of the three numbers.
    """
    return a + b + c

curried_add = curry(add)

# Use the curried function
result = curried_add(1)(2)(3)
print(result)  # Output: 6
```
