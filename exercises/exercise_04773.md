## Exercise
Write a program that prints a sequence of numbers based on the user's input. If the user inputs 'asc', the program prints the numbers in ascending order. If the user inputs 'desc', the program prints the numbers in descending order.

## Reference Solution
```python
def solve():
    n = input("Enter 'asc' or 'desc': ")
    for i in range(10):
        if n == 'asc':
            print(i)
        elif n == 'desc':
            print(9 - i)
        else:
            print("Invalid input")

solve()
```
