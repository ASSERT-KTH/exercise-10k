## Exercise
Write a program that takes a list and a transformation function as input, and applies the transformation function to each element of the list.

## Reference Solution
```python
def transform_list(input_list, transformation_function):
    """
    Applies a transformation function to each element of the input list.

    Args:
        input_list: The list to be transformed.
        transformation_function: A function that takes one argument and returns a transformed value.

    Returns:
        A new list with the transformation function applied to each element of the input list.
    """
    transformed_list = []
    for element in input_list:
        transformed_element = transformation_function(element)
        transformed_list.append(transformed_element)
    return transformed_list

def double(x):
    return x * 2

input_list = [1, 2, 3, 4, 5]
output_list = transform_list(input_list, double)
print(output_list)  # Output: [2, 4, 6, 8, 10]
```
