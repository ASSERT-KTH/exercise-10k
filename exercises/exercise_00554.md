## Exercise
Write a program that takes a string as input and returns the string with each word reversed.

## Reference Solution
```python
def reverse_words(input_string):
    """
    Reverses each word in a given string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The string with each word reversed.
    """
    words = input_string.split()  # Split the string into words
    reversed_words = [word[::-1] for word in words]  # Reverse each word
    return ' '.join(reversed_words)  # Join the reversed words back into a string

# Example usage
def main():
    input_string = "Hello World"
    print("Input String:", input_string)
    print("Reversed Words:", reverse_words(input_string))

if __name__ == "__main__":
    main()
```
