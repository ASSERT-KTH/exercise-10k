## Exercise
Write a program that rotates the elements of an array by a specified number of positions. For example, if the array is [1, 2, 3, 4, 5] and the rotation is 2, the output should be [4, 5, 1, 2, 3].

## Reference Solution
```python
def rotate_array(arr, rotation):
    """
    Rotate the elements of an array by a specified number of positions.

    Args:
        arr (list): The input array.
        rotation (int): The number of positions to rotate.

    Returns:
        list: The rotated array.
    """
    rotation = rotation % len(arr)  # handle cases where rotation > len(arr)
    return arr[-rotation:] + arr[:-rotation]

# Example usage:
arr = [1, 2, 3, 4, 5]
rotation = 2
print(rotate_array(arr, rotation))  # Output: [4, 5, 1, 2, 3]
```
