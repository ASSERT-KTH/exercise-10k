## Exercise
Create a closure to keep track of a counter and increment it each time the function is called.

## Reference Solution
```python
def create_counter():
    """
    Creates a closure to keep track of a counter.

    Returns:
        function: A function that increments the counter when called.
    """
    counter = 0

    def increment():
        nonlocal counter
        counter += 1
        return counter

    return increment

# Example usage:
counter = create_counter()
print(counter())  # Output: 1
print(counter())  # Output: 2
print(counter())  # Output: 3
```
