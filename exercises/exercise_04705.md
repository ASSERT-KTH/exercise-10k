## Exercise
Iterate over the key-value pairs of a dictionary

## Reference Solution
```python
# Define a dictionary
my_dict = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

# Iterate over key-value pairs
for key, value in my_dict.items():
    print(f"{key}: {value}")
```

```python
# Iterate over keys
for key in my_dict.keys():
    print(key)

# Iterate over values
for value in my_dict.values():
    print(value)
```
