## Exercise
Create a function that finds and returns all duplicate elements from a given list. The function should be case-insensitive and consider 'abc' and 'ABC' as the same element. It should return a list of unique duplicate elements.

## Reference Solution
```python
def find_duplicate_elements(input_list):
    """
    This function finds and returns all duplicate elements from a given list.
    The function is case-insensitive and considers 'abc' and 'ABC' as the same element.
    It returns a list of unique duplicate elements.

    Parameters:
    input_list (list): The input list to find duplicate elements from.

    Returns:
    list: A list of unique duplicate elements.
    """

    # Convert the input list to a list of lower case strings to make the function case-insensitive
    lower_case_list = [element.lower() for element in input_list]

    # Initialize an empty dictionary to store the frequency of each element
    frequency_dict = {}

    # Iterate over the lower case list to count the frequency of each element
    for element in lower_case_list:
        # If the element is already in the dictionary, increment its count
        if element in frequency_dict:
            frequency_dict[element] += 1
        # If the element is not in the dictionary, add it with a count of 1
        else:
            frequency_dict[element] = 1

    # Initialize an empty list to store the duplicate elements
    duplicate_elements = []

    # Iterate over the frequency dictionary to find the duplicate elements
    for element, frequency in frequency_dict.items():
        # If the frequency of an element is more than 1, it's a duplicate element
        if frequency > 1:
            # Add the duplicate element to the list
            duplicate_elements.append(element)

    # Return the list of unique duplicate elements
    return duplicate_elements

# Test the function
input_list = ['abc', 'ABC', 'def', 'DEF', 'def']
print(find_duplicate_elements(input_list))  # Output: ['abc', 'def']
```
