## Exercise
Write a program that uses a while loop to reverse a string.

## Reference Solution
```python
def reverse_string(input_string):
    """
    Reverses a string using a while loop.

    Args:
        input_string (str): The input string to be reversed.

    Returns:
        str: The reversed string.
    """
    reversed_string = ""
    i = len(input_string) - 1

    while i >= 0:
        reversed_string += input_string[i]
        i -= 1

    return reversed_string

def main():
    input_string = "Hello, World!"
    print(f"Original String: {input_string}")
    print(f"Reversed String: {reverse_string(input_string)}")

if __name__ == "__main__":
    main()
```
