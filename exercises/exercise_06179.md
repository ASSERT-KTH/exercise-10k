## Exercise
Create a function that takes a number as input and returns the first n numbers in the Fibonacci sequence.

## Reference Solution
```python
def fibonacci(n):
    """
    Returns the first n numbers in the Fibonacci sequence.

    Args:
        n (int): The number of Fibonacci numbers to generate.

    Returns:
        list: A list of the first n Fibonacci numbers.
    """
    if n <= 0:
        return []
    elif n == 1:
        return [0]
    elif n == 2:
        return [0, 1]
    else:
        fib_sequence = [0, 1]
        while len(fib_sequence) < n:
            fib_sequence.append(fib_sequence[-1] + fib_sequence[-2])
        return fib_sequence

# Example usage
if __name__ == "__main__":
    n = 10
    print(f"The first {n} Fibonacci numbers are: {fibonacci(n)}")
```
