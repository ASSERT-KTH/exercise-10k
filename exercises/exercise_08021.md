## Exercise
Create a program that takes in a value and determines its data type. The program should then output the data type and perform a specific action based on the data type. For example, if the input is a string, the program should output the length of the string. If the input is an integer, the program should output the square of the integer. If the input is a float, the program should output the cube of the float.

## Reference Solution
```python
def determine_data_type(value):
    """
    Determine the data type of the input value and perform a specific action based on the data type.

    Args:
        value: The input value to determine the data type of.

    Returns:
        None
    """
    # Check the type of the input value
    data_type = type(value)

    # Output the data type
    print(f"The data type of the input is: {data_type.__name__}")

    # Perform a specific action based on the data type
    if data_type == str:
        # If the input is a string, output the length of the string
        print(f"The length of the string is: {len(value)}")
    elif data_type == int:
        # If the input is an integer, output the square of the integer
        print(f"The square of the integer is: {value ** 2}")
    elif data_type == float:
        # If the input is a float, output the cube of the float
        print(f"The cube of the float is: {value ** 3}")
    else:
        # If the input is of any other data type, output a message indicating that the data type is not supported
        print("The input data type is not supported.")

def main():
    # Example usage:
    values = ["hello", 5, 3.14, True, None]

    for value in values:
        print(f"Input: {value}")
        determine_data_type(value)
        print()  # Empty line for readability

if __name__ == "__main__":
    main()
```
