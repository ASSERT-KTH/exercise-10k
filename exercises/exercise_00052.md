## Exercise
Find the missing number in an array of integers from 1 to n

## Reference Solution
```python
def find_missing_number(nums):
    """
    Find the missing number in an array of integers from 1 to n.

    Args:
        nums (list): A list of integers from 1 to n with one number missing.

    Returns:
        int: The missing number.
    """
    n = len(nums) + 1
    expected_sum = n * (n + 1) // 2
    actual_sum = sum(nums)
    return expected_sum - actual_sum

# Example usage:
nums = [1, 2, 4, 5]
missing_number = find_missing_number(nums)
print(missing_number)  # Output: 3
```

def find_missing_number_xor(nums):
    """
    Find the missing number in an array of integers from 1 to n using XOR.

    Args:
        nums (list): A list of integers from 1 to n with one number missing.

    Returns:
        int: The missing number.
    """
    n = len(nums) + 1
    result = 0
    for i in range(1, n + 1):
        result ^= i
    for num in nums:
        result ^= num
    return result

# Example usage:
nums = [1, 2, 4, 5]
