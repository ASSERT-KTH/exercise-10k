## Exercise
Create a function that takes an array and a predicate function as input and returns True if the predicate function returns True for all elements in the array, False otherwise.

## Reference Solution
```python
def all_true(array, predicate):
    """
    This function checks if a given predicate function returns True for all elements in an array.

    Args:
        array (list): The input list to check.
        predicate (function): A function that takes one argument and returns a boolean value.

    Returns:
        bool: True if the predicate function returns True for all elements in the array, False otherwise.
    """

    # We iterate over each element in the array
    for element in array:
        
        # For each element, we apply the predicate function
        # If the predicate function returns False for any element, we immediately return False
        if not predicate(element):
            return False
    
    # If we have checked all elements and haven't returned False, it means the predicate function returned True for all elements
    # So, we return True
    return True

# Example usage:
# Define a predicate function that checks if a number is even
def is_even(num):
    return num % 2 == 0

# Create an array of numbers
numbers = [2, 4, 6, 8, 10]

# Check if all numbers in the array are even
print(all_true(numbers, is_even))  # Output: True
```
