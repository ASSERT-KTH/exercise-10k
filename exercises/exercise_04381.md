## Exercise
Declare and initialize variables of different data types

## Reference Solution
```python
# Integer variable
integer_var: int = 10

# Float variable
float_var: float = 3.14

# String variable
string_var: str = "Hello, World!"

# Boolean variable
boolean_var: bool = True

# List variable
list_var: list[int] = [1, 2, 3, 4, 5]

# Tuple variable
tuple_var: tuple[int, int, int] = (1, 2, 3)

# Dictionary variable
dict_var: dict[str, int] = {"one": 1, "two": 2, "three": 3}

# Set variable
set_var: set[int] = {1, 2, 3, 4, 5}

# Complex number variable
complex_var: complex = 3 + 4j

# Print the variables
print("Integer:", integer_var)
print("Float:", float_var)
print("String:", string_var)
print("Boolean:", boolean_var)
print("List:", list_var)
print("Tuple:", tuple_var)
print("Dictionary:", dict_var)
print("Set:", set_var)
print("Complex Number:", complex_var)
```
