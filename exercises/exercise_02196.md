## Exercise
Create a function to scramble the characters in a given string. The function should return the scrambled string.

## Reference Solution
```python
import random

def scramble_string(input_str):
    """
    Scramble the characters in a given string.

    Args:
        input_str (str): The string to be scrambled.

    Returns:
        str: The scrambled string.
    """
    # Convert the string into a list of characters
    char_list = list(input_str)
    
    # Use random.shuffle to shuffle the list of characters
    random.shuffle(char_list)
    
    # Join the characters back into a string
    scrambled_str = ''.join(char_list)
    
    return scrambled_str

# Example usage
if __name__ == "__main__":
    input_str = "hello"
    scrambled = scramble_string(input_str)
    print(f"Original string: {input_str}")
    print(f"Scrambled string: {scrambled}")
```
