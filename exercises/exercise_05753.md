## Exercise
Declare variables with different scopes and print their values

## Reference Solution
```python
# Global scope
global_variable = "I am a global variable"

def outer_function():
    # Enclosing scope
    enclosing_variable = "I am an enclosing variable"

    def inner_function():
        # Local scope
        local_variable = "I am a local variable"

        # Accessing variables from different scopes
        print("Accessing global variable:", global_variable)
        print("Accessing enclosing variable:", enclosing_variable)
        print("Accessing local variable:", local_variable)

    inner_function()

# Accessing global variable from global scope
print("Accessing global variable from global scope:", global_variable)

# Trying to access enclosing variable from global scope will raise an error
try:
    print(enclosing_variable)
except NameError:
    print("Error: Enclosing variable is not accessible from global scope")

# Calling the outer function
outer_function()
```
