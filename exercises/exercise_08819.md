## Exercise
Create a function that checks if a given string is a valid palindrome. The string can contain punctuation and spaces. The function should ignore case, spaces, and punctuation when checking for palindrome.

## Reference Solution
```python
def is_palindrome(s):
    """
    Checks if a given string is a valid palindrome. The string can contain punctuation and spaces.
    The function ignores case, spaces, and punctuation when checking for palindrome.

    Parameters:
    s (str): The input string to be checked.

    Returns:
    bool: True if the string is a palindrome, False otherwise.
    """

    # First, we convert the string to lower case to ignore case sensitivity
    s = s.lower()
    
    # Then, we remove all non-alphanumeric characters from the string
    # This will remove spaces and punctuation
    s = ''.join(char for char in s if char.isalnum())
    
    # Now, we compare the string with its reverse
    # If they are equal, the string is a palindrome
    # We use Python's slicing feature to reverse the string
    return s == s[::-1]

# Test the function
print(is_palindrome("A man, a plan, a canal: Panama"))  # True
print(is_palindrome("Not a palindrome"))  # False
```
