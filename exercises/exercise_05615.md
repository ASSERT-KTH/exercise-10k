## Exercise
Develop a program that generates anagrams of a given word, taking the user's input as the word to anagram.

## Reference Solution
```python
import itertools

def generate_anagrams(word):
    """
    Generate all possible anagrams of a given word.

    Args:
        word (str): The input word to anagram.

    Returns:
        list: A list of all possible anagrams.
    """
    # Generate permutations of the input word
    perms = [''.join(p) for p in itertools.permutations(word)]

    # Remove duplicates by converting to a set
    anagrams = set(perms)

    return anagrams

def main():
    # Get user input
    word = input("Enter a word to anagram: ")

    # Generate anagrams
    anagrams = generate_anagrams(word)

    # Print anagrams
    print("Anagrams:")
    for anagram in anagrams:
        print(anagram)

if __name__ == "__main__":
    main()
```
