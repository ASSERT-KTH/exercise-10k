## Exercise
Write a program that prints the ASCII value of a given character using a while loop.

## Reference Solution
```python
def print_ascii(char):
    """
    Prints the ASCII value of a given character.
    
    Args:
        char (str): A single character.
    """
    i = 0
    while i < len(char):
        print(f"The ASCII value of '{char[i]}' is {ord(char[i])}")
        i += 1

# Example usage:
char = input("Enter a character: ")
print_ascii(char)
```
