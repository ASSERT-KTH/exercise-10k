## Exercise
Write a function that uses the reduce function to apply a given function to all elements of a list.

## Reference Solution
```python
import functools
import operator

def apply_function_to_list(lst, func):
    """
    Applies a given function to all elements of a list using reduce.

    Args:
        lst (list): The list of elements to apply the function to.
        func (function): The function to apply to the elements.

    Returns:
        The result of applying the function to all elements.
    """
    return functools.reduce(func, lst)

# Example usage:
numbers = [1, 2, 3, 4, 5]
result = apply_function_to_list(numbers, operator.add)
print(result)  # Output: 15

# Example with a custom function:
def multiply(a, b):
    return a * b

numbers = [1, 2, 3, 4, 5]
result = apply_function_to_list(numbers, multiply)
print(result)  # Output: 120
```
