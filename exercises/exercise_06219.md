## Exercise
Write a function that takes an array of integers as input, and returns a new array where each element is the cumulative product of the elements up to that point in the input array.

## Reference Solution
```python
def cumulative_product(arr):
    res = []
    product = 1
    for x in arr:
        product *= x
        res.append(product)
    return res

# Example usage:
input_arr = [1, 2, 3, 4, 5]
output_arr = cumulative_product(input_arr)
print(output_arr)  # Output: [1, 2, 6, 24, 120]
```
