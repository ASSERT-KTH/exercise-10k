## Exercise
Given an array of strings, write a function to return the element at a specified index.

## Reference Solution
```python
def get_element_at_index(array, index):
    """
    Returns the element at a specified index in the given array.

    Args:
        array (list): The input array of strings.
        index (int): The index of the element to retrieve.

    Returns:
        str: The element at the specified index.

    Raises:
        IndexError: If the index is out of range.
    """
    if index < 0 or index >= len(array):
        raise IndexError("Index out of range")
    return array[index]

# Example usage
def main():
    array = ["apple", "banana", "cherry"]
    index = 1
    try:
        element = get_element_at_index(array, index)
        print(f"Element at index {index}: {element}")
    except IndexError as e:
        print(f"Error: {e}")

if __name__ == "__main__":
    main()
```
