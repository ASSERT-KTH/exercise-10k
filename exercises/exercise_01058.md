## Exercise
Write a program that takes a list as input, and returns the maximum element of the list.

## Reference Solution
```python
def find_max_element(input_list):
    """
    This function finds the maximum element in a given list.

    Args:
        input_list (list): A list of elements.

    Returns:
        The maximum element in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not input_list:
        raise ValueError("Input list is empty")

    return max(input_list)


# Example usage
if __name__ == "__main__":
    input_list = [4, 2, 9, 6, 5, 1]
    try:
        max_element = find_max_element(input_list)
        print(f"Maximum element: {max_element}")
    except ValueError as e:
        print(e)
```
