## Exercise
Write a program that takes three numbers as input and returns the result of the first number multiplied by the second, then added to the third.

## Reference Solution
```python
def calculate_result(a, b, c):
    """
    This function takes three numbers as input, multiplies the first two numbers, 
    and then adds the third number to the result.

    Args:
        a (float): The first number.
        b (float): The second number.
        c (float): The third number.

    Returns:
        float: The result of the calculation.
    """
    return a * b + c

# Example usage:
if __name__ == "__main__":
    num1 = float(input("Enter the first number: "))
    num2 = float(input("Enter the second number: "))
    num3 = float(input("Enter the third number: "))

    result = calculate_result(num1, num2, num3)
    print("The result is:", result)
```
