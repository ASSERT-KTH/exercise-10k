## Exercise
Create a function to count the occurrences of vowels in a given string.

## Reference Solution
```python
def count_vowels(input_string):
    """
    Counts the occurrences of vowels in a given string.

    Args:
        input_string (str): The input string to count vowels from.

    Returns:
        int: The count of vowels in the input string.
    """
    vowels = 'aeiou'
    return sum(1 for char in input_string.lower() if char in vowels)

# Example usage:
def main():
    input_str = "Hello World"
    vowel_count = count_vowels(input_str)
    print(f"The string '{input_str}' contains {vowel_count} vowels.")

if __name__ == "__main__":
    main()
```
