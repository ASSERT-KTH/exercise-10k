## Exercise
Write a program that takes a string and a shift value as input and returns the encrypted string using the Caesar cipher algorithm.

## Reference Solution
```python
def caesar_cipher(s, shift):
    result = ""
    for char in s:
        if char.isupper():
            result += chr((ord(char) - 65 + shift) % 26 + 65)
        elif char.islower():
            result += chr((ord(char) - 97 + shift) % 26 + 97)
        else:
            result += char
    return result

print(caesar_cipher("Python", 3))
print(caesar_cipher("Khoor", -3))
```
