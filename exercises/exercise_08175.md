## Exercise
Create two arrays of 3 integers and combine them into a list of pairs.

## Reference Solution
```python
# Create two arrays of 3 integers
array1 = [1, 2, 3]
array2 = [4, 5, 6]

# Combine them into a list of pairs using zip()
list_of_pairs = list(zip(array1, array2))

print(list_of_pairs)
```
