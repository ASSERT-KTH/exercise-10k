## Exercise
Write a program to print a table of numbers to the screen, with two columns.

## Reference Solution
```python
def print_table(rows):
    """
    Prints a table of numbers to the screen with two columns.

    Args:
        rows (int): The number of rows in the table.
    """
    print("| Number | Square |")
    print("|--------|--------|")
    for i in range(1, rows + 1):
        print(f"| {i:6} | {i**2:6} |")

def main():
    rows = int(input("Enter the number of rows: "))
    print_table(rows)

if __name__ == "__main__":
    main()
```
