## Exercise
Write a function to convert a list of tuples to a dictionary

## Reference Solution
```python
def tuples_to_dict(tuples_list):
    """
    Convert a list of tuples to a dictionary.
    
    Args:
    tuples_list (list): A list of tuples where each tuple contains a key-value pair.
    
    Returns:
    dict: A dictionary where the first element of each tuple is a key and the second element is a value.
    """
    return {key: value for key, value in tuples_list}

# Example usage:
tuples_list = [(1, 'a'), (2, 'b'), (3, 'c')]
print(tuples_to_dict(tuples_list))  # Output: {1: 'a', 2: 'b', 3: 'c'}
```
