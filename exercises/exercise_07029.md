## Exercise
Create a program that takes in a variable and its expected data type, then checks if the variable matches the expected data type. The program should handle the following data types: integer, float, string, boolean. If the variable matches the expected data type, print 'Valid'. Otherwise, print 'Invalid'.

## Reference Solution
```python
def check_data_type(var, expected_type):
    """
    Checks if a variable matches its expected data type.

    Args:
        var (any): The variable to be checked.
        expected_type (str): The expected data type. Can be 'integer', 'float', 'string', or 'boolean'.

    Returns:
        None
    """
    # Create a dictionary to map the expected data type strings to their corresponding Python types
    type_map = {
        'integer': int,
        'float': float,
        'string': str,
        'boolean': bool
    }

    # Check if the expected data type is valid
    if expected_type not in type_map:
        print("Invalid expected data type")
        return

    # Get the actual type of the variable
    actual_type = type(var)

    # Check if the actual type matches the expected type
    if actual_type == type_map[expected_type]:
        print('Valid')
    else:
        print('Invalid')


# Example usage:
if __name__ == "__main__":
    # Test with valid data types
    print("Testing with valid data types:")
    check_data_type(10, 'integer')
    check_data_type(10.5, 'float')
    check_data_type('hello', 'string')
    check_data_type(True, 'boolean')

    # Test with invalid data types
    print("\nTesting with invalid data types:")
    check_data_type('hello', 'integer')
    check_data_type(10, 'float')
    check_data_type(10.5, 'string')
    check_data_type('hello', 'boolean')

    # Test with invalid expected data type
    print("\nTesting with invalid expected data type:")
    check_data_type(10, 'list')
```
