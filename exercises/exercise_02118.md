## Exercise
Write a function to create a new dictionary with the given keys and a default value

## Reference Solution
```python
def create_dict(keys, default_value=None):
    """
    Creates a new dictionary with the given keys and a default value.

    Args:
        keys (list): A list of keys for the dictionary.
        default_value (any, optional): The default value for each key. Defaults to None.

    Returns:
        dict: A new dictionary with the given keys and default values.
    """
    return dict.fromkeys(keys, default_value)

# Example usage:
keys = ['a', 'b', 'c']
default_value = 0

new_dict = create_dict(keys, default_value)
print(new_dict)  # Output: {'a': 0, 'b': 0, 'c': 0}
```
