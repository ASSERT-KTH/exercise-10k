## Exercise
Write a program that swaps the values of two variables without using a third variable.

## Reference Solution
```python
def swap_variables(a, b):
    """
    Swap the values of two variables without using a third variable.

    Args:
        a (int): The first variable.
        b (int): The second variable.

    Returns:
        tuple: A tuple containing the swapped values.
    """
    print("Before swap: a = {}, b = {}".format(a, b))
    a = a + b  # Add b to a
    b = a - b  # Subtract original b from the sum to get original a
    a = a - b  # Subtract new b (original a) from the sum to get original b
    print("After swap: a = {}, b = {}".format(a, b))
    return a, b

# Example usage
if __name__ == "__main__":
    a = 5
    b = 10
    swap_variables(a, b)
```
