## Exercise
Given a list of dictionaries, write a function to create a chain map that allows looking up values from the dictionaries in order.

## Reference Solution
```python
from collections import ChainMap

def create_chain_map(dict_list):
    """
    This function creates a chain map from a list of dictionaries.
    
    Args:
        dict_list (list): A list of dictionaries.
    
    Returns:
        ChainMap: A chain map that allows looking up values from the dictionaries in order.
    """
    
    # First, we check if the input is a list. If not, we raise a TypeError.
    if not isinstance(dict_list, list):
        raise TypeError("Input must be a list of dictionaries.")
    
    # Then, we check if all elements in the list are dictionaries. If not, we raise a TypeError.
    if not all(isinstance(d, dict) for d in dict_list):
        raise TypeError("Input must be a list of dictionaries.")
    
    # Now, we can create the chain map. We pass the list of dictionaries to the ChainMap constructor.
    # The ChainMap constructor takes the dictionaries in the order they are passed and creates a chain map.
    # If a key is not found in the first dictionary, it looks up the key in the second dictionary, and so on.
    chain_map = ChainMap(*dict_list)
    
    # Finally, we return the chain map.
    return chain_map

# Example usage:
dict1 = {'a': 1, 'b': 2}
dict2 = {'b': 3, 'c': 4}
dict3 = {'c': 5, 'd': 6}

dict_list = [dict1, dict2, dict3]

chain_map = create_chain_map(dict_list)

print(chain_map['a'])  # Output: 1
print(chain_map['b'])  # Output: 2
print(chain_map['c'])  # Output: 4
print(chain_map['d'])  # Output: 6
```
