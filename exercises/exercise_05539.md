## Exercise
Declare a dictionary variable, assign key-value pairs to it, and then print its values.

## Reference Solution
```python
# Declare a dictionary variable
person_info = {
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}

# Print dictionary values
print("Dictionary Values:")
for key, value in person_info.items():
    print(f"{key}: {value}")

# Alternatively, print only dictionary values
print("\nOnly Dictionary Values:")
for value in person_info.values():
    print(value)
```
