## Exercise
Create a dictionary from a list of key-value pairs

## Reference Solution
```python
# List of key-value pairs
key_value_pairs = [(1, 'a'), (2, 'b'), (3, 'c')]

# Create a dictionary from the list of key-value pairs
dictionary = dict(key_value_pairs)

print(dictionary)  # Output: {1: 'a', 2: 'b', 3: 'c'}
```

```python
# List of key-value pairs
key_value_pairs = [(1, 'a'), (2, 'b'), (3, 'c')]

# Create a dictionary from the list of key-value pairs
dictionary = {key: value for key, value in key_value_pairs}

print(dictionary)  # Output: {1: 'a', 2: 'b', 3: 'c'}
```
