## Exercise
Write a program that takes a user input and determines its data type. The program should then print out the data type of the input. For example, if the user inputs '123', the program should print 'integer'. If the user inputs '123.45', the program should print 'float'. If the user inputs 'hello', the program should print 'string'.

## Reference Solution
```python
def check_data_type(user_input):
    """
    Checks the data type of the user input.

    Args:
        user_input (str): The input provided by the user.

    Returns:
        str: The data type of the input.
    """
    try:
        # Try to convert the input to an integer
        int(user_input)
        return 'integer'
    except ValueError:
        try:
            # If it's not an integer, try to convert it to a float
            float(user_input)
            return 'float'
        except ValueError:
            # If it's neither an integer nor a float, it's a string
            return 'string'

def main():
    # Get the user input
    user_input = input("Please enter a value: ")

    # Check the data type of the input
    data_type = check_data_type(user_input)

    # Print the data type
    print(f"The data type of '{user_input}' is {data_type}.")

if __name__ == "__main__":
    main()
```
