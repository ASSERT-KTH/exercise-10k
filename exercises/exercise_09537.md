## Exercise
Create a function that takes an array and a filter condition as input, and returns a new array containing only the elements that satisfy the condition. The condition is a function that takes one argument and returns a boolean value.

## Reference Solution
```python
def filter_array(array, condition):
    """
    This function filters an array based on a given condition.

    Args:
        array (list): The input array to be filtered.
        condition (function): A function that takes one argument and returns a boolean value.

    Returns:
        list: A new array containing only the elements that satisfy the condition.
    """

    # Initialize an empty list to store the filtered elements
    filtered_array = []

    # Iterate over each element in the input array
    for element in array:
        # Check if the current element satisfies the condition
        if condition(element):
            # If the condition is met, append the element to the filtered array
            filtered_array.append(element)

    # Return the filtered array
    return filtered_array

# Example usage:
# Define a condition function
def is_even(num):
    return num % 2 == 0

# Test the filter_array function
array = [1, 2, 3, 4, 5, 6]
filtered = filter_array(array, is_even)
print(filtered)  # Output: [2, 4, 6]
```
