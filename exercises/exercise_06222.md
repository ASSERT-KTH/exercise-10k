## Exercise
Write a function that finds the maximum value in a given array of integers.

## Reference Solution
```python
def find_max(array):
    """
    Finds the maximum value in a given array of integers.

    Args:
        array (list): A list of integers.

    Returns:
        int: The maximum value in the array.

    Raises:
        ValueError: If the input array is empty.
    """
    if not array:
        raise ValueError("Input array is empty")

    return max(array)


# Example usage:
if __name__ == "__main__":
    # Define an array of integers
    array = [4, 2, 9, 6, 5, 1]

    try:
        # Find the maximum value in the array
        max_value = find_max(array)
        print(f"The maximum value in the array is: {max_value}")
    except ValueError as e:
        print(e)
```
