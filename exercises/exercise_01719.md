## Exercise
Write a program that uses a for loop to check if a given password meets certain criteria (e.g. length, digits, uppercase and lowercase letters).

## Reference Solution
```python
def check_password(password):
    # Initialize flags
    has_uppercase = False
    has_lowercase = False
    has_digit = False

    # Check if password meets criteria
    if len(password) < 8:
        print("Password should be at least 8 characters long.")
        return False

    for char in password:
        if char.isupper():
            has_uppercase = True
        elif char.islower():
            has_lowercase = True
        elif char.isdigit():
            has_digit = True

    if not has_uppercase:
        print("Password should have at least one uppercase letter.")
        return False
    if not has_lowercase:
        print("Password should have at least one lowercase letter.")
        return False
    if not has_digit:
        print("Password should have at least one digit.")
        return False

    print("Password is valid.")
    return True


def main():
    password = input("Enter a password: ")
    check_password(password)


if __name__ == "__main__":
    main()
```
