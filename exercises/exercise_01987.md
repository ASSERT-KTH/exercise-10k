## Exercise
Write a program to determine the weather forecast based on the temperature using conditional statements

## Reference Solution
```python
def get_weather_forecast(temperature):
    """
    Returns the weather forecast based on the temperature.

    Args:
        temperature (float): The current temperature in Celsius.

    Returns:
        str: The weather forecast.
    """
    if temperature < 0:
        return "Freezing"
    elif temperature < 10:
        return "Cold"
    elif temperature < 20:
        return "Cool"
    elif temperature < 25:
        return "Mild"
    elif temperature < 30:
        return "Warm"
    elif temperature < 35:
        return "Hot"
    else:
        return "Extremely Hot"

def main():
    temperature = float(input("Enter the temperature in Celsius: "))
    forecast = get_weather_forecast(temperature)
    print(f"The weather forecast is: {forecast}")

if __name__ == "__main__":
    main()
```
