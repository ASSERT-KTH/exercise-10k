## Exercise
Write a program that takes a list of pairs of integers as input, and returns two separate lists containing the first and second elements of each pair, respectively.

## Reference Solution
```python
def solve(pairs):
    a = []
    b = []

    for i in range(len(pairs)):
        a.append(pairs[i][0])
        b.append(pairs[i][1])

    return a, b

# Test the function
pairs = [(1, 2), (3, 4), (5, 6)]
a, b = solve(pairs)
print(a)
print(b)
```
