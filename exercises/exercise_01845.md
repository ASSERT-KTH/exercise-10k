## Exercise
Write a program that takes an array of integers as input, and returns a new array containing the same elements in reverse order.

## Reference Solution
```python
def invertirLista(lista):
    invertida = []
    for e in lista:
        invertida = [e] + invertida
    return invertida

# Example usage:
print(invertirLista([1, 2, 3, 4, 5]))  # Output: [5, 4, 3, 2, 1]
```
