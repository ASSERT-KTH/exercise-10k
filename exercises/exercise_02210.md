## Exercise
Write a program that takes a phone number as input and formats it to a standard format (XXX-XXX-XXXX). Print the formatted phone number to the console.

## Reference Solution
```python
import re

def format_phone_number(phone_number: str) -> str:
    # Remove non-digit characters
    phone_number = re.sub(r"\D", "", phone_number)
    
    # Format the phone number
    formatted_phone_number = f"{phone_number[:3]}-{phone_number[3:6]}-{phone_number[6:]}"
    
    return formatted_phone_number

# Test the function
print(format_phone_number("(123) 456-7890"))
```
