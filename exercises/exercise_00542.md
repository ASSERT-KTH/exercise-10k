## Exercise
Write a program that takes a string as input and returns True if the string is a palindrome and False otherwise.

## Reference Solution
```python
def is_palindrome(s: str) -> bool:
    """
    Returns True if the input string is a palindrome, False otherwise.

    Args:
        s (str): The input string to check.

    Returns:
        bool: Whether the string is a palindrome.
    """
    s = ''.join(c for c in s if c.isalnum()).lower()  # remove non-alphanumeric characters and convert to lowercase
    return s == s[::-1]  # compare the string with its reverse

# Example usage
def main():
    test_cases = ["radar", "hello", "A man, a plan, a canal: Panama"]
    for test_case in test_cases:
        print(f"'{test_case}': {is_palindrome(test_case)}")

if __name__ == "__main__":
    main()
```
