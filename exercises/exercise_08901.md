## Exercise
Write a program to calculate the sum of all numbers in a given list.

## Reference Solution
```python
def calculate_list_sum(numbers):
    """
    Calculate the sum of all numbers in a given list.

    Args:
        numbers (list): A list of numbers.

    Returns:
        float: The sum of all numbers in the list.
    """
    return sum(numbers)


def main():
    # Example usage:
    numbers = [1, 2, 3, 4, 5]
    list_sum = calculate_list_sum(numbers)
    print(f"The sum of the list {numbers} is: {list_sum}")


if __name__ == "__main__":
    main()
```
