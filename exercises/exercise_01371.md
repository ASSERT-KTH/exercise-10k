## Exercise
Write a program to calculate the tip for a given bill amount and tip percentage.

## Reference Solution
```python
def calculate_tip(bill_amount, tip_percentage):
    """
    Calculate the tip for a given bill amount and tip percentage.

    Args:
        bill_amount (float): The amount of the bill.
        tip_percentage (float): The percentage of the tip.

    Returns:
        float: The calculated tip amount.
    """
    tip_amount = (bill_amount / 100) * tip_percentage
    return tip_amount

def main():
    # Get the bill amount from the user
    bill_amount = float(input("Enter the bill amount: "))

    # Get the tip percentage from the user
    tip_percentage = float(input("Enter the tip percentage (e.g., 15 for 15%): "))

    # Calculate the tip
    tip_amount = calculate_tip(bill_amount, tip_percentage)

    # Print the result
    print(f"Bill Amount: ${bill_amount:.2f}")
    print(f"Tip Percentage: {tip_percentage}%")
    print(f"Tip Amount: ${tip_amount:.2f}")
    print(f"Total Amount: ${bill_amount + tip_amount:.2f}")

if __name__ == "__main__":
    main()
```
