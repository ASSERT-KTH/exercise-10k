## Exercise
Write a generator that yields the Fibonacci sequence.

## Reference Solution
```python
def fibonacci():
    """
    Generator that yields the Fibonacci sequence.
    
    Yields:
        int: The next number in the Fibonacci sequence.
    """
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

# Example usage:
if __name__ == "__main__":
    fib = fibonacci()
    for _ in range(10):  # Print the first 10 Fibonacci numbers
        print(next(fib))
```
