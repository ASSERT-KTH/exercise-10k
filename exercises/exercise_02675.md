## Exercise
Write a program that takes a string as input and returns the string with the case of each character alternated.

## Reference Solution
```python
def alternate_case(input_string):
    """
    Alternate the case of each character in the input string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The string with the case of each character alternated.
    """
    result = ""
    upper = True  # Start with uppercase

    for char in input_string:
        if char.isalpha():  # Check if character is a letter
            if upper:
                result += char.upper()
            else:
                result += char.lower()
            upper = not upper  # Alternate case
        else:
            result += char  # Non-letter characters remain unchanged

    return result


# Example usage
def main():
    input_string = "Hello, World!"
    print("Input:", input_string)
    print("Output:", alternate_case(input_string))


if __name__ == "__main__":
    main()
```
