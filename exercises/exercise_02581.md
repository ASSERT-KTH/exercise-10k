## Exercise
Write a program to find the maximum value in a given list of numbers.

## Reference Solution
```python
def find_max_value(num_list):
    """
    Find the maximum value in a given list of numbers.

    Args:
        num_list (list): A list of numbers.

    Returns:
        int or float: The maximum value in the list.

    Raises:
        ValueError: If the input list is empty.
    """
    if not num_list:
        raise ValueError("Input list is empty")

    return max(num_list)


def main():
    # Example usage
    numbers = [12, 45, 7, 23, 56, 89, 34]
    max_value = find_max_value(numbers)
    print("Maximum value in the list:", max_value)


if __name__ == "__main__":
    main()
```
