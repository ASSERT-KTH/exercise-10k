## Exercise
Write a program that swaps the values of two variables without using a temporary variable

## Reference Solution
```python
def swap_variables(a, b):
    """
    Swaps the values of two variables without using a temporary variable.

    Args:
        a (int): The first variable.
        b (int): The second variable.

    Returns:
        tuple: A tuple containing the swapped values.
    """
    print(f"Before swap: a = {a}, b = {b}")

    # Swap variables using XOR operator
    a = a ^ b
    b = a ^ b
    a = a ^ b

    print(f"After swap: a = {a}, b = {b}")
    return a, b

# Example usage
a = 5
b = 10
swap_variables(a, b)
```
