## Exercise
Create a program that takes in two numbers and an arithmetic operation (+, -, *, /) as input, and then performs the specified operation on the numbers. The program should handle invalid operations and division by zero.

## Reference Solution
```python
# Author: [Your Name]
# Date: [Today's Date]

def calc(a, b, c):
    if c == '+':
        return a + b
    elif c == '-':
        return a - b
    elif c == '*':
        return a * b
    elif c == '/':
        if b != 0:
            return a / b
        else:
            return "error"
    else:
        return "error"

# Test cases
print(calc(10, 2, '+'))  # Output: 12
print(calc(10, 2, '-'))  # Output: 8
print(calc(10, 2, '*'))  # Output: 20
print(calc(10, 2, '/'))  # Output: 5.0
print(calc(10, 0, '/'))  # Output: error
print(calc(10, 2, '%'))  # Output: error
```
