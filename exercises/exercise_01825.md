## Exercise
Write a program to check if a given array contains any duplicate elements.

## Reference Solution
```python
def has_duplicates(array):
    """
    Checks if a given array contains any duplicate elements.

    Args:
        array (list): The input array to check.

    Returns:
        bool: True if the array contains duplicates, False otherwise.
    """
    return len(array) != len(set(array))

def main():
    # Example usage:
    arrays_to_check = [
        [1, 2, 3, 4, 5],
        [1, 2, 2, 3, 4],
        [1, 1, 1, 1, 1],
        ["apple", "banana", "orange", "apple"]
    ]

    for array in arrays_to_check:
        print(f"Array: {array} | Has duplicates: {has_duplicates(array)}")

if __name__ == "__main__":
    main()
```
