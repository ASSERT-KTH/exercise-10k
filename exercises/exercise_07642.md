## Exercise
Write a function that takes three numbers as input and returns the maximum value.

## Reference Solution
```python
def find_max_value(a: float, b: float, c: float) -> float:
    """
    Returns the maximum value among three input numbers.

    Args:
    a (float): The first number.
    b (float): The second number.
    c (float): The third number.

    Returns:
    float: The maximum value among a, b, and c.
    """
    return max(a, b, c)

# Example usage:
if __name__ == "__main__":
    print(find_max_value(10, 20, 30))  # Output: 30
    print(find_max_value(-5, 0, 5))    # Output: 5
    print(find_max_value(100, 50, 200)) # Output: 200
```
