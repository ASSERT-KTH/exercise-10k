## Exercise
Create a dictionary with 5 key-value pairs where keys are student names and values are their corresponding grades

## Reference Solution
```python
# Define the dictionary
student_grades = {
    "John Doe": 85,
    "Jane Smith": 92,
    "Emily Johnson": 78,
    "Michael Brown": 95,
    "Sarah Lee": 88
}

# Print the dictionary
print("Student Grades:")
for student, grade in student_grades.items():
    print(f"{student}: {grade}")
```
