## Exercise
Write a program to take a user's name and age as input and print a formatted string with their name and age.

## Reference Solution
```python
def print_name_and_age():
    # Get user's name and age as input
    name = input("Please enter your name: ")
    age = input("Please enter your age: ")

    # Print a formatted string with their name and age
    print(f"Hello, my name is {name} and I am {age} years old.")

# Call the function
print_name_and_age()
```
