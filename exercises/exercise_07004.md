## Exercise
Write a program that takes the total bill and the tip percentage as input and returns the total amount to pay.

## Reference Solution
```python
def calculate_total_bill(bill, tip_percentage):
    """
    Calculates the total amount to pay based on the total bill and the tip percentage.

    Args:
        bill (float): The total bill amount.
        tip_percentage (float): The tip percentage.

    Returns:
        float: The total amount to pay.
    """
    tip_amount = bill * (tip_percentage / 100)
    total_amount = bill + tip_amount
    return total_amount

def main():
    # Get user input
    bill = float(input("Enter the total bill: $"))
    tip_percentage = float(input("Enter the tip percentage (%): "))

    # Calculate total amount
    total_amount = calculate_total_bill(bill, tip_percentage)

    # Print result
    print(f"Total amount to pay: ${total_amount:.2f}")

if __name__ == "__main__":
    main()
```
