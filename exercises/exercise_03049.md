## Exercise
Write a function to calculate the grade of a student based on their score.

## Reference Solution
```python
def calculate_grade(score: float) -> str:
    """
    Calculate a student's grade based on their score.

    Args:
        score (float): The student's score.

    Returns:
        str: The student's grade.
    """
    if score >= 90:
        return "A"
    elif score >= 80:
        return "B"
    elif score >= 70:
        return "C"
    elif score >= 60:
        return "D"
    else:
        return "F"

# Example usage
def main():
    score = 85
    grade = calculate_grade(score)
    print(f"A score of {score} corresponds to a grade of {grade}.")

if __name__ == "__main__":
    main()
```
