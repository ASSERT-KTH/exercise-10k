## Exercise
Write a program that takes an array and a comparator function as input and returns the sorted array using the comparator.

## Reference Solution
```python
function sort(arr, comparator) {
  return arr.sort(comparator);
}

function compare(a, b) {
  if (a > b) return 1;
  if (b > a) return -1;
  return 0;
}

const arr = [1, 2, 3, 4, 5];
console.log(sort(arr, compare));
```
