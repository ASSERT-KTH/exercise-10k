## Exercise
Given a list of integers, remove duplicates while preserving the original order

## Reference Solution
```python
def remove_duplicates(nums):
    """
    Removes duplicates from a list of integers while preserving the original order.

    Args:
        nums (list): A list of integers.

    Returns:
        list: The list with duplicates removed.
    """
    seen = set()
    unique_nums = [num for num in nums if not (num in seen or seen.add(num))]
    return unique_nums

# Example usage:
numbers = [1, 2, 3, 2, 4, 5, 5, 6]
unique_numbers = remove_duplicates(numbers)
print(unique_numbers)  # Output: [1, 2, 3, 4, 5, 6]
```
