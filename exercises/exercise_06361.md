## Exercise
Write a program that takes an array and an index as input and returns the element at that index

## Reference Solution
```python
def get_element(array, index):
    """
    Returns the element at the specified index in the array.

    Args:
        array (list): The input array.
        index (int): The index of the element to return.

    Returns:
        The element at the specified index in the array.

    Raises:
        IndexError: If the index is out of range.
    """
    if index < 0 or index >= len(array):
        raise IndexError("Index out of range")
    return array[index]

# Example usage:
array = [1, 2, 3, 4, 5]
index = 2
try:
    element = get_element(array, index)
    print(f"The element at index {index} is: {element}")
except IndexError as e:
    print(f"Error: {e}")
```
