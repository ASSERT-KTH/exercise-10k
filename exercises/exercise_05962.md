## Exercise
Write a program that takes a list and two indices as input, and returns a slice of the list from the first index to the second index.

def slice_list(input_list, start_index, end_index):

def slice_list(input_list, start_index, end_index):
    if start_index < 0 or start_index >= len(input_list):
        raise ValueError("Start index out of range")
    if end_index < 0 or end_index > len(input_list):
        raise ValueError("End index out of range")
    output_list = input_list[start_index:end_index]
    return output_list
