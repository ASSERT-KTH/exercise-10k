## Exercise
Write a program to print a list of numbers from 1 to 10 to the screen, but only include numbers that are greater than 5.

## Reference Solution
```python
def print_numbers_greater_than_five():
    """
    Prints a list of numbers from 1 to 10, excluding numbers less than or equal to 5.
    """
    numbers = [i for i in range(1, 11) if i > 5]
    print(numbers)

# Example usage:
if __name__ == "__main__":
    print_numbers_greater_than_five()
```
