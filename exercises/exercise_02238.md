## Exercise
Write a program that finds the union of two lists.

## Reference Solution
```python
def union_of_lists(list1, list2):
    """
    Returns the union of two lists.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: A new list containing all unique elements from both input lists.
    """
    return list(set(list1 + list2))

def main():
    # Example usage:
    list1 = [1, 2, 3, 4]
    list2 = [3, 4, 5, 6]
    print("List 1:", list1)
    print("List 2:", list2)
    print("Union of List 1 and List 2:", union_of_lists(list1, list2))

if __name__ == "__main__":
    main()
```
