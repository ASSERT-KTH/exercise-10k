## Exercise
Write a function to pivot a dictionary from a long format to a wide format (e.g., transpose a dictionary).

## Reference Solution
```python
def pivot_dict(long_dict, index_key, column_key, value_key):
    """
    Pivot a dictionary from long format to wide format.

    Args:
    - long_dict (dict): Dictionary in long format.
    - index_key (str): Key for the index in the long format dictionary.
    - column_key (str): Key for the column in the long format dictionary.
    - value_key (str): Key for the value in the long format dictionary.

    Returns:
    - wide_dict (dict): Dictionary in wide format.
    """
    # Initialize an empty dictionary to store the pivoted data
    wide_dict = {}

    # Iterate over each item in the long format dictionary
    for item in long_dict:
        # Get the index, column, and value for the current item
        index = item[index_key]
        column = item[column_key]
        value = item[value_key]

        # If the index is not already in the wide dictionary, add it
        if index not in wide_dict:
            wide_dict[index] = {}

        # Add the value to the wide dictionary at the correct index and column
        wide_dict[index][column] = value

    return wide_dict

# Example usage:
long_dict = [
    {"id": 1, "name": "A", "value": 10},
    {"id": 1, "name": "B", "value": 20},
    {"id": 2, "name": "A", "value": 30},
    {"id": 2, "name": "B", "value": 40}
]

wide_dict = pivot_dict(long_dict, "id", "name", "value")
print(wide_dict)
```
