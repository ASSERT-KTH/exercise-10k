## Exercise
Create a program that finds the maximum contiguous subarray sum in a given list of numbers. Use a for loop to iterate over the numbers.

## Reference Solution
```python
def max_subarray_sum(numbers):
    """
    Finds the maximum contiguous subarray sum in a given list of numbers.

    Args:
        numbers (list): A list of integers.

    Returns:
        int: The maximum contiguous subarray sum.
    """
    if not numbers:
        return 0

    max_current = max_global = numbers[0]
    for num in numbers[1:]:
        # Update the maximum sum ending at the current position
        max_current = max(num, max_current + num)
        
        # Update the maximum sum found so far
        max_global = max(max_global, max_current)

    return max_global

# Example usage
numbers = [-2, -3, 4, -1, -2, 1, 5, -3]
max_sum = max_subarray_sum(numbers)
print(f"Maximum contiguous subarray sum: {max_sum}")
```
