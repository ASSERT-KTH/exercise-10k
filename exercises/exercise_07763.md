## Exercise
Write a program that takes a string as input, reverses the order of its characters, and removes any punctuation. The program should handle strings with a mix of uppercase and lowercase letters, numbers, and special characters.

## Reference Solution
```python
#include <iostream>
#include <string>
#include <cctype>

int main() {
    std::string str;

    std::cout << "Please enter a string: ";
    std::getline(std::cin, str);

    for (int i = str.length() - 1; i >= 0; i--) {
        if (std::isalnum(str[i])) {
            std::cout << str[i];
        }
    }

    std::cout << std::endl;

    return 0;
}
```
