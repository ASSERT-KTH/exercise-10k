## Exercise
Write a program that swaps the values of two variables without using a temporary variable.

## Reference Solution
```python
def swap_variables(a, b):
    """
    Swap the values of two variables without using a temporary variable.
    
    Args:
    a (int): The first variable.
    b (int): The second variable.
    
    Returns:
    tuple: A tuple containing the swapped values.
    """
    # Print original values
    print(f"Original values: a = {a}, b = {b}")
    
    # Swap values using arithmetic operations
    a = a + b  # a now holds the sum of a and b
    b = a - b  # b now holds the original value of a
    a = a - b  # a now holds the original value of b
    
    # Print swapped values
    print(f"Swapped values: a = {a}, b = {b}")
    
    return a, b

# Example usage
if __name__ == "__main__":
    a = 5
    b = 10
    swap_variables(a, b)
```
