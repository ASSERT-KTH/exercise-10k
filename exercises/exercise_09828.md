## Exercise
Create a program that calculates the area and perimeter of a rectangle using functions. The function to calculate the area should take the length and width as parameters, and the function to calculate the perimeter should take the length and width as parameters. The program should then print out the calculated area and perimeter.

## Reference Solution
```python
#include <iostream>
using namespace std;

int area(int l, int w) {
    return l * w;
}

int perimeter(int l, int w) {
    return 2 * (l + w);
}

int main() {
    int length = 5;
    int width = 3;
    int a = area(length, width);
    int p = perimeter(length, width);
    cout << "Area: " << a << endl;
    cout << "Perimeter: " << p << endl;
    return 0;
}
```
