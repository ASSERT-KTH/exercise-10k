## Exercise
Write a program that takes in a list of variables and their corresponding data types, then checks if the given values match their respective data types. If a value does not match its data type, the program should print an error message indicating the variable name and its expected data type.

## Reference Solution
```python
def check_data_types(variables):
    """
    This function checks if the given values match their respective data types.

    Args:
        variables (dict): A dictionary containing variable names as keys and tuples of (value, data_type) as values.

    Returns:
        None
    """

    # Iterate over each variable in the input dictionary
    for variable, (value, data_type) in variables.items():
        
        # Initialize a flag to track if the value matches the data type
        is_match = False
        
        # Check the data type and compare it with the given value
        if data_type == 'int':
            # Try to convert the value to an integer
            try:
                int(value)
                is_match = True
            except ValueError:
                pass
        
        elif data_type == 'float':
            # Try to convert the value to a float
            try:
                float(value)
                is_match = True
            except ValueError:
                pass
        
        elif data_type == 'str':
            # All values can be converted to strings, so this is always a match
            is_match = True
        
        elif data_type == 'bool':
            # Check if the value is a boolean
            if value.lower() == 'true' or value.lower() == 'false':
                is_match = True
        
        # If the value does not match the data type, print an error message
        if not is_match:
            print(f"Error: Variable '{variable}' has value '{value}' which does not match its expected data type '{data_type}'")


# Example usage:
variables = {
    'age': ('25', 'int'),
    'height': ('175.5', 'float'),
    'name': ('John Doe', 'str'),
    'is_admin': ('true', 'bool'),
    'invalid_int': ('abc', 'int'),
    'invalid_float': ('def', 'float'),
    'invalid_bool': ('maybe', 'bool')
}

check_data_types(variables)
```
