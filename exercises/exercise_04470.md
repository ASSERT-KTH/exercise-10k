## Exercise
Write a program to print the first N prime numbers using a while loop

## Reference Solution
```python
def is_prime(num):
    """Check if a number is prime."""
    if num < 2:
        return False
    for i in range(2, int(num**0.5) + 1):
        if num % i == 0:
            return False
    return True

def print_primes(n):
    """Print the first N prime numbers."""
    count = 0
    num = 2
    while count < n:
        if is_prime(num):
            print(num, end=" ")
            count += 1
        num += 1

# Example usage:
n = int(input("Enter the number of prime numbers to print: "))
print_primes(n)
```
