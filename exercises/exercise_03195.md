## Exercise
Create a program that takes a time in one format (e.g., 12-hour) and uses if/else statements to convert it to another format (e.g., 24-hour).

## Reference Solution
```python
def convert_time(time_str):
    """
    Converts a time from 12-hour format to 24-hour format.

    Args:
        time_str (str): Time in 12-hour format (e.g., "08:30 AM")

    Returns:
        str: Time in 24-hour format (e.g., "08:30")
    """
    # Split the time string into hours, minutes, and AM/PM
    time_parts = time_str.split()
    hours, minutes = map(int, time_parts[0].split(":"))
    am_pm = time_parts[1]

    # Convert the time to 24-hour format
    if am_pm == "AM":
        if hours == 12:
            hours = 0
    else:
        if hours != 12:
            hours += 12

    # Format the time as a string
    return f"{hours:02d}:{minutes:02d}"

def main():
    # Test the function
    time_str = input("Enter a time in 12-hour format (e.g., 08:30 AM): ")
    converted_time = convert_time(time_str)
    print(f"Time in 24-hour format: {converted_time}")

if __name__ == "__main__":
    main()
```
