## Exercise
Write a program that finds the maximum sum of a subarray within a given one-dimensional array of numbers. The subarray must be a contiguous subset of the original array.

## Reference Solution
```python
def max_subarray_sum(arr):
    """
    Finds the maximum sum of a subarray within a given one-dimensional array of numbers.

    Args:
        arr (list): The input array.

    Returns:
        int: The maximum sum of a subarray.
    """
    max_sum = float('-inf')  # Initialize max_sum to negative infinity
    current_sum = 0  # Initialize current_sum to 0

    for num in arr:
        current_sum = max(num, current_sum + num)  # Update current_sum
        max_sum = max(max_sum, current_sum)  # Update max_sum

    return max_sum

# Example usage
arr = [-2, -3, 4, -1, -2, 1, 5, -3]
max_sum = max_subarray_sum(arr)
print("Maximum subarray sum:", max_sum)
```
