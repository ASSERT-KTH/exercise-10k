## Exercise
Given a dictionary {'name': 'Isabel', 'age': 30, 'city': 'Hobart'}, write a program to print the number of key-value pairs in the dictionary.

## Reference Solution
```python
def print_dict_length(dictionary):
    """Print the number of key-value pairs in a dictionary."""
    print(f"Number of key-value pairs: {len(dictionary)}")

def main():
    # Example dictionary
    person = {'name': 'Isabel', 'age': 30, 'city': 'Hobart'}
    
    # Print the number of key-value pairs
    print_dict_length(person)

if __name__ == "__main__":
    main()
```
