## Exercise
Write a program that takes a person's age as input, and prints 'Child' if the person is under 13, 'Teen' if the person is between 13 and 19, and 'Adult' if the person is 20 or older.

## Reference Solution
```python
# This is a comment explaining the purpose of the program
# It takes a person's age as input and prints their age group

# The program starts here
age = int(input("Enter your age: "))
if age < 13:
    print("Child")
elif age >= 13 and age <= 19:
    print("Teen")
else:
    print("Adult")
```
