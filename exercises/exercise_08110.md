## Exercise
Write a program that takes a string as input and returns the number of words in the string.

## Reference Solution
```python
def count_words(input_string):
    """
    Returns the number of words in the input string.

    Args:
        input_string (str): The input string.

    Returns:
        int: The number of words in the input string.
    """
    words = input_string.split()
    return len(words)

def main():
    input_string = input("Enter a string: ")
    word_count = count_words(input_string)
    print(f"The string contains {word_count} words.")

if __name__ == "__main__":
    main()
```
