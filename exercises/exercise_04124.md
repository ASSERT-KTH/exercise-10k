## Exercise
Determine the range of values for different data types, such as integers and floating-point numbers

## Reference Solution
```python
public class DataTypeExample {
    public static void main(String[] args) {
        byte b = 127; // 8-bit signed integer
        short s = 32767; // 16-bit signed integer
        int i = 2147483647; // 32-bit signed integer
        long l = 9223372036854775807L; // 64-bit signed integer
        float f = 3.4E+38f; // 32-bit floating-point number
        double d = 1.8E+308; // 64-bit floating-point number
        System.out.println("Byte: " + b);
        System.out.println("Short: " + s);
        System.out.println("Int: " + i);
        System.out.println("Long: " + l);
        System.out.println("Float: " + f);
        System.out.println("Double: " + d);
    }
}
```
