## Exercise
Understand how auto-boxing and unboxing work by creating examples of each and printing their values.

## Reference Solution
```python
public class AutoBoxingUnboxing {
    public static void main(String[] args) {
        // Auto-boxing
        int primitiveInt = 10;
        Integer boxedInt = primitiveInt; // auto-boxing
        System.out.println("Boxed Int: " + boxedInt);

        double primitiveDouble = 20.5;
        Double boxedDouble = primitiveDouble; // auto-boxing
        System.out.println("Boxed Double: " + boxedDouble);

        // Unboxing
        Integer boxedIntValue = 30;
        int unboxedInt = boxedIntValue; // unboxing
        System.out.println("Unboxed Int: " + unboxedInt);

        Double boxedDoubleValue = 40.5;
        double unboxedDouble = boxedDoubleValue; // unboxing
        System.out.println("Unboxed Double: " + unboxedDouble);

        // Null Pointer Exception
        Integer nullBoxedInt = null;
        try {
            int nullUnboxedInt = nullBoxedInt; // unboxing
            System.out.println("Null Unboxed Int: " + nullUnboxedInt);
        } catch (NullPointerException e) {
            System.out.println("Null Pointer Exception caught: " + e.getMessage());
        }
    }
}
```
