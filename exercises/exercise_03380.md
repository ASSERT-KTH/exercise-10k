## Exercise
Use a for loop to generate a list of powers of 2 up to a given limit

## Reference Solution
```python
def generate_powers_of_2(limit):
    """
    Generates a list of powers of 2 up to a given limit.
    
    Args:
        limit (int): The upper limit for the powers of 2.
    
    Returns:
        list: A list of powers of 2 up to the given limit.
    """
    powers_of_2 = []
    i = 0
    while 2 ** i <= limit:
        powers_of_2.append(2 ** i)
        i += 1
    return powers_of_2

# However, the problem specifically asks for a for loop. 
# We can achieve this by calculating the number of iterations beforehand.
import math

def generate_powers_of_2_for_loop(limit):
    """
    Generates a list of powers of 2 up to a given limit using a for loop.
    
    Args:
        limit (int): The upper limit for the powers of 2.
    
    Returns:
        list: A list of powers of 2 up to the given limit.
    """
    max_power = math.floor(math.log2(limit))
    powers_of_2 = []
    for i in range(max_power + 1):
        powers_of_2.append(2 ** i)
    return powers_of_2

# Example usage
```
