## Exercise
Write a function that takes a variable number of arguments and returns their sum

## Reference Solution
```python
def sum_variables(*args):
    """
    This function takes a variable number of arguments and returns their sum.

    Args:
        *args: A variable number of arguments.

    Returns:
        The sum of all arguments.

    Raises:
        TypeError: If any argument is not a number.
    """
    total = 0
    for arg in args:
        if not isinstance(arg, (int, float)):
            raise TypeError("All arguments must be numbers")
        total += arg
    return total

# Example usage:
print(sum_variables(1, 2, 3, 4, 5))  # Output: 15
print(sum_variables(10.5, 20.7))  # Output: 31.2
```
