## Exercise
Write a program that takes a string and an integer as input, and returns the encrypted string using the Caesar cipher algorithm, where each character is shifted by the given integer.

## Reference Solution
```python
def caesar_cipher(s, shift):
    result = ""
    for char in s:
        if char.isalpha():
            ascii_offset = 65
            if char.islower():
                ascii_offset = 97
            result += chr((ord(char) - ascii_offset + shift) % 26 + ascii_offset)
        else:
            result += char
    return result

s = input("Enter a string: ")
shift = int(input("Enter a shift value: "))
print(caesar_cipher(s, shift))
```
