## Exercise
Write a function that finds the index of an element in a list.

## Reference Solution
```python
def find_index(lst, element):
    """
    Find the index of an element in a list.

    Args:
        lst (list): The list to search.
        element: The element to find.

    Returns:
        int: The index of the element if found.
        None: If the element is not found.

    Raises:
        ValueError: If the element is not found in the list.
    """
    try:
        return lst.index(element)
    except ValueError:
        return None

# Example usage
my_list = [1, 2, 3, 4, 5]
print(find_index(my_list, 3))  # Output: 2
print(find_index(my_list, 6))  # Output: None
```

```python
def find_all_indices(lst, element):
    """
    Find all indices of an element in a list.

    Args:
        lst (list): The list to search.
        element: The element to find.

    Returns:
        list: A list of indices where the element is found.
    """
    return [i for i, x in enumerate(lst) if x == element]

# Example usage
my_list = [1, 2, 3, 3, 4, 3, 5]
print(find_all_indices(my_list, 3))  # Output: [2, 3, 5]
```
