## Exercise
Create a program that prints a given string in reverse order using a while loop.

## Reference Solution
```python
def print_reverse_string(input_str):
    """
    Prints a given string in reverse order using a while loop.
    
    Args:
        input_str (str): The string to be reversed.
    """
    i = len(input_str) - 1  # Initialize the index to the last character
    while i >= 0:  # Continue until the index is less than 0
        print(input_str[i], end='')  # Print the character at the current index
        i -= 1  # Decrement the index

# Example usage
input_str = "Hello, World!"
print("Original string: ", input_str)
print("Reversed string: ", end='')
print_reverse_string(input_str)
```
