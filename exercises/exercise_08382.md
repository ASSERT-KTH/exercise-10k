## Exercise
Write a program that calculates the power tower of a given number (i.e., the result of raising the number to the power of itself repeatedly).

## Reference Solution
```python
public class PowerTower {
    public static void main(String[] args) {
        System.out.println(tower(3, 4));
    }

    public static double tower(double a, int n) {
        if (n == 1) {
            return a;
        } else if (n == 0) {
            return 1;
        } else {
            return Math.pow(a, tower(a, n - 1));
        }
    }
}
```
