## Exercise
Write a program that takes an array of integers as input and returns the maximum value in the array. The program should iterate through the array, comparing each value to the current maximum, and update the maximum if a larger value is found.

## Reference Solution
```python
#include <iostream>
using namespace std;

int max(int arr[], int n) {
    int max = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    return max;
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int n = sizeof(arr) / sizeof(arr[0]);
    int result = max(arr, n);
    cout << "Maximum value: " << result << endl;
    return 0;
}
```
