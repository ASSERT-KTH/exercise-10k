## Exercise
Given an array of integers, write a program to replace a specific element with a new value.

## Reference Solution
```python
def replace_element(array, target, new_value):
    """
    Replaces a specific element in the array with a new value.

    Args:
        array (list): The input array of integers.
        target (int): The element to be replaced.
        new_value (int): The new value to replace the target element.

    Returns:
        list: The modified array with the target element replaced.
    """
    # Create a copy of the original array to avoid modifying it directly
    modified_array = array.copy()

    # Iterate over the array to find the target element
    for i, element in enumerate(modified_array):
        if element == target:
            # Replace the target element with the new value
            modified_array[i] = new_value

    return modified_array


# Example usage
if __name__ == "__main__":
    array = [1, 2, 3, 4, 5]
    target = 3
    new_value = 10

    print("Original array:", array)
    print("Target element:", target)
    print("New value:", new_value)

    modified_array = replace_element(array, target, new_value)
    print("Modified array:", modified_array)
```
