## Exercise
Perform basic arithmetic operations on complex numbers using variables and print the results.

## Reference Solution
```python
# Define two complex numbers
num1 = 3 + 4j
num2 = 2 - 5j

# Perform addition
result_addition = num1 + num2
print(f"Addition: {num1} + {num2} = {result_addition}")

# Perform subtraction
result_subtraction = num1 - num2
print(f"Subtraction: {num1} - {num2} = {result_subtraction}")

# Perform multiplication
result_multiplication = num1 * num2
print(f"Multiplication: {num1} * {num2} = {result_multiplication}")

# Perform division
result_division = num1 / num2
print(f"Division: {num1} / {num2} = {result_division}")

# Perform modulus (absolute value)
result_modulus = abs(num1)
print(f"Modulus of {num1}: {result_modulus}")

# Perform conjugate
result_conjugate = num1.conjugate()
print(f"Conjugate of {num1}: {result_conjugate}")
```
