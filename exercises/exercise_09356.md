## Exercise
Declare an array to store a list of numbers, then print out the array values.

## Reference Solution
```python
# Declare an array to store a list of numbers
numbers = [1, 2, 3, 4, 5]

# Print out the array values
print("Array values:")
for i, num in enumerate(numbers):
    print(f"Index {i}: {num}")

# Alternatively, you can print the entire array at once
print("\nEntire array:", numbers)
```
