## Exercise
Write a function that takes a string and two integers as input and returns the substring from the start index to the end index.

## Reference Solution
```python
def substring_to_index(text, start, end):
    """
    Returns the substring of text from the start index to the end index.
    """
    return text[start:end]

# Example usage:
print(substring_to_index("Hello, World!", 0, 5))  # Output: "Hello"
```
