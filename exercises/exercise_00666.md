## Exercise
Given a dictionary {'name': 'David', 'age': 25, 'city': 'Beijing'}, write a program to print all the key-value pairs in the dictionary.

## Reference Solution
```python
def print_dict_key_value_pairs(dictionary):
    """Prints all key-value pairs in a dictionary."""
    for key, value in dictionary.items():
        print(f"{key}: {value}")

def main():
    # Example dictionary
    person = {'name': 'David', 'age': 25, 'city': 'Beijing'}
    
    print("Dictionary Key-Value Pairs:")
    print_dict_key_value_pairs(person)

if __name__ == "__main__":
    main()
```
