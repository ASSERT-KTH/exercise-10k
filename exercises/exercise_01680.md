## Exercise
Develop a program that takes a password as input and checks its strength based on the following criteria: the password should have at least 8 characters, should have at least one uppercase letter, one lowercase letter, one digit, and one special character using if/else statements

## Reference Solution
```python
import re

def check_password_strength(password):
    # Define the criteria
    criteria = {
        "length": len(password) >= 8,
        "uppercase": any(char.isupper() for char in password),
        "lowercase": any(char.islower() for char in password),
        "digit": any(char.isdigit() for char in password),
        "special_char": re.search("[^A-Za-z0-9]", password) is not None
    }

    # Check each criterion
    if criteria["length"] and criteria["uppercase"] and criteria["lowercase"] and criteria["digit"] and criteria["special_char"]:
        print("Password is strong.")
    else:
        print("Password is weak. Please ensure it meets the following criteria:")
        if not criteria["length"]:
            print("- At least 8 characters.")
        if not criteria["uppercase"]:
            print("- At least one uppercase letter.")
        if not criteria["lowercase"]:
            print("- At least one lowercase letter.")
        if not criteria["digit"]:
            print("- At least one digit.")
        if not criteria["special_char"]:
            print("- At least one special character.")

def main():
    password = input("Enter a password: ")
    check_password_strength(password)

if __name__ == "__main__":
    main()
```
