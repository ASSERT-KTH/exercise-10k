## Exercise
Write a program that rotates a list by a given number of positions using a for loop.

## Reference Solution
```python
def rotate_list(input_list, num_positions):
    """
    Rotate a list by a given number of positions using a for loop.

    Args:
        input_list (list): The list to be rotated.
        num_positions (int): The number of positions to rotate the list.

    Returns:
        list: The rotated list.
    """
    num_positions = num_positions % len(input_list)
    result = [0] * len(input_list)
    
    # Calculate the starting position for the result list
    start_position = len(input_list) - num_positions
    
    # Copy elements from the input list to the result list
    for i in range(len(input_list)):
        result[i] = input_list[(start_position + i) % len(input_list)]
    
    return result

# Example usage:
input_list = [1, 2, 3, 4, 5]
num_positions = 2
rotated_list = rotate_list(input_list, num_positions)

print("Original List:", input_list)
print("Rotated List:", rotated_list)
```
