## Exercise
Create a program that takes a string as input and outputs the number of vowels in the string.

## Reference Solution
```python
def count_vowels(input_string):
    """
    Counts the number of vowels in a given string.

    Args:
        input_string (str): The string to count vowels from.

    Returns:
        int: The number of vowels in the string.
    """
    vowels = 'aeiouAEIOU'
    return sum(1 for char in input_string if char in vowels)


def main():
    input_string = input("Enter a string: ")
    vowel_count = count_vowels(input_string)
    print(f"The string '{input_string}' contains {vowel_count} vowels.")


if __name__ == "__main__":
    main()
```
