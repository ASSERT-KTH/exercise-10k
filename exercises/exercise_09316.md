## Exercise
Create a function that takes a string as input and returns the string translated to Pig Latin.

## Reference Solution
```python
def translate_to_pig_latin(input_string):
    """
    Translates a given string into Pig Latin.

    Args:
        input_string (str): The input string to be translated.

    Returns:
        str: The translated string in Pig Latin.
    """
    vowels = 'aeiou'
    words = input_string.split()

    translated_words = []
    for word in words:
        # Check if the word starts with a vowel
        if word[0].lower() in vowels:
            # If it starts with a vowel, add 'way' to the end
            translated_word = word + 'way'
        else:
            # If it doesn't start with a vowel, move the consonant cluster to the end and add 'ay'
            consonant_cluster = ''
            for char in word:
                if char.lower() not in vowels:
                    consonant_cluster += char
                else:
                    break
            translated_word = word[len(consonant_cluster):] + consonant_cluster + 'ay'
        
        translated_words.append(translated_word)

    return ' '.join(translated_words)
```
