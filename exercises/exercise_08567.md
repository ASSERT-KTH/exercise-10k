## Exercise
Create a dictionary with 3 key-value pairs and convert it to a set of values.

## Reference Solution
```python
# Create a dictionary with 3 key-value pairs
my_dict = {
    "key1": "value1",
    "key2": "value2",
    "key3": "value3"
}

# Convert the dictionary to a set of values
my_set = set(my_dict.values())

print(my_set)
```
