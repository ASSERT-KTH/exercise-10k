## Exercise
Create a program that simulates a coin flip and prints out the result.

## Reference Solution
```python
import random

def coin_flip():
    """Simulates a coin flip and returns the result."""
    flip = random.choice(["Heads", "Tails"])
    return flip

def main():
    print("Coin Flip Simulator")
    print("--------------------")
    result = coin_flip()
    print(f"The result is: {result}")

if __name__ == "__main__":
    main()
```
