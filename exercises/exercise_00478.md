## Exercise
Write a program that takes a string as input and returns a zigzag pattern of the string

## Reference Solution
```python
def zigzag_pattern(s, num_rows):
    """
    Generate a zigzag pattern of the input string.

    Args:
        s (str): The input string.
        num_rows (int): The number of rows in the zigzag pattern.

    Returns:
        str: The zigzag pattern of the input string.
    """
    if num_rows == 1 or num_rows >= len(s):
        return s

    rows = [''] * num_rows
    index, step = 0, 1

    for char in s:
        rows[index] += char
        if index == 0:
            step = 1
        elif index == num_rows - 1:
            step = -1
        index += step

    return '\n'.join(rows)


def main():
    # Example usage:
    s = "PAYPALISHIRING"
    num_rows = 3
    print(zigzag_pattern(s, num_rows))


if __name__ == "__main__":
    main()
```
