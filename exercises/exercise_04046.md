## Exercise
Write a function that takes a dictionary and another dictionary as input and updates the first dictionary with the key-value pairs from the second dictionary.

## Reference Solution
```python
def update_dict(dict1, dict2):
    """
    Updates the first dictionary with the key-value pairs from the second dictionary.

    Args:
        dict1 (dict): The dictionary to be updated.
        dict2 (dict): The dictionary containing the key-value pairs to update with.

    Returns:
        dict: The updated dictionary.
    """

    # First, we check if both inputs are dictionaries
    if not isinstance(dict1, dict) or not isinstance(dict2, dict):
        raise TypeError("Both inputs must be dictionaries")

    # We use the built-in update method of Python dictionaries to update dict1 with the key-value pairs from dict2
    # This method updates the dictionary in-place, meaning it modifies the original dictionary
    dict1.update(dict2)

    # Finally, we return the updated dictionary
    return dict1

# Example usage:
dict1 = {"a": 1, "b": 2}
dict2 = {"b": 3, "c": 4}

print("Before update:", dict1)
updated_dict = update_dict(dict1, dict2)
print("After update:", updated_dict)
```
