## Exercise
Demonstrate the concept of variable scope by declaring a variable inside and outside a function, and then printing their values.

## Reference Solution
```python
# Global variable
global_variable = "I am global"

def my_function():
    # Local variable
    local_variable = "I am local"
    
    # Accessing global variable
    print("Inside function: Global variable =", global_variable)
    
    # Accessing local variable
    print("Inside function: Local variable =", local_variable)

# Accessing global variable
print("Outside function: Global variable =", global_variable)

# Trying to access local variable outside function will result in error
try:
    print("Outside function: Local variable =", local_variable)
except NameError:
    print("Error: Local variable is not defined outside function")

my_function()
```
