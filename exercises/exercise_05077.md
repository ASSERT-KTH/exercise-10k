## Exercise
Declare two variables and swap their values without using a temporary variable.

## Reference Solution
```python
# Declare two variables
a = 5
b = 10

# Print the original values
print("Original values:")
print(f"a = {a}, b = {b}")

# Swap values using XOR operator
a = a ^ b
b = a ^ b
a = a ^ b

# Print the swapped values
print("\nSwapped values:")
print(f"a = {a}, b = {b}")

# Alternatively, you can use arithmetic operations
a = 5
b = 10

# Print the original values
print("\nOriginal values:")
print(f"a = {a}, b = {b}")

# Swap values using arithmetic operations
a = a + b
b = a - b
a = a - b

# Print the swapped values
print("\nSwapped values:")
print(f"a = {a}, b = {b}")

# However, the most Pythonic way to swap variables is:
a = 5
b = 10

# Print the original values
print("\nOriginal values:")
print(f"a = {a}, b = {b}")

# Swap values using tuple unpacking
a, b = b, a

# Print the swapped values
print("\nSwapped values:")
print(f"a = {a}, b = {b}")
```
