## Exercise
Write a function that handles errors and exceptions using try-except blocks.

## Reference Solution
```python
def divide_numbers(num1, num2):
    """
    Divide two numbers and handle potential errors.

    Args:
        num1 (float): The dividend.
        num2 (float): The divisor.

    Returns:
        float: The result of the division.

    Raises:
        TypeError: If either num1 or num2 is not a number.
        ZeroDivisionError: If num2 is zero.
    """
    try:
        # Attempt to divide the numbers
        result = num1 / num2
        return result
    except TypeError:
        # Handle the case where either num1 or num2 is not a number
        print("Error: Both inputs must be numbers.")
    except ZeroDivisionError:
        # Handle the case where num2 is zero
        print("Error: Cannot divide by zero.")
    except Exception as e:
        # Catch any other unexpected exceptions
        print(f"An unexpected error occurred: {e}")

# Example usage:
print(divide_numbers(10, 2))  # Output: 5.0
print(divide_numbers(10, 0))  # Output: Error: Cannot divide by zero.
print(divide_numbers(10, 'a'))  # Output: Error: Both inputs must be numbers.
```
