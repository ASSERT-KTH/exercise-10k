## Exercise
Write a program that calculates the tax based on the income using if/else statement

## Reference Solution
```python
def calculate_tax(income):
    """
    Calculates the tax based on the income.

    Args:
        income (float): The income to calculate tax for.

    Returns:
        float: The calculated tax.
    """
    if income < 10000:
        tax = 0
    elif income < 20000:
        tax = income * 0.10
    elif income < 50000:
        tax = income * 0.20
    else:
        tax = income * 0.30

    return tax

def main():
    income = float(input("Enter your income: $"))
    tax = calculate_tax(income)
    print(f"Your tax is: ${tax:.2f}")

if __name__ == "__main__":
    main()
```
