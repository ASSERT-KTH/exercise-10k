## Exercise
Return a list of all values in a dictionary

## Reference Solution
```python
def get_dict_values(input_dict):
    """
    Returns a list of all values in a dictionary.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        list: A list of dictionary values.
    """
    return list(input_dict.values())

# Example usage
my_dict = {"a": 1, "b": 2, "c": 3}
print(get_dict_values(my_dict))  # Output: [1, 2, 3]
```
