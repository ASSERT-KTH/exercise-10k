## Exercise
Implement a binary search algorithm to find an element in a sorted array. The function should take in a sorted array and a target value as input and return the index of the target value if it exists in the array, otherwise return -1.

## Reference Solution
```python
class Solution:
    def search(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] == target:
                return mid
            elif nums[mid] < target:
                left = mid + 1
            else:
                right = mid - 1
        return -1
```
