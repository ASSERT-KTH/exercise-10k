## Exercise
Write a function to clear all key-value pairs from a dictionary

## Reference Solution
```python
def clear_dictionary(dictionary):
    """
    Clears all key-value pairs from a dictionary.

    Args:
        dictionary (dict): The dictionary to be cleared.

    Returns:
        None
    """
    dictionary.clear()

# Example usage:
my_dict = {"key1": "value1", "key2": "value2"}
print("Before clearing:", my_dict)
clear_dictionary(my_dict)
print("After clearing:", my_dict)
```
