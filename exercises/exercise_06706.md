## Exercise
Create a program that takes a number as input and returns the product of its digits.

## Reference Solution
```python
def product_of_digits(n):
    product = 1
    for digit in str(n):
        product *= int(digit)
    return product

num = int(input("Enter a number: "))
print("The product of its digits is:", product_of_digits(num))
```
