## Exercise
Create a program that checks if a student has passed or failed based on their grade. If the grade is above 50, the student has passed. Otherwise, they have failed.

## Reference Solution
```python
def check_grade(grade):
    """
    Checks if a student has passed or failed based on their grade.

    Args:
        grade (int): The student's grade.

    Returns:
        str: "Passed" if the grade is above 50, "Failed" otherwise.
    """
    if grade > 50:
        return "Passed"
    else:
        return "Failed"

def main():
    # Example usage
    grade = int(input("Enter the student's grade: "))
    result = check_grade(grade)
    print(f"The student has {result}.")

if __name__ == "__main__":
    main()
```
