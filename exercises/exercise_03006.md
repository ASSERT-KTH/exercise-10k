## Exercise
Write a program to join a list of strings into a single string with a delimiter.

## Reference Solution
```python
def join_strings(string_list, delimiter):
    """
    Join a list of strings into a single string with a delimiter.

    Args:
        string_list (list): A list of strings to be joined.
        delimiter (str): The delimiter to use when joining the strings.

    Returns:
        str: The joined string.
    """
    return delimiter.join(string_list)


# Example usage:
if __name__ == "__main__":
    strings_to_join = ["Hello", "world", "this", "is", "a", "test"]
    delimiter_to_use = ", "

    joined_string = join_strings(strings_to_join, delimiter_to_use)
    print("Joined string:", joined_string)
```
