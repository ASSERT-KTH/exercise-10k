## Exercise
Write a program that uses a for loop to print each element of a given set.

## Reference Solution
```python
def print_set_elements(input_set):
    """
    Prints each element of the given set.

    Args:
        input_set (set): The set to print elements from.
    """
    for element in input_set:
        print(element)


# Example usage
if __name__ == "__main__":
    # Create a sample set
    sample_set = {1, 2, 3, 4, 5}

    # Print set elements
    print("Set Elements:")
    print_set_elements(sample_set)
```
