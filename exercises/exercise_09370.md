## Exercise
Write a program that checks for null and undefined values.

## Reference Solution
```python
// Function to check for null and undefined values
function checkForNullUndefined(value) {
    if (value === null) {
        console.log("The value is null.");
    } else if (value === undefined) {
        console.log("The value is undefined.");
    } else {
        console.log("The value is neither null nor undefined.");
    }
}

// Test the function
checkForNullUndefined(null); // Output: The value is null.
checkForNullUndefined(undefined); // Output: The value is undefined.
checkForNullUndefined("Hello, World!"); // Output: The value is neither null nor undefined.
```
