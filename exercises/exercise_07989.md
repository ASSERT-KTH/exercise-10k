## Exercise
Write a program that removes duplicates from a given list of numbers.

## Reference Solution
```python
def remove_duplicates(num_list):
    """
    Removes duplicates from a given list of numbers.

    Args:
        num_list (list): A list of numbers.

    Returns:
        list: A new list with duplicates removed.
    """
    return list(set(num_list))

# Example usage
numbers = [1, 2, 2, 3, 4, 4, 5, 6, 6, 7, 8, 8, 9]
unique_numbers = remove_duplicates(numbers)

print("Original List:", numbers)
print("List without Duplicates:", unique_numbers)
```

```python
def remove_duplicates_ordered(num_list):
    """
    Removes duplicates from a given list of numbers while preserving the original order.

    Args:
        num_list (list): A list of numbers.

    Returns:
        list: A new list with duplicates removed and original order preserved.
    """
    seen = set()
    return [x for x in num_list if not (x in seen or seen.add(x))]

# Example usage
numbers = [1, 2, 2, 3, 4, 4, 5, 6, 6, 7, 8, 8, 9]
unique_numbers = remove_duplicates_ordered(numbers)

print("Original List:", numbers)
print("List without Duplicates (ordered):", unique_numbers)
```
