## Exercise
Write a program that takes an array of integers and a rotation amount as input, and returns a new array containing the same elements rotated by the specified amount.

## Reference Solution
```python
export default function rotate(arr, numRotations) {
  return arr.slice(numRotations).concat(arr.slice(0, numRotations));
}
```

```python
const arr = [1, 2, 3, 4, 5];
const numRotations = 2;
const rotatedArr = rotate(arr, numRotations);
console.log(rotatedArr); // Output: [3, 4, 5, 1, 2]
```
