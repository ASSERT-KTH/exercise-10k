## Exercise
Write a program that takes an array and a reduction function as input and returns the result of applying the reduction to all elements in the array

## Reference Solution
```python
from functools import reduce

def apply_reduction(array, reduction_function):
    """
    Applies a reduction function to all elements in an array.

    Args:
        array (list): The input array.
        reduction_function (function): A function that takes two arguments and returns one value.

    Returns:
        The result of applying the reduction function to all elements in the array.
    """
    if not array:
        raise ValueError("Array cannot be empty")

    return reduce(reduction_function, array)

# Example usage:
array = [1, 2, 3, 4, 5]
reduction_function = lambda x, y: x + y

result = apply_reduction(array, reduction_function)
print(result)  # Output: 15
```
