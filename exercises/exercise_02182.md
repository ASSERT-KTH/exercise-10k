## Exercise
Create a function to check if a given string is a palindrome. The function should return True if the string is a palindrome and False otherwise.

## Reference Solution
```python
def check_palindrome(s: str) -> bool:
    """
    Checks if a given string is a palindrome.

    Args:
        s (str): The input string to be checked.

    Returns:
        bool: True if the string is a palindrome, False otherwise.
    """

    # First, we remove any leading or trailing whitespace from the string
    s = s.strip()

    # Then, we convert the string to lowercase to make the comparison case-insensitive
    s = s.lower()

    # Now, we compare the string with its reverse
    # If they are equal, the string is a palindrome
    # We use Python's slicing feature to reverse the string
    # The syntax s[start:stop:step] returns a slice of the string
    # By using ::-1, we start at the end of the string and move backwards to the start, stepping backwards by 1 character each time
    return s == s[::-1]
```
