## Exercise
Given an array of integers, sort the array in a wiggle sort order (i.e., the array should alternate between smaller and larger elements). The function should take in an array of integers as input and return the wiggle sorted array.

## Reference Solution
```python
def wiggle_sort(nums):
    nums.sort()
    mid = len(nums) // 2
    small, big = mid - 1, mid
    res = []
    while small >= 0 or big < len(nums):
        if small >= 0:
            res.append(nums[small])
            small -= 1
        if big < len(nums):
            res.append(nums[big])
            big += 1
    return res

# Example usage:
arr = [1, 5, 1, 1, 6, 4]
res = wiggle_sort(arr)
print(res)  # Output: [1, 6, 1, 5, 1, 4]
```
