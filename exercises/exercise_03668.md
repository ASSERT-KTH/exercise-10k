## Exercise
Create a function that takes a number as input and returns its absolute value.

## Reference Solution
```python
# Define a function that takes one argument, 'number'.
def absolute_value(number):
    """
    This function calculates and returns the absolute value of a given number.

    Args:
        number (int or float): The input number.

    Returns:
        int or float: The absolute value of the input number.
    """

    # Check if the input is an integer or a float, because absolute value is only defined for these types.
    if not isinstance(number, (int, float)):
        raise TypeError("Input must be an integer or a float.")

    # Use the built-in abs() function in Python to calculate the absolute value.
    # The abs() function returns the absolute value of a number, which is the number without its sign.
    absolute = abs(number)

    # Return the calculated absolute value.
    return absolute

# Test the function with some examples.
print(absolute_value(-5))  # Output: 5
print(absolute_value(10))  # Output: 10
print(absolute_value(-3.7))  # Output: 3.7
```
