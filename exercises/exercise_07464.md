## Exercise
Write a program that converts a given float into an integer

## Reference Solution
```python
def float_to_int(float_num):
    """
    Converts a float into an integer.

    Args:
        float_num (float): The float number to be converted.

    Returns:
        int: The integer equivalent of the float number.
    """
    return int(float_num)

# Example usage:
if __name__ == "__main__":
    float_num = 10.5
    print(f"Float number: {float_num}")
    int_num = float_to_int(float_num)
    print(f"Integer equivalent: {int_num}")
```
