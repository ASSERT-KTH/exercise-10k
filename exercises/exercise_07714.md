## Exercise
Remove all digits from a given string.

## Reference Solution
```python
import re

def remove_digits(input_string):
    """
    Removes all digits from a given string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The input string with all digits removed.
    """
    return re.sub('\d', '', input_string)

# Example usage:
input_string = "Hello123World456"
result = remove_digits(input_string)
print(result)  # Output: "HelloWorld"
```
