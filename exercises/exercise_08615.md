## Exercise
Write a program that takes a string as input and returns a new string with the words in reverse order.

## Reference Solution
```python
def reverse_words(input_string):
    """
    Reverses the order of words in a given string.

    Args:
        input_string (str): The input string to be processed.

    Returns:
        str: A new string with the words in reverse order.
    """
    words = input_string.split()  # Split the string into words
    reversed_words = words[::-1]  # Reverse the list of words
    output_string = ' '.join(reversed_words)  # Join the reversed words into a string
    return output_string

# Example usage
def main():
    input_str = "Hello World this is Python"
    print("Input String:", input_str)
    print("Reversed Words:", reverse_words(input_str))

if __name__ == "__main__":
    main()
```
