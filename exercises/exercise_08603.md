## Exercise
Write a program that takes two strings as input and returns their concatenation.

## Reference Solution
```python
def concatenate_strings(str1, str2):
    """
    Concatenates two strings.

    Args:
        str1 (str): The first string to be concatenated.
        str2 (str): The second string to be concatenated.

    Returns:
        str: The concatenation of str1 and str2.
    """
    return str1 + str2

def main():
    # Get user input
    str1 = input("Enter the first string: ")
    str2 = input("Enter the second string: ")

    # Concatenate the strings
    result = concatenate_strings(str1, str2)

    # Print the result
    print("The concatenation of the two strings is: ", result)

if __name__ == "__main__":
    main()
```
