## Exercise
Write a program to find the symmetric difference of two or more arrays.

## Reference Solution
```python
def symmetric_difference(*arrays):
    """
    Calculate the symmetric difference of two or more arrays.

    Args:
        *arrays: Variable number of arrays.

    Returns:
        A set containing the symmetric difference of the input arrays.
    """
    result = set(arrays[0])
    for array in arrays[1:]:
        result = result.symmetric_difference(set(array))
    return result

# Example usage:
if __name__ == "__main__":
    array1 = [1, 2, 3, 4]
    array2 = [3, 4, 5, 6]
    array3 = [5, 6, 7, 8]

    print("Symmetric difference of array1 and array2:")
    print(symmetric_difference(array1, array2))

    print("\nSymmetric difference of array1, array2, and array3:")
    print(symmetric_difference(array1, array2, array3))
```
