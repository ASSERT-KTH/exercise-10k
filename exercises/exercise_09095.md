## Exercise
Given an integer array, find the maximum contiguous subarray and return its sum. A subarray is a contiguous part of an array. For example, if we have the array [-2, 1, -3, 4, -1, 2, 1, -5, 4], the maximum subarray is [4, -1, 2, 1] and its sum is 6.

## Reference Solution
```python
def max_subarray_sum(nums):
    """
    Returns the maximum sum of a contiguous subarray within the given array.

    Args:
        nums (list): The input integer array.

    Returns:
        int: The maximum sum of a contiguous subarray.
    """
    if not nums:
        return 0

    max_current = max_global = nums[0]

    for num in nums[1:]:
        max_current = max(num, max_current + num)
        max_global = max(max_global, max_current)

    return max_global

# Example usage:
nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
print(max_subarray_sum(nums))  # Output: 6
```
