## Exercise
Write a function that takes a function as an argument and returns a new function that memoizes the input function's results.

```python
def memoize(func):
    """
    This function takes another function as an argument and returns a new function 
    that memoizes the input function's results. It uses a dictionary to store the 
    results of the input function for different arguments, so that if the input 
    function is called again with the same arguments, it can simply return the 
    stored result instead of recalculating it.

    Args:
        func (function): The function to be memoized.

    Returns:
        function: A new function that memoizes the input function's results.
    """

    # Create a dictionary to store the results of the input function
    cache = {}

    # Define a new function that will memoize the input function's results
    def memoized_func(*args):
        # Check if the result for the given arguments is already in the cache
        if args in cache:
            # If it is, return the cached result
            return cache[args]
        else:
            # If not, calculate the result using the input function
            result = func(*args)
            # Store the result in the cache
            cache[args] = result
            # Return the result
            return result

    # Return the memoized function
    return memoized_func


# Example usage:
def add(a, b):
    print("Calculating...")
    return a + b

memoized_add = memoize(add)
print(memoized_add(2, 3))  # Calculates and prints 5
print(memoized_add(2, 3))  # Returns the cached result 5 without calculating
```
