## Exercise
Write a program to find the longest common prefix of two or more strings.

## Reference Solution
```python
def longest_common_prefix(strings):
    """
    Find the longest common prefix of two or more strings.

    Args:
        strings (list): A list of strings.

    Returns:
        str: The longest common prefix.
    """
    if not strings:
        return ""

    prefix = ""
    for chars in zip(*strings):
        if len(set(chars)) == 1:
            prefix += chars[0]
        else:
            break

    return prefix

# Example usage
strings = ["hello", "hello world", "hello again"]
print(longest_common_prefix(strings))  # Output: "hello"

strings = ["abc", "abcd", "abce"]
print(longest_common_prefix(strings))  # Output: "abc"

strings = ["foo", "bar", "baz"]
print(longest_common_prefix(strings))  # Output: ""
```
