## Exercise
Create a string and perform basic string manipulation operations (e.g. trimming, replacing), then print out the manipulated string.

## Reference Solution
```python
def manipulate_string():
    # Create a sample string
    original_string = "   Hello, World!   "

    # Print the original string
    print("Original String:")
    print(original_string)

    # Trim the string (remove leading and trailing whitespace)
    trimmed_string = original_string.strip()
    print("\nTrimmed String:")
    print(trimmed_string)

    # Replace a substring
    replaced_string = trimmed_string.replace("World", "Universe")
    print("\nString with 'World' replaced by 'Universe':")
    print(replaced_string)

if __name__ == "__main__":
    manipulate_string()
```
