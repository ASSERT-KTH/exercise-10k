## Exercise
Write a function that takes two lists as input and returns a list of common elements

## Reference Solution
```python
def common_elements(list1, list2):
    """
    Returns a list of common elements between two input lists.

    Args:
        list1 (list): The first input list.
        list2 (list): The second input list.

    Returns:
        list: A list of common elements.
    """
    return [element for element in list1 if element in list2]

# Example usage
list1 = [1, 2, 3, 4, 5]
list2 = [4, 5, 6, 7, 8]
print(common_elements(list1, list2))  # Output: [4, 5]
```

def common_elements_set(list1, list2):
