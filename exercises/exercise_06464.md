## Exercise
Write a function that takes a dictionary and a key as input and removes the key-value pair from the dictionary

## Reference Solution
```python
def remove_key_value_pair(dictionary, key):
    """
    Removes a key-value pair from a dictionary.

    Args:
        dictionary (dict): The dictionary to remove the key-value pair from.
        key: The key to remove.

    Returns:
        dict: The updated dictionary.
    """
    if key in dictionary:
        del dictionary[key]
    return dictionary

# Example usage:
my_dict = {"a": 1, "b": 2, "c": 3}
print("Original dictionary:", my_dict)

updated_dict = remove_key_value_pair(my_dict, "b")
print("Updated dictionary:", updated_dict)
```
