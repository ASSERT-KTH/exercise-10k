## Exercise
Create a program that uses nested for loops to print the multiplication table up to n.

## Reference Solution
```python
def print_multiplication_table(n):
    """
    Prints the multiplication table up to n.
    
    Args:
    n (int): The maximum number for the multiplication table.
    """
    # Print the header row
    print("   |", end=" ")
    for i in range(1, n + 1):
        print(f"{i:4d}", end=" ")
    print()
    print("-" * (5 * n + 5))

    # Print the multiplication table
    for i in range(1, n + 1):
        print(f"{i:3d} |", end=" ")
        for j in range(1, n + 1):
            print(f"{i * j:4d}", end=" ")
        print()

def main():
    n = int(input("Enter the maximum number for the multiplication table: "))
    print_multiplication_table(n)

if __name__ == "__main__":
    main()
```
