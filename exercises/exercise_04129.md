## Exercise
Create a dictionary to store student information, then print out the information

## Reference Solution
```python
# Define a function to create and print student information
def create_and_print_student_info():
    # Create a dictionary to store student information
    student_info = {
        "name": "John Doe",
        "age": 20,
        "grade": "Sophomore",
        "major": "Computer Science",
        "courses": ["Math", "Science", "History"]
    }

    # Print out the student information
    print("Student Information:")
    for key, value in student_info.items():
        if isinstance(value, list):
            print(f"{key.capitalize()}: {', '.join(value)}")
        else:
            print(f"{key.capitalize()}: {value}")

# Call the function to create and print student information
create_and_print_student_info()
```
