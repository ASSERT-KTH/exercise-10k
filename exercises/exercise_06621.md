## Exercise
Write a function that takes a single argument and returns its square

## Reference Solution
```python
def square(x):
    """
    Returns the square of the input value.

    Args:
        x (int or float): The input value.

    Returns:
        int or float: The square of the input value.
    """
    return x ** 2
```

```python
print(square(5))  # Output: 25
print(square(3.5))  # Output: 12.25
```
