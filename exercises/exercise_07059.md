## Exercise
Write a program that takes a string and a substring as input and prints out whether the substring is present in the string or not.

## Reference Solution
```python
def check_substring(string, substring):
    """
    Checks if a substring is present in a string.

    Args:
        string (str): The original string.
        substring (str): The substring to be searched.

    Returns:
        bool: True if the substring is present, False otherwise.
    """
    return substring in string

def main():
    # Get user input
    string = input("Enter the original string: ")
    substring = input("Enter the substring to search: ")

    # Check if substring is present
    result = check_substring(string, substring)

    # Print the result
    if result:
        print(f"'{substring}' is present in '{string}'")
    else:
        print(f"'{substring}' is not present in '{string}'")

if __name__ == "__main__":
    main()
```
