## Exercise
Write a function that takes a name as an argument and returns a personalized greeting message, with a default argument for the greeting

## Reference Solution
```python
def personalized_greeting(name, greeting="Hello"):
    """
    This function generates a personalized greeting message.

    Args:
        name (str): The name of the person to be greeted.
        greeting (str, optional): The greeting to be used. Defaults to "Hello".

    Returns:
        str: A personalized greeting message.
    """

    # First, we define the function with two parameters: name and greeting.
    # The greeting parameter has a default value of "Hello", which means it's optional when calling the function.

    # We want to return a string that includes the greeting and the name.
    # To do this, we use an f-string, which allows us to embed expressions inside string literals.
    
    # We start by defining the greeting message.
    # We use an f-string to insert the greeting and name into the message.
    greeting_message = f"{greeting}, {name}!"

    # Finally, we return the greeting message.
    return greeting_message

# Example usage:
print(personalized_greeting("John"))  # Outputs: Hello, John!
print(personalized_greeting("Jane", "Hi"))  # Outputs: Hi, Jane!
```
