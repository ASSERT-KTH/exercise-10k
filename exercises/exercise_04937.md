## Exercise
Create a program that calculates the length of the hypotenuse of a right triangle given the lengths of the other two sides. The program should use the formula hypotenuse = sqrt(a ** 2 + b ** 2) to calculate the hypotenuse.

## Reference Solution
```python
import math

def get_hypotenuse(a, b):
    return math.sqrt(a ** 2 + b ** 2)

print(get_hypotenuse(3, 4))  # Output: 5.0
```
