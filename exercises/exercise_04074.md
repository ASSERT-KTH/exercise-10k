## Exercise
Write a program that takes a string and a dictionary as input and returns the string with all placeholders replaced with the corresponding values from the dictionary.

## Reference Solution
```python
def replace_placeholders(text, params_dict):
    for key, value in params_dict.items():
        text = text.replace(f"{{{key}}}", value)
    return text
```
