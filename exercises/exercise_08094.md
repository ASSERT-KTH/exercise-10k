## Exercise
Write a program that takes two strings as input and returns the concatenated string.

## Reference Solution
```python
def concatenate_strings(str1, str2):
    """
    Concatenates two input strings.

    Args:
        str1 (str): The first string.
        str2 (str): The second string.

    Returns:
        str: The concatenated string.
    """
    return str1 + str2

def main():
    # Get input from the user
    str1 = input("Enter the first string: ")
    str2 = input("Enter the second string: ")

    # Concatenate the strings
    concatenated_str = concatenate_strings(str1, str2)

    # Print the result
    print("The concatenated string is: ", concatenated_str)

if __name__ == "__main__":
    main()
```
