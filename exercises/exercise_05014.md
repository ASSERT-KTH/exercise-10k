## Exercise
Write a program that declares a variable 'x' inside a function and assigns it the value 10. Then, try to access the variable outside the function. Print the result to the console.

## Reference Solution
```python
def my_function():
    x = 10  # declare and assign variable 'x' inside the function

def main():
    my_function()  # call the function to declare 'x'
    
    try:
        print(x)  # try to access 'x' outside the function
    except NameError as e:
        print(f"Error: {e}")  # print the error message

if __name__ == "__main__":
    main()
```
