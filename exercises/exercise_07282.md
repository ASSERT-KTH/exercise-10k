## Exercise
Create a function that takes a name as an argument and returns a personalized greeting

## Reference Solution
```python
def personalized_greeting(name: str) -> str:
    """
    Returns a personalized greeting based on the input name.

    Args:
        name (str): The name of the person to greet.

    Returns:
        str: A personalized greeting message.
    """
    return f"Hello, {name}!"

# Example usage
def main():
    name = "John"
    print(personalized_greeting(name))  # Output: Hello, John!

if __name__ == "__main__":
    main()
```
