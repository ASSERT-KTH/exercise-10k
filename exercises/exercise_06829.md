## Exercise
Write a program that takes a string input from the user and returns the reversed string. For example, if the input is 'hello', the output should be 'olleh'.

## Reference Solution
```python
def reverse_string(input_str):
    """
    This function takes a string input from the user and returns the reversed string.

    Args:
        input_str (str): The input string to be reversed.

    Returns:
        str: The reversed input string.
    """
    return input_str[::-1]


def main():
    # Get the input string from the user
    input_str = input("Please enter a string: ")

    # Reverse the input string
    reversed_str = reverse_string(input_str)

    # Print the reversed string
    print("The reversed string is: ", reversed_str)


if __name__ == "__main__":
    main()
```
