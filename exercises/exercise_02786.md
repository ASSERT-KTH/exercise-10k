## Exercise
Write a program that replaces a specified element in a given array with a new value.

## Reference Solution
```python
def replace_element(array, old_value, new_value):
    """
    Replaces all occurrences of a specified element in a given array with a new value.

    Args:
        array (list): The input array.
        old_value: The value to be replaced.
        new_value: The new value to replace with.

    Returns:
        list: The modified array with the old value replaced by the new value.
    """
    return [new_value if x == old_value else x for x in array]


# Example usage:
array = [1, 2, 3, 4, 2, 5, 2]
old_value = 2
new_value = 10

print("Original array:", array)
print("Modified array:", replace_element(array, old_value, new_value))
```
