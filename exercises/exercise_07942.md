## Exercise
Write a program to convert a user's input from a string to an integer and print the result.

## Reference Solution
```python
def convert_to_int():
    # Get user's input
    user_input = input("Please enter a string: ")

    try:
        # Attempt to convert the input to an integer
        integer = int(user_input)
        print(f"The integer representation of '{user_input}' is: {integer}")
    except ValueError:
        # Handle the case where the input cannot be converted to an integer
        print(f"'{user_input}' cannot be converted to an integer.")

# Call the function
convert_to_int()
```
