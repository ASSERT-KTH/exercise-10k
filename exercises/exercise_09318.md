## Exercise
Create a function that takes a string as input and returns the string with all special characters removed.

## Reference Solution
```python
import re

def remove_special_characters(input_string):
    """
    This function removes all special characters from a given string.

    Args:
        input_string (str): The input string that may contain special characters.

    Returns:
        str: The input string with all special characters removed.
    """
    # Use regular expression to replace all non-alphanumeric characters (except spaces) with an empty string
    cleaned_string = re.sub('[^A-Za-z0-9\s]', '', input_string)
    
    return cleaned_string

# Example usage
if __name__ == "__main__":
    input_str = "Hello, World! This is a test string with special characters like @,#,$, etc."
    print("Original string: ", input_str)
    print("Cleaned string: ", remove_special_characters(input_str))
```
