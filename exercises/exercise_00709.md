## Exercise
Write a function to return a dictionary sorted by its keys

## Reference Solution
```python
def sort_dict_by_keys(input_dict):
    """
    Returns a new dictionary sorted by its keys.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        dict: A new dictionary sorted by its keys.
    """
    return dict(sorted(input_dict.items()))

# Example usage:
input_dict = {'c': 3, 'b': 2, 'a': 1}
sorted_dict = sort_dict_by_keys(input_dict)
print(sorted_dict)
```
