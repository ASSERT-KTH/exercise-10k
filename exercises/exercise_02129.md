## Exercise
Create a function that takes an array and an index as input and returns a new array containing all elements except the one at the specified index.

## Reference Solution
```python
def remove_element(array, index):
    """
    This function takes an array and an index as input and returns a new array containing all elements except the one at the specified index.
    """
    if index < 0 or index >= len(array):
        return array

    new_array = []
    for i in range(len(array)):
        if i != index:
            new_array.append(array[i])

    return new_array

# Example usage
array = [1, 2, 3, 4, 5]
index = 2
print(remove_element(array, index))  # Output: [1, 2, 4, 5]
```
