## Exercise
Write a program that asks the user for a time in standard format (e.g. 3:45 PM) and converts it to military time format (e.g. 15:45).

## Reference Solution
```python
from datetime import datetime

def convert_to_military_time(time_str):
    """Converts standard time format to military time format."""
    try:
        # Parse the input time string
        time = datetime.strptime(time_str, "%I:%M %p")
        
        # Return the time in military format
        return time.strftime("%H:%M")
    except ValueError:
        # Handle invalid input
        return "Invalid time format. Please use HH:MM AM/PM."

def main():
    # Ask the user for a time
    time_str = input("Enter a time in standard format (e.g. 3:45 PM): ")
    
    # Convert the time to military format
    military_time = convert_to_military_time(time_str)
    
    # Print the result
    print(f"Military time: {military_time}")

if __name__ == "__main__":
    main()
```
