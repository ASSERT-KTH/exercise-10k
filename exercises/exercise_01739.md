## Exercise
Write a program that takes two lists of numbers as input and returns True if one list is a rotation of the other, and False otherwise.

## Reference Solution
```python
def isRotation(lst1, lst2):
    if len(lst1) != len(lst2):
        return False
    for i in range(len(lst1)):
        if lst1 == lst2[i:] + lst2[:i]:
            return True
    return False

print(isRotation([1, 2, 3, 4], [3, 4, 1, 2])) # True
print(isRotation([1, 2, 3, 4], [2, 1, 3, 4])) # False
```
