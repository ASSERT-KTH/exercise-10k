## Exercise
Write a program that finds the maximum value in a list of numbers using a while loop.

## Reference Solution
```python
def find_max(numbers):
    """
    Find the maximum value in a list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        int: The maximum value in the list.
    """
    # Check if the list is empty
    if len(numbers) == 0:
        return None

    # Initialize the maximum value with the first element of the list
    max_value = numbers[0]

    # Initialize the index to the second element (index 1)
    index = 1

    # Loop through the list until we reach the end
    while index < len(numbers):
        # If the current element is greater than the max value, update the max value
        if numbers[index] > max_value:
            max_value = numbers[index]
        # Move on to the next element
        index += 1

    # Return the maximum value
    return max_value


# Example usage
numbers = [12, 45, 7, 23, 56, 89, 34]
max_value = find_max(numbers)
print("Maximum value:", max_value)
```
