## Exercise
Create a program that calculates the result of a given exponentiation operation.

## Reference Solution
```python
def exponentiation(base, exponent):
    """
    Calculate the result of a given exponentiation operation.

    Args:
        base (float): The base number
        exponent (float): The exponent

    Returns:
        float: The result of the exponentiation operation
    """
    return base ** exponent

def main():
    # Get user input
    base = float(input("Enter the base number: "))
    exponent = float(input("Enter the exponent: "))

    # Calculate and print the result
    result = exponentiation(base, exponent)
    print(f"{base} to the power of {exponent} is {result}")

if __name__ == "__main__":
    main()
```
