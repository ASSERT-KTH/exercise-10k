## Exercise
Write a program that prints the multiplication table for a given number.

## Reference Solution
```python
def print_multiplication_table(number, max_multiplier=10):
    """
    Prints the multiplication table for a given number up to max_multiplier.

    Args:
        number (int): The number for which the multiplication table is to be printed.
        max_multiplier (int, optional): The maximum multiplier. Defaults to 10.
    """
    print(f"Multiplication table for {number}:")
    for i in range(1, max_multiplier + 1):
        print(f"{number} x {i} = {number * i}")


def main():
    num = int(input("Enter a number: "))
    max_multiplier = int(input("Enter the maximum multiplier (default: 10): ") or 10)
    print_multiplication_table(num, max_multiplier)


if __name__ == "__main__":
    main()
```
