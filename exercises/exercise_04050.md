## Exercise
Write a function that takes a dictionary as input and returns a copy of the dictionary.

## Reference Solution
```python
def copy_dictionary(input_dict):
    """
    This function creates a copy of the input dictionary.

    Args:
        input_dict (dict): The dictionary to be copied.

    Returns:
        dict: A copy of the input dictionary.
    """

    # First, we check if the input is indeed a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary")

    # We use the built-in dict.copy() method to create a shallow copy of the dictionary
    # This method returns a new dictionary object which is a copy of the original dictionary
    copied_dict = input_dict.copy()

    # Finally, we return the copied dictionary
    return copied_dict

# Example usage:
original_dict = {"a": 1, "b": 2}
copied_dict = copy_dictionary(original_dict)
print(copied_dict)  # Output: {'a': 1, 'b': 2}
```
