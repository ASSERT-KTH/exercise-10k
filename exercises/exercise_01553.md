## Exercise
Write a function to count the occurrences of a specified element in a given list of integers.

## Reference Solution
```python
def count_element_occurrences(lst, target):
    """
    Counts the occurrences of a specified element in a given list of integers.

    Args:
        lst (list): The list of integers.
        target (int): The element to be counted.

    Returns:
        int: The number of occurrences of the target element.

    Raises:
        TypeError: If the input is not a list or the target is not an integer.
    """
    if not isinstance(lst, list) or not isinstance(target, int):
        raise TypeError("Invalid input type")

    return lst.count(target)


# Example usage:
if __name__ == "__main__":
    numbers = [1, 2, 3, 2, 4, 2, 5]
    target_element = 2
    count = count_element_occurrences(numbers, target_element)
    print(f"The element {target_element} occurs {count} times in the list.")
```
