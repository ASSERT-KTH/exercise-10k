## Exercise
Write a program that finds the first duplicate in an array of integers. The program should take an array of integers as input and output the first duplicate element, or -1 if no duplicates are found.

## Reference Solution
```python
import java.util.HashSet;
import java.util.Set;

public class Main {
    public static int firstDuplicate(int[] a) {
        Set<Integer> seen = new HashSet<>();
        for (int i : a) {
            if (!seen.add(i)) {
                return i;
            }
        }
        return -1;
    }

    public static void main(String[] args) {
        int[] a = {2, 1, 3, 5, 3, 2};
        System.out.println(firstDuplicate(a));  // Output: 3
    }
}
```
