## Exercise
Write a program that takes in a user's input and determines the data type of the input. The input can be an integer, float, or string. The program should then print out the data type of the input.

## Reference Solution
```python
def determine_data_type(user_input):
    """
    Determine the data type of the user's input.

    Args:
        user_input (str): The user's input.

    Returns:
        str: The data type of the user's input.
    """

    # Try to convert the input to an integer
    try:
        int(user_input)
        return "Integer"
    except ValueError:
        pass

    # Try to convert the input to a float
    try:
        float(user_input)
        return "Float"
    except ValueError:
        pass

    # If the input cannot be converted to an integer or float, it's a string
    return "String"


def main():
    # Get the user's input
    user_input = input("Please enter a value: ")

    # Determine the data type of the input
    data_type = determine_data_type(user_input)

    # Print the data type
    print(f"The data type of '{user_input}' is {data_type}.")


if __name__ == "__main__":
    main()
```
