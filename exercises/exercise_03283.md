## Exercise
Write a program that prints a multiplication table for a given number using a for loop.

## Reference Solution
```python
def print_multiplication_table(number, max_multiplier=10):
    """
    Prints a multiplication table for the given number.

    Args:
        number (int): The number for which the multiplication table is printed.
        max_multiplier (int, optional): The maximum multiplier in the table. Defaults to 10.
    """
    print(f"Multiplication Table for {number}:")
    for i in range(1, max_multiplier + 1):
        print(f"{number} x {i} = {number * i}")


def main():
    # Example usage
    number = int(input("Enter a number: "))
    print_multiplication_table(number)


if __name__ == "__main__":
    main()
```
