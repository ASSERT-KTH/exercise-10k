## Exercise
Write a program that calculates the sum of all even numbers in a given range using a for loop or a while loop. The range is defined by a start and end value, which will be provided as input.

## Reference Solution
```python
#include <stdio.h>

int main() {
    int x, y, sum = 0;

    scanf("%d", &x);
    scanf("%d", &y);

    if (x > y) {
        x = y;
        y = x;
    }

    for (int i = x; i <= y; i++) {
        if (i % 2 == 0) {
            sum += i;
        }
    }

    printf("%d\n", sum);

    return 0;
}
```
