## Exercise
Write a function that takes an array and a predicate function as input, and returns two arrays: one containing the elements that satisfy the predicate, and one containing the elements that do not.

## Reference Solution
```python
def partition_array(array, predicate):
    """
    This function takes an array and a predicate function as input, 
    and returns two arrays: one containing the elements that satisfy the predicate, 
    and one containing the elements that do not.

    Args:
        array (list): The input array to be partitioned.
        predicate (function): A function that takes one argument and returns a boolean value.

    Returns:
        tuple: A tuple of two lists. The first list contains the elements that satisfy the predicate, 
        and the second list contains the elements that do not.
    """

    # Initialize two empty lists to store the elements that satisfy the predicate and those that do not
    satisfied = []
    
    # Initialize another list for elements that do not satisfy the predicate
    not_satisfied = []

    # Iterate over each element in the input array
    for element in array:
        # For each element, apply the predicate function
        if predicate(element):
            # If the predicate function returns True, add the element to the satisfied list
            satisfied.append(element)
        else:
            # If the predicate function returns False, add the element to the not_satisfied list
            not_satisfied.append(element)

    # Return a tuple containing the satisfied and not_satisfied lists
    return satisfied, not_satisfied

# Example usage:
# Define a predicate function that checks if a number is even
def is_even(num):
    return num % 2 == 0

# Create an array of numbers
numbers = [1, 2, 3, 4, 5, 6]

# Partition the array using the is_even predicate
even_numbers, odd_numbers = partition_array(numbers, is_even)

# Print the results
print("Even numbers:", even_numbers)
print("Odd numbers:", odd_numbers)
```
