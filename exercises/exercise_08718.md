## Exercise
Ask the user for a decimal number and print out its octal equivalent.

## Reference Solution
```python
def decimal_to_octal(decimal_number):
    """
    Convert a decimal number to its octal equivalent.

    Args:
        decimal_number (int): The decimal number to convert.

    Returns:
        str: The octal equivalent of the decimal number.
    """
    return oct(decimal_number).replace("0o", "")


def main():
    # Ask the user for a decimal number
    decimal_number = int(input("Enter a decimal number: "))

    # Convert the decimal number to octal and print the result
    octal_number = decimal_to_octal(decimal_number)
    print(f"The octal equivalent of {decimal_number} is: {octal_number}")


if __name__ == "__main__":
    main()
```
