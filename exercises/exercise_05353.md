## Exercise
Create a function that takes an array and an index as input, and inserts a new element at the specified index in the array.

## Reference Solution
```python
def insert_element(array, index, new_element):
    """
    Inserts a new element at the specified index in the array.

    Args:
        array (list): The input array.
        index (int): The index at which the new element is to be inserted.
        new_element: The new element to be inserted.

    Returns:
        list: The modified array with the new element inserted.

    Raises:
        IndexError: If the index is out of range.
    """
    if index < 0 or index > len(array):
        raise IndexError("Index out of range")
    
    # Use list slicing to insert the new element at the specified index
    return array[:index] + [new_element] + array[index:]

# Example usage
array = [1, 2, 3, 4, 5]
index = 2
new_element = 10

print("Original array:", array)
print("Index:", index)
print("New element:", new_element)

modified_array = insert_element(array, index, new_element)
print("Modified array:", modified_array)
```
