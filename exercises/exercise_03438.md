## Exercise
Write a function that takes a hexadecimal string as input and returns the decimal equivalent

## Reference Solution
```python
def hex_to_decimal(hex_string):
    """
    Converts a hexadecimal string to its decimal equivalent.

    Args:
        hex_string (str): The input hexadecimal string.

    Returns:
        int: The decimal equivalent of the input hexadecimal string.

    Raises:
        ValueError: If the input string is not a valid hexadecimal number.
    """
    try:
        decimal = int(hex_string, 16)
        return decimal
    except ValueError:
        raise ValueError("Invalid hexadecimal string")

# Example usage
def main():
    hex_string = "A2"
    decimal = hex_to_decimal(hex_string)
    print(f"The decimal equivalent of {hex_string} is {decimal}")

if __name__ == "__main__":
    main()
```
