## Exercise
Create a program that takes a string input from the user, reverses the string, and checks if the original string is a palindrome. A palindrome is a string that reads the same forwards and backwards.

## Reference Solution
```python
def reverse_string(input_str):
    """Reverses a given string."""
    return input_str[::-1]

def is_palindrome(input_str):
    """Checks if a given string is a palindrome."""
    reversed_str = reverse_string(input_str)
    return input_str.lower() == reversed_str.lower()

def main():
    # Get user input
    input_str = input("Enter a string: ")

    # Reverse the string
    reversed_str = reverse_string(input_str)
    print(f"Reversed string: {reversed_str}")

    # Check if the string is a palindrome
    if is_palindrome(input_str):
        print(f"'{input_str}' is a palindrome.")
    else:
        print(f"'{input_str}' is not a palindrome.")

if __name__ == "__main__":
    main()
```
