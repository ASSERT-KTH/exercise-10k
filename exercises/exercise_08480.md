## Exercise
Write a program to calculate the result of an exponentiation operation

## Reference Solution
```python
def exponentiation(base, exponent):
    """
    Calculates the result of an exponentiation operation.

    Args:
        base (float): The base number.
        exponent (float): The exponent.

    Returns:
        float: The result of the exponentiation operation.
    """
    return base ** exponent

def main():
    # Get user input for base and exponent
    base = float(input("Enter the base: "))
    exponent = float(input("Enter the exponent: "))

    # Calculate the result
    result = exponentiation(base, exponent)

    # Print the result
    print(f"{base} raised to the power of {exponent} is {result}")

if __name__ == "__main__":
    main()
```
