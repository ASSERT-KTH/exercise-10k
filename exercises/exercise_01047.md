## Exercise
Write a program that takes an array and an index as input, and returns the array with the element at the given index removed.

## Reference Solution
```python
def remove_by_index(arr, index):
    if index < 0 or index >= len(arr):
        return arr
    return arr[:index] + arr[index+1:]

# Test case
arr = [10, 20, 30, 40, 50]
index = 2
print(remove_by_index(arr, index))
```
