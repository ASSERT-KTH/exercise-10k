## Exercise
Create a function that takes a name as an argument and returns a greeting message.

## Reference Solution
```python
def create_greeting(name: str) -> str:
    """
    Creates a greeting message for the given name.

    Args:
        name (str): The name to be used in the greeting.

    Returns:
        str: A greeting message.
    """
    return f"Hello, {name}!"

# Example usage
def main():
    name = "John"
    print(create_greeting(name))

if __name__ == "__main__":
    main()
```
