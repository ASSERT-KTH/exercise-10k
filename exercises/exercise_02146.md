## Exercise
Write a program that formats the current date and time as a string in the format 'YYYY-MM-DD HH:MM:SS'.

## Reference Solution
```python
import time
import datetime

# Get the current time in seconds since the epoch (January 1, 1970)
t = time.time()

# Convert the time in seconds to a struct_time object
st = time.localtime(t)

# Create a string representation of the current date and time
s = str(datetime.datetime(st.tm_year, st.tm_mon, st.tm_mday, st.tm_hour, st.tm_min, st.tm_sec))

print(s)
```
