## Exercise
Write a program that takes a sentence as input and prints out the sentence with the first letter of each word capitalized

## Reference Solution
```python
def capitalize_first_letters(sentence):
    """
    Capitalizes the first letter of each word in a sentence.

    Args:
        sentence (str): The input sentence.

    Returns:
        str: The sentence with the first letter of each word capitalized.
    """
    return sentence.title()

def main():
    # Get the sentence from the user
    sentence = input("Please enter a sentence: ")

    # Capitalize the first letter of each word
    capitalized_sentence = capitalize_first_letters(sentence)

    # Print the result
    print("Capitalized sentence:", capitalized_sentence)

if __name__ == "__main__":
    main()
```
