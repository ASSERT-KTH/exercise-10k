## Exercise
Create a program that takes in a list of values as input and determines the data type of each value. The program should then print out the value and its corresponding data type.

## Reference Solution
```python
def determine_data_type(input_list):
    """
    This function takes in a list of values, determines the data type of each value, 
    and prints out the value and its corresponding data type.
    
    Args:
        input_list (list): A list of values of different data types.
    """
    
    # Iterate over each value in the input list
    for value in input_list:
        # Use the type() function to get the data type of the value
        data_type = type(value).__name__
        
        # Print the value and its data type
        print(f"Value: {value}, Data Type: {data_type}")

# Example usage:
input_list = [1, 2.5, "Hello", True, None, [1, 2, 3], {"a": 1, "b": 2}, (1, 2, 3), {"apple", "banana"}]
determine_data_type(input_list)
```
