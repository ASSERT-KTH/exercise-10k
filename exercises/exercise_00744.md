## Exercise
Write a program that takes a person's name, age, and occupation as input and prints out a profile summary

## Reference Solution
```python
def get_profile_info():
    """Get person's profile information"""
    name = input("Enter your name: ")
    age = int(input("Enter your age: "))
    occupation = input("Enter your occupation: ")
    return name, age, occupation

def print_profile_summary(name, age, occupation):
    """Print person's profile summary"""
    print("\nProfile Summary:")
    print("----------------")
    print(f"Name: {name}")
    print(f"Age: {age}")
    print(f"Occupation: {occupation}")

def main():
    name, age, occupation = get_profile_info()
    print_profile_summary(name, age, occupation)

if __name__ == "__main__":
    main()
```
