## Exercise
Write a program that simulates a coin flip. The program should generate a random result (heads or tails) and print it out.

## Reference Solution
```python
import random

def coin_flip():
    """
    Simulates a coin flip and returns the result.
    """
    # Generate a random number, either 0 or 1
    result = random.randint(0, 1)
    
    # If the result is 0, it's heads, otherwise it's tails
    if result == 0:
        return "Heads"
    else:
        return "Tails"

def main():
    # Simulate a coin flip and print the result
    print("The coin landed on:", coin_flip())

if __name__ == "__main__":
    main()
```
