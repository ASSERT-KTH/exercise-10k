## Exercise
Create a function that takes a number as an argument and returns True if it is odd, False otherwise

## Reference Solution
```python
def is_odd(n: int) -> bool:
    """
    Checks if a given number is odd.

    Args:
        n (int): The number to check.

    Returns:
        bool: True if the number is odd, False otherwise.
    """
    return n % 2 != 0
```

```python
print(is_odd(3))  # True
print(is_odd(4))  # False
print(is_odd(-1)) # True
print(is_odd(0))  # False
```
