## Exercise
Write a function that takes a string as input and returns True if the string contains only whitespace characters, False otherwise.

## Reference Solution
```python
def check_whitespace(s):
    """
    This function checks if a given string contains only whitespace characters.

    Args:
        s (str): The input string to be checked.

    Returns:
        bool: True if the string contains only whitespace characters, False otherwise.
    """

    # We use the strip() method, which removes leading and trailing whitespace characters from the string
    # If the string contains only whitespace characters, this will result in an empty string
    stripped_string = s.strip()

    # We then check if the stripped string is empty
    # If it is, we return True, indicating that the original string contained only whitespace characters
    # Otherwise, we return False
    return stripped_string == ""

# Test the function
print(check_whitespace("   "))  # Should return True
print(check_whitespace("Hello"))  # Should return False
```
