## Exercise
Create a function to alternate the case of all characters in a given string

## Reference Solution
```python
def alternate_case(s):
    """
    Alternate the case of all characters in a given string.

    Args:
        s (str): The input string.

    Returns:
        str: The string with alternating case.
    """
    result = ""
    upper = True  # Start with uppercase

    for char in s:
        if char.isalpha():  # Check if the character is a letter
            if upper:
                result += char.upper()
            else:
                result += char.lower()
            upper = not upper  # Toggle the case
        else:
            result += char  # Non-letter characters are not changed

    return result

# Example usage:
print(alternate_case("Hello World"))  # Output: "HeLlO wOrLd"
```
