## Exercise
Write a function that takes a string as input and returns the string with all duplicate characters removed

## Reference Solution
```python
def remove_duplicates(input_str):
    """
    Removes duplicate characters from a string.

    Args:
        input_str (str): The input string.

    Returns:
        str: The string with all duplicate characters removed.
    """
    seen = set()
    output_str = ""

    for char in input_str:
        if char not in seen:
            seen.add(char)
            output_str += char

    return output_str


# Example usage:
if __name__ == "__main__":
    input_str = "hello world"
    print("Input String:", input_str)
    print("Output String:", remove_duplicates(input_str))
```
