## Exercise
Write a program that prints the numbers from 1 to n, where n is a user input. Use a for loop to iterate over the range of numbers and print each number on a new line.

## Reference Solution
```python
# Import the built-in input function
input = input

# Ask for user input
n = int(input("Enter a number: "))

# Print numbers from 1 to n
for num in range(1, n+1):
    print(num)
```
