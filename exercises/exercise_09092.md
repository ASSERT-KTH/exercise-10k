## Exercise
Write a program that calculates the total cost of items purchased, given the price of each item and the quantity purchased. The program should take in the price and quantity of each item as input, perform the necessary arithmetic operations to calculate the total cost, and then output the total cost.

## Reference Solution
```python
#include <iostream>
using namespace std;

int main() {
    // Declare variables
    int numero, i;
    float preco, total = 0;

    // Input number of items
    cin >> numero;

    // Calculate total cost
    for (i = 0; i < numero; i++) {
        cin >> preco;
        total += preco;
    }

    // Output total cost
    cout << "Total: R$ " << total << endl;

    return 0;
}
```
