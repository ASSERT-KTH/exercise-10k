## Exercise
Create a function that finds the intersection of two lists.

## Reference Solution
```python
def list_intersection(list1, list2):
    """
    Finds the intersection of two lists.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: A new list containing elements that are common to both input lists.
    """
    return [element for element in list1 if element in list2]

# Example usage:
def main():
    list1 = [1, 2, 3, 4, 5]
    list2 = [4, 5, 6, 7, 8]
    print("List 1:", list1)
    print("List 2:", list2)
    print("Intersection:", list_intersection(list1, list2))

if __name__ == "__main__":
    main()
```

def list_intersection_set(list1, list2):
    """
    Finds the intersection of two lists using sets.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: A new list containing elements that are common to both input lists.
    """
    return list(set(list1) & set(list2))

# Example usage:
def main():

    list2 = [4, 5, 6, 7, 8]
