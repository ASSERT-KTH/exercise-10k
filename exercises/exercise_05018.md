## Exercise
Write a program that prompts the user to enter their name and then prints out a greeting message with the name. Print the greeting message to the console.

## Reference Solution
```python
# Prompt the user for their name
def get_user_name():
    user_name = input("Please enter your name: ")
    return user_name

# Print out a greeting message
def print_greeting(name):
    print(f"Hello, {name}!")

# Main function
def main():
    user_name = get_user_name()
    print_greeting(user_name)

# Run the main function
if __name__ == "__main__":
    main()
```
