## Exercise
Write a program that takes an array of integers as input and returns the sorted array in descending order.

## Reference Solution
```python
# -*- coding: utf-8 -*-

def sort_descending(arr):
    arr = sorted(arr, reverse=True)
    return arr

arr = [5, 2, 9, 1, 7, 3]
print(sort_descending(arr))
```
