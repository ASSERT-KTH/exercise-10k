## Exercise
Write a program that takes an array of integers and a pivot value as input, and returns two new arrays: one containing the integers less than the pivot, and one containing the integers greater than the pivot.

## Reference Solution
```python
def pivot(arr, pivot):
    less = []
    greater = []
    for num in arr:
        if num < pivot:
            less.append(num)
        elif num > pivot:
            greater.append(num)
    return less, greater

# Example usage:
arr = [1, 2, 3, 4, 5, 6]
pivot_value = 4
less, greater = pivot(arr, pivot_value)
print(less)  # [1, 2, 3]
print(greater)  # [5, 6]
```
