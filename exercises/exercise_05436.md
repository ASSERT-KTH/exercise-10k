## Exercise
Write a program that checks if a phone number is valid or not. If the phone number contains 10 digits and is in the format (XXX) XXX-XXXX, it is valid. If the phone number does not meet these conditions, it is invalid.

## Reference Solution
```python
import re

def is_valid_phone_number(phone_number):
    """
    Checks if a phone number is valid.
    
    A phone number is valid if it contains 10 digits and is in the format (XXX) XXX-XXXX.
    
    Args:
        phone_number (str): The phone number to be validated.
    
    Returns:
        bool: True if the phone number is valid, False otherwise.
    """
    pattern = re.compile(r'^\(\d{3}\) \d{3}-\d{4}$')
    return bool(pattern.match(phone_number))

def main():
    # Example usage
    phone_numbers = [
        "(123) 456-7890",
        "(123) 456-789",
        "123-456-7890",
        "(123) 456-78901",
        "1234567890"
    ]
    
    for phone_number in phone_numbers:
        print(f"Phone Number: {phone_number}, Valid: {is_valid_phone_number(phone_number)}")

if __name__ == "__main__":
    main()
```
