## Exercise
Write a program that simulates rolling a dice and prints the result (1-6).

## Reference Solution
```python
import random

def roll_dice():
    """Simulates rolling a dice and returns the result."""
    return random.randint(1, 6)

def main():
    print("Rolling the dice...")
    result = roll_dice()
    print(f"The result is: {result}")

if __name__ == "__main__":
    main()
```
