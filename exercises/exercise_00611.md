## Exercise
Write a program to calculate a percentage of a given number.

## Reference Solution
```python
def calculate_percentage(number, percentage):
    """
    Calculate a percentage of a given number.

    Args:
        number (float): The number to calculate the percentage of.
        percentage (float): The percentage value.

    Returns:
        float: The calculated percentage.
    """
    return (percentage / 100) * number

def main():
    # Get user input
    number = float(input("Enter a number: "))
    percentage = float(input("Enter a percentage value: "))

    # Calculate percentage
    result = calculate_percentage(number, percentage)

    # Print result
    print(f"{percentage}% of {number} is {result}")

if __name__ == "__main__":
    main()
```
