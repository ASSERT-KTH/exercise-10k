## Exercise
Write a program that takes a string as input and checks if it's empty or not using conditional statements.

## Reference Solution
```python
#include <stdio.h>
#include <string.h>

void checkString(char s[]) {
    if (strcmp(s, "") == 0) {
        printf("The string is empty\n");
    } else {
        printf("The string is not empty\n");
    }
}

int main() {
    char s[] = "";
    checkString(s);
    return 0;
}
```
