## Exercise
Create a program that defines a function to calculate the result of a mathematical operation. The function should take in three parameters: two numbers and an operator. The operator can be '+', '-', '*', or '/'. The function should return the result of the operation. If the operator is not one of the above, the function should return an error message.

## Reference Solution
```python
def calc(num1, num2, operator):
    if operator == "+":
        return num1 + num2
    elif operator == "-":
        return num1 - num2
    elif operator == "*":
        return num1 * num2
    elif operator == "/":
        if num2 != 0:
            return num1 / num2
        else:
            return "Error: Division by zero"
    else:
        return "Error: Invalid operator"

print(calc(1, 2, "+"))  # Output: 3
print(calc(1, 2, "-"))  # Output: -1
print(calc(1, 2, "*"))  # Output: 2
print(calc(1, 2, "/"))  # Output: 0.5
print(calc(1, 0, "/"))  # Output: "Error: Division by zero"
print(calc(1, 2, "%"))  # Output: "Error: Invalid operator"
```
