## Exercise
Write a program that takes two strings as input and returns a new string with the characters of the two strings interleaved.

## Reference Solution
```python
def interleave(s1, s2):
    result = ''
    for i in range(max(len(s1), len(s2))):
        if i < len(s1):
            result += s1[i]
        if i < len(s2):
            result += s2[i]
    return result
```
