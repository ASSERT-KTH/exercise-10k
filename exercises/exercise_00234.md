## Exercise
Write a program to reverse the order of words in a given string.

## Reference Solution
```python
def reverse_word_order(input_string):
    """
    Reverses the order of words in a given string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The string with the order of words reversed.
    """
    words = input_string.split()  # split the string into words
    reversed_words = words[::-1]  # reverse the list of words
    reversed_string = ' '.join(reversed_words)  # join the reversed words into a string
    return reversed_string

# Example usage
def main():
    input_string = "Hello World this is Python"
    print("Input string:", input_string)
    reversed_string = reverse_word_order(input_string)
    print("Reversed string:", reversed_string)

if __name__ == "__main__":
    main()
```
