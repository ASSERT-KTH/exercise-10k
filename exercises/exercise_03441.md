## Exercise
Write a program that checks if a number is greater than 10 and prints 'Greater' if true, otherwise prints 'Lesser'

## Reference Solution
```python
def check_number(num):
    """
    Checks if a number is greater than 10.

    Args:
        num (int): The number to check.

    Returns:
        None
    """
    if num > 10:
        print("Greater")
    else:
        print("Lesser")

def main():
    num = int(input("Enter a number: "))
    check_number(num)

if __name__ == "__main__":
    main()
```
