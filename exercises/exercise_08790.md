## Exercise
Write a program that takes a weight in pounds as input and returns its equivalent in kilograms.

## Reference Solution
```python
def pounds_to_kilograms(pounds: float) -> float:
    """
    Converts a weight in pounds to kilograms.

    Args:
        pounds (float): The weight in pounds.

    Returns:
        float: The weight in kilograms.
    """
    kilograms = pounds * 0.45359237
    return kilograms

def main():
    pounds = float(input("Enter a weight in pounds: "))
    kilograms = pounds_to_kilograms(pounds)
    print(f"{pounds} pounds is equal to {kilograms:.2f} kilograms.")

if __name__ == "__main__":
    main()
```
