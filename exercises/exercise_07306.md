## Exercise
Create a function that takes a string as an argument and returns its version with leading and trailing whitespace removed

## Reference Solution
```python
def remove_whitespace(input_string: str) -> str:
    """
    Removes leading and trailing whitespace from a string.

    Args:
        input_string (str): The input string.

    Returns:
        str: The input string with leading and trailing whitespace removed.
    """
    return input_string.strip()

# Example usage
def main():
    input_str = "   Hello, World!   "
    print(f"Original string: '{input_str}'")
    print(f"String without whitespace: '{remove_whitespace(input_str)}'")

if __name__ == "__main__":
    main()
```
