## Exercise
Write a program that maps a function to each value in a dictionary and returns a new dictionary with the mapped values.

## Reference Solution
```python
def map_dict(func, d):
    return {k: func(v) for k, v in d.items()}

# Example usage:
d = {'a': 1, 'b': 2, 'c': 3}
print(map_dict(lambda x: x * 2, d))  # Output: {'a': 2, 'b': 4, 'c': 6}
```
