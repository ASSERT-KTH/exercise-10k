## Exercise
Write a program to generate a sequence of variables with increasing values, using a loop or other mechanism.

## Reference Solution
```python
# Define the number of variables you want to generate
num_variables = 10

# Initialize an empty dictionary to store the variables
variables = {}

# Loop through the range of the number of variables
for i in range(1, num_variables + 1):
    # Generate the variable name and value
    variable_name = f"var_{i}"
    variable_value = i * 10

    # Store the variable in the dictionary
    variables[variable_name] = variable_value

# Print the generated variables
for variable, value in variables.items():
    print(f"{variable} = {value}")
```
