## Exercise
Write a function to convert a dictionary to a list of tuples

## Reference Solution
```python
def dict_to_list_of_tuples(input_dict):
    """
    Converts a dictionary to a list of tuples.

    Args:
        input_dict (dict): The dictionary to convert.

    Returns:
        list: A list of tuples, where each tuple is a key-value pair from the dictionary.
    """
    return list(input_dict.items())

# Example usage
if __name__ == "__main__":
    example_dict = {"a": 1, "b": 2, "c": 3}
    result = dict_to_list_of_tuples(example_dict)
    print(result)  # Output: [('a', 1), ('b', 2), ('c', 3)]
```
