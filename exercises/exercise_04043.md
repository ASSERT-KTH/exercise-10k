## Exercise
Write a function that takes a dictionary as input and returns a list of all the values in the dictionary.

## Reference Solution
```python
def get_dict_values(input_dict):
    """
    This function takes a dictionary as input and returns a list of all the values in the dictionary.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        list: A list of all the values in the dictionary.
    """

    # First, we check if the input is indeed a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary")

    # We initialize an empty list to store the dictionary values
    dict_values = []

    # We use the .values() method of the dictionary to get a view object that displays a list of all values in the dictionary
    # The .values() method returns a dict_values object, which is a view object that displays a list of all values in the dictionary
    dict_values_view = input_dict.values()

    # We convert the dict_values object to a list using the list() function
    # This is because the problem requires us to return a list of all the values in the dictionary
    dict_values = list(dict_values_view)

    # Finally, we return the list of dictionary values
    return dict_values
```
