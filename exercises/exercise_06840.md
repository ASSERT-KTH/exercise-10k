## Exercise
Write a program that takes a user's name as input and prints out a personalized greeting message. The message should be in the format 'Hello, <name>!'. The program should also handle cases where the user enters an empty string or multiple names.

## Reference Solution
```python
# Importing Modules
import os
import time

# Printing the Greeting Message
def print_greeting(name):
    greeting = "Hello, " + name + "!"
    print(greeting)

# Getting User Input
def get_name():
    name = input("Please enter your name: ")
    return name

# Main Program
def main():
    name = get_name()
    print_greeting(name)

# Running the Main Program
if __name__ == "__main__":
    main()
```
