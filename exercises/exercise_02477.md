## Exercise
Create a function that generates all possible permutations of a given string.

## Reference Solution
```python
import itertools

def generate_permutations(input_string):
    """
    Generates all possible permutations of a given string.

    Args:
        input_string (str): The string for which permutations are to be generated.

    Returns:
        list: A list of all possible permutations of the input string.
    """
    # Use itertools.permutations to generate all permutations
    permutations = [''.join(p) for p in itertools.permutations(input_string)]
    
    return permutations

# Example usage
def main():
    input_string = "abc"
    permutations = generate_permutations(input_string)
    
    print("Permutations of '{}'".format(input_string))
    for i, permutation in enumerate(permutations):
        print("{}. {}".format(i+1, permutation))

if __name__ == "__main__":
    main()
```
