## Exercise
Write a function to clear all key-value pairs from a dictionary using the clear() method

## Reference Solution
```python
def clear_dictionary(input_dict):
    """
    Clears all key-value pairs from a dictionary using the clear() method.

    Args:
        input_dict (dict): The dictionary to be cleared.

    Returns:
        None
    """
    # Check if the input is a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary")

    # Clear the dictionary
    input_dict.clear()

    # Print the cleared dictionary for verification
    print("Cleared Dictionary:", input_dict)

# Example usage
if __name__ == "__main__":
    example_dict = {"key1": "value1", "key2": "value2", "key3": "value3"}
    print("Original Dictionary:", example_dict)
    clear_dictionary(example_dict)
```
