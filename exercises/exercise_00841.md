## Exercise
Create a function that takes another function as an argument and returns the number of parameters the given function has.

## Reference Solution
```python
import inspect

def count_function_parameters(func):
    """
    This function takes another function as an argument and returns the number of parameters the given function has.

    Args:
        func (function): The function to be inspected for its number of parameters.

    Returns:
        int: The number of parameters in the given function.
    """

    # First, we need to check if the input is a function. If it's not, we raise a TypeError.
    if not callable(func):
        raise TypeError("Input must be a function.")

    # We use the inspect module's signature function to get the function's signature.
    # This will give us a Signature object that contains information about the function's parameters.
    signature = inspect.signature(func)

    # We then use the parameters attribute of the Signature object to get a dictionary of the function's parameters.
    # The number of parameters is simply the length of this dictionary.
    # However, we need to exclude the 'self' parameter for instance methods, so we use a conditional expression to do this.
    # If the function is an instance method (i.e., it has a 'self' parameter), we subtract 1 from the total number of parameters.
    num_params = len(signature.parameters) - (1 if any(param.name == 'self' for param in signature.parameters.values()) else 0)

    # Finally, we return the number of parameters.
    return num_params
```
