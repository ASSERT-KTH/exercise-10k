## Exercise
Write a program that takes a string as input and returns the reversed string with each character shifted by 3 places in the ASCII table. For example, 'a' becomes 'd', 'b' becomes 'e', etc.

## Reference Solution
```python
a = input()
b = a[::-1]
print(''.join(chr((ord(i) + 3)) for i in b))
```
