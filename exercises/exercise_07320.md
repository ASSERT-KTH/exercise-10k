## Exercise
Write a program that takes input from the user and handles any errors that may occur.

## Reference Solution
```python
def get_user_input():
    """
    Asks the user for input and handles any potential errors.
    """
    while True:
        try:
            user_input = input("Please enter a number: ")
            # Attempt to convert the input to a float
            number = float(user_input)
            return number
        except ValueError:
            print("That's not a valid number. Please try again.")

def main():
    """
    Main function to test the get_user_input function.
    """
    print("Welcome to the user input program!")
    number = get_user_input()
    print("You entered the number:", number)

if __name__ == "__main__":
    main()
```
