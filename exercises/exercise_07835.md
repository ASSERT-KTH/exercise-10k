## Exercise
Create a program that takes two numbers as input and performs addition, subtraction, multiplication, and division based on the user's choice. The program should then output the result of the chosen operation.

## Reference Solution
```python
# Define a function to add two numbers
def add(x, y):
    return x + y

# Define a function to subtract two numbers
def sub(x, y):
    return x - y

# Define a function to multiply two numbers
def mul(x, y):
    return x * y

# Define a function to divide two numbers
def div(x, y):
    if y != 0:
        return x / y
    else:
        return "Error: Division by zero"

# Define a function to perform an operation based on user input
def op(x, y, n):
    if n == 1:
        return add(x, y)
    elif n == 2:
        return sub(x, y)
    elif n == 3:
        return mul(x, y)
    elif n == 4:
        return div(x, y)
    else:
        return "Error: Invalid operation choice"

# Main program
x = float(input("Enter the first number: "))
y = float(input("Enter the second number: "))

print("Choose an operation:")
print("1. Addition")
print("2. Subtraction")
print("3. Multiplication")
print("4. Division")

n = int(input("Enter your choice (1-4): "))

result = op(x, y, n)

print("Result:", result)
```
