## Exercise
Create a function that takes a list and a predicate function as input and returns the first element in the list for which the predicate function returns True. If no such element is found, return None.

## Reference Solution
```python
def find_first_element(lst, predicate):
    """
    This function takes a list and a predicate function as input, 
    and returns the first element in the list for which the predicate function returns True.
    
    If no such element is found, it returns None.

    Args:
        lst (list): The list to be searched.
        predicate (function): A function that takes one argument and returns a boolean value.

    Returns:
        The first element in the list for which the predicate function returns True, or None if no such element is found.
    """

    # Check if the input is a list
    if not isinstance(lst, list):
        raise TypeError("Input must be a list")

    # Check if the predicate is a callable function
    if not callable(predicate):
        raise TypeError("Predicate must be a callable function")

    # Iterate over each element in the list
    for element in lst:
        
        # Apply the predicate function to the current element
        # If the predicate function returns True, return the current element
        if predicate(element):
            return element
    
    # If we have iterated over the entire list and haven't returned an element, 
    # it means that no element satisfied the predicate function, so return None
    return None
```
