## Exercise
Write a program that takes a list as input and returns a new list containing only the unique elements

## Reference Solution
```python
def get_unique_elements(input_list):
    """
    Returns a new list containing only the unique elements from the input list.
    
    Args:
        input_list (list): The input list.
    
    Returns:
        list: A new list containing only the unique elements.
    """
    return list(set(input_list))

def main():
    # Example usage:
    input_list = [1, 2, 2, 3, 4, 4, 5, 6, 6]
    unique_list = get_unique_elements(input_list)
    print("Input List:", input_list)
    print("Unique List:", unique_list)

if __name__ == "__main__":
    main()
```
