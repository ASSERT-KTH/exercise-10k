## Exercise
Create a program that prompts the user to input their name, age, and favorite food. The program should then output a greeting message with the user's name, their age, and their favorite food.

## Reference Solution
```python
# Input Section
name = input("What is your name? ")
age = int(input("How old are you? "))
food = input("What is your favorite food? ")

# Processing Section
greeting = f"Hello my name is {name} and I am {age} years old. My favorite food is {food}."

# Output Section
print(greeting)
```
