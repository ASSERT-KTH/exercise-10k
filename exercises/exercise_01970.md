## Exercise
Create a function that finds the maximum contiguous subarray of a given array of numbers. The function should take an array as an argument and return the maximum subarray.

## Reference Solution
```python
def max_contiguous_subarray(arr):
    """
    Find the maximum contiguous subarray of a given array of numbers.

    Args:
        arr (list): A list of numbers.

    Returns:
        list: The maximum contiguous subarray.
    """
    if not arr:
        return []

    max_current = max_global = arr[0]
    start = end = 0
    temp_start = 0

    for i in range(1, len(arr)):
        if arr[i] > max_current + arr[i]:
            max_current = arr[i]
            temp_start = i
        else:
            max_current += arr[i]

        if max_current > max_global:
            max_global = max_current
            start = temp_start
            end = i

    return arr[start:end+1]

# Example usage:
arr = [-2, -3, 4, -1, -2, 1, 5, -3]
max_subarray = max_contiguous_subarray(arr)
print("Maximum contiguous subarray:", max_subarray)
```
