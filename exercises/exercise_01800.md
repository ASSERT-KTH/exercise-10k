## Exercise
Write a function to validate a dictionary against a given schema and return a boolean value indicating whether the dictionary is valid according to the schema.

## Reference Solution
```python
import jsonschema

def validate_dictionary(data, schema):
    """
    Validate a dictionary against a given schema.

    Args:
        data (dict): The dictionary to be validated.
        schema (dict): The schema to validate against.

    Returns:
        bool: True if the dictionary is valid according to the schema, False otherwise.
    """
    try:
        jsonschema.validate(instance=data, schema=schema)
        return True
    except jsonschema.exceptions.ValidationError:
        return False

# Example usage
if __name__ == "__main__":
    schema = {
        "type": "object",
        "properties": {
            "name": {"type": "string"},
            "age": {"type": "integer"}
        },
        "required": ["name", "age"]
    }

    data = {
        "name": "John Doe",
        "age": 30
    }

    print(validate_dictionary(data, schema))  # Output: True

    invalid_data = {
        "name": "Jane Doe"
    }

    print(validate_dictionary(invalid_data, schema))  # Output: False
```
