## Exercise
Write a program that uses nested for loops to print a pyramid pattern of asterisks (*).

## Reference Solution
```python
def print_pyramid(n):
    """
    Prints a pyramid pattern of asterisks of height n.
    
    Args:
    n (int): The height of the pyramid.
    """
    for i in range(n):
        # Print leading spaces
        for j in range(n - i - 1):
            print(" ", end="")
        
        # Print asterisks
        for k in range(2 * i + 1):
            print("*", end="")
        
        # Move to the next line
        print()

# Test the function
print_pyramid(5)
```
