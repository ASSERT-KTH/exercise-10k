## Exercise
Create a program that takes two numbers and an arithmetic operation as input and calculates the result. The program should support addition, subtraction, multiplication, and division.

## Reference Solution
```python
# -*- coding: utf-8 -*-

def calculate(num1, num2, op):
    if op == '+':
        result = num1 + num2
    elif op == '-':
        result = num1 - num2
    elif op == '*':
        result = num1 * num2
    elif op == '/':
        result = num1 / num2
    return result

num1 = float(input('Enter the first number: '))
op = input('Enter the arithmetic operation (+, -, *, /): ')
num2 = float(input('Enter the second number: '))

result = calculate(num1, num2, op)
print('The result is:', result)
```
