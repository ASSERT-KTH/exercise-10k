## Exercise
Create a program that uses a for loop to print the numbers from 1 to 100, but replaces multiples of 3 and 5 with 'Fizz' and 'Buzz' respectively.

## Reference Solution
```python
#include <iostream>

int main() {
    for (int i = 1; i <= 100; ++i) {
        if (i % 3 == 0 && i % 5 == 0) {
            std::cout << "FizzBuzz" << std::endl;
        } else if (i % 3 == 0) {
            std::cout << "Fizz" << std::endl;
        } else if (i % 5 == 0) {
            std::cout << "Buzz" << std::endl;
        } else {
            std::cout << i << std::endl;
        }
    }
    return 0;
}
```
