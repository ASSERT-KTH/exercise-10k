## Exercise
Write a function to map dictionary values to a new dictionary

## Reference Solution
```python
def map_dict_values(input_dict, mapping_func):
    """
    Maps dictionary values to a new dictionary using a given function.

    Args:
        input_dict (dict): The input dictionary.
        mapping_func (function): A function that takes a value and returns a new value.

    Returns:
        dict: A new dictionary with mapped values.
    """
    return {key: mapping_func(value) for key, value in input_dict.items()}

# Example usage:
if __name__ == "__main__":
    # Define an example dictionary
    example_dict = {"a": 1, "b": 2, "c": 3}

    # Define a mapping function (e.g., square the value)
    def square_value(x):
        return x ** 2

    # Map dictionary values to a new dictionary
    new_dict = map_dict_values(example_dict, square_value)

    # Print the result
    print("Original Dictionary:", example_dict)
    print("Mapped Dictionary:", new_dict)
```
