## Exercise
Write a program to find the Cartesian product of two or more lists.

## Reference Solution
```python
import itertools

def cartesian_product(*lists):
    """
    Calculate the Cartesian product of two or more lists.

    Args:
        *lists: Variable number of lists.

    Returns:
        A list of tuples, where each tuple is a combination of elements from each input list.
    """
    return list(itertools.product(*lists))

# Example usage
if __name__ == "__main__":
    list1 = [1, 2, 3]
    list2 = ['a', 'b']
    list3 = [True, False]

    result = cartesian_product(list1, list2, list3)
    print("Cartesian Product:")
    for combination in result:
        print(combination)
```
