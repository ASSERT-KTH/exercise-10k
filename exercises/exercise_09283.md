## Exercise
Create a program that takes a user input and determines its data type. The program should be able to identify integers, floats, and strings. If the input is not a valid number, it should be considered a string.

## Reference Solution
```python
def check_data_type(user_input):
    """
    Checks the data type of the user input.
    
    Args:
        user_input (str): The input provided by the user.
    
    Returns:
        str: The data type of the user input.
    """
    try:
        # Try to convert the input to an integer
        int(user_input)
        return "Integer"
    except ValueError:
        try:
            # If it's not an integer, try to convert it to a float
            float(user_input)
            return "Float"
        except ValueError:
            # If it's not a float, it's a string
            return "String"

def main():
    user_input = input("Please enter a value: ")
    data_type = check_data_type(user_input)
    print(f"The data type of '{user_input}' is: {data_type}")

if __name__ == "__main__":
    main()
```
