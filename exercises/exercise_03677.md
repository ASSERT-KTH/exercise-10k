## Exercise
Write a function that takes a string as input and returns the number of vowels it contains.

## Reference Solution
```python
def count_vowels(input_string: str) -> int:
    """
    Counts the number of vowels in a given string.

    Args:
        input_string (str): The string to count vowels in.

    Returns:
        int: The number of vowels in the string.
    """
    vowels = 'aeiou'
    return sum(1 for char in input_string.lower() if char in vowels)

# Example usage:
def main():
    input_str = "Hello World"
    print(f"The string '{input_str}' contains {count_vowels(input_str)} vowels.")

if __name__ == "__main__":
    main()
```
