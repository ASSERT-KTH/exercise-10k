## Exercise
Write a function to convert a dictionary to a set of keys

## Reference Solution
```python
def dict_to_set_of_keys(input_dict):
    """
    Converts a dictionary to a set of its keys.

    Args:
        input_dict (dict): The dictionary to be converted.

    Returns:
        set: A set of keys from the input dictionary.
    """
    return set(input_dict.keys())

# Example usage:
if __name__ == "__main__":
    example_dict = {"apple": 1, "banana": 2, "cherry": 3}
    keys_set = dict_to_set_of_keys(example_dict)
    print(keys_set)
```
