## Exercise
Create a function that takes in a dictionary and returns a new dictionary containing the same key-value pairs, but with the values sorted in ascending order

## Reference Solution
```python
def sort_dict_values(input_dict):
    """
    Creates a new dictionary with the same key-value pairs as the input dictionary,
    but with the values sorted in ascending order.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        dict: A new dictionary with sorted values.
    """
    # First, we sort the input dictionary items based on their values
    sorted_items = sorted(input_dict.items(), key=lambda item: item[1])

    # Then, we create a new dictionary with the sorted items
    sorted_dict = dict(sorted_items)

    return sorted_dict

# Example usage:
input_dict = {'a': 5, 'b': 1, 'c': 3, 'd': 2, 'e': 4}
sorted_dict = sort_dict_values(input_dict)

print("Input Dictionary:")
print(input_dict)
print("\nSorted Dictionary:")
print(sorted_dict)
```
