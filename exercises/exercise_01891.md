## Exercise
Swap the values of two variables without using a temporary variable

## Reference Solution
```python
# Define two variables
a = 5
b = 10

# Swap values using arithmetic operations
a = a + b  # a = 15
b = a - b  # b = 5
a = a - b  # a = 10

print("a =", a)  # Output: a = 10
print("b =", b)  # Output: b = 5
```
