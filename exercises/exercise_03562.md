## Exercise
Write a function that returns all keys from a given dictionary

## Reference Solution
```python
def get_dictionary_keys(dictionary):
    """
    Returns all keys from a given dictionary.

    Args:
        dictionary (dict): The dictionary to retrieve keys from.

    Returns:
        list: A list of dictionary keys.
    """
    return list(dictionary.keys())

# Example usage
if __name__ == "__main__":
    example_dict = {"apple": 1, "banana": 2, "cherry": 3}
    keys = get_dictionary_keys(example_dict)
    print(keys)
```
