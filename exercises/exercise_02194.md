## Exercise
Create a function to generate all possible anagrams of a given string. The function should return a list of all anagrams.

## Reference Solution
```python
from itertools import permutations

def generate_anagrams(input_string):
    """
    Generate all possible anagrams of a given string.

    Args:
        input_string (str): The input string for which anagrams are to be generated.

    Returns:
        list: A list of all unique anagrams.
    """

    # If the input string is empty, return an empty list
    if not input_string:
        return []

    # First, we need to import the permutations function from the itertools module.
    # This function returns all possible orderings of the input string.

    # Generate all permutations of the input string
    perms = permutations(input_string)

    # Initialize an empty set to store unique anagrams
    anagrams = set()

    # For each permutation, join the characters into a string and add it to the set
    for perm in perms:
        # Join the characters in the permutation into a string
        anagram = ''.join(perm)
        
        # Add the anagram to the set
        anagrams.add(anagram)

    # Convert the set of anagrams to a list and return it
    return list(anagrams)

# Example usage:
print(generate_anagrams('abc'))
```
