## Exercise
Create a program where the user can input the number of rows and columns, then use nested for loops to print a rectangle of asterisks (*) with the specified number of rows and columns.

## Reference Solution
```python
def print_rectangle(rows, cols):
    """
    Print a rectangle of asterisks (*) with the specified number of rows and columns.

    Args:
        rows (int): The number of rows in the rectangle.
        cols (int): The number of columns in the rectangle.
    """
    # Use nested for loops to print the rectangle
    for _ in range(rows):
        for _ in range(cols):
            print("*", end=" ")
        print()  # Move to the next line

def main():
    # Get the number of rows and columns from the user
    rows = int(input("Enter the number of rows: "))
    cols = int(input("Enter the number of columns: "))

    # Print the rectangle
    print_rectangle(rows, cols)

if __name__ == "__main__":
    main()
```
