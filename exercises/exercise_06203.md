## Exercise
Write a function that calculates the length of a given list.

## Reference Solution
```python
def calculate_list_length(input_list):
    """
    Calculate the length of a given list.

    Args:
        input_list (list): The input list.

    Returns:
        int: The length of the input list.
    """
    return len(input_list)

# Example usage:
def main():
    example_list = [1, 2, 3, 4, 5]
    print(f"Length of the list: {calculate_list_length(example_list)}")

if __name__ == "__main__":
    main()
```
