## Exercise
Write a program that converts an integer to a string.

## Reference Solution
```python
def int_to_str(n: int) -> str:
    """
    Converts an integer to a string.

    Args:
        n (int): The integer to be converted.

    Returns:
        str: The string representation of the integer.
    """
    return str(n)

def main():
    # Example usage
    num = 12345
    print(f"Integer: {num}")
    print(f"String: {int_to_str(num)}")

if __name__ == "__main__":
    main()
```
