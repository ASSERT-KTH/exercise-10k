## Exercise
Write a program that converts a time from 12-hour format to 24-hour format (e.g., 3:00 PM to 15:00)

## Reference Solution
```python
from datetime import datetime

def convert_to_24_hour(time_str):
    """
    Converts a time from 12-hour format to 24-hour format.

    Args:
        time_str (str): Time in 12-hour format (e.g., "3:00 PM")

    Returns:
        str: Time in 24-hour format (e.g., "15:00")
    """
    in_time = datetime.strptime(time_str, "%I:%M %p")
    out_time = in_time.strftime("%H:%M")
    return out_time

# Example usage
time_str = "3:00 PM"
print(f"Converting {time_str} to 24-hour format: {convert_to_24_hour(time_str)}")
```
