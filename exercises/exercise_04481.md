## Exercise
Create a function that takes in a mathematical operation and two numbers, then performs the operation and returns the result.

## Reference Solution
```python
def mathematical_operation(operation, num1, num2):
    """
    Performs a mathematical operation on two numbers.

    Args:
        operation (str): The mathematical operation to perform. Can be 'add', 'subtract', 'multiply', or 'divide'.
        num1 (float): The first number.
        num2 (float): The second number.

    Returns:
        float: The result of the mathematical operation.

    Raises:
        ValueError: If the operation is not one of 'add', 'subtract', 'multiply', or 'divide'.
        ZeroDivisionError: If the operation is 'divide' and the second number is zero.
    """

    # Define a dictionary to map operation strings to their corresponding functions
    operations = {
        'add': lambda x, y: x + y,
        'subtract': lambda x, y: x - y,
        'multiply': lambda x, y: x * y,
        'divide': lambda x, y: x / y
    }

    # Check if the operation is valid
    if operation not in operations:
        raise ValueError("Invalid operation. Must be one of 'add', 'subtract', 'multiply', or 'divide'.")

    # Perform the operation
    try:
        result = operations[operation](num1, num2)
    except ZeroDivisionError:
        raise ZeroDivisionError("Cannot divide by zero.")

    return result

# Example usage:
if __name__ == "__main__":
    print(mathematical_operation('add', 5, 3))  # Output: 8
    print(mathematical_operation('subtract', 5, 3))  # Output: 2
    print(mathematical_operation('multiply', 5, 3))  # Output: 15
    print(mathematical_operation('divide', 5, 3))  # Output: 1.6666666666666667
```
