## Exercise
Write a program that takes two lists as input, and returns a new list containing pairs of elements from the input lists.

## Reference Solution
```python
def pair_elements(list1, list2):
    """
    Returns a new list containing pairs of elements from the input lists.
    The pairs are created by matching elements at the same index from both lists.
    If the lists are of different lengths, the resulting list will be the length of the shorter list.
    """
    return list(zip(list1, list2))

# Example usage:
list1 = [1, 2, 3]
list2 = ['a', 'b', 'c']
print(pair_elements(list1, list2))  # Output: [(1, 'a'), (2, 'b'), (3, 'c')]
```

def pair_elements(list1, list2):
    """
    Returns a new list containing pairs of elements from the input lists.
    The pairs are created by matching elements at the same index from both lists.
    If the lists are of different lengths, the resulting list will be the length of the shorter list.
    """
    return [(x, y) for x, y in zip(list1, list2)]

# Example usage:
list1 = [1, 2, 3]
list2 = ['a', 'b', 'c']
print(pair_elements(list1, list2))  # Output: [(1, 'a'), (2, 'b'), (3, 'c')]

import itertools

def pair_elements(list1, list2):
    """
    Returns a new list containing all possible pairs of elements from the input lists.
    """
    return list(itertools.product(list1, list2))

# Example usage:
list1 = [1, 2]
list2 = ['a', 'b']
print(pair_elements(list1, list2))  # Output: [(1, 'a'), (1, 'b'), (2, 'a'), (2, 'b')]
