## Exercise
Write a program that asks the user for a number and prints out the next number in the sequence.

## Reference Solution
```python
def get_next_number():
    # Get user input
    num = input("Please enter a number: ")

    # Check if input is a valid integer
    try:
        num = int(num)
    except ValueError:
        print("Invalid input. Please enter a valid integer.")
        return get_next_number()

    # Print the next number in the sequence
    print("The next number in the sequence is:", num + 1)

# Call the function
get_next_number()
```
