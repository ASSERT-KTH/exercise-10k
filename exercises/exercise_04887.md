## Exercise
Return a list of all the values in a dictionary.

## Reference Solution
```python
def get_dict_values(dictionary):
    """
    Returns a list of all the values in a dictionary.
    
    Args:
    dictionary (dict): The input dictionary.
    
    Returns:
    list: A list of dictionary values.
    """
    return list(dictionary.values())

# Example usage:
my_dict = {'a': 1, 'b': 2, 'c': 3}
print(get_dict_values(my_dict))  # Output: [1, 2, 3]
```
