## Exercise
Write a program to calculate the sum of all elements in an array.

## Reference Solution
```python
def calculate_array_sum(array):
    """
    Calculate the sum of all elements in an array.

    Args:
        array (list): The input array.

    Returns:
        int: The sum of all elements in the array.
    """
    return sum(array)


def main():
    # Example usage
    array = [1, 2, 3, 4, 5]
    array_sum = calculate_array_sum(array)
    print(f"The sum of the array is: {array_sum}")


if __name__ == "__main__":
    main()
```
