## Exercise
Write a program to create a simple web page using a web framework.

## Reference Solution
```python
# Import the Flask library
from flask import Flask

# Create a new Flask application
app = Flask(__name__)

# Define a route for the root URL
@app.route('/')
def home():
    # Return a simple HTML page
    return '<h1>Welcome to my simple web page!</h1>'

# Define a route for the about URL
@app.route('/about')
def about():
    # Return a simple HTML page
    return '<h1>About me...</h1><p>This is a simple web page created using Flask.</p>'

# Run the application
if __name__ == '__main__':
    app.run(debug=True)
```
