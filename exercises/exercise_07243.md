## Exercise
Write a program that takes a user's input and converts it into different data types (integer, float, string). The program should then print out the input in each of the converted data types.

## Reference Solution
```python
def convert_input_to_different_data_types():
    # Get the user's input
    user_input = input("Please enter a value: ")

    # Try to convert the input to an integer
    try:
        integer_input = int(user_input)
        print(f"Input as an integer: {integer_input}")
    except ValueError:
        print(f"Input '{user_input}' cannot be converted to an integer.")

    # Try to convert the input to a float
    try:
        float_input = float(user_input)
        print(f"Input as a float: {float_input}")
    except ValueError:
        print(f"Input '{user_input}' cannot be converted to a float.")

    # The input is already a string, so we just print it
    print(f"Input as a string: {user_input}")

def main():
    convert_input_to_different_data_types()

if __name__ == "__main__":
    main()
```
