## Exercise
Write a program that deletes a key-value pair from a dictionary.

## Reference Solution
```python
def delete_key_value_pair(dictionary, key):
    """
    Deletes a key-value pair from a dictionary.

    Args:
        dictionary (dict): The dictionary to delete the key-value pair from.
        key (any): The key of the key-value pair to delete.

    Returns:
        dict: The updated dictionary.
    """
    if key in dictionary:
        del dictionary[key]
    return dictionary

# Example usage
if __name__ == "__main__":
    my_dict = {"name": "John", "age": 30, "city": "New York"}
    print("Original dictionary:", my_dict)

    key_to_delete = "age"
    updated_dict = delete_key_value_pair(my_dict, key_to_delete)
    print("Updated dictionary after deleting '{}' key:".format(key_to_delete), updated_dict)
```
