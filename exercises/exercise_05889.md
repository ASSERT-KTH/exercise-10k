## Exercise
Write a function that takes a list and a binary function as input and returns the result of applying the binary function to all elements of the list.

## Reference Solution
```python
def apply_binary_function(lst, func):
    """
    Applies a binary function to all elements of a list.

    Args:
        lst (list): A list of elements.
        func (function): A binary function.

    Returns:
        result: The result of applying the binary function to all elements of the list.
    """

    # Check if the list is empty
    if not lst:
        return None  # or any other default value

    # Initialize the result with the first element of the list
    # This is because we need to start the reduction process from somewhere
    result = lst[0]

    # Iterate over the rest of the list
    for i in range(1, len(lst)):
        # Apply the binary function to the current result and the current element
        # and update the result with the new value
        result = func(result, lst[i])

    # After iterating over all elements, return the final result
    return result

# Example usage:
# Define a binary function that adds two numbers
def add(a, b):
    return a + b

# Create a list of numbers
numbers = [1, 2, 3, 4, 5]

# Apply the binary function to all elements of the list
result = apply_binary_function(numbers, add)
print(result)  # Output: 15
```
