Create a program with a basic bug (e.g. off-by-one error) and debug it using a debugger or print statements, then print out the debugged output.

public class Main {
    public static void main(String[] args) {
        System.out.println("Debugged output: ");
        printElements(new int[]{1, 2, 3, 4, 5});
    }

    public static void printElements(int[] arr) {
        for (int i = 0; i < arr.length; i++) {  // Change: i = 0 to i = 1
            System.out.println(arr[i]);
        }
    }
}
