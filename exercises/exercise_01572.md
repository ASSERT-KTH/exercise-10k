## Exercise
Create a function that registers and returns a new function with a given name.

## Reference Solution
```python
# Create a dictionary to store the registered functions
_registered_functions = {}

def register_function(name, func):
    """
    Registers and returns a new function with a given name.

    Args:
        name (str): The name of the function to be registered.
        func (function): The function to be registered.

    Returns:
        function: The registered function.
    """
    
    # Check if the function name already exists in the dictionary
    if name in _registered_functions:
        raise ValueError("Function with this name already exists")

    # Add the function to the dictionary
    _registered_functions[name] = func
    
    # Return the registered function
    return func

# Example usage:
def add(a, b):
    """Adds two numbers."""
    return a + b

registered_add = register_function('add', add)

print(registered_add(5, 10))  # Outputs: 15
```
