## Exercise
Write a program that takes an integer as input and checks if it is even or odd. Use if/else statements to determine the result.

## Reference Solution
```python
def check_even_odd(num):
    if num % 2 == 0:
        print(f"{num} is even.")
    else:
        print(f"{num} is odd.")

# Test the function
num = int(input("Enter an integer: "))
check_even_odd(num)
```

def check_even_odd(num):
