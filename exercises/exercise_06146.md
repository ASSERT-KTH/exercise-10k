## Exercise
Write a function to replace a substring with another string in a given string.

## Reference Solution
```python
def replace_substring(original_string, old_substring, new_substring):
    """
    Replaces a substring with another string in a given string.

    Args:
        original_string (str): The original string.
        old_substring (str): The substring to be replaced.
        new_substring (str): The new substring to replace with.

    Returns:
        str: The modified string with the old substring replaced by the new substring.
    """
    return original_string.replace(old_substring, new_substring)

# Example usage:
original_string = "Hello, world! This is a test string."
old_substring = "test"
new_substring = "sample"

modified_string = replace_substring(original_string, old_substring, new_substring)
print(modified_string)
```
