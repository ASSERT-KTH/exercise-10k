## Exercise
Write a program that takes an array and an element as input and returns the index of the first occurrence of the element in the array

## Reference Solution
```python
def find_element_index(array, element):
    """
    This function finds the index of the first occurrence of an element in an array.

    Args:
        array (list): The input array.
        element: The element to be searched.

    Returns:
        int: The index of the first occurrence of the element. Returns -1 if the element is not found.

    Raises:
        TypeError: If the input array is not a list.
    """

    # Check if the input array is a list
    if not isinstance(array, list):
        raise TypeError("Input array must be a list.")

    # Try to find the index of the element
    try:
        # Use the index method of the list to find the index of the element
        index = array.index(element)
        return index
    except ValueError:
        # If the element is not found, return -1
        return -1


# Example usage:
array = [1, 2, 3, 4, 5]
element = 3
print(find_element_index(array, element))  # Output: 2

array = [1, 2, 3, 4, 5]
element = 6
print(find_element_index(array, element))  # Output: -1
```
