## Exercise
Create a function that takes a name and age, with age defaulting to 25 if not provided, using keyword arguments

## Reference Solution
```python
def create_person(*, name, age=25):
    """
    Creates a person object with a name and age.

    Args:
    name (str): The name of the person.
    age (int, optional): The age of the person. Defaults to 25.

    Returns:
    dict: A dictionary representing the person object.
    """
    return {"name": name, "age": age}

# Example usage:
if __name__ == "__main__":
    person1 = create_person(name="John")
    person2 = create_person(name="Jane", age=30)

    print(person1)  # Output: {'name': 'John', 'age': 25}
    print(person2)  # Output: {'name': 'Jane', 'age': 30}
```
