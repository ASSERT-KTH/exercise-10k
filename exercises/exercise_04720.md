## Exercise
Filter a dictionary based on a condition

## Reference Solution
```python
def filter_dict(input_dict, condition):
    """
    Filter a dictionary based on a condition.

    Args:
    input_dict (dict): The dictionary to filter.
    condition (function): A function that takes a key-value pair and returns True if the pair should be included in the output.

    Returns:
    dict: A new dictionary containing only the key-value pairs that satisfy the condition.
    """
    return {key: value for key, value in input_dict.items() if condition(key, value)}

# Example usage:
input_dict = {"a": 1, "b": 2, "c": 3, "d": 4}
condition = lambda key, value: value > 2

filtered_dict = filter_dict(input_dict, condition)
print(filtered_dict)  # Output: {"c": 3, "d": 4}
```

```python
def filter_dict(input_dict, condition):
    """
    Filter a dictionary based on a condition.

    Args:
    input_dict (dict): The dictionary to filter.
    condition (function): A function that takes a key-value pair and returns True if the pair should be included in the output.

    Returns:
    dict: A new dictionary containing only the key-value pairs that satisfy the condition.
    """
    return dict(filter(condition, input_dict.items()))

# Example usage:
input_dict = {"a": 1, "b": 2, "c": 3, "d": 4}
condition = lambda pair: pair[1] > 2

filtered_dict = filter_dict(input_dict, condition)
print(filtered_dict)  # Output: {"c": 3, "d": 4}
```
