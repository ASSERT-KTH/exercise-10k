## Exercise
Write a function that takes a dictionary and a key as input and removes and returns the corresponding value using the pop() method

## Reference Solution
```python
def remove_and_return(dictionary, key):
    """
    Removes and returns the value corresponding to the given key from the dictionary.

    Args:
        dictionary (dict): The dictionary from which to remove the value.
        key: The key of the value to be removed.

    Returns:
        The removed value, or None if the key is not found in the dictionary.
    """
    return dictionary.pop(key, None)

# Example usage:
if __name__ == "__main__":
    my_dict = {"a": 1, "b": 2, "c": 3}
    key_to_remove = "b"
    removed_value = remove_and_return(my_dict, key_to_remove)
    print(f"Removed value: {removed_value}")
    print(f"Updated dictionary: {my_dict}")
```
