## Exercise
Write a program that takes a student's score as input and calculates their grade based on the following scale: A (90-100), B (80-89), C (70-79), D (60-69), F (below 60). Print the grade to the console.

## Reference Solution
```python
# Author: [Your Name]
# Date: [Current Date]

# Importing Modules
import sys

# Defining Functions
def grade(score):
    if score >= 90:
        return 'A'
    elif score >= 80:
        return 'B'
    elif score >= 70:
        return 'C'
    elif score >= 60:
        return 'D'
    else:
        return 'F'

# Main Program
try:
    score = int(sys.stdin.readline().strip())
    print(grade(score))
except ValueError:
    print("Invalid input. Please enter an integer score.")
```
