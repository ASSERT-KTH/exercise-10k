## Exercise
Create a program that uses nested for loops to print a pattern of asterisks (*).

## Reference Solution
```python
# Function to print asterisk pattern
def print_asterisk_pattern(size):
    for i in range(size):
        for j in range(size - i - 1):
            print(" ", end=" ")
        for k in range(2 * i + 1):
            print("*", end=" ")
        print()

# Test the function
print_asterisk_pattern(5)
```
