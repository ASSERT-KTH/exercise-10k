## Exercise
Create a dictionary from two lists of keys and values.

## Reference Solution
```python
# Define the lists of keys and values
keys = ['a', 'b', 'c']
values = [1, 2, 3]

# Use the zip function to create a dictionary
dictionary = dict(zip(keys, values))

# Print the resulting dictionary
print(dictionary)
```
