## Exercise
Write a program that takes a string input from the user, checks if it is a palindrome, and then reverses the string. The program should output whether the string is a palindrome and the reversed string.

## Reference Solution
```python
def check_palindrome(s):
    """
    Checks if the input string is a palindrome.
    
    Args:
        s (str): The input string.
    
    Returns:
        bool: True if the string is a palindrome, False otherwise.
    """
    return s == s[::-1]

def reverse_string(s):
    """
    Reverses the input string.
    
    Args:
        s (str): The input string.
    
    Returns:
        str: The reversed string.
    """
    return s[::-1]

def main():
    # Get the input string from the user
    user_input = input("Please enter a string: ")
    
    # Check if the string is a palindrome
    is_palindrome = check_palindrome(user_input)
    
    # Reverse the string
    reversed_string = reverse_string(user_input)
    
    # Print the results
    print(f'Is "{user_input}" a palindrome? {is_palindrome}')
    print(f'The reversed string is: {reversed_string}')

if __name__ == "__main__":
    main()
```
