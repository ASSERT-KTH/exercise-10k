## Exercise
Write a program that reduces a list of numbers to a single value by applying a specified operation (e.g. sum, product, etc.).

## Reference Solution
```python
def reduce_list_to_single_value(lst, func):
    if len(lst) == 0:
        return 0
    elif len(lst) == 1:
        return lst[0]
    else:
        result = lst[0]
        for num in lst[1:]:
            result = func(result, num)
        return result

# Example usage:
numbers = [1, 2, 3, 4, 5]
sum_result = reduce_list_to_single_value(numbers, lambda x, y: x + y)
print("Sum:", sum_result)

product_result = reduce_list_to_single_value(numbers, lambda x, y: x * y)
print("Product:", product_result)
```
