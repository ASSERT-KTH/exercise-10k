## Exercise
Create a dictionary with names of colors as keys and their hex codes as values. Remove a specific key-value pair from the dictionary and print the updated dictionary.

## Reference Solution
```python
# Define a dictionary of colors and their hex codes
colors = {
    "red": "#FF0000",
    "green": "#00FF00",
    "blue": "#0000FF",
    "yellow": "#FFFF00",
    "purple": "#FF00FF"
}

# Print the original dictionary
print("Original Dictionary:")
print(colors)

# Remove a specific key-value pair from the dictionary
key_to_remove = "green"
if key_to_remove in colors:
    del colors[key_to_remove]
else:
    print(f"The key '{key_to_remove}' does not exist in the dictionary.")

# Print the updated dictionary
print("\nUpdated Dictionary:")
print(colors)
```
