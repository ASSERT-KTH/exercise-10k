## Exercise
Create an array of 5 integers and find the maximum element.

## Reference Solution
```python
def find_max_element():
    # Create an array of 5 integers
    array = [12, 45, 7, 23, 56]

    # Find the maximum element
    max_element = max(array)

    return max_element

# Example usage
if __name__ == "__main__":
    max_element = find_max_element()
    print("Maximum element in the array:", max_element)
```
