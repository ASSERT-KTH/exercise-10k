## Exercise
Create a string with placeholders and format it using a specific formatting scheme, then print out the formatted string.

## Reference Solution
```python
# Create a string with placeholders
string_with_placeholders = "Hello, my name is {} and I am {} years old."

# Define the values to replace the placeholders
name = "John"
age = 30

# Format the string using the str.format() method
formatted_string = string_with_placeholders.format(name, age)

# Print the formatted string
print(formatted_string)
```
