## Exercise
Ask the user for a string and an integer, then print out the string repeated that many times.

## Reference Solution
```python
def repeat_string():
    # Ask the user for a string
    user_string = input("Please enter a string: ")

    # Ask the user for an integer
    while True:
        try:
            repeat_count = int(input("Please enter the number of times to repeat the string: "))
            break
        except ValueError:
            print("Invalid input. Please enter a whole number.")

    # Print out the string repeated that many times
    print((user_string + " ") * repeat_count).strip()

# Example usage
if __name__ == "__main__":
    repeat_string()
```
