## Exercise
Write a program that uses a for loop to print out the numbers from 1 to 10, then uses a while loop to print out the numbers from 11 to 15.

## Reference Solution
```python
# Printing numbers from 1 to 10 using a for loop
for i in range(1, 11):
    print(i)

# Printing numbers from 11 to 15 using a while loop
i = 11
while i <= 15:
    print(i)
    i += 1
```
