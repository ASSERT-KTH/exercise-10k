## Exercise
Create a function that uses the map() function to apply a given function to a list of numbers

## Reference Solution
```python
def apply_function_to_list(func, numbers):
    """
    Applies a given function to a list of numbers using map().

    Args:
        func (function): The function to be applied to the list of numbers.
        numbers (list): A list of numbers.

    Returns:
        list: A list of results after applying the function to the input list.
    """
    return list(map(func, numbers))

# Example usage:
def square(x):
    """Returns the square of a number."""
    return x ** 2

numbers = [1, 2, 3, 4, 5]
result = apply_function_to_list(square, numbers)
print(result)  # Output: [1, 4, 9, 16, 25]
```
