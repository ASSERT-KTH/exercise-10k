## Exercise
Write a program that converts a given tuple into a list

## Reference Solution
```python
def tuple_to_list(input_tuple):
    """
    Converts a given tuple into a list.

    Args:
        input_tuple (tuple): The tuple to be converted.

    Returns:
        list: The list equivalent of the input tuple.
    """
    return list(input_tuple)


# Example usage:
def main():
    # Define a tuple
    my_tuple = (1, 2, 3, 4, 5)

    # Convert the tuple to a list
    my_list = tuple_to_list(my_tuple)

    # Print the results
    print("Original Tuple:", my_tuple)
    print("Converted List:", my_list)


if __name__ == "__main__":
    main()
```
