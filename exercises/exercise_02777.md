## Exercise
Write a program that takes a word as input and scrambles its letters.

## Reference Solution
```python
import random

def scramble_word(word):
    """
    Scrambles the letters of a given word.

    Args:
        word (str): The word to be scrambled.

    Returns:
        str: The scrambled word.
    """
    # Convert the word to a list of characters
    word_list = list(word)
    
    # Shuffle the list of characters
    random.shuffle(word_list)
    
    # Join the list of characters back into a string
    scrambled_word = ''.join(word_list)
    
    return scrambled_word

def main():
    # Get the word from the user
    word = input("Enter a word: ")
    
    # Scramble the word
    scrambled_word = scramble_word(word)
    
    # Print the scrambled word
    print("Scrambled word: ", scrambled_word)

if __name__ == "__main__":
    main()
```
