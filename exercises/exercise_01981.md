## Exercise
Write a program to determine if a given number is even or odd using conditional statements

## Reference Solution
```python
def check_even_odd(num):
    if num % 2 == 0:
        print(f"{num} is even.")
    else:
        print(f"{num} is odd.")

# Example usage
num = int(input("Enter a number: "))
check_even_odd(num)
```
