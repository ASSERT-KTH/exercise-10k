## Exercise
Create a program that takes in an integer and checks if it's even or odd. If the number is even, print 'Even', otherwise print 'Odd'

## Reference Solution
```python
def check_even_or_odd(num: int) -> None:
    """
    Checks if a number is even or odd and prints the result.

    Args:
        num (int): The number to check.
    """
    if num % 2 == 0:
        print("Even")
    else:
        print("Odd")

def main() -> None:
    # Get the number from the user
    num = int(input("Enter an integer: "))

    # Check if the number is even or odd
    check_even_or_odd(num)

if __name__ == "__main__":
    main()
```
