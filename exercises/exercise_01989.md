## Exercise
Develop a program to convert time from 12-hour format to 24-hour format using conditional statements

## Reference Solution
```python
def convert_to_24_hour(time_12_hour):
    components = time_12_hour.split()
    time = components[0].split(":")
    hours = int(time[0])
    minutes = int(time[1])
    seconds = int(time[2])
    period = components[1]

    if period == "AM":
        if hours == 12:
            hours = 0
    else:
        if hours != 12:
            hours += 12

    return f"{hours:02d}:{minutes:02d}:{seconds:02d}"

# Example usage:
time_12_hour = "12:30:00 PM"
print(f"12-hour time: {time_12_hour}")
print(f"24-hour time: {convert_to_24_hour(time_12_hour)}")

time_12_hour = "06:30:00 AM"
print(f"\n12-hour time: {time_12_hour}")
print(f"24-hour time: {convert_to_24_hour(time_12_hour)}")
```
