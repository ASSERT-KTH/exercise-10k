## Exercise
Write a function that takes a dictionary and a function, then applies the function to each value in the dictionary and returns a new dictionary with the results

## Reference Solution
```python
def apply_function_to_dict_values(func, dictionary):
    """
    Applies a given function to each value in a dictionary and returns a new dictionary with the results.

    Args:
        func (function): The function to apply to each value.
        dictionary (dict): The dictionary to operate on.

    Returns:
        dict: A new dictionary with the function applied to each value.
    """
    return {key: func(value) for key, value in dictionary.items()}

# Example usage:
def double_value(x):
    """Doubles the given value."""
    return x * 2

example_dict = {'a': 1, 'b': 2, 'c': 3}
result_dict = apply_function_to_dict_values(double_value, example_dict)

print(result_dict)  # Output: {'a': 2, 'b': 4, 'c': 6}
```
