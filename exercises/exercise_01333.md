## Exercise
Write a function that takes a string as input and returns a scrambled version of the string.

## Reference Solution
```python
import random

def scramble_string(input_string: str) -> str:
    """
    Scrambles a string by shuffling its characters.
    
    Args:
    input_string (str): The string to be scrambled.
    
    Returns:
    str: The scrambled version of the input string.
    """
    # Convert the string to a list of characters
    char_list = list(input_string)
    
    # Shuffle the list of characters
    random.shuffle(char_list)
    
    # Join the shuffled characters back into a string
    scrambled_string = ''.join(char_list)
    
    return scrambled_string

# Example usage
if __name__ == "__main__":
    input_str = "Hello, World!"
    scrambled_str = scramble_string(input_str)
    print(f"Input: {input_str}")
    print(f"Scrambled: {scrambled_str}")
```
