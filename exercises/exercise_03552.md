## Exercise
Write a program that takes two lists as input, and returns a new list containing the common elements.

## Reference Solution
```python
def common_elements(list1, list2):
    """
    Returns a new list containing the common elements in list1 and list2.
    
    Args:
        list1 (list): The first list.
        list2 (list): The second list.
    
    Returns:
        list: A new list containing the common elements.
    """
    return [element for element in list1 if element in list2]

# Example usage:
def main():
    list1 = [1, 2, 3, 4, 5]
    list2 = [4, 5, 6, 7, 8]
    common = common_elements(list1, list2)
    print("Common elements:", common)

if __name__ == "__main__":
    main()
```

def common_elements_set(list1, list2):
    """
    Returns a new list containing the common elements in list1 and list2 without duplicates.
    
    Args:
        list1 (list): The first list.
        list2 (list): The second list.
    
    Returns:
        list: A new list containing the common elements without duplicates.
    """
    return list(set(list1) & set(list2))

# Example usage:
def main():
    list1 = [1, 2, 3, 4, 5, 4]
    list2 = [4, 5, 6, 7, 8]
