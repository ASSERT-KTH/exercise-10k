## Exercise
Write a program to rotate an array by a specified number of positions.

## Reference Solution
```python
def rotate_array(arr, num_positions, direction='clockwise'):
    """
    Rotate an array by a specified number of positions.

    Args:
        arr (list): The input array to be rotated.
        num_positions (int): The number of positions to rotate the array.
        direction (str, optional): The direction of rotation. Defaults to 'clockwise'.

    Returns:
        list: The rotated array.
    """
    num_positions = num_positions % len(arr)  # Handle cases where num_positions is greater than len(arr)
    
    if direction == 'clockwise':
        return arr[-num_positions:] + arr[:-num_positions]
    elif direction == 'counter-clockwise':
        return arr[num_positions:] + arr[:num_positions]
    else:
        raise ValueError("Invalid direction. Please specify 'clockwise' or 'counter-clockwise'.")

# Example usage
if __name__ == "__main__":
    arr = [1, 2, 3, 4, 5]
    num_positions = 2

    print("Original array:", arr)
    print("Rotated array (clockwise):", rotate_array(arr, num_positions))
    print("Rotated array (counter-clockwise):", rotate_array(arr, num_positions, direction='counter-clockwise'))
```
