## Exercise
Write a program that takes a list and a predicate as input, and returns a new list containing only the elements that satisfy the predicate.

## Reference Solution
```python
class Solution:
    def filter_list(self, input_list, predicate):
        # Your code here
        return [element for element in input_list if predicate(element)]

# Example usage
def is_even(num):
    return num % 2 == 0

numbers = [1, 2, 3, 4, 5, 6]
even_numbers = Solution().filter_list(numbers, is_even)
print(even_numbers)  # Output: [2, 4, 6]
```
