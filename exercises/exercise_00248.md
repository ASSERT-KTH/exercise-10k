## Exercise
Write a function that takes a list and an element as input, and returns a new list with the element inserted at a specified position.

## Reference Solution
```python
def insert_element_at_position(list, element, position):
    if position < 0 or position > len(list):
        raise ValueError("Invalid position")

    return list[:position] + [element] + list[position:]

# Test the function
my_list = [1, 2, 3, 4, 5]
print(insert_element_at_position(my_list, 10, 2))  # Output: [1, 2, 10, 3, 4, 5]
```
