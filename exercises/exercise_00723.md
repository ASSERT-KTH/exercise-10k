## Exercise
Write a program that takes two numbers as input and outputs their division result and remainder.

## Reference Solution
```python
# Function to calculate division result and remainder
def calculate_division_result_and_remainder(dividend, divisor):
    # Check if divisor is not zero
    if divisor == 0:
        return "Error: Division by zero is not allowed."
    else:
        # Calculate division result and remainder
        division_result = dividend // divisor
        remainder = dividend % divisor
        return division_result, remainder

# Main function
def main():
    # Take input from user
    dividend = int(input("Enter the dividend: "))
    divisor = int(input("Enter the divisor: "))

    # Calculate division result and remainder
    result = calculate_division_result_and_remainder(dividend, divisor)

    # Check if result is an error message
    if isinstance(result, str):
        print(result)
    else:
        division_result, remainder = result
        print(f"{dividend} divided by {divisor} is {division_result} with a remainder of {remainder}.")

# Run the main function
if __name__ == "__main__":
    main()
```
