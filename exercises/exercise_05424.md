## Exercise
Write a program that converts temperature from Celsius to Fahrenheit or vice versa. If the user chooses to convert from Celsius to Fahrenheit, use the formula F = (C * 9/5) + 32. If the user chooses to convert from Fahrenheit to Celsius, use the formula C = (F - 32) * 5/9.

## Reference Solution
```python
def celsius_to_fahrenheit(celsius):
    """
    Converts temperature from Celsius to Fahrenheit.

    Args:
        celsius (float): Temperature in Celsius.

    Returns:
        float: Temperature in Fahrenheit.
    """
    return (celsius * 9/5) + 32


def fahrenheit_to_celsius(fahrenheit):
    """
    Converts temperature from Fahrenheit to Celsius.

    Args:
        fahrenheit (float): Temperature in Fahrenheit.

    Returns:
        float: Temperature in Celsius.
    """
    return (fahrenheit - 32) * 5/9


def main():
    while True:
        print("\nTemperature Conversion Menu:")
        print("1. Convert Celsius to Fahrenheit")
        print("2. Convert Fahrenheit to Celsius")
        print("3. Quit")

        choice = input("Enter your choice (1, 2, or 3): ")

        if choice == "1":
            celsius = float(input("Enter temperature in Celsius: "))
```
