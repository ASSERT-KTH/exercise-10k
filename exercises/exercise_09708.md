## Exercise
Create a program that takes a string as input and checks if it contains at least one uppercase letter, one lowercase letter, one digit, and one special character. The program should return 'Valid' if the string meets all the conditions, otherwise return 'Invalid'. The string length should be between 8 and 32 characters.

## Reference Solution
```python
def check_string(input_str):
    """
    This function checks if the input string meets certain conditions.
    
    The string should contain at least one uppercase letter, one lowercase letter, 
    one digit, and one special character. The length of the string should be between 
    8 and 32 characters.

    Args:
        input_str (str): The input string to be checked.

    Returns:
        str: 'Valid' if the string meets all the conditions, otherwise 'Invalid'.
    """

    # First, we check the length of the string. If it's not between 8 and 32 characters, 
    # we immediately return 'Invalid'.
    if not 8 <= len(input_str) <= 32:
        return 'Invalid'

    # Initialize flags for uppercase, lowercase, digit, and special character.
    has_uppercase = False
    has_lowercase = False
    has_digit = False
    has_special_char = False

    # Iterate over each character in the string.
    for char in input_str:
        # Check if the character is an uppercase letter.
        if char.isupper():
            has_uppercase = True
        # Check if the character is a lowercase letter.
        elif char.islower():
            has_lowercase = True
        # Check if the character is a digit.
        elif char.isdigit():
            has_digit = True
        # Check if the character is not a letter or a digit (i.e., it's a special character).
        elif not char.isalnum():
            has_special_char = True

    # If all flags are True, the string meets all the conditions, so we return 'Valid'.
    if has_uppercase and has_lowercase and has_digit and has_special_char:
        return 'Valid'
    # If any of the flags are False, the string does not meet all the conditions, so we return 'Invalid'.
    else:
        return 'Invalid'

# Test the function
print(check_string("P@ssw0rd"))  # Should return 'Valid'
print(check_string("password"))  # Should return 'Invalid'
```
