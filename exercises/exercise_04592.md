## Exercise
Write a function that takes a list of elements as input and returns all possible permutations

## Reference Solution
```python
import itertools

def generate_permutations(input_list):
    """
    Generates all possible permutations of a given list of elements.

    Args:
        input_list (list): The list of elements for which permutations are to be generated.

    Returns:
        list: A list of tuples, where each tuple represents a permutation of the input list.
    """
    return list(itertools.permutations(input_list))

# Example usage
if __name__ == "__main__":
    input_list = [1, 2, 3]
    permutations = generate_permutations(input_list)
    print("Permutations:")
    for i, permutation in enumerate(permutations):
        print(f"Permutation {i+1}: {permutation}")
```
