## Exercise
Perform basic Boolean operations (AND, OR, NOT) on given Boolean variables.

## Reference Solution
```python
def boolean_and(a, b):
    """
    Performs a logical AND operation on two Boolean variables.
    
    Args:
    a (bool): The first Boolean variable.
    b (bool): The second Boolean variable.
    
    Returns:
    bool: The result of the AND operation.
    """
    return a and b

def boolean_or(a, b):
    """
    Performs a logical OR operation on two Boolean variables.
    
    Args:
    a (bool): The first Boolean variable.
    b (bool): The second Boolean variable.
    
    Returns:
    bool: The result of the OR operation.
    """
    return a or b

def boolean_not(a):
    """
    Performs a logical NOT operation on a Boolean variable.
    
    Args:
    a (bool): The Boolean variable.
    
    Returns:
    bool: The result of the NOT operation.
    """
    return not a

# Example usage:
if __name__ == "__main__":
    a = True
    b = False

    print("Boolean AND:", boolean_and(a, b))
    print("Boolean OR:", boolean_or(a, b))
    print("Boolean NOT (a):", boolean_not(a))
    print("Boolean NOT (b):", boolean_not(b))
```
