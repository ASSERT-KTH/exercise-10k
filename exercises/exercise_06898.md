## Exercise
Create a program that takes in a user's input and determines the data type of the input. The program should be able to differentiate between integers, floats, and strings. If the input is an integer or a float, the program should print out the value and its corresponding data type. If the input is a string, the program should print out the string and its length.

## Reference Solution
```python
def determine_data_type(input_value):
    """
    Determine the data type of the input value.
    
    Parameters:
    input_value (str): The input value from the user.
    
    Returns:
    None
    """
    try:
        # Attempt to convert the input to an integer
        int(input_value)
        print(f"The value {input_value} is of type integer.")
    except ValueError:
        try:
            # If not an integer, attempt to convert the input to a float
            float(input_value)
            print(f"The value {input_value} is of type float.")
        except ValueError:
            # If not a float, it's a string
            print(f"The string '{input_value}' has a length of {len(input_value)}.")

def main():
    # Get the user's input
    input_value = input("Please enter a value: ")
    
    # Determine the data type of the input value
    determine_data_type(input_value)

if __name__ == "__main__":
    main()
```
