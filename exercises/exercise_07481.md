## Exercise
Write a program that converts a given integer into a float

## Reference Solution
```python
def convert_int_to_float(n):
    """
    Converts an integer into a float.

    Args:
        n (int): The integer to be converted.

    Returns:
        float: The converted float value.
    """
    return float(n)


def main():
    # Example usage
    num = int(input("Enter an integer: "))
    float_num = convert_int_to_float(num)
    print(f"The float equivalent of {num} is {float_num}")


if __name__ == "__main__":
    main()
```
