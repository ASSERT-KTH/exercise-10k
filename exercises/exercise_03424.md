## Exercise
Write a function that takes a list of strings as input and returns the longest common prefix

## Reference Solution
```python
def longest_common_prefix(strings):
    """
    Returns the longest common prefix of a list of strings.

    Args:
        strings (list): A list of strings.

    Returns:
        str: The longest common prefix.
    """
    if not strings:
        return ""

    prefix = min(strings, key=len)

    for i, char in enumerate(prefix):
        for string in strings:
            if string[i] != char:
                return prefix[:i]

    return prefix

# Example usage
strings = ["hello", "hello world", "hello python"]
print(longest_common_prefix(strings))  # Output: "hello"

strings = ["apple", "app", "application"]
print(longest_common_prefix(strings))  # Output: "app"

strings = ["test", "example", "sample"]
print(longest_common_prefix(strings))  # Output: ""
```
