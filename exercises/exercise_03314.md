## Exercise
Write a program that takes two arrays as input, and returns a new array containing the elements of the two arrays, but paired together.

## Reference Solution
```python
def pair_elements(arr1, arr2):
    """
    This function takes two arrays as input and returns a new array containing 
    the elements of the two arrays, but paired together.

    Args:
        arr1 (list): The first input array.
        arr2 (list): The second input array.

    Returns:
        list: A new array containing the paired elements of the two input arrays.
    """
    # Check if the input arrays are of the same length
    if len(arr1) != len(arr2):
        raise ValueError("Input arrays must be of the same length")

    # Use zip to pair together elements from the two arrays
    paired_array = list(zip(arr1, arr2))

    return paired_array


# Example usage:
arr1 = [1, 2, 3]
arr2 = ['a', 'b', 'c']

paired_arr = pair_elements(arr1, arr2)
print(paired_arr)  # Output: [(1, 'a'), (2, 'b'), (3, 'c')]
```
