## Exercise
Write a program that takes an array of integers as input, reverses the order of its elements, and then sorts the reversed array in ascending order. The program should output the sorted reversed array.

## Reference Solution
```python
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    int n;
    cin >> n;

    vector<int> v(n);
    for (int i = 0; i < n; i++) {
        cin >> v[i];
    }

    reverse(v.begin(), v.end());
    sort(v.begin(), v.end());

    for (int i = 0; i < n; i++) {
        cout << v[i] << " ";
    }

    return 0;
}
```
