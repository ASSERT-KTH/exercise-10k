## Exercise
Write a program to find the first duplicate in a given list.

## Reference Solution
```python
def find_first_duplicate(lst):
    """
    Find the first duplicate in a given list.

    Args:
        lst (list): The input list.

    Returns:
        The first duplicate element, or None if no duplicates are found.
    """
    seen = set()
    for element in lst:
        if element in seen:
            return element
        seen.add(element)
    return None

# Example usage:
def main():
    lst = [2, 1, 3, 5, 3, 2]
    duplicate = find_first_duplicate(lst)
    if duplicate is not None:
        print(f"The first duplicate is: {duplicate}")
    else:
        print("No duplicates found.")

if __name__ == "__main__":
    main()
```
