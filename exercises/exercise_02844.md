## Exercise
Write a function to remove a key-value pair from a dictionary.

## Reference Solution
```python
def remove_key_value_pair(dictionary, key):
    """
    Removes a key-value pair from a dictionary.

    Args:
        dictionary (dict): The dictionary from which to remove the key-value pair.
        key: The key of the key-value pair to remove.

    Returns:
        dict: The updated dictionary with the key-value pair removed.
    """
    if key in dictionary:
        del dictionary[key]
    return dictionary

# Example usage
if __name__ == "__main__":
    my_dict = {"name": "John", "age": 30, "city": "New York"}
    print("Original Dictionary:", my_dict)
    updated_dict = remove_key_value_pair(my_dict, "age")
    print("Updated Dictionary:", updated_dict)
```
