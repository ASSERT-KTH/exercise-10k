## Exercise
Write a function that takes a dictionary as input and returns a list of all the values

## Reference Solution
```python
def get_dict_values(input_dict):
    """
    Returns a list of all values in the input dictionary.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        list: A list of dictionary values.
    """
    return list(input_dict.values())

# Example usage
if __name__ == "__main__":
    example_dict = {"a": 1, "b": 2, "c": 3}
    print(get_dict_values(example_dict))  # Output: [1, 2, 3]
```
