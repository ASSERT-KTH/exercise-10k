## Exercise
Write a program that takes a string as input, reverses it, and checks if the reversed string is a palindrome. The program should also handle cases where the input string contains punctuation, digits, or special characters. The program should output the reversed string and a boolean value indicating whether the reversed string is a palindrome.

## Reference Solution
```python
def reverse_and_check_palindrome(input_string):
    # Remove all non-alphanumeric characters from the input string and convert to lowercase
    cleaned_string = ''.join(e for e in input_string if e.isalnum()).lower()

    # Reverse the cleaned string
    reversed_string = cleaned_string[::-1]

    # Check if the reversed string is a palindrome
    is_palindrome = cleaned_string == reversed_string

    # Return the reversed string and the palindrome check result
    return reversed_string, is_palindrome


def main():
    # Read input from the user
    input_string = input("Enter a string: ")

    # Call the function and print the result
    reversed_string, is_palindrome = reverse_and_check_palindrome(input_string)
    print(f"Reversed string: {reversed_string}")
    print(f"Is palindrome: {is_palindrome}")


if __name__ == "__main__":
    main()
```
