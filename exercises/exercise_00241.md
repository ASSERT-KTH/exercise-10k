## Exercise
Write a program that takes an array and an index as input, and returns the element at that index. Handle out-of-bounds errors.

## Reference Solution
```python
def get_element(array, index):
    """
    Returns the element at the specified index in the array.

    Args:
        array (list): The input array.
        index (int): The index of the element to be retrieved.

    Returns:
        The element at the specified index if it exists, otherwise raises an IndexError.

    Raises:
        IndexError: If the index is out of bounds.
        TypeError: If the input array is not a list or the index is not an integer.
    """

    # Check if the input array is a list and the index is an integer
    if not isinstance(array, list) or not isinstance(index, int):
        raise TypeError("Input array must be a list and index must be an integer.")

    # Check if the index is within the bounds of the array
    if index < 0 or index >= len(array):
        raise IndexError("Index out of bounds.")

    # Return the element at the specified index
    return array[index]

# Example usage:
array = [1, 2, 3, 4, 5]
index = 2
try:
    element = get_element(array, index)
    print(f"Element at index {index}: {element}")
except (IndexError, TypeError) as e:
    print(f"Error: {e}")
```
