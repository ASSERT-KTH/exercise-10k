## Exercise
Write a program that simulates a coin flip and prints out the result.

## Reference Solution
```python
import random

def coin_flip():
    """Simulates a coin flip and returns the result."""
    return "Heads" if random.randint(0, 1) == 0 else "Tails"

def main():
    print("Coin Flip Simulator")
    print("--------------------")
    flip_result = coin_flip()
    print(f"The coin landed on: {flip_result}")

if __name__ == "__main__":
    main()
```
