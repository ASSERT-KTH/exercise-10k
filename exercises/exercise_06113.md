## Exercise
Write a program that takes an array of integers and a mapping function as input, and returns a new array containing the results of applying the mapping function to each element in the original array.

## Reference Solution
```python
function map(array, fn) {
  const res = [];
  for (let i = 0; i < array.length; i++) {
    res[res.length] = fn(array[i]);
  }
  return res;
}
```
