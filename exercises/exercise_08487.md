## Exercise
Create a program that checks the data type of a given variable and prints out the type. The variable can be of any data type such as integer, float, string, boolean, etc.

## Reference Solution
```python
def check_data_type(var):
    """Checks the data type of a given variable and prints it"""
    var_type = type(var)
    print(f"The data type of the variable is: {var_type.__name__}")

# Test the function with different data types
check_data_type(10)  # integer
check_data_type(3.14)  # float
check_data_type("Hello, World!")  # string
check_data_type(True)  # boolean
check_data_type([1, 2, 3])  # list
check_data_type((1, 2, 3))  # tuple
check_data_type({"name": "John", "age": 30})  # dictionary
check_data_type(set([1, 2, 3]))  # set
check_data_type(None)  # NoneType
```
