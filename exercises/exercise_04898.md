## Exercise
Return a view object that displays a list of all values in a dictionary.

## Reference Solution
```python
def get_dict_values(dictionary):
    """
    Returns a view object that displays a list of all values in a dictionary.

    Args:
        dictionary (dict): The input dictionary.

    Returns:
        dict_values: A view object that displays a list of all values in the dictionary.
    """
    return dictionary.values()

# Example usage:
my_dict = {'a': 1, 'b': 2, 'c': 3}
dict_values = get_dict_values(my_dict)
print(dict_values)  # Output: dict_values([1, 2, 3])

# To print the values as a list, you can use the list() function:
print(list(dict_values))  # Output: [1, 2, 3]
```
