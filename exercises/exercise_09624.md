## Exercise
Create a function that takes a name and age as arguments, with age defaulting to 25 if not provided.

## Reference Solution
```python
# Define a function named 'greet' with two parameters: 'name' and 'age'. 
# The 'age' parameter has a default value of 25.
def greet(name, age=25):
    """
    Prints out a greeting message with the provided name and age.

    Args:
        name (str): The person's name.
        age (int, optional): The person's age. Defaults to 25.
    """

    # Print out a greeting message that includes the name and age.
    print(f"Hello, my name is {name} and I am {age} years old.")

# Example usage:
greet("John")  # Outputs: Hello, my name is John and I am 25 years old.
greet("Alice", 30)  # Outputs: Hello, my name is Alice and I am 30 years old.
```
