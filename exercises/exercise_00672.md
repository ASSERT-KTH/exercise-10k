## Exercise
Given a dictionary {'name': 'Jackson', 'age': 25, 'city': 'Canberra'}, write a program to clear all key-value pairs from the dictionary.

## Reference Solution
```python
def clear_dictionary(dictionary):
    """
    Clears all key-value pairs from a dictionary.

    Args:
        dictionary (dict): The dictionary to be cleared.
    """
    dictionary.clear()

# Example usage
person = {'name': 'Jackson', 'age': 25, 'city': 'Canberra'}
print("Before clearing:")
print(person)

clear_dictionary(person)
print("After clearing:")
print(person)
```
