## Exercise
Write a program that takes a string as input and generates all possible anagrams of the string. Print the anagrams to the console.

## Reference Solution
```python
import itertools

def generate_anagrams(input_string):
    """
    Generates all possible anagrams of the input string.

    Args:
        input_string (str): The input string to generate anagrams for.

    Returns:
        list: A list of all possible anagrams.
    """
    # Convert the input string to a list of characters
    chars = list(input_string)

    # Use itertools.permutations to generate all possible anagrams
    anagrams = set("".join(p) for p in itertools.permutations(chars))

    return anagrams

def main():
    # Get the input string from the user
    input_string = input("Enter a string: ")

    # Generate the anagrams
    anagrams = generate_anagrams(input_string)

    # Print the anagrams
    print("Anagrams:")
    for anagram in anagrams:
        print(anagram)

if __name__ == "__main__":
    main()
```
