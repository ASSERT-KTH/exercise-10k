## Exercise
Explain the concept of variable scope and provide examples of how it affects variable accessibility.

## Reference Solution
```python
x = 10  # global variable

def my_function():
    print(x)  # prints 10

my_function()
print(x)  # prints 10
```

```python
def my_function():
    y = 20  # local variable
    print(y)  # prints 20

my_function()
print(y)  # raises NameError: name 'y' is not defined
```

```python
for i in range(5):
    z = i  # block variable
    print(z)  # prints 0, 1, 2, 3, 4

print(z)  # raises NameError: name 'z' is not defined
```

```python
def outer_function():
    a = 30  # function variable

    def inner_function():
        print(a)  # prints 30

    inner_function()
    print(a)  # prints 30

outer_function()
print(a)  # raises NameError: name 'a' is not defined
```
