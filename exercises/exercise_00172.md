## Exercise
Write a program to demonstrate static typing, where the data type of a variable is known at compile time.

## Reference Solution
```python
# Define a function with static typing
def greeting(name: str) -> str:
    """Return a greeting message."""
    return f"Hello, {name}!"

# Define a function with static typing
def add_numbers(a: int, b: int) -> int:
    """Return the sum of two numbers."""
    return a + b

# Test the functions
print(greeting("John"))  # Output: Hello, John!
print(add_numbers(5, 7))  # Output: 12

# Try to pass incorrect types to the functions
try:
    print(greeting(123))
except TypeError:
    print("Error: name must be a string")

try:
    print(add_numbers("five", 7))
except TypeError:
    print("Error: a and b must be integers")
```
