## Exercise
Write a program that takes a name and email address as input and generates a personalized email.

## Reference Solution
```python
def generate_email(name, email_address, message="Hello, this is a test email."):
    """
    Generates a personalized email.

    Args:
        name (str): The recipient's name.
        email_address (str): The recipient's email address.
        message (str): The email message. Defaults to "Hello, this is a test email.".

    Returns:
        str: The generated email.
    """
    email = f"Subject: Personalized Email\n\n"
    email += f"Dear {name},\n\n"
    email += message + "\n\n"
    email += f"Best regards,\nYour Name\n"
    email += f"Email: {email_address}"
    return email

def main():
    name = input("Enter your name: ")
    email_address = input("Enter your email address: ")
    email = generate_email(name, email_address)
    print("Generated Email:")
    print(email)

if __name__ == "__main__":
    main()
```
