## Exercise
Create a function that applies a transformation to each element of a list.

## Reference Solution
```python
def transform_list(input_list, transformation):
    """
    Applies a transformation to each element of a list.

    Args:
        input_list (list): The input list.
        transformation (function): A function that takes one argument and returns a transformed value.

    Returns:
        list: The list with the transformation applied to each element.
    """
    return [transformation(element) for element in input_list]

# Example usage:
def square(x):
    """Returns the square of a number."""
    return x ** 2

numbers = [1, 2, 3, 4, 5]
transformed_numbers = transform_list(numbers, square)
print(transformed_numbers)  # Output: [1, 4, 9, 16, 25]
```
