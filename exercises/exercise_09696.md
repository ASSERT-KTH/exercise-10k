## Exercise
Create a program that checks if a person is eligible to vote based on their age. The program should ask for the person's age and print 'You are eligible to vote' if they are 18 or older, and 'You are not eligible to vote' otherwise.

## Reference Solution
```python
# add your solution here
age = int(input("Enter your age: "))

if age >= 18:
    print("You are eligible to vote")
else:
    print("You are not eligible to vote")
```
