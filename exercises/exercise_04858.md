## Exercise
Write a program that takes a sorted list and an element as input, and returns the index of the element if it exists in the list using a binary search algorithm.

## Reference Solution
```python
def binary_search(sorted_list, target):
    """
    Searches for an element in a sorted list using a binary search algorithm.

    Args:
        sorted_list (list): A sorted list of elements.
        target (int): The element to be searched.

    Returns:
        int: The index of the element if found, -1 otherwise.
    """
    low = 0
    high = len(sorted_list) - 1

    while low <= high:
        mid = (low + high) // 2
        guess = sorted_list[mid]

        if guess == target:
            return mid
        elif guess > target:
            high = mid - 1
        else:
            low = mid + 1

    return -1

# Example usage:
my_list = [1, 3, 5, 7, 9]
print(binary_search(my_list, 3))  # Output: 1
print(binary_search(my_list, 2))  # Output: -1
```
