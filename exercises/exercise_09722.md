## Exercise
Write a program that takes two decimal numbers as input and returns their sum.

## Reference Solution
```python
def add_numbers(num1, num2):
    """
    This function takes two decimal numbers as input and returns their sum.

    Args:
        num1 (float): The first decimal number.
        num2 (float): The second decimal number.

    Returns:
        float: The sum of num1 and num2.
    """
    return num1 + num2

# Example usage:
num1 = float(input("Enter the first decimal number: "))
num2 = float(input("Enter the second decimal number: "))

result = add_numbers(num1, num2)
print(f"The sum of {num1} and {num2} is {result}.")
```
