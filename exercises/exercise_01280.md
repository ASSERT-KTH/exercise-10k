## Exercise
Write a program to apply a transformation to each element of a list and reduce the result to a single value.

## Reference Solution
```python
import functools

# Define a transformation function
def transform(x):
    """Return the square of x"""
    return x ** 2

# Define a reduction function
def reduce_list(lst):
    """Return the sum of all elements in the list"""
    return functools.reduce(lambda x, y: x + y, lst)

# Apply transformation and reduce
def transform_and_reduce(lst):
    """Apply transformation to each element and reduce the result"""
    transformed_list = list(map(transform, lst))
    return reduce_list(transformed_list)

# Example usage
numbers = [1, 2, 3, 4, 5]
result = transform_and_reduce(numbers)
print("Result:", result)
```
