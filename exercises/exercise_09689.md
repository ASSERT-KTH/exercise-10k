## Exercise
Create a program that finds the index of a given element in a list. If the element is not found, return -1. The program should handle lists of varying sizes and elements of different data types.

## Reference Solution
```python
def find_index(lst, elem):
    """
    Find the index of a given element in a list.

    Args:
        lst (list): The list to search in.
        elem: The element to search for.

    Returns:
        int: The index of the element if found, -1 otherwise.
    """
    try:
        # Attempt to find the index of the element
        index = lst.index(elem)
        return index
    except ValueError:
        # If the element is not found, return -1
        return -1

# Example usage
my_list = [1, 2, 3, 4, 5]
print(find_index(my_list, 3))  # Output: 2
print(find_index(my_list, 6))  # Output: -1
```
