## Exercise
Write a function to return a list of dictionary's values

## Reference Solution
```python
def get_dict_values(dictionary):
    """
    Returns a list of dictionary's values.

    Args:
        dictionary (dict): The dictionary to extract values from.

    Returns:
        list: A list of dictionary's values.
    """
    return list(dictionary.values())

# Example usage:
example_dict = {"a": 1, "b": 2, "c": 3}
print(get_dict_values(example_dict))  # Output: [1, 2, 3]
```
