## Exercise
Create a function that takes an array as input and returns a string representation of the array.

## Reference Solution
```python
// Given an array, returns a string representation of the array
function arrayToString(array) {
  return JSON.stringify(array);
}

// Example usage:
const array = [1, 2, 3, 4, 5];
const str = arrayToString(array);
console.log(str); // Output: [1,2,3,4,5]
```
