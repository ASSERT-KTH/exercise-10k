## Exercise
Write a program that takes an array of integers as input and returns the maximum value in the array. The program should use a basic algorithm to iterate through the array and find the maximum value.

## Reference Solution
```python
#include <stdio.h>

int mx(int* arr) {
    int max = arr[0];
    for (int i = 1; i < 4; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    return max;
}

int main() {
    int arr[4] = {12, 3, 4, 15};
    printf("%d\n", mx(arr));
    return 0;
}
```
