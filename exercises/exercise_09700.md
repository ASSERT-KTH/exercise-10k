## Exercise
Declare and initialize variables of different data types to store a student's information, including name, age, and grade. The program should then print out the stored information.

## Reference Solution
```python
def main():
    # Declare and initialize variables of different data types
    name = "John Doe"  # string
    age = 20  # integer
    grade = 3.8  # float

    # Print out the stored information
    print(f"Name: {name}")
    print(f"Age: {age}")
    print(f"Grade: {grade}")

if __name__ == "__main__":
    main()
```
