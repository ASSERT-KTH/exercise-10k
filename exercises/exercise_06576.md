## Exercise
Write a function that splits a dictionary into two separate dictionaries

## Reference Solution
```python
def split_dictionary(input_dict, keys):
    """
    Splits a dictionary into two separate dictionaries.

    Args:
        input_dict (dict): The input dictionary to be split.
        keys (list): A list of keys to be included in the first dictionary.

    Returns:
        tuple: A tuple containing two dictionaries.
    """
    dict1 = {key: input_dict[key] for key in keys if key in input_dict}
    dict2 = {key: value for key, value in input_dict.items() if key not in keys}
    return dict1, dict2

# Example usage
input_dict = {
    "apple": 1,
    "banana": 2,
    "cherry": 3,
    "date": 4,
    "elderberry": 5
}
keys = ["apple", "cherry", "elderberry"]

dict1, dict2 = split_dictionary(input_dict, keys)
print("Dictionary 1:")
print(dict1)
print("Dictionary 2:")
print(dict2)
```
