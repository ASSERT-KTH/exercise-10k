## Exercise
Write a program that takes a string input from the user, converts it to an integer, and then prints the square of the converted integer.

## Reference Solution
```python
def sq():
    # Get user input
    x = input()

    # Try to convert the input string to an integer
    try:
        x = int(x)
    except ValueError:
        print("Invalid input. Please enter a valid integer.")
        return

    # Calculate the square of x
    y = x ** 2

    # Print the result
    print(y)

# Call the function
sq()
```
