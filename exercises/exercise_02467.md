## Exercise
Create a function that counts the number of vowels in a given string.

## Reference Solution
```python
def count_vowels(input_string: str) -> int:
    """
    Counts the number of vowels in a given string.

    Args:
        input_string (str): The input string to count vowels from.

    Returns:
        int: The number of vowels in the input string.
    """
    vowels = 'aeiouAEIOU'
    return sum(1 for char in input_string if char in vowels)

# Example usage
def main():
    input_str = "Hello World"
    vowel_count = count_vowels(input_str)
    print(f"The number of vowels in '{input_str}' is: {vowel_count}")

if __name__ == "__main__":
    main()
```
