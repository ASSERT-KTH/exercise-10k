## Exercise
Write a program that takes a string input from the user, reverses the string, and checks if the original string is a palindrome. A palindrome is a word, phrase, number, or other sequence of characters that reads the same forward and backward (ignoring spaces, punctuation, and capitalization).

## Reference Solution
```python
def reverse_string(input_string):
    """Reverses the input string."""
    return input_string[::-1]

def is_palindrome(input_string):
    """Checks if the input string is a palindrome."""
    # Remove spaces and punctuation, and convert to lowercase
    cleaned_string = ''.join(e for e in input_string if e.isalnum()).lower()
    
    # Compare the cleaned string with its reverse
    return cleaned_string == reverse_string(cleaned_string)

def main():
    # Get user input
    user_input = input("Please enter a string: ")
    
    # Reverse the string
    reversed_string = reverse_string(user_input)
    print("Reversed string:", reversed_string)
    
    # Check if the string is a palindrome
    if is_palindrome(user_input):
        print("The string is a palindrome.")
    else:
        print("The string is not a palindrome.")

if __name__ == "__main__":
    main()
```
