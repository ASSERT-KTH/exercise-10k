## Exercise
Write a program that takes a string input from the user and checks if it is a palindrome. If the string is a palindrome, the program should print 'Valid palindrome' along with the reversed string. If not, it should print 'Not a palindrome' and the reversed string. Assume the input string will only contain alphanumeric characters and spaces.

## Reference Solution
```python
def check_palindrome(input_string):
    """
    Checks if the input string is a palindrome and prints the result along with the reversed string.

    Args:
        input_string (str): The input string to be checked.
    """

    # Remove spaces from the input string and convert it to lowercase for case-insensitive comparison
    cleaned_string = input_string.replace(" ", "").lower()
    
    # Reverse the cleaned string
    reversed_string = cleaned_string[::-1]
    
    # Check if the cleaned string is equal to its reverse
    if cleaned_string == reversed_string:
        print('Valid palindrome')
    else:
        print('Not a palindrome')
    
    # Print the reversed string
    print('Reversed string:', reversed_string)


def main():
    # Take a string input from the user
    input_string = input("Enter a string: ")
    
    # Call the function to check the palindrome
    check_palindrome(input_string)


if __name__ == "__main__":
    main()
```
