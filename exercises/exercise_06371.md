## Exercise
Write a program that takes an array and a predicate function as input and returns the first element that satisfies the predicate

## Reference Solution
```python
def find_first_element(array, predicate):
    """
    Finds the first element in the array that satisfies the predicate function.

    Args:
        array (list): The input array.
        predicate (function): A function that takes one argument and returns a boolean value.

    Returns:
        The first element that satisfies the predicate, or None if no such element exists.
    """
    for element in array:
        if predicate(element):
            return element
    return None

# Example usage:
# Define a predicate function
def is_even(x):
    return x % 2 == 0

# Create an array
array = [1, 3, 5, 7, 9, 10, 12]

# Find the first even number in the array
first_even = find_first_element(array, is_even)

print(first_even)  # Output: 10
```
