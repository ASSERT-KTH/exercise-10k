## Exercise
Create a program to retrieve all values from a given dictionary.

## Reference Solution
```python
def get_dict_values(dictionary):
    """
    Retrieves all values from a given dictionary.

    Args:
        dictionary (dict): The dictionary to retrieve values from.

    Returns:
        list: A list of values from the dictionary.
    """
    return list(dictionary.values())

def main():
    # Example usage
    example_dict = {
        "name": "John",
        "age": 30,
        "city": "New York"
    }

    values = get_dict_values(example_dict)
    print("Dictionary Values:")
    for value in values:
        print(value)

if __name__ == "__main__":
    main()
```
