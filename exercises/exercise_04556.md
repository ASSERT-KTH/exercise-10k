## Exercise
Write a program that alternates the case of characters in a given string.

## Reference Solution
```python
def alternate_case(s):
    """
    Alternates the case of characters in a given string.

    Args:
        s (str): The input string.

    Returns:
        str: The string with alternating case.
    """
    result = ""
    upper = True  # Start with uppercase

    for char in s:
        if char.isalpha():  # Check if character is a letter
            if upper:
                result += char.upper()
            else:
                result += char.lower()
            upper = not upper  # Toggle case for next character
        else:
            result += char  # Non-letter characters remain unchanged

    return result


# Example usage
if __name__ == "__main__":
    input_str = "Hello, World!"
    print("Original string:", input_str)
    print("Alternating case:", alternate_case(input_str))
```
