## Exercise
Create a variable inside and outside a function, then print their values to understand the concept of variable scope.

## Reference Solution
```python
# Variable defined outside the function (global scope)
global_variable = 10

def my_function():
    # Variable defined inside the function (local scope)
    local_variable = 20
    
    # Accessing the global variable inside the function
    print("Global variable inside the function:", global_variable)
    
    # Accessing the local variable inside the function
    print("Local variable inside the function:", local_variable)

# Accessing the global variable outside the function
print("Global variable outside the function:", global_variable)

# Trying to access the local variable outside the function will result in an error
try:
    print("Local variable outside the function:", local_variable)
except NameError:
    print("Error: Local variable is not defined outside the function.")

# Calling the function
my_function()
```
