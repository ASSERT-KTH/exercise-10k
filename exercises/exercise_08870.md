## Exercise
Write a function to calculate the sum of rotated numbers of a given number

## Reference Solution
```python
def calculate_rotated_sum(n):
    """
    Calculate the sum of rotated numbers of a given number.
    
    Args:
        n (int): The input number.
    
    Returns:
        int: The sum of rotated numbers.
    """
    # Convert the number to a string to easily rotate the digits
    num_str = str(n)
    
    # Initialize the sum
    total_sum = 0
    
    # Rotate the number and calculate the sum
    for i in range(len(num_str)):
        # Rotate the number by i positions
        rotated_num = int(num_str[i:] + num_str[:i])
        
        # Add the rotated number to the sum
        total_sum += rotated_num
    
    return total_sum

# Example usage:
def main():
    num = 1234
    rotated_sum = calculate_rotated_sum(num)
    print(f"The sum of rotated numbers of {num} is: {rotated_sum}")

if __name__ == "__main__":
    main()
```
