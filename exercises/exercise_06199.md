## Exercise
Write a function that takes a string as input and returns True if the string contains only lowercase characters, False otherwise.

def check_lowercase(input_string):
    """
    This function checks if a given string contains only lowercase characters.

    Args:
        input_string (str): The input string to be checked.

    Returns:
        bool: True if the string contains only lowercase characters, False otherwise.
    """

    # First, we check if the input is indeed a string. If it's not, we raise a TypeError.
    if not isinstance(input_string, str):
        raise TypeError("Input must be a string.")

    # We use the built-in all() function in Python, which returns True if all elements in an iterable are true.
    # We also use a generator expression to iterate over each character in the string.
    # The islower() method returns True if all cased characters in the string are lowercase and there is at least one cased character, otherwise False.
    return all(
        # For each character in the string, we check if it's lowercase using the islower() method.
        char.islower() 
        # We iterate over each character in the input string.
        for char in input_string
    )
