## Exercise
Write a function that uses the compiler to compile a string as Python code into bytecode.

## Reference Solution
```python
def compile_string(code_string):
    """
    Compile a string as Python code into bytecode.

    Args:
        code_string (str): A string containing Python code.

    Returns:
        code: The compiled bytecode.
    """

    # First, we check if the input is a string. If not, we raise a TypeError.
    if not isinstance(code_string, str):
        raise TypeError("Input must be a string.")

    # We use the compile function to compile the string into bytecode.
    # The compile function takes three arguments: the string to compile, the filename, and the mode.
    # We use '<string>' as the filename and 'exec' as the mode, which means the string can contain any type of Python code.
    compiled_code = compile(code_string, '<string>', 'exec')

    # Finally, we return the compiled bytecode.
    return compiled_code

# Example usage:
code_string = "print('Hello, World!')"
compiled_code = compile_string(code_string)
exec(compiled_code)  # Prints: Hello, World!
```
