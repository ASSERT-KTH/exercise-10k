## Exercise
Write a function that takes a dictionary and a key as input and removes and returns the corresponding value

## Reference Solution
```python
def remove_and_return(dictionary, key):
    """
    Removes and returns the value corresponding to the given key from the dictionary.

    Args:
        dictionary (dict): The dictionary from which to remove the key-value pair.
        key: The key of the key-value pair to remove.

    Returns:
        The removed value if the key exists in the dictionary, otherwise None.
    """
    if key in dictionary:
        value = dictionary[key]
        del dictionary[key]
        return value
    else:
        return None

# Example usage:
if __name__ == "__main__":
    example_dict = {"apple": 1, "banana": 2, "cherry": 3}
    print("Original dictionary:", example_dict)

    removed_value = remove_and_return(example_dict, "banana")
    print("Removed value:", removed_value)
    print("Updated dictionary:", example_dict)
```
