## Exercise
Use the reduce function to calculate the sum of all elements in a list.

## Reference Solution
```python
from functools import reduce

def calculate_sum(lst):
    """
    Calculate the sum of all elements in a list using reduce function.
    
    Args:
        lst (list): A list of numbers.
    
    Returns:
        int: The sum of all elements in the list.
    """
    return reduce(lambda x, y: x + y, lst)

# Example usage:
numbers = [1, 2, 3, 4, 5]
result = calculate_sum(numbers)
print("Sum of list elements:", result)
```
