## Exercise
Write a function to check if a string consists only of digits.

## Reference Solution
```python
def check_digits_only(input_string: str) -> bool:
    """
    Checks if a string consists only of digits.

    Args:
        input_string (str): The input string to be checked.

    Returns:
        bool: True if the string consists only of digits, False otherwise.
    """
    return input_string.isdigit()

# Example usage:
if __name__ == "__main__":
    print(check_digits_only("12345"))  # True
    print(check_digits_only("123abc"))  # False
    print(check_digits_only("abcdef"))  # False
    print(check_digits_only(""))  # False
```
