## Exercise
Create a function that takes in a list and returns all permutations of the list.

## Reference Solution
```python
import itertools

def get_permutations(input_list):
    """
    Returns all permutations of the input list.

    Args:
        input_list (list): The list for which to generate permutations.

    Returns:
        list: A list of tuples, where each tuple is a permutation of the input list.
    """
    return list(itertools.permutations(input_list))

def main():
    # Example usage:
    input_list = [1, 2, 3]
    permutations = get_permutations(input_list)
    for i, permutation in enumerate(permutations):
        print(f"Permutation {i+1}: {permutation}")

if __name__ == "__main__":
    main()
```
