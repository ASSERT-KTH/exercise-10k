## Exercise
Write a function to check if a dictionary is empty

## Reference Solution
```python
def is_dict_empty(dictionary):
    """
    Checks if a dictionary is empty.

    Args:
        dictionary (dict): The dictionary to check.

    Returns:
        bool: True if the dictionary is empty, False otherwise.
    """
    return len(dictionary) == 0

# Example usage:
my_dict = {}
print(is_dict_empty(my_dict))  # Output: True

my_dict = {"key": "value"}
print(is_dict_empty(my_dict))  # Output: False
```
