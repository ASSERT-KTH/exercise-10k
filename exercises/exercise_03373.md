## Exercise
Use a for loop to find the smallest number in a given list of numbers

## Reference Solution
```python
def find_smallest_number(num_list):
    """
    This function finds the smallest number in a given list of numbers.

    Args:
        num_list (list): A list of numbers.

    Returns:
        int: The smallest number in the list.
    """
    smallest_num = num_list[0]  # Initialize smallest_num with the first element of the list
    for num in num_list:
        if num < smallest_num:
            smallest_num = num
    return smallest_num

# Example usage
numbers = [64, 34, 25, 12, 22, 11, 90]
smallest = find_smallest_number(numbers)
print("The smallest number in the list is:", smallest)
```
