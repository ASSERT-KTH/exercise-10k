## Exercise
Write a function to check if a given number is an abundant number.

## Reference Solution
```python
def is_abundant(n):
    """
    Checks if a given number is an abundant number.

    Args:
        n (int): The number to check.

    Returns:
        bool: True if the number is abundant, False otherwise.
    """
    if n <= 0:
        raise ValueError("Input must be a positive integer")

    # Calculate the sum of proper divisors
    sum_of_divisors = sum(i for i in range(1, n) if n % i == 0)

    # Check if the sum of divisors is greater than the number
    return sum_of_divisors > n


# Example usage:
if __name__ == "__main__":
    numbers_to_check = [12, 18, 20, 24]
    for num in numbers_to_check:
        if is_abundant(num):
            print(f"{num} is an abundant number")
        else:
            print(f"{num} is not an abundant number")
```
