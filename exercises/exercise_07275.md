## Exercise
Create a function that takes a string as input and returns True if it contains at least one uppercase letter, one lowercase letter, and one digit. Otherwise, return False.

## Reference Solution
```python
def check_string(s):
    """
    This function checks if a given string contains at least one uppercase letter, one lowercase letter, and one digit.

    Args:
        s (str): The input string to be checked.

    Returns:
        bool: True if the string contains at least one uppercase letter, one lowercase letter, and one digit. Otherwise, False.
    """

    # First, we'll assume that the string does not meet the conditions
    has_uppercase = False
    has_lowercase = False
    has_digit = False

    # Then, we'll iterate over each character in the string
    for char in s:
        # For each character, we'll check if it's an uppercase letter
        if char.isupper():
            # If it is, we'll set has_uppercase to True
            has_uppercase = True
        # We'll also check if it's a lowercase letter
        elif char.islower():
            # If it is, we'll set has_lowercase to True
            has_lowercase = True
        # And we'll check if it's a digit
        elif char.isdigit():
            # If it is, we'll set has_digit to True
            has_digit = True

    # Finally, we'll return True if the string has at least one uppercase letter, one lowercase letter, and one digit
    # Otherwise, we'll return False
    return has_uppercase and has_lowercase and has_digit
```
