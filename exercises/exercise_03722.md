## Exercise
Create a program that reads two integer numbers from the user and prints their sum.

## Reference Solution
```python
def main():
    # Read two integer numbers from the user
    num1 = int(input("Enter the first integer number: "))
    num2 = int(input("Enter the second integer number: "))

    # Calculate and print the sum
    total = num1 + num2
    print("The sum of", num1, "and", num2, "is:", total)

if __name__ == "__main__":
    main()
```
