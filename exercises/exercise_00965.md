## Exercise
Write a function to iterate over the key-value pairs in a dictionary and print each pair.

## Reference Solution
```python
def print_dict_key_value_pairs(dictionary):
    """
    This function iterates over the key-value pairs in a dictionary and prints each pair.

    Args:
        dictionary (dict): The dictionary to be iterated over.

    Returns:
        None
    """

    # First, we check if the input is indeed a dictionary
    if not isinstance(dictionary, dict):
        raise TypeError("Input must be a dictionary")

    # We use the items() method of the dictionary to get a view object that displays a list of a dictionaryâ€™s key-value tuple pairs
    dict_items = dictionary.items()

    # Then, we iterate over each key-value pair in the dictionary
    for key, value in dict_items:
        # For each pair, we print the key and the corresponding value
        print(f"Key: {key}, Value: {value}")

# Example usage:
example_dict = {"name": "John", "age": 30, "city": "New York"}
print_dict_key_value_pairs(example_dict)
```
