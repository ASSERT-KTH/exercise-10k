## Exercise
Write a program that calculates the greatest common divisor (GCD) and least common multiple (LCM) of two numbers.

## Reference Solution
```python
import math

def calculate_gcd_lcm(num1, num2):
    # Calculate GCD using math.gcd function
    gcd = math.gcd(num1, num2)
    
    # Calculate LCM using the formula: lcm(a, b) = |a*b| / gcd(a, b)
    lcm = abs(num1 * num2) // gcd
    
    return gcd, lcm

# Get input from user
num1 = int(input("Enter the first number: "))
num2 = int(input("Enter the second number: "))

# Calculate GCD and LCM
gcd, lcm = calculate_gcd_lcm(num1, num2)

# Print the results
print(f"GCD of {num1} and {num2} is: {gcd}")
print(f"LCM of {num1} and {num2} is: {lcm}")
```
