## Exercise
Write a function to calculate the sum of digits of a given number.

## Reference Solution
```python
def sum_of_digits(n: int) -> int:
    """
    Calculate the sum of digits of a given number.

    Args:
        n (int): The input number.

    Returns:
        int: The sum of digits of the input number.
    """
    return sum(int(digit) for digit in str(abs(n)))
```

```python
def main():
    # Test the function with a positive number
    print(sum_of_digits(12345))  # Output: 15

    # Test the function with a negative number
    print(sum_of_digits(-12345))  # Output: 15

    # Test the function with a single-digit number
    print(sum_of_digits(5))  # Output: 5

if __name__ == "__main__":
```
