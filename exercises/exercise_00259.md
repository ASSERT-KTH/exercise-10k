## Exercise
Write a program that takes multiple lists as input, and returns a new list that contains only the elements that are common to all lists.

## Reference Solution
```python
# Create lists
list1 = [1, 2, 3, 4, 5]
list2 = [4, 5, 6, 7, 8]
list3 = [1, 2, 3, 7, 8, 9]

# Find the intersection of the lists
common_elements = set(list1) & set(list2) & set(list3)

# Print the common elements
print(common_elements)
```
