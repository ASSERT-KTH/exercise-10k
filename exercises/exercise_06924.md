## Exercise
Create a function that takes three numbers as arguments and returns the maximum value

## Reference Solution
```python
def find_max(a: float, b: float, c: float) -> float:
    """
    Returns the maximum value among three numbers.

    Args:
    a (float): The first number.
    b (float): The second number.
    c (float): The third number.

    Returns:
    float: The maximum value among a, b, and c.
    """
    return max(a, b, c)

# Example usage:
if __name__ == "__main__":
    print(find_max(10, 20, 30))  # Output: 30
    print(find_max(-5, 0, 5))    # Output: 5
    print(find_max(100, 50, 75)) # Output: 100
```
