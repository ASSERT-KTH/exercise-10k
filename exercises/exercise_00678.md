## Exercise
Given a dictionary {'name': 'Patrick', 'age': 25, 'city': 'Townsville'}, write a program to use the keys() method to print all the keys in the dictionary.

## Reference Solution
```python
def print_dictionary_keys(dictionary):
    """
    Prints all the keys in a given dictionary using the keys() method.

    Args:
    dictionary (dict): The input dictionary.
    """
    # Use the keys() method to get all the keys in the dictionary
    keys = dictionary.keys()
    
    # Print the keys
    print("Dictionary Keys:")
    for key in keys:
        print(key)


# Example usage
if __name__ == "__main__":
    # Define the dictionary
    person_data = {'name': 'Patrick', 'age': 25, 'city': 'Townsville'}
    
    # Print the dictionary keys
    print_dictionary_keys(person_data)
```
