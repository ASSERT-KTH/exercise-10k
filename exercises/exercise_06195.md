## Exercise
Write a function that takes a string and a substring as input and returns True if the string starts with the substring, False otherwise.

## Reference Solution
```python
def check_start_string(main_string, sub_string):
    """
    This function checks if a given string starts with a specified substring.

    Args:
        main_string (str): The main string to be checked.
        sub_string (str): The substring to be searched at the start of the main string.

    Returns:
        bool: True if the main string starts with the substring, False otherwise.
    """

    # First, we need to check if both inputs are strings. If not, we raise a TypeError.
    if not isinstance(main_string, str) or not isinstance(sub_string, str):
        raise TypeError("Both inputs must be strings.")

    # Now, we use the built-in str.startswith() method in Python, which returns True if the string starts with the specified value, False otherwise.
    # This method is case-sensitive and also works with tuples of strings.
    return main_string.startswith(sub_string)

# Example usage:
print(check_start_string("Hello World", "Hello"))  # Returns: True
print(check_start_string("Hello World", "World"))  # Returns: False
```
