## Exercise
Write a program that takes an array of integers as input, reverses the order of its elements, and returns the reversed array.

## Reference Solution
```python
def reverse_array(arr):
    n = len(arr)
    for i in range(n // 2):
        arr[i], arr[n - i - 1] = arr[n - i - 1], arr[i]
    return arr

arr = [1, 2, 3, 4, 5]
print(reverse_array(arr))  # Output: [5, 4, 3, 2, 1]
```
