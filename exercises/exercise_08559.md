## Exercise
Create a dictionary with 3 key-value pairs and get the keys and values separately.

## Reference Solution
```python
# Create a dictionary with 3 key-value pairs
my_dict = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

# Get the keys
keys = my_dict.keys()
print("Keys:", keys)

# Get the values
values = my_dict.values()
print("Values:", values)

# Alternatively, you can use list() function to convert the dict_keys and dict_values objects to lists
keys_list = list(my_dict.keys())
values_list = list(my_dict.values())
print("Keys as list:", keys_list)
print("Values as list:", values_list)
```
