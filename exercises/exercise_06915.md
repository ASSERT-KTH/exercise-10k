## Exercise
Write a program that takes a string as input, reverses it, and checks if the reversed string is the same as the original string (i.e., it's a palindrome). The program should return the reversed string and a boolean indicating whether the string is a palindrome.

## Reference Solution
```python
def reverse_string(input_str):
    input_str = input_str[::-1]
    return input_str, input_str == input_str[::-1]

input_str = input("Enter a string: ")
reversed_str, is_palindrome = reverse_string(input_str)

print(f"Reversed string: {reversed_str}")
print(f"Is palindrome: {is_palindrome}")
```
