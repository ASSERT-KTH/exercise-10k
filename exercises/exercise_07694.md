## Exercise
Write a function that calculates the maximum of two numbers.

## Reference Solution
```python
def calculate_max(a: float, b: float) -> float:
    """
    Calculates the maximum of two numbers.

    Args:
        a (float): The first number.
        b (float): The second number.

    Returns:
        float: The maximum of the two numbers.
    """
    return max(a, b)

# Example usage
if __name__ == "__main__":
    num1 = 10.0
    num2 = 20.0
    max_value = calculate_max(num1, num2)
    print(f"The maximum of {num1} and {num2} is {max_value}")
```
