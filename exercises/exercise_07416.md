## Exercise
Write a program that takes an array of integers as input and returns the sum of the elements at the even indices and the sum of the elements at the odd indices separately. The program should handle arrays of varying lengths.

## Reference Solution
```python
#include <iostream>

int main() {
    int size;
    std::cout << "Enter the size of the array: ";
    std::cin >> size;

    int* arr = new int[size];

    std::cout << "Enter the elements of the array: ";
    for (int i = 0; i < size; i++) {
        std::cin >> arr[i];
    }

    int evenSum = 0;
    int oddSum = 0;

    for (int i = 0; i < size; i++) {
        if (i % 2 == 0) {
            evenSum += arr[i];
        } else {
            oddSum += arr[i];
        }
    }

    std::cout << "Sum of elements at even indices: " << evenSum << std::endl;
    std::cout << "Sum of elements at odd indices: " << oddSum << std::endl;

    delete[] arr;
    return 0;
}
```
