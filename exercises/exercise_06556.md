## Exercise
Write a function that replaces all occurrences of a specified character with another character.

## Reference Solution
```python
def replace_char(original_string, char_to_replace, replacement_char):
    """
    Replaces all occurrences of a specified character with another character.

    Args:
        original_string (str): The string in which to replace characters.
        char_to_replace (str): The character to be replaced.
        replacement_char (str): The character to replace with.

    Returns:
        str: The modified string with replaced characters.
    """
    return original_string.replace(char_to_replace, replacement_char)

# Example usage:
if __name__ == "__main__":
    original_string = "Hello, World!"
    char_to_replace = "o"
    replacement_char = "0"

    modified_string = replace_char(original_string, char_to_replace, replacement_char)
    print(f"Original String: {original_string}")
    print(f"Modified String: {modified_string}")
```
