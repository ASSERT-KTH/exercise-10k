## Exercise
Write a program to demonstrate the use of a fat arrow function.

## Reference Solution
```python
// Define a traditional function
function traditionalFunction(name) {
    return `Hello, ${name}!`;
}

// Define a fat arrow function
const fatArrowFunction = (name) => {
    return `Hello, ${name}!`;
};

// Define a concise fat arrow function
const conciseFatArrowFunction = name => `Hello, ${name}!`;

// Test the functions
console.log(traditionalFunction("John")); // Output: Hello, John!
console.log(fatArrowFunction("Jane")); // Output: Hello, Jane!
console.log(conciseFatArrowFunction("Bob")); // Output: Hello, Bob!
```
