## Exercise
Write a program that takes a 2D array as input, and returns a new array containing all elements from the subarrays.

## Reference Solution
```python
def flatten_array(arr):
    """
    This function takes a 2D array as input and returns a new array containing all elements from the subarrays.

    Args:
        arr (list): A 2D list of elements.

    Returns:
        list: A new list containing all elements from the subarrays.
    """
    # Check if the input is a list
    if not isinstance(arr, list):
        raise ValueError("Input must be a list")

    # Initialize an empty list to store the result
    result = []

    # Iterate over each subarray in the input array
    for subarray in arr:
        # Check if the subarray is a list
        if not isinstance(subarray, list):
            raise ValueError("Input must be a 2D list")
        
        # Extend the result list with the elements of the subarray
        result.extend(subarray)

    # Return the result list
    return result

# Example usage:
arr = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
print(flatten_array(arr))  # Output: [1, 2, 3, 4, 5, 6, 7, 8, 9]
```
