## Exercise
Write a program that takes a list and two indices as input, and returns a slice of the list from the first index to the second.

## Reference Solution
```python
def list_slice(the_list, index1, index2):
    result = []
    for i in range(index1, index2):
        result.append(the_list[i])
    return result

def main():
    a = [1, 2, 3, 4, 5, 6]
    print(list_slice(a, 1, 4))  # Output: [2, 3, 4]

if __name__ == "__main__":
    main()
```
