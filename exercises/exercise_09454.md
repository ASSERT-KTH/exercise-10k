## Exercise
Create a program that checks if a given letter is a vowel or consonant and prints a corresponding message.

## Reference Solution
```python
def check_letter(letter):
    """
    Checks if a given letter is a vowel or consonant and prints a corresponding message.

    Args:
        letter (str): A single character to check.
    """

    # Convert the letter to lowercase to handle both uppercase and lowercase letters
    letter = letter.lower()

    # Check if the letter is an alphabet letter
    if not letter.isalpha():
        print("The input is not a letter.")
        return

    # Check if the letter is a vowel
    if letter in 'aeiou':
        print(f"The letter '{letter}' is a vowel.")
    else:
        print(f"The letter '{letter}' is a consonant.")

def main():
    # Get the letter from the user
    letter = input("Enter a letter: ")

    # Check the letter
    check_letter(letter)

if __name__ == "__main__":
    main()
```
