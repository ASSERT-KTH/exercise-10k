## Exercise
Create a function that finds the maximum of three numbers.

## Reference Solution
```python
def find_max(a, b, c):
    """
    Find the maximum of three numbers.

    Args:
        a (int or float): The first number.
        b (int or float): The second number.
        c (int or float): The third number.

    Returns:
        int or float: The maximum of the three numbers.
    """
    return max(a, b, c)

# Example usage:
if __name__ == "__main__":
    num1 = 10
    num2 = 20
    num3 = 30
    max_value = find_max(num1, num2, num3)
    print(f"The maximum of {num1}, {num2}, and {num3} is: {max_value}")
```
