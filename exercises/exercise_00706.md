## Exercise
Write a function to return a list of dictionary's key-value pairs as tuples

## Reference Solution
```python
def dict_to_tuples(input_dict):
    """
    Returns a list of dictionary key-value pairs as tuples.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        list: A list of tuples, where each tuple is a key-value pair from the input dictionary.
    """
    return list(input_dict.items())

# Example usage
if __name__ == "__main__":
    example_dict = {"apple": 1, "banana": 2, "cherry": 3}
    print(dict_to_tuples(example_dict))
```
