## Exercise
Write a program that takes an object as input and checks if it's empty or not using conditional statements.

let obj: {
  name: string;
  id: number;
  is_admin: boolean;
};

// Initialization of the object
obj = {
  name: "John Doe",
  id: 123,
  is_admin: true

} else {
