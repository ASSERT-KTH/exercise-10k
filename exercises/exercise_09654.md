## Exercise
Write a program that asks for a sentence and repeats it.

## Reference Solution

def repeat_sentence():
    """
    Asks the user for a sentence and repeats it back to them.
    """
    sentence = input("Please enter a sentence: ")
    print("You entered:", sentence)

def main():
    repeat_sentence()

if __name__ == "__main__":
    main()
