## Exercise
Write a program to calculate the binary representation of a given integer

## Reference Solution
```python
def calculate_binary(n):
    """
    Calculate the binary representation of a given integer.

    Args:
        n (int): The input integer.

    Returns:
        str: The binary representation of the input integer.
    """
    return bin(n)[2:]

def main():
    # Get the input integer from the user
    n = int(input("Enter an integer: "))

    # Calculate the binary representation
    binary = calculate_binary(n)

    # Print the result
    print(f"The binary representation of {n} is: {binary}")

if __name__ == "__main__":
    main()
```
