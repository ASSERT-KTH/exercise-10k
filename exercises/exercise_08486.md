## Exercise
Given an array of integers, write a function to find the maximum contiguous subarray (a subarray with the largest sum). The function should return the sum of the maximum subarray and its starting and ending indices.

## Reference Solution
```python
# Problem: Find the maximum contiguous subarray (a subarray with the largest sum)
# Input: An array of integers
# Output: A tuple containing the sum of the maximum subarray and its starting and ending indices

def max_sub_array_sum(a):
    size = len(a)
    max_so_far = float('-inf')
    max_ending_here = 0
    start = 0
    end = 0
    temp_start = 0

    for i in range(0, size):
        max_ending_here = max_ending_here + a[i]
        if (max_so_far < max_ending_here):
            max_so_far = max_ending_here
            start = temp_start
            end = i

        if max_ending_here < 0:
            max_ending_here = 0
            temp_start = i + 1

    return (max_so_far, start, end)

# Example usage:
a = [1, 2, 3, 4, 5]
result = max_sub_array_sum(a)
print("Maximum contiguous subarray sum:", result[0])
print("Starting index:", result[1])
print("Ending index:", result[2])
```
