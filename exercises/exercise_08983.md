## Exercise
Write a function that takes a string and an integer as input and returns the encrypted string using the Caesar cipher algorithm.

## Reference Solution
```python
def encrypt(text, s):
    result = ""

    for i in range(len(text)):
        char = text[i]

        if char.isupper():
            result += chr((ord(char) + s - 65) % 26 + 65)
        else:
            result += chr((ord(char) + s - 97) % 26 + 97)

    return result

text = "ATTACKATDAWN"
s = 1
print("Text : " + text)
print("Shift : " + str(s))
print("Cipher: " + encrypt(text, s))
```
