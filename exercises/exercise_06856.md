## Exercise
Write a program that takes in a user's input and determines the data type of the input. If the input is an integer, it should print 'Integer'. If the input is a float, it should print 'Float'. If the input is a string, it should print 'String'.

## Reference Solution
```python
def determine_data_type(user_input):
    """
    Determine the data type of the user's input.

    Args:
        user_input (str): The user's input.

    Returns:
        None
    """

    # Try to convert the input to an integer
    try:
        int(user_input)
        print('Integer')
        return
    except ValueError:
        pass

    # Try to convert the input to a float
    try:
        float(user_input)
        print('Float')
        return
    except ValueError:
        pass

    # If the input cannot be converted to an integer or a float, it must be a string
    print('String')


def main():
    # Get the user's input
    user_input = input("Please enter something: ")

    # Determine the data type of the user's input
    determine_data_type(user_input)


if __name__ == "__main__":
    main()
```
