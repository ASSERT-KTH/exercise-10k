## Exercise
Given an array of strings, write a function to sort the elements in descending order based on their lengths.

## Reference Solution
```python
def sort_strings_by_length(arr):
    """
    Sorts an array of strings in descending order based on their lengths.

    Args:
    arr (list): A list of strings.

    Returns:
    list: A list of strings sorted in descending order by length.
    """
    return sorted(arr, key=len, reverse=True)

# Example usage:
strings = ["apple", "banana", "cat", "elephant", "dog"]
sorted_strings = sort_strings_by_length(strings)
print(sorted_strings)
```
