## Exercise
Write a function that filters a dictionary based on a given condition

## Reference Solution
```python
def filter_dict(input_dict, condition):
    """
    Filters a dictionary based on a given condition.

    Args:
        input_dict (dict): The dictionary to be filtered.
        condition (function): A function that takes a key-value pair and returns a boolean.

    Returns:
        dict: A new dictionary containing only the key-value pairs for which the condition is True.
    """
    return {key: value for key, value in input_dict.items() if condition(key, value)}

# Example usage
if __name__ == "__main__":
    # Define a dictionary
    my_dict = {
        "apple": 5,
        "banana": 10,
        "cherry": 7,
        "date": 3
    }

    # Define a condition function
    def condition(key, value):
        return value > 5

    # Filter the dictionary
    filtered_dict = filter_dict(my_dict, condition)

    # Print the filtered dictionary
    print("Original Dictionary:")
    print(my_dict)
    print("Filtered Dictionary:")
    print(filtered_dict)
```
