## Exercise
Create a dictionary with 5 key-value pairs. Modify a specific key-value pair.

## Reference Solution
```python
# Create a dictionary with 5 key-value pairs
my_dict = {
    "name": "John",
    "age": 30,
    "city": "New York",
    "country": "USA",
    "occupation": "Software Engineer"
}

# Print the original dictionary
print("Original Dictionary:")
print(my_dict)

# Modify a specific key-value pair
my_dict["age"] = 31

# Print the modified dictionary
print("\nModified Dictionary:")
print(my_dict)
```
