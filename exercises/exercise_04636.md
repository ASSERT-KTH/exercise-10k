## Exercise
Write a program that takes two lists as input, and returns the union of the two lists.

## Reference Solution
```python
def list_union(list1, list2):
    """
    Returns the union of two lists.

    Args:
        list1 (list): The first list.
        list2 (list): The second list.

    Returns:
        list: The union of list1 and list2.
    """
    return list(set(list1) | set(list2))

# Example usage:
def main():
    list1 = [1, 2, 3, 4]
    list2 = [3, 4, 5, 6]
    print("List 1:", list1)
    print("List 2:", list2)
    print("Union:", list_union(list1, list2))

if __name__ == "__main__":
    main()
```
