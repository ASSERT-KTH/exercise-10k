## Exercise
Write a function to find the maximum contiguous subarray of an array of integers. The function should take an array as input and return the maximum sum of a contiguous subarray. If the array is empty, return 0.

## Reference Solution
```python
def maxSubArray(nums):
    if not nums:
        return 0
    maxi = nums[0]
    res = nums[0]
    for i in range(1, len(nums)):
        res = max(nums[i], res + nums[i])
        maxi = max(maxi, res)
    return maxi
```
