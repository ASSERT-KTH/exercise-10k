## Exercise
Write a function that takes a dictionary as input and returns two lists, one for keys and one for values

## Reference Solution
```python
def separate_dict(input_dict):
    """
    This function takes a dictionary as input and returns two lists,
    one for keys and one for values.

    Args:
        input_dict (dict): The input dictionary.

    Returns:
        tuple: A tuple containing two lists, one for keys and one for values.
    """
    keys_list = list(input_dict.keys())
    values_list = list(input_dict.values())

    return keys_list, values_list

# Example usage
input_dict = {"name": "John", "age": 30, "city": "New York"}
keys, values = separate_dict(input_dict)

print("Keys:", keys)
print("Values:", values)
```
