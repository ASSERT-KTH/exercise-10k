## Exercise
Calculate the mode of an array.

## Reference Solution
```python
def calculate_mode(array):
    """
    Calculate the mode of an array.

    Args:
        array (list): The input array.

    Returns:
        list: A list of mode(s).
    """
    # Create a frequency dictionary
    frequency_dict = {}
    for element in array:
        if element in frequency_dict:
            frequency_dict[element] += 1
        else:
            frequency_dict[element] = 1

    # Find the maximum frequency
    max_frequency = max(frequency_dict.values())

    # Find the mode(s)
    modes = [element for element, frequency in frequency_dict.items() if frequency == max_frequency]

    return modes

# Example usage
array = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]
modes = calculate_mode(array)
print("Mode(s):", modes)
```
