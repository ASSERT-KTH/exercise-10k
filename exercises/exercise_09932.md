## Exercise
Create a program that takes in a user's name and age as input, then outputs a greeting message with the name and age. The twist is that the program should handle invalid inputs, such as non-integer values for age or empty strings for name. The program should continue to prompt the user for input until valid inputs are provided.

## Reference Solution
```python
import re

def get_name():
    while True:
        name = input("Enter your name: ")
        if re.match("^[a-zA-Z ]*$", name) and name.strip() != "":
            return name
        else:
            print("Invalid input. Please enter a valid name.")

def get_age():
    while True:
        try:
            age = int(input("Enter your age: "))
            if age < 0:
                print("Invalid input. Please enter a non-negative integer.")
            else:
                return age
        except ValueError:
            print("Invalid input. Please enter a valid integer.")

def greet(name, age):
    print(f"Hello, {name}! You are {age} years old.")

def main():
    name = get_name()
    age = get_age()
    greet(name, age)

main()
```
