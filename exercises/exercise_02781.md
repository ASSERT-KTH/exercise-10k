## Exercise
Create a function that takes a list and an index as input, and returns the element at the specified index. If the index is out of range, return an error message.

## Reference Solution
```python
def get_element_at_index(input_list, index):
    """
    This function returns the element at a specified index in a given list.
    
    Args:
        input_list (list): The list from which to retrieve the element.
        index (int): The index of the element to be retrieved.
    
    Returns:
        any: The element at the specified index if it exists, otherwise an error message.
    """

    # First, we check if the index is within the valid range for the given list
    # We use the len() function to get the number of elements in the list
    # The valid range for a list is from 0 to len(list) - 1
    if index < 0 or index >= len(input_list):
        
        # If the index is out of range, we return an error message
        return "Error: Index out of range"
    
    # If the index is within the valid range, we return the element at that index
    # We use the indexing syntax list[index] to access the element
    else:
        return input_list[index]

# Example usage:
my_list = [1, 2, 3, 4, 5]
print(get_element_at_index(my_list, 2))  # Output: 3
print(get_element_at_index(my_list, 10))  # Output: Error: Index out of range
```
