## Exercise
Write a program that converts temperature from Celsius to Fahrenheit using conditional statements.

## Reference Solution
```python
def convert_to_fahrenheit(celsius):
    """
    Converts temperature from Celsius to Fahrenheit.

    Args:
        celsius (float): Temperature in Celsius.

    Returns:
        float: Temperature in Fahrenheit.
    """
    if celsius < -273.15:
        raise ValueError("Temperature cannot be lower than absolute zero (-273.15°C).")

    fahrenheit = (celsius * 9/5) + 32
    return fahrenheit


def main():
    print("Temperature Conversion Program")

    while True:
        try:
            celsius = float(input("Enter temperature in Celsius (or -999 to quit): "))

            if celsius == -999:
                print("Exiting program.")
                break

            fahrenheit = convert_to_fahrenheit(celsius)
            print(f"{celsius}°C is equal to {fahrenheit}°F")

        except ValueError as e:
            print(f"Invalid input: {e}")


if __name__ == "__main__":
    main()
```
