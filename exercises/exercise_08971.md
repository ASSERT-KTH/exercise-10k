## Exercise
Write a function that takes a string and a substring as input and returns the index of the substring in the string.

## Reference Solution
```python
def find_substring(string, sub_string):
    try:
        index = string.index(sub_string)
        return index
    except ValueError:
        return -1

string = input()
sub_string = input()
print(find_substring(string, sub_string))
```
