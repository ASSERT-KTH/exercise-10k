## Exercise
Write a program that takes an integer as input and prints out all numbers from 1 to the given number.

## Reference Solution
```python
def print_numbers(n):
    """
    Prints out all numbers from 1 to the given number.
    
    Args:
        n (int): The upper limit of the range of numbers to be printed.
    """
    for i in range(1, n + 1):
        print(i)

def main():
    # Get user input
    num = int(input("Enter a positive integer: "))

    # Validate user input
    if num <= 0:
        print("Please enter a positive integer.")
    else:
        print_numbers(num)

if __name__ == "__main__":
    main()
```
