## Exercise
Write a program to reverse the order of elements in a given array.

## Reference Solution
```python
def reverse_array(array):
    """
    Reverses the order of elements in the given array.

    Args:
        array (list): The input array to be reversed.

    Returns:
        list: The reversed array.
    """
    return array[::-1]

# Example usage:
if __name__ == "__main__":
    array = [1, 2, 3, 4, 5]
    print("Original array:", array)
    reversed_array = reverse_array(array)
    print("Reversed array:", reversed_array)
```

```python
def reverse_array_in_place(array):
    """
    Reverses the order of elements in the given array in-place.

    Args:
        array (list): The input array to be reversed.
    """
    array.reverse()

# Example usage:
if __name__ == "__main__":
    array = [1, 2, 3, 4, 5]
    print("Original array:", array)
    reverse_array_in_place(array)
    print("Reversed array:", array)
```
