## Exercise
Create a program that takes a dictionary of student grades as input, where each key is a student's name and each value is another dictionary containing the grades for different subjects. The program should calculate the average grade for each subject across all students and return the results as a dictionary where the keys are the subject names and the values are the average grades.

## Reference Solution
```python
def average_grades(grades):
    subjects = {}
    for student in grades.values():
        for subject, grade in student.items():
            if subject not in subjects:
                subjects[subject] = []
            subjects[subject].append(grade)

    averages = {}
    for subject, grades in subjects.items():
        averages[subject] = sum(grades) / len(grades)

    return averages

# Example usage:
grades = {
    'Alice': {'Math': 90, 'Science': 85, 'English': 95},
    'Bob': {'Math': 80, 'Science': 90, 'English': 80},
    'Charlie': {'Math': 95, 'Science': 85, 'English': 90}
}

print(average_grades(grades))
```
