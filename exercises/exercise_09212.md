## Exercise
Create a function that finds the index of a specific element in a given array. If the element is not found, return -1. The function should handle arrays with duplicate elements and elements that do not exist in the array.

## Reference Solution
```python
def find_index(array, target):
    """
    This function finds the index of a specific element in a given array.
    
    If the element is not found, it returns -1. The function handles arrays with 
    duplicate elements and elements that do not exist in the array.

    Args:
        array (list): The list of elements to search through.
        target: The element to search for.

    Returns:
        int: The index of the target element if found, -1 otherwise.
    """

    # First, we check if the array is empty to avoid any potential errors
    if not array:
        return -1  # Return -1 if the array is empty

    # We use the enumerate function to get both the index and value of each element in the array
    # This allows us to easily access the index of the target element if it's found
    for i, element in enumerate(array):
        
        # For each element in the array, we check if it matches the target
        if element == target:
            return i  # Return the index of the target element if it's found

    # If we've iterated through the entire array and haven't found the target, we return -1
    return -1  # Return -1 if the target element is not found


# Test the function
array = [1, 2, 3, 4, 5]
print(find_index(array, 3))  # Output: 2
print(find_index(array, 6))  # Output: -1
```
