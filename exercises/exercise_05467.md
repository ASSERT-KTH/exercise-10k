## Exercise
Write a program that takes a list, an index, and an element as input and inserts the element at the specified index.

## Reference Solution
```python
def insert_element(lst, index, element):
    """
    Inserts an element at a specified index in a list.

    Args:
        lst (list): The input list.
        index (int): The index where the element should be inserted.
        element: The element to be inserted.

    Returns:
        None
    """
    lst.insert(index, element)

# Example usage:
lst = [1, 2, 3, 5]
index = 3
element = 4

print("Original list:", lst)
insert_element(lst, index, element)
print("List after insertion:", lst)
```
