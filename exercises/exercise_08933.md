## Exercise
Swap the values of two variables without using a temporary variable.

## Reference Solution
```python
def swap_variables_arithmetic(a, b):
    print("Before swap: a = {}, b = {}".format(a, b))
    a = a + b
    b = a - b
    a = a - b
    print("After swap: a = {}, b = {}".format(a, b))

# Example usage:
swap_variables_arithmetic(5, 10)
```

def swap_variables_tuple(a, b):
    print("Before swap: a = {}, b = {}".format(a, b))

    print("After swap: a = {}, b = {}".format(a, b))
