## Exercise
Partition a list around a given pivot element.

## Reference Solution
```python
def partition_list(lst, pivot):
    """
    Partition a list around a given pivot element.

    Args:
        lst (list): The input list to be partitioned.
        pivot (any): The pivot element.

    Returns:
        tuple: A tuple of three lists: elements less than the pivot, elements equal to the pivot, and elements greater than the pivot.
    """
    less_than_pivot = [x for x in lst if x < pivot]
    equal_to_pivot = [x for x in lst if x == pivot]
    greater_than_pivot = [x for x in lst if x > pivot]

    return less_than_pivot, equal_to_pivot, greater_than_pivot

# Example usage
lst = [1, 2, 3, 4, 5, 2, 3, 4, 5]
pivot = 3

less_than_pivot, equal_to_pivot, greater_than_pivot = partition_list(lst, pivot)

print("Elements less than the pivot:", less_than_pivot)
print("Elements equal to the pivot:", equal_to_pivot)
print("Elements greater than the pivot:", greater_than_pivot)
```
