## Exercise
Write a program to calculate the exponentiation of two numbers

## Reference Solution
```python
def calculate_exponentiation(base, exponent):
    """
    Calculate the exponentiation of two numbers.

    Args:
        base (float): The base number.
        exponent (float): The exponent.

    Returns:
        float: The result of the exponentiation.
    """
    return base ** exponent

def main():
    # Get user input
    base = float(input("Enter the base number: "))
    exponent = float(input("Enter the exponent: "))

    # Calculate exponentiation
    result = calculate_exponentiation(base, exponent)

    # Print result
    print(f"{base} raised to the power of {exponent} is {result}")

if __name__ == "__main__":
    main()
```
