## Exercise
Create a function that takes an array and an element as input and returns True if the element is in the array, False otherwise.

## Reference Solution
```python
def check_element(array, element):
    """
    This function checks if an element is present in a given array.

    Args:
        array (list): The input array to be searched.
        element: The element to be searched for in the array.

    Returns:
        bool: True if the element is found in the array, False otherwise.
    """

    # First, we need to check if the input array is actually a list
    if not isinstance(array, list):
        raise TypeError("Input array must be a list")

    # Now, we can use the 'in' operator in Python, which checks if a value is present in a sequence
    # This operator will return True if the element is found, and False otherwise
    return element in array

# Example usage:
array = [1, 2, 3, 4, 5]
element = 3
print(check_element(array, element))  # Output: True

element = 6
print(check_element(array, element))  # Output: False
```
