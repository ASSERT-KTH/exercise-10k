## Exercise
Write a program that rotates the elements of a given array to the right by a specified number of steps. For example, if the array is [1, 2, 3, 4, 5] and the number of steps is 2, the output should be [4, 5, 1, 2, 3].

## Reference Solution
```python
# Author: [Your Name]
# Date: [Today's Date]
# Description: Rotates the elements of a given array to the right by a specified number of steps.

def rotate_once(arr):
    tmp = arr[-1]
    for i in range(len(arr) - 1, 0, -1):
        arr[i] = arr[i - 1]
    arr[0] = tmp
    return arr

def rotate_n_times(arr, n):
    for _ in range(n):
        arr = rotate_once(arr)
    return arr

arr = [1, 2, 3, 4, 5]
n = 2
result = rotate_n_times(arr, n)
print(result)
```
