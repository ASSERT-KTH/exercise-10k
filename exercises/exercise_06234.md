## Exercise
Write a function that checks if a given list contains consecutive numbers.

## Reference Solution
```python
def has_consecutive_numbers(num_list):
    """
    Checks if a given list contains consecutive numbers.

    Args:
        num_list (list): A list of integers.

    Returns:
        bool: True if the list contains consecutive numbers, False otherwise.
    """
    # Sort the list in ascending order
    num_list.sort()

    # Check for consecutive numbers
    for i in range(len(num_list) - 1):
        if num_list[i + 1] - num_list[i] != 1:
            return False

    return True

    print(has_consecutive_numbers([1, 3, 5, 7, 9]))  # Expected output: False

    # Test case 3: List with duplicate numbers
    print(has_consecutive_numbers([1, 2, 2, 3, 4]))  # Expected output: False
```
