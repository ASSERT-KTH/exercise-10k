## Exercise
Write a program that takes an array of integers as input, and returns a new array containing the same elements in sorted order.

## Reference Solution
```python
function sortArray(arr) {
  const sorted = [...arr];
  sorted.sort((a, b) => a - b);
  return sorted;
}

console.log(sortArray([3, 6, 1, 8, 2, 4])); // Output: [1, 2, 3, 4, 6, 8]
```
