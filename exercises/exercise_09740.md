## Exercise
Write a program that takes an angle as input and returns its tangent.

## Reference Solution
```python
import math

def calculate_tangent(angle_in_radians):
    """
    Calculate the tangent of an angle in radians.

    Args:
        angle_in_radians (float): The angle in radians.

    Returns:
        float: The tangent of the angle.
    """
    return math.tan(angle_in_radians)

def main():
    # Get the angle from the user
    angle_in_degrees = float(input("Enter an angle in degrees: "))

    # Convert the angle to radians
    angle_in_radians = math.radians(angle_in_degrees)

    # Calculate the tangent
    tangent = calculate_tangent(angle_in_radians)

    # Print the result
    print(f"The tangent of {angle_in_degrees} degrees is {tangent}")

if __name__ == "__main__":
    main()
```
