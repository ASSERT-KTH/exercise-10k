## Exercise
Write a program that calculates the factorial of a given number using a for loop.

## Reference Solution
```python
def factorial(n):
    if n == 0:
        return 1
    else:
        result = 1
        for i in range(1, n + 1):
            result *= i
        return result

# Test the function
num = int(input("Enter a number: "))
print("The factorial of", num, "is", factorial(num))
```
