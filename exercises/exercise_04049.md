## Exercise
Write a function that takes a dictionary as input and removes all key-value pairs from the dictionary.

## Reference Solution
```python
def clear_dictionary(input_dict):
    """
    This function takes a dictionary as input and removes all key-value pairs from the dictionary.

    Args:
        input_dict (dict): The dictionary to be cleared.

    Returns:
        dict: The cleared dictionary.
    """

    # First, we check if the input is indeed a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary")

    # We can use the built-in clear method of Python dictionaries to remove all key-value pairs
    # This method does not return anything, it modifies the dictionary in-place
    input_dict.clear()

    # Finally, we return the cleared dictionary
    return input_dict

# Example usage:
my_dict = {"a": 1, "b": 2, "c": 3}
print("Before clearing:", my_dict)
clear_dictionary(my_dict)
print("After clearing:", my_dict)
```
