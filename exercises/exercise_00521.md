## Exercise
Write a function to find the maximum value in an array of integers.

## Reference Solution
```python
def find_max_value(array):
    """
    Finds the maximum value in an array of integers.

    Args:
        array (list): A list of integers.

    Returns:
        int: The maximum value in the array.

    Raises:
        ValueError: If the input array is empty.
    """
    if not array:
        raise ValueError("Input array is empty")

    return max(array)

# Example usage:
def main():
    array = [12, 45, 7, 23, 56, 89, 34]
    try:
        max_value = find_max_value(array)
        print(f"Maximum value in the array: {max_value}")
    except ValueError as e:
        print(e)

if __name__ == "__main__":
    main()
```
