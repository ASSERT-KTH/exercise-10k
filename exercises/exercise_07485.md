## Exercise
Write a program that finds the intersection of two sets

## Reference Solution
```python
def set_intersection(set1, set2):
    """
    Returns the intersection of two sets.

    Args:
        set1 (set): The first set.
        set2 (set): The second set.

    Returns:
        set: The intersection of set1 and set2.
    """
    return set1 & set2

def main():
    # Example usage
    set1 = {1, 2, 3, 4, 5}
    set2 = {4, 5, 6, 7, 8}

    intersection = set_intersection(set1, set2)
    print("Set 1:", set1)
    print("Set 2:", set2)
    print("Intersection:", intersection)

if __name__ == "__main__":
    main()
```
