## Exercise
Write a function that finds the union of a list of lists.

## Reference Solution
```python
def union_of_lists(lists):
    """
    Finds the union of a list of lists.

    Args:
        lists (list): A list of lists.

    Returns:
        list: A new list containing all unique elements from the input lists.
    """
    return list(set([item for sublist in lists for item in sublist]))

# Example usage:
if __name__ == "__main__":
    lists = [[1, 2, 3], [2, 3, 4], [3, 4, 5]]
    print("Input lists:", lists)
    print("Union of lists:", union_of_lists(lists))
```
