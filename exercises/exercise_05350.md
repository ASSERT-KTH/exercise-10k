## Exercise
Write a function that takes a list as input, and returns a new list containing the elements of the input list in reverse order.

## Reference Solution
```python
def reverse_list(input_list):
    """
    Returns a new list containing the elements of the input list in reverse order.

    Args:
        input_list (list): The input list to be reversed.

    Returns:
        list: A new list containing the elements of the input list in reverse order.
    """
    return input_list[::-1]

# Example usage:
def main():
    input_list = [1, 2, 3, 4, 5]
    reversed_list = reverse_list(input_list)
    print("Input List:", input_list)
    print("Reversed List:", reversed_list)

if __name__ == "__main__":
    main()
```
