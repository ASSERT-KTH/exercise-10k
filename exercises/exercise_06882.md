## Exercise
Write a program that declares and initializes variables of different data types (integer, float, string, boolean) and then prints out their values and respective data types. Use a loop to iterate over the variables and print their information.

## Reference Solution
```python
# Variables and Data Types

# Declare and Initialize Variables
int_var = 5
float_var = 5.5
str_var = "Hello"
bool_var = False

# Print Variables and Their Data Types
print("int_var:", int_var, type(int_var))
print("float_var:", float_var, type(float_var))
print("str_var:", str_var, type(str_var))
print("bool_var:", bool_var, type(bool_var))

# Using a Loop to Print Variables
var_dict = {
    "int_var": int_var,
    "float_var": float_var,
    "str_var": str_var,
    "bool_var": bool_var
}

for var_name, var_value in var_dict.items():
    print(f"{var_name}: {var_value} ({type(var_value)})")
```
