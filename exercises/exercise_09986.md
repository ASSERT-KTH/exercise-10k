## Exercise
Understand the difference between local and global variables and use them correctly

## Reference Solution
```python
def greet(name):
    # Local variable
    message = f"Hello, {name}!"
    print(message)

greet("John")  # Output: Hello, John!
print(message)  # Error: message is not defined
```

```python
# Global variable
message = "Hello, World!"

def greet():
    print(message)

greet()  # Output: Hello, World!
print(message)  # Output: Hello, World!
```

```python
# Global variable
count = 0

def increment():
    global count  # Declare count as global
    count += 1

def get_count():
    return count

increment()
print(get_count())  # Output: 1
```

```python
# Global variable
message = "Hello, World!"

def change_message():
    global message  # Declare message as global
    message = "Goodbye, World!"

change_message()
print(message)  # Output: Goodbye, World!
```
