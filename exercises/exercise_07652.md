## Exercise
Write a function to find the maximum of two given numbers.

## Reference Solution
```python
def find_max(num1: float, num2: float) -> float:
    """
    Returns the maximum of two given numbers.

    Args:
    num1 (float): The first number.
    num2 (float): The second number.

    Returns:
    float: The maximum of num1 and num2.
    """
    return max(num1, num2)

# Example usage:
if __name__ == "__main__":
    num1 = 10.0
    num2 = 20.0
    max_value = find_max(num1, num2)
    print(f"The maximum of {num1} and {num2} is {max_value}")
```
