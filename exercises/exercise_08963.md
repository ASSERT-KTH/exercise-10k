## Exercise
Write a function that takes a string as input and returns the string in upper case.

## Reference Solution
```python
def convert_to_upper_case(input_string: str) -> str:
    """
    Converts the input string to upper case.

    Args:
        input_string (str): The input string to be converted.

    Returns:
        str: The input string in upper case.
    """
    return input_string.upper()

# Example usage:
def main():
    input_str = "Hello, World!"
    print(f"Original string: {input_str}")
    print(f"Upper case string: {convert_to_upper_case(input_str)}")

if __name__ == "__main__":
    main()
```
