## Exercise
Write a program that reads a string from the user and prints it out with a prefix added to each line.

## Reference Solution
```python
def print_lines_with_prefix(input_string, prefix):
    """
    Prints each line of the input string with the specified prefix.

    Args:
        input_string (str): The input string to be processed.
        prefix (str): The prefix to be added to each line.
    """
    lines = input_string.split('\n')
    for line in lines:
        print(prefix + line)


def main():
    # Read the input string from the user
    input_string = input("Please enter a string: ")

    # Specify the prefix
    prefix = "> "

    # Print the lines with the prefix
    print_lines_with_prefix(input_string, prefix)


if __name__ == "__main__":
    main()
```
