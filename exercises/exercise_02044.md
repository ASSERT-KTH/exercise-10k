## Exercise
Write a function to update the value associated with a given key in a dictionary.

## Reference Solution
```python
def update_dict_value(dictionary, key, value):
    """
    Updates the value associated with a given key in a dictionary.
    
    Args:
        dictionary (dict): The dictionary to be updated.
        key (any): The key to be updated or added.
        value (any): The new value associated with the key.
    
    Returns:
        dict: The updated dictionary.
    """
    dictionary[key] = value
    return dictionary

# Example usage
my_dict = {"name": "John", "age": 30}
print("Original Dictionary:", my_dict)

updated_dict = update_dict_value(my_dict, "age", 31)
print("Updated Dictionary:", updated_dict)

new_dict = update_dict_value(my_dict, "city", "New York")
print("Dictionary with new key-value pair:", new_dict)
```
