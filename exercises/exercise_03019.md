## Exercise
Write a program to check whether a given string contains a specified substring.

## Reference Solution
```python
def check_substring(main_string, sub_string):
    """
    Checks if the main string contains the specified substring.

    Args:
        main_string (str): The main string to search in.
        sub_string (str): The substring to search for.

    Returns:
        bool: True if the substring is found, False otherwise.
    """
    return sub_string in main_string


def main():
    # Example usage
    main_str = input("Enter the main string: ")
    sub_str = input("Enter the substring to search for: ")

    if check_substring(main_str, sub_str):
        print(f"'{sub_str}' is found in '{main_str}'")
    else:
        print(f"'{sub_str}' is not found in '{main_str}'")


if __name__ == "__main__":
    main()
```
