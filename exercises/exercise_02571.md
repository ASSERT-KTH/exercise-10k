## Exercise
Write a program that prints the first n natural numbers using a for loop.

## Reference Solution
```python
def print_natural_numbers(n):
    """
    Prints the first n natural numbers.

    Args:
        n (int): The number of natural numbers to print.
    """
    for i in range(1, n + 1):
        print(i)

def main():
    # Example usage
    n = 10  # Change this to print a different number of natural numbers
    print(f"Printing the first {n} natural numbers:")
    print_natural_numbers(n)

if __name__ == "__main__":
    main()
```
