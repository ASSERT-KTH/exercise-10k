## Exercise
Write a program that takes a number as input and outputs whether it is palindromic or not.

## Reference Solution
```python
def is_palindrome(n):
    """
    Checks if a number is palindromic.

    Args:
        n (int): The number to check.

    Returns:
        bool: True if the number is palindromic, False otherwise.
    """
    return str(n) == str(n)[::-1]

def main():
    # Get user input
    num = input("Enter a number: ")

    # Check if the input is a valid integer
    try:
        num = int(num)
    except ValueError:
        print("Invalid input. Please enter a valid integer.")
        return

    # Check if the number is palindromic
    if is_palindrome(num):
        print(f"{num} is a palindromic number.")
    else:
        print(f"{num} is not a palindromic number.")

if __name__ == "__main__":
    main()
```
