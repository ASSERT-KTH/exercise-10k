## Exercise
Write a program that finds the maximum value in an array of integers. The program should take an array as input, iterate through each element, and keep track of the maximum value encountered so far. The program should then return the maximum value.

## Reference Solution
```python
def find(array):
    max_num = array[0]
    for num in array:
        if num > max_num:
            max_num = num
    return max_num

# Test the function
print(find([1, 2, 3, 4, 5]))  # Output: 5
print(find([-1, -2, -3, -4, -5]))  # Output: -1
```
