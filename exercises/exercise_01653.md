## Exercise
Assign values to variables using tuple unpacking.

## Reference Solution
```python
# Create a tuple
my_tuple = (1, 2, 3)

# Unpack the tuple into variables
a, b, c = my_tuple

print(a)  # Output: 1
print(b)  # Output: 2
print(c)  # Output: 3
```

```python
my_tuple = (1, 2, 3, 4, 5)

a, b, *rest = my_tuple

print(a)  # Output: 1
print(b)  # Output: 2
print(rest)  # Output: [3, 4, 5]
```

```python
my_tuple = (1, 2, 3, 4, 5)

*a, b, c = my_tuple

print(a)  # Output: [1, 2]
print(b)  # Output: 3
print(c)  # Output: 4

# or

a, *b, c = my_tuple

print(a)  # Output: 1
print(b)  # Output: [2, 3, 4]
print(c)  # Output: 5
```

```python
my_tuple = (1, 2, 3)

a, _, c = my_tuple

print(a)  # Output: 1
print(c)  # Output: 3
```
