## Exercise
Write a program to access and print the elements of an array using indexing.

## Reference Solution
```python
# Define an array
my_array = [1, 2, 3, 4, 5]

# Access and print the elements of the array using indexing
for i in range(len(my_array)):
    print(f"Element at index {i}: {my_array[i]}")

# Alternatively, you can use enumerate to get both index and value
print("\nUsing enumerate:")
for index, value in enumerate(my_array):
    print(f"Element at index {index}: {value}")

# Accessing specific elements using indexing
print("\nAccessing specific elements:")
print(f"First element: {my_array[0]}")
print(f"Last element: {my_array[-1]}")
print(f"Middle element: {my_array[len(my_array) // 2]}")
```
