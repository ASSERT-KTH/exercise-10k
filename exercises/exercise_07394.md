## Exercise
Write a program that takes in a string input and determines the corresponding data type in Python. The input string can represent an integer, float, or boolean value. If the input string cannot be converted to any of these data types, it should be classified as a string.

## Reference Solution
```python
def determine_data_type(input_str):
    """
    This function determines the corresponding data type in Python for a given string input.
    
    It checks if the input string can be converted to an integer, float, or boolean value. 
    If it cannot be converted to any of these data types, it is classified as a string.

    Parameters:
    input_str (str): The input string to be evaluated.

    Returns:
    str: The corresponding data type in Python for the input string.
    """

    # First, we check if the input string can be converted to a boolean value
    if input_str.lower() == 'true':
        # If it's 'true', we return 'boolean'
        return 'boolean'
    elif input_str.lower() == 'false':
        # If it's 'false', we return 'boolean'
        return 'boolean'

    # Next, we try to convert the input string to an integer
    try:
        # We use a try block to attempt the conversion
        int(input_str)
        # If the conversion is successful, we return 'integer'
        return 'integer'
    except ValueError:
        # If the conversion fails, we move on to the next step
        pass

    # Then, we try to convert the input string to a float
    try:
        # We use a try block to attempt the conversion
        float(input_str)
        # If the conversion is successful, we return 'float'
        return 'float'
    except ValueError:
        # If the conversion fails, we move on to the next step
        pass

    # If the input string cannot be converted to any of the above data types, we return 'string'
    return 'string'

# Test the function
print(determine_data_type('123'))  # Should print: integer
print(determine_data_type('123.45'))  # Should print: float
print(determine_data_type('true'))  # Should print: boolean
print(determine_data_type('false'))  # Should print: boolean
print(determine_data_type('hello'))  # Should print: string
```
