## Exercise
Create a program that takes two numbers as input and performs addition, subtraction, multiplication, and division based on the user's choice. The program should handle invalid inputs and division by zero errors.

## Reference Solution
```python
#!/usr/bin/python3

# Define a function to add two numbers
def add(x, y):
    return x + y

# Define a function to subtract y from x
def sub(x, y):
    return x - y

# Define a function to multiply two numbers
def mul(x, y):
    return x * y

# Define a function to divide x by y
def div(x, y):
    if y == 0:
        return "Error: Division by zero is not allowed"
    return x / y

# Main program
print("1. Addition")
print("2. Subtraction")
print("3. Multiplication")
print("4. Division")

while True:
    choice = input("Enter your choice (1/2/3/4): ")

    if choice in ['1', '2', '3', '4']:
        num1 = float(input("Enter first number: "))
        num2 = float(input("Enter second number: "))

        if choice == '1':
            print(num1, "+", num2, "=", add(num1, num2))

        elif choice == '2':
            print(num1, "-", num2, "=", sub(num1, num2))

        elif choice == '3':
            print(num1, "*", num2, "=", mul(num1, num2))

        elif choice == '4':
            if num2 != 0:
                print(num1, "/", num2, "=", div(num1, num2))
            else:
                print("Error: Division by zero is not allowed")
    else:
        print("Invalid choice")
```
