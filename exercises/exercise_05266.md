## Exercise
Write a program to determine whether a given character is a vowel or consonant using if/else statements

## Reference Solution
```python
def check_vowel_or_consonant(char):
    """
    Checks if a character is a vowel or consonant.

    Args:
        char (str): The input character.

    Returns:
        str: "Vowel" if the character is a vowel, "Consonant" if it's a consonant, and "Invalid input" otherwise.
    """
    char = char.lower()  # Convert to lowercase for simplicity

    if char in 'aeiou':
        return "Vowel"
    elif char.isalpha():  # Check if the character is a letter
        return "Consonant"
    else:
        return "Invalid input"

def main():
    char = input("Enter a character: ")
    result = check_vowel_or_consonant(char)
    print(f"The character '{char}' is a {result}.")

if __name__ == "__main__":
    main()
```
