## Exercise
Write a program that serializes a dictionary to a JSON string.

## Reference Solution
```python
import json

def dict_to_json(dictionary):
    """
    Serialize a dictionary to a JSON string.

    Args:
        dictionary (dict): The dictionary to be serialized.

    Returns:
        str: The JSON string representation of the dictionary.
    """
    try:
        json_string = json.dumps(dictionary, indent=4)
        return json_string
    except TypeError as e:
        print(f"Error: {e}")
        return None

# Example usage
if __name__ == "__main__":
    example_dict = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }

    json_string = dict_to_json(example_dict)
    print(json_string)
```
