## Exercise
Write a function that takes a dictionary and a key as input and returns the corresponding value. If the key does not exist, return 'Key not found'

## Reference Solution
```python
def get_dict_value(dictionary, key):
    """
    Retrieves the value associated with a given key from a dictionary.

    Args:
        dictionary (dict): The dictionary to search in.
        key (any): The key to search for.

    Returns:
        any: The value associated with the key if found, 'Key not found' otherwise.
    """
    return dictionary.get(key, 'Key not found')

# Example usage:
my_dict = {'name': 'John', 'age': 30}
print(get_dict_value(my_dict, 'name'))  # Output: John
print(get_dict_value(my_dict, 'city'))  # Output: Key not found
```
