## Exercise
Write a function that takes two numbers and an operator as input and returns the result of the operation.

## Reference Solution
```python
def arithmetic_operation(num1, num2, operator):
    """
    Perform an arithmetic operation on two numbers.

    Args:
        num1 (float): The first number.
        num2 (float): The second number.
        operator (str): The arithmetic operator. Can be '+', '-', '*', '/'.

    Returns:
        float: The result of the operation.

    Raises:
        ValueError: If the operator is not one of '+', '-', '*', '/'.
        ZeroDivisionError: If the operator is '/' and num2 is 0.
    """
    if operator == '+':
        return num1 + num2
    elif operator == '-':
        return num1 - num2
    elif operator == '*':
        return num1 * num2
    elif operator == '/':
        if num2 == 0:
            raise ZeroDivisionError("Cannot divide by zero")
        return num1 / num2
    else:
        raise ValueError("Invalid operator. Must be one of '+', '-', '*', '/'.")

# Example usage:
def main():
    print(arithmetic_operation(10, 2, '+'))  # Output: 12
    print(arithmetic_operation(10, 2, '-'))  # Output: 8
    print(arithmetic_operation(10, 2, '*'))  # Output: 20
    print(arithmetic_operation(10, 2, '/'))  # Output: 5.0

if __name__ == "__main__":
    main()
```
