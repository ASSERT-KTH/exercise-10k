## Exercise
Write a program that retrieves the middle element of an array. If the array has an even number of elements, retrieve the average of the two middle elements.

## Reference Solution
```python
def get_middle_element(array):
    """
    Retrieves the middle element of an array. If the array has an even number of elements, 
    retrieves the average of the two middle elements.

    Args:
        array (list): Input list of numbers.

    Returns:
        float: The middle element or the average of the two middle elements.
    """
    # Check if the array is empty
    if not array:
        raise ValueError("Array is empty")

    # Calculate the middle index
    middle_index = len(array) // 2

    # Check if the array has an even number of elements
    if len(array) % 2 == 0:
        # If the array has an even number of elements, return the average of the two middle elements
        return (array[middle_index - 1] + array[middle_index]) / 2
    else:
        # If the array has an odd number of elements, return the middle element
        return array[middle_index]

# Example usage
numbers = [1, 2, 3, 4, 5]
print(get_middle_element(numbers))  # Output: 3

numbers = [1, 2, 3, 4]
print(get_middle_element(numbers))  # Output: 2.5
```
