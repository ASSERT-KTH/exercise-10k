## Exercise
Create a dictionary with 5 key-value pairs. Iterate over the dictionary and print out each key-value pair.

## Reference Solution
```python
def main():
    # Create a dictionary with 5 key-value pairs
    my_dict = {
        "name": "John",
        "age": 30,
        "city": "New York",
        "country": "USA",
        " occupation": "Software Engineer"
    }

    # Iterate over the dictionary and print out each key-value pair
    for key, value in my_dict.items():
        print(f"Key: {key}, Value: {value}")

if __name__ == "__main__":
    main()
```
