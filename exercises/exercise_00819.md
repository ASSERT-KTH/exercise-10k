## Exercise
Store a collection of values in an array variable and print the values.

## Reference Solution
```python
# Define the array
array_values = [1, 2, 3, 4, 5]

# Print the array values
print("Array Values:")
for i, value in enumerate(array_values):
    print(f"Index {i}: {value}")

# Alternatively, you can print the entire array at once
print("\nEntire Array:")
print(array_values)
```
