## Exercise
Write a program that prints numbers from 1 to 10 using a for loop.

## Reference Solution
```python
# Define a function to print numbers
def print_numbers():
    # Use a for loop to iterate from 1 to 10
    for i in range(1, 11):
        print(i)

# Call the function to print numbers
print_numbers()
```
