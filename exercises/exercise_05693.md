## Exercise
Create a program that takes an array as input and checks if it's empty or not using conditional statements.

## Reference Solution
```python
def check_empty_array(input_array):
    if len(input_array) == 0:
        print("The array is empty.")
    else:
        print("The array is not empty.")

# Example usage:
array1 = []
array2 = [1, 2, 3, 4, 5]

print("Checking array1:")
check_empty_array(array1)

print("\nChecking array2:")
check_empty_array(array2)
```
