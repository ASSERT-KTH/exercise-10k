## Exercise
Write a program that uses a loop to print out the numbers from 1 to 10, then uses another loop to print out the numbers from 10 to 1.

## Reference Solution
```python
# Iteration 1:
for i in range(1, 11):
    print(i)

# Iteration 2:
for i in range(10, 0, -1):
    print(i)
```
