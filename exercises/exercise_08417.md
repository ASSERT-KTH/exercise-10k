## Exercise
Write a program that takes a string as input and returns the reversed string. The program should handle strings of any length and should not use any built-in string reversal functions.

## Reference Solution

#!/usr/bin/perl

use strict;
use warnings;

sub reverse_string {
    my $string = shift;
    my $reverse = "";
    my $len = length($string);
    my $i = $len - 1;

    while ($i >= 0) {
        $reverse .= substr($string, $i, 1);
        $i--;
    }

    return $i;
}

my $input = "Hello World";

print "Input: $input\n";
print "Reversed: " . reverse_string($input) . "\n";
