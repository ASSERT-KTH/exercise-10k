## Exercise
Create a program that takes a student's score as input and outputs the corresponding grade based on the following criteria: A (90-100), B (80-89), C (70-79), D (60-69), F (below 60). Use conditional statements to determine the grade.

## Reference Solution
```python
# Author: [Your Name]
# Date: [Today's Date]

# Define a function to determine the grade
def get_grade(score):
    # Use conditional statements to determine the grade
    if score >= 90 and score <= 100:
        return 'A'
    elif score >= 80 and score <= 89:
        return 'B'
    elif score >= 70 and score <= 79:
        return 'C'
    elif score >= 60 and score <= 69:
        return 'D'
    else:
        return 'F'

# Main program
# Get user input
score = float(input("Enter your score: "))

# Call the get_grade function
grade = get_grade(score)

# Print the result
print("Your grade is:", grade)
```
