## Exercise
Write a program to convert a given integer to a string and a floating-point number to an integer.

## Reference Solution
```python
def convert_data_types(input_int, input_float):
    """
    This function converts an integer to a string and a floating-point number to an integer.

    Args:
    input_int (int): The integer to be converted to a string.
    input_float (float): The floating-point number to be converted to an integer.

    Returns:
    str: The string representation of the input integer.
    int: The integer representation of the input floating-point number.
    """

    # Convert the integer to a string
    int_to_str = str(input_int)
    
    # Convert the floating-point number to an integer
    # We use the built-in int() function to truncate the decimal part
    float_to_int = int(input_float)

    return int_to_str, float_to_int


# Example usage
input_int = 123
input_float = 45.67

result_str, result_int = convert_data_types(input_int, input_float)

print(f"Integer {input_int} as a string: {result_str}")
print(f"Floating-point number {input_float} as an integer: {result_int}")
```
