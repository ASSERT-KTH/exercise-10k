## Exercise
Create a program that takes a string as input and prints it in reverse using a while loop.

## Reference Solution
```python
def print_reverse_string(input_string):
    """
    Prints the input string in reverse using a while loop.

    Args:
        input_string (str): The string to be printed in reverse.
    """
    # Initialize a variable to keep track of the current index
    index = len(input_string) - 1

    # Use a while loop to iterate over the string from the end to the beginning
    while index >= 0:
        # Print the character at the current index
        print(input_string[index], end='')

        # Decrement the index to move to the previous character
        index -= 1

# Test the function
input_string = input("Enter a string: ")
print_reverse_string(input_string)
```
