## Exercise
Retrieve the value of a specific key from a dictionary

## Reference Solution
```python
def retrieve_value(dictionary, key):
    """
    Retrieves the value of a specific key from a dictionary.

    Args:
        dictionary (dict): The dictionary to retrieve the value from.
        key: The key to retrieve the value for.

    Returns:
        The value associated with the key if it exists, otherwise None.
    """
    return dictionary.get(key)

# Example usage:
my_dict = {"name": "John", "age": 30, "city": "New York"}
print(retrieve_value(my_dict, "name"))  # Output: John
print(retrieve_value(my_dict, "country"))  # Output: None
```

```python
def retrieve_value(dictionary, key):
    """
    Retrieves the value of a specific key from a dictionary.

    Args:
        dictionary (dict): The dictionary to retrieve the value from.
        key: The key to retrieve the value for.

    Returns:
        The value associated with the key if it exists, otherwise None.
    """
    try:
        return dictionary[key]
    except KeyError:
        return None

# Example usage:
```
