## Exercise
Create a function to generate the Fibonacci sequence up to the nth term.

## Reference Solution
```python
def generate_fibonacci(n):
    """
    Generate the Fibonacci sequence up to the nth term.

    Args:
        n (int): The number of terms in the sequence.

    Returns:
        list: A list of Fibonacci numbers up to the nth term.
    """
    if n <= 0:
        return []
    elif n == 1:
        return [0]
    elif n == 2:
        return [0, 1]
    else:
        fib_sequence = [0, 1]
        while len(fib_sequence) < n:
            fib_sequence.append(fib_sequence[-1] + fib_sequence[-2])
        return fib_sequence

# Example usage:
def main():
    n = 10
    print(f"Fibonacci sequence up to {n} terms:")
    print(generate_fibonacci(n))

if __name__ == "__main__":
    main()
```
