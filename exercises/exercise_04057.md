## Exercise
Write a function that takes a dictionary as input and returns a new dictionary containing the key-value pairs sorted by key.

## Reference Solution
```python
def sort_dict_by_key(input_dict):
    """
    This function takes a dictionary as input, sorts its key-value pairs by key, 
    and returns a new dictionary containing the sorted key-value pairs.

    Args:
        input_dict (dict): The input dictionary to be sorted.

    Returns:
        dict: A new dictionary containing the key-value pairs sorted by key.
    """

    # First, we'll check if the input is indeed a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary")

    # We'll use the built-in sorted function to sort the dictionary items by key
    # The sorted function returns a new list containing the dictionary items in sorted order
    sorted_items = sorted(input_dict.items())

    # We'll use the dict function to convert the sorted list back into a dictionary
    # In Python 3.7 and later, dictionaries remember the order of items inserted
    # So, this will give us a new dictionary with the key-value pairs sorted by key
    sorted_dict = dict(sorted_items)

    # Finally, we'll return the sorted dictionary
    return sorted_dict

# Example usage:
input_dict = {'c': 3, 'b': 2, 'a': 1}
sorted_dict = sort_dict_by_key(input_dict)
print(sorted_dict)  # Output: {'a': 1, 'b': 2, 'c': 3}
```
