## Exercise
Create a program that takes a user's input and determines the data type of the input. The program should be able to identify integers, floats, and strings. It should then print out the data type of the input.

## Reference Solution
```python
def determine_data_type(user_input):
    """
    Determine the data type of the user's input.

    Args:
        user_input (str): The user's input.

    Returns:
        str: The data type of the input.
    """
    try:
        # Attempt to convert the input to an integer
        int(user_input)
        return "Integer"
    except ValueError:
        try:
            # Attempt to convert the input to a float
            float(user_input)
            return "Float"
        except ValueError:
            # If it's not an integer or float, it's a string
            return "String"


def main():
    # Get the user's input
    user_input = input("Please enter a value: ")

    # Determine the data type of the input
    data_type = determine_data_type(user_input)

    # Print the data type
    print(f"The data type of '{user_input}' is {data_type}.")


if __name__ == "__main__":
    main()
```
