## Exercise
Create a program that takes the total bill and tip percentage as input, and returns the tip amount and total bill with tip.

## Reference Solution
```python
def calculate_tip(total_bill, tip_percentage):
    """
    Calculate the tip amount and total bill with tip.

    Args:
    - total_bill (float): The total bill amount.
    - tip_percentage (float): The tip percentage.

    Returns:
    - tip_amount (float): The calculated tip amount.
    - total_bill_with_tip (float): The total bill with tip.
    """
    # Calculate the tip amount
    tip_amount = (tip_percentage / 100) * total_bill
    
    # Calculate the total bill with tip
    total_bill_with_tip = total_bill + tip_amount
    
    return tip_amount, total_bill_with_tip

def main():
    # Example usage
    total_bill = 100.0
    tip_percentage = 15.0
    
    tip_amount, total_bill_with_tip = calculate_tip(total_bill, tip_percentage)
    
    print(f"Total Bill: ${total_bill:.2f}")
    print(f"Tip Percentage: {tip_percentage}%")
    print(f"Tip Amount: ${tip_amount:.2f}")
    print(f"Total Bill with Tip: ${total_bill_with_tip:.2f}")

if __name__ == "__main__":
    main()
```
