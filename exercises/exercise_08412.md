## Exercise
Write a program that uses a for loop to iterate through the numbers 1 to 10 and prints each number and its square. If the number is even, also print its double.

## Reference Solution
```python
for i in range(1, 11):
    print(i, i ** 2)
    if i % 2 == 0:
        print(i * 2)
```
