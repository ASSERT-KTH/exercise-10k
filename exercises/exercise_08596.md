## Exercise
Write a function to check if a dictionary is a superset of another dictionary

## Reference Solution
```python
def is_superset(dict1, dict2):
    """
    Check if dict1 is a superset of dict2.

    Args:
        dict1 (dict): The potential superset dictionary.
        dict2 (dict): The dictionary to check against.

    Returns:
        bool: True if dict1 is a superset of dict2, False otherwise.
    """
    return all(key in dict1 and dict1[key] == value for key, value in dict2.items())

# Example usage
if __name__ == "__main__":
    dict1 = {"a": 1, "b": 2, "c": 3}
    dict2 = {"a": 1, "b": 2}
    print(is_superset(dict1, dict2))  # Output: True

    dict3 = {"a": 1, "b": 3}
    print(is_superset(dict1, dict3))  # Output: False
```
