## Exercise
Write a program that takes a string and two indices as input and returns the substring between the indices.

## Reference Solution
```python
import sys

def substring_between_indices(s, start, end):
    if start < 0 or end > len(s) or start > end:
        return None
    else:
        return s[start:end]

s = sys.stdin.readline().strip()
start = int(input())
end = int(input())

print(substring_between_indices(s, start, end))
```
