## Exercise
Write a program that takes the mass and velocity of an object as input and outputs its kinetic energy.

## Reference Solution
```python
# -*- coding: utf-8 -*-

# Created on 2015-9-7
# @author: Administrator

def kinetic_energy(m, v):
    energy = 0.5 * m * v * v
    return energy

m = 1.0
v = 2.0
print(kinetic_energy(m, v))
```
