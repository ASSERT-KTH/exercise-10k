## Exercise
Write a program to convert a given string to uppercase or lowercase.

## Reference Solution
```python
def convert_case(input_string, case):
    """
    Converts the input string to the specified case.

    Args:
        input_string (str): The input string to be converted.
        case (str): The case to convert the string to. Can be 'upper' or 'lower'.

    Returns:
        str: The converted string.
    """
    if case == 'upper':
        return input_string.upper()
    elif case == 'lower':
        return input_string.lower()
    else:
        raise ValueError("Invalid case. Please specify 'upper' or 'lower'.")

def main():
    input_string = input("Enter a string: ")
    print("1. Convert to uppercase")
    print("2. Convert to lowercase")
    choice = input("Enter your choice (1/2): ")

    if choice == '1':
        print(convert_case(input_string, 'upper'))
    elif choice == '2':
        print(convert_case(input_string, 'lower'))
    else:
        print("Invalid choice. Please try again.")

if __name__ == "__main__":
    main()
```
