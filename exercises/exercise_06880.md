## Exercise
Write a program that takes a string input from the user and checks if it contains only alphabets and spaces. If the input is valid, print 'Valid input' otherwise print 'Invalid input'. The program should also handle cases where the user enters an empty string.

## Reference Solution
```python
# Author: [Your Name]
# Date: [Today's Date]
# Purpose: Check if a string contains only alphabets and spaces

def check():
    user_input = input("Enter a string: ")
    if not user_input:  # Check for empty string
        print("Invalid input")
    else:
        for char in user_input:
            if not char.isalpha() and not char.isspace():
                print("Invalid input")
                return
        print("Valid input")

check()
```
