## Exercise
Create two functions and compose them to create a new function.

## Reference Solution
```python
# Define the first function: addFive
def addFive(x: int) -> int:
    """
    Adds 5 to the input number.

    Args:
    x (int): The input number.

    Returns:
    int: The input number plus 5.
    """
    return x + 5

# Define the second function: multiplyByTwo
def multiplyByTwo(x: int) -> int:
    """
    Multiplies the input number by 2.

    Args:
    x (int): The input number.

    Returns:
    int: The input number multiplied by 2.
    """
    return x * 2

# Compose the functions to create a new function: transformNumber
def transformNumber(x: int) -> int:
    """
    Adds 5 to the input number and then multiplies the result by 2.

    Args:
    x (int): The input number.

    Returns:
    int: The transformed number.
    """
    return multiplyByTwo(addFive(x))

# Example usage
if __name__ == "__main__":
    input_number = 10
    result = transformNumber(input_number)
    print(f"Transformed {input_number} to {result}")
```
