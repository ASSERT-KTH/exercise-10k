## Exercise
Write a function to convert a dictionary into a list of key-value pairs.

## Reference Solution
```python
def dict_to_list(input_dict):
    """
    This function takes a dictionary as input and returns a list of key-value pairs.

    Args:
        input_dict (dict): The dictionary to be converted.

    Returns:
        list: A list of tuples, where each tuple contains a key-value pair from the input dictionary.
    """

    # Initialize an empty list to store the key-value pairs
    pair_list = []

    # Iterate over each key-value pair in the input dictionary
    for key, value in input_dict.items():
        # For each pair, create a tuple with the key and value
        pair = (key, value)
        
        # Append this tuple to the list
        pair_list.append(pair)

    # After iterating over all pairs, return the list
    return pair_list

# Example usage:
input_dict = {'a': 1, 'b': 2, 'c': 3}
print(dict_to_list(input_dict))  # Output: [('a', 1), ('b', 2), ('c', 3)]
```
