## Exercise
Create a dictionary with 10 key-value pairs. Filter out the key-value pairs where the value is greater than 10.

## Reference Solution
```python
# Create a dictionary with 10 key-value pairs
data = {
    "a": 5,
    "b": 15,
    "c": 8,
    "d": 12,
    "e": 3,
    "f": 20,
    "g": 9,
    "h": 1,
    "i": 11,
    "j": 7
}

# Filter out key-value pairs where the value is greater than 10
filtered_data = {key: value for key, value in data.items() if value <= 10}

# Print the original and filtered dictionaries
print("Original Dictionary:")
print(data)
print("\nFiltered Dictionary:")
print(filtered_data)
```
