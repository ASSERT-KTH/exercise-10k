## Exercise
Filter a dictionary to only include key-value pairs where the value meets a certain condition

## Reference Solution
```python
# Define the dictionary
students = {
    "John": 20,
    "Alice": 17,
    "Bob": 25,
    "Charlie": 19,
    "David": 15
}

# Define the condition function
def is_adult(age):
    """Check if a person is an adult (above 18 years old)"""
    return age > 18

# Filter the dictionary
adult_students = {name: age for name, age in students.items() if is_adult(age)}

# Print the filtered dictionary
print(adult_students)
```
