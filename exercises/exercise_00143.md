## Exercise
Create a function that takes in a dictionary and a new key-value pair, and updates the dictionary with the new pair

## Reference Solution
```python
def update_dictionary(input_dict, new_key, new_value):
    """
    Updates a dictionary with a new key-value pair.

    Args:
        input_dict (dict): The dictionary to be updated.
        new_key (any): The key of the new key-value pair.
        new_value (any): The value of the new key-value pair.

    Returns:
        dict: The updated dictionary.
    """
    # Check if the input is a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary")

    # Update the dictionary with the new key-value pair
    input_dict[new_key] = new_value

    # Return the updated dictionary
    return input_dict

# Example usage:
my_dict = {"name": "John", "age": 30}
new_key = "city"
new_value = "New York"

updated_dict = update_dictionary(my_dict, new_key, new_value)
print(updated_dict)
```
