## Exercise
Write a function that creates a copy of a given dictionary.

## Reference Solution
```python
def copy_dictionary(input_dict):
    """
    This function creates a copy of a given dictionary.

    Args:
        input_dict (dict): The dictionary to be copied.

    Returns:
        dict: A copy of the input dictionary.
    """

    # First, we check if the input is indeed a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary")

    # We use the built-in dict.copy() method to create a copy of the dictionary
    # This method returns a shallow copy of the dictionary, which means it creates a new dictionary and then (to the extent possible) inserts references to the original elements
    copied_dict = input_dict.copy()

    # Finally, we return the copied dictionary
    return copied_dict

# Example usage:
original_dict = {"a": 1, "b": 2, "c": 3}
copied_dict = copy_dictionary(original_dict)
print(copied_dict)  # Output: {'a': 1, 'b': 2, 'c': 3}
```
