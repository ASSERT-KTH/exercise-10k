## Exercise
Create a program to validate if a given string contains only alphanumeric characters.

## Reference Solution
```python
def is_alphanumeric(input_string):
    """
    Validate if a given string contains only alphanumeric characters.

    Args:
        input_string (str): The string to be validated.

    Returns:
        bool: True if the string is alphanumeric, False otherwise.
    """
    return input_string.isalnum()

def main():
    # Example usage:
    input_string = input("Enter a string: ")
    if is_alphanumeric(input_string):
        print(f"'{input_string}' is alphanumeric.")
    else:
        print(f"'{input_string}' is not alphanumeric.")

if __name__ == "__main__":
    main()
```
