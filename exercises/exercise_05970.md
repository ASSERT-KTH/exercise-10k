## Exercise
Write a program that takes a list as input, and returns a new list that is the reverse of the input list.

## Reference Solution
```python
def reverse_list(input_list):
    """
    Returns a new list that is the reverse of the input list.

    Args:
        input_list (list): The list to be reversed.

    Returns:
        list: A new list that is the reverse of the input list.
    """
    return input_list[::-1]

# Example usage:
input_list = [1, 2, 3, 4, 5]
print(reverse_list(input_list))  # Output: [5, 4, 3, 2, 1]
```
