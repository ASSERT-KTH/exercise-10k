## Exercise
Write a function to find the maximum contiguous subarray of an array of integers. The function should take an array as input and return the maximum sum of a subarray. If the array is empty, return 0.

## Reference Solution
```python
def maxSubArray(nums):
    if not nums:
        return 0

    max_sum = float('-inf')
    current_sum = 0

    for num in nums:
        current_sum = max(num, current_sum + num)
        max_sum = max(max_sum, current_sum)

    return max_sum
```
