## Exercise
Write a program that takes a string and a partition size as input and returns a list of substrings of the specified size.

## Reference Solution
```python
function partition(s, n) {
  const result = [];
  for (let i = 0; i < s.length; i += n) {
    result.push(s.slice(i, i + n));
  }
  return result;
}

// Test the function
const str = "abcabcabcabc";
const partitionSize = 3;
console.log(partition(str, partitionSize));
```
