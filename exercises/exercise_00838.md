## Exercise
Given two strings, calculate a similarity score between them using a fuzzy matching algorithm.

## Reference Solution
```python
from fuzzywuzzy import fuzz
from fuzzywuzzy import process

def calculate_similarity_score(str1, str2):
    """
    Calculate the similarity score between two strings using Levenshtein distance and ratio.

    Args:
        str1 (str): The first string.
        str2 (str): The second string.

    Returns:
        tuple: A tuple containing the Levenshtein distance and the similarity ratio.
    """
    levenshtein_distance = fuzz.ratio(str1, str2)
    return levenshtein_distance

def main():
    str1 = "kitten"
    str2 = "sitting"
    similarity_score = calculate_similarity_score(str1, str2)
    print(f"The similarity score between '{str1}' and '{str2}' is: {similarity_score}")

if __name__ == "__main__":
    main()
```
