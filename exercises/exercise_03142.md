## Exercise
Write a function to find the maximum value in an array of integers.

## Reference Solution
```python
def find_max_value(array):
    """
    This function finds the maximum value in an array of integers.

    Args:
        array (list): A list of integers.

    Returns:
        int: The maximum value in the array.

    Raises:
        ValueError: If the array is empty.
    """
    if not array:
        raise ValueError("Array is empty")

    return max(array)

# Example usage:
def main():
    array = [4, 2, 9, 6, 5, 1]
    try:
        max_value = find_max_value(array)
        print("Maximum value:", max_value)
    except ValueError as e:
        print(e)

if __name__ == "__main__":
    main()
```
