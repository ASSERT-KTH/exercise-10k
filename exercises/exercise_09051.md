## Exercise
Write a program that checks if a given number is odd or even using an if statement

## Reference Solution
```python
def check_even_or_odd(num):
    """
    Checks if a number is even or odd.

    Args:
        num (int): The number to check.

    Returns:
        str: "Even" if the number is even, "Odd" otherwise.
    """
    if num % 2 == 0:
        return "Even"
    else:
        return "Odd"

def main():
    # Example usage
    num = int(input("Enter a number: "))
    result = check_even_or_odd(num)
    print(f"{num} is {result}")

if __name__ == "__main__":
    main()
```
