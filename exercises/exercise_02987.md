## Exercise
Write a program that reverses a given list of numbers.

## Reference Solution
```python
def reverse_list(numbers):
    """
    Reverses a given list of numbers.

    Args:
        numbers (list): A list of numbers.

    Returns:
        list: The reversed list of numbers.
    """
    return numbers[::-1]

# Example usage:
numbers = [1, 2, 3, 4, 5]
print("Original List:", numbers)
print("Reversed List:", reverse_list(numbers))
```

```python
def reverse_list_in_place(numbers):
    """
    Reverses a given list of numbers in-place.

    Args:
        numbers (list): A list of numbers.
    """
    numbers.reverse()

# Example usage:
numbers = [1, 2, 3, 4, 5]
print("Original List:", numbers)
reverse_list_in_place(numbers)
print("Reversed List:", numbers)
```
