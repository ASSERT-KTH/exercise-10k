## Exercise
Write a program that takes a list of numbers as input and outputs the sum of their squares.

## Reference Solution
```python
def sum_of_squares(numbers):
    return sum([x**2 for x in numbers])

# Example usage:
numbers = [1, 2, 3, 4, 5]
result = sum_of_squares(numbers)
print("The sum of the squares of the numbers is:", result)
```
