## Exercise
Write a program that takes two strings as input and returns true if they are anagrams of each other, and false otherwise.

## Reference Solution
```python
def are_anagrams(str1, str2):
    # Remove spaces and convert to lower case
    str1 = str1.replace(" ", "").lower()
    str2 = str2.replace(" ", "").lower()
    
    # Check if sorted strings are equal
    return sorted(str1) == sorted(str2)

# Example usage:
print(are_anagrams("Listen", "Silent"))  # Returns: True
print(are_anagrams("Hello", "World"))  # Returns: False
```
