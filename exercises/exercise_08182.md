## Exercise
Write a program that prints a pattern of asterisks (*) using a nested for loop.

## Reference Solution
```python
def print_asterisk_pattern(side_length):
    """
    Prints a square pattern of asterisks with the specified side length.

    Args:
        side_length (int): The length of the side of the square pattern.
    """
    for i in range(side_length):
        for j in range(side_length):
            print("*", end=" ")
        print()

def main():
    side_length = int(input("Enter the side length of the pattern: "))
    print_asterisk_pattern(side_length)

if __name__ == "__main__":
    main()
```
