## Exercise
Write a program to extract a subset of elements from a list using slicing.

## Reference Solution
```python
def extract_subset(input_list, start, end):
    """
    Extracts a subset of elements from the input list using slicing.

    Args:
        input_list (list): The input list.
        start (int): The starting index of the subset.
        end (int): The ending index of the subset.

    Returns:
        list: The extracted subset of elements.
    """
    # Extract the subset using slicing
    subset = input_list[start:end]

    return subset


# Example usage
input_list = [1, 2, 3, 4, 5, 6, 7, 8, 9]
start_index = 2
end_index = 6

subset = extract_subset(input_list, start_index, end_index)
print("Input List:", input_list)
print("Extracted Subset:", subset)
```
