## Exercise
Create a dictionary with 3 key-value pairs and serialize it to a JSON string.

## Reference Solution
```python
import json

def create_and_serialize_dict():
    # Create a dictionary with 3 key-value pairs
    data = {
        "name": "John Doe",
        "age": 30,
        "city": "New York"
    }

    # Serialize the dictionary to a JSON string
    json_string = json.dumps(data, indent=4)

    return json_string

# Example usage:
if __name__ == "__main__":
    json_string = create_and_serialize_dict()
    print(json_string)
```
