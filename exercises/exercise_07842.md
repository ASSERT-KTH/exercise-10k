## Exercise
Write a program that takes a user input and checks its data type. If the input is an integer, it should print 'Integer'. If the input is a float, it should print 'Float'. If the input is a string, it should print 'String'. If the input is none of the above, it should print 'Unknown'

## Reference Solution
```python
def check_data_type(user_input):
    """
    Checks the data type of the user input.

    Args:
        user_input: The input provided by the user.

    Returns:
        str: The data type of the user input.
    """
    try:
        # Try to convert the input to an integer
        int(user_input)
        return 'Integer'
    except ValueError:
        try:
            # If not an integer, try to convert to a float
            float(user_input)
            return 'Float'
        except ValueError:
            # If not a float, check if it's a string
            if isinstance(user_input, str):
                return 'String'
            else:
                # If none of the above, return 'Unknown'
                return 'Unknown'

def main():
    user_input = input("Please enter a value: ")
    data_type = check_data_type(user_input)
    print(data_type)

if __name__ == "__main__":
    main()
```
