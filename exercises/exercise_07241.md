## Exercise
Write a program that takes an array of integers as input, reverses the order of its elements, and then sorts the array in ascending order. The program should output the modified array.

## Reference Solution
```python
#include <bits/stdc++.h>
using namespace std;

int main() {
    int t;
    cin >> t;

    while (t--) {
        int n;
        cin >> n;

        int a[n];
        for (int i = 0; i < n; i++) {
            cin >> a[i];
        }

        reverse(a, a + n);

        sort(a, a + n);

        for (int i = 0; i < n; i++) {
            cout << a[i] << " ";
        }
        cout << endl;
    }

    return 0;
}
```
