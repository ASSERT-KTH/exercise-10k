## Exercise
Declare constant variables to store values that do not change

## Reference Solution
```python
# Using UPPERCASE_WITH_UNDERSCORES convention
PI = 3.14

# Using a decorator
import functools

def constant(f):
    @functools.wraps(f)
    def wrapper(*args, **kwargs):
        if args or kwargs:
            raise TypeError("Constant function doesn't accept arguments")
        return f()
    return wrapper

@constant
def PI():
    return 3.14

# Using a class
class Constants:
    PI = 3.14
```
