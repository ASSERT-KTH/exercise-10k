## Exercise
Create a function that takes an array and an index as input, and returns the element at that index in the array. If the index is out of bounds, return an error message.

## Reference Solution
```python
def get_element(array, index):
    """
    This function returns the element at a specified index in an array.
    
    If the index is out of bounds, it returns an error message.

    Parameters:
    array (list): The input array.
    index (int): The index of the element to be retrieved.

    Returns:
    any: The element at the specified index if it exists, otherwise an error message.
    """

    # First, we check if the input array is actually a list
    if not isinstance(array, list):
        raise TypeError("Input array must be a list")

    # Then, we check if the index is an integer
    if not isinstance(index, int):
        raise TypeError("Index must be an integer")

    # We then check if the index is within the bounds of the array
    # If the index is less than 0 or greater than or equal to the length of the array, it's out of bounds
    if index < 0 or index >= len(array):
        # If the index is out of bounds, we return an error message
        return "Error: Index out of bounds"

    # If the index is within bounds, we return the element at that index
    # We use the index to access the element in the array
    return array[index]
```
