## Exercise
Use a while loop to check if a given string is a palindrome

## Reference Solution
```python
def is_palindrome(s):
    """
    Checks if a given string is a palindrome.

    Args:
        s (str): The input string to check.

    Returns:
        bool: True if the string is a palindrome, False otherwise.
    """
    s = ''.join(c for c in s if c.isalnum()).lower()  # remove non-alphanumeric characters and convert to lowercase
    left = 0
    right = len(s) - 1

    while left < right:
        if s[left] != s[right]:
            return False
        left += 1
        right -= 1

    return True

# Example usage
if __name__ == "__main__":
    strings = ["radar", "hello", "A man, a plan, a canal, Panama"]
    for s in strings:
        print(f"'{s}' is a palindrome: {is_palindrome(s)}")
```
