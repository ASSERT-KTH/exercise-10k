## Exercise
Write a program to format a given date in a specific format (e.g. DD/MM/YYYY).

## Reference Solution
```python
from datetime import datetime

def format_date(date, date_format):
    """
    Format a given date in a specific format.

    Args:
        date (str): The date to be formatted (in YYYY-MM-DD format).
        date_format (str): The desired format (e.g., DD/MM/YYYY).

    Returns:
        str: The formatted date.
    """
    # Parse the input date
    parsed_date = datetime.strptime(date, "%Y-%m-%d")

    # Format the parsed date
    formatted_date = parsed_date.strftime(date_format)

    return formatted_date


# Example usage
date = "2022-09-01"
date_format = "%d/%m/%Y"

formatted_date = format_date(date, date_format)
print(f"Formatted date: {formatted_date}")
```
