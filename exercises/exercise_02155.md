## Exercise
Write a program that reads an HTML file and prints out the file's contents.

## Reference Solution
```python
def read_html_file(file_path):
    """
    Reads an HTML file and returns its contents.

    Args:
        file_path (str): The path to the HTML file.

    Returns:
        str: The contents of the HTML file.
    """
    try:
        with open(file_path, 'r') as file:
            contents = file.read()
            return contents
    except FileNotFoundError:
        print(f"File not found: {file_path}")
        return None

def main():
    file_path = input("Enter the path to the HTML file: ")
    contents = read_html_file(file_path)
    if contents:
        print(contents)

if __name__ == "__main__":
    main()
```
