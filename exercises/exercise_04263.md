## Exercise
Write a function that takes a dictionary and a key-value pair as input and updates the dictionary

## Reference Solution
```python
def update_dictionary(input_dict, key, value):
    """
    Updates a dictionary with a given key-value pair.

    Args:
        input_dict (dict): The dictionary to be updated.
        key (any): The key to be added or updated in the dictionary.
        value (any): The value associated with the key.

    Returns:
        dict: The updated dictionary.
    """
    # Check if the input is a dictionary
    if not isinstance(input_dict, dict):
        raise TypeError("Input must be a dictionary")

    # Update the dictionary with the key-value pair
    input_dict[key] = value

    return input_dict

# Example usage:
if __name__ == "__main__":
    my_dict = {"name": "John", "age": 30}
    print("Original Dictionary:", my_dict)

    updated_dict = update_dictionary(my_dict, "city", "New York")
    print("Updated Dictionary:", updated_dict)
```
