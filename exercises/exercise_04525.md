## Exercise
Write a program that takes an array as input and returns the reversed array.

## Reference Solution
```python
def reverse_array(arr):
    """
    Reverses the input array.

    Parameters:
    arr (list): The input array.

    Returns:
    list: The reversed array.
    """
    return arr[::-1]

# Example usage:
if __name__ == "__main__":
    arr = [1, 2, 3, 4, 5]
    print("Original array:", arr)
    print("Reversed array:", reverse_array(arr))
```
