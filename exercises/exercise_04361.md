## Exercise
Write a program that takes a list of key-value pairs as input and creates a dictionary from it.

## Reference Solution
```python
def create_dict(pairs):
    """
    Create a dictionary from a list of key-value pairs.

    Args:
        pairs (list): A list of tuples, where each tuple is a key-value pair.

    Returns:
        dict: A dictionary created from the input key-value pairs.

    Raises:
        ValueError: If the input list is empty or if a key is repeated.
    """
    if not pairs:
        raise ValueError("Input list is empty")

    result = {}
    for pair in pairs:
        if len(pair) != 2:
            raise ValueError("Invalid key-value pair: {}".format(pair))
        key, value = pair
        if key in result:
            raise ValueError("Duplicate key: {}".format(key))
        result[key] = value

    return result

# Example usage:
pairs = [("name", "John"), ("age", 30), ("city", "New York")]
print(create_dict(pairs))  # Output: {'name': 'John', 'age': 30, 'city': 'New York'}
```
