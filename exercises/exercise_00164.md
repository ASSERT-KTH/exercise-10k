## Exercise
Write a program to perform basic boolean operations (AND, OR, NOT) on two boolean variables.

## Reference Solution
```python
# Define two boolean variables
bool1 = True
bool2 = False

# Perform AND operation
and_result = bool1 and bool2

# Perform OR operation
or_result = bool1 or bool2

# Perform NOT operation on bool1 and bool2
not_bool1 = not bool1
not_bool2 = not bool2

# Print the results
print(f"Boolean 1: {bool1}")
print(f"Boolean 2: {bool2}")
print(f"AND operation: {bool1} AND {bool2} = {and_result}")
print(f"OR operation: {bool1} OR {bool2} = {or_result}")
print(f"NOT operation on Boolean 1: NOT {bool1} = {not_bool1}")
print(f"NOT operation on Boolean 2: NOT {bool2} = {not_bool2}")
```
